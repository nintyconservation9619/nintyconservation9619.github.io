詳解

鍵サイズが 128bit の AES による CMAC を計算するためのクラスです。

詳細
このクラスは以下の状態を持ちます。 インスタンスの生成直後は未初期化状態です。

  • 未初期化状態
  • 処理可能状態
  • 完了状態

構築子と解体子

nn::crypto::Aes128CmacGenerator::Aes128CmacGenerator ( )

デフォルトコンストラクタです。

詳細
生成されるインスタンスは未初期化状態です。

size_t  keySize 
)

鍵サイズが 128 bit の AES による CMAC の計算を初期化します。

引数
[in]pKey鍵データへのポインタ。
[in]keySize鍵データのバイトサイズ。
事前条件

事後条件

詳細
鍵データを入力し、MAC 計算を初期化します。 鍵データは内部にコピーされます。

void nn::crypto::Aes128CmacGenerator::GetMac ( void *  pMac,
size_t  macSize 
)

MAC 計算を完了し、最終的な MAC を取得します。

引数
[out]pMacMAC を格納するバッファへのポインタ。
[in]macSizepMac が指すバッファのバイトサイズ。
事前条件
  • macSize >= MacSize
  • インスタンスは処理可能状態または完了状態である。
事後条件
  • pMac に MAC が書き込まれる。
  • インスタンスは完了状態になる。

詳細
MAC 計算が完了されます。 別の MAC 計算を行う場合は Initialize() を呼んで新たに計算を開始してください。
