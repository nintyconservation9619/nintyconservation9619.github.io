列挙型メンバ詳解

enum nn::g3d::AnimObj::BindFlag
inherited

関連付けに関するフラグです。

列挙値
BindFlag_Enabled 

有効です。

BindFlag_SkipCalculate 

更新処理を無視するが、最後の結果が Apply されます。

BindFlag_SkipApply 

Apply を無視するので、単独では無駄な更新が行われます。

BindFlag_Disabled 

Calculate も Apply もスキップします。

virtual void nn::g3d::ModelAnimObj::BindFast ( const ResModel *  pModel)
pure virtual

アニメーション対象に関連付けます。

引数
[in]pModelアニメーション対象のモデルリソースへのポインター。
戻り値
関連付けが成功の場合は true、失敗の場合は false を返します。

詳細
リソース同士を PreBind によって事前に関連付けることにより、 BindFast 時には名前引きを行わない比較的高速な関連付けを行います。
