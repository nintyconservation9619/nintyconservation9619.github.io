詳解

ユーザーアカウントのプレゼンスを扱うクラスです。

詳細
プレゼンスは、フレンドに自分の情報（オンライン状態や遊んでいるアプリケーション等）をリアルタイムに伝搬する機能です。

本クラスで設定した情報は、フレンドの本体上で FriendPresence を経由して取得されます。

ユーザーはプレゼンスの公開範囲を設定することができ、フレンドに一切の情報を送信しないようにすることも選択できます。
そのため、必ずフレンドにデータが伝わる保証はありません。

プレゼンスの公開範囲は、以下の通りです。

  • 非公開
  • お気に入りフレンドまで
  • フレンド全員

システムがプレゼンスのフィルタリングを行うため、アプリケーションがプレゼンスの公開範囲を気にすることはありません。

自分のプレゼンスの状態の遷移に関しては、 PresenceStatus を参照してください。

オンラインプレイの開始・終了の宣言とプレゼンスの更新を同時に行いたい場合、以下の関数を使用してください。

  • UserPresence::DeclareOpenOnlinePlaySession
  • UserPresence::DeclareCloseOnlinePlaySession

どちらの関数も呼び出さなかった場合、オンラインプレイの開始・終了の宣言は現状を維持します。

nn::Result nn::friends::UserPresence::Initialize ( )

初期化します。（ユーザーアカウント自動選択）

戻り値
処理結果。
事前条件
  • Open 状態のユーザーアカウントが 1 人である。

詳細
本関数は、Open 状態のユーザーアカウントが 1 人の場合のみ、自動的にそのユーザーを選択します。
Open 状態のユーザーアカウントが 1 人以外の場合、ASSERT して失敗します。

ユーザーアカウントを指定しないこと以外は、標準の UserPresence::Initialize と同じです。

nn::Result nn::friends::UserPresence::SetDescription ( const char *  description)

ゲームモード説明文を設定します。

引数
[in]descriptionゲームモード説明文。
戻り値
処理結果。
事前条件
  • description != nullptr

詳細
ゲームモード説明文は、「マイページ」上で表示される文字列です。
フレンドが同じアプリケーションを起動していなくても、アプリケーションで何をしているかをフレンドに知らせることができます。

ゲームモード説明文に文字数の制限はありません。（「マイページ」は、ゲームモード説明文をスクロール表示します）

本関数は、アプリ固有領域に "sys_description" というキーでゲームモード説明文を登録します。
細かい制限に関しては、UserPresence::SetAppValue を参照してください。

const char *  value 
)

アプリ固有情報を設定します。【非推奨】

引数
[in]keyキー。
[in]valueバリュー。
戻り値
処理結果。
事前条件
  • key != nullptr
  • value != nullptr

詳細
本関数の利用は非推奨です。
リリース済みのアプリケーションを最新の SDK でリビルドする等、互換性維持の目的でのみ利用可能です。

アプリ固有情報を設定することで、任意のデータをフレンドに伝搬することができます。

アプリ固有情報の記録領域のサイズは PresenceAppFieldSize です。
1 つのキーバリューに付き、キーの文字数＋バリューのバイト数＋2 バイト（NULL 終端用）のサイズを消費します。

設定できるキーバリューには以下の制限があります。

  • キー
    ASCII の英数字、及び、"_" が利用可能です。
    "sys_" で始まるキーはシステムが予約しているため利用できません。
  • バリュー
    UTF-8 文字列のみ設定可能です。
参照
  • FriendPresence::GetAppValue
void nn::friends::UserPresence::DeclareOpenOnlinePlaySession ( )

オンラインプレイの開始を宣言します。

詳細
本関数は、オンラインプレイの開始の宣言とプレゼンスの更新を同時に行うことができます。
詳しくは、 nn::friends::DeclareOpenOnlinePlaySession を参照してください。

宣言の反映は UserPresence::Commit 呼び出し時です。
UserPresence::DeclareCloseOnlinePlaySession を呼び出していた場合、宣言は上書きされます。
