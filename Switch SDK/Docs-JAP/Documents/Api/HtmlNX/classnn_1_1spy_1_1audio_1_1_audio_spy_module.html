void *  buffer,
size_t  bufferSize 
)

初期化を行います。

詳細
AudioSpyModule が使用するワークメモリをユーザが渡します。 InitializeArg 構造体に初期化に必要なパラメータを設定し、 GetRequiredMemorySize() で必要なメモリサイズを取得してください。

引数
[in]args初期化パラメータです。
[in]bufferAudioSpyModule が使用するワークメモリへのポインタです。
[in]bufferSizeワークメモリのサイズです。
戻り値
成功したら true, 失敗したら false を返します。
事前条件
  • buffer は nn::DefaultAlignment にアラインしている
  • bufferSize >= GetRequiredMemorySize()
  • IsInitialized() == false
  • IsRegistered() == false
  • args.pAudioRendererParameter != nullptr
