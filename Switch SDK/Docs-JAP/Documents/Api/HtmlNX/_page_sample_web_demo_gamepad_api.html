Gamepad API を使ってコントローラー操作を読み取るサンプルオフライン HTML

概要

Gamepad API を使ってコントローラー操作を読み取るオフライン HTML のサンプルです。
本サンプルでは下記の機能を使っています。

  • Gamepad API を使ってコントローラー操作を読み取る
  • 標準のコントローラー操作を無効にする

ファイル構成

本サンプルオフライン HTML は Samples/Sources/Applications/WebDemo/html_document/sample/gamepad_api.htdocs 以下にあります。

必要な環境

本サンプルは NX 実機環境のみサポートしています。

操作方法

コントローラーを操作すると、Gamepad API で読み取った情報が画面に表示されます。
いずれかのボタンを 5 秒以上押し続けると、オフライン Web アプレットを終了して、アプリケーションを経由してメニューへ戻ります。

注意事項

特にありません。

実行手順

WebDemo のサンプルプログラムをビルドし、実行してください。
メニューから本サンプルを選択してください。

解説

メニューから本サンプルを選ぶと、アプリケーションを経由してオフライン Web アプレットを起動し、本サンプルオフライン HTML を表示します。

Gamepad API を使ってコントローラー操作を読み取る

コントローラー操作を Gamepad オブジェクトのプロパティとして読み取り、画面に表示しています。

標準のコントローラー操作を無効にする

下記の方法で標準のコントローラー操作を無効にしています。

  • nn::web::ShowOfflineHtmlPageArg::SetPointerEnabled() を使って、L スティックの操作を空間ナビに限定する。
  • JS 拡張 window.nx.footer.unsetAssign() を使って、標準のコントローラー操作の機能を無効にする。
  • meta 要素の viewport 属性を使って、ズーム機能を無効にする。
