template<typename TTarget >
void nn::gfx::TQueue< TTarget >::Present ( TSwapChain< Target > *  pSwapChain,
int  presentInterval 
)

スワップチェーンの内容をディスプレイに描写します。

引数
[in]pSwapChain対象のスワップチェーンへのポインター
[in]presentInterval画面の垂直同期を同期する数
事前条件
  • キューが初期化されている
  • pSwapChain != NULL
  • pSwapChain が初期化されている
事後条件
  • ディスプレイに描写するコマンドが提出されている

詳細
presentInterval に 0 を指定した場合はディスプレイの垂直同期を同期しません。
低レベルグラフィックス API などに依存してこの呼び出しは垂直同期をとるまで処理を返さない場合があります。 CPU 上で同期することを確実にしたい場合はこの後に Sync を呼び出してください。
