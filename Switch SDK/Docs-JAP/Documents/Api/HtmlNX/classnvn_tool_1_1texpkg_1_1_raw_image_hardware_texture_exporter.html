) const

Get HW texture swizzled data size.

This size may be bigger than the pitch linear image size. See NVN programming guide for details on the block format.

引数
[in]targetThe texture target.
[in]widthWidth of the texture.
[in]heightHeight of the texture.
[in]depthDepth of the texture.
[in]formatTexel format of the texture.
[in]mipLevelsNumber of mipmaps the texture contains.
[in]subimagesNumber of subimages (layers) the texture contains.
[in]flagsHardware texture storage flags. See RawImageHardwareTextureFlagsEnum.
戻り値
Swizzles texture data size in bytes, or 0 if failure.
