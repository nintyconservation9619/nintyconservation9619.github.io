nn::os::SemaphoreType 構造体

セマフォオブジェクトを定義する構造体です。 [詳解]

#include <nn/os/os_SemaphoreTypes.h>

詳解

セマフォオブジェクトを定義する構造体です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • InitializeSemaphore()
  • FinalizeSemaphore()
  • AcquireSemaphore()
  • TryAcquireSemaphore()
  • TimedAcquireSemaphore()
  • ReleaseSemaphore(SemaphoreType* semaphore)
  • ReleaseSemaphore(SemaphoreType* semaphore, int count)
  • GetCurrentSemaphoreCount()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, SemaphoreType* semaphore)

上記の関数を利用するには nn/os.h をインクルードする必要があります。

nn::os::Semaphore ユーティリティクラスもあわせてご確認ください。

各種例:
OsMultiWait.cpp, OsMultiWaitUtility.cpp, OsSemaphore.cpp.
