列挙型メンバ詳解

enum nn::atk::viewer::SoundController::State

サウンドの状態を示します。

列挙値
State_Disabled 

正しいサウンドIDが設定されていない等、無効な状態です。

State_Stopped 

停止中です。

State_Playing 

再生中です。

State_Paused 

一時停止中です。
