virtual void nn::spy::SpyModule::OnRequested ( bool  isRequested)
protectedvirtual

内部用機能のため使用禁止です。

Spy.exe からのデータ要求を通知します。

Spy.exe からデータ要求が来たときに、各データインフォに、 データ要求の有無が通知されます。

セッションの終了時には isRequested = false で呼び出されます。

導出クラスでは、最初に基底クラスの実装を呼びだしてください。

引数
[in]isRequestedデータが要求されているときは true が指定されます。
