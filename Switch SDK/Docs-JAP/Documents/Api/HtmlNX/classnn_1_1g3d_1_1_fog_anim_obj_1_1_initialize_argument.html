詳解

内部用機能のため使用禁止です。 FogAnimObj::Initialize() に渡して初期化を行うパラメータです。

構築子と解体子

nn::g3d::FogAnimObj::InitializeArgument::InitializeArgument ( )

コンストラクタです。

インスタンスで使用するすべてのフォグアニメーションリソースに対して Reserve() するか、 SetMaxCurveCount() でバッファーサイズの計算に必要なパラメータを指定する必要があります。
