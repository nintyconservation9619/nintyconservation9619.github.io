詳解

MD5 ハッシュを計算するクラスです。

詳細
このクラスは以下の状態を持ちます。 インスタンスの生成直後は未初期化状態です。

  • 未初期化状態
  • 処理可能状態
  • 完了状態
覚え書き
MD5 は脆弱性が指摘されており、過去システムとの互換性以外の用途では非推奨です。

void nn::crypto::Md5Generator::GetHash ( void *  pHash,
size_t  hashSize 
)

ハッシュ計算を完了し、最終的なハッシュ値を取得します。

引数
[out]pHashハッシュ値を格納するバッファへのポインタ。
[in]hashSizepHash が指すバッファのバイトサイズ。
事前条件
  • hashSize >= HashSize
  • インスタンスは処理可能状態または完了状態である。
事後条件
  • pHash に計算されたハッシュ値が書き込まれる。
  • インスタンスは完了状態になる。

詳細
ハッシュ計算が完了されます。 別のハッシュ計算を行う場合は Initialize() を呼んで新たに計算を開始してください。
