typedef TextCheckResult( * nn::swkbd::TextCheckCallback) (void *pOutDialogTextBuf, size_t *pOutDialogTextLengthSize, String *pStr)

入力結果の文字列を確認するコールバック関数の型です。
pStr に格納された入力結果文字列を確認し、 必要に応じた nn::swkbd::TextCheckResult を返すようにしてください。
ダイアログの表示を行う場合は、pOutDialogTextBuf に対してダイアログに表示する文字列を指定してください。

引数
[out]pOutDialogTextBufダイアログに表示する文字列用バッファです。
nn::swkbd::ShowKeyboardArg::textCheckWorkBuf で指定したバッファを使用しています。
nn::swkbd::KeyboardConfig::isUseUtf8 で設定した文字コードで入力し、 末尾にヌル終端をつけてください。
入力した文字数は pOutDialogTextLengthSize と等しくなるようにしてください。
[out]pOutDialogTextLengthSizeダイアログに表示する文字数です。
nn::swkbd::DialogTextMaxLength 以下になるようにしてください。
[in]pStr入力結果文字列です。

enum nn::swkbd::KeyboardMode

ソフトウェアキーボードの盤面の種類です。

詳細
入力する内容に対応したモードを指定することで、適切な配列のキーボードを使用することができます。

  • フルキーボードモード ( Full )
    • 各リージョンで指定できるすべてのキーボードを出すことができます。
    • キーボードの種類を限定したい場合は文字範囲の制限を加えてください。

  • 数字キーボードモード ( Numeric )
    • 数字入力に特化したキーボードが利用できます。

  • ASCII キーボードモード ( ASCII )
    • ASCII のみ入力できるキーボードが利用できます。

  • フルキーボード(ラテン文字入力優先)モード ( FullLatin )
    • 選択できるキーボードの種類は「フルキーボードモード」と同じです。
    • 「最後に起動したキーボード」によらず常に半角英数字用のキーボードが初めに表示されます。
    • 終了時のキーボードが「最後に起動したキーボード」として記録されることもありません。
    • 予測変換の学習は行われます。
列挙値
KeyboardMode_Full 

フルキーボードモードです。

KeyboardMode_Numeric 

数字キーボードモードです。

KeyboardMode_ASCII 

ASCII キーボードモードです。

KeyboardMode_FullLatin 

フルキーボード(ラテン文字入力優先)モードです。

KeyboardMode_Max 

列挙子の総数です。

関数詳解

nn::Result nn::swkbd::ShowKeyboard ( String *  pOutResultString,
const ShowKeyboardArg &  showKeyboardArg 
)

ソフトウェアキーボードを表示します。

引数
[out]pOutResultString入力結果文字列です。
キャンセル時はバイトサイズが 0 で返ります。
[in]showKeyboardArg表示時に必要な設定パラメータです。
戻り値
ソフトウェアキーボード表示結果を返します。 特に問題なければ ResultSuccess を返します。

詳細
ソフトウェアキーボードが完了するまで、この関数の呼び出し先はブロックされます。
また、エラービューアなどシステムによる UI が表示されている間に本関数を呼び出した場合も、起動中の UI が終了するまで本関数は内部で待機し、
先に起動中の UI が終了すると待機が解除されます。

showKeyboardArg の ShowKeyboardArg::workBuf, ShowKeyboardArg::workBufSize など事前に設定済みである必要があります。

各種例:
SwkbdSimple.cpp.
void nn::swkbd::InitializeKeyboardConfig ( KeyboardConfig *  pOutKeyboardConfig)

ソフトウェアキーボード起動用に、内部的に持っている設定値を初期化します。

詳細
nn::swkbd::SetUserWordList() 関数など、関数経由でパラメータを設定する際、 内部的に保持しているソフトウェアキーボード起動用の設定値を書き換えています。
そのため、ソフトウェアキーボード表示用のパラメータを書き換える際は、 まず初めに nn::swkbd::InitializeKeyboardConfig() 関数を呼び出してください。
