nn::web::ShowOfflineHtmlPageArg::ShowOfflineHtmlPageArg ( const nn::ApplicationId &  id,
const char *  pDocumentPath 
)
explicit

(削除予定) オフライン HTML を開く為の最低限の情報を持つクラスを作成します。

このAPIの利用は非推奨です。代替APIや利用条件の有無については詳細を確認してください。

非推奨:
本 API は削除予定です。 代わりに ShowOfflineHtmlPageArg(const char* pDocumentPath) を使用してください。
引数
[in]idApplicationId を指定してください。
[in]pDocumentPath開きたいオフライン HTML のパスを指定してください。

詳細
引数で指定した ApplicationId は使用されず、起動元の ApplicationId が使われます。 pDocumentPathは NULL 終端されている必要があります。 DocumentPathMaxLength -1 文字以下にする必要があります。
