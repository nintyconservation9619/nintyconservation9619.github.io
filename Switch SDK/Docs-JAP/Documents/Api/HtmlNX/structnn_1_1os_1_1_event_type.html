nn::os::EventType 構造体

イベントオブジェクトを定義する構造体です。 [詳解]

#include <nn/os/os_EventTypes.h>

詳解

イベントオブジェクトを定義する構造体です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • InitializeEvent()
  • FinalizeEvent()
  • SignalEvent()
  • WaitEvent()
  • TryWaitEvent()
  • TimedWaitEvent()
  • ClearEvent()
  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, EventType *event)

上記の関数を利用するには nn/os.h をインクルードする必要があります。

nn::os::Event ユーティリティクラスもあわせてご確認ください。

各種例:
Demo1/Demo1.cpp, LdnAccessPoint.cpp, LdnStation.cpp, MoviePlayerSimple.cpp, OsEvent.cpp, OsMultiWait.cpp, OsMultiWaitUtility.cpp, Parallel.cpp, PerfSimple.cpp.
