メンバ詳解

int32_t GLSLCprogramOutputInfo::location

Location this output is assigned to.

-1 if not a user-assigned or a vertex/fragment input/output.

int32_t GLSLCprogramOutputInfo::locationNdx

Identifies the fragment color index of the output variable.

If this is for a non-fragment shader, -1 is written.
