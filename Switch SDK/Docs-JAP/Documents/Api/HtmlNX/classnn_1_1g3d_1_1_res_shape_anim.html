BindResult nn::g3d::ResShapeAnim::PreBind ( const ResModel *  pModel)

事前にモデルと関連付けを行います。

引数
[in]pModelモデルへのポインター。
戻り値
BindResult を返します。

詳細
リソース同士を PreBind() によって事前に関連付けることにより、 ShapeAnimObj::BindFast 時には名前引きを行わない比較的高速な関連付けを行います。

戻り値

詳細
