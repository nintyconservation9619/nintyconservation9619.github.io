詳解

テクスチャービューを初期化するための情報を表すクラスです。

各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, MiiCompressedTexture.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiTransparentAdjuster.cpp.

構築子と解体子

nn::gfx::TextureViewInfo::TextureViewInfo ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解

void nn::gfx::TextureViewInfo::SetDefault ( )

各パラメーターを既定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetImageDimension( nn::gfx::ImageDimension_2d );
  • SetDepthStencilTextureMode( nn::gfx::DepthStencilFetchMode_DepthComponent );
  • SetImageFormat( nn::gfx::ImageFormat_Undefined );
  • EditSubresourceRange().SetDefault();
  • SetChannelMapping( nn::gfx::ChannelMapping_Red, nn::gfx::ChannelMapping_Green, nn::gfx::ChannelMapping_Blue, nn::gfx::ChannelMapping_Alpha );
  • SetTexturePtr( NULL );
各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, MiiCompressedTexture.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiTransparentAdjuster.cpp.
