詳解

Biquad フィルタを表す抽象クラスです。

BiquadFilterCallback クラスから継承し、 ユーザー側で Biquad フィルタを実装することができます。

BiquadFilterCallback の使用方法については、 SoundSystem::SetBiquadFilterCallback 関数を参照してください。

Biquad フィルタの係数を返します。

nn::atk ライブラリ内部の Biquad フィルタ係数が必要なタイミングで呼び出されます。 BiquadFilterCallback を継承したクラスで、 この関数にフィルタ係数を実装する必要があります。

type は、コールバッククラスを SoundSystem::SetBiquadFilterCallback で登録したときのタイプ番号です。 BiquadFilterType に記述されている BIQUAD_FILTER_TYPE_USER_MIN ～ BIQUAD_FILTER_TYPE_USER_MAX までの値が入ります。

value はフィルタのかかり具合を表す 0.0f ～ 1.0f の値です。 この値に応じたフィルタ係数を coef に格納するように実装します。

引数
[out]pOutValue計算したフィルタ係数を格納するための構造体です。
[in]typeフィルタの種類です (BIQUAD_FILTER_TYPE_USER_MIN ～BIQUAD_FILTER_TYPE_USER_MAX)。
[in]valueフィルタのかかり具合です (0.0f ～ 1.0f)
