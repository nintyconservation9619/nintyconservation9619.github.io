詳解

コンストラクタとデストラクタで自動的に計測の開始・終了を行うクラスです。

事前条件
LoadMeterCenter が初期化されている。

詳細
LoadMeterCenter が保持する、インデクス指定したユーザー計測用の CpuMeter で計測します。
