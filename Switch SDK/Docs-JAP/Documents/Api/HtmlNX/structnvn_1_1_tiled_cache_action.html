詳解

Used in TiledCacheDecideAction to suggest whether the tiled cache should be flushed or enabled/disabled.

列挙型メンバ詳解

enum nvn::TiledCacheAction::Enum

Enumerated values for the type TiledCacheAction.

列挙値
ENABLE 

Tiled caching should be enabled so subsequent work can be binned.

DISABLE 

Tiled caching should be disabled so subsequent work is not binned.

If tiled caching is enabled, this action will first flush the binner.

FLUSH 

Tiled cache binner is flushed with tiling and subsequent work is binned in a new render pass.

FLUSH_NO_TILING 

Tiled cache binner is flushed as untiled and subsequent work is binned in a new render pass.

ENABLE_RENDER_TARGET_BINNING 

Enables subsequent binning of render target state changes.

If tiled caching is enabled, this action will first flush the binner.

DISABLE_RENDER_TARGET_BINNING 

Disables subsequent binning of render target state changes.

If tiled caching is enabled, this action will first flush the binner.
