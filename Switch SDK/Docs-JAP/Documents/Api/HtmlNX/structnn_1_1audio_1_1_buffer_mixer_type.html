nn::audio::BufferMixerType 構造体

BufferMixer エフェクトです。 [詳解]

#include <nn/audio/audio_EffectTypes.h>

詳解

BufferMixer エフェクトです。

詳細
BufferMixer エフェクトは、SubMix/FinalMix が管理するミックスバッファ内でのミックスを行います。
入力に指定したミックスバッファの内容を、出力に指定したミックスバッファにミックスします。
このミックス処理における入出力関係は nn::audio::SetBufferMixerInputOutput() で設定されます。
またミックス時にはボリューム操作も同時に施します。この入出力関係それぞれのミックス処理で用いられるボリュームは nn::audio::SetBufferMixerVolume() で指定します。
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • AddBufferMixer(AudioRendererConfig* pOutConfig, BufferMixerType* pBufferMixer, SubMixType* pSubMix)
  • AddBufferMixer(AudioRendererConfig* pOutConfig, BufferMixerType* pBufferMixer, FinalMixType* pFinalMix)
  • RemoveBufferMixer(AudioRendererConfig* pOutConfig, BufferMixerType* pBufferMixer, SubMixType* pSubMix)
  • RemoveBufferMixer(AudioRendererConfig* pOutConfig, BufferMixerType* pBufferMixer, FinalMixType* pFinalMix)
  • GetBufferMixerInputOutput(const BufferMixerType* pBufferMixer, int8_t* outInput, int8_t* outOutput, int* pOutCount, int count)
  • SetBufferMixerInputOutput(BufferMixerType* pBufferMixer, const int8_t* input, const int8_t* output, int count)
  • GetBufferMixerVolume(const BufferMixerType* pBufferMixer, int index)
  • SetBufferMixerVolume(BufferMixerType* pBufferMixer, int index, float volume)
  • IsBufferMixerEnabled(const BufferMixerType* pBufferMixer)
  • SetBufferMixerEnabled(BufferMixerType* pBufferMixer, bool enable)
  • IsBufferMixerRemovable(const BufferMixerType* pBufferMixer)

上記の関数を利用するには nn/audio.h をインクルードする必要があります。

各種例:
AudioEffect.cpp, AudioEffectAuxUtility.cpp, AudioPerformanceMetrics.cpp, AudioRenderer.cpp.
