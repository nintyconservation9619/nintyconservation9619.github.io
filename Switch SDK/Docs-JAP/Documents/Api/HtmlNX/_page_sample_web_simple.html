nn::web ライブラリを使って Web アプレットを起動し、Web ページを開くシンプルなサンプルプログラム

概要

nn::web ライブラリを使って Web アプレットを起動し、Web ページを開くシンプルなサンプルプログラムです。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/WebSimple 以下にあります。

必要な環境

本サンプルプログラムは NX 実機環境のみサポートしています。

操作方法

画面上に操作方法が表示されます。

入力 動作
A ボタン 指定した Web ページを開く
B ボタン 指定ページに到達したら自動で Web アプレットを終了するように設定して Web ページを開く
X ボタン 起動時の画面と Web ページの背景を指定して Web ページを開く
Y ボタン 動画専用起動で Web アプレットを起動して動画コンテンツを再生する
+ ボタン サンプルアプリケーションを終了する

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

125 void UseCallbackUrl() NN_NOEXCEPT
126 {
127  // Web アプレット起動時に開く Web ページの URL
128  static const char* requestUrl =
129  "https://nintendo-sdk-sample.cdn.nintendo.net/page/callback_url_sample/index.html";
130  // Callback URL
131  static const char* callbackUrl =
132  "https://nintendo-sdk-sample.cdn.nintendo.net/page/callback_url_sample/callback_url/index.html";
133  // Callback 可能 URL
134  static const char* callbackableUrl =
135  "https://nintendo-sdk-sample.cdn.nintendo.net/page/callback_url_sample/callbackable_url/index.html";
136 
137  nn::web::ShowWebPageArg showWebPageArg(requestUrl);
138  // Callback URL を指定した場合は、この URL に Web アプレットが到達するとウェブページの表示を終了します。
139  showWebPageArg.SetCallbackUrl(callbackUrl);
140  // Callback 可能 URL を指定した場合は、この URL から Callback URL に Web アプレットが到達した時のみウェブページの表示を終了します。
141  showWebPageArg.SetCallbackableUrl(callbackableUrl);
142 
143  nn::web::WebPageReturnValue webPageReturnValue;
144  const nn::Result result = nn::web::ShowWebPage(&webPageReturnValue, showWebPageArg);
145  NN_ABORT_UNLESS(result.IsSuccess());
146 
147  nn::web::WebExitReason exitReason = webPageReturnValue.GetWebExitReason();
148  NN_LOG("Exit Reason %d\n", exitReason);
149  if( exitReason == nn::web::WebExitReason_CallbackUrlReached )
150  {
151  if( webPageReturnValue.GetLastUrlSize() > 0 )
152  {
153  NN_LOG("LastUrl %s\n", webPageReturnValue.GetLastUrl());
154  }
155  }
156 }

サンプルプログラムの解説

先のサンプルプログラムの主な処理の流れは以下の通りです。

  • 描画のための初期化処理と背景色の設定
  • メインループ
    • キー入力処理の更新
    • 文字の表示処理
    • 特定のキーの入力があれば、Web アプレットを起動して Web ページを表示

アクセス可能な URL の指定

Web アプレットが開くことができる URL は、accessible-urls.txt という テキストファイルを ROM に付帯情報として含めて指定する必要があります。 本サンプルでは、任天堂の Web サイトと サンプル Web ページが置かれた Web サイトのみを閲覧可能と設定しています。
