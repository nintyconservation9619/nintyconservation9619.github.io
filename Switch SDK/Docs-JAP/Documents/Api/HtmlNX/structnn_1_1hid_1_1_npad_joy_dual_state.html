nn::hid::NpadJoyDualState 構造体

Npad の Joy-Con 2本持ちをモデルとした操作スタイル (NpadStyleJoyDual) の入力状態を表す構造体です。 [詳解]

#include <nn/hid/hid_NpadJoyDual.h>

公開変数類

 Npad の左アナログスティックの状態です。
 
AnalogStickState analogStickR
 Npad の右アナログスティックの状態です。
 
NpadAttributesSet attributes
 Npad の状態を表す属性値です。

 Npad のデジタルボタンの状態です。
 
int64_t samplingNumber
 Npad の入力状態が更新される度に増加する値です。
 

詳解

Npad の Joy-Con 2本持ちをモデルとした操作スタイル (NpadStyleJoyDual) の入力状態を表す構造体です。

詳細
Joy-Con 2本持ち操作で扱えるデジタルボタン ( 方向ボタン, A, B, X, Y, L, R, ZL, ZR, Joy-Con (L) SL, Joy-Con (L) SR, Joy-Con (R) SL, Joy-Con (R) SR, Plus, Minus ) と 左右アナログスティックの入力状態を提供します。 左右の Joy-Con のいずれかが接続されると、attributes::IsConnected のビットがセットされます。 左右の個別の接続状態は attributes::IsLeftConnected と attributes::IsRightConnected で取得できます。

方向ボタンは禁則処理が行われません

各種例:
HidNpadSimple.cpp.
