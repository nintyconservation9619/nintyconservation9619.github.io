詳解

エフェクトシステム生成時に設定するコンフィグです。

nn::vfx::Config::SetEffectHeap(), nn::vfx::Config::SetEffectDynamicHeap() でのヒープの指定はシステムの起動に必須になります。 必要に応じて、適切にバッファサイズを設定してください。 詳しくは、エフェクト チュートリアル ドキュメントをご覧ください。

参照
nn::vfx::Config::SetEffectHeap(), nn::vfx::Config::SetEffectDynamicHeap()
各種例:
VfxBinarySegmentation/main.cpp, VfxCustomShader/main.cpp, VfxManualEmitterSet/main.cpp, VfxSimple/main.cpp.
