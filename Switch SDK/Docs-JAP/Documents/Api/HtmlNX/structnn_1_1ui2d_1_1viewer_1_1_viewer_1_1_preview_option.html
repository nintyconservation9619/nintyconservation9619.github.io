メンバ詳解

char nn::ui2d::viewer::Viewer::PreviewOption::path[PathLengthMax]

レイアウト・バイナリリソースのパスを指定します。

実機の場合は、アーカイブのFSにおけるパスを指定します。

PCの場合は、バイナリリソースの出力されたディレクトリの、 PCファイルシステムにおけるパスを指定します。
