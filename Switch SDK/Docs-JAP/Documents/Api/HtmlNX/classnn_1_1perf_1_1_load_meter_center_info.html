FrameworkDemo/Main.cpp, GfxPrimitiveRenderer.cpp, Parallel.cpp, PerfSimple.cpp.

関数詳解

void nn::perf::LoadMeterCenterInfo::SetCoreCount ( int  coreCount)

現在の環境でアプリケーションが動作する CPU コア数を設定します。

引数
[in]coreCountCPU コア数です。

詳細
設定した個数分のコア別に計測を行う CpuMeter が作成されます。
正常な GPU 計測を行うためにも CPU コア数の設定が必要です。

各種例:
FrameworkDemo/Main.cpp, GfxPrimitiveRenderer.cpp, Parallel.cpp, PerfSimple.cpp.

void nn::perf::LoadMeterCenterInfo::SetCpuBufferCount ( int  bufferCount)

CpuMeter の計測結果保存用のバッファの数を指定します。何フレーム前までの計測結果を保持するかを決定します。

引数
[in]bufferCount計測結果を保存する配列バッファの数です。ダブルバッファとなる 2以上を指定する必要があります。
