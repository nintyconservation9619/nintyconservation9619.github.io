詳解

深度ステンシルステートを初期化するための情報を表すクラスです。

各種例:
GfxShaderCompile.cpp, GfxSimple.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計17項目).

構築子と解体子

nn::gfx::DepthStencilStateInfo::DepthStencilStateInfo ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解

void nn::gfx::DepthStencilStateInfo::SetDefault ( )

各パラメーターを既定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetDepthComparisonFunction( nn::gfx::ComparisonFunction_Less );
  • SetDepthTestEnabled( false );
  • SetDepthWriteEnabled( false );
  • SetStencilTestEnabled( false );
  • SetDepthBoundsTestEnabled( false );
  • SetStencilReadMask( static_cast< uint8_t >( ~0 ) );
  • SetStencilWriteMask( static_cast< uint8_t >( ~0 ) );
  • EditFrontStencilStateInfo().SetDefault();
  • EditBackStencilStateInfo().SetDefault();
各種例:
GfxShaderCompile.cpp, GfxSimple.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計17項目).
