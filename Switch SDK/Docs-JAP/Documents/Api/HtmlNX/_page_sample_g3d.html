G3d

3D ライブラリのサンプル解説です。

はじめに

ここでは、3D ライブラリ(g3d) のサンプルについて解説します。
g3d の機能は nn::g3d の名前空間にあります。

サンプル

nn::g3d のサンプルコードは Samples/Sources/Applications/G3dDemo に配置してあります。 実行するとランチャーメニューが表示され、実機の場合Joy-Con、DebugPad、タッチスクリーン、 PCの場合キーボード操作によってデモを選択、実行できます。

入力 動作
十字ボタン上下またはカーソルキー上下 デモを選択します。
A ボタンまたは A キー デモを実行します。

実機の場合タッチスクリーン、PCの場合マウス操作によってデモの選択、実行も可能です。

nn::g3d::mii を使用したサンプルコードは Samples/Sources/Applications 以下の
MiiG3d で始まるディレクトリに配置してあります。 サンプルを以下に一覧します。

ページ 内容
G3dDemo Simple サンプルプログラム Simple の解説です。g3d を使用したモデル描画のコード例を示します。
G3dDemo NnsG3dSimple サンプルプログラム NnsG3dSimple の解説です。 nns::g3d を使用し、モデル表示を行います。
G3dDemo Edit サンプルプログラム Edit の解説です。ビューアーライブラリーを使用して 3DEditor によるリアルタイム編集を行います。
G3dDemo SkeletalAnimation サンプルプログラム SkeletalAnimation の解説です。
G3dDemo ShapeAnimation サンプルプログラム ShapeAnimation の解説です。シェイプアニメーションを使用したモデル表示をします。
G3dDemo Outline サンプルプログラム Outline の解説です。ジオメトリシェーダーを使用し、モデルのアウトライン表示を行います。
G3dDemo Parallel サンプルプログラム Parallel の解説です。並列処理を使用したモデル更新をします。
G3dDemo Town サンプルプログラム Town の解説です。実際のゲームに近い内容のサンプルプログラムです。
MiiG3dSimple nn::g3d::ResModel への Mii モデルアサイン機能のサンプルプログラムの解説です。
MiiG3dHeadwear G3d で Mii に帽子をかぶせるサンプルプログラムの解説です。

注意点

リソースのコンバートには5分から10分ほどの時間がかかります。コンバート中はVisual Studio上にログなどが出力されませんのでご留意ください。ログはコンバート完了後に出力されます。
