プラットフォーム情報マクロ

マクロ説明
NN_BUILD_APISET_GENERIC 汎用 API セットを使用している場合に定義されます。
NN_BUILD_APISET_NX NX API セットを使用している場合に定義されます。
NN_BUILD_TARGET_PLATFORM_WIN アプリケーションが動作するプラットフォームが Windows の場合に定義されます。
NN_BUILD_TARGET_PLATFORM_NX アプリケーションが動作するプラットフォームが NX 実機環境の場合に定義されます。
NN_BUILD_TARGET_PLATFORM_OS_WIN アプリケーションが動作するプラットフォームにおいて OS が Windows である場合に定義されます。
NN_BUILD_TARGET_PLATFORM_OS_NN アプリケーションが動作するプラットフォームにおいて OS が NX 実機環境で使用されている OS である場合に定義されます。
NN_BUILD_TARGET_PLATFORM_ADDRESS_32 アプリケーションが動作するプラットフォームにおいてアドレスが 32 ビット幅の場合に定義されます。
NN_BUILD_TARGET_PLATFORM_ADDRESS_64 アプリケーションが動作するプラットフォームにおいてアドレスが 64 ビット幅の場合に定義されます。
NN_BUILD_TARGET_PLATFORM_ENDIAN_BIG アプリケーションが動作するプラットフォームにおいてエンディアンがビッグエンディアンの場合に定義されます。
NN_BUILD_TARGET_PLATFORM_ENDIAN_LITTLE アプリケーションが動作するプラットフォームにおいてエンディアンがリトルエンディアンの場合に定義されます。

アサート

マクロ説明
NN_ENABLE_ASSERT アサートを有効化します。NN_DISABLE_ASSERT と同時に定義することはできません。
NN_DISABLE_ASSERT アサートを無効化します。NN_ENABLE_ASSERT と同時に定義することはできません。
NN_ASSERT (condition, ...) 条件を満たしていることをテストします。
NN_ASSERT_EQUAL (lhs, rhs) 値が等しいことをテストします。 満たされるべき条件は lhs == rhs です。
NN_ASSERT_NOT_EQUAL (lhs, rhs) 値が等しくないことをテストします。 満たされるべき条件は lhs != rhs です。
NN_ASSERT_NOT_NULL (pointer) ポインタがヌルポインタではないことをテストします。 満たされるべき条件は pointer != nullptr です。
NN_ASSERT_LESS (lhs, rhs) 左辺が右辺より小さいことをテストします。 満たされるべき条件は lhs < rhs です。
NN_ASSERT_LESS_EQUAL (lhs, rhs) 左辺が右辺より小さいか、または値が等しいことをテストします。 満たされるべき条件は lhs <= rhs です。
NN_ASSERT_GREATER (lhs, rhs) 左辺が右辺より大きいことをテストします。 満たされるべき条件は lhs > rhs です。
NN_ASSERT_GREATER_EQUAL (lhs, rhs) 左辺が右辺より大きいか、または値が等しいことをテストします。 満たされるべき条件は lhs >= rhs です。
NN_ASSERT_STRING_EQUAL (lhs, rhs) C文字列の内容が等しいことをテストします。 満たされるべき条件は strcmp(lhs, rhs) == 0 です。
NN_ASSERT_STRING_NOT_EQUAL (lhs, rhs) C文字列の内容が等しくないことをテストします。 満たされるべき条件は strcmp(lhs, rhs) != 0 です。
NN_ASSERT_ALIGNED (pointer, align) 与えられたアドレス・ポインタのアラインメントをテストします。 満たされるべき条件は pointer & (align - 1) == 0 です。
NN_ASSERT_RANGE (value, begin, end) 値が範囲に含まれていることをテストします。 満たされるべき条件は (begin <= value) && (value < end) です。
NN_ASSERT_MINMAX (value, min, max) 値が最小値以上かつ最大値以下であることをテストします。 満たされるべき条件は (min <= value) && (value <= max) です。

コンパイル時アサート

マクロ説明
NN_STATIC_ASSERT (condition) コンパイル時に条件式が成立することをテストします。

アボート

マクロ説明
NN_ABORT (...) 実行を停止します。このマクロは常に有効です。
NN_ABORT_UNLESS (condition, ...) 条件を満たしていることをテストし、満たされていない場合に停止します。このマクロは常に有効です。
NN_ABORT_UNLESS_EQUAL (lhs, rhs) 値が等しいことをテストし、そうではない場合に停止します。 満たされるべき条件は lhs == rhs です。 このマクロは常に有効です。
NN_ABORT_UNLESS_NOT_EQUAL (lhs, rhs) 値が等しくないことをテストし、そうではない場合に停止します。 満たされるべき条件は lhs != rhs です。 このマクロは常に有効です。
NN_ABORT_UNLESS_NOT_NULL (pointer) ポインタがヌルポインタではないことをテストし、そうではない場合に停止します。 満たされるべき条件は pointer != nullptr です。 このマクロは常に有効です。
NN_ABORT_UNLESS_LESS (lhs, rhs) 左辺が右辺より小さいことをテストします。 満たされるべき条件は lhs < rhs です。
NN_ABORT_UNLESS_LESS_EQUAL (lhs, rhs) 左辺が右辺より小さいか、または値が等しいことをテストします。 満たされるべき条件は lhs <= rhs です。
NN_ABORT_UNLESS_GREATER (lhs, rhs) 左辺が右辺より大きいことをテストします。 満たされるべき条件は lhs > rhs です。
NN_ABORT_UNLESS_GREATER_EQUAL (lhs, rhs) 左辺が右辺より大きいか、または値が等しいことをテストします。 満たされるべき条件は lhs >= rhs です。
NN_ABORT_UNLESS_STRING_EQUAL (lhs, rhs) C文字列の内容が等しいことをテストします。 満たされるべき条件は strcmp(lhs, rhs) == 0 です。
NN_ABORT_UNLESS_STRING_NOT_EQUAL (lhs, rhs) C文字列の内容が等しくないことをテストします。 満たされるべき条件は strcmp(lhs, rhs) != 0 です。
NN_ABORT_UNLESS_ALIGNED (pointer, align) 与えられたアドレス・ポインタのアラインメントをテストします。 満たされるべき条件は pointer & (align - 1) == 0 です。
NN_ABORT_UNLESS_RANGE (value, begin, end) 値が範囲に含まれていることをテストし、そうではない場合に停止します。 満たされるべき条件は (begin <= value) && (value < end) です。 このマクロは常に有効です。
NN_ABORT_UNLESS_MINMAX (value, min, max) 値が最小値以上かつ最大値以下であることをテストし、そうではない場合に停止します。 満たされるべき条件は (min <= value) && (value <= max) です。 このマクロは常に有効です。
NN_ABORT_UNLESS_RESULT_SUCCESS (result) 指定した nn::Result に対して成功であるかどうかをテストし、そうではない場合に停止します。 満たされるべき条件は result.IsSuccess() です。 このマクロは常に有効です。

ログ出力

マクロ説明
NN_ENABLE_LOG ログ出力を有効化します。NN_DISABLE_LOG と同時に定義することはできません。
NN_DISABLE_LOG ログ出力を無効化します。NN_ENABLE_LOG と同時に定義することはできません。
NN_LOG (...) 引数に指定された文字列を、ログとして出力します。
NN_VLOG (format, args) 引数に指定された文字列を、ログとして出力します。引数を可変長引数リストで与えることができます。
NN_PUT (text, length) 引数に指定された文字列を、ログとして出力します。出力する文字列のバイトサイズを引数で指定します。

ブレーク

マクロ説明
NN_DIAG_BREAK () 呼び出した箇所に、ブレークポイント命令を挿入します。

os ライブラリの機能

マクロ説明
NN_OS_ALIGNAS_THREAD_STACK スタック領域のアライメントを指定するマクロです。
NN_OS_ALIGNAS_FIBER_STACK ガードなしファイバスタック領域のアライメントを指定するマクロです。
NN_OS_ALIGNAS_GUARDED_STACK ガード付きスタック領域のアライメントを指定するマクロです。
NN_OS_MUTEX_INITIALIZER (recursive) nn::os::MutexType オブジェクトを静的に初期化します。
NN_OS_CONDITION_VARIABLE_INITIALIZER () nn::os::ConditionVariableType オブジェクトを静的に初期化します。

htc, htcs ライブラリの機能

マクロ説明
NN_ENABLE_HTC htc, htcs ライブラリを有効化します。NN_DISABLE_HTC と同時に定義することはできません。
NN_DISABLE_HTC htc, htcs ライブラリを無効化します。NN_ENABLE_HTC と同時に定義することはできません。

spy ライブラリの機能

マクロ説明
NN_ENABLE_SPY spy_Config.h の中で定義されると spy ライブラリが有効になります。デフォルトでは Debug と Develop ビルドで定義されます。このマクロを定義して spy ライブラリを有効化する際には、htc ライブラリが有効化されている必要があります。
NN_BUILD_CONFIG_SPY_ENABLED spy ライブラリが有効なときに定義されます。
