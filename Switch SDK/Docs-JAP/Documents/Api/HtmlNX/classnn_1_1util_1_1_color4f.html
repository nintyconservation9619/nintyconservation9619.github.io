各種例:
MiiHeadwearExample.cpp, MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiTransparentAdjuster.cpp.
