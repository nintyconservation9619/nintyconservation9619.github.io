nn::atk::SoundStartable::StartInfo::StreamSoundInfo 構造体

ストリームサウンドに関するパラメータ構造体です。 [詳解]

#include <nn/atk/atk_SoundStartable.h>

公開メンバ関数

 StreamSoundInfo () NN_NOEXCEPT
 コンストラクタです。
 

公開変数類

size_t externalDataSize
 内部用機能のため使用禁止です。 メモリ上においてあるストリームサウンドのサイズです。
 
const char * externalPath
 サウンドアーカイブ管理外の bfstm ファイルのパスです。フルパスで指定する必要があります。
 
bool forcePlayPrefetchFlag
 内部用機能のため使用禁止です。 指定している場合、メモリ配置の問題でプリフェッチデータが再生できない際に強制的にストリーム再生します。
 
const void * pExternalData
 内部用機能のため使用禁止です。 メモリ上においてあるストリームサウンドへのアドレスです。
 
const void * prefetchData
 メモリ上においてあるプリフェッチデータへのアドレスです。開始オフセットを指定している場合はプリフェッチデータは使用されなくなります。
 
detail::driver::StreamBufferPool * pStreamBufferPool
 内部用機能のため使用禁止です。 再生に使用する外部ストリームバッファプールを指定します。
 
StreamRegionCallback regionCallback
 ストリームジャンプ機能において、ジャンプ先のリージョンを指定するためのコールバック関数です。
 
void * regionCallbackArg
 StreamRegionCallback に渡すユーザー引数です。
 

詳解

ストリームサウンドに関するパラメータ構造体です。

ストリームサウンド再生時に渡されます。

externalPath にストリームサウンドバイナリへのパスを指定すると、 サウンドアーカイブ中で設定されているストリームサウンドが参照する ストリームサウンドバイナリの代わりに、指定したものを使用して再生することができます。 NULL を指定すると、 元々サウンドアーカイブで設定されているストリームサウンドバイナリが使用されます。
