詳解

テクスチャ生成シェーダー

詳細
Mask,Facelineテクスチャを生成するためのシェーダークラスです。
Mask、Facelineテクスチャを簡単に作成するためのユーティリティとして用意しています。
このクラスを使わずにアプリ側でシェーダーを用意してもかまいません。

各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).

TextureShaderを初期化します。

注意
pMemory およびは、TextureShader::Finalize() まで保持する必要があります。
初期化済みのTextureShaderを再初期化することは出来ません。
Finalize() してから呼び出します。
初期化済みのresShaderFileはTextureShader::Finalize() まで
初期化済みの状態で保持する必要があります。
引数
[in]pMemoryTextureShaderが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]resShaderFile初期化済みのシェーダーファイル
[in]shaderCodeTypeシェーダーコードの種類
[in]pDeviceデバイス
[in]infoTextureShaderの初期化用のinfo
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
事後条件
成功したら TextureShader::IsInitialized() = true
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).
