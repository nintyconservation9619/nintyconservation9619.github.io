詳解

SHA-256 ハッシュを計算するクラスです。

詳細
このクラスは以下の状態を持ちます。 インスタンスの生成直後は未初期化状態です。

  • 未初期化状態
  • 処理可能状態
  • 完了状態

size_t nn::crypto::Sha256Generator::GetContext ( Sha256Context *  pContext) const

ハッシュ計算の中間状態であるハッシュコンテキストを取得します。

引数
[out]pContextハッシュコンテキストを格納するバッファへのポインタ。
戻り値
入力済みデータのうち、ハッシュ計算が行われていないデータのバイト数。
事前条件
  • インスタンスは処理可能状態である。
事後条件
  • pContext にハッシュコンテキストが書き込まれる。

詳細
取得したハッシュコンテキストを InitializeWithContext() で初期化することでハッシュ計算を再開することができます。

Update() で入力されたデータのうちブロックサイズに満たなかったデータはブロックサイズに達するまで未処理の状態でバッファされます。 未処理のデータについては計算結果はコンテキストには反映されていないため、この関数の返り値が 0 でない場合、正しく計算を再開するためには以下のどちらかの処理が必要になります。

  1. ハッシュ計算の再開時に返り値のサイズだけ巻き戻した位置からデータを入力する
  2. GetBufferedData() で取得できるデータをコンテキストとあわせて保存しておき、ハッシュ計算の再開後に最初に入力する
size_t nn::crypto::Sha256Generator::GetBufferedDataSize ( ) const

入力済みの未処理データのサイズを取得します。

戻り値
入力済みの未処理データのバイト数。

詳細
Update() で入力されたデータのうちブロックサイズに満たなかったデータはブロックサイズに達するまで未処理の状態でバッファされます。

この関数ではそのデータのサイズを取得できます。

void nn::crypto::Sha256Generator::GetBufferedData ( void *  pData,
size_t  dataSize 
) const

入力済みの未処理データを取得します。

引数
[out]pDataデータを取得するバッファへのポインタ。
[in]dataSizepData が指すバッファのバイトサイズ。
事前条件
  • dataSize >= GetBufferedDataSize()
事後条件
  • pData にデータが書き込まれる。

詳細
Update() で入力されたデータのうちブロックサイズに満たなかったデータはブロックサイズに達するまで未処理の状態でバッファされます。

この関数ではそのデータを取得できます。

データの取得に必要になるバッファのサイズは最大で BlockSize - 1 バイトです。
