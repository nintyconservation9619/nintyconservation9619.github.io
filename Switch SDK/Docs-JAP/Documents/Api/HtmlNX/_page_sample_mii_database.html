データベースから取得した Mii を表示するサンプルプログラムの解説です。

概要

データベースから取得した Mii を表示するサンプルプログラムです。 nn::mii::Database を使用して Mii データを取得する方法を示しています。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/MiiDatabase 以下にあります。

必要な環境

特にありません。

操作方法

特にありません。

注意事項

NX 実機での実行には、Mii テストモードを有効にする必要があります。 DevMenu 、 DevMenuCommand の 開発支援機能を参照してください。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは、データベースから Mii のデータを取得して 取得したデータを使用して Mii の描画を行います。

コンバート処理

  • Samples/Sources/Applications/MiiDatabase/Resources/SampleShader.glsl

    Mii 描画用簡易シェーダです。

サンプルプログラム実行時の処理

このサンプルプログラムの処理の流れは以下の通りです。

  • 描画のための各種リソースの初期化
  • Mii 用のサンプラの作成
  • データベースの初期化
  • サンプル用のデータベースをインポート
  • nn::mii::CharInfo をデータベースから取得
  • データベースの破棄
  • Faceline、Mask テクスチャ用シェーダ（nn::mii::TextureShader）の初期化
  • Mii リソース（nn::mii::Resource）の初期化
  • nn::mii::CharModel の初期化
  • nn::mii::Faceline の初期化
  • nn::mii::Mask の初期化
  • デスクリプタスロットの割り当て
  • nn::mii::FacelineGpuBuffer の初期化
  • nn::mii::MaskGpuBuffer の初期化
  • Faceline テクスチャのレンダリング
  • Mask テクスチャのレンダリング
  • Mii 描画用のシェーダの初期化
  • Mii 描画用の描画処理の初期化
  • Mii 描画のためのコンスタントバッファの設定
  • Mii の描画
