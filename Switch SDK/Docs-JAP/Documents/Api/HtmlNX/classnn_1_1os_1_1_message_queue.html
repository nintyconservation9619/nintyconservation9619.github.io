詳解

メッセージキューを扱うためのクラスです。

詳細
メッセージキューはデータの送受信を使ってスレッド間で同期を 確立するための機能です。

nn::os::MessageQueue クラスは、nn::os::MessageQueueType オブジェクトを 使用する同期制御 API の呼び出しをラッピングしたユーティリティクラスです。
本クラスが提供するメンバ関数は、nn::os::MessageQueueType 型を 引数にとる nn::os::SendMessageQueue() などの API を発行する形で 実装されています。

コンストラクタです。
メッセージキューオブジェクトを構築し、指定されたパラメータで 初期化します。
メッセージキューのバッファ領域には、uintptr_t 型の配列を渡して下さい。 ここには count * sizeof(uintptr_t) だけの領域が確保されていなければ なりません。
詳細は nn::os::InitializeMessageQueue() を参照して下さい。
