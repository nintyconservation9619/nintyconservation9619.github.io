enum nn::g3d::SkeletonObj::Flag

スケルトンの状態を表すフラグです。

列挙値
Flag_BlockBufferValid 

内部用機能のため使用禁止です。 ユニフォームブロックが構築済みであるかどうかを表すフラグです。

Flag_InvalidLocalMtx 

内部用機能のため使用禁止です。 古いローカル行列の一部がキャッシュに乗っているかどうかを表すフラグです。

Flag_InvalidWorldMtx 

内部用機能のため使用禁止です。 古いワールド行列の一部がキャッシュに乗っているかどうかを表すフラグです。

enum nn::g3d::SkeletonObj::ScaleMode

スケルトンに関するScaleモードです。

列挙値
ScaleMode_None 

スケール計算を行わないことを表すフラグです。

ScaleMode_Std 

通常の方式でスケール計算を行うことを表すフラグです。

ScaleMode_Maya 

Maya 方式でスケール計算を行うことを表すフラグです。

ScaleMode_Softimage 

Softimage 方式でスケール計算を行うことを表すフラグです。

RotateMode_Quat 

クォータニオン方式で回転計算を行うことを表すフラグです。

bool nn::g3d::SkeletonObj::SetupBlockBuffer ( nn::gfx::Device *  pDevice,
nn::gfx::MemoryPool *  pMemoryPool,
ptrdiff_t  offset,
size_t  memoryPoolSize 
)

メモリプールを渡して、ユニフォームブロックを構築します。

引数
[in]pDeviceデバイスへのポインター。
[in]pMemoryPoolメモリプールへのポインター。
[in]offsetメモリプールの使用する領域へのオフセット。
[in]memoryPoolSizeoffset以降で使用可能なメモリプールのサイズ。
戻り値
構築に成功した場合は true、失敗した場合は false を返します。
事前条件
  • pDevice が初期化されている。
  • pMemoryPool が初期化されている。
  • offset で指されたメモリプールの位置が nn::gfx::Buffer::GetBufferAlignment() のアラインメント上にある。
  • memoryPoolSize は CalculateBlockBufferSize() 以上である。
  • ユニフォームブロックが未構築である。
事後条件
  • ユニフォームブロックが構築されている

void nn::g3d::SkeletonObj::CalculateSkeleton ( int  bufferIndex)

ワールド行列から行列パレットを計算します。

引数
[in]bufferIndex計算を反映させるユニフォームブロックのインデックス。
事前条件
  • インデックスはスケルトンが持つユニフォームブロックのバッファー数の範囲内。

詳細
GPU から参照されるバッファーを書き換えるため、 前回の描画で GPU が参照し終わった後に呼ぶ必要があります。 構築時のオプションでダブルバッファー化するか、 フレームバッファーのコピーアウト中などに呼ぶことを想定しています。
