nn::atk::BiquadFilterCoefficients 構造体

Biquad フィルタ係数の構造体です。 [詳解]

#include <nn/atk/atk_Global.h>

公開変数類

 denominator[1] の係数を表します。

詳解

Biquad フィルタ係数の構造体です。

各係数については、Nintendo SDK リファレンスマニュアルの BiquadFilterParameter の説明をご参照ください。 各係数がどの係数に対応しているかは、それぞれの係数の説明に記載されています。
