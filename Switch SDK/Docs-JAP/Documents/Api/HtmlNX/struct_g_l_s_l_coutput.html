詳解

Main GLSLC output structure.

This structure contains information about the overall GLSLCoutput followed by numSections section headers.

Data for each section is stored following the last header, with the section header indicating the size of the data and its offset relative to the beginning of the GLSLCoutput structure.

The size parameter contains the total size of the GLSLCoutput structure (including data).

The dataOffset is the offset to the first data block from the beginning of the GLSLCoutput structure.

The structure is stored in contiguous memory, using the offsets in each section header to denote the data location in the GLSLCoutput structure.

各種例:
AssetFileLoadingHelper.cpp, IntermediateFileManager.cpp.
