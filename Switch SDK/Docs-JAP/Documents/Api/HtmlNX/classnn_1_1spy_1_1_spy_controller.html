void nn::spy::SpyController::SetCurrentApplicationFrame ( int  value)

現在のアプリケーションフレームを設定します。

引数
[in]value現在のアプリケーションフレーム値を指定します。

詳細
nn::spy が送信するデータには、この関数で設定したアプリケーションフレームが付加されます。 アプリケーションフレームを設定しておくと、Spy.exe のタイムラインにてアプリケーションフレーム単位のグリッド表示が可能になります。 この関数は、できるだけアプリケーションフレーム処理の先頭で呼び出すようにしてください。

参照
GetCurrentApplicationFrame
各種例:
SpyAtk.cpp, SpyAudio.cpp.

bool nn::spy::SpyController::RegisterModule ( SpyModule &  module)

Spy モジュールを登録します。

詳細
モジュールの登録は Initialize() と Open() の間に行います。

引数
[in]moduleSpy モジュールです。
戻り値
登録に成功した場合は true、失敗した場合は false を返します。
事前条件
  • Initialize() は呼ばれた。
  • Finalize() は呼ばれていない。
  • Open() は呼ばれていない、または Open() が呼ばれた後に Close() が呼ばれた。
各種例:
SpyAtk.cpp, SpyAudio.cpp.
void nn::spy::SpyController::UnregisterModule ( SpyModule &  module)

Spy モジュールの登録を解除します。

詳細

引数

事前条件

  • Open() は呼ばれていない、または Open() が呼ばれた後に Close() が呼ばれた。
