 Enumerated values for the type Format. [詳解]
 

詳解

Identifies how data elements stored in texture or buffer objects are encoded.

Specifies how data elements stored in texture or buffer objects are encoded and decoded, described in detail in the programming guide. Not all formats are supported for all uses (texture, vertex pulling); see the formats table to determine what formats may be used in what situations.
