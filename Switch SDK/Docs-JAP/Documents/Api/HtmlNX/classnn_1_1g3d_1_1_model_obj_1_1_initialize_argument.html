void nn::g3d::ModelObj::InitializeArgument::Skeleton ( SkeletonObj *  pSkeleton)

共有スケルトンを設定します。

引数
[in]pSkeletonSkeletonObj へのポインター。
事前条件
  • 設定する SkeletonObj は設定するリソースと同一のスケルトンリソースを扱うインスタンスである。

詳細
構築済み SkeletonObj を渡すことで他の ModelObj とスケルトンを共有することができます。 共有することで計算処理をスキップすることができます。
