enum nn::g3d::ResCameraAnim::Flag

カメラアニメーションに関するフラグです。

列挙値
Flag_CurveBaked 

カーブがベイク済みです。

Flag_PlayPolicyLoop 

ループすることを表します。

Flag_RotAim 

カメラの視線を注視点から計算します。

Flag_RotEulerZxy 

カメラの視線を回転角から計算します。

Flag_ProjOrtho 

平行投影で計算します。

Flag_ProjPersp 

透視投影で計算します。
