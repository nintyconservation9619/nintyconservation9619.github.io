詳解

一時的に使用するデスクリプタスロットを管理するデスクリプタプールです。

詳細
描画処理で一時的に使用するデスクリプタスロットを動的に確保し、 描画の完了後にまとめて解放するための機能を提供します。

引数

void nn::gfx::util::TransientDescriptorAllocator::Free ( )

このオブジェクトが記録している履歴のうち最も古いものに記録されているデスクリプタスロットを解放します。

事前条件
IsInitialized() == true
GetHistoryLength() > 0
事後条件
最も古い履歴に記録されているデスクリプタスロットを解放します。
GetHistoryLength() の返す値が 1 減少します。

詳細
スロットの解放は Begin() の呼びだしから End() の呼び出しまでを 1 単位として行われます。 各々の Allocate() で獲得したスロットを個別に解放することはできません。
解放されたスロットは直ちに獲得可能な状態になります。
同一インスタンスに対するこの関数の複数回の呼び出しはスレッドセーフではありません。 同一インスタンスに対するこの関数とスロットの獲得を行う関数の同時呼出しはスレッドセーフです。

End()

void nn::gfx::util::TransientDescriptorAllocator::FillHistory ( )

履歴を確保数 0 の履歴で埋めます。

事前条件
IsInitialized() == true
IsRecording() == false
事後条件
GetHistoryLength() == GetHistoryLengthMax()

詳細
履歴がいっぱいになるまで確保数 0 の履歴を追加します。 以下のコードと等価です。

TransientDescriptorAllocator transient;
while(transient.GetHistoryLength() < transient.GetHistoryLengthMax())
{
transient.Begin();
transient.End();
}

既に履歴が記録されている場合、この関数呼び出しによって追加される履歴がより新しい記録となります。
同一インスタンスに対するこの関数の呼び出しはスレッドセーフではありません。
