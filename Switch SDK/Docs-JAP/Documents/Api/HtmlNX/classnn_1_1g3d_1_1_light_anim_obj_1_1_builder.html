詳解

LightAnimObj の構築を行うクラスです。

構築子と解体子

nn::g3d::LightAnimObj::Builder::Builder ( )

コンストラクタです。

詳細
インスタンスで使用するすべてのライトアニメーションリソースに対して Reserve() するか、 SetMaxCurveCount() でバッファーサイズの計算に必要なパラメータを指定する必要があります。
