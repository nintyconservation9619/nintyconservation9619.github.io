詳解

ミューテックスを扱うためのクラスです。

詳細
ミューテックスはスレッド間で排他制御を行なうための同期オブジェクトです。

nn::os::Mutex クラスは、nn::os::MutexType オブジェクトと 関連する同期制御 API の呼び出しをラッピングしたユーティリティクラスです。 本クラスが提供するメンバ関数は、nn::os::MutexType 型を引数にとる nn::os::LockMutex() などの API を発行する形で実装されています。

また、このクラスは C++ 標準の BasicLockable 要件と Lockable 要件を 満たしているため、std::lock_guard や std::unique_lock などの クラステンプレートの型引数とすることができます。
このため、Lock() と lock() のように同じ処理を行なうメンバ関数が 複数存在しています。

各種例:
AudioEchoback.cpp.

void nn::os::Mutex::Lock ( )

ミューテックスをロックします。

詳細
ミューテックスをロックします。 ロックに成功するまでブロッキングします。
詳細は nn::os::LockMutex() を参照して下さい。

bool nn::os::Mutex::TryLock ( )

ミューテックスのロックを試みます。

戻り値
ロックに成功したか否か

詳細
ミューテックスのロックを試み、ロックの成否を bool 型で返します。
本関数は、如何なる時もブロッキングしません。
詳細は nn::os::TryLockMutex() を参照して下さい。

void nn::os::Mutex::Unlock ( )

ミューテックスをアンロックします。

詳細
ミューテックスをアンロックします。 ミューテックスは事前にロックされていなければなりません。
詳細は nn::os::UnlockMutex() を参照して下さい。

ミューテックスが自スレッドによって獲得済みか否かを返します。

詳細
ミューテックスが自スレッドによって獲得済みなら true を、 そうでなければ false を返します。
