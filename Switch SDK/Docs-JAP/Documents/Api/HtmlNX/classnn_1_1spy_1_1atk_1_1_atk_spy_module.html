void *  buffer,
size_t  bufferSize 
)

初期化を行います。

詳細
AtkSpyModule が使用するワークメモリをユーザが渡します。 InitializeArg 構造体に初期化に必要なパラメータを設定し、 GetRequiredMemorySize() で必要なメモリサイズを取得してください。

引数
[in]args初期化パラメータです。
[in]bufferAtkSpyModule が使用するワークメモリへのポインタです。
[in]bufferSizeワークメモリのサイズです。
戻り値
初期化に成功した場合は true を返します。
事前条件
  • buffer は nn::DefaultAlignment にアラインしている
  • bufferSize >= GetRequiredMemorySize()
  • nn::atk::SoundSystem::IsInitialized() == true
  • IsInitialized() == false
  • IsRegistered() == false
  • args.isPerformanceMetricsEnabled == true のときは args.isPerformanceMetricsInitializationEnabled == true
  • args.audioPerformanceMetricsInfoCount >= InitializeArg::AudioPerformanceMetricsCountMin
  • args.isWaveformEnabled == true のときは args.isWaveformInitializationEnabled == true
  • args.isWaveformInitializationEnabled == true の時は nn::atk::SoundSystem が nn::atk::SoundSystem::SoundSystemParam::enableCircularBufferSink == true で初期化されている

波形データを送信します。

詳細
nn::atk::SoundSystem::ReadCircularBufferSink() で取得した波形データを送信します。

nn::atk は nn::audio::RequestUpdateAudioRenderer() の 完了時刻を提供しないので、代わりに、引数 tick には nn::atk::SoundSystem::ReadCircularBufferSink() の呼び出し直前の Tick を指定します。

引数
[in]buffer波形のサンプルデータが格納されたバッファへのポインタです。
[in]bufferSizeバッファに格納されたサンプルデータのサイズです。
[in]ticknn::atk::SoundSystem::ReadCircularBufferSink() の呼び出し直前の Tick です。
戻り値
成功したら true, 失敗したら false を返します。
事前条件
  • tick <= nn::os::GetSystemTick()
  • tick は前回の呼び出し時の tick 以上でなければなりません。
  • IsInitialized() == true
  • Initialize() で波形データ送信のためのパラメータが指定されている。
