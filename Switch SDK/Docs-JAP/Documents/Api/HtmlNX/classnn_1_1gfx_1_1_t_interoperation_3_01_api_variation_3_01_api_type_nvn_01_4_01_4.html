詳解

template<>
class nn::gfx::TInteroperation< ApiVariation< ApiTypeNvn > >

NVN API との相互運用を行うためのクラスです。

詳細
元のオブジェクトの作成者がオブジェクトの所有権をもちます。
Initialize が呼ばれていない gfx オブジェクトに対して Finalize を呼ぶことはできません。
元のオブジェクトを作成したコンテキストとそれを使用するコンテキストは同じまたは共有関係にある必要があります。
