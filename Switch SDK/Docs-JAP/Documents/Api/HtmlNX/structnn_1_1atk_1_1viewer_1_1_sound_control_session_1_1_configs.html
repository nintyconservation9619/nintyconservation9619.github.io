メンバ詳解

size_t nn::atk::viewer::SoundControlSession::Configs::channelStreamBufferSize

チャンネル毎の受信用ストリームのバッファサイズです。


指定しないとデフォルト値 DefaultChannelStreamBufferSize が使用されます。
通常の利用において、値を変更する必要はありません。

int nn::atk::viewer::SoundControlSession::Configs::maxItemName

アイテム名の長さの上限値です。


指定しないとデフォルト値 DefaultMaxItemName が使用されます。
非常に長いアイテム名を対象にしたい場合や、使用する名前が短いことが事前にわかっていて、 少しでもメモリ使用量を減らしたい場合に設定を変更してください。
