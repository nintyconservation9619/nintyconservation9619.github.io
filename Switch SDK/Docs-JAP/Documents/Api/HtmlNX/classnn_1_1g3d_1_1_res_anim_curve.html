enum nn::g3d::ResAnimCurve::FrameType

アニメーションカーブの frame quantization のタイプです。

列挙値
FrameType_Quant32 

float 浮動小数点数のフレームです。

FrameType_Quant16 

S10.5 固定小数点数のフレームです。

FrameType_Quant8 

U8 整数のフレームです。

enum nn::g3d::ResAnimCurve::KeyType

アニメーションカーブの　key quantizationのタイプです。

列挙値
KeyType_Quant16 

float 浮動小数点数 * fScale + fOffset のキーです。

KeyType_Quant8 

S16 整数 * fScale + fOffset のキーです。

CurveType_BakedFloat 

コマ形式のカーブです。

CurveType_StepInt

CurveType_BakedInt 

コマ形式のカーブです。

CurveType_StepBool 

ステップ形式のカーブです。

CurveType_BakedBool 

コマ形式のカーブです。

enum nn::g3d::ResAnimCurve::WrapMode

繰り返し方法です。

列挙値
WrapMode_Clamp 

最初または最後のフレームにクランプします。

WrapMode_Repeat 

カーブを繰り返します。

WrapMode_Mirror

WrapMode_RelativeRepeat 

端を起点として相対的にカーブを繰り返します。
