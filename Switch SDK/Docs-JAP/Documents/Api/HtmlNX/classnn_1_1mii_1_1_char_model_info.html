CharModelInfo& nn::mii::CharModelInfo::SetDefault ( )

デフォルト値を設定

戻り値
自身の参照を返します。

詳細
デフォルト値は以下の通りです。
CharModelInfoData::facelineFormat = nn::gfx::ImageFormat_R8_G8_B8_A8_Unorm
CharModelInfoData::maskFormat = nn::gfx::ImageFormat_R8_G8_B8_A8_Unorm
CharModelInfoData::facelineMipCount = 1
CharModelInfoData::maskMipCount = 1
CharModelInfoData::facelineHeight = 512
CharModelInfoData::maskResolution = 512
CharModelInfoData::maskCount = 1
CharModelInfoData::createFlag = CreateFlag_Default
CharModelInfoData::scale = 1.0f
CharModelInfoData::coordinateUp = CoordinateType_DefaultUp
CharModelInfoData::coordinateFront = CoordinateType_DefaultFront

各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).
