詳解

大きなメモリプール上のメモリを小分けにして使用するためのアロケータです。

詳細
管理領域用のメモリは、ユーザが指定したコールバック関数によって確保・解放されます。

[in]isThreadSafeAllocate(), Free() をスレッドセーフで呼び出すようにするかどうかのフラグです。

pMemoryPool は有効なメモリプールのポインタでなければなりません。
baseOffset >= 0
baseOffset % allocatableAlignmentMax == 0
size >= 1 && size <= SizeMax
allocatableAlignmentMax >= 1 && allocatableAlignmentMax <= AlignmentMax
allocatableAlignmentMax は 2 の累乗でなければなりません。

const size_t nn::gfx::util::MemoryPoolAllocator::AllocatorUnitSize = (1 << 8)
static

このクラスで管理するメモリユニットのサイズです。

詳細
これより小さいサイズでアロケートしても、内部で AllocatorUnitSize サイズ分のメモリプールを消費します。
