詳解

一般的なネットワーク情報を格納する構造体です。

詳細
LDN ライブラリで構築されたネットワークに限らず、 一般的なアクセスポイントで構築されたネットワークがもつ情報を格納する構造体です。

channel はアクセスポイントがネットワークを展開している無線チャンネルです。 2.4GHz 帯の 1, 6, 11 および 5GHz 帯の 36, 40, 44, 48 のいずれか 1 つの無線チャンネルが格納されています。

linkLevel には LinkLevelMin 以上、 LinkLevelMax 以下の通信品質が格納されています。 ステーションでは、自身とアクセスポイントの間の信号強度から通信品質が算出されます。 ステーション・ステーション間の信号強度は通信品質に反映されないため、通信品質が良好であっても、 ステーション・ステーション間ではフレームの損失が多くなる場合があります。 ステーションが既にアクセスポイントから切断されている場合の通信品質は LinkLevelMin です。 アクセスポイントでは、全てのステーションとの間の信号強度の最小値から通信品質が算出されます。 ただし、ステーションが存在しない場合の通信品質は常に LinkLevelMax です。 通信品質が低下する場面としては以下のような状況が考えられます。

  • 周囲の電波状況に問題がある
  • アクセスポイントとステーションの間の距離が離れすぎている
  • アクセスポイントとステーションの間に障害物がある

通信品質は最近受信したフレームの信号強度を元にしています。 アプリケーションが全くフレームを受信しなかったとしても、 システムで制御用にフレームを送受信しているため通信品質は更新されますが、 通信品質の更新が遅くなり正確さを欠く状態になることに注意してください。

networkType には NetworkType のいずれかの値が格納されています。
