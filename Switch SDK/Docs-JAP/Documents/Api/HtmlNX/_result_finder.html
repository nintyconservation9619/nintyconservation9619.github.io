Result 値検索

nn::Result クラスを返り値として返却する API のエラー内容を検索し、詳細を表示します。

nn::Result の内部表現である 16 進数 32bit の値を入力してください。

使い方

  1. nn::Result の内部表現である 32bit の値を確認します。
    • 内部表現の 32bit 値は nn::Result::GetInnerValueForDebug() 関数で取得することができます。
    • 同様の値は NN_ABORT_UNLESS_RESULT_SUCCESS() マクロに nn::Result の値を渡した際のログからでも確認できます。
      NN_ABORT_UNLESS_RESULT_SUCCESS() マクロで実行停止した場合、例えば以下のようなログが出力されます。
      Failed: result
      Module: 3
      Description: 8
      InnerValue: 0x00001003
      上記の InnerValue の値が内部表現の 32bit 値にあたります。
  2. 取得した 32bit の値を本ページ上部のテキストボックスに入力します。
  3. 検索結果が表示されます。

検索結果

検索結果として、Result 値に関する情報が表示され、また、この Result 値を捕捉することのできる Result の一覧が表形式で表示されます。
「検索結果」欄には詳細値が一致した Result の情報が表示されます。
検索結果と表の各要素は以下を表します。

  • Name
    • 該当の Result クラス名です。
  • Text
    • 該当の Result の説明です。
  • Module
    • 該当の Result の属するモジュールです。
  • Description
    • 該当の Result のモジュール内における詳細値です。
  • Range
    • 該当の Result が捕捉する詳細値の範囲です。
