詳解

This module is responsible for format conversion.

Conversion uses doubles as the intermediate format, and supports Floyd-Steinberg dithering.

Convert format from a single image mipmap level to another mipmap level.

Convert format from a single image mipmap level to another mipmap level. double[4] is used as the intermediate conversion format. Must source and dest must have space already allocated, and dimensions & format matching. Does NOT do scaling.

引数
[in]srcSource image to convert from.
[out]dstDestination image to convert to.
[in]ditherWhether to enable dithering during format conversion.
戻り値
TP_OK if the operation was successful or an error code otherwise.
