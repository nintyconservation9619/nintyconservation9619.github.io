template<typename StorageT, typename Tag = void>
struct nn::util::BitPack< StorageT, Tag >

ビットフィールド代替型です。

詳細
ビットフラグの集合としてのみ利用する型は BitFlagSet を利用してください。

テンプレート引数
StorageT値を格納する整数型です。int より大きな型は使用できません。
Tag異なる型としてインスタンス化するためのタグです。
