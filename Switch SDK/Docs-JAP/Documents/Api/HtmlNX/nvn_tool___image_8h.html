Get the offset into the pixel of a data slice.

Takes into account pitch and BPP so forth. Eg. DataSlice(3) + GetSlicePixelOffset(1, 2) will be the pixel address of pixel[1][2][3].

引数
[in]lpThe mipmap level.
[in]xThe X (width) co-ordinate of the pixel offset.
[in]yThe Y (height) co-ordinate of the pixel offset.
戻り値
Offset into a dataslice at which the pixel is at, in bytes.
