圧縮テクスチャを使用して Mii を描画するサンプルの解説です。

概要

圧縮テクスチャに変換した Faceline と Mask テクスチャを使用して Mii 描画をするサンプルプログラムです。 似顔絵ライブラリで提供している TextureShader を使用して圧縮なしの Faceline と Mask テクスチャを作成後、 nn::gfx::util::TextureCompressor を使用して圧縮された Faceline と Mask のテクスチャを作成します。 作成した圧縮テクスチャを使用して Mii の顔モデルを 3D 空間上に表示するまでの手順を示しています。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/MiiCompressedTexture 以下にあります。

必要な環境

特にありません。

操作方法

特にありません。

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは、ランタイム圧縮ライブラリを使用して圧縮フォーマットのテクスチャに変換した Faceline と Mask テクスチャを使用して Mii の表示を行います。

コンバート処理

  • Samples/Sources/Applications/MiiCompressedTexture/Resources/SampleShader.glsl Mii 描画用簡易シェーダです。
  • Sources/Resources/GfxUtilTextureCompression/TextureCompression_ShaderVariation.glsl
  • Sources/Resources/GfxUtilTextureCompression/TextureCompression_VertexShader.glsl ランタイムテクスチャ圧縮用シェーダです。

サンプルプログラム実行時の処理

このサンプルプログラムの処理の流れは以下の通りです。

  • 描画のための各種リソースの初期化
  • Mii 用のサンプラの作成
  • nn::mii::CharInfo を取得
  • Faceline、Mask テクスチャ用シェーダ（nn::mii::TextureShader）の初期化
  • Mii リソース（nn::mii::Resource）の初期化
  • nn::mii::CharModel の初期化
  • nn::mii::Faceline の初期化
  • nn::mii::Mask の初期化
  • デスクリプタスロットの割り当て
  • nn::mii::FacelineGpuBuffer の初期化
  • nn::mii::MaskGpuBuffer の初期化
  • Faceline テクスチャのレンダリング
  • Mask テクスチャのレンダリング
  • 圧縮先のテクスチャの準備
  • 圧縮先のテクスチャビューの準備
  • 圧縮用のシェーダの準備
  • ランタイム圧縮ライブラリを使用して圧縮テクスチャを作成
  • 圧縮したテクスチャを CharModel に登録
  • Mii 描画用のシェーダの初期化
  • Mii 描画用の描画処理の初期化
  • Mii 描画のためのコンスタントバッファの設定
  • Mii の描画
