nn::irsensor::ImageTransferProcessorExConfig 構造体

イメージ転送プロセッサの詳細設定をおこなう構造体です。 [詳解]

#include <nn/irsensor/irsensor_ImageTransferProcessorTypes.h>

公開変数類

IrCameraConfig irCameraConfig
 IR カメラの設定です。
 
bool isExternalLightFilterEnabled
 外部光フィルタ機能が有効かどうか。
 
ImageTransferProcessorFormat origFormat
 切り取り元画像フォーマットです。出力画像フォーマットサイズ以上の領域を設定します。
 
ImageTransferProcessorFormat trimmingFormat
 切り取った出力画像フォーマットです。
 
int16_t trimmingStartX
 切り取り元画像上での、切り取り開始点の X 座標です。ピクセル単位です。
 
int16_t trimmingStartY
 切り取り元画像上での、切り取り開始点の Y 座標です。ピクセル単位です。
