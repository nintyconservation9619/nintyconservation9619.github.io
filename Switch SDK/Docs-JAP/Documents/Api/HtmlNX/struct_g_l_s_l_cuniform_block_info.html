int32_t GLSLCuniformBlockInfo::bindings[GLSLC_NUM_SHADER_STAGES]

Bindings associated with this block in each shader stage, where array index n corresponds to NVNshaderStage n.

If the block is unused in a particular shader stage, -1 is output for that array entry. The stagesReferencedIn member can be used to determine which array entries contain valid binding values.

各種例:
AssetFileLoadingHelper.cpp, IntermediateFileManager.cpp.
