nn::swkbd::ShowKeyboardArg 構造体

ソフトウェアキーボードの起動引数を定義する構造体です。 [詳解]

#include <nn/swkbd/swkbd_AppearArg.h>

公開変数類

KeyboardConfig keyboardConfig
 ソフトウェアキーボードの設定引数です。
 
void * textCheckWorkBuf
 ソフトウェアキーボードの入力確定後に、アプリ側で入力結果文字列を確認する際に用いるワークバッファです。
nn::swkbd::GetRequiredTextCheckWorkBufferSize() で取得できるバイトサイズ分確保したバッファを設定してください。
アプリ側で確認する必要のない場合、設定する必要はありません。
 
size_t textCheckWorkBufSize
 ソフトウェアキーボードの入力確定後に、アプリ側で入力結果文字列を確認する際に用いるワークバッファのバイトサイズです。
nn::swkbd::GetRequiredTextCheckWorkBufferSize() で取得できるバイトサイズを指定してください。
 
void * workBuf
 ソフトウェアキーボードに渡すワークバッファです。
nn::swkbd::SetInitialText() で設定した初期入力文字列や、 アプリ側で指定したユーザー辞書が格納されます。
nn::swkbd::GetRequiredWorkBufferSize() で取得できるバイトサイズ分確保したバッファを設定してください。
nn::os::MemoryPageSize でアライメントされたものでなければなりません。
 
size_t workBufSize
 ソフトウェアキーボードに渡すワークバッファのバイトサイズです。
nn::swkbd::GetRequiredWorkBufferSize() で取得できるバイトサイズを指定してください。
