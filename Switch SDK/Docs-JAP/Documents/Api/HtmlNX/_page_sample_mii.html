Mii

似顔絵ライブラリのサンプル解説です。

はじめに

ここでは、似顔絵ライブラリのサンプルプログラムについて解説します。
似顔絵ライブラリの機能は nn::mii の名前空間にあります。

サンプル一覧

サンプルコードは Samples/Sources/Applications/ 以下に Mii で始まるディレクトリ名で配置してあります。
サンプルを以下に一覧します。

サンプル 内容
MiiCompressedTexture 圧縮テクスチャを使用して Mii を描画するサンプルの解説です。
MiiDatabase データベースから取得した Mii を表示するサンプルプログラムの解説です。
MiiDynamicMask Mii の表情を動的に切り替えるサンプルプログラムの解説です。
MiiHeadwearExample アクセサリーや帽子などの頭部装飾物をつけたMiiを表示するサンプルプログラムの解説です。
MiiMipMap Faceline、Mask テクスチャのミップマップサンプルプログラムの解説です。
MiiPremultipliedAlphaIcon 背景とのブレンドに乗算済みアルファを使用したアイコンの生成機能サンプルプログラムの解説です。
MiiShaderExample Mii を MiiEdit アプレットと同等に描画するサンプルプログラムの解説です。
MiiSimple Mii の描画サンプルプログラムの解説です。
MiiSimpleIcon Mii アイコン描画サンプルプログラムの解説です。
MiiSimpleIconBody 体つき Mii アイコン描画サンプルプログラムの解説です。
MiiSimpleVariableIconBody 体格反映体つき Mii アイコン描画サンプルプログラムの解説です。
MiiSystemResource システムから取得した似顔絵リソースを使用して Mii を描画するサンプルプログラムの解説です。
MiiTransparentAdjuster 線形補間による透明ピクセルの色にじみ防止機能サンプルプログラムの解説です。
