引数
[in]imglibNVN image library entry points.
[in]converterFormat converter.
[in]mipmapGeneratorMipmap generator.
[in]combinerImage stitcher / combiner.
[in]srcImagesArray of numImages pointers to source images.
[out]dstImagesArray of pointers to destination images. Each pointer either points to an allocated destination image, or points directly to a given source image. Take care not to double-free images on the latter case.
[in]numImagesNumber of source & destination images to process.
[in]mipFilterWhich mipfilter to use. To disable mipmap generation, set this to MIPFILTER_INVALID.
[in]minMipSizeGenerate mipmaps until N x N where N = minMipSize.
[in]srcFormatReinterpret source images as this format. BPP must match between the real image format and the reinterpreted format. Set to NVN_FORMAT_NONE to disable.
[in]dstFormatConvert input images into this format. Set to NVN_FORMAT_NONE to disable conversion.
[in]ditherWhether to enable dithering when converting.
[in]stitchTextureArrayStitch all input textures into a single 2D texture array.
[in]assembleMipmapsCombine all input textures into a single image with mipmap chain.
戻り値
TP_OK if the operation was successful or an error code otherwise.
