詳解

フレンドのプレゼンスを扱うクラスです。

詳細
フレンドが設定したプレゼンスの情報を取得する機能を持ちます。
本クラスで取得できる情報は、フレンドの本体上で UserPresence を経由して設定されます。

プレゼンスはリアルタイムな情報を伝搬する機能であるため、本体ストレージ上にキャッシュは残りません。

以下の場合、有効なプレゼンスデータは取得できません。

  • ネットワークに接続していない。
  • フレンドサーバーに障害が発生しており、フレンドのプレゼンスが取得できない。
  • フレンドが自身のプレゼンス公開範囲を制限している。

最後に遊んだアプリケーションを取得します。

戻り値
最後に遊んだアプリケーション。

詳細
プレゼンスの状態が PresenceStatus_Online または PresenceStatus_OnlinePlay の場合、現在遊んでいるアプリケーションとなります。

有効なプレゼンスデータが存在しない場合、本関数はゼロクリアされたアプリケーション情報を返します。
