nn::hid::NpadJoyRightState 構造体

#include <nn/hid/hid_NpadJoyRight.h>

詳解

Npad の Joy-Con (R) をモデルとした操作スタイル (NpadStyleJoyRight) の入力状態を表す構造体です。

詳細
Joy-Con (R) の操作スタイルで扱えるデジタルボタン ( A, B, X, Y, R, ZR, Joy-Con (R) SL, Joy-Con (R) SR, Plus ) と 右アナログスティックの入力状態を提供します。 その他の入力状態は無入力 ( デジタルボタンの押下は無く、アナログスティックはニュートラル位置 ) となります。 attributes::IsConnected または attributes::IsRightConnected で、コントローラーの接続状態を取得できます。

各種例:
IrSensorClusteringBasic.cpp, IrSensorMomentBasic.cpp.
