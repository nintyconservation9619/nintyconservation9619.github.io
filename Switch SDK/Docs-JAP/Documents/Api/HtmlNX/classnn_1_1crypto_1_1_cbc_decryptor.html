詳解

template<typename BlockCipher>
class nn::crypto::CbcDecryptor< BlockCipher >

CBC 暗号利用モードによる復号化を行うクラスです。

テンプレート引数
BlockCipher対象とするブロック暗号の実装クラス

詳細
CBC は NIST SP 800-38A で規定される暗号利用モードです。

このクラスはブロック暗号の実装クラスをテンプレート引数とするテンプレートクラスです。 実際の使用には Aes128CbcDecryptor などを利用してください。

テンプレートとして指定するブロック暗号クラスは以下の条件を満たす実装になっている必要があります。 各関数の引数など詳細な説明については AesDecryptor クラスを参照してください。

  1. BlockSize という処理するブロックサイズが定義された定数を持つ。
  2. DecryptBlock という単一ブロックを復号化する関数を持つ。

このクラスは以下の状態を持ちます。 インスタンスの生成直後は未初期化状態です。

  • 未初期化状態
  • 処理可能状態

本クラスではパディングはサポートされません。 データフォーマットとして特定のパディングが必要な場合は、データ側にパディングを付与して使用してください。

CBC モードで復号化の処理を初期化します。

[in]pIvIV データへのポインタ。
[in]ivSizeIV データのバイトサイズ。
事前条件
  • pBlockCipher で指定するインスタンスは DecryptBlock を実行できる。
  • ivSize == IvSize
事後条件

詳細
pBlockCipher には復号化に使用するブロック暗号を実装するインスタンスを指定します。 あらかじめ鍵の設定など必要な初期化を行い、DecryptBlock が実行できる状態になっている必要があります。

pBlockCipher で指定したインスタンスはコピーされません。 一連の処理が完了するまで対象 pBlockCipher に対する処理は行わないでください。

IV データは内部にコピーされます。

template<typename BlockCipher>
size_t nn::crypto::CbcDecryptor< BlockCipher >::Update ( void *  pDst,
size_t  dstSize,
const void *  pSrc,
size_t  srcSize 
)

CBC モードで復号化を行います。

引数
[out]pDst復号化されたデータを受け取るバッファへのポインタ。
[in]dstSizepDst が指すバッファのバイトサイズ。
[in]pSrc復号化するデータが格納されているバッファへのポインタ。
[in]srcSize復号化するデータのバイトサイズ。
戻り値
実際に復号化されたバイト数。
事前条件
  • インスタンスは処理可能状態である。
  • dstSize >= ((srcSize + GetBufferedDataSize()) / BlockSize) * BlockSize
  • pDst, dstSize で指定する領域と pSrc, srcSize で指定する領域に重なりがない。
事後条件
  • pDst に復号化された結果が書き込まれる。

詳細
CBC モードでは復号化はブロック単位で処理されるため、1 回の呼び出しで dstSize として必要になるサイズは最大で (srcSize + ブロックサイズ - 1) になります。 例えば 1 バイトずつ Update を呼び出したとすると積算サイズがブロックサイズになるまでは処理は行われず、その間は返り値として 0 が返されます。 積算サイズがブロックサイズに到達した呼び出しで 1 ブロック分が処理され、ブロックサイズ分が返り値として返されることになります。

CBC モードでは srcSize の合計がブロックサイズの倍数である必要があります。

dstSize の合計として必要になるバッファサイズは srcSize の合計と同じだけ必要になります。

template<typename BlockCipher>
size_t nn::crypto::CbcDecryptor< BlockCipher >::GetBufferedDataSize ( ) const

入力済みの未処理データのサイズを取得します。

戻り値
入力済みの未処理データのバイト数。

詳細
Update で入力されたデータのうちブロックサイズに満たなかったデータはブロックサイズに達するまで未処理の状態でバッファされます。

この関数ではそのデータのサイズを取得できます。
