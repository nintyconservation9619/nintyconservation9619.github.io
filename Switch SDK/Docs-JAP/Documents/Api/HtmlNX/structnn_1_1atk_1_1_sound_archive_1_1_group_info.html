詳解

グループの設定情報の構造体です。

サウンドアーカイブで定義されている、 グループの情報を取得するための構造体です。

groupFileSize には、サウンドアーカイブ内に格納される「グループファイル」 のファイルサイズが含まれています。

SoundMaker 上で設定するグループの出力方式が「埋め込み」の場合は、 当該グループを SoundDataManager::LoadData でロードする際に必要なメモリサイズに相当します。

一方、出力方式が「リンク」の場合は、 グループファイルには「当該グループに含まれるアイテム情報」 しか含まれていない (＝実データは含まれていない) ため、 LoadData の際に必要なメモリサイズとは異なります。
