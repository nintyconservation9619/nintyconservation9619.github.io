詳細

注意
pMemory およびは、TransparentAdjuster::Finalize() まで保持する必要があります。
初期化済みのTransparentAdjusterを再初期化することは出来ません。
Finalize() してから呼び出します。
初期化済みのresShaderFileはTransparentAdjuster::Finalize() まで
初期化済みの状態で保持する必要があります。

・カラーターゲットの設定
