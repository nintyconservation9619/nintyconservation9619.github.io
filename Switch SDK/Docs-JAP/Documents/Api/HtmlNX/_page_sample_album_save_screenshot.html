アプリケーションが生成した画像を画面写真として保存・ SNS 投稿する

目次

  • 概要
  • ファイル構成
  • 必要な環境
  • 操作方法
  • 解説

アプリケーションが生成した画像を画面写真としてアルバムに保存する方法および保存して SNS 投稿する方法を示します。

概要

アルバムライブラリをアプリケーションから使用する方法を示します。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/AlbumSaveScreenshot 以下にあります。

必要な環境

本サンプルプログラムは開発機環境でビルドして実行することができます。

操作方法

一般的なサンプルプログラムと同様に本プログラムをビルドし、実行してください。

サンプルプログラムは以下を順番に実行した後、終了します。

  • アルバムへの画面写真の保存
  • アルバムへ画面写真を保存して SNS 投稿

解説

このサンプルプログラムは次の順序で処理を行います。

  • アルバムへの画面写真の保存
    • アルバムライブラリの初期化
    • 保存する画像データの生成
    • 画像データを画面写真として保存
    • アルバムライブラリの終了
  • アルバムへ画面写真を保存して SNS 投稿
    • アルバムライブラリの初期化
    • 保存する画像データの生成
    • 画像データを画面写真として保存して SNS 投稿
    • アルバムライブラリの終了

177  nn::album::Initialize();

181  NN_LOG("Screenshot width = %d, height = %d\n", width, height);
182 
183  // 保存する画像データを生成します。
184  size_t imageDataSize = 0;
185  auto pImageData = CreateImageData(&imageDataSize, width, height);
186 
187  {
188  NN_LOG("SaveScreenshot\n");
189  auto result = nn::album::SaveScreenshot(
190  pImageData,
191  imageDataSize,
192  ScreenShotSize,
193  ReportOption
194  );
195  if(result.IsFailure())
196  {
197  if(nn::album::ResultAlbumUnknown::Includes(result))
198  {
199  if(nn::album::ResultAlbumError::Includes(result))
200  {
201  // 画面写真の作成や保存に失敗した場合にこのエラーが返ります。
202  NN_LOG(" -> Screenshot Failed(AlbumError)\n");
203  }
204  else if(nn::album::ResultAlbumFull::Includes(result))
205  {
206  // 画面写真を保存するストレージが満杯だった場合にこのエラーが返ります。
207  NN_LOG(" -> Screenshot Failed(AlbumFull)\n");
208  }
209  else if(nn::album::ResultAlbumFileCountLimit::Includes(result))
210  {
211  // 画面写真の数がアルバムに保存できる最大数に達している場合にこのエラーが返ります。
212  NN_LOG(" -> Screenshot Failed(AlbumFileCountLimit)\n");
213  }
214  else
215  {
216  NN_LOG(" -> Screenshot Failed(Unknown)\n");
217  }
218  }
219  else if(nn::album::ResultNotSupported::Includes(result))
220  {
221  NN_LOG(" -> NotSupported\n");
222  }
223  else
224  {
225  NN_LOG(" -> Error(Unknown)\n");
226  }
227  return;
228  }
229  }
230  NN_LOG(" -> OK\n");
231 
232  // API 呼出後、保存した画像データは破棄できます。
233  DestroyImageData(pImageData);
234 
235  // アルバムライブラリを終了します。
236  nn::album::Finalize();
237 
238  // 保存結果の自動通知が消えるのを待ちます。
239  // このサンプルでは連続で保存を行うため、通知が消えるまで待っていますが、

241  NN_LOG("Waiting for 5 sec ...\n");
242  nn::os::SleepThread(nn::TimeSpan::FromSeconds(5));

本サンプルで使用している nn::album::SaveScreenshot() では 記録メディアへの静止画像ファイルの書き込みが行なわれます。

記録メディアへの書き込み回数や書き込み容量に関しては、 別途ガイドラインが規定されていますので、 そちらに注意して API を不必要に乱発しないようにして下さい。

なお、書き込み頻度を調査するためのデバッグ機能も別途提供されています。 詳細は「FS ライブラリ／デバッグ機能／アクセス頻度確認」を参照して下さい。
