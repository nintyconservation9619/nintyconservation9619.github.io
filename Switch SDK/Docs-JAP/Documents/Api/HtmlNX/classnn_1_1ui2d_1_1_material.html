列挙型メンバ詳解

enum nn::ui2d::Material::Flags

マテリアルが保持する情報のフラグです。

列挙値
Flags_UserAllocated 

ユーザーがマテリアルを構築したかを示します。

Flags_TextureOnly 

テクスチャのみを設定します。

Flags_ThresholdingAlphaInterpolation 

閾値によるアルファ補間が有効であるかを示します。

Flags_BlackColorFloat 

黒カラーを float カラーとして扱うかどうか。

Flags_WhiteColorFloat 

白カラーを float カラーとして扱うかどうか。

Flags_DynamicAllocatedColorData 

カラーデータが動的に確保されたメモリ領域に保存されているかどうか。

Flags_Use2ForShaderVariationCountPerBlend 

ブレンド毎のバリエーション数として 2 を使うか(バイナリバージョンが 8.2.0より古い) かどうか。ライブラリ内部で利用するので、指定しないでください。

Flags_UseDetailedCombinerWithVariationVariable 

詳細コンバイナにバリエーション変数を使うか。(バイナリバージョンが 8.3.0より古い) かどうか。ライブラリ内部で利用するので、指定しないでください。

Flags_MaxFlags

詳細
引数で指定したマテリアルをコピーしたマテリアルを作成します。 このコンストラクタを使用して異なるレイアウトに属するマテリアルとしてコピーされた場合、キャプチャテクスチャの参照情報が更新されず 初期化した際のレイアウトが保持しているキャプチャテクスチャを参照する状態となります。 別レイアウトのマテリアルとしてコピーする場合は MaterialCopyContext を指定するコピーコントラクタを使用してください。

マテリアルの元々のレイアウトデータの状態ではなく、そのときの状態でコピー しますのでご注意ください。つまり、アニメーションを再生した後はそのままの 状態でコピーします。

また、アニメーションについては、何もバインドされていない状態になります。
