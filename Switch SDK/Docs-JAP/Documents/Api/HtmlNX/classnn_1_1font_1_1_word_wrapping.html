static bool nn::font::WordWrapping::CalculateWordWrapping ( uint32_t *  pWriteSize,

const uint16_t *  pInputText,
WordWrapCallback &  func,
const WordWrapConfig &  config = WordWrapConfig() 
)
static

ワードラッピングを行います。UTF-16版。

引数
[out]pWriteSizepOutTextBuffer に格納された文字列の長さを返します。
[out]pOutTextBuffer適切な位置に改行コードの追加された文字列が格納されます。
[in]outTextBufferSizepOutTextBuffer に格納できる文字数を指定します。終端文字を含みます。
[in]pInputText処理の対象となる文字列です（要ヌル終端）。
[in]func一行の表示範囲を得るコールバックです。
[in]configワードラップの動作を指定します。
戻り値
処理に成功した場合は true を返します。処理を中断した場合は false を返します。

詳細
文字列に改行が含まれる場合、改行コードはLFである必要があります。CRやCRLFは扱えません。

改行して良い場所が無かった場合には func が示す位置で強制的に改行します。

以下の場合は処理を中断します。その際でも pOutTextuBuffer はヌル終端されます。

  • pOutTextuBuffer のサイズが足りない場合。
  • config で指定された行数の上限を超えた場合。

pWriteSize にアドレスを指定すると、pOutTextBuffer に格納された文字列の長さが書き込まれます。
