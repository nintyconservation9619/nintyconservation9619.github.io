詳解

非同期処理を扱うクラスです。

詳細
非同期処理の処理待ちやキャンセルを行うことが可能です。
非同期処理が完了するまで、非同期関数に渡したオブジェクトを破棄してはいけません。

各種例:
FriendsProfileImage.cpp, FriendsSimple.cpp.

nn::Result nn::friends::AsyncContext::Cancel ( )

処理の中断を要求します。

戻り値
処理結果。

詳細
本関数は処理の中断を要求するだけであり、本関数の呼び出し直後に非同期処理が完了するわけではありません。
非同期処理を中断して完了した場合、 AsyncContext::GetResult は ResultCanceled を返します。

nn::Result nn::friends::AsyncContext::GetResult ( ) const

非同期処理の結果を取得します。

戻り値
非同期処理の結果。
戻り値一覧:

ResultCanceled 非同期 API のキャンセル要求が行われました。

ResultInternetRequestNotAccepted インターネット接続要求が受理されていない状態で非同期 API を実行しました。

ResultHttpError 通信中にエラーが発生しました。

ResultServerError サーバーがエラーを返しました。

nn::account::ResultNetworkCommunicationError 通信中にエラーが発生しました。

nn::account::ResultNetworkServiceAccountUnavailable ネットワークサービスアカウントが利用可能な状態ではありませんでした。

詳細
非同期処理の結果を返します。
非同期関数の呼び出し結果ではないことに注意してください。

AsyncContext::HasDone で取得したフラグが true になるまで、本関数を呼び出してはいけません。

非同期処理が以下のいずれかの場合、エラービューアでエラーメッセージを表示してください。

  • ResultInternetRequestNotAccepted
  • ResultHttpError
  • ResultServerError

バックグラウンドでプロフィール情報を取得する等、アプリケーションの進行に影響がない通信処理を行っていた場合、エラーを表示する必要はありません。

非同期処理の結果が nn::account 名前空間のものである場合、アカウントライブラリのマニュアルに従ってエラーハンドリングを行ってください。

各種例:
FriendsProfileImage.cpp, FriendsSimple.cpp.
