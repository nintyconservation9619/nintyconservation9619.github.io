bool nn::ui2d::ArchiveHandle::Initialize ( void *  pArchiveStart,
const char *  pResourceRootDirectory 
)

アーカイブおよび各種リソースを初期化します。

詳細
アーカイブのフォーマットが正しくなければ失敗します。

pArchiveStartで与えるアーカイブデータは、 nn::ui2d::ArchiveResourceAlignment バイトで アライメントしてください。

詳しくは、 nn::ui2d::ArcResourceAccessor::Attach メソッドの説明をご参照ください。

引数
[in]pArchiveStartアーカイブの先頭アドレスです。
[in]pResourceRootDirectoryリソースのルートディレクトリ名です。
戻り値
アーカイブの設定が成功した場合は true を、失敗した場合は false を返します。
参照
ArcResourceAccessor::Attach
