virtual void nn::perf::CpuMeter::Next ( )
virtual

自身とアタッチされている LoadMeterBase の計測結果の保存先バッファを切り替えます。毎フレームの終了時に呼び出してください。

詳細
アタッチされている LoadMeterBase の Next() も内部で再帰的に呼ばれるため、親負荷メーターの Next() だけを呼んでください。
この関数が呼ばれない場合、正常な測定結果を得ることができません。

nn::perf::LoadMeterBaseを再実装しています。
