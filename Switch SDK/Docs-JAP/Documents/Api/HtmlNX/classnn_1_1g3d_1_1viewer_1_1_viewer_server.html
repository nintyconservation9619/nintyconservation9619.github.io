NN_DETAIL_G3D_VIEWER_API ViewerResult nn::g3d::viewer::ViewerServer::Open ( )

サーバーと 3DEditor を通信可能な状態にすることを要求します。

戻り値
処理結果を返します。
戻り値一覧:

ViewerResult_Success - 正常に処理が終了しました。

ViewerResult_ViewerDisabled - nn::g3d::viewer が無効化されています。

ViewerResult_ServerNotInitialized - サーバーが未初期化です。

ViewerResult_ServerClosing - サーバーがクローズ中です。

ViewerResult_UnknownError - 不明なエラーです。

詳細
サーバーを通信可能な状態にするためには、この関数をコールした後に、 nn::g3d::viewer::ViewerServer::Poll() を呼ぶ必要があります。 通信可能かどうかは nn::g3d::viewer::ViewerServer::IsOpened() で判定することができます。

NN_DETAIL_G3D_VIEWER_API void nn::g3d::viewer::ViewerServer::Close ( )

サーバーと 3DEditor の通信を切断することを要求します。

詳細
サーバーを切断状態にするためには、この関数をコールした後に、 nn::g3d::viewer::ViewerServer::Poll() を呼ぶ必要があります。 切断されているかどうかは nn::g3d::viewer::ViewerServer::IsOpened() で判定することができます。
