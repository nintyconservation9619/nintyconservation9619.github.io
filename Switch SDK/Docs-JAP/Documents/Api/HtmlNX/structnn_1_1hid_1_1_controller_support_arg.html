nn::hid::ControllerSupportArg 構造体

コントローラーサポートの UI 呼び出しの起動パラメータです。 [詳解]

#include <nn/hid/hid_ControllerSupportArg.h>

公開メンバ関数

void SetDefault ()
 コントローラーサポートの UI 呼び出しの起動パラメータにデフォルト値を設定します。 [詳解]
 

公開変数類

bool enableExplainText
 コントローラーサポートの UI で表示するコントローラー番号ごとの文字列を使用するか指定します。
 
bool enableIdentificationColor
 コントローラーサポートの UI で表示するコントローラー番号ごとの識別用の色を使用するか指定します。
 
bool enableLeftJustify
 コントローラーサポートの終了時にコントローラー番号を前詰めするかを指定します。 false を指定するとコントローラー番号に空きがある状態で終了する場合があります。 一人専用モードでは無効です。
 
bool enablePermitJoyDual
 2本持ちのコントローラーがある状態での決定を許可するかを指定します。 false を指定すると2本持ちのコントローラーがある状態で決定できなくなります。 ローカル通信中など、2本持ちを禁止したい場合に使用することを想定しています。
 
bool enableSingleMode
 コントローラーサポートを一人専用モードで起動するかを指定します。
 
bool enableTakeOverConnection
 既に接続済みのコントローラーの接続を維持するかを指定します。 false を指定すると全てのコントローラーが切断されます。
 
char explainText [4][ExplainTextMaxBufferSize]
 コントローラーサポートの UI で表示するコントローラー番号ごとの文字列です。 最大 nn::hid::ExplainTextMaxLength 文字が指定可能です。 値を指定していても、enableExplainText が false の場合は反映されません。 文字列の幅などが適切に表示されているかは実際の表示を見て確認してください。
 
nn::util::Color4u8Type identificationColor [4]
 コントローラーサポートの UI で表示するコントローラー番号ごとの識別用の色を指定します。 値を指定していても、enableIdentificationColor が false の場合は反映されません。
 
int8_t playerCountMax
 無線コントローラーの接続を受け付ける最大人数です。 一人専用モードでは無効です。
 
int8_t playerCountMin
 無線コントローラーの接続を受け付ける最少人数です。 一人専用モードでは無効です。
 

詳解

コントローラーサポートの UI 呼び出しの起動パラメータです。

関数詳解

void nn::hid::ControllerSupportArg::SetDefault ( )

コントローラーサポートの UI 呼び出しの起動パラメータにデフォルト値を設定します。

詳細
下記のデフォルト値が設定されます。

  • playerCountMin : 0
  • playerCountMax : 4
  • enableTakeOverConnection : true
  • enableLeftJustify : true
  • enablePermitJoyDual : true
  • enableSingleMode : false
  • enableIdentificationColor : false
  • enableExplainText : false
