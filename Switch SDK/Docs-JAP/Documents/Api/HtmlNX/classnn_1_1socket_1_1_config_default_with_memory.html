詳解

Class specifying a default configuration.

Fixed size memory pool is included.

詳細
Default values provided by this class are adequate for most applications. It must be instantiated with global or static scope. As this class contains a memory pool, when instantiated the resulting object may consume several megabytes of memory.

各種例:
FriendsProfileImage.cpp, LdnAccessPoint.cpp, LdnSimple/Main.cpp, LdnStation.cpp, LibcurlBasic.cpp, LibcurlSslContextImport.cpp, NifmNetworkConnectionSimple.cpp, SocketBasic.cpp, SocketResolver.cpp, SslSimple.cpp.
