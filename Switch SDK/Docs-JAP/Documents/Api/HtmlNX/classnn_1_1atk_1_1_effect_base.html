詳解

エフェクトの抽象クラスです。

エフェクトの使用方法については、SoundSystem::AppendEffect などを参照してください。

ユーザー定義の信号処理を行うエフェクトを実装したい場合は nn::atk::EffectAux を参照してください。

virtual bool nn::atk::EffectBase::IsRemovable ( ) const
pure virtual

エフェクトが削除可能かどうかを取得します。

nn::atk::SoundSystem::ClearEffect または nn::atk::SoundSystem::RemoveEffect を呼び出す前に、 本関数で削除可能であることを確認してください。

戻り値
エフェクトが削除可能であれば true を返します。
