nn::irsensor::ClusteringData 構造体

クラスタリングプロセッサでオブジェクトを表す構造体です。 [詳解]

#include <nn/irsensor/irsensor_ClusteringProcessorTypes.h>

公開変数類

float averageIntensity
 オブジェクトの平均輝度値です（0 以上 IrCameraIntensityMax 以下）。
 
Rect bound
 オブジェクトの外接矩形です（ClusteringProcessorConfig::windowOfInterest で指定した領域内）。
 
::nn::util::Float2 centroid
 オブジェクトの重心座標です（ClusteringProcessorConfig::windowOfInterest で指定した領域内）。
 
int32_t pixelCount
 オブジェクトの面積です（ClusteringProcessorOutObjectPixcelCountMax 以下、かつ ClusteringProcessorConfig::objectPixelCountMin 以上 ClusteringProcessorConfig::objectPixelCountMax 以下）。
 

詳解

クラスタリングプロセッサでオブジェクトを表す構造体です。
