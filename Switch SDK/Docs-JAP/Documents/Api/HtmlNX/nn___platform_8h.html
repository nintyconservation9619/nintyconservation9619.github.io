マクロ定義

#define NN_BUILD_APISET_GENERIC   1
 汎用 API セットを使用している場合に定義されます。
 
#define NN_BUILD_APISET_NX   1
 NX API セットを使用している場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_ADDRESS_32   1
 アプリケーションが動作するプラットフォームにおいてアドレスが 32 ビット幅の場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_ADDRESS_64   1
 アプリケーションが動作するプラットフォームにおいてアドレスが 64 ビット幅の場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_ENDIAN_BIG   1
 アプリケーションが動作するプラットフォームにおいてエンディアンがビッグエンディアンの場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_ENDIAN_LITTLE   1
 アプリケーションが動作するプラットフォームにおいてエンディアンがリトルエンディアンの場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_NX   1
 アプリケーションが動作するプラットフォームが NX 実機環境の場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_OS_NN   1
 アプリケーションが動作するプラットフォームにおいて OS が NX 実機環境で使用されている OS である場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_OS_WIN   1
 アプリケーションが動作するプラットフォームにおいて OS が Windows である場合に定義されます。
 
#define NN_BUILD_TARGET_PLATFORM_WIN   1
 アプリケーションが動作するプラットフォームが Windows の場合に定義されます。
