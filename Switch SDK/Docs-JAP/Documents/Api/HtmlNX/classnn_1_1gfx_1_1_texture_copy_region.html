詳解

テクスチャーコピーオペレーションの範囲を表すクラスです。

各種例:
Town.cpp, VfxSimple/main.cpp.

構築子と解体子

nn::gfx::TextureCopyRegion::TextureCopyRegion ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解

void nn::gfx::TextureCopyRegion::SetDefault ( )

各パラメーターを既定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetOffsetU( 0 );
  • SetOffsetV( 0 );
  • SetOffsetW( 0 );
  • SetWidth( 1 );
  • SetHeight( 1 );
  • SetDepth( 1 );
  • EditSubresource().SetDefault();
  • SetArrayLength( 1 );
各種例:
Town.cpp, VfxSimple/main.cpp.
