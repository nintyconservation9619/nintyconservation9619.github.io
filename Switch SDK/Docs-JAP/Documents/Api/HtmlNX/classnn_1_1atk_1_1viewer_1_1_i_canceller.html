virtual bool nn::atk::viewer::ICanceller::IsCanceled ( void *  param)
pure virtual

キャンセルの有無を取得します。

対象の処理が完了するか、この関数が true を返すまで、処理が続行されます。

引数
[in]paramユーザーパラメータを指定します。
戻り値
true を返すと、対象をキャンセルします。
false を返すと、処理を続行します。

nn::atk::viewer::TimeoutCanceller, nn::atk::viewer::Cancellerで実装されています。

pure virtual

キャンセルオブジェクトの状態をリセットします。

引数
[in]paramユーザーパラメータを指定します。

nn::atk::viewer::TimeoutCanceller, nn::atk::viewer::Cancellerで実装されています。
