詳解

Facelineテクスチャ描画用バッファ

詳細
TextureShader でFacelineテクスチャを描画するときに
必要なバッファなどをまとめたクラスです。
このクラスは、1つのFacelineテクスチャを描画するごとに1つ必要です。

注意
TextureShader を使わない場合は不要です。
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).

)

FacelineGpuBufferを初期化します。

注意
pMemory および pMemoryPoolは、FacelineGpuBuffer::Finalize() まで保持する必要があります。
初期化済みのFacelineGpuBufferを再初期化することは出来ません。
Finalize() してから呼び出します。
引数
[in]pMemoryFacelineGpuBufferが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pDeviceデバイス
[in]pMemoryPoolFacelineGpuBufferが使用するメモリプール
[in]memoryPoolOffsetFacelineGpuBufferが使用するメモリプールのオフセット
[in]memoryPoolSizeFacelineGpuBufferが使用するメモリプールのサイズ
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& ! FacelineGpuBuffer::IsInitialized()
&& オフセット、サイズが正常
事後条件
成功したら FacelineGpuBuffer::IsInitialized() = true
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).

書き込み先テクスチャを設定

引数
[in]pDeviceデバイス
[in]pTexture書き込み先のテクスチャのポインタ
[in]infoTextureShader のTextureShaderInfo
[in]width書き込み先のテクスチャの幅
[in]height書き込み先のテクスチャの高さ
[in]mipCountミップマップ数
事前条件
pDevice != NULL
&& pTexture != NULL && pTexture が初期化されている
&& TextureShaderInfo にテクスチャとして有効なフォーマットが設定されている
&& 0 < width && 0 < height && 0 < mipCount

詳細
書き込み先のテクスチャを設定します。
内部で nn::gfx::ColorTargetViewを構築します。
一つ前のColorTargetViewをFinalizeしてから処理します。
このFacelineGpuBuffer渡すをTextureShaderの初期化時に設定した
TextureShaderInfoを渡してください。

各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).
