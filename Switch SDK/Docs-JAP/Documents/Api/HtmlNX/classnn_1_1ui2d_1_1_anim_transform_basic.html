[in]pResAccessorリソースアクセサです。
[in]pResリソースのアニメーションブロックへのポインタです。

詳細
Bind() でアニメーションに関連付けるアニメーションブロックデータと アニメーションデータに必要なリソースを設定します。

また、 Bind() で必要になるBindPairの配列 を アニメーションブロックに含まれるアニメーション対象の個数分確保します。

この関数の実装では SetAnimResource() を使用して、アニメーション ブロックデータを登録しなければなりません。

virtual void nn::ui2d::AnimTransformBasic::ForceBindPane ( Pane *  pDstPane,
const Pane *  pSrcPane 
)
virtual

強制的に別のペインのアニメーションの関連付けを行います。

詳細
pDstPane で指定したペインに、pSrcPaneで指定したペインの アニメーションを関連付けます。

本来異なるペインに付いているアニメーションをバインドすることになりますので、 例えばペインの種類が異なっていたりすると不正な動作になってしまいます。 十分に注意して使用してください。

なお、関連付けたアニメーションはUnbindPaneで解除できます。

引数
[in]pDstPane強制的に関連付けたい先のペインへのポインタです。
[in]pSrcPane強制的に関連付けたい元のアニメーションを持つペインの名前です。
