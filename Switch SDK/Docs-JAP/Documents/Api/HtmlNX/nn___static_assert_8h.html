詳解

コンパイル時に条件式が成立することをテストする API です。

マクロ定義詳解

#define NN_STATIC_ASSERT (   condition)

コンパイル時に条件式が成立することをテストします。

引数
[in]conditionコンパイル時定数である条件式

詳細
条件式 condition が true の場合、何も起こりません。 条件式 condition が false の場合、コンパイルエラーになります。

各種例:
AccountApplication.cpp, CodecOpusCodecProcessingTime.cpp, CodecOpusDecoder.cpp, CodecOpusEncoder.cpp, CodecOpusMultiStreamDecoder.cpp, MiiCompressedTexture.cpp, MiiG3dHeadwear.cpp, MiiHeadwearExample.cpp.
