BindResult nn::g3d::ResVertexShapeAnim::PreBind ( const ResShape *  pTarget)

事前にシェイプと関連付けを行います。

引数
[in]pTargetResShape へのポインター。
戻り値
BindResult を返します。

詳細
リソース同士を PreBind() によって事前に関連付けることにより、 BindFast 時には名前引きを行わない比較的高速な関連付けを行います。
