ゲームの取扱説明書を想定したサンプルオフライン HTML

概要

ゲームの取扱説明書を想定したオフライン HTML のサンプルです。
本サンプルでは下記の機能を使っています。

  • 空間ナビを独自のフォーカス表現に変更する
  • L スティックの操作を空間ナビに限定する
  • フッターを非表示にする
  • オフライン HTML から Web アプレットを呼び出す
  • JS 拡張を使って SE を鳴らす

ファイル構成

本サンプルオフライン HTML は Samples/Sources/Applications/WebDemo/html_document/sample/instruction_manual.htdocs 以下にあります。

必要な環境

本サンプルは NX 実機環境のみサポートしています。

操作方法

入力 動作
L スティックまたは上下左右ボタン 空間ナビを移動します。
A ボタン、タッチ 項目を決定します。
B ボタン ページを戻る、またはオフライン Web アプレットを終了して、アプリケーションを経由してメニューへ戻ります。

注意事項

特にありません。

実行手順

WebDemo のサンプルプログラムをビルドし、実行してください。
メニューから本サンプルを選択してください。

解説

メニューから本サンプルを選ぶと、アプリケーションを経由してオフライン Web アプレットを起動し、本サンプルオフライン HTML を表示します。

空間ナビを独自のフォーカス表現に変更する

meta 要素の focus-ring-visibility 属性を使って、空間ナビのフォーカス枠を非表示にしています。
button 要素にフォーカスしている間、 CSS の ::before 、::after 疑似要素を使って独自のフォーカス枠を表示しています。

L スティックの操作を空間ナビに限定する

nn::web::ShowOfflineHtmlPageArg::SetPointerEnabled() を使って、L スティックの操作を空間ナビに限定しています。

フッターを非表示にする

nn::web::ShowOfflineHtmlPageArg::SetFooterEnabled() を使って、フッターを非表示にしています。

オフライン HTML から Web アプレットを呼び出す

オフライン HTML の "Official Website" の項目を選ぶと、"http://localhost/" から始まる URL へ遷移します。
オフライン Web アプレットは、この URL をコールバック URL と判定してオフライン Web アプレットを終了し、URL をアプリケーションへ返します。
アプリケーションは nn::web::OfflineHtmlPageReturnValue::GetLastUrl() を使って Web アプレットで開く URL を取得し、Web アプレットを起動して Web ページを表示します。
Web ページから戻ったら、再びオフライン HTML を表示します。

JS 拡張を使って SE を鳴らす

button 要素に nx-se-disabled 属性を追加して、標準の SE を無効にしています。
JS 拡張 window.nx.playSystemSe() を使って、システム SE を鳴らしています。
