SDK サンプル解説書

SDK サンプルの解説書です。

はじめに

ここでは、SDK のサンプルプログラムについて解説します。

サンプルについて

サンプルコードは Samples/Sources/Applications/ 以下にあります。

一部のサンプルは、サンプルで共通に使う処理をまとめたサンプル用のライブラリ libnns* を使用しています。 libnns* のソースコードは Samples/Sources/Libraries/ 以下にあります。

サンプルや libnns* のソースコードはコピー＆ペーストをしてアプリケーションで使用しても構いませんが、 使い方を分かりやすく例示するために冗長な部分があったり、 用いているパラメータが最適化されていない場合もあります。 内容をよくご確認ください。

不具合やご質問への対応等についてはサンプルや libnns* ライブラリに関しても通常のライブラリ同様に、サポート対象となります。

注意
サンプルやデモプログラムなどで使用されている画像データや音楽データの著作権は、一部を除き任天堂株式会社に帰属します。 これらのデータは、サンプルやデモプログラム以外ではご使用いただけません。
