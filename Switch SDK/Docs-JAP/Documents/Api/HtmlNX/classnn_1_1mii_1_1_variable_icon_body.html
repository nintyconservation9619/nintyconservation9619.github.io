VariableIconBodyを初期化します。

引数
[in]pMemoryVariableIconBodyが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pDeviceデバイス
[in]pMemoryPoolVariableIconBodyが使用するメモリプール
[in]memoryPoolOffsetVariableIconBodyが使用するメモリプールのオフセット
[in]memoryPoolSizeVariableIconBodyが使用するメモリプールのサイズ
[in]pBody体リソースファイルのポインタ
[in]bodySizepBodyのサイズ
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& オフセット、サイズが正常
&& charInfoが壊れていない。
事後条件
成功したら VariableIconBody::IsInitialized() = true

詳細
pBodyは、初期化後破棄できます。

注意
pMemory 、 pMemoryPool は、
VariableIconBody::Finalize() まで保持する必要があります。
初期化済みのVariableIconBodyを再初期化することは出来ません。
Finalize() してから呼び出します。
初期化時に渡したメモリプールが CpuCached な場合、
この関数の呼び出し後に、メモリプールをフラッシュする必要があります。
各種例:
