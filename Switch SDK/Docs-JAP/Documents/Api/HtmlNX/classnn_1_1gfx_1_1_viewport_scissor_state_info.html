void nn::gfx::ViewportScissorStateInfo::SetViewportStateInfoArray ( const ViewportStateInfo *  pViewportStateInfoArray,
int  viewportStateCount 
)

ビューポートステート情報の配列を設定します。

引数
[in]pViewportStateInfoArrayビューポートステート情報の配列へのポインター
[in]viewportStateCountビューポートステートの数
事前条件
  • viewportStateCount >= 0 && viewportStateCount <= 65535
事後条件
  • ビューポートステート情報の配列とその数が設定されている

詳細
ポインター先の内容はオブジェクトの初期化後に破棄できます。 シザリングが有効の場合、ビューポートステートの数はシザーステートの数と等しい必要があります。

Edit.cpp, GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, GfxSimple.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計19項目).
