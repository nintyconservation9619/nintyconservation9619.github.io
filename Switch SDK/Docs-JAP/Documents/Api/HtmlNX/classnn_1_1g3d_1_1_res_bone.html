enum nn::g3d::ResBone::Flag

ボーンに関するフラグです。

列挙値
Flag_Visibility 

ビジビリティの初期値です。

Flag_BillboardNone 

ビルボード計算をしません。

Flag_BillboardChild 

階層ビルボードの子として計算を行います。

Flag_BillboardWorldViewVector 

World 方式でビルボード計算を行います。

Flag_BillboardWorldViewPoint 

WorldViewpoint 方式でビルボード計算を行います。

Flag_BillboardScreenViewVector 

Screen 方式でビルボード計算を行います。

Flag_BillboardScreenViewPoint 

ScreenViewpoint 方式でビルボード計算を行います。

Flag_BillboardYaxisViewVector 

YAxis 方式でビルボード計算を行います。

Flag_BillboardYaxisViewPoint 

YAxisViewpoint 方式でビルボード計算を行います。

Flag_SegmentScaleCompensate 

内部用機能のため使用禁止です。

MayaSSC で計算します。

Flag_ScaleUniform 

Scale が等方です。

Flag_ScaleVolumeOne 

Scale の体積が１です。

Flag_RotateZero 

Rotate が 0 です。

Flag_TranslateZero 

Translate が 0 です。

Flag_ScaleOne 

Scale が１です。

Flag_RotTransZero 

Rotate, Translate が 0 です。

Flag_Identity 

トランスフォームが単位行列です。

Flag_HiScaleUniform 

累積 Scale が等方です。

Flag_HiScaleVolumeOne 

累積 Scale の体積が１です。

Flag_HiRotateZero 

累積 Rotate が 0 です。

Flag_HiTranslateZero 

累積 Translate が 0 です。

Flag_HiScaleOne 

累積 Scale が１です。

Flag_HiRotTransZero 

累積 Rotate, Translate が 0 です。

Flag_HiIdentity 

累積トランスフォームが単位行列です。

Flag_BillboardIndexNone 

階層ビルボードの影響を受けるボーンがありません。

enum nn::g3d::ResBone::RotateMode

ボーンに関する Rotate モード

列挙値
RotateMode_Quat 

Quaternion 方式で回転を計算します。

RotateMode_EulerXyz 

Euler 角方式で回転を計算します。

enum nn::g3d::ResBone::MirroringState

内部用機能のため使用禁止です。 ボーンの位置情報を示すフラグです。モーションミラーリングの計算に使用します。

列挙値
MirroringState_CenterRotate 

回転前の中央のボーン。

MirroringState_CenterPostRotate 

回転値が入った中央のボーン

MirroringState_SideRoot 

回転後にローカル軸の向きが変わった後の中央のボーン。

MirroringState_Side 

ジョイントのミラーで生成された左右のボーンのルート。
