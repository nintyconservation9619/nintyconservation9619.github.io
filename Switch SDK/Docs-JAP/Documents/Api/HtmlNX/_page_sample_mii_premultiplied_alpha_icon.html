背景とのブレンドに乗算済みアルファを使用したアイコンの生成機能サンプルプログラムの解説です。

概要

似顔絵ライブラリで提供しているアイコンシェーダの乗算済みアルファ機能を使用してアイコン生成するサンプルです。 乗算済みアルファの有効・無効を切り替えて生成したアイコンと顔モデルを直接描画したものを比較します。 左側 : 乗算済みアルファを使用したアイコン 真ん中 : Mii の顔モデルを描画 右側 : 通常のアイコン 半透明メガネと背景の重なりの色が左側と真ん中でほぼ一致しているのに対し、右側とは不一致なことが分かります。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/MiiPremultipliedAlphaIcon 以下にあります。

必要な環境

特にありません。

操作方法

特にありません。

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは、Mii のアイコンを生成し、 SetPremultipliedAlphaBlendForBackground() の有効無効による画像の違いを比較できます。

コンバート処理

  • Samples/Sources/Applications/MiiPremultipliedAlphaIcon/Resources/SampleShader.glsl

    アイコンテクスチャと Mii 作成・編集アプレットと同等の Mii の描画を行うシェーダです。

サンプルプログラム実行時の処理

このサンプルプログラムの処理の流れは以下の通りです。

  • 描画のための各種リソースの初期化
  • 似顔絵リソースファイルの読み込み
  • Mii リソース（nn::mii::Resource）の初期化
  • nn::mii::CharInfo を取得
  • nn::mii::CharModel の初期化
  • nn::mii::Faceline の初期化
  • nn::mii::Mask の初期化
  • デスクリプタスロットの割り当て
  • Faceline、Mask テクスチャ用シェーダ（nn::mii::TextureShader）の初期化
  • nn::mii::FacelineGpuBuffer の初期化
  • nn::mii::MaskGpuBuffer の初期化
  • アイコン書き込み先テクスチャの初期化
  • 結果を格納するテクスチャの初期化
  • デプスステンシル用テクスチャの初期化
  • アイコンテクスチャ用シェーダ（nn::mii::IconShader）の初期化
  • nn::mii::IconGpuBuffer の初期化
  • Mii 用のサンプラの作成
  • テクスチャビューの作成とデスクリプタプールへの登録
  • nn::mii::TextureShader を使用して Faceline と Mask のテクスチャのレンダリング
  • nn::mii::IconShader を使用して乗算済みアルファがオフの Mii アイコンのレンダリング
  • nn::mii::IconShader を使用して乗算済みアルファがオンの Mii アイコンのレンダリング
  • アイコンの描画と同等のシェーダを使用して Mii の顔モデルをレンダーターゲットに直接描画（真ん中）
  • 乗算済みアルファでないテクスチャ(左)と乗算済みアルファのテクスチャ(右)をそれぞれ専用のブレンドステートを使用して画面表示
