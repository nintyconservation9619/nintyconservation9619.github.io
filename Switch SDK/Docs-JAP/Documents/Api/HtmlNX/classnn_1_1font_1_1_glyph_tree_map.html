詳解

テクスチャキャッシュ内のグリフを管理するためのTreeMapです。

詳細
左傾赤黒木という平衡二分木のアルゴリズムを利用してます。

ライブラリユーザが直接使用するクラスではないため、詳しいDoxyコメントは 付けていません。
