const ResBone *  pBone 
) const

引数
[out]pResultアニメーション結果を格納するバッファーへのポインター。

詳細
アニメーションがスケール、回転、移動の初期値を持たない場合、pBone で示されるボーンのスケール、回転、移動を使用して初期化します。 アニメーションがスケール、回転、移動の初期値を持たず、かつ pBone が NULL の場合、スケール 1、回転 0、移動 0 で初期化します。
