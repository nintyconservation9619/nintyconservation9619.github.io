enum nn::image::PixelFormat

画像データのピクセルフォーマットを表す列挙子

列挙値
PixelFormat_Rgba32 

RGBA各8bitで表現された計32bitのピクセルデータ

PixelFormat_Rgb24 

RGB各8bitで表現された計24bitのピクセルデータ

enum nn::image::JpegStatus

JPEGライブラリが返し得るステータスコード

詳細
JPEGライブラリが提供する機能のうち処理の成否があるものは、ここに挙げるステータスコードのうちいずれかを返却 します。 返るステータスコードやその要因は各APIごとに異なります。詳細は各APIのリファレンスを参照してください。

列挙値
JpegStatus_Ok 

成功

JpegStatus_WrongFormat 

不正な形式のデータの入力

JpegStatus_UnsupportedFormat 

サポートしていない形式のデータの入力

JpegStatus_OutOfMemory 

ワークメモリの不足

JpegStatus_ShortOutput 

出力バッファの不足

JpegStatus_OutputOverabounds 

アプリケーションによる入力データの過剰

enum nn::image::JpegSamplingRatio

JPEGデータのサンプリング比

詳細
JPEGエンコードを行う際に指定可能な、サンプリング比を表します。 一般に420, 422, 444の順で画質が向上し、出力データサイズが増加します。

列挙値
JpegSamplingRatio_444 

Y,Cb,Crをそれぞれ4:4:4の割合でサンプリングします。

JpegSamplingRatio_422 

Y,Cb,Crをそれぞれ4:2:2の割合でサンプリングします。

JpegSamplingRatio_420 

Y,Cb,Crをそれぞれ4:2:0の割合でサンプリングします。

enum nn::image::ExifOrientation

画像の方向

詳細
JPEGデータのメタ情報の一つである 「画像方向」を表します。

Rotate とつく項目は「画像がどう回転したか」を示すもので、 「撮影機器がどう回転したか」を示さないことに注意してください。 撮影機器の回転角を考えるときは、符号が逆になります。 また、回転角は時計回りでみた値を示します。

列挙値
ExifOrientation_Normal 

回転していない状態

ExifOrientation_FlipHorizontal 

画像を左右反転した状態

ExifOrientation_Rotate180 

画像を180度回転した状態

ExifOrientation_FlipVertical 

画像を上下反転した状態

ExifOrientation_FlipTopRightToLeftBottom 

画像を、左上と右下を結ぶ直線を軸に反転した状態

ExifOrientation_Rotate270 

画像を時計回りに270度回転した状態

ExifOrientation_FlipTopLeftToRightBottom 

画像を、右上と左下を結ぶ直線を軸に反転した状態

ExifOrientation_Rotate90 

画像を時計回りに90度回転した状態
