詳解

内部用機能のため使用禁止です。 SkeletalAnimBlender::Initialize() に渡して初期化を行うパラメータです。

構築子と解体子

nn::g3d::SkeletalAnimBlender::InitializeArgument::InitializeArgument ( )

コンストラクタです。

インスタンスで使用するすべてのスケルトンリソースに対してReserve() するか、 SetMaxBoneCount()、でバッファーサイズの計算に必要なパラメータを指定する必要があります。
