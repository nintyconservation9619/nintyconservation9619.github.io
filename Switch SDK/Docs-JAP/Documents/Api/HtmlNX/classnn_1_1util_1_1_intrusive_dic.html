template<class T, class NodeTraits>
class nn::util::IntrusiveDic< T, NodeTraits >

侵入型辞書です。

テンプレート引数
T要素の型です。要素は IntrusiveDicNode をメンバに持つか、 IntrusiveDicBaseNode を継承した型である必要があります。
NodeTraits要素の特性を示す型です。要素が IntrusiveDicNode をメンバに持つ場合は IntrusiveDicMemberNodeTraits を、IntrusiveDicBaseNode を継承した型である場合は IntrusiveDicBaseNodeTraits を指定する必要があります。
