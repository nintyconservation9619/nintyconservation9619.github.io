詳解

複数区間計測可能な GPU 負荷メータークラスです。

詳細
GPU タイムスタンプを取得して計測を行いますが、最終的には CPU 時間に変換された Tick 値が Section 構造体に格納されます。

virtual void nn::perf::GpuMeter::Next ( )
virtual

自身とアタッチされている負荷メーターで GPU に記録されたタイムスタンプの値を読み出し、計測結果の格納先バッファを切り替えます。

詳細
毎フレームの終了時に呼び出してください。この関数が呼ばれない場合、正常な測結果を得ることができません。
「バッファ数 - 2」フレーム前にコマンドに積んだ計測結果を集計します。
アタッチされている子負荷メータの Next() も内部で再帰的に呼ばれるため、親負荷メータの Next() だけを呼んでください。

nn::perf::LoadMeterBaseを再実装しています。
