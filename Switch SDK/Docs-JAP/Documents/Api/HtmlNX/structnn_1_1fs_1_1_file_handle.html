nn::fs::FileHandle 構造体

ファイルを扱うためのハンドルです。 [詳解]

#include <nn/fs/fs_File.h>

詳解

ファイルを扱うためのハンドルです。

詳細
あるファイルハンドルに対して、内部の状態異常による操作の失敗が発生した場合、それ以降そのファイルハンドルに対しては異常状態フラグが立ちます。
異常状態にあるファイルハンドルに対して Close 以外の操作が行われた場合、適切なハンドリングを促すために開発者に通知が行われます。
