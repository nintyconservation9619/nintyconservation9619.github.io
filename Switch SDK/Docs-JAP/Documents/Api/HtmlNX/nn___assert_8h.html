条件を満たしていることをテストする API の宣言

マクロ定義詳解

#define NN_ASSERT (   condition,
  ... 
)    NN_DETAIL_ASSERT_IMPL((condition), NN_MACRO_STRINGIZE(condition), ##__VA_ARGS__);

条件を満たしていることをテストします。

詳細
条件式 condition が true に評価される場合、何も行いません。

条件式 condition が false に評価される場合、Assertion 失敗ハンドラを呼び出します。

NN_ENABLE_ABORT_MESSAGE が定義されている場合、指定されたメッセージが Assertion 失敗ハンドラに渡されます。 NN_DISABLE_ABORT_MESSAGE が定義されている場合、空のメッセージが渡されます。 どちらのマクロも定義されていない場合は、NintendoSDK のビルドタイプによって渡されるメッセージが決定されます:

  • Debug または Develop の場合 : 指定されたメッセージ
  • Release の場合 : 空のメッセージ

このマクロは、NN_ENABLE_ASSERT が定義されている場合に有効です。 NN_DISABLE_ASSERT が定義されている場合、空の処理に展開されます。

このマクロは既定で有効です。
