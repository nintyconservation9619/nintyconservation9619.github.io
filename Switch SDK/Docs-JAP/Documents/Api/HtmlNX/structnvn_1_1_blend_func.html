詳解

Identifies a coefficient to be multiplied with each component of the source or destination color when blending.

For the default BlendEquation::ADD mode, blending is performed by evaluating 'C = Fs * Cs + Fd * Cd' for each color component. In these equations, Cs represents the fragment shader output (source) color and Cd represents the color value stored in the framebuffer. The BlendFunc enum specifies values used for the blend factor terms Fs and Fd. There are separate blend functions for RGB and alpha color components.
