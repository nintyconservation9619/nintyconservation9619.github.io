bool nn::g3d::ModelObj::SetupBlockBuffer ( nn::gfx::Device *  pDevice,
nn::gfx::MemoryPool *  pMemoryPool,
ptrdiff_t  offset,
size_t  memoryPoolSize 
)

メモリプールを渡して、ユニフォームブロックを構築します。

引数
[in]pDeviceデバイスへのポインター。
[in]pMemoryPoolメモリプールへのポインター。
[in]offsetメモリプールの使用する領域へのオフセット。
[in]memoryPoolSizeoffset以降で使用可能なメモリプールのサイズ。
戻り値
構築に成功した場合は true、失敗した場合は false を返します。
事前条件
  • pDevice が初期化されている。
  • pMemoryPool が初期化されている。
  • offset で指されたメモリプールの位置が nn::gfx::Buffer::GetBufferAlignment() のアラインメント上にある。
  • memoryPoolSize は CalculateBlockBufferSize() 以上である。
  • ユニフォームブロックが未構築である。
事後条件
  • ユニフォームブロックが構築されている

void nn::g3d::ModelObj::CalculateBounding ( int  lodLevel)

指定したLODレベルのバウンディング球を計算します。

引数
[in]lodLevelLODレベル。

詳細
すべての ShapeObj を走査し、モデルのバウンディング球を計算します。 ShapeObj が指定した LOD レベルを持たない場合、LOD レベル 0 のデータで代用し、 計算を行います。

void nn::g3d::ModelObj::CalculateSkeleton ( int  bufferIndex)

ワールド行列から行列パレットを計算します。

引数
[in]bufferIndex計算を反映させるユニフォームブロックのインデックス。
事前条件
  • インデックスはスケルトンが持つユニフォームブロックのバッファー数の範囲内。

詳細
スケルトンのユニフォームブロックに計算結果は反映されます。

GPU から参照されるバッファーを書き換えるため、 前回の描画で GPU が参照し終わった後に呼ぶ必要があります。 構築時のオプションでダブルバッファー化するか、 描画完了待ち後に呼ぶことを想定しています

各種例:
SkeletalAnimation.cpp.
