nn::atk::viewer 名前空間

サウンドインゲーム編集ライブラリの名前空間です。

このライブラリを利用すると、ゲームで利用中のサウンドアーカイブの内容を SoundMaker から編集することができるようになります。
詳しい使い方については、各クラスの解説と edit デモを参照ください。
【注意】
このライブラリには、β版のクラスが含まれています。
これらのクラスが提供する機能は、予告なく変更される可能性があります。 [詳解]
