サンプルプログラム Town の解説です。実際のゲームに近い内容のサンプルプログラムです。

概要

遠景や地形、キャラクタなど複数のモデルを表示する、実際のゲームに近い内容の処理を行うサンプルです。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/G3dDemo 以下にあります。

必要な環境

特になし。

操作方法

Joy-Con、DebugPad、PCの場合キーボード、マウスによる入力を用いて、以下の操作が可能です。

各モードにおいて共通操作

入力 動作
Start ボタン、スペースキーまたはタッチスクリーン長押し、Joy-Con の+ボタン メニューに戻ります。
左スティックまたはマウス左ボタン押し カメラ回転。マウスの場合、画面の中心点からクリック位置への方向、距離によって回転します。
右スティックまたはマウス右ボタン押し カメラ移動。マウスの場合、画面の中心点からクリック位置への方向、距離によって移動します。
L ボタンまたは L キー エディットモードとデバッグモードを切り替えます。
R ボタンまたは R キー メニュー表示のオン、オフを切り替えます。

エディットモード

入力 動作
十字ボタン上下またはカーソルキー上下 エディット項目の選択を切り替えます。
十字ボタン左右またはカーソルキー左右 選択されているエディット項目の値を増減します。
Selected Model 選択対象のモデルを変更します。
Selected Shape 選択中のモデルに含まれる選択対象のシェイプを変更します。
Anim Speed エディット対象のモデルのアニメーションスピードを変更します。
A ボタンまたは A キー 3dEditor が接続されている場合、モデルをアタッチします。接続されていない場合は何もしません。
B ボタンまたは B キー 3dEditor が接続されている場合、画面輝度平均値計算用シェーダーをアタッチします。接続されていない場合は何もしません。
X ボタンまたは X キー 3dEditor が接続されている場合、シャドウシェーダーをアタッチします。接続されていない場合は何もしません。
Y ボタンまたは Y キー 3dEditor が接続されている場合、G バッファー用シェーダーをアタッチします。接続されていない場合は何もしません。
ZR ボタンまたは Z キー 選択しているモデルにカメラが移動します。

デバッグモード 1

入力 動作
十字ボタン上下またはカーソルキー上下 デバッグ項目の選択を切り替えます。
十字ボタン左右またはカーソルキー左右 選択されているデバッグ項目の値を切り替えます。
Selected Model 選択対象のモデルを変更します。
Selected Shape 選択中のモデルに含まれる選択対象のシェイプを変更します。
Model Animation モデルアニメーションのオン、オフを切り替えます。
Light Animation ライトアニメーションのオン、オフを切り替えます。
Wireframe ワイヤーフレーム表示のオン、オフを切り替えます。
Ice 雪シェーダーのオン、オフを切り替えます。
Shadow シャドウ処理のオン、オフを切り替えます。
Water 水面処理のオン、オフを切り替えます。
Geometry G バッファー処理のオン、オフを切り替えます。
Light ライトプリパス処理のオン、オフを切り替えます。
Average 画面の輝度変化の平滑処理のオン、オフを切り替えます。

デバッグモード 2

入力 動作
十字ボタン上下またはカーソルキー上下 デバッグ項目の選択を切り替えます。
十字ボタン左右またはカーソルキー左右 選択されているデバッグ項目の値を切り替えます。
Selected Model 選択対象のモデルを変更します。
Selected Shape 選択中のモデルに含まれる選択対象のシェイプを変更します。
Bounding Sphere バウンディングスフィア表示のオン、オフを切り替えます。
Bounding Box バウンディングボックス表示のオン、オフを切り替えます。
Single Shape 選択されているシェイプの単体表示のオン、オフを切り替えます。
Lod Level LODレベルの色分け表示のオン、オフを切り替えます。

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、メニューから Town を選択し、実行してください。

解説

サンプルプログラムの処理の流れは以下の通りです。

  • コマンドバッファーを初期化
  • 各種レンダーターゲットを初期化
  • シェーダーアーカイブ読み込み、初期化
  • モデルリソース読み込み、初期化およびテクスチャーの関連付け
  • シェーダー、動的テクスチャーとモデルの関連付け
  • モデルとアニメーションの関連付け
  • ループ開始
  • 前フレームに作成したコマンドリストの実行開始
  • 各行列計算
  • モデル更新およびシェーダー更新
  • 水面描画のコマンド生成
  • デプスマップ生成のコマンド生成
  • Gバッファー生成のコマンド生成
  • ライトプリパスのコマンド生成
  • 最終描画結果のためのコマンド生成（不透明)
  • 最終描画結果のためのコマンド生成（半透明)
  • 画面輝度計算のコンピュートシェーダーのためのコマンド生成
  • コマンドリストの実行完了待ち
  • GPU待ちが必要な計算処理
  • スキャンバッファーをスワップ
  • ループ開始に戻る
