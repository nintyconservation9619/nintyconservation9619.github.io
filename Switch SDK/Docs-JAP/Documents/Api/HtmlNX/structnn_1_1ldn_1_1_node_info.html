詳細
ネットワークに参加しているノードの情報です。

ipv4Address にはノードに割り当てられた IPv4 アドレスが格納されており、 socket ライブラリではこのアドレスを使用して通信を行います。 LDN ネットワークに接続した後、切断されるまで変化することはありません。 また、一度切断されたとしても、接続先のネットワークが同じであれば、 基本的には前回と同じ IPv4 アドレスが 割り当てられます。 ただし、ステーションが切断されている間に 多数の新しいステーションがネットワークへの接続と切断を繰り返した場合には 古い IPv4 アドレスが別のステーション向けに再利用される可能性があるため、 必ずしも再接続時に前回と同じ IPv4 アドレスが割り当てられることは保証されません。

nodeId にはノード番号が格納されており、 LdnNetworkInfo 構造体の nodes のインデックスに対応しています。 アクセスポイントには必ずノード番号 0 が、 ステーションにはノード番号 1 ～ StationCountMax が割り当てられます。 一度 LDN ネットワークに接続すると切断されるまで変化することはありません。 IPv4 アドレスとは異なり再接続時に同じノード番号が優先利用されることはありません。

isConnected はそのノードの接続状態で、 この値が true であればノードが接続中であることを意味します。 isConnected の値が false の場合、 ipv4Address や nodeId には有効な値が格納されていません。
