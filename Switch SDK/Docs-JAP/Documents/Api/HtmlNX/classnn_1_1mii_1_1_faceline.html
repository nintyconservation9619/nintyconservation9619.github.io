Facelineを初期化します。

引数
[in]pMemoryFacelineが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pDeviceデバイス
[in]pMemoryPoolFacelineが使用するメモリプール
[in]memoryPoolOffsetFacelineが使用するメモリプールのオフセット
[in]memoryPoolSizeFacelineが使用するメモリプールのサイズ
[in]pTemporaryBuffer一時バッファ用の構造体へのポインタ
[in]resourceFacelineの構築用リソース
[in]charInfo生成するMiiのCharInfo
[in]isVerticalFlip上下反転するかのフラグ
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& Resource::IsInitialized()
&& ! Faceline::IsInitialized()
&& pTemporaryBuffer != NULL
&& オフセット、サイズが正常
&& charInfoが壊れていない。
事後条件
成功したらFaceline::IsInitialized() = true
注意
pMemory および pMemoryPool は、Faceline::Finalize() まで保持する必要があります。
pMemoryPool には MemoryPoolProperty_CpuCached が設定されたメモリプールを渡してください。
resource は、Resource の設定条件によって
Faceline::Finalize() まで保持の必要があります。
初期化済みのFacelineを再初期化することは出来ません。
Finalize() してから呼び出します。
pTemporaryBufferは初期化中のみ必要です。
初期化が終われば破棄してかまいません。
上下反転フラグ isVerticalFlip はアプリケーションの座標系設定に応じて設定してください。
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).
