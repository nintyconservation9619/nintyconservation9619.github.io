#define NN_FUNCTION_LOCAL_STATIC (   type,
  name,
  ... 
)

C++11 以降の関数ローカルな static 変数定義します。

引数
[in]type定義する型
[in]name定義する変数の名前
[in]...変数の初期化部

詳細
VC++2012 と VC++2013 環境で、関数内における以下のコードの C++11 以降でのエミュレートを行います。

1 static type name __VA_ARGS__

name で指定された変数の初期化はスレッドセーフに行われます。

それ以外の環境では、上記のコードそのままに展開されます。

使用例：

1 class C
2 {
3 public:
4  C();
5  NN_IMPLICIT C(int);
6  C(int, int);
7 };
8 void F()
9 {
10  NN_FUNCTION_LOCAL_STATIC(C, x0); // static C x0;
11  NN_FUNCTION_LOCAL_STATIC(C, x1a, = 10); // static C x1a = 10;
12  NN_FUNCTION_LOCAL_STATIC(C, x1b, (10)); // static C x1b(10);
13  NN_FUNCTION_LOCAL_STATIC(C, x2, (10, 20)); // static C x2(10, 20);
14 }
各種例:
Demo1/Demo1.cpp, HidNpadIntegrate_Main.cpp.
#define NN_DEPRECATED

対象の利用は非推奨であることを宣言します。

詳細
関数に対しては、宣言・定義の先頭に本マクロを記述してください。 構造体・クラスに対しては、struct/class キーワードと識別子の間に本マクロを記述してください。 これ以外の場所に記述した場合、環境によっては正しくビルドできません。

使用例：

1 NN_DEPRECATED
2 void DoSomething()
3 {
4 };
5 
6 class NN_DEPRECATED C
7 {
8 };
