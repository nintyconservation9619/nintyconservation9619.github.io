nn::irsensor::ClusteringProcessorConfig 構造体

クラスタリングプロセッサの設定をおこなう構造体です。 [詳解]

#include <nn/irsensor/irsensor_ClusteringProcessorTypes.h>

公開変数類

IrCameraConfig irCameraConfig
 IR カメラの設定です。
 
bool isExternalLightFilterEnabled
 外部光フィルタ機能が有効かどうか。
 
int objectIntensityMin
 検出するオブジェクトを構成するピクセルの最小輝度値です（0 以上 IrCameraIntensityMax 以下）。
 
int objectPixelCountMax
 検出するオブジェクトの最大サイズです（0 以上 ClusteringProcessorObjectPixelCountMax 以下）。
 
int objectPixelCountMin
 検出するオブジェクトの最小サイズです（0 以上 ClusteringProcessorObjectPixelCountMax 以下）。
 
Rect windowOfInterest
 撮影画像で処理対象となる矩形領域です（左上隅のピクセルを (0, 0) とする、幅 IrCameraImageWidth、高さ IrCameraImageHeight の領域内）。
