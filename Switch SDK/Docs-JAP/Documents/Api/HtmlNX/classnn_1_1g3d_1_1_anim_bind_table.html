アニメーションの関連付け状態を管理するクラスです。

列挙型メンバ詳解

enum nn::g3d::AnimBindTable::BindFlag

内部用機能のため使用禁止です。 関連付けに関するフラグです。

列挙値
BindFlag_Enabled 

有効です。

BindFlag_SkipCalculate 

更新処理を無視するが、最後の結果が Apply されます。

BindFlag_SkipApply 

Apply を無視するので、単独では無駄な更新が行われます。

BindFlag_Disabled 

Calculate も Apply もスキップします。
