TM ライブラリの使用方法を例示したサンプルです。

概要

ここでは、TM ライブラリを使用した、ホスト側ツールの作成方法を説明します。

TM ライブラリを使用したプロジェクトの作成方法については、 TM ライブラリ の関数リファレンス も併せて参照して下さい。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/TmSimple 以下にあります。

必要な環境

あらかじめ Target Manager をインストールしておく必要があります。

操作方法

  1. 追加可能な機材の一覧が出力されるので、追加したい機材に対応する数値を入力してください
  2. サンプルを実行可能な機材の一覧が出力されるので、サンプルを実行したい機材に対する数値を入力してください 数値入力の際は、-1 などの範囲外の数値を入力することで、その操作をスキップすることができます

注意事項

実行手順

サンプルプログラムをビルドし、実行してください。

325 void RunProgram()
326 {
327  int32_t targetCount;
328  RETURN_IF_FAILURE(nn::tm::GetTargetCount(&targetCount));
329  if (targetCount == 0)
330  {
331  printf("No target is found. Skip RunProgram\n");
332  return;
333  }
334 
335  std::unique_ptr<nn::tm::TargetHandle[]> pTargetHandles(new nn::tm::TargetHandle[targetCount]);
336  RETURN_IF_FAILURE(nn::tm::GetTargets(pTargetHandles.get(), &targetCount, targetCount));
337 
338  printf("Please input the option number of the target to run:\n");
339  ShowRegisteredTargetsWithoutDetails(pTargetHandles.get(), targetCount);
340  int selectedTarget = GetInputNumber();
341  if (0 <= selectedTarget && selectedTarget < targetCount)
342  {
343  nn::tm::TargetHandle target = pTargetHandles[selectedTarget];
344  // Guarantee that the target is connected
345  RETURN_IF_FAILURE(nn::tm::ConnectTarget(target));
346  printf("Loading %s...\n", GetSampleNspPath().c_str());
347  RETURN_IF_FAILURE(nn::tm::LaunchTargetProgram(target, GetSampleNspPath().c_str(), "", "", ""));
348 
349  nn::tm::TargetProgramName programName;
350  nn::tm::Error getNameError = nn::tm::GetTargetProgramName(&programName, target);
351  if (getNameError == nn::tm::Error_Ok)
352  {
353  printf("Running program is %s\n", programName.name);
354  }
355  else
356  {
357  nn::tm::ErrorText errorText;
358  nn::tm::GetErrorTextEn(&errorText, getNameError);
359  printf("GetTargetProgramName failed because %s(%d).\n", errorText.text, getNameError);
360  }
361 
362  nn::tm::TargetProgramState programState;
363  nn::tm::Error getStateError = nn::tm::GetTargetProgramState(&programState, target);
364  if (getStateError == nn::tm::Error_Ok)
365  {
366  printf("Program state is %s\n", ProgramStateName[programState + 1]);
367  }
368  else
369  {
370  nn::tm::ErrorText errorText;
371  nn::tm::GetErrorTextEn(&errorText, getStateError);
372  printf("GetTargetProgramName failed because %s(%d).\n", errorText.text, getStateError);
373  }
374 
375  printf("Terminating the application... ");
376  RETURN_IF_FAILURE(nn::tm::KillTargetProgram(target));
377  printf("Success\n");
378  }
379 }

サンプルプログラムの解説

先のサンプルプログラムの全体像は以下の通りです。

  • この PC で動作する Target Manager に登録可能なターゲットのリストアップを行う
  • この Target Manager にすでに登録されているターゲットのリストアップを行う
  • 登録可能なターゲットを提示し、指定されたターゲットを Target Manager に登録する
  • 登録されているターゲットを提示し、指定されたターゲット上でサンプルを実行する
