各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSystemResource.cpp.

構築子と解体子

nn::gfx::DepthStencilViewInfo::DepthStencilViewInfo ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解

void nn::gfx::DepthStencilViewInfo::SetDefault ( )

各パラメーターを既定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetImageDimension( nn::gfx::ImageDimension_2d );
  • SetMipLevel( 0 );
  • EditArrayRange().SetDefault();
  • SetTexturePtr( NULL );
各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSystemResource.cpp.
