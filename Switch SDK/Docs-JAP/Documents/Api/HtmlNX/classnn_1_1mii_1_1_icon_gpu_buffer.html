詳解

アイコンテクスチャ描画用バッファ

詳細
IconShader でアイコンテクスチャを描画するときに
必要なバッファなどをまとめたクラスです。
このクラスは、1つのアイコンテクスチャを描画するごとに1つ必要です。

注意
IconShader を使わない場合は不要です。
各種例:
MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon.cpp, MiiTransparentAdjuster.cpp.

IconGpuBufferを初期化します。

注意
pMemory および pMemoryPoolは、IconGpuBuffer::Finalize() まで保持する必要があります。
初期化済みのIconGpuBufferを再初期化することは出来ません。
Finalize() してから呼び出します。
引数
[in]pMemoryIconGpuBufferが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pDeviceデバイス
[in]pMemoryPoolIconGpuBufferが使用するメモリプール
[in]memoryPoolOffsetIconGpuBufferが使用するメモリプールのオフセット
[in]memoryPoolSizeIconGpuBufferが使用するメモリプールのサイズ
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& ! IconGpuBuffer::IsInitialized()
&& オフセット、サイズが正常
事後条件
成功したらIconGpuBuffer::IsInitialized() = true
各種例:
MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon.cpp, MiiTransparentAdjuster.cpp.

CharModelを設定する
ガンマの値を直接指定するタイプの本関数は廃止予定です。
ガンマの値を指定しない SetCharModel を使用してください。

引数
[in]model構築済みmodel
[in]charInfoキャラインフォ
[in]modelTypeモデルの種類
[in]nose鼻の種類
[in]maskマスクの番号
[in]gammaTypeガンマの種類

詳細
CharModelから必要な情報を設定します。
仮でガンマの種類を渡すようにしています。
CharModel作成時に使用したリソースにあわせたガンマを設定してください。

注意
初期化時に渡したメモリプールが CpuCached な場合、
この関数の呼び出し後に、メモリプールをフラッシュする必要があります。
各種例:
MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon.cpp, MiiTransparentAdjuster.cpp.
