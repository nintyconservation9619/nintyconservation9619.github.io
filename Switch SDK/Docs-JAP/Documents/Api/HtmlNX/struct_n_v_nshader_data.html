詳解

Data structure representing the shader data.

The data structure is for one shader stage. The contents of this struct contain everything NVN needs to set up the program for use with nvnProgramSetShaders

各種例:
AssetFileLoadingHelper.cpp, MultiThreadedAssetFileLoadingHelper.cpp.

メンバ詳解

const void* NVNshaderData::control

Control section from the offline compiler.

This address should be aligned to 8 bytes.
