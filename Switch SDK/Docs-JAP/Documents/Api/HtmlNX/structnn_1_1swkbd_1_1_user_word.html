nn::swkbd::UserWord 構造体

ユーザー辞書登録単語構造体 [詳解]

#include <nn/swkbd/swkbd_Types.h>

公開変数類

char16_t reading [UnfixedStringLengthMax+1]
 読み。文字列の最後に NULL終端 を指定する必要があります。
 
char16_t word [UnfixedStringLengthMax+1]
 表記。文字列の最後に NULL終端 を指定する必要があります。
 

詳解

ユーザー辞書登録単語構造体

詳細

  • 独自の単語を変換候補に提示したい場合に使用します。
  • reading に指定した文字列をソフトウェアキーボードで入力すると、word が予測変換結果に出力されます。
  • 例えば「任天堂」という単語を登録する場合、reading に「にんてんどう」、word に「任天堂」と指定してください。
  • ユーザーがカタカナを入力したとき、辞書に問い合わせるときにはひらがなに変換されるので、
    読みの登録はひらがなを用いてください。
  • 表記と読みは、ともに UTF-16 のみ対応しています。
