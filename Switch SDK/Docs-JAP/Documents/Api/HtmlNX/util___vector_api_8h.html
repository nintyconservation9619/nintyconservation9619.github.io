マクロ定義詳解

#define NN_UTIL_VECTOR_2F_INITIALIZER   NN_UTIL_GENERAL_VECTOR_2F_INITIALIZER

nn::util::Vector2fType に対する初期化子として利用できるマクロです。

詳細
本マクロは初期化リストとなり、以下のように記述することで nn::util::Vector2fType を初期化できます。

1 nn::util::Vector2fType vector = NN_UTIL_VECTOR_2F_INITIALIZER(0.f, 1.f);
#define NN_UTIL_VECTOR_3F_INITIALIZER   NN_UTIL_GENERAL_VECTOR_3F_INITIALIZER

nn::util::Vector3fType に対する初期化子として利用できるマクロです。

詳細
本マクロは初期化リストとなり、以下のように記述することで nn::util::Vector3fType を初期化できます。

1 nn::util::Vector3fType vector = NN_UTIL_VECTOR_3F_INITIALIZER(0.f, 1.f, 2.f);
