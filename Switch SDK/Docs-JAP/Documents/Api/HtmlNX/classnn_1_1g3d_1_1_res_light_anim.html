enum nn::g3d::ResLightAnim::Flag

ライトアニメーションに関するフラグです。

列挙値
Flag_CurveBaked 

カーブがベイク済みです。

Flag_PlayPolicyLoop 

ループすることを表します。

Flag_CurveEnable 

ライトの有効無効のアニメーションが存在します。

Flag_BaseEnable 

ライトの有効無効の初期値が存在します。

Flag_BasePos 

位置の初期値が存在します。

Flag_BaseDir 

角度の初期値が存在します。

Flag_BaseDistAttn 

距離減衰の初期値が存在します。

Flag_BaseAngleAttn 

角度減衰の初期値が存在します。

Flag_BaseColor0 

カラー0の初期値が存在します。

Flag_BaseColor1 

カラー1の初期値が存在します。
