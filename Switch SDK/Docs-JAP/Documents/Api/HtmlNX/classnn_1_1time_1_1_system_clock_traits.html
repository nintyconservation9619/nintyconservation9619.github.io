詳解

NintendoSDK のシステムクロックが扱う型を定義しているクラスです。

詳細
C++ 標準の TrivialClock の要件を満たし、 std::chrono::system_clock に準ずる機能を提供するクロックが扱う型は、このクラスの型の別名として定義されています。 このため、それらのクロックが返す time_point はすべて同一の型として扱うことができます。
