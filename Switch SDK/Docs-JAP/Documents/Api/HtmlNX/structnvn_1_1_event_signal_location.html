Mode determining when CommandBuffer::SignalEvent signals the event.

列挙型メンバ詳解

enum nvn::EventSignalLocation::Enum

Enumerated values for the type EventSignalLocation.

列挙値
TOP 

Signal the event when the command is processed by the top of the GPU pipeline.

VERTEX_PIPE 

Signal the event when vertex processing pipeline has completed work on all previously submitted draw calls.

Does not wait for rasterization and fragment processing.

BOTTOM 

Signal the event when work for all previously submitted GPU commands has completed.
