アプリ側で作成した RectDrawer を設定してセットアップを行います。

詳細
メモリープールは、nn::gfx::MemoryPoolProperty_CpuUncached、nn::gfx::MemoryPoolProperty_GpuCached プロパティを持つプールを指定してください。 サイズは、CalculateMemoryPoolSize() で、オフセットのアラインメントは、CalculateMemoryPoolAlignment() で求めます。 RectDrawer の Initialize および Finalize はアプリ側で行ってください。

lodBias に渡した値は、nn::gfx::SamplerInfo::SetLodBias に設定されます。 ミップマップを利用していて LodBias を調整したい場合にお使いください。

引数

[in]charMaxレイアウト内で描画可能な文字の最大数
[in]pExternalMemoryPoolメモリープールです。
[in]externalMemoryPoolOffset利用するメモリープール領域へのオフセットです。
[in]externalMemoryPoolSizeメモリープールのサイズです。
[in]pFontDrawerアプリ側で作成した RectDrawer です。
[in]lodBias算出されるミップマップレベルに適用するオフセット値です。
