nn::os::ThreadType 構造体

スレッドオブジェクトを定義する構造体です。 [詳解]

#include <nn/os/os_ThreadTypes.h>

詳解

スレッドオブジェクトを定義する構造体です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • CreateThread(ThreadType* thread, ThreadFunction function, void* argument, void* stack, size_t stackSize, int priority, int coreNumber)
  • CreateThread(ThreadType* thread, ThreadFunction function, void* argument, void* stack, size_t stackSize, int priority)
  • DestroyThread()
  • StartThread()
  • GetCurrentThread()
  • WaitThread()
  • YieldThread()
  • SleepThread()
  • ChangeThreadPriority()
  • GetThreadPriority()
  • GetThreadCurrentPriority()
  • SetThreadName()
  • SetThreadNamePointer()
  • GetThreadNamePointer()
  • GetCurrentCoreNumber()
  • SetThreadCoreMask()
  • GetThreadCoreMask()
  • GetThreadAvailableCoreMask()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, ThreadType* thread)

上記の関数を利用するには nn/os.h をインクルードする必要があります。
