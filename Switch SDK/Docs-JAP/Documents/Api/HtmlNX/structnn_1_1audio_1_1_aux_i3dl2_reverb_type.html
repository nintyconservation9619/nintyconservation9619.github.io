nn::audio::AuxI3dl2ReverbType 構造体

nn::audio::AuxType に適用する I3dl2Reverb エフェクトのユーティリティです。 [詳解]

#include <nn/audio/audio_AuxUtilityEffectTypes.h>

詳解

nn::audio::AuxType に適用する I3dl2Reverb エフェクトのユーティリティです。

詳細
nn::audio::AuxType によって得られたサンプルデータを入力として I3dl2Reverb エフェクト処理を行うユーティリティです。
nn::audio::ReadAuxSendBuffer() によって得られたサンプルデータを、このエフェクトへの入力に指定することで、 Interactive 3D Audio Level 2 (I3DL2) 仕様に部分的に準拠した環境リバーブエフェクトを出力します。 このエフェクトによって施される I3dl2Reverb エフェクトは nn::audio::I3dl2ReverbType による処理と同一の内容です。
ただしこのエフェクト処理は nn::audio::ProcessAuxI3dl2Reverb() を呼び出した CPU スレッド上で処理されます。
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

(ワークバッファサイズ取得)

  • GetRequiredBufferSizeForAuxI3dl2Reverb(int sampleRate, int channelCountMax)

(初期化関連)

  • InitializeAuxI3dl2Reverb()
  • ResetAuxI3dl2Reverb()

(接続先設定)

  • SetAuxI3dl2ReverbInputOutput()
  • GetAuxI3dl2ReverbInputOutput()

(パラメータ)

  • SetAuxI3dl2ReverbParameters()
  • GetAuxI3dl2ReverbParameters()

(エフェクト処理)

  • ProcessAuxI3dl2Reverb()

上記の関数を利用するには nn/audio.h をインクルードする必要があります。

各種例:
AudioEffectAuxUtility.cpp.
