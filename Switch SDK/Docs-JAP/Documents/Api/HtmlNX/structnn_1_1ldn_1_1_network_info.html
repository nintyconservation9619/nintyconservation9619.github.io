nn::ldn::NetworkInfo 構造体

スキャン結果を格納する構造体です。 [詳解]

#include <nn/ldn/ldn_Types.h>

公開変数類

 一般的なネットワーク情報を格納する構造体です。
 
LdnNetworkInfo ldn
 LDN 特有のネットワーク情報です。
 
NetworkId networkId
 ネットワークの識別子です。
 

詳解

スキャン結果を格納する構造体です。

詳細
ネットワークの情報を格納する構造体で、 Scan 関数の結果を格納し、 Connect 関数の引数としても使用します。 LDN ライブラリの利用者がこの構造体の内容を変更する必要はありません。

common には常に有効な値が格納されていますが、 networkId と ldn は一般的なアクセスポイントやネットワークがもつパラメータではありませんので、 common.networkType == NetworkType_Ldn を満たす場合にのみ有効です。

この型の変数は大きなメモリ領域を必要としますのでスタック上に確保することは推奨されません。 スタック上に確保する場合には実行時にスタックが溢れないように注意してください。

各種例:
LdnAccessPoint.cpp, LdnStation.cpp.
