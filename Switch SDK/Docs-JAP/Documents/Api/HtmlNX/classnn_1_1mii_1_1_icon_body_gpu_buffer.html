詳解

体つきアイコンテクスチャ描画用バッファ

詳細
IconShader で体つきアイコンテクスチャを描画するときに
必要なバッファなどをまとめたクラスです。
このクラスは、1つのアイコンテクスチャを描画するごとに1つ必要です。

注意
IconShader を使わない場合は不要です。
各種例:
MiiSimpleIconBody.cpp.

戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& ! IconBodyGpuBuffer::IsInitialized()
&& オフセット、サイズが正常
事後条件
成功したら IconBodyGpuBuffer::IsInitialized() = true
注意
pMemory および pMemoryPoolは、IconBodyGpuBuffer::Finalize() まで保持する必要があります。
初期化済みのIconBodyGpuBufferを再初期化することは出来ません。
Finalize() してから呼び出します。

詳細
CharModelから必要な情報を設定します。
仮でガンマの種類を渡すようにしています。
CharModel作成時に使用したリソースにあわせたガンマを設定してください。

注意
初期化時に渡したメモリプールが CpuCached な場合、
この関数の呼び出し後に、メモリプールをフラッシュする必要があります。
