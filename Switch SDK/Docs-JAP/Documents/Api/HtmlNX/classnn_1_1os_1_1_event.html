詳解

イベントを扱うためのクラスです。

詳細
イベントはスレッド間で同期を確立するための機能です。

nn::os::Event クラスは、nn::os::EventType オブジェクトを使用する 同期制御 API の呼び出しをラッピングしたユーティリティクラスです。
本クラスが提供するメンバ関数は、nn::os::EventType 型を引数にとる nn::os::WaitEvent() などの API を発行する形で実装されています。

構築子と解体子

イベントの初期状態は非シグナル状態となります。
詳細は nn::os::InitializeEvent() を参照して下さい。

関数詳解

void nn::os::Event::Wait ( )

イベントがシグナル状態になるまで待機します。

詳細
イベントがシグナル状態になるまで待機します。
イベントが nn::os::EventClearMode_AutoClear の場合、シグナル状態を検知すると 同時にイベントは非シグナル状態にクリアされます。
詳細は nn::os::WaitEvent() を参照して下さい。

bool nn::os::Event::TryWait ( )

イベントがシグナル状態かをポーリングします。

戻り値
イベントのポーリング成否を返します。

詳細
イベントがシグナル状態かどうかをポーリングします。
イベントがシグナル状態なら true を、そうでなければ false を返します。
イベントが nn::os::EventClearMode_AutoClear の場合、シグナル状態を検知すると 同時にイベントは非シグナル状態にクリアされます。
詳細は nn::os::TryWaitEvent() を参照して下さい。

戻り値

詳細
イベントをシグナル状態になるまで時限付きで待機します。
指定された timeout 時間までにイベントがシグナル状態になれば true を、 そうでなければ false を返します。
イベントが nn::os::EventClearMode_AutoClear の場合、シグナル状態を検知すると 同時にイベントは非シグナル状態にクリアされます。
詳細は nn::os::TimedWaitEvent() を参照して下さい。
