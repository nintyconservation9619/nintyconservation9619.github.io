詳解

初期化用のオプションパラメータを格納する構造体です。

各パラメータには、コンストラクタでデフォルト値が設定されます。
通常の利用において、値を変更する必要はありません。

参照
Initialize()

各種例:

メンバ詳解

size_t nn::atk::viewer::SoundEdit::Option::sndEditHeapSize

インゲーム編集ライブラリが内部で使用するヒープのサイズを指定します。

指定しないとデフォルト値 DefaultSndeditHeapSize が使用されます。
通常の利用において、値を変更する必要はありません。

size_t nn::atk::viewer::SoundEdit::Option::threadStackSize

インゲーム編集ライブラリが内部で使用するスレッドのスタックサイズを指定します。

指定しないとデフォルト値 DefaultThreadStackSize が使用されます。
通常の利用において、値を変更する必要はありません。
