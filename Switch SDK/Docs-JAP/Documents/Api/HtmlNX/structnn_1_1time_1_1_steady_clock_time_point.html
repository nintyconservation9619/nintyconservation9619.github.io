詳解

単調増加クロックが指し示す値を表す構造体です。

詳細
nn::time::GetSpanBetween 関数によって2つの値のあいだの経過時間を計算することができます。
アプリケーションが動作していないあいだの経過時間を計算したい場合、この構造体の内容をセーブデータに保存して次回の起動へ受け渡してください。

各種例:
TimeStandardSteadyClock.cpp.
