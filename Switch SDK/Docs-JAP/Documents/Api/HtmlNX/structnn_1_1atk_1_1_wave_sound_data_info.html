nn::atk::WaveSoundDataInfo 構造体

ウェーブサウンドデータのパラメータセットです。 [詳解]

#include <nn/atk/atk_WaveSoundPlayer.h>

公開変数類

int channelCount
 ウェーブサウンドデータのチャンネル数です。
 
int64_t compatibleLoopEnd
 ウェーブサウンドデータがループする時のループ終了位置を、 ウェーブサウンドの先頭からのサンプル数で表します。 ループしない時は、データの終端をサンプル数で表します。 [詳解]
 
int64_t compatibleLoopStart
 ウェーブサウンドデータがループする時のループ開始位置を、 ウェーブサウンドの先頭からのサンプル数で表します。 [詳解]
 
int64_t loopEnd
 ウェーブサウンドデータがループする時のループ終了位置を、 ウェーブサウンドの先頭からのサンプル数で表します。 ループしない時は、データの終端をサンプル数で表します。 (再生される最後のサンプルの次のサンプルを指します)
 
bool loopFlag
 ウェーブサウンドデータがループするなら true、 終端で終了するなら false となります。
 
int64_t loopStart
 ウェーブサウンドデータがループする時のループ開始位置を、 ウェーブサウンドの先頭からのサンプル数で表します。
 
int sampleRate
 ウェーブサウンドデータのサンプルレートです。
 

詳解

ウェーブサウンドデータのパラメータセットです。

この情報は WaveSoundHandle::ReadWaveSoundDataInfo か SoundArchivePlayer::ReadWaveSoundDataInfo から取得できます。

WaveSoundHandle::ReadWaveSoundDataInfo

メンバ詳解

int64_t nn::atk::WaveSoundDataInfo::compatibleLoopStart

ウェーブサウンドデータがループする時のループ開始位置を、 ウェーブサウンドの先頭からのサンプル数で表します。

0.3.0 より loopStart の値が、ハードウェアの制約による 補正がかかる前の値を返すように変更されました。 以前の値を取得したい場合にはこちらの値を使用してください。 ただし、このメンバの使用は非推奨で、将来的に削除予定です。

int64_t nn::atk::WaveSoundDataInfo::compatibleLoopEnd

ウェーブサウンドデータがループする時のループ終了位置を、 ウェーブサウンドの先頭からのサンプル数で表します。 ループしない時は、データの終端をサンプル数で表します。

0.3.0 より loopEnd の値が、ハードウェアの制約による 補正がかかる前の値を返すように変更されました。 以前の値を取得したい場合にはこちらの値を使用してください。 ただし、このメンバの使用は非推奨で、将来的に削除予定です。
