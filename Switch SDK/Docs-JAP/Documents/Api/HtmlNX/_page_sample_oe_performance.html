アプリケーション動作環境操作ライブラリを用いた性能選択操作の解説です。

概要

アプリケーション動作環境操作ライブラリを用いて性能を選択する方法を解説します。

アプリケーション動作環境操作ライブラリの使い方については、 アプリケーション動作環境操作ライブラリの関数リファレンス も併せて参照して下さい。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/OePerformance 以下にあります。

必要な環境

本サンプルプログラムは NX プラットフォームでのみビルドと実行が可能です。

操作方法

特になし。

注意事項

このデモは画面上に何も表示しません。実行結果はログに出力されます。

実行手順

サンプルプログラムをビルドし、実行してください。

サンプルプログラムの解説

サンプルプログラムはノーマルモードおよびブーストモードで使用する性能を更新します。

サンプルプログラムの処理の流れは以下の通りです。

  • デフォルトの性能の組み合わせを設定します。
    • ノーマルモード時のデフォルトの性能を設定します。
      • 対象の性能モードを nn::oe::PerformanceMode 列挙型で引数に指定してください。
      • 使用したい性能を nn::oe::PerformanceConfiguration 型で引数に指定してください。
      • nn::oe::PerformanceConfiguration 型の定義は今後追加される可能性があります。
      • nn::oe::SetPerformanceConfiguration 関数は通常成功します。
    • ブーストモード時のデフォルトの性能を設定します。
  • ノーマルモード時の GPU のクロックレートを、デフォルトの性能よりも高い値にします。
  • ブーストモード時の GPU とメモリのクロックレートを、デフォルトの性能よりも低い値にします。
  • プログラム終了時に性能設定は全て破棄されデフォルトの性能設定に戻ります。

このサンプルプログラムの実行結果を以下に示します。

Set configuration <Cpu1020Mhz Gpu307Mhz Emc1331Mhz> to <Normal Mode>.
Set configuration <Cpu1020Mhz Gpu768Mhz Emc1600Mhz> to <Boost Mode>.
Set configuration <Cpu1020Mhz Gpu384Mhz Emc1331Mhz> to <Normal Mode>.
Set configuration <Cpu1020Mhz Gpu307Mhz Emc1331Mhz> to <Boost Mode>.
