bool nn::ui2d::ArcResourceAccessor::Attach ( void *  pArchiveStart,
const char *  pResourceRootDirectory 
)

アーカイブデータを登録します。

詳細
pArchiveStartで与えるアーカイブデータは、 nn::ui2d::ArchiveResourceAlignment バイトで アライメントしてください。

これは、アーカイブデータ内に含まれているテクスチャデータを、別の場所にコピーせずに そのままGPUで扱うことができるようにするためです。アーカイブ内のファイルのデータ アラインメントは、レイアウトアーカイバが自動的に行います。

nn::ui2d::ArchiveResourceAlignment は、要求される最も大きいアライメントに合わせて 定義しています。

引数
[in]pArchiveStartアーカイブデータへのポインタです。
[in]pResourceRootDirectoryリソース検索のルートとなるディレクトリ名です。
戻り値
登録に成功したら true、失敗したら falseを返します。

void* nn::ui2d::ArcResourceAccessor::Detach ( )

登録したアーカイブデータを解除します。

詳細
レイアウトとリソースは1対1ではなく、複数のレイアウトから同じリソースを共有することができるため、 本関数を呼ぶことで、Finalize メソッドを呼んでもリソースを解放しないようにできます。

戻り値
登録されていたアーカイブデータへのポインタを返します。
