詳解

biquad フィルタのパラメータです。

詳細
enable により有効・無効の切り替えが可能です。

biquad フィルタの伝達関数は以下のように表現されます。 numerator および denominator の値は Q14 フォーマットの 固定小数点値として扱われます。 

        numerator[0] +  numerator[1] * z^{-1}  +  numerator[2] * z^{-2}
H(z) = ------------------------------------------------------------------
          (1 << 14)  - denominator[0] * z^{-1} - denominator[1] * z^{-2}
各種例:
AudioPerformanceMetrics.cpp, AudioRenderer.cpp.
