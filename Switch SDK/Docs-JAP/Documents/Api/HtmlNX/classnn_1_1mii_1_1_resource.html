Resourceを初期化します。

注意
pMemory は、Resource::Finalize() まで保持する必要があります。
初期化済みのResourceを再初期化することは出来ません。
Finalize() してから呼び出します。
引数
[in]pMemoryResourceが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pTextureResourceテクスチャリソースファイルへのポインタ
[in]textureResourceSizeテクスチャリソースのサイズ
[in]pShapeResourceシェイプリソースファイルへのポインタ
[in]shapeResourceSizeシェイプリソースのサイズ
[in]infoResourceInfo
[in]pDeviceデバイス
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& modelInfo.IsValid() != NULL
&& ! Resource::IsInitialized()
&& オフセット、サイズが正常
&& charInfoが壊れていない。
&& pTextureResource と textureResourceSize が正常
&& pShapeResource と shapeResourceSize が正常
&& pTextureResource と pShapeResource がともに
&& リソースのアライメント要求 ResourceMemoryAlignment を満たしている。
事後条件
成功したら Resource::IsInitialized() = true
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).
