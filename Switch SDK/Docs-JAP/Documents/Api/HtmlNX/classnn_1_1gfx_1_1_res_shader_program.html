const nngfxToolShaderCompilerShaderReflection* nn::gfx::ResShaderProgram::GetShaderCompilerReflection ( ) const

シェーダーコンパイラーから出力されたリフレクション情報を取得します。

戻り値
シェーダーコンパイラーから出力されたリフレクション情報へのポインターを返します。

詳細
シェーダーのコンバート時に、シェーダーコンパイラーから出力されたリフレクション情報を含めるためのオプションが指定されている必要があります。 詳細はシェーダーコンバーターのドキュメントを参照してください。

template<typename TTarget >
ShaderInitializeResult nn::gfx::ResShaderProgram::Initialize ( TDevice< TTarget > *  pDevice)

シェーダーを初期化します。

テンプレート引数
TTarget対象の低レベルグラフィックス API
引数
[in]pDeviceデバイスへのポインター
戻り値
シェーダーの初期化の結果を返します。
戻り値一覧:

ShaderInitializeResult_Success 成功しました。

ShaderInitializeResult_InvalidFormat info に設定されているシェーダーフォーマットがこの環境では無効です。

ShaderInitializeResult_SetupFailed コンパイル失敗などによりシェーダーのセットアップに失敗しました。

事前条件
  • pDevice != NULL
  • pDevice が初期化されている
  • シェーダーが初期化されていない
  • シェーダーが属しているシェーダコンテナーが初期化されている
事後条件
  • シェーダーが初期化されている

詳細
nn::gfx::Shader の Initialize に相当する処理を行います。

各種例:
GfxSimple.cpp, GfxSimpleCompute.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計17項目).
