enum nn::g3d::ResSkeleton::ScaleMode

スケルトンに関するscaleモードです。

列挙値
ScaleMode_None 

スケール計算をしません。

ScaleMode_Std 

標準的な方法でスケールの計算をします。

ScaleMode_Maya 

Maya 方式でスケールの計算をします。

ScaleMode_Softimage 

Softimage 方式でスケールの計算をします。

enum nn::g3d::ResSkeleton::RotateMode

スケルトンに関するRotateモードです。

列挙値
RotateMode_Quat 

クォータニオン方式で回転を計算します。

RotateMode_EulerXyz 

オイラー角方式で回転を計算します。

enum nn::g3d::ResSkeleton::ResetGuardFlag

リセット時にリセットしないものを指定するフラグです。

列挙値
ResetGuardFlag_None 

すべてリセットします。

ResetGuardFlag_UserPtr 

ユーザーポインターの設定をリセットしません。

enum nn::g3d::ResSkeleton::MirroringMode

内部用機能のため使用禁止です。 モーションミラーリングの計算方式を示すフラグです。

列挙値
MirroringMode_X 

ミラー方向が X 軸です。

MirroringMode_XY 

前半のミラー方向が X 軸で、後半は Y 軸です。

MirroringMode_XZ 

前半のミラー方向が X 軸で、後半は Y 軸です。
