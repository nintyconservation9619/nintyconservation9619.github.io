詳解

Buffer utility for passing media data between decoders and extractors.

詳細
Buffer utility class can be used for passing decoder buffers to extractors. Buffer can be constructed with buffer object allocated data memory, client passed data memory or to hold only buffer information.

各種例:
MovieDecoderPlayer.cpp.

Constructor to create an empty buffer object.

詳細
A new buffer object is created. No data memory is allocated

movie::Buffer::Buffer ( size_t  capacity)

Constructor to create a buffer object with memory allocation.

引数
[in]capacityThe size of data memory allocation

詳細
Creates a movie::Buffer object that allocates memory of size capacity This memory is owned by the movie::Buffer and will be released when destructed.

size_t  capacity 
)

Create a buffer object, data points to client provided memory pointer.

引数
[in]dataMemory data pointer from client
[in]capacityData size

詳細
The buffer object is created. Memory does not belong to buffer object. This memory will not be de-allocated when buffer object is deleted. Client need to deallocate the client provided memory.
