Web

Web アプレット・オフライン Web アプレットの機能を使用するサンプルプログラムの解説です。

はじめに

ここでは、Web アプレット・オフライン Web アプレットのサンプルについて解説します。

サンプル

サンプルコードは Samples/Sources/Applications/ 以下の Web で始まるディレクトリ名で配置してあります。
サンプルを以下に一覧します。

ページ 内容
WebSimple nn::web ライブラリを使って Web アプレットを起動し、Web ページを開くシンプルなサンプルプログラム
WebOfflineSimple nn::web ライブラリを使ってオフライン Web アプレットを起動し、オフライン HTML を開くシンプルなサンプルプログラム
WebDemo Web アプレット・オフライン Web アプレットの機能の活用方法を示すサンプルプログラム
WebDemoInstructionManual ゲームの取扱説明書を想定したサンプルオフライン HTML
WebDemoPauseMenu ゲームのポーズダイアログを想定したサンプルオフライン HTML
WebDemoQuickHelp ゲームをポーズしてクイックヘルプを表示することを想定したサンプルオフライン HTML
WebDemoRaceResult レースゲームのリザルトシーンを想定したサンプルオフライン HTML
WebDemoShootingMovie 3D シューティングゲームの操作テクニック説明を想定したサンプルオフライン HTML
WebDemoDirectPlayback 動画を直接再生するサンプルオフライン HTML
WebDemoKeyboardEvents Keyboard Events を使ってコントローラー操作を読み取るサンプルオフライン HTML.
WebDemoGamepadApi Gamepad API を使ってコントローラー操作を読み取るサンプルオフライン HTML.
WebDemoSpatialNavigation CSS を使って空間ナビを様々な独自のフォーカス表現に変更するサンプルオフライン HTML.
