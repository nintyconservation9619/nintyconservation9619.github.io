nn::ui2d::ResAnimationBlock 構造体

アニメーションを定義するデータブロックです。 [詳解]

#include <nn/ui2d/ui2d_Resources.h>

公開変数類

uint16_t animContCount
 アニメーション内容の数です。
 
uint32_t animContOffsetsOffset
 animContOffsets へのオフセットです(データブロック先頭から)。
 
nn::font::detail::BinaryBlockHeader blockHeader
 ヘッダです。
 
uint16_t fileCount
 参照するテクスチャリソースの数です。
 
uint16_t frameSize
 フレーム数です。
 
uint8_t loop
 ループする場合は 1 を格納します。
 
char padding [1]
 (未使用です)
 

詳解

アニメーションを定義するデータブロックです。

詳細
blockHeader の kind には DataBlockKindPaneAnimInfo が指定されます。

この構造体に以下のデータ構造が続きます。

・ uint32_t fileNameOffsets[fileCount]
・ uint8_t namePool[]
・ uint32_t animContOffsets[paneCount]
・ ResAnimationContent animCont0
・ …

fileNameOffsets 配列には参照するテクスチャリソース名への オフセット(fileNameOffsetsより)を格納します。

animContOffsets 配列にはアニメーション内容への オフセット(データブロック先頭より)を格納します。
