詳解

3Dサウンドのパラメータです。

このパラメータを用いて 3D サウンドの計算を行います。

controlFlag, decayCurve, decayRatio, dopplerFactor, soundUserParam は SoundMaker で各サウンドに設定した値が入ります。

actorUserParam は、Sound3DActor::SetUserParam で設定された値です。

dopplerFactor は、ドップラー効果のかかりり具合を表します。 0 のときは音が変化せず、値が大きくなるほど変化が大きくなります。
