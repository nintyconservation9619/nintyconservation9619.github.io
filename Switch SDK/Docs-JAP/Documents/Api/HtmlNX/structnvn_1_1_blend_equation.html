詳解

Specifies the equation used for blending source and destination colors.

Blending is performed by evaluating a function for each component of the source and destination colors (Cs and Cd), called the blend equation. Several blend equations also involve source and destination blend factors (Fs and Fd), called the blend function.

列挙値
ADD 

Compute C = Fs * Cs + Fd * Cd.

SUB 

Compute C = Fs * Cs - Fd * Cd.

REVERSE_SUB 

Compute C = Fd * Cd - Fs * Cs.

MIN 

Compute min(Cs, Cd).

MAX 

Compute max(Cs, Cd).
