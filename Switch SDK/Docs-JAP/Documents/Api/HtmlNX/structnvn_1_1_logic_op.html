詳解

Specifies a bit-wise operation performed on source and destination colors.

Logical operations are performed by evaluating a function for each component of the source and destination colors (Cs and Cd). Before performing logical operations, the source color is converted to the format used to store destination values. If enabled, the same operation is performed on all render targets. There is no support for separate logical operations on color and alpha components.
