nn::audio::FinalMixType 構造体

FinalMix を表す構造体です。 [詳解]

#include <nn/audio/audio_FinalMixTypes.h>

静的公開メンバ関数

static float GetVolumeMax ()
 設定可能なボリュームの最大値 (128.0f) を返します。
 
static float GetVolumeMin ()
 設定可能なボリュームの最小値 (0.0f) を返します。
 

詳解

FinalMix を表す構造体です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

(確保・解放)

  • AcquireFinalMix()
  • ReleaseFinalMix()

(情報取得)

  • GetFinalMixBufferCount()
  • GetFinalMixNodeId()

(音量)

  • GetFinalMixVolume()
  • SetFinalMixVolume()

(Sink)

  • AddDeviceSink()
  • RemoveDeviceSink()

(SubMix)

  • SetSubMixDestination(AudioRendererConfig *pOutConfig, SubMixType *pSource, FinalMixType *pDestination)
  • SetSubMixMixVolume(SubMixType *pSource, FinalMixType *pDestination, float volume, int sourceIndex, int destinationIndex)
  • GetSubMixMixVolume(const SubMixType *pSource, const FinalMixType *pDestination, int sourceIndex, int destinationIndex)

(Voice)

  • SetVoiceDestination(AudioRendererConfig *pOutConfig, VoiceType *pSource, FinalMixType *pDestination)
  • SetVoiceMixVolume(VoiceType *pSource, FinalMixType *pDestination, float volume, int sourceIndex, int destinationIndex)
  • GetVoiceMixVolume(const VoiceType *pSource, const FinalMixType *pDestination, int sourceIndex, int destinationIndex)

(エフェクト)

  • AddBufferMixer(AudioRendererConfig *pOutConfig, BufferMixerType *pBufferMixer, FinalMixType *pFinalMix)
  • RemoveBufferMixer(AudioRendererConfig *pOutConfig, BufferMixerType *pBufferMixer, FinalMixType *pFinalMix)
  • AddDelay(AudioRendererConfig pOutConfig, DelayType *pDelay, void buffer, size_t bufferSize, FinalMixType *pFinalMix, nn::TimeSpan delayTimeMax, int channelCountMax)
  • RemoveDelay(AudioRendererConfig *pOutConfig, DelayType *pDelay, FinalMixType *pFinalMix)
  • AddReverb(AudioRendererConfig pOutConfig, ReverbType *pReverb, void buffer, size_t bufferSize, FinalMixType *pFinalMix, int channelCount)
  • RemoveReverb(AudioRendererConfig *pOutConfig, ReverbType *pReverb, FinalMixType *pFinalMix)
  • AddAux(AudioRendererConfig pOutConfig, AuxType *pAux, void buffer, size_t bufferSize, FinalMixType *pFinalMix, void *pSendBuffer, void *pReturnBuffer, size_t bufferSize)
  • RemoveAux(AudioRendererConfig *pOutConfig, AuxType *pAux, FinalMixType *pFinalMix)

上記の関数を利用するには nn/audio.h をインクルードする必要があります。
