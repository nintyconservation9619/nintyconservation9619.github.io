enum nn::vi::PixelFormat

Pixel formats for use with the display server.

詳細
Not all formats may be supported on all platforms.

列挙値
PixelFormat_Rgba8888 

32-bit R8G8B8A8

PixelFormat_Rgbx8888 

32-bit R8G8B8X8

PixelFormat_Rgb888 

24-bit R8G8B8

PixelFormat_Rgb565 

16-bit R5G6B5

PixelFormat_Bgra8888 

32-bit B8G8R8A8

PixelFormat_Rgba5551 

16-bit R5G5B5A1

PixelFormat_Rgba4444 

16-bit R4G4B4A4

PixelFormat_StandardRgba8888 

32-bit R8G8B8A8 (sRGB)

PixelFormat_StandardRgbx8888 

32-bit R8G8B8X8 (sRGB)

PixelFormat_Unknown 

Display server is unable to support this format.

enum nn::vi::ScalingMode

Scaling modes for use with the display server.

詳細
Not all platforms will support every scaling mode.

列挙値
ScalingMode_None 

All content outside the layer is cropped.

ScalingMode_Exact 

Only content that matches the layer dimensions will be used for updates.

ScalingMode_FitToLayer 

Content is scaled to match the layer dimensions.

ScalingMode_ScaleAndCrop 

Content is scaled to the layer's smaller dimension then cropped.

ScalingMode_PreserveAspectRatio 

Content is scaled while preserving the aspect ratio.

A pillarbox or letterbox may be added as appropriate.

void nn::vi::Initialize ( )

Initializes VI library.

事後条件
VI library is initialized.

詳細
This is safe to call multiple times. If the library is already initialized, no operations occur.

There must be a matching call to Finalize for each call to this function or the VI library will not be finalized.

事前条件
（NXプラットフォームのみ）:
Call nv::SetGraphicsAllocator and nv::InitializeGraphics prior to this function.

詳細

（NXプラットフォームのみ）:
There are no explicit checks for the preconditions at this time, but it is expected to change in the future.
各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, GfxSimple.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp, TutorialBaseClass.Nx.cpp, TutorialBaseClass.Windows.cpp, VkSimple.cpp (計19項目).
void nn::vi::Finalize ( )

Finalizes VI library.

事前条件
No preconditions.
事後条件
VI library has been shutdown.

詳細
Note that nn::vi::Layer and nn::vi::Display handles will be invalidated and should not be used after calling this function.

This is safe to call multiple times. If the library is not initialized, no operations occur.

void nn::vi::CloseDisplay ( Display *  pDisplay)

Close the display.

引数
[in]pDisplayDisplay handle.
事前条件
VI library is initialized.
pDisplay != nullptr and pDisplay is a valid handle.
事後条件
pDisplay invalidated.

詳細
When a particular display is closed, all layers created by this process on that display are destroyed. The display is effectively blanked for the process.

After closing, pDisplay should not be used.

Result nn::vi::CreateLayer ( Layer **  pOutLayer,
Display *  pDisplay 
)

Creates a new layer on a particular display.

引数
[out]pOutLayerLayer handle.
[in]pDisplayDisplay handle.
戻り値
Whether the layer was created.
戻り値一覧:

ResultSuccess Layer created successfully.

ResultOperationFailed An error occurred.

事前条件
VI library is initialized.
pOutLayer != nullptr
pDisplay != nullptr and pDisplay is a valid handle.
事後条件
A valid handle is written to pOutLayer.

詳細
The layer is visible by default. Until a frame is submitted by the process, the layer will be blank.

The textures associated with this layer have restrictions due to a limitation with downscaling:
width <= 1920
height <= 1080
This restriction may be lifted in the future.

A particular display may only support a limited number of layers system-wide. If a process exceeds its quota then this function will return ResultOperationFailed. At minimum, a process can use a single layer to display content. To query how many layers can be created by the process, use ListDisplays().

If this API is used with a graphics API that does not have an explicit swap chain, like EGL, then the rendering dimensions will use default values of 1280x720.

各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, GfxSimple.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp, TutorialBaseClass.Nx.cpp, TutorialBaseClass.Windows.cpp, VkSimple.cpp (計19項目).

The textures associated with this layer have restrictions due to a limitation with downscaling:
width <= 1920
