nn::os::MessageQueueType 構造体

メッセージキューオブジェクトを定義する構造体です。 [詳解]

#include <nn/os/os_MessageQueueTypes.h>

詳解

メッセージキューオブジェクトを定義する構造体です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • InitializeMessageQueue()
  • FinalizeMessageQueue()
  • SendMessageQueue()
  • TrySendMessageQueue()
  • TimedSendMessageQueue()
  • JamMessageQueue()
  • TryJamMessageQueue()
  • TimedJamMessageQueue()
  • ReceiveMessageQueue()
  • TryReceiveMessageQueue()
  • TimedReceiveMessageQueue()
  • PeekMessageQueue()
  • TryPeekMessageQueue()
  • TimedPeekMessageQueue()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, MessageQueueType* messageQueue, MessageQueueWaitType waitType);

上記の関数を利用するには nn/os.h をインクルードする必要があります。

nn::os::MessageQueue ユーティリティクラスもあわせてご確認ください。

各種例:
OsMessageQueue.cpp, OsMultiWait.cpp, OsMultiWaitUtility.cpp.
