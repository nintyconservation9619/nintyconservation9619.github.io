enum nn::g3d::AnimObj::BindFlag
inherited

関連付けに関するフラグです。

列挙値
BindFlag_Enabled 

有効です。

BindFlag_SkipCalculate 

更新処理を無視するが、最後の結果が Apply されます。

BindFlag_SkipApply 

Apply を無視するので、単独では無駄な更新が行われます。

BindFlag_Disabled 

Calculate も Apply もスキップします。

virtual void nn::g3d::MaterialAnimObj::BindFast ( const ResModel *  pModel)
virtual

アニメーション対象に関連付けます。

引数
[in]pModelアニメーション対象の ResModel へのポインター。
事前条件
  • pModel と設定されている ResMaterialAnim のリソースが ResMaterialAnim::PreBind によって事前に関連付けられている。

詳細
リソース同士を ResMaterialAnim::PreBind によって事前に関連付けることにより、 BindFast 時には名前引きを行わない比較的高速な関連付けを行います。

void nn::g3d::MaterialAnimObj::RevertTo ( ModelObj *  pModelObj) const

自身のマテリアルアニメーションが書き換えるパラメーターをマテリアルリソースの値に戻します。

引数
[in]pModelObjパラメータを戻すモデルインスタンス。
事前条件
  • m_pRes != NULL
  • IsTargetBound()
  • pModelObj != NULL

詳細

  • モデルのマテリアルが持つパラメーターに対して複数のマテリアルアニメーションで変更を加えていた場合、 この関数を呼び出すとパラメーターがマテリアルリソースの値に上書きされてしまうので注意してください。
  • SetBindFlag() で Apply が無効化されているマテリアルは処理がスキップされます。
