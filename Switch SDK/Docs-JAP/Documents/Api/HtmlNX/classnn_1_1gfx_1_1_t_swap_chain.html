TMemoryPool< Target > *  pMemoryPool,
ptrdiff_t  memoryPoolOffset,

)

スワップチェーンを初期化します。

引数
[in]pDeviceデバイスへのポインター
[in]info初期化用の情報
[in]pMemoryPoolスキャンバッファを配置するメモリプールへのポインター
[in]memoryPoolOffsetバイトでのスキャンバッファーを配置するメモリープールへのオフセット
[in]memoryPoolSizeバイトでのスキャンバッファーが使用するメモリープールのサイズ
事前条件
  • pDevice != NULL
  • pDevice が初期化されている
  • スワップチェーンが初期化されていない
  • IsMemoryPoolRequired が true の場合、pMemoryPool != NULL
  • IsMemoryPoolRequired が true の場合、memoryPoolSize が CalculateScanBufferSize で取得されるサイズ以上
事後条件
  • スワップチェーンが初期化されている

詳細
渡すメモリープールは圧縮可能として作成されている必要があります。

各種例:

template<typename TTarget >
AcquireScanBufferResult nn::gfx::TSwapChain< TTarget >::AcquireNextScanBufferIndex ( int *  pOutScanBufferIndex,
TSemaphore< Target > *  pSemaphore,
TFence< Target > *  pFence 
)

次の Present 対象のスキャンバッファーのインデックスを取得します。

引数
[out]pOutScanBufferIndex次の Present 対象のスキャンバッファーのインデックスを受けとるためのポインター
[in]pSemaphore対象のスキャンバッファーが使用可能になる際にシグナル状態になるセマフォへのポインター
[in]pFence対象のスキャンバッファーが使用可能になる際にシグナルに状態なるフェンスへのポインター
戻り値
スキャンバッファーの取得処理の結果を返します。
戻り値一覧:

AcquireScanBufferResult_Success スキャンバッファーの取得に成功しました。

AcquireScanBufferResult_Failed スキャンバッファーの取得に失敗しました。

事前条件
  • スワップチェーンが初期化されている
  • pOutScanBufferIndex != NULL
  • pSemaphore != NULL
  • pSemaphore が初期化されている
  • pSemaphore がシグナル状態でない
  • pFence != NULL の場合、pFence が初期化されている
  • pFence != NULL の場合、pFence がシグナル状態でない

詳細
GL 版では pSemaphore と pFence に対する Sync は効果をもちません。

各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp.
