nn::ldn::LdnNetworkInfo 構造体

LDN 特有のネットワーク情報を格納する構造体です。 [詳解]

#include <nn/ldn/ldn_Types.h>

公開変数類

Bit8 advertiseData [AdvertiseDataSizeMax]
 スキャン結果に含まれる任意データです。
 
uint16_t advertiseDataSize
 スキャン結果に含まれる任意データのバイトサイズです。
 
int8_t nodeCount
 現時点でネットワークに接続しているノードの台数です。
 
int8_t nodeCountMax
 ネットワークに参加可能なノードの最大台数です。アクセスポイントを含みます。
 
NodeInfo nodes [NodeCountMax]
 ネットワークに参加しているノードの情報です。
 
Bit16 securityMode
 アクセスポイントの SecurityMode です。
 
Bit8 stationAcceptPolicy
 ステーションの接続受付ポリシー (AcceptPolicy) です。
 

詳解

LDN 特有のネットワーク情報を格納する構造体です。

詳細
nodeCount は nodes の中で isConnected == true を満たすノードの数です。 isConnected == true を満たすノードが nodes の前方に詰めて格納されることはありませんので、 接続中のノードの情報を取得したい場合には nodes から isConnected == true を満たす NodeInfo を探索して参照してください。

advertiseData と advertiseDataSize にはアクセスポイントが SetAdvertiseData 関数で設定したデータとそのサイズが格納されています。

この型の変数は大きなメモリ領域を必要としますのでスタック上に確保することは推奨されません。 スタック上に確保する場合には実行時にスタックが溢れないように注意してください。
