OE

アプリケーション動作環境操作ライブラリのサンプル解説です。

はじめに

ここでは、アプリケーション動作環境操作ライブラリのサンプルについて解説します。
アプリケーション動作環境操作ライブラリの機能は nn::oe の名前空間にあります。

サンプル

サンプルコードは Samples/Sources/Applications/ 以下に Oe で始まるディレクトリ名で配置してあります。
サンプルを以下に一覧します。

ページ 内容
起動パラメータ取得サンプル アプリケーション起動パラメータの取得方法の解説です。
モード変更通知 アプリケーション動作環境操作ライブラリを用いたモード変更通知の解説です。
性能選択 アプリケーション動作環境操作ライブラリを用いた性能選択操作の解説です。
疑似本体 ID 取得 アプリケーション動作環境操作ライブラリを用いた擬似本体 ID 取得の解説です。
デフォルトディスプレイの画面解像度取得 アプリケーション動作環境操作ライブラリを用いたデフォルトディスプレイの画面解像度取得の解説です。
無操作状態の制御 無操作状態での画面焼け軽減等の機能の開始時間を制御する方法の解説です。
独自の操作形態を持つアプリケーション向けの無操作状態の制御 無操作状態の継続時間を独自の定義によりリセットする方法の解説です。
一時的な自動スリープの無効化 アプリケーション中で一時的に自動スリープを無効化する機能の解説です。
言語設定取得 アプリケーション動作環境操作ライブラリを用いた言語設定取得のサンプルプログラムの解説です。
