nninitStartup() から利用可能な API リストです。

説明

nninitStartup() 内で使用することのできる API リストです。 以下のライブラリで提供されている関数やクラス、マクロが利用できます。

  • nn::Result クラス
  • nn::TimeSpan クラス
  • nn::util ライブラリ
    • NN_LOG() マクロ
    • NN_ASSERT() 系のマクロ
    • NN_ABORT() 系のマクロ
  • nn::init ライブラリ
    • 全ての API、クラス が使用できます。
  • nn::os ライブラリ
    • 全ての API、クラス が使用できます。
