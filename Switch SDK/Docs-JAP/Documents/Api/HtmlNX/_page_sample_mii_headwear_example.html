概要

アクセサリーや帽子などの頭部装飾物をつけたMiiを表示するサンプルプログラムです。 nn::mii::CharModel から頭部装飾物用のトランスフォームを取得する方法を示しています。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/MiiHeadwearExample 以下にあります。

必要な環境

特にありません。

操作方法

特にありません。

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは、Mii にアクセサリーや帽子などの頭部装飾物を着けて表示します。

コンバート処理

  • Samples/Sources/Applications/MiiHeadwearExample/Resources/SampleShader.glsl

    Mii と頭部装飾物描画用簡易シェーダです。

頭部装飾モデルデータ

  • Samples/Sources/Applications/MiiHeadwearExample/Resources/Acce_front.nfsb
  • Samples/Sources/Applications/MiiHeadwearExample/Resources/Acce_side.nfsb
  • Samples/Sources/Applications/MiiHeadwearExample/Resources/Acce_top.nfsb
  • Samples/Sources/Applications/MiiHeadwearExample/Resources/Cap_sample00.nfsb
  • Samples/Sources/Applications/MiiHeadwearExample/Resources/Cap_sample01.nfsb

  • Samples/Sources/Applications/MiiHeadwearExample/Resources/Headwear.nfsb

    6 種類のサンプル用の頭部装飾モデルデータ（nfsb ファイル）を用意しています。 nfsb ファイルは独自の形式のバイナリデータです。

サンプルプログラム実行時の処理

このサンプルプログラムの処理の流れは以下の通りです。

  • 描画のための各種リソースの初期化
  • Mii 用のサンプラの作成
  • nn::mii::CharInfo を取得
  • Faceline、Mask テクスチャ用シェーダ（nn::mii::TextureShader）の初期化
  • Mii リソース（nn::mii::Resource）の初期化
  • nn::mii::CharModel の初期化
  • nn::mii::Faceline の初期化
  • nn::mii::Mask の初期化
  • デスクリプタスロットの割り当て
  • nn::mii::FacelineGpuBuffer の初期化
  • nn::mii::MaskGpuBuffer の初期化
  • Faceline テクスチャのレンダリング
  • Mask テクスチャのレンダリング
  • Mii と頭部装飾用モデル描画用のシェーダの初期化
  • Mii 描画用の描画処理の初期化
  • Mii 描画のためのコンスタントバッファの設定
  • 頭部装飾用モデルファイルの読み込み
  • 頭部装飾用モデルの頂点ステートの初期化
  • 頭部装飾用モデルの頂点属性用バッファの作成
  • 頭部装飾用モデル描画のためのコンスタントバッファの設定
  • 頭部装飾用モデルの描画
  • Mii の描画
