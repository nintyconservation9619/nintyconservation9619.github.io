template<typename TProfile >
void nn::atk::AtkProfileReader< TProfile >::Initialize ( void *  buffer,
size_t  bufferSize,
int  profileMaxCount 
)

内部用機能のため使用禁止です。 プロファイルリーダーを初期化します。

初期化には GetRequiredMemorySize で得られた値以上のサイズのバッファが必要です。 また、引数の profileMaxCount には GetRequiredMemorySize と同じ値を指定してください。

プロファイル情報は Read 関数を呼び出すことで取得することができます。

引数
[in]bufferGetRequiredMemorySize で得られたサイズ以上のバッファです。
[in]bufferSizebuffer のサイズです。
[in]profileMaxCount蓄えるプロファイルの最大数を指定します。GetRequiredMemorySize と同じ値を指定してください。
事前条件
  • buffer != NULL
  • bufferSize >= GetRequiredMemorySize で得られた値
  • profileMaxCount > 0
template<typename TProfile >
int nn::atk::AtkProfileReader< TProfile >::Read ( TProfile *  pProfile,
int  maxCount 
)

内部用機能のため使用禁止です。 プロファイル情報を取得します。

引数で指定した pProfile に最大 maxCount の個数のタスクプロファイル情報を格納しますが、 プロファイルリーダーで読み取れるプロファイルの数が maxCount に満たない場合は、 読み取ることができる個数分だけ格納されます。

引数
[in]pProfile読み取ったプロファイル情報の格納先を指定します。
[in]maxCount読み取るプロファイル情報の最大数を指定します。
戻り値
格納したタスクプロファイル情報の個数を返します。
事前条件
  • pProfile != NULL
