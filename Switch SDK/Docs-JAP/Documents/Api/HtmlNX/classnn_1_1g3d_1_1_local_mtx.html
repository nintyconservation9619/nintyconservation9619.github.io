詳解

ボーンのローカル行列を表すクラスです。

flag が 0 の場合は scale と mtxRT をそのまま使用して計算されます。 さらに次のビットが立っている場合は、ライブラリーがそれぞれのビットに応じた 特殊計算や最適化計算を行うことがあります。
