詳解

POSIX 標準の時刻を表す構造体です。

詳細
協定世界時 (UTC) の1970年1月1日午前0時0分0秒からの経過秒数で時刻を保持します。
nn::TimeSpan との加減算が可能ですが、結果は秒単位に切り捨てられます。

各種例:
TimeStandardUserSystemClock.cpp.
