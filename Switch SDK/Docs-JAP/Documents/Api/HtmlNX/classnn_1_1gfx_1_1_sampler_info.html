nn::gfx::SamplerInfo::SamplerInfo ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解

void nn::gfx::SamplerInfo::SetDefault ( )

各パラメーターを既定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetFilterMode( nn::gfx::FilterMode_MinLinear_MagLinear_MipLinear );
  • SetAddressU( nn::gfx::TextureAddressMode_ClampToEdge );
  • SetAddressV( nn::gfx::TextureAddressMode_ClampToEdge );
  • SetAddressW( nn::gfx::TextureAddressMode_ClampToEdge );
  • SetComparisonFunction( nn::gfx::ComparisonFunction_Never );
  • SetBorderColorType( nn::gfx::TextureBorderColorType_White );
  • SetMaxAnisotropy( 1 );
  • SetMinLod( -1000.0f );
  • SetMaxLod( 1000.0f );
  • SetLodBias( 0.0f );
各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, GfxSimple.cpp, HidControllerSequence_Main.cpp, Town.cpp.
