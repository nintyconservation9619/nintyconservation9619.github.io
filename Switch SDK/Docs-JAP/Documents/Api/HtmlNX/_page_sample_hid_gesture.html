ジェスチャの取得

nn::hid ライブラリを用いてジェスチャを取得するサンプルプログラムの解説です。

概要

ジェスチャの状態を表示します。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/HidGesture 以下にあります。

必要な環境

Windows 環境においては、タッチまたはマウス操作が入力として扱われます。

操作方法

サンプルプログラムを実行するとウィンドウが立ち上がります。 ウインドウ上には、左半分に矩形状に数字が、右半分には操作ログが表示され、 矩形にはピンチや回転といった連続的な操作の結果が、 操作ログにはタップやスワイプと言った単発的な操作の結果が反映されます。 操作ログからはコンテキスト番号を確認することもできます。 また、タッチ位置にはカーソルが描画されます。

注意事項

コンソールウィンドウには何も表示されません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

サンプルプログラムの処理の流れは以下の通りです。

  • ジェスチャ API を Initialize する。
  • 利用可能なジェスチャを全て取得する。
  • eventNumber から新規に発生したジェスチャを選り分ける。
  • 新規に発生したジェスチャを画面表示に反映させる。
