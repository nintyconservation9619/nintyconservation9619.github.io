詳解

バッファー・テクスチャー間でのコピーオペレーションの範囲を表すクラスです。

構築子と解体子

nn::gfx::BufferTextureCopyRegion::BufferTextureCopyRegion ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解

void nn::gfx::BufferTextureCopyRegion::SetDefault ( )

各パラメーターを規定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetBufferOffset( 0 );
  • SetBufferImageWidth( 0 );
  • SetBufferImageHeight( 0 );
  • EditTextureCopyRegion().SetDefault();
