template<typename CharType>
class nn::font::DefaultWordWrapCallbackBase< CharType >

ワードラッピングの処理中に呼び出されるコールバックのデフォルト実装です。

詳細
本クラスは一般的なワードラッピング処理を行います。 改行方法を細かく調整したい場合は nn::font::WordWrapCallbackBase を継承して コールバッククラスを作成してください。
