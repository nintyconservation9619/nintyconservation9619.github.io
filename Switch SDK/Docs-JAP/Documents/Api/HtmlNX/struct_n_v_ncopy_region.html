詳解

Data structure describing a region of a texture used in copy commands.

覚え書き
Copy commands accessing a texture accept one or more pointers to copy region structures. NULL pointers are not supported.
各種例:
DebugTextRenderer.cpp, NvnTutorial10.cpp.
