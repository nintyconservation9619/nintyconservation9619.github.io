詳解

サウンド再生処理結果を表すクラスです。

ResultCode_ErrorNotEnoughInstance は、 再生しようとしたサウンドに必要な数のインスタンスが 作られていないことを表します。 サウンドアーカイブを使って再生している場合は、 サウンドアーカイブ中に設定されている インスタンス数を増やす必要があります。

列挙型メンバ詳解

enum nn::atk::SoundStartable::StartResult::ResultCode

サウンド再生処理結果を表すコードです。

列挙値
ResultCode_Success 

再生は成功しました。

ResultCode_ErrorLowPriority 

プレイヤーで同時再生可能なサウンド数を超え、かつ、 他のサウンドよりプレイヤー優先度がが低かったため、再生に失敗しました。

ResultCode_ErrorInvalidLabelString 

指定したラベル文字列に対応するサウンドが見つからないため、再生に失敗しました。

ResultCode_ErrorInvalidSoundId 

指定したサウンド ID に対応するサウンドが見つからないため、再生に失敗しました。

ResultCode_CanceledBySinglePlay 

先着優先に設定された単一再生機能によって、指定したサウンド ID に対応するサウンドの再生が中止されました。

ResultCode_ErrorCanceledForPrioritizingOldest 

このAPIの利用は非推奨です。代替APIや利用条件の有無については詳細を確認してください。

非推奨:
こちらの API は削除予定です。代わりに ResultCode_CanceledBySinglePlay を使用してください。 先着優先に設定された単一再生機能によって、指定したサウンド ID に対応するサウンドの再生が中止されました。
ResultCode_ErrorNotDataLoaded 

(現在は使われていません)

ResultCode_ErrorNotSequenceLoaded 

シーケンスデータが未ロードでプレイヤーヒープも存在しないため、再生に失敗しました。

ResultCode_ErrorNotBankLoaded 

バンクデータが未ロードでプレイヤーヒープも存在しないため、再生に失敗しました。

ResultCode_ErrorNotWsdLoaded 

ウェーブサウンドデータが未ロードでプレイヤーヒープも存在しないため、再生に失敗しました。

ResultCode_ErrorNotWarcLoaded 

波形アーカイブデータが未ロードでプレイヤーヒープも存在しないため、再生に失敗しました。

ResultCode_ErrorNotEnoughPlayerHeap 

プレイヤーヒープの準備が整っていないため、再生に失敗しました。

ResultCode_ErrorCannotOpenFile 

(現在は使われていません)

ResultCode_ErrorNotAvailable 

SoundArchivePlayer::Initialize が呼ばれていない、あるいは、SoundArchivePlayer に設定した SoundArchive に対する初期化 (FsSoundArchive::Open, MemorySoundArchive::Initialize, AddonSoundArchive::Initialize) が呼ばれていないため、 再生に失敗しました。

ResultCode_ErrorCannotAllocateTrack 

(現在は使われていません)

ResultCode_ErrorNotEnoughInstance 

優先度が低いため、サウンドインスタンスの確保に失敗し、再生に失敗しました。 サウンドインスタンスの数は、SoundMaker の [プロジェクトの設定] - [サウンドアーカイブ] タブの「◯◯サウンドの数」で設定できます。 また、優先度を高くするには、プレイヤー優先度を高くしてください。

ResultCode_ErrorInvalidParameter 

不正なパラメータが指定されたため、再生に失敗しました。 現在は、StartInfo の actorPlayerId が不正な値 (＝0～3以外の値) の場合に、 本エラーが発生します。

ResultCode_ErrorInvalidSequenceStartLocationLabel 

StartInfo の sequenceSoundInfo で指定されている startLocationLabel が不正なため、 再生に失敗しました。

ResultCode_ErrorActorNotInitialized 

SoundActor あるいは Sound3DActor が初期化前のため、 再生に失敗しました。

ResultCode_ErrorInvalidWarcId 

(現在は使われていません)

ResultCode_ErrorInvalidBankData 

ロード済みのバンクデータが不正なため、波形アーカイブデータが見つからず、再生に失敗しました。

ResultCode_ErrorUser 

サウンドライブラリ内部では使われていません。 "ResultCode_ErrorUser" ～ "ResultCode_ErrorUnknown - 1" は、 ユーザーが自由に利用することができます。

ResultCode_ErrorUnknown 

なんらかの理由で、再生に失敗しました。
