Mii アイコン描画サンプルプログラムの解説です。

概要

似顔絵ライブラリで提供しているアイコン生成用シェーダを使用して Mii アイコンの作成とその描画をするサンプルプログラムです。 nn::mii::IconShader を使用して Mii アイコンを作成する手順を示しています。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/MiiSimpleIcon 以下にあります。

必要な環境

特にありません。

操作方法

特にありません。

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは、Mii のアイコンを生成して表示を行います。

コンバート処理

  • Samples/Sources/Applications/MiiSimpleIcon/Resources/SampleShader.glsl

    アイコン描画用簡易シェーダです。

サンプルプログラム実行時の処理

このサンプルプログラムの処理の流れは以下の通りです。

  • 描画のための各種リソースの初期化
  • 似顔絵リソースファイルの読み込み
  • Mii リソース（nn::mii::Resource）の初期化
  • nn::mii::CharInfo を取得
  • nn::mii::CharModel の初期化
  • nn::mii::Faceline の初期化
  • nn::mii::Mask の初期化
  • デスクリプタスロットの割り当て
  • Faceline、Mask テクスチャ用シェーダ（nn::mii::TextureShader）の初期化
  • nn::mii::FacelineGpuBuffer の初期化
  • nn::mii::MaskGpuBuffer の初期化
  • アイコン書き込み先テクスチャの初期化
  • デプスステンシル用テクスチャの初期化
  • アイコンテクスチャ用シェーダ（nn::mii::IconShader）の初期化
  • nn::mii::IconGpuBuffer の初期化
  • Mii 用のサンプラの作成
  • アイコン書き込み先テクスチャビューの作成とデスクリプタプールへの登録
  • nn::mii::TextureShader を使用して Faceline と Mask のテクスチャのレンダリング
  • nn::mii::IconShader を使用して Mii アイコンのレンダリング
  • Mii アイコンテクスチャの描画
