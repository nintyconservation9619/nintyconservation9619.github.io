template<int TViewportCount>
class nn::gfx::util::ViewportScissorStateHolder< TViewportCount >

nn::gfx::ViewportScissorState に必要な追加メモリをメンバとして確保するラッパークラスです。

テンプレート引数
TViewportCountビューポートの数。 nn::gfx::ViewportScissorState::SetViewportStateInfoArray() に与えるビューポートステート情報の数と同じかそれ以上の値を指定してください。
