システムから取得した似顔絵リソースを使用して Mii を描画するサンプルプログラムの解説です。

概要

システムから取得した似顔絵リソースを使用して Mii の描画をするサンプルプログラムです。 システムから似顔絵リソースを取得する方法を示しています。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/MiiSystemResource 以下にあります。

必要な環境

本サンプルは NX 実機環境のみサポートしています。

操作方法

特にありません。

注意事項

Mii のシステムリソースは、NX 実機でのみサポートされます。 本サンプルは NX 実機向けのサンプルです。実機以外の環境での動作はサポートされません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは、システムから取得した Mii のリソースを使用して Mii の表示を行います。

コンバート処理

  • Samples/Sources/Applications/MiiSystemResource/Resources/SampleShader.glsl

    Mii 描画用簡易シェーダです。

サンプルプログラム実行時の処理

このサンプルプログラムの処理の流れは以下の通りです。

  • 描画のための各種リソースの初期化
  • Mii 用のサンプラの作成
  • nn::mii::CharInfo を取得
  • Faceline、Mask テクスチャ用シェーダ nn::mii::TextureShader の初期化
  • システムリソースから似顔絵リソースを取得
  • nn::mii::Resource の初期化
  • nn::mii::CharModel の初期化
  • nn::mii::Faceline の初期化
  • nn::mii::Mask の初期化
  • デスクリプタスロットの割り当て
  • nn::mii::FacelineGpuBuffer の初期化
  • nn::mii::MaskGpuBuffer の初期化
  • Faceline テクスチャのレンダリング
  • Mask テクスチャのレンダリング
  • Mii 描画用のシェーダの初期化
  • Mii 描画用の描画処理の初期化
  • Mii 描画のためのコンスタントバッファの設定
  • Mii の描画
