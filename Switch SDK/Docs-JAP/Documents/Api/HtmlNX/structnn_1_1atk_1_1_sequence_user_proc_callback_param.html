'userproc' コマンドで呼び出されるコールバックで使用される構造体です。 [詳解]

#include <nn/atk/atk_SequenceSoundPlayer.h>

公開変数類

bool cmpFlag
 比較コマンドで設定される真偽のフラグです。 フラグの状態を参照、あるいは変更することができます。
 
volatile int16_t * globalVariable
 グローバル変数の配列です。 globalVariable[0] から globalVariable[15] まで指定できます。 変数の値を参照、あるいは変更することができます。
 
volatile int16_t * localVariable
 ローカル変数の配列です。 localVariable[0] から localVariable[15] まで指定できます。 変数の値を参照、あるいは変更することができます。
 
volatile int16_t * trackVariable
 トラック変数の配列です。 trackVariable[0] から trackVariable[15] まで指定できます。 変数の値を参照、あるいは変更することができます。
