電力計測ライブラリを用いた消費電力計測サンプルプログラムの解説です。

概要

ここでは、電力計測ライブラリを用いて消費電力を計測する方法を解説します。

電力計測ライブラリの使い方については、 電力計測ライブラリの関数リファレンス も併せて参照して下さい。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/PcmBasic 以下にあります。

必要な環境

本サンプルプログラムは NX プラットフォームでのみビルドと実行が可能です。

操作方法

とくになし

注意事項

このデモは画面上に何も表示されません。実行結果はログに出力されます。

実行手順

サンプルプログラムをビルドし、実行してください。

サンプルプログラムの解説

サンプルプログラムは、指定可能な計測ポイントすべてで消費電力を計測し、その結果を表示します。

サンプルプログラムの処理の流れは以下の通りです。

  • 電力計測ライブラリの初期化
  • 計測ポイントごとの消費電力の測定と表示
    • 計測したい計測ポイントに対応する nn::pcm::MeasuringPoint_ から始まる列挙値を引数に指定してください。
  • 実行結果の確認
    • 指定した計測ポイントで計測された消費電力（ mW 単位）が戻り値に格納されます。

このサンプルプログラムを SDEV で実行したときの結果を以下に示します。 計測ポイントを搭載していない EDEV で実行した場合は、すべての計測ポイントで "Not implemented on this board." となります。

PCM demo start.

[Measurement Result]
  54 mW (@ GPU)
 750 mW (@ CPU)
 928 mW (@ DDR)

PCM demo end.
