nn::os::TlsSlot 構造体

TLS のスロット番号を表す型です。 [詳解]

#include <nn/os/os_ThreadLocalStorageCommon.h>

詳解

TLS のスロット番号を表す型です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • AllocateTlsSlot()
  • FreeTlsSlot()
  • GetTlsValue()
  • SetTlsValue()
  • GetUsedTlsSlotCount()

上記の関数を利用するには nn/os.h をインクルードする必要があります。

ThreadLocalStorage ユーティリティクラスもあわせてご確認ください。

各種例:
OsThreadLocalStorage.cpp, PerfSimple.cpp.
