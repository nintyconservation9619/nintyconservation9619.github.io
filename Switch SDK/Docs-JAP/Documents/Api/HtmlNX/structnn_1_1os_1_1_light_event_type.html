nn::os::LightEventType 構造体

軽量イベントオブジェクトを定義する構造体です。 [詳解]

#include <nn/os/os_LightEventTypes.h>

詳解

軽量イベントオブジェクトを定義する構造体です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • InitializeLightEvent()
  • FinalizeLightEvent()
  • SignalLightEvent()
  • WaitLightEvent()
  • TryWaitLightEvent()
  • TimedWaitLightEvent()
  • ClearLightEvent()

上記の関数を利用するには nn/os.h をインクルードする必要があります。

nn::os::LightEvent ユーティリティクラスもあわせてご確認ください。
