void nn::util::RelocationTable::Relocate ( )

バイナリファイルのメモリ上の配置を固定し、データにアクセス可能な状態にします。

事前条件
対応するバイナリファイルヘッダの IsRelocated() が false であること。

詳細
バイナリファイル中の BinTPtr が持つオフセット値をポインタに変換し、変換済みフラグをバイナリファイルヘッダに設定します。

void nn::util::RelocationTable::Unrelocate ( )

バイナリファイルをメモリ上で移動可能な状態に戻し、データにアクセス不可能な状態にします。

事前条件
対応するバイナリファイルヘッダの IsRelocated() が true であること。

詳細
バイナリファイル中の BinTPtr が持つポインタをオフセット値に逆変換し、未変換フラグをバイナリファイルヘッダに設定します。
