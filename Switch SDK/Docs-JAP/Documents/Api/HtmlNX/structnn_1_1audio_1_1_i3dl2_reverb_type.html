nn::audio::I3dl2ReverbType 構造体

I3dl2Reverb エフェクトです。 [詳解]

#include <nn/audio/audio_EffectTypes.h>

公開型

enum  Preset
 I3dl2ReverbType に指定できるプリセットパラメータを表す列挙型です。
 

詳解

I3dl2Reverb エフェクトです。

詳細
Interactive 3D Audio Level 2 (I3DL2) 仕様に部分的に準拠した環境リバーブです。

メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

(ワークバッファサイズ取得)

  • GetRequiredBufferSizeForI3dl2Reverb(int sampleRate, int channelCountMax)

(確保・解放)

  • AddI3dl2Reverb(AudioRendererConfig* pOutConfig, I3dl2ReverbType* pOutReverb, void* buffer, size_t bufferSize, FinalMixType* pFinalMix, int channelCountMax)
  • AddI3dl2Reverb(AudioRendererConfig* pOutConfig, I3dl2ReverbType* pOutReverb, void* buffer, size_t bufferSize, SubMixType* pSubMix, int channelCountMax)
  • RemoveI3dl2Reverb(AudioRendererConfig* pOutConfig, I3dl2ReverbType* pReverb, FinalMixType* pFinalMix)
  • RemoveI3dl2Reverb(AudioRendererConfig* pOutConfig, I3dl2ReverbType* pReverb, SubMixType* pSubMix)
  • IsI3dl2ReverbRemovable(const I3dl2ReverbType* pReverb)

(接続先設定)

  • GetI3dl2ReverbInputOutput(const I3dl2ReverbType* pReverb, int8_t* outInput, int8_t* outOutput, int* pOutCount, int count)
  • SetI3dl2ReverbInputOutput(I3dl2ReverbType* pReverb, const int8_t* input, const int8_t* output, int count)

(有効・無効設定)

  • SetI3dl2ReverbEnabled(I3dl2ReverbType* pReverb, bool enable)
  • IsI3dl2ReverbEnabled(const I3dl2ReverbType* pReverb)

(パラメータ)

  • SetI3dl2ReverbRoomGain(I3dl2ReverbType* pReverb, float room)
  • GetI3dl2ReverbRoomGain(const I3dl2ReverbType* pReverb)
  • SetI3dl2ReverbRoomHfGain(I3dl2ReverbType* pReverb, float roomHf)
  • GetI3dl2ReverbRoomHfGain(const I3dl2ReverbType* pReverb)
  • SetI3dl2ReverbLateReverbDecayTime(I3dl2ReverbType* pReverb, nn::TimeSpan time)
  • GetI3dl2ReverbLateReverbDecayTime(const I3dl2ReverbType* pReverb)
  • SetI3dl2ReverbLateReverbHfDecayRatio(I3dl2ReverbType* pReverb, float hfDecayRatio)
  • GetI3dl2ReverbLateReverbHfDecayRatio(const I3dl2ReverbType* pReverb)
  • SetI3dl2ReverbReflectionsGain(I3dl2ReverbType* pReverb, float reflections)
  • GetI3dl2ReverbReflectionsGain(const I3dl2ReverbType* pReverb)

  • GetI3dl2ReverbReflectionDelayTime(const I3dl2ReverbType* pReverb)
  • SetI3dl2ReverbReverbGain(I3dl2ReverbType* pReverb, float reverb)
  • GetI3dl2ReverbReverbGain(const I3dl2ReverbType* pReverb)
  • SetI3dl2ReverbLateReverbDelayTime(I3dl2ReverbType* pReverb, nn::TimeSpan time)
  • GetI3dl2ReverbLateReverbDelayTime(const I3dl2ReverbType* pReverb)

  • GetI3dl2ReverbHfReference(const I3dl2ReverbType* pReverb)
  • SetI3dl2ReverbDryGain(I3dl2ReverbType* pReverb, float dryGain)
  • GetI3dl2ReverbDryGain(const I3dl2ReverbType* pReverb)
