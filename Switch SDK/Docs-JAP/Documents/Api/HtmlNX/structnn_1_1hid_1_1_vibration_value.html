詳解

振動値を表す構造体です。

詳細
振動値はある瞬間の振動の振幅と周波数の組です。 低帯域と高帯域それぞれの振幅と周波数の値をセットで表します。

各種例:
HidVibrationRollingBall.cpp.

詳細
デフォルトの振動値は振幅が 0.0f, 周波数はそれぞれデフォルト値になります。
