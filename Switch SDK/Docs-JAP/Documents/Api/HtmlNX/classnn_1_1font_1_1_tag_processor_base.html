列挙型メンバ詳解

template<typename CharType >
enum nn::font::TagProcessorBase::Operation

呼び出し元 (TextWriter) が行うべき処理の列挙子です。

列挙値
Operation_Default 

行頭では次の文字との間に文字間を開けません。 行頭以外の位置では次の文字との間に文字間を開けます。

Operation_NoCharSpace 

行頭以外の位置でタグ処理が行われた場合、 デフォルトでは次の文字との間に文字間を空けますが、 これを空けないようにします。

Operation_CharSpace 

行頭でタグ処理が行われた場合、 デフォルトでは次の文字との間に文字間を空けませんが、 これを空けるようにします。

Operation_NextLine 

改行時に必要な処理のうち X 座標の調整のみを行います。 Y 座標の調整は行いません。 また、次の文字との間に文字間を開けません。

Operation_EndDraw 

文字列の途中で文字列描画を終了する場合に使用します。

Operation_MaxOperation 

内部用機能のため使用禁止です。
