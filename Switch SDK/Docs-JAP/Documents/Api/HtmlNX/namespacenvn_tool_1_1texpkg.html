詳解

テクスチャパッケージの名前空間です。テクスチャイメージをハードウェア依存のテクスチャパッケージ形式に変換する機能を提供します。

型定義詳解

typedef uint32_t(* nvnTool::texpkg::GetImageSliceDataSizeFunction) (RawImageLib *imglib, uint32_t width, uint32_t height, uint32_t depth, NVNformat format, uint32_t *pitch)

Custom alignment function callback.

Custom alignment function callback. Format loaders will need to implement this for format-specific alignments. Alternatively the format loader can re-align at load, and just use the default (which assumes packed no align).

列挙型詳解

enum nvnTool::texpkg::RawImageHardwareTextureFlagsEnum

Hardware texture conversion flags.

列挙値
NVN_TEXPKG_STORAGE_FLAGS_SPARSE 

Whether to align the image for use with NX sparse.

This is similar to nvn::TextureFlags::SPARSE. Cannot be used with NVN_TEXPKG_FLAGS_MINIMAL_LAYOUT.

NVN_TEXPKG_STORAGE_FLAGS_MINIMAL_LAYOUT 

Whether choose minimal texture storage layout in order to save memory.

Note that this may impact performance. This is similar to nvn::TextureFlags::MINIMAL_LAYOUT. Cannot be used with NVN_TEXPKG_FLAGS_SPARSE.
