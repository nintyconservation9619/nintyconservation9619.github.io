nn::os::TimerEventType 構造体

タイマーイベントオブジェクトを定義する構造体です。 [詳解]

#include <nn/os/os_TimerEventTypes.h>

詳解

タイマーイベントオブジェクトを定義する構造体です。

詳細
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

  • InitializeTimerEvent()
  • FinalizeTimerEvent()
  • StartOneShotTimerEvent()
  • StartPeriodicTimerEvent()
  • StopTimerEvent()
  • WaitTimerEvent()
  • TryWaitTimerEvent()
  • SignalTimerEvent()
  • ClearTimerEvent()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, TimerEventType* timerEvent)

上記の関数を利用するには nn/os.h をインクルードする必要があります。

nn::os::TimerEvent ユーティリティクラスもあわせてご確認ください。

各種例:
HidVibrationPlayer.cpp, OsTimerEvent.cpp.
