I3dl2Reverb エフェクトパラメータです。

詳細
I3dl2Reverb が施す I3dl2Reverb エフェクト処理のパラメータセットです。 各パラメータが持つ意味はオーディオエフェクトの説明の「I3dl2 リバーブパラメータ」を参照してください。

AudioEffect.cpp, AudioEffectAuxUtility.cpp.
