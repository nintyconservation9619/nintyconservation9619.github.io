NVN

NX 専用グラフィックス API である NVN ライブラリのサンプル解説です。

はじめに

ここでは、NVN ライブラリ (nvn) のサンプルについて解説します。
NVN の機能は nvn 名前空間にあります。

サンプル

サンプルコードは Samples/Sources/Applications/ 以下に Nvn で始まるディレクトリ名で配置してあります。
サンプルを以下に一覧します。

サンプル 内容
NVN Tutorial 01: Getting Started With NVN This tutorial demonstrates how to setup an application to use the NVN graphics API and performs some basic operations with it.
NVN Tutorial 02: GPU Offline Build Tools This tutorial demonstrates how to use the GLSLC offline shader compiler and the NVN texture packager to prepare assets to be consumed by an application using NVN.
NVN Tutorial 03: Asset File Loading This tutorial demonstrates rendering a simple model loaded from an asset file generated by Tutorial02.
NVN Tutorial 04: Multithreaded Asset File Loading, Cube Maps, and Mipmaps This tutorial builds off of Tutorial03 and introduces asset file loading in parallel on multiple threads and rendering multiple models with different asset data and shaders.
NVN Tutorial 05: Setting Up Render State And Rendering Debug Text This tutorial builds off of Tutorial03 and shows various combinations of render states that can be used while rendering geometry as well as a way to render debug text to the screen.
NVN Tutorial 06: Uniform Block Memory Management This tutorial demonstrates a simple way to manage uniform buffer and command buffer memory while trying to limit the amount of times that the CPU has to wait for the GPU in order to safely write uniform and command data.
NVN Tutorial 07: Multithreaded Renderer This tutorial demonstrates how to use NVN to render objects on multiple threads.
NVN Tutorial 08: GPU Statistics This tutorial builds off of Tutorial07 by adding functionality to display GPU performance counters on the screen.
NVN Tutorial 09: Shader Specialization This tutorial builds off of Tutorial08 by adding online shader compiling and shader specializations.
NVN Tutorial 10: NVN and OpenGL Interoperation This tutorial demonstrates OpenGL/NVN interoperability through sharing texture resources.
NVN Tutorial 11: Runtime Texture Compression This tutorial shows how to load a Jpeg image from a file and compress it at runtime through a shader.
NVN Tutorial Library The NVN Tutorial Library contains common code shared by NVN Tutorial 01 and 03-09.
