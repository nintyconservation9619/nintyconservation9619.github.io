nv::MemoryInfo 構造体

A structure for retrieving memory allocation status. [詳解]

#include <nv/nv_MemoryManagement.h>

公開変数類

size_t firmwareMemoryAllocatableSize
 Largest block that can be currently allocated from firmware memory.
 
size_t firmwareMemoryFreeSize
 Number of free bytes currently in firmware memory.

 Largest block that could be allocated from firmware memory when firmware memory usage peaked.
 
size_t firmwareMemoryMinimumFreeSize
 Number of free bytes in firmware memory when firmware memory usage peaked.
 
size_t reserved [4]
