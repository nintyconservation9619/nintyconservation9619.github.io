nn::atk::SoundStartable::StartInfo::WaveSoundInfo 構造体

ウェーブサウンドに関するパラメータ構造体です。 [詳解]

#include <nn/atk/atk_SoundStartable.h>

 WaveSoundInfo 構造体のパラメータを有効にするためのビットフラグの定義です。 [詳解]
 

公開メンバ関数

 WaveSoundInfo () NN_NOEXCEPT
 コンストラクタです。
 

公開変数類

int enableParameterFlag
 構造体に設定された値を有効にするためのビットフラグです。 デフォルト値では全て無効になっています。
 
bool isContextCalculationSkipMode
 内部用機能のため使用禁止です。 ADPCM 再生でオフセットを指定した場合に、再生に用いるコンテキスト計算を再生前に計算せずに再生を行います。 この結果、オフセット指定時の再生による CPU 処理は削減されますが、 バイナリに存在する波形データとは出力結果が一致しない状態で再生されます。
 
int release
 音源に与えられるリリースパラメータです。 0 ～ 127 の間で指定します。
 
const void * waveAddress
 使用する波形データのアドレスです。
 
int8_t waveType
 nn::atk::WaveType を指定します。
 

詳解

ウェーブサウンドに関するパラメータ構造体です。

ウェーブサウンド再生時に渡されます。

waveAddress に波形データのアドレスを指定すると、 サウンドアーカイブ中で設定されている波形データの代わりに、 メモリ上に置かれた波形データを使用して再生することができます。 NULL を指定すると、 元々サウンドアーカイブで設定されている波形データが使用されます。

列挙型メンバ詳解

enum nn::atk::SoundStartable::StartInfo::WaveSoundInfo::EnableParameterFlagBit

WaveSoundInfo 構造体のパラメータを有効にするためのビットフラグの定義です。

それぞれの値の論理和を enableParameterFlag に設定します。

release に設定された値を有効にします。

EnableParameterFlagBit_ContextCalculationSkipMode 

内部用機能のため使用禁止です。 ContextCalculationSkipMode に設定した値を有効にします
