構築子と解体子

nn::gfx::TextureInfo::TextureInfo ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解

void nn::gfx::TextureInfo::SetDefault ( )

各パラメーターを既定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetImageStorageDimension( nn::gfx::ImageStorageDimension_2d );
  • SetImageFormat( nn::gfx::ImageFormat_Undefined );
  • SetGpuAccessFlags( 0 );
  • SetTileMode( nn::gfx::TileMode_Optimal );
  • SetWidth( 1 );
  • SetHeight( 1 );
  • SetDepth( 1 );
  • SetMipCount( 1 );
  • SetArrayLength( 0 );
  • SetSwizzle( 0 );
  • SetMultiSampleCount( 1 );
各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp, Town.cpp, VfxSimple/main.cpp (計17項目).
