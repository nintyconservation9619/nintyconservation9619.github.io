マクロ定義詳解

#define NN_ABORT (   ...)    NN_DETAIL_ABORT_UNLESS_IMPL(false, "NN_ABORT", ##__VA_ARGS__)

実行を停止します。このマクロは常に有効です。

詳細
実行を停止する前にアボートオブザーバが呼び出されます。

NN_ENABLE_ABORT_MESSAGE が定義されている場合、指定されたメッセージがアボートオブザーバに渡されます。 NN_DISABLE_ABORT_MESSAGE が定義されている場合、空のメッセージが渡されます。 どちらのマクロも定義されていない場合は、NintendoSDK のビルドタイプによって渡されるメッセージが決定されます:

  • Debug または Develop の場合 : 指定されたメッセージ
  • Release の場合 : 空のメッセージ

このマクロは、常に有効です。

[in]...ログとして出力するメッセージの書式指定文字列と引数

)    NN_DETAIL_ABORT_UNLESS_IMPL((condition), NN_MACRO_STRINGIZE(condition), ##__VA_ARGS__)

条件を満たしていることをテストし、満たされていない場合に停止します。このマクロは常に有効です。

詳細
条件式 condition が true に評価される場合、何も行いません。

条件式 condition が false に評価される場合、実行を停止します。

実行を停止する前にアボートオブザーバが呼び出されます。

NN_ENABLE_ABORT_MESSAGE が定義されている場合、指定されたメッセージがアボートオブザーバに渡されます。 NN_DISABLE_ABORT_MESSAGE が定義されている場合、空のメッセージが渡されます。 どちらのマクロも定義されていない場合は、NintendoSDK のビルドタイプによって渡されるメッセージが決定されます:

  • Debug または Develop の場合 : 指定されたメッセージ
  • Release の場合 : 空のメッセージ

このマクロは、常に有効です。
