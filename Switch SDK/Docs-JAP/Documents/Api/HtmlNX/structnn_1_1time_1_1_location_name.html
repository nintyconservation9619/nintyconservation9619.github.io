詳解

地域名を扱う構造体です。

詳細
LoadTimeZoneRule() 関数に指定します。

アプリケーション起動中に LocationName を固定しておきたい場合には、必要に応じてキャッシュしても構いません。
また、その値を次回起動に引き継ぎたい場合には、その時点での LocationName をセーブデータ等に保存しても構いません。
ただし、キャッシュした LocationName から計算される値や判定（現在時刻や日付またぎなどの判定など) は、 システム側のものとずれが出る可能性がある点に注意してください。

タイムゾーンを扱う API について も併せて参照してください。
