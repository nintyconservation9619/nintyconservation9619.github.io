bool nn::mii::Database::IsUpdated ( int  sourceFlags)

状態更新が発生したかを判定します。

引数
[in]sourceFlagsSourceFlag
戻り値
更新があればtrueを返します。

詳細
Miiに更新・追加・削除が発生したかを判定します。
システムが保持するMiiが更新された場合にtrueを返します。 更新は、アプリ・アプレットの遷移が発生したときに起こります。
アプリ・アプレットから戻ってきたときにこの関数は、 trueを返す場合があります。
このメソッドはワンショットです。
呼び出すと状態がクリアされることに注意してください。

bool nn::mii::Database::IsFullDatabase ( ) const

データベースの容量が満杯かを判定します。

戻り値
満杯ならtrueを返します。

詳細
GetCount(SourceFlag_Database) == DatabaseMiiCountの判定は行わず、こちらのメソッドを使ってください。 GetCount(SourceFlag_Database) != DatabaseMiiCountの場合でも満杯の場合があります。

nn::Result nn::mii::Database::Convert ( CharInfo *  pInfo,
const Ver3StoreData &  ver3StoreData 
)

3DS/WiiU の StoreData を CharInfo に変換します。

引数
[out]pInfo変換した CharInfo の格納先を指定します。
[in]ver3StoreData変換する Ver3StoreData を指定します。
戻り値
処理の結果を表す nn::Result 値を返します。
  • nn::ResultSuccess 変換に成功
  • nn::mii::ResultBroken のハンドリングが必要です。
詳細
3DS/WiiU の StoreData を CharInfo に変換します。
NG ワード を含むニックネームは、ニックネームの文字数分の "?" で置き換えられます。
不正な文字コードを含むニックネームは "???" に置き換えられます。
Mii のニックネームとして使用可能な文字コードは NintendoSDK/Resources/Mii/fontset 以下の文字フィルタファイルを参照してください。
