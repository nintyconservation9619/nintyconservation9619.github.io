pMemory および pMemoryPool は、CharModel::Finalize() まで保持する必要があります。

レンダーターゲットとなるテクスチャも含まれるため、Compressible 設定が必要です。
resource は、Resource の設定条件によって
CharModel::Finalize() まで保持の必要があります。
初期化済みのCharModelを再初期化することは出来ません。
Finalize() してから呼び出します。
pTemporaryBufferは初期化中のみ必要です。
初期化が終われば破棄してかまいません。

描画パラメータを取得

詳細
指定の条件で描画パラメータを取り出します。

引数
[in]type取得する描画パラメータの種類
[in]model取得するモデルの種類
[in]nose取得する鼻の種類
[in]mask取得するマスクスロット番号
戻り値
DrawParam を返します。
描画の必要が無い場合はNULLを返します。
事前条件
IsInitialized() == true
&& CharModelInfo::GetCreateFlag() に該当種類(model,nose)が含まれている
&& 0 <= index < CharModelInfo::GetMaskNum()
注意
取り出した DrawParam は、取得時の条件で固定されています。
テクスチャデスクリプタスロットは、
SetTextureDescriptorSlot() で変更されることに注意してください。
DrawType_Mask 以外は、maskを無視します。
DrawType とモデル生成の種類( CreateModelType 、CreateNoseType )には依存関係があります。
CreateModelType に依存する DrawType は以下の通りです。
  • DrawType_Forehead
  • DrawType_Hair
  • DrawType_Hat
これらの DrawType は CreateModelType の値によって返すパラメータが異なります。
また、初期化時に設定した生成フラグに含まれない CreateModelType を指定した場合は、nullptr を返します。
CreateNoseType に依存する DrawType は以下の通りです。
  • DrawType_Nose
  • DrawType_Noseline
これらの DrawType は CreateNoseType の値によって返すパラメータが異なります。
また、初期化時に設定した生成フラグに含まれない CreateNoseType を指定した場合は、nullptr を返します。
その他の DrawType は CreateModelType や CreateNoseType に依存しません。
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSystemResource.cpp.
