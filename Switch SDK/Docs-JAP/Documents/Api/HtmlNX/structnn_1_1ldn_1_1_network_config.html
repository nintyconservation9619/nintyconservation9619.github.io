アクセスポイントが構築するローカル・ネットワークに関する設定情報です。 [詳解]

詳解

アクセスポイントが構築するローカル・ネットワークに関する設定情報です。

詳細
アクセスポイントが構築するローカル・ネットワークに関する設定情報です。

intentId はローカル・ネットワークの用途を表明する識別子です。 詳細については IntentId のリファレンスを参照してください。

nodeCountMax ではネットワークに接続できるノードの最大数を設定します。 ノードの接続上限は 1 以上 NodeCountMax 以下に設定可能で、 この数字にはアクセスポイント自身を含んでいます。

channel には無線チャンネルを指定します。 LDN ライブラリで使用できる無線チャンネルは製品機と同じ 2.4GHz 帯の 1, 6, 11 と、 開発機でのみ使用できる 5GHz 帯の 36, 40, 44, 48 のいずれかです。 ここで指定したチャンネルによって自動的に周波数帯域が決まりますので、 他に 2.4GHz 帯と 5GHz 帯を切り替えるために必要な作業はありません。 特に無線チャンネルを指定する必要が無い場合は AutoChannel を与えることで、 2.4GHz 帯の 1, 6, 11 の 3 つのチャンネルの中から、 混雑していない最適なチャンネルをシステムが検索して使用します。 AutoChannel 指定時に 5GHz 帯のチャンネルが選択されることはありません。 また、無線チャンネルの指定は開発機でのみ有効で、 製品機ではここでの指定は無視されて常にシステムが無線チャンネルを決定します。

localCommunicationVersion はローカル通信バージョンです。 アプリケーションへのパッチの適用などで通信の仕様が変わる場合に、 ローカル通信バージョンをインクリメントしておくことで、通信の互換性がないことを表明できます。 ローカル通信バージョンには LocalCommunicationVersionMin 以上、 LocalCommunicationVersionMax 以下の値を設定してください。
