詳解

A collection of typed name/value pairs.

This data structure is used to pass media data between different modules.

詳細
Objects are unsorted and stored in order of insertion. Only one item of a given name may exist at any time. Setting will overwrite any existing item even if it's a different type. Collection holds at most 64 different name/value pairs.

各種例:
MovieDecoderPlayer.cpp.
