線形補間による透明ピクセルの色にじみ防止機能サンプルプログラムの解説です。

概要

似顔絵ライブラリで提供している色にじみ防止機能を使い、 画像を拡大したときに生じるにじみを比較するサンプルプログラムです。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/MiiTransparentAdjuster 以下にあります。

必要な環境

特にありません。

操作方法

特にありません。

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは、Mii のアイコンを生成し、 nn::mii::TransparentAdjusterの有無により画像の違いを比較できます。

コンバート処理

  • Samples/Sources/Applications/MiiTransparentAdjuster/Resources/SampleShader.glsl

    テクスチャ描画用簡易シェーダです。

サンプルプログラム実行時の処理

このサンプルプログラムの処理の流れは以下の通りです。

  • 描画のための各種リソースの初期化
  • 似顔絵リソースファイルの読み込み
  • Mii リソース（nn::mii::Resource）の初期化
  • nn::mii::CharInfo を取得
  • nn::mii::CharModel の初期化
  • nn::mii::Faceline の初期化
  • nn::mii::Mask の初期化
  • デスクリプタスロットの割り当て
  • Faceline、Mask テクスチャ用シェーダ（nn::mii::TextureShader）の初期化
  • nn::mii::FacelineGpuBuffer の初期化
  • nn::mii::MaskGpuBuffer の初期化
  • アイコン書き込み先テクスチャの初期化
  • 結果を格納するテクスチャの初期化
  • デプスステンシル用テクスチャの初期化
  • アイコンテクスチャ用シェーダ（nn::mii::IconShader）の初期化
  • nn::mii::IconGpuBuffer の初期化
  • Mii 用のサンプラの作成
  • テクスチャビューの作成とデスクリプタプールへの登録
  • nn::mii::TextureShader を使用して Faceline と Mask のテクスチャのレンダリング
  • nn::mii::IconShader を使用して Mii アイコンのレンダリング
  • nn::mii::TransparentAdjuster の初期化とパラメータ設定
  • nn::mii::TransparentAdjusterGpuBuffer の初期化とパラメータ設定
  • nn::mii::TransparentAdjuster を使って Mii アイコン画像から結果テクスチャを出力
  • Mii アイコンテクスチャ(左)と結果テクスチャ(右)を画面表示
