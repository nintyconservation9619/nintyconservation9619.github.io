引数に指定された文字列を、ログとして出力します。

引数
[in]...printf 形式で指定する、ログとして出力する文字列の書式指定文字列と、それに与える引数

詳細
このマクロは、NN_ENABLE_LOG が定義されている場合に有効です。 NN_DISABLE_LOG が定義されている場合、空の処理に展開されます。

このマクロは既定で有効です。

NN_LOG() に設定されるログの重大性は Info、冗長性は 0 です。

NN_LOG() に与える文字列のエンコーディングは UTF-8 です。

Windows 環境で、やむをえずロケール既定のエンコーディングを使用する場合は、 NintendoSDK のヘッダファイルをインクルードする前に、 NN_LOG_USE_DEFAULT_LOCALE_CHARSET マクロを定義することで、 ロケール既定のエンコーディングを使用することができます。

詳しくは、NintendoSDK ドキュメントの「ログ・アサートの文字コードに関する注意点」をご参照ください。
