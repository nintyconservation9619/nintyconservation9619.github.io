ARMv7A 動作環境での例外情報の詳細を格納する構造体です。

詳細
ARMv7A 動作環境において UserExceptionInfoDetail 型へと typedef されます。
各メンバ変数の間には適度にパディングが入る可能性があるため、 アクセスする場合には、個々のメンバ変数名を使ってアクセスして下さい。
