詳解

ルートシグネチャーを初期化するための情報を表すクラスです。

このAPIの利用は非推奨です。代替APIや利用条件の有無については詳細を確認してください。

非推奨:
廃止予定です。nn::gfx::CommandBuffer::SetTextureAndSampler 等のバインド API を使用してください。

構築子と解体子

nn::gfx::RootSignatureInfo::RootSignatureInfo ( )

コンストラクターです。

事後条件
  • 各パラメーターがゼロ初期化されている

関数詳解
