#define NN_PERF_INITIALIZE_METER (   pDevice,

)    nn::perf::LoadMeterCenter::Initialize( pDevice, loadMeterCenterInfo, pMemory, memorySize, pMemoryPool, memoryPoolOffset, memoryPoolSize )

LoadMeterCenter を初期化します。

引数
[in]pDeviceデバイスへのポインタです。
[in]loadMeterCenterInfo初期化用の情報です。
[in]pMemory初期用のバッファへのポインタ
[in]memorySizeバイトでのバッファサイズ
[in]pMemoryPoolGpuタイムスタンプが書き込まれるメモリプールへのポインタ
[in]memoryPoolOffsetGpuタイムスタンプが書き込まれるメモリプールへのオフセット
[in]memoryPoolSizeGpuタイムスタンプが書き込まれるメモリプールのサイズ
事前条件
  • pDevice != NULL
  • pDevice が初期化されている。
  • info に GpuMeter の計測区間数が 1 以上の値で設定されている。

詳細
以下のメーターを作成します。

  • フレーム計測用 CpuMeter : 1つ
  • コア計測用 CpuMeter : LoadMeterCenterInfo で指定された数
  • ユーザー計測用 CpuMeter : LoadMeterCenterInfo で指定された数
  • GPU 計測用 GpuMeter : 1つ
    フレーム計測用 CpuMeter にすべての CpuMeter と GpuMeter がアタッチされます。
各種例:
FrameworkDemo/Main.cpp, GfxPrimitiveRenderer.cpp, PerfSimple.cpp.
