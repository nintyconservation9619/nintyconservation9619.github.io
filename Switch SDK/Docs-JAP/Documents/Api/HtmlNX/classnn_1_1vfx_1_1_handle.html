詳解

エフェクト操作ハンドルクラスです。

生成済みのエミッタセットへは、このクラスを取得できます。

各種例:
VfxBinarySegmentation/main.cpp, VfxCustomShader/main.cpp, VfxManualEmitterSet/main.cpp, VfxSimple/main.cpp.

関数詳解

void nn::vfx::Handle::Invalidate ( )

ハンドルを無効化します。

放出済みのエミッタセットはそのままに、ハンドルだけ無効化します。 ハンドルを使いまわす時に使用します。
