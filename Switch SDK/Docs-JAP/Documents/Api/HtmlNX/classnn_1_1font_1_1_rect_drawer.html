uint32_t  charCount 
)
static

RectDrawer が必要なワークバッファのサイズを取得します。

詳細
メモリープール用のサイズも合算しています。 Initialize() で別にメモリープールを指定する場合は、この関数の戻り値から、CalculateMemoryPoolSize() の値を減算して利用してください。

引数
[in]pDeviceデバイス
[in]charCount1度に描画できる文字数です。
戻り値
頂点バッファ・コマンドバッファのメモリサイズを返します。( メモリは、GetWorkBufferAlignment で アライメント調整する必要があります。)

詳細
メモリープールは、nn::gfx::MemoryPoolProperty_CpuUncached、nn::gfx::MemoryPoolProperty_GpuCached プロパティを持つプールを指定してください。 サイズは、CalculateMemoryPoolSize() で、オフセットのアラインメントは、CalculateMemoryPoolAlignment() で求めます 。

引数

[in]pWorkBuffer頂点用バッファ( GetWorkBufferAlignment() で アライメント調整する必要があります。)

[in]pExternalMemoryPoolメモリープールです。
[in]externalMemoryPoolOffset利用するメモリープール領域へのオフセットです。
