ゲームのポーズダイアログを想定したサンプルオフライン HTML

概要

ゲームのポーズダイアログを想定したオフライン HTML のサンプルです。
本サンプルでは下記の機能を使っています。

  • 背景にアプリケーションのキャプチャー画像をぼかした画像を利用する
  • コールバック URL と戻り値を使って結果をアプリケーションへ返す
  • 任天堂外字を表示する
  • Web Audio を使って SE を鳴らす

ファイル構成

本サンプルオフライン HTML は Samples/Sources/Applications/WebDemo/html_document/sample/pause_menu.htdocs 以下にあります。

必要な環境

本サンプルは NX 実機環境のみサポートしています。

操作方法

アプリケーションの操作は下記の通りです。

入力 動作
B ボタン メニューへ戻ります。
+/- ボタン ゲーム中にポーズしたことを想定して、オフライン Web アプレットを起動してオフライン HTML を表示します。

オフライン HTML の操作は下記の通りです。

入力 動作
L スティックまたは上下ボタン 空間ナビを移動します。
A ボタン、タッチ 項目を決定します。
B ボタン ページを戻る、またはオフライン Web アプレットを終了して、アプリケーションへ戻ります。
+/- ボタン オフライン Web アプレットを終了して、アプリケーションへ戻ります。

注意事項

特にありません。

実行手順

WebDemo のサンプルプログラムをビルドし、実行してください。
メニューから本サンプルを選択してください。

解説

本サンプルの流れは下記の通りです。

  • メニューから本サンプルを選ぶと、アプリケーションへ遷移します。
  • アプリケーションはゲーム中を想定した画像を表示します。
  • +/- ボタンを押すと、ゲーム中にポーズしたことを想定して、オフライン Web アプレットを起動してオフライン HTML を表示します。
  • オフライン HTML の表示を終了すると、アプリケーションは、その結果に応じて再びゲーム中を想定した画像を表示するか、メニューへ戻ります。

背景にアプリケーションのキャプチャー画像をぼかした画像を利用する

nn::web::ShowOfflineHtmlPageArg::SetBackgroundKind() の引数に OfflineBackgroundKind_ApplicationCaptureBlur を指定します。

コールバック URL と戻り値を使って結果をアプリケーションへ返す

オフライン HTML の "Exit Stage" または "Exit Game" の項目を選ぶと、"http://localhost/" から始まる URL へ遷移します。
オフライン Web アプレットは、この URL をコールバック URL と判定してオフライン Web アプレットを終了し、URL をアプリケーションへ返します。
アプリケーションは返ってきた URL を nn::web::OfflineHtmlPageReturnValue::GetLastUrl() で取得し、 URL が "http://localhost/?id=" から始まっていたら、"http://localhost/?id=" に後続する数字に応じたログ出力を行った後にメニューへ戻ります。

任天堂外字を表示する

下ボタン、Joy-Con の + ボタン、- ボタンの任天堂外字を表示しています。

Web Audio を使って SE を鳴らす

Web Audio を使って、MP4 フォーマットの音声ファイルを再生しています。
AudioBufferSourceNode.onended イベントハンドラを使って、再生が終わるのを待ってからページを遷移しています。
