詳解

カレンダー情報を扱う構造体です。

詳細
西暦、月、日、時、分、秒からなる日時を扱う構造体です。

nn::TimeSpan との加減算が可能ですが、結果は秒単位に切り捨てられます。

各種例:
TimeClaimPeriodicBenefit.cpp, TimeStandardUserSystemClock.cpp.

関数詳解

bool nn::time::CalendarTime::IsValid ( ) const

有効なカレンダー情報を保持しているか判定します。

戻り値
有効かどうか
戻り値一覧:

true 有効

false 無効

詳細
year が 0 以下の場合は必ず false が返ります。

以下をすべて満たす場合に true を返します。

  • year, month, day が有効な日付である( nn::time::IsValidDate() と同等の判定 )
  • 0 <= hour <= 23
  • 0 <= minute <= 59
  • 0 <= second <= 59
