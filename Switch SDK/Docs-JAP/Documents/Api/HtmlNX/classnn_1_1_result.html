詳解

処理の結果を汎用的に表す型です。

詳細
このクラスは失敗する可能性のある関数の返り値として、汎用的に使用される型です。 このクラスのインスタンスは、すべての Result 値を格納することができます。

このクラスのデフォルトコンストラクタで生成されたインスタンスは、有効な Result 値を持ちません。 有効な Result 値は、各ライブラリで定義されるエラー Result クラスのインスタンスや、 ResultSucess クラスのインスタンスから変換することで生成されます。

構築子と解体子

nn::Result::Result ( )

有効な Result 値を持たないインスタンスを生成するデフォルトコンストラクタです。

詳細
このコンストラクタで生成されたインスタンスは、有効な Result 値を持たないため、 非 static メンバ関数を呼び出すことはできません。

nn::Result::operator ResultSuccess ( ) const

ResultSuccess 型への変換関数です。

戻り値
ResultSuccess 型のインスタンスを返します。
事前条件
*this が有効な Result 値を持っている。

詳細
持っている Result 値が成功を表すものでなかった場合(!this->IsSuccess() の場合)、 変換は失敗し、UnhandledResult ハンドラが呼ばれ、処理は返りません。
