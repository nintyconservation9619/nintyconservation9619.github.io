nn::audio::AuxReverbType 構造体

nn::audio::AuxType に適用する Reverb エフェクトのユーティリティです。 [詳解]

#include <nn/audio/audio_AuxUtilityEffectTypes.h>

詳解

nn::audio::AuxType に適用する Reverb エフェクトのユーティリティです。

詳細
nn::audio::AuxType によって得られたサンプルデータを入力として Reverb エフェクト処理を行うユーティリティです。
nn::audio::ReadAuxSendBuffer() によって得られたサンプルデータを、このエフェクトへの入力に指定することで、 閉じた空間内の音の響きをエミュレートする Reverb エフェクトを出力します。 このエフェクトによって施される Reverb エフェクトは nn::audio::ReverbType による処理と同一の内容です。
ただしこのエフェクト処理は nn::audio::ProcessAuxReverb() を呼び出した CPU スレッド上で処理されます。
メンバー変数を直接変更することは禁止です。
次の関数を介して、本構造体を利用してください。

(ワークバッファサイズ取得)

  • GetRequiredBufferSizeForAuxReverb(int sampleRate, int channelCountMax)

(初期化関連)

  • InitializeAuxReverb()
  • ResetAuxReverb()

(接続先設定)

  • SetAuxReverbInputOutput()
  • GetAuxReverbInputOutput()

(パラメータ)

  • SetAuxReverbParameters()
  • GetAuxReverbParameters()

(エフェクト処理)

  • ProcessAuxReverb()

上記の関数を利用するには nn/audio.h をインクルードする必要があります。

各種例:
AudioEffectAuxUtility.cpp.
