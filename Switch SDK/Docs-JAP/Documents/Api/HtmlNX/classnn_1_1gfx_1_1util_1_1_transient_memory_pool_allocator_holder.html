class nn::gfx::util::TransientMemoryPoolAllocatorHolder< THistoryLengthMax >

TransientMemoryPoolAllocator とその初期化に必要なメモリブロックを包含してオブジェクトの構築を行うクラスです。

テンプレート引数

このオブジェクトと包含する TransientMemoryPoolAllocator を初期化します。

引数
[in]pMemoryPoolメモリプールです。
[in]baseOffset管理するメモリの先頭のオフセットです。
[in]size管理するメモリのサイズ（バイト）です。
[in]allocatableAlignmentMaxこのアロケータから確保するメモリに要求する最大のアライメントです。
事前条件
このオブジェクトは未初期化でなければなりません。
各パラメータの事前条件は TransientMemoryPoolAllocator::Initialize() と同一です。
事後条件
このオブジェクトは初期化済になります。
参照
TransientMemoryPoolAllocator::Initialize()

詳細
指定したパラメータを用いてこのオブジェクトが包含する TransientMemoryPoolAllocator のインスタンスを初期化します。 初期化に使用するメモリブロックにはこのオブジェクトが包含しているメモリ領域を使用します。 履歴の最大数には HistoryLengthMax が指定されます。
同一インスタンスに対するこの関数の呼び出しはスレッドセーフではありません。

template<int THistoryLengthMax>
TransientMemoryPoolAllocator* nn::gfx::util::TransientMemoryPoolAllocatorHolder< THistoryLengthMax >::Get ( )

このオブジェクトが包含する TransientMemoryPoolAllocator を取得します。

戻り値
のオブジェクトが包含する TransientMemoryPoolAllocator のポインタを返します。

詳細
取得される TransientMemoryPoolAllocator の初期化および破棄はこのオブジェクトの Initialize(), Finalize() により行います。 取得される TransientMemoryPoolAllocator の生存期間はこのオブジェクトの生存期間と同一です。
この関数はスレッドセーフです。
