テクスチャーを初期化します。

引数
[in]pDeviceデバイスへのポインター
[in]info初期化用の情報
[in]pMemoryPoolテクスチャーのミップマップデータを配置するメモリープールへのポインター
[in]memoryPoolOffsetバイトでのテクスチャーのミップマップデータを配置するメモリープールへのオフセット
[in]memoryPoolSizeバイトでのテクスチャーが使用するメモリープールのサイズ
事前条件
  • pDevice != NULL
  • pDevice が初期化されている
  • テクスチャーが初期化されていない
  • IsMemoryPoolRequired が true の場合、pMemoryPool != NULL
  • IsMemoryPoolRequired が true の場合、memoryPoolSize が CalculateMipDataSize で取得されるサイズ以上
事後条件
  • テクスチャーが初期化されている
