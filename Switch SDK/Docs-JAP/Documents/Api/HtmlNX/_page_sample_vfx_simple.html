サンプルプログラム VfxSimple の解説です。

概要

nn::vfx を使用してエフェクトバイナリを再生する最もシンプルなサンプルです。

エフェクトランタイムの機能の使い方については、 Vfx の関数リファレンス も併せて参照して下さい。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/VfxSimple 以下にあります。

必要な環境

画面表示が利用可能である必要があります。

操作方法

このサンプルを実行するとエフェクトが自動的に再生され、一定回数描画した後終了します。

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

576 void CalculateCallback(nns::gfx::GraphicsFramework* pGraphicsFramework, void* pUserData)
577 {
578  NN_UNUSED(pGraphicsFramework);
579  CalculateCallbackUserData* pCallbackUserData = reinterpret_cast<CalculateCallbackUserData*>(pUserData);
580 
581  // 300フレーム毎に次のエミッタを再生する
582  if ((*pCallbackUserData->pFrame != 0) && (*pCallbackUserData->pFrame % 300) == 0)
583  {
584  if (pCallbackUserData->emitterSetMaxCount > *pCallbackUserData->pEmitterSetId)
585  {
586  g_pVfxSystem->KillEmitterSetGroup(pCallbackUserData->groupID);
587  (*pCallbackUserData->pEmitterSetId)++;
588  g_pVfxSystem->CreateEmitterSetId(pCallbackUserData->pEmitterSetHandle, *pCallbackUserData->pEmitterSetId, pCallbackUserData->resourceId, pCallbackUserData->groupID);
589  if (pCallbackUserData->pEmitterSetHandle->IsValid())
590  {
591  GetViewParamaters(pCallbackUserData->pEmitterSetHandle, pCallbackUserData->pViewMatrix, pCallbackUserData->pCameraPos, pCallbackUserData->pScaleValue, pCallbackUserData->pIsCircleMove);
592  NN_SDK_LOG("%s\n", pCallbackUserData->pEmitterSetHandle->GetEmitterSet()->GetName());
593  }
594  (*pCallbackUserData->pMoveTime) = 0;
595  }
596  }
597  (*pCallbackUserData->pFrame)++;
598 
599  // 最後のエミッタの再生時間
600  if (pCallbackUserData->emitterSetMaxCount <= *pCallbackUserData->pEmitterSetId)
601  {
