VertexAttribState & nvn::VertexAttribState::SetDefaults ( )

Set default state for the VertexAttribState object.

覚え書き
This is semantically equivalent to calling
VertexAttribState::SetFormat(NVN_FORMAT_NONE, 0);
VertexAttribState::SetStreamIndex(0);

except that the object will be disabled until the first valid call to VertexAttribState::SetFormat().

VertexAttribState & nvn::VertexAttribState::SetFormat ( Format  format,
ptrdiff_t  relativeOffset 
)

Specifies the format (in memory) for values for a single vertex attribute.

Specifying format is mandatory, and doing so also enables the vertex attribute for fetching.

引数
formatFormat of the specified vertex attribute.
relativeOffsetOffset (in bytes) of the values for the vertex attribute relative to the base address from the vertex buffer binding.
