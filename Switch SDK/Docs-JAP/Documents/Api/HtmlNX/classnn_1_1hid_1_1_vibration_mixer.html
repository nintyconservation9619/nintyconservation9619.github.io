詳解

振動値のミキサーとしての機能を提供する振動ノードです。

詳細
VibrationMixer は、接続元の振動ノードで発生した振動値を受け取って ミックスする機能を提供する振動ノードです。

ミックスされた振動値は、 nn::hid::VibrationNode::Update が呼ばれるたびに 接続先の振動ノードに伝えられます。

構築子と解体子

nn::hid::VibrationMixer::VibrationMixer ( )

コンストラクタです。
本関数はスレッドセーフではありません。
本関数とnn::hid::VibrationNode::Update 関数を同時に呼ばないように排他制御をしてください。

事後条件
GetMixMode() == MixMode_MaxAmplitudePerSubband

static void nn::hid::VibrationNode::Update ( )
staticinherited

振動値フローの有向グラフ全体を更新して、 それぞれの振動ノードが接続先に振動値を伝えます。

詳細
この関数が呼ばれると、すべての振動ノードの内部状態が更新されます。
その際、隣接している振動ノード間では、接続元から接続先に対して振動値が伝えられます。
接続元の更新処理は接続先の更新処理よりも先に行われます。
この関数は任意のタイミングで実行することができますが、 タイマー等を利用して、DefaultVibrationSampleInterval 間隔で 実行することが推奨されます。

本関数はスレッドセーフではありません。
本関数と本クラスおよび派生クラスのコンストラクタおよびデストラクタを同時に呼ばないように排他制御をしてください。
