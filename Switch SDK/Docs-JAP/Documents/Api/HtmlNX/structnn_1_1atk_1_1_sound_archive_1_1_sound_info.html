nn::atk::SoundArchive::SoundInfo 構造体

サウンドの設定情報の構造体です。 [詳解]

#include <nn/atk/atk_SoundArchive.h>

公開変数類

uint8_t actorPlayerId
 アクターでサウンドを再生するときに使用されるアクタープレイヤーの番号です。
 
FileId fileId
 サウンドアーカイブ内でのファイル ID です。
 
bool isFrontBypass
 フロントバイパスするか否かを示すフラグです。
 
PanCurve panCurve
 パンのカーブの種類です。
 
PanMode panMode
 パンモードの種類です。
 
ItemId playerId
 サウンドを再生するときに使用されるプレイヤーの ID です。
 
uint8_t playerPriority
 サウンドを再生するときのプレイヤープライオリティの値です。
 
uint8_t remoteFilter
 リモコンフィルタの値です。
 
uint16_t singlePlayEffectiveDuration
 単一再生の効果時間です。
 
SinglePlayType singlePlayType
 単一再生の種類です。
 
uint8_t volume
 サウンドを再生するときの音量です。
 

詳解

サウンドの設定情報の構造体です。

isFrontBypass が true で、SoundSystem::SetOutputMode で設定する出力モードが OutputMode_Surround の場合、 当該サウンドはフロントチャンネルをバイパスします。
