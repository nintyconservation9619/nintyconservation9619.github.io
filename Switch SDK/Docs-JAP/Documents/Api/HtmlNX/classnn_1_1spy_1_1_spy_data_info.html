詳解

nn::spy ライブラリ、Spy.exe で扱うデータ情報を格納します。

構築子と解体子

nn::spy::SpyDataInfo::SpyDataInfo ( const char *  dataName,
const uint32_t  dataVersion 
)

コンストラクタです。

引数
[in]dataNameデータ名を指定します。 名前は ASCII 文字列で終端文字を含め、128 文字以下にしてください。
[in]dataVersionデータバージョンを指定します。 バージョンは、2 バイト刻みで メジャー、マイナー、マイクロ、(未使用) の 4 つの数値を示します。 バージョン番号 0x00000000 は不正値とみなされます。

関数詳解

nn::spy::detail::SpyDataId nn::spy::SpyDataInfo::GetDataId ( ) const

内部用機能のため使用禁止です。

セッション中に有効なDataIdの値を取得します。

詳細
Spy.exe により割り当てられたデータ名に対応する DataId です。
