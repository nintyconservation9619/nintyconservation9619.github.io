template<typename TTarget >
SyncResult nn::gfx::TFence< TTarget >::Sync ( TimeSpan  timeout)

フェンスがシグナル状態になるのを同期します。

引数
[in]timeout同期する最大待ち時間
戻り値
同期処理の結果を返します。
戻り値一覧:

SyncResult_Success 成功しました。

SyncResult_TimeoutExpired フェンスがシグナル状態になる前に timeout で指定されている時間を過ぎました

事前条件
  • フェンスがキューにセットされている

詳細
フェンスがシグナル状態になるまで、または指定された時間に達するまで処理を返しません。
