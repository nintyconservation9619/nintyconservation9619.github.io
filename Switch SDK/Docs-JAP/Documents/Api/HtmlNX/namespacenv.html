詳解

NX において、複数のグラフィックス API から共通して必要となる機能を提供する名前空間です。

関数詳解

void nv::SetGraphicsAllocator ( AllocateFunction  allocate,
FreeFunction  free,
ReallocateFunction  reallocate,
void *  userPtr 
)

Set memory allocator for graphics subsystem.

引数
[in]allocateA function pointer for allocating memory.
[in]freeA function pointer for freeing memory.
[in]reallocateA function pointer for reallocating memory.
[in]userPtrUser pointer that will be passed to callbacks.

詳細
This function must be called before using any graphics APIs. The relative order of nv::InitializeGraphics, nv::SetGraphicsAllocator and nv::SetGraphicsDevtoolsAllocator calls does not matter, as long as the application makes all its calls before using any graphics API's. Failure to do so will result in the application being aborted.

void nv::InitializeGraphics ( void *  address,
size_t  size 
)

Specify the address and size of memory that is donated to graphics firmware.

詳細
This function must be called before using any graphics APIs. The relative order of nv::InitializeGraphics, nv::SetGraphicsAllocator and nv::SetGraphicsDevtoolsAllocator calls does not matter, as long as the application makes all its calls before using any graphics API's. Failure to do so will result in the application being aborted. The donated memory block will be unmapped from process address space and cannot be used afterwards.

引数
[in]addressAddress of memory being donated to graphics firmware.
[in]sizeSize of memory being donated to graphics firmware.
各種例:
GfxPrimitiveRenderer.cpp, GfxShaderCompile.cpp, GfxSimple.cpp, GfxSimpleCompute.cpp, HidControllerSequence_Main.cpp, HidGesture_Main.cpp, HidKeyboardAndMouse_Main.cpp, HidNpadIntegrate_Main.cpp, HidNpadSixAxisSensor_Main.cpp, HidSimple_Main.cpp, HidVibrationBasic.cpp, HidVibrationGenerator.cpp, HidVibrationPlayer.cpp, HidVibrationRollingBall.cpp, IrsensorViewer.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp, MovieDecoderPlayer.cpp, MoviePlayerSimple.cpp, PcmBasic.cpp, SwkbdSimple.cpp, TutorialBaseClass.Nx.cpp, VkSimple.cpp (計34項目).
void nv::FinalizeGraphics ( )

Release resources from graphics subsystem.

詳細
This function must be called once the client is no longer making use of the graphics subsystem. It will release the resources in use from the memory donated in nv::InitializeGraphics(), and also free the memory received from the allocator previously set up with the nv::SetGraphicsAllocator() function.

各種例:
HidNpadIntegrate_Main.cpp, HidVibrationBasic.cpp, HidVibrationGenerator.cpp, HidVibrationPlayer.cpp, HidVibrationRollingBall.cpp, MovieDecoderPlayer.cpp, MoviePlayerSimple.cpp.
void nv::GetGraphicsMemoryInfo ( MemoryInfo *  memoryInfo)

Retrieve information about current memory usage status of firmware.

詳細
The function returns current memory usage status of the memory heap that was provided for firmware through nv::InitializeGraphics.

引数
[out]memoryInfoA pointer to structure that will receive the info.
void nv::DumpGraphicsMemoryInfo ( )

Dump detailed information about graphics memory status to log.

詳細
This method will not print information for SDK users. To debug memory leaks, consider using nvn::Device::WalkDebugDatabase() instead. Please see the API Object Tracking section of the NVN Programming Guide.
