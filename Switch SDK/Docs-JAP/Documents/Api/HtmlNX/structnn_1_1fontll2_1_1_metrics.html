#include <nn/fontll2/fontll2_ScalableFontEngine.h>

 著作権情報文字列です。

 予約されたメンバです。
 
BoundingBox fontBoundingBox
 境界矩形情報です。デザイン単位における値が格納されています。
 
char fontFamilyName [FontFamilyNameLengthMax]
 フォントファミリー名です。
 
uint32_t fontInTtcCount
 TrueType collection 中に含まれるフォント数です。
 
char fontName [FontNameLengthMax]
 フォント名です。
 
uint16_t fontType
 フォントの種類を表す内部メンバーです。
 
uint32_t glyphCount
 フォントが持つグリフ数です。
 
uint16_t headMacStyle
 head テーブルにおける MacStyle フラグ（表示形式に関するフラグ）です。
 
int16_t hheaAscent
 hhea テーブルにおけるアセント（ベースラインからグリフ上端まで）量です。デザイン単位における値が格納されています。
 
int16_t hheaDescent
 hhea テーブルにおけるディセント（ベースラインからグリフ下端まで）量です。デザイン単位における値が格納されています。
 
uint16_t hheaLeading
 hhea テーブルにおけるレディング（行と行の間隔）量です。デザイン単位における値が格納されています。
 
uint32_t iconCount
 フォントが持つアイコン数です。
 
uint32_t indexInTtc
 TrueType collection でのフォント番号です。
 
CharacterMapping mappings [MappingLengthMax]
 内部メンバーです。
 
uint16_t metricsResolution
 フォント解像度です。デザイン単位で格納されている値に乗算してピクセルサイズに変換する場合に利用します。
 
int16_t os2Ascent
 os2 テーブルにおけるアセント（ベースラインからグリフ上端まで）量です。デザイン単位における値が格納されています。
 
int16_t os2Descent
 os2 テーブルにおけるディセント（ベースラインからグリフ下端まで）量です。デザイン単位における値が格納されています。
 
uint16_t os2FsSelection
 os2 テーブルにおけるフォントセレクションフラグ（表示形式に関するフラグ）です。
 
uint16_t os2Leading
 os2 テーブルにおけるレディング（行と行の間隔）量です。デザイン単位における値が格納されています。
 
int16_t os2WinAscent
 hhea テーブルにおける Windows形式のアセント（ベースラインからグリフ上端まで）量です。デザイン単位における値が格納されています。
 
int16_t os2WinDescent
 hhea テーブルにおけるWindows形式のディセント（ベースラインからグリフ下端まで）量です。デザイン単位における値が格納されています。
 
uint16_t unitsPerEm
 デザイン単位を emサイズ（通常アルファベット M の大きさ）で割った値です。
