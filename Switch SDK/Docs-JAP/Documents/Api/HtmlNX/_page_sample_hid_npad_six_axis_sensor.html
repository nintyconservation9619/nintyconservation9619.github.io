Npadを使った SixAxisSensor の状態の取得

目次

  • 概要
  • ファイル構成
  • 必要な環境
  • 操作方法
  • 実行手順
  • 解説

Npadを使って SixAxisSensor の状態を取得するためのサンプルプログラムの解説です。

概要

コントローラの最新の SixAxisSensor の状態を取得する方法について説明します。

ファイル構成

本サンプルプログラムは ../../../Samples/Sources/Applications/HidNpadSixAxisSensor 以下にあります。

必要な環境

Windows 環境で動作させる場合は、事前に PC に Bluetooth ドングルを接続した上で、PC とコントローラをペアリングしてください。 SDEV 環境で動作させる場合は、事前に SDEV とコントローラをペアリングしてください。

操作方法

サンプルプログラムを実行すると Windows 環境ではウィンドウ上に、NX 開発機環境ではタッチスクリーン上に それぞれの SixAxisSensor の状態が表示されます。

コントローラを操作すると状態の変化が反映されます。 コントローラは 4台同時接続までサポートされています。

左列から順に以下の状態を表示します。

  • SixAxisSensor の加速度・角速度・回転角・姿勢の値の表示
  • SixAxisSensorState::acceleration の XY・YZ・ZX成分を 2次元座標系で表示
  • SixAxisSensorState::angularVelocity の XY・YZ・ZX成分を 2次元座標系で表示
  • SixAxisSensorState::angle の X・Y・Z軸周りの回転を 2次元座標系で表示
  • SixAxisSensorState::GetQuaternion の姿勢情報を 3次元ワイヤーフレームで表示

各コントローラの表示は、ControllerManager に追加された順番に、 Red, Orange, Yellow, Green に色分けされて表示されます。

サンプルプログラムを終了させるには、コントローラの + ボタンと - ボタンを同時に押すか、 VisualStudio または Nintendo Target Manager 経由で終了させてください。

コントローラの + ボタン押下で、以下の処理が行われます。

  • 3 次元ワイヤーフレームの姿勢を初期化
  • ポインタ座標をスクリーン中心位置に初期化

実行手順

サンプルプログラムをビルドし、実行してください。

任意のボタン押下で、ボタンを押下した操作形態に対応するコントローラーの最新の SixAxisSensorState が コンソールウィンドウにダンプされます。 ダンプされた加速度・角速度・回転角度・各軸姿勢ベクトルは、それぞれ左から x,y,z 軸成分が表示されます。

解説

サンプルプログラムの全体像は以下の通りです。

  • Npad ID を取得
  • Npad を初期化
  • SixAxisSensorHandle の取得
  • 6軸センサー機能のアクティベート
  • 現在のボタン入力状態を取得
  • 現在の6軸センサー入力状態を取得
  • 現在のタイムスタンプからパケット落ち率を推定
