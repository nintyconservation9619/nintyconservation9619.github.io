詳解

Option flags that control compilation and GLSLC output.

This struct is used as part of the GLSLCoptions struct.

メンバ詳解

uint32_t GLSLCoptionFlags::outputAssembly

Include an assembly dump section in the GLSLCoutput.

There will be a separate assembly dump section for each shader stage.

各種例:
AssetFileLoadingHelper.cpp, IntermediateFileManager.cpp.
uint32_t GLSLCoptionFlags::outputGpuBinaries

Output individual program binary sections in the GLSLCoutput.

There will be one binary per stage.

各種例:
AssetFileLoadingHelper.cpp, IntermediateFileManager.cpp.
uint32_t GLSLCoptionFlags::outputThinGpuBinaries

Controls whether the resulting output contains a GPU program that is a thin binary (only contains program code for NX platforms).

各種例:
AssetFileLoadingHelper.cpp.
uint32_t GLSLCoptionFlags::tessellationAndPassthroughGS

Compiles tessellation evaluation and passthrough geometry shaders (NV_geometry_shader_passthrough) to work together as a pair.

When a tessellation evaluation or passthrough geometry shader compiled with this option is bound for rendering, the other stage must also have a shader compiled with this option bound. Using such shaders without a similarly compiled shader in the other stage will produce undefined results.

uint32_t GLSLCoptionFlags::prioritizeConsecutiveTextureInstructions

Optimize texture batches for intra-warp cache locality in compute shaders.

Does not affect non-compute shaders.

uint32_t GLSLCoptionFlags::enableFastMathMask

A mask indicate which shader stages to enable fast math optimizations.

Fast math optimizations allow the compiler to make decisions which might not strictly adhere to IEEE standards. The mask corresponds to an NVNshaderStageBits type.

NOTE: The function glslcGetDefaultOptions will enable fast math optimizations in fragment shaders by default (i.e. will initialize enableFastMathMask with a value of NVN_SHADER_STAGE_FRAGMENT_BIT).

NOTE: It is recommended that fast math be enabled for fragment shaders only, while other stages such as vertex/geometry/tessellation/compute might be more sensitive to precision differences in the presence of fast math optimizations and might produce incorrect results.

uint32_t GLSLCoptionFlags::enableMultithreadCompilation

Option to enable multithreaded compilation using separate GLSLCcompileObject.

As of package version 50, up to 64 concurrent GLSLCcompileObjects can have this enabled. Prior to package version 50, up to 16 concurrent GLSLCcompileObjects can have this enabled. Use the GLSLC API function glslcGetVersion to query the library's package version.
