nn::ldn::ScanFilter 構造体

スキャンのフィルタを格納する構造体です。 [詳解]

#include <nn/ldn/ldn_Types.h>

公開変数類

Bit32 flag

NetworkId networkId
 ネットワークの識別子です。
 
Bit32 networkType
 スキャン対象となるネットワークです。 NetworkType の値を論理和で指定してください。

 アクセスポイントの SSID です。
 

詳解

スキャンのフィルタを格納する構造体です。

詳細
Scan 関数の結果をフィルタリングするための条件を設定する構造体です。

まず、 networkType, networkId, ssid のうち、 フィルタリングに使用したいメンバ全てに値を設定してください。 ここで指定した値が一致するネットワークだけが探索結果に含まれます。

その後、フィルタリングに使用するメンバを LDN ライブラリに知らせるために、 flag に ScanFilterFlag で定義された定数の論理和を格納してください。 例えば、 networkType と networkId.intentId でフィルタリングしたい場合、 ScanFilterFlag_NetworkType | ScanFilterFlag_IntentId を指定します。
