詳解

時計の各種情報を保持するクラスです。

このクラスは C++ 標準の TriviallyCopyable の要件を満たしており、 std::memcpy() が可能なクラスです。

アプリケーションが動作していない期間を含め、本クラスを扱いたい場合には、 オブジェクトをセーブデータに保存して次回の起動へ受け渡してください。

メンバ関数はスレッドアンセーフです。

参照
nn::time::CalculateStandardUserSystemClockDifferenceByUser(), nn::time::CalculateSpanBetween()
各種例:
TimeCalculateUserSystemClockDifference.cpp, TimeClaimPeriodicBenefit.cpp.

構築子と解体子

nn::time::ClockSnapshot::ClockSnapshot ( )

コンストラクタです。

詳細
オブジェクトが保持する情報は以下になります

  • nn::time::PosixTime, nn::time::CalendarTime, nn::time::CalendarAdditionalInfo は協定世界時 (UTC) の1970年1月1日午前0時0分0秒 を指す値
  • nn::time::LocationName は協定世界時 (UTC) を指す値
  • nn::time::SystemClockContext, nn::time::SteadyClockTimePoint は無効な値

関数詳解

static void nn::time::ClockSnapshot::CreateWithStandardSystemClock ( ClockSnapshot *  pOut)
static

システムの最新時刻でオブジェクトを生成します。

引数
[out]pOutオブジェクトを格納するバッファへのポインタを指定します。
事前条件
  • TIME ライブラリが初期化済である
  • pOut != nullptr

詳細
nn::time::StandardUserSystemClock, nn::time::StandardNetworkSystemClock から得られるシステムの最新時刻を使ってオブジェクトを生成します。

ネットワーク時計が補正されていない場合には、 ネットワーク時計の時刻として協定世界時 (UTC) の1970年1月1日午前0時0分0秒を保持します。

各種例:
TimeCalculateUserSystemClockDifference.cpp, TimeClaimPeriodicBenefit.cpp.
