template<int THistoryLengthMax>
class nn::gfx::util::TransientBufferAllocatorHolder< THistoryLengthMax >

TransientBufferAllocator とその初期化に必要なメモリブロックを包含してオブジェクトの構築を行うクラスです。

テンプレート引数
THistoryLengthMax構築する TransientBufferAllocator が記録する履歴の最大数です。
参照
TransientBufferAllocator

ptrdiff_t  baseOffset,

このオブジェクトと包含する TransientBufferAllocator を初期化します。

引数
[in]pBufferバッファです。
[in]baseOffset管理する領域の先頭のオフセットです。
[in]size管理する領域のサイズ（バイト）です。
[in]allocatableAlignmentMaxこのアロケータから確保する領域に要求する最大のアライメントです。
事前条件
このオブジェクトは未初期化でなければなりません。
各パラメータの事前条件は TransientBufferAllocator::Initialize() と同一です。
事後条件
このオブジェクトは初期化済になります。
参照
TransientBufferAllocator::Initialize()

詳細
指定したパラメータを用いてこのオブジェクトが包含する TransientBufferAllocator のインスタンスを初期化します。 初期化に使用するメモリブロックにはこのオブジェクトが包含しているメモリ領域を使用します。 履歴の最大数には HistoryLengthMax が指定されます。
同一インスタンスに対するこの関数の呼び出しはスレッドセーフではありません。

事前条件

事後条件

詳細

template<int THistoryLengthMax>
TransientBufferAllocator* nn::gfx::util::TransientBufferAllocatorHolder< THistoryLengthMax >::Get ( )

このオブジェクトが包含する TransientBufferAllocator を取得します。

戻り値
のオブジェクトが包含する TransientBufferAllocator のポインタを返します。

詳細
取得される TransientBufferAllocator の初期化および破棄はこのオブジェクトの Initialize(), Finalize() により行います。 取得される TransientBufferAllocator の生存期間はこのオブジェクトの生存期間と同一です。
この関数はスレッドセーフです。
