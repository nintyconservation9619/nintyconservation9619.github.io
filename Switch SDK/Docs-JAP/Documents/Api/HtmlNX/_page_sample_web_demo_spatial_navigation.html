CSS を使って空間ナビを様々な独自のフォーカス表現に変更するサンプルオフライン HTML

概要

CSS を使って空間ナビを様々な独自のフォーカス表現に変更するオフライン HTML のサンプルです。
本サンプルでは下記の機能を使っています。

  • 空間ナビを独自のフォーカス表現に変更する
  • タッチ操作を禁止

ファイル構成

本サンプルオフライン HTML は Samples/Sources/Applications/WebDemo/html_document/sample/spatial_navigation.htdocs 以下にあります。

必要な環境

本サンプルは NX 実機環境のみサポートしています。

操作方法

入力 動作
L スティックまたは上下左右ボタン 空間ナビを移動します。
B ボタン、X ボタン オフライン Web アプレットを終了して、アプリケーションを経由してメニューへ戻ります。

注意事項

特にありません。

実行手順

WebDemo のサンプルプログラムをビルドし、実行してください。
メニューから本サンプルを選択してください。

解説

メニューから本サンプルを選ぶと、アプリケーションを経由してオフライン Web アプレットを起動し、本サンプルオフライン HTML を表示します。

空間ナビを独自のフォーカス表現に変更する

meta 要素の focus-ring-visibility 属性を使って、空間ナビのフォーカス枠を非表示にしています。
button 要素に対して CSS の :focus 疑似クラスを使って、フォーカスした時のスタイルを指定しています。

タッチ操作を禁止

nn::web::ShowOfflineHtmlPageArg::SetTouchEnabledOnContents() を使って、タッチ操作を無効にしています。
