詳細
AES (Rijndael) 暗号による復号化を実装するクラスです。

このクラスは鍵サイズをテンプレート引数とするテンプレートクラスです。 実際の使用には AesDecryptor128, AesDecryptor192, AesDecryptor256 型を使用してください。

暗号に対する十分な知識があり、何をしているかを十分に理解しているのでない限り、このクラスの DecryptBlock() を直接使用すべきではありません。

このクラスは以下の状態を持ちます。 インスタンスの生成時は未初期化状態です。

  • 未初期化状態
  • 処理可能状態

1ブロックの復号化を行います。

引数
[out]pDst復号化されたデータを受け取るバッファへのポインタ。
[in]dstSizepDst が指すバッファのバイトサイズ。
[in]pSrc復号化するデータが格納されているバッファへのポインタ。
[in]srcSize復号化するデータのバイトサイズ。
事前条件
  • dstSize == BlockSize && srcSize == BlockSize
  • インスタンスは処理可能状態である
事後条件
  • pDst に BlockSize の復号化された結果が書き込まれる。

詳細
復号化ではブロックサイズ分の暗号文を同サイズの平文に変換します。 pSrc からブロックサイズ分のデータを読み取り、復号化して pDst にその結果を書き込みます。
