詳解

フレンドを扱うクラスです。

詳細
あるユーザーアカウントに関するフレンドの情報を参照することができます。
複数のユーザーアカウントが同一ユーザーをフレンドにしていたとしても、ユーザーアカウントごとに参照できる内容は異なります。

各種例:
FriendsPresence.cpp, FriendsSimple.cpp.

nn::Result nn::friends::Friend::GetProfileImage ( size_t *  outSize,
void *  buffer,
size_t  size 
) const

本体ストレージ上にキャッシュしたユーザーのプロフィール画像を取得します。

引数
[out]outSize取得したサイズ。
[in]bufferバッファ。
[in]sizeバッファのサイズ。
戻り値
処理結果。


戻り値一覧:
ResultProfileImageCacheNotFound フレンドのプロフィール画像のキャッシュが存在しません。
デフォルトのユーザーアイコンを表示するか、別途 URL を取得してダウンロードしてください。
事前条件
  • outSize != nullptr
  • buffer != nullptr
  • size > 0
  • 初期化時に指定したユーザーアカウントが Open 状態である。

詳細
プロフィール画像は、256 × 256 の JPEG 画像で保存されています。
ファイルの最大サイズは 100KB 以内に収まります。

別サイズのプロフィール画像を取得したい場合、 GetProfileList で URL を取得し、ダウンロードしてください。

bool nn::friends::Friend::IsNewly ( ) const

新規に成立したフレンドかどうかを確認します。

戻り値
新規に成立したフレンドかどうか。

詳細
新規に成立したフレンドには、「マイページ」上で新着マークが付きます。
このフラグは、ユーザーが「マイページ」上で新着マークを見た時に false になります。

新着マークは「マイページ」上での表示用であるため、アプリケーション内で表示してはいけません。
