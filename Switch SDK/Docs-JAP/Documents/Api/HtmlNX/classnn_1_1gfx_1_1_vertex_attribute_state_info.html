関数詳解

void nn::gfx::VertexAttributeStateInfo::SetDefault ( )

各パラメーターを既定値に設定するためのヘルパー関数です。

事後条件
  • 各パラメーターが設定されている

詳細
以下を呼ぶことに相当します。

  • SetSemanticIndex( 0 );
  • SetShaderSlot( -1 );
  • SetBufferIndex( 0 );
  • SetOffset( 0 );
  • SetFormat( nn::gfx::AttributeFormat_Undefined );
  • SetNamePtr( NULL );
各種例:
GfxShaderCompile.cpp, GfxSimple.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計16項目).

void nn::gfx::VertexAttributeStateInfo::SetShaderSlot ( int  value)

シェーダーでのスロット番号を設定します。

引数
[in]valueスロット番号
事前条件
  • value >= -1 && value <= 32767
事後条件
  • シェーダーでのスロット番号が設定されている

詳細
スロット番号はシェーダーから問い合わせた値を設定してください。
SetNamePtr に NULL 以外が設定されている場合は SetNamePtr の値が優先されます。
無効なスロット番号を設定するには -1 を設定します。

各種例:
GfxSimple.cpp, MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSystemResource.cpp.
