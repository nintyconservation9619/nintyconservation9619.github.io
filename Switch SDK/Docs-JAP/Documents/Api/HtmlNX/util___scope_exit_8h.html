詳解

スコープを抜ける際に実行するコードを記述するためのマクロの定義しています。

マクロ定義詳解

#define NN_UTIL_SCOPE_EXIT

スコープを抜ける際の処理を記述するためのマクロです。

詳細
本マクロに続けて、ブロックを記述すると、 ブロック内の処理が、スコープを抜ける際に実行されます。 なお、ブロックの閉じ中括弧の後にセミコロンが必要です。

使用例：

1 {
2  FILE* f = std::fopen(path, "r");
3  NN_UTIL_SCOPE_EXIT
4  {
5  std::fclose(f);
6  };
7 
8  ... std::fread などの呼び出し
9 
10  // スコープ終了時に自動的に std::fclose(f) が呼ばれる
11 }
各種例:
AccountApplication.cpp, AccountApplicationAuthorization.cpp, BcatPractical.cpp, OeLaunchParameter.cpp, OsUserExceptionHandlerSample.cpp.
