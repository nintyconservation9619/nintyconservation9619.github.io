リサンプラのサンプルプログラムの解説です。

概要

リサンプラを利用してサンプルレート変換を行うサンプルです。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/AudioOutWithResampler 以下にあります。

必要な環境

オーディオ出力が利用可能である必要があります。

操作方法

サンプルプログラムを実行すると、矩形波を再生されます。 キーボードや DebugPad による入力を用いてプログラムを操作することが出来ます。

入力 動作
[Start/Space] プログラムの終了

注意事項

特にありません。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

このサンプルプログラムは Samples/Sources/Applications/AudioOut とリサンプラを組み合わせたサンプルです。 44.1kHz の波形を 48kHz の AudioOut を用いて再生するために、サンプルレート変換を行います。

サンプルプログラムの処理の流れは以下の通りです。

  • 利用可能なオーディオ出力を列挙
  • デフォルトのオーディオ出力をオープン
  • オーディオ出力の情報を取得、確認
  • リサンプラを準備
  • 再生用バッファを準備、登録
  • 再生を開始
  • オーディオ再生を実行
  • 再生を終了
  • オーディオ出力をクローズ
  • 再生用バッファを破棄

リサンプラの初期化は nn::audio::InitializeResampler() により行います。入出力オーディオデータの サンプルレートと、チャンネル数はこの時点で指定する必要があり、初期化後に変更することはできません。 また、初期化時にはサイズ nn::audio::GetRequiredBufferSizeForResampler() のワークバッファを渡す必要があります。 このバッファはリサンプラを使用終了するまで、解放してはいけません。

リサンプル処理は nn::audio::ProcessResamplerBuffer() により行われます。 リサンプル処理の入出力サンプル数はサンプルレートの比により決まります。 出力バッファのサイズはあらかじめ nn::audio::GetResampledOutBufferSize() で計算しておき、 そのサイズ以上のバッファを用意しておく必要があります。
