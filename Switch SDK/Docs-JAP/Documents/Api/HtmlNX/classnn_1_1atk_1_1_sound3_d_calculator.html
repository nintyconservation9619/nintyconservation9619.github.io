詳解

3D サウンドの標準的なパラメータ計算処理をまとめたクラスです。

Sound3DEngine クラスから呼び出されています。

カスタマイズして独自の 3D サウンドエンジンを作成する場合には、 このクラスの関数を利用することができます。

3D サウンドの音量と優先度の標準的なパラメータ計算を行います。

3D サウンドでは音量と優先度が連動しますので、 この関数でまとめて計算を行います。

この関数は、過去の互換性維持のために残されています。 現在の nn::atk ライブラリでは、計算処理削減のため、 当関数の代わりに actorDistance 引数付きのオーバーロード関数を呼び出しています。

引数
[out]pOutVolume音量の計算結果を格納するポインタです。
[out]pOutPriority優先度の計算結果を格納するポインタです。
[in]manager計算に使用する 3D サウンドマネージャです。
[in]listener計算に使用する 3D サウンドリスナーです。
[in]actorParam計算に使用する 3D サウンドパラメータです。
