詳解

Maskテクスチャ描画用バッファ

詳細
TextureShader でMaskテクスチャを描画するときに
必要なバッファなどをまとめたクラスです。
このクラスは、1つのMaskテクスチャを描画するごとに1つ必要です。

注意
TextureShader を使わない場合は不要です。
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).

MaskGpuBufferを初期化します。

詳細
初期化後のデフォルト値は以下の通りです。

注意
pMemory および pMemoryPoolは、MaskGpuBuffer::Finalize() まで保持する必要があります。
初期化済みのMaskGpuBufferを再初期化することは出来ません。
Finalize() してから呼び出します。
引数
[in]pMemoryMaskGpuBufferが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pDeviceデバイス
[in]pMemoryPoolMaskGpuBufferが使用するメモリプール
[in]memoryPoolOffsetMaskGpuBufferが使用するメモリプールのオフセット
[in]memoryPoolSizeMaskGpuBufferが使用するメモリプールのサイズ
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& ! MaskGpuBuffer::IsInitialized()
&& オフセット、サイズが正常
事後条件
成功したらMaskGpuBuffer::IsInitialized() = true
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).

書き込み先テクスチャを設定

引数
[in]pDeviceデバイス
[in]pTexture書き込み先のテクスチャへのポインタ
[in]infoTextureShader のTextureShaderInfo
[in]resolution書き込み先のテクスチャの解像度
nn::mii::CharModelInfo::SetDynamicTextureResolution(int facelineHeightValue,int mask) で設定した Mask テクスチャの解像度を指定してください。
[in]mipCount書き込み先のテクスチャのミップマップ数
事前条件
pDevice != NULL
&& pTexture != NULL && pTexture が初期化されている
&& info に有効なテクスチャフォーマットが設定されている
&& 0 < resolution && 0 < mipCount

詳細
書き込み先のテクスチャを設定します。
Mask テクスチャは縦と幅のサイズが等しいテクスチャです。
内部で nn::gfx::ColorTargetViewを構築します。
一つ前のColorTargetViewをFinalizeしてから処理します。
このMaskGpuBuffer渡すをTextureShaderの初期化時に設定した
TextureShaderInfoを渡してください。

各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).
