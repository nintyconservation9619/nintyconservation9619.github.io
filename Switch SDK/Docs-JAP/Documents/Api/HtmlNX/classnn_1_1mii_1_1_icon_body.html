IconBodyを初期化します。

引数
[in]pMemoryIconBodyが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pDeviceデバイス
[in]pMemoryPoolIconBodyが使用するメモリプール
[in]memoryPoolOffsetIconBodyが使用するメモリプールのオフセット
[in]memoryPoolSizeIconBodyが使用するメモリプールのサイズ
[in]pBody体リソースファイルのポインタ
[in]bodySizepBodyのサイズ
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& オフセット、サイズが正常
&& charInfoが壊れていない。
事後条件
成功したら IconBody::IsInitialized() = true

詳細
pBodyは、初期化後破棄できます。

注意
pMemory 、 pMemoryPool は、
IconBody::Finalize() まで保持する必要があります。
初期化済みのIconBodyを再初期化することは出来ません。
Finalize() してから呼び出します。
初期化時に渡したメモリプールが CpuCached な場合、
この関数の呼び出し後に、メモリプールをフラッシュする必要があります。
各種例:
MiiSimpleIconBody.cpp.
