Maskを初期化します。

詳細
初期化後のデフォルト値は以下の通りです。

注意
pMemory および pMemoryPoolは、Mask::Finalize() まで保持する必要があります。
resource は、Resource の設定条件によって
Mask::Finalize() まで保持の必要があります。
初期化済みのMaskを再初期化することは出来ません。
Finalize() してから呼び出します。
pTemporaryBufferは初期化時にのみ必要です。
初期化後に破棄しても問題ありません。
pMemoryPool には MemoryPoolProperty_CpuCached が設定されたメモリプールを渡してください。
上下反転フラグ isVerticalFlip はアプリケーションの座標系設定に応じて設定してください。
引数
[in]pMemoryMaskが使用するメモリ
[in]memorySizepMemoryのサイズ
[in]pDeviceデバイス
[in]pMemoryPoolMaskが使用するメモリプール
[in]memoryPoolOffsetMaskが使用するメモリプールのオフセット
[in]memoryPoolSizeMaskが使用するメモリプールのサイズ
[in]pTemporaryBuffer一時バッファへのポインタ
[in]resourceMaskの構築用リソース
[in]charInfo生成するMiiのCharInfo
[in]expression使用する表情フラグ
[in]isVerticalFlip上下反転フラグ
戻り値
成功するとnn::ResultSuccessを返します。
事前条件
pMemory != NULL
&& pDevice != NULL
&& pMemoryPool != NULL
&& Resource::IsInitialized()
&& ! Mask::IsInitialized()
&& pTemporaryBuffer != NULL
&& オフセット、サイズが正常
&& charInfoが壊れていない。
&& メモリ計算時と同じ ExpressionFlag を指定している。
事後条件
成功したらMask::IsInitialized() = true
各種例:
MiiCompressedTexture.cpp, MiiDatabase.cpp, MiiDynamicMask.cpp, MiiG3dHeadwear.cpp, MiiG3dSimple.cpp, MiiHeadwearExample.cpp, MiiMipMap.cpp, MiiPremultipliedAlphaIcon.cpp, MiiShaderExample.cpp, MiiSimple.cpp, MiiSimpleIcon.cpp, MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody.cpp, MiiSystemResource.cpp, MiiTransparentAdjuster.cpp (計15項目).
