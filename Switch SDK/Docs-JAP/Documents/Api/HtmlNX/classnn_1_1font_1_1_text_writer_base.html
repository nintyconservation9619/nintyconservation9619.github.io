enum nn::font::TextWriterBase::PositionFlag

SetDrawFlag() 関数で使用する文字列の配置を行う文字列描画フラグです。

列挙値
PositionFlag_HorizontalAlignLeft 

複数行文字列を左寄せする。

PositionFlag_HorizontalAlignCenter 

複数行文字列を中央寄せする。

PositionFlag_HorizontalAlignRight 

複数行文字列を右寄せする。

PositionFlag_HorizontalOriginLeft 

原点を文字列の左端に置く。

PositionFlag_HorizontalOriginCenter 

原点を文字列の水平方向中央に置く。

PositionFlag_HorizontalOriginRight 

原点を文字列の右端に置く。

PositionFlag_VerticalOriginTop 

原点を文字列の上端に置く。

PositionFlag_VerticalOriginMiddle 

原点を文字列の垂直方向中央に置く。

PositionFlag_VerticalOriginBottom 

原点を文字列の 1 行目ベースラインに置く。

PositionFlag_VerticalOriginBaseLine 

原点を文字列の下端に置く。
