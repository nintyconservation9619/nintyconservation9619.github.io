詳解

振動値に対する調整係数を表す構造体です。

詳細
振動値調整係数は、振動値の各メンバ変数に対して乗算される係数の組です。 振動値調整係数を振動値に適用することによって、振動値を変化させることができます。

関数詳解

static VibrationModulation nn::hid::VibrationModulation::Make ( )
static

デフォルトの振動値調整係数を生成します。

戻り値
デフォルトの振動値調整係数を返します。

詳細
デフォルトの振動値調整係数はすべての係数が 1.0f のため、元の振動値に何も調整を加えないという意味になります。
