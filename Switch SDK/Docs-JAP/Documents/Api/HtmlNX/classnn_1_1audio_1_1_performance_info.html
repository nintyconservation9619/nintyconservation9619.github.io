関数詳解

bool nn::audio::PerformanceInfo::SetBuffer ( const void *  frameBuffer,
size_t  bufferSize 
)

パフォーマンス情報へのバッファを設定し、初期化します。

引数
[in]frameBufferパフォーマンス情報を保持するバッファ
[in]bufferSizeパフォーマンス情報を保持するバッファのサイズ
戻り値
指定したバッファにパフォーマンス情報が含まれていなければ false を返します。
事前条件
  • frameBuffer != nullptr

詳細
nn::audio::SetPerformanceFrameBuffer() では「次にパフォーマンス情報を書き出すバッファ」を指定し、 返り値として「その時点までのパフォーマンス情報が格納されたバッファ」を返します。 この返されたバッファに含まれるパフォーマンス情報を参照してインスタンスを初期化します。 nn::audio::SetPerformanceFrameBuffer() によって、オーディオレンダラが書き出すバッファを変更したのちに、 返り値のアドレスを本 API の引数に与えてください。

各種例:
AudioPerformanceMetrics.cpp.

uint32_t nn::audio::PerformanceInfo::GetFrameIndex ( )

現在の PerformanceFrame のインデックスを取得します。

戻り値
インデックスを返します。

詳細
PerformanceFrame は各フレーム毎にインデックスを持ち、 この値はオーディオフレーム毎にインクリメントされる連続な値となっています。 MoveToNextFrame() の前後で、このインデックスが連続であることを確認することで、 PerformanceFrame の欠落の有無を確認することができます。 インデックスが uint32_t で表現できる最大値を超えた場合 0 に戻ります。

各種例:
AudioPerformanceMetrics.cpp.
