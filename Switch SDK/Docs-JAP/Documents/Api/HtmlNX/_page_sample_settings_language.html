nn::settings ライブラリの言語コードを操作するサンプルプログラムの解説です。

概要

言語コードの扱い方について説明します。

ファイル構成

本サンプルプログラムは Samples/Sources/Applications/SettingsLanguage 以下にあります。

必要な環境

特にありません。

操作方法

特にありません。

注意事項

このサンプルはグラフィックの描画を行いません。実行結果はログに出力されます。

実行手順

サンプルプログラムをビルドし、実行してください。

解説

サンプルプログラムの処理の流れは以下の通りです。

  • 言語コードのインスタンス作成
  • 言語コードの表示
  • 言語コードの比較
  • 言語コードの受け渡し
