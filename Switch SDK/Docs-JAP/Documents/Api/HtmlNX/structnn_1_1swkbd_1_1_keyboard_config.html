nn::swkbd::KeyboardConfig 構造体

ソフトウェアキーボードの設定引数を担う構造体です。 [詳解]

#include <nn/swkbd/swkbd_AppearArg.h>

公開変数類

char16_t guideText [GuideTextMaxLength+1]
 ガイド文字列です。 プレビュー欄に文字が１つも入力されていない時に表示されます。
未入力時に入力の目的や、入力例などを表示する文字列を指定してください。
また、複数行の時にも表示したい説明がある場合はこちらを指定してください。
最大 nn::swkbd::GuideTextMaxLength 文字が指定可能です。
 
char16_t headerText [HeaderTextMaxLength+1]
 ヘッダーテキストに表示する文字列です。
何を入力するべきかを簡潔に表示する文字列を指定してください。
プレビュー欄が１行の時のみ、プレビュー欄上部に表示されます。
最大 nn::swkbd::HeaderTextMaxLength 文字が指定可能です。
 
InitialCursorPos initialCursorPos
 ソフトウェアキーボード起動時の、プレビュー欄に表示されるテキストカーソルの初期位置です。
テキストカーソルを先頭に合わせるか、末尾に合わせるかを指定できます。
 
InputFormMode inputFormMode
 プレビュー欄の表示の種類です。
プレビュー欄を1行の見た目で表示するか、複数行の見た目で表示するかを設定できます。
 
uint32_t invalidCharFlag
 入力禁止文字フラグです。
入力禁止したい文字（キー）群を表すビットフラグ（nn::swkbd::InvalidChar の各値を指定）を指定してください。
 
bool isPredictionEnabled
 予測変換機能を使用するか否かを指定できます。
使用する場合は true を設定ください。
 
bool isUseBlurBackground
 ソフトウェアキーボード開始時に、現在表示中の画面をぼかすかどうかを指定できます。
ソフトウェアキーボードの開始時、起動したアプリ側の画面がキャプチャされ、 キャプチャ結果の画像がソフトウェアキーボードの背景に描画されます。
true にすると、キャプチャ結果の画像がぼかした状態で描画されます。
 
bool isUseNewLine
 改行を有効にするかどうかを指定できます。
true にすると、改行ボタンが押せるようになります。
InputFormMode が InputFormMode_MultiLine でなければ、改行は有効になりません。
 
bool isUseUtf8
 ソフトウェアキーボード終了時に返る文字列を UTF-8 にするかどうかを指定できます。
true にすると、ソフトウェアキーボード終了時に返る文字列が UTF-8 になります。
false にすると、ソフトウェアキーボード終了時に返る文字列が UTF-16 になります。
 
KeyboardMode keyboardMode
 ソフトウェアキーボードの盤面の種類です。
nn::swkbd::KeyboardMode を元に、表示したい盤面の種類を指定してください。
 
char16_t leftOptionalSymbolKey
 数字キーボードモードに追加できる左側の文字です。
nn::swkbd::KeyboardMode が KeyboardMode_Numeric の時のみ有効です。
キーを使用しない場合は 0 を指定してください。
 
char16_t okText [OkTextMaxLength+1]
 OKボタンの文字列です。
最大で nn::swkbd::OkTextMaxLength 文字までです。
 
PasswordMode passwordMode
 パスワードキーボードモードのときの入力文字隠蔽モードです。
指定していない場合、入力した文字がそのまま表示されます。
 
char16_t rightOptionalSymbolKey
 数字キーボードモードに追加できる右側の文字です。
nn::swkbd::KeyboardMode が KeyboardMode_Numeric の時のみ有効です。
キーを使用しない場合は 0 を指定してください。
 
int32_t separateTextPos [SepareteTextPosMax]
 区切り文字を挿入する位置を指定できます。
指定した位置の後に区切り文字が挿入されます。
すべてに入力文字範囲外の値を指定すると、区切り文字が挿入されなくなります。
 
char16_t subText [SubTextMaxLength+1]
 サブテキストに表示する文字列です。
ヘッダーテキストに書かれた内容を補足する文字列を指定してください。
プレビュー欄が１行の時のみ、ヘッダーテキストより下部に表示されます。
最大 nn::swkbd::SubTextMaxLength 文字が指定可能です。
 
int32_t textMaxLength
 入力テキストとして有効な最大文字数です。
プレビュー欄の右下に「入力中の文字数/最大文字数」が表示されます。
最大 nn::swkbd::TextMaxLength まで指定可能です。
 
int32_t textMinLength
 入力テキストとして有効な最小文字数です。
プレビュー欄の右下に「入力中の文字数/最大文字数」が表示されます。
最大 nn::swkbd::TextMaxLength まで指定可能です。
 

詳解

ソフトウェアキーボードの設定引数を担う構造体です。
