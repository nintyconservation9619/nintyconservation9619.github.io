詳解

Data structure describing how counter values are written to buffer memory.

This structure can be used to interpret values written in reports from nvnCommandBufferReportCounter for all counters types except for NVN_COUNTER_TYPE_ZCULL_STATS.

各種例:
StatTracker.cpp.

メンバ詳解

uint64_t NVNcounterData::timestamp

GPU timestamp indicating when the counter was written.

On the NX device, the timestamp values written in a counter report are measured in units of 1.625 nanoseconds since the device was initialized, and have a resolution of 52 nanoseconds.

On the Windows reference implementation, the timestamp values written in a counter report are measured in units of 1.0 nanoseconds since 00:00 GMT on January 1, 1970, and have a resolution of 32 nanoseconds.

各種例:
StatTracker.cpp.
