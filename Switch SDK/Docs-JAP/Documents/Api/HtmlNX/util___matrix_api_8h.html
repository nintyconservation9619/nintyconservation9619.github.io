#define NN_UTIL_MATRIX_ROW_MAJOR_4X3F_INITIALIZER   NN_UTIL_GENERAL_MATRIX_ROW_MAJOR_4X3F_INITIALIZER

nn::util::MatrixRowMajor4x3fType に対する初期化子として利用できるマクロです。

詳細
