DepthStencilState & nvn::DepthStencilState::SetDepthWriteEnable ( NVNboolean  enable)

Specify whether the depth buffer should be updated by the depth test.

If depth testing is disabled, the depth buffer is not updated, regardless of the depth write enable.

引数
enableEnable (TRUE) or disable (FALSE) writes to the depth buffer.

DepthStencilState & nvn::DepthStencilState::SetStencilFunc ( Face  faces,
StencilFunc  func 
)

Specifies the comparison function used when stencil testing is enabled.

There are separate stencil functions for front- and back-facing primitives. This function can update the value of either or both functions in the DepthStencilState object. If stencil testing is disabled, the stencil function has no effect.

引数
facesSpecifies the face(s) (front and/or back) using the new stencil function.
funcComparison function used when stencil testing is enabled.
DepthStencilState & nvn::DepthStencilState::SetStencilOp ( Face  faces,
StencilOp  fail,
StencilOp  depthFail,
StencilOp  depthPass 
)

Specifies operations performed on stored stencil values by the stencil test.

There are separate sets of stencil operations for front- and back-facing primitives. This function can update the value of either or both sets of operations in the DepthStencilState object. If stencil testing is disabled, the stencil operations have no effect.

引数
facesSpecifies the face(s) (front and/or back) using the new stencil operations.
failOperation performed on the stencil value when the stencil test fails.
depthFailOperation performed on the stencil value when the stencil test passes and the depth test fails.
depthPassOperation performed on the stencil value when the stencil and depth tests pass.
