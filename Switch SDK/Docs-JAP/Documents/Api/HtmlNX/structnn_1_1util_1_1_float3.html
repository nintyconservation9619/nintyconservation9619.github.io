nn::util::Float3 に対する初期化子として利用できるマクロです。

詳細
本マクロは初期化リストとなり、以下のように記述することで nn::util::Float3 を初期化できます。

nn::util::Float3 vector = NN_UTIL_FLOAT_3_INITIALIZER(0.f, 1.f, 2.f);
各種例:
GfxPrimitiveRenderer.cpp, MiiHeadwearExample.cpp.
