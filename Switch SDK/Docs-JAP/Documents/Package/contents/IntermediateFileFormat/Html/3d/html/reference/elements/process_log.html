<process_log>

説明

一括処理に関する記録を保存する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <process_log_array> : <process_log> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
process string
[0-9A-Za-z_]+		 一括処理の種類です。
処理の内容については中間ファイルオプティマイザのヘルプを参照してください。
argument string
-		 最適化処理の引数です。
process の説明で argument の説明が無い場合は、空文字列を指定します。

内容

最適化処理結果の概要です。
詳細については中間ファイルオプティマイザのヘルプを参考にしてください。
