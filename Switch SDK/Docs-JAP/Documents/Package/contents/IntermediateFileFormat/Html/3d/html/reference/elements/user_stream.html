<user_stream>

説明

ストリーム型ユーザーデータを格納する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <user_data> : ランタイムで取得できる、ユーザー独自のデータ（ユーザーデータ）を格納する要素です。

属性

名前 型と値 説明
size int
0 ～ MAX		 データの byte 数を指定します。
stream_index int
-1 ～ 65534		 データを格納した <stream> のインデックスです。
<stream> の type は byte を指定し、column は 16 を指定します。
size が 0 の場合は -1 を指定します。

内容

内容はありません。
