<tex_pattern_mat_anim>

説明

テクスチャーパターンアニメーションの対象となるマテリアルを指定する要素です。

親要素

  • <tex_pattern_mat_anim_array> : <tex_pattern_mat_anim> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
mat_name string
[0-9A-Za-z\-\._]+		 アニメーション対象となる <material> の名前です。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <pattern_anim_target> [1～255] : テクスチャーサンプラーのテクスチャーをアニメーションする要素です。
