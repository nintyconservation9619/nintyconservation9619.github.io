<bone_array>

説明

<bone> の配列要素です。

親要素

  • <skeleton> : 座標系の定義に利用する <bone> を取り纏める要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素はルートボーンから深さ優先でソートし、兄弟は<bone> name の文字コード順でソートします。
ソート順はワールド行列計算等のランタイム動作に影響します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <bone> [1～65535] : スキニング等に利用する座標系を定義するボーンの要素です。
