<interleave_array>

説明

<interleave> の配列要素です。

この要素が省略された場合、すべての頂点属性を一つの頂点バッファでインターリーブします。
この要素は DCC ツールから出力しません。

親要素

  • <shading_model> : シェーディングモデルを定義する要素です。

属性

名前 型と値 説明
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。

内容

子要素はソートせず、シェーダーアノテーションによるユーザーの指定順を保持します。
ただし、id_set="" を指定した <interleave> は、必ず最後に配置します。
id_set="" を指定した <interleave> がアノテーションで定義されていない場合は、シェーダーコンバータが id_set="" binarize="true" を指定した <interleave> を末尾に追加します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <interleave> [1～65535] : 頂点属性のインターリーブを定義する要素です。
