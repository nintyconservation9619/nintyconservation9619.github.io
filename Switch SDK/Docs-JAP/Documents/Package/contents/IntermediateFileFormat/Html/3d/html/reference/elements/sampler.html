<sampler>

説明

テクスチャーサンプラーを定義する要素です。

親要素

  • <sampler_array> : <sampler> の配列要素です。

属性

名前 型と値 説明 DCC
sampler_index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
name string
[0-9A-Za-z_]+		 実機側でテクスチャーサンプラーの関連付けに利用する名前です。
参照元の <pattern_anim_target> name と対になっています。
DCC ツールから出力する際に、以下の命名規則で名前を指定します。
_a0～_a254: albedo
_o0～_o254: opacity
_e0～_e254: emission
_n0～_n254: normal
_t0～_t254: tangent
_s0～_s254: specular
_r0～_r254: reflection
設定
hint string
[0-9A-Za-z_]*		 ツール側でテクスチャーサンプラーの関連付けに利用するヒント情報です。
参照元の <original_texsrt> <original_texsrt_anim> <pattern_anim_target> hint と対になっています。
DCC ツールから出力する際に、以下の命名規則でヒントを指定します。
albedo0～albedo254
opacity0～opacity254
emission0～emission254
normal0～normal254
tangent0～tangent254
specular0～specular254
reflection0～reflection254
設定
tex_name string
[0-9A-Za-z\-\._]*		 サンプラーで用いる <texture> の名前です。
ftx ファイル名から拡張子を取り除いた名前を指定します。
英数字と - . _ からなる、1 文字以上の文字列を指定します。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <wrap> [1] : テクスチャーサンプル時のラップを指定する要素です。
  • <filter> [1] : テクスチャーサンプル時のフィルターを指定する要素です。
  • <lod> [1] : テクスチャーサンプル時の LOD を指定する要素です。
