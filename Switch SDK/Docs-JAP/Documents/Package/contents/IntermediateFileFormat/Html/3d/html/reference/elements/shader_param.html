<shader_param>

説明

シェーダーパラメータを設定する要素です。

シェーダーパラメータの順序はシェーダー割り当て時に決定し、UBO への配置順と一致します。
この要素は DCC ツールから出力しません。

親要素

  • <shader_param_array> : <shader_param> の配列要素です。

属性

名前 型と値 説明

0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。

[0-9A-Za-z_]+		 設定する <uniform_var> の識別子です。

<original_color> <original_texsrt> を元にしなかった場合は、空文字列を指定します。

depend string
[0-9A-Za-z_]*		 シェーダーパラメータの変換で依存するシェーダーパラメータの識別子です。

内容

type に合わせて、パラメータ値を定義します。
bool 値は 0、1 で指定します。
