<nw4f_3dif>

説明

3D 中間ファイルフォーマットのルート要素です。

詳細については テキスト中間ファイルを参照してください。

親要素

  • ルート要素です。

属性

名前 型と値 説明 DCC
version enum
4.0.0		 中間ファイルのバージョン番号です。
 設定

内容

ファイルの拡張子に合わせた要素が一つ出現します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <file_info> [0～1] : 中間ファイルの作成や変更に関する記録を保存する要素です。

  • 以下の要素のいずれかが出現します。[] 内の数値は要素の出現回数です。

    • <model> [1] : モデルファイル (.fmda, .fmdb) で指定する要素です。
    • <material> [1] : マテリアルを定義する要素です。
    • <texture> [1] : テクスチャーファイル (.ftxa, .ftxb) で指定する要素です。
    • <skeletal_anim> [1] : スケルタルアニメーションファイル (.fska, .fskb) で指定する要素です。
    • <shader_param_anim> [1] : シェーダーパラメータアニメーションファイル (.fspa, .fspb)、カラーアニメーションファイル (.fcla .fclb)、テクスチャー SRT アニメーションファイル (.ftpa .ftpb) で指定する要素です。
    • <tex_pattern_anim> [1] : テクスチャーパターンアニメーションファイル (.ftpa, .ftpb) で指定する要素です。
    • <bone_visibility_anim> [1] : ボーンビジビリティアニメーションファイル (.fvba, .fvbb) で指定する要素です。
    • <mat_visibility_anim> [1] : マテリアルビジビリティアニメーションファイル (.fvma, .fvmb) で指定する要素です。
    • <shape_anim> [1] : シェイプアニメーションファイル (.fsha, .fshb) で指定する要素です。
    • <scene_anim> [1] : シーンアニメーションファイル (.fsna, .fsnb) で指定する要素です。
    • <shader_config> [1] : シェーダー設定ファイル (.fsca, .fscb) で指定する要素です。
    • <shader_definition> [1] : シェーダー定義ファイル (.fsda, .fsdb) で指定する要素です。
    • <shader_variation> [1] : シェーダーバリエーションファイル (.fsva, .fsvb) で指定する要素です。
    • <material_anim> [1] : マテリアルアニメーションファイル (.fmaa, .fmab) で指定する要素です。
    • <combiner_shader> [1]
