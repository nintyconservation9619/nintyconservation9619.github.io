<input>

説明

頂点バッファ内に頂点属性が含まれる際の、順序や量子化型を指定します。

親要素

  • <input_array> : <input> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
attrib_index int
0 ～ 254		 シェーダーに入力する <vtx_attrib> のインデックスです。
設定

内容

内容はありません。
