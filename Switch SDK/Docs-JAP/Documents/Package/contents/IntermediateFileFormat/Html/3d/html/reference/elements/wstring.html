<wstring>

説明

Unicode 文字列を格納する要素です。

親要素

  • <user_wstring> : Unicode 文字列ユーザーデータを格納する要素です。

属性

属性はありません。

内容

文字コードは UTF-8 で、文字セットは UCS2、改行は CRLF です。
& < > を &amp; &lt; &gt; にエスケープします。

出力例
