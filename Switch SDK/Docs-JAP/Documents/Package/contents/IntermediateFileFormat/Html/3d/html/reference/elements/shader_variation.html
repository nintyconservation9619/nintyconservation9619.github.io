<shader_variation>

説明

シェーダーバリエーションファイル (.fsva, .fsvb) で指定する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <nw4f_3dif> : 3D 中間ファイルフォーマットのルート要素です。

属性

名前 型と値 説明
version enum
4.0.0

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <shader_variation_info> [1] : <shader_variation> に関する情報の要素です。
  • <target_shader_array> [1] : <target_shader> の配列要素です。
  • <stream_array> [0～1] : <stream> の配列要素です。
  • <tool_data> [0～1] : NintendoWare のツール用データを格納する要素です。
  • <user_tool_data> [0～1] : ユーザー独自のツール用データを格納する要素です。
