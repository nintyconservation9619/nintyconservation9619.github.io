<stream>

説明

頂点属性や頂点インデックスのデータ列を格納する要素です。

親要素

  • <stream_array> : <stream> の配列要素です。

属性

名前 型と値 説明 DCC
stream_index int
0 ～ MAX		 配列内におけるインデックスです。
参照元の <vtx_attrib> <mesh> <texture_info> <original_image> <original_hermite> <original_linear> <original_step> <shader_src> <user_int> <user_float> <user_string> <user_wstring> <hermite_curve> <linear_curve> <step_curve> stream_index と対になっています。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
type enum
float
int
byte
string
wstring		 データ列の値の型を指定します。
float データ列の値が実数です。
int データ列の値が符号付き 32bit 整数です。
byte データ列の値が符号無し 8bit 整数（16進数表記）です。
string ASCII 文字列データです。
文字コードは UTF-8 で、文字セットは ASCII、改行は CRLF です。
& < > を &amp; &lt; &gt; にエスケープします。
wstring Unicode 文字列データです。
文字コードは UTF-8 で、文字セットは UCS2、改行は CRLF です。
& < > を &amp; &lt; &gt; にエスケープします。

 設定
count int
1 ～ MAX		 データ列の値の個数を指定します。
type が string、wstring の場合は、常に 1 です。
設定
column int
0 ～ 16		 中間ファイルで値をレイアウトする際に利用する列数です。
データ列の内容に合わせて、読みやすい列数を指定します。
type が string、wstring の場合は、常に 0 です。
設定

内容

一行につき column 個の値を、size 個定義します。
