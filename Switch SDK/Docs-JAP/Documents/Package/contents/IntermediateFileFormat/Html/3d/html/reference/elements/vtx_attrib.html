<vtx_attrib>

説明

頂点属性を定義する要素です。

親要素

  • <vtx_attrib_array> : <vtx_attrib> の配列要素です。

属性

名前 型と値 説明 DCC
attrib_index int
0 ～ 254		 配列内におけるインデックスです。
参照元の <input> <attrib_assign> attrib_index と対になっています。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
name string
[0-9A-Za-z_]+		 実機側で頂点属性の関連付けに利用する名前です。
参照元の <attrib_assign> <target_attrib> attrib_name と対になっています。
DCC ツールから出力する際に、以下の命名規則で名前を指定します。
_p0 ～ _p254: position
_n0 ～ _n254: normal
_t0 ～ _t254: tangent
_b0 ～ _b254: binormal
_c0 ～ _c254: color
_u0 ～ _u254: uv
_i0 ～ _i31: blend index
_w0 ～ _w31: blend weight
設定

固定
count int
1 ～ MAX		 頂点の数を指定します。
 設定
stream_index int
0 ～ MAX		 頂点属性の実データを格納する <stream> のインデックスです。
従法線は UV の V に対して逆方向になります。
接線と従法線の W 値は、算出元の UV が表向きであれば 1 を、裏向きであれば -1 を指定します。
接線と従法線、法線と従法線はそれぞれ直交し、長さは 1 です。
テクスチャー座標はテクスチャー画像の左上を 0 とします。
blendweight が 0 である場合の blendindex は、頂点で最大の blendweight を持つ blendindex を指定します。
