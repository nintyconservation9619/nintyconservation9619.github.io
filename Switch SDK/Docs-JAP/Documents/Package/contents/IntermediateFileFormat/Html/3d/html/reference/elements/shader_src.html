<shader_src>

説明

シェーダー定義を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader_src_array> : <shader_src> の配列要素です。

属性

名前 型と値 説明
src_index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
include_path string
[^*?\\|"'<> ]*		 シェーダーソースのインクルードパスです。
path string
[^*?\\|"'<> ]+		 シェーダーソースのインクルードパスからの相対パスです。
stream_index int
0 ～ 65534		 シェーダーソースを格納する <stream> のインデックスです。
<stream> の type は wstring を指定します。
<stream> の count は 1 を指定します。
<stream> の column は 0 を指定します。
文字コードは UTF-8 で、文字セットは UCS2、改行は CRLF です。
& < > を &amp; &lt; &gt; にエスケープします

内容

内容はありません。
