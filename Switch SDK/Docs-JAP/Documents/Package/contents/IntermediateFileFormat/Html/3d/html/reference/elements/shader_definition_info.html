<shader_definition_info>

説明

<shader_definition> に関する情報の要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader_definition> : シェーダー定義ファイル (.fsda, .fsdb) で指定する要素です。

属性

名前 型と値 説明
code_page int
0 ～ 65535		 シェーダーソースコードのコードページを指定します。
config_path string
[^*?\\|"'<> ]+		 シェーダー定義ファイルの生成元であるシェーダー設定ファイルのパスです。
シェーダーコンバータで設定したルートパスからの相対パスか、ファイル名です。

内容

内容はありません。
