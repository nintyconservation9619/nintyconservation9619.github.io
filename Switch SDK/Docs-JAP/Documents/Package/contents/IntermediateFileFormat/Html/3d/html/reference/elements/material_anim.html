<material_anim>

説明

マテリアルアニメーションファイル (.fmaa, .fmab) で指定する要素です。

マテリアルアニメーションの名前はファイル名から拡張子を取り除いた部分です。

親要素

  • <nw4f_3dif> : 3D 中間ファイルフォーマットのルート要素です。

属性

名前 型と値 説明 DCC
version enum
4.0.0		 -

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <process_log_array> [0～1] : <process_log> の配列要素です。
  • <material_anim_info> [1] : <material_anim> に関する情報の要素です。
  • <tex_pattern_array> [0～1] : <tex_pattern> の配列要素です。
  • <per_material_anim_array> [0～1] : <per_material_anim> の配列要素です。
  • <original_per_material_anim_array> [0～1] : <original_per_material_anim> の配列要素です。
  • <user_data_array> [0～1] : <user_data> の配列要素です。
  • <stream_array> [0～1] : <stream> の配列要素です。
  • <comment> [0～1] : 編集用コメント等のメタ情報を格納する要素です。
  • <tool_data> [0～1] : NintendoWare のツール用データを格納する要素です。
  • <user_tool_data> [0～1] : ユーザー独自のツール用データを格納する要素です。
