<material_info>

説明

<material> に関する情報の要素です。

親要素

  • <material> : マテリアルを定義する要素です。

属性

名前 型と値 説明 DCC
visibility bool
true false		 マテリアルに関連付けられた <shape> の可視状態を指定します。
DCC ツールからの出力時は、true を指定します。
固定
mesh_adjacency bool
true false		 関連するメッシュのバイナリー変換時に隣接トライアングルの情報を出力する場合に、true を指定します。
DCC ツールからの出力時は、常に false を指定します。
固定
compress_enable bool
true false		 中間ファイルオプティマイザのマテリアル圧縮において圧縮可能なマテリアルとする場合に、true を指定します。
設定

内容

内容はありません。
