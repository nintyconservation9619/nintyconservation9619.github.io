<bone_vis_bone_anim>

説明

<bone> の visibility をアニメーションする要素です。

ボーンが描画に利用されているかどうかは、以下の条件で判定します。
rigid_body || (matrix_index[0] != -1) || (matrix_index[1] != -1)

親要素

  • <bone_vis_bone_anim_array> : <bone_vis_bone_anim> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
bone_name string
[0-9A-Za-z\-\._]+		 アニメーション対象 <bone> の名前です。
設定
parent_name string
[0-9A-Za-z\-\._]*		 <bone> parent_name と同じ定義です。
 設定
rigid_body bool
true false		 <bone> rigid_body と同じ定義です。
 設定
matrix_index int[2]
-1 ～ 1364		 <bone> matrix_index と同じ定義です。
 設定
binarize_visibility bool
true false		 ボーンのアニメーションをバイナリーファイルに出力する場合に、true を指定します。
DCC ツールからの出力時は、true を指定します。
固定
compress_enable bool
true false		 <bone> compress_enable と同じ定義です。
 設定
base_value float
MIN ～ MAX		 アニメーションの初期値を指定します。
アニメーションが付いている場合には、0 フレームの値を指定します。
設定

内容

アニメーションが無く、base_value を固定値として対象に設定する場合は、内容がありません。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <step_curve> [0～1] : キーの間を補間しないステップカーブの要素です。
