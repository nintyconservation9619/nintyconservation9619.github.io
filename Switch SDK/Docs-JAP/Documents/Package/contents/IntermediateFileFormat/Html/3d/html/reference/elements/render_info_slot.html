<render_info_slot>

説明

シェーディングモデルが受け入れる描画情報を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <render_info_slot_array> : <render_info_slot> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
name string
[0-9A-Za-z_]+		 描画情報の名前です。
type enum
int
float
string		 描画情報の値の型を指定します。
count int
1 ～ 65535		 描画情報の値の最大個数です。
optional bool
true false		 描画情報の値を最大個数分指定しなくてもよい場合に、true を指定します。
choice string
-		 描画情報の値の範囲です。
描画情報の型により、記法が異なります。
単一の値を指定する場合は、値をそのまま指定します。
種類記法描画情報の値
数値
range		"[min, max]"
min max は数値です。		数値
文字列
list		"add, sub, mul, div"
英数字と _ からなる列挙値です。
"add:加算, sub:減算" といった表記により UI のラベルを指定できます。		文字列のいずれか
数値
list		"-1, 0, 1, 2"
数値からなる列挙値です。
"-1:加算, 0:減算" といった表記により UI のラベルを指定できます。		数値
default string
[^"']*		 <render_info> を作成する際のデフォルト値です。
count が 2 以上の場合は、スペース区切りで複数の値を指定します。

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <ui_label> [0～1] : 変数を編集する UI のラベルを定義する要素です。
  • <ui_comment> [0～1] : 変数を編集する UI のコメントを定義する要素です。
  • <ui_group> [0～1] : 変数を編集する UI のグループ分けを定義する要素です。
  • <ui_order> [0～1] : 変数を編集する UI の並び順優先度を定義する要素です。
  • <ui_item> [0～1] : 変数を編集する UI の型を切り替える要素です。
  • <ui_visible> [0～1] : 変数を編集する UI の表示を切り替える要素です。
