<target_attrib>

説明

キーシェイプの頂点属性を定義する要素です。
シェイプアニメーションのアニメーション対象となる頂点属性と、その順序を定義します。

親要素

  • <target_attrib_array> : <target_attrib> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
attrib_name string
[0-9A-Za-z_]+		 キーシェイプが使用する <vtx_attrib> の名前です。
設定

内容

内容はありません。
