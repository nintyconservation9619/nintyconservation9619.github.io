<sampler_array>

説明

<sampler> の配列要素です。

親要素

  • <material> : マテリアルを定義する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 255		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素はソートせず、ユーザーの指定順を保持します。
DCC ツールからの出力時は <sampler> hint 定義順で出力します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <sampler> [1～255] : テクスチャーサンプラーを定義する要素です。
