<camera_anim>

説明

ユーザー定義カメラをアニメーションする要素です。

親要素

  • <camera_anim_array> : <camera_anim> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
camera_name string
[0-9A-Za-z\-\._]+		 アニメーション対象であるユーザー定義カメラの名前です。
設定
frame_count int
0 ～ 16777215		 アニメーションのフレーム数です。
設定
loop bool
true false		 アニメーションをループ再生する場合に、true を指定します。
 設定
rotate_mode enum
aim
euler_zxy		 ユーザー定義カメラの回転モードを指定します。
aim 注視点 (aim) と捻り (twist) で回転します。
euler_zxy ZXY 軸回転 (rotate) で回転します。

 設定
projection_mode enum
ortho
persp		 ユーザー定義カメラの投影モードを指定します。
ortho aspect near far ortho_height を用いて、正射影で投影します。
persp aspect near far persp_fovy を用いて、透視射影で投影します。

 設定

内容

target が position_x、position_y、position_z、aspect、near、far の <camera_anim_target> を常に持ちます。
rotate_mode が aim の場合は、target が aim_x、aim_y、aim_z、twist の <camera_anim_target> を持ちます。
rotate_mode が euler の場合は、target が rotate_x、rotate_y、rotate_z の <camera_anim_target> を持ちます。
projection_mode が ortho の場合は、target が ortho_height の <camera_anim_target> を持ちます。
projection_mode が persp の場合は、target が persp_fovy の <camera_anim_target> を持ちます。
<camera_anim_target> は target の定義順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <camera_anim_target> [0～15] : ユーザー定義カメラの各パラメータをアニメーションする要素です。
  • <user_data_array> [0～1] : <user_data> の配列要素です。
  • <comment> [0～1] : 編集用コメント等のメタ情報を格納する要素です。
  • <tool_data> [0～1] : NintendoWare のツール用データを格納する要素です。
  • <user_tool_data> [0～1] : ユーザー独自のツール用データを格納する要素です。
