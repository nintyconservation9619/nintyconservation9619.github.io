<filter>

説明

テクスチャーサンプル時のフィルターを指定する要素です。

親要素

  • <sampler> : テクスチャーサンプラーを定義する要素です。

属性

名前 型と値 説明 DCC
mag enum
point
linear		 テクスチャー拡大時のフィルターを指定します。
DCC ツールからの出力時は、常に linear を指定します。
point ポイントサンプリングします。
linear サンプリングする際にリニアフィルターを適用します。

 固定
min enum
point
linear		 テクスチャー縮小時のフィルターを指定します。
DCC ツールからの出力時は、常に linear を指定します。
point ポイントサンプリングします。
linear サンプリングする際にリニアフィルターを適用します。

 固定
mip enum
none
point
linear		 テクスチャーミップマップ間のフィルターを指定します。
DCC ツールからの出力時は、デフォルトでは常に linear を指定します。
DCC ツールのコンフィグを設定することにより、常に point を指定することもできます。
none ミップマップを適用しません。
point ミップマップを適用します。
linear ミップマップを適用し、ミップマップ間をリニア補間します。

 固定
max_aniso enum
aniso_1
aniso_2
aniso_4
aniso_8
aniso_16		 アニソトロピックフィルターの最大値を指定します。
DCC ツールからの出力時は、常に aniso_1 を指定します。
固定

内容

内容はありません。
