<macro>

説明

シェーダーのマクロを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <macro_array> : <macro> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
name string
[0-9A-Za-z_]+		 マクロの名前です。
value string
[^"']*		 マクロの値です。

内容

内容はありません。
