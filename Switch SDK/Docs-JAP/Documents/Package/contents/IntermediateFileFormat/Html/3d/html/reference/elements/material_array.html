<material_array>

説明

<material> の配列要素です。

親要素

  • <model> : モデルファイル (.fmda, .fmdb) で指定する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素は <material> name の文字コード順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <material> [1～65535] : マテリアルを定義する要素です。
