<variation>

説明

シェーダーのオプション変数で指定した値の範囲を制限する要素です。
要素を省略した場合は、デフォルト値のみを利用するバリエーションを定義します。

この要素は DCC ツールから出力しません。

親要素

  • <variation_array> : <variation> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
id string
[0-9A-Za-z_]+		 バリエーションを指定するオプション変数の識別子です。
choice string
[^"']+		 オプション変数の値の範囲に対する制限です。
シェーダーのオプション変数で指定した choice を上書きします。
変数の型により、記法が異なります。
単一の値を指定する場合は、値をそのまま指定します。
* を指定するとシェーダーのソースに基づいて自動的に決定します。
種類記法<shader_option>
value
bool"bool"0 または 1
整数
range		"[min, max]"
min max は整数です。		整数
列挙
list		"add, sub, mul, div"
英数字と _ からなる列挙値です		列挙値のいずれか
default string
[0-9A-Za-z+\-_]*		 オプション変数のデフォルト値です。
<option_var> choice の記法で単一の値を指定します。
default が空の場合は、ソースの記載に基づいて自動でデフォルトを決定します。
choice で指定した範囲にソースのデフォルトが含まれていない場合、指定した範囲の最初の選択肢がデフォルトとして扱われます。
branch bool
true false		 オプション変数を静的分岐として扱うかどうかを指定します。
true なら静的分岐、false ならバリエーションです。

内容

内容はありません。
