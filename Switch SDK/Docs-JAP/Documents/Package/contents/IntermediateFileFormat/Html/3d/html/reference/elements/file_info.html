<file_info>

説明

中間ファイルの作成や変更に関する記録を保存する要素です。

親要素

  • <nw4f_3dif> : 3D 中間ファイルフォーマットのルート要素です。

属性

属性はありません。

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <create> [1] : 中間ファイルの作成に関する記録を保存する要素です。
  • <modify> [0～1] : 中間ファイルの変更に関する記録を保存する要素です。

   1│<nw4f_3dif version="4.0.0">

   4│        src_path="C:/src_dir/src_file.mb"
   5│    />
   6│    <modify tool_name="3dIntermediateFileUpdater" tool_version="3.11.0" />
   7│</file_info>
