<vertex_shape_anim>

説明

<shape> の形状をアニメーションする要素です。

親要素

  • <vertex_shape_anim_array> : <vertex_shape_anim> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
shape_name string
[0-9A-Za-z\-\._]+		 アニメーション対象 <shape> の名前です。
設定
base_name string
[0-9A-Za-z\-\._]+		 ベース形状となる <key_shape> の名前です。
ベース形状の重みは 1 - (<shape_anim_target> の重み合計) になります。
設定

内容

DCC ツールからの出力時は、子要素は DCC ツール上のターゲット形状の順で出力します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <shape_anim_target> [1～255] : <key_shape> の重みをアニメーションする要素です。
