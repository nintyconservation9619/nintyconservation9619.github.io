<option>

説明

シェーダープログラムに設定する、シェーダーのオプションを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <option_array> : <option> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
id string
[0-9A-Za-z_]+		 設定する <option_var> の識別子です。
choice string
[^"']+		 オプションの値を <variation> choice と同じ記法で指定します。

内容

内容はありません。
