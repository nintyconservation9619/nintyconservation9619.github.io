<force_include_file>

説明

強制的にインクルードするシェーダーソースファイルを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <force_include_file_array> : <force_include_file> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
path string
[^*?\\|"'<> ]+		 シェーダーソースの先頭に強制的にインクルードするシェーダーソースファイルのパスです。
インクルードパスからの相対パスで指定します。

内容

内容はありません。
