<material>

説明

マテリアルを定義する要素です。

親要素

  • <material_array> : <material> の配列要素です。
  • <combiner_shader>
  • <nw4f_3dif> : 3D 中間ファイルフォーマットのルート要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
name string
[0-9A-Za-z\-\._]+		 マテリアルの名前です。
参照元の <original_material> <shader_param_mat_anim> <original_material_anim> <tex_pattern_mat_anim> <mat_vis_mat_anim> mat_name と対になっています。
設定
version enum
4.0.0		 この属性は省略できます。
 -

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <process_log_array> [0～1] : <process_log> の配列要素です。
  • <material_info> [1] : <material> に関する情報の要素です。
  • <render_state> [1] : 深度テスト、アルファテスト、ブレンドといったレンダーステートを定義する要素です。
  • <sampler_array> [0～1] : <sampler> の配列要素です。
  • <shader_assign> [0～1] : マテリアルに割り当てるシェーダーを指定します。
  • <user_data_array> [0～1] : <user_data> の配列要素です。
  • <stream_array> [0～1] : <stream> の配列要素です。
  • <comment> [0～1] : 編集用コメント等のメタ情報を格納する要素です。
  • <tool_data> [0～1] : NintendoWare のツール用データを格納する要素です。
  • <user_tool_data> [0～1] : ユーザー独自のツール用データを格納する要素です。
