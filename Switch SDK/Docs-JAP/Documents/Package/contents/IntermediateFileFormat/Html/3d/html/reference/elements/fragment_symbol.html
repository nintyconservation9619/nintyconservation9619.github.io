<fragment_symbol>

説明

フラグメントシェーダーにおける変数のシンボルを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <option_var> : シェーダーのオプション変数を定義する要素です。
  • <sampler_var> : シェーダーのサンプラー変数を定義する要素です。
  • <block_var> : シェーダーのユニフォームブロック変数を定義する要素です。
  • <shader_storage_block_var> : シェーダーのシェーダーストレージブロック変数を定義する要素です。
  • <uniform_var> : シェーダーのユニフォーム変数を定義する要素です。
  • <buffer_var> : シェーダーのバッファー変数を定義する要素です。

属性

名前 型と値 説明
name string
[0-9A-Za-z\.\[\]_]+		 フラグメントシェーダーにおける変数の名前です。

内容

内容はありません。
