<shading_model>

説明

シェーディングモデルを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shading_model_array> : <shading_model> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
name string
[0-9A-Za-z_]+		 シェーディングモデルの名前です。
生成元の <shader> name と対になっています。
また、参照元の <shader_assign> shading_model と対になっています。
material_shader bool
true false		 マテリアル用のシェーディングモデルである場合に、true を指定します。
revision int
0 ～ MAX		 シェーディングモデルのリビジョン番号です。

内容

<vertex_stage>、<geometry_stage> 、<fragment_stage> 、<compute_stage> のいずれかが必ず存在します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <vertex_stage> [0～1] : シェーダーの頂点シェーダーステージを定義する要素です。
  • <geometry_stage> [0～1] : シェーダーのジオメトリシェーダーステージを定義する要素です。
  • <fragment_stage> [0～1] : シェーダーのフラグメントシェーダーステージを定義する要素です。
  • <compute_stage> [0～1] : シェーダーの演算シェーダーステージを定義する要素です。
  • <macro_array> [0～1] : <macro> の配列要素です。
  • <option_var_array> [0～1] : <option_var> の配列要素です。
  • <attrib_var_array> [0～1] : <attrib_var> の配列要素です。
  • <sampler_var_array> [0～1] : <sampler_var> の配列要素です。
  • <block_var_array> [0～1] : <block_var> の配列要素です。
  • <shader_storage_block_var_array> [0～1] : <shader_storage_block_var> の配列要素です。
  • <render_info_slot_array> [0～1] : <render_info_slot> の配列要素です。
  • <textblock_array> [0～1] : <textblock> の配列要素です。
  • <interleave_array> [0～1] : <interleave> の配列要素です。
  • <group_array> [0～1] : <group> の配列要素です。
  • <page_array> [0～1] : <page> の配列要素です。
  • <streamout> [0～1] : ストリームアウトするバリイング変数を定義する要素です。
