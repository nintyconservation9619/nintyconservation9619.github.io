<ui_max>

説明

変数を編集する UI の最大値を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <uniform_var> : シェーダーのユニフォーム変数を定義する要素です。

属性

名前 型と値 説明
value float
MIN ～ MAX		 変数を編集する UI の最大値です。

内容

内容はありません。
