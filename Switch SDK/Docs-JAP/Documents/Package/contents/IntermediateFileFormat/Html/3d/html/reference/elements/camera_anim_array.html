<camera_anim_array>

説明

<camera_anim> の配列要素です。

親要素

  • <scene_anim> : シーンアニメーションファイル (.fsna, .fsnb) で指定する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素はソートせず、ユーザーの指定順を保持します。
DCC ツールからの出力時は <camera_anim> name の文字コード順で出力します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <camera_anim> [1～65535] : ユーザー定義カメラをアニメーションする要素です。
