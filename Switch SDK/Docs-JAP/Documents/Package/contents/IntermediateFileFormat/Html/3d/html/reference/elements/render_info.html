<render_info>

説明

描画情報を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <render_info_array> : <render_info> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
name string
[0-9A-Za-z_]+		 描画情報の名前です。
type enum
int
float
string		 描画情報の値の型を指定します。
count int
0 ～ MAX		 描画情報の値の個数を指定します。

内容

描画情報の値を、英数字からなる文字列で指定します。
スペース、タブ、改行で区切ることにより、複数の値を指定します。
