<alpha_test>

説明

アルファテストを定義する要素です。

親要素

  • <render_state> : 深度テスト、アルファテスト、ブレンドといったレンダーステートを定義する要素です。

属性

名前 型と値 説明 DCC
enable bool
true false		 アルファテストを有効にする場合に、true を指定します。
DCC ツールからの出力時は、常に false を指定します。
<render_state> mode が opaque であれば false を、mask であれば true を指定します。
固定
func enum
never
less
equal
lequal
greater
nequal
gequal
always		 アルファテストの関数を指定します。
DCC ツールからの出力時は、常に gequal を指定します。
never 常にテストをパスしません。
less アルファ値が小さい場合に、テストをパスします。
equal アルファ値が同じ場合に、テストをパスします。
lequal アルファ値が同じか小さい場合に、テストをパスします。
greater アルファ値が大きい場合に、テストをパスします。
nequal アルファ値が同じでない場合に、テストをパスします。
gequal アルファ値が同じか大きい場合に、テストをパスします。
always 常にテストをパスします。

 固定
value float
MIN ～ MAX		 アルファテストの比較値を指定します。
DCC ツールからの出力時は、常に 0.5 を指定します。
固定

内容

内容はありません。
