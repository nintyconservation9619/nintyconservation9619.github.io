<shader_variation_info>

説明

<shader_variation> に関する情報の要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader_variation> : シェーダーバリエーションファイル (.fsva, .fsvb) で指定する要素です。

属性

名前 型と値 説明
shader_archive string
[0-9A-Za-z_]+		 シェーダーアーカイブの名前です。
シェーダー定義ファイルのファイル名から拡張子を取り除いた名前を指定します。

内容

内容はありません。
