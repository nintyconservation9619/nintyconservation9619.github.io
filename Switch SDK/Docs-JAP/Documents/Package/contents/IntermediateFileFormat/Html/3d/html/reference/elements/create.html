<create>

説明

中間ファイルの作成に関する記録を保存する要素です。

親要素

  • <file_info> : 中間ファイルの作成や変更に関する記録を保存する要素です。

属性

名前 型と値 説明 DCC
tool_name string
-		 中間ファイルを作成したツールの名前です。
 設定
tool_version string
-		 中間ファイルを作成したツールのバージョンです。
 設定
src_path string
[^*?\\|"'<> ]*		 中間ファイルを作成した際に、元ファイルがあった場合のフルパスです。
 設定

内容

内容はありません。
