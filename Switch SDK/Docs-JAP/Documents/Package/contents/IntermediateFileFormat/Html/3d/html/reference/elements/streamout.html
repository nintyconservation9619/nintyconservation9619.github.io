<streamout>

説明

ストリームアウトするバリイング変数を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shading_model> : シェーディングモデルを定義する要素です。

属性

名前 型と値 説明
varying string
[0-9A-Za-z _]*		 空白区切りでバリイング変数を定義します。

内容

内容はありません。
