<user_wstring>

説明

Unicode 文字列ユーザーデータを格納する要素です。

親要素

  • <user_data> : ランタイムで取得できる、ユーザー独自のデータ（ユーザーデータ）を格納する要素です。

属性

名前 型と値 説明 DCC
count int
0 ～ MAX		 データの個数を指定します。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <wstring> [0～65535] : Unicode 文字列を格納する要素です。
