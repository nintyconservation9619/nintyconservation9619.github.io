<wrap>

説明

テクスチャーサンプル時のラップを指定する要素です。

親要素

  • <sampler> : テクスチャーサンプラーを定義する要素です。

属性

名前 型と値 説明 DCC
u enum
repeat
mirror
clamp
mirror_once		 U 方向のラップを指定します。
中間ファイルをマージする際、DCC ツールの設定は無視します。
repeat テクスチャーを繰り返します。
mirror テクスチャーを反転しながら繰り返します。
clamp テクスチャー端の画素を引き伸ばします。
mirror_once テクスチャーを座標のマイナス方向に一度だけ繰り返し、以降はテクスチャー端の画素を引き伸ばします。

 設定
v enum
repeat
mirror
clamp
mirror_once		 V 方向のラップを指定します。
中間ファイルをマージする際、DCC ツールの設定は無視します。
repeat テクスチャーを繰り返します。
mirror テクスチャーを反転しながら繰り返します。

mirror_once テクスチャーを座標のマイナス方向に一度だけ繰り返し、以降はテクスチャー端の画素を引き伸ばします。

 設定
w enum
repeat
mirror
clamp
mirror_once		 W 方向のラップを指定します。
DCC ツールからの出力時は、常に clamp を指定します。
repeat テクスチャーを繰り返します。
mirror テクスチャーを反転しながら繰り返します。
clamp テクスチャー端の画素を引き伸ばします。
mirror_once テクスチャーを座標のマイナス方向に一度だけ繰り返し、以降はテクスチャー端の画素を引き伸ばします。
