<lod_offset>

説明

各 LOD レベルでの頂点インデックスのオフセット値です。

この要素は DCC ツールから出力しません。

親要素

  • <vertex> : 頂点データを定義する要素です。

属性

名前 型と値 説明
count int
1 ～ 255		 LOD のベースレベルを除いた段数を指定します。

内容

count 個の整数値を指定します。
