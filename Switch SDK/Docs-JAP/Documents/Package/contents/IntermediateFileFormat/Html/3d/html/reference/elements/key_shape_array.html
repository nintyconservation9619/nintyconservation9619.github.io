<key_shape_array>

説明

<key_shape> の配列要素です。

<shape> がシェイプアニメーションに対応していない場合は、この要素を省略します。

親要素

  • <shape> : モデルの形状を定義する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 255		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

DCC ツールからの出力時は、<key_shape> index="0" にベース形状、
index="1" 以降に DCC ツール上の順番でターゲット形状が出力されます。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <key_shape> [1～255] : シェイプアニメーション用のキーシェイプを定義する要素です。
