<attrib_var>

説明

シェーダーの頂点属性変数を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <attrib_var_array> : <attrib_var> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
id string
[0-9A-Za-z_]+		 頂点属性変数の識別子です。
参照元の <attrib_assign> id と対になっています。
hint string
[0-9A-Za-z_]*		 頂点属性変数の関連付けに利用するヒント情報です。
DCC ツールからの出力に合わせてヒントをつける場合、<vtx_attrib> hint と同じ命名規則で指定します。
type enum
int
int2
int3
int4
uint
uint2
uint3
uint4
float
float2
float3
float4		 頂点属性変数の論理的な型です。
<vtx_attrib> type と同じ定義です。

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <vertex_symbol> [1] : 頂点シェーダーにおける変数のシンボルを定義する要素です。
  • <ui_label> [0～1] : 変数を編集する UI のラベルを定義する要素です。
  • <ui_comment> [0～1] : 変数を編集する UI のコメントを定義する要素です。
  • <ui_group> [0～1] : 変数を編集する UI のグループ分けを定義する要素です。
  • <ui_order> [0～1] : 変数を編集する UI の並び順優先度を定義する要素です。
  • <ui_editable> [0～1] : 変数を UI で編集可能かどうかを切り替える要素です。
