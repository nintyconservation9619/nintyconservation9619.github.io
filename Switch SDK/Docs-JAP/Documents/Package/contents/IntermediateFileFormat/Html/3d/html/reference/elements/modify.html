<modify>

説明

中間ファイルの変更に関する記録を保存する要素です。

<file_info> および <create> がある場合にのみ、 <modify> を指定します。
この要素は DCC ツールから出力しません。

親要素

  • <file_info> : 中間ファイルの作成や変更に関する記録を保存する要素です。

属性

名前 型と値 説明
tool_name string
-		 中間ファイルを変更したツールの名前です
tool_version string
-		 中間ファイルを変更したツールのバージョンです。

内容

内容はありません。
