<original_step>

説明

DCC ツールから出力した、キーの間を補間しないステップカーブの要素です。

親要素

  • <original_color_anim_target> : DCC ツールにおけるカラーをアニメーションする要素です。
  • <original_texsrt_anim_target> : DCC ツールにおけるテクスチャー SRT をアニメーションする要素です。

属性

名前 型と値 説明 DCC
count int
1 ～ 65535		 カーブが持つキーの数です。
 設定
pre_wrap enum
clamp
repeat
mirror
relative_repeat		 最初のキーより前のフレームで評価した際の動作を指定します。
clamp 最初のキーの値を返します。
repeat カーブが単純に繰り返した値を返します。
mirror カーブが反転しつつ繰り返した値を返します。

 設定
post_wrap enum
clamp
repeat
mirror
relative_repeat		 最後のキーより後のフレームで評価した際の動作を指定します。
clamp 最後のキーの値を返します。
repeat カーブが単純に繰り返した値を返します。
mirror カーブが反転しつつ繰り返した値を返します。

 設定
baked bool
true false		 カーブをベイクした場合に、true を指定します。
 設定
stream_index int
0 ～ MAX		 カーブの実データを格納する <stream> のインデックスです。
一行毎にキーのフレーム、値を指定します。
 設定
