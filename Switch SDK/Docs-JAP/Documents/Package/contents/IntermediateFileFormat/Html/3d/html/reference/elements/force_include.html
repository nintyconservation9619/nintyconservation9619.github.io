<force_include>

説明

強制的にインクルードしたシェーダーソースを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <force_include_array> : <force_include> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
src_index int
0 ～ 65534		 強制的にインクルードしたシェーダーソースを定義する <shader_src> のインデックスです。

内容

内容はありません。
