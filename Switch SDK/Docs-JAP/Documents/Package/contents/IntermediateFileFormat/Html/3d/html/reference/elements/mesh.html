<mesh>

説明

ポリゴンを定義する頂点インデックスの要素です。

親要素

  • <mesh_array> : <mesh> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
mode enum
triangles
triangle_fan
triangle_strip		 頂点インデックスのプリミティブを指定します。
DCC ツールからの出力時は、常に triangles を指定します。
triangles トライアングルズによる頂点インデックス指定です。
triangle_fan トライアングルファンによる頂点インデックス指定です。
triangle_strip トライアングルストリップによる頂点インデックス指定です。

 固定
quantize_type enum
none
uint_16
uint_32		 頂点インデックスの型を指定します。
DCC ツールからの出力時は、常に none を指定します。
中間ファイルオプティマイザのポリゴン最適化時に、uint_16 か uint_32 を指定します。
none 量子化の精度を指定していません。
バイナリー化する場合は u32 として扱います。
uint_16 16bit 整数による量子化を指定します。
uint_32 32bit 整数による量子化を指定します。

 固定
count int
1 ～ MAX		 頂点インデックス数を指定します。
 設定
stream_index int
0 ～ MAX		 頂点インデックスの実データを格納する <stream> のインデックスです。
視点から見て、反時計回りのポリゴンを表とします。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <submesh_array> [1] : <submesh> の配列要素です。
