<original_color>

説明

DCC ツールにおけるマテリアルのカラー設定を定義する要素です。

親要素

  • <original_color_array> : <original_color> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
hint string
[0-9A-Za-z_]+		 ツール側でカラーの関連付けに利用するヒント情報です。
参照元の <shader_param> original_hint と対になっています。
diffuse: ディフューズカラーです。
opacity: 不透明カラーです。
ambient: アンビエントカラーです。
emission: エミッションカラーです。
specular: スペキュラカラーです。
設定
color float[3]
MIN ～ MAX		 カラーの値を RGB の順に指定します。
設定

内容

内容はありません。
