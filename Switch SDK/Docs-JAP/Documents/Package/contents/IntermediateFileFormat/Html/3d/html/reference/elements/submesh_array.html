<submesh_array>

説明

<submesh> の配列要素です。

親要素

  • <mesh> : ポリゴンを定義する頂点インデックスの要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素はソートせず、ユーザーの指定順を保持します。
DCC ツールからの出力時は常に一つです。
<submesh> の offset と count による範囲指定は、0 から順に頂点インデックスの全領域を指定する必要があります。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <submesh> [1～65535] : 描画呼び出しを定義する要素です。
