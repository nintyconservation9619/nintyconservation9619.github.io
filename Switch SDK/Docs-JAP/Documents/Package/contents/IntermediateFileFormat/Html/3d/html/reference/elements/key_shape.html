<key_shape>

説明

シェイプアニメーション用のキーシェイプを定義する要素です。

親要素

  • <key_shape_array> : <key_shape> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
name string
[0-9A-Za-z\-\._]+		 要素の名前です。
参照元の <vertex_shape_anim> shape_name と対になっています。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <target_attrib_array> [1] : <target_attrib> の配列要素です。
