<depth_test>

説明

深度テストを定義する要素です。

親要素

  • <render_state> : 深度テスト、アルファテスト、ブレンドといったレンダーステートを定義する要素です。

属性

名前 型と値 説明 DCC
enable bool
true false		 深度テストを有効にする場合に、true を指定します。
DCC ツールからの出力時は、常に true を指定します。
<render_state> mode が custom でなければ true を指定します。
固定
write bool
true false		 フレームバッファへの深度書き込みを有効にする場合に、true を指定します。
DCC ツールからの出力時は、常に true を指定します。
<render_state> mode が opaque か mask であれば true を、translucent であれば false を指定します。
固定
func enum
never
less
equal
lequal
greater
nequal
gequal
always		 深度テストの関数を指定します。
DCC ツールからの出力時は、常に lequal を指定します。
<render_state> mode が custom でなければ lequal を指定します。
never 常にテストをパスしません。
less 深度値が小さい場合に、テストをパスします。
equal 深度値が同じ場合に、テストをパスします。
lequal 深度値が同じか小さい場合に、テストをパスします。
greater 深度値が大きい場合に、テストをパスします。
nequal 深度値が同じでない場合に、テストをパスします。
gequal 深度値が同じか大きい場合に、テストをパスします。
always 常にテストをパスします。

 固定

内容

内容はありません。
