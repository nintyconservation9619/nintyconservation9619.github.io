<shader_definition>

説明

シェーダー定義ファイル (.fsda, .fsdb) で指定する要素です。

シェーダー定義ファイルから生成するシェーダーアーカイブの名前は、シェーダー定義ファイルの名前から拡張子を取り除いた部分です。
この要素は DCC ツールから出力しません。

親要素

  • <nw4f_3dif> : 3D 中間ファイルフォーマットのルート要素です。

属性

名前 型と値 説明
version enum
4.0.0

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <shader_definition_info> [1] : <shader_definition> に関する情報の要素です。
  • <force_include_array> [0～1] : <force_include> の配列要素です。
  • <shading_model_array> [1] : <shading_model> の配列要素です。
  • <shader_src_array> [1] : <shader_src> の配列要素です。
  • <stream_array> [0～1] : <stream> の配列要素です。
