<vtx_buffer>

説明

頂点バッファを定義する要素です。

親要素

  • <vtx_buffer_array> : <vtx_buffer> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <input_array> [1] : <input> の配列要素です。
