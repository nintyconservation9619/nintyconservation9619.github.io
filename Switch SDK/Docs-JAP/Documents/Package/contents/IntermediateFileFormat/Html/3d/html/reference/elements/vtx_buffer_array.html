<vtx_buffer_array>

説明

<vtx_buffer> の配列要素です。

親要素

  • <vertex> : 頂点データを定義する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 255		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

DCC ツールからの出力時は、全頂点データをインターリーブで出力するため、
<vtx_buffer> は一つだけ出力します。
<shader_assign> に <shading_model> を割り当てた際に、<interleave> の内容に合わせて、<vtx_buffer> を組み替えます。
<interleave> 毎に <vtx_buffer> を定義し、<interleave> id_set と <attrib_assign> id が合致したら、<vtx_buffer> に <input> を登録します。
合致しなかった <attrib_assign> id については、<interleave> id_set="" に対応した <vtx_buffer> に登録します。
最終的に <input> が登録されなかった <vtx_buffer> は削除します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <vtx_buffer> [1～255] : 頂点バッファを定義する要素です。
