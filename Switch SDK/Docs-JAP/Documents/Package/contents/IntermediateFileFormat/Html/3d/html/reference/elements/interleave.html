<interleave>

説明

頂点属性のインターリーブを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <interleave_array> : <interleave> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
id_set string
[0-9A-Za-z _]*		 インターリーブする頂点属性の id をスペース区切りで指定します。
空文字列は未指定の頂点属性すべてを指定します。
binarize bool
true false		 インターリーブした頂点属性をバイナリー化する場合に true を指定します。

内容

内容はありません。
