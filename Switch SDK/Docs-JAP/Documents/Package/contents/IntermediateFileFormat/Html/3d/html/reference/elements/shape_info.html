<shape_info>

説明

<shape> に関する情報の要素です。

親要素

  • <shape> : モデルの形状を定義する要素です。

属性

名前 型と値 説明 DCC
mat_name string
[0-9A-Za-z\-\._]+		 描画に使用する <material> の名前です。
設定
bone_name string
[0-9A-Za-z\-\._]+		 リジッドボディの描画やビジビリティアニメーションに使用する <bone> の名前です。
設定
original_material_name string
-		 シェイプの元データのマテリアル名を指定します。
シェイプ圧縮等で複数のシェイプが結合された場合はカンマ区切りで複数の名前を指定します。
設定
original_bone_name string
-		 シェイプの元データのボーン名を指定します。
シェイプ圧縮等で複数のシェイプが結合された場合はカンマ区切りで複数の名前を指定します。
設定
vertex_index int
0 ～ 65534		 形状定義に利用する <vertex> のインデックスです。
 設定
vertex_skinning_count int
0 ～ 128		 一つの頂点に影響を与えるボーン数です。
すなわち頂点あたりのボーンインデックス数、ボーンウェイト数です。
リジッドボディの場合は 0 になります。
設定
local_coordinate bool
true false		 頂点列がボーンローカル座標系で格納されている場合(シェイプがリジッドボディー、またはリジッドスキニングの場合)に、true を指定します。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
original_shape_hash string
[^"']*		 シェイプの元データのハッシュ値を指定します。
DCC ツールからの出力時は、常に空文字列を指定します。
固定
original_shape_count int
1 ～ 65535		 シェイプ圧縮で圧縮する前のシェイプ数を指定します。
DCC ツールからの出力時は、常に 1 を指定します。
固定
delete_near_vertex_mode string
[^"']*		 適用済みの近似頂点削除のモードを指定します。
DCC ツールからの出力時は、常に空文字列を指定します。
固定
divide_submesh_mode string
[^"']*		 適用済みサブメッシュ分割のモードを指定します。
DCC ツールからの出力時は、常に空文字列を指定します。
固定
optimize_primitive_mode string
[^"']*		 適用済みプリミティブ最適化のモードを指定します。
DCC ツールからの出力時は、常に空文字列を指定します。
空文字列 プリミティブ最適化を適用していません。
default 通常の設定でプリミティブ最適化を適用済みです。
force 時間を掛けたプリミティブ最適化を適用済みです。
brute_force より時間を掛けたプリミティブ最適化を適用済みです。
forsyth http://tomforsyth1000.github.io/papers/fast_vert_cache_opt.html で解説されるアルゴリズムで最適化済みです。

 固定
polygon_reduction_mode string
[^"']*		 適用済みポリゴンリダクション最適化のモードを指定します。
DCC ツールからの出力時は、常に空文字列を指定します。
固定

内容

内容はありません。
