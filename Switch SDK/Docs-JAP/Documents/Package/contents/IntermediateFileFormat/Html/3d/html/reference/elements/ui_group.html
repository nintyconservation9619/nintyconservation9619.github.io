<ui_group>

説明

変数を編集する UI のグループ分けを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <option_var> : シェーダーのオプション変数を定義する要素です。
  • <attrib_var> : シェーダーの頂点属性変数を定義する要素です。
  • <sampler_var> : シェーダーのサンプラー変数を定義する要素です。
  • <uniform_var> : シェーダーのユニフォーム変数を定義する要素です。
  • <render_info_slot> : シェーディングモデルが受け入れる描画情報を定義する要素です。
  • <textblock> : シェーディングモデルが受け入れるテキストブロックを定義する要素です。
  • <group> : 編集用の UI をまとめるグループを定義する要素です。

属性

名前 型と値 説明
group_name string
[0-9A-Za-z_]+		 変数を編集する UI が所属するグループの名前です。

内容

内容はありません。
