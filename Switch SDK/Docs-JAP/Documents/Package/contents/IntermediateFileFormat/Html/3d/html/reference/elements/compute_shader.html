<compute_shader>

説明

演算シェーダー設定を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader> : シェーダー設定を定義する要素です。

属性

名前 型と値 説明
path string
[^*?\\|"'<> ]+		 演算シェーダーソースのパスです。
<include_path> path からの相対パスで指定します。

内容

内容はありません。
