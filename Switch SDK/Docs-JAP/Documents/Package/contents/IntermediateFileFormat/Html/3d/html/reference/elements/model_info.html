<model_info>

説明

<model> に関する情報の要素です。

親要素

  • <model> : モデルファイル (.fmda, .fmdb) で指定する要素です。

属性

名前 型と値 説明 DCC
material_count int
0 ～ 65535		 マテリアル ( <material> ) の数です。
 設定
bone_count int
0 ～ 65535		 ボーン ( <bone> ) の数です。
 設定
shape_count int
0 ～ 65535		 シェイプ ( <shape> ) の数です。
 設定
smooth_skinning_shape int
0 ～ 65535		 スムーススキニングのシェイプの数です。
 設定
rigid_skinning_shape int
0 ～ 65535		 リジッドスキニングのシェイプの数です。
 設定
smooth_skinning_matrix int
0 ～ 65535		 スムーススキニングの行列の数です。
 設定
rigid_skinning_matrix int
0 ～ 65535		 リジッドスキニングの行列の数です。
 設定
total_process_vertex int
0 ～ MAX		 モデル内のすべての頂点のうち、頂点シェーダーが処理する頂点の数です。
GPU の頂点キャッシュにより、total_index 以下の値になります。
 設定
unite_pos_quantize bool
true false		 頂点座標の量子化型をモデル内でもっとも精度が高くなる型で統一する場合に、true を指定します。
DCC ツールからの出力時は、常に false を指定します。
 固定
dcc_preset string
-		 DCC プラグインのプリセット名です。
 設定
dcc_magnify float
0 < ～ MAX		 DCC ツールで指定していた全体スケールです。
 設定
dcc_start_frame int
MIN ～ MAX		 DCC ツールで指定していたアニメーションの開始フレームです。
アニメーションのあるリジッドボディを出力する際に、開始フレームの状態でモデルを出力します。
 設定

内容

内容はありません。
