<tool_data>

説明

NintendoWare のツール用データを格納する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <texture> : テクスチャーファイル (.ftxa, .ftxb) で指定する要素です。
  • <material> : マテリアルを定義する要素です。
  • <model> : モデルファイル (.fmda, .fmdb) で指定する要素です。
  • <bone> : スキニング等に利用する座標系を定義するボーンの要素です。
  • <material_anim> : マテリアルアニメーションファイル (.fmaa, .fmab) で指定する要素です。
  • <shape_anim> : シェイプアニメーションファイル (.fsha, .fshb) で指定する要素です。
  • <skeletal_anim> : スケルタルアニメーションファイル (.fska, .fskb) で指定する要素です。
  • <scene_anim> : シーンアニメーションファイル (.fsna, .fsnb) で指定する要素です。
  • <camera_anim> : ユーザー定義カメラをアニメーションする要素です。
  • <light_anim> : ユーザー定義ライトをアニメーションする要素です。
  • <fog_anim> : ユーザー定義フォグをアニメーションする要素です。
  • <shader_param_anim> : シェーダーパラメータアニメーションファイル (.fspa, .fspb)、カラーアニメーションファイル (.fcla .fclb)、テクスチャー SRT アニメーションファイル (.ftpa .ftpb) で指定する要素です。
  • <shader_variation> : シェーダーバリエーションファイル (.fsva, .fsvb) で指定する要素です。
  • <tex_pattern_anim> : テクスチャーパターンアニメーションファイル (.ftpa, .ftpb) で指定する要素です。
  • <bone_visibility_anim> : ボーンビジビリティアニメーションファイル (.fvba, .fvbb) で指定する要素です。
  • <mat_visibility_anim> : マテリアルビジビリティアニメーションファイル (.fvma, .fvmb) で指定する要素です。

属性

属性はありません。

内容

NintendoWare のツール用データを XML 形式で格納します。
3DEditor が利用する要素は、要素名の接頭語として nw3de_ を使用します。
中間ファイルのフォーマッターでは、属性を改行しません。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <*> [1～制限無し] : ユーザー独自のツール用データを定義する要素です。
