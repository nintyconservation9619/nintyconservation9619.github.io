<mat_visibility_anim_info>

説明

<mat_visibility_anim> に関する情報の要素です。

この要素は DCC ツールから出力しません。

親要素

  • <mat_visibility_anim> : マテリアルビジビリティアニメーションファイル (.fvma, .fvmb) で指定する要素です。

属性

名前 型と値 説明
frame_count int
0 ～ 16777215		 アニメーションのフレーム数です。
loop bool
true false		 アニメーションをループ再生する場合に、true を指定します。
frame_resolution int
1 ～ MAX		 アニメーションのベイク、量子化、コマ化におけるフレーム解像度です。
フレームの分割数を指定します。
dcc_preset string
-		 DCC プラグインのプリセット名です。
dcc_start_frame int
MIN ～ MAX		 DCC ツールで指定していたアニメーションの開始フレームです。
dcc_end_frame int
MIN ～ MAX		 DCC ツールで指定していたアニメーションの終了フレームです。
dcc_fps float
0 < ～ MAX		 DCC ツールで指定していたアニメーションのフレームレートです。
bake_all bool
true false		 DCC ツールですべてのアニメーションをベイクした場合に、true を指定します。

内容

内容はありません。
