<skeletal_anim_info>

説明

<skeletal_anim> に関する情報の要素です。

親要素

  • <skeletal_anim> : スケルタルアニメーションファイル (.fska, .fskb) で指定する要素です。

属性

名前 型と値 説明 DCC
frame_count int
0 ～ 16777215		 アニメーションのフレーム数です。
設定
loop bool
true false		 アニメーションをループ再生する場合に、true を指定します。
 設定
frame_resolution int
1 ～ MAX		 アニメーションのベイク、量子化、コマ化におけるフレーム解像度です。
フレームの分割数を指定します。
 設定
dcc_preset string
-		 DCC プラグインのプリセット名です。
 設定
dcc_magnify float
0 < ～ MAX		 DCC ツールで指定していた全体スケールです。
DCC ツール外でアニメーションをベイクする際に、bake_tolerance_translate に乗算します。
 設定
dcc_start_frame int
MIN ～ MAX		 DCC ツールで指定していたアニメーションの開始フレームです。
 設定
dcc_end_frame int
MIN ～ MAX		 DCC ツールで指定していたアニメーションの終了フレームです。
 設定
dcc_fps float
0 < ～ MAX		 DCC ツールで指定していたアニメーションのフレームレートです。
 設定
bake_all bool
true false		 DCC ツールですべてのアニメーションをベイクした場合に、true を指定します。
 設定
scale_mode enum
standard
maya
softimage		 ボーン行列のスケール計算方法です。
DCC ツールに合わせた値を指定します。
standard 通常の方式でスケールを計算します。
maya Maya のセグメントスケール補正を考慮して、スケールを計算します。
softimage Softimage のスケール計算方式を考慮して、スケールを計算します。

 設定
rotate_mode enum
euler_xyz
quaternion		 ボーン行列の回転計算方法です。
DCC ツールからの出力時には euler_xyz を指定します。
euler_xyz オイラー角の回転を XYZ の順に計算します。
オイラー角の回転値は度数で指定します。
quaternionクォータニオンで計算します。

 固定

0 ～ MAX		 アニメーションをベイクする際の、スケールに対する許容誤差を絶対値で指定します。
 設定

0 ～ MAX		 アニメーションをベイクする際の、回転に対する許容誤差です。

 設定

0 ～ MAX		 アニメーションをベイクする際の、移動に対する許容誤差を絶対値で指定します。
 設定
quantize_tolerance_scale float
0 ～ MAX		 アニメーションを量子化する際の、スケールに対する許容誤差です。
アニメーション結果の値の範囲に対する割合で指定します。
 設定
quantize_tolerance_rotate float
0 ～ MAX		 アニメーションを量子化する際の、回転に対する許容誤差です。
絶対値で度数を指定します。
 設定

0 ～ MAX		 アニメーションを量子化する際の、移動に対する許容誤差です。
アニメーション結果の値の範囲に対する割合で指定します。
 設定
