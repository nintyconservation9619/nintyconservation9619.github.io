<pattern_anim>

説明

テクスチャーサンプラーのテクスチャーをアニメーションする要素です。

親要素

  • <tex_pattern_anim_array> : <pattern_anim> の配列要素です。

属性

名前 型と値 説明 DCC
sampler_name string
[0-9A-Za-z_]+		 実機側でテクスチャーサンプラーとの関連付けに利用する名前です。
仕様は <sampler> name と同一です。
設定
hint string
[0-9A-Za-z_]*		 ツール側でテクスチャーサンプラーとの関連付けに利用するヒント情報です。
仕様は <sampler> hint と同一です。
設定
base_value float
MIN ～ MAX		 アニメーションの初期値を指定します。
アニメーションが付いている場合には、0 フレームの値を指定します。
設定

内容

アニメーションが無く、base_value を固定値として対象に設定する場合は、内容がありません。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <step_curve> [0～1] : キーの間を補間しないステップカーブの要素です。
