<skeleton>

説明

座標系の定義に利用する <bone> を取り纏める要素です。

親要素

  • <model> : モデルファイル (.fmda, .fmdb) で指定する要素です。

属性

属性はありません。

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <skeleton_info> [1] : <skeleton> に関する情報の要素です。
  • <bone_array> [1] : <bone> の配列要素です。
