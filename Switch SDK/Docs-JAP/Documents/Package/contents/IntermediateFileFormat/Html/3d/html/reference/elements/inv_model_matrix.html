<inv_model_matrix>

説明

バインドポーズの逆行列要素です。

<bone> の matrix_index が "-1 -1" でない場合に定義します。

親要素

  • <bone> : スキニング等に利用する座標系を定義するボーンの要素です。

属性

属性はありません。

内容

バインドポーズのモデル座標系からからボーンのローカル座標系への 3×4 変換行列を、12 個の float で指定します。
