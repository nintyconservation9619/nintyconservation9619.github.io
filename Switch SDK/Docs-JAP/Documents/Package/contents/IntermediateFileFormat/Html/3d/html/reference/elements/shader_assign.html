<shader_assign>

説明

マテリアルに割り当てるシェーダーを指定します。

この要素は DCC ツールから出力しません。

親要素

  • <material> : マテリアルを定義する要素です。

属性

名前 型と値 説明
shader_archive string
[0-9A-Za-z_]+		 シェーダーアーカイブの名前です。
シェーダー定義ファイルのファイル名から拡張子を取り除いた名前を指定します。
shading_model string
[0-9A-Za-z_]+		 割り当てる <shading_model> の名前です。
revision int
0 ～ MAX		 割り当てる <shading_model> のリビジョン番号です。

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <render_info_array> [0～1] : <render_info> の配列要素です。
  • <shader_option_array> [0～1] : <shader_option> の配列要素です。
  • <sampler_assign_array> [0～1] : <sampler_assign> の配列要素です。
  • <shader_param_array> [0～1] : <shader_param> の配列要素です。
  • <attrib_assign_array> [0～1] : <attrib_assign> の配列要素です。
