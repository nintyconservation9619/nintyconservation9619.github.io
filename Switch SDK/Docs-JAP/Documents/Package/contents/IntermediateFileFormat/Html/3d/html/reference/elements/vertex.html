<vertex>

説明

頂点データを定義する要素です。

DCC ツールからの出力時には、<shape> と対で出力します。
中間ファイルオプティマイザにて、<vertex> をマージします。

親要素

  • <vertex_array> : <vertex> の配列要素です。

属性

名前 型と値 説明 DCC
vertex_index int
0 ～ 65534		 配列内におけるインデックスです。
参照元の <shape_info> vertex_index と対になっています。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <lod_offset> [0～1] : 各 LOD レベルでの頂点インデックスのオフセット値です。
  • <vtx_attrib_array> [1] : <vtx_attrib> の配列要素です。
  • <vtx_buffer_array> [0～1] : <vtx_buffer> の配列要素です。
