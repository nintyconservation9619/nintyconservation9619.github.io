<camera_anim_target>

説明

ユーザー定義カメラの各パラメータをアニメーションする要素です。

親要素

  • <camera_anim> : ユーザー定義カメラをアニメーションする要素です。

 設定
base_value float
MIN ～ MAX		 アニメーションの初期値を指定します。
アニメーションが付いている場合には、0 フレームの値を指定します。
設定

内容

アニメーションが無く、base_value を固定値として対象に設定する場合は、内容がありません。

以下の要素のいずれかが出現します。[] 内の数値は要素の出現回数です。

  • <hermite_curve> [0～1] : キーの間を滑らかに補間するエルミートカーブの要素です。
  • <linear_curve> [0～1] : キーの間を線形補間するリニアカーブの要素です。
  • <step_curve> [0～1] : キーの間を補間しないステップカーブの要素です。
