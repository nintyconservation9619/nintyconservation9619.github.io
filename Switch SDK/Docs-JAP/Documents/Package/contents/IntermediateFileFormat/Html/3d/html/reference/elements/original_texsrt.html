<original_texsrt>

説明

DCC ツールにおけるマテリアルのテクスチャー SRT 設定を定義する要素です。

親要素

  • <original_texsrt_array> : <original_texsrt> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
hint string
[0-9A-Za-z_]+		 ツール側でテクスチャーサンプラーとの関連付けに利用するヒント情報です。
仕様は <sampler> hint と同一です。
参照元の <shader_param> original_hint と対になっています。
設定
uv_hint string
[0-9A-Za-z_]*		 頂点属性との関連付けに利用できるヒント情報です。
値を指定する際の仕様は <vtx_attrib> hint と同一です。
設定
mode enum
maya
3dsmax
softimage		 テクスチャー SRT の計算方式を指定します。
mayaMaya の計算方式です。
3dsmax3ds Max の計算方式です。
softimageSoftimage の計算方式です。

 設定
scale float[2]
MIN ～ MAX		 テクスチャー SRT のスケール値を XY の順に指定します。
設定
rotate float
MIN ～ MAX		 テクスチャー SRT の回転値を指定します。
設定
translate float[2]
MIN ～ MAX		 テクスチャー SRT の移動値を XY の順に指定します。
設定

内容

内容はありません。
