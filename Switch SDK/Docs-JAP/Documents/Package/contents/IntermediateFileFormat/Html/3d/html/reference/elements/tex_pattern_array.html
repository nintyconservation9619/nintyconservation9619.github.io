<tex_pattern_array>

説明

<tex_pattern> の配列要素です。

親要素

  • <material_anim> : マテリアルアニメーションファイル (.fmaa, .fmab) で指定する要素です。
  • <tex_pattern_anim> : テクスチャーパターンアニメーションファイル (.ftpa, .ftpb) で指定する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素はソートせず、ユーザーの指定順を保持します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <tex_pattern> [1～65535] : テクスチャーパターンアニメーションで用いるパターンテクスチャーの要素です。
