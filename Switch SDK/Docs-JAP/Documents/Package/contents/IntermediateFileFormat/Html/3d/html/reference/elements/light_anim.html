<light_anim>

説明

ユーザー定義ライトをアニメーションする要素です。

親要素

  • <light_anim_array> : <light_anim> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
light_name string
[0-9A-Za-z\-\._]+		 アニメーション対象であるユーザー定義ライトの名前です。
設定
frame_count int
0 ～ 16777215		 アニメーションのフレーム数です。
設定
loop bool
true false		 アニメーションをループ再生する場合に、true を指定します。
 設定
type string
[0-9A-Za-z_]*		 ユーザー定義ライトのタイプを指定します。
DCC ツールからの出力時は ambient directional point spot のいずれかを指定します。
設定
dist_attn_func string
[0-9A-Za-z_]*		 ユーザー定義ライトの距離減衰関数を指定します。
DCC ツールからの出力時は、空文字列を指定します。
固定
angle_attn_func string
[0-9A-Za-z_]*		 ユーザー定義ライトの角度減衰関数を指定します。
DCC ツールからの出力時は、空文字列を指定します。
固定

内容

<light_anim_target> target の position_～、direction_～、aim_～、dist_attn_～、angle_attn_～、color0_～、color1_～ をそれぞれセットで持ちます。
<light_anim_target> は target の定義順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <light_anim_target> [0～20] : ユーザー定義ライトの各パラメータをアニメーションする要素です。
  • <user_data_array> [0～1] : <user_data> の配列要素です。
  • <comment> [0～1] : 編集用コメント等のメタ情報を格納する要素です。
  • <tool_data> [0～1] : NintendoWare のツール用データを格納する要素です。
  • <user_tool_data> [0～1] : ユーザー独自のツール用データを格納する要素です。
