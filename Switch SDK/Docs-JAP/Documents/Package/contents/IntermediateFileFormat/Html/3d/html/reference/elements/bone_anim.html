<bone_anim>

説明

<bone> のスケール、回転、移動をアニメーションする要素です。

ボーンが描画に利用されているかどうかは、以下の条件で判定します。
rigid_body || (matrix_index[0] != -1) || (matrix_index[1] != -1)

親要素

  • <bone_anim_array> : <bone_anim> の配列要素です。

内容

アニメーションの有無にかかわらず、スケールと回転と移動の XYZ 値アニメーションを <bone_anim_target> で指定します。
回転の W 値アニメーションは <skeletal_anim_info> の rotate_mode が quaternion の場合にのみ指定します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <bone_anim_target> [9～10] : <bone> の各パラメータをアニメーションする要素です。
  • <user_data_array> [0～1] : <user_data> の配列要素です。
