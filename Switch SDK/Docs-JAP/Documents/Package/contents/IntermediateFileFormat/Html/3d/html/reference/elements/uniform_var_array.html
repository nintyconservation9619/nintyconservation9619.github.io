<uniform_var_array>

説明

<uniform_var> の配列要素です。

この要素は DCC ツールから出力しません。

親要素

  • <block_var> : シェーダーのユニフォームブロック変数を定義する要素です。

属性

名前 型と値 説明
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。

内容

子要素はソートせず、シェーダーソースにおけるユーザーの指定順を保持します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <uniform_var> [1～65535] : シェーダーのユニフォーム変数を定義する要素です。
