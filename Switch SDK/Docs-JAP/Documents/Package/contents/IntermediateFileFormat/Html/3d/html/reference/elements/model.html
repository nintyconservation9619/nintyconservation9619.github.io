<model>

説明

モデルファイル (.fmda, .fmdb) で指定する要素です。

モデルの名前はファイル名から拡張子を取り除いた部分です。

親要素

  • <nw4f_3dif> : 3D 中間ファイルフォーマットのルート要素です。

属性

名前 型と値 説明 DCC
version enum
4.0.0		 -

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <process_log_array> [0～1] : <process_log> の配列要素です。
  • <model_info> [1] : <model> に関する情報の要素です。
  • <material_array> [0～1] : <material> の配列要素です。
  • <skeleton> [1] : 座標系の定義に利用する <bone> を取り纏める要素です。
  • <vertex_array> [0～1] : <vertex> の配列要素です。
  • <shape_array> [0～1] : <shape> の配列要素です。
  • <original_material_array> [0～1] : <original_material> の配列要素です。
  • <user_data_array> [0～1] : <user_data> の配列要素です。
  • <stream_array> [0～1] : <stream> の配列要素です。
  • <comment> [0～1] : 編集用コメント等のメタ情報を格納する要素です。
  • <tool_data> [0～1] : NintendoWare のツール用データを格納する要素です。
  • <user_tool_data> [0～1] : ユーザー独自のツール用データを格納する要素です。
