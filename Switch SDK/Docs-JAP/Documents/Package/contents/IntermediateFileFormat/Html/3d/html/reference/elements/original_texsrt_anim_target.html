<original_texsrt_anim_target>

説明

DCC ツールにおけるテクスチャー SRT をアニメーションする要素です。

親要素

  • <original_texsrt_anim> : DCC ツールにおけるテクスチャー SRT アニメーションの対象となるサンプラーを指定する要素です。

属性

名前 型と値 説明 DCC
target enum
scale_x
scale_y
rotate
translate_x
translate_y		 テクスチャー SRT のアニメーションするパラメータを指定します。
scale_x スケールの X 値をアニメーションします。
scale_y スケールの Y 値をアニメーションします。
rotate 回転値をアニメーションします。
translate_x 移動の X 値をアニメーションします。
translate_y 移動の Y 値をアニメーションします。

 設定
base_value float
MIN ～ MAX		 アニメーションの初期値を指定します。
アニメーションが付いている場合には、0 フレームの値を指定します。
設定

内容

アニメーションが無く、base_value を固定値として対象に設定する場合は、内容がありません。

以下の要素のいずれかが出現します。[] 内の数値は要素の出現回数です。

  • <original_hermite> [0～1] : DCC ツールから出力した、キーの間を滑らかに補間するエルミートカーブの要素です。
  • <original_linear> [0～1] : DCC ツールから出力した、キーの間を線形補間するリニアカーブの要素です。
  • <original_step> [0～1] : DCC ツールから出力した、キーの間を補間しないステップカーブの要素です。
