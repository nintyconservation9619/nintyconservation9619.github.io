<hermite_curve>

説明

キーの間を滑らかに補間するエルミートカーブの要素です。

親要素

  • <shape_anim_target> : <key_shape> の重みをアニメーションする要素です。
  • <bone_anim_target> : <bone> の各パラメータをアニメーションする要素です。
  • <camera_anim_target> : ユーザー定義カメラの各パラメータをアニメーションする要素です。
  • <light_anim_target> : ユーザー定義ライトの各パラメータをアニメーションする要素です。
  • <fog_anim_target> : ユーザー定義フォグの各パラメータをアニメーションする要素です。
  • <param_anim_target> : シェーダーパラメータをアニメーションする要素です。

属性

名前 型と値 説明 DCC
count int
1 ～ 65535		 カーブが持つキーの数です。
 設定
frame_type enum
none
frame32
frame16
frame8		 アニメーションキーのフレーム数の量子化精度を指定します。
中間ファイルオプティマイザで精度を設定します。
DCC ツールからの出力時には none を指定します。
none 量子化精度を指定していません。
frame32 32bit 浮動小数を指定します。
frame16 16bit 符号付き固定少数を指定します。
整数部は 10bit で、小数部は 5bit です。
frame8 8bit 符号無し整数を指定します。

 固定
key_type enum
none
key32
key16
key8		 アニメーションキーの値と傾きの量子化精度を指定します。
中間ファイルオプティマイザで精度を設定します。
DCC ツールからの出力時には none を指定します。
none 量子化精度を指定していません。
key32 32bit 浮動小数を指定します。
key16 16bit 符号付き整数を指定します。
key8 8bit 符号付き整数を指定します。

 固定
scale float
MIN ～ MAX		 量子化後の値や傾きに乗算するスケールです。
scale を乗算した後に offset を加算します。
DCC ツールからの出力時には 1 を指定します。
固定
offset float
MIN ～ MAX		 量子化後の値や傾きに加算するオフセットです。
scale を乗算した後に offset を加算します。
DCC ツールからの出力時には 0 を指定します。
 固定
pre_wrap enum
clamp
repeat
mirror
relative_repeat		 最初のキーより前のフレームで評価した際の動作を指定します。
clamp 最初のキーの値を返します。
repeat カーブが単純に繰り返した値を返します。
mirror カーブが反転しつつ繰り返した値を返します。
relative_repeat カーブ端の値から相対的に繰り返した値を返します。

 設定
post_wrap enum
clamp
repeat
mirror
relative_repeat		 最後のキーより後のフレームで評価した際の動作を指定します。
clamp 最後のキーの値を返します。
repeat カーブが単純に繰り返した値を返します。
mirror カーブが反転しつつ繰り返した値を返します。
relative_repeat カーブ端の値から相対的に繰り返した値を返します。

 設定
baked bool
true false		 カーブをベイクした場合に、true を指定します。
 設定
stream_index int
0 ～ MAX		 カーブの実データを格納する <stream> のインデックスです。
一行毎にキーのフレーム、値、前の傾き、後の傾きを実数で指定します。
フレームは 0 以上の値を指定します。
 設定
