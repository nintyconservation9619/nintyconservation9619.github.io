<submesh>

説明

描画呼び出しを定義する要素です。

プログラムでのカリング等に利用します。

親要素

  • <submesh_array> : <submesh> の配列要素です。

属性

名前 型と値 説明 DCC
submesh_index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
offset int
0 ～ MAX		 描画呼び出しを開始する頂点インデックスのインデックスオフセットを指定します。
DCC ツールからの出力時は、常に 0 を指定します。
固定
count int
1 ～ MAX		 offset の位置から描画する頂点インデックス数を指定します。
設定

内容

内容はありません。
