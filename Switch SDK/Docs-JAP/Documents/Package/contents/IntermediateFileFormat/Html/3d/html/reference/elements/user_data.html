<user_data>

説明

ランタイムで取得できる、ユーザー独自のデータ（ユーザーデータ）を格納する要素です。

親要素

  • <user_data_array> : <user_data> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
name string
[0-9A-Za-z_]+		 ユーザーデータの名前です。
設定

内容

以下の要素のいずれかが出現します。[] 内の数値は要素の出現回数です。

  • <user_int> [1] : 整数型ユーザーデータを格納する要素です。
  • <user_float> [1] : 実数型ユーザーデータを格納する要素です。
  • <user_string> [1] : ASCII 文字列ユーザーデータを格納する要素です。
  • <user_wstring> [1] : Unicode 文字列ユーザーデータを格納する要素です。
  • <user_stream> [1] : ストリーム型ユーザーデータを格納する要素です。
