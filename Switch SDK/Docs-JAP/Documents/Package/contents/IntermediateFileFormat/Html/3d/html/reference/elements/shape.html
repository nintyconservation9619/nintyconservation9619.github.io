<shape>

説明

モデルの形状を定義する要素です。

親要素

  • <shape_array> : <shape> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
name string
[0-9A-Za-z\-\._]+		 要素の名前です。
参照元の <vertex_shape_anim> shape_name と対になっています。 DCC ツールからの出力時は「ボーン名__マテリアル名」を指定します。ここでいうボーン名は、シェイプの親ボーンに変換される DCC ツール上のノード名と同じです。
名前が重複する場合は 2 番目以降の名前に「_1」「_2」と番号を付加します。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <shape_info> [1] : <shape> に関する情報の要素です。
  • <mesh_array> [1] : <mesh> の配列要素です。
  • <key_shape_array> [0～1] : <key_shape> の配列要素です。
