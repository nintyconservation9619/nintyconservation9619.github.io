<skeleton_info>

説明

<skeleton> に関する情報の要素です。

親要素

  • <skeleton> : 座標系の定義に利用する <bone> を取り纏める要素です。

属性

名前 型と値 説明 DCC
scale_enable bool
true false		 true を指定すると、ボーン行列を計算する際にスケールを計算します。
DCC ツールからの出力時には、常に true を指定します。
固定
scale_mode enum
standard
maya
softimage		 ボーン行列のスケール計算方法です。
DCC ツールに合わせた値を指定します。
standard 通常の方式でスケールを計算します。
maya Maya のセグメントスケール補正を考慮して、スケールを計算します。
softimage Softimage のスケール計算方式を考慮して、スケールを計算します。

 設定
rotate_mode enum
euler_xyz
quaternion		 ボーン行列の回転計算方法です。
DCC ツールからの出力時には euler_xyz を指定します。
euler_xyz オイラー角の回転を XYZ の順に計算します。
オイラー角の回転値は度数で指定します。
quaternionクォータニオンで計算します。

 固定
motion_mirroring_enable bool
true false		 モーションのミラーリングを行う場合、true を指定します。
DCC ツールからの出力時には、常に false を指定します。
固定

内容

内容はありません。
