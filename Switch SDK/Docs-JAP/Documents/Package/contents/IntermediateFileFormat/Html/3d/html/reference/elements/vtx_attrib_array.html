<vtx_attrib_array>

説明

<vtx_attrib> の配列要素です。

親要素

  • <vertex> : 頂点データを定義する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 255		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素は <vtx_attrib> hint の定義順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <vtx_attrib> [1～255] : 頂点属性を定義する要素です。
