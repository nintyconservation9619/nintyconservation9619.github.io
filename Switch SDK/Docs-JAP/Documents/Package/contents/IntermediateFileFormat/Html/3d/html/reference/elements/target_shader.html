<target_shader>

説明

シェーダーバリエーションを生成する対象シェーダーを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <target_shader_array> : <target_shader> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
shading_model_name string
[0-9A-Za-z_]+		 シェーダーバリエーションを生成するシェーディングモデルの名前です。

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <shader_program_array> [1] : <shader_program> の配列要素です。
