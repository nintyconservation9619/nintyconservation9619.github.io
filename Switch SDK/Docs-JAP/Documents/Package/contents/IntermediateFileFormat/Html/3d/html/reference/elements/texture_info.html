<texture_info>

説明

<texture> に関する情報の要素です。

親要素

  • <texture> : テクスチャーファイル (.ftxa, .ftxb) で指定する要素です。

属性

名前 型と値 説明 DCC
dcc_preset string
-		 DCC プラグインのプリセット名です。
 設定
hint string
[0-9A-Za-z_]*		 テクスチャーの用途をヒント情報として指定します。
<sampler> hint に似た、albedo、opacity、emission、normal、tangent、specular、reflection といった値を指定します。
normal が指定されていれば法線を正規化する等、各ツールで hint に合わせて動作を変える場合があります。
設定
linear bool[4]
true false		 <original_image> をコンバートしたテクスチャーデータのガンマ値がリニア変換されているかどうかを指定します。
<original_image> の rgba チャンネルに対して、true を指定したチャンネルは、ガンマ値が sRGB からリニア変換されたテクスチャーデータがテクスチャーファイルに格納されています。
falseを指定したチャンネルは、<original_image>と同じガンマ値のテクスチャーデータがテクスチャーファイルに格納されています。
ただし quantize_type が srgb_ で始まるフォーマットは、GPUのテクスチャーフェッチ時にリニア変換が行われるため、<original_image>と同じガンマ値のテクスチャーデータがテクスチャーファイルに格納されています。
srgb_ で始まるフォーマットの場合は "true true true false" または "true true true true" を指定します。
また、2 チャンネルの実機向けフォーマットについては、<original_image> が RGB であれば R と G が、RGBA であれば R と A が、ガンマ補正の対象となるチャンネルになります。
DCC ツールからの出力時は、コンフィグ設定に合わせてフラグを指定します。

depth int
1 ～ MAX		 dimension が cube の場合、6 を指定します。
dimension が 1d_array か 2d_array の場合は、テクスチャーの枚数を指定します。
