<sampler_assign_array>

説明

<sampler_assign> の配列要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader_assign> : マテリアルに割り当てるシェーダーを指定します。

属性

名前 型と値 説明
length int
1 ～ 255		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。

内容

子要素は <sampler_assign> id を、シェーダーの id 定義順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <sampler_assign> [1～255] : シェーダーにテクスチャーのサンプラーを割り当てる要素です。
