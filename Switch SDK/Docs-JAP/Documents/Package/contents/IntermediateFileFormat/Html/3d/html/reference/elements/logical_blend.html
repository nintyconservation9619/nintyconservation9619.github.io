<logical_blend>

説明

フレームバッファとの論理ブレンドを定義する要素です。

親要素

  • <render_state> : 深度テスト、アルファテスト、ブレンドといったレンダーステートを定義する要素です。

属性

inv_or		 論理ブレンドのオペレーションを指定します。
DCC ツールからの出力時は、常に copy を指定します。
<render_state> mode が custom でなければ、copy を指定します。
詳細は SDK を確認してください。
固定

内容

内容はありません。
