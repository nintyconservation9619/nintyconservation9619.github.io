<shader>

説明

シェーダー設定を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader_array> : <shader> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
name string
[0-9A-Za-z_]+		 シェーダー設定の名前です。
生成先の <shading_model> name と対になっています。
material_shader bool
true false		 マテリアル用のシェーダー設定である場合に、true を指定します。

内容

<vertex_shader>、<geometry_shader> 、<fragment_shader> のいずれかが必ず存在します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <vertex_shader> [0～1] : 頂点シェーダー設定を定義する要素です。
  • <geometry_shader> [0～1] : ジオメトリシェーダー設定を定義する要素です。
  • <fragment_shader> [0～1] : フラグメントシェーダー設定を定義する要素です。
  • <compute_shader> [0～1] : 演算シェーダー設定を定義する要素です。
  • <macro_array> [0～1] : <macro> の配列要素です。
  • <variation_array> [0～1] : <variation> の配列要素です。
