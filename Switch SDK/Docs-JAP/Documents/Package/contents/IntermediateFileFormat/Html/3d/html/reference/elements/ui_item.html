<ui_item>

説明

変数を編集する UI の型を切り替える要素です。

この要素は DCC ツールから出力しません。

親要素

  • <option_var> : シェーダーのオプション変数を定義する要素です。
  • <uniform_var> : シェーダーのユニフォーム変数を定義する要素です。
  • <render_info_slot> : シェーディングモデルが受け入れる描画情報を定義する要素です。

属性

名前 型と値 説明
value enum
auto
color
check
combiner		 変数を編集する UI の型です。
auto UI の型を自動的に判別します。
color カラー型の UI を指定します。
check チェックボックスの UI を指定します。
チェックが無効の場合は <ui_min>、有効の場合は <ui_max> の値になります。
<ui_min> が定義されていない場合は 0、<ui_max> が定義されていない場合は 1 として扱います。
combiner 開発用の設定です。使用しないでください。

内容

内容はありません。
