<attrib_assign>

説明

シェーダーに割り当てる頂点属性を指定する要素です。

マテリアルに関連付いている全シェイプに対する頂点属性割り当ての論理和を指定します。
この要素は DCC ツールから出力しません。

親要素

  • <attrib_assign_array> : <attrib_assign> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
id string
[0-9A-Za-z_]+		 割り当てる <attrib_var> の識別子です。
attrib_name string
[0-9A-Za-z_]+		 シェーダーに割り当てる <vtx_attrib> の名前です。

内容

内容はありません。
