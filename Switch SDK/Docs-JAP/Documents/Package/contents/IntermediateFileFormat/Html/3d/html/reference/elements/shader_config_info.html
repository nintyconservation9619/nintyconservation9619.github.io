<shader_config_info>

説明

<shader_config> に関する情報の要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader_config> : シェーダー設定ファイル (.fsca, .fscb) で指定する要素です。

属性

名前 型と値 説明
code_page int
0 ～ 65535		 シェーダーソースコードの文字コードを表すコードページを指定します。

内容

内容はありません。
