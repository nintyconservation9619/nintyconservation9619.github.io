<tex_pattern>

説明

テクスチャーパターンアニメーションで用いるパターンテクスチャーの要素です。

親要素

  • <tex_pattern_array> : <tex_pattern> の配列要素です。

属性

名前 型と値 説明 DCC
pattern_index int
0 ～ 65534		 配列内におけるインデックスです。
<pattern_anim_target> が持つカーブの評価値から参照されます。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
tex_name string
[0-9A-Za-z\-\._]*		 テクスチャーパターンアニメーションで用いる <texture> の名前です。
ftx ファイル名から拡張子を取り除いた名前を指定します。
設定

内容

内容はありません。
