<fog_anim_target>

説明

ユーザー定義フォグの各パラメータをアニメーションする要素です。

親要素

  • <fog_anim> : ユーザー定義フォグをアニメーションする要素です。

属性

名前 型と値 説明 DCC
target enum
dist_attn_start
dist_attn_end
color_r
color_g
color_b		 アニメーション対象を指定します。
dist_attn_start 距離減衰の開始距離をアニメーションします。
dist_attn_end 距離減衰の終了距離をアニメーションします。
color_r カラーの R をアニメーションします。
color_g カラーの G をアニメーションします。
color_b カラーの B をアニメーションします。

 設定
base_value float
MIN ～ MAX		 アニメーションの初期値を指定します。
アニメーションが付いている場合には、0 フレームの値を指定します。
設定

内容

アニメーションが無く、base_value を固定値として対象に設定する場合は、内容がありません。

以下の要素のいずれかが出現します。[] 内の数値は要素の出現回数です。

  • <hermite_curve> [0～1] : キーの間を滑らかに補間するエルミートカーブの要素です。
  • <linear_curve> [0～1] : キーの間を線形補間するリニアカーブの要素です。
  • <step_curve> [0～1] : キーの間を補間しないステップカーブの要素です。
