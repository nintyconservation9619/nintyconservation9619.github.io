<bone_anim_target>

説明

<bone> の各パラメータをアニメーションする要素です。

親要素

  • <bone_anim> : <bone> のスケール、回転、移動をアニメーションする要素です。

属性

translate_z		 アニメーション対象を指定します。
scale_x スケールの X 値をアニメーションします。
scale_y スケールの Y 値をアニメーションします。
scale_z スケールの Z 値をアニメーションします。
rotate_x 回転の X 値をアニメーションします。
rotate_y 回転の Y 値をアニメーションします。
rotate_z 回転の Z 値をアニメーションします。
rotate_w 回転の W 値をアニメーションします。
<skeletal_anim_info> の rotate_mode が quaternion の場合に指定します。
translate_x 移動の X 値をアニメーションします。
translate_y 移動の Y 値をアニメーションします。
translate_z 移動の Z 値をアニメーションします。

 設定
base_value float
MIN ～ MAX		 アニメーションの初期値を指定します。
アニメーションが付いている場合には、0 フレームの値を指定します。
設定

内容

アニメーションが無く、base_value を固定値として対象に設定する場合は、内容がありません。

以下の要素のいずれかが出現します。[] 内の数値は要素の出現回数です。

  • <hermite_curve> [0～1] : キーの間を滑らかに補間するエルミートカーブの要素です。
  • <linear_curve> [0～1] : キーの間を線形補間するリニアカーブの要素です。
  • <step_curve> [0～1] : キーの間を補間しないステップカーブの要素です。
