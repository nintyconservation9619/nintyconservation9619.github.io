<original_color_array>

説明

<original_color> の配列要素です。

親要素

  • <original_material> : DCC ツールにおけるシェーダーパラメータ関連のマテリアル設定を定義する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 255		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素は <original_color> hint の定義順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <original_color> [1～255] : DCC ツールにおけるマテリアルのカラー設定を定義する要素です。
