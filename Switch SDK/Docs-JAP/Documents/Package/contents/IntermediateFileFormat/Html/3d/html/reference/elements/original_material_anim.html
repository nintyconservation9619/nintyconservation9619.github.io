<original_material_anim>

説明

DCC ツールにおけるマテリアルアニメーションの、対象となるマテリアルを指定する要素です。

親要素

  • <original_material_anim_array> : <original_material_anim> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
mat_name string
[0-9A-Za-z\-\._]+		 アニメーション対象となる <material> の名前です。
設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <original_color_anim_array> [0～1] : <original_color_anim> の配列要素です。
    拡張子が .ftsa .ftsb の場合、この要素は含まれません。
  • <original_texsrt_anim_array> [0～1] : <original_texsrt_anim> の配列要素です。
    拡張子が .fcla .fclb の場合、この要素は含まれません。
