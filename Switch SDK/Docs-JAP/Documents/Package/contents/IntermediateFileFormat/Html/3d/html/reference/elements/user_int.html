<user_int>

説明

整数型ユーザーデータを格納する要素です。

親要素

  • <user_data> : ランタイムで取得できる、ユーザー独自のデータ（ユーザーデータ）を格納する要素です。

属性

名前 型と値 説明 DCC
count int
0 ～ MAX		 データの個数を指定します。
設定

内容

count 個の整数値を指定します。
