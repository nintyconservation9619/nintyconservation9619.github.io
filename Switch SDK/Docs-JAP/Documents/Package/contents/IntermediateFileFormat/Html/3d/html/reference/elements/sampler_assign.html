<sampler_assign>

説明

シェーダーにテクスチャーのサンプラーを割り当てる要素です。

この要素は DCC ツールから出力しません。

親要素

  • <sampler_assign_array> : <sampler_assign> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 254		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
id string
[0-9A-Za-z_]+		 割り当てる <sampler_var> の識別子です。
sampler_name string
[0-9A-Za-z_]+		 シェーダーに割り当てる <sampler> の名前です。

内容

内容はありません。
