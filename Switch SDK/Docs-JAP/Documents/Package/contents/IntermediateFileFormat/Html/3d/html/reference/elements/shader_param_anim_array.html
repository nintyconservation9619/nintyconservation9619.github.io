<shader_param_anim_array>

説明

<param_anim> の配列要素です。

この要素は DCC ツールから出力しません。

親要素

  • <per_material_anim> : マテリアルアニメーションの、対象となるマテリアルを指定する要素です。

属性

名前 型と値 説明
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。

内容

子要素は <param_anim> id を、シェーダーの id 定義順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <param_anim> [1～65535] : シェーダーパラメータアニメーションの対象となるパラメータを指定する要素です。
