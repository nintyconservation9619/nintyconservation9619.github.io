<ui_editable>

説明

変数を UI で編集可能かどうかを切り替える要素です。

この要素は DCC ツールから出力しません。

親要素

  • <attrib_var> : シェーダーの頂点属性変数を定義する要素です。

属性

名前 型と値 説明
value bool
true false		 変数を UI で編集できる場合に、true を指定します。

内容

内容はありません。
