<original_image_array>

説明

<original_image> の配列要素です。

親要素

  • <texture> : テクスチャーファイル (.ftxa, .ftxb) で指定する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素は <original_image> face、slice_index の順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <original_image> [1～65535] : <texture> の元となる二次元画像データの要素です。
