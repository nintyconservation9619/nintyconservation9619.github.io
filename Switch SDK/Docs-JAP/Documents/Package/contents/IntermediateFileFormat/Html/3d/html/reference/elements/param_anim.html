<param_anim>

説明

シェーダーパラメータアニメーションの対象となるパラメータを指定する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shader_param_anim_array> : <param_anim> の配列要素です。
  • <param_anim_array> : <param_anim> の配列要素です。

属性

名前 型と値 説明
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
id string
[0-9A-Za-z_]+		 アニメーション対象である <shader_param> の id を指定します。
仕様は <shader_param> id と同一です。
original_hint string
[0-9A-Za-z_]*		 <original_color_anim> <original_texsrt_anim> を元にアニメーションを割り当てた際に、<original_color_anim> <original_texsrt_anim> hint を指定します。
<original_color_anim> <original_texsrt_anim> を元にしなかった場合は、空文字列を指定します。

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <param_anim_target> [1～65535] : シェーダーパラメータをアニメーションする要素です。
