<tex_pattern_anim_array>

説明

<pattern_anim> の配列要素です。

親要素

  • <per_material_anim> : マテリアルアニメーションの、対象となるマテリアルを指定する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 65535		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定

内容

子要素は <pattern_anim> mat_name の文字コード順でソートします。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <pattern_anim> [1～255] : テクスチャーサンプラーのテクスチャーをアニメーションする要素です。
