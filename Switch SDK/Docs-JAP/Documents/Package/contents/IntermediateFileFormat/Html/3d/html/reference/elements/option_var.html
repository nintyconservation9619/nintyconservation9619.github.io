<option_var>

説明

シェーダーのオプション変数を定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <option_var_array> : <option_var> の配列要素です。

属性

名前 型と値 説明

0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。

[0-9A-Za-z_]+		 オプション変数の識別子です。
参照元の <shader_option> id と対になっています。

staticオプション変数を <shader_option> で設定します。

.+		 オプション変数の値の範囲です。
変数の型により、記法が異なります。
単一の値を指定する場合は、値をそのまま指定します。
種類記法<shader_option>
value
bool"bool"0 または 1
整数

内容

<vertex_symbol>、<geometry_symbol> 、<fragment_symbol> 、<compute_symbol> のいずれかが必ず存在します。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <vertex_symbol> [0～1] : 頂点シェーダーにおける変数のシンボルを定義する要素です。
  • <geometry_symbol> [0～1] : ジオメトリシェーダーにおける変数のシンボルを定義する要素です。
  • <fragment_symbol> [0～1] : フラグメントシェーダーにおける変数のシンボルを定義する要素です。
  • <compute_symbol> [0～1] : 演算シェーダーにおける変数のシンボルを定義する要素です。
  • <ui_label> [0～1] : 変数を編集する UI のラベルを定義する要素です。
  • <ui_comment> [0～1] : 変数を編集する UI のコメントを定義する要素です。
  • <ui_group> [0～1] : 変数を編集する UI のグループ分けを定義する要素です。
  • <ui_order> [0～1] : 変数を編集する UI の並び順優先度を定義する要素です。
  • <ui_item> [0～1] : 変数を編集する UI の型を切り替える要素です。
  • <ui_visible> [0～1] : 変数を編集する UI の表示を切り替える要素です。
