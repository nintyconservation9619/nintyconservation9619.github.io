<shader_config>

説明

シェーダー設定ファイル (.fsca, .fscb) で指定する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <nw4f_3dif> : 3D 中間ファイルフォーマットのルート要素です。

属性

名前 型と値 説明
version enum
4.0.0

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <shader_config_info> [1] : <shader_config> に関する情報の要素です。
  • <include_path_array> [0～1] : <include_path> の配列要素です。
  • <force_include_file_array> [0～1] : <force_include_file> の配列要素です。
  • <shader_array> [1] : <shader> の配列要素です。
  • <stream_array> [0～1] : <stream> の配列要素です。
