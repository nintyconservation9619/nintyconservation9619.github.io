<original_texsrt_anim>

説明

DCC ツールにおけるテクスチャー SRT アニメーションの対象となるサンプラーを指定する要素です。

親要素

  • <original_texsrt_anim_array> : <original_texsrt_anim> の配列要素です。
    拡張子が .fcla .fclb の場合、この要素は含まれません。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
hint string
[0-9A-Za-z_]+		 ツール側でテクスチャーサンプラーとの関連付けに利用するヒント情報です。
仕様は <sampler> hint と同一です。
参照元の <param_anim> original_hint と対になっています。
設定
mode enum
maya
3dsmax
softimage		 テクスチャー SRT の計算方式を指定します。
mayaMaya の計算方式です。
3dsmax3ds Max の計算方式です。
softimageSoftimage の計算方式です。

 設定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <original_texsrt_anim_target> [1～5] : DCC ツールにおけるテクスチャー SRT をアニメーションする要素です。
