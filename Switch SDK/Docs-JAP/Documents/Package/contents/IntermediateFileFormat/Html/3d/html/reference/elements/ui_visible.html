<ui_visible>

説明

変数を編集する UI の表示を切り替える要素です。

この要素は DCC ツールから出力しません。

親要素

  • <option_var> : シェーダーのオプション変数を定義する要素です。
  • <sampler_var> : シェーダーのサンプラー変数を定義する要素です。
  • <uniform_var> : シェーダーのユニフォーム変数を定義する要素です。
  • <render_info_slot> : シェーディングモデルが受け入れる描画情報を定義する要素です。

属性

名前 型と値 説明
value enum
true
false
param_only
anim_only		 変数を編集する UI を表示状態を指定します。
true 編集 UI を表示します。
false 編集 UI を表示しません。
param_only パラメータ用編集 UI を表示し、アニメーション用編集 UI を表示しません。
<sampler_var> の子要素の場合は、true と同じ動作をします。
anim_only パラメータ用編集 UI を表示せず、アニメーション用編集 UI を表示します。
<sampler_var> の子要素の場合は、false と同じ動作をします。

内容

内容はありません。
