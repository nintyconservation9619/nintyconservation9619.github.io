<material_visibility_anim>

説明

<material_info> の visibility をアニメーションする要素です。

この要素は DCC ツールから出力しません。

親要素

  • <per_material_anim> : マテリアルアニメーションの、対象となるマテリアルを指定する要素です。

属性

名前 型と値 説明
base_value float
MIN ～ MAX		 アニメーションの初期値を指定します。
アニメーションが付いている場合には、0 フレームの値を指定します。

内容

アニメーションが無く、base_value を固定値として対象に設定する場合は、内容がありません。

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <step_curve> [0～1] : キーの間を補間しないステップカーブの要素です。
