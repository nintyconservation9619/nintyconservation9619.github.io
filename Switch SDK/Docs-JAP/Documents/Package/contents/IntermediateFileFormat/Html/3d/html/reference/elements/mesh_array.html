<mesh_array>

説明

<mesh> の配列要素です。

親要素

  • <shape> : モデルの形状を定義する要素です。

属性

名前 型と値 説明 DCC
length int
1 ～ 255		 配列内の要素数です。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
DCC ツールからの出力時は、常に 1 を指定します。
固定

内容

以下の要素が順番に出現します。[] 内の数値は要素の出現回数です。

  • <mesh> [1～255] : ポリゴンを定義する頂点インデックスの要素です。
