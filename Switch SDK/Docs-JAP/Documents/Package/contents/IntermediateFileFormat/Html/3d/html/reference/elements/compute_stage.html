<compute_stage>

説明

シェーダーの演算シェーダーステージを定義する要素です。

この要素は DCC ツールから出力しません。

親要素

  • <shading_model> : シェーディングモデルを定義する要素です。

属性

名前 型と値 説明
src_index int
0 ～ 65534		 演算シェーダーのシェーダーソースを定義する <shader_src> のインデックスです。

内容

内容はありません。
