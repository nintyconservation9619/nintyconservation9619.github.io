<original_image>

説明

<texture> の元となる二次元画像データの要素です。

親要素

  • <original_image_array> : <original_image> の配列要素です。

属性

名前 型と値 説明 DCC
index int
0 ～ 65534		 配列内におけるインデックスです。
この値は可読性を向上するためのヒントであり、ツール類で読み込んだ際には無視します。
設定
slice_index int
0 ～ MAX		 <texture_info> の dimension が 3d の場合は
depth のインデックスを指定します。
dimension が 1d_array、2d_array、cube_array の場合は配列のインデックスを指定します。それ以外の場合は 0 を指定します。
固定
face enum
none
positive_x
negative_x
positive_y
negative_y
positive_z
negative_z		 <texture_info> の dimension が cube、cube_array の場合に、 右手座標系でのキューブ面の方向を指定します。それ以外の場合は none を指定します。
none キューブテクスチャーで無い場合に指定します。
positive_x 正の X 方向の面です。
negative_x 負の X 方向の面です。
positive_y 正の Y 方向の面です。
negative_y 負の Y 方向の面です。
positive_z 正の Z 方向の面です。
negative_z 負の Z 方向の面です。

 固定
format enum
rgba8
rgb8
rgba32f
rgb32f		 元画像データのフォーマットを指定します。
rgba8 符号無し 8bit 整数の RGBA フォーマットです。
rgb8 符号無し 8bit 整数の RGB フォーマットです。
rgba32f 実数の RGBA フォーマットです。
rgb32f 実数の RGB フォーマットです。

 設定
width int
1 ～ MAX		 元画像データの幅です。
設定
height int
1 ～ MAX		 元画像データの高さです。
設定
size int
1 ～ MAX		 元画像データの byte 数です。
設定
original_path string
[^*?\\|"'<> ]*		 元画像ファイルへの相対パスです。
ドライブを跨ぐ場合は、拡張子付きのファイル名のみを指定します。
設定
stream_index int
0 ～ MAX		 元画像データを格納した <stream> のインデックスです。
<stream> の type と column は、format に合わせて以下のように指定します。
format type column
rgba8 byte 16
rgb8 byte 12
rgba32f float 4
rgb32f float 3

 設定

内容

内容はありません。
