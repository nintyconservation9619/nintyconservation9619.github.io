NintendoSDK NX Addon 1.7.1 (based on Generic 1.7.1) の変更点

不具合修正

1-7-1S01: (NGC) nmeta の設定によっては、NG ワードチェックが動作しない不具合の修正

NintendoSDK Firmware for NX 4.0.0 環境の場合、nmeta の設定によっては、NGワードチェックが動作しない不具合がありましたので、これを修正しました。

なお、本不具合は、NX Addon 4系より前のバージョンでは使用禁止の nmeta の設定をした場合にのみ発生する不具合です。
