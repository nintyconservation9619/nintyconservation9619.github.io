NintendoSDK NX Addon 1.6.0 (based on Generic 1.6.0) の変更点

改善

1-6-0S01: (GFX) nvn 版で GetGpuAddress の結果をキャッシュしないように変更

NX Addon 1.5.0 にて、nvn 版で nn::gfx::Buffer::GetGpuAddress が呼ばれたときに nvn をコールしないようにし、事前取得された結果を返すように変更しましたが、この変更を元に戻しました。

1-6-0S02: (TOOL) AuthoringEditor 上でのレーティング情報の入力項目を改善

レーティング情報の入力を分かりやすい形に改善しました。
AuthoringEditor 上で仕向け地を選択すると、その地域で必要なレーティングの入力項目のみが表示されます。

不具合修正

1-6-0S03: (3Dツール) 3D シェーダーコンバーターで、Windows7 環境での実行時に応答がなくなることがある不具合の修正

3D シェーダーコンバーター (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dShaderConverter.exe) で、Windows7 環境での実行時、--shader-cache-directory オプションを使用しているときに応答がなくなることがある不具合を修正しました。

NintendoWare NX Addon 1.6.0 (based on Generic 1.6.0) の変更点

互換性の失われる仕様変更

★1-6-0W01: (3D-G3D) バイナリーリソースファイルのフォーマット更新

nn::gfx の構造体変更に追従するためにバイナリーリソースフォーマットを更新しました。これにより、以前のバージョンの 3D バイナリーコンバーター(%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dBinaryConverter.exe)で作成したバイナリーリソースファイルとの互換が切れますので、リソースの再コンバートが必要です。

改善

1-6-0W02: (3Dツール) 3D バイナリーコンバーターから出力されるバイナリーファイルサイズの削減

3D バイナリーコンバーター (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dBinaryConverter.exe) から出力されるバイナリーデータに不要な領域が含まれていたため、それらを削除しました。この変更により nn::g3d::ResMesh と nn::g3d::ResVertex の情報を持つバイナリーのファイルサイズが削減される場合があります。

不具合修正

1-6-0W03: (オーディオ-ATK) ストリームサウンドのループ終端付近に StartOffset を指定した場合にアサートが発生する不具合の修正

ループする波形を使用するストリームサウンドにおいて StartOffset を指定し、設定されているループ範囲の終端付近が再生開始位置となった場合 、アサートが発生することがある不具合を修正しました。

1-6-0W04: (3Dツール) 中間ファイルオプティマイザーのポリゴンリダクション機能を用いたモデル中間ファイルが 3D バイナリーコンバーターで変換に失敗する不具合の修正

中間ファイルオプティマイザー(%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dIntermediateFileOptimizer.exe) のポリゴンリダクション機能を用いて出力したモデル中間ファイル (.fmdb) が 3D バイナリーコンバーター (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dBinaryConverter.exe) で変換に失敗する不具合があり、これを修正しました。
