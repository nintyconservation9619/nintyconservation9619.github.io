NintendoSDK NX Addon 0.12.14 (based on Generic 0.17.14) の変更点

機能追加

13-15S01: (TOOL) AuthoringEditor で .nsp ファイルの概要表示に対応

AuthoringEditor に .nsp ファイルの概要表示機能を追加しました。
nsp ファイルが保持する ROM ハッシュ、アプリケーション ID 、使用ミドルウェアといった概要情報の表示が可能です。

NintendoWare NX Addon 0.12.14 (based on Generic 0.17.14) の変更点

不具合修正

13-15W01: (エフェクト-VFX) VFX ランタイムで、寿命ランダムを使用した際にパーティクルが途中で消える不具合の修正

VFX ランタイムで、寿命ランダム機能を使用した場合に、まれに放出中のパーティクルが途中で消える不具合を修正しました。カスタム処理でパーティクルの Kill() を呼び出している場合、特に頻繁に発生する傾向があります。
原因はランタイム内部のパーティクル生成に関する最適化処理でしたが、現状効果の薄い最適化になっている状態でしたので、この最適化を取りやめることで修正を行いました。
