NintendoSDK NX Addon 0.12.7 (based on Generic 0.17.7) の変更点

改善

13-8S01: (NGC) nn::ngc::ProfanityFilter::Finalize() の追加

nn::ngc::ProfanityFilter のインスタンスを終了するための関数 nn::ngc::ProfanityFilter::Finalize() を追加しました。
nn::ngc::ProfanityFilter::Finalize() を実行せずに nn::ngc::ProfanityFilter インスタンスを破棄した場合はデストラクタ内で自動的に nn::ngc::ProfanityFilter::Finalize() が実行されます。

不具合修正

13-8S02: (NGC) 複数の nn::ngc::ProfanityFilter インスタンスで NGC の機能を使用した時に不正なメモリにアクセスすることがある問題の修正

複数の nn::ngc::ProfanityFilter インスタンスを使用する時、最初のインスタンスに渡したワークバッファ領域を2つ目以降のインスタンスが使用する不具合があったため、これを修正しました。

NintendoWare NX Addon 0.12.7 (based on Generic 0.17.7) の変更点

改善

13-8W01: (エフェクト-VFX) vfx ランタイムが内部で生成するパーティクル用のアトリビュートバッファにインデックス属性を追加

vfx ランタイムが内部で生成するパーティクル用のアトリビュートバッファをインデックスバッファとしても利用できるようにインデックス属性を付加しました。
