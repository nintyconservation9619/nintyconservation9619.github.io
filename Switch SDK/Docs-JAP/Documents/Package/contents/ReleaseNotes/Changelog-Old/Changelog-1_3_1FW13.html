NintendoSDK NX Addon 1.3.1 (based on Generic 1.3.1) FW 2.0.0-13 の変更点

不具合修正

★1-3-1FW13S01: (BLUETOOTH) Joy-ConのHomeボタンでスリープ復帰できない不具合の修正

インターネット設定をした本体をスリープさせると、取り外したJoy-ConのHomeボタンでスリープ復帰できないことがある不具合を修正しました。

1-3-1FW13S02: (Display) 起動時、Switch ロゴが消えた後暗転状態でハングすることがある不具合の修正

起動時、Switch ロゴが消えた後暗転状態でハングすることがある不具合を修正しました。

1-3-1FW13S03: (General) ニンテンドーアカウント連携済みのアカウントがある状態で本体を起動してから24時間以上経過すると 2162-0002 の Fatal Error が発生することがある不具合の修正

ニンテンドーアカウント連携済みのアカウントがある状態で本体を起動してから24時間以上経過すると、下記の Fatal Error が発生することがある不具合があり、これを修正しました。
ErrorCode: 2162-0002
Program: 0x0100000000000032
