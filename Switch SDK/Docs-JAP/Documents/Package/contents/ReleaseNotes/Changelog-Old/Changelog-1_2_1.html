1-2-1S01: (FS) FsAccessLogChecker を version 4.0 に更新

FsAccessLogChecker (%NINTENDO_SDK_ROOT%\Tools\FsAccessLogChecker\FsAccessLogChecker.exe) version 4.0 にて、以下の対応を行いました。

  • nn::fs::MountSaveDataReadOnly() のアクセスログに対応
  • セーブデータアクセスの分類を "Nand" に変更
  • 不正なアクセスログの解析失敗メッセージを改善
