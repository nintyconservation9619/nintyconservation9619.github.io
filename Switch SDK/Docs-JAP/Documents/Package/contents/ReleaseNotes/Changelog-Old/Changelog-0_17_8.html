NintendoSDK NX Addon 0.12.8 (based on Generic 0.17.8) の変更点

不具合修正

13-9S01: (AUDIO) nn::audio::SetDelayInputOutput() の不具合修正

nn::audio::DelayType 利用時に nn::audio::SetDelayInputOutput() による入出力切り替えを行った場合に、オーディオレンダラの挙動が不安定になり、システムが停止する不具合がありましたが、これを修正しました。

この不具合は nn::audio::DelayType を追加後に nn::audio::SetDelayInputOutput() による入出力が設定されていない状態でnn::audio::RequestUpdateAudioRenderer() が呼ばれると、発生することがありました。

またこの修正で nn::audio::SetDelayInputOutput() 呼び出し時には、nn::audio::DelayType の状態がリセットされるようになります。
nn::audio::DelayType の動作中に nn::audio::SetDelayInputOutput() 呼び出した場合、波形にノイズが混じる可能性がありますのでご注意ください。

13-9S02: (FS) FsAccessLogChecker での書き込み容量算出の不具合修正

FsAccessLogChecker (%NINTENDO_SDK_ROOT%\Tools\FsAccessLogChecker\FsAccessLogChecker.exe) に、誤って nn::fs::SetFileSize() で指定されたサイズも書き込み容量として計上する不具合があり、これを FsAccessLogChecker version 1.2 で修正しました。
