NintendoSDK NX Addon 0.12.12 (based on Generic 0.17.12) の変更点

機能追加

13-13S01: (General) DevMenu へのグローバルIPアドレス、NATタイプ、MAC アドレス表示機能追加

DevMenu で、グローバル IP アドレス、NAT タイプ、MAC アドレスを Network メニューより表示できるようにしました。

改善

★13-13S02: (HID) NFC の通信特性改善のための Joy-Con FW の更新

NFC の通信特性の改善を行いました。
この変更を反映するには Joy-Con (R) FW をバージョン 3.48.3.0b に更新する必要があります。

13-13S03: (OE) 廃止予定性能のビルド時の警告の追加

以下の廃止予定性能のビルド時に警告を出力するようにしました。

  • nn::oe::PerformanceConfiguration_Cpu1020MhzGpu307MhzEmc1600Mhz
  • nn::oe::PerformanceConfiguration_Cpu1020MhzGpu384MhzEmc1600Mhz


不具合修正

13-13S04: (TOOL) InitializeSdev で SDEV MP の初期化に失敗することがある不具合の修正

InitializeSdev で SDEV MP を初期化しようとした場合、「[ERROR] Write serial number to HostBridge failed.」と表示され失敗することがある不具合があり、これを修正しました。

13-13S05: (TOOL) リソースデータに含めるエントリの名前によって nsp 作成に失敗する不具合の修正

名前にマルチバイト文字を含んだエントリがリソースデータに含まれていると nsp 作成に失敗する不具合があり、これを修正しました。

NintendoWare NX Addon 0.12.12 (based on Generic 0.17.12) の変更点

不具合修正

13-13W01: (オーディオ-ATK) ロード遅延やポーズ時、ストリームサウンドで再生位置のサンプル数が巻き戻る不具合の修正

ロード遅延やポーズ時、 nn::atk::StreamSoundHandle::GetPlaySamplePosition の返り値が、正しい値より 10000 サンプル程度減少した値を返す場合がある不具合を修正しました。
