NintendoSDK NX Addon 0.12.10 (based on Generic 0.17.10) の変更点

機能追加

★13-11S01: (FRIENDS) マイページアプレット呼び出しライブラリの追加

マイページアプレットに関する以下の API を追加しました。

- nn::friends::ShowFriendList()
- nn::friends::ShowUserDetailInfo()
- nn::friends::StartSendingFriendRequest()
- nn::friends::ShowMethodsOfSendingFriendRequest()
- nn::friends::StartFacedFriendRequest()
- nn::friends::ShowReceivedFriendRequestList()
- nn::friends::ShowBlockedUserList()

アプレットの詳細については ライブラリアプレット／マイページアプレット をご覧ください。

★13-11S02: (General) SystemSoftwareKit の統合

SystemSoftwareKit に含まれていた内容を NintendoSDK NX Addon および NintendoSDK SystemUpdater for NX パッケージに統合しました。
これ以降、開発機をを初期化した際には標準で SystemSoftware が含まれるようになります。

★13-11S03: (General) 最終動作確認環境（PROD MODE) でインフラ通信が使えるようにするための対応

最終動作確認環境（PROD MODE) でインフラ通信を使えるようにするため、DevMenuApp に Debug > Network Service を Enabled に設定する機能を追加しました。

★13-11S04: (HID) コントローラサポートアプレット呼び出しライブラリの追加

コントローラサポートアプレットに関する以下の API を追加しました。

- nn::hid::ShowControllerSupport()
- nn::hid::SetExplainText()

アプレットの詳細については ライブラリアプレット／コントローラーサポートアプレット をご覧ください。

★13-11S05: (SWKBD) ソフトウェアキーボード呼び出しライブラリの追加

nn::swkbd 名前空間にソフトウェアキーボードに関する API を追加しました。
アプレットの詳細については ライブラリアプレット／ソフトウェアキーボードアプレット をご覧ください。
また、サンプルとして SwkbdSimple サンプルを追加しました。

★13-11S06: (WEB) Web アプレットおよびオフライン Web アプレット呼び出しライブラリの追加

nn::web 名前空間に Web アプレットおよびオフライン Web アプレットに関する API を追加しました。

各アプレットの詳細は ライブラリアプレット／Webアプレット 、ライブラリアプレット／オフラインWebアプレット をご覧ください。

★13-11S07: (アカウント) アカウント管理アプレット呼び出しライブラリの追加

アカウント管理アプレットに関する以下の API を追加しました。
また、サンプルとして AccountApplication サンプルを追加しました。

- nn::account::ShowUserSelector()
- nn::account::ShowUserCreator()

アプレットの詳細については ライブラリアプレット／アカウント管理アプレット をご覧ください。

13-11S08: (NIFM) プロキシ設定取得 API の追加

利用中のネットワーク接続に指定されているプロキシの設定を取得する nn::nifm::GetCurrentProxySetting() を追加しました。

改善

13-11S09: (FS) Devmenu - FS Speed Emulation Mode の OFF と Faster の結合

現状の FS の{{DocLink=177646410}}速度エミュレーション機能{{DocLink}}では OFF と Faster mode は同じ挙動であるため、{{DocLink=107328771}}DevMenu{{DocLink}} - Debug - FS Speed Emulation Mode の "OFF" と "Faster" を "OFF / Faster" に結合しました。将来、OFF より Faster mode の方が速くなる予定のため、その際には "OFF と "Faster" へ分離する予定です。

不具合修正

13-11S10: (AUDIO) nn::audio::SetReverbInputOutput() の不具合修正

nn::audio::ReverbType 利用時に nn::audio::SetReverbInputOutput() による入出力切り替えを行った場合に、オーディオレンダラの挙動が不安定になり、システムが停止する不具合がありましたが、これを修正しました。

この不具合は nn::audio::ReverbType を追加後に nn::audio::SetReverbInputOutput() による入出力が設定されていない状態でnn::audio::RequestUpdateAudioRenderer() が呼ばれると、発生することがありました。

またこの修正で nn::audio::SetReverbInputOutput() 呼び出し時には、nn::audio::ReverbType の状態がリセットされるようになります。
nn::audio::ReverbType の動作中に nn::audio::SetReverbInputOutput() 呼び出した場合、波形にノイズが混じる可能性がありますのでご注意ください

13-11S11: (MOVIE) End of Stream が返る前にDecoder::Stop()を呼び出すと処理が返ってこなくなる不具合の修正

Decoder へ End of Stream を送ってからその結果が返ってくる前にDecoder::Stop() を呼び出すと、処理が返ってこなくなる不具合を修正しました。

13-11S12: (TOOL) AuthoringEditor が出力する meta ファイルのレーティング団体名の誤りを修正

AuthoringEditor が出力する meta ファイルの一部レーティング団体名に誤りがあり、 AuthoringTool への入力時にエラーが発生する不具合があったため修正しました。
名称の修正を行ったレーティング団体は次の通りです。

  • (誤) PEGI Portugal → (正) PEGIPortugal
  • (誤) PEGI+BBFC → (正) PEGIBBFC


13-11S13: (TOOL) BackupSaveData でジャーナルサイズよりセーブデータ内のデータ総サイズが大きい場合にインポートエラーが発生する不具合の修正

BackupSaveData でジャーナルサイズよりセーブデータ内のデータ総サイズが大きい場合にインポートエラーが発生する不具合を修正しました。

ただしジャーナルサイズよりファイルサイズが大きいファイルが保存されたセーブデータのインポートエラーは引き続き発生します。追って修正予定です。
