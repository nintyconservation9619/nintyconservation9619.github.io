NintendoSDK NX Addon 0.12.11 (based on Generic 0.17.11) の変更点

機能追加

★13-12S01: (OE) 製品で使用できる性能（nn::oe::PerformanceConfiguration）の変更

nn::oe::SetPerformanceConfiguration で設定できる性能を変更しました。

  • 開発環境のみで使用可能であった以下の性能が製品環境で使用可能になりました。
    • nn::oe::SetPerformanceConfiguration_Cpu1020MhzGpu384MhzEmc1331Mhz
  • 製品環境または開発環境で使用可能であった以下の性能を近い将来に使用不可能にすることを予告します。
    • nn::oe::SetPerformanceConfiguration_Cpu1020MhzGpu307MhzEmc1600Mhz
    • nn::oe::SetPerformanceConfiguration_Cpu1020MhzGpu384MhzEmc1600Mhz


13-12S02: (DIAG) Log Viewer に、NX Binary Log 形式をテキスト形式に変換する機能を追加

Menu > Convert NX Binary Log から、変換対象の NX Binary Log と、テキストファイルの出力先を選択することで、ファイル形式を変換できます。

不具合修正

13-12S03: (DIAG) Log Viewer で、NX Binary Log 形式のファイルを開いたときに、不正なタイムスタンプが表示される不具合を修正

NX Binary Log には、タイムスタンプが含まれませんが、誤ってファイルを開いたときの時刻をタイムスタンプとして表示していたため、表示されないように修正しました。

13-12S04: (DIAG) Log Viewer で、巨大な NX Binary Log 形式のファイルを開いたときに、異常終了する不具合を修正

1 GB を超えるような、巨大な NX Binary Log 形式のファイルを開いたときに、Log Viewer が操作を受け付けなくなった後、メモリ不足で異常終了する不具合を修正しました。

13-12S05: (MOVIE) 動画再生中に movie API から処理が返らなくなる不具合の修正

動画再生中、たまにmovie API から処理が返らなくなることがあり、その不具合を修正しました。

13-12S06: (TOOL) BackupSaveData でジャーナルサイズよりファイルサイズが大きいファイルが保存されたセーブデータのインポートがエラーになる不具合の修正

BackupSaveData でジャーナルサイズよりファイルサイズが大きいファイルが保存されたセーブデータをエクスポート後インポートするとエラーになる不具合を修正しました。

NintendoWare NX Addon 0.12.11 (based on Generic 0.17.11) の変更点

不具合修正

13-12W01: (レイアウト-UI2D) LayoutEditor で、長い時間がかかるバイナリ変換を行うプレビュー処理で、意図せずタイムアウトと判定され、プレビューに失敗する不具合を修正

LayoutEditor で、長い時間がかかるバイナリ変換を行うプレビュー処理で、意図せずタイムアウトと判定され、プレビューに失敗する不具合があり、これを修正しました。
