5-3-0W63: (レイアウトツール) LayoutEditor で整列ペイン機能のデフォルトが「無効」になっている不具合を修正

LayoutEditor で、整列ペイン機能のデフォルトが「無効」となっており、プロジェクト設定ファイル で「有効」にしなければ利用出来ない不具合を修正しました。

5-3-0W64: (レイアウトツール) LayoutEditor で部品の上書きを使用時にテクスチャコンバイナのプロパティが正しく切り替わらない不具合を修正

LayoutEditor で部品の上書きを使用時にテクスチャコンバイナのプロパティが正しく切り替わらない不具合を修正しました。

5-3-0W65: (レイアウトツール) LayoutEditor のカーブエディタで Undo を行ったときにキーの傾きが元に戻らない不具合を修正

LayoutEditor のカーブエディタで Undo を行ったときにキーの傾きが元に戻らない不具合を修正しました。

5-3-0W66: (レイアウトツール) LayoutEditor のテクスチャウインドウで詳細表示を選択したときにテクスチャの右端が表示されない不具合を修正

LayoutEditor のテクスチャウインドウで詳細表示を選択したときにテクスチャの右端が表示されない不具合を修正しました。

5-3-0W67: (レイアウトツール) LayoutViewer 実機版で、タッチ操作によるボタンの挙動確認が正しく動作しない不具合を修正

LayoutViewer 実機版で、タッチ操作によるボタンの挙動確認が正しく動作しない不具合があり、これを修正しました。

5-3-0W68: (レイアウトツール) キャプチャペインの上書き設定が正しく読み込まれない不具合を修正

LayoutEditor へデータを読み込む際にキャプチャペインの上書き設定が正しく読み込まれない不具合があり、これを修正しました。
また、LayoutEditor 内でのキャプチャペインの上書き設定を含むパーツのコピー＆ペースト時にも上書き設定が正しくコピーされない不具合も同時に修正しました。

5-3-0W69: (エフェクト-VFX) EffectMaker モデル連携描画機能の改善

EffectMaker モデル連携描画機能にて、読み込ませる fmdb ファイルのマテリアルの状態によっては正しく描画されない問題の改善をおこないました。

5-3-0W70: (エフェクト-VFX) nn::vfx::viewer::ViewerSystemで発生するメモリリークを修正

ツールコネクタクラスの開放が行われていなかったので修正しました。また、同クラスの生成をvfxヒープから行うようにしました。

5-3-0W71: (SPY) Spy.exe でラウドネス値が不正になる不具合の修正

Spy.exe で長時間の波形の記録を行うと、ラウドネスメーターパネルのラウドネス値(INT)が不正になる不具合を修正しました。

Spy.exe で保存されたデータのフォルダが、他のデータのサブフォルダになる場合がある不具合を修正しました。
