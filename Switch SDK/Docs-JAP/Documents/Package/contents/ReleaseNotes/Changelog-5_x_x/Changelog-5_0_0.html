★5-0-0W01: (オーディオ-ATK) ストリームサウンドバイナリのコンバート仕様の変更

ストリームサウンド用の波形ファイルにリージョン情報とループ情報が両方設定されていた場合、これまではループ情報が無効化される仕様でしたが、今回の変更後はリージョン情報とループ情報の両方が出力されるようになります。

具体的には、ループ情報を持つストリームサウンドを再生するときにリージョンコールバックを指定した場合には、アサートで停止します。
