5-1-0W96: (レイアウトツール) LayoutEditor の部品パレットの更新時に部品が見つからなかったときに異常終了する不具合を修正

LayoutEditor の部品パレットの更新時に開かれているレイアウトの部品が見つからなかったときに異常終了する不具合があり、これを修正しました。

5-1-0W97: (レイアウトツール) LayoutEditor の分割モードで拡張ユーザ情報のアニメーションが正しく出力されない不具合を修正
