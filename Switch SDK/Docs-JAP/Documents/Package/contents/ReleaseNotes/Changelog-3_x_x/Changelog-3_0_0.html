nvn に以下の非互換の変更を行いました。

The behavior of nvnWindowBuilderSetNativeWindow has changed on the Windows reference implementation which could potentially break existing Windows NVN applications. In this version of NVN, passing a valid HWND as the native window parameter will cause the driver to create an internal GL context to swap buffers. Applications that create their own "donor" GL context must pass NULL as the native window parameter. If any applications create their own GL context and pass an NVWD as the native window parameter, there could be rendering corruptions as both the internal GL context and the application's GL context might conflict with each other. These applications must be modified to either pass NULL as the native window, or stop creating their own GL context and pass HWND as the native window to let the driver create its own internal GL context. See the API documentation of nvnWindowBuilderSetNativeWindow for more information.

EffectMaker を利用して、EffectViewer や アプリ で再生中 のチャイルドエミッタをリアルタイム編集した際に、一部のパラメータが正しく編集できない不具合を修正しました。

EffectMaker において、NW4F や NW4C で作成した eset ファイルをアップデートしたとき、エミッタセットに付けられるエミッタが上限である 16 個より多く付いてしまうことがあり、この場合はエラーを表示するようにしました。

EffectMaker において、カスタムアクションユーザー定義の数が eset ファイルを保存したときより増えていた場合、eset ファイルを開いたときにカスタムアクションコールバック ID が「コ－ルバックを利用しない」から他の ID に書き換わる問題を修正しました。

3-0-0W106: (エフェクトツール) EffectMaker カラーの長さを維持しない不具合を修正

EffectMaker において RGB モードで各カラーの値を 1 以上に設定し HSV モードに切り替えて色相を編集すると各カラーの値が 1 以下に丸められる不具合を修正しました。

EffectMaker でバイナリコンバートを行ったとき、キャッシュファイルの読み込みに失敗してバイナリコンバートエラーが発生する問題を修正しました。

3-0-0W108: (エフェクトツール) EffectMaker テクスチャプレビューの表示上の不具合を修正

EffectMaker においてフォーマット「srgb_bc7」で、テクスチャプレビューとテクスチャ詳細ウィンドウの見た目が異なる問題について修正しました。
