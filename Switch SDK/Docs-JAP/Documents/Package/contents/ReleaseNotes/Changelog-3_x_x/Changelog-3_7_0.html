NintendoSDK NX Addon 3.7.0 (based on Generic 3.7.0) の変更点

改善

3-7-0S01: (TOOL) AuthoringTool にパッチ間差分情報の表示機能を追加

AuthoringTool に、パッチ間のファイル単位の差分情報を表示する diffpatch コマンドを追加しました。

3-7-0S02: (TOOL) AuthoringTool のパッチ生成処理時間の改善

AuthoringTool が行っているデータ変換処理を最適化し、特にパッチ生成における処理時間を改善しました。

3-7-0S03: (TOOL) パッチ作成時のセーブデータ・一時ストレージ・キャッシュストレージのサイズ更新内容のチェックを改善

パッチによるセーブデータ・一時ストレージ・キャッシュストレージのサイズ更新を行う場合に、一部のケースでパッチ作成時に不正な値の入力や更新ができてしまっていたため、チェックの内容を改善しました。

3-7-0S04: (TOOL) 追加コンテンツ作成時にタグ名の文字コードをチェックする機能の追加

AuthoringTool で追加コンテンツを作成する時に、 nmeta ファイルの Tag に 指定できない文字が設定されている場合にエラーを発生させるようにしました。
詳細は 機能／システム／AOC ライブラリ／追加コンテンツの作成方法 を参照してください。

3-7-0S05: (TOOL) 追加コンテンツ作成時にタグ名の文字列長をチェックする機能の追加

AuthoringTool で追加コンテンツを作成する時に、 nmeta ファイルの Tag に 128 文字を超える文字列を指定するとエラーを発生させるようにしました。
詳細は 機能／システム／AOC ライブラリ／追加コンテンツの作成方法 を参照してください。

不具合修正

3-7-0S06: (FS) FsAccessLogChecker を version 12.3 に更新

FsAccessLogChecker (%NINTENDO_SDK_ROOT%\Tools\FsAccessLogChecker\FsAccessLogChecker.exe) を version 12.3 に更新しました。以下の変更が含まれます。

  • 解析結果の "FS Access Log Details" が入る位置を修正
  • 改行コード CR に対応
  • マルチバイト文字を含む Shift-JIS に対応
  • インデックス付きキャッシュストレージ機能とnn::fs::GetSaveDataSize() のチェックに対応
  • SameAddressRead 項目のファイルに関連しない読み込み操作の出力形式を変更
  • オプション --warning_ratio を追加し、SameAddressRead 項目に閾値を超えていない上位の値を常時出力するよう対応


3-7-0S07: (TOOL) AuthoringTool の list コマンドで削除ファイルが表示されない不具合の修正

AuthoringTool の list コマンドの --patched-only オプションでパッチに含まれるファイルのみを表示する際に、パッチで削除されたファイルが "Listing removed files from original" として表示されない場合が有る不具合を修正しました。

3-7-0S08: (TOOL) AuthoringTool の replace コマンドでファイルサイズが変わるような編集を行った場合にファイル読み込みが遅くなる場合がある不具合の修正

AuthoringTool の replace コマンドでアプリに含まれるリソースデータ及び Html ドキュメントのファイルをファイル単位で編集し、
ファイルサイズが元のサイズから変更された場合、アプリ及びそのアプリを元にして作成しされたパッチにおいて、
リソースデータ及び Html ドキュメントのファイルの読み込みが本来の速度より遅くなる不具合があり、これを修正しました。

またこの修正により、replace コマンドを使用した場合にパッチサイズが意図せず大きくなる事もなくなりました。

3-7-0S09: (TOOL) Nintendo NX CPU Profiler で、サンプリングできない期間がある問題を回避

Nintendo NX CPU Profiler で、サンプリングできない期間がある問題を回避するため、サンプリング方法を旧方式に戻しました。旧方式では、複数コアにアフィニティが設定されているスレッドは観測できないため、プロファイラ利用中は単一コアのみにアフィニティを設定して利用してください。

3-7-0S10: (TOOL) Visual Studio をアンインストールして Help Viewer が残っている場合に Document パッケージのインストールが失敗する不具合の修正

Microsoft Visual Studio をアンインストールして Microsoft Help Viewer が残っている状態で Document パッケージをインストールしようとすると失敗する不具合を修正しました。
