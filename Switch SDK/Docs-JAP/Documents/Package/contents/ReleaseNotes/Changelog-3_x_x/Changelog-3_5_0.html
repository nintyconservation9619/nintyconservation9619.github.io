NintendoSDK NX Addon 3.5.0 (based on Generic 3.5.0) の変更点

不具合修正

3-5-0S01: (TOOL) InitializeEdev で EDEV 複数台接続時に違う本体が初期化される場合がある不具合の修正

InitializeEdev で EDEV 複数台接続時に違う本体が初期化される場合がある不具合を修正しました。

NintendoWare NX Addon 3.5.0 (based on Generic 3.5.0) の変更点

不具合修正

3-5-0W01: (3D-G3D) nn::g3d::ShapeObj::MakeSubMeshRange() の 引数 meshIndex に 0 以外の値を代入していても pRangeArray の lodLevel に 0 が格納されている不具合の修正

nn::g3d::ShapeObj::MakeSubMeshRange() の 引数 meshIndex に 0 以外の値を代入していても pRangeArray の lodLevel に常に 0 が格納されている不具合があり、これを修正しました。

これにより、指定した lodlevel のモデルが描画されるようになるため、描画処理の高速化を行うことが可能です。
しかし、今まで意図せず lodlevel 0 のモデルが描画されていたのに対し、設定を行った lodlevel のモデルが描画されるようになるため描画結果が変化する恐れがあります。

3-5-0W02: (レイアウトツール) LayoutEditor の部品パレットでリロードを繰り返すたびに、使用メモリ量が増加してツールが異常終了してしまう不具合を修正

LayoutEditor の部品パレットでリロードを繰り返すたびに、使用メモリ量が増加してツールが異常終了してしまう不具合があり、これを修正しました。
