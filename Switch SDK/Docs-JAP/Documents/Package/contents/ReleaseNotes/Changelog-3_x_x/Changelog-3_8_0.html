NintendoSDK NX Addon 3.8.0 (based on Generic 3.8.0) の変更点

互換性の失われる仕様変更

★3-8-0S01: (General) Visual Studio 2013 のサポート廃止

事前の予告通り、Visual Studio 2013 のサポートを廃止しました。

★3-8-0S02: (TOOL) AuthoringTool にアクセス可能な URL をチェックする機能を追加

AuthoringTool に --error-unpublishable オプションを設定している場合、アクセス可能な URL に "^https://" と "---- " 以外を指定するとエラーになるように改善しました。
詳細は 機能／ライブラリアプレット／Webアプレット・オフラインWebアプレット／Webアプレット を参照してください。

機能追加

3-8-0S03: (EC) 期間券のコース詳細画面を表示する関数の追加

期間券のコース詳細を表示するための関数 nn::ec::ShowShopSubscriptionList() を追加しました。

改善

3-8-0S04: (TOOL) AuthoringTool に NintendoSDK に含まれるアイコンの使用をチェックする機能を追加

AuthoringTool 実行時に --error-unpublishable オプションを指定した場合のチェック項目を追加しました。
アプリケーションイメージに含めるアイコンに NintendoSDK に含まれるアイコン (NintendoSDK_Application.bmp) を設定しているとエラーを発生するように対応しました。

3-8-0S05: (TOOL) AuthoringTool に脆弱性のある NEX の使用をチェックする機能の追加

AuthoringTool 実行時に --error-unpublishable オプションを指定すると、脆弱性のある NEX を使用している場合にエラーになるように改善しました。
詳細は Nintendo Developer Portal の「技術情報／マスター提出関連情報」を参照してください。

不具合修正

3-8-0S06: (NFP) NfpDebugTool が強制終了する不具合を修正

NfpDebugTool でタグを読み込む際、ツールが強制終了する不具合を修正しました。

3-8-0S07: (TOOL) .NET Framework 4.7.1 をインストールすると AuthoringTool 実行時にエラー終了することがある問題の修正

.NET Framework 4.7.1 もしくは Windows 10 Fall Creators Update（バージョン1709）もしくは .NET Framework 4.7.1 をインストールすると AuthoringTool 実行時にエラー終了することがあり、これを修正しました。

3-8-0S08: (TOOL) AuthoringTool で小さなサイズのアイコンにプログレッシブ Jpeg を指定できることがある不具合の修正

AuthoringTool でアプリケーションイメージを作成する時に、小さなサイズのアイコンにプログレッシブ Jpeg を指定できてしまう不具合があり、これを修正しました。
詳細は 機能／ビルド／アプリケーションイメージの作成／アプリケーションイメージに含めるアイコンについて を参照してください。

3-8-0S09: (TOOL) NintendoSDK アプリケーション・ライブラリプロジェクトのSDKルート選択ページに関する動作の調整

Visual Studio 上にて「NintendoSDK アプリケーション」「NintendoSDK ライブラリ」プロジェクトを作成するためのウィザード表示にて"SDK の位置を指定する" のページで "次へ" ボタンがページ表示準備が終わる前にクリックすることができた不具合を修正しました。

さらにSDKルートとして不正なパスを指定した場合は"次へ"ボタンを押下してもページ移動できないようにしました。
