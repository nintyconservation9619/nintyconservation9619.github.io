NintendoSDK NX Addon 3.3.0 (based on Generic 3.3.0) の変更点

互換性の失われる仕様変更

★3-3-0S01: (IRSENSOR) 対応するコントローラファームのバージョンを Firmware 3.48.3.b に巻き戻し

NXAddon 3.0.0～NXAddon 3.2.0 まで、コントローラファームの更新を行うことで、コントローラに関連する不具合修正、機能追加を行っていましたが、NXAddon 3.3.0 より対応するコントローラファームを古いものにました。
対応するファームウェアバージョンは、 Firmware 3.48.3.b となります。
これに伴い、NXAddon 3.2.0 から、以下の機能が巻き戻りますので、注意が必要です。
■ モーションIRカメラ

  • モーメント、クラスタリングモード時のデータ遅延を2フレーム程度改善
  • ジェスチャライブラリの更新
  • モーションIRカメラの起動後、数フレーム間設定が反映されない不具合の修正
  • モーションIRカメラ使用中にコントローラ切断後、再接続した後にデータロス率が上がってしまう不具合の修正
  • プロセッサ動作中にモード、パラメータを高速に切り替える機能を追加
  • コントローラのファームウェア更新の必要性をモーションIRカメラを起動する前に確認できる機能の追加

■ その他

  • Joy-Con (R) のファームウェアアップデート速度の改善


なお、これらの削除された機能については将来の SDK にて改めて追加される予定となっています。

機能追加

3-3-0S02: (BCAT) データ配信機能の実践的な利用サンプルの追加

データ配信機能の実践的な利用サンプルを追加しました。
利用上の注意点の記述や、適切なエラーハンドリングが行われているサンプルです。

3-3-0S03: (DevMenu) アプリケーションに起動パラメータを渡す開発用機能を追加

SD カードにインストールしたファイルを、起動パラメータとして、アプリケーションに渡す機能を追加しました。

詳しくは、DevMenu のマニュアル から、「起動パラメータのインストール、アンインストール」を参照してください。

3-3-0S04: (WEB) JS 拡張を有効にする機能を追加

Web アプレット / オフライン Web アプレットで JS 拡張を有効にする以下の API を追加しました。

  • nn::web::ShowWebPageArg::SetJsExtensionEnabled()
  • nn::web::ShowOfflineHtmlPageArg::SetJsExtensionEnabled()


3-3-0S05: (WEB) URL 表記をドメイン名に変更する API を追加

Web アプレットで URL 表記をドメイン名に変更する以下の API を追加しました。

  • nn::web::ShowWebPageArg::SetDisplayUrlKind()


3-3-0S06: (WEB) ポインター操作を無効にする機能を追加

Web アプレットでポインター操作を無効にする以下の API を追加しました。

  • nn::web::ShowWebPageArg::SetPointerEnabled


3-3-0S07: (WEB) 動画専用起動を行う為の機能を追加

オフライン Web アプレットで動画専用起動を行う為の以下の API を追加しました。

  • nn::web::ShowOfflineHtmlPageArg()


改善

★3-3-0S08: (AOC) 追加コンテンツ nsp の暗号化鍵の変更

NX Addon 3.2より、追加コンテンツ nsp が内部的に使用している暗号化鍵の変更を行いました。

過去に Nintendo Developers Portal に提出済みの追加コンテンツを NX Addon 3.X 系でリビルドを行うと、作成されたコンテンツのハッシュが変更されます。同一バージョンで内容が異なるコンテンツが登録されることになり、エラーとなりますので注意してください。

NX Addon 3.X で作成された追加コンテンツを提出する場合には、同一 nsp にそれ以前の SDK で提出した更新のないコンテンツを含めないようにしてください。

3-3-0S09: (UTIL) コンパイラオプション -Wfloat-conversion を有効にした場合に nn/util/detail/util_MatrixApi.neon.h から警告が発生する問題の修正

コンパイラオプション -Wfloat-conversion を有効にした場合に nn/util/detail/util_MatrixApi.neon.h から警告が発生する問題を修正しました。

不具合修正

3-3-0S10: (APPLET) 一部のアプリをメニューから終了した際に正常終了できなくなっていた不具合を修正

NX Addon 3.0.0 以降のファームウェアを使用した際に、一部のアプリを HOME メニューや DevMenu から終了した際に正常終了できなくなっていた不具合を修正しました。

3-3-0S11: (AUDIO) 複数個のオーディオレンダラを同時に使用した時に正しく DSP リソース配分が効かない事がある不具合の修正

複数個のオーディオレンダラを同時に使用した時、正しく DSP リソース配分が効かない事がある不具合の修正があり、これを修正しました。

具体的には、アプリケーションが複数個のオーディオレンダラを同時に利用している場合、かつ、オーディオフレームあたりの DSP 処理時間の合計値が 3500 us を超えている場合に、3500 us 以上使えてしまうことがありました。

3-3-0S12: (FS) FsAccessLogChecker version 8.1 でマウント操作のカウントを修正

FsAccessLogChecker (%NINTENDO_SDK_ROOT%\Tools\FsAccessLogChecker\FsAccessLogChecker.exe) でアクセスログのマウント対象を誤って識別して読み出し操作をカウントすることがある不具合があり、これを FsAccessLogChecker version 8.1 で修正しました。

3-3-0S13: (HID) UnsetVibrationDeviceHandleを実行時に振動を停止する処理を追加しました。

UnsetVibrationDeviceHandleを実行時に、振動ハンドルに対して、振動を停止する処理を追加しました。

3-3-0S14: (HID) 無線通信状態でモーション IR センサー利用中に本体レールジョイントすると IR が On のままになる不具合を修正

NxAddon 3.1.0 以降でモーション IR センサーを使用中に右コンをレールに接続しても IR がオフにならないという不具合が含まれていることがわかったため修正を行いました。

3-3-0S15: (LDN) nn::ldn::DefaultLocalCommunicationId の定義変更

nn::ldn::DefaultLocalCommunicationId は 0 として定義されていましたが、この SDK で 0xFFFFFFFFFFFFFFFF に変更されました。

この変更に伴って、NX Addon 3.1.0 や NX Addon 3.2.0 でビルドした nn::ldn::DefaultLocalCommunicationId を使用するアプリケーションが NX Addon 3.3.0 以降に含まれる FW では正しく動作しなくなる可能性があります。NX Addon 3.3.0 でビルドしなおしてください。

3-3-0S16: (TOOL) AuthoringEditor の一部レーティング機関の UI 表示が正しくない不具合を修正

AuthoringEditor において PEGIBBFC レーティングの選択肢が ドキュメントの nmeta の書き方 にあるリストより少なくなっている不具合を修正しました。

3-3-0S17: (TOOL) AuthoringEditor 上での nsp 比較時、Web アプレットからアクセス可能な URL リストの比較結果が正しくない不具合の修正

AuthoringEditor 上からアプリケーション nsp 同士を比較した際、 Web アプレットからアクセス可能な URL リスト (nmeta の AccessibleUrlsFilePath 要素) の比較結果が正しくない不具合を修正しました。

3-3-0S18: (TOOL) 中間ファイル関連のツールに正しく中間ファイルが読み込めない不具合の修正

NintendoWare パッケージをインストールしていないときに、中間ファイル関連のツール(%NINTENDO_SDK_ROOT%\Tools\Graphics\GraphicsTools\IntermediateFile~.exe)に中間ファイルを読み込むとエラーが発生したり、正しく中間ファイルを読み込むことができない不具合があり、これを修正しました。

NintendoWare NX Addon 3.3.0 (based on Generic 3.3.0) の変更点

不具合修正

3-3-0W01: (レイアウト-UI2D) 「テクスチャだけ上書き」の機能によって設定したテクスチャが未使用扱いになる不具合の修正

「テクスチャだけ上書き」の機能によって設定したテクスチャが未使用扱いになる不具合を修正しました。

3-3-0W02: (レイアウトツール) LayoutConverter で、-u オプションを指定しても、変更のないテクスチャが更新される不具合の修正

LayoutConverter で、-u オプションを指定しても、変更のないテクスチャが更新される不具合があり、修正しました。従来は LayoutConverter が出力する ftxb ファイルに <file_info> の項目が含まれていましたが、これを含めないようにしました。

3-3-0W03: (SPY) データを閉じると再生ボタンが有効になる不具合の修正

Spy.exe でデータファイルを開いたりアプリケーションからデータを受信した後、「すべて閉じる」ボタンでデータを閉じると、「再生」ボタンが有効になってしまう不具合を修正しました。
