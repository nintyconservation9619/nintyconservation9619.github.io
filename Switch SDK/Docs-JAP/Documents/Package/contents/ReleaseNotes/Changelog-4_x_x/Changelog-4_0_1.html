NintendoSDK NX Addon 4.0.1 (based on Generic 4.0.1) の変更点

不具合修正

4-0-1S01: (General) アプリケーションの言語に中国語（簡体字）と中国語（繁体字）を指定したときに正しく反映されない不具合の修正

NX Addon 4.0 では、中国語（簡体字）と中国語（繁体字）において、以下の３点の不具合がありました。

  • 中国語（簡体字）と中国語（繁体字）のアイコンが正しく取得出来ない
  • 中国語（簡体字）と中国語（繁体字）の設定が逆になって取得される
  • 中国語（簡体字）と中国語（繁体字）の共有フォントが取得できない


NX Addon 4.0.1 では、これらの不具合が修正されています。

NintendoWare NX Addon 4.0.1 (based on Generic 4.0.1) の変更点

不具合修正

4-0-1W01: (レイアウト-UI2D) 「カラー補間だけ上書き」と「テクスチャだけ上書き」の両方を設定すると未使用テクスチャの判定がおかしくなる不具合の修正

「カラー補間だけ上書き」と「テクスチャだけ上書き」の両方を設定すると未使用テクスチャの判定がおかしくなる不具合を修正しました。

4-0-1W02: (レイアウト-UI2D) 詳細なコンバイナモードを設定したペインと、３枚以上のテクスチャを設定したペインの含まれるレイアウトファイルをランタイムで利用する場合に正しく処理が行われない問題を修正

詳細なコンバイナモードを設定したペインと、３枚以上のテクスチャを設定したペインの含まれるレイアウトファイルをランタイムで利用する場合に正しく処理が行われない問題があり、これを修正いたしました。

4-0-1W03: (レイアウトツール) LayoutEditor のテクスチャコンバイナの表示が崩れる不具合の修正

LayoutEditor のテクスチャコンバイナの表示が崩れアルファ合成設定が見切れる不具合があり、これを不具合の修正しました。

4-0-1W04: (レイアウトツール) ワークスペースの切り替え後にカーブエディタを開くと以上終了する不具合の修正

ワークスペースの切り替え後にカーブエディタを開くと以上終了する不具合があり、これを修正しました。
