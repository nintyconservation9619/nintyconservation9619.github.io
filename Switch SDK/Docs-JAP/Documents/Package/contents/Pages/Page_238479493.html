派生部品機能を利用する

使い方手順

派生部品機能の使い方について説明します。

派生部品レイアウトを作る

  1. 部品パレットのリストから、部品レイアウトを選択
  2. 右クリックメニューから、「派生部品を作る」を実行
  3. 選択部品を派生元にした、新規派生部品レイアウトが作成される

ペインを追加する

  1. 新規派生部品レイアウトを開く。
  2. 通常通り、ペインを作成する。( 例 : [ メニュー ] → [ ペイン ] → [ ペイン作成 ] )

派生元ペインにアニメを追加する

  1. 派生部品レイアウトで、派生元ペインを選択する。
    ( レイアウトウィンドウでは選択不能なので、親子階層ウィンドウから選択します )
  2. カーブエディタを開く。
  3. 黒色で表示されたカーブは、派生元部品で設定済みのアニメーション。項目ツリーには赤い錠前アイコンが表示れている。改変や削除はできない。
  4. 設定済みカーブ以外にキーを追加する

追加したペインのアニメを元部品のアニメと連動させる

  1. 前述の手順で、ペインを追加し、アニメーションを設定する。
  2. 新規にアニメーション区間タグを追加する。
    元部品の機能アニメーションに対応する区間タグをベースに複製するのが便利 ( [ リストで区間タグを選択 ] → [ 選択を複製ボタン ] ) 。
  3. 新規にアニメーション区間タグの対象グループを追加する。このグループには追加したペインを含める
  4. 追加ペインを含むグループを区間タグの対象グループに設定する
  5. 部品・コントロール設定ウィンドウを開き、機能アニメーションで先ほどの新規区間タグを指定しなおす。
  6. 元部品のアニメーションと一緒に、追加ペインのアニメーションが再生される部品が完成

派生元部品を編集する

  1. 元部品の編集を完了し、ファイル保存をします。
  2. 先ほど編集をした部品の派生部品について、部品パレットの派生元カラムにチェックマークが表示されます。
    これは、元部品が変更され再度保存しなおす必要があることを示しています。
  3. チェックの追加派生部品を開いて、再度ファイル保存しなおします。
    この操作で元部品の編集が派生部品に反映されます。
    部品パレットのリロードだけでは十分ではない点にご注意ください。
  4. 派生部品を利用した、ボディレイアウトでも元部品の編集が正しく反映されるようになります。
