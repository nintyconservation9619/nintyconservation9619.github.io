グループウインドウ

 グループウインドウ

グループウインドウの各部位の名称と役割について説明します。

グループ機能は関連のある複数のペインをまとめる機能です。
以下のような、親子階層では表現しにくい集団を表現するときに使用します。

  • 複数の階層をまたぐようなペインの集団をまとめる場合
  • ペインが同時に複数の集団に所属しているような場合

レイアウトの中にあるグループを確認するには、グループウィンドウを使用します。

グループウインドウ 操作以下の項目
A
  • 切り取り
  • コピー
  • 貼り付け
  • 削除
 グループのコピー & ペーストを行います。
  • 選択グループの編集
  • グループ解除・グループから除外
  • 既に設定済みのグループ内容を編集します。
  • 選択グループに対してグループを解除（削除）します。
    選択ペインに対してグループから除外します。
  • 選択をロック
  • 選択をロック解除
  • 選択を隠す
  • 選択を表示
 グループのロックや、表示、非表示を設定します。
  • 選択ペインを自動的に表示
 選択ペイン表示の「自動 / 手動」を切り替えます。
B
階層を開きます。
C
階層を閉じます。
