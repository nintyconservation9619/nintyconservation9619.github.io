表示

表示に関する操作を行います。

A オブジェクトビューのサブメニューを表示します。
B オブジェクトビューを一覧表示にした際の項目の表示順や、非表示設定をする列項目設定サブメニューを表示します。
C 選択されているオブジェクトのプロパティウィンドウを開きます。
D   ページ変更時にサイズ調整する/しないを切替えます。
E ツールバーの表示/非表示を切り替えます。
F ステータスバーの表示/非表示を切り替えます。
G ログバーの表示/非表示を切り替えます。

オブジェクトビューサブメニュー

オブジェクトビューの表示に関する操作を行います。

A オブジェクトビューを 1 画面レイアウトにします。
B オブジェクトビューを 2 画面（上下）レイアウトにします。
C オブジェクトビューを 2 画面（左右）レイアウトにします。
D オブジェクトビューを 3 画面（下を分割）レイアウトにします。
E オブジェクトビューを 3 画面（上を分割）レイアウトにします。
F オブジェクトビューを 3 画面（右を分割）レイアウトにします。
G オブジェクトビューを 3 画面（左を分割）レイアウトにします。
H オブジェクトビューを 4 画面レイアウトにします。
I オブジェクトビューの分割サイズを既定の状態にします。
2 画面以上のレイアウト時のみ有効です。

列項目設定サブメニュー

列項目設定の保存や読み込みを行います。

A 列項目の設定ダイアログが表示されます。
B 列項目の設定をプリセットファイル ( .3deuipreset ) として保存します。
C 保存した列項目のプリセットファイル ( .3deuipreset ) を読み込みます。

列項目の設定ダイアログ

オブジェクトビューを一覧表示にした際の項目の表示順や、非表示設定をします。

A モデル一覧表示の設定です。
B マテリアル一覧表示の設定です。
マテリアル一覧の設定では、各カテゴリ毎に列項目の設定ができます。
C シェイプ一覧表示の設定です。
D テクスチャ一覧表示の設定です。
E ボーン一覧表示の設定です。
F 各列項目が上から下に現在の順序で一覧表に表示されます。
項目のチェックを外すと非表示になります。
G 選択している項目の表示順序を上の項目と入れ替えます。
H 選択している項目の表示順序を下の項目と入れ替えます。
I 列項目の順序と表示を既定の状態に戻します。
J 列項目の設定全てにチェックを入れます。
K 列項目の設定全てのチェックを外します。
L 列項目の設定変更を実行します。
M 列項目の設定変更をキャンセルします。
列項目の設定は、変更前の状態になります。

マテリアル一覧の列項目の設定

マテリアル一覧では、表示する情報量が多いのでカテゴリ毎の列項目の設定ができます。

A マテリアル一覧で表示したい情報をカテゴリ毎に選択します。
B 選択したカテゴリ内にある項目が表示されます。
マテリアル一覧で表示したい項目にチェックを入れます。
