端でαを薄くする割合

ここでは「範囲内ループ」で利用する「箱」の端でパーティクルのアルファ値を変化させる機能について説明します。

概要

「端でαを薄くする割合」は、「箱」の範囲内でパーティクルの透明度を設定できる機能です。
「箱」の際でパーティクルの一部が突然出現または消えることによる見た目の違和感を緩和できます。
以下は の概要です。

端でαを薄くする割合

箱の端から中心を 0 ～ 1 の範囲として割合で設定します（箱の中心が1、箱の端が0）。
パーティクルのアルファ値は、この位置から箱の端にかけて α値 = 1 ～ 0 の範囲でグラデーションします。
値は x y z 軸それぞれについて設定し、最終的な α 値は各軸の α 値の乗算になります。
