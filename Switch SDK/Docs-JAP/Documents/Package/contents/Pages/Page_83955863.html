アニメーションのバイナリー出力設定プラグイン

transform ノードおよび joint ノードにアニメーションのバイナリー出力フラグを設定します。

設定の検索もできます。

スケルタルアニメーションの移動のバイナリー出力フラグを OFF にして、体格の異なるキャラクタで回転アニメーションを共用するといった用途に使用します。

使用方法

「Nintendo > Set Animation Binarization...」を選択するとウィンドウが表示されます。

Selected Transform Node 欄には、選択されている transform ノードまたは joint ノードが表示されます。

参考：

4 つ以上のノードが選択されている場合は省略表示となり、右端に選択された数が表示されます。

設定方法

transform ノード または joint ノードを選択（複数選択可）すると、Set 欄に選択したノードの現在の設定が表示されます。複数選択の場合、最初に選択したノードの設定が表示されます。

チェックボックスをクリックしてバイナリー出力フラグを変更します。
複数選択の場合、選択したノードを一括して変更できます（変更したバイナリー出力フラグだけ反映されます）。

ウィンドウを閉じずに別のノードを選択すると、別のノードを設定することができます。

Binarize Skeletal Animation

スケルタルアニメーションの移動（Translate）、回転（Rotate）、スケール（Scale）について、バイナリー出力フラグを指定します。

 デフォルトはすべて ON です。

検索方法

Search 欄のチェックボックスを ON にして、Search ボタンをクリックすると、該当する transform ノードと joint ノードが選択されます。
Search ボタンの左にあるラジオボタンを and にすると、各バイナリー出力フラグの設定をすべて満たすものを検索します。
or にすると、いずれか 1 つ以上の項目を満たすものを検索します。

参考：

1 つもチェックされていない項目は検索の条件から除外されます。
