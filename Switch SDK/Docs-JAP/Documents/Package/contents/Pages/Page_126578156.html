概要

InitializeEdev は EDEV 本体のファームウェアを初期化するためのコマンドラインツールです。

パッケージ

InitializeEdev は NintendoSDK NX Addon パッケージに含まれています。

配置

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/InitializeEdev.exe

なお動作のためにはこの exe ファイル以外にも依存するファイルがいくつかあります。

コマンド定義

オプション形式

InitializeEdev [target options] [other options]

target options:
    --target <serial number>                           初期化するターゲットのシリアル番号を指定します

other options:
    --target-version                                   初期化で用いるファームウェアの種類を指定します
    --display-available-version                        利用できるファームウェアの種類をすべて出力します
    --verbose                                          ログを詳細に出力します
    --version                                          ツール自身のバージョンを表示します

なおシリアル番号指定時はチェックディジットはつけてもつけなくても動作します。

終了コード

ツールの結果成否は終了コードで確認することができます。

0   - 正常終了
非0 - エラー終了

使い方

以下に利用ケースに応じた InitializeEdev の実行例を示していきます。

EDEV のシリアル番号を指定して初期化する

InitializeEdev --target XAW03121000000

 

なお、これ以外のオプションについては現在は利用するケースがまだないため例示を省略します。

その他機能の説明

製品機に近いファーム(PROD MODE)による初期化

「製品機に近い状態になるファーム(PROD MODE)」を使って初期化することもできます。

InitializeEdev.exe --target XAL0310000XXXX --target-version "0.X.Y-Z + Prod"

のようにオプション --target-version の後ろに "+ Prod" を追加すると製品機に近いファームで初期化します。

さらに、

InitializeEdev.exe --target XAL0310000XXXX --target-version "0.X.Y-Z + ProdWithLog"

オプション --target-version の後ろに "+ ProdWithLog" を追加すると microSD カードからのログ出力機能付きの製品機に近いファームで初期化します。

注意：

「製品機に近いファームによる初期化(PROD MODE)」機能は EDEV MP のみで利用できます。それ以外の EDEV 本体で実行するとエラーになりますのでご注意ください。

注意事項

バージョンダウン初期化に関して

現バージョンの InitializeEdev で初期化した EDEV を NX Addon 0.10 以前の InitializeEdev (および InitializeEdevWin) を用いてバージョンダウン初期化することはできません。
