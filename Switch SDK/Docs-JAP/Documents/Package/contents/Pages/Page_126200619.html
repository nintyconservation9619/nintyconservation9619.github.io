ビットマップフォントを作成する

ここではFontConverter(GUI版)を使用し、以下の様なオリジナルのビットマップフォントを作成する手順について説明します。

このチュートリアルでは以下の 手順でフォントを作成しています。

1 フォント作成の準備 FontConverter(GUI版) を起動し、フォント作成のテンプレートにする為の画像を用意します。
2 画像編集ソフトでフォントの作成 テンプレート画像を画像編集ソフトに読み込み、これをベースにオリジナルのフォントを作成します。
3 FontConverter からフォントファイルの出力 作成したオリジナルフォントを FontConverter(GUI版) に読み込み、フォントファイルに変換して出力します。
4

LayoutEditor での利用

作成したオリジナルフォントを LayoutEditor のテキストペインで利用します。

1.フォントの作成準備

FontConverter(GUI版) を起動し、テンプレート用画像を用意します。

関連リンク FontConverter(GUI版) の起動方法については「作業の流れ」を参照してください。

「Windows Font」タブを選択する

「入力」項目から「Windows Font」タブを選択します。
以下の項目を設定します。

フォント名 サイズ 説明
MS ゴシック
 		16 px Windows PC にインストールされたフォントが選択できます。
入力フォントとして「 MS　ゴシック」をサイズ 16 pixel で利用します。

「Image」タブを選択する

「出力」項目から「Image」タブを選択します。
「画像ファイル」項目の「 ファイル選択」を押し、表示されたダイアログで出力先をと出力時のファイル名を設定します。
ここではファイル名を「Template.bmp」とします。

「入力」項目で選択したフォントを画像ファイルとして出力できます。
出力した画像ファイルは、テンプレートとして画像編集ソフトに読み込み、これをベースにオリジナルのフォントを作成します。

2.画像編集ソフトでフォントの作成

テンプレート画像を画像編集ソフトで開きます。ここでは画像編集ソフトとして「Photoshop」を使用します。
以下イメージ図のように、テンプレート画像を下のレイヤーとし、これをベースに別レイヤーでオリジナルのフォントを作成します。

関連リンク 「Photoshop」で作業するとき、テンプレート画像の文字部分とグリッドや幅線などのフォント情報をレイヤー分けしておくと作業しやすくなります。
レイヤーを分ける便利な Tips については「フォント生成用レイヤーを分離する」を参照してください。

3.FontConverter からフォントファイルの出力

再び FontConverter (GUI版) の作業に移ります。
作成したオリジナルフォント「IconFont_16.tga」を読み込み、フォントファイルに変換して出力します。

「Image」タブを選択する

「入力」項目から「Image」タブを選択します。
画像編集ソフトで作成した画像ファイル「IconFont_16.tga」を選択します。

カラーフォーマットを変更する

「Image」タブ設定「カラーフォーマット」のプリセットを「LA8( –> BC5 )」に変更します。

関連リンク 以下は関連ページへのリンクです。

カラーフォーマット よくある質問「カラーフォーマットの選び方」について説明します。
カラーフォーマットの選び方 ビットマップフォントで使用できる画像のカラーフォーマットについて説明します。

「bffnt / bffna」タブを選択する

「出力」項目から「bffnt / bffna」タブを選択します。
「出力ファイル」項目の「 ファイル選択」を押し、表示されたダイアログで出力先をと出力時のファイル名を設定します。
ここではファイル名を「IconFont_16.bffnt」とします。

「入力」項目で選択した画像ファイルをLayoutEditor で利用可能なフォントファイルとして出力できます。

フォントファイルに変換する

「 変換」を押してフォントファイルを出力します。
