参照ファイルの検索操作

ここではエフェクトブラウザの検索機能の使い方について、エミッタセットが参照するテクスチャの検索する手順を例に説明します。

エフェクトブラウザを開く

EffectMaker ツールバー「 エフェクトブラウザ」ボタンをマウス左ボタンクリックし、「エフェクトブラウザ」のウィンドウを開きます。

「検索」タブの選択

「ナビゲーション」ペインで「 検索タブ」を選択することで、検索条件を設定する項目が表示されます。

検索の設定

以下の手順で検索の設定を行います。

検索基点パスの設定

「 テキストボックス」に検索の基準となるフォルダのパスを指定します。
ここではパッケージの以下の場所を設定してください。

\Samples\Sources\Resources\Vfx\Eset\

検索対象ファイルの選択

検索機能で探したいファイルの種類を「 ラジオボタン」で選択します。
ここではテクスチャを検索する為、「 テクスチャ」に設定します。

検索条件の設定

検索条件をプリセットから選択します。
ここでは「参照テクスチャ」を選択します。
この条件では「検索基点パス」以下のエミッタセットが参照するテクスチャファイルを検索します。

関連リンク 検索条件の詳細についてはこちらを参照してください。

検索の実行

「検索ボタン」をマウス左ボタンクリックで検索を開始します。
「ファイルリスト」に検索の結果ヒットしたテクスチャファイルが表示されていることを確認します。

検索範囲の設定

「 検索範囲を検索基点パスに絞り込む」に設定することで、検索が再実行されます。
ファイルリストには検索範囲を「検索基点パス」以下に絞り込んだ結果が表示します。
