ConvertNxbinlog

概要

ConvertNxbinlog は、nxbinlog ファイルをテキスト形式ファイルに変換するコマンドラインツールです。

プレーンテキスト形式、および json 形式への変換に対応しています。

参考：

nxbinlog は、開発機が SD カードに出力するログのファイル形式です。SD カードログの利用方法は 開発支援機能の設定の「SD Card Logging Mode」を参照してください。

パッケージ

ConvertNxbinlog は、NintendoSDK NX Addon で提供されます。

配置

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/ConvertNxbinlog.exe

使い方

-i オプションに変換する nxbinlog ファイル、-o オプションに出力ファイルを指定して実行します。--json オプションを指定すると、json 形式に変換します。省略した場合は、プレーンテキスト形式に変換します。

$ ConvertNxbinlog.exe --help
ConvertNxbinlog.exe: Convert nxbinlog to ohter format file.

使用方法: ConvertNxbinlog.exe -i<arg> -o<arg> [--json]

使用可能な引数:
  -i, --input <arg>          Input file name.
  -o, --output <arg>         Output file name.
  --json                     Convert to json file.
