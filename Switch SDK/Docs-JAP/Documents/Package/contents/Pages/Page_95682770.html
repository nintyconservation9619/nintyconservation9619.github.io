UV スケールアニメ

ここでは登録したテクスチャの UV をスケールする機能「UV スケールアニメ」について説明します。

概要

「UV スケールアニメ」は登録したテクスチャの UV 座標にスケールをかけることでテクスチャにスケールをかける機能です。
以下は「 設定項目」の概要です。

UV スケールアニメ

テクスチャ UV 座標の「スケール」を各軸個別に設定できます。
X がテクスチャ座標 U 、 Y がテクスチャ座標 V に相当します。

設定を ON にすると「 UV スケールアニメ」が表示されます。以下はグループ内の設定項目です。

加算値

テクスチャ UV のスケールに毎フレーム加算する値を設定します。
加算値はエミッタ起動時から適用されます。

初期値

テクスチャ UV 座標のスケールに初期値を設定します。

初期値ランダム

初期値に指定した範囲の乱数を設定します。
初期値を乱数の上限、下限側はランダムに変化します。以下は乱数の下限をもとめる計算式です。

  • 乱数の下限 : 初期値 - 「初期値ランダム」の計算結果

設定例

以下は「UV スケールアニメ」の各機能の設定例です。
「放出間隔」は 10 に設定しています。

A 加算値
 指定の軸に対して毎フレーム加算する値として 0.01 を設定しています。
B 初期値 指定の軸に対して初期値として 2 を設定しています。
C 初期値ランダム

指定の軸に対してランダム値として 2 を設定しています。
放出毎の初期値は「初期値」2 に対し、上限を更に +2 した範囲内でランダムな値を取ります。
