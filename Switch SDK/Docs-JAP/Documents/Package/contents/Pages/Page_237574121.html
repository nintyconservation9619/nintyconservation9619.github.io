7 エクスポートアニメーション

3 エクスポート

Maya からアニメーションをエクスポートする手順の説明をします。

モデルを選択

エクスポートする対象のモデルを選択して設定します。

アウトライナなどからTopLを選択します。

エクスポート

Maya の 「メニューバー」→「 Nintendo」→「Export with Setting...」を開きます。

下図と表の順番に操作してエクスポートします。

 

順番 カテゴリ 操作
1 Export Target 「Selection」 にします。
2 Output File Name

「TrafficSignal」と入力します。
3 OutPut Folder

フォルダーアイコンをクリックして以下の「AnimationSample」フォルダーを指定します。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\TrafficSignal
4 General Options Magnify を「1.000」と入力します。Texture SRT Mode を「Maya」にします。
5 Animation Options

Start / End Frame を「All」にします。Maya で設定した再生範囲（0F～50F）のアニメーションを出力します。

アニメーションをループ再生するように指定します。Loop にチェックを入れます。( この指定は次の 7 のいずれかの Animeation Data のチェックを入れる必要が有ります)
6 Output File　Selection 「Texture Data」にチェックを入れます。
7 Output File　Selection

「Search Model Animation」をクリックすると、アニメーションキーの有効な出力対象にチェックが入ります。

以下の 4 つにチェックが入ることを確認します。

「Bone Visibility Animation Data」

「Color Animation Data」

「Texture SRT Animation Data」

「Texture Pattern Animation Data」

8 Export

「Export」をクリックします。１～ 7 の設定でファイルを出力します。

確認

下記のファイルが出力されていることを確認します。

フォルダー
%NINTENDO_SDK_ROOT% \ Samples\Sources\Applications\G3dDemo\Resources\TrafficSignal 
ファイル 内容

/textures/SampleBodyAlbedo.ftxb

/textures/SampleLidAlbedo.ftxb

/textures/TexturePattern.0.ftxb

/textures/TexturePattern.1.ftxb

/textures/TexturePattern.2.ftxb

/textures/TexturePattern.3.ftxb

/textures/TexturePattern.4.ftxb

/textures/TextureSRT.ftxb

テクスチャー

 カラーアニメーション
