エフェクトデータの編集

ここでは作成したエフェクトデータの編集方法と編集内容の保存方法について説明します。

準備

関連リンク クイックスタート「エフェクトの作成」を完了させた状態から作業を開始します。

ここまでの手順において、ワークスペースのエミッタセット「  SamplePreset_JustOne」直下には以下の様にエミッタ「 Emitter1」が作られています。

エミッタの選択

エミッタの選択方法について説明します。
「 SamplePreset_JustOne」のエミッタ「 Emitter1」をマウス左ボタンクリックで選択すると、プロパティウィンドウ右側に選択したエミッタの設定項目が表示されます。

タブの切り替え

プロパティウィンドウ上部には、ワークスペースで選択したノードの機能が種類ごとに分類された「 タブ」が並びます。
設定したい「 タブ」をマウス左ボタンクリックすることでタブ内の「 設定項目」を切り替えることができます。
以下は「 SamplePreset_JustOne」のエミッタ「 Emitter1」を選択した時のタブ項目です。

参考：

「 タブ」内の「 設定項目」がデフォルトから変更された場合、タブの色が「 オレンジ」に変わります。

値の編集

選択したエミッタの編集方法について説明します。
「 SamplePreset_JustOne」のエミッタ「 Emitter1」のタブ項目「エミッタ」を選択し、
「エミッタトランスフォーム」グループ→「位置(±)」Xの値を以下の方法で編集します。

  • 「 設定項目」の右側にある「 ダイアル」の上でマウス左ボタンでクリックした後、ドラッグ移動させることにより、「  テキストボックス 」の値を変更できます。以下は操作説明です。
  • マウス左ボタンクリックから ← → 移動で値を増減できます。
  • マウス左ボタンクリックから↑ ↓ 移動で← → 操作時の値の増減度合いを調節できます。
  • 「  値 = 0」をマウス左ボタンクリックすることで、直接値を設定できます。ここでは値を「  値 = 10.123」にします。
  • 「 スピナボタン上」を押すごとにの値を 1 段階上げられます。「 スピナボタン下」を押すごとにの値を 1 段階下げられます。
  • 「 アジャストボタン」を押すことで「 ダイアル」で感覚的に値を決めた後、「 アジャストボタン」で少数点の端数を切り捨てられます。
    例えば「  値 = 10.123」の時に「 アジャストボタン」を1回押しすと、少数部を 1 段上の位「 値 = 10.12」に丸めます。
  • 「 ランダム設定」は上位項目の設定に対して乱数値を設定します。

値を編集したことで、ワークスペース上の「 SamplePreset_JustOne」と「 Emitter1」の右側に編集したことを示す「 編集済みマーク」が付いていることを確認します。

参考：

「 設定項目」がデフォルトから変更された場合、色が「 オレンジ」に変わります。

編集内容の保存

編集したエミッタセットを保存します。
ワークスペースのエミッタセット「 SamplePreset_JustOne」を選択した状態で、以下いずれかの操作を行います。

  • マウス右ボタンクリックで開くメニューから [ 保存] を選択します。
  • メインメニュー項目 [ファイル]→[ 保存] を選択します。
  • ショートカットキー Ctrl + S を使用します。

ダイアログが表示されますので、ファイル名と保存先を指定してから「OK」ボタンを押します。
「  SamplePreset_JustOne」の右側の「  編集済みマーク」が消えたことを確認します。

エミッタセットを選択して閉じる

エミッタセットを閉じる方法について説明します。
ワークスペースのエミッタセット「 SamplePreset_JustOne」を選択した状態で、以下いずれかの操作を行います。

  • マウス右ボタンクリックで開くメニューから[ 閉じる] を選択します。
  • メインメニュー項目 [ファイル]→[ 閉じる] を選択します。
  • ショートカットキー Ctrl + W を使用します。

エミッタセット「 SamplePreset_JustOne」は既に保存されている為、ファイルは閉じられ、ワークスペース上からもとり除かれます。

参考：

この時、エミッタセットの編集結果が未保存の場合は保存を確認するダイアログが表示されます。
