NVN ライブラリ

はじめに

NVN は NintendoSDK NX Addonでサポートされる低レベル 3D グラフィックス API です。この API はハードウェアの仕様に直結した設計となっており、例えばハードウェアの差異を吸収する様なレイヤが内部に入っていないため CPU のオーバーヘッドを減らす事に成功しています。

NVN は NintendoSDK のユーザが自前のゲームエンジン等へ組み込む際の API の取り回しの観点から GFX ライブラリよりも細かい粒度でオブジェクトの操作をしたい場合等を想定して提供されています。しかし上述の様にハードウェアに特化した仕様になっているため、互換性の観点から NintendoSDK としてはグラフィックス API は NVN をラップした上位の GFX ライブラリを使用し、必要な部分にのみ 低レベルグラフィックス API との相互運用 機能を利用して NVN を使用する事を推奨しています。

GPU ドライバのインストール方法 

PC 上で NVN を使用するためには専用のドライバのインストールが必要です。

インストーラーのファイルは Nintendo Dev Interface から「NVIDIA Graphics Driver」パッケージをインストールすることで入手することができます。パッケージのインストール時に表示されるダイアログからお使いの OS に適したインストーラーを起動することができます。ダイアログからインストーラーを起動しなかった場合には NintendoSDK/Installers/NvnDriver 以下にあるバッチファイルを実行することでドライバをインストールすることができます。バッチファイルは OS ごとに用意されておりますのでお使いの OS に合ったものを実行してください。

対応 GPU とサポートされる機能

GPU アーキテクチャは Kepler 第1世代以降をサポートしますが使用できる機能に若干の制約があります。特に、Maxwell 第2世代未満の GPU を使用する際は NX ハードウェアのいくつかの機能が使用できないことにご注意ください。詳細は NVN プログラミングマニュアル  を参照してください。

また、PC 上で動作する NVN アプリケーションに対して NVN 専用のグラフィックスデバッガを利用する場合、Maxwell 第2世代以外の GPU を使用する際はグラフィックスデバッガのいくつかの機能が使用できないことにご注意ください。詳細は NVN グラフィックスデバッガ パッケージ内に含まれるドキュメントを参照してください。

制限事項

NVN はシェーダ言語として GLSL / SPIR-V を使用しますが、これらを NVN から使用する場合には Khronos の適合性試験（Conformance Test）によってテストされていません。そのため Khronos の GLSL / SPIR-V 仕様と差異がある可能性があります。
