NintendoSDK

NintendoSDK は以下の機能を提供します。
機能によっては一部制限があったり、将来的に変更される可能性がありますのでご注意ください。

ビルドシステム

NintendoSDK のライブラリやツールを利用してアプリケーションを作成するための仕組みを提供します。

現在サポートしているプラットフォーム、プラットフォームツールセットは次の通りです。

プラットフォーム
  • Win32
  • x64
プラットフォームツールセット
  • v140 (Visual Studio 2015)
  • v141 (Visual Studio 2017) (NX Addon 4.x 以降のみサポート)

ビルドシステムに関する詳細は こちら をご確認ください。

ライブラリ

NintendoSDK が提供するライブラリは以下の通りです。
これらのライブラリをリンクすることによって、OS や FS、パッド、通信、グラフィックス、オーディオといった基本的な機能が利用可能になります。

参考：

ここで示される機能群は「汎用 API セット」と呼ばれます。

具体的なプラットフォーム上でのみサポートされる機能については、○○Addon というパッケージ群により、「○○ (ハード名) API セット」と呼ばれる API セットとして提供されます。

ライブラリ サポートする機能 現バージョンの制限・補足
ACCOUNT
  • 本体システム上で作成したユーザーアカウントの一覧の取得
  • ネットワークサービスアカウントに関する機能
  • ニンテンドーアカウントに関する機能


AUDIO
  • 音声を扱うデバイスとの入出力
  • 「ボイス」 を単位としたピッチ変換・フィルタ・ミキシング
  • 「サブミックス・ファイナルミックス」を単位としたエフェクト処理・AUX 処理


BCAT
  • データ配信
CODEC
  • adpcm, opus, aac などのオーディオ圧縮フォーマットの、ターゲット上でのエンコード・デコード処理、および PC 上でのエンコードツール
  • opus, aac の機能はオープンソースソフトウェア (OSS) を使用されています。
    • 利用には当該 OSS のライセンス条項に同意する必要があります。
CRYPTO
  • データの暗号化・復号化
  • 電子署名の生成・検証
  • 暗号学的ハッシュ関数によるハッシュ値の計算
CURL
  • cURL による HTTP/HTTPS 通信
DIAG
  • ログ出力、アサーション、バックトレースの取得
ERR
  • エラー表示
FONTLL
  • 本体内蔵フォントを含むスケーラブルフォントの読み取りとメインメモリへのレンダリング
FRIENDS
  • フレンドリストの取得
  • ブロックリストの取得
  • プレゼンスの更新
  • 任意のユーザーのユーザープロフィール（ニックネームとプロフィール画像 URL）の取得
  • いっしょに遊んだユーザーの追加
FS
  • アプリのリソースデータの読み込み
  • セーブデータの読み書き
  • 開発用途でのホスト PC 上のファイルシステムの読み書き
  • 利用可能なセーブデータの種類に制限があります。
GFX
  • 低レベルグラフィックス API を抽象化したグラフィックス機能
  • 現在、GL 版の実装についていくつかの 制限 があります。
HID
  • ヒューマンインターフェイスデバイス (ゲームパッド、キーボード、タッチスクリーン、加速度センサなど) の入力制御
  • 振動機能
  • 下記をサポートしています。
    • ゲームパッド
    • タッチスクリーン
    • デバッグ用ゲームパッド
    • キーボード
    • デバッグ用マウス
  • 次の機能追加を予定しています。
    • 自動入力や入力の記録・再生などのデバッグ機能 （一部対応済み）
HTC
  • Target Manager を介してホスト PC とやり取りを行う各種機能
  • 制限に関しては HTC ライブラリの制限 を参照してください。
HTCS
  • ホスト上アプリとターゲット上アプリ間の Target Manager を介したソケット通信
  • 制限に関しては HTCS ライブラリの制限 を参照してください。
INIT
  • メモリアロケータの初期化
  • プログラムの起動と初期化 も併せて参照してください。
JPEG
  • JPEG のデコードおよび、 Exif 情報の取得
  • Exif 情報の付加を伴う JPEG のエンコード


KPR
  • キー入力の文字コード変換
  • キー入力の文字コード組み立て（デッドキー処理、ローマ字入力など）
MEM
  • メモリ領域の割り当てを行うためのスレッドセーフなメモリアロケータ機能
    • 従来の拡張ヒープ、フレームヒープ、ユニットヒープ
    • 拡張ヒープよりも確保、解放速度が優れる汎用アロケータ (StandardAllocator クラス)
  • スレッドセーフな整数アロケータ機能
    • NumberLineAllocator クラス


Mii（似顔絵ライブラリ）
  • システムからの Mii データ取得
  • システムからの描画リソース取得
  • Mii データを使ったMiiの描画
  • Mii 作成・編集アプレットと同等の描画機能
  • Mii のアイコン画像作成
  • Windows環境では、システムからの描画リソース取得をサポートしていません。
NGC
  • NG ワードチェック
NIFM
  • ネットワーク接続管理
  • 現在のバージョンにおける制限に関しては 利用上の注意・制限等 を参照してください。
OS
  • マルチコアに対応したマルチスレッド
  • ファイバ
  • セマフォ、イベント、メッセージキュー、タイマーイベント
  • 多重待ち
  • ミューテックス、条件変数、バリア、ReaderWriter ロック
  • メモリヒープ
  • チック
PCM
  • ボード上の消費電力の計測
  • 現在のバージョンにおける制限に関しては PCM ライブラリの制限 を参照してください。
PCTL
  • ペアレンタルコントロールによる機能の制限チェック
  • Windows環境では、実際に制限されることを確認するための設定変更を行うことができません。
PDM
  • プレイ情報の取得
  • Windows環境では、実際に情報を取得することはできません。
PERF
  • パフォーマンス計測結果集計
  • CPU パフォーマンス計測
  • GPU パフォーマンス計測
  • アプリケーションレベルの計測機能が提供されます。
SETTINGS
  • 本体設定やデバッグ設定、エミュレーション設定などの取得・設定


SOCKET
  • バークレイソケット互換の TCP/IP 通信
    • Windows 上では WinSock のラッパーとして動作します
SSL
  • SSL/TLS 通信
TIME
  • 時刻の取得
  • POSIX 時間と年月日時分秒の形式との相互変換


UTIL

  • ビット操作

  • 文字列操作

  • コンテナ（リスト・辞書）

  • 算術演算（三角関数・ベクトル・行列）

  • 圧縮・展開
VI
  • レイヤーの作成や、フォーマットやサイズの設定などの基本的なディスプレイ出力の機能
ZLIB
  • zlib による圧縮・展開
  • gzip file access functions は使用できません。


Target Manager

Target Manager は PC や LAN に接続されているターゲット上や、Windows 上で実行されるアプリケーションを管理するツールです。

開発機材の集中管理や、スクリプトによるターゲットの自動制御、HTCS の管理など、様々な機能を提供する予定です。

Target Manager に関する詳細は こちら をご確認ください。

NintendoWare

NintendoWare はゲームソフトのグラフィックスやサウンドの開発に使用することができるミドルウェアです。

NintendoWare に関する詳細は こちら をご確認ください。

警告：
  • NintendoWare のモジュールは NintendoSDK 0.7.0 にで大規模な変更を行いました。詳しくは 新NintendoWareライブラリについて を参照してください。
  • NintendoWare のマニュアルは現在制作中です。

Addon で追加される機能

各種プラットフォーム向けの Addon と呼ばれるパッケージ群をインストールすることで、それぞれのプラットフォームに応じて機能を足していくことが可能です。

それぞれの Addon で提供される機能一覧に関しては、「○○で追加される機能」 というページをご覧ください。
