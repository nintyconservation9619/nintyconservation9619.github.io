エミッタ形状「プリミティブ」

ここではパーティクルの放出位置が DCC プラグインで作成したモデル頂点のエミッタ形状「プリミティブ」の設定項目について説明します。

概要

エミッタ形状「プリミティブ」は、DCC ツールで作成した自由な形状のプリミティブモデルファイルを EffectMaker に読み込み、エミッタとして利用できます。
ワールド座標系に配置される読み込んだプリミティブモデルの各頂点から頂点法線方向にパーティクルを放出します。
以下は「 設定項目」の概要です。

「エミッタ形状」を「プリミティブ」に設定すると「 プリミティブ」が表示されます。以下はグループ内の設定項目です。

ファイル指定

モデルファイルを読み込むことができます。以下はボタンの種類です。

ロード ファイル選択ウィンドウが開き、モデルファイルが選択できます。
サンプルプリミティブを開く Primitive フォルダを開きます。良く使用するモデルファイルはこの場所に置くことができます。
Primitive フォルダは以下の場所に設定されています。
EffectMaker を起動したフォルダ"\Primitives
リロード 設定したモデルファイルが更新されます。
クリア 設定したモデルファイルを解除します。

プリミティブを読み込むと、そのプリミティブに関して以下の情報がコメントとして表示されます。
プリミティブの負荷や内包するアトリビュート情報を確認できます。

  • 頂点数
  • 内包するアトリビュート( pos（頂点位置） / normal（法線） / tangent（接線） / color（頂点カラー） / uv（テクスチャ座標） )

関連リンク 以下は関連ページへのリンクです。

プリミティブファイル EffectMaker におけるプリミティブファイルの扱い方について説明します。

放出タイプ

エミッタの法線方向にパーティクルを放出する時の挙動タイプを以下のプリセットから選択します。

一斉 パーティクルはプリミティブの頂点全てから法線方向へ放出されます。
ランダム パーティクルはプリミティブ頂点内のランダムな場所から法線方向へ放出されます。
インデックス順 パーティクルはプリミティブ頂点の描画順に法線方向へ放出されます。
