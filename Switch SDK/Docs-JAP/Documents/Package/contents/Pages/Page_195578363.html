ファイル

レイアウトエディタウインドウのメニューバー「ファイル」の項目について説明します。
新規作成 新規にレイアウトウインドウを作成します。
開く 保存されたレイアウトウインドウのファイルを開きます。
追加で開く 選択しているレイアウトウインドウに、レイアウトウインドウのデータを追加します。
保存 レイアウトウインドウを保存します。
別名で保存 レイアウトウインドウを、別の名前で保存します。
閉じる レイアウトウインドウを閉じます。
バイナリ出力 「バイナリファイル出力ウインドウ」が開きます。
最近使ったファイル 最近使ったファイルを表示します。選択すると、ファイルが開きます。
部品ルート以下のレイアウトファイルを再出力

部品の入ったフォルダへのパスを部品ルートと呼びます。

部品ルート以下のすべてのレイアウトファイルを一度ツールに読み込んで保存しなおします。
指定フォルダ以下のレイアウトファイルを一括同期

指定フォルダ以下の、部品を使っている、すべてのボディレイアウトについて、部品レイアウトの拡張ユーザ情報の変更を反映させます。

ボディレイアウトで上書きしているパラメータはそのまま維持されます。
終了 LayoutEditor を終了します。

サムネール保存機能を追加

LayoutEditor にサムネール用の画像を保存する機能を追加することができます。

「プロジェクト設定ファイル」から設定を有効にすると「レイアウトファイルを保存した」「一括同期を行った」等の時、サムネール画像をレイアウトファイルの横の .NwThumbs フォルダ内に「レイアウトファイル名.flyt.png」として保存します。
また、レイアウトエディタウインドウのメニューバー「ファイル」に、「指定フォルダ以下のサムネール画像を更新」のメニューが追加されます。

保存された画像は「サムネールエクスプローラ拡張」を使用して表示することができます。

メニューが追加された、レイアウトエディタウインドウのメニューバー「ファイル」

A メニューバー「ファイル」に追加された項目、「指定フォルダ以下のサムネール画像を更新」
