初速のランダム

ここではエミッタセット内すべてのパーティクルの「初速のランダム」に対して、ここで設定した値を乗算する機能「初速のランダム」について説明します。

概要

「初速のランダム」は、エミッタセット内すべてのパーティクルの「初速度のランダム」に対して、ここで設定した値を乗算できる機能です。
