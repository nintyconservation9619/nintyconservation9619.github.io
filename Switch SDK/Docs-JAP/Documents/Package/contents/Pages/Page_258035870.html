キャプチャーボタンの動作

キャプチャーボタンが提供する機能について説明します。

機能概要

スクリーンショット撮影機能

キャプチャーボタンを短時間押す（押してすぐ離す）ことで、実行中のアプリケーションのスクリーンショットを保存できます。

スクリーンショット撮影機能は API から利用することもできます。詳細は 画面写真機能 を参照してください。

動画撮影機能

参考：

動画撮影機能は NintendoSDK Firmware for NX 4.0.0-x.x 以降において、有効化したアプリケーションに対して動作します。

キャプチャーボタンを長押しすることで、実行中のアプリケーションによる映像と音声を動画ファイルとして保存できます。

システムが長押しされたと判定した時点から遡り、最大30秒分の動画を保存します。

NX Addon 4.2.0 以降（もしくは NX Addon 3 系では 3.6.0 以降）でビルドされたアプリケーションは、この動画撮影機能がデフォルトで有効となっています。

保存される動画ファイルの仕様は以下の通りです。

項目 内容
フォーマット MP4
映像 H.264 / 720p / 30fps
音声 AAC / 48kHz / stereo
再生時間 最大 30 秒
注意：

動画撮影は、実行中のアプリケーションがインフォーカス状態のときのみ撮影可能です。

HOME メニューへの遷移、ライブラリアプレットの呼び出し、あるいはスリープ復帰を挟んだ場合、それ以前の内容は失われます。

ファイルの保存先

SD カードが挿さっていれば、そちらを保存先として優先します。
SD カードは本体起動時に挿入されている必要があります。本体起動中に SD カードの抜き差しを行った場合は、本体の再起動が必要となります。

撮影したスクリーンショットおよび動画は、本体機能の「アルバム」から閲覧できます。「アルバム」の詳細は「本体機能ガイド」を参照してください。

機能の有効化または無効化

スクリーンショット撮影機能および動画撮影機能を制御する手段を提供しています。

  • nmeta の設定による制御方法（nmeta ファイルの書き方）
  • API によって撮影禁止区間を指定する方法（動作環境制御機能）

アプリケーションへの影響

動画撮影機能を有効化したアプリケーションには性能面で影響があります。

詳細は 実装に際しての注意事項 (NX Addon) を参照してください。
