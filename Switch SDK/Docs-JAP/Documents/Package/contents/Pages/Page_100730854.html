ランダムカラー設定のUI

「ランダムカラー設定の UI」では複数のカラーを登録できます。
放出時のパーティクルカラーは登録したカラーからランダムに選択されます。

A 挙動タイプ

選択項目から「ランダム」を選択することで、複数のカラーが登録できる本 UI が表示されます。
B カラーのON/OFF ランダムカラーの要素を「 チェックボックス」ON/OFF で切り替えます。
ON にすることで、そのカラーが適用されます。
C カラーの登録 左クリックすることで、カラーを設定できるボタンです。

関連リンク 以下は関連ページへのリンクです。

カラーの設定 「カラーコンバイナ」計算式で使用する Color0 や Color1 のカラーを設定する機能について説明します。
カラーの挙動タイプを「ランダム」に変更することで、本UIが使用可能になります。
