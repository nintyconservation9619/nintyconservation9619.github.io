基本設定

基本設定

このタブではエミッタ全体の設定を行います。

基本設定

「 基本設定」は、エミッタと放出されるパーティクルの基本挙動を設定する機能群です。

パーティクル追従タイプ エミッタのトランスフォームアニメーションに対するパーティクルの追従設定について説明します。
CPUで動作させる パーティクルの挙動計算方法を切り替える機能「終了処理」について説明します。
開始処理 エフェクト再生開始時の処理を決める機能「開始処理」について説明します。
終了処理 再生中のエフェクトを強制的に停止した時の処理を決める機能「終了処理」について説明します。

描画設定

「 描画設定」は、放出されるパーティクルの描画に関する設定を行う機能群です。
以下はグループ内設定項目の一覧です。

Zバッファ、アルファテスト 描画時の深度やアルファのテストを行う機能「Zバッファ、アルファテスト」について説明します。
パーティクルを描画 パーティクルを描画するかどうかを決める機能「パーティクルを描画」について説明します。
表示面 パーティクルについて、表示する面を決める機能「表示面」について説明します。
パーティクルソート 放出されたパーティクルごとの並び順を決める機能「パーティクルソート」について説明します。
描画パス ゲームアプリケーションで指定する描画パスを設定する機能「描画パス」について説明します。

乱数設定

「 乱数設定」は、パーティクル挙動の乱数値を制御する機能群です。
以下はグループ内設定項目の一覧です。

乱数タイプ パーティクル挙動の乱数値を制御する機能「乱数タイプ」について説明します。

継承設定

「 継承設定」は、エミッタから階層下のチャイルドエミッタへの継承項目について設定する機能群です。

関連リンク チャイルドエミッタの詳細は 「チャイルドエミッタ」 を参照してください。
以下はグループ内設定項目の一覧です。

パーティクルにエミッタ割当 親エミッタから放出されるパーティクルをチャイルドエミッタの発生元にするかを決める機能です。
描画パス 親エミッタの描画パスを自身の描画時に考慮するかどうかを決める機能です。
速度 親パーティクルの速度を自身の速度に乗算するかを決める機能です。
カラースケール 親パーティクルのカラースケールを自身のカラースケールに乗算するかを決める機能です。
カラー 親パーティクルのカラーを自身の カラー0 、カラー1 に乗算するかを決める機能です。
アルファ 親パーティクルのアルファを自身の アルファ0 、アルファ1 に乗算するかを決める機能です。
スケール 親パーティクルのスケール値を自身のスケール値 に乗算するかを決める機能です。
回転 親パーティクルの回転値を自身の回転値 に乗算するかを決める機能です。
