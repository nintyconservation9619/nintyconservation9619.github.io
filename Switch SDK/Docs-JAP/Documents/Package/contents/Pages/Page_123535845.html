SDEV の準備

SDEV の準備に必要な手順は、次の 3 ステップです。

  • SDEV の電源を ON にする
  • PC と SDEV を接続する
  • クラシックコントローラ PRO を接続する


参考：

事前に SDEV の背面の左下にある「ディップスイッチ」（下図の赤枠）が全て上側になっていることを確認してください。

SDEV の電源を ON にする

SDEV の上面にある「ACアダプター接続端子」（下図の赤丸）に、付属の「Wii U本体ACアダプター」を接続します。

SDEV には主電源の ON/OFF を行うスイッチがないため、主電源の ON/OFF を行うには「Wii U本体ACアダプター」の挿抜が必要です。

SDEV 本体上面にある電源ボタンを 6 秒間長押しした場合、システムは OFF になりますが、SDEV の主電源は ON のままです。

SDEV の電源が ON になったことは、以下の 2 つの方法で確認できます。

  • SDEV 上面にある 「Ethernet ポート」の LED が点灯する（10秒ほどで消灯します）
  • SDEV 裏面左下部の 3 連の LED のうち真ん中のLEDが点灯する。
参考：

他の PC と接続していた SDEV を使用する場合は、以下の手順で電源を ON にしてください。

  1. SDEV 本体の背面にある「復旧ボタン」を押しながら、SDEV 本体の電源を入れます
  2. 「復旧ボタン」の右側にある「ゲームカートリッジ検知 LED」「電源 LED」「SD カード検知 LED」がすべて点灯してから、「復旧ボタン」から手を離します
  3. 「復旧用のファームウェア」が起動し、SDEV の PC との接続設定がリセットされます


PC と SDEV を接続する

SDEV の上面にある「Ethernet ポート」（下図の赤丸）に、「Ethernet ケーブル」を接続します。


参考：

PC と SDEV との接続は、SDEV の初期化の「SDEV を初期化する」で行う手順を通じて確認することが出来ます。

初期化 UI で SDEV が表示されていない場合は、接続上の問題があるかも知れません。


以下のどちらかの方法で PC と SDEV を Ethernet で接続します。

  1. PC と SDEV を「Ethernet ケーブル」で直結する
    • SDEV を直結しているネットワークカードの設定が「IP アドレスを自動取得する」である必要があります
    • 設定方法は「直結時の IP アドレス設定方法」を参照してください
  2. PC が接続されているネットワーク上のネットワーク装置（スイッチまたはルーター）とSDEV をEthernet ケーブルで接続する
    • DHCP サーバーが存在するネットワークでも、存在しないネットワークでも正常に動作します

SDEV との通信速度を向上する方法

PC のネットワーク アダプターがジャンボフレームに対応している場合 SDEV との通信速度を向上させることができます。

デバイスマネージャーを開き、ネットワーク アダプター -> （対象のアダプターの）プロパティ -> 詳細設定 で ジャンボフレームを「4KB」または「4088 Bytes」に設定します。

※ネットワーク アダプターによっては JumboPacket と表記されていることがあります。

注意：

環境によってはジャンボフレームが使えない可能性がありますので、通信がうまくいかない場合は無効にしてください。

クラシックコントローラ PRO を接続する

SDEV の上面にある「デバッグ用コントローラ接続端子」（下図の赤丸）に、「クラシックコントローラ PRO」を接続します。

クラシックコントローラ PRO を、デバッグ用のコントローラとして使用することができます。
