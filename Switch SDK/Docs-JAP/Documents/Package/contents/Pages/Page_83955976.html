マニュアルの表記規則

このマニュアルで記述されている表記規則について説明します。

メニューの表記

メニューとメニュー項目は、[メニュー名]→[メニュー項目名] と表記します。 例として、「ファイル」メニューの「開く」項目を示す場合は、[ファイル]→[開く] となります。

キーバインディングの表記

複数のキーを同時に使うキーバインディングは、＋マークで表記します。 例として、Ctrl キーを押しながら S キーを押す場合は、Ctrl+S となります。
