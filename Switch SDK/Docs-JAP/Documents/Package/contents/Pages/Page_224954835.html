【ユーザー設定】プリミティブ

プリミティブ

プリミティブファイル使用時の環境設定を行います。

参照外のプリミティブを読み込んだ時に警告ダイアログを表示

エミッタセットファイル（.eset）を開いた時、エミッタやチャイルドに登録されているパーティクル形状「プリミティブ」用のモデルファイル(.fmdb) が参照できない場所にある場合に警告を表示するかを ON/OFF で切り替えます。

参照元のプリミティブが変更された時に自動的に読み込む

エミッタやチャイルドに登録されているパーティクル形状「プリミティブ」用のモデルファイル（.fmdb） が更新された時、自動で再読み込みするかをON/OFF で切り替えます。
