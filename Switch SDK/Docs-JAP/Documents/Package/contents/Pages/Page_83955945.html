3DEditor の概要

3DEditor は中間ファイルを編集するツールです。
ビューアライブラリと通信し、編集状況を確認しながら編集できるリアルタイム編集に対応します。

3DEditor の編集項目

3DEditor で編集できる項目は以下になります。

  • モデルのパラメータ（ビルボード、ユーザーデータなど）
  • マテリアルの固定機能（テクスチャサンプラ）
  • マテリアルへのシェーダ割り当てとパラメータ編集
  • テクスチャの編集（ユーザーデータなど）
  • マテリアルアニメーションの作成と編集
  • ボーンビジビリティアニメーションの作成と編集
  • スケルタルアニメーションの編集(ユーザーデータなど)
  • シーンアニメーションの作成と編集

ビューアライブラリ

ビューアライブラリ とは、ゲームプログラムに組み込んで利用する、プレビュー用のランタイムライブラリです。
ビューアライブラリは 3DEditor とリアルタイムに通信し、編集状況を表示します。

ビューアライブラリを組み込むと、以下のような利点があります。

  •   ゲームプログラムやビューアプログラムの起動無しに、ゲームとプレビュー環境を高速に切り替えることができます。
  •   ライティング、フルスクリーンフィルタといった、ゲーム特有の描画処理を適用した上で、パラメータの調整ができます。

対応アニメーションの種類

3DEditor が対応しているアニメーションは以下になります。

拡張子 説明
.fskb / .fska スケルタルアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fskb、テキスト中間ファイルの場合は .fska です。
.fmab / .fmaa

マテリアルアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fmab、テキスト中間ファイルの場合は .fmaa です。
.fspb / .fspa シェーダパラメータアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fspd、テキスト中間ファイルの場合は .fspa です。
.fclb / .fcla カラーアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fclb、テキスト中間ファイルの場合は .fcla です。
.ftsb / .ftsa テクスチャSRTアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .ftsb、テキスト中間ファイルの場合は .ftsa です。
.ftpb / .ftpa テクスチャパターンアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .ftpb、テキスト中間ファイルの場合は .ftpa です。
.fvbb / .fvba ボーンビジビリティアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fvbb、テキスト中間ファイルの場合は .fvba です。
.fvmb / .fvma マテリアルビジビリティアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fvmb、テキスト中間ファイルの場合は .fvma です。
.fshb / .fsha シェイプアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fshb、テキスト中間ファイルの場合は .fsha です。
.fsnb / .fsna シーンアニメーション中間ファイルです。
バイナリ中間ファイルの場合は .fsnb、テキスト中間ファイルの場合は .fsna です。
