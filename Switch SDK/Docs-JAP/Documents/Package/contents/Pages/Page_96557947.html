重力の「方向」

方向

ここでは放出されたパーティクルに対して重力がかかる方向を指定するする機能「方向」について説明します。

概要

「方向」は、「重力」で設定した力を与える方向を決める機能です。X,Y,Z軸であらゆる方向に指定できます。
以下は「 設定項目」の概要です。

方向

パーティクル放出時の進行方向をを X Y Z で設定します。
値は -1 から 1 の範囲で「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は「方向」の設定を変更したときの比較画像です。

A :X =  1

B :X = -1

C :Y =  1

D :Y = -1

E :Z =  1

F :Z = -1
