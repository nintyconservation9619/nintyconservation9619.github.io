文字コードの組み立て

機能概要

複数のキー入力から構成される文字入力を受け付けて文字コード (UTF-16) 列を組み立てる機能を提供します。

キー入力は GetKeyCode() によって文字コードに変換された形式で指定します。


文字コードの組み立てを行う際は、まず KeyCodeComposer クラスのインスタンスを作成します。

KeyCodeComposer は下表の組み立てモードをサポートします。目的に応じて KeyCodeComposer::SetMode() で指定してください。

現在の組み立てモードは KeyCodeComposer::GetMode() で確認できます。組み立てモードを変更すると組み立て途中の文字列は破棄されます。

組み立てモード 説明
KeyCodeComposerMode_Default 入力された文字コードをそのまま出力します。
KeyCodeComposerMode_DeadKey デッドキーによる修飾を処理します。
KeyCodeComposerMode_RomajiHiragana 日本語のローマ字入力（ひらがな）を処理します。
KeyCodeComposerMode_RomajiKatakana 日本語のローマ字入力（カタカナ）を処理します。

KeyCodeComposer は、文字コードを格納するための長さ KeyCodeCompositionCountMax のキューを保持しており、

KeyCodeComposer::Enqueue() で指定されたキー入力を一つずつキューの末尾に追加し、その度に組み立て処理を進めます。

キューへの追加はキュー先頭の文字コードが組み立ての完了した値として確定した時点で打ち切られます。

キー入力の受け付けがどこまで進んだかは KeyCodeComposer::Enqueue() の返り値で確認できます。

組み立てられた文字コード列は KeyCodeComposer::Dequeue() を呼び出すことでキューから取り出すことができます。

全ての確定した文字コードを取り出すことで再び KeyCodeComposer::Enqueue() によるキー入力の追加が可能になります。

KeyCodeComposer::Dequeue() で取り出すことが可能な文字コードの数は GetDequeueableCount() で取得できます。

キューの内容と長さは、それぞれ  KeyCodeComposer::GetComposition() と KeyCodeComposer::GetCompositionCount() で参照することが可能です。

キュー中の文字コードから未確定部分を削除する場合は  KeyCodeComposer::Remove() を呼び出します。

キュー中の文字コードを強制的に確定させる場合は KeyCodeComposer::Flush() を呼び出します。

キュー中の文字コードを破棄する場合は KeyCodeComposer::Clear() を呼び出します。

使用方法

#include <nn/nn_Common.h>
#include <nn/nn_Log.h>
#include <nn/nn_Macro.h>
#include <nn/kpr/kpr_KeyCodeComposer.h>
#include <nn/util/util_CharacterEncoding.h>

extern "C" void nnMain()
{
    const uint16_t src[] = { 'k', 'o', 'n', 'n', 'n', 'i', 'c', 'h', 'i', 'h', 'a' };

    uint16_t dst[NN_ARRAY_SIZE(src)] = {};

    int srcHead = 0;
    int dstHead = 0;

// 1. KeyCodeComposer クラスのインスタンスを作成します。
    nn::kpr::KeyCodeComposer composer;

// 2. 組み立てモードを設定します。
    composer.SetMode(nn::kpr::KeyCodeComposerMode_RomajiHiragana);

    while (srcHead < NN_ARRAY_SIZE(src))
    {
// 3. 組み立て処理を進めます。
        srcHead += composer.Enqueue(&src[srcHead], NN_ARRAY_SIZE(src) - srcHead);

// 4. 組み立てられた文字コード列を取り出します。
        dstHead += composer.Dequeue(&dst[dstHead], NN_ARRAY_SIZE(dst) - dstHead);
    }

    char utf8[NN_ARRAY_SIZE(dst) * 4 + 1] = {};
    nn::util::ConvertStringUtf16NativeToUtf8(utf8, NN_ARRAY_SIZE(utf8), dst, dstHead);

    NN_LOG("%s\n", utf8);
}
