オプション

EffectMaker やビューアの使用環境を設定する「オプションウィンドウ」について説明します。

【ユーザー設定】 ユーザー毎の設定項目です。
インターフェイス EffectMaker の UI について、設定できます。
ビューア設定 ビューアについて設定できます。
ファイルイベント ファイル操作時に実行したいユーザーコマンドファイルを設定できます。
詳細 EffectMaker が利用するメモリやデータの管理方法について設定できます。
【プロジェクト設定】 ゲームプロジェクト毎の設定項目です。基本的にユーザー毎の設定は推奨しません。
カスタムシェーダ EffectMaker で使用するカスタムアクション用ファイルを指定します。
カスタムアクション EffectMaker で使用するカスタムシェーダ用フォルダを指定します。
エミッタ拡張パラメータ EffectMaker で使用するエミッタ拡張パラメータ用フォルダを指定します。
ファイルイベント ファイル操作時に実行したいユーザーコマンドファイルを設定できます。
【プラットフォーム設定】 ゲームプラットフォームに関する情報を表示します。
