NintendoWare Spy ニュース

NintendoSDK 1.0.0 における NintendoWare Spy の変更点を紹介します。

Audio ラウドネスメーターパネルが強化されました

Audio ラウドネスメーターパネルの機能が強化されました。

  • True Peak が表示できるようになりました。
  • メーター表示がピークホールドに対応しました。

操作方法は Spy/Spy.exe/Audio ラウドネスメーターパネル を参照してください。

（評価中）Spy デバッグパネルを追加しました

注意：

この機能は現在、評価中です。予告なく変更・削除される場合がありますので、ご注意ください。

Spy デバッグパネルを追加しました。

Spy ライブラリの動作状態を表示します。今のところ、データ送信用バッファの使用率をモニターすることができます。

操作方法は Spy/Spy.exe/Spy デバッグパネル を参照してください。

このパネルに表示されるデータは Spy ライブラリの nn::spy::DebugModule クラスで送信されます。

時間軸の変更

タイムラインパネルや Audio 波形パネルなどの横軸が時間のグラフ表示において、時間軸の扱いが変わりました。

時間の単位が AppFrame のときは、アプリケーションフレームが記録された時間にグリッド線が表示されるようになりました。

  • 従来はアプリケーションフレームが等間隔になるように縮尺が変化していました。
