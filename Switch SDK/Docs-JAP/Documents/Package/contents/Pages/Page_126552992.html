顔モデル

顔モデルとは Mii の頭部分のモデルのことを指します。

似顔絵ライブラリでは Mii データ（CharInfo）に設定されたパラメータに応じた顔モデルを描画するための仕組みを提供します。

顔モデル構成パーツ

設定されたパラメータに応じた顔モデルを描画するため、 顔モデルは複数のシェイプの組み合わせで構成されます。1 体分の Mii の顔モデルを描画するためのシェイプとテクスチャのデータは nn::mii::CharModel で管理しています。

顔モデルを構成する各パーツと nn::mii::CharModel::DrawType 列挙型は関連しており、以下の種類があります。

顔モデルを構成するパーツの中には、その Mii に不要なパーツもあります。例えば、メガネや顎鬚などのパーツは Mii によっては描画するパーツがない場合があります。不要なパーツに関する実装上の注意点に関しては CharModel の機能説明のページを確認してください。

顔モデルの種類

顔モデルには、通常のモデルに加え、帽子などの装飾品を被らせるためのモデルなどいくつか種類があります。

似顔絵ライブラリでは用途に合わせた顔モデルを生成する機能を提供しています。

顔モデルの種類 サンプル 解説
nn::mii::CreateModelType_Normal 通常の顔モデルとして生成します。
nn::mii::CreateModelType_Hat 帽子用顔モデルとしてモデルを生成します。帽子、髪の毛、おでこのパーツが通常モデルと異なります。
nn::mii::CreateModelType_FaceOnly 被り物用顔モデルとして生成します。帽子、髪の毛、おでこのパーツが描画されません。

さらに、フルフェイスのヘルメットのような被り物用に鼻の高さを制限した顔モデルの生成可能です。

鼻の種類 サンプル 解説
 nn::mii::CreateNoseType_Normal 通常の鼻の高さのモデルを生成します。
 nn::mii::CreateNoseType_Flatten

鼻の高さを調整してモデルを生成します。鼻の高さが制限され、一定の値以下に調整されたモデルになります。

顔モデルのモデルの種類（nn::mii::CreateModelType）と鼻の高さの種類（nn::mii::CreateNoseType）は組み合わせて指定することが可能です。詳しくは、CharModel の機能説明のページの顔モデルの生成条件の項目を参照してください。

注意事項

顎鬚の種類

顎鬚には、シェイプとして表現される顎鬚とテクスチャとして表現される顎鬚があります。

シェイプで表現される顎鬚の場合、顔モデルのパーツの一つとして表現されます。テクスチャで表現される顎鬚の場合、顔輪郭テクスチャの一部として表現されます。

シェイプ顎鬚 テクスチャ顎鬚

帽子の種類

帽子パーツとして描画されるものには、帽子以外に髪留めがあります。

髪留めも帽子として扱っています。帽子モデルでも髪留めが描画されることに注意してください。

通常モデル 帽子モデル
