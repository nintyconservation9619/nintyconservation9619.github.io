 nn::g3dニュース

3Dテクスチャーコンバーター

--gpu-encoding false を指定しない場合に GPU エンコーディング不可能ならエラーにするように仕様変更

--gpu-encoding オプションに false を指定していない場合に GPU エンコーディング不可能ならエラーにするように仕様変更しました。 従来のように可能なら GPU でエンコードして、不可能なら CPU でエンコードしたい場合は --gpu-encoding auto を指定してください。

--gpu-encoding の値 動作内容
未指定 true 指定時と同様の動作
true 可能なら GPU エンコードし、不可能ならエラー
false 常に CPU エンコード
auto 可能なら GPU エンコードし、不可能なら CPU でエンコード（これまでの true の動作）

Photoshop プラグインや Maya プラグインではコンフィグファイルの gpu_encoding="true"/"false"/"auto" で動作を変更することができます。

bc3、bc7、astcフォーマットに圧縮するときのアルファ成分品質の改善



テクスチャーコンバーター（%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dTextureConverter.exe）でbc3、 bc7、astcフォーマットに圧縮するときのアルファ成分品質の改善をしました。

改善前 改善後

現在の環境でGPUエンコーディングが可能かどうかを判定できるように

テクスチャーコンバーター（%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dTextureConverter.exe）で以下のオプションを指定することで現在の環境がGPUエンコーディング可能な環境かどうかを判定できるようになりました。

3dTextureConverter.exe --check-gpu-encoding

GPUエンコーディングが可能なら3Dテクスチャーコンバーターの終了コード（%ERRORLEVEL%）が0になります。

3Dシェーダーコンバーター

マシン分散コンパイルを利用するための機能追加

マシン並列で分散コンパイルができるように一部のシェーダーバリエーションのみをコンパイルする --variation-begin-ratio オプション、--variation-end-ratio オプションと、コンパイル以外の処理をスキップする --skip-convert オプションを追加しました。詳しい利用方法は3D シェーダーコンバーター、分散コンパイルを参照してください。

G3dDemo サンプル

ShapeAnimation デモで使用するリソースを変更

ShapeAnimation デモで、シェイプアニメーションを行うリソースを更新しました。

  • 頂点数とキーシェイプ数が増えたことで、CPU と GPU の処理の差が分かりやすくなりました。
  • 頂点位置だけでなく、法線もアニメーションするように変更しました。

 nn::atkニュース

サウンドアーカイブをサウンドアーカイブプレイヤーの初期化後に追加できる機能の追加

サウンドアーカイブをサウンドアーカイブプレイヤーの初期化後に追加できる、追加サウンドアーカイブの機能を追加しました。 

この機能により、必要な場面でサウンドアーカイブプレイヤーにサウンドアーカイブを追加し、必要がなくなった段階で削除することができるようになります。
特定の場面でのみ必要とされるサウンドや、 追加コンテンツに含まれるサウンドでの使用に適しています。

詳しくは、以下のページをご確認ください。

  • ガイド／追加サウンドアーカイブ
  • プログラミングガイド／リソースの読み込み

nn::spyニュース

nn::atk の追加サウンドアーカイブ機能への対応

nn::atk の追加サウンドアーカイブ機能への対応のため nn::spy::atk::AtkSpyModule::InitializeArg 構造体に addonSoundArchiveCountMax メンバー変数が追加されています。
