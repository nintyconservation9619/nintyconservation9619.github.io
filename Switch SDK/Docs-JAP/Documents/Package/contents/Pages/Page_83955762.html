プレイヤー

プレイヤー

プロジェクトツリーで、プレイヤーのノードを選択することで、サウンドセットタブにプレイヤーを表示することができます。
この時表示されるリストを、プレイヤーリストと呼びます。

プレイヤーの新規作成と追加

プレイヤーを新規作成または追加する場合は、サウンドセットタブのリスト領域を選択して、[プロジェクト]→[アイテムの追加]→[プレイヤー…] を実行します。

プレイヤーリストで、右クリックを行うと現れるコンテキストメニューによる操作でも、プレイヤーを新規作成または追加することができます。

プレイヤーの編集

プレイヤーの編集はプレイヤーリストで行います。

プレイヤーリストの定義要素

ここではプレイヤーリストの各定義要素の意味を解説します。

名前

プレイヤーの名前を指定します。

名前に使用できる文字には制限があり、以下の文字が使えます。

  • 先頭は、半角英字とアンダーバー [a-zA-Z_]
  • 先頭以外は、半角英数字とアンダーバー [a-zA-Z0-9_]

サウンドリミット

プレイヤーで同時に再生できるサウンドの数を指定します。

プレイヤーはこの値で指定された最大再生数を超えるサウンドのリクエストを受けた場合には、各サウンドのプレイヤープライオリティによって再生するサウンドを選別します。この選別に漏れたサウンドは、実際の発音の有無に関わらず停止されます。
サウンドの最大再生数の指定は仮想ボイスの優先順位で、物理ボイスを規定する設定ではない点に注意してください。

この値はシーケンスサウンドやウェーブサウンドのような「サウンド」の最大再生数を指定します。
各サウンドには複数の MIDI チャンネルやボイスが存在しますが、これら個別のボイスに対する優先順位の設定はチャンネルプライオリティによって行います。チャンネルプライオリティは各サウンドの再生が、物理ボイスとして発音される段階で、そのボイス単位に適用されます。

ヒープサイズ

プレイヤーが使用する、プレイヤーヒープのサイズを指定します。

プレイヤーヒープについての詳しい解説は、「プレイヤーヒープについて」をご覧ください。
