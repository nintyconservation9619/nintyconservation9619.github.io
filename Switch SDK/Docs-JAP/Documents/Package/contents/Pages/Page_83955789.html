概要

パラメータパネルとは？

パラメータパネルは、サウンドの各種パラメータを GUI によって編集するパネルです。

サウンドセットタブで選択している各種サウンドリストに対する、音量や優先度などの値や、チェックボックスの状態の入力を行ったり、バンクタブで選択しているインストルメントに対するエンベロープやパンなどの入力を行います。
エンベロープの値は、グラフで視覚的に確認しながら操作することができます。

 

パラメータパネルの役割

パラメータパネルは、サウンドセットタブ、バンクタブと連動して動作します。
数値は、スライダーやテキストボックス横の上下ボタンにより、リスト上での編集よりも柔軟に操作ができます。

バンクタブにおいては、インストリストやパーカッションリストと組み合わせて利用することで、数値表示ではイメージしにくいエンベロープなどの設定を補助します。
サンプルマップと一緒に使うことによって、サンプルマップでは編集できないエンベロープやパンの指定を行うことができます。

パラメータパネルはフローティングダイアログとして表示され、SoundMaker のメインウインドウの外に移動することができます。

パラメータパネルの操作

パラメータパネルを開く

[表示]→[パラメータパネル] またはメインウインドウのツールバーにあるパラメータパネルアイコン により、パラメータパネルを開くことができます。
この操作には Ctrl + I および Alt + 8 のショートカットキーが割り振られています。
これらの操作は、ウインドウの表示・非表示のトグル動作になります。

各リストがアクティブな状態でないと、パラメータパネル内の各設定値は表示されません。

パラメータパネルが操作する対象

パラメータパネルは、常にアクティブなサウンドセットタブ、バンクタブに対してパラメータの表示と編集を行います。

パラメータの編集

パラメータパネルでの編集操作について、バンクタブを対象とした場合の説明をします。

エンベロープの編集

バンクタブ内の任意の場所でインストルメントかリージョンを選択し、パラメータパネルでエンベロープの操作を行うことで、選択したインストルメントまたはリージョンのエンベロープの値を編集することができます。

値の編集は、各フェーズのスライダーの操作で行います。
作成したエンベロープカーブは、パラメータパネル内に表示されるグラフで確認することができます。アタック、ホールド、ディケイ、リリースの各時間については、グラフの上部にミリ秒単位で正確に表示されます。

マルチリージョンのインストルメントをインストリストで選択しているなど、エンベロープが編集不可能な時は、スライダーの操作は受け付けません。

音量の編集

音量の値は、インストルメントとリージョンに各々独立して指定することができます。

バンクタブ内の任意の場所でインストルメントかリージョンを選択し、パラメータパネルで音量の操作を行うことで、選択したインストルメントまたはリージョンの音量の値を編集することができます。

ピッチの編集

ピッチの値は、インストルメントとリージョンに各々独立して指定することができます。

バンクタブ内の任意の場所でインストルメントかリージョンを選択し、パラメータパネルでピッチの操作を行うことで、選択したインストルメントまたはリージョンのピッチの値を編集することができます。

ピッチの操作は半音単位、セント単位（半音の 1 / 100）のいずれか、またはこれらを組み合わせて指定を行います。

パンの編集

バンクタブ内の任意の場所でリージョンを選択し、パラメータパネルでパンの操作を行うことで、選択したリージョンのパンの値を編集することができます。
値の編集は、パンのスライダーの操作で行います。

マルチリージョンのインストルメントをインストリストで選択しているなど、パンが編集不可能な時は、スライダーの操作は受け付けません。
