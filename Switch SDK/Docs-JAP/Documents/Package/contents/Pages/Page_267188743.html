本体更新ツールによる更新
注意：

NX Addon 4.3.0 から追加された機能です。

本体更新ツールとは

NX Addon 4.3.0 から開発機の本体更新ツールを追加しました。 (ただし EDEV 向け PC ツールは NX Addon 4.4.1 から)
これは初期化ツールによる初期化に比べて設定等が保存されたままファームウェアのバージョンアップができるという特徴があります。

ただし、NX Addon 4.0.0 以降からの更新しかできません。それ以前のファームが書かれた開発機の更新はできませんのでご注意ください。

更新方法

DevKitUpdaterパッケージ (と SystemUpdater for NX パッケージ) をインストールし、以下のいずれかの方法で更新することができます。

SDEV の PC からの更新

DevKitVersionUpdater パッケージに含まれるツール SystemUpdateSdev.exe を用いることで更新できます。

詳しくは SDK ドキュメントをご覧ください。

EDEV の PC からの更新

DevKitVersionUpdater パッケージに含まれるツール SystemUpdateEdev.exe を用いることで更新できます。

詳しくは SDK ドキュメントをご覧ください。

SDEV/EDEV の NX 開発カードからの更新

ファイル DevKitUpdarXXX.nsp を NX 開発カードに書き込んで更新することができます。

  1. DevKitVersionUpdater パッケージと NX Addon パッケージをインストールする
  2. 開発カードを開発機に挿す

  3. 以下のコマンドを実行して開発カードにイメージを書き込む

    Tools\CommandLineTools\RunOnTarget.exe TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- gamecard write %NINTENDO_SDK_ROOT%\Resources\Firmwares\NX\DevKitUpdaterEdevK5.nsp
  4. 書き込んだカードを更新したい SDEV or EDEV に挿す
  5. 下の写真のような緑のアイコンが表示されるので起動する (HomeMenu の場合も表示されます)

  6. LCD に更新画面が表示されるのでしばらく待ちこのような画面になれば更新完了です

ただし、この方法は SDEV の場合 DEV MODE の更新では利用できません。上記 PC からの更新方法を使って下さい。

SDEV/EDEV の microSD カードからの更新

microSD カードを用いて更新することも可能です。

この方法を使う場合は NX Addon パッケージも必要です。

  1. PC に microSD カードを挿し、ファイル %NINTENDO_SDK_ROOT%\Resources\Firmwares\NX\DevKitUpdaterXXX.nsp をそのカードにコピーする
    1. 上記 XXX の部分は EDEV (DEV MODE) の場合は EdevK5、SDEV/EDEV (PROD MODE) の場合は NxProd、SDEV/EDEV (ログ出力機能付き PROD MODE) の場合は NxProdLog になります
  2. コピーした microSD カードを更新したい SDEV or EDEV に挿す
  3. DevMenu の "Install from SD" 機能を使って microSD カードに書いたファイル DevKitUpdaterXXX.nsp をインストールする
  4. インストールしたタイトルの緑アイコンが表示されるのでを起動する
  5. LCD に更新画面が表示されるのでしばらく待ちこのような画面になれば更新完了です

ただし、この方法は SDEV の場合 DEV MODE の更新では利用できません。上記 PC からの更新方法を使って下さい。

制約事項

ツールリファレンスの注意事項も参照して下さい

  • バージョンダウンおよび同一バージョンへの更新はできません
    • ツールは成功しますが更新されません
  • NX Addon 3.X 以前のファームが書かれた開発機からは利用できません
