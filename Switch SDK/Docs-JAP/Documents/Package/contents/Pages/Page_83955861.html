テクスチャーパターンアニメーション設定プラグイン

テクスチャーパターンアニメーションを設定している場合に出力されるテクスチャーは、デフォルトでは file ノードのイメージの名前（Image Name）アトリビュートで指定したものとイメージ番号（Image Number）（フレーム拡張子（Frame Extension）アトリビュート）に打たれたキーの値に対応するものだけですが、このプラグインを使用すると特定のフレーム拡張子に対応するテクスチャーを強制的に出力することができます。

フレーム拡張子アトリビュートにエクスプレッションやアニメーションレイヤを接続する場合および直接キーを設定しない場合は、かならずこのプラグインで出力するテクスチャーのリストを設定する必要があります。

参考：

file ノードのイメージシーケンスの使用（Use Image Sequence）が ON であれば、イメージ番号（フレーム拡張子アトリビュート）にアニメーションが設定されていなくても、このプラグインで設定したテクスチャーは出力されます。

使用方法

「Nintendo > Set Texture Pattern...」を選択するとウィンドウが表示されます。

Selected File Node 欄には、選択されている file ノードが表示されます。

設定方法

file ノード（または マテリアルノード）を 1 つだけ選択すると、Set 欄には選択したノードの現在の設定が表示されます。

中間ファイルに出力するイメージ番号（フレーム拡張子）のリストを入力し、Enter キーで決定します。
このとき、file ノードのイメージシーケンスの使用（フレーム拡張子の使用（Use Frame Extension）アトリビュート）が OFF だとエラーになります。

リストの書式は

1,2,3

のように出力する番号を「,(カンマ)」で区切って並べます。

この例では、file ノードの Image Name が "name.1.tga" だとすると

　"name.1.tga"
　"name.2.tga"
　"name.3.tga"

の 3 つのテクスチャーがフレーム拡張子アトリビュートにキーが打たれてなくても強制的に出力されます。指定した番号に対応するテクスチャーが存在しないと出力時に警告が表示されます。またマイナスの番号を指定することはできません。

連続した番号なら

1,2,3,4,5

と書くかわりに

1-5

のように「-（ハイフン）」でつないで指定することもできます。

3-6,10

のように指定すると 3,4,5,6,10 と書いた場合と同じになります。

カンマのかわりに半角スペースで区切ることも可能です。

次のような書き方はエラーになります。

1-3-5 （ハイフンで 2 つ以上の数値をつないでいる）
-3    （ハイフンから始まっている）
5-    （ハイフンで終わっている）

01,02 と書いた場合は 1,2 と書いた場合と同じ効果になります。
"name.1.tga" と "name.01.tga" の両方が存在する場合は、常に 0 がついていないほうのテクスチャーが出力されます。

ウィンドウを閉じずに別のノードを選択すると、別のノードを設定することができます。
