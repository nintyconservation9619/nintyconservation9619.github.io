エミッタセットノードの操作

ここではワークスペースにおけるエミッタセットノードの操作方法について説明します。

準備

関連リンク クイックスタート 「エフェクトの作成」 を完了させた状態から作業を開始します。

ここまでの手順において、ワークスペースのエミッタセット「 EmitterSet1」直下には以下の様に複数のエミッタが作られています。

エミッタセット名の変更

作成したエミッタセット名の変更方法について説明します。
ワークスペースのエミッタセット「 EmitterSet1」を選択した状態で、以下いずれかの操作を行います。

  • メインメニュー項目［ファイル］→［ 名前を変更］を選択します。
  • マウス右ボタンクリックで開くメニューから「 名前変更」を選択します。
  • ショートカットキー F2 を使用します。

名前の変更が可能になりますので、エミッタセット名を「Flow」と入力し、Enter キーで確定します。
「 EmitterSet1」が「 Flow」に変更されます。

エミッタセットの複製

選択したエミッタの複製方法について説明します。
ワークスペースのエミッタセット「 Flow」を選択した状態で以下いずれかの操作を行います。

  • メインメニュー項目 [編集]→[ 複製]を選択します。
  • マウス右ボタンクリックで開くメニューから「 複製」を選択します。
  • ツールバーにある「 複製」ボタンを押します。
  • ショートカットキー Ctrl + D を使用します。

ワークスペース上にはエミッタセット「 Flow_Copy1」が追加されています。

エミッタセットの並び替え

複数エミッタセットの並び替え方法について説明します。
ワークスペース上でエミッタセット「 Flow」が「 Flow_Copy1」の上にある状態で「 Flow」を選択します。
以下の操作を行います。

  • Alt キーを押しながらマウス左ボタンクリックで「 Flow」を選択、ドラッグで「 Flow_Copy1」の下に移動します。

ワークスペース上でエミッタセット「 Flow_Copy1」の下に「 Flow」が移動します。
