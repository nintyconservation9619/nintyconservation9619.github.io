はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている、パフォーマンス計測機能を提供する PERF ライブラリについて説明します。

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象としています。

各章の説明

「マニュアル」では、個々の PERF ライブラリ機能を使用する上で理解しておかなければならない特筆事項を説明しています。

関連資料

PERF の関数リファレンス と併せて参照して下さい。

PERF ライブラリの機能

PERF ライブラリは、CPU と GPU のパフォーマンス計測機能および計測結果の集計機能を提供します。
PERF を使用することで、システムの tick の差分を集計しパフォーマンス情報を簡易的に確認できるようになります。

PERF ライブラリが提供する機能

現在の PERF ライブラリでは以下の機能を提供しています。

  • パフォーマンス計測結果集計
  • CPU パフォーマンス計測
  • GPU パフォーマンス計測
参考：

perf ではアプリケーションレベルでの計測と集計を行います。
レジスタを読み出してハードウェアレベルでのプロファイリングを行う CPU および GPU プロファイラツールは今後のリリースで別途提供される予定です。

それぞれの機能については マニュアル を参照して下さい。
