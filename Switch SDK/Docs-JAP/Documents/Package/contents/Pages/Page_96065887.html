スケールアニメーション

ここではカーブエディタでパーティクルスケールのアニメーションを設定する機能「スケールアニメーション」について説明します。

概要

「スケールアニメーション」は、カーブエディタにより、パーティクル寿命の範囲でパーティクルのスケールアニメーションができる機能です。
例えばカーブエディタの値で 100 ％から 0 ％のアニメーションを作成した場合、パーティクル全体を基準値から徐々に消滅させることができます。
以下は「 設定項目」の概要です。

スケールアニメーション

横軸を「パーティクル寿命」（時間）を 100 % とした割合、縦軸をパーティクルの基本サイズである「基準値」（値）としたグラフでアニメーションを作成及び編集します。
