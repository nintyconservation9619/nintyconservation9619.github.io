NintendoSDK 0.17.0 で行われた NintendoWare オーディオ (nn::atk) の変更点について説明します。

リリース時優先度固定の機能が無効化されない不具合の修正

SoundMaker でリリース時優先度固定のオプションを無効に設定しても、リリース時の優先度が最低にならない不具合を修正しました。
この修正により、アプリケーション実行中の発音優先度判定の結果が変わるため、サウンドの聞こえ方が以前と異なるようになる可能性がある点にご注意ください。
