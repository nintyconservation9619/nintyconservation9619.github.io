レイアウトにおけるリニア空間

レイアウトではリニアワークフローに対応しています。

レイアウトでのメリットは？

主に3D描画にメリットが多く、レイアウトでのわかり易いメリットは少ないです。
ただし、3Dとガンマ特性をそろえると、

  • 最終的な統合のために余計な追加処理が回避できる
  • 3Dモデルと、レイアウトで色あわせをする際に簡単

のように余計な手間が少なくなり、これが一番のメリットといえます。

プロジェクトの途中で方式を切り替えた場合、どのくらい影響があるのか？

グラデーションのかかり方や、アニメーションの変化の仕方が変わるのでレイアウトデータの調整は必須と考えてください。
ほぼ、すべてのレイアウトでカラー設定やアニメーション設定をやり直す必要があります。

  • テクスチャ
  • 頂点カラー
  • カラー補間
  • 各種カラーアニメーション
  • 透明度を設定したペイン
