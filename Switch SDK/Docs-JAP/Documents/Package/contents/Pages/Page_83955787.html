モデルプロパティウィンドウ プレビューページ

選択したモデルのプレビューに関する情報を確認して編集するページです。

A バインドするモデルを選択します。
バインドするモデルは、3DEditor に読み込まれているモデルの中から選択します。
B A で選択したモデルのボーンの中から、バインドするボーンを選択します。
C バインドするボーンの表示順序を「名前順」「番号順」か選択します。
D スケール を X Y Z で編集します。
E スケールの X Y Z を連動して編集する場合は、チェックを入れます。
F 回転を X Y Z で編集します。
G 移動を X Y Z で編集します。
H 編集した設定をリセットして初期状態に戻します。
リセットには全てリセットか、「スケール」「回転」「移動」の個別リセットが選択できます。
I LOD が設定されているモデルの場合に、LOD のレベルを選択してプレビューすることが可能です。
J 頂点量子化を無効にしてモデルをプレビューする場合は、チェックを外します。

LOD プレビューの切り替え

シェイプ LOD を持つモデルを読み込んだ場合は、LOD の各レベルをプルダウンメニューから選択してプレビューできます。
