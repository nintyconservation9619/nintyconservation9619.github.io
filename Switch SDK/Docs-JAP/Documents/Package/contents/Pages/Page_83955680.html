コンバイナ

  このタブではパーティクルのカラーやアルファの合成設定を行います。

カラー・アルファコンバイナ

「 カラー・アルファコンバイナ」は、パーティクルのカラーやアルファ要素をコンバイナ式により合成する機能です。
以下はグループ内設定項目の一覧です。

カラーコンバイナ

パーティクルのカラー要素を合成する機能です。
アルファコンバイナ   パーティクルのアルファ要素を合成する機能です。

パーティクルシェーダ

「 パーティクルシェーダ」は、パーティクルに対してシェーダにより特殊効果を与える機能群です。
以下はグループ内設定項目の一覧です。

シェーダタイプ 登録したテクスチャでパーティクルの描画に特殊な効果を与える機能です。
