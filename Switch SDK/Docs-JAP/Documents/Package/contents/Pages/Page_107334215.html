テセレーションステート

テセレーションステートは、レンダリングパイプラインにおけるテセレーション機能についてのいくつかのステートをまとめて表現します。

情報

テセレーションステートの情報は nn::gfx::TessellationStateInfo として定義されています。

パッチの制御点の数

SetPatchControlPointCount 関数により、パッチプリミティブの制御点の数を指定します。

オブジェクト

テセレーションステートのオブジェクトは nn::gfx::TessellationState として定義されています。

初期化と破棄

テセレーションステートの初期化と破棄はオブジェクトの規則に従います。

テセレーションステートのセット

nn::gfx::CommandBuffer クラスの SetTessellationState 関数を呼ぶことによってテセレーションステートをセットします。
