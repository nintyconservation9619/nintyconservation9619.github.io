3 ボーンビジビリティーアニメーション

ボーンビジビリティーアニメーションを作成します。

可視性

アウトライナなどからモデル「Visivilty_Model」を選択して「可視性」にアニメーションキーを作成します。

下表のようにキーを作成します。

 

  0 フレーム 25 フレーム
可視性 オン オフ

下図のようなボーンビジビリティーアニメーションになることを確認します。

出力対象

可視性アニメーションの出力対象は「可視性」のキーフレームです。

「Nintendo Export Options」の「Output File Selection」の A「Bone Visibility Animation Data」に対応しています。

関連リンク 詳しくは ボーンビジビリティーアニメーション を参照してください。
