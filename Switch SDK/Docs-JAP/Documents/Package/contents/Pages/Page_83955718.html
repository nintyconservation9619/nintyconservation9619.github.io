プロジェクトファイルフォーマット

プロジェクトファイルは、XML形式のファイルです。
ここでは、プロジェクトファイルに含まれる要素について説明します。

出力例

プロジェクトファイルフォーマットの出力例です。

<?xml version="1.0" encoding="utf-8"?>
<Project Version="3.6.0.0">
    <file_info>
        <create user="user name" host="host name" date_time="2000-01-01T00:00:00+09:00" tool_name="3DEditor" tool_version="3.4.0" />
        <modify user="user name" host="host name" date_time="2000-01-01T00:00:00+09:00" tool_name="3DEditor" tool_version="3.6.0" />
    </file_info>
    <Scene PreviewDefaultAnimationSet="true">
        <Animation Name="Scene.fsnb" />
    </Scene>
    <Item Path="Human.fmdb" PreviewDefaultAnimationSet="false">
        <AnimationSet Name="run" Preview="true">
            <Animation Name="Human_run.fskb" ItemIndex="2" Preview="true" />
        </AnimationSet>
        <Animation Name="Human_walk.fskb" ItemIndex="3" Preview="true" />
        <Material>
            <Name>human_all</Name>
            <OptimizeShader>true</OptimizeShader>
            <OptimizeShaderOnReload>true</OptimizeShaderOnReload>
        </Material>
        <Preview ScaleX="2" ScaleY="2" ScaleZ="2" RotateX="0" RotateY="0" RotateZ="0" TranslateX="0" TranslateY="0" TranslateZ="0" Visible="true">
    </Item>
    <Item Relative="SearchPath" Base="Textures" Path="HumanA.ftxb" />
    <Item Relative="Project" Path="Human_run.fskb" />
    <Item Relative="Project" Path="Human_walk.fskb" />
    <Item Relative="Project" Path="../shaders/shader.fsdb" />
    <Item Relative="SearchPath" BasePath="%NINTENDO_SDK_ROOT%/../Textures" Path="HumanB.ftxb" />
    <comment color="1 0 0" comment="comment" />
</Project>

<Project>

プロジェクトファイルのルート要素です。

属性

名前 説明
Version プロジェクトファイルのバージョン番号です。

内容

以下の要素が出現します。

  • <file_info> : プロジェクトファイルの作成や変更に関する記録を保存する要素です。
    省略可能です。
  • <Scene> : シーンのアニメーションセットの情報を格納する要素です。
    1個出現します。
  • <Item> : プロジェクトに含まれるファイルを表します。
    0個以上出現します。
  • <comment> : 編集用コメント等のメタ情報を格納する要素です。
    省略可能です。

<file_info>

プロジェクトファイルの作成や変更に関する記録を保存する要素です。
プロジェクトファイルを読み込んだときに <file_info> が存在しなければ、保存時にも出力されません。

内容

以下の要素が高々 1 回出現します。

  • <create> : プロジェクトファイルの作成に関する記録を保存する要素です。
  • <modify> : プロジェクトファイルの変更に関する記録を保存する要素です。

<create>

プロジェクトファイルの作成に関する記録を保存する要素です。

属性

名前 説明
user プロジェクトファイルを作成したユーザーの名前です。
host プロジェクトファイルを作成したマシンの名前です。
date_time プロジェクトファイルを作成した日時です。ISO8601 に準拠した、「YYYY-MM-DDThh:mm:ssTZD」書式で指定します。
tool_name プロジェクトファイルを作成したツールの名前です。
tool_version プロジェクトファイルを作成したツールのバージョン番号です。

<modify>

プロジェクトファイルの変更に関する記憶を保存する要素です。

属性

名前 説明
user プロジェクトファイルを変更したユーザーの名前です。
host プロジェクトファイルを変更したマシンの名前です。
date_time プロジェクトファイルを変更した日時です。ISO8601 に準拠した、「YYYY-MM-DDThh:mm:ssTZD」書式で指定します。
tool_name プロジェクトファイルを変更したツールの名前です。
tool_version プロジェクトファイルを作成したツールのバージョン番号です。

<Scene>

シーンのアニメーションセット情報を格納する要素です。

属性

名前 説明
PreviewDefaultAnimationSet 既定アニメーションセットを接続先アプリケーションでのプレビュー対象とする場合、 true を指定します。

内容

以下の要素が、それぞれ 0 回以上出現します。

  • <AnimationSet> : シーンの既定でないアニメーションセットを表します。
  • <Animation> : シーンの既定アニメーションセットに含まれるアニメーションを表します。

<Item>

プロジェクトに含まれるファイルを表します。

属性

名前 説明
Relative Path の基点の種類を表します。Project, SearchPath, StartUpFolder のいずれかを指定します。それぞれ、プロジェクトファイルのパス、追加参照パス、Startup フォルダのパスを表します。
Base 追加参照パスの名前です。Path が追加参照パスからの相対パスでない場合は省略します。
BasePath 追加参照パスのパスです。Path が % で囲われた環境変数から始まる絶対パスからの相対パスでない場合は省略します。
Path ファイルパスです。
PreviewDefaultAnimationSet 既定アニメーションセットを接続先アプリケーションでのプレビュー対象とする場合、trueを指定します。<Item> がモデルファイルを指さない場合は省略します。
RetargetingHostModelName スケルタルアニメーションのリターゲット先モデルの名前です。<Item> がスケルタルアニメーションファイルを指さない場合やリターゲット先モデルが指定されていない場合は省略します。
参考：

Version 3.5.1.0 までは Relative に StartUp を指定でき、追加参照パスを表していました。
3DEditor では StartUp が指定された旧バージョンのプロジェクトファイルも読み込むことができます。

内容

<Item> がモデルファイルを指す場合は、以下の要素が出現します。

  • <AnimationSet> : モデルにバインドされる既定でないアニメーションセットを表します。
    0 回以上出現します。
  • <Animation> : モデルの既定アニメーションセットに含まれるアニメーションを表します。
    0 回以上出現します。
  • <Preview> : モデルのプレビューに関する情報を表します。
    高々1回出現します。
  • <Material> : モデルに含まれるマテリアルのプレビューに関する情報を表します。
    0 回以上出現します。

<AnimationSet>

アニメーションセットを表します。

属性

名前 説明
Name アニメーションセットの名前です。
Preview アニメーションセットを接続先アプリケーションでのプレビュー対象とする場合、trueを指定します。

内容

以下の要素が 0 回以上出現します。

  • <Animation> : アニメーションセットに含まれるアニメーションです。

 

<Animation>

アニメーションセットに含まれるアニメーションを表します。

属性

名前 説明
Name アニメーションのファイル名です。
ItemIndex アニメーションの <Item> の <Project> 内のインデックスです。
Preview アニメーションセット内のアニメーションを接続先アプリケーションでのプレビュー対象とする場合、 true を指定します。

<Preview>

モデルのプレビューに関する情報です。

属性

名前 説明
BindModelName バインドするモデルを指定します。
バインドするモデルは、3DEditor に読み込まれているモデルの中から選択します。
BindBoneName BindModelName で指定したモデルのボーンの中から、バインドするボーンを指定します。
ScaleX スケールの X を指定します。
ScaleY スケールの Y を指定します。
ScaleZ スケールの Z を指定します。
RotateX 回転の X を指定します。
RotateY 回転の Y を指定します。
RotateZ 回転の Z を指定します。
TranslateX 移動の X を指定します。
TranslateY 移動の Y を指定します。
TranslateZ 移動の Z を指定します。
Visible モデルをプレビュー対象とする場合、trueを指定します。
ShowInObjView モデルをオブジェクトビューで表示する場合、trueを指定します。

<Material>

モデルに含まれるマテリアルのプレビューに関する情報です。

内容

以下の要素が、高々 1 回出現します。

  • <Name> : マテリアルの名前です。
  • <OptimizeShader> : 最適化シェーダを送信する場合、trueを指定します。
  • <OptimizeShaderOnReload> : マテリアルの変更に追従してシェーダを再生成する場合、trueを指定します。

<comment>

3DEditor で使われているプロジェクトの編集用データです。

属性

名前 説明
color 編集用カラーを "1 0.5 0" といったフォーマットで指定します。未指定の場合は空文字列を指定します。
comment 編集用コメント文を一行で指定します。
