シーケンスサウンドセット

プロジェクトツリーで、シーケンスサウンドセットのノードを選択することで、サウンドセットタブにシーケンスサウンドセットのリストを表示することができます。
この時表示されるリストは、サウンドセットタブ内を上下に分割して表示されます。
分割されたタブのうち、上をシーケンスサウンドセットリストと呼び、下をシーケンスサウンドリストと呼びます。

シーケンスサウンドセットの役割

シーケンスサウンドセットには、シーケンスサウンドと同様のデータを登録できますが、複数のシーケンスサウンドを１つのシーケンスサウンドセットにまとめることができます。
グループへの登録は、シーケンスサウンドセット単位でも行うことができますので、シーケンスサウンドセットを使うことで、シーケンスデータのグループ管理が楽になります。（シーケンスサウンドセット内のシーケンス単体もグループに登録可能です。）
シーケンスサウンドは主に BGM 用、シーケンスサウンドセットは主に SE 用というような使い方が考えられます。

シーケンスサウンドセットの新規作成と追加

シーケンスサウンドセットを新規作成または追加する場合は、サウンドセットタブのリスト領域を選択して、[プロジェクト]→[アイテムの追加]→[シーケンスサウンドセット…] を実行します。

シーケンスサウンドセットリストで、右クリックを行うと現れるコンテキストメニューによる操作でも、シーケンスサウンドセットを新規作成または追加することができます。

シーケンスサウンドの新規作成と追加および編集

シーケンスサウンドの新規作成と追加、および編集については、『シーケンスサウンド』 のページの各項目を参照してください。
