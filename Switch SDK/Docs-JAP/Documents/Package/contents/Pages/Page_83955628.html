機能説明

ここでは、GFX におけるそれぞれの機能についての詳細を説明します。

新機能と旧機能

低レベルグラフィックス API の世代間には、新しい概念の追加などいくつかの大きな変更があります。

GFX はこれら新しい方式を用意しますが、移行用のためや、新しい低レベルグラフィックス API で動作させる必要のない場合のために、古い方式のインターフェイスもサポートします。 ただし、古い方式では新しい低レベルグラフィックス API では動作しない可能性があります。一方で、新しい方式ではすべての低レベルグラフィックス API が動作することを目指しています。これは、古い低レベルグラフィックス API で新しい方式を使う場合はエミュレーションが行われることを意味します。

新機能と旧機能の対応表

以下は新機能とそれに対応する旧機能の一覧です。
低レベルグラフィックス API のバリエーションによっては、 旧機能のほうでは動作できません。

新機能 旧機能
メモリープール メモリープールを使わずオブジェクトを作成
デスクリプタープール コマンドバッファーに直接的にリソースをセット
リソースステート キャッシュのフラッシュと無効化
パイプライン シェーダーと各種ステート

アルファテストについて

アルファテストの機能は現在、gfx によって提供されていません。アルファテストと同等の表現を行いたい場合は、ピクセルシェーダーにおいて discard を使ってフラグメントを破棄することによって、またはもし低レベルグラフィックス API にアルファテストの機能があれば低レベルグラフィックス API と併用することによって、代用することができます。
