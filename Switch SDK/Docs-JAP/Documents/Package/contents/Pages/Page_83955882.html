マテリアルアトリビュート設定プラグイン

マテリアルに NintendoWare 用のマテリアルのアトリビュートを設定します。このプラグインで設定した内容は、3ds Max の画面上には反映されませんのでご注意ください。

Set Material Attribute は、以下のいずれかの方法で設定してください。

マテリアル毎に個別に設定する方法

NintendoWare のデータ作成プラグインは、複数のマテリアルに対して同時に検索したり設定できるように設計しています。

マテリアルエディタ上でマテリアル毎に NintendoWare 用のマテリアル情報を設定したい場合は、以下の方法で設定することも可能です。

マテリアルを選択し、マテリアルエディタの[Nintendo]メニューから [Add Material Attribute] を実行すると、NintendoWare 用のマテリアル情報が付加されます。

上記プラグインを使った場合も自動的に以下の情報が付加されます。 ここでマテリアル毎に直接設定を変更することもできます。
[Nintendo]メニューから [Delete Material Attribute] を実行することで、マテリアルに付加した NintendoWare 用の情報を削除することもできます。

マテリアルにもユーザーデータを設定する事ができます。
マテリアルを選択しマテリアルエディタの [Nintendo] メニューから [Set Material User Data] を実行すると、設定ウィンドウが表示されます。
設定方法はユーザーデータ設定プラグイン（Set User Data）を参照してください。

[マルチ/サブ マテリアル] にマテリアル情報を設定した場合は、サブマテリアル全てに同じマテリアル情報が出力されます。
また、サブマテリアルにそれぞれマテリアル情報を設定することで、サブマテリアルごとに違うマテリアル情報を設定できます。
サブマテリアルにマテリアル情報が設定されていない場合は、マルチ/サブマテリアルの情報が出力されます。

参考：

NintendoWare 専用の付加情報を設定した場合も、逆に削除した場合も、3ds Max のワークビューやデータ構成には影響は与えません。

複数のマテリアルを一括して設定する方法

Nintendo > Set Material Attribute を実行するとウィンドウが表示されます。

このウインドウを使って、複数のマテリアルに対して同時に設定を変更したり、検索することができます。
オブジェクトを複数選択することで、それらに割り当てられているマテリアルを選択します。

Selected Material に名前が表示されているマテリアルが設定対象となります。
この状態で Set 項目の内容を変更すると、変更した項目の内容が反映されます。
複数のマテリアルに対して同時に変更することもできます。 ただし、変更していない項目については設定は変わりません。

  • Compress
     Export プラグインの Compress Material オプションを ON にしたときに、圧縮されないようにするためのフラグを設定します。
    Compress if possible にすると、他のマテリアルとアトリビュートが同じなら圧縮されます。これがデフォルトです。
    Don’t compress にすると、他のマテリアルとアトリビュートが同じでも圧縮されずに独立したマテリアルとして出力されます。

検索方法

Search 欄で検索条件を入力し、Search ボタンを押すと、マテリアルエディタに登録されているマテリアルのうち、該当するマテリアルの名前が Selected Material 欄に表示されます。

Target 欄で選択したマテリアルの名前が Set 欄に表示され、設定対象となります。
