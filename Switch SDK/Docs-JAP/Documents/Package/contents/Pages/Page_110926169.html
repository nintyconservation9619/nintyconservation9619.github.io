フォント使用時にビューアへのデータ転送を高速化したい

質問

フォント使用時にビューアへのデータ転送を高速化する方法を知りたいです。

回答

フォントファイル(bffnt) とともに、フォントソーファイル(xlor, bmp)を配置しておくと、そのレイアウトデータが使用しているフォントのみを自動生成してビューアに送る為、転送速度を高速化できます。

例えば、フォントと同一フォルダに、XXXX.bffnt XXXX.xlor XXXX.bmp と配置するか、./font/XXXX.bffnt ./xlor/XXXX.xlor ./bmp/XXXX.bmp のように配置します。

自動生成が成功すると、プレビュー転送時にメインウィンドウ下部のステータスバーに、「転送高速化のためにサイズ削減したフォントを自動生成しました。処理時間 ---- ... ms」と通知されます。

関連リンク フォント関連のファイルについては「フォントコンバータファイルフォーマット」を参照してください。
