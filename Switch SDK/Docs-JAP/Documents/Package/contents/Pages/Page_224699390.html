5 マルチエクスポート

モデルとアニメーションを個別設定で出力します。アニメーションを指定フレーム間ごとに個別に出力します。

Maya で連続したアニメーションを作っておき、指定フレームの区画ごとに出力できます。
Maya 1 シーンを管理するだけで、モデル、複数のアニメーションなどの出力設定を便利に行うことができます。

一度設定すると以降は一度に 4 種を出力できるようになります。

モデルを選択

エクスポートする対象のモデルを選択して設定します。

アウトライナなどからTopLを選択します。

Nintendo Multi Export ウインドウ

「Nintendo」→ 「Multi Export...」を選んで「Nintendo Multi Export」ウインドウを開きます。

「出力アイテム」を追加する

「Add」ボタンをクリックすると「Input Multi Export Item」ウインドウが開きます。
そのまま「OK」をクリックして登録します。

登録すると「Nintendo Multi Export」ウインドウにリスト表示されます。

「出力アイテム」を設定する（モデルとテクスチャー）

モデルとテクスチャの出力設定をします。

ウインドウに追加されたリストをダブルクリックして「Input Multi Export Item」を開きます。

下図と表の順番に設定して「Export_Model」を設定します。モデルとテクスチャーを出力対象にします。

順番
カテゴリ
操作
1 Output File Name 「Export_Model」と入力します。出力するファイル名です。
2 Node(Display Layer)

「Get」ボタンを押します。選択している「TopL」が登録されます。

（出力対象を直接記入することもできます）
3 Output File Selection

「Model Data」と「Texture Data」だけにチェックを入れます。モデルとテクスチャーが出力対象になります。
4 OK 「OK」ボタンを押して更新します。

設定して OK ボタンを押すと以下のようにリストが更新されます。

Output File Name が「Export_Model」になったことなどを確認します。

上図  A 「Output Folder」は未記入の場合は 3.エクスポート「Nintendo Export Options」で設定したフォルダーに出力されます。フォルダーアイコンをクリックして任意のフォルダーを指定することもできます。

上図  B スキニングを設定していないモデルの場合 Start Frame で設定した開始フレームの状態で出力されますが、この例ではスキニングを使用しているので出力されるモデルに影響しません。

 

「出力アイテム」を複製して設定する（待機アニメーション）

待機アニメーション（0～50フレーム）の出力設定をします。

リストを複製して設定リストを追加します。（設定内容ごと複製できるので手数が少なくできます）

リストから「Export_Model」を選択して「Duplicate」ボタンを押します。

複製できたら（どちらでもいいので）ダブルクリックして「Input Multi Export Item」を開きます。

下図と表の順番に設定して「Export_Wait」を設定します。待機アニメーション（0～50フレームループ）を出力対象にします。

順番
カテゴリ
操作
1 Output File Name 「Export_Wait」と入力します。出力するファイル名です。
2

Start / End Frame

Loop

「Range 」を選んで「0」「50」と入力します。０ フレームから 50 フレームのアニメーションが出力対象になります。

「Loop」のチェックをオンにします。オンにするには次の 3 にチェックを入れる必要があります。

 
3 OutPut File Selection

「Skeletal Animation Data」だけにチェックを入れます。スケルタルアニメーションが出力対象になります。
4 OK

「OK」ボタンを押してリストに登録します。

Output File Name に「Export_Wait」がリストに追加されたことを確認します。

2
