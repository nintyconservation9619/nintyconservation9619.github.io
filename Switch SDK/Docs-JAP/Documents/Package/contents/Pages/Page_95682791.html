乱数値を固定

ここでは「テクスチャ0」 ～ 「テクスチャ2」の乱数値を固定にする機能「乱数値を固定」について説明します。

概要

「乱数値を固定」は「 テクスチャ0」 ～ 「 テクスチャ2」サブタブで値をランダムにする機能の乱数値を固定にできる機能です。
テクスチャ間で乱数値を一致させることによって、例えばアルベドマップと法線マップの位置を一致させた状態でテクスチャ座標をアニメーションできます。
以下は「 設定項目」の概要です。

乱数値を固定

「テクスチャ0」～「テクスチャ2」で以下の機能の乱数値を固定にできます。

テクスチャパターンアニメ 「ループ」の「開始テーブルランダム」
「ランダム」
テクスチャ座標アニメ 「初期値ランダム」全般

設定例

以下は「乱数地を固定」を ON/OFF したときの比較画像と設定内容です。

テクスチャ0 文字「 A 」の画像です。
テクスチャ1 文字「 A 」の法線マップ画像です。
UVスクロールアニメ
「初期値ランダム」		 X = 0.5 放出毎のテクスチャ座標の初期値がランダムに変化しています。
X 方向（U方向）で ±0.5 ずつオフセットしています。
サンプルカスタムシェーダ
Custom Shader Sample		 使用 EffectViewer で法線マップを確認できます。
関連リンク 「カスタムシェーダを利用する」を参照してください。
ON

「 テクスチャ 0 の画像」と「 テクスチャ 1 の法線マップ用画像」の乱数値が揃い、違和感なく見えます。
OFF 「 テクスチャ 0 の画像」と「 テクスチャ 1 の法線マップ用画像」の乱数値が揃っていません。
参考：

「テクスチャ 0」と「テクスチャ 1」は同じ設定に揃えてください。
上の例の場合、「テクスチャ 0」側が以下の設定では「乱数値を固定」を ON にしても乱数値が揃いません。

  •  「縦半分の画像」を登録
  • 「繰り返し数」設定を 2×1
  • 「ラップ」設定を「 ミラー反転した画像」
