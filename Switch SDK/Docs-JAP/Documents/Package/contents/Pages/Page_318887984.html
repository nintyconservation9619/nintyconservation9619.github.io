CDR ライブラリ

はじめに

CDR ライブラリは、nxdmp の操作を行うためのライブラリです。

C:\Program Files\Nintendo\Oasis\ 以下にインストールされます。

構成

C:\Program Files\Nintendo\Oasis\lib 以下に libnn_cdr_x64.dll, libnn_cdr_x64.lib がインストールされます。
ヘッダは C:\Program Files\Nintendo\Oasis\include\cdr.h を使用してください。
lib ファイルはインポートライブラリとなっているため、lib を使用したツールでも、実行時には dll が参照できる必要があります。

将来的には .NET 用ライブラリの提供もされる予定です。

使用方法

C++ アプリからの使用

ここでは C++ コンソールアプリからどのように TM ライブラリを使用するかを説明します。

  1. 「プロジェクトプロパティ > 全般 > 追加のライブラリディレクトリ」に、「C:\Program Files\Nintendo\Oasis\lib\」を追加します。
  2. 「プロジェクトプロパティ > リンカー > 入力 > 追加の依存ファイル」に、「libnn_tm_x64.lib;」を追加します。
  3. cdr.h をインクルードし、実装します。
  4. 各関数の使用方法については、cdr.h のコメントを参照してください。
