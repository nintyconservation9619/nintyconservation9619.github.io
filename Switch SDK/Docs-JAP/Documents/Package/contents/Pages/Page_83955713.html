ショートカット一覧

 以下は EffectMaker におけるショートカットキーの一覧です。

メインメニュー(メインメニューに表示される項目のショートカットキーです。)
Ctrl + N [新規作成] → [エミッタセット] エミッタセットを新規に作成します。
Ctrl + Shift + N [新規作成] → [エミッタ] エミッタセットの直下にエミッタを新規に作成します。
Ctrl + O [ファイル] → [開く] ファイルを開きます。
Ctrl + W [ファイル] → [閉じる] ワークスペース上で選択しているファイルを閉じます。
Ctrl + Alt + W [ファイル] → [全て閉じる] ワークスペース上のすべてのファイルを閉じます。
Ctrl + S [ファイル] → [保存] ワークスペース上で選択しているノードを保存します。
Ctrl + Shift + S [ファイル] → [別名で保存] ワークスペース上で選択しているノードを別名で保存します。
Ctrl + Alt + S [ファイル] → [全て上書き保存] ワークスペース上のすべてのノードを上書き保存します。
F2 [ファイル] → [名前を変更] ワークスペース上で選択しているノードの名前を変更します。
Ctrl + Q [ファイル] → [終了] EffectMaker を終了します。
Ctrl + Z [編集] → [元に戻す] 編集前の状態に内容を戻します。
Ctrl + Y [編集] → [やり直し] 元に戻した編集をやり直します。
Ctrl + C [編集] → [コピー] ワークスペース上で選択しているノードの内容をコピーします。
Ctrl + V [編集] → [貼り付け] コピーした内容を選択しているノードに貼り付けます。
Ctrl + D [編集] → [複製] ワークスペース上で選択しているノードを複製します。
Delete [編集] → [削除] ワークスペース上で選択しているノードを削除します。
Ctrl + Shift + H 機能の有効 / 無効 選択しているノードの機能の有効 / 無効を切り替えます。
F9 [表示] → [プロパティウィンドウ] プロパティウィンドウを表示します。
F10 [表示] → [エフェクトブラウザ] エフェクトブラウザを表示します。
F11 [表示] → [カーブエディタウィンドウ] カーブエディタウィンドウを表示します。
F12 [表示] → [履歴ウィンドウ] メインウィンドウのドッキングエリア内に履歴ウィンドウを表示します。
Ctrl + Shift + M [表示] → [ツールバー] ツールバーの表示 / 非表示を切り替えます。
F5 [プレビュー] → [再転送] ビューアにデータを転送します。
Ctrl + K [ツール] → [オプション] オプションダイアログを開きます。
F1 [ヘルプ] → [ヘルプ] ヘルプを表示します。
プロパティウィンドウ(ワークスペースでの専用ショートカットキーです。)
Alt + ノードドラッグ操作 - 選択しているエミッタセットノードやエミッタノードを上下の同種ノードと入れ替えます。
Ctrl + E [エクスプローラで開く] 選択しているエミッタセットノードの保存先をエクスプローラで開きます。
プロパティウィンドウ(タブ操作専用のショートカットキーです。)
ESC - プロパティの編集中でもタブ操作に移行します。
Ctrl + Tab　または → - 選択状態を右側のタブに切り替えます。
Ctrl + Shift + Tab　または ← - 選択状態を左側のタブに切り替えます。
テクスチャの詳細ウィンドウ
Ctrl + Num + - テクスチャイメージを拡大表示します。
Ctrl + Num - - テクスチャイメージを縮小します。
Ctrl + 1 - RGB チャンネルを表示します。
Ctrl + 2 - R チャンネルを表示します。
Ctrl + 3 - G チャンネルを表示します。
Ctrl + 4 - B チャンネルを表示します。
Ctrl + 5 - A チャンネルを表示します。
Ctrl + M   ミップマップ部の表示 / 非表示を切り替えます。
Ctrl + D   ミップマップの表示方向（縦 / 横）を切り替えます。
