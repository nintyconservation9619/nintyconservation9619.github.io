プリミティブのアトリビュート情報

「エミッタ形状」や「パーティクル形状」において、「プリミティブ」を使用した場合、登録したプリミティブの情報を表示します。
プリミティブが意図したデータ構造になっていることや、余計な情報を含んでいないか等、ポリゴンモデルのアトリビュート情報を確認出来ます。
以下はパーティクル形状に「プリミティブ」を読み込んだ時の設定内容です。

以下は、上の画像で表示されている情報の内容について説明しています。

頂点数 606 頂点数が 606 頂点あることを表しています。
アトリビュート pos 頂点座標があることを表しています。
normal 法線情報があることを表しています。
tanegnt 接線情報があることを表しています。
uv0,uv1 UV0 と UV1 の2つのUV座標を持つことを表しています。
color 頂点カラーが着けられていることを表しています。
UVセット名 ( _u0,_u1 ) UV0 と UV1 のUVセット名を表しています。
