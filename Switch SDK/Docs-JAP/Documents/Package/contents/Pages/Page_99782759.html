SSL ライブラリ

はじめに

本ドキュメントの目的

このドキュメントでは、Nintendo SDK に付属する SSL ライブラリについて説明します。

対象読者

このドキュメントは、SSL/TLS または HTTPS を使用した安全なソケット通信を必要とする開発者を対象としています。

関連資料

このドキュメントは、nn::ssl 名前空間 API リファレンスと合わせて使用してください。

SSL ライブラリに用意されている機能

任天堂の SSL ライブラリは、Secure Socket Layer（SSL）プロトコルと Transport Layer Security（TLS）プロトコルを実装し、サーバ/クライアント認証を伴う、ソケットを使用した安全なネットワーク接続を実現します。SSL ライブラリは、SSL クライアント専用であることから、基礎となるネットワーク接続でクライアントソケットのみを受け入れます。

SSL ライブラリは、以下の機能を備えています。

サポート対象のプロトコル キー交換アルゴリズム 暗号化アルゴリズム
  • TLSv1.0
  • TLSv1.1
  • TLSv1.2
  • ECDHE-PSK
  • DHE-PSK
  • ECDHE
  • RSA



  • AES-CBC（128 bit および 256 bit）
  • AES-GCM（128 bit および 256 bit）
  • ChaCha20 Poly1305（256 bit）
  • 3DES（112 bit）
  • RC4（128 bit）
  • SHA-1
  • SHA-2（224 bit、256 bit、384 bit、および 512 bit）
  • MD5
  • HMAC
