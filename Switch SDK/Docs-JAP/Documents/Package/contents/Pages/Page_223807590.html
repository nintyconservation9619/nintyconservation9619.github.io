3 エクスポート

Maya からモデル、テクスチャ、アニメーションをエクスポートする手順の説明をします。

モデルを選択

エクスポートする対象のモデルを選択して設定します。

アウトライナなどからTopLを選択します。

エクスポート

Maya の 「メニューバー」→「 Nintendo」→「Export with Setting...」を開きます。

下図と表の順番に操作してエクスポートします。

 

順番 カテゴリ 操作
1 Preset

「TutorialDefault」を選びます。
2 Export Target 「Selection」 にします。
3 Output File Name

「ExportTripod」と入力します。
4 OutPut Folder

フォルダーアイコンをクリックして以下の「SampleTripod」フォルダーを指定します。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod 

5 General Options Magnify を「1.000」と入力します。Texture SRT Mode を「Maya」にします。
6 Animation Options

Start / End Frame を「Range」にして「0」「220」と入力します。0 から 220 フレームの範囲のアニメーションを出力します。

アニメーションをループ再生するように指定します。Loop にチェックを入れます。( この指定は次の 6 の Skeletal Animeation Data のチェックを入れる必要が有ります)

7 Output File　Selection 「Model Data」「Texture Data 」「Skeletal Animeation Data」の 3 つにチェックを入れます。
8 Export

「Export」をクリックします。１～ 6 の設定でファイルを出力します。

確認

下記のファイルが出力されていることを確認します。　モデル（fmdb）とアニメーション（fskb）ファイルは同じ名前で出力されます。

フォルダー
%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod 
ファイル 内容
ExportTripod.fmdb モデルファイル
ExportTripod.fskb スケルタルアニメーション
/textures/SampleTripodA01Albedo.ftxb テクスチャ

関連リンク より詳しい操作は エクスポートプラグイン使用方法 を参照してください。
