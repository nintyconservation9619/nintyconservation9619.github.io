マテリアルプロパティウィンドウ

選択したマテリアルのプロパティを確認、編集するウィンドウです。

A

マテリアルページを表示します。現在は何も表示されません。
B 全般ページを表示します。
C プレビューページを表示します。
D サンプラページを表示します。
E

シェーダページを表示します。
ツリービューでは、シェーダ定義ファイルで「グループ化されているパラメータ」のページ名やグループ名を子階層として表示します。
中間ファイル 3.3.0 からグループ化されているパラメータをさらにページ毎にわけることができるようになりました。
F オリジナル情報ページを表示します。
G 親マテリアルページを表示します。
H ユーザーデータページを表示します。
I

マテリアル設定ファイルをエクスポートします。
操作すると、マテリアル設定ファイルのエクスポートダイアログが開いてエクスポートに関する編集ができます。
J

マテリアル設定ファイルをインポートします。
操作すると、ファイル選択ウィンドウが開いてインポートするマテリアル設定ファイル ( .f3mt ) を選択できます。
マテリアル設定ファイルを選択すると、マテリアル設定ファイルのインポートダイアログが開いてインポートに関する編集ができます。

ページ名やグループ名を子階層として表示

マテリアルプロパティウィンドウのツリービューでは、シェーダ定義ファイル ( .fsd ) で設定したページ名や、グループ化されているパラメータのグループ名が表示されます。

参考：

シェーダ定義ファイル ( .fsd ) の詳細については、別途用意されている 3D 中間ファイル仕様書を参照してください。

  1. ページ名
    シェーダ定義ファイル(.fsd)に、<page_array> がある場合は、シェーダの直下に表示されます。
    「シェーダ」直下のノードの一番上に「既定ページ」を設け、その中には「頂点属性」、<group> の page_name が空欄のもの page_name が <page_array> にないものを表示します。
  2. グループ名
    シェーダ定義ファイル(.fsd) に　<page_array> がない場合は、シェーダの直下に表示されます。
    <page_array> が設定されていた場合は、設定されたページの子階層として表示されます。

ページやグループの右クリックメニュー

ページやグループ名の右クリックメニューで「コピー」「貼り付け」「既定値に戻す」ができます。

関連リンク メインウィンドウのオブジェクトビューをマテリアル一覧にした場合に表示する項目を設定できます。

マテリアル設定ファイルのエクスポートダイアログ

マテリアル設定ファイル ( .f3mt ) のエクスポートに関する操作を行います。

A 

インポートする際の初期適用項目を選択します。
チェックされた項目は、インポート時に既定の適用項目になります。

サンプラやシェーダのグループは、個々に適用項目として割り当てを行えます。 
B 

エクスポートする項目の設定内容を表示します。
C 

適用する項目全ての内容を展開して表示します。
D 

展開して表示されている設定内容をすべて折りたたみます。
E

エクスポートするマテリアル設定ファイルにコメントを追記します。
F

エクスポートします。
G  エクスポートをキャンセルします。

マテリアル設定ファイルのインポートダイアログ

マテリアル設定ファイル ( .f3mt )のインポートに関する操作を行います。

A 

インポート先のマテリアルを選択します。
マテリアルが複数選択されている場合は、チェックボックスでインポート先のマテリアルを任意に選択できます。
B 

インポート先のマテリアルが使用されているモデルです。
C 

インポートに適用する項目をチェックボックスで選択します。
初期設定は、エクスポートダイアログで設定した初期適用項目になります。
※サンプラやシェーダのグループは、個々に適用項目として割り当てが行えます。 
D 

インポートに適用する項目の設定内容を表示します。
E

適用する項目全ての内容を展開して表示します。
F

展開して表示されている設定内容をすべて折りたたみます。
G

マテリアル設定ファイル ( .f3mt ) に記載されているコメントです。
H

インポートを開始します。
I

インポートをキャンセルします。

 

同名アニメーションファイルがバインドされていた場合

  • インポート先のマテリアルを含むモデルに同名のアニメーションがバインドされている場合は、そのアニメーションが編集されます。
  • インポート先のマテリアルを含むモデルに同名のアニメーションがバインドされていない場合は、アニメーションが新規作成されます。
    ただし、パス未確定の同名アニメーションがあれば、そのアニメーションが編集されます。

サンプラやシェーダの適用項目について補足

サンプラや、シェーダのグループは、ダイアログで個々に適用項目として割り当てを行えます。

サンプラ

サンプラの適用項目の割り当て方法は、右クリックメニューで「置き換え」か「追加」に切り替えることができます。

シェーダ

シェーダは、シェーダ設定とグループ単位でインポート時の適用項目を選択できます。

 

参考：

シェーダ設定では、シェーダ定義とシェーディングモデルを割り当てます。
グループに属さないパラメータは、その他としてまとめられます。
