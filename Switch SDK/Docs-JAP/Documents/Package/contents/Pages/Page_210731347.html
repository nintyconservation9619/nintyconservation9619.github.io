チャイルドエミッタの不具合修正による注意点

本バージョンで修正された、チャイルドエミッタの不具合について、修正箇所と注意点について説明します。

チャイルドエミッタの基本設定タブ「エミッタSRTに完全追従」の機能に不具合がありました。
以下は、「エミッタSRTに完全追従」に設定したデータについて、修正前後の比較動画です。
█ が親エミッタ、█ がチャイルドエミッタです。

注意：

修正前に以下の正しくない挙動で見た目調整したエフェクトは、修正により見た目が変わる可能性があります。

修正前 修正後
「エミッタに追従しない」の挙動になっています。 親エミッタに追従しており、「エミッタSRTに完全追従」が効いています。

チャイルドのエミッタタブ「エミッタトランスフォーム」の機能に不具合がありました。
以下は、チャイルドのエミッタのエミッタトランスフォーム「位置」を X = 8 に設定したデータについて、修正前/後の比較動画です。

修正前 修正後
ワールド座標系で X 方向にオフセットがかかった見え方です。 エミッタ座標系で X 方向にオフセットがかかっており、正しい見え方です。

カスタムアクションの修正

カスタムアクションで、コールバック関数の選択操作を省略できるようにしました！
カスタムアクションノードを追加した時点で登録したコールバック関数がデフォルトとして選択状態になります。
これにより、毎回「コールバックID」からカスタムアクションプログラムを選択する必要がなくなりました。
