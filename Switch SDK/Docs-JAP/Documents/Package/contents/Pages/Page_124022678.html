上書きプロパティ設定ウインドウ

「上書きプロパティ設定ウインドウ」では、「上書き設定」をするペインを指定します。
部品レイアウトに配置したペインは、部品レイアウトに「上書き設定」を追加しておくことで、部品ペインとして配置した後でもパラメータを上書きできるようになります。

[ 部品・コントロール設定ウインドウ ] → [ 上書き ] → [ 設定 ] から「上書きプロパティ設定ウインドウ」が開きます。
上書きプロパティ設定ウインドウでは以下の操作ができます。

上書きプロパティ設定ウインドウ
A 上書きするペイン

一覧から、個々の部品ペインで設定を上書きするペインにチェックを入れて選択します。
ここで選択されたペインは、「部品・コントロール設定ウインドウ」にリスト化されます。
B 説明

選択しているペインの説明が表示されます。

ここで入力した説明は、「部品・コントロール設定ウインドウ」にリスト化された「コメント」に表示されます。
