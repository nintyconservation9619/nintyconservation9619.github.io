SLConverter

 

はじめに

目的

SLConverter は DirectX の HLSL シェーダ言語から NX で利用される GLSL シェーダ言語へ変換するツールです。

HLSL で開発されたシェーダを NX で利用することを目的にしています。

動作環境

NintendoSDK の動作環境に準拠します。

対象ユーザ

SLConverter は以下のような開発者を主な対象としています。

  • HLSL で記述された既存のシェーダファイルを NX に移植したい開発者

利用するメリット

SLConverter を利用することにより以下のようなメリットがあります。

  • DirectX のエフェクトファイル形式も対応しています。内部のテクニック毎にシェーダ変換を行うことが出来ます。
  • マルチスレッドを使いシェーダ変換を高速化することが出来ます。
  • VertexAttribute の位置をコンフィグファイルで任意に指定することが出来ます。
  • 行列の処理等、HLSL と GLSL の違いを吸収して変換することが出来ます。 
  • シェーダモデル 5.0 の一部機能を利用することが出来ます。詳細はシェーダモデル 5.0 への対応を参照してください。
