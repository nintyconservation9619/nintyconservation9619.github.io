ユーザーコマンド

ユーザーコマンドは、ユーザーが設定したコマンド（定型の処理）を、メニューやツールボタンから実行する機能です。

それぞれのプロジェクトで行う定型の処理（バッチファイル等で実装される処理やアプリケーションの実行）を、SoundMaker から実行することができます。

ユーザーコマンドの設定

ユーザーコマンドの設定は、『プロジェクト設定ダイアログ』で行います。

ユーザーコマンドは、

  • コマンドの名前
  • コマンド
  • アイコンファイルパス

で構成されており、最大 8 個まで登録することができます。

「コマンドの名前」は、メニューやツールボタンのツールチップに表示される文字列として、また「コマンド」は実際に実行される文字列になります。

「アイコンファイルパス」に画像を指定すると、メニューおよびツールボタンにアイコンを設定できます。

 

設定されたコマンドは、メニューの [ツール]→[ユーザーコマンド] およびツールバーに追加されます。
コマンドが 1 つも設定されていない場合、ツールボタンは何も表示されません。
