近距離アルファ

ここではカメラから近い距離にあるパーティクルに対して半透明処理を施す機能「近距離アルファ」について説明します。

概要

「近距離アルファ」は、カメラから近い距離にあるパーティクルに対して頂点シェーダ内でアルファ値を 0.0 にする機能です。
描画面積の大きなパーティクルがカメラに被さることにより発生する、めり込みによる表示の問題と描画負荷の問題を解決できます。

「近距離アルファ」と「カメラ前でスケール制限（近）」 「近距離アルファ」と「カメラ前でスケール制限」により描画負荷を軽減させる方法について紹介します。


以下は「 設定項目」の概要です。

近距離アルファ

「近距離アルファ」を使用するかを「 チェックボックス」ON/OFF で切り替えます。
ON に設定すると「 近距離アルファ」が表示されます。

以下は「 設定項目」の概要です。

αが 0 になる距離

アルファ値が 0 になる距離をカメラからの距離で設定します。

αが 1 になる距離

アルファ値が 1 になる距離をカメラからの距離で設定します。

設定例

以下は「近距離アルファ」の値を変更したときの比較画像です。

A

αが 0 になる距離 = 30

αが 1 になる距離 = 50

エフェクトはカメラから 50（5 グリッド）のところから 30（3 グリッド）までの範囲で透明になります。
カメラに近づくほど透明になる設定です。
B

αが 0 になる距離 = 10

αが 1 になる距離 = 30		 デフォルト設定です。
エフェクトはカメラから 30（3 グリッド）のところから 10（1 グリッド）までの範囲で透明になります。
エフェクトがカメラにかなり近づいた時に透明になる設定です。
C

αが 0 になる距離 = 50

αが 1 になる距離 = 30

エフェクトはカメラから 30（3 グリッド）のところから 50（5 グリッド）までの範囲で透明になります。

A   とは逆に、カメラから遠ざかるほど透明になる設定です。
