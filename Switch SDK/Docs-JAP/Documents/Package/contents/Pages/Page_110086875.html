グループ機能を利用する

ここでは、グループ機能の使用方法について説明します。

グループを作成する

グループを作成するには、レイアウトウィンドウで複数のペインを選択します。
この状態で、[ 右クリックメニュー ] → [ グループを作成 ] を選びます。
選択されたペインが、グループとして登録されます。
レイアウト中のグループを確認するには、グループウィンドウを使用します。
グループウィンドウは、[ メニュー ] → [ ウィンドウ ] → [ グループ ] とすることで表示・非表示を切り替えることができます。

グループを削除する

グループを削除するには、グループウィンドウを使用します。
削除したいグループに対応するツリー項目を選択し、[ 機能 ] → [ 削除 ] を実行します。

グループを編集する

グループを編集するには、ペイングループ編集ウィンドウを使用します。
グループ編集をしたいペインを選択して右クリックメニューから、[ グループを編集 ] とします。
また、グループのメンバーを変更する場合は、ペインを選択後 [ 右クリックメニュー ] → [ グループを編集 ] で表示される、 「ペイングループ編集ダイアログ」を利用します。
