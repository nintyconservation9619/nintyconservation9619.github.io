サウンドプロジェクトを使った共同作業の例

サウンドプロジェクトとサウンドセットの利用方法は自由です。
ひとりでプロジェクトの作成を行う場合には、ひとつのサウンドプロジェクトとひとつのサウンドセットで作業を行っても問題はありません。

ひとりで進めるプロジェクトの場合でも、サウンドの目的別にサウンドセットを分類したり、また複数のスタッフで進める大規模なプロジェクトでは、サウンドセットファイルを他の SoundMaker の利用者と共有することで、共同作業を行うもできます。
