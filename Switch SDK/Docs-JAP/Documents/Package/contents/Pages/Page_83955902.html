基本設定

このタブではエミッタ全体の設定を行います。

ファイル情報

「 基本設定」は、エミッタと放出されるパーティクルの基本挙動を設定する機能群です。
以下はグループ内設定項目の一覧です。

ファイル名　 ファイルの名前を表示する機能です。
ファイルパス ファイルの保存先パスを表示する機能です。
コメント コメントを編集できる機能です。
ラベルカラー ラベルカラーを設定できる機能です。

再生設定

「 再生設定」は、エフェクトの再生設定を行う機能群です。
以下はグループ内設定項目の一覧です。

呼び出し開始フレーム エフェクトを呼び出すタイミングを設定できる機能です。
開始前に強制計算 エフェクトの計算処理をあらかじめ回しておき、ゲーム開始時にパーティクルが表示された状態にできる機能です。
マニュアルエミッタ放出 ゲームプログラマーがエミッタを好きな位置に放出できる機能です。
