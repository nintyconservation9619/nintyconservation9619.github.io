NW4F から NintendoSDK への 移行ガイド

概要

このページでは、NintendoWare For Cafe ( NW4F ) から NintendoSDK-NintendoWare に移行するユーザーのための移行手順を示します。
ただし、Cafe では動作させることは出来ないため、Windows プラットフォームに移行していただく必要があります。

目次
  • 概要
  • 移行手順
  • 変更点
  • モジュールごとの変更点

移行手順

下記の手順に従って、NW4F の環境を NintendoSDK-NintendoWare に移行することができます。

  1. 次のパッケージをインストールする
    • NintendoSDK
    • NintendoSDK-NintendoWare
    • NintendoSDK-NintendoWare-Samples
    • NintendoSDK-Documents
     

  2. NX アプリケーションのプロジェクトを作成する にしたがってユーザーアプリケーション用の VisualStudio プロジェクトを作成する


  3. コンバータのパス・オプションを変更する

    手順自体は上記のとおりですが、詳細な変更内容については、次項「変更点」をご確認ください。

コンバータ・ツール

次のように変更されました。

  • ツールの exe 名から、接頭辞「NW4F_」を削除しました
    • 例：NW4F_EffectMaker.exe → EffectMaker.exe
  • Softimage プラグインは提供されません
  • NW4F_AnimSoundMaker.exe は提供されません

 
名前空間

nw::g3d

nn::g3d

nw 名前空間から nn 名前空間 へ移行しました。

また、一部モジュールはライブラリ名の変更が行われています。
nw::font nn::font
nw::lyt nn::ui2d
nw::eft nn::vfx
nw::snd nn::atk
