はじめに

ここでは NFC ライブラリのステート遷移について説明します。

NFC ライブラリには以下の二種類のステートがあります。NFC ライブラリを使用するにはこれらの状態を適切に把握する必要があります。

  • NFC ライブラリのステート
  • NFC デバイスのステート

ステートは nn::nfc::DeviceState で定義されており、nn::nfc::GetDeviceState() で現在のステートを取得することができます。

ステートが示す状態

NFC デバイスの各ステートは次のように定義されています。

表 10. NFC デバイスのステート
ステート 状態の説明
DeviceState_Init タグを探していない状態です。NFC ライブラリ初期化直後はこの状態です。
DeviceState_Search タグを探している状態です。
DeviceState_Active タグが検知された状態です。
DeviceState_Deactive タグを検知した後、タグが離れた状態です。
DeviceState_Keep タグのセッションが維持された状態です。
DeviceState_Unexpected

想定していない状態です。接続されていないデバイスが指定された場合に発生することがあります。
