位相の変化速度

ここでは位相が変化する速度を設定出来る機能「位相の変化速度」について説明します。

概要

「位相の変化速度」は、波形成分の位相を変化させることができる機能です。
位相の変化速度を持たせると、同じ軌道を通っているパーティクルの軌道を徐々に変えることができます。
以下は「 設定項目」の概要です。

位相の変化速度

「位相の変化速度」は、位相の変化する速度を設定します。
設定した値が 0 では位相が変化しません。

設定例

以下は「各成分で位相を統一」の値を変更したときの比較画像です。

A 位相の変化速度 = 0 デフォルト設定です。
パーティクルの位相は変化せず、一定の波形上を移動します。
B 位相の変化速度 = 0.5 A と比較すると、パーティクルに影響する波形の位相が変化し、徐々に波形の軌道が変わります。
