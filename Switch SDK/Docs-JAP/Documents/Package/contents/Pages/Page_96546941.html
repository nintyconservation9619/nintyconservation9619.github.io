位相のばらつき（%）

ここでは GPU ノイズの波形をパーティクルごとにばらつかせる機能「位相のばらつき」について説明します。

概要

「位相のばらつき（%）」は、「各成分で位相を統一」で揃えた波形の位相をばらつかせる割合を設定する機能です。
位相をばらつかせると、全体としての統一感はありつつも、パーティクルごとに多少の個性を持たせた軌道になります。
以下は「 設定項目」の概要です。

位相のばらつき（%）

位相をばらつかせる範囲を「 ダイヤル」や「 テキストボックス」により 1 周期を 100 %としたときの割合で設定します。
100%を指定すると、位相を統一しないのと同じ状態になります。

1周期 を乱数の上限、下限側はランダムに変化します。以下は乱数の下限をもとめる計算式です。
