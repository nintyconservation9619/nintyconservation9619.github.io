TM ライブラリ

はじめに

TM ライブラリは、Nintendo Target Manager の各種機能を、PC 上で動くツールなどから直接使用するためのライブラリです。

NintendoSDK そのものではなく Target Manager に紐づくため、C:\Program Files\Nintendo\Oasis\ 以下にインストールされます。

構成

{ VS2015 | VS2017 } x { x86 | x64 } の組み合わせで、4通りのライブラリが C:\Program Files\Nintendo\Oasis\lib 以下に含まれます。
どれについてもヘッダは共通で、C:\Program Files\Nintendo\Oasis\include\tm.h を使用してください。

また、.NET 用ライブラリとして、Nintendo.Tm.dll も提供されています。


使用方法

C++ アプリからの使用

ここでは C++ コンソールアプリからどのように TM ライブラリを使用するかを説明します。

  1. 「プロジェクトプロパティ > 全般 > 追加のライブラリディレクトリ」に、「C:\Program Files\Nintendo\Oasis\lib\」を追加します。
  2. 「プロジェクトプロパティ > リンカー > 入力 > 追加の依存ファイル」に、「libnn_tm_v120_x86.lib;」を追加します。
    1. バージョンやアーキテクチャの部分は、ご使用の VS のバージョンやビルドするプラットフォーム構成に従って読み替えてください。
  3. tm.h をインクルードし、実装します。
  4. 各関数の使用方法については、tm.h のコメントを参照してください。
参考：

Debug ビルドの場合、「warning LNK4098: defaultlib 'MSVCRT' は他のライブラリの使用と競合しています。/NODEFAULTLIB:library を使用してください。」という警告が出ることがあります。
その場合は、「プロジェクトプロパティ > リンカー > 入力 > 特定の既定のライブラリの無視」に、「MSVCRT」 を指定してください。

.NET アプリからの使用

  1. 「ソリューションエクスプローラ > 参照 > 参照(B)...」から、「C:\Program Files\Nintendo\Oasis\lib\Nintendo.Tm.dll」を追加します。
  2. 実装します。各関数の使用方法については、サンプルを参照してください。
