メインウィンドウ

3DEditor のメインウィンドウです。

A タイトルバー

ソフトウェア名や、編集中のファイル名が表示されます。
B メインメニュー 各種機能を呼び出します。
各種機能のショートカットキーは、こちら を参照してください。 
C ツールバー

メニュー項目で、使用頻度の高い機能を「アイコン」で表示します。
D ファイルビュー

3DEditor に読み込まれているファイルをツリー状に表示して、選択したファイル情報を表示します。
E アニメーションコントロール ビュー上で再生されるアニメーションをコントロールします。

F

ログバー

オリジナルアニメーションの適用時や、マテリアルシェーダアサインなどを行った際のメッセージをログとして残します。
ログバーの上部にある三角マークや、「メインメニュー」 → [表示] でログバーの表示 / 非表示を切り替えることができます。 

G

ステータスバー

メニュー項目の説明や、ファイル入出力の状態を表示します。 

H

オブジェクトビュー

3DEditor に読み込まれているファイルに含まれる各種オブジェクトを様々な方法で表示します。 
表示は任意に切り替えることができます。
