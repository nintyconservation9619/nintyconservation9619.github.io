Tick クラス

前提

Tick クラスは、システムチックを扱うためのクラスです。システムチックは、システムで決められた周期に従って単調増加する 64bit の符号付き整数値であり、 nn::os::Tick クラスで表現されます。システムチックの進む速さはターゲット環境に依存していますが、その周波数（１秒間に進むカウント数）を取得する機能が提供されています。詳細は 時間管理 を参照して下さい。

本体起動時のシステムチックの初期値は不定です。初期値が 0 であることを想定した実装を行ってはいけません。

機能概要

nn::os::Tick クラスは以下のメンバ関数を持ちます。API リファレンスはリンク先を参照して下さい。

名前 説明 備考
Tick(int64_t tick) コンストラクタ

チック値 tick をとってオブジェクトを初期化します。
Tick(nn::TimeSpan timespan) コンストラクタ TimeSpan 値をとってオブジェクトを初期化します。
GetInt64Value() チック値の取得 現在保持しているチック値を std::int64_t 型で返します。
ToTimeSpan() nn::TimeSpan 型への変換 現在保持しているチック値を nn::TimeSpan 型に変換して返します。
その他
その他のメンバ関数については nn::os::Tick クラスの API リファレンスを参照して下さい。



Tick クラス使用時の注意事項

通常、システムチック値は以下のようなコードで取得して使用します。

#include <nn/os.h>

auto tick = nn::os::GetSystemTick();

取得した tick から、tick.GetInt64Value() とすることで、システムチックの即値を int64_t 型で取得することができますが、この値を float や int32_t などのより小さなサイズの型へ変換して等値比較や大小比較を行わないで下さい。int64_t よりも小さな型や低精度の型に変換すると、元となるシステムチック値の情報が欠落し、正しい比較演算等が行われなくなります。
