テクスチャ・フォント・部品の集約

「テクスチャ・フォント・部品の集約」について説明します。

「テクスチャ・フォント・部品の集約」機能は、レイアウトデータの受け渡しのさいに、関連ファイルのセットを自動で生成します。
受け渡しをしたいレイアウトデータを選択し、「テクスチャ・フォント・部品の集約」を行うと、受け渡しをしたいデータを抜き出して（集約して）フォルダにまとめることができます。
レイアウトやテクスチャを、一つ一つ選択して受け渡し用のデータを作り直す（パスを張りなおす）必要はありません。

テクスチャ・フォント・部品 の集約 ウインドウ
A 集約パス

集約したデータを書き出すパスを入力します。

集約されて書き出されたデータは、指定されたパスを参照します。
B
レイアウト名でサブフォルダを作成する

チェックを入れると、レイアウト名と同じ名前のフォルダを作成して、集約したデータの書き出し先に指定します。

このフォルダを受け渡しすることで、簡単にレイアウトデータの引渡しを行うことができます。
C
テクスチャをレイアウト毎にサブフォルダで分類 ( 部品のテクスチャは「Common」へ )

チェックを入れると、集約するテクスチャをレイアウト毎にサブフォルダに分類します。

複数のレイアウトのテクスチャを分類しておきたい時に使用します。

部品のテクスチャは、Common 内に分類します。
D 集約先パス

集約したデータを書き出すフォルダを指定します。

「 B レイアウト名でサブフォルダを指定する」のチェックを入れると、書き出し先にレイアウト名の付いたフォルダが作られ、集約先として指定されます。
E 実行 指定したパスに集約を実行します。
