3DEditor

3DEditor について新しく追加された機能について紹介します。

マテリアル参照機能を試験的に公開しました

注意：

この機能は今後、予告なく仕様や動作が変更される場合がありますので、利用する際はご注意下さい。

正式公開機能となる際は改めてニュースにてお知らせします。

あるマテリアルのシェーダーパラメーター設定を別のマテリアルが参照することができるマテリアル参照機能を試験的に公開しました。この機能を利用することで、複数のマテリアル間で設定を共有できるようになるため、マテリアルの管理が行いやすくなります。

関連リンク ワークフローへの組み込み方など、詳細は マテリアル参照 をご覧ください。
