アルファコンバイナ

ここではエミッタ内のアルファ要素を合成できる機能「アルファコンバイナ」について説明します。

概要

「アルファコンバイナ」は、エミッタ内のアルファ要素を合成する機能です。
以下はアルファコンバイナで扱えるアルファ要素と計算式上での名称です。

テクスチャのアルファ要素 テクスチャ 0 ､1 ､ 2 タブに登録するテクスチャのアルファ情報です。
「テクスチャ 0」～「テクスチャ 2」で利用できます。
パーティクルのアルファ要素 アルファタブに設定するアルファ情報です。
「計算式」で利用できます。
プリミティブの頂点アルファ要素 パーティクルタブで登録したプリミティブファイルが持つ頂点アルファ情報です。
「プリミティブ」で利用できます。
