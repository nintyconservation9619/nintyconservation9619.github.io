リニア編集モード

EffectMakerの「リニア編集モード」について説明します。

概要

「リニア編集モード」とは、シェーダで正しいライティング計算を行うさいのガンマ補正に EffectMaker のカラー編集環境やテクスチャの表示状態を合わせるオプション設定です。

リニア編集モードに関連する機能

「リニア編集モード」に切り替えることで、以下の機能がリニア編集用に切り替わります。

RGBを二乗して取得 リニア編集モード時に UI が表示されます。
登録したテクスチャの輝度を二乗することにより、リニア変換時の結果に近似できる機能です。
カラー編集の UI カラー編集の UI がガンマ補正された状態に切り替わります。
テクスチャプレビュー リニアワークフローに合わせてテクスチャプレビューをフォーマットに応じて補正します。

カラーに関する UI の変更

「リニア編集モード」に切り替えると、カラーに関する UI のカラーがガンマ補正された状態に切り替わります。
以下は、ガンマ補正が行われる UI です。

  • カラースライダ
  • カラーピッカー
  • カラーアニメーション UI

テクスチャプレビュー

「リニア編集モード」の場合、リニアテクスチャは適切にガンマ補正され、ビューアでの見え方に合わせた表示に切り替わります。 また、リニア編集モード時に、リニアではないテクスチャを読み込んだ場合は警告が出力されます。 警告の表示/非表示はオプションで抑制できます。

リニアテクスチャ

リニアテクスチャについて説明します。

リニアテクスチャの種類

エフェクトで主に利用されるテクスチャフォーマットは大きく以下 3 種類のフォーマットに分けられます。
EffectMaker でのテクスチャの扱いもこのフォーマットを基準に挙動が変化します。

フォーマット モード 解説
unorm 通常 通常のテクスチャ。
linear unorm リニア PhotoShop エクスポート時にリニア変換される。
sRGB リニア sRGB テクスチャ。シェーダ内でフェッチ時にリニア変換される。
