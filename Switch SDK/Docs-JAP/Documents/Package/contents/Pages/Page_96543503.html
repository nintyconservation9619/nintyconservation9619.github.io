各成分で位相を統一

ここではパーティクルが利用する波形の位相を毎回揃えるかを設定する機能「各成分で位相を統一」について説明します。

概要

「各成分で位相を統一」は、パーティクル放出毎に利用する波形の、位相のずらし幅を統一し、位相を揃えるかを設定できる機能です。
位相を統一することにより大量のパーティクルの動きを統一できます。

以下図では、波形の1周期を 100% としたときの位相を 25% 分ずらした例です。

以下の図は、A , B , C の3つのパーティクルで波形成分の位相を統一する、しないを設定したときのイメージです。

波形成分の位相を統一しない場合（デフォルト）

各パーティクルで利用する波形の位相が変わります。

パーティクル A パーティクル B パーティクル C

波形成分の位相を統一する場合

すべてのパーティクルで、利用する波形の位相が統一されます。

パーティクル A パーティクル B パーティクル C

以下は「 設定項目」の概要です。

各成分で位相を統一

各波形成分で位相を統一するかを「 チェックボックス」ON/OFF で切り替えます。
設定を ON にすると「 各成分で位相を統一」が表示されます。以下はグループ内の設定項目です。

位相の変化速度 揃えた波形の位相を動かす速度を設定する機能です。
位相のばらつき（%） 波形を揃えた上で、位相のばらつき度合いを設定する機能です。
