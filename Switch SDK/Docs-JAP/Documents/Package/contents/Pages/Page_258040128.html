注意事項

このバージョンでの注意事項についてお知らせします。


モデルファイルの量子型書き換えスクリプトを同梱

パーティクル形状「プリミティブ」として uv0 と uv1 の量子化型（quantize_type アトリビュート）が一致していないモデルファイル（ .fmdb）は読み込めません。
「中間ファイルフィルター」を利用して量子化型を書き換えることで一致させることができます。
本バージョンでは、パッケージにこの「中間ファイルフィルター」を利用した、モデルファイルの書き換えスクリプト（ .py と .bat ）が同梱されるようになりました！

ドッキング対応した EffectViewer

「PC ビューアを起動」ボタンで EffectViewer が EffectMaker のドッキングエリア内に配置できるようになりました！
以下は新しい「EffectViewer」です。

ウィンドウのツールバーには、ビューアのデバッグ機能で使用頻度が多い機能が表示されています。
以下はツールバーに表示される機能の説明です。


床の表示

カメラを操作したときにエフェクトの姿勢を把握しやすくできる「床」を表示します。
以下の選択項目があります。

  • 「 床を非表示」：床を非表示にします。
  • 「 床をグリッドで表示」：グリッドを表示します。
  • 「 床をブロックで表示」：不透明描画のブロックを表示します。

サイズの指定

1 グリッドまたは 1 ブロックのサイズを設定します。

エフェクト作成時にパーティクルサイズの基準にできます。

分割数の指定

床をグリッドで表示しているとき、1 グリッドの分割数を設定します。


グレアの強さ

グレアの強さを指定します。


エミッタやフィールドの表示

関連リンク エミッタに関する以下の情報を表示します。

  • 「エミッタ形状を表示」
  • 「チャイルドエミッタ形状を表示」
  • 「フィールド情報を表示」



パーティクルの重なりチェック

パーティクル同士の重なり度合を確認できます。


デバッグ情報の表示

「GPU計測結果の表示」を含むエフェクトの情報を非表示にする機能です。

「サムネールを撮影」機能の追加

「プレビュー」のメニュー項目及びメインウィンドウの「ツールバー」に「 サムネールを撮影」機能が追加されました！
ビューアに表示されているエミッタセットファイル、モデルファイルのサムネールを撮影します。

「 サムネールを撮影」を実行すると、エミッタセットファイルの保存先フォルダに「.NwThumbs」フォルダを自動生成し、撮影した画像を .png ファイルとして保存します。

保存した .png ファイルは、例えば「エフェクトブラウザ」でファイルのサムネールとして表示されます。
以下はエフェクトブラウザで「お気に入り」にある「VfxTemplate」のエミッタセットを表示した画像です。エフェクトのサムネールが表示されています。
