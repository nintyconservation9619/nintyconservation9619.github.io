オートバックアップの場所を開く機能の追加

メインメニューからオートバックアップにより保存されたエミッタセットファイルの場所を直接開けるようになりました！

「オートバックアップ」により「 エミッタセットファイル（ .eset）」は、EffectMaker.exe と同じ場所にある AutoBackup フォルダに保存されます。
AutoBackup フォルダは、以下の方法で簡単に開くことができます。

  • メニューバー[ヘルプ] → [オートバックアップの場所を開く]を選択します。
  • 「 オプション設定」のダイアログを開き、設定項目「詳細」の「 オートバックアップ」にある「フォルダを開く」ボタンを押します。

カスタムシェーダのソース表示機能を拡張

カスタムシェーダのソース表示機能においてプリプロセスの有無を選択できる機能を追加しました。
この機能は主にプログラマがアーティストの作成したエフェクトデータを確認するときなどに使用します。
