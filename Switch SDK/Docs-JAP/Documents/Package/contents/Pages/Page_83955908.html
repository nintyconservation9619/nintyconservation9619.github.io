「エミッタ」タブ

「 エミッタ設定にスケールをかける」は、エミッタセット内のすべてのエミッタ形状スケールに対して設定を行う機能です。
以下はスケールがかかる対象をまとめた表です。

以下は「 設定項目」の概要です。

エミッタ形状スケール（±）

エミッタセット内のすべてのエミッタ形状スケールに対して乗算する値を X Y Z の各軸ごとに設定します。

関連リンク 以下は関連ページへのリンクです。  

エミッタ形状 パーティクルの放出形状と範囲を決める機能です。
