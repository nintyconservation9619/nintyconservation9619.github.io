詳細

  EffectMaker が利用するメモリやデータの管理方法について設定できます。

メモリ管理

「 メモリ管理」は、EffectMaker で使用するキャッシュ用のメモリサイズを設定する機能です。
キャッシュサイズをオーバーした場合は、古いキャッシュから順に削除されます。

オートバックアップ

「 オートバックアップ」は、指定した間隔でEffectMaker のデータを自動保存する機能です。

オートバックアップを実行 「バックアップ間隔」で指定した間隔で、自動的に開いているエミッタセットファイル（ .eset）を保存します。
起動時に一定期間過ぎたファイルを削除 起動時に、指定した日数経過したファイルを削除するかをON/OFF で切り替えます。

「オートバックアップ」により「 エミッタセットファイル（ .eset）」は、NW_EffectMaker.exe と同じ場所にある AutoBackup フォルダに保存されます。
AutoBackup フォルダは、以下の方法で簡単に開くことができます。

  • メニューバー[ヘルプ] → [オートバックアップの場所を開く]を選択します。
  • 「 オプション設定」のダイアログを開き、設定項目「詳細」の「 オートバックアップ」にある「フォルダを開く」ボタンを押します。
