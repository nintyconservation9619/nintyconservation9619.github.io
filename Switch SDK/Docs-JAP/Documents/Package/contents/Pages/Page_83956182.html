メインウィンドウ

メインウィンドウ のメインメニュー項目に対応するショートカットキーの一覧です。

操作 対応するメニュー 説明
[ Ctrl ] + [ O ] [ファイル] → [開く] ファイルを開きます。
[ Ctrl ] + [ W ] [ファイル] → [閉じる] 選択しているファイルを閉じます。
編集されているファイルの場合は、ファイル保存ダイアログが開きます。
[ Ctrl ] + [ Shift ] + [ W ]
[ Ctrl ] + [ Alt ] + [ W ]		 [ファイル] → [全て閉じる] 読み込んでいるファイルを全て閉じます。
編集されているファイルがある場合は、ファイル保存ダイアログが開きます。
[ Ctrl ] + [ S ] [ファイル] → [上書き保存] 上書き保存します。
[ Ctrl ] + [ Shift ] + [ S ] [ファイル] → [別名で保存] 読み込んでいるファイルを別名にして保存します。
[ Ctrl ] + [ Alt ] + [ S ] [ファイル] → [全て上書き保存] 読み込んでいるファイルを全て上書き保存します。
[ F2 ] [ファイル] → [名前の変更] 選択しているファイル名を変更します。
[ Ctrl ] + [ M ] [ファイル] → [マージ] → [モデル] 選択しているモデルをマージします。
実行するとファイル選択ウィンドウが開き、マージするモデルファイルを選択できます。
[ Ctrl ] + [ Q ] [ファイル] → [終了] 3DEditor を終了します。
[ Ctrl ] + [ Z ]
[ Z ]		 [編集] → [元に戻す] 操作を元に戻します。
[ Ctrl ] + [ Y ] [編集] → [やり直し] 操作をやり直します。
[ Ctrl ] + [ P ] [表示] → [プロパティウィンドウ] 選択しているファイルのプロパティウィンドウを開きます。
[ Ctrl ] + [ 1 ] [表示] → [オブジェクトビュー] → [1画面] オブジェクトビューを 1 画面表示にします。
[ Ctrl ] + [ 2 ] [表示] → [オブジェクトビュー] → [2画面：上下] オブジェクトビューを 2 画面上下表示にします。
[ Ctrl ] + [ 3 ] [表示] → [オブジェクトビュー] → [2画面：左右] オブジェクトビューを 2 画面左右表示にします。
[ Ctrl ] + [ 4 ] [表示] → [オブジェクトビュー] → [3画面：下を分割] オブジェクトビューを 3 画面で下を分割表示にします。
[ Ctrl ] + [ 5 ] [表示] → [オブジェクトビュー] → [3画面：上を分割] オブジェクトビューを 3 画面で上を分割表示にします。
[ Ctrl ] + [ 6 ] [表示] → [オブジェクトビュー] → [3画面：右を分割] オブジェクトビューを 3 画面で右を分割表示にします。
[ Ctrl ] + [ 7 ] [表示] → [オブジェクトビュー] → [3画面：左を分割] オブジェクトビューを 3 画面で左を分割表示にします。
[ Ctrl ] + [ 8 ] [表示] → [オブジェクトビュー] → [4画面] オブジェクトビューを 4 画面表示にします。
[ F5 ] [プレビュー] → [アプリケーションへ接続（切断）] アプリケーションへ接続します。
アプリケーションに接続済みの場合は、切断されます。
[ F1 ] [ヘルプ] → [ヘルプ] 3DEditor のヘルプを開きます。

オブジェクトビュー

メインウィンドウ上の オブジェクトビュー がアクティブの場合に対応するキーの一覧です。

操作 説明
[ Ctrl ] + [ A ] オブジェクトビュー内のオブジェクトを全て選択します。
[ + ] ビューを拡大します。 （スケマティック表示時のみ有効）
[ - ] ビューを縮小します。 （スケマティック表示時のみ有効）
[ A ] 全体を中央表示します。 （スケマティック表示時のみ有効）
[ F ] 選択オブジェクトを中央に表示します。 （スケマティック表示時のみ有効）
[ F3 ] 選択オブジェクトのコメントを編集します。 （スケマティック表示時のみ有効）

アニメーションコントロール

メインウィンドウ上の、アニメーションコントロール に対応するキーの一覧です。

操作 説明
[ v ] アニメーションを再生 / 一時停止します。
[ Shift ] + [ v ] アニメーションを停止します。（カレントフレームがアニメーション開始フレームになります）
[ Alt ] + [ , ] アニメーションをコマ戻しします。
[ Alt ] + [ . ] アニメーションをコマ送りします。

ログバー

メインウィンドウ上のログバーに表示されたログに対するショートカットキーの一覧です。
ログバーが選択されているときに使用できます。

操作 説明
[ Ctrl ] + [ A ] ログを全て選択します。
[ Ctrl ] + [ C ] 選択したログをコピーします。
[ Del ] ログを全てクリアします。

カーブエディタ

カーブエディタ の操作に対応するキーの一覧です。

操作 説明
[ S ] カーブビューに表示されているカレントタイム（赤線）があるフレームにキーを追加します。
[ Delete ]
[ BackSpace ]		 選択キーを削除します。
[ Ctrl ] + [ C ] 選択キーをコピーします。
[ Ctrl ] + [ V ] コピーまたは、カットしたキーをペーストします。
[ Ctrl ] + [ X ] 選択キーをカットします。
[ A ] カーブ全体をビューに収めます。
[ F ] 選択キーをビューに収めます。
[ Ctrl ] + [ + ] ビューを拡大します。
[ Ctrl ] + [ - ] ビューを縮小します。
[ < ] [ > ] カーブビューに表示されている全てのキーで、カレントフレームが前後にあるキーの位置にジャンプします。
[ ← ] [ → ] カーブビューでキーの選択状態を左右にある次のキーにシフトします。
[ E ] キーモードを「最も近いキーの移動」にします。
[ I ] キーモードを（ I キーを押している間 ）「キーの挿入」にします。
[ R ] キーモードを「スケール操作」にします。
[ W ] キーモードを「複数移動」にします。
[ C ] 他のキーのフレーム、値にスナップする機能の ON / OFF を切り替えます。

テクスチャ詳細ウィンドウ

テクスチャ詳細ウィンドウ で使用するショートカットキーの一覧です。

操作 説明
[ Ctrl ] + [ Num + ] ビューを拡大します。
[ Ctrl ] + [ Num - ] ビューを縮小します。
[ Ctrl ] + [ 1 ] RGB チャンネルを表示します。
[ Ctrl ] + [ 2 ] R チャンネルを表示します。
[ Ctrl ] + [ 3 ] G チャンネルを表示します。
[ Ctrl ] + [ 4 ] B チャンネルを表示します。
[ Ctrl ] + [ 5 ] A チャンネルを表示します。
[ Ctrl ] + [ M ] ミップマップ部の表示 / 非表示を切り替えます。
[ Ctrl ] + [ D ] ミップマップの表示方向（縦 / 横）を切り替えます。

スライダーコントロール

スライダーコントロール に対応したキーの一覧です。

操作 説明
[ R ] 設定したスライダーコントロールの値を元に戻します。
[ + ] スライダーの範囲を拡大します。
[ - ] スライダーの範囲を縮小します。
