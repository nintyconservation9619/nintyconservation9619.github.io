ウインドウペインプロパティ編集ページ

「ウインドウペインプロパティ編集ページ」についての情報を入力します。

「ウインドウペインプロパティ編集ページ」は、「ウインドウペイン」のプロパティウインドウを設定するのに必要な「共通項目」です。
ウインドウペイン関係の設定画面では、各設定画面共通で「ウインドウペインプロパティ編集ページ」が表示されます。

ウインドウペインプロパティ編集ページ

プロパティの編集対象となる部位を指定します。

図の中から編集をする箇所を、マウス左クリックで選択します。

選択された部位は図の中でピンク色に表示され、選択箇所の名称が切り替わります。

また、選択可能な箇所はウインドウペインの種類や、貼るテクスチャ数設定によって変化します。

フレーム部分のテクスチャの向きは、左上を基本としています。

  • コンテンツ
  • 左上
  • 左下
  • 右上
  • 右下
  • 上フレーム
  • 下フレーム
  • 左フレーム
  • 右フレーム

種類 :

ウインドウペインの種類を決めます。

ウインドウペインの種類は３種あり、それぞれ形状が違います。

テクスチャを設定できる部位もウインドウペインの種類ごとに違います。

  • 全方位（従来挙動）
  • 水平
  • 水平ーコンテンツ無し

テクスチャ数 :

ウインドウペインの枠部分に貼る、テクスチャの数を指定します。

左上の 1 部分のみにテクスチャを設定して四方に引き伸ばす設定や、

全てのフレームの分割に違うテクスチャを貼るという貼り方もあります。

  • 1 枚 ( 左上だけ指定 )
  • 4 枚 ( カドだけ指定 )
  • 8 枚 ( すべて指定 )
