ノイズの移動速度（±）

ここでは、ノイズテーブルの移動速度を設定する機能「ノイズの移動速度（±）」について説明します。

概要

「ノイズの移動速度（±）」は、ノイズテーブルの移動速度を XYZ 各軸ごとに設定できる機能です。
ノイズを移動することによって、パーティクルの移動方向を動的に変化させることができます。

以下は、「ノイズの移動速度」で X 方向の値を変えたときの説明用イメージです。

X = 0 X = 1
ノイズテーブルは動きません。 ノイズテーブルが移動します。

以下は「 設定項目」の概要です。

ノイズの移動速度（±）

ノイズテーブルの移動速度の値を「 ダイヤル」や「 テキストボックス」により座標系 X Y Z で設定します。
値が大きいほどノイズテーブルが速く動き、パーティクルの移動方向が大きく変化します。

設定例

下は「ノイズの移動速度（±）」 XYZ の値を変えた場合の比較画像です。

X,Y,Z = 0 X,Y,Z = 0.1 X,Y,Z = 1
パーティクルは一定の軌道で移動します。 パーティクルの移動方向が変わります。 パーティクルの移動方向が大きく変わります。
