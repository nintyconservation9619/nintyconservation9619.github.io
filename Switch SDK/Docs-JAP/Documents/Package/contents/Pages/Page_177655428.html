最小移動距離 / 1フレーム

ここではエミッタが移動しない、または微小な移動のとき、移動したとみなすことでパーティクルを放出できる機能「最小移動距離 / 1フレーム」について説明します。

エミッタが移動しない、または微小な移動の場合でも、最少移動距離を下回る移動の場合は 最小移動距離を移動したとみなします。

例えば、エミッタの移動距離が ０ 、「放出間隔」が １ 、「最小移動距離 / 1フレーム」が 0.1 の場合、移動した距離を 0.1 とみなし、 10 フレームに 1 回放出します。

エミッタが停止した状態でもパーティクルを表示しておきたい時の使い方です。

概要

「最小移動距離/1フレーム」は、1 フレームのエミッタ移動量を設定した値移動したとみなす機能です。
エミッタが移動しない、または微小な移動のように、最少移動距離を下回る移動の場合は最小移動距離を移動したとみなします。
エミッタが動かない状態でもパーティクルを表示しておきたい場合に使います。

以下は「 設定項目」の概要です。

最小移動距離 / 1フレーム

「最小移動距離 / 1フレーム」の値を「 ダイヤル」や「 テキストボックス」により設定します。
