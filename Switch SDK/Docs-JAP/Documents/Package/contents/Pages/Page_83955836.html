ライブラリが使用するファイル

このセクションでは、アプリケーションの実装において、実際にストレージ上に配置し、Atk ライブラリから利用する各ファイルについての解説を行います。

サウンドアーカイブファイル

Atk ライブラリが利用するサウンドデータの本体です。ファイルの拡張子は .bfsar で、SoundMaker によって作成されます。

ストリームサウンド、ウェーブサウンド、シーケンスサウンドの各サウンドと、プレイヤーやグループ、波形 アーカイブなどの情報が含まれています。シーケンスデータやバンク、ウェーブサウンドの波形データも含んでいますが、ストリームサウンドの波形データのみは、後述するストリームファイルを外部ファイルとして参照します。

サウンドアーカイブファイルのストレージ上での配置場所は任意です。

アイテム

サウンドアーカイブファイルの構成要素であるストリームサウンド、ウェーブサウンド、シーケンスサウンド、バンク、波形アーカイブ、プレイヤー、グループなどをアイテムと呼びます。

アイテムはすべて名前（文字列）で管理され、ひとつのサウンドアーカイブ内で合計約 43 億（2 の 32 乗）個のアイテムを利用することができます。

参照ファイルパス

サウンドアーカイブファイルは、ストリームサウンドの再生にストリームファイルを外部ファイルとして参照します。

ストリームファイルのパスは、サウンドアーカイブファイルのあるディレクトリからの相対パスで保存されています。

ストリームファイル

サウンドアーカイブファイルが利用するストリームサウンドの波形データです。ファイルの拡張子は .bfstmで、SoundMaker によって作成されます。

ストリームファイルのストレージ上での配置場所は、必ずサウンドアーカイブファイルに保存されたファイルパスによって参照可能な場所に配置する必要があります。

SoundMaker による出力先とストレージでの配置場所について

ストリームファイルは SoundMaker で作成される際に、通常はコンバートするサウンドプロジェクトファイルと同階層にある output ディレクトリに出力されます。この出力先は SoundMaker のプロジェクト設定によって、output 以下の任意のディレクトリに指定することができます。

プロジェクト設定で指定した output 以下のストリームファイルの位置は、サウンドアーカイブファイルにストリームファイルの参照ファイルパスとして保存されます。

output 以下をすべてそのままストレージの任意の場所に配置して利用することができます。
