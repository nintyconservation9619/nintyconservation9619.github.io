はじめに

DIAG ライブラリとは

DIAG ライブラリは、ログ出力・アサート・アボートなどの、プログラムのデバッグに使用する API を提供するライブラリです。

 

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象とします。

 

関連資料

DIAG ライブラリでは、多くの機能をマクロの形で提供していますので、マクロのリファレンスを参照してください。

また、DIAG の関数リファレンスも併せて参照してください。

これらのマクロ・関数に入力する文字コードについては、ログ・アサートの文字コードに関する注意点 に説明があります。

アサート・アボート時に出力される定型的な文字列およびその理由については、 プログラムの異常終了 をご覧ください。
