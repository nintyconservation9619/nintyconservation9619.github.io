はじめに

本ドキュメントの目的

NintendoSDK では HLSL から GLSL へのコンバーターとして HlslCrossCompiler と SLConverter を提供します。本ページではそれぞれのツールの概要を説明し、どちらを使用するかを選択する情報を提供します。

対象読者

NintendoSDK で HLSL コンバーターを使用して開発される方を対象としています。

SLConverter

頂点・ピクセルシェーダーのみをサポートし、シェーダーモデル 5.0 の限られた機能しか使用する事はできません。また、新規機能追加の予定はありません。従来から SLConverter を使用しているユーザー向けのツールです。

HlslCrossCompiler

HLSLcc (https://github.com/Unity-Technologies/HLSLcc) からフォークした hlslcc.dll (Tools/Graphics/GraphicsTools/hlslcc.dll) を利用しています。独自にテストや改善を行いコンピュートシェーダー、頂点・ピクセルシェーダーをサポートし、シェーダーモデル 5.0 の殆どの機能を使用する事ができます。また、全シェーダーステージへの対応・NVN の機能追加への追従等今後も継続してアップデートされます。
