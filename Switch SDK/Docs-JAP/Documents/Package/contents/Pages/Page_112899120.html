フォントのサイズに関する仕様

はじめに

本ページでは、フォントサイズの仕様について解説します。

フォントサイズについて

フォントサイズとは、フォントコンバータで Windows Font を入力とする場合と、スケーラブルフォントを利用する場合でのみ出てくる概念で、単位は px です。

フォントコンバータにおいて、Windows Font を入力とするときの文字幅に「フォントで指定される文字幅を使用する」を指定すると、ビットマップフォントのセルの横幅が指定したフォントサイズの値に設定されます。

スケーラブルフォントをプログラム中で GetWidth() した値は、フォントサイズの値と一致します。およそフォントサイズの値をフォントの横幅として描かれる感じで描画されますが、必ずしも個別の字形の横幅がフォントサイズの値と一致するわけではありません。

フォントコンバータで指定するフォントサイズと、スケーラブルフォントにおけるフォントサイズは共通した概念です。したがって、これらに同じ値を設定するとほぼ同じ描画結果になります。
