Barrier クラス

機能概要

Barrier クラスは、複数スレッド間で待ち合わせを行うための同期機能です。
予め決めておいた数の待機スレッドが揃った段階で、それら全てのスレッドが同時に待ち解除されて実行を再開します。

Barrier クラスは、nn::os::BarrierType オブジェクトを利用した バリア機能 をラッピングしたもので以下のメンバ関数を持ちます。API リファレンスはリンク先を参照して下さい。

名前 説明 備考
Barrier コンストラクタ InitializeBarrier() 同等の初期化を実行
~Barrier デストラクタ FinalizeBarrier() 同等の破棄を実行
Await 待ち合わせを行う AwaitBarrier() 同等のロックを実行

 

コンストラクタでは、待ち合わせを行うスレッドの数を指定することが出来ます。それぞれの機能詳細はバリア機能を参照して下さい。
