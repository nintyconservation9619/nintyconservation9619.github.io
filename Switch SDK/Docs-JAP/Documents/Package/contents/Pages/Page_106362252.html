エミッタ拡張パラメータ

エミッタ拡張パラメータとは？

ユーザーデータ定義を利用し、エミッタ単位でユーザーが拡張パラメータを付加することができます。エフェクトメーカー上で編集したエミッタ拡張パラメータは、エフェクトランタイム上で参照することができ、アプリとエフェクト再生の連携などに利用できます。

関連リンク

カスタムマイズとは 読み進める前に リンク先ページ内、「ユーザーデータ定義」を参照ください。
ユーザーデータ定義 ユーザデータ定義の仕様です。

 

エミッタ拡張パラメータを 編集・保存・バイナライズ するための準備

エフェクトメーカーでエミッタ拡張パラメータを編集・保存、バイナライズ するには、以下の準備を行う必要があります。

  • ユーザーデータ 定義ファイルの作成
  • エミッタ 拡張パラメータ 設定ファイルの作成
  • プロジェクト設定ファイルへの記述

ユーザーデータ定義ファイル の作成

作成したいパラメータ定義 と カスタムUI をエフェクトメーカー上で認識・表示させるために、ユーザーデータ定義 ページを参照し、ユーザーデータ定義ファイルを作成します。

作成したエミッタ 拡張パラメータ構造体 の先頭に以下のブロックを記述し、 UserDataDefinition 要素の、Kind 属性を EmitterExtParams とし、 エミッタ 拡張パラメータであることを宣言します。

エミッタ 拡張パラメータ 設定ファイルの作成

作成した ユーザーデータ定義ファイル をエフェクトメーカーに読み込ませるために、エミッタ 拡張パラメータ設定ファイルを作成します。

1
2
3
4		 
<?xml version="1.0"?>
<ArrayOfEmitterExtParamsDefinition xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
    <EmitterExtParamsDefinition Udd2Definition="EmitterExtParamsDefinition.h" />
</ArrayOfEmitterExtParamsDefinition>
参考：

カスタムデータ コンフィグファイルは、文字コード UTF-8 で保存する必要があります。

 プロジェクト設定ファイルへの登録

エフェクトメーカーでカスタムデータを利用するために、プロジェクト設定ファイルに、作成したカスタムデータコンフィグファイルのパスを記述します。詳しくは、プロジェクト設定ファイル を参照ください。
エミッタ 拡張パラメータ設定ファイルをエフェクトメーカーオプションで設定することでプロジェクト設定ファイルを利用せずに動作を確認することもできます。

エフェクトメーカー上でUI表示の確認

エフェクトメーカーを起動し、 エミッタセット新規作成 し、エミッタを選択右クリックから　「エミッタ拡張パラメータを追加」　を実行することで、エミッタに対してエミッタ拡張パラメータの追加が行えます。

// カスタムデータ構造体
struct ResCustomData
{
    nn::util::Float4    color;
    nn::util::Float3    rgb;
    float               alpha;
    nn::util::Float3    position;
    nn::util::Float2    texCoord;
    float               coefficient;
    int                 paramInt;
    uint32_t            flags;
    uint32_t            choices;
    uint32_t            choices2;
    char                usrString[32];
};

// エミッタセット生成
vfxSystem->CreateEmitterSetId( &handle, 0, resId, groupId, heap );
nn::vfx::EmitterSet* emitterSet = m_handle.GetEmitterSet();
 
// カスタムデータを取得
for ( int i = 0; i < emitterSet->GetCreatedEmitterCount(); i++ )
{
    const nn::vfx::Emitter* emitter = emitterSet->GetEmitter( i );
    const ResCustomData* customData = reinterpret_cast<const ResCustomData*>( emitter->GetCustomDataParameter() );
}

/// <?xml version="1.0" encoding="UTF-8"?>
/// <UserDataDefinition Kind="EmitterExtParams"
///                     Name="エミッタ拡張パラメータサンプル"
///                     Version="1">
#include <nn/util/util_MathTypes.h>
struct ResCustomData
{
    /// <ParameterDefinition Name="カラー"
    ///                      Type="ColorRgba">
    /// <ColorSetting Default="1 1 1 1" />
    nn::util::Float4  color;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="カラーのみ"
    ///                      Type="ColorRgb">
    /// <ColorSetting Default="1 0.7 0.5" />
    nn::util::Float3  rgb;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="アルファのみ"
    ///                      Type="ColorA">
    /// <ColorSetting Default="0.3" />
    float   alpha;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="位置座標"
    ///                      Type="Slider3f">
    /// <SliderSetting Default="0.1 0.2 0.3" Minimum="-1000000.0" Maximum="1000000.0" Step="1" />
    nn::util::Float3  position;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="テクスチャオフセット"
    ///                      Type="Slider2f">
    /// <SliderSetting Default="0.5 0.5" Minimum="0.0" Maximum="1.0" Step="0.01" EnableTransform="False" />
    nn::util::Float2  texCoord;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="パラメータ ( Float )"
    ///                      Type="Slider1f">
    /// <SliderSetting Default="0.5" Minimum="0.0" Maximum="1.0" Step="0.01" />
    float   coefficient;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="パラメータ ( 整数 )"
    ///                      Type="Slider1i">
    /// <SliderSetting Default="10" Minimum="0" Maximum="1000000" Step="1" />
    int     paramInt;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="フラグ"
    ///                      Type="CheckButtons"
    ///                      NeedReload="true">
    /// <OptionGroup>
    ///   <Option Value="スイッチ1" />
    ///   <Option Value="スイッチ2" />
    ///   <Option Value="スイッチ3" />
    /// </OptionGroup>
    unsigned int flags;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="選択肢"
    ///                      Type="RadioButtons">
    /// <OptionGroup Default="1">
    ///   <Option Value="選択肢1" />
    ///   <Option Value="選択肢2" />
    ///   <Option Value="選択肢3" />
    ///   <Option Value="選択肢4" />
    /// </OptionGroup>
    unsigned int choices;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="選択肢2"
    ///                      Type="RadioButtons">
    /// <OptionGroup Default="1">
    ///   <Option Value="選択肢2-1" />
    ///   <Option Value="選択肢2-2" />
    ///   <Option Value="選択肢2-3" />
    ///   <Option Value="選択肢2-4" />
    /// </OptionGroup>
    unsigned int choices2;
    /// </ParameterDefinition>
    /// <ParameterDefinition Name="ユーザー定義文字列"
    ///                      Type="StringComboBox">
    /// <OptionGroup>
    ///   <Option Value="UserDefinitionString1" />
    ///   <Option Value="UserDefinitionString2" />
    ///   <Option Value="UserDefinitionString3" />
    ///   <Option Value="UserDefinitionString4" />
    /// </OptionGroup>
    char    usrString[32];
    /// </ParameterDefinition>
};
