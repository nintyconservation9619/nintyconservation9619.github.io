放出

このタブではパーティクルの放出に関する設定を行います。

放出設定

「 放出設定」は、パーティクルの放出方法について設定する機能群です。
以下はグループ内設定項目の一覧です。

ワンタイム

パーティクルの放出時間を決める機能です。
放出開始フレーム パーティクルの放出タイミングを設定する機能です。
放出間隔タイプ パーティクルの放出間隔を設定する機能です。

チャイルドエミッタ時の設定

以下はチャイルドエミッタ用の設定項目です。
関連リンク チャイルドエミッタの詳細は 「チャイルドエミッタ」 を参照してください。

放出タイミング（%） チャイルドパーティクルの放出タイミングを設定する機能です。

初速設定

「 初速設定」は、放出されるパーティクルの初速について設定する機能群です。
以下はグループ内設定項目の一覧です。

全方向初速 エミッタ法線方向への初速を設定する機能です。
指定方向初速 指定した方向への初速を設定する機能です。
Y軸拡散初速

エミッタ座標系で Y 軸からの拡散速度を設定する機能です。
拡散初速 指定した軸方向への初速を設定する機能です。
初速のランダム 乱数値を与え、初速にばらつきを持たせる機能です。
参考：

各項目の計算結果を、上から順番に加算した結果が放出時の初速です。 

その他運動設定

「 その他運動設定」は、放出後のパーティクル速度について設定する機能群です。
以下はグループ内設定項目の一覧です。

空気抵抗 放出されたパーティクルの速度を減衰または加速させる機能です。
重力 放出されたパーティクルに対して指定した方向に加速度を与える機能です。
エミッタ速度継承率 エミッタの速度をパーティクル放出時に初速として継承する機能です。
運動量ランダム 放出時にパーティクル個々が持つ運動量をランダムにする機能です。
