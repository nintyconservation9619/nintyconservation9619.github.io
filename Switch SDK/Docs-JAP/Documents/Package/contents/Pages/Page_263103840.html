中間ファイルバージョンの更新

3D 中間ファイルバージョンを 3.7.0 から 4.0.0 へ更新しました。主な変更点については 3D 中間ファイル仕様書 の更新履歴をご覧ください。

既存の中間ファイルは 中間ファイルアップデーター で更新して下さい。この中間ファイルバージョン更新により、3D バイナリーコンバーターの出力するバイナリーリソースファイル(.bfres)に影響はありません。

最適化シェーダーのコンパイル時情報をマテリアルごとに確認できるように

最適化シェーダーのコンパイル時情報を、マテリアルごとに確認できるようになりました。

注意：

NintendoSDK 5.0.0 ではこの機能に不具合があり、以下の場合に正しい情報が出力されません。

  • 同一シェーディングモデルが割り当てられたモデルが複数読み込こまれているとき
  • シェーダーのリアルタイム編集を元に戻す操作を行ったとき
  • マテリアルのオプションを編集して以前の状態に戻したとき

マテリアルにラベルを設定できるように

マテリアル、モデル、ボーン、テクスチャー、アニメーションにラベルを設定できるようになりました。

注意：

現在は親マテリアル選択時のマテリアル名のみが設定したラベル名に置きかわります。

ラベルがあれば「ラベル(マテリアル名)」、なければ「マテリアル名」が表示されます。
