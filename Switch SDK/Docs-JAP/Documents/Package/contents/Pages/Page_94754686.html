パーティクル形状「プリミティブ」

ここではパーティクル形状「プリミティブ」について説明します。

概要

パーティクル形状に「プリミティブ」を指定することで、DCC ツールで作成した自由な形状のポリゴン形状をパーティクルとして利用できます。
利用可能なアトリビュートは DCC ツール側でコントロールできます。

以下は「 設定項目」の概要です。
「パーティクル形状」の設定を「プリミティブ」にすると グループ『プリミティブ』が表示されます。以下はグループ内の設定項目です。

ファイル指定

プリミティブファイルの登録や更新、削除を行います。以下はボタンの種類です。

ロード ファイル選択ウィンドウが開き、モデルファイル（.fmdb）が選択できます。
サンプルプリミティブを開く Primitives フォルダを開きます。良く使用するモデルファイル（.fmdb）はこの場所に置くことができます。
Primitives フォルダは以下の場所に設定されています。
"EffectMaker を起動したフォルダ"\Primitives
リロード 設定したモデルファイル（.fmdb）が更新されます。
クリア 設定したモデルファイル（.fmdb）を解除します。

プリミティブを読み込むと、そのプリミティブに関して以下の情報がコメントとして表示されます。
プリミティブの負荷や内包するアトリビュート情報を確認できます。

  • 頂点数
  • 内包するアトリビュート( pos（頂点位置） / normal（法線） / tangent（接線） / color（頂点カラー） / uv（テクスチャ座標）/ LOD数 )
