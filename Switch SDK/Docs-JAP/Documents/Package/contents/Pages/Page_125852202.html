4 マテリアルの編集

demo シェーダにテクスチャを追加、設定する手順を説明します。
パッケージ内にあるサンプルのテクスチャを追加、マテリアルを調整することでモデルの質感を調整します。

このチュートリアルでは以下の UI について触れています。

プロパティウインドウ マテリアルのプロパティウインドウを開きます。
シェーダパラメータの編集

スペキュラや法線マップを例にして、マテリアルに割り当てられたシェーダパラメータを編集します。

参考：

完成データは以下の場所にあります。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\SampleTripod_Finish.fmdb

 

プロパティウインドウ

体部分のマテリアルを編集します。

スケマティック表示のツリーからマテリアル「 BodyMt 」をダブルクリックしてプロパティウインドウを開きます。

 

参考：

どのマテリアルが選択されているかビューアで確認できます。

3DEditor のスケマティック表示のツリーのマテリアルをクリックすると対応個所がビューアで点滅します。
シェイプ、モデルも同様に確認できます。

下図のウインドウが開きます。
このページでは、以降はプロパティウインドウ内での操作を行います。

関連リンク 詳しくはマテリアルプロパティウィンドウを参照してください。

シェーダパラメータを編集

 スペキュラや法線マップを例にして、マテリアルに割り当てられたシェーダパラメータを編集します。

項目の選択はプロパティウインドウのツリーリスト(下図)から選択します。
ここでは「 スペキュラ 」と「 法線 」を編集します。

スペキュラの調整

プロパティウインドウの ツリーリストからスペキュラを選択します。
スペキュラ強度とスペキュラ絞りの値を以下のように変更します。

値を変更するとリアルタイムにビューアに表示されます。

スペキュラ強度 スペキュラ絞り
1.0 10.0

スペキュラマスクの追加

スペキュラマスクの「チェックボックス」を ON にします。

下記のテクスチャをスペキュラマスクの「未指定」にドラッグ＆ドロップします。

スペキュラマスク

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\textures\SampleTripodA01Specular.ftxb

下図の赤枠にドロップします。

テクスチャが登録されると下図のようになります。

リアルタイムにビューアに表示されます。
下図のように黒い部分のスペキュラが無くなります。

参考：

★は、保存時から変更された箇所に表示されます。
変更した箇所を元に戻す、または保存すると★は表示されなくなります。

法線マップの追加

プロパティウインドウの ツリーリスト から 法線を選択します。

法線マップの有効化の「チェックボックス」を ON にします。

下記のテクスチャを法線マップの「未指定」にドラッグ＆ドロップします。

法線マップ

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\textures\SampleTripodA01Normal.ftxb

テクスチャが登録されると下図のようになります。

下図のように凹凸が付きます。

プロパティウインドウを閉じて、マテリアルの編集を終了します。

登録したテクスチャ一覧や変更はサンプラで行います。
関連リンク 詳しくはマテリアルプロパティウィンドウサンプラページを参照してください。
