成分 0 ～ 成分 3の振幅設定

ここでは波形成分「成分0（±） ～ 成分3（±）」について振幅を調整する機能について説明します。

概要

「成分 0（±） ～ 成分 3（±）」は、「ノイズの強さ」で設定した振幅に重み付けができる機能です。
数値を大きくすると GPU ノイズに使用する成分の振幅が大きくなり、小さくすると振幅が小さくなります。
以下は「 設定項目」の概要です。

「各成分の設定」を ON にすると「 各成分の振幅に対する重み付け」が表示されます。以下はグループ内の設定項目です。

成分 0（±） ～ 成分 3（±）

GPU ノイズに使用する波形成分が、ノイズの強さに対してどの程度の重み付けをするか設定します。
以下は、各成分の振幅をもとめる計算式です。

  • 各成分の振幅 : ノイズの強さ × 各成分の振幅に対する重み付け
