店頭体験版向け機能

概要

店頭体験版アプリケーションは店頭試遊台上で動作します。店頭試遊台は店頭体験版モードと呼ばれる通常とは異なるモードで動作し、店頭体験版モード向けの機能が用意されています。
開発機を店頭体験版モードで動作させる方法は 店頭体験版の動作確認環境の構築 を参照してください。

参考：

店頭体験版モード向けの機能は、NX Addon 5.3.0以降で使用可能です。

使用方法

店頭体験版モード判定機能

nn::oe::IsInStoreModeEnabled()を使用することで、店頭体験版モードが有効であるか否かを判定することできます。
店頭体験版モードでなくても本機能を使用することができ、ユースケースとしては店頭体験版モードか否かで処理を分岐させたい場合が考えられます。

  1. OEライブラリのヘッダーファイルをincludeしてください。

    #include <nn/oe/oe_InStoreDemoApis.h>

  2. OEライブラリの店頭体験版モード判定機能を呼び出してください。

    nn::oe::IsInStoreDemoModeEnabled();

店頭体験版アプリケーション終了機能

nn::oe::ExitApplicationAndGoBackToInStoreDemoMenu()を実行すると、店頭体験版アプリケーションが終了され店頭体験版メニューに戻ります。
店頭体験版モードでない場合には内部的にアボートするので注意してください。

  1. OEライブラリのヘッダーファイルをincludeしてください。

    #include <nn/oe/oe_InStoreDemoApis.h>

  2. OEライブラリの店頭体験版アプリケーション終了機能を呼び出してください。

    nn::oe::ExitApplicationAndGoBackToInStoreDemoMenu();
