SoundMaker や AtkPlayer でサウンドを再生する

1. 概要

SoundMaker や AtkPalyer でサウンドを再生する手順です。

SoundMaker は PC 版のみで PC 上でサウンド再生します。
AtkPlayer には PC 版と実機版があり、PC 版はキーボードで操作、実機版はコントローラーで操作してサウンドを再生します。

参考：

複数のサウンドを再生する場合、「SoundMaker や AtkPlayer で複数のサウンドを再生する」をご確認ください。

2. SoundMaker でサウンドを再生、停止する

2.1 SoundMaker でサウンドを再生する

1. サウンドセットタブのリスト領域で、再生したいアイテムをクリックして選択します。

2. メインウインドウ上部やリスト内の再生ボタンをクリックすると、PC スピーカーからサウンドが再生されます。

2.2 SoundMaker で再生中のサウンドを停止する

メインウインドウ上部の「すべて停止」ボタンをクリックすると、再生中のサウンドが停止します。

3. AtkPlayer でサウンドを再生、停止する

3.1 AtkPlayer でサウンドを再生する

3.1.1 AtkPlayer で再生操作を行う

1. AtkPlayer を起動します。

2. SoundArchivePlayer パネルで再生したいサウンド名を表示させます。← → ボタン（キー）で再生対象のサウンドを切り替えることができます。

3. A ボタン（キー）でサウンドが再生されます。

3.1.2 SoundMaker で再生操作を行う

1. AtkPlayer を起動します。

2. AtkPlayer と SoundMaker を接続します。

3. サウンドセットタブのリスト領域で、再生したいアイテムをクリックして選択します。

4. メインウインドウ上部やリスト内の再生ボタンをクリックすると、AtkPlayer でサウンドが再生されます。        

3.2. AtkPlayer で再生中のサウンドを停止する

3.2.1 AtkPlayer でサウンドの停止操作を行う

SoundArchivePlayer パネルにて B ボタン（キー）操作を行うとサウンドが停止します。

3.2.2 SoundMaker でサウンドの停止操作を行う

AtkPlayer と接続中の SoundMaker で操作を行います。

メインウインドウ上部のすべて停止ボタンをクリックすると、再生サウンドを停止します。

4. 困ったときは

SoundMaker

再生サウンドが聴こえない

  • ヘッドホンやスピーカーが PC に接続されていることを確認してください。
  • [ツール]→[オプション]より、オーディオタブの出力先デバイスがサウンドの出力先として適切か確認してください。

「すべて停止」ボタンでサウンドが停止しない

  • ツールバーの【全発音停止】で停止してください。
    再生サウンドのパラメータ「Env R」が 0 またはそれに近い値になっているインストルメントでは、パーカッションモードが有効なリージョンを鳴らしているケースが考えられます。

AtkPlayer

サウンドが停止しない

  • AtkPlayer を再起動してください。
    再生サウンドのパラメータ「Env R」が 0 またはそれに近い値になっていることが考えられます。ループする波形の Env R を 0 に設定して再生すると、停止操作ではサウンドを停止できなくなります。

SoundArchivePlayer パネルでサウンド名を切り替えても、インストルメントが見当たらない

    • AtkPlayer はインストルメント単位での再生に対応していません。
