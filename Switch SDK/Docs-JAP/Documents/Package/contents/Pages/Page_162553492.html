オーディオパフォーマンス情報

nn::audio のパフォーマンス

nn::spy::audio ライブラリは、nn::audio ライブラリからパフォーマンス情報を収集し、Spy.exe に送信します。

パフォーマンス情報は、以下の nn::audio ライブラリのコンポーネントごとに処理負荷を取得します。
各コンポーネントの詳細については、オーディオレンダラの解説ページを参照ください。 

コンポーネント Spy.exe における表示内容の説明
Voice

再生中の Voice の処理負荷を表示します。
SubMix

SubMix の処理負荷を表示します。

SubMix に紐づくエフェクトの処理負荷も含みます。
FinalMix

FinalMix の処理負荷を表示します。

FinalMix に紐づくエフェクトの処理負荷も含みます。
Sink

Sink の処理負荷を表示します。

nn::atk のパフォーマンス

nn::spy::atk ライブラリは、nn::atk ライブラリからパフォーマンス情報を収集し、Spy.exe に送信します。

nn::audio のパフォーマンス情報に加え、nn::atk の内部スレッドの処理負荷も表示します。
各スレッドの説明は、Atk のスレッド解説ページを参照ください。

スレッド 項目 説明
サウンドスレッド NwFrame nn::atk ライブラリの内部状態の更新やパラメータ計算の処理負荷を表示します。
タスクスレッド bfstm load bfstm ファイルロードの開始と終了時刻を表示します。
タスクスレッド opus load opus ファイルロードの開始と終了時刻を表示します。
タスクスレッド opus decode opus デコードの開始と終了時刻を表示します。
