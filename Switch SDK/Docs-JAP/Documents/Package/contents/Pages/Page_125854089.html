出力ファイル

FontConverter が出力するファイルについて説明します。
「出力」設定のタブにより、変換されるファイルが異なります。

「Image」タブでの変換

FontConverter が解釈することのできる特殊な形式の画像ファイル (.bmp)に変換します。
関連リンク 詳細については「フォントコンバータファイルフォーマット」を参照してください。

出力した画像ファイルは、画像編集ソフトで、デザインに修飾をほどこすことができます。
修飾後の画像ファイル (.bmp)は、以下の設定により LayoutEditor で使用可能なフォントの元データーとして利用できます。

  • 「入力」の「Image」タブに修飾後の画像ファイル (.bmp)を設定します。
  • 「出力」の「bffnt / bffna」タブにビットマップフォントバイナリファイル (.bffnt)を設定し、変換ボタンを押します。

「bffnt / bffna」タブでの変換

フォントライブラリが処理するビットマップフォントバイナリ (.bffnt)に変換します。
変換するにより、LayoutEditor で使用可能になります。
