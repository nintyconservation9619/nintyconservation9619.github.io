シェーダー定義プロパティウィンドウ シェーダー定義ページ

選択したシェーダー定義ファイル情報を確認するページです。

A シェーダーソースコードのコードページを表示します。
コードページは、中間ファイルの <shader_definition_info> で指定します。
<shader_definition_info>　の詳細は、別途用意されている中間ファイルフォーマットのドキュメントで確認してください。
B シェーダー定義ファイルの生成元であるシェーダー設定ファイルのパスです。
シェーダーコンバータで設定したルートパスからの相対パスか、ファイル名が表示されます。
C シェーダー定義ファイル内にあるプログラムとプログラムで使用されているシェーダー ( .glsl ) を一覧表示します。
D C. の一覧で選択したプログラムの頂点シェーダー、フラグメントシェーダー、ジオメトリシェーダー、演算シェーダーのシェーダーソースコードをタブで管理して表示します。
