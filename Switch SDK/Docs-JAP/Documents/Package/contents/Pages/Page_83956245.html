コピー＆貼り付け機能を利用する

コピー＆貼り付け制御を利用する

ここでは、コピーと貼り付け機能の説明を行います。

ペインを複製する

一般的な、コピー、貼り付けと同様に、以下の操作でコピー貼り付け操作を行うことが可能です。

[ メニュー ] → [ 編集 ] → [ コピー ]
[ メニュー ] → [ 編集 ] → [ 貼り付け ]

新規に、コピーしたペインと同一のパラメータを持つペインが複製されたことが確認できます。



ここで、アニメーションに関連する情報は、コピーされない点に注意してください。
アニメーション情報をコピーするには、後述の「コピー貼り付け制御ウィンドウ」を利用します。
階層構造については、コピーされます。グループ情報に関しては、コピーされません。

ペイン情報貼り付け制御ウィンドウを利用する

以下のような操作を行う場合に、ペイン情報貼り付け制御ウィンドウが利用できます。

・既存のペインに対して、ペイン情報の一部分だけを貼り付けて上書きしたい。
・複数のペインに対して、一斉に貼り付け処理を行いたい。
・アニメーションデータを貼り付けたい。

ペイン情報貼り付け制御ウィンドウの表示切り替えは、
[ メニュー ] → [ ウィンドウ ] → [ ペイン情報貼り付け制御 ] で行うことが可能です。

ペイン情報の一部分だけを貼り付けて上書きする

まず、コピー元ペインを選択してコピーを実行します。このとき、コピー元のペインは、単数ペインである必要があります。
次に、貼り付け先のペインを選択します。ペイン情報貼り付け制御ウィンドウの表示が変更され、有効な項目が変化することが確認できます。
貼り付け元と、貼り付け先が、同じ情報項目を持っている場合は、有効な項目として判定されます。
有効な項目のうち、コピーしたい項目にチェックをします。
ウィンドウ下部の「選択ペインに貼り付け」をクリックします。
選択ペインに対して、指定の情報が貼り付けられていることが確認できます。

ベクトル成分貼り付け制御

この機能を使用すると、貼り付ける情報のなかで、ベクトル情報に相当する情報について、貼り付ける成分を制御することができます。
たとえば、サイズパラメータの X 成分だけを貼り付けることによって、ペインの幅だけを貼り付けるような操作を行うことが可能になります。


この設定は、ベクトル情報形式のパラメータ全体に対する設定になります。( つまり、位置とサイズで共有される設定となります。 )
色成分貼り付け制御も同様の制御を色 ( RGBA ) 形式のパラメータについて行うことが可能です。

アニメーション情報の貼り付け

通常のパラメータと同様に、アニメーションを貼り付けることが可能です。
貼り付け先のキー情報を置き換えるを有効にした場合、貼り付け操作前に、現在のキーフレームを消去します。
そのため、貼り付け先の情報が、貼り付け元の情報によって、完全に置き換えられます。
