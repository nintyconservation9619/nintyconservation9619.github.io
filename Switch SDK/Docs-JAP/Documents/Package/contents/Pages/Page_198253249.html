フレーム値にオフセットを加算ウインドウ

「フレーム値にオフセットを加算ウインドウ」では、選択したアニメーション区間タグのフレーム値を一括で編集します。

フレーム値にオフセットを加算ウインドウ
A 全てチェック
 表示されているアニメーション区間タグすべてにチェックが入ります。
B アニメーション区間タグ名 追加されているアニメーション区間タグが表示されます。
C フレーム長を一括設定する チェックが入っているとアニメーション区間タグのフレーム長を一括で設定できます。
D フレーム長 ここで設定する値でフレーム長が一括設定されます。
E フレームにオフセット値を加算する チェックが入っているとアニメーション区間タグのフレームにオフセット値が加算されます。
F 加算オフセット ここで設定する値がアニメーション区間タグのフレームに加算されます。
