メインウィンドウ

以下は FontConverter 「メインウィンドウ」では～の操作ができます。

A 「入力」項目
入力ファイルの指定や入力設定を行います。
B フィルタを使う

フォントを作るさいに必要な文字だけを抽出する「フィルタ」機能が利用できます。

関連リンク フィルタの書式については 「文字フィルタファイル(.xllt)」 を参照してください。
C 「出力」項目 出力時先や出力設定を行います。
D ステータスバー 変換状況をグラフィカルな表示でお知らせします。
E 「変換」ボタン マウス左クリックで変換を開始します。

関連リンク 以下は関連ページへのリンクです。
