Keyboard (NX Addon)

NX Addon の導入によって、下記の環境においても Keyboard の利用が可能になります。

SDEV 環境における構成

USB キーボード

USB 接続されたキーボードの状態から入力状態を決定します。

複数台のキーボードが接続された場合は、それらの押下状態の論理和が入力状態となり、 ロックキーの状態は共有されます。

最大で 8 台までのキーボードが認識され、それ以降に接続されたキーボードは単に無視されます。

他にも HID 機器が USB 接続されている場合は、その数の分だけ認識されるキーボードの台数の上限が下がります。

Input Director による入力転送

Input Director でホスト PC から転送されたキーボードの状態から入力状態を決定します。

USB キーボードからの入力が存在する場合は、 それとの論理和が最終的な入力状態となります。

EDEV 環境における構成

SDEV 環境と同一です。
