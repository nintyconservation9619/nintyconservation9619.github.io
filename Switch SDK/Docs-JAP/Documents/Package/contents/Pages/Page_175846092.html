寿命

ここではパーティクルの寿命を設定する機能「寿命」について説明します。

概要

「寿命」は、パーティクルがエミッタから放出されてから消滅するまでの時間を設定できる機能です。

 

参考：

パーティクル「寿命」の時間軸で行われるアニメーションを「パーティクル時間アニメ」と呼びます。

 関連リンク 「パーティクル時間」の詳細については「エミッタ時間とパーティクル時間」を参照してください。


以下は「 設定項目」の概要です。

寿命

パーティクルがエミッタから放出されてから消滅するまでの時間を「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は「寿命」の値を変更したときの比較画像です。

A 20 パーティクルが放出されてから20フレームで消滅します。
B 50 パーティクルが放出されてから50フレームで消滅します。
C 100
 パーティクルが放出されてから100フレームで消滅します。
