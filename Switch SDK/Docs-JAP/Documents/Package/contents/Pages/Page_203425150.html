警告メッセージ一覧

VFXランタイムから出力される警告の内容と解決方法の一覧です。

出力タイミング メッセージ 発生する条件 解決方法
エフェクトシステム生成時

[Stripe] Memory Allocate Error!!

[ConnectedStripe] Memory Allocate Error!!

[SuperStripe] Memory Allocate Error!!

 ストライプシステムの初期化時( vfx システム初期化時 )に、静的ヒープからバッファを確保できなかった。 nn::vfx::Config::SetStaticHeap で設定するヒープのバッファを増強してください。
エフェクトバイナリ登録時

vfx system deleted the registered binary.

resource id : XX.

 指定リソースIDに登録済みバイナリが存在した為、登録済みバイナリを削除した場合。

vfx システム破棄前に登録済みエフェクトバイナリを破棄( nn::vfx::System::ClearResource )してください。
エフェクトバイナリ削除時 resource to clear does not exist.
resource id : XX.

vfx システムからエフェクトバイナリを破棄する際、指定したリソースIDに既に登録済みエフェクトバイナリが存在しなかった。

 vfx システムからエフェクトバイナリを破棄する際、指定リソースIDが使用中か確認を行ってください。

エフェクト生成時

 A resource appointed at the time of emitter set generation is not registered.

エミッタセット生成時に指定されたリソースが登録されていない。

CreateEmitterSet 呼び出し時に渡している リソースID の確認、または、利用するエフェクトバイナリがシステムに登録済みか確認を行ってください。
There is not emitter set information to be formed within an appointed resource. 指定されたリソース内に生成するエミッタセット情報がない CreateEmitterSet 呼び出し時に渡している リソースID の確認、または、利用するエフェクトバイナリがシステムに登録済みか確認を行ってください。
EmitterResource Setup Failed.

何らかのバッファ不足等により、エミッタを再生する為のリソースセットアップに失敗した。

シェーダ や 描画用プリミティブ の セットアップに失敗しています。
利用するエフェクトバイナリ時に他の警告やエラーが出力されていないか確認を行ってください。
CustomShader Callback not unestablished. CustomShaderID:XX

エミッタパラメータでカスタムシェーダを利用するように設定されているが、利用するカスタムシェーダコールバックが設定されていない。

 カスタムシェーダのパラメータとコールバック関数の設定状況に問題がないか確認を行ってください。

CustomAction Callback is not unestablished.

CustomActionID:%d

エミッタパラメータでカスタムアクションを利用するように設定されているが、利用するカスタムアクションコールバックが設定されていない。

 カスタムアクションのパラメータとコールバック関数の設定状況に問題がないか確認を行ってください。
There is no available EmitterSet instance. 利用可能な エミッタセット インスタンスが検索できなかった。 nn::vfx::Config::SetEmitterSetCount でエミッタセットインスタンスを増強してください。
There is no available Emitter instance. 利用可能な エミッタ インスタンスが検索できなかった。 nn::vfx::Config::SetEmitterCount でエミッタインスタンスを増強してください。

DynamicHeap Alloc Failed.

Alloc size : XX Byte.

 動的ヒープが不足している。 nn::vfx::Config::SetDynamicHeap で設定するヒープのバッファを増強してください。

GpuBuffer Alloc Failed. Alloc size : XX Byte.

GpuBuffer Alloced : AllocedSize/BufferSize.

 Gpuバッファが不足している。 nn::vfx::Config::SetGpuBufferSize で設定するGpuバッファサイズを増強してください。
エフェクト削除時 EmitterSet has been already deleted. 削除済みエミッタセットが削除された。 削除済みエミッタセットが削除されるようなパスがなか確認を行ってください。
エフェクトバイナリ登録時

vfx system deleted the registered binary.

resource id : XX.

 指定リソースIDに登録済みバイナリが存在した為、登録済みバイナリを削除した場合。

vfx システム破棄前に登録済みエフェクトバイナリを破棄( nn::vfx::System::ClearResource )してください。
エフェクトバイナリ削除時 resource to clear does not exist.
resource id : XX.

vfx システムからエフェクトバイナリを破棄する際、指定したリソースIDに既に登録済みエフェクトバイナリが存在しなかった。

 vfx システムからエフェクトバイナリを破棄する際、指定リソースIDが使用中か確認を行ってください。
エフェクト描画時 Particle sort has failed. More buffer size is needed. パーティクルソートバッファが不足している。 nn::vfx::Config::SetParticleSortBufferCount で設定するソートバッファを増強してください。

TemporaryBuffer Alloc Failed. Alloc size : XX Byte.

TemporaryBuffer Alloced : AllocedSize/BufferSize.

 テンポラリバッファが不足している。 nn::vfx::Config::SetTemporaryBufferSize で設定するGpuバッファサイズを増強してください。
ストライプ There is no available Stripe instance. 利用可能な ストライプ インスタンスが検索できなかった。 nn::vfx::Config::SetStripeCount / SetSuperStripeCount で生成可能なストライプ数の増強を行ってください。
参考：

nn::vfx::Config や 各バッファに関する情報は、ランタイム内部で消費するメモリについて を参照してください。
