Zバッファ、アルファテスト

ここではパーティクル描画時の深度やアルファのテストを行う機能「Zバッファ、アルファテスト」について説明します。

概要

「Zバッファ、アルファテスト」は、パーティクル描画時の深度テストやアルファテストを設定できる機能です。
良く使用する設定の組み合わせは、プリセットとして用意されています。また、「カスタム」では自由な組み合わせが設定できます。
以下は「 設定項目」の概要です。

Zバッファ、アルファテスト

パーティクル描画時の Zバッファとの比較テストと アルファテストの組み合わせを、以下のプリセットから選択します。
