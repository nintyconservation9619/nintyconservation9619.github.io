カラーコンバイナの計算式

ここではカラーコンバイナの計算式について説明します。

概要

カラーコンバイナの「計算式」は、カラー要素を合成する際に適用する計算式を設定できる機能です。
表現の目的に合った計算式に変更することでエフェクトのクオリティを上げることができます。
以下はカラーコンバイナ式で入力値として扱うカラー要素のイメージとその略語です。

Comb_RGB

コンバイナ設定上段の RGB 値です。
ここでは登録したテクスチャのアルファチャンネルを RGB 値として出力しています。
█
C0 カラータブで設定したカラー 0 の RGB 値です。
ここでは固定値として RGB = 1.0,0.5,0.0 を設定しています。
█ C1 カラータブで設定したカラー 1 の RGB 値です。
ここでは固定値として RGB = 0.0,0.5,1.0 を設定しています。

カラー計算式

上段の RGB 値（Comb_RGB）と 「カラー」タブで設定する RGB 値との合成方法を、用意された計算式から選択します。
以下は合成の結果、出力されるイメージと計算式です。

C0 カラー 0 のカラーをそのまま出力します。
C0 × Comb_RGB カラー 0 と Comb_RGB の出力結果を乗算した結果を出力します。
C0 × Comb_RGB ＋ C1 × (1 － Comb_RGB) Comb_RGB の出力結果中、白カラーをカラー 0 、黒カラーをカラー1 として合成した結果を出力します。
C0 × Comb_RGB ＋ C1 カラー 0 と Comb_RGB を乗算した値にカラー1 を加算した結果をを出力します。
例えばポストエフェクト「グレア」を使用している場合、カラー 1 で効果の度合いを調整できます。

設定例

以下はポストエフェクト「グレア」が利用できる環境下で２つの式を比較した画像です。

「グラデーションエディタ」の設定です。
アニメーションの最初と終わりは黒です。
アニメーションの中間地点で青カラーを 1 以上に設定することでグレア効果を強くしています。
A C0 × Comb_RGB ＋ C1 × (1 － Comb_RGB)

テクスチャの 白い部分に カラー 0 、黒い部分をカラー 1 を適用しています。
テクスチャの 黒い部分のみがグレアの影響を受けて光らせる使用例です。
B C0 × Comb_RGB ＋ C1 カラー 0 とテクスチャを乗算したあと カラー 1 の値を加算しています。
全体的にグレアの影響を受けて光らせる使用例です。
