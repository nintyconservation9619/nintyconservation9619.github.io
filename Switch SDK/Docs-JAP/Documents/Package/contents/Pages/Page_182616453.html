カラーピッカー

「カラーピッカー」では、色の指定を視覚的に設定できます。

 

A 現在のカラー 現在のカラーが表示されます。
B

現在のカラーをカラーパレットに登録します。

※アルファ成分は登録されません。
カラーパレット

カラーパレットの色を左クリックで選択します。
選択した色を現在のカラーとして設定します。
C カラー色調 カラーの色調（Hue）を変更します。
D カラーの彩度 / 明度 カラーの彩度（Saturation）と明度（Value）を変更します。
E 16 進数 入力フォーム RGB値を 16 進数で入力できます。
F

「リニアガンマ変換」ガンマ値に影響を与えます。

「 sRGB ガンマ変換」ガンマ値に影響を与えます。
G カラー成分の表示形式を、RGB 形式 / HSB 形式で切り替えます。
カラー成分の表示 カラーの値を数値入力や、スライダーで設定します。
