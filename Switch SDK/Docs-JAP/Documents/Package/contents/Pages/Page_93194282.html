プロパティウィンドウの基本操作

プロパティウィンドウとは、3DEditor に読み込んだモデルや、アニメーションのパラメータを確認、編集するウィンドウです。

ここでは、プロパティウィンドウの基本操作を説明します。

関連リンク プロパティウィンドウの項目は選択したオブジェクトにより変化しますので、詳細は各プロパティウィンドウを確認してください。

プロパティウィンドウを開く

プロパティウィンドウは、下記方法などで開くことができます。

  • ファイルビューでファイル名をダブルクリック
  • オブジェクトビューでオブジェクトをダブルクリック
  • オブジェクトビューで選択したオブジェクトの右クリックメニューにある「プロパティウィンドウ」を選択
参考：

下記の方法でもプロパティウィンドウを開くことが出来ます。

  • メインメニュー「表示」→「プロパティウィンドウ」
  • ツールバー「プロパティウィンドウを開く」
  • ショートカットキー「Ctrl+P」

 

プロパティウィンドウのアイコン

プロパティウィンドウの基本操作は右上にあるアイコンで操作します。

アイコン 機能 説明
プロパティのコピー ツリービューで選択されている項目のプロパティをコピーします。（項目に子項目がある場合は、子項目も含めてコピーします） 
プロパティの貼り付け プロパティのコピーでコピーした項目を子項目を含めて貼り付けます。
マテリアル設定ファイルのエクスポート マテリアルの設定をマテリアル設定ファイル （ .f3mt ） にエクスポートします。（この機能はマテリアルプロパティウィンドウのみ有効です）
エクスポートした内容は、他のマテリアルにインポートすることができます。 
関連リンク エクスポートをする際は、エクスポートダイアログが表示されエクスポートする項目を細かく設定できます。
マテリアル設定ファイルのインポート

マテリアル設定ファイル （ .f3mt ） を任意のマテリアルにインポートします。（この機能はマテリアルプロパティウィンドウのみ有効です）
関連リンク インポートを実行する際は、インポートダイアログが表示され適用する項目を細かく設定できます。
プロパティウィンドウの複製

選択しているプロパティウィンドウをそのまま複製して、別ウィンドウで開くことができます。
ヘルプを開く アイコンを押すとプロパティウィンドウの HTML ヘルプが開きます。
