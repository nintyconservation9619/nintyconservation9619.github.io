キャプチャペイン

「キャプチャペイン」は、フレームバッファやレイアウトのレンダリング結果をキャプチャして テクスチャとして使用します。
プロパティウインドウの中の「キャプチャペイン」では、LayoutEditor のキャプチャペイン情報編集をします。

参考：

キャプチャペイン機能を使用するには、「プロジェクト設定ファイル」でキャプチャペイン機能を有効にする必要があります。

キャプチャペインプロパティ

プロパティウインドウの「キャプチャペイン」から、キャプチャテクスチャ設定をします。

キャプチャテクスチャ設定項目

フレームバッファキャプチャ有効

 レイアウトウィンドウ上「キャプチャ領域」のフレームバッファをテクスチャとして取り込む設定です。

初回フレームのみキャプチャ

テクスチャを初回フレームに一度だけ生成する設定です。

「キャプチャペイン」階層下のペインをアニメーションさせたい場合は「 OFF」にします。
テクスチャスケール キャプチャペインの解像度を最大として、設定した割合でスケールをかけてテクスチャを生成します。
クリアカラー

以下のケースでテクスチャカラーを設定します。

  • フレームバッファのキャプチャを行わない場合
  • キャプチャ領域がフレームバッファのテクスチャサイズからはみ出た場合

特にアルファ値に 0 を設定することで、生成するテクスチャのアルファ初期値が 0 となります。

その結果「キャプチャペイン」階層下にあるペイン外側の透過部を、生成するテクスチャでは透明にできます。
参考：

キャプチャされるテクスチャのサイズが変わるようなデータはランタイムで再現できません。
キャプチャペインのサイズやスケールをアニメーションさせないようにご注意ください。
