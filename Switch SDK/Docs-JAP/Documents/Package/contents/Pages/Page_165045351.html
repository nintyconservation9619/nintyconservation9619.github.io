エフェクトを違和感なく消したい

エフェクトを違和感なく消す方法について説明します。

Nの文字の消す 煙エフェクトを消す
A 透明にして消す 「 カラータブ」のアルファ設定でアルファ値をアニメーションさせて消します。
B 小さくして消す 「 スケールタブ」のスケールアニメーションで消します。
C ちぎって消す

「 コンバイナタブ」で専用のアルファ式に変更すると利用できます。
「 カラータブ」のアルファ設定でアルファ値をアニメーションさせて消します。

 

A 透明にして消す

パーティクルを透明にして消す方法について説明します。
以下はこの方法の特徴です。

利点
  • 厚みがないものや消滅時の尺が短いエフェクトに向きます。
  • エフェクトのサイズを維持した状態で消したいときに使えます。

 
欠点
  • 立体物や爆発など、量感をもとめられるエフェクトには不向きです。
  • 半透明描画なので、描画優先を気にする必要があります。
  • 表示面積が大きいと描画負荷がかかります。

B 小さくして消す

パーティクルにスケールをかけ、小さくして消す方法について説明します。
以下はこの方法の特徴です。

利点
  • 厚みがないものや消滅時の尺が短いエフェクトに向きます。
  • エフェクトのサイズを維持した状態で消したいときに使えます。
  • 「Zバッファ・アルファテスト」の設定が「抜き」や「不透明」で透明にできないときに使えます。
欠点
  • 立体物や爆発など、量感をもとめられるエフェクトには不向きです。
  • 半透明描画なので、描画優先を気にする必要があります。
  • 表示面積が大きいと描画負荷がかかります。

C ちぎって消す

アルファの閾値をアニメーションさせることで、パーティクルをちぎって消す方法について説明します。
消え方はテクスチャのアルファ画像でコントロールできます。

この方法は、以下のようなケースで使用します。

 

「アルファコンバイナ」では、ちぎって消す為の計算式が３種類用意されています。
以下に各式の特徴について説明します。

A （Comb_A － （1 － A0））× 2  NintendoWare for CTR のデータ互換の為に残している計算式です。
スケール値が 2 に固定されている為、コントラストを調整することができません。
B （Comb_A － A0） × A1 B と C は同じ式ですが、A1 のアニメーションを調整することでコントラストを微妙にコントロールできます。
C
D Clamp ( （Comb_A － A0） × 4 ) × A1

B と C に加え、A0 の値で白の範囲をコントロールできます。
