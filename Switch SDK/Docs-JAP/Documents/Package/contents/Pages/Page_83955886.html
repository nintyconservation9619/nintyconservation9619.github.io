概要

SoundProjectWaveExporter.exe (以下、SoundProjectWaveExporter) は、NintendoWare SoundMaker プロジェクトファイルに登録されている、すべてのサウンドに対して一括で波形ファイル出力するためのコマンドラインツールです。

ここでは、SoundProjectWaveExporter の使い方と機能について説明します。

SoundProjectWaveExporter の使い方

SoundProjectWaveExporter の実行方法

SoundProjectWaveExporter を使うには、コマンドラインから以下の形式で実行します。

SoundProjectWaveExporter.exe [オプション] <プロジェクトファイルパス>

例）

$ cd NintendoSDK/Tools/Audio/SoundMaker
$ ./SoundProjectWaveExporter.exe -o temp ../../SampleData/Sound/simple/simple.fspj

出力を行うと、指定したディレクトリに波形ファイルを出力します。

プロジェクトファイルパス

プロジェクトファイル (.fspj) のパスを指定します。

コマンドラインオプション

SoundProjectWaveExporter のオプションは、ハイフンから始まる文字または文字列によって指定します。

オプション 説明
-o, --output <output-folder> 出力フォルダを指定します。
-t, --time <time>

最大出力時間 (秒)を指定します。

[デフォルト値：30]
-r, --sampling-rate <rate>

（内部用機能のため使用禁止です。）

サンプルレート 32kHz の場合は "32" を

サンプルレート 48kHz の場合は "48" を指定します。

[デフォルト値: 32]
-m, --monaural モノラル出力します。
-s, --silent ログを出力しません。
--sound-type

波形出力するサウンドのタイプを限定します。

wave, sequence, stream が指定できます。

"," で区切って複数のタイプを指定できます。

(例) --sound-type wave,sequence
--split-sequence-sound-tracks シーケンスサウンドの各トラックを別ファイルに出力します。

 

SoundProjectWaveExporter の機能

SoundMaker では、サウンドを単数、または複数指定して波形ファイル出力をする機能がありますが、本ツールでは、SoundMaker プロジェクトファイルに登録されているすべてのサウンドに対して、一括で波形ファイルを出力します。

出力時の動作

  • ログ出力は、"標準出力" に出力されます。
  • -s オプションが指定されていても、エラーは出力されます。
  • -o オプションで指定したフォルダが存在しない場合は、指定した名前でフォルダを作成します。
  • SoundMaker で「アイテムを無効にする」設定をしている波形ファイルは、出力されません。
  • --sound-type オプションで波形出力するサウンドのタイプを限定したときは、指定のタイプのサウンドのみが出力されます。

シーケンスサウンドのトラック毎の波形化の動作

--split-sequence-sound-tracks オプションを指定すると、シーケンスサウンドはトラック毎に別の波形ファイルに出力されます。

  • 波形ファイルの名前には {sequence sound name}_01.wav のように 1 から始まるトラック番号が付加されます。
  • トラックが無音だった場合は、そのトラックの波形ファイルは削除されます。
