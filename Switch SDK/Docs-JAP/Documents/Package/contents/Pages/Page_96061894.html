フレネルアルファ

ここでは視線方向に対してポリゴン法線方向のアルファ値を変化させる機能「フレネルアルファ」について説明します。

概要

「フレネルアルファ」は、視線方向とパーティクルポリゴンの法線方向のなす角度を設定値として、ポリゴン面のアルファ値に変換する機能です。
法線方向に変化がある、パーティクル形状に「プリミティブ」を使用した場合に効果的です。

以下は「 設定項目」の概要です。

フレネルアルファ

フレネルアルファの処理方法を以下のプリセットから選択します。

利用しない フレネルアルファを利用しません。
頂点で処理 フレネルアルファを頂点シェーダで処理します。
フラグメントで処理 フレネルアルファをフラグメントシェーダでピクセル単位に処理します。
