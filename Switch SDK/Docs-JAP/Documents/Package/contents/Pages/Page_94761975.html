アルファコンバイナの計算式

ここではアルファコンバイナの計算式について説明します。

概要

アルファコンバイナの「計算式」は、アルファ要素を合成する際に適用する計算式を設定できる機能です。
表現の目的に合った計算式に変更することでエフェクトのクオリティを上げることができます。

以下はアルファコンバイナ式の入力値として扱うアルファ要素のイメージとその略語です。

Comb_A コンバイナ設定上段の アルファ値です。
ここでは登録したテクスチャのアルファ値をそのまま出力しています。
A0 カラータブで設定したアルファ 0 の A 値です。
ここでは説明の為に 0.0（黒）→ 1.0（白）→ 0.0（黒）でアニメーションさせています。
█
A1 カラータブで設定したアルファ 1 の A 値です。
ここでは固定値として A = 0.5 を設定しています。

アルファ計算式

上段の アルファ値（Comb_A）と、「カラー」タブで設定する アルファ値との合成方法を、用意された計算式から選択します。
以下は合成結果のイメージと計算式です。

Comb_A × A0

コンバイナ設定上段の 出力結果とアルファ 0   の出力結果を乗算した結果を出力します。
Comb_A × A0 × A1 コンバイナ設定上段の 出力結果と カラー 0 を乗算した値に アルファ   1 を 乗算 した結果をを出力します。
以下の計算式は、テクスチャアルファチャンネルの閾値をアニメーションさせることによりエフェクトを消す技法に使用します。
関連リンク ここで使用するエフェクトを消す技法については 「エフェクトを違和感なく消したい」の「C ちぎって消す」を参照してください。
（Comb_A － （1 － A0））× 2 コンバイナ設定上段の 出力結果からアルファ 0 の反転結果を引いた値を２倍して出力します。
 NintendoWare for CTR のデータ互換の為に残している計算式です。
（Comb_A － A0） × A1

コンバイナ設定上段の 出力結果からアルファ 0 の値を引き、アルファ 1 の値で倍率をかけて出力します。
アルファ 1 により上の式より微妙な調整が可能です。
Clamp ( （Comb_A － A0） × 4 ) × A1 コンバイナ設定上段の 出力結果からアルファ 0 の値を引いた値を４倍します。
出力された結果を 0 ～ 1 にの範囲内で切り取り、アルファ 1 の値で倍率をかけて出力します。。
