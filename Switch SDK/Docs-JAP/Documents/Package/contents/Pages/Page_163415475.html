コントローラーの準備

はじめに

ここでは、Joy-Con やSwitch Proコントローラーなどの NX 向けのコントローラーを利用する方法について案内します。

NX向けコントローラーを利用した開発を行いたい場合には、本案内に沿って環境を整えてください。

概要 説明 補足
コントローラーのファームウェアアップデート

コントローラー内部のファームウェアを更新します。

SDEV とコントローラーをBluetoothで接続しファームウェア更新を行います。

コントローラーを利用するためには、必ずファームウェアの更新を行ってください。

SDEV / EDEV 環境でコントローラーを接続する方法 SDEV /EDEV にコントローラーを接続するためには、事前にペアリング(本体へのコントローラーの登録処理) を行う必要があります。  
(Windows7 のみ) Bluetooth のドライバーソフトウェアを切り替える Windows7 のPCにコントローラーを接続する場合は、Bluetooth ドライバーソフトウェアを切り替える必要があります。 Windows8.1 / Windows10 では不要です
PC 環境でコントローラーをペアリングする方法 (デバイスの追加) PC 環境でコントローラーを接続するためには、Windows の機能を利用してペアリングします。
