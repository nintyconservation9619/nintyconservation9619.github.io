Audio ラウドネスメーターパネル

Audio が出力した波形から計算したラウドネス値を表示するパネルです。

画面説明


NintendoSDK 1.0.0 での画面です。


 項目 説明
A 表示・非表示

グラフやメーターに表示する項目を選択します。
B グラフ

平均ラウドネス値、ピーク値、RMS 値をグラフ表示します。


平均ラウドネス値またはピーク値のグラフを１つだけ表示すると、目標値がグラフの背景に色で表示されます。


True Peak をオンにすると、ピーク値のグラフには True Peak の計算値が表示されます。
C メーター

現在の時間における平均ラウドネス値や、各チャンネルのピーク値、RMS 値をメーター表示します。

平均ラウドネス値およびピーク値の目標値に対する上下が色で示されます。


数値表示の下段は現在の時間における値を表示し、上段はそのピークホールドを表示します。


チャンネルはピーク値と RMS 値のどちらを表示するかを選べます。

True Peak がオンのときはピーク値には True Peak の計算値が表示されます。
D 現在値 現在の時間における平均ラウドネス値を見やすい大きめのサイズで表示します。
平均ラウドネス値の目標値に対する上下が色で示されます。

nn::audio API で出力デバイスごとに音量を設定している場合は、TV への出力音量が考慮されます。

 最終的に出力される波形は、さらに後段でダウンミックスされる可能性がありますが、 Spy は、そのダウンミックス前の波形に対して計測を行います。

ラウドネス値の目標値については、NX アプリケーション向けの「サウンド制作ガイド」を参照ください。

操作方法

操作対象 操作 手順
表示・非表示 クリック

グラフとメーターに表示するアイテムを選択します。


メーターの Peak と RMS はどちらか一方だけが選択できます。2ch、6ch、off のいずれか一つが選択できます。
グラフ 左クリック 現在の時間を、指定位置に移動します。
右ドラッグ グラフ表示を横方向にスクロールします。
Ctrl + ホイール グラフ表示を横方向に拡大、縮小します。
