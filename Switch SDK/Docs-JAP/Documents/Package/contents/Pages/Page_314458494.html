透視投影に対応したテクスチャプロジェクションマッピングモードを追加

「マッピング方法」に、レイアウトのカメラがパースペクティブのさい、投影するテクスチャ画像はスクリーン空間で描画されるプリセットを追加しました！
レイアウトのカメラをパースペクティブ投影にしたさいに、ペインを X または Y 軸で回転させても貼られたテクスチャはスクリーン座標系で投影した表示にできます。

以下はレイアウトのカメラが「 パースペクティブ投影」のさい、「正射投影マッピング」と「透視投影マッピング」の比較画像です。

A 正射投影マッピング

回転によりペインにパースがかかる為、「 テクスチャ画像」が歪んで見えます。
B 透視投影マッピング

「 テクスチャ画像」はスクリーン空間で投影される為、パースによる歪みが発生しません。

「親子階層」ウインドウの操作改善

「 親子階層」ウインドウでの階層化操作を改善しました！
レイアウトウィンドウとの連携操作や階層化操作が改善されています。
以下は「親子階層」ウインドウ上での操作で改善されたショートカットキーの一覧です。

以下のキー操作はレイアウトウィンドウで即座に反映されるようになりました！
[ A ] レイアウトウィンドウを全体表示します。
[ F ] レイアウトウィンドウで選択したペインを全体表示します。
以下のキー操作により、ペイン階層化の作業がしやすくなりました！
[ P ] 選択したペインを最後に選択したペインの階層下に配置します。
[ Shift ] + [ P ] 選択したペインの階層関係を解消します。
[ G ] ヌルペインを作成し、選択したペインを階層下に配置します。
以下のキー操作により、複数ペインの表示 / 非表示がしやすくなりました！

[ Alt ] + 「表示 / 非表示アイコン」クリック

選択ペイン階層下のペインのみを表示し、その他すべてのペインを非表示にします。
再度同操作で直前の状態に戻ります。

サンプルの追加

「インダイレクト」機能と「しきい値によるアルファ補間」機能を利用したサンプルを追加しました。

\indirect\indirect.flyt

「インダイレクト」の表現サンプルです。

背景と「GAME CLEAR」 の文字をインダイレクトで歪ませています。

ローカライズなどで文字列を変更可能にする為、「GAMECLEAR」の部分は「キャプチャペイン」の機能を使用しています。

\AlphaThreshold\AlphaThreshold.flyt

「しきい値によるアルファ補間」の表現サンプルです。


「RESULT」 の文字列と罫線を「しきい値によるアルファ補間」で出現させています。
