ランダム

ここではパーティクルの動きを秩序のない挙動にできる「ランダム」について説明します。

概要

「ランダム」はパーティクルをランダムな方向に移動させるフィールドです。
「乱数の種」を利用して秩序のない挙動が作れます。  

ランダム

「 ランダム」は、パーティクルをランダムな方向に移動させる機能群です。

ランダム幅

パーティクルの速度に対して加算する値を設定する機能です。

ランダム適用間隔

ランダムを適用する間隔を設定する機能です。
