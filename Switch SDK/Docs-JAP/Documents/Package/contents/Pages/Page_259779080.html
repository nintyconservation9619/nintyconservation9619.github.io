サウンドプロジェクトを新規作成する

1. 概要

サウンドプロジェクトの新規作成例です。

SoundMaker - 基本編 の「サウンドプロジェクトの作成」も併せてご確認ください。

サウンドプロジェクトの新規作成方法は、登録するサウンドセットの作成方法により、大きく 3 つに分かれます。

  • 2.1 サウンドセットを新規に作成する
  • 2.2 既存のサウンドセットを指定する
  • 2.3 空のサウンドプロジェクト

2. 作成例

2.1 サウンドセットを新規に作成する

空のサウンドセットを登録したサウンドプロジェクトを作成します。

1. [ファイル]→[プロジェクトの新規作成] をクリックします。

2. プロジェクト名およびパスを設定し、「サウンドセットを新規に作成する」を選択して次へをクリックします。

3. 完了をクリックすると、空のサウンドセットが登録されたサウンドプロジェクトが作成されます。

2.2 既存のサウンドセットを指定する

作成済みサウンドセットを登録した、サウンドセットを作成します。

1. [ファイル]→[プロジェクトの新規作成] をクリックします。

2. プロジェクト名およびパスを設定し、「既存のサウンドセットを指定する」を選択して次へをクリックします。

3. 参照するサウンドセットのパスを指定し、次へをクリックします。

4. 完了をクリックすると、既存のサウンドセットを参照するサウンドプロジェクトが作成されます。

2.3 空のサウンドプロジェクト

参照するサウンドセットが無い、サウンドプロジェクトを作成します。

1. [ファイル]→[プロジェクトの新規作成] をクリックします。

2. プロジェクト名およびパスを設定し、「空のプロジェクト」を選択して次へをクリックします。

3 完了をクリックすると、サウンドセットを参照しないサウンドプロジェクトが作成されます。
