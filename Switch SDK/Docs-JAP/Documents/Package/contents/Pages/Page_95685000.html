HTC ライブラリ

はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている HTC (Host Target Communication) ライブラリについて説明します。

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象としています。

関連資料

nn::htc ライブラリの関数リファレンスと併せて参照してください。

HTC ライブラリの機能

HTC ライブラリはホスト PC とやり取りを行う各種 API を提供します。

切断エミュレーション

Target Manager と接続されていない状態でのアプリケーションの動作を検証するための機能として切断エミュレーション機能があります。
この機能が有効になっている場合、Target Manager と接続中であっても htc, htcs の各 API は常に切断中と同様のエラーを返すようになります。

切断エミュレーションの利用方法については 開発支援機能の設定の「HTC Disconnection Emulation Mode」を参照してください。

HTC ライブラリの制限

本機能は開発環境でのみ使用可能です。製品環境では利用できません。

また、HTC ライブラリは標準ではビルドタイプ Release でビルドするとコンパイルエラーになります。
ビルドタイプ Release で利用したい場合は NN_ENABLE_HTC を定義してください。

なお、Sample で HTC、HTCS ライブラリを使用しているものには、ソリューション構成 Release のプロパティシートで NN_ENABLE_HTC を定義していますので、注意してください。
