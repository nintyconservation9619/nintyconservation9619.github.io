カラーバッファ屈折

ここではテクスチャの RGB 値を利用して背景を屈折させる機能「カラーバッファ屈折」について説明します。

概要

「カラーバッファ屈折」は、「テクスチャ 0」タブに登録したテクスチャの RGB 値を利用して背景を屈折させる機能です。
例えば背景に岩があり、「屈折用テクスチャ」を登録したパーティクルを下方向に自由落下させることで滝のエフェクトが表現できます。
以下は「 設定項目」の概要です。

「シェーダタイプ」を「カラーバッファ屈折」にすると「 カラーバッファ屈折」と「 係数アニメ」が表示されます。
以下はグループ内の設定項目です。

係数

X 軸とY 軸方向に対する屈折の度合いを「 ダイヤル」や「 テキストボックス」により設定します。
またこの値は「係数アニメ」で歪み度合いをアニメーションできます。
関連リンク 詳細については「係数アニメ」を参照してください。

カメラが遠いほど強く歪む

パーティクルはカメラから離れるほど、屈折が大きく掛かる機能です。
関連リンク 詳細については 「カメラが遠いほど強く歪む」   を参照してください。

アルファ値を反映

屈折用テクスチャのアルファ画像と「カラー」タブで設定したアルファ値を屈折の値に乗算するかを「 チェックボックス」ON/OFF で切り替えます。
関連リンク 詳細については「アルファ値を反映」を参照してください。

設定例

以下は「カラーバッファ屈折」の値を変更したときの比較画像と設定内容です。

テクスチャ 0

R チャンネル側画像です。
X 方向（U 方向）に屈折させます。
黒は - 方向、白は + 方向に屈折します。

G チャンネル側画像です。
Y 方向（V 方向）に屈折させます。
黒は - 方向、白は + 方向に屈折します。
床の表示
「ブロック」
屈折させる対象画像です。
床の表示タイプを「ブロック」にしています。
アルファ値を反映
「 チェックボックス OFF」に設定されています。
係数アニメ

設定：5%=0.0 → 50,51%=1.0 → 95%=0.0
A X,Y = 0.01.,0

屈折テクスチャの X 方向（U 方向）に設定値を乗算し、屈折度合いを決めています。
更に「係数アニメ」を乗算して屈折をアニメーションさせています。
B X,Y = 0.,0.01 屈折テクスチャの Y 方向（V 方向）に設定値を乗算し、屈折度合いを決めています。
更に「係数アニメ」を乗算して屈折をアニメーションさせています。

テクスチャ 0 に登録した屈折用画像は、unorm_8_8 、Component Slection は RGGG で出力しています。
チャンネル RGGG のうち、「赤」の部分が横の屈折、「緑」の部分が縦の屈折として利用されます。
