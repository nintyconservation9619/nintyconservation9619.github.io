位置をエミッタに追従

ここではエミッタが移動した時、磁力の「位置」を追従させるかを決める機能「位置をエミッタに追従」について説明します。

概要

エミッタが移動した時、磁力の「位置」をエミッタに追従させるかを設定します。
エミッタの「パーティクル追従タイプ」を「エミッタに追従しない」に設定した場合に効果があります。

以下は「 設定項目」の概要です。

位置をエミッタに追従

エミッタが移動した時、磁力の「位置」を追従させるかを「 チェックボックス」ON/OFF で切り替えます。
以下はエミッタに追従を ON にした場合、「磁力」によりパーティクルの速度に加算される値を求める計算式です。

  • 速度に加算する値 : （ その時のエミッタ位置+ 磁力の位置 - その時のパーティクル位置 - その時のパーティクル速度 ） × 「磁力」

設定例

以下は「位置をエミッタに追従」機能を ON/OFF した場合の比較画像です。

「位置をエミッタに追従」が OFF の場合

各パーティクルを放出した位置に磁力が発生します。

「位置をエミッタに追従」が ON の場合

エミッタの位置に磁力が追従します。

各パーティクルは移動する磁力に引かれます。
