放出停止

ここではエミッタからパーティクルの放出を停止する機能「放出停止」について説明します。

概要

「放出停止」は、Fade コマンド実行時にエミッタからパーティクルの放出を停止できる機能です。
ゲームアプリのプログラマが、再生中のパーティクルの放出を強制的に止めることができます。
以下は「 設定項目」の概要です。

放出停止

「放出停止」の使用 / 未使用を「 チェックボックス」ON/OFF で切り替えます。

参考：

パーティクルが寿命無限の場合は、放出を停止した上で「αフェード」もしくは「スケールフェード」を ON に設定することを推奨します。
これにより既に放出されたパーティクルが残り続けることを防ぎます。
関連リンク 寿命無限については「寿命無限」を参照してください。
