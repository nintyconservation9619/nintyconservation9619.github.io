固定乱数値

ここではエフェクトの見た目を毎回同じにする為の乱数値を設定する機能「固定乱数値」について説明します。

概要

「固定乱数値」は、乱数値を直接指定できる機能です。
「乱数タイプ」設定を 「固定」に設定し、同じ乱数値を指定したエミッタ間で乱数要素のある機能の挙動を揃えることができ、エフェクトの見た目を毎回同じにできます。

「乱数タイプ」設定を 「固定」に設定すると「 固定」が表示されます。以下はグループ内の設定項目です。
以下は「 設定項目」の概要です。

固定乱数値

乱数値を設定します。この値が乱数列を発生させる「タネ」になります。
同じ乱数の「タネ」からは、エフェクトを呼び出す度に同じ乱数列が発生します。

設定例

以下は「乱数タイプ」設定を 「固定」に設定し、複数のエミッタで「固定乱数値」を指定したエフェクトの比較画像です。
「▲ 三角」と「■ 四角」の組合わせ、「▲ 三角」と「■ 四角」の組合わせで乱数値を変えて説明します。

▲■ 乱数値「1」 乱数値が「1」に設定した 2 つのエミッタが出す粒は、毎回同じ軌道を取ります。
▲■ 乱数値「10」 乱数値を「10」に設定した 2 つのエミッタが出す粒は、毎回同じ軌道を取ります。
