境界ペイン

ここではボタンなどのあたり判定として利用できる「境界ペイン」について説明します。

概要

「境界ペイン」は、ペインの当たり判定を指定できる機能です。
レイアウトに境界ペインを配置することで、プログラム側で当たり判定の範囲の判別に利用できます。

境界ペインは以下のように利用します。

  • カーソルがペインに乗った時の当たり判定

以下は「境界ペイン」の設定項目です。

ペイン基本

ペインの名前、位置、形状、中心位置などを設定します。

拡張ユーザ情報

ペインにユーザー独自の拡張情報を設定します。
