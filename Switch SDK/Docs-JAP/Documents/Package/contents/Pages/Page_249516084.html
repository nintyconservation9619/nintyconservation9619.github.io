アニメーション区間のモード切り替えでよくある質問

アニメーション区間のモード「 結合モード」と「 分割モード」でよくある質問です。

質問 回答
「結合モード」「分割モード」とは何ですか？

「結合モード」とは、ひとつのアニメーションを区間タグで区切って利用する従来の動作のことを指します。

「分割モード」とは、アニメーションと区間タグが 1:1 に紐付けされるようになるレイアウトエディタの新機能です。


「分割モード」に切り替えることで、区間タグ毎に別々のアニメーションを作成することができます。

「結合モード」と「分割モード」で混在はできますか？ 「結合モード」と「分割モード」は、レイアウトファイル毎に設定することができます。
「結合モード」と「分割モード」はどちらがおすすめですか？

以下の点から「分割モード」がお勧めです。

  • アニメーションが分割されて表示されるので、把握や管理がしやすい
  • 区間の長さが変わった際の調整がしやすい

まずは、以下のように部分的に「分割モード」を取り入れてみることをお勧めします。

  • 新規作成分だけ「分割モード」を使って作る
  • 一部のファイルだけ「分割モード」に変換してみる
「分割モード」で文字毎アニメーション SRT を区間タグ毎に設定できません。

「文字毎の変換」では SRT ( スケール、回転、移動 ) は、区間タグ毎にアニメーションカーブを設定できない特殊なパラメータです。

複数の区間タグで「文字毎アニメーション」を利用したい場合は、時間オフセットと時間幅を区間タグ毎に設定してください。

関連リンク 「文字毎の変換アニメーション」については こちら を参照してください。
「分割モード」で区間タグを切り替えるとペインの位置が移動します。

ペインにアニメーションが設定されている場合、区間タグを切り替えることで切り替え先のアニメーションが反映されます。

アニメーションが設定されていない場合は、ファイル読み込み時の初期状態に戻ります。
「結合モード」から「分割モード」に切り替えた際の質問と回答です。
レイアウトのアニメーションは作り直しになりますか？

「分割モード」に切り替えたレイアウトのアニメーションは、そのまま利用可能です。

「分割操作」および「結合操作」は、区間タグ範囲内のカーブの形状が保持されるように変換が行なわれます。
その為、区間タグ毎に出力したバイナリファイルは「分割モード」と「結合モード」で同一の結果が得られます。
アニメーションキーの追加ができなくなりました。

「分割モード」では、区間タグを選択した状態でアニメーションキーを追加することができます。
「結合モード」に戻したらアニメーションカーブの形状が変わりました。

一度「分割モード」に切り替えてしまったレイアウトファイルは、「結合モード」に戻してもカーブの形状が元に戻るとは限りません。

ただし、上述の通り区間タグ毎に出力したバイナリファイルは、同一の結果が得られるので実行結果に差異はありません。
複数の区間アニメーションを切り替えて再生した結果が「結合モード」の時と変わってしまいました。

「分割モード」はデフォルトで「区間タグのフレーム区間内にキーが無い場合は、アニメーションを出力しない」が ON に設定されています。
この為、両モードで再生時の挙動が変わってしまいます。
