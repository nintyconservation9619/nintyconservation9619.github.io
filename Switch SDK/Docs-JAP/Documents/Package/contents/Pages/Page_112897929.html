フェードイン時間

ここでは 放出されるパーティクルのフェードインに要する時間を設定する機能「フェードイン時間」について説明します。

概要

パーティクル放出時にここで設定した値をかけてパーティクルを出現させることができます。
「αフェード」や「スケールフェード」が適用された時に利用できる機能です。
以下は「 設定項目」の概要です。

フェードイン時間

Fadeコマンドが実行されてからの経過時間を設定します。

設定例

以下は「αフェード」ON で「フェードイン時間」の設定値を変えた場合の比較画像です。

A 0 Fade コマンドが実行された時点でパーティクルは出現します。
B 100 100フレームをかけ、アルファフェードとスケールフェードで徐々に出現します。
