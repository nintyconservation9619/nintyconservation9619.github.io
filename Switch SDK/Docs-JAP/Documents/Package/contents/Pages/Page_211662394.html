レイアウトの自動再転送機能

コマンド等の実行毎に、自動でビューアに再転送ができるようになりました !!

実行する毎に再転送されるので、ユーザーは結果をダイレクトに確認することができます。
自動再転送の ON / OFF は、切り替えることができます。
( 転送エラーが発生した場合は機能はオフになります )

レイアウトの自動再転送機能
A プラットフォーム切り替えコンボボックス 使用するプラットフォームに切り替えて使用します。
B 自動再転送切り替えアイコン

自動転送を切り替えるアイコンです。

アイコンをマウスの左クリックすることで切り替わります。
レイアウトの編集時にビューアに転送を行いません。
レイアウトの編集毎にビューアに転送します。
C 接続先切り替えコンボボックス 接続先をコンボボックスから指定します。
D ビューア転送のアニメーション再生切り替えアイコン

ビューアに転送する時のアニメーション再生 ON / OFF 切り替えのアイコンです。

アイコンをマウスの左クリックすることで切り替わります。
ビューアでアニメーションを再生しません ( ボタンの挙動の確認ができます。)
ビューアでアニメーションを再生します。

レイアウト環境設定ウインドウの統合

レイアウト環境設定ウインドウが統合されました !!

二箇所にありました「レイアウト環境設定ウインドウ」ですが、統合されて一箇所になります !!
設定画面を統合することにより、メニューがシンプルになり、環境設定が使いやすくなりました。

レイアウト環境設定ウインドウは、
「レイアウトエディタウインドウ / ウインドウ / レイアウト設定 / 環境設定」に統合されます。
「レイアウトエディタウインドウ / ファイル / レイアウト環境設定ウインドウ」は廃止されました。

「レイアウトエディタウインドウ / ウインドウ / レイアウト設定 / 環境設定」に統合
「レイアウトエディタウインドウ / ウインドウ / レイアウト設定 / 環境設定」に統合されました。
「レイアウトエディタウインドウ / ファイル / レイアウト環境設定ウインドウ」は廃止

「レイアウトエディタウインドウ / ファイル / レイアウト環境設定ウインドウ」は廃止されて、表示されなくなりました。

ステージ式の前後入れ替え

「詳細なコンバイナモード」でステージ式の前後入れ替えが出来るようになりました !!

LayoutEditor のプロパティウィンドウ、テクスチャコンバイナ「詳細なコンバイナモード」にステージ式を入れ替えるボタンを追加しました。

詳細なコンバイナモードに追加された、ステージ式の前後入れ替えボタン
A ステージ式の前後入れ替えボタン

ステージ式を、前後のステージ式と入れ替えるボタンです。

B の「ステージ選択」で選択状態（ウインドウに表示状態）にしたステージを、前後のステージ式と入れ替えます。
B ステージ選択

ウインドウに表示するステージ式を選択します。
