 nn::ui2d ニュース

degamma-paramater 指定時に、フォントの影カラーなどが正しくガンマ変換処理されない不具合の修正

LayoutConverter で、degamma-paramater オプションを指定している場合に、フォントの影カラーや、文字毎の変換のカラーが正しくガンマ変換処理されない不具合があり、これを修正しました。

不具合に合わせて、正しく表示されるようにレイアウトデータ側で調整している場合、不具合が修正されることによって逆に表示が正しく行われなくなる可能性がありますのでご注意ください。（degamma-paramater は、移植作品など、過去に SRGB ガンマワークフロー前提で作成されたレイアウトデータをそのまま利用する用途で指定されるオプションです。）
