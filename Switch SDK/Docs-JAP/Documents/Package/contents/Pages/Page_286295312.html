アルバムアプレット

本体機能のアルバムを呼び出すライブラリアプレットです。

アルバムの詳細については、「本体機能ガイド」の「システムアプリ＞撮影機能・アルバム」ページを参照してください。

提供する機能

提供している機能と API の対応は、次の表の通りです。

機能 対応するAPI 備考
アルバム内のファイルの一覧表示 nn::album::ShowAlbumFiles

呼び出し元のアプリに紐づくファイルのみ表示対象となります。

ヘッダファイルのインクルード

プログラム中に下記のコードを記述してください。

#include <nn/album/album_Applet.h>

アルバムアプレットの起動

アルバムアプレットの機能は対応した API を呼び出すことで利用できます。
アルバムアプレットを起動する API は、アプレットが終了するまで処理をブロックします。
