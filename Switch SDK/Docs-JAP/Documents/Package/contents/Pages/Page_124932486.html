FontConverter(GUI版)の概要

FontConverter(GUI版)の概要について説明します。
フォントコンバータは、フォントライブラリが処理する、ビットマップフォントバイナリ (.bffnt) を作成します。

FontConverterでできること

  • WindowsフォントからBMP, TGA, TIFF形式の画像ファイル生成
  • BMP, TGA, TIFF形式の画像からフォントバイナリを生成
  • Windowsフォントからフォントバイナリを生成
  • 必要な文字だけを抽出したフォントバイナリを作るフィルタ

使用上の注意

FontConverter 使用にあたっての注意事項について説明します。

フォントのライセンスについて

FontConverter には、いかなるフォントのライセンスも付属しません。
フォントを利用したソフトを発売するには別途フォントメーカーとのライセンス契約が必要になります。

自動フォントリンクの無効化

Windows には自動フォントリンクという機能があります。
これは自動的に他のフォントを使って、フォントに含まれていない文字を表示する機能です。
ライセンスされていないフォントを無自覚に商品に使用してしまう危険性を考慮して、フォントコンバータでは自動フォントリンク機能を無効化しています。

内部でのUnicode変換

エンコーディングの変換にはWindowsの変換ルールにもとづいて変換が行われます。
例えばShift JISで文字コード0x8160の「～」は、本来のUnicodeの定義ではU+301Cとなりますが、Windows(=FontConverter)ではU+FF5Eとなります。
