Atk ストリームサウンドパネル

Atk で再生中のストリームサウンドに関する情報を表示するパネルです。

画面説明


 項目 説明
A サウンド名 再生中のストリームサウンドのサウンド名です。
B 充填率 ストリームサウンドの充填率です。
C 遅延状態 ストリームサウンドの再生がロード遅延により一時停止している場合、「Delayed」の文字が表示されます。
D 充填率のグラフ表示 ストリームサウンドの充填率を、棒グラフ表記したものです。
E ファイルリードキャッシュの残量

ストリームサウンドのファイルリードキャッシュ機能使用時に、キャッシュの残量が表示されます。

キャッシュ表示には nn::spy::atk::AtkSpyModule::PushAtkProfiles() を呼ぶ際に nn::atk::SoundArchivePlayer を引数として渡す必要があります。
