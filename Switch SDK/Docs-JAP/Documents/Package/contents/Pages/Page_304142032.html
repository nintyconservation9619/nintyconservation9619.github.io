派生元への関連付けを修正

部品レイアウト一覧の右クリックメニューで「派生元への関連付けを修正」を選ぶと、以下のダイアログウィンドウが表示されます。

派生元への関連付けを修正　ダイアログ

A 現在の名前

現在の派生元になっている部品レイアウトファイル名を表示します。
B 新しい名前

新しく派生元にしたい部品レイアウトファイルをプルダウンメニューから指定します。

または、派生関係を消して通常の部品にします。
C 編集対象

編集対象となる派生部品レイアウトファイルをチェックボックスで指定します。

編集対象には、部品フォルダ内で「現在の派生元」から派生しているすべての部品レイアウトファイルが表示されます。
