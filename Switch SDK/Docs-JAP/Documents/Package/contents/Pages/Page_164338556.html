テクスチャ

テクスチャ

LayoutEditor のペインの中の「テクスチャ設定」をします。

追加/削除 テクスチャーの「追加/削除」をします。

設定

テクスチャーの設定をします。

 

追加/削除

テクスチャーの「追加/削除」をします。

登録したテクスチャを順送りで切り替えます。

追加...

テクスチャウインドウに登録したテクスチャに中から、テクスチャを追加します。

削除...

登録したテクスチャを削除します。
変更... 登録したテクスチャを変更し、違うテクスチャに入れ替えます。

設定

テクスチャーの設定をします。

カラーイメージ

選択しているテクスチャのカラーイメージを表示します。

アルファイメージ

選択しているテクスチャのアルファイメージを表示します。

テキストボックス

選択しているテクスチャのファイル名を表示します。

フォーマット :

選択しているテクスチャのフォーマットを表示します。
サイズ : 選択しているテクスチャのサイズを表示します。
イメージ設定

「繰り返し方」「フィルタリング」の設定をします。（タブを切り替えて表示）


関連リンク 詳細については「イメージ設定 」を参照してください。
マッピング設定

マッピングの設定をします。（タブを切り替えて表示）


関連リンク 詳細については「マッピング設定 」を参照してください。
テクスチャパターン

テクスチャパターンリストを設定します。（タブを切り替えて表示）


関連リンク 詳細については「テクスチャパターン 」を参照してください。
参考：

各タブの設定項目はリンク先で詳しく解説します。
