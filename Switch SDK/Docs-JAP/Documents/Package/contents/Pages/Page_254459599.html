使用パターンテーブル数

ここではアニメーションで使用するパターン数を設定する機能「使用パターンテーブル数」について説明します。

概要

「使用パターンテーブル数」は、アニメーションで使用するパターン数を設定できる機能です。
以下は「 設定項目」の概要です。

使用パターンテーブル数

アニメーションで使用するパターン数を「テーブル」の範囲で「 ダイヤル」や「 テキストボックス」により設定します。
設定内容は「パターンテーブル」の UI に表示されます。

関連リンク 以下は関連ページへのリンクです。

パターンテーブル 「テクスチャパターンアニメ」で使用するパターンを指定し、パターンをアニメーションさせる順番が変更できる UI 「パターンテーブル」について説明します。
テーブル簡易入力 「テクスチャパターンアニメ」で使用するパターンを指定し、パターンをアニメーションさせる順番を設定する機能「テーブル簡易入力」について説明します。

設定例

以下は再生方法「ライフフィット」で「使用パターンテーブル数」の値を変更したときの比較画像です。

A

使用パターンテーブル数 = 16
