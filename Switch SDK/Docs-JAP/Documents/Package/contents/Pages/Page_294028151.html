マルチスケーラブルフォントプロパティウィンドウ

以下は「マルチスケーラブルフォントプロパティウィンドウ」の UI 説明です。

A フォントサイズ フォントサイズを指定します。
B 代替文字 代替文字を指定します。
C 行間オフセット

行間オフセットを指定します。
D スケーラブルフォント設定順序入れ替えボタン スケーラブルフォント設定一覧で選択しているアイテムの位置を変更します。
E スケーラブルフォント設定一覧 子ノードのスケーラブルフォント設定の一覧を表示します。
上に表示されるほど優先度が高くなります。
