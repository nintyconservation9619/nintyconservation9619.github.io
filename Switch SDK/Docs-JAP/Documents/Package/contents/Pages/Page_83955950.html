炎を作る

ここでは一つのエミッタで以下の様な炎のエフェクトを作成する手順について説明します。

このチュートリアルではアニメーションを設定する以下 UI の使用例について触れています。

グラデーションエディタ

カラーやアルファをアニメーションさせるUIです。
カーブエディタ パーティクルのスケールやフィールドなどの値をアニメーションさせるUIです。

完成データは以下の場所にあります。

\Samples\Sources\Resources\Vfx\Eset\Fire_04.eset
タブが「 オレンジのタブ」やラベル背景が「 オレンジの設定項目」のようにように変わっている箇所はデフォルトから修正が加えられています。
サンプルのどこが設定されているかを把握する時に便利です。

エフェクト作成の準備

あらかじめ EffectMaker と EffectViewer を起動した状態から、エフェクト作成にあたっての準備を行います。

関連リンク EffectMaker と EffectViewer の起動方法については 「作業の流れ」 を参照してください。

「カメラ」タブを選択する

ワークスペース上で「 ビューアノード」を選択します。
プロパティウィンドウから「カメラ」タブを選択します。
ビューアのカメラに関する設定項目が表示されます。

関連リンク 以下は関連ページへのリンクです。

ワークスペース

エフェクトを構成する各ノードを階層構造で表示する「ワークスペース」について説明します。
プロパティウィンドウ ノードの操作やプレビューができる「ワークスペース」とノードの機能設定ができる UI から構成される「プロパティウィンドウ」について説明します。

ビューアのカメラを設定する

エフェクトが確認しやすい状態にする為、「 ビュー・投影」の以下の「 項目」を設定します。

項目名 値 説明
カメラの位置 X,Y,Z = 0,4,10 エフェクトがビューア画面に収まる様にカメラの位置と注視点を調整します。
注視点 X,Y,Z = 0,2,0

エミッタセットの作成

以下の手順で新規にエミッタセットを作成します。

1
メインメニュー項目 [ファイル]→[新規作成]→[ エミッタセット] を選択します。
2
ダイアログが表示されますので、エミッタセット名を「Fire」と入力し「OK」ボタンを押します。
ワークスペースでエミッタセット「 Fire」が作成されます。
エミッタセット「 Fire」 直下にはエミッタ「 Emitter1」が配置されています。

ビューアを確認する

新規で作成したエフェクトはビューアで以下の様に表示されていることを確認します。
以降はビューアで変化を確認しながらエフェクトを編集します。

エミッタ名を変更する

ワークスペースでエミッタ「 Emitter1」を選択します。
以下の手順でエミッタノードの名前を変更します。

1
メインメニュー項目［ファイル］→［ 名前を変更］を選択します。
2
名前の変更が可能になりますので、エミッタ名を「Main」と入力します。
エミッタセット「 Fire」 直下のエミッタ「 Emitter1」は「 Main」に変更されます。

以降はエミッタ「 Main」を選択した状態で説明します。

放出と初速の設定

パーティクルの量と動きを調整し、炎の動きを作成します。

「放出」タブを選択する

プロパティウィンドウの「 タブ」から「放出」を選択します。
パーティクルがエミッタから放出するタイミングや初速の設定項目が表示されます。

「放出」タブを設定する

「 放出設定」の以下の項目を設定します。

項目名 値 説明
放出間隔 1 パーティクルは、エミッタから毎フレーム放出されます。

「 その他運動設定」の以下の項目を設定します。

項目名 値 説明
空気抵抗 0.95 エミッタから全方向に放出されたパーティクルは失速し、やがて停止します。
重力 1 パーティクルの速度に重力をかけ、勢いよく立ち昇る炎にします。
方向 Y = 1.0 重力のかかる方向を上方向に変更します。
上昇気流により立ち昇る炎を表現できます。

関連リンク 値の編集方法については 「エフェクトデータの編集」 を参照してください。

パーティクルの設定

放出されたパーティクルが消えるタイミングを設定します。

「パーティクル」タブを選択する

プロパティウィンドウの「 タブ」から「パーティクル」を選択します。エミッタから放出されるパーティクルの設定項目が表示されます。

「パーティクル」タブを設定する

「 パーティクル寿命」の以下の「 項目」を設定します。

項目名 値 説明
寿命 50 パーティクルはエミッタから放出されてから 50 フレーム目に消滅します。
寿命ランダム 30（%） 「寿命」で指定した値を上限とし、下限側は以下の式で計算されます。
50 - （50 × 30%） = 35
パーティクルの寿命は35 フレームから 50 フレームの間でランダムに変化します。

以下はこの段階でのビューアの見え方です。
炎の基本的な動きが作成できました。

テクスチャとカラーの設定

パーティクルのテクスチャとカラーを調整して炎らしくします。
パーティクルのテクスチャとして炎の形と色の画像ともうひとつ、炎の熱による発光度合いを調整する為の画像を使用します。

「テクスチャ0」タブを選択する

プロパティウィンドウの「 タブ」から「テクスチャ」タブを選択します。
「 サブタブ」は「テクスチャ0」が選択されていることを確認します。

「テクスチャ0」 タブを設定する。

「 テクスチャ」の以下の「 項目」でテクスチャファイルを登録します。

項目名 設定パス テクスチャ 説明

ファイル指定

\Samples\Sources\Resources\Vfx
\Eset\Textures\Fire_00.ftxb

「 ロードボタン」を押すことで開くウィンドウからテクスチャファイルを選択します。
炎の色とシルエットのテクスチャです。

正常に登録された場合、「 テキストボックス」にファイル名が表示されます。

「テクスチャ1」タブを選択する

「 サブタブ」から「テクスチャ1」を選択します。

「テクスチャ1」 タブを設定する

「 テクスチャ」の以下の「 項目」でテクスチャファイルを登録します。

項目名 設定パス テクスチャ 説明
ファイル指定
\Samples\Sources\Resources\Vfx
\Eset \Textures\Fire_01.ftxb

「 ロードボタン」を押すことで開くウィンドウからテクスチャファイルを選択します。

テクスチャ0 に対して乗算するテクスチャです。

正常に登録された場合、「 テキストボックス」にファイル名が表示されます。

「 テクスチャ座標アニメ」の以下の「 項目」で「テクスチャ1」の UVスクロールアニメを設定します。

項目名 値 説明
UVスクロールアニメ ON 「UVスクロールアニメ」が設定可能になりますので、以下の様に設定します。
加算値 X,Y = 0.001,-0.001 テクスチャは右上方向に向かってスクロールアニメーションします。
初期値ランダム X,Y = 0.5 放出時のスクロールアニメーションの開始位置をランダムに変化させることで、自然に見える様にします。

「 テクスチャ座標アニメ」の以下の「 項目」で「テクスチャ1」の UVスケールアニメを設定します。

項目名 値 説明
UVスケールアニメ ON 「UVスケールアニメ」が設定可能になりますので、以下の様に設定します。
加算値 X,Y = 0.005 テクスチャのディテールが徐々に細かくなる様にアニメーションします。
初期値 X,Y = 0.5 放出時のテクスチャ適用範囲を設定します。初期値はディテールを粗めにしています。

スケールと回転の設定

勢いよく燃え上がり、やがて消滅する炎の様子をパーティクルスケールのアニメーションで表現します。
また、パーティクル個々にほんの少し回転アニメーションを加えることで、上昇する炎の動きにゆらぎを与えます。

「スケール」タブを選択する

プロパティウィンドウ「 タブ」から「スケール」を選択します。
パーティクルサイズやスケールアニメーションに関する設定項目が表示されます。

パーティクルサイズの基準値を設定する

「 基本スケール」の以下の「 項目」で放出時のパーティクルサイズを設定します。
ここで設定した値はパーティクルにスケールアニメーションをかける際の基準値となります。

項目名 値 説明
基準値 X,Y,Z = 5 放出時のパーティクルの基本サイズを設定します。
ランダム量 X,Y,Z = 30%

「基準値」に対してランダム量を設定し、パーティクルが一定サイズで放出されることを防ぎます。
この設定では、パーティクルサイズは「基準値」の 70％ から基準値の間でランダムに変化します。

パーティクルスケールをアニメーションさせる

「 スケール（パーティクル時間アニメ）」の以下の「 項目」でパーティクルサイズをアニメーションさせます。

関連リンク アニメーションを設定する UI の操作方法については 「カーブエディタの操作」 を参照してください。

項目名 時間（f） 時間（%） 値 説明
スケールアニメーション 0 0 1.0

放出時は値を1.0に設定します。「基準値」がそのまま適用されます。
25 50 1.0

寿命の半分までは値を1.0に設定します。放出時と同サイズを継続します。
50 100 0.5

最後のフレームではスケール値を 0.5 に設定します。
透明度のアニメーションと合わせることで、燃焼エネルギーの減少を印象付けられます。

「回転」タブを選択する

プロパティウィンドウ「 タブ」から「回転」を選択します。
パーティクルの傾きや回転アニメーションに関する設定項目が表示されます。

パーティクルを回転アニメーションさせる

「 回転設定」の以下の「 項目」でパーティクルの Z 軸を中心に回転アニメーションを設定します。

項目名 値 説明
回転軸
 		Z軸 ON パーティクルはZ軸での回転のみ設定可能になりますので、以下の様に設定します。
回転方向ランダム ON 放出時の回転方向が時計回り、反時計回りでランダムに変化します。
回転速度(±)
4

パーティクルの回転アニメーション「回転速度(±)」の値に対してランダム範囲を指定します。

パーティクルは、毎フレーム 0 ～ 4 度の範囲でランダムな値が加算されることにより回転アニメーションします。
複数のパーティクルが重なることにより、炎がメラメラと揺らめいている様に見えます。
