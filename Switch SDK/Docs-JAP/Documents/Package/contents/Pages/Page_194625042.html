データ配信

データ配信とは

データ配信機能とは、任天堂が管理しているサーバーに登録されたデータを複数の本体に一斉に配信する機能です。配信はサーバーのデータを本体に同期する形式で行われます。


サーバーには一階層の深さを持つ複数のディレクトリを登録できます。ディレクトリの中には複数のファイルを登録できます。ファイルごとに配信の開始日時、および終了日時を予約設定できます。

同期の際は、ディレクトリとファイルのツリー構造がそのままの形で本体にコピーされます。ディレクトリやファイルがサーバーから削除されると、本体からも削除されます。

本ドキュメントではディレクトリとファイルをまとめてデータと呼んでいます。


アプリケーションが起動しているとき、任意のタイミングで同期を行うことができます。また、アプリケーションが起動していないときは、本体システムが自動的に同期を行います。
