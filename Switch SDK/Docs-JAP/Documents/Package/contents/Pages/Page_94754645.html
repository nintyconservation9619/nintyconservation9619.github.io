パーティクルオフセット

ここではゲーム内のオブジェクトとパーティクルの重なりを調整する機能「パーティクルオフセット」について説明します。

概要

「パーティクルオフセット」は、パーティクルの深度情報やカメラ方向での位置にオフセットできる機能です。
エミッタ単位で設定することで、パーティクルの重なりを調整できます。
以下は「 設定項目」の概要です。

パーティクルオフセット

パーティクルのオフセット方法を、以下のプリセットから選択します。

Z値をオフセット パーティクルの深度値を操作します。
「カメラ方向にオフセット」の様なパースによるパーティクルサイズの変化はありません。
カメラ方向にオフセット パーティクルの描画位置を、カメラに対して前後にオフセットさせます。
サイズ固定カメラオフセット パーティクルの描画位置を、カメラに対して前後にオフセットさせるとき、パースを打ち消す処理を入れます。
利用しない この機能を利用しない為、負荷がかかりません。

設定例

以下は「F の文字」を貼った立方体パーティクルに、各プリセットを設定したときの比較画像です。
「F の文字」パーティクルと重なるように、背景にあるチェック模様の板状パーティクルを前後に移動させています。

A Z値をオフセット 「オフセット値」の設定により、深度値に 20 を設定しています。
背景のパーティクルとは重ならず、且つパーティクルのサイズに変化がありません。
B カメラ方向にオフセット

「オフセット値」の設定によりワールド座標系で Z方向に 20 移動しています。

背景のパーティクルとは重なりませんが、パースがかかる為、パーティクルのサイズに変化があります。
C サイズ固定カメラオフセット 「オフセット値」の設定によりワールド座標系で Z方向に 20 移動しています。
スクリーン座標系で頂点単位に調整することで、パースを打ち消す処理が入る為、サイズに変化がありません。
D 利用しない 背景のパーティクルと重なります。
