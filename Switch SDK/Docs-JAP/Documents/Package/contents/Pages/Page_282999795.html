キャプチャペイン

ここではフレームバッファやレイアウトのレンダリング結果をキャプチャし、画像に変換する「キャプチャペイン」について説明します。

概要

「キャプチャペイン」は、フレームバッファやレイアウトのレンダリング結果をキャプチャして テクスチャとして利用できるペインです。
「親子階層」ウィンドウで「 キャプチャペイン」階層下のペインは通常のペイン描画は行われません。

キャプチャペインが生成したテクスチャ画像は「テクスチャ」ウィンドウに「█ 黄色帯」で通常テクスチャと区別して表示されます。
このテクスチャは、通常のテクスチャ同様、ペインの「プロパティウィンドウ」→「 テクスチャ」に登録して利用できます。
キャプチャペインでは主に以下の設定ができます。

  • テクスチャとしてキャプチャする範囲
  • フレームバッファをキャプチャするか、初回フレームのみをキャプチャするかの設定
  • ヌキ部分のカラー値やアルファ値を指定するクリアカラー

以下は「キャプチャペイン」の設定項目です。

ペイン基本

ペインの名前、位置、形状、中心位置などを設定します。

使用例

以下は「キャプチャペイン」の使用例です。

キャプチャペインの設定 キャプチャペインで生成されたテクスチャ画像を使った表現

黄色い枠がキャプチャペインの範囲です。
「 キャプチャペイン」の子階層には以下のペインを配置しています。

「 ピクチャペイン」「 テキストボックスペイン」

「 キャプチャペイン」のキャプチャ範囲でテクスチャ画像が生成されます。

テクスチャ画像は左右に分割した 2 つの「 ピクチャペイン」で利用します。
左右からマスクテクスチャをアニメーションしてテキストを消しています

制限事項

キャプチャペインの制限事項です。

  • 親子関係に関する制限
    • キャプチャテクスチャに依存関係が発生するような構造の親子関係は作成できません。
      例えばキャプチャペインの子供にキャプチャペインで作成したテクスチャを使用しているピクチャペインを配置するような構造です。
  • 回転、スケールに関する制限
    • キャプチャテクスチャを作成するペインに回転やスケールが設定されているとペインの形状に沿ってキャプチャすることができません。
