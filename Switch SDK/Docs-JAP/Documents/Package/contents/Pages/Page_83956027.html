サンプラーアトリビュート設定プラグイン

file ノードに NintendoWare 用のサンプラーのアトリビュートを設定します。ただし、Maya の画面上には反映されません。

設定の検索もできます。

使用方法

「Nintendo > Set Sampler Attribute...」を選択するとウィンドウが表示されます。

Selected file Node 欄には、選択されている file ノードが表示されます。

参考：

4 つ以上の file ノードが選択されている場合は省略表示となり、右端に選択された数が表示されます。

設定方法

file ノードを選択（複数選択可）すると、Set 欄に選択したノードの現在の設定が表示されます。
複数選択の場合、最初に選択したノードの設定が表示されます。

ウィンドウの各項目を操作してアトリビュートを変更します。
複数選択の場合、選択したノードを一括して変更できます（変更したアトリビュートだけ反映されます）。

ウィンドウを閉じずに別のノードを選択すると、別のノードを設定することができます。

Name サンプラーの名前を入力します。
空欄にすると file ノードの接続先に応じた名前で出力されます（詳細は下記）。デフォルトは空欄です。
コンフィグファイル で sampler_merge_priority="true" としている場合、「_（アンダーバー）」から始まる名前のみ設定できます（アンダーバーから始まらない名前を入力するとエラーになります）。
Hint

サンプラーのヒント情報を入力します。
サンプラーの名前を入力している場合に入力できます。
空欄にすると file ノードの接続先に応じたヒント情報で出力されます（詳細は下記）。デフォルトは空欄です。
参考：

同じマテリアルで使用しているサンプラーは、名前やヒント情報が重複してはいけません。

参考：

サンプラーのマージの詳細は、 中間ファイルオプティマイザーのヘルプ を参照してください。

参考：

envCube ノード（環境立方体）の各面に file ノードが接続されている場合は、 前面 の file ノードを選択して、サンプラーのアトリビュートを設定してください。
envCube ノードの右面に接続された file ノードでキューブマップの DDS ファイルを使用している場合は、 右面 の file ノードを選択して、サンプラーのアトリビュートを設定してください。

参考：

マテリアルの半透明の深度（Translucence Depth）アトリビュートに接続されたテクスチャーはデフォルトでは出力されませんが、サンプラーの名前を設定していれば出力されます。
半透明の深度アトリビュートに接続されたテクスチャーで Hint が空欄の場合、Name に入力した値がヒント情報にも出力されます。
半透明の深度アトリビュートに接続されたテクスチャーの中間ファイルの <sampler_array> での順番は最後になります（UV セットも使用されている UV セットの中で最後に出力されます）。

検索方法

Search 欄の Name に文字列を入力して、右の Search ボタンをクリックすると、サンプラーの名前に入力した文字列が含まれる file ノードがすべて選択されます。
「*（アスタリスク）」を入力して検索すると、サンプラーの名前が空欄でない file ノードがすべて選択されます。
何も入力しないで検索すると、サンプラーの名前が空欄になっている file ノードがすべて選択されます。

Hint の検索についても同様です。

デフォルトの名前とヒント情報

サンプラーの名前が空欄の場合、file ノードの接続先に応じて次のような名前で出力されます。
独自の名前を設定する場合、デフォルトの名前との重複に注意してください。
ヒント情報についても同様です。

接続先のアトリビュート デフォルトの名前 デフォルトのヒント情報
カラー（Color） _a0, _a1, ... albedo0, albedo1, ...
透明度（Transparency） _o0, _o1, ... opacity0, opacity1, ...
白熱光（Incandescence） _e0, _e1, ... emission0, emission1, ...
スペキュラカラー（Specular Color） _s0, _s1, ... specular0, specular1, ...
反射カラー（Reflected Color） _r0, _r1, ... reflection0, reflection1, ...
バンプマッピング（Bump Mapping） _n0, _n1, ... normal0, normal1, ...
