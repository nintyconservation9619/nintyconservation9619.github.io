アルファ値を反映

ここでは歪み用テクスチャのアルファ画像と「カラー」タブで設定したアルファ値を「屈折」の値に乗算する機能「アルファ値を反映」について説明します。

概要

「アルファ値を反映」は、以下のアルファ要素を「カラーバッファ屈折」の係数に乗算するかを決める機能です。

  • カラータブのアルファ
  • テクスチャ0～2に登録した画像のアルファ


例えば背景に岩があり、「屈折用テクスチャ」を登録したパーティクルを下方向に自由落下させることで滝のエフェクトが表現できます。
以下は「 設定項目」の概要です。

アルファ値を反映

屈折用テクスチャのアルファ画像と「カラー」タブで設定したアルファ値を屈折の値に乗算するかを「 チェックボックス」ON/OFF で切り替えます。

設定例

以下は「アルファ値を反映」を ON/OFF したときの比較画像です。

テクスチャ 0

R チャンネル側画像です。
X 方向（U 方向）に屈折させます。
黒は - 方向、白は + 方向に屈折します。

G チャンネル側画像です。
Y 方向（V 方向）に屈折させます。
黒は - 方向、白は + 方向に屈折します。
床の表示
「ブロック」
 屈折させる対象画像です。
床の表示タイプを「ブロック」にしています。
係数 X = 0.01
Y = 0.01 		屈折テクスチャの縦横両方向に設定値を乗算しています。
係数アニメ
設定：5%=0.0 → 50,51%=1.0 → 95%=0.0
ON

テクスチャ0 に登録したテクスチャ画像の A チャンネルの値 0 が屈折に乗算されています。
OFF テクスチャ0 に登録したテクスチャ画像の A チャンネルの影響は受けません。
参考：

テクスチャ 0 に登録した屈折用画像は、unorm_8_8 、Component Slection は RGGG で出力しています。
チャンネル RGGG のうち、「赤」の部分が横の屈折、「緑」の部分が縦の屈折として利用されます。
