ドキュメントガイド

ドキュメントガイドについて

ドキュメントガイドでは、 NintendoSDK 全体のドキュメント構成について紹介します。

また、各ドキュメントの読み進め方を説明します。

ドキュメントの構成

パッケージ全体に関するドキュメントの一覧

ドキュメント名 概要
NintendoSDK オーバービュー 
 NintendoSDK について説明します。
ドキュメントガイド（本ドキュメント） ドキュメントの構成を説明します。
環境構築ガイド
開発環境のセットアップ手順について説明します。
チュートリアル

NintendoSDK を用いた開発に関する様々な項目を説明します。

ざっとサンプルを動作させたい場合は、 クイックスタート もご確認ください。
機能 NintendoSDK が提供する機能の詳細について説明します。
APIリファレンス

各種 API を説明します。

このリファレンスに記載のないAPIは使用しないでください。
SDK サンプル解説書 SDK のサンプルプログラムについて解説します。
注意事項

アプリケーションの開発にあたって注意あるいは配慮すべき情報です。

事前に内容を確認してください。
既知の不具合 将来のバージョンで修正を予定している不具合情報です。
トラブルシューティング トラブルに見舞われた際の回避方法を説明します。
トラブルシューティングに書かれた項目は、基本的には仕様となるため、
既知の不具合 とは異なり修正予定はありません。
変更履歴 SDKのこれまでの変更内容についてまとめています。

機能に関するドキュメントの構成

機能ごとに提供されるドキュメントの基本的な構成は下記の通りです。

ドキュメント名 概要
オーバービュー

概要を説明します。

開発者向けに、アプリケーションの設計や実装に関する内容も含みます。
マニュアル 機能を使ったアプリケーションの設計や実装方法を説明します。
サンプル サンプルの動かし方やソースコードの説明をします。

読み進め方

いくつかの状況に対して、各種ドキュメントをどのように読み進めるべきかを紹介します。

パッケージの導入を行いたい場合

  1. NintendoSDK オーバービュー
  2. ドキュメントガイド (本ドキュメント)
  3. 環境構築ガイド 

特定のモジュールをとりあえず触ってみたい場合

  1. （各モジュールの）オーバービュー
  2. チュートリアル
  3. サンプルのドキュメント

実装開始時あるいは実装中に調べたい場合

  • チュートリアル、サンプルのドキュメント
    • サンプルを実行しつつ読むことで実際の動作を確認しながら学習ができる。
    • 実装の参考になるコード片を参照することができる。
  • マニュアル
    • 詳細な仕様を調べることができる。
  • API リファレンス
    • API に関する仕様を調べることができる。
