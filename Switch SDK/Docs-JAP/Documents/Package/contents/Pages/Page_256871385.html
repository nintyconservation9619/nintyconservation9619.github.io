カーブエディタに「前後区間のキーを表示」機能を追加 ( 分割モード専用機能 )

編集中のアニメーションカーブの前後に、直前直後の「アニメーション区間タグ」の「直近のキー」を表示して参照できる、カーブ作成の補助をする機能が追加されました !!

扱っている「アニメーション区間」前後の「直近のキー」を、カーブエディタに参考表示することで、直前直後のアニメーションを直接確認しながらの作業が可能となりました。
※この機能を使用するには、直前直後のアニメーション区間タグに、同じ種類のアニメーションが存在する必要があります。

使用方法

カーブエディタウインドウの「前後区間のキーを表示」アイコンをクリックすると機能が有効になります。

「アニメーション区間タグ編集ウインドウ」の表示順序を参考に、現在選択中の「アニメーション区間タグ」の前後の区間タグにある、アニメーションキーを参考表示します。
参考表示したアニメーションキーに、現在編集中のアニメーションカーブの最初と最後の値をスナップすることも可能です。


前後の「アニメーション区間タグ」のアニメーションキーをアニメーションカーブの前後 ( 範囲外 ) に参照表示

A

「 前後区間のキーを表示」の使用例です。

一つ前のアニメーション区間タグのアニメーションキーを参照表示 ( 例：2.00 )
B

「 前後区間のキーを表示」の使用例です。

一つ後のアニメーション区間タグのアニメーションキーを参照表示 ( 例：2.50 )
参照表示したキーの値に合わせて、選択したカーブのスナップ機能を使用

C

「 選択したカーブを前区間にスナップ」の使用例です。

参照表示した一つ前のアニメーション区間タグのキーの値に、アニメーションカーブの先頭のキーがスナップされました。
D

「 選択したカーブを後区間にスナップ」の使用例です。

参照表示した一つ後のアニメーション区間タグのキーの値に、アニメーションカーブの最後のキーがスナップされました。
アイコン 名称 機能

前後区間のキーを表示 前後の「アニメーション区間タグ」のアニメーションに、同種のアニメーションが存在する場合、直近のキーを参照表示します。
アイコンをクリックすると「表示 / 非表示」が切り替わります。

選択したカーブを前区間にスナップ アイコンをクリックすることで、選択したアニメーションカーブの先頭のキーを「前後区間のキーを表示」で参照表示した「前区間キー」の値にスナップします。
「前区間のキー」を表示状態にすると、選択可能となります。

選択したカーブを後区間にスナップ アイコンをクリックすることで、選択したアニメーションカーブの最後のキーを「前後区間のキーを表示」で参照表示した「後区間キー」の値にスナップします。
「後区間のキー」を表示状態にすると、選択可能となります。

サムネール保存機能を追加

LayoutEditor にサムネール用の画像を保存する機能を追加しました !!

「プロジェクト設定ファイル」から設定を有効にすると「レイアウトファイルを保存した」「一括同期を行った」等の時、サムネール画像をレイアウトファイルの横の .NwThumbs フォルダ内に「レイアウトファイル名.flyt.png」として保存します。
また、レイアウトエディタウインドウのメニューバー「ファイル」に、「指定フォルダ以下のサムネール画像を更新」のメニューが追加されます。

保存された画像は「サムネールエクスプローラ拡張」を使用して表示することができます。

メニュー追加された、レイアウトエディタウインドウのメニューバー「ファイル」

A メニューバー「ファイル」に追加された項目、「指定フォルダ以下のサムネール画像を更新」
