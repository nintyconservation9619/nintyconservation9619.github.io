adpcm デコーダ

機能概要

adpcm デコーダは adpcm データをデコードして PCM データに変換するための機能を提供します。
同様の機能は AUDIO ライブラリ の オーディオレンダラ に組み込まれた Voice 処理でも実現されますが、nn::codec が提供するのはデコード単独の機能です。

adpcm デコード機能として、nn::codec は以下の API を提供します。

  • nn::codec::DecodeAdpcm()
  • nn::codec::GetAdpcmLoopContext()

デコード機能は nn::codec::DecodeAdpcm() で完結します。nn::codec::GetAdpcmLoopContext() はループ再生時に必要になるコンテキスト情報を取得するための手段として提供されています。

それぞれの API は、adpcm データのヘッダ部分に含まれる nn::codec::AdpcmParameter や nn::codec::AdpcmContext などの情報が必要になります。
