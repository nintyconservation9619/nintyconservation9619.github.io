SRT制御

このタブではエミッタセットに対してゲームアプリケーション側で制御可能な設定や、3Dモデルとエフェクトとの連携設定を行います。

行列設定

「 行列設定」は、パーティクルに対するスケールのかけ方を設定する機能です。
以下はグループ内設定項目の一覧です。

行列設定タイプ　

パーティクルに対するスケールのかけ方を設定する機能です。

モデル連携

「 モデル連携」は、 EffectMaker に登録したポリゴンモデルとエフェクトを連携させる機能群です。
以下はグループ内設定項目の一覧です。

マトリクス適用タイプ エミッタセットが接続ボーンの回転とスケールに接続するかを決める機能です。
接続モデル 連携するモデルを「モデル設定ノード」に登録したモデルファイルから選択する機能です。
接続ボーン エフェクトを放出させたいボーンを「接続モデル」で選択した連携モデルから選択する機能です。

関連リンク 以下は関連ページへのリンクです。

3Dモデルと炎を連携させる アニメーションする3Dモデルとエフェクトを連携させる手順について説明します。
モデル設定ノード ビューアで確認する為のモデルファイルとアニメーションファイルの設定を行います。

エミッタセット行列

「 エミッタセット行列」は、エミッタセット内のエミッタ全体に対して「エミッタトランスフォーム」で設定した値にオフセットを与える機能群です。
エフェクト全体のスケールを変えたいときに使用します。
また、モデルに連携時に、接続先ボーンから位置にオフセットを与える時にも利用できます。
以下はグループ内設定項目の一覧です。

座標（±） エミッタの位置にオフセットを与える機能です。
エミッタビルボード エミッタをカメラ方向に向かせる機能です。
「無し」に設定した場合は各軸個別にエミッタの回転角度を設定できます。
スケール エミッタのスケールに設定した値を乗算する機能です。
参考：

「エミッタセット行列」の設定を処理する座標系はエミッタとパーティクルで以下のように異なります。

エミッタ ワールド座標系で処理されます。
パーティクル エミッタの座標系で処理されます。
