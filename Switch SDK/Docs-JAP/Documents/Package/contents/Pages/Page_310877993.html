概要

ペアリング済みのコントーローラー数、接続されているコントローラー状態の確認、ペアリング情報の削除を行います。
接続されているコントローラーがある場合にのみ、その情報がページに表示されます。ペアリング済みだけれども未接続のコントローラー情報は表示されません。

使い方

コントローラーのペアリング

｢Start Controller Pairing｣ ボタンを押すことで、開発機とコントローラーのペアリングを開始できます。
ペアリングをキャンセルしたい場合は ｢Cancel Controller Paring｣ を押してください。

コントローラーペアリング設定の削除

｢Delete All Controller Paring Settings｣ ボタンを押すことでペアリング設定を削除できます。

コントローラー接続状態、充電値残量の表示

ペアリング済みのコントローラー数、接続されているコントローラーの状態を確認できます。

Device Style

本体に接続されているコントローラーの接続スタイルを確認できます。スタイルの種類は以下の通りです。

Style 内容
Joy Left Joy-Con (L)が接続されていることを表します。
Joy Right Joy-Con (R)が接続されていることを表します。
Handheld 本体ジョイントレールにJoy-Conが装着されていることを表します。
Pro Con Switch Proコントローラーが接続されていることを表します。
Other 上記以外のコントローラーが接続されていることを表します。

プレイヤーランプ

本体に接続されているJoy-Conが、画面に表示されているコントローラーの内のどれに該当するかをプレイヤーランプ点灯パターンから確認できます。
接続スタイルがHandheldの場合はプレイヤーランプは無点灯で表示されます。コントローラー番号は左右Joy-Conそれぞれに最大4個まで割り振られます。

Firmware Version

接続されているコントローラーのFirmware Versionを表示します。最新バージョンの詳細は コントローラーファームウェアのリリースノート のページを参照してください。
現時点では接続スタイルがHandheld時はFirmware Versionが表示されないため、Firmware Versionを確認するには開発機本体からコントローラーを外した状態で接続してください。

Battery

コントローラーの充電池状態を確認できます。DevMenuでは、充電値残量はアイコンにより次の5段階で表現されます。

定義 説明
Empty 充電池残量がゼロであることを表します。
Critical 充電池残量がほとんど残っていないことを表します。残量が赤く表示されます。
Low 充電池残量が少なくなっていることを表します。
Medium 充電池残量が約半分であることを表します。
High

充電池残量が十分であることを表します。

接続スタイルがHandheldの場合、以下に基づいて充電池状態が表示されます。

  • 接続されているJoy-Conが 1 つの場合は、そのJoy-Conの充電池状態を表示する
  • 接続されているJoy-Conが左右 2 つの場合は、Joy-Con (R)の充電池状態を表示する

ジャイロスコープ、加速度センサー設定の変更

ジャイロスコープと加速度センサーの設定を変更できます。

プレイヤーランプ点灯パターン毎に各センサーの「ゼロ点」と「感度」を変更できます。

  • 「Apply Changes」を行うと、コントローラーが切断されます。再接続すると変更した値が反映されます
  • コントローラーが接続されていないプレイヤーランプ点灯パターンに対しても設定することができ、コントローラーが接続されると設定値が反映されます
  • 変更した値は開発機を再起動するとデフォルト値に戻ります

振動設定の変更

「Enable Vibration」にて、コントローラーの振動設定のON/OFFを切替えられます。

Proコントローラー有線USB通信機能設定の変更

「Enable Pro Controller USB Communication」にて、Proコントローラーの有線USB通信機能設定のON/OFFを切替えられます。
