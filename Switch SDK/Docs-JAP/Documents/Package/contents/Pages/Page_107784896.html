NVIDIA GameWorks

概要

GameWorks は、NVN を用いたグラフィックスライブラリおよびサンプル集のパッケージです。

パッケージ

GameWorks は、NintendoSDK NX Addon とは別のパッケージで提供されます。また、他の NintendoSDK パッケージよりも遅れてリリースされる場合があります。

以下にパッケージを一覧します。

パッケージ名 説明
NVIDIA GameWorks NVN を用いたグラフィックスライブラリおよびサンプル集です。

配置

GameWorks は、以下のように NintendoSDK  を配置したディレクトリ直下に配置されます。

NintendoSDK/
GameWorks/
    extensions/
    Samples/
    tools/

マニュアル

パッケージの詳細については、パッケージ内のドキュメントを参照してください。
