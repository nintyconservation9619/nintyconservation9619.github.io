寿命無限

ここではパーティクルの寿命を無限にする機能「寿命無限」について説明します。

概要

「寿命無限」は、エミッタから放出された全てのパーティクル寿命を無限にできる機能です。
例えば、エミッタにパーティクルを一つだけ表示させたままにする場合などに利用できます。

参考：
  • 「寿命無限」は放出設定「ワンタイム」が ON の場合に機能します
    関連リンク 「ワンタイム」の詳細については ワンタイム を参照してください。
  • 再生中のエフェクトを強制的に停止しする機能「終了処理」の「αフェード」を ON にすることで、寿命無限のパーティクルを消滅できます。
    関連リンク 「終了処理」の詳細については 終了処理 を参照してください。
  • 「寿命無限」を使用した場合、初速などの設定をビューアに反映させる為には F5 キーによるビューアへの再転送やプレビューコントロールでアニメーションを停止→再生する必要があります。
    関連リンク 「プレビューコントロール」については プレビューコントロール を参照してください。

以下は「 設定項目」の概要です。

寿命無限

寿命を無限に設定するかを「 チェックボックス」ON/OFF で切り替えます。
寿命を無限にすると、スケールや回転等、寿命にあわせてアニメーションする機能に影響があります。

設定をOFF にすると「 寿命無限OFF(寿命を指定)」が表示されます。以下はグループ内の設定項目です。

寿命  パーティクルの寿命を設定する機能「寿命」について説明します。
