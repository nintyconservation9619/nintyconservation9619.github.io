テスト配信

テスト配信とは

ファイルごとに設定した配信開始日時を無視して、特定の本体ごとにカスタマイズした組み合わせのデータを配信することができます。この仕組みを「テスト配信」と呼びます。

テスト配信は開発環境において複数の配信状況を並行してデバッグするとき、および製品環境において最終確認をするときに利用します。

テスト配信の受信設定

テスト配信を受信するには、以下の操作が必要です。

詳細は「BCAT管理ツール取扱説明書」を参照してください。

配信先の本体の指定

BCAT管理ツールを操作して、対象の本体のシリアル番号、および対象の本体に連携しているニンテンドーアカウントを指定してください。

本体の接続先の切り替え

本体を操作して、テスト配信用のサーバーに接続できるようにしてください。

テスト配信の優先度

本体にテスト配信の受信設定を行うと、通常の配信は受信されなくなります。

あらかじめ通常の配信で受信していたデータが本体に存在する場合は、テスト配信の内容で上書きされます。
