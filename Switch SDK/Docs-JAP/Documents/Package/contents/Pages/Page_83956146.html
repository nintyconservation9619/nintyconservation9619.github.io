サンプルマップ

サンプルマップ

バンクタブは上下に分割されて表示されます。上領域はインストリストで、下領域にはサンプルマップとパーカッションリストのタブが配置されています。
この下領域で、サンプルマップのタブを選択した時に表示されるのがサンプルマップです。

 

サンプルマップの役割

サンプルマップはインストルメントを編集する専用のエディタです。
インストリストで選択したインストルメントの編集を行います。

サンプルマップは、鍵盤の表示と、その上にリージョンの領域をブロックで視覚的に示すオーソドックスなスタイルを採用したインストルメントのエディタで、主にマルチリージョンのインストルメントの作成に適しています。

サンプルマップの操作

ここではサンプルマップで可能な操作を解説します。

リージョンの表示

リージョンが表示される領域には、上部に薄い水色のバーが付属しています。このバーをリージョンヘッダーと呼びます。
ベロシティリージョンをまとめて操作する場合には、このリージョンヘッダーをクリックして利用します。

 

キーリージョン・ベロシティリージョンの作成

リージョンの領域（リージョンヘッダーを含む）で右クリックして現れるコンテキストメニューで、キーリージョンの分割またはベロシティリージョンの分割ができます。

キーリージョンの削除や移動などでできた、ブランク（空）の領域をダブルクリックする操作でも、新しいキーリージョンを作成することができます。

また、ベロシティリージョンはキーリージョンの一部として扱われます、キーリージョンの無い場所に、ベロシティリージョンだけを作成することはできません。

 

リージョン分割の制限

リージョンヘッダーをクリックして、ベロシティリージョンを複数選択した状態からのベロシティリージョンの分割や、同一キーリージョン内でベロシティリージョン分割されていて、全ベロシティリージョンが選択されていない状態からのキーリージョンの分割などは行えません。

 

リージョンの参照波形ファイルの指定

リージョンの領域をダブルクリックするか、右クリックして現れるコンテキストメニューから「参照波形の変更」を選択することで、リージョンの参照波形ファイルを指定できます。

オリジナルキーについて、指定した波形ファイルのファイル名や付加情報に基づいて、自動的に適切と思われる値に設定されます。
（例えば "Test.A3.aif" というファイル名の "A3" という部分）

フォルダを開く・参照ファイルを開く

右クリックして現れるコンテキストメニューから「フォルダを開く」または「参照ファイルを開く」を選択することで、そのファイルに対するそれぞれの動作を行えます。

キーリージョンの操作

リージョンヘッダーをドラッグ＆ドロップすることで、キーリージョンの内容を他のキーリージョンと入れ替えることができます。この時、入れ替えるキーリージョンにベロシティリージョンが存在すれば、その情報も同時に入れ替わります。

リージョンヘッダーを Ctrl + ドラッグ＆ドロップすることで、キーリージョンの内容を他のキーリージョンにコピーすることができます。
入れ替えと同様に、コピー元のキーリージョンにベロシティリージョンが存在すれば、その情報も同時にコピーされます。

ベロシティリージョンの操作

リージョンヘッダー以外の領域（ベロシティリージョン自体）をマウスでドラッグした場合には、ベロシティリージョンに対する操作として扱われます。

リージョンヘッダーを使った操作と同様の操作で、通常のドラッグ＆ドロップでは移動、もしくは入れ替え、Ctrl + ドラッグ＆ドロップでコピーを行います。

ベロシティリージョンの境界をドラッグ中には、その時の値がツールチップに表示されます。

リージョンの圧縮方式の指定

リージョンが参照する波形ファイルの圧縮方式の指定は、鍵盤の下のスペースにある、圧縮方式の項目で行います。

  • ADPCM
    波形ファイルを専用の ADPCM 形式にコンバートします。
    圧縮によりループの先頭が、ブロックの先頭に自動的に移動されるので、ループ位置のタイミングを利用した再生制御を行う場合などには注意が必要です。
  • PCM16
    16bit の PCM データにコンバートします。

リージョンのオリジナルキーの指定

リージョンのオリジナルキーの編集は、リージョンの表示領域と鍵盤の間にあるバーで行います。
バーのある鍵盤の位置の時、選択中のリージョンの波形ファイルが元のピッチで再生されます。

 

ベロシティの指定

リージョンの表示領域の左にあるスライダーで、鍵盤による再生時のベロシティを指定できます。

インストルメント波形の参照回数

サウンドプロジェクト内にある、すべてのシーケンスサウンドが使用する、インストルメント波形の参照回数を調べることができます。

[プレビュー]→[インストルメント波形の参照回数を計測] を実行します。

 

「インストルメントの参照波形の計測」ダイアログが表示されるので、最大計測時間を設定して計測ボタンを押します。
計測は、各サウンドの再生開始位置から行われます。

 

参照回数は、サンプルマップのリージョン領域に参照回数の数値として表示されると共に、リージョンの赤色の濃淡としても表示されます。

 

シーケンスサウンドのファイルパス、またはファイルが外部で更新された場合には、参照回数の数値の横に '*' が表示され、参照データが正しくない状態であることを知らせます。

インストルメントの参照波形の計測については、「波形の参照回数」の項目も参照してください。
