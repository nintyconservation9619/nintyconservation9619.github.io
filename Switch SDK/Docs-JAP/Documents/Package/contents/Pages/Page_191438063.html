ニンテンドーアカウントとの連携

事前準備

開発環境のためのニンテンドーアカウントの取得

開発環境では、開発環境のために作成されたニンテンドーアカウントを取得する必要があります。
開発環境のためのニンテンドーアカウントを作成する方法について、詳細は開発環境でのニンテンドーアカウント作成手順 ページを参照してください。 

ニンテンドーアカウントとの連携

次のいずれかの手順によって、ユーザーアカウントをニンテンドーアカウントと連携できます。

  • DevMenu を使用する
    1. 「Accounts」メニューからユーザーアカウントの詳細画面を開く
    2. 「Ensure NSA with UI」を押す
      • 本体システムのユーザーインターフェースが表示され、ニンテンドーアカウントとの連携処理を実施できます。
  • HOME メニューを使用する
    • ユーザー詳細画面や本体設定での誘導に従いニンテンドーアカウントを連携する

ニンテンドーアカウントとの連携の状態

ニンテンドーアカウントの状態の更新

ニンテンドーアカウントの状態はサーバー上で随時更新され得ますが、それが更新された場合に Nintendo SDK 上で感知されるまでには時間差が生じます。
Nintendo SDK ではこの時間差について次のように定めています。

ニンテンドーアカウントの状態の更新は Nintendo SDK に対し、ネットワークへの接続を維持した状態でおよそ 24 時間以内に反映されます。

ニンテンドーアカウントの状態の手動更新

開発環境においては、次の手順でニンテンドーアカウントの状態を即時反映することができます。

  • DevMenu の 「Accounts」メニューからユーザーアカウントの詳細画面を開く
  • 「Update NA info cache」を押す
    • ニンテンドーアカウントサービスとの通信を行い、最新の状態を取得し、 Nintendo SDK 上で反映します。

ニンテンドーアカウントとの連携の状態の不健全性

ニンテンドーアカウントや本体システムの状態により、ユーザーアカウントとニンテンドーアカウントとの連携状態が不健全になる場合があります。 
連携状態が不健全になると、次の影響を生じます。

  • ニンテンドーアカウントを利用できない
  • ネットワークサービスアカウントを利用できない

意図せずこの状態が生じた場合は、次の手順でニンテンドーアカウントとの連携状態を回復することができます。

  • 「ネットワークサービスアカウントの利用」ページの「ネットワークサービスアカウントの可用性の回復」に記載の手順に従う
