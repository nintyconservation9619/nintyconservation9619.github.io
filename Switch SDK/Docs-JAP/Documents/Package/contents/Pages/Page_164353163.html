テクスチャパターン

テクスチャパターンは、LayoutEditor のペインの登録テクスチャから、テクスチャパターンリストを作成します。

「テクスチャパターンアニメーション」は、ここで製作した「テクスチャパターンリスト」を使用し、製作します。

テクスチャパターンリスト

テクスチャパターンリストタブを選択すると、テクスチャパターンリストを製作する画面が表示されます。

テクスチャパターンリスト作成画面
テクスチャパターンリスト

テクスチャパターンアニメーションに使用する、テクスチャリストです。

アニメーションに使用するテクスチャをリストアップし、登録番号を付けます。

テクスチャパターンアニメーションは、この登録番号を元に作成します。
追加

「テクスチャパターンリスト」にテクスチャを追加するために、「テクスチャパターンの追加ウインドウ」を開きます。
削除 「テクスチャパターンリスト」から、登録したテクスチャを削除します。
番号 テクスチャパターンリストに入れた、テクスチャの登録番号です。
↑ テクスチャパターンリストの中の、選択したテクスチャの登録番号を上げます。
↓ テクスチャパターンリストの中の、選択したテクスチャの登録番号を下げます。
クリア テクスチャパターンリスト内を全削除します。

テクスチャパターンの追加

テクスチャパターンリストにテクスチャを追加するには、「テクスチャパターンの追加ウインドウ」を使用します。

テクスチャパターンの追加ウインドウ
テクスチャパターンの追加 テクスチャウインドウに登録されたテクスチャが、一覧表示されます。
チェックを入れ、OK ボタンを押すことで、テクスチャパターンリストに登録されます。
( 複数同時選択の場合、テクスチャパターンリストに登録される番号は選択順ではありません )
