6 テクスチャーパターンアニメーション

テクスチャーパターンのアニメーションを作成します。

パターンアニメーション用テクスチャー

テクスチャーパターンアニメーションのテクスチャーを貼替えます。

マテリアル「PatternMt」のファイルアトリビュートに設定されているテクスチャーを以下の「TexturePattern.0.tga」に貼替えます。

場所 %NINTENDO_SDK_ROOT%Samples\Sources\Applications\G3dDemo\Resources\TrafficSignal\sourceimages\TexturePattern.0.tga
参考：

上記フォルダーには以下の 5 枚の連番テクスチャーが格納されています。

  • TexturePattern.0.tga
  • TexturePattern.1.tga
  • TexturePattern.2.tga
  • TexturePattern.3.tga
  • TexturePattern.4.tga

テクスチャーパターンアニメーション用のテクスチャーファイルは、これらのように拡張子の前にフレーム拡張子に対応する数字を付けておきます。 

パターンアニメーションを確認、エクスポートするには同じフォルダーに連番のテクスチャーが必要です。


イメージシーケンス

マテリアル「PatternMt」のファイルアトリビュートの「イメージ番号」にアニメーションキーを作成します。

Color アトリビュートに接続されている file ノードをアトリビュートエディタで開きます。
「イメージシーケンスの使用にチェック」→「イメージ番号を右クリック」→「エクスプレッションの削除」をしてキーを作成します。

下表のようにキーを作成します。

 
0 フレーム
10 フレーム
20 フレーム
30 フレーム
 40 フレーム
イメージ番号 0 1 2 3 4

アニメーションのカーブを選んで「ステップ接線」に変更します。
意図したタイミングでテクスチャーが切り替わるようになります。

ハイパーシェードなどでアニメーションを設定した file ノードを選択し、グラフエディタ上でアニメーションのカーブを選んで「ステップ接線」に変更します。

下図のようなイメージシーケンスのアニメーションになることを確認します。

出力対象

テクスチャーパターンのアニメーションの出力対象はファイルアトリビュートの「イメージ番号」のキーフレームです。

「Nintendo Export Options」の「Output File Selection」の D「Texture Pattern Animation Data」に対応しています。
