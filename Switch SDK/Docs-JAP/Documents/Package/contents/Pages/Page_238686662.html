文字毎の変換を利用する

文字毎の変換の利用方法について説明します。

LayoutEditor での設定方法

手順１：テキストボックスペインの、｢文字毎の変換｣属性を有効にする

  • テキストボックスを選択し、ダブルクリックしプロパティウィンドウを開きます。
  • 左ツリーから、｢テキストボックス｣を選択し、「文字毎の変換」グループから、「有効」チェックを入れます。チェックを入れると、各種設定パラメータが有効になります。
  • ｢有効｣にしなければ追加処理負荷はほとんどありません。
  • 適宜、「オフセット時間」、「時間幅」を設定します。良くわからない場合はとりあえず、そのままでも構いません。
  • 今回新たに使い始める場合は、効果が分り易いように、テキストボックスの文字数は少し多めにして改行されない状態にしておくとよいと思います。

手順２：テキストボックスに、文字毎の変換カーブを設定する

  • カーブエディタを開きます。
  • 手順１で｢文字毎の変換｣属性を有効にした、テキストボックスを選択します。
  • 左ツリーから次のように項目を選択します。[ ペインの名前 ] → [ テキストボックス ] → [ テキストボックス ] → [ 文字毎の変換 : 移動 ]
  • キーフレームを設定し、変換カーブを設定します。正しく設定されていれば編集画面で、文字に変化が現れるはずです。
  • 手順１で設定した、「オフセット時間」や「時間幅」を調整することで、設定した変換カーブを評価する範囲を指定することができます。

手順３：アニメーションさせる（任意）

  • カーブエディタで、文字毎の変換：時間オフセットにキーを打ちます ( [ ペインの名前 ] → [ テキストボックス ] → [ テキストボックス ] → [ 文字毎の変換 : 時間オフセット ] )
  • この設定で、手順１にあった、「オフセット時間」がアニメーションし、変換カーブの評価開始時間が変化します。アニメーションを再生すると、文字がアニメーションするはずです。
  • カーブエディタで、手順２で設定した変換カーブを選択すると、評価開始時間が｢青い線｣で確認できます。( 通常のアニメーション再生時間は赤い線です )
