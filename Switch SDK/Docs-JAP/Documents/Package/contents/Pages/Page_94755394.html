「ディレクショナルビルボード」と「ディレクショナルポリゴン」

ここでは「ディレクショナルビルボード」と「ディレクショナルポリゴン」の違いについて説明します。

概要

”ディレクショナル～”と付く項目はパーティクルの速度をパーティクルの向きに影響させるパーティクルタイプです。
「ディレクショナルビルボード」と「ディレクショナルポリゴン」の２種類があります。

ビルボード系とディレクショナル系の違い

以下は「ビルボード」と「ディレクショナルビルボード」を比較しています。各エミッタ共に全方向に速度を与えています。

A ビルボード 指向性のあるパーティクルでは違和感があります。
B ディレクショナルビルボード 速度方向にパーティクルが傾く為、指向性のあるパーティクルでは有効な場合があります。
参考：

パーティクルの速度に依存するディレクショナル系パーティクルは、パーティクルの速度が 0 の場合表示されません。

「ディレクショナルビルボード」と「ディレクショナルポリゴン」の違い

以下の様にカメラを動かす事で「ディレクショナルビルボード」と「ディレクショナルポリゴン」の違いが解ります。

A ディレクショナルビルボード

パーティクルの Y 軸が速度方向を向き、Y 軸を回転させて Z 軸が常にカメラ方向を向くパーティクルタイプです。

Y軸方向に長いパーティクルは円柱状に見えます。長く伸び、曲線運動をとる火花の様なエフェクトに有効です。
B ディレクショナルポリゴン

パーティクルの Y 軸が速度方向を向くパーティクルタイプです。

「ディレクショナルビルボード」と違い、Y軸での回転アニメーションが利用できます。ひらひらと宙を舞う、木の葉の様なエフェクトに有効です。
