ソフトパーティクル

ここではソフトパーティクルについて説明します。
「ソフトパーティクル」は、モデルとパーティクルが重なったとき、パーティクルの境目に半透明処理を施す機能です。
この処理により、境界に見られるラインを目立たなくし、エフェクトを自然に見せることができます。

以下は「ソフトパーティクル」の機能を使用しない場合と使用した場合の比較画像です。
着地した雪の、床と重なった「 〇 赤丸部分」を確認することで違いがわかります。

A 未使用（デフォルト） 雪と床の重なりが目立ち、不自然に見えます。
B 使用 雪と床の重なった箇所に半透明処理を入れている為重なりが目立ちません。

ソフトパーティクルはパーティクルの深度値をオフセットする機能の後に処理されます。
以下は「Z値をオフセット」と「Zファイティング抑制」使用/未使用時の比較画像です。

A 利用しない（デフォルト） 「Z値をオフセット」の処理なしで「ソフトパーティクル」の処理のみが適用されています。
B オフセット値（±）= 1 「Z値をオフセット」の処理により 1 手前にオフセットがかかった位置で「ソフトパーティクル」の処理が適用されています。
C Zファイティング抑制 : ON 「Zファイティング抑制」の処理により手前にオフセットがかかった位置で「ソフトパーティクル」の処理が適用されています。
