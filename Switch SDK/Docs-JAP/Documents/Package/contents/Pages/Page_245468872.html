検索ウィンドウ

指定した範囲のエミッタセットファイル（ .eset）から、指定した設定のファイルを探し出し、設定内容を一括変換できる「検索ウィンドウ」が追加されました！
例えば多くのエフェクトファイル中に設定内容に問題のあるファイルが複数混在しているとき、これまでは該当するファイルを探し出し、ひとつずつ開いて修正する必要がありました。
本機能の利用により、「検索」タブで問題のあるファイルを検索し、ヒットしたファイルに対して「置換」タブで一括修正できます。

検索

「検索」タブでは指定したフォルダまたはそれ以下のフォルダに対し、指定した条件下にある設定内容のファイルを検索できます。

A 検索フォルダ 検索したいフォルダをテキストボックスにドラッグアンドドロップまたはフォルダボタンを押して指定します。
B 検索条件 検索条件をコンボボックスで選択し、追加ボタンで条件を追加します。
C 検索条件の合成 複数の検索条件を合成し、合成式を上から順番に評価して検索を行います。
D 検索を実行 設定した条件で検索を行います。
E 検索結果 条件に該当するエフェクトファイル名が表示されます

置換

「置換」タブでは検索結果に表示されたエミッタセットファイルの設定内容を指定した内容で置換できます。
安全のため、置換を実行する前にデータのバックアップをとることをおすすめします。

A 置換式 置換する項目について置換式を追加します。
B 置換を実行 検索で検索した全てのエミッタファイルに置換式が適用されます。
C 検索結果 置換されたエフェクトファイルが表示されます。

既知の不具合

検索ウィンドウは開発途中のため下記の不具合が存在します。今後のリリースで順次修正予定です。

  • 検索タブの検索条件数よりも置換タブの置換式条件数が多い状態で「検索の実行」か「置換の実行」をすると致命的なエラーが発生します。
  • 検索タブで検索条件に設定していないアルファベットの項目を式に入れて検索を実行すると致命的なエラーが発生します。
  • 検索タブ「検索条件」か置換タブ「置換式」に複数の条件を設定後、ゴミ箱ボタンで条件を削除すると、一番右のプルダウンが「と等しい」に自動的に変わってしまいます。

また、検索できない条件も存在します。

  • アニメーションカーブについてはカーブデータの形状そのものをデータ列として検索することに対応していません。
  • カラーの検索については対応していません。
  • パーティクルの位相ランダム、初期位相、位相ランダムについては対応していません。
