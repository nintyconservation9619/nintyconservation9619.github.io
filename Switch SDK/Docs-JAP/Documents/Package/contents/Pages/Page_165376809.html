カラー補間設定

カラー補間設定

LayoutEditor のペインの中の「カラー補間設定」をします。

しきい値によるアルファ補間 「しきい値によるアルファ補間」の設定入力への切り替えを行います。
黒カラー補間 「黒カラー補間」の色やアルファ値を設定します。
白カラー補間 「白カラー補間」の色やアルファ値を設定します。

 しきい値によるアルファ補間

「しきい値によるアルファ補間」の設定入力への切り替えを行います。

しきい値によるアルファ補間

チェック無しの場合（初期設定）

テクスチャのカラーに「黒カラー補間」と「白カラー補間」の入力値で影響を与えます。
しきい値によるアルファ補間

チェック有りの場合

「黒カラー補間」と「白カラー補間」で設定したアルファ値を「しきい値」として、アルファ値の補間を行います。

関連リンク 詳細については「しきい値によるアルファ補間」を参照して下さい。


 黒カラー補間

「黒カラー補間」の色やアルファ値を設定します。

黒カラー補間

黒カラー補間の値 ( R,G,B,A ) を入力します。

 

 白カラー補間

「白カラー補間」の色やアルファ値を設定します。

白カラー補間

白カラー補間の値 ( R,G,B,A ) を入力します。
