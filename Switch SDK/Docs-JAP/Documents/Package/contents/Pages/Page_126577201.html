お気に入りのサンプルを読み込む

 ここでは「 エフェクトブラウザ」を使ったファイルの読み込み方法について、「お気に入り」のサンプルを用いて説明します。

準備

あらかじめ EffectMaker と EffectViewer を起動した状態から作業を進めます。

関連リンク EffectMaker と EffectViewer の起動方法については 「作業の流れ」 を参照してください。

エフェクトブラウザを起動する

EffectMaker ツールバーの「 エフェクトブラウザ」ボタンを押してエフェクトブラウザを開きます。

エフェクトブラウザの作業

以降はエフェクトブラウザの操作について説明します。
関連リンク 詳細については「エフェクトブラウザ」を参照してください。

「お気に入り」タブを選択する

エフェクトブラウザのウィンドウから「お気に入り」タブを選択します。

お気に入りを選択する

「お気に入り」タブのナビゲーションペインから、サンプル用のお気に入り「 VfxSample」を選択します。

ファイルを選択する

ファイルリストに表示された「 Bubble_Fish.work」を選択します。

選択したファイルをエクスポートする

ファイルを選択した状態で、ツールバーの「 ワークスペースへエクスポート」をマウス左ボタンクリックします。
この操作により、ワークスペースに「 Bubble_Fish.work」とに関連付けされたファイルが読み込まれます。
 

参考：

ワークスペース上の作業状態は、「 ワークスペースファイル（ .work）」として保存できます。
保存したワークスペースファイル（.work）を開くことで、保存した時の状態へ復帰させることができます。
関連リンク 詳細については「ワークスペースファイル」を参照してください。

メインウィンドウの作業

以降はメインウィンドウの操作について説明します。

ワークスペースを確認する

ワークスペースを確認します。
ワークスペースのノードに「 Bubble_Fish.work」と関連付けされたファイルが読み込まれていることを確認します。

以下は確認するノードの種類とそのノードに登録されたファイルです。

ノードの種類 ファイル
エミッタセット 「 Bubble_Fish.eset」、「 Drop_02.ftxb」、「 SphereMap.fmdb」
プレビュー 「 Bubble_fin1.prev」、「 Bubble_fin4.prev」、「 Bubble_head.prev」
モデル設定 「 Fish.fmdb」、「 Fish.fskb」
