サンプリング時のフィルタ設定

マテリアルプロパティウィンドウのサンプラページからテクスチャサンプル時のフィルタをプルダウンメニューで指定します。

以下に各設定の内容を説明します。

拡大

テクスチャ拡大時のフィルタを指定します。

フィルタ 説明
ポイント ポイントサンプリングします。
リニア サンプリングする際に、リニアフィルタを適用します。

縮小

テクスチャ縮小時のフィルタを指定します。

フィルタ 説明
ポイント ポイントサンプリングします。
リニア サンプリングする際に、リニアフィルタを適用します。

ミップマップ

ミップマップ間のフィルタを指定します。

フィルタ 説明
なし ミップマップを適用しません。
ポイント ミップマップを適用します。
リニア ミップマップを適用し、ミップマップ間をリニア補間します。

アニソトロピック

アニソトロピックフィルタの最大値を指定します。
