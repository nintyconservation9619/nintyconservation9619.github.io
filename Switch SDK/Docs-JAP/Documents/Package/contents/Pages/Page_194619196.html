Vulkan ライブラリ

はじめに

Vulkan とは Khronos Group が策定しているリアルタイムグラフィックス・コンピュート API です。

Vulkan は NintendoSDK NX Addon でサポートされており、Nintendo SDK ユーザが Vulkan を利用する既存のタイトル、フレームワークの移植を目的とした場合を想定して提供されています。

PC 動作環境

NintendoSDK の動作環境に準拠します。

GPU ドライバのインストール方法 

ご利用の PC に Vulkan の環境をセットアップしてください。

Nintendo Dev Interface から入手することが出来る「NVIDIA Graphics Driver」パッケージは Vulkan をサポートしています。

サポートされるバージョン

Nintendo Switch™ は Vulkan® 1.0 に準拠しています。
