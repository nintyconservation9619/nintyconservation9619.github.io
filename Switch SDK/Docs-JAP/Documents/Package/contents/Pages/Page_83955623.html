中間ファイルチェッカー

中間ファイルチェッカーの使用方法と注意事項を説明します。

  • 概要
  • コマンドラインオプション
  • 利用例
  • チェック項目

概要

正しい中間ファイルかどうかをチェックします。

何らかの問題が起こった際に、中間ファイルに問題が無いかを確認するために利用します。

中間ファイルを独自ツール等で書き出した際には、このツールで中間ファイルに問題が無いことを確認してください。

実行ファイル (3dIntermediateFileChecker.exe) やショートカットに、中間ファイルやフォルダーをドラッグ＆ドロップすると、中間ファイルをチェックします。
中間ファイル以外のファイルは無視します。

問題があった場合は、エラーメッセージを表示してキー入力待ち状態になります。
問題が無かった場合は、そのまま終了します。

コマンドラインオプション

3dIntermediateFileChecker.exe [--args-file ""] [--version] [-h] [-s] [-j ""] 
[--enable-extension ""] [--disable-extension ""] [中間ファイル|フォルダ]... 

正しい中間ファイルかどうかをチェックします。

--args-file ""

引数ファイルを指定します。
引数ファイルに記述した内容を、引数として指定できます。 
引数ファイルは複数行に渡って各オプションを記述できます。 
行頭に # がある行はコメントとして扱います。 
オプション値指定の "" によるエスケープに改行を含めることはできません。

--version

バージョン情報を表示します。

-h --help

ヘルプを表示します。

-s --silent

処理中のメッセージ表示を抑制します。

-j "" --jobs ""

実行ジョブ数を指定します。

--enable-extension ""

中間ファイルの中で、処理する拡張子を指定します。
複数記述する際は , で区切ります。
例）--enable-extension "fmda,ftxb"

--disable-extension ""

中間ファイルの中で、処理しない拡張子を指定します。
複数記述する際は , で区切ります。
例）--disable-extension "fmda,ftxb"

中間ファイル|フォルダー

処理するファイルやフォルダーを指定します。
フォルダーを指定した場合、指定フォルダー以下のファイルを再帰的に処理します。
中間ファイル以外は無視します。

ワイルドカード（?: 一文字、*: 複数文字）に対応しています。
例) "a?b*"
一文字目が a で三文字目が b のファイルを変換します。

利用例

中間ファイルチェッカーの利用方法を説明します。

中間ファイルのチェック

check.fmdb をチェックします。

3dIntermediateFileChecker.exe check.fmdb

フォルダーのチェック

check フォルダーおよびサブフォルダー内の中間ファイルをチェックします。

3dIntermediateFileChecker.exe check\

チェック項目

中間ファイルチェッカーは以下の項目をチェックします。

  • XML スキーマが正しいか
  • ストリームの浮動小数点数の値に無限大、非数がないか
  • モデルの法線、接線、従法線の長さが 1 であるか
