マテリアル

マテリアルについて注意事項を説明します。

  • マテリアル名
  • シェーダの種類
  • 有効なアトリビュート

マテリアル名

中間ファイルには、シェーディンググループごとに 1 つのマテリアルが出力されます。

参考：

単一のマテリアルが複数のシェーディンググループに接続されている場合、最初に出力されるノードで使用されているマテリアルはそのままの名前で出力されますが、その他のマテリアルはマテリアル名の後ろに "_1"、"_2" のようにアンダーバーと数字が付加されます。

シェーダの種類

スペキュラや反射カラーを使用しない場合は Lambert シェーダを使用してください。

スペキュラや反射カラーを使用する場合は Blinn、Phong、Phong E、異方性（Anisotropic） シェーダのいずれかを使用してください。

参考：

異方性シェーダを使用すると、法線マッピングが設定されていなくても接線と従法線が出力されます（カラー用のテクスチャーがある場合は、最上位レイヤのテクスチャーの UV セットに対応した接線と従法線を出力します。
カラー用のテクスチャーもない場合は、現在の UV セットに対応した接線と従法線を出力します）。

有効なアトリビュート

次のアトリビュートが中間ファイルに反映されます。

また、マテリアルアトリビュート設定プラグインで、NintendoWare 用のアトリビュートを設定できます。

カラー（Color） ディフューズカラー
透明度（Transparency） 不透明度
アンビエントカラー（Ambient Color） アンビエントカラー
白熱光（Incandescence） エミッションカラー
スペキュラカラー（Specular Color） スペキュラーカラー

カラー、白熱光、スペキュラカラーアトリビュートに file ノードが接続されている場合

file ノードのカラーゲイン（Color Gain）アトリビュートが中間ファイルに反映されます。
file ノードのカラー オフセット（Color Offset）アトリビュートは反映されません。

カラー、白熱光、スペキュラカラーアトリビュートに layeredTexture ノードが接続されている場合

最上位レイヤに接続された file ノードのカラーゲインアトリビュートが中間ファイルに反映されます。

透明度アトリビュートに file ノードが接続されている場合

file ノードのアルファ値のゲイン（Alpha Gain）アトリビュートが不透明度に反映されます。
file ノードのアルファ値のオフセット（Alpha Offset）アトリビュートは反映されません。

参考：
拡散（Diffuse）アトリビュートは中間ファイルには反映されません。
Maya 上での明るさを対象プラットフォームでの表示に近づけたい場合は拡散アトリビュートを 1.0 にしておいてください。
