履歴ポリゴン化の間隔

ここでは記録された履歴の中からストライプの為に参照する履歴を決める機能「履歴ポリゴン化の間隔」について説明します。

概要

「履歴ポリゴン化の間隔」は、記録された履歴の中から、ストライプのポリゴン頂点を生成するさいに参照する履歴を決める機能です。
値を大きくとることによって、ストライプの形状は粗くなりますが、ストライプ生成に確保するメモリ領域を節約できます。
