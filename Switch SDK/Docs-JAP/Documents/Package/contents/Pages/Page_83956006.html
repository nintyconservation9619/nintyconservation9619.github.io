中間ファイルコンバーター

概要

一般的なフォーマットのファイルを中間ファイルに変換します。

DCC ツールを利用できない環境で、制限つきで中間ファイルを出力するために利用します。

あくまで制限つきで利用するためのツールであり、一般的なフォーマットを自在に変換できるコンバーターではありません。
一般的なフォーマットから本格的な中間ファイルを生成するには、一旦 DCC ツールに読み込んだ上でエクスポートしてください。

実行ファイル(3dIntermediateFileConverter.exe)やショートカットに、変換対象のファイルをドラッグ＆ドロップすると、中間ファイルに変換します。
変換対象ではないファイルは無視します。

問題があった場合は、エラーメッセージを表示してキー入力待ち状態になります。
問題が無かった場合は、そのまま終了します。

コマンドラインオプション

3dIntermediateFileConverter.exe [--args-file ""] [--version] [-h] [-s] [-j ""]
[-o ""] [--magnify ""] [--disable-invert-uv] [--texture-format-srgb] [--gpu-encoding ""] [ファイル]...

一般的な 3D データのファイルを中間ファイルに変換します。

--args-file ""

引数ファイルを指定します。
引数ファイルに記述した内容を、引数として指定できます。 
引数ファイルは複数行に渡って各オプションを記述できます。 
行頭に # がある行はコメントとして扱います。 
オプション値指定の "" によるエスケープに改行を含めることはできません。

--version

バージョン情報を表示します。

-h --help

ヘルプを表示します。

-s --silent

処理中のメッセージ表示を抑制します。

-j "" --jobs ""

実行ジョブ数を指定します。

ファイル

変換するファイルを指定します。
対応するファイル: fbx obj png bmp jpg jpeg tga fmd

-o "" --output ""

出力ファイル名を指定します。
このオプションを指定しない場合、拡張子を中間ファイルの拡張子に変更して同じフォルダーに保存します。

--magnify ""

出力ファイルの頂点座標と移動値への倍率を指定します。
対応するファイル: fbx obj fmd

--disable-invert-uv

UV 座標の上下を反転しないようにします。
UV 座標系が左上原点で下方向が正であることを想定して作成されている場合に使用します。
対応するファイル: fbx obj

--texture-format-srgb

テクスチャーを SRGB フォーマットでコンバートする場合に使用します。
対応するファイル: fbx obj png bmp jpg jpeg tga

--gpu-encoding


テクスチャーを GPU エンコーディングするかどうかを指定します。詳しくは 3D テクスチャーコンバーター の --gpu-encoding オプションの説明を参照してください。

対応するファイル: fbx obj png bmp jpg jpeg tga

制限事項

fbx ファイルの制限

  • モデル中間ファイルへの変換のみサポートしています(アニメーションはサポートしていません)
  • 動作確認している FBX は Maya から出力した FBX バージョン 2013 (7300) の FBX のみです
  • 三角分割されているポリゴンのみ変換することができます
  • スキニングは影響するボーン数が１つであっても、スムーススキニングとして扱われます。この制約は将来的に排除される予定です。
  • モデル中間ファイルの以下の属性について、DCC プラグインでは DCC ツール側の設定が反映されますが、中間ファイルコンバーターを利用した際は固定の値が設定されます
要素 属性名 設定される値
<bone> visibility true
scale_compensate false
type 空文字列
side none
compress_enable true
<material_info> compress_enable true

利用例

中間ファイルコンバーターの利用方法を説明します。

obj ファイルの変換

target.obj を target.fmdb に変換します。

3dIntermediateFileConverter.exe target.obj

png ファイルの変換

source.png を destination.ftxb に変換します。

3dIntermediateFileConverter.exe source.png -o destination.ftxb

fmd ファイルの変換

source.fmdb を2倍の大きさに拡大し destination.fmdb に変換します。

3dIntermediateFileConverter.exe source.fmdb --magnify "2.0" -o "destination.fmdb"


サンプルの説明

中間ファイルコンバーターのデモは Samples/Sources/Tools/3dTools/3dIntermediateFileConverter/ 以下にあります。

再帰的な obj ファイルの変換 (RecursiveObjConvert)

RecursiveObjConvert.bat に obj ファイルやフォルダーをドラッグ＆ドロップすると、fmdb に変換します。
