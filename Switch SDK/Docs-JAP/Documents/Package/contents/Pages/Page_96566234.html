継承設定「カラー」

ここではチャイルドエミッタの継承設定「カラー」について説明します。

概要

継承設定「カラー」は、上位ノードの親エミッタから「カラー」の値を継承できる機能です。
以下は「 設定項目」の概要です。

カラー

親エミッタからのカラー継承を「 チェックボックス」ON/OFF で切り替えます。
ONにした場合、親エミッタの「カラー」をパーティクル放出時のカラーに乗算します。
継承するカラーは、「カラー0」、「カラー1」各々に指定できます。

OFF 子エミッタのパーティクルカラーがそのまま反映されています。
