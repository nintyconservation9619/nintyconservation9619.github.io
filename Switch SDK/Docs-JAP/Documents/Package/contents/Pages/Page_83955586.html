プロパティウィンドウ 追加参照パスページ

選択したファイルに対する追加参照パスの設定や編集をするページです。

A 複数の追加参照パスがある場合に選択したパスを上下に移動します。
B 追加参照パスを新規に追加します。
追加ボタンを押すと追加参照パスの編集ウィンドウが表示されます。
C 選択した追加参照パスを削除します。
D 選択した追加参照パスを編集します。
編集ボタンを押すと追加参照パスの編集ウィンドウが表示されます。
E 設定されている追加参照パスが表示されます。
F 追加参照パスで設定されているフォルダ以下にあるサブフォルダの検索を行うか情報が表示されます。

追加参照パスの編集ウィンドウ

追加参照パスの追加や編集ができます。

A 参照フォルダが表示されます。
B

フォルダ参照ウィンドウから参照フォルダを選択できます。
C 参照フォルダの絶対パスが表示されます。
D パスの基点が表示されます。
E 参照フォルダ以下にあるサブフォルダも検索する場合はチェックを入れます。
F OKボタンを押すと参照フォルダが追加参照パスとして追加されます。
G 追加参照パスの編集をキャンセルします。
