LayoutEditor

LayoutEditor に新しく追加された機能について紹介します。

テキストボックスペインで行単位での幅調整ができるようになりました !!

LayoutEditor 上でテキストボックスペインの幅を行単位で編集して、より豊かな表現ができるようになりました !!

この機能は、新聞や雑誌の記事のような、画像に沿った文字列 ( 文章 ) のレイアウトを作りたい時などに、とても便利です !!
何行にもわたる長文の表示形態を、一つのペインの中で自在に変化させることができます。

 

[ プロパティウインドウ ] → [ テキストボックスペイン ] → [ 行ごとに幅を指定 ]

テキストボックスペインの各行ごとに、文字列の始まる箇所と終わる箇所を数値入力して、行の幅を指定することのできる機能です。

ペイン基本で設定したテキストボックスペインの X 方向のサイズ ( 幅 ) から、行の内側方向に向けて、自由に幅を設定できます。
左側 ( 行の始まる箇所 ) 設定、右側 ( 行の終わる箇所 ) 設定ともに、行の内側方向がマイナス方向となる数値入力となります。
プラスの数値入力 ( テキストボックスペインのサイズからはみ出す幅調整 ) は出来ません。

16 行までの設定が可能です。

行ごとに幅を指定　全 16 行 ( プロパティウインドウ ) 「行ごとに幅を指定」の数値が反映されたテキストボックスペイン
 有効 機能の ON / OFF スイッチ チェックボックスにチェックを入れると、機能が有効となります。
サイズ X : テキストボックスペインの X の大きさ

テキストボックスペインの X 方向のサイズを表示します。
プロパティウインドウの「ペイン基本」で設定した X 方向のサイズです。

「行単位での幅調整」で幅の調整をすると、各行のサイズが変化するので参考数値となります。

プロパティウィンドウのカラー補間のピッカー部分を RGB と A で分離してドラッグ出来るようになりました !!

LayoutEditor プロパティウインドウのカラーボタンを RGB と A で分離してドラッグできるようになりました !!

 

プロパティウインドウの、カラーボタンの左側をドラッグすると RGB、右側をドラッグすると A のカラーを複製します。

カラー補間設定 ( プロパティウインドウ )
  • RGB と A の値を別々にドラッグできます。
  • 左が RGB 、右が A です。

グラフィックス中間ファイルバージョンの更新

グラフィックス中間ファイルのバージョンを 3.7.0 に更新しました。
以下の場所にあるプログラマ向けサンプルのテクスチャファイルを使用するさいは、ファイルのアップデートが必要です。 

Samples\Sources\Resources\Ui2d\Common\Textures\SampleTexture.ftxb

以下の中間ファイルアップデータを利用することで、既存の中間ファイルのバージョンを更新できます。

Tools\Graphics\GraphicsTools\IntermediateFileUpdater.exe
