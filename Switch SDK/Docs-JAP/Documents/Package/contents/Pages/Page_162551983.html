3DEditor

3DEditor に新しく追加された機能について紹介します。

モデルのマージ機能を追加

過去に廃止されたモデルのマージ機能が改めて追加されました。
選択したモデルに他のモデルファイルの設定を反映させることができるようになりました。

3DEditor のチーム設定ファイル

チーム設定ファイルの機能を追加しました。

バイナリ化されないボーンビジビリティアニメーションのカーブをカーブエディタに非表示にする機能を追加

バイナリ化されないボーンビジビリティアニメーションのカーブを非表示にできる機能を追加しました。

チーム設定の <ShowOnlyBinarizeEnabledBoneVisibilityAnimCurve> を有効にすることで、非表示にすることができます。

Maya エクスポートプラグインのコンフィグファイルで export_bone_visibility_all を false に設定している場合、Maya でキーを打っていないボーンビジビリティアニメーションが 3DEditor のカーブエディタに表示されなくなります。

チーム関連ファイルフォーマット

チーム設定ファイルに含まれる要素を説明します。

<ShowOnlyBinarizeEnabledBoneVisibilityAnimCurve> の項目をご覧ください。

接続するプラットフォーム毎の接続設定をチーム設定に追加

接続先プラットフォームの接続設定をカスタマイズできるようになりました。

接続先アプリケーションのプラットフォームを特定のものに限定したり、3D シェーダコンバータやテクスチャコンバータの追加引数をプラットフォーム毎に変更できるようになりました。

チーム設定ファイルに含まれる要素を説明します。

接続先アプリケーションに送信する中間ファイルに対して送信前に任意のコマンドを呼び出す機能を追加

接続先アプリケーションに送信する中間ファイルに対して中間ファイルフィルタを適用するなど、任意の処理を挟めるようになりました。

関連リンク 以下は関連ページへのリンクです。

チーム関連ファイルフォーマット

チーム設定ファイルに含まれる要素を説明します。

<PreBinarizeCommand> の項目をご覧ください。

シェーダー最適化時のシェーダーバリエーション作成をスキップする機能を追加

シェーダー最適化時に特定のマテリアルについてシェーダーバリエーション作成をスキップし、接続先アプリケーションで保有するシェーダーアーカイブを割り当てられるようになりました。

これにより、シェーダー最適化時の 3DEditor のシェーダーコンバート時間を短縮することができます。

チーム設定ファイルに含まれる要素を説明します。

ユーザー設定やアプリケーション設定をチーム設定毎に切り替えられるように

これまで、ユーザー設定やアプリケーション設定はマシン単位で共通の設定が使用されていましたが、チーム設定単位で切り替えられるようになりました。

チーム設定の<SettingName>で設定名を指定することで、ユーザー設定やアプリケーション設定が設定名に基づいて作成されます。
