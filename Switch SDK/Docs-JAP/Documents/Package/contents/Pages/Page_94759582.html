ランダムとノイズ

「ランダム」と「ノイズ」の違いについて、フィールド「ランダム」と「GPUノイズ」を例に説明します。

「ランダム」と「ノイズ」は、ともにパーティクルの動きや設定項目の値に対してパターンのないような挙動にできる機能です。
「ランダム」は秩序がありませんが、「ノイズ」は法則性があります。
以下はパーティクルの動きに対してランダムな手法を用いた「ランダム」とノイズの手法を用いた「GPUノイズ」特徴をまとめた表です。
