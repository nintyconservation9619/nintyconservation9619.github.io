カーブエディタに項目の文字列フィルタ機能追加

カーブエディタの編集項目を検索できるようになりました。

カーブエディタのキー移動/スケールツール改善

カーブエディタのキー移動/スケールツールがより使いやすくなりました。

 

シェーダパラメータのサブグループ機能追加

マテリアルプロパティウィンドウのシェーダグループにサブグループを作れるようになりました。

サブグループを作る場合、シェーダアノテーションでは下記のように group に対して別のグループを group_id で指定します。

// @@ group="group_lighting" label="ライティング"
// @@ group="group_hemisphere_light" label="半球ライト"
// @@ group="group_directional_light" label="指向性ライト"
/* @@ group_lighting:
group_id="group_hemisphere_light"
group_id="group_directional_light"
*/

テクスチャをシェーダのテクスチャ選択UIに直接ドロップで登録可能に

マテリアルプロパティウィンドウのシェーダのテクスチャ選択UIに直接テクスチャをドロップ登録することができるようになりました。
