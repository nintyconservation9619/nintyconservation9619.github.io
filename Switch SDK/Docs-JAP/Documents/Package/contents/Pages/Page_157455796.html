はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている LDN (Local Directional Network) ライブラリの機能について説明します。

対象読者

NintendoSDK を使って、アプリケーションのローカル通信機能を開発する方を対象とします。

関連資料

LDN の関数リファレンスと併せて参照してください。

LDN ライブラリの機能

LDN ライブラリは、中継器やインターネットを介さずに複数の端末を接続し、それぞれの端末上で動作するアプリケーション間で通信する機能を提供します。

ただし、LDN ライブラリが提供する機能はネットワークの構築と探索・接続に限定されており、構築されたネットワーク上で実際に通信を行うためには SOCKET ライブラリを使用する必要があります。
