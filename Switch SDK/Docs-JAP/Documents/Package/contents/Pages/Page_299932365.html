シザーペイン

ここでは「親子階層」ウィンドウで階層下にあるペインの描画対象を矩形で切り抜く「シザーペイン」について説明します。

概要

「 シザーペイン」は、階層下にあるペインを描画対象として、矩形で切り抜くペインです。
矩形範囲外の描画処理をスキップし、負荷を軽減できます。

以下は「シザーペイン」の設定項目です。

ペイン基本 ペインの名前、位置、形状、中心位置などを設定します。
拡張ユーザ情報 ペインにユーザー独自の拡張情報を設定します。

関連リンク 以下は関連ページへのリンクです。

マスク

ペインのプロパティ「 マスク」に登録したマスク画像によりペインを切り抜く機能「マスク」について説明します。

切り抜き形状をマスク画像で制御できます。
