パーティクル放出最大数

ここでは放出されるパーティクル数に制限をかけます。

概要

「パーティクル放出最大数」は、表示されるパーティクル数を設定した値以下に制限する機能です。
パーティクル寿命が長いなどの原因で、同時の表示されるパーティクル個数が多くなり過ぎた結果必要以上に処理負荷がかかることを防止できます。

以下は「 設定項目」の概要です。

パーティクル放出最大数

「パーティクル放出最大数」の閾値を「 ダイヤル」や「 テキストボックス」により整数値で設定します。
