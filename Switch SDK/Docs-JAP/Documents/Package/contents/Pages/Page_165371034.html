「システム分類」の概要

各関数は、それらの関数が所属するシステム（物理、グラフィック、音声など）に応じて自動的に分類されます。この分類は、主に各システムで共通に受け入れられる用語を使用した文字列一致によって実現されます（例えば、物理は "broadphase"、演算は "matrix" など）。関数によっては誤って分類されることも十分に考えられますが、ほとんどの場合は正確であるため、理解のための大きな助けになります。

選択された関数は、それらの関数に対して選択されたシステム分類に基づいてカラー表示されます。このカラー表示は、「関数」タブ、「コールツリー」タブ、および「サンプルグラフ」タブに反映されます。各色の意味については、「サンプルグラフ」タブの各コアの上に表示されます。

「関数」タブのツールバーにある「システム分類」ドロップダウンボックスを使用して、システム分類を基準に関数をフィルタ処理することができます。
