Webアプレット・オフラインWebアプレット

Webアプレット・オフラインWebアプレットは、アプリケーションが指定したWebページを、主要なHTML5の機能や動画再生機能を備えたWebブラウザーを用いて表示するライブラリアプレットです。

Webアプレットは、指定されたインターネット上の URL にあるWebページを表示するライブラリアプレットです。
オフラインWebアプレットは、アプリケーションに付加されたオフラインHTMLを表示するライブラリアプレットです。

本ドキュメントの目的

本ドキュメントでは、Webアプレット・オフラインWebアプレットの起動と終了の手順、主な機能やスペック、アプリケーションに合わせたカスタマイズの方法、使用例などを説明します。
また、両アプレットに共通した情報と各アプレットの情報に分けて説明します。
