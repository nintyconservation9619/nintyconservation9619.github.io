中間ファイルアップデーター

中間ファイルアップデーターの使用方法と注意事項を説明します。

  • 概要
  • コマンドラインオプション
  • 利用例
  • ユーザーツールデータの更新

概要

中間ファイルのバージョンを最新にします。

古い中間ファイルはバイナリファイルに変換できませんので、本ツールで中間ファイルのバージョンを最新に保ってください。
ただし、DCC ツールからの再出力、3DEditor での編集、中間ファイルのマージ等によっても中間ファイルを更新しますので、ワークフローによっては利用する必要が無い場合もあります。

実行ファイル (3dIntermediateFileUpdater.exe) やショートカットに、中間ファイルやフォルダをドラッグ＆ドロップすると、中間ファイルを更新して上書き保存をします。

ただし、更新項目が無い場合は上書き保存をしません。
中間ファイル以外のファイルは無視します。

中間ファイルの上書きが問題となる場合は、中間ファイルをコピーした後に利用してください。

各バージョンにおけるアップデーターの動作については、中間ファイル仕様書の更新履歴を確認してください。

コマンドラインオプション

3dIntermediateFileUpdater.exe [--args-file ""] [--version] [-h] [-s] [-j ""]
[--enable-extension ""] [--disable-extension ""] [--disable-file-info] [中間ファイル|フォルダ]... 

中間ファイルのバージョンを最新にします。

--args-file ""

引数ファイルを指定します。
引数ファイルに記述した内容を、引数として指定できます。 
引数ファイルは複数行に渡って各オプションを記述できます。 
行頭に # がある行はコメントとして扱います。 
オプション値指定の "" によるエスケープに改行を含めることはできません。

--version

バージョン情報を表示します。

-h --help 

ヘルプを表示します。

-s --silent

処理中のメッセージ表示を抑制します。

-j "" --jobs ""

実行ジョブ数を指定します。

--enable-extension ""

中間ファイルの中で、処理する拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--enable-extension "fmda,ftxb"

--disable-extension ""

中間ファイルの中で、処理しない拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--disable-extension "fmda,ftxb"

--disable-file-info

中間ファイルを更新する際に <file_info> を更新しません。 

中間ファイル|フォルダー

処理するファイルやフォルダーを指定します。

フォルダーを指定した場合、指定フォルダー以下のファイルを再帰的に処理します。
中間ファイル以外は無視します。

ワイルドカード（?: 一文字、*: 複数文字）に対応しています。
例) "a?b*"
一文字目が a で三文字目が b のファイルを変換します。

利用例

中間ファイルアップデーターの利用方法を説明します。

中間ファイルの更新

old_version.fmdb を最新版に更新します。

3dIntermediateFileUpdater.exe old_version.fmdb

フォルダーの更新

old_version フォルダーやサブフォルダー内の中間ファイルを最新版に更新します。

3dIntermediateFileUpdater.exe old_version\

ユーザーツールデータの更新

ユーザーツールデータの更新といったユーザー独自の更新処理は中間ファイルアップデーターで更新することは出来ません。
これらを更新するためには 中間ファイルフィルター を利用して下さい。
