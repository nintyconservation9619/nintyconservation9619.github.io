4 カラーアニメーション

カラーアニメーションを作成します。

カラー

マテリアル「ColorMt」の共通マテリアルアトリビュートの「カラー」にアニメーションキーを作成します。

下表のようにキーを作成します。

 
0 フレーム
25 フレーム
 50 フレーム
カラー 黒 赤 黒

下図のような赤色が点滅するアニメーションになることを確認します。

出力対象

カラーアニメーションの出力対象はマテリアルの「スペキュラーシェーディング」のキーフレームです。

注意：

マテリアルのカラーなどのアニメーションは定義するシェーダーによって適用されるカーブが変わります。

demo.fsdb ではカラーを使用しますが、エクスポートプラグインは以下のカーブに対応しています。

  • カラー（テクスチャーノードが接続されている場合はカラーゲイン）
  • 透明度（テクスチャーノードが接続されている場合はアルファ値のゲイン）
  • アンビエントカラー
  • 白熱光
  • スペキュラーカラー

 

「Nintendo Export Options」の「Output File Selection」の B「Color Animation Data」に対応しています。

関連リンク 詳しくは カラーアニメーション を参照してください。

参考：

カラー、テクスチャー SRT、テクスチャーパターンの各アニメーションを .fmab 形式で一つにまとめて出力します。
これらのアニメーションはエクスポートプラグインのコンフィグの指定によって別々に出力することも出来ます。

関連リンク 詳しくは エクスポートプラグイン使用方法 を参照して下さい。
