親マテリアルをチーム設定で指定したフォルダの中のマテリアルから一覧で選択できるように

チーム設定で<ParentMaterialPath>を指定することにより、指定フォルダ内に含まれるモデル中間ファイルのマテリアル一覧から親マテリアルを設定できるようになりました。

3dEditor.teamsettings に下の例のようにパスを追記すると、 3DEditor を起動するだけで親マテリアルを選択することができるようになりました。

<TeamSettings>
    <FileIo>
        <ParentMaterialPath Recursive="true">%MY_PROJECT%\ParentMaterials</ParentMaterialPath>
        ...
    </FileIo>
    ...
</TeamSettings>


注意：

同じシェーダー定義とシェーディングモデルが割り当てられているモデル中間ファイルのマテリアルのみ一覧に表示されます。
