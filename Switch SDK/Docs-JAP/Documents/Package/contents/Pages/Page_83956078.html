レンダーターゲットステート

レンダーターゲットステートは、カラーターゲットと深度ステンシルターゲットに関するステートをまとめて表現します。

情報

レンダーターゲットステートの情報は nn::gfx::RenderTargetStateInfo として定義されています。

深度ステンシルフォーマット

SetDepthStencilFormat により、深度ステンシルのフォーマットを設定します。
このフォーマットは、実際の深度ステンシルのフォーマットと一致している必要があります。

カラーターゲットステート

SetColorTargetStateInfoArray によりカラーターゲットステートの配列を設定します。

カラーターゲットステートの配列は、それぞれ順番にカラーターゲットに対応します。

カラーターゲットフォーマット

SetFormat により、カラーターゲットのフォーマットを設定します。
このフォーマットは、実際のカラーターゲットのフォーマットと一致している必要があります。 

レンダーターゲットステートのセット

パイプラインを使う実行モデルを利用する場合は、パイプラインの情報にレンダーターゲットステートの情報を記述します。

パイプラインを使わない実行モデルについて、レンダーターゲットステートをセットする必要はありません。
