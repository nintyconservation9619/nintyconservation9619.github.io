プロパティウィンドウ

プロパティウィンドウは、3DEditor に読み込んだオブジェクトを編集するウィンドウです。
編集項目は、選択したオブジェクトにより異なります。 

プロパティウィンドウの種類

3DEditor で用意されているプロパティウィンドウは以下になります。

プロパティウィンドウ 名 説明
プロジェクトプロパティウィンドウ

プロジェクトのパス情報や、作成に関する記録が確認できます。
モデルプロパティウィンドウ

選択したモデルのプロパティを確認、編集するウィンドウです。
ボーンプロパティウィンドウ

選択したボーンのプロパティを確認、編集するウィンドウです。
シェイププロパティウィンドウ

選択したシェイプのプロパティを確認、編集するウィンドウです。
マテリアルプロパティウィンドウ

選択したマテリアルのプロパティを確認、編集するウィンドウです。
テクスチャプロパティウィンドウ

選択したテクスチャのプロパティを確認、編集するウィンドウです。

アニメーションプロパティウィンドウ

選択したアニメーションのプロパティを確認、編集するウィンドウです。（アニメーションの種類により、編集項目が変化します）
シェーダ定義プロパティウィンドウ

選択したシェーダ定義ファイルを確認、編集するウィンドウです。

プロパティウィンドウの基本機能

各プロパティウィンドウの基本的な機能は以下になります。

A 対象オブジェクトの選択状態を固定します。
選択状態を固定すると、他のオブジェクトのプロパティウィンドウが別ウィンドウで開きます。
B
設定しているオブジェクトの名前を表示します。
同名のオブジェクトをプロパティウィンドウで区別できるようにオブジェクト名の後に、使用しているモデル名を表示します。
例：マテリアル名 - モデル名のようになります。
  
C

アクティブなプロパティウィンドウの選択しているプロパティページ以下の設定を子階層を含めてコピーします。
D

コピーしたプロパティページの設定を貼り付けます。

複数の項目が選択されているときは選択している各項目に貼り付けが実行されます。
E
マテリアル設定ファイルをエクスポートします。
関連リンク この操作は、マテリアルプロパティウィンドウで有効になります。
F
マテリアル設定ファイルをインポートします。
関連リンク この操作は、マテリアルプロパティウィンドウで有効になります。
G 選択しているプロパティウィンドウの複製を、別ウィンドウで開きます。
H ヘルプファイルを開きます。
I プロパティウィンドウのツリーリストです。
対象オブジェクトの分類分けしたプロパティページを選択します。
J 作業用カラーを編集します。
K 作業用ラベルを編集します。
L 作業用コメントを編集します。
M 選択したプロパティページの内容を確認し、編集します。
N
データが編集してあり、セーブしていない場合は、項目に赤い★マーク ( 更新マーク ) が付きます。
関連リンク ファイルビュー上でも、データが編集された項目には、同じ ★マークが付きます。
