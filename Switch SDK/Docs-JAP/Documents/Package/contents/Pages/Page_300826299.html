ノードのコピー＆ペースト

ノードのコピー＆ペーストについて説明します。

子ノードとして貼り付ける

コピー元のノードを選択し、右クリックコンテキストメニューから「コピー」を実行します。
コピー先のノードを選択し、右クリックコンテキストメニューから「子ノードとして貼り付け」を実行します。
コピー先のノードの子ノードとして、コピー元のノードツリーが貼り付けられます。

ノードを貼り付ける

コピー元のノードを選択し、右クリックコンテキストメニューから「コピー」を実行します。
コピー先のノードを選択し、右クリックコンテキストメニューから「ノードを貼り付け」を実行します。
コピー先のノードに、コピー元のノードツリーが貼り付けられます。

プロパティを貼り付ける

コピー元のノードを選択し、右クリックコンテキストメニューから「コピー」を実行します。
コピー先のノードを選択し、右クリックコンテキストメニューから「プロパティを貼り付け」を実行します。
コピー先のノードに、コピー元のノードプロパティが貼り付けられます。
子ノードは変更されません。
