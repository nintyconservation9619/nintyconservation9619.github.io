パターンアニメーション用テクスチャを作る

ここでは Photoshop プラグイン「画像結合プラグイン」を利用した、パターンアニメーション用テクスチャ（フリップブックテクスチャ）の作成手順について炎のパターンを例に説明します。
このプラグインは、個々に分割された複数のイメージから構成された 1 枚のテクスチャにまとめます。
分割されたイメージは、「テクスチャパターンアニメ」でテクスチャ UV のステップ移動によりアニメーションできます。

テクスチャ作成の準備

あらかじめ Photoshopプラグインがインストールされた Photoshop を起動した状態から、テクスチャ作成にあたっての準備を行います。

関連リンク Photoshop プラグインのセットアップ方法については Photoshopプラグインの 「はじめに」 を参照してください。

「画像統合プラグイン」を開く

サンプルで用意した画像ファイルを「画像結合プラグイン」で結合します。
Photoshop メインメニュー項目 [ファイル]→[ 自動処理]→ [Nintendo Unite Images...] を選択し、「画像結合プラグイン」を開きます。

画像結合プラグインを設定する

結合する画像をまとめたフォルダを設定します。

項目名  設定パス(結合する画像データの場所) 説明
Input Folder
 		 \Samples\Sources\Resources\Vfx \Eset\Textures\UniteImages\ 結合する画像をまとめたフォルダを指定します。

画像の縦横サイズ、結合する画像の数などを設定し、「OK」ボタンを押して画像を結合します。

項目名  値 説明
Number of Pixels in an Image

Width : 128
Same Value : チェックオン

結合する画像の横幅と縦幅を 128 に設定します。
Number of Images in a Row (in a Column)

Horizonal : 8
Vertical : 4
Same Value : チェックオフ

1 行に配置する画像の数を 8 、行数を 4 に設定します。
Process  Unite Alpha : チェックオフ  結合元の画像にアルファがない場合は、チェックをオフにします。
