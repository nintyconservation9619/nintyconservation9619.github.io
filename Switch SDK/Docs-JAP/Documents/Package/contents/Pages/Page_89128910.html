機能説明

ここでは、HID ライブラリが提供する各機能を、API の使用方法と合わせて説明します。

機能一覧

機能 概要
DebugPad デバッグコントローラ操作を実現するための機能です。
Gesture ジェスチャ操作を実現するための機能です。
Keyboard キーボード操作を実現するための機能です。
Mouse マウス操作を実現するための機能です。
Npad 種々の操作形態をプラットフォームで定義されたコントローラーモデルにならって制御するための機能です。
SixAxisSensor 6軸センサーの操作を実現するための機能です。
TouchScreen タッチスクリーン操作を実現するための機能です。
Xpad

Xpad は非サポートです。Npad への移行をお願いします。

※ 将来の SDK にて削除予定です
振動 振動子を操作するための機能です。
