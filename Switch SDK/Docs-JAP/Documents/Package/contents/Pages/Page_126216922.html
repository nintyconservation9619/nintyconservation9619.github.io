アニメーション

概要

NintendoWare 3D のアニメーションの機能について説明します。アニメーションの作成方法については、各プラグインおよび 3DEditor の項目を参照してください。

アニメーションの種類

g3d では以下のアニメーションをサポートしています。

スケルタルアニメーション

骨格 ( ボーン ) を階層的に連結したモデルであるスケルトンを定義し、それを所定の姿勢 ( キーフレーム ) へとアニメーションさせ、それに従いキャラクターの構成頂点の位置を追従させるアニメーションです。

マテリアルアニメーション

マテリアルに適用されるアニメーションです。複数のマテリアルに適用するアニメーションを１つのアニメーションデータに持つことが可能です。マテリアル毎に適用されるアニメーションの中にはシェーダーパラメーターアニメーション、テクスチャパターンアニメーション、マテリアルビジビリティアニメーションを含めることが可能です。シェーダーパラメーターアニメーションおよびテクスチャパターンアニメーションにおいては１つのマテリアルに対して、複数アニメーションを１つのアニメーションデータに含めることが可能です。

シェーダーパラメーターアニメーション

シェーダーが参照するパラメーターを変化させるアニメーションです。マテリアルの反射率を変化させたり、テクスチャのUVアニメーションに使用することが出来ます。

テクスチャパターンアニメーション

モデルが参照するテクスチャを変化させるアニメーションです。

マテリアルビジビリティアニメーション

マテリアルの表示、非表示を切り替えるアニメーションです。

シェイプアニメーション

複数のキーシェイプ ( 基本形状 ) をブレンドするアニメーションです。g3d ではキーシェイプをブレンドするためのウェイト ( 重み ) 値の計算のみをサポートしています。そのウェイトを使用し、ブレンドする処理はアプリケーションで実装する必要があります。

ボーンビジビリティアニメーション

ボーンの表示、非表示を切り替えるアニメーションです。１つのボーンに関連付けられた複数のシェイプを同時に表示、非表示を切り替えることが可能です。

シーンアニメーション

カメラアニメーション、ライトアニメーション、フォグアニメーションの 3 つがあります。g3d はアニメーションの計算のみをサポートしています。アニメーションの結果をカメラ、ライト、フォグに反映される処理はアプリケーションで実装する必要があります。

カメラアニメーション

カメラのアニメーションです。カメラ種別は投影モードと回転モードから選択できます。投影モードは「透視射影」、「正射影」、回転モードは「注視点と捻り」、「ZXY軸回転」から選択できます。

ライトアニメーション

ライトのアニメーションです。ライト種別は環境光、ディレクショナルライト、スポットライト、ポイントライトから選択できます。

フォグアニメーション

フォグのアニメーションです。距離減衰、フォグカラーなどがアニメーションできます。

リターゲティング

リターゲティングとは、ある特定のモデル用に作成されたスケルタルアニメーションを別のモデルに適用することを可能にする機能です。
下図の「リターゲティングなし」の図のように通常、移動成分を持つスケルタルアニメーションはモデル間のスケールの違いが無視された状態で適用されてしまいます。
リターゲティングを有効にすることでモデル間のスケールの違いを考慮し、スケルタルアニメーションの移動成分が補正された上でアニメーションが適用されます。

制約

リターゲティング機能を使用するためにモデルが以下の条件を満たす必要があります。
ここではスケルタルアニメーションが作成されたモデルをオリジナルモデル、オリジナルモデル用に作成されたスケルタルアニメーションを適用しようとするモデルをターゲットモデルと呼ぶことにします。

  • オリジナルモデルとターゲットモデルはボーン構成が同じでなければなりません。
    但し、ターゲットモデルがオリジナルモデルにないボーンを持つことは出来ます。
  • オリジナルモデルとターゲットモデルはボーン名が同一でなければなりません。
  • オリジナルモデル及びターゲットモデルのバインドポーズ時に、同じ座標に2つ以上のボーンが配置されている場合、該当するボーンにはリターゲティングが適用されません。
    リターゲティングを利用する場合、アニメーション対象の複数ボーンを同じ座標に配置しないようにしてください。

使用方法

リターゲティングを使用するためには nn::g3d::SkeletalAnimObj のバインド処理時にリターゲティング情報を生成する必要があります。
リターゲティング情報を生成する nn::g3d::SkeletalAnimObj::Bind 関数を呼び出すようにしてください。

リターゲティングは 3DEditor でプレビューすることができます。詳しくは 3DEditor の アニメーションプロパティウィンドウ を参照してください。
