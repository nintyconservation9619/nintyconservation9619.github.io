NintendoSDK 0.14.0 で追加された機能をグラフィカルに紹介します。

マテリアルアニメーションに追加参照パスを設定可能に

3DEditor やアサインユーティリティにおいて、モデルの追加参照パスと同じようにマテリアルアニメーションにも追加参照パスを設定できるようになりました。

関連リンク 以下は関連ページへのリンクです。

3DEditor 3D で扱う中間ファイルを編集する UI ツールです。モデルやアニメーション中間ファイルの編集ができます。
アサインユーティリティ
3D で扱う中間ファイルに各種データの割り当てを行うコマンドラインツールです。
追加参照パス

ファイル読み込み時などに、そのファイルが参照するファイルの探索範囲を追加できます。
追加参照パスページ

3DEditor のプロパティウィンドウで、モデルやマテリアルアニメーションに対して追加参照パスを設定します。

Photoshop CC 2015.5 に対応

Photoshop プラグインが Photoshop CC 2015.5 に対応しました。

中間ファイルの src_path に出力されるパスを相対パス指定可能に

中間ファイルの <create> 要素の src_path 属性に特定のフォルダからの相対パスを出力できるようになりました。
相対パスで出力するには各ツールで設定を行う必要があります。

ツール 指定方法
Maya プラグイン

コンフィグファイルの project_root で相対パスのルートとなるフォルダを指定します。
Photoshop プラグイン コンフィグファイルの project_root で相対パスのルートとなるフォルダを指定します。
中間ファイルコンバータ --project-root オプションで相対パスのルートとなるフォルダを指定します。
テクスチャコンバータ --project-root オプションで相対パスのルートとなるフォルダを指定します。
参考：

使用例として、テクスチャ中間ファイル（.ftx）がどのフォトショップファイル（.psd）から作られているかを、ルートとなるフォルダからの相対パスから特定できるようになります。
