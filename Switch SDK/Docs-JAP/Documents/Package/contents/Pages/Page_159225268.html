FontConverter を使用したサンプルデータの一覧です。

ビットマップフォント
IconFont_16.bffnt
IconFont_16.ffnt

FontConverter(GUI版)を使用して作成したオリジナルビットマップフォントです。

フォントデータは以下の場所にあります。
 

「プラットフォーム用ビットマップフォント」
\Samples\Sources\Resources\Ui2d\Common\Fonts\（プラットフォーム名）\IconFont_16.bffnt

 

 「プラットフォームに依存しないビットマップフォント」

 \Samples\Sources\Resources\Ui2d\Common\Fonts\IconFont_16.ffnt
関連リンク  フォントファイル ffnt については「対象プラットフォームに依存しないビットマップフォント」を参照してください。

 

アウトラインフォント
IconFont-Regular.otf
IconFont-Regular.ttf

AdobeIllustrator と Fontgrapher を使用して作成したオリジナルアウトラインフォントです。

フォントデータは以下の場所にあります。

「OpenType フォント」

\Samples\Sources\Resources\Font\IconFont-Regular.otf


「TrueType フォント」
\Samples\Sources\Resources\Font\IconFont-Regular.ttf
