プロパティの編集

ペインを選択して、 [ 右クリックメニュー ] → [ プロパティ ] とします。
画面にプロパティウィンドウが表示されます。
（プロパティウインドウの開き方は他にも複数存在します）

プロパティウィンドウでは、選択状態のペインに対して様々なパラメータを編集することが可能です。
「ペインの位置、形状」のパラメータを入力して、ペインの変化を確認して下さい。
LayoutViewer でプレビューし、最終的な表示結果の変化を確認しましょう。

  •  初期状態
    （ 128 × 128 の大きさのピクチャペインを座標 0,0,0 に配置しています。）

  • 位置、サイズを編集

  • 位置、スケール、回転を編集
    サイズとスケールの違いに注意しましょう。

操作のやり直し

LayoutEditor ではアンドゥ / リドゥに対応しています（一部を例外を除きます）。
誤った設定を入力してしまった場合の取り消しや、取り消しのキャンセルができます。

操作の取り消し（アンドゥ）

レイアウトウィンドウをアクティブにして、以下いずれかを実行すると操作が取り消され、操作以前の状態に戻ります。
連続で操作の取り消しを実行することで、操作をさかのぼることができます。

  • メニューから取り消し
    [ メニュー ] → [ 編集 ] → [ 取り消し ] で操作を取り消します。
  • ショートカットで取り消し
    ショートカット [ Ctrl ＋ Z ] で操作を取り消します。

操作のやり直し（リドゥ）

レイアウトウィンドウをアクティブにして、以下いずれかを実行すると、取り消した操作をやり直します。
取り消した操作のやり直しを連続で実行していくこともできます。

  • メニューからやり直し
    [ メニュー ] → [ 編集 ] → [ やり直し ] で操作をやり直します。
  • ショートカットでやり直し
    ショートカット [ Ctrl ＋ Shift ＋ Z ] で操作をやり直します。

ファイル保存

任意のファイル名を指定してファイルを保存します。
レイアウトファイルの拡張子は flyt, flan です。

すべてを保存

以下いずれかの操作を行うと、作成したレイアウトデータのすべてを保存します。
初回の保存時には、保存するファイル名を入力するダイアログが表示されます。

  • メニューからすべてを保存
    [ メニュー ] → [ ファイル ] → [ 保存 ] → [ すべて ]
  • ショートカットですべてを保存
    ショートカット [ Ctrl ＋ S ]

別名で保存

以下いずれかの操作を行うと、作成したレイアウトデータを別名保存します。
同じ名前のデータがある場合、上書きするかの確認ダイアログが開きます。

  • メニューからすべてを保存
    [ メニュー ] → [ ファイル ] → [ 別名で保存 ] → [ すべて ]
  • ショートカットですべてを保存
    ショートカット [ Ctrl ＋Shift ＋ S ]
