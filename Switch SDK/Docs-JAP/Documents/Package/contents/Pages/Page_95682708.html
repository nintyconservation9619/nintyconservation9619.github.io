適用するUVセット

ここではパーティクル形状が「プリミティブ」の時、登録テクスチャの UV セットを選択できる機能「適用するUVセット」について説明します。

概要

「適用するUVセット」は、パーティクル形状として登録した、プリミティブ用モデルが保持する UV セットを変更できる機能です。
テクスチャごとに対応させる UV セットを選択し、パーティクルのシェーダで利用できます。
「パーティクル」タブの「パーティクル形状」でプリセットから「プリミティブ」を選択することで表示される設定項目です。
以下は「 設定項目」の概要です。

適用するUVセット

登録したモデルファイルの UV セット中、２ つまで表示され、その中から 1 つを選択できます。
関連リンク パーティクル形状「プリミティブ」については 「プリミティブ」 を参照してください。

モデルに 複数の UVセットを持たせるにはDCCツールでの設定が必要です。
