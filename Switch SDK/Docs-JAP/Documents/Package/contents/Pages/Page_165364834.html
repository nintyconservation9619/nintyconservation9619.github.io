右側ウィンドウのグループの概要

右側ウィンドウには以下のタブがあります。

  • 「情報」：プロファイルの概要と情報。
  • 「チェッカー」：プロファイルおよびアセンブリコードで実行された静的チェックと動的チェック。
  • 「コールツリー」：関数コールツリー。プロファイルにコールスタックの情報が存在する場合にのみ表示されます。
  • 「サンプルグラフ」：メインのグラフ表示ウィンドウ。
  • 「コードカバレッジ」：プロファイリングで検出された関数と検出されなかった関数を示します。
  • 「アセンブリ」：1 つの関数でアセンブリ行がヒットした回数を示します。
