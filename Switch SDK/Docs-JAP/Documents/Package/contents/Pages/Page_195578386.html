編集

レイアウトエディタウインドウのメニューバー「編集」の項目について説明します。
取り消し 編集を取り消します。
やり直し 取り消した編集をやり直します。
切り取り 選択したペインを切り取ります。
コピー 選択したペインをコピーします。
貼り付け コピーしたペインを貼り付けます。
削除 選択したペインを削除します。
アニメーションのコピー・貼り付けを行う チェックが入っているとペインの「コピー・貼り付け」をする際、アニメーション情報も一緒に「コピー・貼り付け」されます。
ワークスペースの記憶 サブウインドウの配置など現在のワークスペースの状態を記憶します。
ワークスペースの復元 記憶したワークスペースを復元します。
ワークスペースを初期状態に戻す 記憶したワークスペースを初期状態に戻します。
ツール環境設定 「ツール環境設定ウインドウ」が開きます。
