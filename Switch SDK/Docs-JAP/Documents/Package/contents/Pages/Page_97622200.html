スケール

ここではエミッタのスケールに対してオフセットを設定できる機能「スケール」について説明します。

概要

「スケール」は、エミッタのスケールに対するオフセットをワールド座標系で設定します。
エミッタ及びパーティクルサイズやスピードも設定値の影響を受ける為、エフェクト全体のサイズ変更に使用できます。
以下はスケールがかかる対象をまとめた表です。

スケール

エミッタのスケールに対するオフセットを「 ダイヤル」や「 テキストボックス」により設定します。
