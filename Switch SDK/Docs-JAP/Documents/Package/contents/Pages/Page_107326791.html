テクスチャの「ファイルパスを再設定」で別名ファイルも指定できるように改善

すでに LayoutEditor で使用されているテクスチャを別名のテクスチャに入れ替えることが出来るようになりました！

テクスチャウインドウ内の変更したいテクスチャを選択状態にして「 ファイルパスの再設定」を選択すると、再設定するテクスチャの選択ウインドウが開きます。
テクスチャサイズの変更があった場合でも、ピクチャペインのサイズは変更されません。

以下はテクスチャウインドウの画像です。

使用例

ピクチャペインの作業中に、違う名前のテクスチャに変更する事になり「 ファイルパスの再設定」で入れ替える、というような使い方が考えられます。

以前までは同じ名前のテクスチャしか再設定できませんでした。
名前の制約が無くなり、より便利になりました。

ピクチャペインのサイズをテクスチャ側のサイズに戻す機能を追加

サイズ変更したピクチャペインをテクスチャ本来のサイズに戻すことができる機能が追加されました！

メインメニュー項目 [ ペイン ] → [ テクスチャのサイズにリサイズ ] で実行されます。
リサイズは、各ピクチャペインのプロパティウインドウで指定した中心に向かって実行されます。

この機能はサイズに影響します。スケールには影響しません。

１．サイズ変更した（リサイズ前の）ピクチャペイン ２．テクスチャ本来のサイズにリサイズしたピクチャペイン
A サイズ変更された状態のピクチャペイン ピクチャペインが横長にサイズ変更されている状態です。テクスチャが伸びています。
B リサイズされたピクチャペイン [ テクスチャのサイズにリサイズ ] を実行し、テクスチャ本来のサイズにリサイズされました。
C ピクチャペインの中心（右下） このピクチャペインは中心が右下に設定されているため、右下を中心にリサイズされました。

使用例

ピクチャペインのサイズを編集している時、テクスチャの元のサイズに戻すために [ テクスチャのサイズにリサイズ ] でリセットする、という使い方ができます。

今までは、一度サイズ変更したピクチャペインを元のテクスチャの大きさに戻したい場合には数値入力を行う必要がありました。
ボタン一発でピクチャペインのサイズをテクスチャの実解像度にリセットできます。

設定されたフレームレート（ FPS ）でアニメーションが再生されるように改善

指定した FPS でアニメーションが再生できるようになりました！

メインメニュー項目 [ 編集 ] → [ ツール環境設定 ] → [ フレームレート ] のチェックを入れると、30FPS でアニメーションが再生されます。
現在のアニメーションの FPS がどうなっているかは、カーブエディタウインドウに表示されます。

以下はカーブエディタウインドウの画像です。

A FPS 表示箇所
 現在のフレームレート（ FPS ）が表示されます。初期値は 60FPS です。

使用例

アニメーションの 60FPS と 30FPS の見え方の違いを、FPS を切り替えることで容易に確認できます。

部品の「拡張ユーザ情報」の値を部分的に変更する機能

部品ペインで上書きしている拡張ユーザ情報の編集機能が以下のように改善されました!

部品ペインには、部品レイアウトから引き継いだプロパティ設定を部分的に変更する設定項目「上書き」があります。
以下は「上書き」で上書き可能な「拡張ユーザ情報」の編集項目です。

A 追加ボタン ユーザー情報をリストに追加します。
a はこのレイアウトで新たに変更または追加されたユーザー情報です。
B 削除ボタン 選択しているユーザー情報を未使用にします。参照先部品データからは削除されません。
b はこのレイアウトで使用されないユーザー情報です。
C
 		リセットボタン 参照先部品データの状態に戻します。
c はこのレイアウトで変更されないユーザー情報です。
D 編集ボタン

「拡張ユーザ情報」ダイアログが開き、選択しているユーザー情報を編集できます。

d はこのレイアウトで編集されたユーザー情報です。
E 選択項目の並び替えボタン 選択しているユーザ情報をリスト上、上下方向で並び替えます。
注意：

部品レイアウトの拡張ユーザー情報を編集した後、メインメニュー項目 [ 部品ペインを再読み込み (Ctrl + R) ] → [ ペイン ]を行ったとき、灰色で表示されたユーザー情報には部品レイアウトの変更が反映されます。
但し、その他のユーザー情報には反映されません。

参考：

「拡張ユーザ情報」とは、ユーザー独自の付加情報が設定できる機能です。ゲームアプリの機能やワークフローで利用できます。

使用例

例えばあるゲームアプリのフローでは、独自に作成したシェーダの値を「拡張ユーザ情報」により設定しているとします。
参照先部品ペインのユーザー情報を部品を利用するレイアウト側で部分的に変更できる為、シェーダカラーのみを変更するなどの使い方が考えられます。
