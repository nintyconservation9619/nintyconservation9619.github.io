反射回数（±）

ここではパーティクルがコリジョンにヒットする回数を設定する機能「反射回数（±）」について説明します。

概要

「反射回数（±）」は、パーティクルがコリジョンにヒットしたときを含めた反射回数を設定できる機能です。
エミッタタイプが「CPU」のとき、及びヒット時の挙動が「反射」の場合に機能します。
以下は「 設定項目」の概要です。

反射回数（±）

「ヒット時の挙動」が「反射」の場合に機能します。
パーティクルヒット時に、コリジョンとの反射回数を整数値で「 ダイヤル」や「 テキストボックス」により設定します。

値を -1 にすると、反射回数は無限になります。

設定例

以下は「反射回数（±）」の値を変更した場合の比較画像です。

A -1
デフォルト設定です。パーティクルはコリジョンで無限に反射します。
B 0 パーティクルはコリジョンに反射せずに通過します。
C 2 パーティクルはコリジョンに 2 回反射した後コリジョンを通過します。
