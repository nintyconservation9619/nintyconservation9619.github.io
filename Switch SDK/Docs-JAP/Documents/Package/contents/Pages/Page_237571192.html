2 Maya シーンファイルの用意

モデルをインポートする手順を説明します。
インポートしたモデルに 50 フレームのアニメーションを作成します。

このチュートリアルではエクスポートする Maya シーンを用意するために既存の fmdb を Maya にインポートします。通常は Maya のシーンファイルを開きます。
パッケージ内にあるサンプルモデルを読み込みます。 

Nintendo Import ウインドウを開く

Maya の「メニューバー」 → 「Nintendo」→ 「Import...」を開きます。

Nintendo Import ウインドウを設定して読み込む

各項目を設定して読み込みます。

下図と表の順番に操作して読み込みます。

順番 カテゴリ 操作
1 General Options

Process Mode を「Import」にします。

Magnify を「1.0000」にします。Create Texture File のチェックをオンにします。
2 Import Options fmd File

フォルダーアイコンをクリックして以下の「. fmdb」ファイルを指定します。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\TrafficSignal\TrafficSignal.fmdb
3 Import Options Import fmd 「Import fmd」をクリックします。設定したファイルを Maya にインポートします。

インポートに成功すると下図のモデルが表示されます。Nindendo Import ウインドウを閉じます。

Maya のアニメーション作業エリアを 0 フレームから 50 フレームの範囲に設定します。

関連リンク インポートの詳細は インポートプラグイン を参照してください。
