NintendoWare Audio ニュース

NintendoSDK 3.0.0 における変更点を紹介します。

リソースの管理についてのドキュメントとサンプルの追加

Atk はオーディオリソースをロードしたい単位に分割して作成し、ゲーム上で読み込むための仕組みを提供しています。

リソースの管理についてのドキュメントをガイドの リソース管理 に、

グループ機能を使ったリソース読み込みのサンプルプログラムを AtkGroup サンプル に追加しました。

リソース管理の詳細につきましてはこれらを参照ください。

ストリームジャンプ機能の追加

あらかじめ波形に設定しておいたリージョン間を行き来しながら再生することができるストリームジャンプ機能を追加しました。
ストリームジャンプ機能の概要やデータ作成のルールについては、ガイドのストリームサウンドをご確認ください。
また組み込み方法については プログラミングガイドのストリームサウンド や AtkStreamSound サンプル をご確認ください。
