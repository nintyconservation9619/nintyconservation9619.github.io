速度方向（±）

ここではエミッタセット内すべてのパーティクルの速度に対して、ここで設定した方向に値を加算する機能「速度方向（±）」について説明します。

概要

「速度方向（±）」は、エミッタセット内すべてのパーティクルの速度に対して、ここで設定した方向に値を加算できる機能です。
パーティクルの初速に対して操作されるので、フィールド(速度を弄るタイプ)は、初速に対して毎フレーム変更できます。

エミッタプラグインノードとプレビューノードから計算した初速をフィールドで変更できます。
速度方向を含む初速計算のイメージは以下のようになります。

  • パーティクルの初速 : （エミッタノードの初速設定　×　プレビューノードの初速設定にスケールをかける） + 速度方向（±）

関連リンク  以下は関連ページへのリンクです。

エミッタノード「放出」 パーティクルの放出に関する設定を行います。
プレビューノード「放出」 エミッタノードの「放出」機能に対してゲームアプリケーション側で制御可能な設定を行います。

以下は「 設定項目」の概要です。

速度方向（±）

パーティクルの速度に加算する値を X Y Z 軸それぞれで「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は「速度方向（±）」の値を設定した場合の比較画像です。
エミッタが放出するパーティクルの初速は 0 で、-Y 方向に重力を設定しています。

A X,Y,Z = 0 デフォルト設定です。速度に値を加算しないので、重力の影響を受けて下方向に加速します。
B Y = 0.5

A と比較すると、上方向に放出されたあと、重力の影響を受けて減速します。
C X,Y = 0.5
 A と比較すると、右上方向に放出されたあと、重力の影響を受けて上方向の速度が減速します。
