カーブビュー

カーブエディタのカーブビューでは、アニメーションカーブの作成 / 編集（キーの追加 / コピーと貼り付け）を行います。

関連リンク アニメーション編集が不可の場合は、編集不可である理由をカーブビュー上に表示します。

A

カーブビュー領域です。
カーブや、キーの選択及び編集操作を行います。
編集時に、フレームや値をクランプして作業を行えます。
アニメーションフレーム数の範囲外は、暗い背景（濃いグレー）で表示されます。
B カーブが存在するノードを選択している場合に、ノードのパスが表示されます。
ノードパスの表示 / 非表示は、カーブビューの右下にある表示設定パネルのカーブ名の表示で変更することができます。
C アニメーションカーブが色分け表示されます。
カラーアニメーションは、R G B で表示されます。
その他のアニメーションカーブにはユニークな色が設定されて表示されます。
アニメーションカーブをダブルクリックすると、該当するカーブツリーが展開されて見える位置にスクロールします。
D カレントタイムです。
[S] キーを使用してキーを追加 / 挿入するさいは、このカレントタイム（赤線）があるフレームにキーが打たれます。
ドラッグすることで、任意のフレームに移動できます。
E タイムスライダーです。
タイムスライダー領域の上部は、選択されているカーブに対応した情報が表示されアニメーションの処理をフレーム単位で編集できます。 
F 移動 / スケールツールを表示します。
G アニメーション設定パネルを表示します。
H 表示設定パネル（カーブビュー上に表示する情報）を表示します。 

カーブビュー上のマウス操作

カーブビューでのマウス操作は以下になります。

操作 説明
右ボタンクリック 選択カーブに対するコンテキストメニューが開きます。
左ボタンクリック キーを選択します。
左ボタン範囲選択 範囲選択内に入ったキーや、カーブを選択状態にします。
中ボタンドラッグ キーのモードが「最も近いキーの移動」か「複数移動」だった場合、キーを移動します。
最も近いキーの移動では、中ボタンをドラッグした場所から一番近いキーを移動します。
複数移動では、選択状態にあるキーを移動します。
キーのモードが「スケール操作」の場合は、ドラッグした箇所を基準にしてカーブのスケールを操作します。
キーのモードが「キーの挿入」「キーの追加」だった場合、挿入や追加したキーの移動ができます。
・「キーの挿入」モードでは、カーブに沿ってキーが移動します。
・「キーの追加」モードでは、キーを自由に移動できます。
中ボタンクリック キーのモードが「キーの挿入」だった場合、中ボタンを押したフレームにカーブを評価した「値」と「傾き」を基にしてキーを挿入します。
キーのモードが「キーの追加」だった場合、中ボタンを押した場所にキーを追加します。
Shift+中ボタンドラッグ キーのモードが「最も近いキーの移動」か「複数移動」の場合に有効です。
キーの移動を縦、または横に制限して移動します。
Alt+中ボタンドラッグ ビューをスライドします。
Alt+右ボタン 左右ドラッグ ビューのスケールを均等に変更します。
Alt+Shift+右ボタン 左右ドラッグ開始 ビューのフレーム方向のスケールを変更します。
Alt+Shift+右ボタン 上下ドラッグ開始 ビューの値方向のスケールを変更します。

コンテキストメニュー

カーブビュー上でマウスの右クリックを行うと、選択しているカーブに対するコンテキストメニューが開きます。

項目 説明
キーの挿入 選択しているカーブ上にあるカレントフレームの位置にキーを挿入します。
選択カーブのみを表示 カーブビュー上で選択しているカーブ以外を非表示にします。
カラーをコピー カラーアニメーションのカーブを選択している際に表示されるカラーパネルや、カラーバーの色をコピーします。
カラーを貼り付け コピーしたカラーを貼り付けます。貼り付け場所はカラーバー上で任意の場所になります。


移動 / スケールツール

「移動 / スケールツール」アイコンをクリックすると、カーブの移動やスケールを変更するためのパネルがカーブビュー上に表示されます。

A カーブの移動ツールです。
カーブビュー 上で選択したカーブ（複数選択可）を設定した数値分上下左右に移動します。
※上下矢印で「値」、左右矢印で「フレーム」を移動させます。 
B カーブのスケールを変更するツールです。
カーブビュー上で選択したカーブ（複数選択可）を対象（値 か フレーム）毎に倍率の数値でスケールを切り替えます。
スケールをかける基準点を「最小値」「最大値」「入力値」から選択できます。 


アニメーション設定パネル

カーブエディタで編集しているアニメーションのフレーム数や、量子化許容誤差を編集するパネルです。

カーブビューの右下にある「アニメーション設定」アイコンをクリックすると表示されます。

A アニメーションのフレーム数を数値入力や、ポップアップスライダーで編集します。
対応しているアニメーションのフレーム範囲は、0 ～ 16777215 です。
B フレーム数を変更した時に、既存カーブのフレームを自動で伸縮する場合にチェックを ON にします。
C アニメーションをループ再生する場合にチェックを ON にします。
D

量子化許容誤差の指定を行います。

数値を大きくするとデータ量が小さくなります。そのぶんアニメーションの精度が落ちます。

数値を変えることで、データ量と精度の調整を行います。
Ｅ

量子化による削減率を一覧します。

量子化許容誤差の指定の値によってどれくらい削減されているかを表示します。

量子化許容誤差の設定

アニメーションを量子化する際の許容誤差を設定します。アニメーションの量子化についての詳細は、3D 機能説明のアニメーションの量子化をご覧ください。

項目 説明
スケール

アニメーションを量子化する際の、テクスチャ行列やスケルタルアニメーションなどにおけるスケールに対する許容誤差です。
「(カーブの最大値) － (カーブの最小値)」に対する割合を指定します。
回転 アニメーションを量子化する際の、テクスチャ行列の回転、もしくはスケルタルアニメーションの回転に対する許容誤差です。
絶対値で度数を指定します。
移動

アニメーションを量子化する際の、テクスチャ行列の移動、もしくはスケルタルアニメーションの移動に対する許容誤差です。

「(カーブの最大値) － (カーブの最小値)」に対する割合を指定します。

スケール、回転、移動の各項目の後ろに記されている(50% 44/88)のような表示は、量子化後のアニメーションカーブのバイナリーサイズ削減率を示しています。

例えば、移動(50% 44/88)と表示されている場合は、「移動」カーブのバイナリーサイズが元は 88 バイトだったところ、量子化により 50 % 削減され、44 バイトになるということを示しています。

アニメーション設定パネルの右クリックメニュー

アニメーション設定パネルのフレーム数を設定するテキストボックス上でマウスの右クリックを行うと、下記コンテキストメニューが表示されます。

  • コピー
    テキストボックス内のフレーム数をコピーします。
  • 貼り付け
    コピーしたフレーム数を貼り付けます。
  • 最大値を入力
    3DEditor のカーブエディタが対応している最大フレーム数 （ 16777215 ） を自動で入力します。

スケルタルアニメーションのアニメーション設定パネル

スケルタルアニメーションをカーブエディタで開いた際は、アニメーション設定パネル でスケルタルアニメーションのループ再生と、量子化許容誤差の設定 ができます。

参考：

DCC ツールからループ設定をマージすると DCC ツール側の設定が優先されるため 3DEditor で設定した内容が上書きされますのでご注意ください。

表示設定パネル（カーブビュー上に表示する情報）

カーブビューの右下にある「表示設定」アイコンをクリックすると、表示設定パネルが表示され、カーブビューに表示する情報をチェックボックスで選択できます。
