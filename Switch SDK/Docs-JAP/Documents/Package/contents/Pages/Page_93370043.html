エミッタ形状「球」

ここではパーティクルの放出位置が球体内のエミッタ形状「球」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「球」は、ワールド座標系に配置される球体の外周面上でランダムな位置から球状に広がった法線方向にパーティクルを放出します。
以下は「 設定項目」の概要です。

「エミッタ形状」を「球」に設定すると「 球」が表示されます。以下はグループ内の設定項目です。

エミッタ半径

球の半径を X Y Z で設定します。

弧の開き方

弧の開き方を以下のプリセットから選択できます。

経度 球体の Z 軸正方向を基準に東西へ広がる角度を 0 ～ 360 度の範囲で設定します。
緯度 球体の Y 軸正方向（天頂）を基準に天底までのなす角度を 0 ～ 180 度の範囲で設定します。
