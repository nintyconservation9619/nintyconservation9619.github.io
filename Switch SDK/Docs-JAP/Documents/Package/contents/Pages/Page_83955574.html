スタートアップパス

概要

スタートアップパスは ユーザー設定と、チーム設定のそれぞれに複数設定できます。

スタートアップパスを設定することにより 3DEditor は起動時に以下のファイルを開きます。

  1. スタートアップパスで指定されたサブフォルダを含むフォルダ以下の中間ファイル
  2. スタートアップパスで指定された中間ファイルとプロジェクトファイル
  3. 3dEditor.exe と同じフォルダにある名前が Startup のフォルダのサブフォルダも含めて下にある中間ファイルとプロジェクトファイル

スタートアップパスで開いたファイルは、3DEditor のファイルメニュー「全て閉じる」では閉じません。
3DEditor から閉じたい場合は、ファイルを選択してファイルメニューの閉じるを使用します。

スタートアップパスの設定

スタートアップパスには「フォルダパス」または「ファイルパス」が設定できます。
パスの文字列には % で囲われた環境変数を含めることができます。3DEditor は % で囲われた環境変数を環境変数の値に置き換えて解釈します。

通常のファイル読み込みと同様に、起動時のファイル読み込みもアンドゥできます。

スタートアップパスを設定する際は、以下を注意してください。

  • プロジェクトファイルを起動時に開く場合は、それ以外のファイルを開く設定に含めないでください。
    起動時に限らずプロジェクトファイルと、別ファイルを同時に開くことはできません。
  • スタートアップパスは絶対パスまたは追加参照パスからの相対パスを指定してください。
