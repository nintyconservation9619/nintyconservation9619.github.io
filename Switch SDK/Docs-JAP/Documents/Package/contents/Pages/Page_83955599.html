運動量ランダム

ここではパーティクル放出時に個々が持つ運動量をランダムにする機能「運動量ランダム」について説明します。

概要

「運動量ランダム」は、パーティクル個々の動きにランダム性を持たせる機能です。
速度をはじめパーティクル放出後に加えられる力（重力やその他フィールド）などがパーティクル個々が持つ運動量の影響を受けます。
以下は「 設定項目」の概要です。

運動量ランダム

パーティクル個々が放出時に持つランダム係数を指定します。
この値が毎フレームパーティクル個々の運動量に乗算され、その結果が位置に加算されます。
