テクスチャ画像の変更ダイアログ

サンプラで使用するテクスチャの変更を行います。

マテリアルプロパティウィンドウのサンプラページにあるテクスチャ情報の変更から操作します。
A テクスチャのカラーイメージです。
B テクスチャのアルファイメージです。
C 表示深さです。配列テクスチャを選択した場合にアクティブになります。
D 3DEditor に読み込まれているテクスチャを一覧表示します。一覧からサンプラで使用するテクスチャを選択します。
E 選択したテクスチャの情報を表示します。
F 一覧表で選択しているテクスチャをサンプラで使用することを決定します。
G テクスチャの変更をキャンセルします。
