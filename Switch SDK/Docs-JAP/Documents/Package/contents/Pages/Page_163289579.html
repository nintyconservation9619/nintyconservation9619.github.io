追加コンテンツの管理

追加コンテンツの管理には SDK に含まれるターゲットツールである、DevMenuCommand を使用します。

DevMenuCommand を使用することで下記の管理機能を使うことができます。

  • 追加コンテンツのインストール
  • 追加コンテンツの列挙（※）
  • 追加コンテンツのアンインストール（※）

 

※同じアプリケーション ID を持つアプリケーションと追加コンテンツがひとまとめにされます。個別の削除や列挙はできません。

インストール

追加コンテンツの作成方法 を参考にパッケージング済みの追加コンテンツが用意されていることが前提です。

開発機に接続されている状態で下記のコマンドで追加コンテンツをインストールすることができます。

# Nand へのインストール
%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent install <AocPath>
 
# Sd カードへのインストール
%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent install <AocPath> -s sdcard

AocPath はフルパスで入力する必要があります。

 

インストールが成功した場合は下記のような出力が得られます。

Preparing to install D:\NintendoSDK\Samples\Outputs\NX-NXFP2-a64\Applications\AocSimple\Develop\SampleAoc.nsp...
5120 / 5120
[SUCCESS]

 

 

列挙

開発機にインストールされている追加コンテンツは次のコマンドで、そのアプリケーション ID を列挙することができます。

%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent list

 

列挙に成功した場合は下記のような出力が得られます。

2 add-on-content(s) found.
id                  index  release-version  required-app-release-version
0x0100326002324000      1                1                             0
0x0100326002324000      2                1                             0
[SUCCESS]

 

 

アンインストール

開発機にインストールされている追加コンテンツは次のコマンドでアンインストールすることができます。

%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent uninstall <ApplicationId> [--index <index>]

ApplicationId には追加コンテンツの列挙で取得した ID や、追加コンテンツのメタデータで指定したアプリケーション ID を指定します。

アンインストールに成功した場合は下記のような出力が得られます。

[SUCCESS]
