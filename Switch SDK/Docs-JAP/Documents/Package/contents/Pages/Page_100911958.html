はじめに

本体設定の言語設定を扱う機能です。

現在の言語設定を取得する方法はプラットフォームごとに異なるためアドオンとして提供されます。

 

言語設定の取り得る値は下記のとおりです。 値は IETF 言語タグ形式の言語コードとの対応関係を持ちます。

下記の値は GetAvailableLanguageCodes() を呼び出すことで取得可能です。

値 IETF 言語タグ
日本語 ja
米国英語 en-US
フランス語 fr

イタリア語 it

中国語 zh-CN
韓国語 ko

中国語（簡体字） zh-Hans

中国語（簡体字）と 中国語（繁体字）は NX Addon 4.0.0 からサポートされます。

 

Windows 環境における構成

Windows のシステムロケールとの間には特に関係を持ちません。

言語設定の使用方法

#include <nn/nn_Log.h>
#include <nn/settings/settings_Language.h> 
 
extern "C" void nnMain()
{ 
// 言語設定の候補を表示します。
    int count = nn::settings::GetAvailableLanguageCodeCount();
 
    nn::settings::LanguageCode* codes = new nn::settings::LanguageCode[count]();
 
    count = nn::settings::GetAvailableLanguageCodes(codes, count);
 
    for (int i = 0; i < count; ++i)
    {
        NN_LOG("%d %s\n", i, codes[i].string);
    }
 
    delete[] codes;
}
