はじめに

本ドキュメントでは、 NintendoSDK の提供する JPEG ライブラリ機能について説明します。

JPEGライブラリとは

JPEGライブラリは NintendoSDK に収録される、 JPEGデータや関連するデータ構造を処理するための次のライブラリ機能を指します。

  • nn::image::JpegDecoder クラス
  • nn::image::JpegEncoder クラス
  • nn::image::ExifExtractor クラス
  • nn::image::ExifBuilder クラス

対象読者

以下の方を対象としています。

  • NintendoSDK JPEG ライブラリを使用した アプリケーション を企画・開発される方
  • NintendoSDK JPEG ライブラリを使用した ミドルウェア を企画・開発される方
  • その他、 NintendoSDK JPEG ライブラリ を何らかの目的に使用される方

ライセンス

JPEG ライブラリは、オープンソースソフトウェアの libjpeg をバックエンドに使用しています。
libjpeg のソフトウェアライセンス条項に関して、  SDK パッケージ同梱のライセンスファイル (Documents/Licenses/libjpeg_8c/README.txt)を参照してください。

参考：

JPEG ライブラリの機能はすべてのアプリのバイナリに含まれますので、本体設定内で表記を行います。各アプリでの表記は不要です。
