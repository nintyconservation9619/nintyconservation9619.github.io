ライブラリ

概要

PC 版 NVN ではヘッダファイルのみが提供されます。 ライブラリの静的リンクは不要で NVN API のエントリポイントはランタイムで NVN ドライバから関数ポインタを取得します。
NintendoSDK NX Addon で提供される NVN ヘッダファイルにそのヘルパー関数が含まれています。具体的な使用方法は NVN プログラミングマニュアル やサンプルを参照してください。

NX 版 NVN ではヘッダファイルに加えて、専用のライブラリが提供されます。

ヘッダファイル

NVN は、C インタフェースを定義する nvn/nvn.h と C++ インタフェースを定義する nvn/nvn_Cpp.h を提供します。
その他の必要なヘッダファイルも合わせて、以下のディレクトリに配置されます。 

$NINTENDO_SDK_ROOT/Include/nvn
