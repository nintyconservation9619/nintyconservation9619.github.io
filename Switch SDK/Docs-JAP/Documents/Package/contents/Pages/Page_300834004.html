マスク機能について

マスク機能とは、ペインの描画結果をマスク画像で切り抜く機能です。

LayoutEditor 上でペインのプロパティから「マスク」の項目を設定することで使用できます。

マスク機能は内部的にキャプチャ機能を使用していますので、マスク機能を利用した flyt を扱う場合には、キャプチャ機能を使うための対応がプログラム側で必要になります。

キャプチャ機能をプログラム側で使う方法は、「キャプチャ機能について」のページをご覧ください。
