連番テクスチャの読み込みウインドウ

「連番テクスチャの読み込みウインドウ」について説明します。

「連番テクスチャの読み込み」機能を利用すると、指定したフォルダ内に存在する、特定の命名規則にしたがって番号付けされたファイル群 ( 連番テクスチャ ) をまとめて読み込んで、それらのテクスチャによるパターンアニメーションを行うペインを自動生成することができます。

[ メニューバー ] → [ ツール ] → [ 連番テクスチャの読み込み ]

連番テクスチャの読み込みウインドウ
A 読み込み先フォルダ 連番テクスチャが入っている、読み込み先のフォルダのパスを入力します。
B 開始フレーム 連番テクスチャのパターンアニメーション開始フレームを入力します。
C パターンのフレーム間隔 連番テクスチャパターンアニメーションのテクスチャを切り替える間隔をフレームで入力します。
D 繰り返し回数

連番テクスチャのパターンアニメーションを繰り返す回数を入力します。

( 繰り返し方法をミラーにした場合は、ミラー再生分も繰り返し回数に入力します )
E 繰り返し方法 リピート テクスチャをテクスチャ名の番号 ( 連番 ) 順に再生します。
ミラー 番号順に再生し、最後の番号から逆番号順に再生します。
F 読み込み実行 設定を決定し、読み込みを実行します。
テクスチャウインドウ内に、読み込み先フォルダの連番テクスチャが読み込まれます。
参考：

設定した連番テクスチャのアニメーション再生の長さよりも、カーブエディタで指定したアニメーションの長さが短い場合は、連番テクスチャのアニメーションは途中で終了し、最初に戻ります。
