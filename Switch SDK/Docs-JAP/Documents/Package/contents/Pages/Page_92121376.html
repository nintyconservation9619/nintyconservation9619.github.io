 EffectMaker の UI について、設定できます。

ダイアログ

「 ダイアログ」は、ダイアログウィンドウの設定を行う機能です。
以下はグループ内設定項目の一覧です。

XPのスタイルを使用

EffectMaker の保存や読み込みダイアログの UI デザインを XP に切り替えるかを ON/OFF で切り替えます。
起動時に「今日のヒント」を表示 EffectMaker の起動時に「今日のヒント」ダイアログを表示するかを ON/OFF で切り替えます。

テキストボックスクリック時の挙動

「 テキストボックスクリック時の挙動」は、テキストボックスクリック操作時の挙動を設定する機能です。
テキストボックス内をマウスの左クリック操作で全選択状態にするかを ON/OFF で切り替えます。

使用するテキストエディタ

「 使用するテキストエディタ」は、 EffectMaker で使用するテキストエディタを設定する機能です。
ここで選択したテキストエディタは、 EffectMaker の以下の機能で利用します。

  • エラーダイアログの表示
  • カスタムシェーダ「シェーダコード」の表示
