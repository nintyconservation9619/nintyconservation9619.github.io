はじめに

DevMenuは、開発機を操作する各種機能を提供する開発機専用メニューです。

表示名 詳細説明ページ 提供する主機能
Application アプリケーション管理
  • インストールされたアプリケーションのリスト、起動、終了、削除およびインストール
  • ゲームカードへのアプリケーションの書き込み
Save Data セーブデータ管理
  • セーブデータのリストと削除
  • セーブデータのmicroSDカードへのバックアップ
  • セーブデータのmicroSDカードからのリストア
Controller コントローラー管理
  • ペアリング済みのコントーローラー数や接続されているコントローラー状態の表示
  • 新規コントローラーとのペアリング
  • ペアリング情報の削除
Accounts アカウント管理
  • ユーザーの追加と削除、ユーザー情報の変更
  • ネットワークサービスアカウントの登録
Error エラー情報
  • 過去に発生したエラー情報の表示、エクスポート
Device デバイス設定の変更
  • 開発機の各種設定
Network ネットワーク設定
  • ネットワーク接続状態の確認
  • 各種テスト機能
Debug 開発支援機能の設定
  • 開発支援機能の設定
Time Test 時計テスト機能の設定
  • ネットワーク時計の設定
  • Steady Clock の設定
Error エラー情報
  • 過去に発生したエラー情報の表示、エクスポート
Shop ショップ機能の設定
  • ショップの切り替え
  • コンテンツをダウンロードできる本体の登録と解除
参考：

NX Addon 5.3.0以降で利用可能です。
Firmware ファームウェア情報
  • ファームウェアバージョン情報の表示

DevMenuはスタンドアローンで動作するため、開発機がホストに接続されていなくても上記の機能を使用することができます。

DevMenuの機能をコマンドラインで実行するために DevMenuCommand も用意されています。

使用方法

DevMenu の起動

DevMenuは開発機の電源を入れると自動的に起動します。DevMenuはアプリケ－ション起動後はバックグラウンドに常駐します。

操作方法

DevMenuの操作はデバッグパッドまたは画面へのタッチを使用して行います。ここではデバッグパッドでの操作を基本として説明します。

全体的な操作

画面左側フレームの各ページにフォーカスを合わせることでページの内容が表示されます。　
ページタブとページ本体間のフォーカス遷移は、十字ボタンの左右またはA/Bボタンを押すことで行えます。

操作形態による実行可能な処理の制限

現状、タッチでのみ行える処理が存在します。以下が該当する操作になります。今後のリリースで改善予定です。

  • ページタブフレームのスクロール

開発者向けオーバーレイ通知

DevMenu環境では開発者向けオーバーレイ通知が表示されます。

  • 開発者向けオーバーレイ通知は言語設定に関わらず英文テキストで表示されます。
  • アイコンやサムネイル画像の表示、SEの再生は行われません。

製品機での動作を確認する際には、HOMEメニュー環境に変更して動作確認を行ってください。
