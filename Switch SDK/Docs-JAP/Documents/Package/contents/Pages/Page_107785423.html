VI ライブラリ/機能

機能

VI ライブラリには、ディスプレイとレイヤを中心に扱う 2 つの主な機能セットがあります。

  • 使用方法：レイヤの作成

ディスプレイ

VI ライブラリを使用するアプリケーションでは以下の処理が可能です。

  • 使用可能なディスプレイを列挙する
  • 他の操作で使用するディスプレイを開いたり、閉じたりします。
  • 表示機能を問い合わせます。

レイヤ

アプリケーションのコンテンツは、レイヤのハンドルで制御します。アプリケーションでは、これらのハンドルを使用して以下の操作が可能であることを想定できます。

  • レイヤを作成または破棄する
  • グラフィックス API で使用するネイティブウィンドウを取得する
