 nn::g3d ニュース

テクスチャーコンバーターのテクスチャー中間ファイル関連機能の廃止

テクスチャーコンバーター(%NINTENDO_SDK_ROOT%\Tools\Graphics\GraphicsTools\TextureConverter.exe)のテクスチャー中間ファイル関連の機能を次回以降リリースのバージョンで廃止予定です。今後、テクスチャー中間ファイルの変換には NintendoWare パッケージに新しく追加された 3D テクスチャーコンバーター(%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dTextureConverter.exe)を使用して下さい。 

中間ファイル関連ツール、ツールの機能の移動

以下のツールのパスを変更しました。変更前のパスは次回以降リリースのバージョンで廃止予定です。移行期間中は廃止予定となっているツールも使用できますが、移行期間終了後にパッケージから削除する予定です。

Use3dTools という名前の空ファイルを %NINTENDO_SDK_ROOT%\Tools\Graphics\GraphicsTools 以下に作成しておくと、移行前のツールが使われたときにエラーを発生させることができます。移行できているかどうかの確認に使用してください。

 nn::ui2d ニュース

レイアウトコンバーターへフィル最適化をおこなう機能を追加

レイアウトコンバーターにピクチャペインのフィル最適化のため、透明なピクセルを描画しないメッシュ形状を作成する機能を追加しました。詳細についてはメッシュ形状トリミング最適化についてを参照してください。

Pane クラスで拡張ユーザー情報関連のメンバを整理してメモリ削減

Pane クラスで、拡張ユーザー情報関連のメンバを整理してメモリ削減をしました。ポインタメンバーが一つ減って、64ビット環境なら、ペイン一つあたり、8バイトメモリが減少します。

この変更によって、拡張ユーザー情報のアニメーションを利用しているペインでは、SetExtUserDataList で他のユーザー情報にポインタを差し替えることができなくなります（この操作はメモリリークを発生しますので、アサートを入れて処理を禁止しています）。意図してこのような操作を行うことは考えにくく、問題になるケースは少ないと考えていますが念のためご注意をお願いします。
