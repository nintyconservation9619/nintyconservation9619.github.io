レイアウトエディタウインドウ

レイアウトエディタウインドウを構成する、各部位の名称と役割について説明します。

  • A メニューバー
  • B ツールバー

A メニューバー

以下の項目が並びます。項目の上でマウス左クリックによりメニューが表示されます。
並んだ項目は、レイアウトウインドウを表示すると変化します。

ファイル
レイアウトの作成や、ファイルの入出力、環境設定に関する操作を行います。
編集 やり直し、複製、削除等の編集作業を行います。
ウインドウ レイアウトの作成に必要なサブウインドウの「表示 / 非表示」を行います。
ヘルプ LayoutEditor のヘルプや情報を表示します。

C 作業エリア

レイアウト作業を行うエリアです。
複数のレイアウトウインドウを開いて同時に作業をする事ができます。
同時に複数のレイアウトを用いてタブから切り替え、それぞれのレイアウトの違いやアニメーションの比較をしたりできます。
また、レイアウト間でペインの複製をする事もできます。

ドッキングウインドウ

作業エリア内で各操作ウインドウをドッキングさせ、作業内容に合わせて好みの配置にすることができます。

「 グループウインドウ」、「 親子階層ウインドウ」、「 カーブエディタウインドウ」、「 部品パレットウインドウ」等、これらのウィンドウをまとめて表示させたり、取り外して独立した ( フローティング状態 ) ウィンドウにできます。
LayoutEditor の作業内容にあわせて好みの配置にすることができます。

ワークスペース

「 B ツールバー」の「ワークスペース」をプルダウンから切り替えることにより、それぞれの作業内容に適したウインドウ表示 ( モード ) に切り替えることができます。
「ワークスペース」は、「レイアウト」「アニメーション」「部品」から選択します。

「レイアウト」「アニメーション」「部品」それぞれのモード毎、別々にウインドウをカスタマイズすることができます。
モード毎にカスタマイズされたウインドウの状態は、ツールが設定を記憶していますので、ユーザー個人の使いやすいウインドウ配置設定を構築することが可能です。

各ウインドウを、レイアウトエディタウインドウの作業エリア内で配置

ウィンドウの上辺部分をマウス左ボタンでドラッグすると、レイアウトエディタウインドウ上にアイコンが表示されます。
配置箇所を表す以下のアイコン上までウィンドウを移動させ、ドロップすることで配置できます。

レイアウトエディタウインドウ作業エリアの上辺にスナップするように配置されます。
レイアウトエディタウインドウ作業エリアの右辺にスナップするように配置されます。
レイアウトエディタウインドウ作業エリアの下辺にスナップするように配置されます。
レイアウトエディタウインドウ作業エリアの左辺にスナップするように配置されます。

ドッキングさせたいウィンドウ内で配置

ウィンドウを選択し、そのままドッキングさせたいウィンドウまで移動すると「  アイコン」が表示されます。
配置箇所を表す以下のアイコン上にウィンドウをドロップすることで配置できます。

ドッキングさせたいウィンドウの上側に配置されます。
ドッキングさせたいウィンドウの右側に配置されます。
ドッキングさせたいウィンドウの下側に配置されます。
ドッキングさせたいウィンドウの左側に配置されます。

ウィンドウ同士が重ねて配置されます。

表示させたいウィンドウを「タブ」で切り替えます。

ワークスペースの記憶

ユーザーがカスタムしたワークスペースをツールに記憶させることができます。

[ 編集 ] → [ ワークスペースの記憶 ]

ワークスペースの復元

ワークスペースの復元を行うと、「ワークスペースの記憶」でツールに記憶させたワークスペースを復元することができます。

[ 編集 ] → [ ワークスペースの復元 ]
ツールバーの「 ワークスペースの配置情報を復元」アイコンでも可能です。
ショートカット：Ctrl + Shift + T

 

参考：

レイアウトウインドウの領域には、ツールウインドウを重ねることはできません。
また、レイアウトウインドウはフローティング状態にすることはできません。

複数のレイアウトウインドウを開いた場合、レイアウトウインドウの切り替えは、表示上辺の「タブ」で行います。
