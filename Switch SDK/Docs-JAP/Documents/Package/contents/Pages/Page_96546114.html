各成分の設定

ここではパーティクルが移動する軌道となる波形を設定する機能「各成分の設定」について説明します。

概要

パーティクルが移動する軌道となる波形は 4 つの波形成分が合成されて形成されています。
「各成分の設定」によって「成分0～3」の4 つの波形個別に周期と振幅に対する重み付けが設定できます。

以下は 0 から 3の波形成分が足されてひとつの波形が形成されるイメージです。

以下は「 設定項目」の概要です。

各成分の設定

各成分の波形を設定するかを「 チェックボックス」ON/OFF で切り替えます。
設定を ON にすると「 グループ」が表示されます。以下は各グループ内の設定項目です。

成分 0 ～ 成分 3 の周期設定

グループ『各々成分の「基本周期」に対する比率』の設定項目です。
各波形成分の周期（フレーム数）を設定する機能です。

成分 0 ～ 成分 3 の振幅設定

グループ『各成分の振幅に対する重み付け』の設定項目です。
各波形成分の振幅に対して設定した値を乗算することで重み付けする機能です。

設定例

以下は波形成分 0 から 3 と合成した波形の上を移動するパーティクルです。

A 成分 0 デフォルト設定での成分 0 の波形イメージです。
B 成分 1 デフォルト設定での成分 1 の波形イメージです。
C 成分 2 デフォルト設定での成分 2 の波形イメージです。
D 成分 3 デフォルト設定での成分 3 の波形イメージです。
E 合成波形 全ての波形成分を合成した波形のイメージです。
