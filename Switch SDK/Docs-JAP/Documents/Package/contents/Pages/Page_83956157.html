カスタムアクション・カスタムシェーダと連動するコールバック

コールバック関数の呼び出し制御は特徴的で、複数登録されたコールバック関数セットの中からどのコールバック関数セットを実行するかエフェクトメーカ ( UI ) 上で指定する仕組みになってます。よって利用するには、エフェクトメーカ、エフェクトランタイム、それぞれにおいてプログラマ側で仕込みを準備する必要があります。詳しくは カスタムアクション ・ カスタムシェーダ　それぞれのドキュメントを参照ください。

エミッタ単位

エミッタの処理単位で呼び出されるコールバックで、以下のタイミングでコールバック関数を設定することが出来ます。

呼び出しタイミング 型 説明

エミッタ初期化時

 EmitterInitializeCallback

エミッタ生成時に呼び出されるコールバックです。

エミッタ単位で独自にワークを準備したい場合などに利用します。
エミッタ破棄時 EmitterFinalizeCallback エミッタ破棄時に呼び出されるコールバックです。

エミッタ計算処理前

 EmitterPreCalculateCallback エミッタ計算処理前に呼び出されるコールバックです。

エミッタ計算処理後

 EmitterPostCalculateCallback

エミッタ計算処理後に呼び出されるコールバックです。

計算済み全パーティクルの情報にアクセスしたい場合などに利用します。

エミッタ描画前

 EmitterDrawCallback

エミッタ描画前に呼び出されるコールバックです。

コールバック返り値で true を返すことで、自身のエミッタ描画処理をキャンセルすることができます。

エミッタ描画設定後

 RenderStateSetCallback

エミッタ描画直前で呼び出されるコールバックです。

カスタムシェーダ利用時に、独自のコンスタントバッファやテクスチャを設定したり、

デフォルトで設定された描画設定をキャンセルしたい場合などに利用します。

関連リンク

カスタムシェーダの例 カスタムシェーダのチュートリアルです。RenderStateSetCallbackを利用したコードサンプルがあります。

 

パーティクル単位

パーティクル単位で呼び出されるコールバックで、以下のタイミングでコールバック関数を設定することが出来ます。パーティクル単位でコールバックが呼び出されるため、利用する際は放出されるパーティクル数に注意が必要です。また、CPUエミッタでは全てのコールバックが利用できますが、GPUエミッタ、GPU-SOエミッタでは、パーティクル放出時/削除時コールバックのみ利用できます。

パーティクルに対しての操作は、コールバック関数の引数で渡される　nn::vfx::ParticleCalculateArg クラスを介して行います。nn::vfx::ParticleCalculateArg クラスには、パーティクルの位置や速度、カラーなどのGetter / Setter が実装されています。

呼び出しタイミング 型 制限 説明

パーティクル放出時

 ParticleEmitCallback 無し パーティクル放出時に放出されるパーティクルごとに呼び出されるコールバックです。

パーティクル削除時

 ParticleRemoveCallback 無し パーティクル削除時に削除されるパーティクルごとに呼び出されるコールバックです。

パーティクル計算時

 ParticleCalculateCallback

CPUエミッタ

時のみ有効

 パーティクル計算処理時にパーティクルごとに呼び出されるコールバックです。

 

ランタイムへのコールバックの設定方法

エフェクトシステムに対してコールバックを設定するには、nn::vfx::CallbackSet 構造体を利用します。nn::vfx::CallbackSet 構造体には、エミッタ単位、パーティクル単位のコールバック関数ポインタのメンバがありますので、利用したいタイプのコールバック関数ポインタメンバのみ初期化したうえで、エフェクトシステムに登録します。

//  パーティクル計算処理コールバック関数
static void _ParticleCalcCallback( nn::vfx::ParticleCalcArg& arg )
{
    nw::math::VEC3 worldPos;
    arg.GetWorldPos( &worldPos );
    worldPos.y += 1.0f;
    arg.SetWorldPos( worldPos );
}

// カスタムアクションID 1 に対してパーティクル計算コールバックを設定する
nn::vfx::CallbackSet set;
set.particleCalc = _ParticleCalcCallback;
system->SetCallback( nw::vfx::CallbackId_CustomAction1, set );
