3D ニュース

NintendoSDK 0.17.0 で追加された機能をグラフィカルに紹介します。

3DEditor

3DEditor について新しく追加された機能について紹介します。

GUI からモデル中間ファイルに対して任意の処理を実行できるようになりました

特定のモデルの特定のマテリアルの量子化設定を削除するなど、特定のモデル中間ファイルに対する任意の処理をコンテキストメニューから実行することができるようになりました。

関連リンク 詳しくは <UserCommand> をご覧ください。

接続先アプリケーションにモデルに対する任意のスクリプトを送信できるようになりました

特定のモデルにフォーカスする処理など、接続先アプリケーション上の特定のモデルに対する任意の処理をコンテキストメニューから実行することができるようになりました。

関連リンク 詳しくは <RuntimeUserScript> をご覧ください。

注意：

「ユーザコマンド」と「ランタイムユーザスクリプト」はチーム設定で何かコマンドを設定しないとUI上には表示されません。

 

Maya プラグイン

Maya プラグインに新しく追加された機能について紹介します。

モデルのマージエクスポート時に Maya 側で設定したボーンの可視性を反映させられるようになりました

コンフィグファイルに以下を記述することで、モデルのマージエクスポート時に Maya 側で設定したボーンの可視性が反映されるようになります。

bone_visibility_merge_priority="true"

関連リンク 詳細はエクスポートプラグイン使用方法の  bone_visibility_merge_priority をご覧ください。

LOD エクスポート時にサブメッシュ分割を適用できるようになりました

今までサブメッシュ分割は NintendoWare のポリゴンリダクションとしか併用することができませんでしたが、任意の LOD モデルを利用する LOD エクスポート時にもサブメッシュ分割を適用できるようになりました。

NintendoWare のポリゴンリダクションと併用する場合に比べて、ランタイムでの利用に制約がありますので、プログラマーと相談の上、ご利用下さい。制約については 3D プログラミングガイドの LOD を参照してください。

関連リンク サブメッシュについてはモデルの「モデルのジオメトリデータ構成」を参照してください。
