プレビュープレイヤー

プレビュープレイヤーとは？

プレビュープレイヤーは、AtkPlayer での SoundArchivePlayer パネルに相当する操作を SoundMaker 上で行い、設定を転送して実機でのサウンドの確認を行うペインです。

プレビュープレイヤーで再生を行うと、PCエミュレーション または AtkPlayer に接続中は AtkPlayer 上にてサウンド再生されます。

 

プレビュープレイヤーの操作

プレビュープレイヤーを開く

[表示]→[プレビュープレイヤー] または、ツールバーの「プレビュープレイヤーの表示」ボタンにより、プレビュープレイヤーを開くことができます。

またこの操作には Alt + 4 のショートカットキーが割り振られています。 これらの操作は、ウインドウの表示・非表示のトグル動作になります。

 

 

プレビュープレイヤーは、メインウインドウ下部のペインにドッキングウインドウとして現れます。

ウインドウ下部にある"プレビュープレイヤー"タブ部分またはタイトルバーのコンテキストメニューから、プレビュープレイヤーウインドウを [フローティング]、[ドッキング]、[非表示] に変えることができます。
タイトルバー部分のダブルクリックでも [フローティング]、[ドッキング] の切り替えが行えます。

また、ドッキングウインドウ時は、タイトルバー右側の最小化アイコンをクリックするか、スプリッタ (境界線) をダブルクリックすることにより、タブのみの最小化表示になります。

プレビュープレイヤーの操作

プレビュープレイヤーのプレイリストにサウンドを登録または入れ替えを行うには、以下の方法があります。

  • サウンドセットタブの選択状態のインデックスセルをクリックし、プレビュープレイヤーのプレイリスト内にドラッグ＆ドロップすることにより、サウンドが登録できます。
  • サウンドセットタブ内のリストで登録するサウンドを選択し、リスト内やツールバーの「再生」ボタンをクリック、または [F9]・スペースキーを押下することにより、プレイリスト内の選択されている場所にサウンドが登録され、再生することができます。

　　　　　

サウンドが登録されたプレイリストでは、それぞれ「再生」、「一時停止」、「停止」ボタンを押下することにより、サウンドの確認を行います。
ショートカットキーとして、スペースキーおよび [F9] キーで "再生"、[F10] キーで "すべて停止" を行うことができます。

また、「ループ再生」ボタンを押下して有効にすることにより、サウンドのループ再生を行えます。
ボタンをクリックする毎に ON, OFF のトグル動作をします。
