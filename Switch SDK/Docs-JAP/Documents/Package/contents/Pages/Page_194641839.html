レンジスライダー

カーブエディタウインドウの「レンジスライダー」に表示される機能を説明します。

「レンジスライダー」は、再生するアニメーションの範囲を設定します。

レンジスライダー
A 開始フレーム アニメーションの開始フレームです。
B 終了フレーム アニメーションの終了フレームです。
C 再生開始フレーム アニメーション再生範囲の再生を開始するフレームです。
D 再生終了フレーム アニメーション再生範囲の再生を終了するフレームです。
E アニメーション再生範囲

アニメーション再生時には、この範囲を繰り返し再生します。

( C と D で指定したフレーム範囲です )

この範囲が タイムスライダー に表示されます。
