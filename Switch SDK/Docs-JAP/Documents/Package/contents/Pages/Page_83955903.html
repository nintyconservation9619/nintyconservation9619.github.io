基本設定

ファイル情報

このタブではエミッタセットファイルの情報を見たり、エミッタセットの基本設定を行います。

基本設定

「 ファイル情報」は、エミッタセットファイルの基本情報を見たり、ファイルにコメントやラベルカラーが設定できる機能群です。
また、ファイルに付随する情報として、「コメント」や「ラベルカラー」を設定できます。
これら情報は、「エフェクトブラウザ」のファイルリストで閲覧したり、編集できます。
以下はグループ内設定項目の一覧です。

ファイル名 ファイルの名前を表示する機能です。
ファイルパス ファイルの保存先パスを表示する機能です。
コメント

コメントを編集できる機能です。
ラベルカラー

ラベルカラーを設定できる機能です。

設定例

「コメント」と「ラベルカラー」はエフェクトブラウザでも確認及び編集できます。
以下は設定したエミッタセッットファイルを「ファイル情報」と「エフェクトブラウザ」で表示した画像です。

A ファイル情報 B エフェクトブラウザ
A コメント ファイルに対するメモや他人にデータを渡す時のメッセージとして利用できます。
半角、全角で入力できます。
B ラベルカラー
 ファイルを好みに合わせて色で仕分ける時に利用できます。
エフェクトブラウザのリスト表示ではラベルカラーでソートする機能もあります。
