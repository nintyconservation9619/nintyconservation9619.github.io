ユーザーデータリソース

概要

ユーザーデータリソースは、リソースのコンバート時に付加される任意のユーザーデータを表現します。

ユーザーデータリソース

ユーザーデータリソースは nn::gfx::ResUserData によって定義されています。

名前の取得

GetName によりユーザーデータリソースの名前を取得します。

データ数の取得

GetCount によりユーザーデータリソースが所有するデータの数を取得します。

型の習得

GetType によりユーザーデータリソースが所有するデータの型を取得します。

データの取得

以下の表において、GetType で得られる型に対応するデータ取得関数を呼ぶことでデータを取得できます。

型 データ取得関数
nn::gfx::ResUserData::Type_Int GetInt
nn::gfx::ResUserData::Type_Float GetFloat
nn::gfx::ResUserData::Type_String GetString
nn::gfx::ResUserData::Type_Stream GetStream
