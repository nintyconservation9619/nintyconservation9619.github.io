ペアフォントプロパティウィンドウ

以下は「ペアフォントプロパティウィンドウ」の UI 説明です。

A フォント1の種類 フォント1の種類です。プロパティウィンドウ上では編集不可です。
ツリービューからペアフォントノードに子ノードを追加すると設定されます。
B フォント2の種類 フォント2の種類です。プロパティウィンドウ上では編集不可です。
ツリービューからペアフォントノードに子ノードを追加すると設定されます。
C 入れ替え

フォント1の種類とフォント2の種類を入れ替えます。
