NVN グラフィックスデバッガ

概要

NVN グラフィックスデバッガは、NVN 専用のグラフィックスデバッガです。ドローコール、ステート、リソースのデバッグ、パフォーマンス解析、描画フレームのキャプチャ等の機能を提供します。

NVN グラフィックスデバッガはアプリケーションのデバッグ目的に使われることを意図しています。このライブラリを使用したROM を Nintendo Developer Portal (NDP) にマスター ROM としてアップロードすることはできません。

パッケージ

NVN グラフィックスデバッガは、NintendoSDK NX Addon とは別のパッケージで提供されます。また、他の NintendoSDK パッケージよりも遅れてリリースされる場合があります。

以下にパッケージを一覧します。

パッケージ名 説明
NVN Graphics Debugger NVN 専用のグラフィックスデバッガです。

配置

NVN グラフィックスデバッガは、以下のように NintendoSDK を配置したディレクトリ直下に配置されます。

NintendoSDK/
NvnGraphicsDebugger/

マニュアル

パッケージの詳細については、パッケージ内のドキュメントを参照してください。
