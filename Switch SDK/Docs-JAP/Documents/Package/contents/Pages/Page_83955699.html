テキスト中間ファイルコンバーター

テキスト中間ファイルコンバーターの説明と注意事項を説明します。

  • 概要
  • コマンドラインオプション
  • 利用例

概要

バイナリー中間ファイル (.f??b) をテキスト中間ファイル (.f??a) に変換します。

バイナリー中間ファイルのデータ列をテキストエディター等で確認するため、テキスト中間ファイルに変換する際に利用します。

実行ファイル (3dIntermediateFileBinaryToAscii.exe) やショートカットに、バイナリー中間ファイルやフォルダーをドラッグ＆ドロップすると、バイナリー中間ファイルのあるフォルダーにテキスト中間ファイルを保存します。
バイナリー中間ファイル以外のファイルは無視します。

テキスト中間ファイルを他の場所に出力したい場合は、バイナリー中間ファイルをコピーしてから変換するか、変換後のテキスト中間ファイルを移動してください。

コマンドラインオプション

3dIntermediateFileBinaryToAscii.exe [--args-file ""] [--version] [-h] [-s] [-j ""] [--disable-xsd-check]
[--enable-extension ""] [--disable-extension ""] [中間ファイル|フォルダ]...

バイナリー中間ファイル (.f??b) をテキスト中間ファイル (.f??a) に変換します。

--args-file ""

引数ファイルを指定します。
引数ファイルに記述した内容を、引数として指定できます。 
引数ファイルは複数行に渡って各オプションを記述できます。 
行頭に # がある行はコメントとして扱います。 
オプション値指定の "" によるエスケープに改行を含めることはできません。

--version

バージョン情報を表示します。

-h --help

ヘルプを表示します。

-s --silent

処理中のメッセージ表示を抑制します。

-j "" --jobs ""

実行ジョブ数を指定します。

--disable-xsd-check

スキーマ (.xsd) による中間ファイルのフォーマットチェックを無効化します。
本オプション有効時は、最新版の中間ファイルしか処理できなくなります。

--enable-extension ""

中間ファイルの中で、処理する拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--enable-extension "fmda,ftxa"

--disable-extension ""

中間ファイルの中で、処理しない拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--disable-extension "fmda,ftxa"

中間ファイル|フォルダー

処理するファイルやフォルダーを指定します。
バイナリー中間ファイル以外は無視します。
フォルダーを指定した場合、指定フォルダー以下のファイルを再帰的に処理します。
中間ファイル以外は無視します。

ワイルドカード（?: 一文字、*: 複数文字）に対応しています。
例) "a?b*"
一文字目が a で三文字目が b のファイルを変換します。

利用例

テキスト中間ファイルコンバーターの利用方法を説明します。

バイナリー中間ファイルの変換

binary.fmdb を binary.fmda に変換します。

3dIntermediateFileBinaryToAscii.exe binary.fmdb

フォルダーの変換

binary フォルダーやサブフォルダー内のバイナリー中間ファイルを、テキスト中間ファイルに変換します。

3dIntermediateFileBinaryToAscii.exe binary\
