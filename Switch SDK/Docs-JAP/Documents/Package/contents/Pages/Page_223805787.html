OTF の全角文字を TTF のように等幅で表示したい

質問

OTF の全角文字が、TTF で表示したときのような等幅にならなくて困っています。等幅で表示する方法を教えてください。

回答

font ライブラリでは、全角文字がフォントの意図する自然な見た目になるように、OTF に含まれるプロポーショナルメトリクス情報(palt)という情報を使ってプロポーショナルで表示しています。

ただし全角と半角の記号の文字幅を使い分けたいなどの理由で、全角文字を等幅で表示したいという場合には、以下のように fcpx ファイル内の <scalableFontDescription> タグ属性で ignorePalt を true に指定すると、palt 情報が無視されて等幅で表示されるようになります。

<scalableFontDescription boldWeight="0" borderWidth="0" ttcFontIndex="0" ignorePalt="true">

fcpx ファイルの詳しいフォーマットは「複合フォント定義ファイルフォーマット」をご覧ください。
