アルファテスト

flyt ファイルをレイアウトエディタにドロップすれば読み込んで確認することができます。

確認できるサンプル

  • Write_Sample.flyt

フラグメントオペレーションの『アルファテスト』を使用し、文字が書かれていく表現のサンプルです。

アルファテスト

  • 「P_fontN_00」のピクチャペインについて
    • 0段目のテクスチャのアルファイメージに、書かれて欲しい順に白→黒にグラデーションにします。
    • フラグメントオペレーションの『アルファテスト』の項目の『デフォルト設定を利用する』のチェックを外します。
    • 条件を「アルファ＞値」に設定します。この設定により表示非表示をテクスチャで制御できるようになります。
    • アルファテストに値を１から０に減少していくアルファテストアニメーションを設定します。
    • ※その他の文字も上記に準じた構成になっています。
  • 「P_Light_00 P_Light_01」のピクチャペインについて
    • 文字の書かれている先端に光を表示しています。
      →フラグメントオペレーションは「加算」です。
