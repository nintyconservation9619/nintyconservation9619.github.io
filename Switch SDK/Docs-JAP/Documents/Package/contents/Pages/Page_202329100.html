ペイン

レイアウトエディタウインドウのメニューバー追加項目「ペイン」について説明します。

ペイン作成 ペインを作成します。
ピクチャペインの作成は「テクスチャウインドウ」から行います。
親子設定 ペイン間に親子関係を設定します。
親子解除 設定した親子関係を解除します。
親の平行移動操作を子に影響させる チェックが入っていると親ペインの平行移動にあわせて子ペインも平行移動します。
調整点のドラッグでサイズを調整する チェックが入っている状態で調整点をドラッグすると、サイズを調整します。
チェックが外れている状態で調整点をドラッグすると、スケールを調整します。
グループを作成 選択したペインでグループを作成します。
グループを編集 選択したペインが含まれるグレープの編集を行います。
選択ペインをロック 選択したペインをロックします。
ロックをすべて解除 ペインのロックがすべて解除されます。
ロックしたペインを隠す ロックされたペインを隠し、レイアウトウインドウやビューアに表示しなくなります。
境界ペインをロック 境界ペインをロックします。
選択ペインを隠す 選択ペインを隠し、レイアウトウインドウやビューアに表示しなくなります。
ペインをすべて表示 隠したペインをすべて表示します。
隠したペインをロック 隠したペインをロックします。
整列

複数のペインを選択し、「整列一覧」から整列方法を選んで整列します。
テクスチャのサイズにリサイズ ピクチャペインのサイズをテクスチャと同じサイズにリサイズします。
プロパティ 選択したペインのプロパティウインドウが開きます。
部品ペインを再読み込み レイアウトウインドウ上の部品ペインを再読み込みします。
選択ヌル以下を部品化 選択状態のヌル以下を部品化します。
選択ペインの部品レイアウトを開く 選択状態のペインの部品レイアウトを開きます。

整列一覧

複数のペインを選択した状態で「整列」を実行すると、ペインを並び替えることができます。

整列一覧のコマンド
左揃え

選択したペインの左辺を基準になるペインの左辺に揃えます。

選択したペインの中で親子階層ウィンドウの一番上にあるペインが基準になります。
左右中央揃え

選択したペインの水平方向中央を基準になるペインの水平方向中央に揃えます。

選択したペインの中で親子階層ウィンドウの一番上にあるペインが基準になります。
右揃え

選択したペインの右辺を基準になるペインの右辺に揃えます。

選択したペインの中で親子階層ウィンドウの一番上にあるペインが基準になります。
上揃え

選択したペインの上辺を基準になるペインの上辺に揃えます。

選択したペインの中で親子階層ウィンドウの一番上にあるペインが基準になります。
上下中央揃え

選択したペインの垂直方向中央を基準になるペインの垂直方向中央に揃えます。

選択したペインの中で親子階層ウィンドウの一番上にあるペインが基準になります。
下揃え

選択したペインの下辺を基準になるペインの下辺に揃えます。

選択したペインの中で親子階層ウィンドウの一番上にあるペインが基準になります。
左右に分布

選択した複数のペインの範囲の中で、ペインの中心同士の距離が左右均等になるように配置されます。

参考：

中心同士の距離を均等にするので、大きさの違うペインが含まれる場合は、ペインの端からの距離が均等にならない場合があります。
将来的に、ペインの端からの距離を均等にするモードを追加する予定です。
上下に分布

選択した複数のペインの範囲の中で、ペインの中心同士の距離が上下均等になるように配置されます。

参考：

中心同士の距離を均等にするので、大きさの違うペインが含まれる場合は、ペインの端からの距離が均等にならない場合があります。

将来的に、ペインの端からの距離を均等にするモードを追加する予定です。
