ブレンドステート

概要

ブレンドステートは、主にパイプラインにおけるブレンディング固定機能について、いくつかのステートをまとめて表現します。

それぞれのカラーターゲットについてのブレンドステートは、ブレンドターゲットステートとして表現されます。

情報

ブレンドステートの情報は nn::gfx::BlendStateInfo として定義されています。

論理オペレーション

SetLogicOperationEnabled により論理オペレーションを有効または無効に設定し、SetLogicOperation 関数により適用する論理オペレーションを指定します。
論理オペレーションを有効に設定した場合はブレンディングは行われません。

アルファ値のカバレッジ値への変換

 SetAlphaToCoverageEnabled 関数により、マルチサンプリングを行っている際にアルファ値をカバレッジ値に変換するかどうかを設定します。

デュアルソースブレンド

SetDualSourceBlendEnabled 関数により、同じ出力バッファに 2 つのカラーを出力するデュアルソースブレンドを有効または無効に設定します。
デュアルソースブレンドを有効に設定することは、利用できるレンダーターゲットの数に制限を課します。

別々のブレンド

それぞれのレンダーターゲットについて別々のブレンドターゲットステートを使うには、SetIndependendBlendEnalbe に true を設定します。
SetIndependentBlendEnabled 関数に false を設定した場合は、SetBlendTargetStateInfoArray により指定されるブレンドターゲットステートについて、すべて 0 番目のブレンドターゲットステートの設定を反映するようになり、1 番目以降のブレンドターゲットステートは無視されるようになります。

 ブレンド定数

ブレンドファクターにおいてブレンド定数を使う場合は、SetBlendConstant によりブレンド定数の値を設定します。
赤、緑、青、アルファ要素それぞれについて値を設定します。

ブレンドターゲットステート

SetBlendTargetStateInfoArray によりブレンドターゲットステートの配列を設定します。

ブレンドターゲットステートの配列は、それぞれ順番にカラーターゲットに対応します。

ブレンディング

SetBlendEnabled 関数により、ブレンディングを有効または無効に設定します。

ブレンディングの式

以下の関数により、ブレンディングに使われる式を制御します。

要素 関数 設定するパラメーター
カラー要素 SetSourceColorBlendFactor ソースのブレンドファクター
SetDestinationColorBlendFactor デスティネーションのブレンドファクター
SetColorBlendFunction ブレンド関数
アルファ要素 SetSourceAlphaBlendFactor ソースのブレンドファクター
SetDestinationAlphaBlendFactor デスティネーションのブレンドファクター
SetAlphaBlendFunction ブレンド関数

書き込みマスク

SetChannelMask により、カラーターゲットに書き込む要素を組み合わせで制御します。
すべての要素に書き込む場合は nn::gfx::ChannelMask_All を指定してください。 

オブジェクト

ブレンドステートのオブジェクトは nn::gfx::BlendState として定義されています。

初期化と破棄

ブレンドステートの初期化と破棄はオブジェクトの規則に従います。

ブレンドステートのセット

パイプラインを使う実行モデルを利用する場合は、パイプラインの情報にブレンドステートの情報を記述します。

パイプラインを使わない実行モデルを利用する場合は、nn::gfx::CommandBuffer クラスの SetBlendState を呼ぶことによってブレンドステートをセットします。
