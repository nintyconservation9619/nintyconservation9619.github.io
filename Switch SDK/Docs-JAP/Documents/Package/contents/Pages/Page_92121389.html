ファイルイベント

ファイル操作時に実行したいユーザーコマンドファイルを設定できます。
コマンドには引数として、処理される全ファイルのリストが記述された一時ファイルのパス（例：C:\Users\UserName\AppData\Local\Temp\tmp123A.tmp）が渡されます。

1
2
3		 
C:\Data\EmitterSet1.eset
C:\Data\EmitterSet2.eset
C:\Data\EmitterSet3.eset

本設定項目は【ユーザー設定】と【プロジェクト設定】の両方にあります。
関連リンク 設定による挙動については「【ユーザー設定】と【プロジェクト設定】共通の設定項目」を参照してください。

ファイル保存前コマンド

「 ファイル保存前コマンド」は、ファイルを保存する直前に実行したいユーザーコマンドファイルを指定する機能です。

ファイル保存後コマンド

「 ファイル保存後コマンド」は、ファイルを保存した直後に実行したいユーザーコマンドファイルを指定する機能です。

ファイル削除前コマンド

「 ファイル削除前コマンド」は、エフェクトブラウザで未使用ファイルを Unused フォルダに移動する直前に実行したいユーザーコマンドファイルを指定する機能です。

ファイル削除後コマンド

「 ファイル削除後コマンド」は、エフェクトブラウザで未使用ファイルを Unused フォルダに移動した直後に実行したいユーザーコマンドファイルを指定する機能です。

ファイルオープン前コマンド

「 ファイルオープン前コマンド」は、ファイルを開く直前に実行したいユーザーコマンドファイルを指定する機能です。
