メーターパネル

現在の時間の詳細情報を表示します。

 現在は、処理負荷の情報にのみ対応しています。

画面説明

  項目 説明
A nn::audio 現在の各処理負荷の合計を表示します。
B メーター

現在の各処理負荷をバーで表示します。

nn::audio

処理負荷の現在値を表示します。
Voice, SubMix, FinalMix, Sink の処理負荷の合計値で、単位はマイクロ秒です。

メーター

各処理負荷を色付きのバーで表示します。
バー内の数値は処理負荷を表します。単位はマイクロ秒です。

以下は、バーの色とコンポーネントの対応表です。

バーの色 コンポーネント
赤色 Voice
黄色

SubMix
緑色 FinalMix
青色 Sink

操作方法

操作 手順
右ドラッグ 横方向にスクロールします。
Ctrl + ホイール 横方向に拡大、縮小します。
