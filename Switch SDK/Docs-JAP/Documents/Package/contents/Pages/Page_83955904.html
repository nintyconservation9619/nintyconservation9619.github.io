ユーザーデータ

このタブではエミッタセット単位でユーザー独自の付加情報を設定します。
ゲームアプリの機能やワークフローで利用できます。

ビット

「 ビット」は、ユーザーデータをフラグで設定できます。
以下はグループ内設定項目の一覧です。

ビット [0] ～ [15]

16 個のフラグを「 チェックボックス」ON/OFF で設定します。

数値

「 数値」は、ユーザーデータをスライダで設定できます。
以下はグループ内設定項目の一覧です。

数値1 ～ 数値2  0 ～ 255 の 256 段階の範囲で「 ダイヤル」や「 テキストボックス」により値を設定します。

設定例

以下は、ゲームアプリで「ユーザーデータ」を使った例です。

A ビットのフラグ [0] = ON　 ゲームの中ではこのエミッタセットのパーティクル位置に敵キャラを配置します。
B 数値 = 128
 ゲーム中では追従する敵キャラの種類を指定する為の ID 番号として利用します。
