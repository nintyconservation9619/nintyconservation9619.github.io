ウインドウペインを利用する

ここでは、ウインドウペインの説明を行います。
ウインドウペインは、テクスチャを複数枚組み合わせて、ウインドウ(窓枠)オブジェクトを作成するための機能です。
フレームと呼ばれる枠の部分と、コンテンツと呼ばれる枠の内側の部分に、画像を設定してウインドウを作成することが可能です。

長所と短所

ピクチャペインと比べると、1枚のテクスチャでサイズを大きく拡大した場合に起こりがちな、枠の部分テクスチャの引き伸ばしが起こりません。
その結果としてテクスチャサイズを削減する効果もあります。
反面、複雑な設定を行うと、処理負荷が高くなります。
「マテリアル設定を共有する指定」や「水平ウインドウ」を指定して、設定の自由度を下げることで処理負荷を下げることが可能です。

ウインドウペインを作成する

レイアウトファイルを新規作成し、[ 右クリックメニュー ] → [ ペインの作成 ] → [ ウインドウペイン ] を選択します。
レイアウト領域に緑枠に白色のペインが表示されます。


この状態のウインドウペインは、画像が設定されていない状態です。
正しくウインドウとして表示するためにはテクスチャを設定する必要があります。

テクスチャの設定

ウインドウペインのテクスチャ設定について説明します。

テクスチャの設定を行うには、まずテクスチャウインドウを使用して画像をテクスチャとして登録しておく必要があります。

テクスチャウインドウ テクスチャウインドウについて説明しています。( UI 説明 )


テクスチャの登録が完了したら、ペインのプロパティウインドウを開き、左ツリー項目から、ウインドウペイン情報項目を選択します。
右側に、ウインドウペインプロパティ編集ページが表示されます ( 図中 A ) 。
ここで、ウインドウの種類と設定対象となる部位を選びます。


まず、ウインドウコントロール ( 図中 C ) をクリックして、設定対象を指定します。ここでは「左上」を選択します。
4隅が赤く表示されて、画像の設定対象がウインドウフレームの 4 隅に変更されます。

さらに、左ツリー項目から、テクスチャ項目を選択しテクスチャ設定ページを表示し、テクスチャを設定します。
先ほど指定したウインドウペインの4隅に対してテクスチャの設定が行われます。

ウインドウフレームの設定は、細かく設定することができます ( 図中 B ) 。

別の部位への設定も ｢部位を選ぶ｣ → ｢設定する｣ の手順を繰り返しを行って対処します。

 


設定が完了すると図のような、ウインドウペインが表示されます。
