シザーペインを使う場合のプログラム側の対応

シザーペインについて

シザーペインとは、子階層の描画結果を指定領域で切り抜くペインです。

LayoutEditor 上でシザーペインを配置することで使用できます。

シザーペインは内部的にハードウェアの機能を使ってシザーを設定しており、描画後には本来のシザーの設定に戻す処理を行っていますが、戻すべきビューポート・シザーの範囲はあらかじめプログラム側で指定しておく必要があります。

戻すべきビューポート・シザーの範囲を、nn::ui2d::DrawInfo クラスの初期化時に以下のように指定してください。

// ビューポートとシザー設定のデフォルト値を設定する。
const int width = 1280; // デフォルトの幅を指定
const int height = 720; // デフォルトの高さを指定

nn::gfx::ViewportStateInfo  viewportInfo;
viewportInfo.SetDefault();
viewportInfo.SetWidth(static_cast<float>(width));
viewportInfo.SetHeight(static_cast<float>(height));

nn::gfx::ScissorStateInfo  scissorInfo;
scissorInfo.SetDefault();
scissorInfo.SetWidth(width);
scissorInfo.SetHeight(height);

g_pDrawInfo->SetDefaultViewportScissorInfo(viewportInfo, scissorInfo);
