ここでは、帽子の作成について説明します。

帽子作成前の確認

Mii に帽子を被せて表示をするには、頭の上半分をカットした帽子用の顔モデルを表示する必要があります。

帽子用の顔モデルは、プログラマが似顔絵ライブラリを使用して表示させます。詳しくは顔モデル構造ページの顔モデルの種類の項目を参照してください。

帽子の作成

ここでは、帽子の作成手順について説明します。

最初にデザイナーが帽子のモデルを作成し、それをプログラマが実装するという順序で説明します。

帽子のモデルデータの作成（デザイナー作業）

帽子を作成する際の基準となるサンプルを用意しています。3DCG ソフトウエアで Cap_template を表示し、確認してください。

Cap_templete には 2 つの帽子があり、すべての Mii の髪型に合うように設計されています。そのため、独自の帽子を作成する場合は、これら2つの帽子よりも小さくならないようにしてください。

野球帽のように前髪が少し見える帽子を作成する場合は Cap_template00、ニット帽のように前髪が隠れる帽子の場合は Cap_template01 を参考にしてください。また、実際に Cap_template を使って作成した野球帽とニット帽のサンプル Cap_sample があります。こちらも参考にしてください。

以下に作成時の注意点を示します。

  • 帽子モデルは、原点から少し浮いた状態の位置にあります。帽子はこの位置で作成してください。作成後は必ずピボットポイントを原点に合わせてください。
  • 帽子の底面にはポリゴンを張るようにしてください。下から見た時の髪の断面が見える恐れがあります。

帽子モデルの表示（プログラマ作業）

似顔絵ライブラリを使用して Mii の顔モデルを表示し、合わせて帽子も表示してください。

帽子は、頭の上半分がカットされた帽子用の顔モデルに装着します。帽子用の顔モデルを作成するには、nn::mii::CharModel の初期化時に渡す nn::mii::CharModelInfo に設定する CreateFlag  に nn::mii::CreateFlag_Hat を指定します。nn::mii::CharModel::GetDrawParam() で指定する nn::mii::CreateModelType を nn::mii::CreateModelType_Hat にすることで、帽子用の顔モデルを表示するためのパラメータが取得できます。

帽子は顔輪郭モデルに用意されている set_hair という Null を基準に装着します。set_hair は髪モデルの位置を示す Null で、nn::mii::CharModel::GetHatPartsTranslate() を使用することで、移動の情報を取得することができます。set_hair については被り物ページの被り物作成前の確認の項目も合わせて参照してください。

帽子作成後の確認

実際のアプリで Mii のすべての髪型に帽子を装着し、確認を行ってください。

帽子が小さすぎる場合は、もう一度、帽子作成の基準となるサンプル Cap_template よりも小さくなっていないかを確認してください。
