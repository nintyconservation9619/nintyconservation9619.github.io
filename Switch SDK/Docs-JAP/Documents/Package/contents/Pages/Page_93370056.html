エミッタ形状「立方体」

ここではパーティクルの放出位置が立方体外周のエミッタ形状「立方体」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「立方体」は、ワールド座標系に配置される立方体の外周面上でランダムな位置から箱状に広がった法線方向にパーティクルを放出します。
以下は「 設定項目」の概要です。

「エミッタ形状」を「立方体」に設定すると「 立方体」が表示されます。以下はグループ内の設定項目です。

辺の長さ

立方体の各辺の長さを X Y Z で設定します。
