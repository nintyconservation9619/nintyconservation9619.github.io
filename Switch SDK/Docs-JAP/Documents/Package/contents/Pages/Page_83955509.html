テクスチャー SRT アニメーション

テクスチャー SRT アニメーションは、place2dTexture ノードの 繰り返し UV（Repeat UV）、フレームの回転（Rotate Frame）、フレームの移動（Translate Frame）アトリビュートをアニメーションさせて設定します。

place2dTexture ノードの有効範囲（Coverage）アトリビュートは (1, 1)、オフセット（Offset）アトリビュートは (0, 0)、回転 UV（Rotate UV）アトリビュートは 0 にしておいてください。

エクスポートオプションの Texture SRT Mode で、テクスチャー SRT の計算方法を指定することができます。

テクスチャー SRT アニメーションはアニメーションを設定したアトリビュートについてのみデータが出力されます。データが出力されなかったアトリビュートは、fmd ファイルの状態で表示されます。
よって、アトリビュートが一定値でも fmd ファイルの状態と異なる場合は、アトリビュートにキーを 1 つ以上打ってからテクスチャー SRT アニメーションを出力してください。

マテリアルの反射カラー（Reflected Color）アトリビュートに接続されたテクスチャーについては、テクスチャー SRT アニメーションは出力されません。

エクスポートオプションのマテリアル圧縮が有効になっていると、マテリアルがまとめられた結果、テクスチャー SRT アニメーションが設定されていないマテリアルまでアニメーションしてしまう場合があります。
その場合はマテリアルアトリビュート設定プラグインで、テクスチャー SRT アニメーションを設定したマテリアルが圧縮されないようにしてください。

file ノードのアトリビュートに直接アニメーションを設定した場合には対応していません。
