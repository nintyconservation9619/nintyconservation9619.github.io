ソフトパーティクル「頂点で処理」

ここでは「ソフトパーティクル」の処理をパーティクルの頂点単位で行う機能「頂点で処理」について説明します。

概要

ソフトパーティクルの機能「頂点で処理」は、ソフトパーティクルの処理をプリミティブポリゴンの頂点単位で適用する機能です。
主に、プリミティブパーティクルで使用します。頂点数が多いプリミティブパーティクルほど、アルファの階調がきれいに見えます。

以下は「 設定項目」の概要です。

「ソフトパーティクル」の設定を「頂点で処理」にすると「 頂点で処理」が表示されます。以下はグループ内の設定項目です。

フェード開始α値オフセット パーティクルがモデルと重なった箇所の透明度を設定する機能「フェード開始α値オフセット」について説明します。
サンプリング半径 パーティクルがモデルと重なった箇所から厚みを持たせで透明度を設定する機能「サンプリング半径」について説明します。

関連リンク ソフトパーティクルの詳細については こちら を参照してください。
