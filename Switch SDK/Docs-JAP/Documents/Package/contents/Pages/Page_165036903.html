Audio 波形パネル

Audio が出力した波形を表示するパネルです。

画面説明


 項目 説明
A 波形リスト チャンネルごとの波形をリスト形式で表示します。
B オレンジ色の縦線 現在の時間を示します。

 nn::audio API で出力デバイスごとに音量を設定している場合は、TV への出力音量が考慮されます。

 最終的に出力される波形は、さらに後段でダウンミックスされる可能性がありますが、 Spy は、そのダウンミックス前の波形を記録します。

操作方法

操作対象 操作 手順
波形リストの波形表示部 左クリック 現在の時間を、指定位置に移動します。
右ドラッグ 波形表示を横方向にスクロールします。
Ctrl + ホイール 波形表示を横方向に拡大、縮小します。
Alt + ホイール 縦方向に拡大、縮小します。
波形リストのヘッダ部 境界線を左ドラッグ ヘッダを区切る境界線をドラッグすると、表示の縦幅を変更します。
