カラーフォーマットの選び方

質問

カラーフォーマットの選び方について教えてください。

回答

  • 通常のフォントは A8 を選びます。
  • ふち付きフォント（アルファ情報を明度とは別に持つフォント）の場合は、LA8 と 2チャンネル形式を指定します。
    • ふち付きフォントで、フチの表示 ON/OFF を変えられるのが LA4Packed形式です。
      • 圧縮ノイズが気になる場合 LA4PackedNoCompress を使います。ただし容量は2倍になります。
  • RGB~ は画像フォント用です。
    • リニアワークフローを採用している場合（担当プログラマに確認してください）は、SRGB 形式を利用します。
  • A4 と A8 、LA4 と LA8 と二つあるが、XXX4 と XXX8 でどちらを選んでも結果は同じです（内部で BC4 フォーマットと解釈される）。
    • 過去のプラットフォームで使っていたデータを同じ設定で変換できるように残されています。
