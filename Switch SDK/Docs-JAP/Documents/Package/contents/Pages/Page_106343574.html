 プロパティウインドウ

プロパティウインドウは、LayoutEditor の ペインを編集するウインドウ です。
ペインの種類によって、設定する項目が変化します。

プロパティウインドウの種類

LayoutEditor で用意されているプロパティウインドウは以下になります。

  • 拡張ユーザ情報

アニメーション関連のUI

プロパティウィンドウの設定項目に「 アニメーションマーク」が表示されている項目はアニメーションが設定できます。

「アニメーションマーク」または「項目名」上でマウスカーソルが「メニューが表示可」の場合、マウス右ボタンクリックによりメニューが表示されます。
メニューからはアニメーションに関する操作を行うことができます。以下はメニューの内容です。（メニューの内容は項目により異なります）

キーを追加 対応する項目にカレントフレームでアニメーションキーを追加します
すべてにキーを追加 対応するグループの項目すべてにアニメーションキーを追加します
Rにキーを追加 カラーのR要素にアニメーションキーを追加します
Gにキーを追加 カラーのG要素にアニメーションキーを追加します
Bにキーを追加 カラーのB要素にアニメーションキーを追加します
Aにキーを追加 カラーのA要素にアニメーションキーを追加します

アニメーションマークの色で項目の状態を確認できます。
以下は項目の表示状態とその説明です。

 項目にアニメーションキーが設定されていない状態です。

項目にアニメーションキーが設定されている状態です。
タイムラインのカレントフレームで値が確定されている状態です。

項目の値が不確定な状態です。
タイムラインを操作したりファイルの保存時に設定した値が無効になります。
タイムラインのカレントフレームにキーを追加すると「 確定状態」になります。
