カーブツリービューの右クリックメニュー

カーブツリービューの右クリックメニューは、グループやカーブの種類により変わります。
基本的には、グループやノード単位のコピーや貼り付け、カーブの削除を行います。

参考：
関連リンク DCC ツールから出力した「カラーアニメーション」「テクスチャ SRT アニメーション」のオリジナルデータを適用してアニメーションカーブを生成することもできます。

ここでは、特有な機能があるアニメーションの右クリックメニューを説明します。

シェーダーパラメータアニメーション / テクスチャ SRT アニメーション / カラーアニメーション

シェーダーパラメータアニメーション / テクスチャ SRT アニメーション / カラーアニメーションの右クリックメニューには、マテリアルシェーダーページを表示する機能があります。
この機能を使用すると、カーブツリービューの右クリックメニューで選択したモデルに対応しているマテリアルプロパティウィンドウのシェーダーページにジャンプします。

テクスチャ SRT / テクスチャ SRT EX のテクスチャ行列のモード変更

テクスチャ SRT やテクスチャ SRT EX グループ のテクスチャ行列の右クリックメニューからは、モード（テクスチャ座標の計算方式）を選択できます。
モードは、「Maya」「3ds Max」「Softimage」から選択します。

テクスチャパターンアニメーション

テクスチャパターンアニメーションの右クリックメニューには、マテリアルサンプラページを表示する機能があります。

選択したノードやカーブのみ表示する

カーブツリービュー上で選択したカーブのみ表示して編集状態にできます。

以下のカーブのみを表示

カーブツリービューで選択したノード以下にあるカーブのみを表示して、編集状態にします。


このカーブのみを表示

 カーブツリービューで選択したカーブのみを表示して、編集状態にします。

オリジナルを適用

DCC ツールから出力されたカラーアニメーション、テクスチャ SRT アニメーションのオリジナルデータからシェーダー定義ファイル( .fsdb )を参照してアニメーションカーブを生成します。

特定のオリジナルカーブを適用

DCC ツールから出力されたカラーアニメーション、テクスチャSRTアニメーションのオリジナルデータを任意に選択してアニメーションカーブを生成します。

特定のオリジナルカーブを適用を選択すると、適用するオリジナルアニメーションの選択ダイアログが表示されます。

適用したいアニメーション名を選択して、OKを押すと任意のアニメーションをオリジナルデータから適用できます。
