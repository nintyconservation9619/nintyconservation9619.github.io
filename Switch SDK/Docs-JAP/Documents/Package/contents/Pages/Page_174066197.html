Atk パフォーマンスパネル

Atk のパフォーマンス情報を表示するパネルです。

画面説明

項目 説明
atk SoundThread atk サウンドスレッド処理の開始から完了までの時間を表示します。
atk TaskThread atk タスクスレッドにおいて、ストリームサウンドのデータロード、opus のデコード処理の開始から完了までの時間を表示します。
DSP

Voice, MainMix(SubMix), FinalMix, Sink の開始から完了までの時間を表示します。

  • Voice については最初のアイテムの開始から最後のアイテムの完了までの時間です。
 

DSP の処理時間は正確ですが、開始、終了時刻は正確な値を取得できないため、精度は高くありません。
オレンジ色の縦線 現在の時間を示します。

 各表示項目について、詳しくは オーディオパフォーマンス情報の説明 を参照ください。

PC 版アプリでは、他プロセスなどの影響により、開始、終了時刻や処理時間が大きくブレることがあります。

操作方法

操作 手順
左クリック 現在の時間を、指定位置に移動します。
右ドラッグ 横方向にスクロールします。
Ctrl + ホイール 横方向に拡大、縮小します。
Alt + ホイール 縦方向に拡大、縮小します。
