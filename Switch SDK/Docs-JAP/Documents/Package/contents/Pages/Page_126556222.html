ビットマップフォントとスケーラブルフォントのサイズを合わせる

質問

ビットマップフォントとスケーラブルフォントのサイズを合わせるにはどうしたら良いか教えてください。

回答

フォントサイズに同じ値を指定するとほぼ同じ描画結果になります。
フォントコンバータで以下項目を同じ値に設定してください。

  • ビットマップフォントのサイズ値
  • スケーラブルフォントのサイズ値
    （レイアウトエディタではプロジェクトファイル、プログラムではプログラマが指定します。）
