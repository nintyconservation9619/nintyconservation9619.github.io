座標（±）

ここではエミッタの位置にオフセットを与える機能「座標（±）」について説明します。

概要

「座標（±）」は、エミッタの位置にワールド座標系でオフセットを与える機能です。
以下は「 設定項目」の概要です。

座標（±）

エミッタの位置に対するオフセットを「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は「座標（±）」の値を変更した時の比較画像です。

A (X,Y,Z) = (0,0,0) オフセットを与えていない場合のエミッタ位置です。ワールド座標系の中心に配置されています。
エミッタ位置にアニメーションを設定しています。
B (X,Y,Z) = (-20,0,-20) エミッタの位置にオフセットがかかっています。
エミッタ位置のアニメーションはオフセットがかかった状態からアニメーションしています。
