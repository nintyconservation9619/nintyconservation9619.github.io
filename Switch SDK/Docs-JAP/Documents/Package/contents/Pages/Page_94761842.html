カラーコンバイナ

ここではエミッタ内のカラー要素を合成できる機能「カラーコンバイナ」について説明します。

概要

「カラーコンバイナ」は、エミッタ内のカラー要素を合成する機能です。
以下はカラーコンバイナで扱えるカラー要素です。

テクスチャのカラー要素 テクスチャ 0 ､1 ､ 2 タブに登録するテクスチャのカラー情報です。
「テクスチャ 0」～「テクスチャ 2」で利用できます。
テクスチャ0 の歪み要素 「パーティクルシェーダ」の機能使用時、RGB 値を歪みや屈折表現として適用されます。
歪み用のテクスチャは「テクスチャ 0」に登録することで利用できます。
パーティクルのカラー要素

「カラー」タブで設定するカラー情報です。

「計算式」で利用できます。
プリミティブの頂点カラー要素 パーティクルタブで登録したプリミティブファイルが持つ頂点カラー情報です。
「プリミティブ」で利用できます。

テクスチャ 0 チャンネル

テクスチャのカラー要素を用意された式に通すことで「計算式」への入力値を決めます。
関連リンク プリセットとして用意された式の詳細については 「カラーコンバイナのチャンネル」 を参照してください。
テクスチャ 1
 演算子 上段との合成方法を、符号「×、＋、－」から選択します。
チャンネル カラーチャンネルの利用方法をテクスチャ 0 同様にプリセットから選択し、設定します。
テクスチャ 2
 演算子 上段との合成方法を、符号「×、＋、－」から選択します。
チャンネル カラーチャンネルの利用方法をテクスチャ 0 同様にプリセットから選択し、設定します。
計算式

「カラー」タブで設定する Color0（C0） や Color1（C1） の RGB 値と上段の RGB 値（Comb_RGB）を合成します。

関連リンク 合成の計算式については  「カラーコンバイナの計算式」 を参照してください。
プリミティブ
 演算子 上段との合成方法を、符号「×、＋、－」から選択します。
チャンネル プリミティブが持つ頂点カラーの利用方法をプリセットから選択し、設定します。
