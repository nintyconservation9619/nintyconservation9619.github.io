はじめに

DevMenuOverlay は、開発機にオーバーレイで情報を表示する機能です。

以下は、アプリケーション実行中に電力消費量のグラフとログをオーバーレイ表示しています。

使用方法

DevMenuOverlay の起動

DevMenuOverlay は Boot Menu が DevMenu の状態で開発機の電源を入れると自動的に起動します。

操作方法

十字ボタン上下と A/B ボタンで操作します。

QuickMenu

ホームボタンを押し続けることで表示することができます。

Sleep

開発機をスリープ状態にします。

Close Menu

QuickMenu を閉じます。

Power Consumption Meter

参考：

NintendoSDK SystemUpdater for NX 4.0.0-0.0 以降で使用可能です。 

また、DEV MODE の SDEV でのみ利用できます。 EDEV や PROD MODE の SDEV では利用することができません。

電力消費量のグラフを画面四隅のいずれかにオーバーレイ表示することができます。

赤色のグラフは CPU の電力消費量を指します。
緑色のグラフは GPU の電力消費量を指します。
青色のグラフはメインメモリの電力消費量を指します。

Log Viewer

参考：

NintendoSDK SystemUpdater for NX 5.0.0-1.0 以降で使用可能です。 

TargetManager に表示されるものと同等のログを画面上部あるいは下部にオーバーレイ表示することができます。

ログは最大40行分保存され、Quick Menu を開いている状態で、タッチによるスクロールが可能です。

注意：
  • 1回で 127 byte 以上のログを出力した場合、127 byte で切り捨てられます。
  • オーバーレイの1フレーム(1/30 秒)の間に 8 kbyte 以上のログを出力した場合、8 kbyte で切り捨てられます。

PowerMenu

電源ボタンを押し続けることで表示することができます。



Sleep

開発機をスリープ状態にします。

Shutdown

開発機をシャットダウンします。

Close Menu

PowerMenu を閉じます。
