テーブル簡易入力

ここでは「テクスチャパターンアニメ」で使用するパターンを指定し、パターンをアニメーションさせる順番を設定する機能「テーブル簡易入力」について説明します。

概要

「テーブル簡易入力」は、「テクスチャパターンアニメ」で使用するパターンを指定し、パターンをアニメーションさせる順番を設定できる機能です。
以下は「 設定項目」の概要です。

テーブル簡易入力

「テクスチャパターンアニメ」で使用するパターンを連番で指定する場合はハイフン、1コマづつ指定する場合はカンマ区切りで指定します。
コマ単位の入力を行う場合は使用パターンテーブル数と連動した「 設定可能な白いコマ」に、1コマづつアニメーションさせる順番を指定します。

関連リンク 以下は関連ページへのリンクです。

パターンテーブル 「テクスチャパターンアニメ」で使用するパターンを指定し、パターンをアニメーションさせる順番が変更できる UI 「パターンテーブル」について説明します。
