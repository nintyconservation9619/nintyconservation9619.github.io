ディスタンスフィールド

ここでは「ディスタンスフィールド」について説明します。
ディスタンスフィールドテクスチャを使用すると、エッジの鮮明さを保ったままテクスチャを拡大できます。

以下は同解像度の標準テクスチャとディスタンスフィールド用テクスチャとでビューアの見え方を比較した画像です。

A 標準テクスチャ 通常のテクスチャでは、ピクセルが斜めになる輪郭部分がカクカクして見えます。
B ディスタンスフィールド用テクスチャ ディスタンスフィールドテクスチャでは、滑らかになります。


概要

「ディスタンスフィールド」は、「バイリニア補間」との併用でテクスチャの解像度が画面の表示サイズよりも小さい場合でもきれいに見せることができる技法のことです。

バイリニア補間は以下のようにテクスチャの画素がゲーム画面で拡大表示される際に、足りない情報を補う為の補間方式です。


A 実解像度のテクスチャ画素です。
B ゲーム画面で、A の画素を拡大表示したイメージです。
C B の段階で足りない情報をバイリニア補間で補う様子です。
█ の輝度値は 0.5 、 █ の輝度値は 1.0 とします。

シェーダなどで補間に倍率をかけることで、以下のように █ の輝度値を 1.0 に持ち上げます。
これにより、シャープな輪郭が得られます。

 

A 拡大表示により、バイリニア補間された画素です。
B シェーダなどで倍率をかけることで、画素を 1.0 に持ち上げます。

以下のような拡大時の情報を格納したテクスチャ画像を用意し、バイリニア補間と併用することで、拡大時もテクスチャ解像度以上の見た目を実現できます。
このテクスチャ画像を「ディスタンスフィールドテクスチャ」と呼びます。また、この技法を「ディスタンスフィールド」と呼びます。

「ディスタンスフィールドテクスチャ」です。
高解像度で作成した画像に対して Photoshop プラグインを適用します。
使用するテクスチャの解像度はフィルタ適用時の半分程度が適切です。

テクスチャの品質向上のために

実際に使用するテクスチャ解像度よりも高解像度の状態でディスタンスフィールドテクスチャを作成し、これを縮小して使用することで品質向上できます。
以下は「ディスタンスフィールドフィルタプラグイン」を適用する画像解像度による比較画像です。

A 画像解像度 128×128 で「ディスタンスフィールドフィルタ」をかけたテクスチャを画面に表示したテクスチャです。
輪郭部分のジャギーが目立ちます。
B 画像解像度 256×256 で「ディスタンスフィールドフィルタ」をかけたテクスチャを 128×128 に縮小し、画面に表示したテクスチャです。
A と同じ解像度ですが、輪郭部が高品位です。
