EffectViewerNw4f の制限事項

EffectViewerNw4f ビューアは、EffectMaker のモデル設定ノードに登録した .fmdb ファイルを転送すると強制終了する問題があります。
以下の方法でこの問題を回避できます。

  1. あらかじめ EffectMaker とビューアを閉じておきます。
  2. \NintendoSDK\Tools\Graphics\3dTools\SwitchToNwBinaryMode.bat を実行し、MarkerCreateOldBinary を作成します。
  3. \NintendoSDK\Tools\Graphics\EffectMaker に作成した MarkerCreateOldBinary をコピーします。
  4.  EffectMaker と EffectViewerNw4f ビューアを起動し、接続状態にします。
  5. モデル設定ノードの「ファイル指定」で.fmdb ファイルを登録します。
  6. ビューアで登録したモデルが表示されていることを確認します。

「標準重力」機能の追加

放出タブに、オプション【プロジェクト設定】の設定内容から、物理的に正しい重力値を算出して入力する「標準重力」ボタンが追加されました！
これにより、これまでエフェクトデザイナ個々に見た目あわせで調整されてきた重力値は、「標準重力」により、物理的に正しい値を入力できます。
また、重力値が一意に決まっていることで、ゲーム内のエフェクトを一貫性のある動きに揃えることができます。
