GLSLC

概要

GLSLC は、シェーダソースとバリエーション情報の入力から、NVN 向けのシェーダのコンパイル結果を出力します。

提供形態

実機環境向けとして .nso, Windows 環境向けとして .dll, が提供されます。

また、.dll と組み合わせて利用するインポートライブラリとして .lib が提供されます。

製品環境における GLSLC の利用について

製品環境においては、以下の利点から、Windows 上で事前にコンパイルしたシェーダを利用することを推奨します。

  • 実機上でシェーダをコンパイルする CPU 時間を削減できます。
  • 実機上でシェーダをコンパイルするために必要となるメモリを削減できます。

しかし、製品環境においても、実機上でシェーダをコンパイルする機能を利用することができます。この場合は、上記の利点が得られないことに注意してください。
