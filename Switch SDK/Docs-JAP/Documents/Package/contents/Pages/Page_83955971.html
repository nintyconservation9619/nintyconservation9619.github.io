テクスチャパターンアニメーションを作成する

テクスチャパターンアニメーションを作成する

ここでは、ピクチャペインにテクスチャパターンアニメーション効果を設定する方法を説明します。

テクスチャパターンアニメーション作成の基本的な流れ

テクスチャパターンアニメーションアニメーションは、ピクチャペインの参照するテクスチャ番号の切り替えるアニメーションです。

  1. まず、ペインのテクスチャ設定に、テクスチャパターンリストを登録します。
  2. 次に、そのパターンリストのどのテクスチャを表示テクスチャに使用するかを、
    パターンリスト中の通し番号の形式でアニメーションキーで設定していきます。

テクスチャパターンリストの登録


テクスチャパターンリストを登録するには、ペインのプロパティウィンドウからテクスチャ項目ページを選択します ( A ) 。
テクスチャ項目ページ下部から、テクスチャパターンタブを選択します ( B ) 。

テクスチャパターンリストにはテクスチャパターンの追加ウィンドウで登録したテクスチャが表示されます。 ( A ) 
テクスチャを登録するには、追加ボタンをクリックしてテクスチャパターンの追加ウィンドウを開き、チェックを入れ、OK ボタンを押すことで、テクスチャパターンリストに登録されます。( B ) 。
登録したテクスチャの削除、順番の入れ替え操作も行うことが可能です。
前述のように、パターンアニメを作成時には、テクスチャパターンリストの番号行に表示された数値を使用して、切り替えるテクスチャを指定します。

パターンアニメーションキーフレームの作成

テクスチャパターンリストの登録が完了したら、パターンアニメーションのキーフレームを作成します。

対象となるペインを選択して、カーブエディタウィンドウを表示してください。



編集対象アトリビュートから、ペイン → ピクチャ → マテリアル → テクスチャパターン [ N ] として、テクスチャパターンアニメーション項目を選択してください ( A ) 。
N はテクスチャ番号をあらわす任意の数字です ( 例 N = 0  ) 。 [ PaneNode ] → [ Picture ] → の部分は、マテリアル情報を持つ、他の種類ペインの場合 ( 例：Window ペイン ) には、変化します。



編集対象を設定したら、任意のキーフレームを作成します。
テクスチャパターンアニメーションのカーブ形状は、補間が存在しないステップ形式です ( A ) 。また、キーの値は整数値となります。

ここで、先ほどのテクスチャパターンリストに存在しない番号を指定しないように注意してください。
不正な番号を持つキーは無視されます。

再生確認

作成が完了したら、アニメーションを再生してみます。
作成したカーブに沿って、テクスチャが切り替わっていることが確認できます。

テクスチャパターンアニメーションの保存と読み込み

テクスチャパターンアニメーションの保存は、[ メニュー ] → [ ファイル ] → [ 保存 ] → [ すべて ] とします。
レイアウト情報を保存する flyt ファイルと、 アニメーション情報を保存する flan ファイルが出力されます。

ここで、テクスチャパターンリストがレイアウトファイルではなく、アニメーションファイルに保存される点にご注意ください。

レイアウトファイル単体を読み込んでみる (  [ メニュー ] → [ ファイル ] → [ 開く ] → [ ファイル種類を指定して開く ] → [ レイアウト ]  ) とこのことが確認できます。

連番テクスチャの読み込み機能を利用した、テクスチャパターンアニメーションの作成

連番テクスチャの読み込み機能を利用すると、指定したフォルダ内に存在する、特定の命名規則にしたがって番号付けされたファイル群 ( 連番テクスチャと呼び ます ) をまとめて読み込んで、それらのテクスチャによるパターンアニメを行うペインを自動生成することができます。


連番テクスチャの読み込み機能は、[ メニュー ] → [ ツール ] → [ 連番テクスチャの読み込み ] とします。



設定ダイアログでは、連番テクスチャの入力先をフォルダで指定します。
ダイアログ内のファイルから、連番テクスチャを判定して読み込みを行います。
連番テクスチャは TextureName.XXX.tga ( XXXは任意の番号 ) のような形式で指定してください。


各種パラメータを設定し、読み込みを実行するとテクスチャパターンアニメーションをするピクチャペインがレイアウトに生成されます。
