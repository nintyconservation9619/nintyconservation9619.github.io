項目フィルタのカスタマイズ

ここでは「項目フィルタ」をカスタマイズし、アトリビュートツリーに表示する項目を絞り込む「カスタムフィルタ」の利用手順について説明します。

「カスタムフィルタ」は「項目フィルタ」をカスタマイズし、アトリビュートツリーに表示する項目を絞り込む機能です。
表示したい項目のセットを名前とつけて登録します。リストから登録したセットを選択することで再利用できます。
少ない手順で任意の項目のみを表示できる為、アニメーション編集が楽になります。

「カスタムフィルタ」の登録と利用

「カスタムフィルタ」の登録と利用手順について説明します。

1

カーブエディタウインドウにある「 カスタムフィルタ」のボタンを押します。
2 右側のテキストボックスをマウス左ボタンクリックすることにより、「新規登録（表示対象から）」が表示されます。
3 「新規登録（表示対象から）」を選択し、「フィルタの新規作成」ダイアログを開きます。
4

ダイアログ「ファイル名」に「カスタムフィルタ」の名前を入力します。

「カスタムフィルタ」を利用するさいに内容が解りやすい名前を入力します。
5

ソート表示したい項目のみ「チェックボックス ON」に設定します。
6 「OK」ボタンを押してダイアログを閉じます。
7 手順 2 の操作でリストに登録した「カスタムフィルタ」を選択し、アトリビュートツリーの項目が意図通りソートされることを確認します。
参考：

登録した「カスタムフィルタ」を削除したい場合は「カーブエディタ」ウィンドウの以下メニュー項目から行います。

  • [ 表示 ] → [ 項目フィルタ ] → [ カスタムフィルタを削除 ]

また、フィルタ内容をファイルとして入出力するには以下メニュー項目から行います。

  • [ 表示 ] → [ 項目フィルタ ] → [ インポート]
  • [ 表示 ] → [ 項目フィルタ ] → [ エクスポート ]

使用例

以下は、選択した複数ペインの 位置 X だけを表示させ、一括してカーブを編集する例です。

フィルタ未使用 フィルタ使用

ツリーにはアニメーション編集に関係のないアトリビュートが並びます。
その結果、各ペインの位置 X を選択する手間がかかります。		 本機能により各ペインの位置 X アトリビュートのみが表示されます。
その結果、アニメーションが効率的に編集できます。
