先端α、末端α

ここでは、ストライプ両端のアルファ値を設定できる、「先端α」と「末端α」について説明します。

概要

「先端α」と「末端α」はストライプの先端や末端の頂点アルファの値を設定できる機能です。
テクスチャを使用せずに、ストライプの先端または末端を透明にできます。

以下は、1 から順に 3 まで履歴が生成されてたストライプを例に、先端と末端について説明した図です。

A ストライプの末端です。
図の例ではアルファ値は 1.0 です。		 B ストライプの先端です。
図の例ではアルファ値は 0.5 です。

以下は「 設定項目」の概要です。

先端α

最後に生成されたパーティクル頂点のアルファ値を 0 ～ 1 の範囲で「 ダイヤル」や「 テキストボックス」により設定します。
値が 0 の場合、ストライプの先端から末端に向けて徐々に消えて見えます。

末端α

最初に生成されたパーティクル頂点のアルファ値を 0 ～ 1 の範囲で「 ダイヤル」や「 テキストボックス」により設定します。
値が 0 の場合、ストライプの末端から先端に向けて徐々に消えて見えます。

設定例

以下は「先端α」と「末端α」の値を設定した場合の比較画像です。

A 先端α=1、末端α=1 デフォルト設定です。ストライプ先端から末端にかけてのアルファ値は変化しません。
B 先端α=1、末端α=0 ストライプは「先端」から「末端」にかけて徐々に消えて見えます。
C 先端α=0、末端α=1 ストライプは「末端」から「先端」にかけて徐々に消えて見えます。
