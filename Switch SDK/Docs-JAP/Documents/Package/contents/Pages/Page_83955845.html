オーバービュー

ゲームにおけるエフェクト と NintendoWare エフェクトの構成要素 について説明します。

ゲームにおけるエフェクトの効果

ゲームにおいて、エフェクトを用いることで以下の様なメリットがあります。

画面に豪華さを盛り込むことができます。

背景と一緒に使用することで、そのステージの状況説明に使えます。

プレイヤーの操作と合わせることで手応え感を盛り込むことができます。

EffectMaker で使用する基本用語

EffectMaker はゲームエフェクトを作成、編集できるエディタです。
以下は EffectMaker で使用する用語についての図と解説です。

エミッタ

エミッタはパーティクルの発生源です。

放出するパーティクルに対して「初速」「寿命」「スケール」「カラー」などのパラメータを与えます。

パーティクルの発生位置はエミッタが持つエミッタ形状やエミッタトランスフォームで決定されます。
パーティクル

パーティクルはエミッタから放出される粒子（ポリゴン）です。

エミッタから初速を持って放出され、フィールドの影響を受けながら運動し、寿命が尽きたら消去されます。

EffectMaker では、大量のパーティクルの動きをまとめて制御できます。
フィールド

フィールドを設定することによって、初速度方向以外の運動を付加することができます。

フィールドには、重力や磁力、スピンなどが存在します。

NintendoWare エフェクトの構成要素

NintendoWare エフェクトは、エフェクトのデータの作成やアプリ上での再生を支援します。
NintendoWare エフェクトは、大きく４つの要素で構成されます。

   

EffectMaker

エフェクトデータを作成するための Windwos GUI ツールです。バイナリコンバータも兼務します。

アーティストは EffectMaker を利用してエフェクトを作成しエミッタセット中間ファイル( 拡張子 eset ) を出力します。

関連リンク　EffectMaker

EffectViewer

EffectMaker で作成したエミッタセット中間ファイルのプレビューを行うツールです。

Windows上で動作するアプリケーションと各プラットフォーム実機上で動作するアプリケーションが提供されます。

関連リンク　EffectViewer

エフェクトランタイム

ゲームアプリに組み込んで利用するランタイムライブラリです。

EffectMaker で作成したエミッタセット中間ファイルからコンバートしたエフェクトバイナリを読み込んで各ターゲット上でのエフェクトの 再生・管理 を行います。

関連リンク　エフェクトランタイムオーバービュー

ビューアライブラリ

EffectViewerを自作するためのランタイムライブラリです。

ビューアライブラリをゲームアプリに組み込むことで、EffectMaker とゲームアプリを連動させ、アプリ上でエフェクトの再生や編集を行うことができます。

アーティストは、アプリ上のゲームキャラクタに試作中のエフェクトをアタッチしたり、最終的な描画結果をもとにエフェクト色味の調整などを行うことができます。
