マテリアルプロパティウィンドウ サンプラページ

選択しているマテリアルのサンプラを確認して編集するページです。
サンプラへのテクスチャの追加や変更にも対応しています。

注意：

DCCツールのエクスポートプラグインのサンプラマージ設定が有効な場合に対応する UI 仕様に切り替える機能があります。

A サンプラが複数存在する場合に、リスト内で選択したサンプラを上下に移動して順序を変更します。
B サンプラを新規に追加します。
C リスト内で選択しているサンプラを削除します。
D 子マテリアルでの制限を「なし」か「独自のサンプラ追加を許可しない」に設定します。
E サンプラリストです。
サンプラの情報をリスト表示します（サンプラ単位のコピー & ペーストと子マテリアルで必須とするの設定ができます）。
F

サンプラリストで選択されているサンプラの名前です。

サンプラの値の編集と子マテリアルの制限を設定できます。
G 選択されているサンプラの DCC ツール上での扱いに関するヒントを表示します。
H 選択されているサンプラのテクスチャ情報です。テクスチャの変更も行えます。
I

選択されているサンプラの サンプリング時のラップ設定です。
J

選択されている サンプラのサンプリング時のフィルタ設定です。
K

選択されている サンプラのサンプリング時の LOD 設定です。

マテリアル参照に関する設定

ラベル名を右クリックするとコンテキストメニューが表示されマテリアル参照の設定ができます。

ラップやフィルタ、LOD を右クリックでもマテリアル参照の設定ができます。

関連リンク 親マテリアルの設定は 親マテリアルページ を参照して下さい。

子マテリアルでの制限

親マテリアルのサンプラから、子マテリアルでの制限を「なし」か「独自のサンプラ追加を許可しない」に設定します。
「独自のサンプラ追加を許可しない」に設定した場合、子マテリアルで親マテリアルにないサンプラーの追加が禁止されます。

子マテリアルで必須かどうかを設定

親マテリアルのサンプラから、子マテリアルでサンプラを必須とするかしないかを設定します。

「子マテリアルで必須とする」で「はい」を設定すると、子マテリアルのサンプラに鍵のアイコンが表示され、削除が出来なくなります。

親マテリアルが設定されていて、子に引き継がれている場合は下記のアイコンが表示されます。

サンプラの値の編集

サンプラの値に何を設定するかを指定します。親マテリアルの値を参照するか、自身に設定された値を使用するか、デフォルト値を使用するかを切り替えることができます。

値 説明
親マテリアルの値を参照する

親マテリアルで設定された値を参照して使用します。（自身に設定された値は無効になります）
上書きする 自身に設定された値で参照している親マテリアルの値を上書きします。
デフォルト値を使用する 空文字列が入ります。

子マテリアルの制限を設定

子マテリアルでの「値」の制限を設定します。

子マテリアルでの制限

説明
なし

子マテリアルの「値」を制限しません。子マテリアルでは「値」、「子マテリアルでの制限」どちらも設定できます。
親マテリアルの値で固定する 子マテリアルの「値」は必ず「親マテリアルの値を参照する」に設定されます。子マテリアルのシェーダーパラメーターは親マテリアルの値で固定されます。
必ず子マテリアルで値を設定させる 子マテリアルの「値」は必ず「上書きする」に設定されます。子マテリアルでは「値」、「 子マテリアルでの制限」は設定できなくなります。
