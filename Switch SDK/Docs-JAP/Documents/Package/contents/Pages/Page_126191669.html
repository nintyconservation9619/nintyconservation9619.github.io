境界ペインを利用する

 レイアウトに境界を作り判定領域の指定等に用いる、境界ペインに関して説明をおこないます。

境界ペインを作成する

レイアウトエディタウインドウのメニューバー追加項目 [ ペイン ] から、[ ペインの作成 ] → [ 境界ペイン ] を選択します。
レイアウト領域に、緑枠に緑色のペインが表示されます。

レイアウトウインドウに作成された、境界ペイン 親子階層ウインドウに表示された、境界ペイン

境界ペインは、その性格上、画像等の設定はしません。

プロパティウインドウの「境界ペイン」の設定には、何も表示されません。

「ペイン基本」「拡張ユーザ情報」のみの設定となります。

境界ペインの階層への組み込み

境界ペインの実際の使用方法について、サンプルを参考にして下さい。

NintendoSDK\Samples\Sources\Resources\Ui2d\cannonButton\NormalBtn_00.flyt

境界ペインを組み込んだレイアウト 境界ペインが組み込まれた親子階層

親子階層に、境界ペインが組み込まれています。

境界ペインの形状や大きさを変化させることで、判定領域が変化します。
