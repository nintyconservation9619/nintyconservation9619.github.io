デバッグ用サンプル Mii データ

開発の利便性のために、いくつのかの Mii データ（CharInfo）のサンプルファイルを用意しています。

サンプルファイルは SDK の以下の場所にあります。

$(NintendoSDK)/Resources/Mii/Common/charinfo
注意：

これらの Mii データファイルはデバッグ目的でのみ使用するようにしてください。製品で使用しないように注意してください。

Mii データファイルの利用方法

Mii データファイルは ROM ファイルシステムやホスト PC 上のファイルシステムを使用して読み込み、nn::mii::CharInfo 型にキャストして使用します。 具体的な利用例については、の「MiiSimple」を参照してください。
