「機能アニメーションで使用する区間タグの生成と設定」ウインドウ

「機能アニメーションで使用する区間タグの生成と設定」ウインドウでは以下の操作ができます。

A 未設定の機能アニメーション名
 チェックを入れると「アニメーション区間タグ編集ウインドウ」に機能アニメーションが追加されます。
B アニメーション区間 設定されているアニメーション区間の「開始フレーム」と「終了フレーム」が表示されます。
C アニメーション区間設定・長さ アニメーション区間の「開始フレーム」「終了フレーム」を設定します。結果が「長さ : 」として表示されます。
D 説明表示 アニメーション区間の説明が設定されている場合に表示されます。
E 説明文入力 アニメーション区間の説明を入力します。
