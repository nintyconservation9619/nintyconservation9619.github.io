追加サウンドアーカイブ

概要

追加サウンドアーカイブとは、通常使用するサウンドアーカイブと組み合わせて、再生するサウンドやバンクを追加するためのサウンドアーカイブです。

この機能を利用することで、以下のようなユースケースにも対応できるようになります。

追加コンテンツ配信に利用する

追加コンテンツ（サウンド、バンク）を追加サウンドアーカイブとして作成し、リリース済みのサウンドアーカイブと組み合わせて利用できます。

サウンドアーカイブのバンクは再利用して、シーケンスサウンドデータのみを追加サウンドアーカイブに含めることも可能です。

サウンドアーカイブを分割して運用する

ゲームのリソースロードの単位で追加サウンドアーカイブを作成し、サウンドアーカイブプレイヤーに動的に登録／解除することで、ユーザー管理グループの代替として利用できます。

ユーザー管理グループと異なり、追加サウンドアーカイブはメインのサウンドアーカイブとは別アーカイブになるため、以下の利点があります。

  • アーカイブサイズ、依存する範囲が小さくなるので、コンバート時間の短縮が期待できる
  • 独立性が高くなるので、複数人でメインのサウンドアーカイブと、追加サウンドアーカイブを同時編集してもコンフリクトしにくい

仕様

追加サウンドアーカイブのアイテムの扱いは、以下のようになります。

追加サウンドアーカイブに追加可能なアイテム
  • ウェーブサウンド
  • ストリームサウンド
  • シーケンスサウンド
  • バンク
  • 波形アーカイブ
追加サウンドアーカイブから参照できる
メインサウンドアーカイブのアイテム
  • プレイヤー
  • バンク
追加サウンドアーカイブに追加できないアイテム
  • グループ
  • プレイヤー

追加サウンドアーカイブはプレイヤーを持たず、メインのサウンドアーカイブに存在するプレイヤーを使用して再生します。

追加サウンドアーカイブのシーケンスサウンドは、メイン/追加サウンドアーカイブにあるバンクどちらも使用することができます。

ただし、メインのサウンドアーカイブは追加サウンドアーカイブのアイテムを参照することはできません。
追加サウンドアーカイブから別の追加サウンドアーカイブのアイテムを参照することもできません。


参考：

SoundMaker で利用できないアイテムを登録すると、コンバートエラー、または nn::atk で読み込んだ際に無視されます。
