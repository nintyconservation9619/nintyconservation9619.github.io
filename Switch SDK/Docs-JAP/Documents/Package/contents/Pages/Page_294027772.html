ビットマップフォントプロパティ

ビットマップフォントプロパティウィンドウ

以下は「ビットマップフォントプロパティウィンドウ」の UI 説明です。

A 文字の範囲グループ

複合フォントで利用する文字の範囲を指定します。
文字の範囲は最大で16個まで設定することができます。
B 16進数チェックボックス チェックを入れると開始、終了が16進数で表示されます。
チェックを外すと開始、終了が10進数で表示されます。
C 有効化チェックボックス チェックを入れると範囲設定が有効になり、開始、終了を編集できるようになります。
D 開始位置入力テキストボックス 文字範囲の開始位置を指定します。
E 終了位置入力テキストボックス 文字範囲の終了位置を指定します。
