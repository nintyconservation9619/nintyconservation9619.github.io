バンクの概念とバンクタブの解説を行います。

概要

バンクタブの役割

バンクタブは、バンクファイル（拡張子 .fbnk）の編集を行うペインです。
バンクファイルは、サウンドセットのバンクから参照され、シーケンスサウンドが再生に使用する音色として機能します。

バンクファイルの構造

バンクファイルは、楽器に対応するインストルメントと、これらの楽器をセットにして、プログラム番号を割り当てたインストリストから構成されています。
ここから後のセクションで、各要素の詳しい解説を行います。

バンクタブのタブテキスト

通常、プロジェクトに属するバンクファイルの場合、タブテキストにバンク名が含まれて「バンク名 - ファイル名」のように表示されますが、[ファイル] メニューからプロジェクト未登録のバンクファイルを読み込んだ場合などは、ファイル名のみの表示となります。
