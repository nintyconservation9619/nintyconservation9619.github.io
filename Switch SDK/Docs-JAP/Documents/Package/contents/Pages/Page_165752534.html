Mii データ

Mii データ

1 体分の Mii のデータを管理する CharInfo について説明します。

CharInfo

nn::mii::CharInfo は 1 体の Mii を構成する情報を持つデータ構造です。ニックネームや顔モデルの描画に必要な各パラメータなどの Mii の情報が含まれます。CharInfo は Mii の各パラメータの読み取りのみを許可します。

CharInfo から直接 Mii の各パラメータを取得することはできません。専用のアクセサ（nn::mii::CharInfoAccessor）を使用して各パラメータを取得します。

アプリケーション独自に Mii のデータを保存または通信経由でやり取りする場合は、この形式を使用してください。

CharInfoAccessor

nn::mii::CharInfoAccessor は CharInfo から情報を取得するための機能を提供するクラスです。

CharInfoAccessor を介して以下の Mii のパラメータが取得できます。

パラメータ 説明
ニックネーム Mii のニックネームです。文字コードは UTF 16、 NULL(0x0000)終端された文字列が格納されています。エンディアンは該当プラットフォームのエンディアンに従います。
フォントリージョン ニックネームを表示する際のフォントのリージョン（nn::mii::FontRegion）です。
お気に入りの色 Mii に設定されているお気に入りカラー（nn::mii::FavoriteColor）です。
性別 Mii に設定されている性別（nn::mii::Gender）です。
身長 Mii の身長です。有効な値の範囲は 0（nn::mii::HeightMin） ～ 127（nn::mii::HeightMax） です。
体重 Mii の体格です。有効な値の範囲は 0 （nn::mii::BuildMin） ～ 127（nn::mii::BuildMax） です。
髪の左右反転フラグ Mii の髪型パーツが左右反転しているかのフラグです。髪型が左右反転している場合は true が設定されています。
髪の種類 Mii の髪型パーツの種類です。有効な値の範囲は 0（nn::mii::HairTypeMin）～ 131（nn::mii::HairTypeMax）です。
顔輪郭の種類 Mii の顔輪郭パーツの種類です。有効な値の範囲は 0（nn::mii::FacelineTypeMin）～ 11（nn::mii::FacelineTypeMax）です。
肌の色 Mii の肌の色です。有効な値の範囲は 0 (nn::mii::SkinColorMin) ～ 9（nn::mii::SkinColorMax）です。

Mii のニックネームの取得

nn::mii::CharInfoAccessor::GetNickname() を使用することでニックネームの取得できます。GetNickname() 関数にはフォントリージョンのマスクを指定します。アプリケーションでサポートするフォントリージョンを指定してください。

取得した Mii のデータが持つフォントリージョンが指定したフォントリージョンのマスクに含まれていればそのままニックネームを返します。含まれていない場合は、アスキー文字以外を「？」に置換したものを返します。

Mii のニックネームをアプリケーションで表示する場合の注意点

Mii のニックネームは本体内蔵フォントが使用されており、アプリケーションが対応していない文字が含まれている場合があります。その場合でも、すべて本体内蔵フォントを使用して表示するか、あるいはアプリケーションが対応していない文字を代替文字に置き換えるなどの対応を行い、画面表示が崩れたりアプリケーションの進行に支障が出たりしないようにしてください。

なお、代替文字としては "□"（U+25A1）を推奨します。NXの本体機能が採用している代替文字が "□" であり、挙動が統一できるためです。アプリケーションで使用しているフォントにない文字を空白にしたり非表示にすることは推奨しません。文字が抜けることでニックネームがNGワードに見える場合があるからです。
