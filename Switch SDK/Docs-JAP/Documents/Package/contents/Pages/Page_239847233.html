ドッキングウインドウ

各操作ウインドウをドッキングさせ、作業内容に合わせて好みの配置にできるようになりました !!

「 グループウインドウ」、「 親子階層ウインドウ」、「 カーブエディタウインドウ」、「 部品パレットウインドウ」等、これらのウィンドウをまとめて表示させたり、取り外して独立した ( フローティング状態 ) ウィンドウにできます。
LayoutEditor の作業内容にあわせて好みの配置にすることができます。

ワークスペース

レイアウトエディタウインドウのツールバーにある「ワークスペース」をプルダウンから切り替えることにより、簡単にそれぞれの作業内容に適したウインドウ表示 ( モード ) に切り替えることができます。
「ワークスペース」は、「レイアウト」「アニメーション」「部品」から選択します。

「レイアウト」「アニメーション」「部品」それぞれのモード毎、別々にウインドウをカスタマイズすることができます。
モード毎にカスタマイズされたウインドウの状態は、ツールが設定を記憶していますので、ユーザー個人の使いやすいウインドウ配置設定を構築することが可能です。

文字毎の変換の簡単設定

文字毎の変換設定を簡単にできる機能「文字毎の変換のオフセット/時間幅アニメーションの簡単設定」が追加されました。

これまで「文字毎の変換」機能は、「オフセット」と「時間幅」というパラメータに対して手動でアニメーションカーブを作る必要があったため、複雑な機能になっていました。
今回追加された「文字毎の変換のオフセット/時間幅アニメーションの簡単設定」を使用すると、従来よりも簡単に設定ができます。

[ プロパティウインドウ ] → [ テキストボックスペイン ] → [ 文字毎の変換 ] → [ オフセット/時間幅アニメーションの簡単設定 ] 

文字毎の変換のオフセット/時間幅アニメーションの簡単設定ウインドウ
A 追加

「追加」ボタンで、アニメーションの設定項目 ( 簡単設定 ) を追加します。

ボタンを押す度に、追加されます。
B 削除 「削除」ボタンで、アニメーションの設定項目 ( 簡単設定 ) を削除します。
ボタンを押す度に、削除されます。
C 簡単設定 ( 1 ) 設定画面 ( 1 ) です。
「追加」ボタンにより作成された、アニメーションの設定項目 ( 簡単設定 ) です。
D 簡単設定 ( 2 )

設定画面 ( 2 ) です。

「追加」ボタンにより作成された、アニメーションの設定項目 ( 簡単設定 ) です。
「追加」ボタンを押す度に、( 3 ) 、( 4 ) と増えていきます。
E 左の文字から動かす テキストボックスの、一番左の文字から動かす場合にチェックを入れます。
テキストボックスの、一番右の文字から動かす場合にチェックを外します。
F 終了フレームを自動計算する

アニメーションの終了フレームが自動的に設定されます。
最後の文字が動いている途中でアニメーションを打ち切りたい場合は、チェックを外します。
G 開始フレーム 最初の文字が動き始めるフレームを入力します。
H 最初の文字が動き終わるフレーム 最初の文字が動き終わるフレームを入力します。
I 最後の文字が動き始めるフレーム 最後の文字が動き始めるフレームを入力します。
J 終了フレーム 終了フレームを自動計算しない場合、終了フレームを設定します。
