レイアウトエディタウインドウのメニューバー追加項目「ツール」について説明します。
未使用フォントを削除 使用していないフォントを削除します。
連番テクスチャの読み込み テクスチャのパターンアニメーションを行う連番テクスチャを自動生成する「連番テクスチャの読み込みウインドウ」が表示されます。
テクスチャ・フォント・部品の集約

レイアウトに使用しているテクスチャ・フォント・部品を1つのフォルダに集約します。
ビューアに転送 レイアウトをビューアに転送します。
機能アニメーションの確認は、こちらを利用します。
ビューアに転送（アニメーションモード） レイアウトをビューアにアニメーションモードで転送します。
実機ビューアを起動

実機ビューアを起動します。

参考：

実機ビューアにて確認するときは、実機ビューアを起動してから転送します。
現フレームをPCビューアでキャプチャ

レイアウトウインドウに表示されているフレームを PC ビューアでキャプチャします。

キャプチャした画像はデスクトップに保存されます。

例：レイアウトエディタウインドウの追加項目 [ ツール ] → [ ユーザーコマンド ] → [ notepad ]
