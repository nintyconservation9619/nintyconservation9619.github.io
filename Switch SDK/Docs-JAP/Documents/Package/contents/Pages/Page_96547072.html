成分 0 ～ 成分 3 の周期設定

ここでは波形成分「成分0 ～ 成分3」について周期を調整する機能について説明します。

概要

「成分 0 ～ 成分 3」は、各波形成分の周期を設定できる機能です。
「基本周期」に対する比率を波形成分ごとに変更することで、均一ではない波形を生成し、パーティクルが移動する軌道をランダムにできます。
以下は「 設定項目」の概要です。

「各成分の設定」を ON にすると「 各成分の「基本周期」に対する比率」が表示されます。以下はグループ内の設定項目です。

成分 0 ～ 成分 3

GPU ノイズに使用する波形成分の周期を設定します。
設定した値は「基本周期」に対して乗算し、これを各成分の周期（フレーム数）とします。

数値が大きいほど周期が長くなり、緩やかに変化する波形になります。数値が小さいと周期が短くなり、激しく変化する波形になります。
以下は、各成分の周期をもとめる計算式です。

  • 各成分の周期 : 基本周期 ×各成分の「基本周期」に対する比率

設定例

以下は GPU ノイズに使用する波形成分 0 の「各成分の「基本周期」に対する比率」を変更したときの波形成分のイメージです。
基本周期は 100 フレームとし、波形成分 1～3 の影響は無視しています。

A 0.6 デフォルト設定時の波形成分 0 のイメージです。
B 1.0 A と比較すると、波形周期が長くなり、緩やかな波形になります。
C 0.25
 A と比較すると、波形周期が短くなり、激しく変化する波形になります。
