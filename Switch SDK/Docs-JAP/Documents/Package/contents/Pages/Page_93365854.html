箱をカメラ前に固定

ここでは「範囲内ループ」で利用する「箱」をカメラ前に固定する機能について説明します。

概要

「箱をカメラ前に固定」は、「箱」の位置をカメラの視点方向に固定する機能です。
カメラを前後に移動させた場合でも、「箱」とカメラの距離は維持した状態で表示できます。
以下は の概要です。

箱をカメラ前に固定

箱の位置・大きさ・回転量はカメラ座標系で適用されます。
箱をカメラ位置に追従させることで、以下の様な用途に使用できます。

  • 特定の区画のみに雨を降らせる。
  • （ゲーム設定上は全区画に雨が降っているが、描画負荷の関係上）描画するのは自分の周りに限定にする。

内部実装

「箱をカメラ前に固定」を利用することで、計算に利用する座標系が変わります。
以下は「箱をカメラ前に固定」をON/OFFした場合の座標系の違いを図にしたものです。

A 箱をカメラ前に固定 OFF ワールド座標系 → 「箱」の座標系
B 箱をカメラ前に固定 ON ワールド座標系 → カメラ座標系 → 「箱」の座標系
