中間ファイルフォーマッター

中間ファイルフォーマッターの使用方法と注意事項を説明します。

  • 概要
  • コマンドラインオプション
  • 利用例

概要

中間ファイルを読み易くフォーマットします。

フォーマットの統一されていない中間ファイルを Diff ツール等で比較する際に、このツールであらかじめフォーマットをすると正常に比較できます。

実行ファイル (3dIntermediateFileFormatter.exe) やショートカットに、中間ファイルやフォルダーをドラッグ＆ドロップすると、中間ファイルをフォーマットして上書き保存します。
中間ファイル以外のファイルは無視します。

中間ファイルの上書きが問題となる場合は、中間ファイルをコピーした後に利用してください。

コマンドラインオプション

3dIntermediateFileFormatter.exe [--args-file] [--version] [-h] [-s] [-j ""] [--disable-xsd-check]
[--enable-extension ""] [--disable-extension ""] [中間ファイル|フォルダ]...

中間ファイルを読み易くフォーマットします。

--args-file ""

引数ファイルを指定します。
引数ファイルに記述した内容を、引数として指定できます。
引数ファイルは複数行に渡って各オプションを記述できます。
行頭に # がある行はコメントとして扱います。
オプション値指定の "" によるエスケープに改行を含めることはできません。

--version

バージョン情報を表示します。

-h --help

ヘルプを表示します。

-s --silent

処理中のメッセージ表示を抑制します。

-j "" --jobs ""

実行ジョブ数を指定します。

--disable-xsd-check

スキーマ (.xsd) による中間ファイルのフォーマットチェックを無効化します。
本オプション有効時は、最新版の中間ファイルしか処理できなくなります。

--enable-extension ""

中間ファイルの中で、処理する拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--enable-extension "fmdb,ftxb"

--disable-extension ""

中間ファイルの中で、処理しない拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--disable-extension "fmdb,ftxb"

中間ファイル|フォルダー

処理するファイルやフォルダーを指定します。
中間ファイル以外は無視します。
フォルダーを指定した場合、指定フォルダー以下のファイルを再帰的に処理します。
中間ファイル以外は無視します。

ワイルドカード（?: 一文字、*: 複数文字）に対応しています。
例) "a?b*"
一文字目が a で三文字目が b のファイルを変換します。

利用例

中間ファイルフォーマッターの利用方法を説明します。

中間ファイルのフォーマット

format.fmdb を読み易くフォーマットします。

 3dIntermediateFileFormatter.exe format.fmdb

フォルダーのフォーマット

format フォルダーやサブフォルダー内の中間ファイルを、読み易くフォーマットします。

3dIntermediateFileFormatter.exe format\
