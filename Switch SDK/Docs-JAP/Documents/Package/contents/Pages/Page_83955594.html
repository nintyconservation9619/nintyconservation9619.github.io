マテリアルプロパティウィンドウ シェーダーページ

マテリアルに設定するシェーダー定義ファイルを選択して編集するページです。

関連リンク シェーダーページで使用するスライダーコントロールの表示 / 非表示は、メインメニューのツールにあるユーザー設定で切り替える事ができます。

参考：

シェーダーページに表示されるパラメーターは、3DEditor に読み込んだシェーダー定義ファイル ( .fsdb ) に依存します。


A シェーダー定義ファイルを切替えます。
B

マテリアルに適用するシェーディングモデルを切り替えます。  
C シェーダー定義内で有効になるグループのみを表示します。
D

シェーダーパラメーターの ID を表示するか、非表示にするか切り替えます。

チェックすると、パラメーターのラベルの横に ID が併記されます。
E

カスタムラベルでラベルが変更されていた場合に元ラベルを表示します。

チェックするとカスタムラベルの編集で変更したラベルの横に（元ラベル）が表示されます。
F

シェーディングモデル間でデータを引き継ぎます。

3DEditor が初回起動の場合は、デフォルトで ON の状態です。
G

親マテリアル値で固定されているシェーダーパラメーターのコントロールを強制的に表示するかどうか切り替えます。

H シェーダーページにあるパラメーターを一括してコピーします。
I コピーしたシェーダーページのパラメーターを貼り付けます。
J 変更したパラメーターをシェーダーアサインされた時と同じデフォルトの値に戻します。
K

シェーダーパラメーターのグループです。

シェーダー定義ファイルで、１つのグループとして設定されているパラメーターが１つの枠内に表示されます。

グループに設定されていないパラメーターは枠の外に表示されます。
L

ラベル名です。

右クリックのコンテクストメニューからカスタムラベルの編集やマテリアル参照に関する設定をします。
M カーブエディタへのジャンプボタンです。
O カーブエディターへのジャンプボタンです。チャンネル毎のカーブを選択した状態で開きます。
P シェーダーパラメーターの検索をします。
「Ctrl」+「F」で検索フィールドが表示され、入力した文字列を含む項目をハイライトします。


シェーディングモデル間でデータを引き継ぐが ON の場合

シェーディングモデル間でデータを引き継ぐが ON の場合は、UI 上で表示されていない値も引き継がれます。

【具体例】

シェーダー定義ファイルで、下記のように、visible="false" になっているパラメーターで、この状態のサンプラー割り当ても引き継ぎ対象です。

vec4 param; //@@ id="param" default="1 1 0 0" visible="false"

参考：

editable が false の場合のパラメーターは、引き継がないようにしています。

シェーダーパラメーターの ID を表示

シェーダーパラメーターの ID を表示するか、非表示にするか切り替えます。
チェックすると、パラメーターのラベルの横に ID が併記されます。

ラベル名を右クリックすると、コンテキストメニューが表示されラベルの ID がコピーできます。

ラベル参照時の ID 表示

3DEditor では、同じシェーディングモデル内に含まれる他のラベルを参照することができます。
他ラベルの内容を代わりに表示したり、3DEditor 上のラベル表示の右側に参照先のラベルへページを移動できる参照ボタンを付加したりできます。

関連リンク ラベル参照についての詳細は、シェーダーアノテーションマニュアル の「ラベル属性」を参照してください。

ラベル参照時に、ID 表示が ON された場合は、参照元の ID と 内容が表示されます。

例：
下図は、カスタムbool値をラベル参照しています。
ID 表示を ON にすると、参照先のラベル（参照元のID | 参照元の内容）と表示されます。
A 参照元ラベルの ID
B 参照元ラベルの内容

親マテリアルで非表示されているコントロールの表示

親マテリアルで非表示に設定されているシェーダーパラメーターのコントロールを強制的に表示するかどうか 切り替えます。
チェックすると子マテリアルの表示を「表示しない」にしたマテリアルが表示されます。 



カスタムラベルの編集

シェーダーページにあるカスタムラベルは右クリックのコンテキストメニュー、カスタムラベルを編集で変更できます。
グループのラベル編集もできます。

カスタムラベルを変更を選択するとカスタムラベルの編集ダイアログが表示され編集できます。

また、ラベルの色変更や他の要素のラベルを参照する事も可能になりました。

ラベルの色変更や他の要素のラベルを参照する為の詳細は、別途用意されている シェーダーアノテーションマニュアル の「ラベル属性」を参照してください。

カスタムラベルの元ラベルを表示

カスタムラベルの編集で、ラベルを変更した後に元のラベルを確認できます。
チェックするとカスタムラベルの編集で変更したラベルの横に（元ラベル）が表示されます。

カスタムラベルのコピー / 貼り付け

カスタムラベルのコピー / 貼り付けは、プロパティページ設定のコピー / 貼り付けのみ対応しています。

A 選択している項目のコピーをします。
B

コピーした項目を貼り付けます。

マテリアル参照に関する設定

ラベル名を右クリックするとコンテキストメニューが表示されマテリアル参照の設定ができます。

グループやページを右クリックでもマテリアル参照の設定ができます。

関連リンク 親マテリアルの設定は 親マテリアルページ を参照して下さい。

シェーダーパラメーターの値の編集

シェーダーパラメーターの値に何を設定するかを指定します。親マテリアルの値を参照するか、自身に設定された値を使用するか、デフォルト値を使用するかを切り替えることができます。
設定によってアイコンが表示されます。アイコンは親マテリアルの適用後に最終的に採用される「値」を示すため、参照している親マテリアルの設定によっては、マテリアルに設定した「値」とアイコンの表示が一致しない場合があります。親マテリアルの適用についてはマテリアル参照を参照してください。

値 アイコン表示 説明
親マテリアルの値を参照する

親マテリアルで設定された値を参照して使用します。（自身に設定された値は無効になります）
上書きする なし 自身に設定された値を使用します。参照している親マテリアルの値は使用されません。
デフォルト値を使用する

シェーダー定義ファイルに定義されているシェーダーパラメーターのデフォルト値を使用します。

マテリアル参照設定時のカスタムラベル

  • 子マテリアルのカスタムラベルが空で、親マテリアルのカスタムラベルが空でない場合は、子マテリアルのカスタムラベルには親マテリアルのカスタムラベルが UI 上反映されます。
  • 子マテリアルのカスタムラベルが空でない場合は、親マテリアルのカスタムラベルは無視されて、子マテリアルのカスタムラベルが反映されます。
  • 子マテリアルのカスタムラベルは常に編集可能です。

子マテリアルの制限を設定

子マテリアルでの「値」の制限を設定します。設定によってアイコンが表示されます。

子マテリアルでの制限
 アイコン表示
説明
なし なし

子マテリアルの「値」を制限しません。子マテリアルでは「値」、「子マテリアルでの制限」どちらも設定できます。

親マテリアルの値で固定する

子マテリアルの「値」は強制的に「親マテリアルの値を参照する」として扱われます。子マテリアルのシェーダーパラメーターは親マテリアルの値で固定され、設定項目が表示されなくなります。
必ず子マテリアルで値を設定させる

子マテリアルの「値」は強制的に「上書きする」として扱われます。子マテリアルでは「値」、「 子マテリアルでの制限」は設定できなくなります。

関連リンク 詳しくは マテリアル参照 を参照して下さい。

カラーコントロールと数値コントロールの切り替え

型が float3 または float4 で、UI 型切替が「自動判別」の場合、カラーコントロールと数値コントロールを切り替えることができます。

ラベルを右クリックしたときのコンテキストメニューから指定します。

数値コントロールからカラーコントロールへの切り替え

この時点では数値コントロールです。

ラベル（この場合、「ライト方向 (dir_light_dir): 」）を右クリックし、コンテキストメニューから「カラーコントロールを表示」を選ぶと、カラーコントロールに切り替わります。


カラーコントロールから数値コントロールへの切り替え

この時点は、カラーコントロールです。

ラベル（この場合、「ライト方向 (dir_light_dir): 」）を右クリックし、コンテキストメニューから「数値コントロールを表示」を選ぶと、数値コントロールに切り替わります。

グループのコピー / 貼り付けと既定値に戻す

グループのラベルを右クリックすると、コンテキストメニューが開いてグループのコピーと貼り付けができます。

グループ毎の設定を既定値に戻すことも可能です。

シェーダーやシェーディングモデルが違っても、group_name が同じであれば貼り付けできます。

関連リンク group_name の詳細は、別途用意されている 3D 中間ファイル仕様書 の要素一覧にある <ui_group>　などを参照してください。

マテリアル一覧に表示する / しない

グループのラベルを右クリックして、マテリアル一覧に表示する、しないを設定できます。


スライダーコントロールの表示 / 非表示

関連リンク シェーダーページで使用されるスライダーコントロールの表示 / 非表示をメインメニューのツールにあるユーザー設定のコントロール設定で切り替えることができます。

コントロール設定で、スライダーコントロールを非表示にすると、下記図のようにスライダーがそのまま表示されるようになります。


スライダー範囲の拡大/縮小

スライダー範囲は、拡大/縮小ボタンで変更することができます。

拡大 / 縮小ボタンは、以下の間隔で幅を変更します。

..., 0.1, 0.2, 0.5, 1, 2, 5, 10, 20, 50, 100, 200, ...

ランタイム連携時の描画情報ラベルの色分け

ランタイム連携時には描画情報のラベルには常に下記の色がついている状態になります。

  • [黄色] シェーダー定義ファイルから描画情報を受け取っている状態です。
    ランタイム連携が未接続の状態でも編集可能です。
  • [青色] 3DEditorで編集して、ランタイムへ送った描画情報のパラメーターです。
  • [橙色] ランタイムから送られてきた描画情報のパラメーターです。
  • [灰色] どこからも描画情報を受け取っていない状態です。
    編集不可の状態です。

group condition 評価時のエラー表示

グループのコンディション式が評価できない時に警告マークを表示します。
警告マークにマウスを合わせるとエラー内容が表示されます。

ジャンプボタン

シェーダーパラメーターアニメーション / テクスチャー SRT アニメーション / カラーアニメーション が作成されている状態で使用するボタンです。

カーブエディタに対応するアニメーションカーブや、パラメーターがある場合にアクティブになり、ジャンプボタンを押すとカーブエディタが自動で開いて対応しているカーブやパラメーターが選択表示されます。
対応しているカーブやパラメーターがない場合は、ジャンプボタンはグレーオフしています。

関連リンク 逆のパターンとして、カーブエディタのカーブツリービューの右クリックメニューからマテリアルプロパティウィンドウ シェーダーページの対応項目を表示することも可能です。

アイコン 説明
対応するカーブやパラメーターがない状態でグレーオフしています。
対応するアニメーションやパラメーターがありますが、カーブ自体は設定されていない状態です。
同じパラメーターが複数のアニメーションにある場合は、三角マークが付き開くアニメーションを選択できます。
カーブエディタに対応するアニメーションやパラメーターがあり、カーブが設定されている状態です。
同じパラメーターが複数のアニメーションにある場合は、三角マークが付き開くアニメーションを選択できます。
対応するカーブやパラメーターに複数のカーブが設定されている状態です。
同じパラメーターが複数のアニメーションにある場合は、三角マークが付き開くアニメーションを選択できます。

三角マーク

三角マークをマウスオーバーすると、プルダウンメニューが開きパラメーターが使用されているアニメーションを選択してカーブエディタを表示できます。

テクスチャーをドラック & ドロップでサンプラーに登録

シェーダー定義内にあるテクスチャー情報表示領域に、テクスチャーをドラッグ & ドロップすることによりサンプラーにテクスチャーを追加登録することができます。

シェーダーパラメーターの検索をします。

「Ctrl」+「F」キー操作で検索フィールドが表示されます。

入力した文字列を含む項目がハイライトされます。
