「コードカバレッジ」タブの概要

「コードカバレッジ」タブでは、指定のプロファイルでどの関数が確認され、どの関数が確認されなかったかを追跡できます。これは、テストのカバレッジや予期しない動作を検出するためのカバレッジを決定する際に便利です。

サンプルプロファイリング中に確認された関数

このウィンドウには、プロファイリング中に確認された関数の名前が表示されます。リストの関数の数は、全体のコードサイズと共に見出しにレポートされます。全体のコードサイズは、フィルタが適用されていない場合にのみ表示されます。

「コピー」ボタンをクリックすると、現在のフィルタを使用して現在表示されている項目がクリップボードにコピーされます。

サンプルプロファイリング中の未確認関数

このウィンドウには、プロファイリング中に確認されなかった関数の名前が表示されます。リストの関数の数は、全体のコードサイズと共に見出しにレポートされます。全体のコードサイズは、フィルタが適用されていない場合にのみ表示されます。

「コピー」ボタンをクリックすると、現在のフィルタを使用して現在表示されている項目がクリップボードにコピーされます。

関数のフィルタ処理

「フィルタ」テキストボックスは、「関数」タブと同様に動作します。

右クリックによるコンテキストメニュー

タブ内で右クリックすると、さまざまなオプションを持つコンテキストメニューが表示されます。

  • 「関数名をクリップボードにコピー」：関数のフルネームと引数をクリップボードにコピーします。
  • 「関数の選択」：この関数が「関数」タブで選択状態になります。
  • 「アセンブリ表示」：「アセンブリ」タブにフォーカスが移動し、関数のアセンブリコードが表示されます。
  • 「詳細表示」：新しい「関数の詳細」ウィンドウが開き、関数の詳細が表示されます。
