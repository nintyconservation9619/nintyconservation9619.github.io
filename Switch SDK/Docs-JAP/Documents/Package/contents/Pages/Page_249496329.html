テンプレート一覧

エフェクト作成時のひな型となるテンプレートデータの一覧です。
ゲームでよく見かける表現を用意していますので、開発するゲームアプリケーションに合わせ、テクスチャ等を追加、変更して利用できます。

また、エフェクトで汎用的に用いられるパーツを「プリセット」として用意しています。
作成中のエフェクトデータにプリセットからエミッタを読み込んで利用できます。
テンプレートデータが利用しているプリセットについては表の「使用プリセット」を参照してください。
関連リンク 「プリセット」からエミッタを追加する方法については こちら を参照してください。
