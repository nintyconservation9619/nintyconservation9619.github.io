概要

NintendoWare AudioToolKit ライブラリ(以下、Atk ライブラリ) は、ゲームアプリケーションのサウンド再生を担うライブラリです。このライブラリを利用することで、NintendoWare SoundMaker (以下、SoundMaker) などのツールで作成されたサウンドデータをゲームアプリケーションで再生できるようになります。

このドキュメントでは、Atk ライブラリを使ってサウンドを再生するプログラムを組むために必要な事項を説明しています。まず、開発環境の構築方法について説明し、続いてサンプルプログラムを眺めながらライブラリの具体的な使い方を説明します。

個々のクラスや関数の詳細な説明については、関数リファレンスをご覧ください。
