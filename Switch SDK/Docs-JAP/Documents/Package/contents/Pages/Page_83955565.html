セマフォ

概要

セマフォは、GPU における異なるキューの間をコマンドバッファーの単位で同期させるための機能を提供します。

異なるキューが利用可能かどうかはプラットフォームと低レベルグラフィックス API に依存し、利用可能でない場合はそれぞれのコマンドは直列に実行されるため、セマフォがその効果を発揮することはありません。
たとえば、コピーキューがデータのコピーを終えてからグラフィックスキューにレンダリングを始めさせるなど、異なるキューが利用可能である場合にそれらのキュー間で同期をとるためにセマフォを使います。

 情報

セマフォの情報は nn::gfx::SemaphoreInfo として定義されています。

オブジェクト

セマフォのオブジェクトは nn::gfx::Semaphore として定義されています。

初期化と破棄

セマフォの初期化と破棄は通常のオブジェクトの方法に従います。

セマフォによる同期

セマフォによる同期は、キューに実行させたコマンドを他に待機させたい側がセマフォをセットし、他が実行したコマンドを待機したい側がセマフォを同期することによって行われます。

セット側

同期側に実行を待機させたいコマンドをすべてキューに提出したあと、nn::gfx::Queue の SetSemaphore を呼ぶことによって、セマフォをセットするコマンドをキューに提出します。このセマフォを待機することは、それらのコマンドが GPU によってすべて実行完了するまで待機することを意味します。

同期側

待機したいセマフォを引数として、nn::gfx::Queue の SyncSemaphore を呼ぶことによって、そのセマフォを待機するコマンドをキューに提出します。
