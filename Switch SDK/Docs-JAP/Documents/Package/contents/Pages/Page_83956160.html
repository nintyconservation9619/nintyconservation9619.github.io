インストルメントのパラメータについて

ここではインストルメントのパラメータの種類と、その概念を解説します。

インストパラメータとリージョンパラメータ

SoundMaker のバンクタブでは、内部的にインストルメントが持つパラメータ（全てのリージョンに共通）と、リージョンが持つパラメータを区別しています。

前者をインストパラメータ、後者をリージョンパラメータと呼びます。

各パラメータの扱い方

ここではバンクタブで設定する各パラメータについて、インストパラメータとリージョンパラメータのどちらで扱うことが可能かをまとめます。

エンベロープ

インストパラメータかリージョンパラメータの何れかで設定が可能です。

エンベロープの値をインストパラメータとして扱う（インストルメント単位で設定）か、リージョンパラメータとして扱う（リージョン単位で設定）かは、インストルメントごとに設定することができます。
エンベロープは通常インストパラメータとして扱いますが、マルチリージョンのインストルメントに対して、リージョンパラメータのエンベロープを設定することにより、リージョン単位で個別にエンベロープの値を設定することができます。
適用対象の設定は、エンベロープの適用対象の選択を参照してください。

現在のエンベロープの適用対象は、インストリストのエンベロープの表示で知ることができます。
インストリストの表示が数値で表示されているものは「インストルメント単位」、表示が "*" になっているものは「リージョン単位」となっています。

ひとつのインストルメントに対して、インストパラメータとリージョンパラメータのエンベロープを同時に設定することはできません。

パン

リージョンパラメータとしてのみ扱い、常にリージョン単位で指定を行います。

音量

インストパラメータとリージョンパラメータの両方で、独立して設定が可能です。

ピッチ

インストパラメータとリージョンパラメータの両方で、独立して設定が可能です。

パンモードについて

インストルメントのパンモードは、"デュアル" 固定になっています。
