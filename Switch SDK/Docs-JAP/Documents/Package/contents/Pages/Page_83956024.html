OS ユーティリティクラス

ここでは、OS ライブラリが提供するユーティリティクラスについて説明します。

ユーティリティクラスは、OS ライブラリが提供する各機能をクラスの形で利用できるようにしたものです。

OS ライブラリが提供する機能については、OSライブラリ／マニュアル／機能説明 を参照して下さい。
