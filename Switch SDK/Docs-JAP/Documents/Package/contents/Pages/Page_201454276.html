Atk シーケンス変数パネル

Atk のシーケンス変数を表示するパネルです。

評価中の機能です。予告なく変更・削除される場合があります。
 シーケンス変数はオーディオフレームの単位で変化しますが、現在の実装では、アプリケーションからの転送処理をこれより長い時間間隔でしか行えないため情報が欠落します。

画面説明

   項目 説明
A シーケンスサウンドのリスト 再生されたシーケンスサウンドをリストで表示します。
B 変数の表示

グローバル、ローカル、トラックの各変数の値をリストで表示します。

カラムの数字は変数の番号になります。

操作方法

 操作 手順
左クリック ローカル、トラック変数を表示するシーケンスサウンドを選択します。
