ファイルビュー

ファイルビュー

3DEditor に読み込まれている中間ファイルを表示、選択するビューです。
ファイルビューの上部にはファイル一覧が表示され、下部には選択されているファイル情報が表示されます。

関連リンク 読み込まれたファイルに対して操作 ( 右クリックメニューなど ) します。

A 現在読み込んでいるプロジェクトです。
B シーン
シーンファイルを一覧表示します。
シーンファイルは、アニメーションセット内に作成されます。
シーンアニメーション （ カメラ / ライト / フォグ ） は、シーンファイルの子項目として表示されます。
C モデル
モデルファイル ( .fmdb / .fmda ) を一覧表示します。
バイナリ中間ファイルの場合は .fmdb、テキスト中間ファイルの場合は .fmda です。
バインドされているアニメーションは、アニメーションセットに格納されて、接続したアプリケーションで同時に再生できます。
※モデルファイル名は、太文字で表示されます。
D アニメーション
アニメーションファイルを一覧表示します。
E テクスチャ
テクスチャ中間ファイル ( .ftxb、/ .ftxa ) を一覧表示します。   
F シェーダ定義
シェーダ定義中間ファイル ( .fsdb ) を一覧表示します。
G 三角マークをクリックすると、ファイル情報の表示 / 非表示の切り替えができます。
マークの下部分をドラッグすると、ファイル情報の表示領域を変更することができます。
H 全てのファイルに共通な情報が表示されます。
I ファイルの種類毎に固有の情報が表示されます。

プロジェクト

  • 読み込んでいるプロジェクトを閉じます。
  • プロジェクトを上書き保存します。
    プロジェクトは、プロジェクトファイル ( .f3pj) として保存されます。
  • プロジェクトに名前を付けて保存します。
  • プロジェクトの名前を変更します。
  • プロジェクトを開き直します。
    最終保存時点の状態で開き直しますので、その間の編集は無効になります。
    未保存の編集がある場合は、警告ダイアログが表示されます。
  • プロジェクトファイルのある場所をエクスプローラで開きます。

シーングループ

  • シーンアニメーションを選択して開きます。
  • シーンアニメーションを新規に作成します。
    選択すると、アニメーションの新規作成ダイアログが表示され任意の名前でシーンアニメーションファイルを作成できます。
    関連リンク シーンアニメーションの作成は、アニメーショングループや、ファイルメニューの新規作成からも作成できます。
  • アニメーションセットを新規に作成します。
  • バインドされているアニメーションセットとアニメーションのバインドをすべて解除します。

関連リンク メインメニューのツールや、ツールバーにある、オートアニメーションバインドモードを ON にして「シーンアニメーションを開く」を選択すると、自動的にシーンのアニメーションセットが作成されます。

既定アニメーションセット / アニメーションセット ( シーン )

  • アニメーションセット内に既存のシーンアニメーションファイルを開きます。
  • アニメーションセットに新規にシーンアニメーションを作成します。
  • アニメーションセット内にある全てのシーンアニメーションのバインドを解除します。
  • アニメーションセットを削除します。
    アニメーションセット内にあるアニメーションファイルはバインドを解除されます。
  • アニメーションセットを削除します。
    アニメーションセット内にあるアニメーションのバインドを解除して 3DEditor から閉じます。
  • アニメーションセットをアプリケーションでプレビューします。
  • アニメーションファイルの名前を変更します。

シーンファイル

  • シーンにバインドされているシーンファイルのバインドを解除します。
  • 選択したシーンファイルを閉じます。
  • 選択したシーンファイルを上書き保存します。
  • 選択したシーンファイルに名前を付けて保存します。
  • シーンアニメーション（カメラ/ライト/フォグ）を新規に作成します。
    作成するアニメーション毎にアニメーションの新規作成ダイアログが表示されます。
  • シーンファイルの名前を変更します。
  • シーンファイルを開き直します。
    最終保存時点の状態で開き直しますので、その間の編集は無効になります。
    未保存の編集がある場合は、警告ダイアログが表示されます。
  • 選択したシーンファイルが保存されている場所をエクスプローラで開きます。

モデルグループ

  • モデルファイルを選択して開きます。
  • モデルグループ内にあるモデルファイルを全て閉じます。
  • 選択したモデルを閉じます。
  • モデルグループ内にあるモデルファイルと参照ファイルを全て保存します。
    選択すると、場所を指定して保存ダイアログが開きます。

既定アニメーションセット / アニメーションセット ( モデル )

  • アニメーションセット内に既存のアニメーションファイルを開きます。
  • アニメーションセットに新規にアニメーションを作成します。
  • アニメーションセット内にある全てのアニメーションのバインドを解除します。
  • アニメーションセットを削除します。
    アニメーションセット内にあるアニメーションファイルはバインドを解除されます。
  • アニメーションセットを削除します。
    アニメーションセット内にあるアニメーションのバインドを解除して 3DEditor から閉じます。
  • アニメーションセットをアプリケーションでプレビューするか切り替えられます。
  • アニメーションセット内にあるアニメーションを全て一時停止します。
  • アニメーションセット内にあるアニメーションを全て再生します。
  • アニメーションファイルの名前を変更します。
  • スケルタルアニメーションのリターゲット元モデルを設定します。
    設定されたリターゲット元モデルの確認や変更は、スケルタルアニメーションウィンドウのプレビューページでも行うことができます。 
参考：

スケルタルアニメーションのリターゲットは、同じスケルトン階層を持つモデルに対して有効です。

モデルファイル

  • アニメーションファイルを開きます。
  • アニメーションファイルを新規作成します。
    関連リンク 選択すると、アニメーションの新規作成ダイアログが表示されます。
  • 関連リンク アニメーションセットを新規作成します。
  • 選択したモデルにバインドされているアニメーションセットとアニメーションのバインドをすべて解除します。
  • オブジェクトビューでのモデル表示 / 非表示を切り替えます。
  • 選択したモデルと参照ファイルを閉じます。
  • 選択したモデルファイルのみを閉じます。参照しているアニメーションファイルなどは閉じません。
  • 選択したモデルファイルを上書き保存します。
  • 選択したモデルファイルに名前を付けて保存します。
  • 選択したモデルファイルと参照されているファイル（アニメーションや、テクスチャなど）を保存します。
    場所を指定して保存ダイアログが開きます。
  • 選択したモデルファイルをマージ対象としてマージの設定をします。
    関連リンク選択すると、マージダイアログが表示されます。
  • 選択したモデルファイルの名前を変更します。
  • 選択したモデルファイルを開き直します。
    最終保存時点の状態で開き直しますので、その間の編集は無効になります。
    未保存の編集がある場合は、警告ダイアログが表示されます。
  • 選択したモデルファイルの場所をエクスプローラで開きます。

ユーザーコマンド / ランタイムユーザースクリプト

モデルファイルを右クリックすると開きます。（チーム設定で何かコマンドを設定している必要が有ります）

  • 特定のモデルの特定のマテリアルの量子化設定を削除するなど、特定のモデル中間ファイルに対する任意の処理をコンテキストメニューから実行できます。
    関連リンク 詳しくは UserCommand を参照してください。
  • 特定のモデルにフォーカスする処理など、接続先アプリケーション上の特定のモデルに対する任意の処理をコンテキストメニューから実行できます。
    関連リンク 詳しくは RuntimeUserScript を参照してください。
注意：

「ユーザーコマンド」と「ランタイムユーザースクリプト」はチーム設定で何かコマンドを設定しないとUI上には表示されません。
 「ランタイムユーザースクリプト」を実行するにはランタイムとの接続が必要です。

アニメーションの作成

ファイルビューに表示されているモデルファイル ( .fmdb / .fmda ) か、アニメーションファイルグループを右クリックして「アニメーションの新規作成」を選択します。

選択すると、アニメーションの新規作成ダイアログが開きます。

  • 名前
    アニメーションファイルの名前を入力します。
    モデルファイルの右クリックメニューでアニメーションを作成する場合は、モデルファイル名と同じ名前がデフォルトで入ります。
  • アニメーションの初期設定
    アニメーションのフレーム数の設定と、ループ再生の On / Off が切り替えられます。

    対応しているアニメーションのフレーム数は、0 ～ 16777215 で設定可能です。

フレーム数設定の右クリックメニュー

アニメーションの新規作成ダイアログのフレーム数を設定するテキストボックス上でマウスの右クリックを行うと、下記コンテキストメニューが表示されます。

  • コピー
    テキストボックス内のフレーム数をコピーします。
  • 貼り付け
    コピーしたフレーム数を貼り付けます。
  • 最大値を入力
    3DEditor のカーブエディタが対応している最大フレーム数 （ 16777215 ） を自動で入力します。

同名アニメーションについて

3DEditor は、パスが違う同名のアニメーションファイルを読み込むことができます。
また、パスが確定されているアニメーションファイルを読み込んでいた場合は、同名のアニメーションを新規に作成することもできます。

同名ファイルは、ファイルビュー上で判断できるようにファイル名の後に[ 1/2 ][ 2/2 ]が表示されます。

ファイルビュー上では、同名アニメーションに対して以下の制限がありますので、ご注意ください。

  • 同じモデル以下で同名アニメーションを新規作成することはできません。
  • 名前をつけて保存で、既存ファイルに上書きすることはできません。
  • 移動やコピー先のモデルに同名のアニメーションがバインドされている場合は、移動やコピーができません。
  • 移動やコピーするアニメーションセットに含まれるアニメーションと、同名のアニメーションが移動先のモデルにバインドされているときはアニメーションセットは移動できません。
  • パスが未確定のアニメーションと同名のアニメーションは新規作成することはできません。

アニメーションセット

アニメーションセットとは、複数のアニメーションをまとめたものです。
アプリケーションで再生すると、アニメーションセットに含まれるすべてのアニメーションが同時に再生されます。

アニメーションセットには任意に作成するものと、デフォルトで用意されている「既定アニメーションセット」があります。

既定アニメーションセット
3DEditor では、シーンやモデルに最低一個のアニメーションセットが必要になりますので、デフォルトで「既定アニメーションセット」が用意されています。
任意のアニメーションセットが作成されていない場合に、3DEditor でアニメーションの作成や、読み込みを行うと「既定アニメーションセット」に格納されます。

シーンや、モデルにバインドされるアニメーションはアニメーションセットの子階層として表示されます。

子階層として表示されるアニメーションは、再生 ON / OFF アイコンや、プレビューアイコンで、それぞれ「アニメーションの再生」 「プレビューの表示・非表示」を切り替えることができます。

アニメーションセット内のアニメーション適用順の変更

アニメーションセット内にあるアニメーションは、一番下のアニメーションから適用されプレビューされます。

適用順を変更したい場合は、アニメーションファイルをドラッグして、挿入したい場所にドロップしてください。

アニメーションセットの作成

ファイルビューに表示されているシーングループや、モデルファイル ( .fmdb / .fmda ) を右クリックして「アニメーションセットの新規作成」を選択します。

選択すると、アニメーションセットの新規作成ダイアログが開きますので、任意の名前を入力します。
※ ダイアログには、初期設定で「新規アニメーションセット x」と名前が入っています。 x には数字が自動的に割り振られます。

OK ボタンを押すと、シーングループやモデルファイルの子階層に「アニメーションセット」が作成されますので、まとめたい「アニメーション」をドラッグ＆ドロップします。

アニメーションセットの横にある目アイコンをクリックして再生状態にした後、アプリケーションに接続するとセットに含まれるすべてのアニメーションが同時に再生します。
