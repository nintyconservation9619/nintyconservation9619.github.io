エミッタ

パーティクルの発生源「エミッタ」について説明します。
以下はエミッタの特徴です。

  • エミッタはゲームエフェクトとして描画するものではありません。
  • パーティクルを放出するさいに、「初速」「寿命」「スケール」「カラー」などのパラメータを与えます。
  • エミッタにはさまざまな形状があり、形状により、パーティクルの発生位置や初速方向が異なります。
  • エミッタにはパーティクル 位置 や 速度 の処理方法によりタイプが異なります。
