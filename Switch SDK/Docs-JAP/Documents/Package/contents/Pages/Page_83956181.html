ツールバー ( カーブエディタ )

カーブエディタのツールバーは作業スタイルにより、レイアウトの変更が可能です。

A キーモード、キーの種類、キーフレームや値のクランプがアイコンで選択できます。
各アイコンにオンカーソルすると、機能について説明されます。
B カーブビューの表示方法を切り替えることができます。
各アイコンにオンカーソルすると、表示方法が説明されます。
C アニメーションの再生 / 一時停止を操作します。
一時停止したアニメーションは、ファイルビューのファイル名の右側に一時停止アイコンが表示されます。
D カレントタイムのクランプ機能を ON / OFF します。
E カーブやキーのペースト方法を選択します。
F カーブビューで選択したキーのフレーム数と値を数値入力で設定します。
G

選択したキーのスロープを数値入力で設定します。
H

倍数スナップ

追加するキーのフレームと、値のスナップ単位をプルダウンメニューから選択します。
I

キースナップを ON にした状態でカーブビュー上のキーを移動すると、カーブビュー上にある他のキーに対して、フレームと値がスナップするようになります。
スナップしたキーに対してカーブビュー上で点線が表示されます。 

C キーでキースナップの ON / OFF を切り替えることができます。

ツールバーアイコン

カーブエディタのツールバーで使用されているアイコン群です。

アイコン 説明
キーモードを「複数移動」にします。選択した複数のキーを同時に移動します。
キーモードを「最も近いキーの移動」にします。カーブビュー上でマウスをドラッグした位置から最も近いキーを移動します。
キーモードを「スケール操作」にします。カーブビュー上で、マウスを中ドラッグした箇所を基準にカーブのスケールを操作できます。
キーモードを「キーの挿入」にします。（カーブ形状を崩さずにキーを挿入します）
キーのスロープは挿入した箇所の前後にあるキーに合わせて設定されます。
キーモードを「キーの追加」にします。
キーのスロープは追加した箇所の前後のキーの位置に関係なく水平に設定されます。
カーブの補間タイプを「エルミート」にします。
カーブの補間タイプを「リニア」にします。
カーブの補間タイプを「ステップ」にします。
キーのハンドルスロープを「スプライン」します。
キーを選択して実行した場合は、選択したキーのスロープがスプラインになります。
スロープを選択して実行した場合は、選択したスロープがスプラインになります。
キーのハンドルスロープを「リニア」にします。
キーを選択して実行した場合は、選択したキーのスロープがリニアになります。
スロープを選択して実行した場合は、選択したスロープがリニアになります。
キーのハンドルスロープを「フラット」にします。
キーを選択して実行した場合は、選択したキーのスロープがフラットになります。
スロープを選択して実行した場合は、選択したスロープがフラットになります。
キーのハンドルスロープを「自動調整」にします。
このボタンを ON にしていた場合、キーの追加 / 削除時に前後のキーのスロープを スプラインになるように自動調節を行います。
OFF の場合は、キー追加時のスロープはフラットにして 追加 / 削除時の前後キーのスロープには影響を及ぼしません。
キーのハンドルスロープを分割して編集します。
カーブビューで選択したキーのスロープが分割状態の場合は、自動的にアイコンが選択状態になります。
キーのハンドルスロープを連結して編集します。
カーブビューで選択したキーのスロープが連結状態の場合は、自動的にアイコンが選択状態になります。
フレームをクランプします。クランプする範囲の左右に緑線が表示されます。
値をクランプします。クランプする範囲の上下に緑線が表示されます。
全てをフレームに表示します。
アニメーションの再生範囲をフレームに表示します。
値のクランプ範囲をビューに表示します。複数カーブが選択されている場合は、いちばん広い範囲に合わせます。
カレントタイムをビューの中心にして表示します。
編集しているアニメーションを一時停止状態にします。一時停止するとファイルビューに表示されているファイルに一時停止アイコンが表示されます。

カーブやキーのペースト方法

カーブエディタのペースト方式は以下の方法があり、プルダウンメニューから選択できます。

ペースト方式 説明
挿入 コピーしたキーを挿入してペーストします。
ペースト位置よりも後にあるキーのフレームは、ペーストしたカーブのフレーム分だけ増加します。
置換 コピーしたキーを置換してペーストします。
既にあるキーはペーストしたキー（カーブ）に置き換えられます。
融合 コピーしたキーを融合してペーストします。
既にあるキーを残し、キーをペーストします。
結果、既にあるキーとペーストしたキーが融合したかたちのカーブが作成されます。
ペースト位置 説明
絶対 ペースト先にあるキーの位置に関わらず、コピー元の値でペーストされます。
相対 ペースト先にあるキーの相対位置に移動してペーストされます。
