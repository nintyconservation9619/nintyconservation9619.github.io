ファイルパス

ここではプレビューファイルの保存先パスを確認できる機能「ファイルパス」について説明します。

概要

「ファイルパス」は、ワークスペースで選択しているプレビューファイルの保存先パスを確認できる機能です。
以下は「 設定項目」の概要です。

ファイルパス

保存先のパスを表示します。以下は各ボタンの説明です。

プレビューを現在の設定に上書きして読み込み 開いたダイアログで選択したプレビューファイル（.prev）を現在の設定に上書きして読み込みます。
プレビューを追加して読み込み 開いたダイアログで選択したプレビューファイル（.prev）をエミッタセット以下に追加して読み込みます。
プレビューを保存 開いたダイアログで選択したプレビューファイル（.prev）に上書き保存します。
プレビューを別名で保存 開いたダイアログで選択したプレビューファイル（.prev）に名前を付けて保存します。
