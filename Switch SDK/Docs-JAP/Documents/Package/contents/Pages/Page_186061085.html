グローバルカラーアニメーションのUI

「グローバルカラーアニメーションのUI」ではエミッタ時間によるカラーやアルファにアニメーションを設定できます。

A 行を追加 選択している行の下に、新たな行を追加します。
B 行を削除 選択している行を削除します。
C 行を複製 選択している行を複製します。
D 再生方法 ループ再生させるかどうかを切り替えます。
E 補間タイプ アニメーションの補間タイプを選択します。
F クリア アニメーションの編集内容をリセットし、デフォルトの状態に戻します。
G テーブル 各行に値を設定しアニメーションを編集します。

エミッタ時間とパーティクル時間 エフェクトアニメーションで扱う時間軸「エミッタ時間」と「パーティクル時間」について説明します。
カラー編集のUI 選択しているカラーやアルファの値を設定できる UI です。
グローバル乗算カラー・アルファ エミッタから放出されるパーティクル全体のカラーやアルファに影響を与える機能「グローバル乗算カラー・アルファ」について説明します。
「 エミッタ時間アニメ」を ON にすることで、本UIが使用可能になります。
