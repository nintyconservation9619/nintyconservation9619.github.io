文字毎の変換の簡単設定機能を利用する

文字毎の変換の簡単設定機能を使ったアニメーションの設定方法について説明します。

文字毎の変換とは

  • 「文字毎の変換」とは、下図の (1)～(5) のように、テキストボックスペインのテキストを 1 文字毎にアニメーションさせる機能です


  • これまでこの「文字毎の変換」機能は、「オフセット」と「時間幅」というパラメータに対して手動でアニメーションカーブを作る必要があったため複雑な機能になっていましたが、「簡単設定機能」を使うことで従来よりも解りやすく設定できるようになりました。

LayoutEditor での設定方法

  • ここからは実際に LayoutEditor 上で上記のアニメーションを作成する手順を説明していきます。

手順１：テキストボックスの「文字毎の変換」機能を有効にする

  1. テキストボックスをダブルクリックしてプロパティウィンドウを開きます。
  2. 左ツリーから、｢テキストボックスペイン｣を選択し、「文字毎の変換」グループの「有効」にチェックを入れます。チェックを入れると、「文字毎の変換」機能が有効になります。

手順２：テキストボックスにアニメーションカーブを設定する

  1. カーブエディタを開きます。
  2. ｢文字毎の変換｣属性を有効にしたテキストボックスに、アニメーションさせたいカーブを作成します。1 文字あたり 50 フレーム分動かしたい場合は 50 フレーム分のカーブを作成します。


  • 下図では、「ABCDE」のテキストを設定したペインに対し、0～25 フレームで Y 座標を 0 から 100 に上昇させ、61～66 フレームで Y 座標を 100 から 0 に下降させるカーブを作成しています。

手順３：簡単設定で文字毎のアニメーションを設定する

  1. テキストボックスをダブルクリックして、プロパティウインドウの「テキストボックスペイン」ツリーから、「文字毎の変換」グループの「オフセット/時間幅アニメーションの簡単設定」ボタンをクリックします。
  2. 出てきたダイアログの「追加」ボタンで、アニメーションの個数だけ設定項目を追加します。
  3. テキストボックスの一番左の文字から動かしたい場合は「左の文字から動かす」にチェックを入れ、一番右の文字から動かしたい場合はチェックを外します。
  4. 最初の文字が動き始めるフレームと動き終わるフレームをそれぞれ、「開始フレーム」「最初の文字が動き終わるフレーム」に入力します。
  5. 最後の文字が動き始めるフレームを「最後の文字が動き始めるフレーム」に入力します。


  • 下図では、先ほど作成したカーブに従って、0～25 フレームと 61～66 フレームで最初の「A」の文字が動くように設定しています。
  • 「A」「B」…「E」と順番にアニメーションしていきますが、最後の「E」の文字が動くのは、25(～50) フレームと 95(～100) フレームになるように設定しています。

  • 「終了フレームを自動計算する」をチェックしておくとアニメーションの終了フレームが自動的に設定されますが、もし最後の文字が動いている途中でアニメーションを打ち切りたい場合は、チェックを外すことで明示的に「終了フレーム」を設定することができます。

完成

  • 以上の手順で、文字毎の変換の「オフセット」と「時間幅」のアニメーションが自動的に作成されます。
  • 例題として作成してきたテキストボックスには下図のアニメーションが設定されています。

  1. 0～25 フレームで、最初の「A」の文字が Y 方向を 0 から 100 に上昇します。
  2. 25 フレームになると、最後の「E」の文字が同様の動作を始めます。動作が終了するのは 50 フレーム目です。
  3. 51～60 フレームには文字毎の変換の設定がないため、そのままの状態で静止しています。
  4. 61～66 フレームで、最初の「A」の文字が Y 方向を 100 から 0 に下降します。
  5. 95 フレームになると、最後の「E」の文字が同様の動作を始めます。動作が終了するのは 100 フレーム目です。
  • 「文字毎の変換」機能の詳細に関しては「文字毎の変換」のページをご覧ください。
