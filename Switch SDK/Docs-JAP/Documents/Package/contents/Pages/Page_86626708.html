継承設定「描画パス」

ここではチャイルドエミッタの継承設定「描画パス」について説明します。

概要

継承設定「描画パス」は、上位ノードの親エミッタから描画パスの情報を継承できる機能です。
以下は「 設定項目」の概要です。

描画パス

親エミッタの「描画パス」設定を自身の描画時に考慮するかを「 チェックボックス」ON/OFF で切り替えます。ON にした場合、常に親エミッタより後に描画されます。
設定をONにすると  グループ「描画パス」が表示されます。以下はグループ内の設定項目です。

上位階層より先に描画

継承設定「描画パス」がONの場合に表示される設定項目です。
親エミッタより先に描画するかを「 チェックボックス」ON/OFF で切り替えます。
ON にした場合、ビューアにおいて親エミッタのパーティクルが手前に描画されます。
