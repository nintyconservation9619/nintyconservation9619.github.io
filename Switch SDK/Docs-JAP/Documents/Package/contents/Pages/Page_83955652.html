波形の参照回数

波形の参照回数ウインドウでは、サウンドプロジェクト内にある、すべてのシーケンスサウンドが使用する、インストルメントに含まれる波形ファイルの参照回数を一覧表示します。

 

波形の参照回数の操作

波形の参照回数ウインドウを開く

[表示]→[波形の参照回数] または、ツールバーの「波形の参照回数」ボタンにより、波形の参照回数ウインドウを開くことができます。
これらの操作は、ウインドウの表示・非表示のトグル動作になります。

 

 

ウインドウタイトルバーのコンテキストメニューから、波形の参照回数ウインドウを [フローティング]、[ドッキング]、[非表示] に変えることができます。
ウインドウタイトルバー部分のダブルクリックでも [フローティング]、[ドッキング] の切り替えが行えます。

また、ドッキングウインドウ時は、タイトルバー右側の最小化アイコンをクリックするか、スプリッタ (境界線) をダブルクリックすることにより、タブのみの最小化表示になります。

波形の参照回数ウインドウの操作

サウンドにはループ再生するものもあるので、再生時間を指定する必要があります。

ウインドウ左上にある「計測」ボタンを押すと、「インストルメントの参照波形の計測」ダイアログが表示されるので、最大計測時間を設定して計測ボタンを押します。
計測は、各サウンドの再生開始位置から行われます。

[プレビュー]→[インストルメント波形の参照回数を計測] を実行しても同じ挙動になります。

 

計測を行うと、ウインドウ上部の「インストルメントに含まれる波形ファイルの参照回数」のリストに、ファイルパスと参照回数が表示され、そのリストを選択すると下部の「参照しているインストルメント」のリストに詳細が表示されます。

 

「参照しているインストルメント」リストの番号が付いているインデックスセルをダブルクリックすることにより、該当バンクのインストリストを開きます。
