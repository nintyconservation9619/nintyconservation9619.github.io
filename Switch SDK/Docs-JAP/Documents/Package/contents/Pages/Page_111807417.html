  • 選択ペインを自動的に表示

選択ペイン表示の「自動 / 手動」を切り替えます。

B のアイコンにて「自動 / 手動」を切り替えることもできます。


「自動」の場合、「親子階層ウインドウ」にて「親子階層」を閉じているペインをレイアウトウインドウ上でクリックした時に、親子階層が自動的に開きます。

「手動」の場合、「親子階層ウインドウ」にて「親子階層」を閉じているペインをレイアウトウインドウ上でクリックしても、親子階層は自動的に開きません。
B
選択ペイン表示の「自動 / 手動」を切り替えます。 A の項目「 選択ペインを自動的に表示」を切り替えるアイコンです。
C
選択しているレイアウトウインドウ内、全ての親子階層を展開します。
D
選択しているレイアウトウインドウ内、全ての親子階層を閉じます。
E

この階層以下を隠す / この階層以下を表示

「 アイコン」をクリック、または左側空白箇所をクリックすることで、レイアウトウインドウ上のペインの「表示 / 非表示」を切り替えます。

アイコン出現箇所をドラッグする事で、複数のペインの「表示 / 非表示」を連続で切り替えることができます。

「 Alt + アイコンクリック」で、クリックした項目以外を非表示にできます。（対応予定）


A の「この階層以下を隠す」「この階層以下を表示」と同じ機能です。
F

この階層以下をロック / この階層以下をロック解除

右側空白箇所をクリック、または「 アイコン」をクリックすることで、レイアウトウインドウ上のペインの「ロック / ロック解除」を切り替えます。

アイコン出現箇所をドラッグする事で、複数のペインの「ロック / ロック解除」を連続で切り替えることができます。


A の「この階層以下をロック」「この階層以下をロック解除」と同じ機能です。
G 親子階層表示領域

親子階層が表示されます。

ドラッグ & ドロップする事で、階層を入れ替えることができます。
アイコンを左クリックすることで、親子階層を展開します。
アイコンを左クリックすることで、親子階層を閉じます。
