放出

このタブではミッタセットに対してゲームアプリケーション側で制御可能な設定や、3Dモデルとエフェクトとの連携設定を行います。

放出設定にスケールをかける

「 放出設定にスケールをかける」は、パーティクルの放出設定に対してスケールをかける機能群です。
以下はグループ内設定項目の一覧です。

放出レート　

エミッタセット内のパーティクルすべてに対して放出レートに設定した値を乗算する機能です。
放出間隔 エミッタセット内のパーティクルすべてに対して放出間隔に設定した値を乗算する機能です。

初期設定にスケールをかける

「 初期設定にスケールをかける」は、パーティクルの初速設定に対してスケールをかける機能群です。
以下はグループ内設定項目の一覧です。

全方向初速 エミッタセット内すべてのパーティクルの「全方向初速」に対して、ここで設定した値を乗算する機能です。
指定方向初速 エミッタセット内すべてのパーティクルの「指定方向初速」に対して、ここで設定した値を乗算する機能です。
初速のランダム エミッタセット内すべてのパーティクルの「初速度のランダム」に対して、ここで設定した値を乗算する機能です。

運動設定に対する加算

「 運動設定に対する加算」は、パーティクルの速度に対して値を加算する機能です。
以下はグループ内設定項目の一覧です。

速度方向（±） エミッタセット内すべてのパーティクルの速度に対して、ここで設定した方向に値を加算する機能です。
