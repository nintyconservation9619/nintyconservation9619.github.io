はじめに

このページについて

このページでは、 NintendoSDK のビルドシステムが提供するプロパティシートについて、下記の情報を記載しています。

  • プロパティシートのインストール先ディレクトリと、各プロパティシートの機能。
  • 上記プロパティシートがビルド時に参照、提供するプロパティの一覧。

表記について

このページでは NintendoSDK のインストール先ディレクトリを $(NintendoSdkRoot) で表している箇所があります。

プロパティシート

NintendoSDK が提供するプロパティシートは次のディレクトリに存在します。

ディレクトリパス 説明
$(NintendoSdkRoot)\Build\Vc NintendoSDK 用のビルド設定を行うプロパティシートが置かれています。このディレクトリ内のファイル構成は将来の NintendoSDK のリリースに伴って変更される可能性があります。特に指示がある場合を除き、このディレクトリ内のプロパティシートをプロジェクトへ直接インポートしないでください。
$(NintendoSdkRoot)\Build\VcProjectUtility NintendoSDK のビルドシステムを使用するプロジェクトがインポートすべきプロパティシートが置かれています。開発者はこのディレクトリ内に置かれたプロパティシートを開発中のプロジェクトに自由にインポートすることができます。

VcProjectUtility ディレクトリには、次のようなプロパティシートが存在します。

ファイル名
説明
ImportNintendoSdk.props $(NintendoSdkRoot)\Build\Vc から、NintendoSDK を使用するために必要な他のプロパティシートをインポートします。後述する ビルドシステムが参照するプロパティ に挙げられたプロパティをあらかじめ定義しておく必要があります。

プロパティ一覧

ビルドシステムが参照するプロパティ

ここで挙げられるプロパティは NintendoSDK のプロパティシートをインポートする前に適切に定義されている必要があります。

ConfigurationType

ビルド対象のバイナリ種別の判定に使用します。このプロパティの値は以下のいずれかでなければなりません。

  • Application
  • StaticLibrary

ただし、NX Addon パッケージを使用しており、かつプロパティ Platform の値が NX32 もしくは NX64 の場合に限り、以下の値も使用することができます。

  • DynamicLibrary

NintendoSdkBuildType

使用する NintendoSDK ライブラリのビルドタイプの判定に使用します。詳しくは ビルドターゲットとビルドタイプ を参照してください。

このプロパティの値は以下のいずれかでなければなりません。

  • Debug
  • Develop
  • Release

NintendoSdkRoot

使用する NintendoSDK のルートパスの取得に使用します。このプロパティの値は NintendoSDK がインストールされたディレクトリのパスでなければなりません。

NintendoSdkSpec

使用する NintendoSDK の API セットの判定に使用します。共通パッケージを使用している場合、このプロパティの値は汎用 API セットを意味する Generic でなければなりません。

NX Addon パッケージを使用している場合、プロパティの値として NX API セットを意味する NX を使用することができます。NX 開発機環境、またはその Windows 互換環境向けにビルドを行う場合は NX を選択してください。

Platform

現在のビルド対象プラットフォームの判定に使用します。共通パッケージを使用している場合、このプロパティの値は以下のいずれかでなければなりません。

  • Win32
  • x64

NX Addon パッケージを使用している場合、上記に加えて下記のプロパティの値を使用することができます。これらは、それぞれ NX 開発機環境向けの 32bit 及び 64bit バイナリをビルドするためのプラットフォームです。

  • NX32
  • NX64

PlatformToolset

Visual C++ を使用したビルド時のみ、使用する Visual C++ ツールチェインのバージョンの判定に使用します。このプロパティの値は以下のいずれかでなければなりません。

  • v140
  • v141

ビルドシステムが提供するプロパティ

ここで挙げられるプロパティは開発者の Visual Studio プロジェクト内で自由に参照することができます。

NintendoSdkBuildTarget

ビルドするライブラリやアプリケーションのバイナリの種類を表す識別子です。詳しくは ビルドターゲットとビルドタイプ を参照してください。

NintendoSdkBuildTarget プロパティの値は、NintendoSDK のインクルードディレクトリパスやライブラリディレクトリパスの一部に使用されます。

NintendoSdkDllSuffix

動的ライブラリの検索時、ライブラリ名に付加すべき接尾辞です。

通常このプロパティは、ピリオドを含めた動的ライブラリファイルの拡張子となります。例えば Win32 及び x64 プラットフォームのビルド時は .dll となります。

NintendoSdkIncludePath

NintendoSDK のインクルードディレクトリのリストです。

NintendoSdkLibraryPath

NintendoSDK のライブラリディレクトリのリストです。

NintendoSdkLibSuffix

静的ライブラリの検索時、ライブラリ名に付加すべき接尾辞です。

通常このプロパティは、ピリオドを含めた静的ライブラリファイルの拡張子となります。例えば Win32 及び x64 プラットフォームのビルド時は .lib となります。

NintendoSdkNXAddonVersionMajor

NintendoSDK NX Addon のメジャーバージョン番号です。

NintendoSdkNXAddonVersionMinor

NintendoSDK NX Addon のマイナーバージョン番号です。

NintendoSdkNXAddonVersionMicro

NintendoSDK NX Addon のマイクロバージョン番号です。

NintendoSdkVcRuntimeLibrary

NintendoSDK が使用する Visual C++ ランタイムの種類を表します。 Visual C++ を使用したコンパイルが行われる場合のみ定義されます。

値は、ClCompile アイテムの RuntimeLibrary メタデータと同等です。以下のいずれかを取ります。

  • MultiThreadedDLL
  • MultiThreadedDebugDLL

NintendoSdkVersionMajor

NintendoSDK のメジャーバージョン番号です。

NintendoSdkVersionMinor

NintendoSDK のマイナーバージョン番号です。

NintendoSdkVersionMicro

NintendoSDK のマイクロバージョン番号です。

その他のプロパティについて

NintendoSDK のプロパティシートでは、このドキュメントに記載されていないプロパティを定義することがあります。これらはビルドシステムが内部で使用するプロパティですので、参照することは避けてください。通常これらのプロパティ名には接頭辞 NintendoSdk または _NintendoSdk が付加されています。
