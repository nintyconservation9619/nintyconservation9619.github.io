3DEditor

マテリアル参照機能の仕様変更

これまであったマテリアルプロパティウィンドウの「子マテリアルでの表示」、「子マテリアルでの編集」という２つのコンテキストメニューを「子マテリアルの制限」という１つのメニューに統合しました。

これまで設定した中間ファイルは 3DEditor や 3D コマンドラインツールに読み込むときに自動で新しい形式に変換されます。各メニューの詳細についてはマテリアルプロパティウィンドウのドキュメントをご覧ください。

Maya プラグイン

頂点カラーの出力を抑制する機能を追加

Maya の頂点カラーのカラーセットをカレントに設定していても、中間ファイルに出力しないよう設定できるようになりました。
カレントの頂点カラーのカラーセットが「nw_ignore」で始まる名前の場合は、頂点カラーセットが設定されていても出力しないようになりました。

例として、編集用データ（別のDCCツールで使う為の頂点情報を頂点カラーに設定している、など）を中間ファイルには出したくない場合などに使用します。
