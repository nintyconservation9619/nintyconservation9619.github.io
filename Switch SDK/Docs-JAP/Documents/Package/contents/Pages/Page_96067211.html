プリミティブを利用する

プリミティブを利用することで不要な塗り面積を削減できます。
下図はプリセットプリミティブの「EfPrePrim_RadialAlpha.fmdb」を利用して描画した例で、
図で示すように標準の矩形を利用して描画するよりも、四隅の分だけ塗面積を節約できます。


画面上で非常に小さく描画されるようなシチュエーションでは逆効果になる場合もあります。

プリセットプリミティブの「EfPrePrim_RadialAlpha.fmdb」をパーティクルとして利用したエフェクトです。
「EfPrePrim_RadialAlpha.fmdb」には頂点アルファが使用されており、テクスチャマスクの代替としても利用できます。


見た目的には遜色なく同等の表現が可能です。
