使用方法

画像結合プラグインは、指定したフォルダーに存在する画像ファイルをすべて結合した画像を Photoshop 上に作成します。
パーティクルなど多数のポリゴンを描画する際に、ポリゴンごとにテクスチャーをロードするのではなく、大きなテクスチャーを 1 枚ロードして、その一部が表示されるようにテクスチャー座標を調整して描画する場合に便利です。

[ファイル] → [自動処理] → [Nintendo Unite Images...] を実行します。

  • Input Folder 欄
    結合する画像ファイルが存在するフォルダーを指定します。
    Photoshop が対応しているすべてのファイル形式の画像を使用できます（プラグインで対応しているファイル形式の画像も使用できます）。
  • Number of Pixels in an Image 欄
    1 つの画像の幅（Width）と高さ（Height）をピクセル数で指定します。
    ドロップダウンリストで custom を選択すると、ピクセル数を直接入力できます。
    Same Value を ON にすると、高さが自動的に幅と同じ値になります。
    画像の幅または高さが、ここで指定した値と異なる場合、サイズを変更してから結合します。画像の再サンプルには「バイリニア法」を使用します。
  • Number of Images in a Row (in a Column) 欄
    横方向（Horizontal）と縦方向（Vertical）に配置する画像の個数を指定します。
    ドロップダウンリストで custom を選択すると、個数を直接入力できます。
    Same Value を ON にすると、縦方向の個数が自動的に横方向の個数と同じ値になります。
    各画像ファイルはファイル名のアルファベット順に左上から配置されます。
  • Unite Alpha
    アルファチャンネルを結合するなら ON にします。
    どの画像にもアルファチャンネルがないと判明している場合は OFF にすると処理時間を短縮できます。
  • Fill Color Edge with Black
    各画像のカラーチャンネルの端の 1 ピクセルを黒で塗りつぶすなら ON にします。
  • Fill Alpha Edge with Black
    各画像のアルファチャンネルの端の 1 ピクセルを黒で塗りつぶすなら ON にします。

OK ボタンをクリックすると結合した画像が作成されます。

注意事項

画像統合プラグインを使用する際の注意事項を記載します。

アルファチャンネル

画像ファイルにアルファチャンネルが 2 つ以上存在する場合、チャンネルウィンドウで一番上にあるアルファチャンネルだけが結合されます。

EXR ファイルを結合する場合、Photoshop CC 2015 以降では Unite Alpha が ON なら A 成分をアルファチャンネルとして読み取ります。
Unite Alpha が OFF または CC 2014 以前なら A 成分を透明度として読み取ります。

レイヤー

画像ファイルにレイヤーが複数存在する場合、表示レイヤーを結合した状態の画像を作成します。
