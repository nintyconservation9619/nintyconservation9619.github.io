ファイルリストにはエミッタ「ShotFire」を持つ花火のエフェクトが表示されます。

プリセットをプロジェクト設定に追加

プリセット機能がゲームプロジェクト単位で利用できるようになりました！

よく使用するエフェクトデータは所定のフォルダに登録しておくことで、エフェクトデータ作成時にひな型として呼び出し、利用できます。
以下はプリセット機能をプロジェクトで利用するメリットです。

  • プロジェクト内でエフェクトの表現が統一される。
  • 似た様なデータが複数存在することがなくなる。
  • プリセット化できる部分はその都度最初から作る必要がなくなり、効率が良い。
