一部のバイナリイメージに関しては、エフェクトバイナリコンバート時に　別ファイルとして出力することが可能です。

バイナリコマンドラインオプションについて

エフェクトバイナリに含まれる　テクスチャ、3Dバイナリ、シェーダ をバイナリ化する際のコマンドラインオプションについて説明します。

  利用しているツール コマンドラインオプション
テクスチャ 3dTextureConverter.exe 

-o EffectMaker\Converter\texture_intermediate\output.bntx

--tile-mode=Linear "input01.ftxb" "input02.ftxb" ( *2 )
3Dバイナリ 3dBinaryConverter.dll

--ignore-assign --silent --disable-attribute-alignment-sort

頂点・ピクセルシェーダ ShaderConverter.exe

--glsl-version=430 -sGlsl -aGl --code-type=Source

-o EffectMaker\Converter\shader_intermediate\output.grsf

--vertex-shader=EffectMaker\Converter\shader_intermediate\shader.vsh

--pixel-shader=EffectMaker\Converter\shader_intermediate\shader.psh

--include-directory=EffectMaker\Converter\shader

--variation=EffectMaker\Converter\shader_intermediate\shaderVariation.xml

--print-progress --code-page 65001 ( *3)
コンピュートシェーダ ShaderConverter.exe 

--glsl-version=430 -sGlsl -aGl --code-type=Source

-o EffectMaker\Converter\shader_intermediate\output.grsf

--compute-shader=EffectMaker\Converter\shader_intermediate\computeShader.vsh

--include-directory=EffectMaker\Converter\shader

--variation=EffectMaker\Converter\shader_intermediate\computeShaderVariation.xml

--print-progress --code-page 65001 ( *4 )
参考：

( *2 ) 一度にコンバートするテクスチャftxbファイルパスが列挙されます。

( *3 ) コマンドラインコンバート時に--output-shader-binaryオプションが指定された場合、ShaderConverter.exeの出力パスオプションである-oの内容を書き換えます。

( *4 ) コマンドラインコンバート時に--output-compute-shader-binaryオプションが指定された場合、ShaderConverter.exeの出力パスオプションである-oの内容を書き換えます。
