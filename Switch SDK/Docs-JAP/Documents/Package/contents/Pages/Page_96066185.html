パーティクル時間のカーブエディタ

「パーティクル時間のカーブエディタ」では横軸を時間、縦軸を値としたグラフでパーティクル時間のアニメーションを設定できます。
ここでは「スケールアニメーション」の UI を例に、パーティクル時間のカーブエディタについて説明します。

A プリセット エフェクトでよく使用するアニメーションのカーブ形状をボタン一発で生成します。
B ツールバー カーブエディタ操作する為のツール群です。
C カーブビュー

エフェクトを構成する各ノードをツリー状に表示します。

マウスの右クリックによりワークスペースでの編集メニューが開きます。
D ヘルプ アイコン上にマウスオーバーすることで、アニメーション編集で便利なショートカットの一覧が表示されます。

プリセット

「プリセット」はエフェクトでよく使用するアニメーションのカーブをボタン一発で生成できるボタンです。
以下にボタンの種類ごとのアニメーション挙動について説明します。

2キー パーティクル寿命の範囲でスケール 0  から スケール基準値のサイズに拡大します。
2キー パーティクル寿命の範囲でスケール基準値のサイズ  から スケール 0 に縮小します。
3キー

パーティクル寿命の 50% までスケール 0  から スケール基準値のサイズに拡大します。

50%～100% までスケール基準値のサイズ  から スケール 0 に縮小します。
4キー

パーティクル寿命の 25% までスケール 0  から スケール基準値のサイズに拡大します。

75%～100% までスケール基準値のサイズ  から スケール 0 に縮小します。
3キー

スケール基準値のサイズ  からアニメーションを開始します。

75%～100% までスケール基準値のサイズ  から スケール 0 に縮小します。
4キー

パーティクル寿命の 25% までスケール基準値のサイズ  からスケール基準値の半分のサイズに縮小します。

75%～100% までスケール基準値の半分のサイズから スケール 0 に縮小します。

ツールバー

「ツールバー」はカーブエディタ操作する為のツール群です。

 

カーブビュー

「カーブビュー」はエフェクトを構成する各ノードをツリー状に表示します。

 

ヘルプ

「 ヘルプ」はアイコン上にマウスオーバーすることで、アニメーション編集で便利なショートカットの一覧が表示されます。
