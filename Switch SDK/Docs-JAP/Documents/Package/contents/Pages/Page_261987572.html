KPR ライブラリ

はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている KPR  (Keycode PRocessing) ライブラリの機能について説明します。

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象とします。

関連資料

KPR ライブラリの関数リファレンスと合わせてご利用ください。

KPR ライブラリの機能

KPR ライブラリは、キーボードのキーコードを処理するための API 群です。以下の機能を提供します。

  • 文字コードへの変換
  • 文字コードの組み立て（デッドキー処理、ローマ字入力など）
