カラー

  このタブではエフェクトのカラーやアルファに関する設定を行います。

「 パーティクルタブ」には以下の「   サブタブ」があります。

パーティクル パーティクル個々のカラーやアルファに関する設定を行います。
グローバル エミッタ全体に影響を与えるカラーやアルファ値を設定します。

以下は「 パーティクル」の設定項目です。

カラー計算

「 カラー計算」は、エミッタで利用するパーティクル全体のカラーに乗算する値を設定する機能です。
以下はグループ内設定項目の一覧です。

カラースケール

カラーの設定値に対してスケールをかける機能です。

カラー・アルファ設定

「 カラー・アルファ設定」は、「カラーコンバイナ」計算式で使用するカラーを設定する機能群です。
以下はグループ内設定項目の一覧です。

カラーの設定 「カラーコンバイナ」計算式の Color0 や Color1 として使用する挙動タイプを設定する機能です。
アルファの設定 「カラーコンバイナ」計算式の Alpha0 や Alpha1 として使用する挙動タイプを設定する機能です。

アルファ処理

「 アルファ処理」は、パーティクルのアルファ値に効果を与える機能群です。
以下はグループ内設定項目の一覧です。

ソフトパーティクル モデルと重なったパーティクルに対して半透明処理を施す機能です。
近距離アルファ カメラから近い距離にあるパーティクルに対して半透明処理を施す機能です。
遠距離アルファ カメラから遠い距離にあるパーティクルに対して半透明処理を施す機能です。
フレネルアルファ 視線方向に対してプリミティブ法線方向のアルファ値を変化させる機能です。
デカール モデルとパーティクルが重なった部分以外のアルファ値を変化させる機能です。
モデルにパーティクルを貼り付けた様な効果を与える機能です。

以下は「 グローバル」の設定項目です。

グローバル乗算カラー・アルファ

「 グローバル乗算カラー・アルファ」は、エミッタから放出されるパーティクル全体に影響を与えるカラーやアルファ値を設定します。
以下はグループ内設定項目の一覧です。

グローバル乗算カラー・アルファ
エミッタから放出される全パーティクルのカラーに乗算する機能です。
