はじめに

ここでは、NintendoSDK を利用したライブラリやアプリケーション開発するのに必要な環境の構築方法について案内します。

動作環境の構築

「動作環境」に書かれている動作環境を準備してください。

NintendoSDK パッケージのインストール

Nintendo Dev Interface を利用して、各種 NintendoSDK パッケージをインストールしてください。

各種パッケージの種類については、「パッケージ構成」にリストが掲載されていますので、必要なパッケージを適宜インストールしてください。

開発機のセットアップ・ファームウェアの更新

「NintendoSDK SystemUpdater for NX パッケージ」をインストールすると、開発機 (SDEV・EDEV) のセットアップをすることが可能です。

SDEV 向けの NX アプリケーションを実行、あるいはビルドしたい場合は、まず SDEV の初期化 でその手順をご確認ください。
EDEV で同様のことを行うためには EDEV の初期化 でその手順をご確認ください。

すでに古いバージョンのファームウェアが書き込まれている場合は、 SDEV のアップデート でファームウェアのアップデート手順をご確認ください。

コントローラーのセットアップに関しては、コントローラーの準備 をご確認ください。

次のステップ

NintendoSDK が提供するサンプルのビルド・実行方法や、NX アプリケーションの開発手順を学ぶには、 チュートリアル をご確認ください。 

開発機の初期化や設定など、開発機の制御について学ぶには、 開発機の制御 をご確認ください。
