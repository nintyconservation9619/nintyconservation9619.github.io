指定ファイルを使用するエミッタセット

指定した参照ファイルを使用するエミッタセットを検索する「指定ファイルを使用するエミッタセット」について説明します。
以下は「未使用ファイル」専用の設定項目です。

指定ファイルを使用するエミッタセット ファイル選択ダイアログ
A 「 ダイアログを表示」

マウス左ボタンクリックでファイル選択ダイアログが開きます。
B ファイル選択ダイアログ

ダイアログは「検索基点パス」以下のファイルをリストアップします。
C テキストボックス ダイアログで選択したファイルが登録されるとここにそのファイル名が表示されます。
D 「 クリア」 マウス左ボタンクリックで登録したファイルを解除します。

概要

以下はこのプリセットの図説です。

A 検索基点パス アドレスバーで指定したパスです。検索の基点になります。
B 指定ファイル

ダイアログで選択したファイルです。

この範囲にあるファイルを参照するエミッタセットを検索します。
C  エミッタセット

本機能の検索対象です。
