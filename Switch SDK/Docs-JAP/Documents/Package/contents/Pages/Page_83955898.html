マルチエクスポートプラグイン

ここでは、マルチエクスポートプラグインの機能と使用方法を説明します。

機能

マルチエクスポートプラグインを使用すると、シーン内に複数の出力設定を保存して、複数モデルおよび複数アニメーションをまとめて出力できます。

マルチエクスポートプラグインは、次の設定の組み合わせに従って出力します。この設定の組み合わせを「出力アイテム」と呼びます。

  • 出力する中間ファイル名
  • 中間ファイルを出力するフォルダのパス
  • マージする fmd ファイルのパス
  • 使用する fdes ファイルのパス
  • 出力するノード名
  • 出力するシーンレイヤノード名
  • 有効にするアニメーションレイヤ名
  • 中間ファイルに対する編集用コメント文
  • アニメーションを出力する範囲
  • ループ再生フラグ
  • 各中間ファイルの出力フラグ
  • 追加オプション
参考：

出力アイテムは、シーンのルートノードのカスタムアトリビュートに保存されます。

使用方法

「Nintendo > Multi Export...」を選択するとウィンドウが表示されます。

ウィンドウには現在のシーンに設定されている出力アイテムのリストが表示されます。

  • Output File Name
    出力する中間ファイル名です。
  • fdes File
    使用する fdes ファイル名です。
  • Node
    出力するノード名です。
  • Layer
    出力する表示レイヤ名です。
  • Comment
    中間ファイルに対する編集用コメント文です。
  • Start End
    アニメーションを出力する範囲です。
  • Loop
    ループ再生フラグです。
  • Selection
    各中間ファイルの出力フラグです。ON なら次の文字が表示されます。
    M : fmdb ファイル
    T : ftxb ファイル
    S : fskb ファイル
    V : fvbb ファイル
    C : fclb ファイル
    R : ftsb ファイル
    P : ftpb ファイル
    H : fshb ファイル
    N : fsnb ファイル
  • Options
    追加オプションです。
    出力アイテムに出力フォルダのパス、マージする fmd ファイルのパス、有効にするアニメーションレイヤ名が設定されていれば、リストの項目を選択した際にリストの下に表示されます。
    Add ボタンをクリックすると、新しい出力アイテムを追加できます。リストの項目が選択されている場合は選択した場所に挿入、リストの項目が選択されていない場合はリストの最後に追加となります（リストの何もない場所を クリックすると選択が解除されます）。
    リストの項目を選択して Edit ボタンをクリックするか項目をダブルクリックすると、既存の出力アイテムの内容を変更できます。
    リストの項目を選択（複数選択可）して Duplicate ボタンをクリックすると、選択した出力アイテムを複製できます。
    リストの項目を選択（複数選択可）して Delete ボタンをクリックすると、選択した出力アイテムを削除できます。
    リストの項目を選択（複数選択可）して Move Up ボタンまたは Move Down ボタンをクリックすると、リスト中での出力アイテムの順番を入れ換えることができます（順番は出力される中間ファイルには影響しません）。
    リストの項目を選択して Load fdes to Scene ボタンをクリックすると、出力アイテムの fdes ファイルの設定をシーンにロードして、エクスポートプラグインのオプションウィンドウを表示します（fdes ファイルのパスが空欄の場合は現在のシーンのオプション設定を表示します）。
    リストの項目を選択して Get Playback Range ボタンをクリックすると、3ds Max 上のアニメーションの再生範囲を取得して、出力アイテムの Start / End Frame に設定します。
    リストの項目を選択して Set Playback Range ボタンをクリックすると、出力アイテムの Start / End Frame の範囲が再生されるように 3ds Max 上のアニメーションの再生範囲を変更します。
    リストの項目を選択（複数選択可）して、Export Selected Items ボタンをクリックすると、選択した出力アイテムを出力します。
    Export All Items ボタンをクリックすると、すべての出力アイテムを出力します。
    Add ボタンまたは Edit ボタンをクリックすると、出力アイテム入力ウィンドウが表示されます。
  • Output File Name
    出力する中間ファイル名（拡張子を除く）を入力します。
    右にある Scene ボタンをクリックすると現在のシーン名が入力されます。
    Node ボタンをクリックすると 3ds Max 上で選択中のノード（複数選択されている場合は一番最初に選択されたもの）の名前が入力されます。
  • Output Folder
    中間ファイルを出力するフォルダを入力します。
    空欄の場合、fdes ファイルの設定（fdes File も空欄の場合は現在のシーンの設定）を使用します。
    フルパスでない場合、現在のシーンファイルのフォルダからの相対パスとみなします。
  • Merge fmd File
    ON にして fmd ファイルのパスを指定すると、指定した fmd ファイルをマージします。
    パスが空欄の場合、上書きする出力先ファイルをマージします。
    パスがフルパスでない場合、現在のシーンファイルのフォルダからの相対パスとみなします。
  • fdes File
    使用する fdes ファイルのパスを入力します。
    空欄の場合、現在のシーンのオプション設定で出力します。
    フルパスでない場合、現在のシーンファイルのフォルダからの相対パスとみなします。
    下にある Recent fdes ボタンをクリックすると、最近入力した fdes ファイルの履歴から選んで入力できます。
    Load fdes to Scene ボタンをクリックすると、入力されている fdes ファイルの設定をシーンにロードして、エクスポートプラグインのオプションウィンドウを表示します（パスが空欄の場合は現在のシーンのオプション設定を表示します）。
  • Node
    特定のノードだけ出力する場合に、出力するノードのノード名を入力します（入力したノードの下の階層のノードも出力されます）。
    複数のノード名をスペースで区切って記述できます。
    fdes ファイルがシーン全体を出力する設定であれば何も入力する必要はありません。
    また、fdes ファイルがシーン全体を出力する設定であっても、ノード名を入力すると、入力したノードだけが出力されます。
    右にある Get ボタンをクリックすると、3ds Max 上で選択中のノードの名前が入力されます。
  • Select ボタンをクリックすると、入力したノードを 3ds Max 上で選択します。
  • Layer
    特定のレイヤだけ出力する場合に、出力するレイヤのレイヤ名を入力します。
    複数のレイヤ名をスペースで区切って記述できます。
    fdes ファイルがシーン全体を出力する設定であれば何も入力する必要はありません。
    また、fdes ファイルがシーン全体を出力する設定であっても、レイヤ名を入力すると、入力したレイヤだけが出力されます。
    下にある Get ボタンをクリックすると、3ds Max 上で可視のレイヤの名前が入力されます。
    Set ボタンをクリックすると、入力したレイヤのみ 3ds Max 上で可視にします。
  • Anim Layer
    特定のアニメーションレイヤのみ有効にして出力する場合に、有効にするアニメーションレイヤ名を入力します。
    複数のアニメーションレイヤ名をスペースで区切って記述できます。
    ベース以外のアニメーションレイヤを無効にするには "Base Layer" を入力します。
    空欄の場合、現在有効なアニメーションレイヤを出力します。
    右にある Get ボタンをクリックすると、3ds Max 上で有効なアニメーションレイヤ名が入力されます。
    Set ボタンをクリックすると、入力したアニメーションレイヤのみ 3ds Max 上で有効にします。
  • Comment
    中間ファイルに対する編集用コメント文を指定します。
    「;（セミコロン）」、「"（ダブルクォート）」は使用できません。
  • Start / End Frame
    アニメーションを出力する範囲（開始フレームと終了フレーム）を指定します。
    指定方法はエクスポートプラグインの Start / End Frame オプションと同じです。
  • Loop
    ループ再生フラグを指定します。
    ON にすると、ループ再生するデータとして出力します。
    OFF にすると、一度だけ再生するデータとして出力します。
  • Output File Selection
    各中間ファイルの出力フラグを指定します。
    Get from fdes ボタンをクリックすると、fdes File で指定した fdes ファイルで出力する設定になっている中間ファイルのみ ON にします（パスが空欄の場合は現在のシーンのオプション設定で出力される中間ファイルのみ ON にします）。

  • Additional Options
    追加オプションを オプション文字列の使用方法 と同じ書式で、「"（ダブルクォート）」なしのスペース区切りで指定します。
    1 行に複数のオプションを記述することも、改行して複数行に記述することも可能です。
    追加オプションは fdes ファイルのオプション設定より優先されます。
    たとえば、fdes ファイルの Frame Precision の設定が 1.0 になっている場合に、0.5 で出力するなら、次のように記述します。

    -fp 0.5
    		 
          OK ボタンをクリックすると、出力アイテムが追加または変更されます。
参考：

出力アイテムの fdes ファイルがプリセットを使用する場合、出力アイテムの設定よりプリセットファイルでロックされている設定が常に優先されます。
たとえば、プリセットで Output Folder が「D:/data/」でロックされていれば、出力アイテムの Output Folder が「D:/tmp/」であっても、「D:/data/」に出力されるので注意してください。
Merge fmd File、Start / End Frame、Loop、Output File Selection などについても同様です。
