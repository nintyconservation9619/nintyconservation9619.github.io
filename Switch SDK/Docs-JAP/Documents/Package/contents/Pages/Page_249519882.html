アニメーションの分割モードを利用する

アニメーションの分割モードの利用方法について説明します。

「結合モード」と「分割モード」

「結合モード」とは、ひとつのアニメーションを区間タグで区切って利用する、従来の動作のことを指します。
「分割モード」とは、アニメーションと区間タグが 1:1 に紐付けされるようになる、レイアウトエディタの新機能です。
「分割モード」に切り替えることで、区間タグ毎に別々のアニメーションを作成することが出来るようになります。
機能の詳細は下記リンクからご確認ください。

アニメーション区間タグ

「分割モード」を利用する

アニメーション区間タグウインドウの「分割モードボタン」を押下することで「分割モード」に切り替わります。
