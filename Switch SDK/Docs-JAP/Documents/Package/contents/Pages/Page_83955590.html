放出間隔タイプ「時間」

ここではパーティクルが放出される間隔を「時間」で設定する放出間隔タイプ「時間」について説明します。

概要

放出間隔タイプ「時間」は、パーティクルの放出間隔を時間で設定できる機能です。
放出間隔を「時間」で設定することによって、一度に放出するパーティクルの数（放出レート）を設定できます。
また、放出レートや放出間隔をランダムにできます。

以下は「 設定項目」の概要です。

「放出間隔タイプ」の設定を「時間」にすると「 放出間隔タイプ「時間」」が表示されます。以下はグループ内の設定項目です。

放出レート エミッタが一度に放出するパーティクルの量を設定します。
放出間隔 パーティクルが放出される間隔を時間（フレーム）で設定します。
