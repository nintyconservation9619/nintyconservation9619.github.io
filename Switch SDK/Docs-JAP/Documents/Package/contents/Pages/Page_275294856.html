概要

SystemUpdateEdev は開発機 EDEV のファームウェアを本体更新機能と同等の手順で更新するためのコマンドラインツールです。

InitializeEdev による初期化と比べて設定等が保持されたままファームウェアがバージョンアップできるというメリットがあります。

このツールは NX Addon 4.4.1 以降で追加されました。

パッケージ

DevKitVersionUpdater パッケージに含まれます。

配置

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/SystemUpdateEdev.exe

なお動作のためにはこの exe ファイル以外にも依存するファイルがいくつかあります。

コマンド形式

オプション形式

SystemUpdateEdev [target options] [other options]

target options:
    --target <serial number | EDEV name>               本体更新するターゲットのシリアル番号もしくは名前を指定します

other options:
    --target-version                                   本体更新で用いるファームウェアの種類を指定します
    --display-available-version                        利用できるファームウェアの種類をすべて出力します
    --verbose                                          ログを詳細に出力します
    --version                                          ツール自身のバージョンを表示します

終了コード

ツールの結果成否は終了コードで確認することができます。

0   - 正常終了
非0 - エラー終了

使い方

以下に利用ケースに応じた SystemUpdateEdev の実行例を示していきます。

開発機のIPアドレスを指定して更新する

SystemUpdateSdev --target 192.168.10.11

開発機のシリアル番号を指定して更新する

SystemUpdateEdev --target XAL03100000000

開発機の名前(name)を指定して更新する

SystemUpdateSdev --target (target name)

注意事項

  • バージョンダウンはできません。
  • NX Addon 4.0.0 以降のファームが書き込まれた EDEV に対して実行できます。それより前のファームが書かれた EDEV からは実行できませんのでご注意下さい。
  • 開発機のシリアル番号指定時はチェックディジットはあってもなくても指定できます。
