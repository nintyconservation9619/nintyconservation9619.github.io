概要

InitializeEdevWin は開発機 EDEV のファームウェアを初期化するためのGUIツールです。

このツールのコマンドライン版が InitializeEdev です。自動テストで用いる場合はそちらをご利用ください。

パッケージ

InitializeEdevWin は NintendoSDK NX Addon パッケージに含まれています。

配置

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/InitializeEdevWin.exe

なお動作のためにはこの exe ファイル以外にも依存するファイルがいくつかあります。

注意点

本ツールで EDEV を初期化すると、インストールしたアプリケーションや、セーブデータがすべて削除されます。

オプション形式

この実行ファイルに指定する必要のあるコマンドラインオプションはありません。

exe ファイルを引数なしで実行するとウィンドウが表示されます。

表示画面

メインウィンドウ

InitializeEdevWin を起動すると表示される画面です。

使い方は 1 で初期化する開発機を指定し、2 でファームウェアバージョンを指定し、3の初期化開始ボタンを押すというものです。
初期化開始ボタンを押すとログウィンドウが表示され初期化が開始されます。

なお1の画面に初期化したい開発機が表示されない場合は“Re-Scan”ボタンを押して再検索してください。
またメニューの"Tools" に登録されている項目はありませんのでクリックしてもなにも表示されません。

ログウィンドウ

初期化を実行すると表示される画面です。

プログレスバーで経過が表示され、ログウィンドウにログが表示されます。
初期化が完了すると画面左下の結果表示欄に結果が表示され、画面右下の終了ボタンが有効になります。

 

その他能の説明

ファームウェア種類選択コンボボックスから初期化したいファームウェアを選ぶことで以下に説明する機能が使えます。

製品機に近いファームによる初期化

ファームウェア種類選択コンボボックスで "+ Prod" を含む項目を選択すると製品機に近い状態になるファームを使って初期化します。

また "+ ProdWithLog" を含む項目を選択すると microSD カードからのログ出力機能付きの製品機に近いファームで初期化します。

警告：

この「製品機に近いファーム」で初期化した本体を再度初期化する場合はセーフモードに手動操作で入れないと再初期化できません。

注意：

「製品機に近いファームによる初期化」機能は EDEV MP のみで利用できます。それ以外の EDEV 本体で実行するとエラーになりますのでご注意ください。

 

注意事項

バージョンダウン初期化に関して

現バージョンの InitializeEdevWin で初期化した EDEV を NX Addon 0.10 以前の InitializeEdevWin (および InitializeEdev) を用いてバージョンダウン初期化することはできません。

セーフモードに入った EDEV の初期化に関して

EDEV 本体がセーフモードに入った状態で InitializeEdev を起動した場合はメイン画面に Name 欄が (Safe Mode) と記載された行が表示されます。

 

 

この行を選択後 "Start initialization" をクリックすると EDEV の初期化が行われます。
