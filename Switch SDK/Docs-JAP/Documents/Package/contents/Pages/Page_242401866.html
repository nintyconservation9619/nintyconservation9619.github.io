オーディオニュース

NintendoSDK 4.0.0 における変更点を紹介します。

SoundMaker

「参照ファイルを開く」で使われるアプリケーションをすべてのタイプで指定できるようになりました

SoundMaker で MIDI ファイルと波形ファイルを「参照ファイルを開く」で開くのに使うアプリケーションを指定できるようになりました。

  • テキストシーケンスファイルのみ以前より指定ができました。

エクスプローラー上では波形ファイルをプレビューし SoundMaker から開くときは編集する、といった使い分けが可能になります。

参考：

この変更に伴い、テキストシーケンスファイルの外部エディタの設定はオプションダイアログの [全般] タブから [外部エディタ] タブに移動しています。

詳細は オプション設定 を参照してください。

Opus ストリームサウンドのデコード時にハードウェアアクセラレータの利用を選択できるようになりました

SoundMaker 上で、ハードウェアアクセラレータを利用した Opus デコードの設定を行う機能を追加しました。

[プロジェクト] -> [プロジェクトの設定] -> [コンバート] の項にある、 [ターゲットでの Opus のデコードにはハードウェアを使用するように指定する] をチェックすることで、
全ての Opus ストリームサウンドに対してハードウェアアクセラレータを利用する設定が適用されます。

詳細は 圧縮オーディオ を参照してください。

コマンドラインツール

SoundProjectUtility コマンドラインツールを追加しました

Tools/Audio/SoundMaker/SoundProjectUtility.exe コマンドラインツールを追加しました。

サウンドプロジェクトを対象にした雑多な機能を提供します。今リリースでは以下の機能が利用できます。

  • サウンドプロジェクトが参照しているファイルの一覧を表示します。
  • （評価中）サウンドプロジェクトを SoundMaker が対応する最新バージョンに更新します。

詳細は SoundProjectUtility を参照してください。

SoundProjectWaveExporter にシーケンスサウンドをトラック毎に分割して波形化する機能を追加しました

Tools/Audio/SoundMaker/SoundProjectWaveExporter.exe に、シーケンスサウンドをトラック毎に分割して波形化する機能を追加しました。

  • --split-sequence-sound-tracks オプション。

過去にシーケンスサウンドとして作成された資産を、DAW による編集がしやすいように波形化する用途を想定しています。

詳細は SoundProjectWaveExporter を参照してください。
