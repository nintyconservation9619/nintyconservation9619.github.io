部品レイアウトの作成

ここではメニュー画面のボタン部品として利用できる以下のようなセレクトボタンの作成手順について説明します。

レイアウト作成の準備

あらかじめ LayoutEditor を起動した状態から、レイアウトデータ作成にあたっての準備を行います。

関連リンク LayoutEditor の起動方法については「作業の流れ」を参照してください。

ウインドウを開く

以下のウインドウが開いた状態から作業を開始します。え

  • テクスチャ
  • 親子階層
  • プロパティウインドウ
  • 部品パレット
  • 部品コントロール設定

関連リンク 各ウインドウの UI については「ウインドウ」を参照してください。

保存先フォルダの作成

「プロジェクト設定ファイル」を作成することで、レイアウトデータの保存先を以下の手順で指定します。

1

部品パレットウインドウで「 フォルダを開く」ボタンを押し、「フォルダーの参照」ダイアログを開きます。
2 任意のディレクトリを「部品ルートフォルダ」として選択します。
3 「新しいフォルダーの作成」ボタンを押し、フォルダ名に「Menu」を指定してフォルダを作成します。
4 「OK」ボタンを押して「フォルダーの参照」ダイアログを閉じます。

作成したフォルダが部品ルートとして設定され、フォルダ内にプロジェクト設定ファイルが作成されます。
以降、チュートリアルで作成するレイアウトデータはこのフォルダに保存します。

関連リンク プロジェクト設定ファイルについてはこちらを参照してください。

新規レイアウトの作成

メインメニュー[ ファイル ] → [ 新規作成 ] から新規レイアウトを作成します。

関連リンク 新規レイアウトの作成手順については「レイアウトの作成」を参照してください。

テクスチャの読み込み

使用する以下のテクスチャを以下の場所から「テクスチャ」ウインドウに読み込みます。
Samples/Sources/Resources/Ui2d/Common/Textures/

「テクスチャ」ウインドウを「詳細表示」に切り替え、テクスチャのフォーマットを以下の様に設定します。

サムネイル

 ファイル名 フォーマット 説明

Band.tga A8 :（非圧縮_A）

テクスチャをアルファとして圧縮せずに使用するフォーマットです。

セレクトボタンの枠として利用します。

Select.tga BC4A :（A） テクスチャをアルファとして圧縮して使用するフォーマットです。
セレクトボタンのカーソルとして利用します。

関連リンク テクスチャの登録方法についてはこちらを参照してください。

 ペイン基本

「親子階層」ウインドウで作成された「 ピクチャペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

 ペイン名 Band_Fade このペインはボタン選択時のカーソルになります。

カラー補間設定

「親子階層」ウインドウで作成された「 ピクチャペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

 ペイン名 Select このペインはボタンを押したときの効果になります。

「テキストボックスペイン」の作成

メインメニュー[ ペイン ] → [ ペイン作成 ] → [ 「テキストボックスペイン」 ] を選択します。
開いた「テキストボックス作成」ダイアログで以下のように設定します。

フォントファイルで指定 Samples/Sources/Resources/Ui2d/Common/Fonts/sample.bffnt
文字列 Tile Name

「親子階層」ウインドウで作成された「 テキストボックスペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

 ペイン名 Text このペインはボタン上に配置する文字列になります。

「境界ペイン」の作成

判定領域の指定に「 境界ペイン」を作成します。

メインメニュー[ ペイン ] → [ ペイン作成 ] → [ 境界ペイン ] を選択します。
「親子階層」ウインドウに作成された「 境界ペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本


 ペイン名 Bounding このペインはマウスカーソルをボタン上にのせるときの判定領域です。
サイズ X,Y = 400,50 ボタンよりやや小さめに設定します。
部品倍率の影響 自分を含む部品外枠との距離を維持 ボタンのサイズ変更に連動させます。

関連リンク 「境界ペイン」の概要についてはこちらを参照してください。

レイアウトの部品化

「部品・コントロール設定ウインドウ」でこのレイアウトを「部品レイアウト」に設定します。
「部品・コントロール設定ウインドウ」の設定項目が無効の状態であることを確認します。
「 参照ボタン」を押し、開いた「コントロールの設定」ダイアログで以下のように設定します。

コントロールの種類 配置用部品 [Locator] 配置先「ボディレイアウト」で配置部品として利用できます。
名前 Menu_Bar

「コントロールの設定」ダイアログの OK ボタンを押してダイアログを閉じます。
「部品・コントロール設定ウインドウ」の設定項目が有効になりました。
配置先「ボディレイアウト」「部品ペイン」として利用できます。
