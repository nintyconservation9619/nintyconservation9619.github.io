エミッタ

このタブではエミッタの形状やトランスフォームの設定を行います。

エミッタ設定

「 エミッタ設定」は、エミッタ形状とその大きさを設定する機能群です。
以下はグループ内設定項目の一覧です。

エミッタ形状

パーティクルの放出形状と範囲を決める機能です。
スケール 選択したエミッタ形状スケールをかける機能です。
放出位置ランダム パーティクルをランダムな位置から放出する機能です。

エミッタトランスフォーム

「 エミッタトランスフォーム」は、エミッタの「位置」、「回転」、「スケール」に関する設定とそのアニメーションを設定する機能群です。
以下はグループ内設定項目の一覧です。

エミッタトランスフォーム「位置(±)」 エミッタ配置時の位置を設定する機能です。
エミッタトランスフォーム「回転(±)」 エミッタ配置時の角度を設定する機能です。
エミッタトランスフォーム「スケール」 エミッタ配置時のスケールを設定する機能です。
ランダム幅の更新 エミッタトランスフォームの「位置(±)」と「回転(±)」で設定した乱数値を放出毎に更新するかを決める機能です。
参考：

エミッタの「位置」、「回転」、「スケール」は、「カーブエディタウィンドウ」でアニメーションできます。
