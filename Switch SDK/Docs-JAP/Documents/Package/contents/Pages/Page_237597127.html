以前のファームウェアからの変更点

ECMAScript

  • アロー関数内の this の挙動が ECMAScript 6 に準拠していなかった問題を修正しました。
    ※ただし、現在のバージョンではアロー関数の挙動が ECMAScript 6 に準拠していません。
    　今後の本体更新で ECMAScript 6 に準拠した際に互換性に支障をきたす恐れがありますので、アロー関数の利用を避けてください。
    • Functions > arrow functions > lexical "this" binding
    • Functions > arrow functions > this unchanged by call or apply
    • Functions > arrow functions > can't be bound, can be curried

HTML5

W3C の HTML5 の仕様 (https://www.w3.org/TR/2014/REC-html5-20141028/) に基づいたサポートリストです。

なお、一般的なブラウザと比べて以下の動作が異なりますのでご注意下さい。

  • video 要素の autoplay 属性には対応していません。
  • ルート要素に対する resize イベントは発行されません。iframe で読み込まれたリソースについては resize イベントが発行されます。
