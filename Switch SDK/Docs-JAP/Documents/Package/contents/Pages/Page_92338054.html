ファイルビューの基本操作

ファイルビューは、3DEditor に読み込んだ各種ファイルがツリー状に表示されるビューです。

ここでは、3DEditor のファイルビューの基本操作を説明します。

関連リンク 読み込まれているファイルをダブルクリックすることで、プロパティウィンドウを開いたり、アニメーションの再生や一時停止などの操作を行うことができます。

ファイルビューの機能

ファイルビューには、以下の機能があります。

基本機能

  • 編集されているファイルのアイコンには、赤い★マーク が表示されます。
  • 参照されていないテクスチャや、モデルにバインドされていないアニメーションなどのファイル名は、赤字 で表示されます。
  • テクスチャパターンアニメーションで使用しているテクスチャが参照切れの場合は、テクスチャパターンアニメーションのファイル名が 赤字 で表示されます。
  • 選択されているプロパティの項目をダブルクリックすると、項目に応じたプロパティウィンドウが開きます。
  • 選択されているファイル（ファイルのグループ）を右クリックすると、項目に応じた右クリックメニューが表示されます。
  • アニメーションファイルの新規作成をします。
  • アニメーションファイルは、アニメーションセットや別モデルへ、ドラッグ & ドロップで移動できます。
  • 選択されているファイル名をリネームできます。
  • スタートアップパスで開いたファイルには、 スタートアップアイコンが表示されます。
  • 読み込んだプロジェクトファイル ( .f3pj ) で使用されているアニメーション / アニメーションセットや、モデルの数がノード毎に表示されます。
  • ファイルビューの右クリックメニューで、ファイルを開き直すを選択すると、最終保存状態のファイルを再読み込みしてくれます。
    ※開き直す前までに未保存の編集があった場合は破棄されますのでご注意ください。
    未保存の編集がある場合は、警告ダイアログが表示されます。

ファイル名のリネーム機能

3DEditor で読み込んだファイルの名前をリネームできます。

参考：

ファイル名には、英数字と、-_. (ハイフン、アンダーバー、ドット) が使用できます。
上記以外の文字を使用すると警告ダイアログが表示されます。

アニメーションの再生 / 一時停止 機能

ファイルビューでプレビュー対象になっているアニメーションセット内にあるアニメーションの再生 / 一時停止がアイコンで操作できます。

関連リンク アニメーションセット内にある全てのアニメーションの再生 / 一時停止はアニメーションセットの右クリックメニューから操作できます。

アニメーションプレビューの ON / OFF 機能

アニメーションセット内にある、個々のアニメーションのプレビューをアイコンの操作で ON / OFF できます。

プレビューが OFF の場合は、アイコンがグレーオフします。

アニメーションセットのプレビューを切り替える

複数のアニメーションセットが存在する場合に、ラジオボタンでプレビューするアニメーションセットを切り替えることができます。

オブジェクトビューでモデルデータの表示 / 非表示を切り替える

ファイルビューのモデルファイルのアイコンを操作することで、オブジェクトビューでモデルデータの表示 / 非表示が切り替えられます。

モデルファイルのアイコンをクリックすると、アイコンがグレーオフしてモデルデータがオブジェクトビューで非表示になります。

関連リンク モデルファイルの右クリックメニューからも表示 / 非表示の切り替えができます。

tga ファイルを読み込んだ場合

tga ファイルを読み込んだ場合は、保存不可ですが他のテクスチャ中間ファイルと同じように一覧表示されます。

関連リンク 一覧表示されているテクスチャファイル（.tga ファイルを含む）のサムネイルをドラッグ & ドロップでマテリアルプロパティウィンドウのサンプラページに追加できます。
