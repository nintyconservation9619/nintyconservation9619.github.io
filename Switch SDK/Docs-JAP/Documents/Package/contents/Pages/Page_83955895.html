 プレビューノード

プレビューノードの機能について説明します。
プレビューはプレビューファイル（ .prev）として保存できます。 また、プレビューの設定結果は EffectViewer で確認できます。

基本設定 ファイル情報を見たり、プレビューの基本設定を行います。
SRT制御 エミッタセットに対してゲームアプリケーション側で制御可能な設定や、3Dモデルとエフェクトとの連携設定を行います。
自動移動 実際にエミッタセットを移動させた場合にエフェクトがビューアでどう見えるかを確認できる機能「自動移動」の設定を行います。
エミッタ エミッタノードの「エミッタ」機能に対してゲームアプリケーション側で制御可能な設定を行います。
放出 エミッタノードの「放出」機能に対してゲームアプリケーション側で制御可能な設定を行います。
パーティクル エミッタノードの「パーティクル」機能に対してゲームアプリケーション側で制御可能な設定を行います。
カラー エミッタノードの「カラー」機能に対してゲームアプリケーション側で制御可能な設定を行います。
スケール エミッタノードの「スケール」機能に対してゲームアプリケーション側で制御可能な設定を行います。
