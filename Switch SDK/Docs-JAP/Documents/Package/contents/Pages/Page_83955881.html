データ作成プラグイン

データ作成プラグインとは NintendoWare 用のデータ作成を補助するためのプラグイン群です。

主に、3ds Max 上のノードに対して NintendoWare 用のアトリビュートを設定したり、設定を検索したりするために使用します。
データ作成プラグインは 「Nintendo」メニューから実行します。

データ作成プラグインの種類

データ作成プラグインの一覧を次の表に示します。

プラグイン名 機能内容
マテリアルアトリビュート設定プラグイン
（Set Material Attribute） 		マテリアルに NintendoWare 用のマテリアルのアトリビュートを設定します。
サンプラアトリビュート設定プラグイン
（Set Sampler Attribute） 		テクスチャにサンプラのアトリビュートを設定します。
ノード圧縮不可設定プラグイン
（Set No Compress Node） 		ノードにノード圧縮不可フラグを設定します。
シェイプアニメーション設定プラグイン
（Set Shape Animation） 		ノードにシェイプアニメーションで更新する頂点属性を設定します。
ユーザーデータ設定プラグイン
（Set User Data） 		ノードにユーザーデータを設定します。
