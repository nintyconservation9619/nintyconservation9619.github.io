NX 本体内蔵フォントについて

使用許諾範囲

本フォントは、NX用のソフトウェア、コンテンツおよびこれらに関連するマニュアル等（電子的媒体にて販売・頒布されるものを含む）を開発・作成する目的にのみ使用可能です。上記以外の用途には使用しないでください。

本フォントは、一旦ビットマップフォントに変換してからであれば、必要に応じて改変しても構いませんが、改変したことに起因するトラブルはすべて改変した者の責任となります。

その他、使用許諾に関して質問がある場合は、任天堂の担当者までご相談ください。

管理上の注意事項

本フォントを任天堂の管理者の許可なく第三者に使用させること、並びに販売、貸与、譲渡、頒布等することを禁じます。

任天堂が求める場合には、ただちに本フォントを消去または返還してください。

NintendoWare での利用

bfttf 、xlor、xllt 各ファイルは、NintendoWare NX AddOnパッケージの、Resources/Fonts/NX に入っています。
