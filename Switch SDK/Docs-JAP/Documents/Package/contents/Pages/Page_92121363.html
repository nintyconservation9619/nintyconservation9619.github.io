【ユーザー設定】

ユーザー毎の設定ができます。

元に戻す

「 元に戻す」は、作業手順のアンドゥ操作「元に戻す」回数を設定する機能です。
以下はグループ内設定項目の一覧です。

無限 元に戻す機能を無限に続けるかを ON/OFF で切り替えます。
最大数 操作手順を最大何回まで記録するかを設定します。「無限」が OFF の場合に使用できます。

設定した内容は「OK」ボタンを押すことで確定し、 EffectMaker を再起動することで反映されます。

自動で読み込むワークスペースファイル

「 自動で読み込むワークスペースファイル 」は、 EffectMaker 起動時にワークスペースファイルを自動で読み込むかが設定できる機能です。
「ワークスペースファイル」を設定します。
設定した内容は「OK」ボタンを押すことで確定し、 EffectMaker を再起動することで反映されます。

プリセットフォルダパス

「 プリセットフォルダパス」は、エフェクトの作成時に利用できるプリセット用エミッタセットファイル（ .eset）の置き場所（フォルダ）を変更できる機能です。
フォルダの場所を設定します。
設定した内容は「OK」ボタンを押すことで確定し、 EffectMaker を再起動することで反映されます。

プリセットからエミッタセットを作成 エミッタセットをプリセットから作成する方法について説明します。
プリセットからエミッタを追加 エミッタをプリセットから追加する方法について説明します。

オプション

「 オプション」は EffectMaker で利用するファイルや便利機能が設定できます。

エミッタ エミッタファイルに関するダイアログの表示 / 非表示を設定できます。
テクスチャ テクスチャファイルに関するダイアログの表示 / 非表示を設定できます。
プリミティブ プリミティブファイルに関するダイアログの表示 / 非表示を設定できます。
プレビュー プレビューファイルに関する保存時の挙動を設定できます。

ドキュメントファイルパス

「 ドキュメントファイルパス 」は、 EffectMaker が参照するドキュメントの場所を設定できます。
「ドキュメントファイル」を設定します。
設定した内容は「OK」ボタンを押すことで確定し、 EffectMaker を再起動することで反映されます。

本設定項目は【ユーザー設定】と【プロジェクト設定】の両方にあります。
関連リンク 設定による挙動については 「【ユーザー設定】と【プロジェクト設定】共通の設定項目」 を参照してください。
