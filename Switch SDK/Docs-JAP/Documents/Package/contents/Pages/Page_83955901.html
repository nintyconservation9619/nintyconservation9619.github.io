ファイル

ファイルの入出力と、3DEditor 全般に関する操作を行います。

A

新規作成サブメニューを表示します。
B 3DEditor が対応しているグラフィックス中間ファイルを開きます。
読み込んだマテリアルのパラメータがシェーディングモデルに一致しない場合は、マテリアルの修正ダイアログが表示され修正するマテリアルを選択することができます。
 tga ファイルを開いた場合は、自動的に ftx に変換して読み込みます。
C

3DEditor が対応しているプロジェクトファイルを開きます。
読み込んだマテリアルのパラメータがシェーディングモデルに一致しない場合は、マテリアルの修正ダイアログが表示され修正するマテリアルを選択することができます。
D

選択されているファイルを閉じます。
モデルファイル (.fmdb、.fmda) を閉じた場合は、 モデルが参照しているアニメーションファイル、テクスチャファイルも同時に閉じます。
E

すべてのファイルを閉じます。
閉じるファイルの中に編集されたファイルが含まれていた場合は、変更したファイルの保存ダイアログが表示されます。
スタートアップパスから開かれたファイルは、全て閉じるでは閉じません。
F

3DEditor に読み込まれているファイルを選択して閉じます。
閉じるファイルは、ファイルを選択して閉じるダイアログから選びます。
G

選択されているファイルを上書き保存します。
初めて保存する場合は、保存場所とファイル名を指定するダイアログを表示します。
H

選択されているファイルに名前を付けて保存します。
保存場所とファイル名を指定するダイアログが表示されます。
I

場所を指定して保存ダイアログが開いて、ファイルを指定した場所に保存します。
J

すべてのファイルを上書き保存します。
※3DEditor で編集したファイルが対象になります。3DEditor で編集していないファイルは上書き保存されません。
K

ファイルビューで選択しているファイルの名前を変更します。
L

最近使ったファイルが一覧表示されます。
一覧から選択したファイルを 3DEditor に読み込みます。
M

最近使ったプロジェクトが一覧表示されます。
一覧から選択したプロジェクトを 3DEditor に読み込みます。
N

3DEditor を終了します。

新規作成サブメニュー

ファイルの新規作成に関する操作を行います。

A

プロジェクトを新規に作成します。
B

マテリアルアニメーションファイルを新規に作成します。
C

ボーンビジビリティアニメーションファイルを新規に作成します。
D

シーンアニメーションファイルを新規に作成します。

tga ファイルを開いた場合

3DEditor で tga ファイルを開いた場合は、自動的にテクスチャ中間ファイル（.ftx）に変換して読み込みます。
あくまで、簡易確認用として扱われるため変換フォーマットは 3D テクスチャーコンバーターのデフォルトのみサポートしています。

読み込んだ tga ファイルは保存することもできませんのでご注意ください。
関連リンク ファイルビューでは「保存不可」と表示されます。

 

変更したファイルの保存ダイアログ

ファイルメニューの全て閉じるを選択してファイルを閉じる場合に、閉じるファイルの中に編集されているファイルが存在した場合に表示されるダイアログです。

A 変更されたファイルが一覧表示されます。
チェックボックスが ON になっているファイルが保存されます。
B 一覧に表示されている全てのファイルに対して、チェックボックスの ON / OFF を切り替えます。
C チェックボックスが ON になっているファイルを保存してファイルを閉じます。
D チェックボックスが ON になっているファイルも保存せずにファイルを閉じます。
E ファイルを閉じる操作自体をキャンセルします。

ファイルを選択して閉じるダイアログ

3DEditor から複数のファイルを選択して閉じる場合に表示されるダイアログです。

A 閉じるファイルが一覧表示されます。
チェックボックスが ON になっているファイルが閉じられます。
B 閉じるファイルを拡張子毎に選択します。
選択した拡張子のチェックボックスが全て ON になり閉じるファイルの対象になります。
C 閉じるファイルを拡張子毎に対象外にします。
選択した拡張子のチェックボックスが全て OFF 状態になり閉じるファイルの対象外になります。
D 閉じるファイルの対象ファイル数を拡張子毎に表示します。
E 選択したファイルを閉じます。
F ファイルを選択して閉じるをキャンセルします。

場所を指定して保存ダイアログ

すべてのファイルを指定した場所に保存する場合に表示されるダイアログです。

A 保存先フォルダです。
B フォルダの参照ダイアログから、任意のフォルダを保存先に指定します。
C 保存ファイルの一覧表示です。
初期設定では、全ファイルのチェックボックスが ON になっていますので、保存したくないファイルはチェックボックスを OFF にします。
D 選択した拡張子のチェックボックスを すべて ON にします。
E 選択した拡張子のチェックボックスを すべて OFF にします。
F 保存できる対象ファイルの数です。
G チェックボックスを ON にすると、テクスチャを textures フォルダに保存します。
H チェックボックスを ON にすると、ファイルを上書きする際に確認ダイアログを表示します。
I ファイルの保存を実行します。
J すべて指定場所に保存をキャンセルします。

マテリアルの修正ダイアログ

読み込んだマテリアルのパラメータがシェーディングモデルに一致しない場合に表示されます。

A 修正が必要なマテリアルの一覧表です。
シェーディングモデルに一致させたいマテリアルのチェックボックスを ON にします。
初期状態は、全てにチェックが入った状態になります。
B 次回以降にシェーディングモデルが一致しないマテリアルがあった場合に、修正ダイアログを表示せずに自動的に修正を行って良い場合はチェックを ON にします。
自動修正を行うかどうかの設定は、ユーザー設定に保存されます。
C マテリアルの修正を実行します。
D マテリアルの修正をキャンセルしてダイアログを閉じます。
参考：

「次回以降自動的に修正を行う」の設定を切り替えたい場合は、[メインメニュー] → [ツール] → [ユーザー設定] にある入出力設定で切り替えることができます。

複数のファイルに対してチェックを入れる

ダイアログに一覧で表示されているファイルを、Ctrl キーを押しながら複数選択した後にチェックボックスにチェックを入れると選択した全てのファイルに対してチェックの ON / OFF が設定できます。

マージダイアログ

モデルファイルを右クリックし「モデルファイルをマージ」を選択して開きます。
選択したモデルに他のモデルファイルの設定を反映させることができます。 

A マージする内容が表示されます。
B マージ対象のファイル名とファイルパスが表示されます。
C マージ対象にマージするファイル名とファイルパスが表示されます。
D マージできる項目の種類と、マージできるデータ数が表示されます。
E マージ内容の詳細を、各項目(タブ)ごとに設定します。
参考：

設定の一部を上書きすることをマージと呼んでいます。
設定の全てが上書きされるわけではありません。

 

全般タブ

モデル全般のマージ設定を行います。

A シェイプをマージするかどうかを指定します。
B モデルの頂点データをマージするかどうかを指定します。
C 「ボーンスケールを利用する」設定をマージするかどうかを指定します。
D マージを実行、キャンセルします。
「OK」ボタンで全てのタブの設定が反映された上でマージが実行されます。

ボーンタブ

モデルのボーンのマージ設定を行います。
ボーンのマージは、各ボーンの「ビルボード」と「ボーンに関連するシェイプを表示する」の内容を上書きします。

A

モデルのボーンがリストアップされます。

ボーンの名称が同じ場合は自動的に対応するボーンがリストアップされます。
B リストから選択してボーンを個別に設定します。
C

リストに表示された全てのボーンのマージ設定をします。
D マージを実行、キャンセルします。
「OK」ボタンで全てのタブの設定が反映された上でマージが実行されます。

マテリアルタブ

モデルのマテリアルのマージ設定を行います。
マテリアルのマージは、各マテリアルの「マテリアル圧縮最適化において削除を許可する」設定を除くすべての設定を上書きします。

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

A モデルのマテリアルがリストアップされます。
マテリアルの名称が同じ場合は自動的に対応するマテリアルがリストアップされます。 
B リストから選択してマテリアルを個別に設定します。
C

リストに表示された全てのマテリアルのマージ設定をします。
D マージを実行、キャンセルします。
「OK」ボタンで全てのタブの設定が反映された上でマージが実行されます。
