経度

ここでは球形状エミッタの設定項目「経度」について説明します。

概要

弧の開き方「経度」は球系エミッタの時、エミッタを球体の Z 軸正方向を基準に東西（左右）方向に広げることで扇状にできる機能です。
パーティクル密度を維持したまま放出範囲を半球状にできます。

以下は「 設定項目」の概要です。
「弧の開き方」を「経度」にすると「 経度」が表示されます。以下はグループ内の設定項目です。

弧の幅（角度）

弧の開始位置を中心とした幅を角度で設定します。

弧の開始角度

弧の開始位置を角度で設定します。 デフォルト値では Z 座標の正方向が開始地点です。

弧の開始角度ランダム

弧の開始角度をランダムにするかどうかを「 チェックボックス」ON/OFF で切り替えます。

設定例

以下は「弧の開き方」の「経度」の設定例と設定内容の解説図です。

A エミッタ半径 「エミッタ半径」を 50 に設定しています。
B 弧の幅 「弧の幅」を 90 度に設定しています。
エミッタ形状を球体の Z 軸正方向を基準に東西（左右）方向に 45 度ずつ 90 度広げた扇状の範囲とした設定です。
C 弧の開始角度

「弧の開始角度」を 45 度に設定しています。
「弧の幅」の基準位置を45度ずらした設定です。
