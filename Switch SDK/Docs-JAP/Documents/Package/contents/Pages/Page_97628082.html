エミッタカラーに対する乗算

ここでは、エミッタセット内すべてのエミッタの「グローバル乗算カラー」に対して、ここで設定した値を乗算する機能「エミッタカラーに対する乗算」について説明します。

概要

「エミッタカラーに対する乗算」は、エミッタノードのカラータブ「グローバル乗算カラー」に設定した値を乗算できる機能です。
エミッタ時間アニメを設定したカラーに対して色を制御できます。
以下は の概要です。

エミッタカラーに対する乗算

「エミッタカラー」を使用するかどうかを「 チェックボックス」ON/OFF で切り替えます。
設定を ON にすると「 エミッタカラーに対する乗算」が表示されます。以下はグループ内の設定項目です。

カラー 0 、カラー 1

「カラー 0」はエミッタセット内すべてのエミッタの「グローバル乗算カラー 0」に対して、ここで設定した値を乗算します。
「カラー 1」はエミッタセット内すべてのエミッタの「グローバル乗算カラー 1」に対して、ここで設定した値を乗算します。
