履歴の初速

ここでは、履歴の初速を設定する機能「履歴の初速」について説明します。

概要

「履歴の初速」は、「履歴」に対してランダムな方向に初速を与える機能です。
以下は「 設定項目」の概要です。

履歴の初速

履歴」の初速を「 ダイヤル」や「 テキストボックス」により設定します。
値が大きいほど履歴が大きく動くようになります。
また、値を大きくすることで、履歴を波形で動かすときの振幅を大きくすることができます。

関連リンク 以下は関連ページへのリンクです。

履歴の変化周期 指定軸方向に変化する波形の周期をフレームで設定します。
方向変化率 波形の変化率を設定します。
