OpenGL ライブラリ

はじめに

OepnGL とは Khronos Group が策定しているリアルタイムグラフィックス・コンピュート API です。

OpenGL は NintendoSDK NX Addon でサポートされており、Nintendo SDK ユーザが OpenGL を利用する既存のタイトル、フレームワークの移植を目的とした場合を想定して提供されています。

PC 動作環境

NintendoSDK の動作環境に準拠します。

GPU ドライバのインストール方法 

ご利用の PC に OpenGL の環境をセットアップしてください。

Nintendo Dev Interface から入手することが出来る「NVIDIA Graphics Driver」パッケージは OpenGL をサポートしています。

サポートされるバージョン

Nintendo Switch™ は OpenGL® 4.5 と  OpenGL® ES 3.2 に準拠しています。
