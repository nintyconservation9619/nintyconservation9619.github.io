回転速度

ここではパーティクル全体の回転速度を設定する機能「回転速度」について説明します。

概要

「回転速度」は、放出されたパーティクル全体を「回転軸」で指定した軸を中心に回すことができる機能です。
エフェクト全体を竜巻形状にできます。
以下は「 設定項目」の概要です。

回転速度

回転速度を設定します。符号により、回転方向が変わります。
設定した値はパーティクルの位置に毎フレーム加算される為、等速な回転挙動にできます。

参考：

「 パーティクル時間アニメ」ボタンを押すことにより「 回転速度(パーティクル時間アニメ)」が表示さ、カーブエディタでアニメーションを設定できます。
アニメーションは「 CPUで動作させる」ON で有効にできます。
