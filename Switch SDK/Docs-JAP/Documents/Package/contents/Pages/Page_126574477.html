シェーダーリソースファイル

概要

シェーダーリソースファイルは、シェーダーコンバーターによって作成されたバイナリーイメージを表現します。

シェーダーリソースファイルは、あらかじめコンパイルされたシェーダーを表現することに加えて、シェーダーのバリエーションを扱う仕組みを提供します。

シェーダーファイルリソース

シェーダーファイルリソースは nn::gfx::ResShaderFile として定義されています。

シェーダーファイルリソースはシェーダーコンバーターによって作成されるバイナリーイメージそのものを表し、リソースファイルクラスの共通機能を利用することができます。

シェーダーコンテナーリソースの取得

GetShaderContainer により、シェーダーコンテナーリソースを取得することができます。

シェーダーコンテナーリソース

シェーダーコンテナーリソースは nn::gfx::ResShaderContainer として定義されています。

シェーダーコンテナーリソースの機能を使う前に、まずシェーダーコンテナーリソースを初期化する必要があります。

初期化・終了

Initialize によってシェーダーコンテナーリソースを初期化、Finalize によってシェーダーコンテナーリソースを終了します。

Initialize 時にメモリープールを指定しなかった場合は、リソース内部のメモリープールが使われます。シェーダーコンバーターによるリソース作成時に --disable-memory-pool オプションが指定されている場合は、リソース内部にメモリープールを所有していないため、この方法で初期化することはできません。
使用者側のメモリープールがシェーダーコンテナーリソースを含んでいる場合は、そのメモリープールを指定して Initialize を呼ぶことによって、リソース内部のメモリープールのかわりに使用者側のメモリープールを使うことができます。

バリエーション数の取得

GetShaderVariationCount により、シェーダーコンテナーリソースがもつシェーダーのバリエーション数を取得します。

シェーダープログラムリソースの取得

GetResShaderProgram により、指定されたバリエーションのシェーダープログラムリソースを取得します。

シェーダープログラムリソース

シェーダープログラムリソースは nn::gfx::ResShaderProgram として定義されています。

シェーダー情報の取得

GetShaderInfo により、GFX のシェーダー情報を取得します。

シェーダーオブジェクトの取得

GetShader により、GFX のシェーダーオブジェクトを取得します。初期化前はシェーダーオブジェクトの領域には 0 が埋められています。

初期化・終了

Initialize を呼ぶと、GetShaderInfo によって得られる情報を使って、GetShader によって得られるシェーダーオブジェクトを初期化します。

Finalize を呼ぶと、GetShader によって得られるシェーダーオブジェクトを終了します。
