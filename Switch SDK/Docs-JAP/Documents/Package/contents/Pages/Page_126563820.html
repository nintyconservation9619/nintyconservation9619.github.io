変調

顔モデルパーツの色替えに対応するために、変調を使用して各パーツのカラーを決定しています。

変調に関連する描画パラメータ

nn::mii::DrawParam は、顔モデルや顔輪郭テクスチャ、マスクテクスチャを構成する各パーツの描画パラメータを保持します。

nn::mii::DrawParam から変調にかかわる以下のパラメータを取得できます。顔モデルの描画の際には、このパラメータをユニフォームに設定して適切に変調が行われるようにしてください。

パラメータの種類 説明
mode 変調の種類です。nn::mii::DrawType::ModulateType 列挙型によって表されます。
const0 定数カラー1
const1 定数カラー2
const2 定数カラー3
texture テクスチャ

変調の種類

似顔絵ライブラリで使用する変調の種類は以下の通りです。

nn::mii::DrawType::ModulateType 列挙型と関連しています。詳しくは API リファレンスの nn::mii::DrawType::ModulateType の項目を参照してください。

変調の種類 説明
nn::mii::DrawType::ModulateType_Constant 定数カラー（const0）で描画します。
nn::mii::DrawType::ModulateType_Texture テクスチャをそのまま描画します。
nn::mii::DrawType::ModulateType_RgbLayered テクスチャの RGB 各成分に定数カラー（const0～const2）を掛けて描画します。
nn::mii::DrawType::ModulateType_Glass テクスチャの R 成分に定数カラー（const0）を掛けて描画します。アルファ値は、テクスチャの G 成分に補正値（const1）を使用して補正したアルファ値を使用します。詳しくは API リファレンスを参照してください。
nn::mii::DrawType::ModulateType_Alpha 定数カラー（const0）をカラーとし、アルファはテクスチャのアルファを使用して描画します。
nn::mii::DrawType::ModulateType_AlphaOpa 定数カラー（const0）にテクスチャのアルファの値を掛けて描画します。アルファの値は 1.0f 固定です。
nn::mii::DrawType::ModulateType_IconBody 定数カラーに色が含まれていません。お気に入りの色を取得して使用してください。
