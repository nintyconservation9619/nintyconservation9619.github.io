カーブエディタの基本操作

関連リンク カーブエディタは、アニメーションプロパティウィンドウのカーブを選択しても開きます。
選択したアニメーションのカーブを確認して編集します。

カーブエディタの基本的な機能は、以下になります。

  • アニメーション編集が不可状態時のメッセージ表示
  • アニメーションカーブの色分け表示
  • カーブ編集時の有効範囲の指定
  • カーブとキーの選択状態の確認
  • キーの追加
  • コピーと貼り付け
  • フレームや値のクランプ設定
  • カーブのプレラップとポストラップの指定

カーブエディタ操作時の注意

  1. カーブエディタで水平の直線カーブを作成した場合は、保存時にコンスタントカーブになります。 （保存したアニメーションを再読む込みするとキーが一つになります）
  2. カーブエディタにスケルタルアニメーションのカーブも表示されますが、3DEditor は、スケルタルアニメーションの編集には対応していません。
    ※スケルタルアニメーションは編集不可ですが、カーブエディタ上では視認性向上の為、カラー表示されます。
  3. カーブエディタにシェイプアニメーションのベース形状が表示されますが、3DEditor は、シェイプアニメーションの編集には対応していません。

アニメーション編集が不可状態

特定の条件により、カーブエディタでアニメーションの編集が不可状態の場合があります。
その場合は、カーブビュー上にアニメーションが編集できない理由が表示されます。

アニメーションカーブの色分け表示

カーブエディタでは、カーブに色をつけて表示します。
※カラーカーブは、R G B で表示されます。

カーブツリービューに表示されるノード名の左に表示される四角形の色がカーブと同色になります。

カーブ編集時の有効範囲

カラーアニメーション、テクスチャパターンアニメーション、ビジビリティアニメーションなど、カーブの値を編集する際に「有効範囲外」は紫色に塗りつぶされます。
また、フレームの有効範囲外はグレーオフされて表示されます。

A カーブ編集時の値有効範囲外で、紫色に塗りつぶされます。
B カーブ編集時のフレーム有効範囲外で、グレーオフされます。
参考：

シェーダーで定義する UI 最小値 ( min ) 、UI 最大値 ( max ) からカーブ編集の有効範囲を設定することができます。

カーブとキーの選択

カーブビューでは、カーブ、キー、スロープを選択（複数選択）することができます。

状態 ビューの表示 選択方法
未選択状態 カーブ表示領域で、何もない領域を選択します。
カーブ選択状態 カーブをマウスの左ボタンで範囲選択します。
キー選択状態 カーブに含まれている選択したいキーを左ボタンクリックするか、マウスの左ボタンで範囲選択します。
キーが選択されると、影響を受けるカーブのラインがハイライトされスロープハンドルが表示されます。
スロープ選択状態 キーを選択した状態で、スロープハンドルの先端を左ボタンクリックするか、マウスの左ボタンで範囲選択します。
スロープ連結の統合 / 分割はツールバーのアイコンで変更します。

キーの追加

既存カーブの任意のフレームに、キーを追加する手順を説明します。

1. キーを追加するカーブを選択します。
2. キーを追加したいフレームにカーブビュー上にある赤いラインを合わせて「S」キーを押します。
3. 選択した場所にキーが追加されます。
4. 追加したキーを選択して、ツールバー（カーブエディタ）にある「フレーム」 と「値」に数値を入れます。

関連リンク キーの追加は、カーブビュー上のマウス操作でも可能です。

 

コピーと貼り付け

カーブビュー上で選択したカーブや、キーは「Ctrl+C」「Ctrl+V」でコピーと貼り付けができます。

コピー操作には以下の 2 種類があります。

  • カーブをコピーする
  • キーをコピーする

貼り付け操作は、選択されているカーブに対して行うことができます。

関連リンク カーブやキーの貼り付け方法には、「挿入」「置換」「融合」があり、カーブビューで変更できます。

フレームや値のクランプ

カーブを編集する際に、フレームや値をクランプして作業できます。

関連リンク クランプの有効 / 無効はツールバー（カーブエディタ）にある「フレームをクランプする」「値をクランプする」ボタンで切り替えます。

フレームのクランプを有効にすると、クランプする範囲の左右に緑線が表示されます。

値のクランプを有効にすると、クランプする範囲の上下に緑線が表示されます。

カーブのプレラップとポストラップの指定

カーブエディタのカーブ要素から「プレラップ」「ポストラップ」の指定ができます。

プレラップ clamp 最初のキーの値を返します。 最初のキーより前のフレームでの動作を指定します。
repeat カーブが単純に繰り返した値を返します。
mirror カーブが反転しつつ繰り返した値を返します。
relative_repeat カーブ端の値から相対的に繰り返した値を返します。
ポストラップ clamp 最後のキーの値を返します。 最後のキーより後のフレームでの動作を指定します。
repeat カーブが単純に繰り返した値を返します。
mirror カーブが反転しつつ繰り返した値を返します。
relative_repeat カーブ端の値から相対的に繰り返した値を返します。
