5. UV ハードエッジを保護してリダクションを行う

UV 空間における開放端の事を、 UV ハードエッジと呼びます。また、UV 内の閉じられた一つの図形の事を、Mayaでは UV シェルと呼びます、一般的には UV アトラスと呼ばれます。つまり、UV ハードエッジはUVシェルの開放端という事になります。

開放端と保護角　に UV ハードエッジと、その保護角についての詳細があります。

--property-flag Position|UV　に変更する事で、UV ハードエッジを保護する事が出来ますが、ここでは更に連続する UV ハードエッジの成す角度を指定する事で保護する方法を説明します。

開放端同様に図７のように、2本の UV ハードエッジが連続しているときに、その成す角度Θを指定する事で、アトラスの外周の形が大きく変化しないように指定する事が出来ます。これは開放端の時と全く同じです。


パラメーターD
"--target-rate-lod-level1 0.5 --target-rate-lod-level2 0.25 --property-flag Position|UV --optimize-position-policy OptimalOnSpace --uv-weight 0.1 --lock-uv-hard-edge-angle 90"

以下のバッチファイルで実行する事が出来ます。

%NINTENDO_SDK_ROOT%\Samples\Sources\Tools\3dTools\3dIntermediateFileOptimizer\PolygonReductionTutorial\ExecuteParameterD.bat

これまで他のパラメーターで抑制する効果を得ていたシェル境界の問題に対して、本質的な対応をここでは行います。

パラメーターCに、強い UV ハードエッジの保護（--lock-uv-hard-edge-angle 90）を付加しました。これをパラメーターD と呼びます。設定角については、 中間ファイルオプティマイザ にも記載があります。

 図12_1 UVハードエッジ保護の結果　（--lock-uv-hard-edge-angle 90）

保護角を 90度に設定した結果です。図12_0 に見られるような歪みは UV シェルの形状変化(下段)によって引き起こされています。そのため 90 度指定で直角に近い角度を保護する事で、シェルの形状変化が抑制出来た結果、UV の歪みが消えています。
