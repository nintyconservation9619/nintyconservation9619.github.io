UI 説明

エディタのユーザーインターフェイス（UI）各部の概要を説明します。

レイアウトエディタウインドウ LayoutEditor を起動した時に開くウインドウの各部位の名称と役割について説明します。
レイアウトウィンドウ LayoutEditor のメインウインドウを構成する、各部位の名称と役割について説明します。
プロパティウィンドウ LayoutEditor のノードを設定する為の「プロパティウインドウ」について説明します。
カーブエディタウィンドウ LayoutEditor のアニメーションを作成、編集する「カーブエディタウインドウ」について説明します。
テクスチャウィンドウ LayoutEditor のテクスチャを登録し、フォーマットを選択できる「テクスチャウインドウ」について説明します。

連番テクスチャの読み込みウインドウ

連番テクスチャからアニメーションを自動生成する「連番テクスチャの読み込みウインドウ」について説明します。
グループウィンドウ レイアウトの種類に応じてデータをグループ化できる「グループウインドウ」について説明します。
親子階層ウィンドウ レイアウトの構造を構築できる「親子階層ウインドウ」について説明します。
ペイン情報貼り付け制御 複数のペイン間でコピーする項目を制御できる「ペイン情報貼り付け制御ウインドウ」について説明します。
アニメーション区間タグ編集ウィンドウ アニメーションの範囲を設定できる「アニメーション区間タグ編集ウインドウ」について説明します。
ペイン検索ウィンドウ 目的のペインをサーチできる「ペイン検索ウインドウ」について説明します。
色見本 レイアウト内で良く使用するカラーセットを作成できる「色見本」について説明します。
カラーピッカー 色の指定を視覚的に設定できる「カラーピッカー」について説明します。
アニメーション共有情報ウィンドウ よく使用するアニメーションを共有できる「アニメーション共有情報ウインドウ」について説明します。
部品・コントロール設定ウィンドウ レイアウトデータを部品として登録、設定できる「部品・コントロール設定ウインドウ」について説明します。
部品パレットウィンドウ リストアップされた部品から任意の部品を選択できる「部品パレットウインドウ」について説明します。
LayoutViewer LayoutEditor 用ビューア「LayoutViewer」 について説明します。
レイアウト設定ウインドウ レイアウトウインドウの環境設定について説明します。
