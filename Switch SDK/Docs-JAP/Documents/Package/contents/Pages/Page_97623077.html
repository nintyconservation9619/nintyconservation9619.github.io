放出間隔

ここではエミッタセット内のパーティクルすべてに対して放出間隔に設定した値を乗算する機能「放出間隔」について説明します。

概要

「放出間隔」は、「放出間隔タイプ」が「時間」のときに、エミッタセット内のパーティクルすべてに対して放出間隔に設定した値を乗算する機能です。

放出間隔タイプ「時間」 パーティクルが放出される間隔を「時間」で設定する放出間隔タイプ「時間」について説明します。
放出間隔タイプ「時間」の放出間隔 パーティクルが放出される間隔を時間（フレーム）で設定する「放出間隔」について説明します。

放出間隔

放出間隔に設定する値を「 ダイヤル」や「 テキストボックス」により設定します。
