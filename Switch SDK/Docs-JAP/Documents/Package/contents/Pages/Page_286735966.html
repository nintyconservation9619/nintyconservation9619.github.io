サウンドパラメータの初期値の設定

初期値設定ファイル

サウンドプロジェクトファイルと同じフォルダ内にある、初期値設定ファイル「プロジェクト名.fspj.defaultparam」を編集することで、サウンドパラメータの初期値を設定することができます。

初期値設定ファイルの出力

  1. サウンドプロジェクトを開きます。
  2. サウンドプロジェクトを閉じます。
  3. サウンドプロジェクトファイルと同じフォルダに、初期値設定ファイル「プロジェクト名.fspj.defaultparam」が出力されます。

既に初期値設定ファイルが存在する場合は、初期値設定ファイルを上書きすることはありません。

初期値設定ファイルを削除しても、サウンドプロジェクトを閉じると初期値設定ファイルは再出力されます。

初期値設定ファイルの編集

出力された初期値設定ファイルは、yaml フォーマットで記述されています。

すべてのパラメータはコメントアウトされていますので、必要なパラメータのコメントを外して初期値を設定してください。


  • 例：ストリームサウンドの音量の初期値を 127 にする場合。

ストリームサウンドのラベル「StreamSound:」以下の「#  Volume: 96」が編集対象です。

## ストリームサウンド
StreamSound:
　　：
　～～～
　　：
#  # 音量
#  Volume: 96

↓ 先頭のコメントアウト文字「＃」を外して初期値を 127 にします。

## ストリームサウンド
StreamSound:
　　：
　～～～
　　：
#  # 音量
  Volume: 127
注意：

パラメータ「Volume:」の先頭にあるスペース（空白文字）は削除しないでください。スペースによる字下げはパラメータの認識に必要です。


  • 例：ウェーブサウンドのアイテムカラーをピンクにする場合。

ウェーブサウンドのラベル「WaveSound:」以下の「#  Color: None」が編集対象です。

## ウェーブサウンド
WaveSound:
　　：
　～～～
　　：
#  # アイテムカラー None, Red, Orange, Yellow, Green, Cyan, Blue, Purple, Pink のいずれかを指定します。
#  Color: None

↓ 先頭のコメントアウト文字「＃」を外して初期値を Pink にします。

## ウェーブサウンド
WaveSound:
　　：
　～～～
　　：
#  # アイテムカラー None, Red, Orange, Yellow, Green, Cyan, Blue, Purple, Pink のいずれかを指定します。
  Color: Pink
注意：

パラメータ「Color:」の先頭にあるスペース（空白文字）は削除しないでください。スペースによる字下げはパラメータの認識に必要です。

初期値設定ファイルの読み込み

サウンドプロジェクトと開くと初期値設定ファイルは自動的に読み込まれ、サウンドアイテムの新規作成時に初期値が反映されるようになります。

初期値設定ファイルを編集した場合は、サウンドプロジェクトを開き直してください。

初期値設定ファイルの記述に誤りがあるとエラーになり、初期値は反映されません。
