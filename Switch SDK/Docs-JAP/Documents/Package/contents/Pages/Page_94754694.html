「初速のランダム」と「運動量ランダム」

ここではパーティクルの移動軌跡をランダムにする２つの機能「初速のランダム」と「運動量ランダム」の違いについて説明します。

概要

「初速のランダム」は、パーティクル放出時の初速を毎回ランダムにできる機能です。

「運動量ランダム」は、パーティクル放出時に、パーティクル個々が持つ運動量をランダムにできる機能です。
「フィールド」のようにパーティクルが放出された後にパーティクルの運動に対して影響を与える機能にも作用します。
