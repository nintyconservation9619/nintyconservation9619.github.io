色情報ウインドウ

「色情報ウインドウ」では色見本ウインドウに登録されている色情報の編集ができます。

色情報ウインドウ
A 表示色

左側に RGB 、右側にアルファが表示されます。

クリックすることで「カラーピッカー」が開きます。
B RGBA RBGA の各値を直接設定することができます。
C コメント コメントを設定することができます。
