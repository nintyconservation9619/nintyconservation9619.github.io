１．作業の開始

LayoutEditor を起動し、作業を開始します。

LayoutEditor の起動 

以下の起動ファイルをマウス左ボタンでダブルクリックすることにより LayoutEditor が起動します。

\Tools\Graphics\LayoutEditor\LayoutEditor.exe

２．ファイルを読み込む 

レイアウトデータとなるレイアウトファイル(.flyt)を LayoutEditor に読み込みます。
以下の場所にあるレイアウトファイルを例に説明します。

\Samples\Sources\Resources\Ui2d\expression\flyt\test\Font_Sample.flyt

以下いずれかの操作を行うとレイアウトデータファイル「 Font_Sample.flyt 」が読み込まれます。

  • メニューから開く
    メインメニュー項目 [ ファイル ] → [ 開く ] → [ すべて ] でダイアログが開きます。
    開いたダイアログで「 Font_Sample.flyt 」を選択してから「開く」ボタンを押します。
  • ショートカットで開く
    ショートカット [ Ctrl ＋ O ] でダイアログが開きます。
    開いたダイアログで「 Font_Sample.flyt 」を選択してから「開く」ボタンを押します。
  • ドラッグ & ドロップで開く
    エクスプローラー上で「 Font_Sample.flyt 」を選択し、ワークスペース上にドラッグ & ドロップします。

プレビューには以下の方法があります。

ビューアに転送

作成中のレイアウトデータの状態をビューアに転送します。
アニメーションは転送されません。
アニメーションが付いている場合は、現在表示されているアニメーションフレームの状態でビューア転送されます。

  • メニューから転送
    メインメニュー項目 [ ツール ] → [ ビューアに転送 ] を選択します。
  • ショートカットから転送
    ショートカット [ Shift ＋ F5 ] を押します。

ビューアに転送　（アニメモード）

レイアウトデータとアニメーションデータ両方の転送をします。
アニメーションの再生される長さは、カーブエディタのアニメーション再生フレーム区間となります。
（カーブエディタ、及び再生フレーム区間については、アニメーションの機能説明を参照して下さい。）

  • メニューから転送
    メインメニュー項目 [ ツール ] → [ ビューアに転送　(アニメモード) ] を選択します。
  • ショートカットから転送
    ショートカット [ F5 ] を押します。

ビューアの終了

以下いずれかの操作を行うことでビューアを終了します。

  • メインウインドウから閉じる
    ビューアのメインウインドウ「 閉じる」ボタンを押します。
  • タイトルバーから閉じる
    ビューアウインドウタイトルバーをマウス右ボタンクリックで [ 閉じる ] を選択します。
  • ショートカットで閉じる
    ショートカット [ Alt ＋ F4 ] を押します。

４．作業の完了

以下いずれかの操作を行うと、開いているすべてのファイルを閉じて LayoutEditor が終了します。

  • メインメニューから終了
    メインメニュー項目 [ ファイル ] → [ 終了 ]
  • メインウインドウから終了
    LayoutEditor のメインウインドウ「 閉じる」ボタンを押します。
  • ショートカットキーからの終了
    選択状態のレイアウトデータから順に終了します。
    ショートカット [ Ctrl ＋ Q ] を押します。

未保存のファイルがある場合はダイアログが表示されますので、保存先フォルダを指定してから「OK」ボタンを押します。
