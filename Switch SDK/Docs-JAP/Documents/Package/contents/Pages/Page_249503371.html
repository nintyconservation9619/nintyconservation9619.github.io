レイアウトウィンドウをタブ化しない機能

「レイアウトウインドウをタブ化しない」機能は、複数のレイアウトウインドウを開く時の、表示方法を選択する機能です。
「タブ表示」による切り替え表示、「個別のウインドウ」による別々の表示、の二形態から選択をします。

[ レイアウトエディタウインドウ ] → [ メニューバー ] → [ ウインドウ ] → [ レイアウトウインドウをタブ化しない ]

レイアウトウインドウをタブ化しない

チェックを入れると、レイアウトウインドウを開く際に、個別のウインドウとして開くようになります。
作業形態に合わせて選択して下さい。

※タブ化の設定を切り替えるには、全てのレイアウトウインドウを閉じる必要があります。
