はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている HID  (Human Interface Device) ライブラリの機能について説明します。

 

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象とします。

 

各章の説明

「マニュアル」では、HID ライブラリの提供する個々の機能について使用方法を解説します。

 

関連資料

HID の関数リファレンス、サンプル解説書と併せて参照してください。

またコントローラーの概要については「コントローラーガイド」を参照してください。特にアプリケーションを開発する上で注意すべき点については「アプリケーションのコントローラー対応」を参照してください。

HID ライブラリの機能

HID ライブラリは、ゲームパッド、キーボード、タッチスクリーン、加速度センサなど、各種 HID を制御するための API を提供します。
