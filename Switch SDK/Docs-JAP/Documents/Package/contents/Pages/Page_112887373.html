開始処理

ここではエフェクト再生開始時の処理を決める機能「開始処理」について説明します。

概要

「開始処理」は、エフェクトの再生開始時の挙動を決める機能です。
エフェクト出現時にアルファやスケールで徐々に表示させることで、パーティクルが突然表示されることによる違和感を軽減できます。
以下は「 開始処理」の「 設定項目」です。

開始処理

パーティクル放出時の挙動が設定できます。

αフェード すべてのパーティクルに対して「フェードイン時間」で指定した時間をかけて透明状態から徐々に出現させる機能です。
スケールフェード すべてのパーティクルのサイズを「フェードイン時間」で指定した時間をかけて徐々に拡大して出現させる機能です。
フェードイン時間 フェードインに要する時間をフレームで設定する機能です。「αフェード」や「スケールフェード」をONにすることで利用できます。

設定例

「 アニメーションを停止ボタン」を押すことでパーティクルの放出を停止し、「 アニメーションを再生ボタン」を押すことで放出を開始します。
以下は「開始処理」の各機能を ON/OFF した場合の放出開始時の比較画像です。

A フェード OFF フェード処理なしの状態です。
B アルファフェードON 「フェードイン時間」で指定した時間をかけて透明な状態から徐々に出現します。
C スケールフェードON 「フェードイン時間」で指定した時間をかけてサイズ 0 の状態から徐々に出現します。
