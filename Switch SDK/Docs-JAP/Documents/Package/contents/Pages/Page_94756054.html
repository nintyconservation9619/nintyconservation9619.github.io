ランタイムライブラリの変更点

名前空間

NintendoSDK 0.7.0　にて NintendoWare オーディオライブラリは、 nw 名前空間から nn 名前空間 に移行しました。
nn 名前空間への移行に伴い、名前空間の名称の見直しや統合も行いました。

新オーディオモジュール 旧オーディオモジュール
nn::atk nw::snd
nn::atk::viewer

nw::snd::edit

nw::snd::ctrl

nn::atk は、nw::snd をベースとしつつも、NintendoSDK 向けにインターフェイスや仕様を調整したメインライブラリで、今後、下位ライブラリである nn::audio の新機能への対応や、nn::atk 自体の機能追加を予定しています。
一方、nw::snd は Wii U 向けアプリケーションの移植を考慮したライブラリで、NW4F とのインターフェイス互換を可能な限り維持して提供することを検討中です。

※nw::snd インターフェイスの提供時期は未定です。

低レイヤオーディオライブラリの変更

NintendoWare オーディオライブラリはこれまで WinExt を低レイヤオーディオライブラリとして利用してきましたが、nn::atk ライブラリでは nn::audio ライブラリに変更しました。 

API、メソッド名の変更について

nn::atk ライブラリでは、NintendoSDK としての一貫性を高めるために、各種API、メソッド名の変更が行われています。

SoundSystem にエフェクトを追加する際の注意点

チャンネル設定などの一部のパラメータは SoundSystem::AppendEffect 後には変更できず、AppendEffect 前に設定する必要があるためご注意ください。

SoundSystem::AppendEffect 前にのみパラメータを変更できるAPI 、および任意のタイミングでパラメータを変更できる API は以下のようになっています。

エフェクトのパラメータを Set 関数で設定した直後に、 Get 関数で取得できる値は Set 関数で設定した値となっています。
ただし、 Set 関数で指定したパラメータが実際に反映されるのは、サウンドスレッドの更新処理が行われたあととなります。
Get 関数で得られたパラメータが、現在エフェクトに反映されているパラメータとは必ずしも一致しない点にご注意ください。

ツールの変更点

SoundMaker、各種コマンドラインツールについて

SoundMaker や各種コマンドラインツールは、nw::snd と nn::atk で共通のものを利用します。

ただし、以下の点にご注意ください。

  • NW4F 用の SoundMaker や各種コマンドラインツールは、NintendoSDK では利用できません
    NintendoSDK に同梱されているツールをご利用ください 
  • 今後の機能追加は「nn::atk のみ」となるため、SoundMaker などで新機能を利用すると、nw::snd では利用できないバイナリリソースファイルが出力されることがあります

プレビューツールの変更点

同梱されるプレビューツールについて

これまでの NintendoSDK では、nw::snd を使用したプレビューツールである nw::snd SoundPlayer が同梱されていましたが、
NintendoSDK 0.8.0 から nn::atk を使用するプレビューツールである AtkPlayer に変更されました。

AtkPlayer では nw::snd SoundPlayer が持つ機能に加えて nn::atk の追加機能にも対応していく予定です。
詳しい使い方などは、AtkPlayer をご確認ください。
