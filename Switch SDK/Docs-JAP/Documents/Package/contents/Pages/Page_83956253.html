Biped

Biped を使ったキャラクタの出力にも対応しています。

ただし、Biped は内部で特殊な処理をしており、以下のようなケースでは正常に表示できません。

  • 1.0 以外のスケールは対応しません
    Biped のボーンに対して 1.0 以外のスケールをかけた場合、スケールが考慮された精確な行列情報を取得できないため、正常に表示できません。

キャラクタ設定時には以下の点をご考慮ください。

  • フィギュアモード、及びスキンモードの設定
    Biped の親に Dummy オブジェクト、または PointHelper オブジェクトを使い、原点にてスキンポーズ設定を行ってください。
