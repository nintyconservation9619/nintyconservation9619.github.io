「空気抵抗」を加味

ここでは GPU ノイズに空気抵抗を反映させる機能「「空気抵抗を加味」について説明します。

概要

「「空気抵抗」を加味」は、「放出」タブの機能「空気抵抗」の結果を「GPUノイズ」に反映させるかを設定できる機能です。
この機能を利用し、例えば「空気抵抗」を強くかかけることによって、パーティクルに対するフィールドのかかり方を徐々に弱めることができます。

関連リンク 以下は関連ページへのリンクです。

空気抵抗 放出されたパーティクル全体の初速を減衰または加速させる機能「空気抵抗」について説明します。

以下は「 設定項目」の概要です。

「空気抵抗」を加味

エミッタノード「放出」タブの「空気抵抗」の影響を受けるかを「 チェックボックス」ON/OFF で切り替えます。

設定例

以下は「「空気抵抗」を加味」を ON/OFF した場合の比較画像です。

A 「空気抵抗」を加味 OFF GPUノイズは空気抵抗の影響を受けません。
B 「空気抵抗」を加味 ON 、空気抵抗 0.95 GPUノイズは空気抵抗の影響を受け、パーティクルが徐々に減速します。
C 「空気抵抗」を加味 ON 、空気抵抗 1.05 GPUノイズは空気抵抗の影響を受け、パーティクルが徐々に加速します。
