ノード圧縮不可設定プラグイン

transform ノードおよび joint ノードに圧縮機能を有効にしても圧縮されないようにするフラグを設定します。

設定の検索もできます。

使用方法

「Nintendo > Set No Compress Node...」を選択するとウィンドウが表示されます。

Selected Transform Node 欄には、選択されている transform ノードまたは joint ノードが表示されます。

参考：

4 つ以上のノードが選択されている場合は省略表示となり、右端に選択された数が表示されます。

設定方法

transform ノードまたは joint ノードを選択（複数選択可）すると、Set 欄に選択したノードの現在の設定が表示されます。
複数選択の場合、最初に選択したノードの設定が表示されます。

コンボボックスをクリックして圧縮不可フラグを変更します。
複数選択の場合、選択したノードを一括して変更できます。

Compress this node if possible にすると、圧縮機能が有効になっている場合に圧縮可能ならば圧縮します。これがデフォルトです。
Don’t compress this node にすると、圧縮機能が有効になっていても絶対に圧縮しません。

ウィンドウを閉じずに別のノードを選択すると、別のノードを設定することができます。

検索方法

Search 欄のチェックボックスを ON にして、Search ボタンをクリックすると、該当する transform ノード と joint ノードが選択されます。

参考：

複数のチェックボックスを ON にした場合は、いずれかの設定を満たすものを検索します。
