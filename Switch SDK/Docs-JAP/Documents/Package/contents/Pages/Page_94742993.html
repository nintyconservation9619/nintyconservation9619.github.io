ランダム幅の更新

ここではエミッタトランスフォームの「位置(±)」と「回転(±)」で設定した乱数値を放出毎に更新するかを決める機能「ランダム幅の更新」について説明します。

概要

「ランダム幅の更新」は、エミッタトランスフォーム「位置」、「回転」に乱数が設定されている場合、エミッタ放出毎に乱数値を変更できる機能です。
毎回パーティクルの放出状態をランダムにすることによって、エフェクト全体の見た目にばらつきをも持たせることができます。
以下は「 設定項目」の概要です。

ランダム幅の更新

「ランダム幅」の更新タイミングを「 チェックボックス」ON/OFF で切り替えます。
ON に設定することで、乱数値はパーティクルの放出毎に更新されます。
OFF に設定することで、エミッタをセットする時に一回だけ更新されます。

参考：

エミッタトランスフォーム「位置」、「回転」、「スケール」いずれか１箇所でもアニメーションが設定されている場合は機能しません。
