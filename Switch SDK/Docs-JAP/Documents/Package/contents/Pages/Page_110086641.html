カーブエディタの操作

ここではアニメーションの作成手順を説明します。

アニメーションの基本的な流れ

レイアウトエディタではキーフレームアニメーションという手法でアニメーションを作成します。
作成したレイアウトにキーフレームを設定してアニメーションを作成します。

ペインの位置アニメーションを作成

ペインが X 軸 （ 左右 ） に移動するアニメーションを作成します。

カーブエディタ

以下いずれかの操作をレイアウトエディタウインドウで行い、カーブエディタを開きます。

  • [  カーブエディタ ] をクリックします。
  • [ メニュー ] → [ ウインドウ ] → [ カーブエディタ ] を選択します。

以下はカーブエディタウィンドウです。

ペインの選択

レイアウトエディタウインドウからアニメーションさせる対象のペインを選択します。
カーブエディタの左側ツリーに項目が表示され、選択ペインがアニメーション編集対象となります。

開始キーフレームの作成

カーブエディタウインドウのアトリビュートツリーから [ ペイン ] → [ 位置 ] → [ X ] をクリックして選択状態にします。
以下いずれかの操作を行い開始キーフレームを作成します。

  •  [ メニュー ] → [  キー作成 ] をクリックして開始キーフレームを作成します。
  •  ショートカット [ S ] で開始キーフレームを作成します。

グラフ表示領域にキーフレーム ( 黒い点 ) が作成され、位置 X に[ キー設定済みマーク ] が表示されます。

参考：

表示されない場合は、表示領域がずれている可能性があります。ショートカット [ A ] で全体表示を行ってください。

カレントフレームの移動

以下いずれかの操作を行い、カレントフレームを 50 フレームに変更します。

  • タイムスライダーをクリックとドラッグします。
  • タイムスライダー横のテキストボックスに数値を入力します。

終了キーフレームの作成

以下いずれかの操作を行い終了キーフレームを作成します。

  •  [ メニュー ] → [  キー作成 ] をクリックして終了キーフレームを作成します。
  •  ショートカット [ S ] で終了キーフレームを作成します。

キーフレームの移動

終了キーを矩形選択します。
マウス中ボタンでドラッグしてキーフレームを移動します。

参考：

[ Shift ] を押しながらマウス中ボタンでドラッグを行うと、移動軸 （ 時間軸、値軸 ） を固定して移動できます。

最初にドラッグした方向で、移動軸が固定されます。

アニメーションの再生

再生コントロールの [  再生ボタン ] をクリックします。
アニメーション再生が開始され、ペインがカーブに沿って X 軸に移動するのを確認します。
アニメーションの再生を停止するには、再生コントロールの [  停止ボタン ] をクリックします。

スロープの編集

キーフレームのスロープを編集してアニメーションに変化を付けます。

キーフレームのスロープの編集

キーを矩形選択します。選択したキーが緑色の表示に変わります。
キーの左右に表示されている制御点の左右どちらかを矩形選択します。

スロープを選択すると黄緑色の表示に変わります 。
マウス中ボタンでドラッグを行い、スロープの形状を編集します。

このように、カーブの形を編集することでアニメーションを作成していきます。

ビューアで確認

以下いずれかの操作を行い、ビューアで確認します。

  • レイアウトエディターウインドウの [ メニュー ] → [ ツール ] → [ ビューアに転送 ] を選択します。
  • ショートカット [ F5 ] 

ビューアが起動し転送が開始されます。
転送が終わるとペインが表示されアニメーションを再生します。
