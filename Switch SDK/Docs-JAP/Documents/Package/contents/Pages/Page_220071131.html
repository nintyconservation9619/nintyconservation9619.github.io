フレームバッファ・デプスバッファの UV 原点位置の切り替え

参考：

以下の情報は、NX Addon 4.1 以降のテクスチャ原点位置の切り替え方法になります。

グラフィックス API の設定で、テクスチャ座標の原点を左上、左下にそれぞれ設定可能ですが、エフェクトのシェーダ内で、フレームバッファやデプスバッファからテクスチャ値を参照するための UV 座標の計算処理は、デフォルトでは左下に設定されています。テクスチャ座標の原点切り替えは、プロジェクト設定ファイル から行えます。

 テクスチャ座標の原点の切り替え方法

参考：

以下の情報は、NX Addon 4.0 以前のテクスチャ原点位置の切り替え方法になります。
NX Addon 4.1 以降も当面はサポートされますが、プロジェクト設定ファイル 内、<TextureOriginMode> で指定する方式への乗せ換えをお願いします。

本設定を切り替えるためには、プロジェクト設定ファイルの「追加のプリプロセッサ定義」に、

  • _TEXTURE_ORIGIN_LOWER_LEFT : UV 原点は左下
  • _TEXTURE_ORIGIN_UPPER_LEFT : UV 原点は左上

を設定することで明示的に指定可能です。具体的な記述例は以下の通りです。
