テクスチャフォーマットの選び方

エフェクトでおすすめのテクスチャフォーマットの選び方について説明します。

テクスチャフォーマットについては、 Photoshop の Vfx 用プリセット名を記載しています。
関連リンク Vfx 用プリセットの詳細については 「テクスチャフォーマットのプリセット」 を参照してください。


以下の選択フローを参考に、使用するテクスチャを選んでください。

以下はリニア環境下にあるときのテクスチャフォーマットの選択フローです。

 

以下はプリセット名とテクスチャフォーマットの対応表です。

関連リンク 以下は関連ページへのリンクです。

RGB を二乗して取得 テクスチャの輝度を二乗することにより、リニア変換時の結果に近似できる機能です。
