ウインドウ

レイアウトエディタウインドウのメニューバー「ウインドウ」の項目について説明します。

カーブエディタ 「カーブエディタウインドウ」を開きます。
テクスチャ 「テクスチャウインドウ」を開きます。
グループ 「グループウインドウ」を開きます。
親子階層 「親子階層ウインドウ」を開きます。
ペイン情報貼り付け制御 「ペイン情報貼り付け制御ウインドウ」を開きます。
アニメーション区間タグ編集 「アニメーション区間タグ編集ウインドウ」を開きます。
ペイン検索 「ペイン検索ウインドウ」を開きます。
すべてのプロパティウインドウ

「プロパティウインドウ」を開きます。

ペインを選択している場合は、そのペインのプロパティウインドウを開きます。
レイアウト設定 「レイアウト設定ウインドウ」を開きます。
色見本 「色見本ウインドウ」を開きます。
アニメーション共有情報 「アニメーション共有情報ウインドウ」を開きます。
部品・コントロール設定 「部品・コントロール設定ウインドウ」を開きます。
部品パレット 「部品パレットウインドウ」を開きます。
ドッキングを固定する ドッキングウインドウの状態をロックします。
レイアウトウインドウをタブ化しない レイアウトウインドウの複数表示をタブ化せず、個別のウインドウとして表示します。
ツールバーを隠す レイアウトエディタウインドウのツールバーを非表示にします。
