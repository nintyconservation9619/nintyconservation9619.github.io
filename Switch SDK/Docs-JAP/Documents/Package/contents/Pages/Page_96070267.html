回転順序

ここでは回転軸 XYZ のうち、２つ以上の軸を使用するさいに処理の順番を設定する機能「回転順序」について説明します。

概要

「回転順序」は、「 回転軸」の「 チェックボックス」 ２つ以上の軸の使用を ON にしたとき、回転をかける軸の順番を設定できる機能です。
3Dモデルと回転方法を合わせるこごができます。

以下は「 設定項目」の概要です。
「回転軸」で ２つ以上の軸の使用を ON にすると「 回転順序」が表示されます。以下はグループ内の設定項目です。

回転順序

選択した軸の回転順序を以下のプリセットから選択できます。

Y → Z → X パーティクルは軸 Y → Z → X の順で回転できます。
X → Y → Z パーティクルは軸 X → Y → Z の順で回転できます。
Z → X → Y パーティクルは軸 Z → X → Y の順で回転できます。
