Mii に使用されている色

ここでは、Mii に使用されている色について解説します。

Mii の色は、こちらの ドキュメントにまとまっています。

シート 説明
CommonColor 毛、目、メガネ、口に使われる共通色の一覧表です。
CommonColorOrder Mii作成・編集アプレット の 編集画面のすべてのカラーで表示される色の並び順です。
SkinColor 肌に使われる色の一覧表です。
SkinColorOrder Mii作成・編集アプレット の 編集画面で表示される顔のカラーの並び順です。
FavoriteColor 帽子、体(体つきアイコン、体格反映体つきアイコン)に使われる色の一覧表です。
参考：

色の RGB 値は、sRGB ガンマ空間での値(8bit)です。

また、CommonColor シートの UpperLipColor 列の色は、上唇のあるパーツを使ったときに利用されます。
