カラーコンバイナのチャンネル

ここではカラーコンバイナのチャンネルについて説明します。

概要

カラーコンバイナの「チャンネル」は、登録したテクスチャのどの成分を「計算式」への入力値として適用するかを決める機能です。
テクスチャの使い方に幅を持たせることができる為、複数のエミッタ間で同じテクスチャを使い回す場合に便利です。

以下は「チャンネル」の項目ごとのイメージと項目名、項目ごとの説明です。

RGB 登録したテクスチャのカラー値が、入力値として適用されます。
1.0 - RGB 定数 1.0 からテクスチャのカラー値を減算したカラーが入力値として適用されます。
A 登録したテクスチャの アルファチャンネルの値です。
1.0 - A 定数 1.0 からテクスチャのアルファ値を減算したカラーが入力値として適用されます。
1.0 定数1.0 が入力値として適用されます。
