マッピング設定

ここではペインに対するテクスチャ画像の貼り方を設定できる機能「マッピング設定」について説明します。

概要

「 テクスチャ」に登録したテクスチャ画像をペインに適用するさいの貼り方を「マッピング方法」から選択できます。
以下は「マッピング方法」のプリセットです。

UVマッピング（UV座標0を使用） ペインのテクスチャ座標（UV 座標）に沿ってテクスチャ画像を適用できるマッピング方法です。
テクスチャ画像は常にペインの位置、サイズ、向きの変化に影響します。

正射投影マッピング

ペインに対してテクスチャ画像を投影するマッピング方法です。
ペイン位置相対の投影マッピング ペインの位置を変更したさいに移動した位置を基準にテクスチャ画像を投影する 「正射投影マッピング」です。
透視投影マッピング

ペインに対してテクスチャ画像を投影するマッピング方法です。

レイアウトのカメラが「 パースペクティブ投影」のさい、投影するテクスチャ画像がスクリーン空間で投影される為、パースを無効にできます。
ペイン位置相対の透視投影マッピング ペインの位置を変更したさいに移動した位置を基準にテクスチャ画像を投影する「透視投影マッピング」です。

使用例

以下は投影マッピングの種類と設定内容を変えたときの比較画像です。
レイアウトのカメラは「 正射影投影」です。

A 正射投影マッピング

「 テクスチャ画像」はレイアウト画面の中心を基準に投影されています。

レイアウト全体に投影 ON

「 テクスチャ画像」はレイアウト画面全体に引き延ばされて投影されています。
B ペイン位置相対の投影マッピング

「 テクスチャ画像」はペインの中心を基準に投影されています。
また、ペインの移動に追従しています。
C ペイン位置相対の投影マッピング

 ペインの回転・スケールを考慮して投影 ON

「 テクスチャ画像」はペインの回転に 追従して投影されています。

以下はレイアウトのカメラが「 パースペクティブ投影」のさい、「正射投影マッピング」と「透視投影マッピング」の比較画像です。

A 正射投影マッピング

回転によりペインにパースがかかる為、「 テクスチャ画像」が歪んで見えます。
B 透視投影マッピング

「 テクスチャ画像」はスクリーン空間で投影される為、パースによる歪みが発生しません。
