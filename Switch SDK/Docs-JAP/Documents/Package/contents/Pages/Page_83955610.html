エミッタセットファイル

エフェクトデータ用中間ファイル「エミッタセットファイル」について説明します。

エミッタセットファイルの保存内容

エミッタセットはエミッタセットファイル（ .eset）として保存し、 1 つのエフェクトデータとして扱われます。
プログラマはエミッタセットの単位で描画位置やカラーなどを操作します。
エミッタセットファイルは以下の情報を保持します。

  • エミッタセットの設定内容
  • テクスチャファイルの名前
  • プリミティブファイルの名前

エミッタセットファイルを開く

エミッタセットファイル（ .eset）を開く時、関連するファイルと一緒に読み込まれます。
この時、 関連するファイルが見つからない場合はダイアログが表示され、そのテクスチャファイルは読み込まれません。

以下は関連するファイルの種類です。

テクスチャファイル エミッタセットファイルで利用する画像ファイルです。
プリミティブファイル エミッタセットファイルで利用するプリミティブ用モデルファイルです。
