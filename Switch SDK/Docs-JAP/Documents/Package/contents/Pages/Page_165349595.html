コントローラーのファームウェアアップデート

  • 概要
  • 必要なツール
  • 手順書

概要

本ページではコントローラーのファームウェアを更新する方法を案内します。
コントローラーファームウェアの変更履歴については、コントローラーファームウェアのリリースノート を参照してください。 

注意：

SDEV/EDEV を利用して Joy-Con のファームウェアを更新する際は、Joy-Con を US Writer に接続しないでください。
Joy-Con を US Writer に接続したままファームウェアを更新すると、更新に失敗する場合があります。



注意：

NX Addon 3.1.0 より、本体機能のコントローラーサポートアプレットを使用してファームウェアを更新できるようになります。

ただし、出荷状態の試作品 (SDEV MP 付属の黒い Joy-Con) は、コントローラーサポートアプレットによるファームウェア更新ができません。
これらの機材を使用する際は、必ず最初に本ページの手順に従ってファームウェアを更新してください。 

量産品の Joy-Con 及び Switch Pro コントローラーに関しては、上記の制限はありません。


必要なツール

  • コントローラーファームウェアアップデータ

    • NintendoSDK フォルダ直下の UpdateControllerFirmware.bat

    • NX のコントローラーのファームウェアを更新するためのツールです。 

手順書

  1. コントローラーファームウェアアップデータ の起動
    NintendoSDK フォルダ直下にある UpdateControllerFirmware.bat を実行します。

  2. コントローラーを SDEV/EDEV に接続します。
    コントローラーのシンクロボタンを押して、本体にコントローラーを Bluetooth 接続するか、本体のレール部分に接続してください。
    接続されたコントローラーは Connected controllers にリストアップされます。

    注意：
    本体のレール部分に接続してファームウェアを更新するには、Joy-Con のファームウェアが 3.46.x.x 以降に更新されている必要があります。
    ファームウェアが 3.46.x.x より古い場合は、コントローラーを Bluetooth で接続して更新してください。
    コントローラーの電池が少ない場合は、あらかじめ本体のレール部または通電式の拡張グリップ等を利用して充電を行ってください。

  3. コントローラーのLボタン/ZLボタン または Rボタン/ZRボタンを同時押しし、 離します。
     ファームウェア更新はボタンを離すと開始します。


  4. ファームウェア更新が行われます。
    ファームウェア更新中は、Bluetooth 接続しているコントローラーをレール部分に接続したり、レール接続中のコントローラーをレールから外したりしないでください。

    NFC に対応したコントローラーの更新中には、一度コントローラーの再起動が行われます。
    コントローラーの再起動が行われると自動的に本体への再接続が行われますが、電波状況などにより稀に失敗することがあります。
    再接続に失敗した場合は、コントローラーのボタンを押して手動で再接続を行ってください。
    再接続後更新が再開されます。 


  5. ファームウェア更新が完了すると、画面に "Firmware update completed" と表示されます。 
    更新が完了したコントローラーは、SDEV/EDEV から切断されます。 

    注意：

    電波状況などにより、"verify error occurred." と表示され、アップデートに失敗することがあります。

    アップデートに失敗した場合は、再度アップデートを行ってください。

  6. コントローラーのファームウェア更新は連続して行うことができます。別のコントローラーを更新する場合は、手順 2. から繰り返してください


  7. ファームウェア更新が完了した後は、一度 SDEV/EDEV の再起動を行ってください。
    (将来のSDKで再起動が不要になる予定です)
