プレビューバンク

プレビューバンクとは？

プレビューバンクは、MIDI シーケンサーから入力された MIDI 信号を、どのバンクで再生するのかを定めるペインです。
このペイン（ウインドウ）に登録されたバンクを使用して MIDI 再生します。

 

プレビューバンクの操作

プレビューバンクを開く

[表示]→[プレビューバンク] または、ツールバーの「プレビューバンクの表示」ボタンにより、プレビューバンクを開くことができます。
これらの操作は、ウインドウの表示・非表示のトグル動作になります。

 

 

プレビューバンクは、メインウインドウ下部のペインにドッキングウインドウとして現れます。

ウインドウ下部にある "プレビューバンク" タブ部分またはタイトルバーのコンテキストメニューから、プレビューバンクウインドウを [フローティング]、[ドッキング]、[非表示] に変えることができます。
タイトルバー部分のダブルクリックでも [フローティング]、[ドッキング] の切り替えが行えます。

また、ドッキングウインドウ時は、タイトルバー右側の最小化アイコンをクリックするか、スプリッタ (境界線) をダブルクリックすることにより、タブのみの最小化表示になります。

プレビューバンクの操作

プレビューバンクのリストにバンクを登録または入れ替えを行うには、以下の方法があります。

  • サウンドセットタブのシーケンスサウンドリストまたはバンクリストから選択状態のインデックスセルをクリックし、プレビューバンクのリスト内にドラッグ＆ドロップすることにより、バンクの登録ができます。
  • サウンドセットタブのシーケンスサウンドリストまたはバンクリストから、右クリックによるコンテキストメニューを開き、[プレビューバンクに登録] でバンクの登録ができます。
    この場合は、バンクはプレビューバンクの 0 番に固定で登録されます。
  • 開いているバンクタブのタブ上で右クリックによるコンテキストメニューを開き、[プレビューバンクに登録] でバンクの登録ができます。
    この場合は、バンクはプレビューバンクの 0 番に固定で登録されます。  
  • プロジェクトツリーのバンクノードを展開した子ノード上から、右クリックによるコンテキストメニューを開き、[プレビューバンクに登録] でバンクの登録ができます。
    この場合は、バンクはプレビューバンクの 0 番に固定で登録されます。  

シーケンスサウンドリストからバンクを登録すると、シーケンスサウンドに紐づけられたバンクが登録されます。
プレビューバンクの 0 〜 3 の数字は、シーケンスサウンドのマルチバンク設定における 0 〜 3 に相当します。

 

プレビューバンクに登録したアイテムを削除するには、削除したいアイテムを選択し、メインメニューの [表示]→[削除] を実行、または Delete, Ctrl+D キーを押下します。
