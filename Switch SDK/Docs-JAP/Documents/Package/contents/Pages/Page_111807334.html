 表示切り替え

プルダウンにより、テクスチャウインドウのリスト表示を切り替えます。

  • 詳細表示
  • 画像表示
I - テクスチャファイルビュー 選択したテクスチャデータのグラフィックを表示します。
J - 情報バー 選択したテクスチャのパスを表示します。
詳細表示の各項目
名前 テクスチャの名前を表示します。
フォーマット

テクスチャのフォーマットを表示します。

参考：

 各テクスチャのフォーマット項目をクリックすることで、選択候補ドロップダウンからフォーマットを指定することができます。
幅 テクスチャの幅 ( X 方向 ) を表示します。
高さ テクスチャの高さ ( Y 方向 ) を表示します。
Byte テクスチャのデータサイズを表示します。
実機サイズ

実機で利用されるテクスチャ形式に変換した後の、テクスチャのデータサイズを表示します。

参考：

同じ幅、高さのテクスチャでも、フォーマットを変えるとデータサイズが変化します。
ファイルパス テクスチャのパスを表示します。

TGA画像ファイルに関する制限事項

TGA画像ファイルは 通常の TGAファイルを使用します。
アルファ情報を利用する画像は 32 ビットフォーマット、アルファ情報を利用しない画像には 24 ビットフォーマットで TGA ファイルを作成してください。

テクスチャフォーマット指定

LayoutEditor上でテクスチャフォーマットを指定することで、ゲーム内での、データサイズや表示品質を制御します。
テクスチャフォーマットを指定するには...

  1. リスト表示を画像表示から詳細表示に切り替えます
  2. リストから、テクスチャを選択します
  3. 選択テクスチャのフォーマットをクリックし、選択候補ドロップダウンからフォーマットを指定します。
