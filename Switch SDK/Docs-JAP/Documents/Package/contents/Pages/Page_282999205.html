テキストボックスペイン

ここでは登録したフォントをレイアウト素材として利用し、文字列を扱うことができる「テキストボックスペイン」について説明します。

概要

「テキストボックスペイン」は、登録したフォントをレイアウト素材とし、レイアウトに文字列を配置できる機能です。
テキストボックスペインでは主に以下の設定ができます。

  • フォントファイルの登録
  • 文字列の入力
  • フォントサイズ、文字間隔、傾きの設定
  • テキストカラーの設定

以下は「テキストボックスペイン」の設定項目です。

ペイン基本 ペインの名前、位置、形状、中心位置などを設定します。
テキストボックスペイン ペインで使用するフォント、画面に表示する文字列などを設定する「テキストボックスペイン」の UI について説明します。
カラー補間設定 文字列のカラーとアルファに影響を与える入力値を設定します。
フラグメントオペレーション フラグメントの計算方法をプリセットから設定したり、プリセットの値を変更してカスタムフラグメントを設定します。
拡張ユーザー情報 ペインにユーザー独自の拡張情報を設定します。

関連リンク 以下は関連ページへのリンクです。

テキストボックスペインを利用する 文字列をレイアウトに配置する手順を説明します。

特殊フチ付き形式フォント

ビットマップフォント(Packed4 形式)に対して文字の輪郭に「縁」を付ける機能「特殊フチ付き形式フォント」について説明します。
複合フォント 複数のフォントを合わせて一つのフォントとして扱う「複合フォント」について説明します。

スケーラブルフォント

1つのフォントファイルを指定したサイズで描画できる「スケーラブルフォント」について説明します。
サンプル一覧「ペインを組合わせたボタン」 各種ペインを組合わせたボタンのサンプルです。ペインの基本的な使い方が理解できます。
サンプル一覧「テキストボックスペイン」 テキストボックスペインの機能説明用サンプルです。
