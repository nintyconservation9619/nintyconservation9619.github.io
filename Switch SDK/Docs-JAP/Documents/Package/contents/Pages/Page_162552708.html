  • render_info、shader_option、shader_param
    下表の Mii パラメータは render_info や shader_option、shader_param などを利用してマテリアルに反映させて下さい。

    Mii パラメータ 説明
    カリング用 int 値 nn::mii::DrawParam::GetCullMode() で取得可能な値です。
    半透明判別用 int 値 nn::mii::CharModel::DrawType から決まる値です。nn::mii::DrawType_XluMin 以上のパーツが半透明です。
    定数カラー

    nn::mii::DrawParam::GetConstantColor() で取得する値を設定します。

    float3 が nn::mii::DrawParam::ConstantColorNum 個必要です。
    モジュレーションタイプ nn::mii::DrawParam::GetModulateType() で取得する値を設定します。

  • sampler

vec4 GetAlbedo()
{
    vec4 texel = texture(s_Tex,In.texCoord);
    vec4 albedo;
    switch(modulate)
    {
    case MODULATE_TYPE_CONSTANT:
        albedo = vec4(constColor[0],1.0f);
        break;
    case MODULATE_TYPE_TEXTRUE:
        albedo = texel;
        break;
    case MODULATE_TYPE_LAYERED:
        albedo = vec4(constColor[0] * texel.r
            + constColor[1] * texel.g
            + constColor[2] * texel.b
            , texel.a);
        break;
    case MODULATE_TYPE_ALPHA:
        albedo = vec4(constColor[0] * texel.r ,texel.r);
        break;
    case MODULATE_TYPE_LUMINANCE_ALPHA:
        albedo = vec4(constColor[0] * texel.r ,texel.g);
        break;
    case MODULATE_TYPE_ALPHA_OPA:
        albedo = vec4(constColor[0] * texel.r ,1.0f);
        break;
    default:
        albedo = vec4(1.0f);
        break;
    }
    return albedo;
}

void main()
{
    /// ModulateTypeを考慮してアルベドを取得
    vec4 albedo = GetAlbedo();
    if(albedo.a == 0.0f)
    {
        discard;
    }
    o_Color = albedo;
}

  • shape

  • name
    "[bone_name]__[mat_name]" の命名規則で名前を付けます。シェイプ名をユーザが任意に決める事は他ツールとの連携の都合上お勧めしません。
  • mat_name
    shape に対応する material の名前です。nn::mii::CreateModelType、nn::mii::CreateNoseType が異なる shape でも material は共通です。

  • bone_name
    shape に対応する bone の名前です。

  • mesh

    • count
      ダミーで stream を入れる必要があるので適切な count を設定して下さい。
      サンプルではダミーにポリゴンを 1 枚入れているので count="3" と記述しています。

    • stream_index
      ダミーで頂点インデックス用 stream を用意する必要があります。ここでは、stream_index="5" の stream がダミー頂点インデックスです。

  • bone
    parent bone を 1 つ用意し、それ以外は nn::mii::CreateModelType、nn::mii::CreateNoseType に対応する bone を用意します。

  • visibility
    nn::g3d を用いた Mii の描画では bone の visibility を使用して、nn::mii::CreateModelType や nn::mii::CreateNoseType の切り替えを行います。

    e.g.1. 通常のおでことハット用のおでこの切り替え
    e.g.2. 通常の鼻と高さ調整された鼻の切り替え

    ランタイムでユーザが表示させたい nn::mii::CreateModelType や nn::mii::CreateNoseType ( に対応する bone ) の visibility を設定してください。
