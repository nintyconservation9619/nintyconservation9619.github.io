「チェッカー」タブの概要

「チェッカー」タブには、数十の独立した静的チェッカーおよび動的チェッカーが含まれます。静的チェッカーはアセンブリコードおよび記号を分析し、動的チェッカーは取得したプロファイルデータを処理します。チェッカーはプロファイルが取得されると自動的に実行されます。トリガしたチェッカーのみが表示されます。

「チェッカー管理」切り替えボタン

「チェッカー管理」切り替えボタンを有効にすると、すべてのチェッカーを表示できます。「チェッカー管理」切り替えボタンを無効にすると、無効にしたトリガやトリガしなかったトリガはすべて非表示になります。「チェッカー管理」切り替えボタンを有効にすると、すべてのチェッカーが表示され、チェッカー内のパラメータを調整したり、無効になっているチェッカーを再度有効にしたりできます。

チェッカー

チェッカーは、見出しをクリックすることによって、展開したり非表示にしたりできます。

以下で説明するように、すべてのチェッカーが同様の形式になっています。

「上位」ボタン

  • 「チェッカー再実行」：チェッカーを再実行します。チェッカーのパラメータを変更した場合には、チェッカーの再実行が必要になることがあります。
  • 「テキストのコピー」：「概要」、「説明」、および「修正案」の文字列をクリップボードにコピーします。
  • 「表のコピー」：生成される表の内容をクリップボードにコピーします。
  • 「チェッカー無効」：現在のプロファイルのチェッカーを無効にし、今後のすべてのプロファイルで実行されないようにします。チェッカーを再度有効にするには、「チェッカー管理」を有効にし、見出しの右側に「チェッカー無効化」というマークが付いている無効化されたチェッカーを見つけて、右上の「チェッカーの有効化」ボタンをクリックします。

概要

チェッカーの結果の概要を表示します。この文字列をクリップボードにコピーするには、「テキストのコピー」ボタンをクリックします。

チェッカーのパラメータ

「概要」の下、「Results Table（結果の表）」の上にあるスピナーコントロールでチェッカーのパラメータを変更できます。これらのパラメータを変更すると、新しい値が永続的に保存されます。デフォルト設定に戻すには、「デフォルトにリセット」ボタンをクリックします。

新しいパラメータ値を使用してチェッカーを再実行するには、「チェッカー再実行」ボタンをクリックします。

結果の表

「Results Table（結果の表）」には、チェッカーの実行結果がすべて表示されます。列見出しをクリックして表を再ソートできます。表の結果をクリップボードにコピーするには、「表のコピー」をクリックします。

行を右クリックすると、そのデータまたは関数に適用できる追加オプションが表示されます。

説明

「説明」の文字列は、チェッカーのテスト項目およびテストを行う理由について、概要を示しています。この文字列をクリップボードにコピーするには、「テキストのコピー」ボタンをクリックします。

修正案

「修正案」の文字列は、次に実行する手順についての具体的な提案を示しています。この文字列をクリップボードにコピーするには、「テキストのコピー」ボタンをクリックします。
