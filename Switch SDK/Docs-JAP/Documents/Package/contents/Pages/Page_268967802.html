部品レイアウトの利用

ここでは「部品レイアウト」を配置先「ボディレイアウト」に読み込み、配置する手順について説明します。
以下は「部品レイアウト」と配置先の「ボディレイアウト」です。

部品レイアウト ボディレイアウト

「部品レイアウトの作成」 で作成したセレクトボタンです。 「ボディレイアウトの作成」 で作成したメニュー画面背景です。

レイアウト作成の準備

あらかじめ LayoutEditor を起動した状態から、レイアウトデータ作成にあたっての準備を行います。

関連リンク LayoutEditor の起動方法については 「作業の流れ」 を参照してください。

ウインドウを開く

以下のウインドウが開いた状態から作業を開始します。

  • 親子階層
  • プロパティウインドウ
  • 部品パレット

関連リンク 各ウインドウの UI については 「ウインドウ」 を参照してください。

部品パレットの更新

チュートリアルを通して行っている場合、これまでに作成したレイアウトが「部品パレット」ウインドウに表示されていません。
「部品パレット」ウインドウの「 更新」ボタンを押し、部品パレットを最新の状態に更新することで作成したレイアウトが表示されます。

関連リンク 「部品パレットウィンドウ」についてはこちらを参照してください。

ボディレイアウトを開く

「部品パレット」から「ボディレイアウト」を開きます。

「部品パレット」リストの「Menu.flyt」でマウス右ボタンクリックで開くコンテキストメニューから「開く」を選択します。
LayoutEditor に「Menu.flyt」が読み込まれます。

部品の登録

「部品パレット」ウインドウから部品レイアウトを「ボディレイアウト」に読み込み「 部品ペイン」として配置します。

「部品パレット」ウインドウで「Menu_Bar.flyt」をダブルクリックします。
「親子階層」ウインドウに「 L_parts_00」が追加されます。

部品ペインの移動

「親子階層」ウインドウで「L_parts_00」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

 位置 X,Y = -275,260

「ヌルペイン」の作成

登録した「 部品ペイン」を階層下に配置し、アニメーションさせる為の「 ヌルペイン」を作成します。

メインメニュー[ ペイン ] → [ ペイン作成 ] → [「ヌルペイン」 ] を選択します。
「親子階層」ウインドウに作成された「 ヌルペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

 ペイン名 N_InOut

ペインの親子階層化

「親子階層」ウインドウで、作成した「 N_InOut」の階層下に「 L_parts_00」を配置します。

部品ペインの複製

以下の手順で「 部品ペイン」を複製します。

1

「親子階層」ウインドウで「 L_parts_00」を選択します。
2

メインメニュー[ 編集 ] → [ コピー ] で「 L_parts_00」をコピーします。
3

「親子階層」ウインドウで「 N_InOut」を選択します。
4

メインメニュー[ 編集 ] → [ 貼り付け ] で「L_parts_00」を複製します。
「親子階層」ウインドウで「 L_parts_01」が追加されているを確認します。

複製した部品ペインの移動

「レイアウトウインドウ」上で「 L_parts_01」を移動させます。
「 L_parts_01」を選択し、レイアウトウインドウで配置を確認しながら「↓キー」を押します。

「部品ペインの複製」を繰り返し、複製したペイン名末尾に 02 ～ 07 をつけます。
部品ペインの位置 Y の間隔を 70 ピクセルにし、複製したペインを等間隔に配置します。
最終的に、各ペインの配置は以下表のようになります。

 ペイン名 位置X 位置Y

L_parts_00

 -275 260
L_parts_01 -275 190
L_parts_02 -275 120
L_parts_03 -275 50
L_parts_04 -275 -20
L_parts_05 -275 -90
L_parts_06 -275 -160
L_parts_07 -275 -230

レイアウトデータの保存

編集内容を保存します。
メインメニュー[ ファイル ] → [ 保存 ] で「Menu.flyt」に編集内容を保存します。
