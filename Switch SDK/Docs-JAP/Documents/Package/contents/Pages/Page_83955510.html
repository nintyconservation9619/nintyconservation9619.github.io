カラーアニメーション

カラーアニメーションは、マテリアルノードのカラー（Color）、透明度（Transparency）、アンビエントカラー（Ambient Color）、白熱光（Incandescence）、スペキュラカラー（Specular Color）をアニメーションさせて設定します。

  • カラー、白熱光、スペキュラカラーアトリビュートにfile ノードが接続されている場合は file ノードのカラーゲイン（Color Gain）アトリビュートに設定したアニメーションが中間ファイルに反映されます。
  • カラー、白熱光、スペキュラカラーアトリビュートに layeredTexture ノードが接続されている場合は、最上位レイヤに接続された file ノードのカラーゲインアトリビュートに設定したアニメーションが中間ファイルに反映されます。
  • 透明度アトリビュートに file ノードが接続されている場合は file ノードのアルファ値のゲイン（Alpha Gain）アトリビュートに設定したアニメーションが中間ファイルに反映されます。

カラーアニメーションはアニメーションを設定したアトリビュートについてのみデータが出力されます。
データが出力されなかったアトリビュートは、fmd ファイルの状態で表示されます。よって、アトリビュートが一定値でも fmd ファイルの状態と異なる場合は、アトリビュートにキーを 1 つ以上打ってからカラーアニメーションを出力してください。

エクスポートオプションのマテリアル圧縮が有効になっていると、マテリアルがまとめられた結果、カラーアニメーションが設定されていないマテリアルまでアニメーションしてしまう場合があります。
その場合はマテリアルアトリビュート設定プラグインで、カラーアニメーションを設定したマテリアルが圧縮されないようにしてください。
