範囲内ループ

「範囲内ループ」ノードの機能について説明します。

概要

「範囲内ループ」は、パーティクルを仮想的な「箱」の中に収める機能です。
以下図のように、パーティクルの表示を、エディタで指定した「箱」の範囲内に納めます。
パーティクルはその箱の範囲外に出たら、箱の反対側に戻り、繰り返し表示されます。

パラメータ

「 パラメータ」は、～を設定する機能群です。

「箱」の端でパーティクルのアルファ値を変化させる機能について説明します。

クリッピング設定／クリッピング平面の高さ 「箱」の内側でパーティクルの表示をクリッピングする機能について説明します。

設定例

以下は「範囲内ループ」の設定例です。

A 「箱」の位置を固定した使い方

パーティクルの表示を「箱」の範囲内納めます。

例えば広大なマップの中で部分的に雨を降らせたいといった表現に使用できます。
B 「箱」をカメラ前に固定した使い方

カメラを前後に移動させた場合でも、「箱」とカメラの距離は維持した状態で表示されます。

例えばカメラがキャラクタに追従するシーンで、移動するキャラクタの周りにのみ雨を降らせたいといった表現に使用できます。
C 「箱」の範囲内で再利用する使い方 少ないパーティクルを箱の範囲内で繰り返し描画することで、CPUの計算負荷を軽減できます。
