タグ追加ウインドウ

「タグ追加ウインドウ」ではアニメーション区間タグの追加を行います。

タグ追加ウインドウ
A 名前
 アニメーション区間タグの名前を設定します。
B コメント

アニメーション区間タグのコメントを設定します。
C 区間

区間の開始フレームと終了フレームを設定します。
D 長さ

区間の長さを設定します。
