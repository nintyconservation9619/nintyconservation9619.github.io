検索機能

検索機能

検索ウインドウや検索タブを使い、文字列や特殊な条件からサウンドプロジェクト内のアイテムを検索することや、リスト上から参照の検索をすることができます。

検索ウインドウを使った検索

[編集]→[検索] で検索ウインドウを開きます。
この操作は Ctrl + F のショートカットキーでも行うことができます。

 

検索ウインドウの使い方

種別タブ

検索対象をサウンドセットアイテムとインストルメントに分けて行います。
検索する種別のタブを選択して下さい。

検索範囲

検索する範囲をプルダウンリストから選択します。

プロジェクト全体

プロジェクト全体を対象とします。

指定サウンドセット

検索ウィンドウで、アイテムタブを選択時に「指定サウンドセット」が選択可能になります。
「指定サウンドセット」を設定すると「サウンドセットの選択」ボタンがアクティブになり、サウンドセットの選択ダイアログを開くことができます。
サウンドセットの選択ダイアログでは、プロジェクトに登録されているサウンドセット名の一覧が表示されるので、検索対象にしたいサウンドセットにチェックを入れます。

現在のバンク

検索ウィンドウで、インストルメントタブを選択時にバンクタブがアクティブの場合、「現在のバンク」が選択可能になります。

現在のリスト

アクティブタブにて表示中のリストを対象とします。
アクティブタブが無い場合は、項目が現れません。

検索対象

「検索する文字列」に指定した文字列を、検索対象として含むかどうかをチェックボックスにより指定します。

指定できる項目は、以下の項目です。

  • 名前
  • ファイルパス
  • コメント
  • アイテム参照
  • 再生開始位置（シーケンスサウンドの再生開始位置）
参考：

・ストリームサウンドトラックには [名前] がないので、名前検索を指定してもストリームサウンドトラックは検索されません。
・「アイテム参照」にチェックを入れると、プレイヤー (バンク) を使っているアイテムについて検索できます。

オプション

「検索文字列にワイルドカードを使用する」にチェックを入れることにより、ワイルドカード "*" の使用が可能となります。

  • チェックなしの場合

"*" は文字列として扱われて、部分一致検索となります。

  • チェックありの場合

"*" はワイルドカードとして扱われて、完全一致検索となります。

「検索範囲の終わりに達したら最初から検索する」にチェックを入れると、検索ウインドウでの [前を検索]、[次を検索] によって検索範囲の終わりに達したら、ループして最初から検索されるようになります。

検索ボタン

次を検索（前を検索）

アクティブタブの選択位置から、条件に一致する次の (前の) アイテムを検索し、サウンドセットタブまたはバンクタブで、アイテムの場所を選択表示します。
このボタンを押しても検索ウインドウは閉じません。

ショートカットキー

検索には、ショートカットキーが設定されています。

次を検索： F3 （[検索する文字列] のテキストボックスにフォーカスがある場合は Enter でも可能）

前を検索： Shift + F3 （[検索する文字列] のテキストボックスにフォーカスがある場合は Shift + Enter でも可能）

参照検索

あるプレイヤー (バンク) を使っているアイテムを検索するには、プレイヤーリスト (バンクリスト) のコンテキストメニューから「すべての参照を検索」を選択して実行します。

対象アイテムを参照しているアイテムをプロジェクト全体から検索し、検索タブに一覧表示します。

参照されないアイテムについては、コンバート時に「コンバートログウインドウ」において警告表示され検出できます。
