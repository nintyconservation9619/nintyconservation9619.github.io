5 テクスチャー SRT アニメーション

テクスチャーの移動アニメーションを作成します。

テクスチャー SRT

「フレームの移動」にアニメーションキーを作成します。

マテリアル「TextureSRTMt」の Color アトリビュートで使用している place2dTexture ノードをアトリビュートエディタで開き、2Dテクスチャー配置アトリビュートの「フレームの移動」にアニメーションキーを作成します。

下表のようにキーを作成します。

 
0 フレーム
50 フレーム
フレームの移動 0.000 0.000 1.000 0.000

下図のようにテクスチャーが横移動するアニメーションになることを確認します。

出力対象

テクスチャー SRT アニメーションの出力対象は2Dテクスチャ配置アトリビュートの「フレームの移動」「フレームの回転」「繰り返し UV」のキーフレームです。

「Nintendo Export Options」の「Output File Selection」の C「Texture SRT Animation Data」に対応しています。

関連リンク 詳しくは テクスチャー SRT アニメーション を参照してください。

カラー、テクスチャーSRT、テクスチャーパターン、の各アニメーションを .fmab 形式で一つにまとめて出力します。
テクスチャー SRT アニメーションは単体ファイル fts で出力することが出来ます。
