エミッタトランスフォーム「回転(±)」

ここではエミッタ配置時の角度を設定する機能「回転(±)」について説明します。

概要

「回転(±)」は、エミッタ配置時の角度を設定できる機能です。
以下は「 設定項目」の概要です。

回転

エミッタ配置時の基準となる角度をエミッタ座標系 X Y Z で設定します。
「 ランダム設定」 では配置時の角度をランダムにする範囲を設定します。

 

参考：
関連リンク 毎回パーティクルの放出状態をランダムにする場合は 「ランダム幅を更新」 を参照してください。

設定例

以下はエミッタトランスフォーム「回転(±)」の値を± 各々の方向に対してアニメーションさせたときの比較画像です。

A Y 軸 - 方向

負の方向に設定した場合は時計回りに回転します。
B Y 軸 + 方向

正方向に設定した場合は反時計回りに回転します。


以下の画像は、基準となる角度を 45度 、「 ランダム設定」でランダム範囲を 60度 に設定しています。
