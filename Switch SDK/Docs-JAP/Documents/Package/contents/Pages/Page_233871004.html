デバッグを目的としたコントローラーファームウェアの巻き戻し

  • 概要
  • 必要なツール
  • 手順書

概要

本ページでは、コントローラーファームウェア更新を必要とする機能のデバッグに際して、コントローラーファームウェアを古いバージョンに巻き戻す方法を説明します。

NX Addon 3.1.0 以降では、IRSENSOR ライブラリの新機能を使用する際、コントローラーファームウェアの更新が必要になる場合があります。
コントローラーのファームウェア更新を行うためのコントローラーサポートアプレットは、アプリの動作中に IRSENSOR ライブラリから呼び出されます。 
コントローラーサポートアプレットが呼び出される際の挙動を確認するために、本ページで説明する方法を用いてコントローラーのファームウェアを巻き戻すことで、コントローラーをファームウェア更新が必要な状態にできます。 

IRSENSOR ライブラリ使用時のファームウェア更新に関しては、IRSENSOR ライブラリを参照してください。

参考：

コントローラーファームウェアの巻き戻し機能は NX Addon 3.2.0 から対応しています。

必要なツール

  • コントローラーファームウェアアップデータ (デバッグモード)

    • NintendoSDK フォルダの TargetTools/NX-NXFP2-a64/ControllerFirmwareUpdater/Release/UpdateControllerFirmwareForDebug.bat

    • NX のコントローラーのファームウェアを古いバージョンに巻き戻すためのツールです

手順書

  1. コントローラーファームウェアアップデートツール の起動
    NintendoSDK フォルダの TargetTools/NX-NXFP2-a64/ControllerFirmwareUpdater/Release 下にある UpdateControllerFirmwareForDebug.bat を実行します。

     

  2. ツールが起動した後の操作方法は、コントローラーのファームウェアアップデートと同様です。

ControllerFirmwareUpdaterForDebug.bat を使用してファームウェアを巻き戻すと、ファームウェアは NX Addon 1.x.x 以前と同じ下記のバージョンになります。

コントローラー ファームウェアバージョン
Joy-Con (L) 03.48.00.00
Joy-Con (R) 03.48.03.0b
Switch Pro コントローラー 03.48.03.05
