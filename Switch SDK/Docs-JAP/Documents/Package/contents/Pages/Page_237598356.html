ルート CA 証明書

概要

Web アプレットで利用可能なルート CA 証明書には大きく分けて以下の 2 種類が存在します。

  • SDK の SSL ライブラリから読み込んだ証明
    • こちらの詳細につきましては「機能 > ネットワーク > SSL ライブラリ > ルート CA 証明書」を参照して下さい。
  • Web アプレットで追加で読み込んだ証明書
