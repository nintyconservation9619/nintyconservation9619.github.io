ディレクショナルポリゴン

ここではパーティクルタイプ「ディレクショナルポリゴン」について説明します。

概要

パーティクルタイプ「ディレクショナルポリゴン」は、パーティクルの移動する直前のフレームとの位置を見てその移動方向へ傾けて表示します。
ディレクショナルビルボードの様にパーティクルの回転方向がカメラ方向に影響されない為、エフェクトを板としてさせることができます。

「ディレクショナルポリゴン」使用時の注意・です。
「指定方向初速」が 0 以外の設定値で且つ「指定方向」が XYZ = 0,1,0 の場合、表示面の向きが定まらない為、パーティクルが表示されません。
この場合、「指定方向拡散角度」を0.0001 に設定してください。Y 軸から離れる方向に少量の速度を与えることでパーティクルが表示されます。

設定例

以下は「ディレクショナルポリゴン」の設定例です。
エフェクト全体の形状をサイコロ状にできます。

エミッタ形状
「球（等分割）」
分割数 = 6
パーティクル形状
「パーティクル」

回転軸 = X軸
初期回転(±) = 90
A エミッタ形状
 エミッタは球体を 6 等分した場所から球の法線方向にパーティクルを放出します。
B
 		パーティクル パーティクルは X 軸で 90 度回転させ、面を放出方向に向かせます。
