カメラ距離スケール制限（近）

ここではカメラの寄りでパーティクルサイズに制限をかける機能「カメラ距離スケール制限（近）」について説明します。

概要

「カメラ距離スケール制限（近）」カメラの前で必要以上にパーティクルが大きくならない様に制限をかける機能です。
描画面積の大きなパーティクルがカメラに被さることによる描画負荷を軽減させることができます。
以下は「 設定項目」の概要です。

カメラ距離スケール制限（近）

「カメラ距離スケール制限（近）」を使用するかを「 チェックボックス」ON/OFF で切り替えます。
設定を ON にすると 「 スケール制限距離（近）」が表示されます。以下はグループ内の設定項目です。

スケール制限距離（近）

「 ダイヤル」で設定した値よりもカメラとの距離が近いパーティクルに対してカメラに映るサイズ（スクリーン上の大きさ）に制限をかけます。
