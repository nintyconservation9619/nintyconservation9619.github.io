カラー編集のUI

選択しているカラーやアルファの値を設定します。

A 時間 アニメーションのタイミングをパーティクル寿命に対する割合で設定します。
B カラーの彩度（Saturation）と明度（Value） カラーの彩度（Saturation）と明度（Value）を変更します。
C カラーの色調（Hue） カラーの色調（Hue）を変更します。
D カラー成分の表示形式

カラー成分の表示形式を、RGB 形式 / sRGB 形式  / HSV 形式から選択できます。
表示形式が異なるだけで、設定される値には影響ありません。

  • RGB形式：明度をリニア空間上で調整できます。
  • sRGB形式：明度をノンリニア空間上で調整できます。
  • HSV形式：色相、彩度、明度で調整できます。

関連リンク リニア空間については こちら を参照してください。
E カラーとアルファの設定 カラーとアルファの値を設定します。1以上の値も設定できます。
参考：

RGB 値に1.0 以上の値を設定することで、EffectViewer でグレアの効果が利用できます。
