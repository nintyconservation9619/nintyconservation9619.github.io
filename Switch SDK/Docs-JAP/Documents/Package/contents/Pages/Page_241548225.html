「マニュアルエミッタ放出」の制限事項

ここでは「マニュアルエミッタ放出」で使用できない機能と使用を推奨しない機能についてまとめています。

使用できない機能

以下は「マニュアルエミッタ放出」で使用できない機能です。

対象ノード 説明
エミッタ

以下の放出系機能は使用できません。

  • ワンタイム
  • 放出開始フレーム
  • 放出間隔タイプ「距離」すべての機能
  • 放出間隔タイプ「時間」「放出間隔」
チャイルドエミッタ チャイルドエミッタを含むエミッタセットをマニュアルエミッタ放出することはできません。
プレビュー

以下の機能は使用できません。

  • 開始前に強制計算
  • 放出間隔

使用を推奨しない機能

以下は「マニュアルエミッタ放出」で使用を推奨しない機能です。

対象ノード 説明
エミッタ

以下すべてのエミッタ時間アニメーションが正しく動作しません。

  • エミッタ形状 / スケール
  • エミッタトランスフォーム / 位置
  • エミッタトランスフォーム / 回転
  • エミッタトランスフォーム / スケール
  • 放出設定 / 放出レート
  • 初速設定 / 全方向初速
  • 放出設定 / 指定方向初速
  • その他運動設定 / 重力
  • パーティクル寿命 / 寿命
  • グローバル乗算カラー・アルファ
  • 基本スケール / 基準値
履歴式ストライプ
履歴式ストライプ 2		 パーティクル寿命がマニュアルエミッタ放出間隔未満の場合に正しく動作しません。
履歴式ストライプを使用する場合は、パーティクル寿命を放出間隔以上に設定してください。
