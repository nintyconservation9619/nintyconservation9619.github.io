はじめに

ここでは、UTIL ライブラリが提供する個々の機能に関して説明します。

それに先立ち、本節では Util ライブラリ全体で共通の事項について説明します。

UTIL ライブラリの名前空間

UTIL ライブラリが提供する型、定数、クラス、関数などは全て nn::util:: の名前空間に配置されています。

特に指定される場合を除いて、これより深い階層の名前空間にある型、定数、関数などは使用しないで下さい。
予告なく構成や仕様を変更する場合があります。

UTIL ライブラリのヘッダファイル

必要に応じて下記のヘッダファイルを include してください。

  • ベクトル・行列計算を利用する場合...nn/util/util_Vector.h, nn/util/util_Matrix.h, nn/util/util_Quaternion.h

nn/util/ ディレクトリより深い階層のヘッダは予告なく構成を変更する場合があります。
