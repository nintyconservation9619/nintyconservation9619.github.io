子エミッタ形状の表示

ここではビューアにチャイルドエミッタの形状とエミッタの軸を表示する機能「子エミッタ形状の表示」について説明します。

概要

「子エミッタ形状の表示」は、EffectViewer にエミッタの形状と軸を表示するデバッグ機能です。
また、「パーティクルにエミッタ割当」と「 エミッタ設定」と「 エミッタトランスフォーム」の設定内容が UI 操作に連動して表示されます。
