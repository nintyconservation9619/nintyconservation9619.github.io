テクスチャ

  このタブではパーティクルに貼るテクスチャ画像に関する設定を行います。

「 テクスチャタブ」には以下の「 サブタブ」があります。

テクスチャ
0 ~ 2

「テクスチャ0」、「テクスチャ1」、「テクスチャ2」の3つのサブタブです。

1 つのエミッタで利用可能なテクスチャは「テクスチャ0」～「テクスチャ2」で各 1 つずつ、最大 3 つが設定できます。
共通設定 「テクスチャ0」～「テクスチャ2」で共有する機能が設定できます。
参考：

テクスチャファイルは、エフェクトブラウザやエクスプローラから「 テクスチャ0 ~ 2」のサブタブ内に直接ドラッグ＆ドロップすることで登録できます。

以下は「 テクスチャ0～2」の設定項目です。

テクスチャ

「 テクスチャ」は、エミッタで利用するテクスチャについて設定する機能群です。
以下はグループ内設定項目の一覧です。

ファイル指定

テクスチャファイルのパスが表示され、テクスチャを登録できます。
テクスチャのプレビュー機能や、便利なプリセットボタンがあります。
適用するUVセット パーティクルタイプが「プリミティブ」の時、登録テクスチャの UV セットを選択できる機能です。
RGB を二乗して取得 登録したテクスチャの輝度を二乗することによりリニア空間に近づける機能です。
パターン分割    

テクスチャの分割数を設定できる機能です。
ラップ テクスチャの繰り返し方法を設定する機能です。
繰り返し数 テクスチャの繰り返し数を設定する機能です。
ランダム反転 パーティクル放出時に、テクスチャ座標を反転させるかを設定する機能です。
スフィアマップ 登録したテクスチャをスフィアマップとして適用する機能です。
フィルタモード テクスチャのフィルタ方式を設定する機能です。
ミップマップレベルを制限 テクスチャミップマップの設定ができる機能です。

テクスチャ座標アニメ

「 テクスチャ座標アニメ」は、登録したテクスチャの UV座標 について設定する機能群です。
以下はグループ内設定項目の一覧です。

UVスクロールアニメ テクスチャ UV 座標の位置をアニメーションできる機能です。
UV回転アニメ テクスチャ UV 座標の角度をアニメーションできる機能です。
UVスケールアニメ テクスチャ UV 座標のスケールをアニメーションできる機能です。

テクスチャパターンアニメ

「 テクスチャパターンアニメ」は、 1 枚のテクスチャに描かれた複数画像（パターン）のアニメーションを設定する機能群です。
以下はグループ内設定項目の一覧です。

再生方法 テクスチャ UV をステップ移動させることで、テクスチャのパターンをアニメーションできる機能です。

テクスチャオプション

「 テクスチャの密度を固定」は、テクスチャの表示サイズを固定にする機能について設定する機能群です。
以下はグループ内設定項目の一覧です。

テクスチャの密度を固定

テクスチャの表示サイズを固定化できる機能です。

以下は「 共通設定」の設定項目です。

乱数設定

「 テクスチャの密度を固定」は、「テクスチャ0」 ～ 「テクスチャ2」の乱数値を固定にできる機能について設定する機能群です。
以下はグループ内設定項目の一覧です。

乱数値を固定

「テクスチャ0」 ～ 「テクスチャ2」サブタブで乱数値を固定にできる機能です。
