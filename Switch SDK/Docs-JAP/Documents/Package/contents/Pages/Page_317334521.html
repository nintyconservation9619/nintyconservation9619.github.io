ui2d/font ライブラリから fontll2 ライブラリを使う方法

はじめに

本ページでは、fontll の内部エンジンをバージョンアップさせた fontll2 ライブラリを、ui2d/font ライブラリから使う方法について説明します。

fontll2 とは

fontll2 ライブラリでは、fontll ライブラリの内部エンジンをバージョンアップさせたものです。

fontll ライブラリを使ったときに一部のフォントの描画がおかしくなるなどの問題が発生した場合、fontll2 では修正されていることがありますのでお試しください。

詳しくは、「FONTLL2 ライブラリ」のドキュメントをご覧ください。

ui2d/font ライブラリから fontll2 ライブラリを使う方法

fontll2 ライブラリを使う方法は、下記の通りです。

  1. Include\nn\font\font_TextureCache.h を開き、「// #define NN_FONT_USE_FONTLL2」のコメントを外して NN_FONT_USE_FONTLL2 マクロを定義する。
  2. Sources\Libraries\font\libnn_font 以下のソリューションを開き、font ライブラリをリビルドする。
  3. Libraries 以下にある libnn_fontll2 ライブラリをアプリケーションにリンクする。

これにより、ui2d/font ライブラリは fontll2 を使用するようになります。
