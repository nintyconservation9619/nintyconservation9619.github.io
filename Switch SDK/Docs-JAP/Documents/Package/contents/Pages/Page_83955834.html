パンの動作

用語の解説

パンモード

ステレオ波形に対するパンの動作には、幾つかの種類が考えられます。このパンの動作種類を、パンモードと呼びます。

パンモードの指定は、ステレオの素材にのみ有効な設定です。

バランスパン

パンモードの一種類で、ステレオの左右チャンネルに対して、これらの音量バランスを調整する動作をこのように呼びます。バランスパンは、モノラルの素材に対して行う、通常のパンと同様の動作を行いますので、モノラルとの音量差が生じません。

ストリームサウンド等に使う、既にステレオとしてミックスダウンされた素材に適しています。

デュアルパン

パンモードの一種類で、ステレオの左右チャンネルを個別に、2 個の独立したモノラルボイスとして音量を調整する動作をこのように呼びます。デュアルパンでは、例えばパンを左に変更した場合に、右チャンネルの音が左チャンネルにも出力されます。

ウェーブサウンドで、ステレオサンプリングされた素材に対して、強引にパンを適用したい場合などに利用します。

パンカーブ

左右の音量を決定する際に、利用する音量の減衰カーブの種類をこのように呼びます。

パンカーブの指定は、ステレオ、モノラル両方の素材に有効で、その設定はサラウンドパンの動作にも適用されます。

動作仕様

ステレオ波形に対するパンの動作

ステレオ波形に対するパンの動作は、パンが中央に位置している場合には、左チャンネルが左スピーカー、右チャンネルが右スピーカーに出力され、それぞれのチャンネルの音は混ざり合いません。

この中央での動作は、パンモードによって共通ですが、パンを左右に動かした場合、パンモードによって動作が異なります。

パンを動かした場合の動作の違いを、パンを右に移動した場合を例にして、以下に解説します。

バランスパンの動作

左チャンネルの音量が下がり、右チャンネルの音量は上がるか、中央での音量を維持します。
左右チャンネルの音量のみが変化し、定位自体は変化しません。

デュアルパンの動作

左チャンネルのパンが右に移動し、右チャンネルの音量は変化しません。

結果として、左チャンネルの音が、右チャンネルからも鳴ります。

パンモードの設定

パンモードの設定は、SoundMaker で、ストリームサウンド、またはウェーブサウンド単位で行います。

パンカーブの種類と設定

パンカーブの設定は、SoundMaker で、ストリームサウンド、またはウェーブサウンド単位で行います。

カーブの種類は、平方根、三角関数、線形の3種類から選択します。カーブには更に中央での音量減衰の有無と、中央からの音量の増加を、中央の値でクランプする clamp の設定が可能です。

パンモード・パンカーブの種類とラウドネスとの関係

パンモード・パンカーブの設定によって、アプリケーション上で等倍音量で再生する際のラウドネス値と元波形のラウドネス値が異なる場合が存在します。

SoundMaker にて、音量値を等倍の 127、圧縮形式を PCM16 と設定し、パンがかからない状態で再生した時、ラウドネス値が元波形と一致するように設定したい場合は、以下のようにパンモード・パンカーブを設定してください。

  • モノラル音源の場合: 「任意のパンモード」の「Sqrt(-3dB), SinCos(-3dB)」
  • ステレオ音源の場合: 「バランスパン」の「Sqrt(0dB), SinCos(0dB), Linear(0dB)」(clamp の有無はどちらでも可)
