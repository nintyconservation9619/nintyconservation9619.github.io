NintendoWare について

NintendoWare は、ゲームソフトのグラフィックスやサウンドの開発に使用することができるライブラリとツールの総称です。
ゲームソフトの開発者がゲームの中身の制作に専念できるように、多くのゲームソフトで使用されるような基本的なツールとライブラリを提供します。

 3Dモデル、UI や フォント、3Dエフェクト を扱うためのモジュール群です。
SE、BGM などのゲームサウンドを扱うための開発環境を提供します。

グラフィックス、オーディオ以外のモジュール

モジュール
Spy アプリケーション上の情報を収集し、PC ツール(Spy.exe) のタイムライン上に可視化するためのライブラリとツールのセットです。

NintendoWare の特徴

パフォーマンス

NintendoWare は Nintendo プラットフォームに最適化されています。
NintendoWare を使用することでハードウェアの性能を最大限に引き出すことができます。

組み込みやすさ

NintendoWare は、様々なジャンルのゲームソフト、様々なスタイルのワークフローに組み込めるように、粒度の小さな機能単位から提供しています。ゲームソフトの開発者は必要な機能を組み合わせて独自のゲームエンジンを作成することや、既存のワークフローに組み込むことができます。

安定性

NintendoWare は社内外のゲームソフト開発において利用されており、常に検証が行われています。後方互換性にも配慮して拡張が行われています。

発展性

NintendoWare は社内外の利用者から要望を受け付けており、拡張が続けられています。

ソースコード公開

NintendoWare は、ゲームソフト開発者にライブラリのソースコードを公開しています。デバッグや拡張のために内部実装を確認することができます。

マルチプラットフォーム対応

NintendoWare は、基本的にどのプラットフォーム向けにもアプリ側でコードの変更なくコンパイルできます。
