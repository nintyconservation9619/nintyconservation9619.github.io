 nn::g3dニュース

3D テクスチャープリセットコンバーターでプリセットの部分上書き機能を追加

3D テクスチャープリセットコンバーターでプリセットの部分的な上書きができるようになりました。例えば、以下のように記述すると後に記述した preset_override で Normal1 プリセットに対して サイズを 50 % にリサイズするオプションを追加します。

<presets>
    <preset name="Normal1" format="unorm_bc5" comp_sel="rg01" hint="normal" />
</presets>
<preset_overrides>
    <preset_override name="Normal1" resize_w="50%" />
</preset_overrides>


中間ファイルコンバーターに fbx をモデル中間ファイルに変換する機能を追加

中間ファイルコンバーターで fbx ファイルを fmdb に変換できるようになりました。ただし変換できる fbx ファイルには制限があります。詳しくは 中間ファイルコンバーター をご覧ください。

 nn::ui2dニュース

ウインドウ座標系の原点が左上に設定されているときにキャプチャペインやシザーペインを適切に動作させるための設定を追加

ウインドウ座標系の原点が左上に設定されているときにキャプチャペインやシザーペインを適切に動作させるための設定を追加しました。

ウインドウ座標系の原点を左上に設定している場合は nn::ui2d::DrawInfo::SetLeftTopWindowOrigin() へ true を設定することによりランタイムが適切な領域を処理するようになります。

 nn::atkニュース

ストリームサウンドのループ回数を取得する関数のループ回数増加タイミング変更

ストリームサウンドのループ回数を取得する、 nn::atk::StreamSoundHandle::GetPlayLoopCount() のループ回数増加タイミングを変更しました。

以前のループ回数増加は、 nn::atk::StreamSoundHandle::GetPlaySamplePosition() で取得できる再生位置が、ループにより巻き戻った少し後に行われていました。
今回の変更により、 nn::atk::StreamSoundHandle::GetPlaySamplePosition() で再生位置がループにより巻き戻るタイミングと、ループ回数の増加タイミングが一致するようになります。
