コマンドラインオプション

コマンドラインオプション

LayoutEditor は コマンドライン起動から各種機能を利用して、ファイル入出力や スクリーンキャプチャの保存を行うことができます。

書式

以下の様な書式を使用します。

LayoutEditor.exe 入力ファイルパス [option]

オプション

オプション 意味
-o, --output 読み込んだファイルの中間ファイル出力先フォルダを指定します。
--close-if-loaded ツールに読み込まれたレイアウトを閉じます。
--parts-root-path 部品ルートフォルダを指定します。
--current-frame LayoutEditor の時間フレーム値を指定します。
--save-screen-capture スクリーンキャプチャを保存します。
--capture-file-path スクリーンキャプチャの出力ファイルを指定します。
--capture-size スクリーンキャプチャの画面サイズを指定します（例：--capture-size=1280,720）。
--send-viewer レイアウトをビューアへ転送します。コマンドライン引数に入力ファイルパスが指定されず、既に LayoutEditor が起動している場合は、選択されているレイアウトが転送されます。

例

以下の書式を使用することで、部品フォルダを指定して、ファイルを読み込み画面サイズを指定して、スクリーンキャプチャを保存できます。

LayoutEditor.exe "D:\input.flyt" --parts-root-path="D:\parts_dir" --save-screen-capture --capture-file-path="D:\out\test.bmp" --capture-size=1280,720
