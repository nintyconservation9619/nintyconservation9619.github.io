作業の流れ

ここでは EffectMaker を起動してから作業を完了するまでの一連の流れについて説明します。

1 作業の開始 EffectMaker を起動し、作業を開始します。
2 ファイルを読み込む エミッタセットファイル（.eset）を EffectMaker に読み込みます。
3 ビューアへの接続 EffectViewer を起動、接続します。
4 作業の完了 EffectMaker の作業を完了させます。

1 作業の開始

作業を開始する為に、エフェクトの設定を行う EffectMaker を起動します。

EffectMaker の起動

以下の起動ファイルをマウス左ボタンでダブルクリックすることにより EffectMaker が起動します。

\Tools\Graphics\EffectMaker\EffectMaker.exe

2 ファイルを読み込む

エフェクトデータとなるエミッタセットファイル（.eset）を EffectMaker に読み込みます。
以下の場所にあるエミッタセットファイルを例に説明します。

\Samples\Sources\Resources\Vfx\Eset\Fire_01.eset

以下いずれかの操作を行うとエミッタセットファイル「Fire_01.eset 」が読み込まれます。
ワークスペース上にはエミッタセット「 Fire_01」が表示されます。

  • メインメニュー項目 [ファイル] → [ 開く] でダイアログが開きますので Fire_01.eset を選択してから「OK」ボタンを押します。
  • エクスプローラ上で Fire_01.eset を選択し、ワークスペース上にドラッグ＆ドロップします。
  • F10キーを押して「 エフェクトブラウザ」を開き、Fire_01.eset を選択した状態で「 ワークスペースへエクスポート」ボタンを押します。

3 ビューアの起動と接続

接続先プラットフォームとビューアをプリセットから選択し、ビューアを起動、接続します。

接続先プラットフォームを切り替えたい場合は、ツールバーにある を変更します。

接続先ビューアを切り替える

接続先ビューアを切り替えたい場合は、ツールバーにある「 接続状態→切断」 を変更します。
選択したプラットフォームにより選択項目の内容が変わります。

ビューアの起動

「接続先プラットフォーム」と「接続先ビューア」の設定に適合したビューアを起動します。
「 ビューアを起動ボタン」をクリックすることで ビューアが起動します。
エミッタセット「 Fire_01」の設定内容を確認します。

ビューアに再接続する

万が一ビューアとの接続が切れた場合、以下いずれかの操作を行うと再度ビューアに接続できます。

  • メインメニュー項目 [プレビュー]→ [ 再転送] を選択します。
  • ツールバーにある「 切断状態→接続」ボタンを押します。
  • ショートカットキー F5 を使用します。

4 作業の完了

EffectMaker の作業を完了させます。

EffectMaker を終了する

以下いずれかの操作を行うと、開いているすべてのファイルを閉じて EffectMaker が終了します。

  • メインメニュー項目 [ファイル]→[終了] を選択します。
  • メインウィンドウの「閉じる」ボタンを押します。
  • ショートカットキー Ctrl + Q を使用します。

未保存のファイルがある場合はダイアログが表示されますので、保存先フォルダを指定してから「OK」ボタンを押します。

Windows 版ビューアを終了する

以下いずれかの操作を行うと、Windows 版ビューアが終了します。

  • Windows 版ビューアのウィンドウタイトルバーをマウス右ボタンクリックで開くメニューから「閉じる」を選択します。
  • Windows 版ビューアのウィンドウ右上の「 閉じる」ボタンをマウス左ボタンクリックします。
  • Windows 版ビューアのウィンドウを選択した状態でショートカットキー Alt+F4 を使用します。
