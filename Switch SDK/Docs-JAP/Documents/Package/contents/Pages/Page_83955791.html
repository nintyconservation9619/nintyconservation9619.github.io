SystemEvent クラス

前提

システムイベント機能は、システムからの通知をイベントとして同期待ちできる機能です。

機能概要

SystemEvent クラスは、nn::os::SystemEventType オブジェクトを利用した システムイベント機能 をラッピングしたもので以下のメンバ関数を持ちます。API リファレンスはリンク先を参照して下さい。

名前 説明 備考
SystemEvent コンストラクタ

引数なしのコンストラクタ を利用して未初期化状態のインスタンスを生成します。
~SystemEvent デストラクタ DestroySystemEvent() 同等の破棄を実行
Wait システムイベントの待機 WaitSystemEvent() 同等のシステムイベントの待機
TryWait システムイベントのポーリング TryWaitSystemEvent() 同等のシステムイベントのポーリング
TimedWait システムイベントの時限付き待機 TimedWaitSystemEvent() 同等のシステムイベントの時限付き待機
Clear システムイベントのクリア ClearSystemEvent() 同等のシステムイベントのクリア

 

SystemEvent クラスの使用例

システムイベント機能は、NintendoSDK のモジュールとして提供されている機能を使って利用する前提となっています。単一のサンプルとして開示することが難しいため、ここでは使用例は記載していません。各モジュールの使い方を示したサンプルプログラム等を参照して下さい。
