ライトアトリビュート設定プラグイン

ポイントライトまたはスポットライトのシェイプノードに NintendoWare 用のライトのアトリビュートを設定します。

使用方法

「Nintendo > Set Light Attribute...」を選択するとウィンドウが表示されます。

Selected Light 欄には、選択されているライトの transform ノードが表示されます。

参考：

4 つ以上のノードが選択されている場合は省略表示となり、右端に選択された数が表示されます。

設定方法

ポイントライトまたはスポットライトの transform ノードまたはシェイプノードを選択（複数選択可）すると、Set 欄に選択したライトの現在の設定が表示されます。
複数選択の場合、最初に選択したライトの設定が表示されます。

ウィンドウの各項目を操作してアトリビュートを変更します。複数選択の場合、選択したライトを一括して変更できます（変更したアトリビュートだけ反映されます）。

ウィンドウを閉じずに別のノードを選択すると、別のライトを設定することができます。

Distance Attenuation

距離減衰を使用するなら ON にします。
この項目はライトのシェイプノードの減衰率（Decay Rate）アトリビュートと連動しています。
この項目を ON にした際に、減衰率が「減衰なし（No Decay）」なら「リニア（Linear）」に変更します（「二次（Quadratic）」または「キュービック（Cubic）」なら変更しません）。
減衰率が「リニア」「二次」「キュービック」のいずれでも出力される中間ファイルは同じです。
Start Distance

減衰開始距離を指定します。
この項目は Distance Attenuation が ON の場合に有効です。
End Distance

減衰終了距離を指定します。
この項目は Distance Attenuation が ON の場合に有効です。

Start Distance、End Distance は Maya の画面上には反映されません。

Start Distance、End Distance の項目名を右クリックして、メニューから Set Key を選択すると現在のフレームにキーを設定できます。

このプラグインを実行すると、選択したライトのシェイプノードにカスタムアトリビュートが追加されます。
カスタムアトリビュートは、チャンネルボックスやアトリビュートエディタ（追加のアトリビュート）に NW4F Attn Start Dist（減衰開始距離）、NW4F Attn End Dist（減衰終了距離）という名前で表示されます。

参考：

ライトのタイプを変更すると、カスタムアトリビュートとカスタムアトリビュートに接続されたアニメーションが消えるので注意してください。
ライトのタイプを変更した場合は、このプラグインで Start Distance、End Distance を再設定してください。
