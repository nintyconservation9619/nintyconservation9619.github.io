再生方法「ランダム」

ここではパーティクル放出時に、パーティクルごと、表示する 1 パターンが選ばれるテクスチャパターンアニメーションの再生方法「ランダム」について説明します。

概要

再生方法「ランダム」は、パーティクル放出時に、パーティクルごと、表示する 1 パターンをランダムにできる機能です。
以下は「 設定項目」の概要です。

設定を ON にすると「 ランダム」と「  パターンテーブル」が表示されます。以下はグループ内の設定項目です。

パターン数

表示パターン数を「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は「ランダム」の各機能を ON/OFF した場合の比較画像です。

テクスチャパターン画像
使用パターンテーブル数 16
A

パターン数 = 2

放出時に適用されるコマは 0 , 1 の 2 コマからランダムに選択されています。
B パターン数 = 8 放出時に適用されるコマは 0～7 の 8 コマからランダムに選択されています。
C パターン数 = 16 放出時に適用されるコマは 16 コマ全てからランダムに選択されています。
