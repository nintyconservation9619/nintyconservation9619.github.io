Z ロール速度

ここでは選択しているノードの回転速度を設定する機能「Z ロール速度」について説明します。

概要

「Z ロール速度」は、速度を与えることで選択ノードを回転できる機能です。
円状ルートの移動方向軸に選択ノードを回転させます。

参考：

ストライプの計算方式が「ビルボードストライプ」の場合は効果がありません。

以下は「 設定項目」の概要です。

Z ロール速度

毎フレーム角度に加算する値を「 ダイヤル」で設定します。

設定例

以下は「Z ロール速度」の設定値による比較画像です。

Zロール速度 = 0 Zロール速度 =2
デフォルト 0 では回転しません。

毎フレーム 2 ずつ角度に加算します。

エミッタから四方に複数のパーティクルが放出された状態で効果的です。
