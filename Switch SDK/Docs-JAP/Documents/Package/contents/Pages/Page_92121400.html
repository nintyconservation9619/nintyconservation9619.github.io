【プロジェクト設定】

ゲームプロジェクト毎の設定ができます。
基本的にユーザー毎の設定は推奨しません。

参考：

「 水色枠のグループ」の設定項目は、その場で編集を加えても EffectMaker 再起動時に「プロジェクト設定ファイル」の内容で上書きされます。

プロジェクト設定ファイルパス

「 プロジェクト設定ファイルパス」は、ゲームプロジェクト毎の設定ファイルを登録する機能です。
ゲームアプリケーション独自の描画順を記述したファイルなどが使用できます。

「プロジェクト設定ファイル」には、以下ファイルの設定内容が記述されています。

カスタムシェーダ

「 リセットボタン」を押すことで、設定項目が初期状態にリセットされます。

描画設定のプレビュー

「 描画設定のプレビュー」は、プロジェクト設定ファイルに記述されたゲームプロジェクト毎の描画設定を表示する機能です。

リニアワークフロー

「 リニアワークフロー」は、ゲームプロジェクトでリニア編集モードに変更するかどうかを ON/OFF で切り替える機能です。

関連リンク 以下は関連ページへのリンクです。

リニア空間 カラーの編集環境やテクスチャの表示についての色空間について説明します。

標準重力

「 標準重力」は、放出タブの「標準重力を適用 」のボタンを利用可能にするかを ON/OFF で切り替えます。
ON にすることで、以下の設定が可能になります。

単位長

ゲーム世界の寸法（単位は cm ）
FPS ゲームプロジェクトの基本フレームレート
参考：

以下は、FPS = T、単位長 = M　とした場合の「標準重力」機能が利用する計算式です。
重力値を算出する式 = 重力加速度 9.8 / ( T × T ) × M / 10000

ドキュメントファイルパス

「 ドキュメントファイルパス 」は、 EffectMaker が参照するドキュメントの場所を設定できます。
「ドキュメントファイル」を設定します。
設定した内容は「OK」ボタンを押すことで確定し、 EffectMaker を再起動することで反映されます。

本設定項目は【ユーザー設定】と【プロジェクト設定】の両方にあります。
関連リンク 設定による挙動については 「【ユーザー設定】と【プロジェクト設定】共通の設定項目」 を参照してください。
