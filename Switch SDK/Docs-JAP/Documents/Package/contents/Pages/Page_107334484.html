SDEV・EDEV にコントローラーを接続する方法

概要

SDEV に接続したことのないコントローラーを利用するためには、本体へのコントローラーの登録処理 (ペアリングと呼びます) を行う必要があります。
NX では本体に最大 10 台のコントローラーの情報を登録することができます。 

本体とペアリングを行うためには、以下の 4つの方法があります。

  1. (Joy-Con のみ) 本体のレール部分に Joy-Con を装着して登録する
  2. 本体とコントローラーをコントローラー充電ケーブルでつなげて登録する
  3. Devmenu から登録する
  4. (SDEV 利用時のみ) デバッグコントローラーを使用して、コントローラーを登録する

本体に登録されたコントローラーの情報を削除するためには、 DevMenu を使用します。

  • コントローラーの登録を削除する

 

(Joy-Con のみ) 本体のレール部分に Joy-Con を装着して登録する

  1. SDEV の電源を起動しDevMenu が表示されること、またはアプリが動作していることを確認してください。

  2. Joy-Con (R) を NX 本体(SDEV または EDEV) の右レールに、または Joy-Con (L) を NX 本体の左レールに装着してください

  3. 1秒以内に登録処理が完了します。
    本体に装着されたコントローラーが操作できる状態にある場合、コントローラーの登録処理が完了しています。
    製品ではコントローラーが登録された際に、登録された旨をユーザーに通知する UI が表示されます。      

 

本体とコントローラーをコントローラー充電ケーブルでつなげて登録する

  1. SDEV MP および EDEV MP の電源を起動しDevMenu が表示されること、またはアプリが動作していることを確認してください。

  2. SDEV MP（EDEV MP）に同梱されている SDEV クレードルを取り付けてください。

  3. Joy-Con (R) および Joy-Con (L) を Joy-Con 充電グリップに取り付けてください。

  4. Joy-Con 充電グリップまたは Switch Pro コントローラの USB Type-C 端子と SDEV クレードル の USB 端子を、コントローラー充電ケーブルでつなげてください。

  5. 数秒以内に登録処理が完了します。
    製品ではコントローラーが登録された際に、登録された旨をユーザーに通知する UI が表示されます。

  6. 登録後にコントローラーのボタンを押すと、本体と無線でつながります。

 

注意：

Joy-Con 充電グリップおよび Switch Pro コントローラのUSBファームウェアが古いと、上記動作が正しく行われません。

お使いの前に、こちらのページを参考にUSBファームウェアをアップデートしてください。

 

 

Devmenu から登録する

  1. SDEV の電源を起動しDevMenu が表示されることを確認してください。

  2. DevMenu  の "Controller" メニューを選択してください。

  3. "Controller" メニュー内の "Start controller pairing" を押すと、本体側のコントローラーの登録待ち受け処理が開始します。
  4. コントローラーのシンクロボタンを長押し(1秒以上) してください。
    コントローラーのペアリング処理が開始するとプレイヤーランプがシーケンシャルに点滅するアニメーションが開始します。 
     
  5. アニメーションが停止し、自身の接続ポートを表すプレイヤーランプのみが点灯するとペアリングが成功です。
    プレイヤーランプが消灯した場合はペアリングに失敗しているため、4. の手順をやり直してください。
     
  6. 2台目以降のコントローラーを登録するためには、4. を再度やり直してください。
     

  7. ペアリングが終了したら、"Cancel controller pairing" を選択して、登録処理を終了します。

    注意：

    複数の SDEV/EDEV 本体が、"Start controller pairing" を行い待ち受け状態となっている場合、意図しない本体とコントローラーが登録される可能性があります。
    コントローラーの登録処理が完了したら、必ず "Cancel controller pairing" を押して登録処理を終了してください 

 

(SDEV 利用時のみ) デバッグコントローラーを使用して、コントローラーを登録する

  1. SDEV の電源を起動しDevMenu が表示されることを確認してください。

  2. コントローラーのシンクロボタンを長押し(1秒以上) してください。
    コントローラーのペアリング処理が開始するとプレイヤーランプがシーケンシャルに点滅するアニメーションが開始します。 
     
  3. デバッグコントローラー(クラシックコントローラー) の ホームボタンを長押し(1秒以上)して、SDEVをペアリング状態にいれてください。
    本体側のペアリングが 5秒で終了するため、先に2.の手順を行いコントローラー側のペアリングを開始するようにしてください。

  4. アニメーションが停止し、自身の接続ポートを表すプレイヤーランプのみが点灯するとペアリングが成功です。
    プレイヤーランプが消灯した場合はペアリングに失敗しているため、2. の手順からやり直してください。 

※ 複数台のコントローラーを同時にはペアリングできません、1台ずつ 2-4 の手順を行う必要があります。

コントローラーの登録を削除する

NintnedoSDK ドキュメントの機能 -> 開発機の制御 -> DevMenu ->  コントローラペアリング設定の削除 を参照ください。
