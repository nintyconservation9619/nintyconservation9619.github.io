チュートリアル

ここでは、NX Addon を利用して NX アプリケーションを開発するための基本的な手順を説明します。

サンプルをビルド＆実行する

NX Addon に含まれるサンプルをビルド＆実行する方法を学ぶには、クイックスタート を参照してください。

開発機 (SDEV / EDEV) 上でサンプルアプリケーションを実行するには、事前に SDEV の初期化 / EDEV の初期化 を済ませておく必要がありますので、あわせてご確認ください。

Windows 上でグラフィックス機能を利用するサンプルアプリケーションを実行するには、事前に専用のドライバのインストールが必要です。詳しい手順は、NVN ライブラリ の 「GPU ドライバのインストール方法」を参照してください。

NX アプリケーションのプロジェクトを作成する

NX Addon を利用することで、NX アプリケーション開発のためのプロジェクトファイルの作成が可能になります。
また、既存のプロジェクトファイルを NX アプリケーション向けに設定することもできます。
詳しい手順は 新規プロジェクトの作成 を参照してください。

NX アプリケーションを開発する

NX Addon を利用することで、NX API セットを使ったアプリケーション開発が可能になります。
NX アプリケーションはビルド対象のプラットフォームを切り替えることで、基本的に同じコードを Windows 環境と NX 開発機の両方でデバッグ・実行することができます。
詳しい手順は アプリケーションの開発 を参照してください。

NX アプリケーションを開発機にインストールして実行する

開発した NX アプリケーションは開発機にインストールして実行することでホスト PC を接続せずに動作確認することができます。
詳しい手順は アプリケーションの開発機へのインストール を参照してください。

NX アプリケーションを NX 開発カードに書き込んで実行する

開発した NX アプリケーションは、 NX 開発カードに書き込んで実行することでホスト PC を接続せずに動作確認することができます。
詳しい手順は  NX 開発カードへの書き込み を参照してください。
