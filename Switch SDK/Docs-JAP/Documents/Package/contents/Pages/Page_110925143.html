フォントやテクスチャのリンク切れを解消したのに、ファイルを開くたびにエラーメッセージが表示される

質問

フォントやテクスチャのリンク切れを解消したのに、ファイルを開くたびにエラーメッセージが表示されます。

回答

リンク切れの解消をおこなった際には、リンク切れを起こしていたレイアウトウィンドウをすべて閉じて、部品パレット右上にあるのリロードボタンで部品パレットを更新するようにしてください。
この処理によって、ツール内に読み込まれている古い（リンク切れをおこしている）部品レイアウトが破棄され、エラーメッセージが出なくなります。
