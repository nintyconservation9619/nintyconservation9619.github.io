しきい値によるアルファ補間

 しきい値によるアルファ補間

「しきい値によるアルファ補間」の設定入力への切り替えを行います。

 しきい値によるアルファ補間

チェック無しの場合（初期設定）

テクスチャのカラーに「黒カラー補間」と「白カラー補間」の入力値で影響を与えます。


  • 「黒カラー補間」のアルファ値はテクスチャの透明部分の透明度に影響を与えます。
  • 「白カラー補間」のアルファ値はテクスチャの不透明部分の透明度に影響を与えます。
 しきい値によるアルファ補間

チェック有りの場合

「黒カラー補間」と「白カラー補間」で設定したアルファ値を「しきい値」として、アルファ値の補間を行います。


  • 「黒カラー補間」のアルファ値はテクスチャの透明部分の透明度に影響を与えます。
    ここで指定した数値をしきい値として、数値に満たないテクスチャのアルファ値を透明として扱います。
  • 「白カラー補間」のアルファ値はテクスチャの不透明部分の透明度に影響を与えます。
    ここで設定した数値をしきい値として、数値以上のテクスチャのアルファ値を不透明として扱います。
「しきい値によるアルファ補間」を設定し、テクスチャ境界のぼやけを軽減
参考：

「しきい値によるアルファ補間」は、高性能なアルファテスト（細かな制御ができ、境界が滑らか）として利用することができます。
また、アルファチャンネルをディスタンスフィールド化したテクスチャに適用することで、拡大・縮小した場合のテクスチャ境界のぼやけを軽減することができます。
( アルファチャンネルのディスタンスフィールド化は、 Nintendo Ware の Photoshop プラグインで行うことができます )
