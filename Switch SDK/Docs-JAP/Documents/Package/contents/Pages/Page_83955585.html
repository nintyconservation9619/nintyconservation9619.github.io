MIDI コントロールコード対応表

  • #13 transposeは、値から 64 を引きます。
  • #16 ～ #19 はそれぞれ、ローカル変数 0 ～ 3 に値をセットします。
  • #28、#29 sweep_pitch は、値から 64 を引きます。(#29 sweep_pitch はその後 24 倍します)
  • #64 damper_on / damper_off は、値が 64 以上のとき damper_on、64未満のとき damper_off になります。
  • #65 porta_on / porta_off は、値が64以上のとき porta_on、64 未満のとき porta_off になります。
  • #68 monophonic_on /monophonic_off は、値が 64 以上のとき monophonic_on、64 未満のとき monophonic_off になります。
  • #120 ～ #127 はチャンネルモードメッセージで、コントロールチェンジではありません。
  • #100 ～ #101 の RPN (Registered Parameter Number) は、現在 RPN0 のピッチ・ベンド・センシティビティ (Pitch Bend Sensitivity) のみに対応しています。
  • データエントリーは、#6 の ”Data Entry MSB” のみ対応しています。
