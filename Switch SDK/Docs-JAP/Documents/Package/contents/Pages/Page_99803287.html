バイナリファイルの「ファイル指定」

ここではバイナリファイルが登録できる「ファイル指定」について説明します。

概要

「ファイル指定」はバイナリファイルを登録できる機能です。
出力されたバイナリファイルを読み込むことで、ビューアでそのエフェクトを確認できます。
以下は「 設定項目」の概要です。

ファイル指定

バイナリファイルのパスが表示され、任意にファイルを登録できます。以下はボタンの種類です。

ロード ファイル選択ウィンドウが開き、バイナリファイルが選択できます。
リロード 設定したバイナリファイルが更新されます。
クリア 設定したバイナリファイルを解除します。

以下はバイナリファイル内エミッタセットの再生を制御するボタンの種類です。

前のエミッタセットを再生 直前のエミッタセットをビューアで再生します。
次のエミッタセットを再生 次のエミッタセットをビューアで再生します。
