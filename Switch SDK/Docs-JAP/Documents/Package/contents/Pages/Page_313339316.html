注意：

公開されているパッケージに含まれる API のみを記載しています。一部の限定公開パッケージに含まれる API はここには記載しておりません。

参考情報

  • 利用が禁止されている API は以下の方法のいずれかでチェックすることも可能です。
    • AuthoringTool 実行時にオプション --error-unpublishable をつけることでエラーログが出力される
    • AuthoringEditor で nsp ファイルを開くとマスター ROM 提出不可エラーが表示される
