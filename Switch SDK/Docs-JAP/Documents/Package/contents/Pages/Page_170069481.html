オーディオニュース

NintendoWare Audio ニュース

NintendoSDK 0.15.0 における NintendoWare オーディオ開発環境の変更点を紹介します。

SoundMaker

サウンドの波形出力が可能になりました

SoundMaker や SoundProjectWaveExporter で、任意のサウンドを波形化し、wav ファイルに出力できるようになりました。

詳しい操作方法は、SoundMaker/波形ファイル出力機能、SoundProjectWaveExporter を参照ください。

注意：

既知の不具合として、出力した wav ファイルの頭に無音が入ることがありますが、今後のリリースにて修正予定です。

AtkPlayer

I3DL2 Reverb エフェクトのプレビューができるようになりました

AtkPlayer の Effect パネルでプレビューできるエフェクトに I3DL2 Reverb が追加されました。Reverb, Delay エフェクトと同様にエフェクトの適用, パラメータの調整を行うことができます。

詳しくは、「AtkPlayer」を参照ください。
