ツール環境設定

レイアウトエディタウインドウのメニューバー「編集」から選択できる「ツール環境設定」ついて説明します。

ツール環境設定ウインドウ

A 安全フレームを描画する チェックが入っているとレイアウトウインドウに安全フレームが描画されます。
B フレームカラー 安全フレームの色を指定します。
C フレームマージン 安全フレームを設定する範囲を指定します。
D 隠されたペインをビューアに表示しない チェックが入っていると隠されたペインがビューアに表示されなくなります。
E テキストボックスのエスケープ文字を制御コードに変換する プログラマーが検証目的で使用します。デザイナーは基本的に利用しません。
F 区間タグのフレーム区間内にキーが無い場合は、アニメーションを出力しない

プレビュー時のバイナリコンバータ実行時に --omit-nokey-all を実行します。

ゲームプログラム側のバイナリコンバータ設定に合わせて設定します。
G 描画モード DirectX モードのみ選択できます。
H FOV

FOV の値を設定します。

FOV ：視野 ( Field-of-View )
I

Near

Near の値を設定します。

Near : 近く
J Far

Far の値を設定します。

Far : 遠く
K 「パースペクティブ投影」を初期状態にする チェックを入れると、パースペクティブ投影を初期状態にします。
L デプステストを行う

チェックを入れると、デプステストを行います。

前後関係による描画判定のテストです。
M 30FPS

チェックを入れるとフレームレートが 30FPS になります。

チェックが外れている状態のフレームレートは 60FPS です。
N null ペインの " 子階層に透明度の影響を伝える " の初期値を " 有効 " にする チェックを入れると、null ペインの「子階層に透明度の影響を伝える」の初期値を「有効」にします。
O リニア色空間モード

バイナリコンバータで「テクスチャ」と「色パラメータ」のリニア色空間への変換処理を行います。

ビューアの描画モードを切り替えて、sRGB 色空間に復元して表示を行うようにします。

設定変更をしたら、レイアウトウインドウを閉じて、開きなおしてください。

関連リンク リニア色空間モードについては、「レイアウトにおけるリニア空間」を参照ください。
P 色パラメータをリニア色空間で指定する

指定された場合、コンバート時に色パラメータのリニア色空間へのガンマ補正を行いません。

プログラマーと相談して、チームの共通設定として設定する項目です。

関連リンク 「レイアウトにおけるリニア空間」の、「頂点カラー、カラー補間」を参照ください。
Q フレームバッファに sRGB 書き込みする

フレームバッファに対して、sRGB 書き込みをするかどうかを制御します。
基本的にリニア色空間モードの際には、常にONになります。（チェックは GUI からは無効にできません。

「プロジェクト設定ファイル」を直接編集すれば、任意に設定することができます。）


プログラマーと相談して、チームの共通設定として設定する項目です。
R テクスチャを sRGB フェッチする

テクスチャを sRGB フェッチするかどうかを制御します。
基本的にリニア色空間モードの際には、常にONになります。（チェックは GUI からは無効にできません。

「プロジェクト設定ファイル」を直接編集すれば、任意に設定することができます。）


プログラマーと相談して、チームの共通設定として設定する項目です。
S アニメーション管理モードの警告ウィンドウを表示する チェックを外すと、アニメーション区間タグ編集ウインドウで「結合モード / 分割モード」の切り替え時に警告ウィンドウが表示されなくなります
T 矢印キーで選択ペインを移動する 親子階層ウィンドウで選択しているペインを、レイアウトウィンドウ上で、矢印キーで移動できるようにします。
