概要

NoftWriter は、ntf ファイルをタグに書き込みNOFT (Nintendo Original Format Tag) を作成するツールです。

配置場所

このツールは Nintendo SDK の NX Addon に含まれており、以下の場所に配置されています。

TargetTools/NX-NXFP2-a64/NoftWriter/Release/NoftWriter.nsp

使用方法

準備

amiibo データリストパッケージに含まれている ntf ファイルを SD カード内の任意の場所に書き込みます。ファイル名やフォルダ名に、ASCII 文字以外は使用できません。

amiibo データリストパッケージには CSV 形式のデータリストファイル amiibo_DataList_yyyymmdd.csv が含まれています。ファイル名の「yyyymmdd」はパッケージのバージョンを示す日付を表しています。この CSV 形式のデータリストファイルを SD カードの任意の場所に保存すると、ntf ファイルで定義されているキャラクターの US_English 名が合わせて表示されます。データリストファイルはなくても、NoftWriter の動作には支障ありません。

ツールの起動

ntf ファイルを書き込んだ SD カードを NX に挿入した状態で NoftWriter を起動します。

$ Tools/CommandLineTools/RunOnTarget.exe TargetTools/NX-NXFP2-a64/NoftWriter/Release/NoftWriter.nsp

nsp ファイルを開発機にインストールして使用することもできます。nsp ファイルのインストール方法については、アプリケーションの開発機へのインストール を参照してください。

起動すると NX の画面に下図のような画面が表示されます。SD カードに保存されている全ての ntf ファイルの一覧が表示されます。

 

NTFMaker で作成した開発用の ntf ファイルを利用する場合は、 CSV 形式のデータリストファイルにキャラクター名やファイルの説明を追加することができます。

NVL-AAAA_00_00.ntf,Mario
：
NVL-AACC_00_00.ntf,PAC-MAN
NVL-AACB_00_00.ntf,Mega Man
NVL-DUM1_00_00.ntf, NTF File for Debug 1
NVL-DUM4_00_00.ntf, NTF File for Debug 4

一列目に ntf ファイルの名前、二列目に表示する文字列を記載します。ASCII 文字以外は使用できません。

書き込み

書き込みたい ntf ファイルを選択します。方向ボタンの上下でフォーカスを移動し、A ボタンを押すことで選択できます。ntf ファイルを選択すると NX の画面に下図のような画面が表示されます。

「Write(except lock bits)」 によって作成した NOFT は、後で本ツールを使用して別の ntf ファイルを書き込むことが可能です。「Write」 によって作成した NOFT は、後で別の ntf ファイルを書き込むことはできません。また、「Write(except lock bits)」で作成した NOFT に、「Write」で別の ntf ファイルを上書きすることができますが、書き込んだ以降は、「Write(except lock bits)」で書き込んでも失敗します。

まだ書き込んでいないタグ Write(except lock bits) 書き込み可能
まだ書き込んでいないタグ Write 書き込み可能、以降は書き込み不可
「Write(except lock bits)」で書き込んだタグ Write(except lock bits) 書き込み可能
「Write(except lock bits)」で書き込んだタグ Write 書き込み可能、以降は書き込み不可
「Write」で書き込んだタグ Write(except lock bits) 書き込み不可
「Write」で書き込んだタグ Write 書き込み不可

「Write(except lock bits)」または「Write」を選択してAボタンを押すと書き込み待機画面が表示されます。書き込み可能なタグを NFC デバイスにかざしてください。

参考：

複数のコントローラーを接続している場合、タグはツールの操作に使用したコントローラーにかざしてください。

タグを検知すると書き込み中画面に切り替わります。書き込み中はタグを離さないようにしてください。

注意：

まだ何も書き込んでいないタグに書き込みを行うときは、タグを離さないよう特に注意してください。NoftWriter は初めて書き込むタグに対し、重要な領域への書き込みを行います。このとき書き込みに失敗すると「まだ書き込んでいないタグ」、「Write(except lock bits) で書き込んだタグ」、「Write で書き込んだタグ」のどれにも該当しないタグになってしまう可能性があります。

結果の表示

タグへの書き込みが正常に終了した場合、下図のように NX の画面に「Finished!!」と表示されます。

タグへの書き込みが失敗した場合、下図のように NX の画面に「Write failed...」と表示されます。

注意

誤ったデータを書き込むとそのタグが使用不可能になる恐れがあります。
