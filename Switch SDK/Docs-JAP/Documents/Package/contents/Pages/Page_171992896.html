コンピュートシェーダを一括で処理する

GPU を最適に動作させるために、コンピュートシェーダを一括で処理した上で、各エミッタを描画する機能について説明します。

利用方法

以下の手順でコンピュートシェーダを一括処理できます。

  1. エフェクトシステムを初期化する際に nn::vfx::Config で コンピュートシェーダ一括処理モードに切り替える
  2.  nn::vfx::System::BatchCalculationComputeShaderEmitter を呼び出してコンピュートシェーダを一括処理する
// エフェクトの初期化
nn::vfx::Config  config;
   :
   :
config.SetEnableComputeShaderBatchProcess();
nn::vfx::System* pVfxSystem = new nn::vfx::System( config );


// エフェクトの描画処理
pVfxSystem->SwapBuffer();
pVfxSystem->SetViewParam( processingIndex, &view );

// コンピュートシェーダ駆動を一括で行う
pVfxSystem->BatchCalculationComputeShaderEmitter( pCommandBuffer, nullptr );

// エミッタの描画を行う
pVfxSystem->Draw( processingIndex, pCommandBuffer, groupId );
