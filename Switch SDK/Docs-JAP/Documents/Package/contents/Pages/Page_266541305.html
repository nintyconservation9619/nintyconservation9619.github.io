はじめに

ここでは、NFC ライブラリの各機能の利用方法についてまとめます。

フロー図では各関数の基本的な使用例を記載しています。実際のアプリケーション開発では状況に応じて適切に対処してください。発生したエラーへの対処方法については、Nintendo Switch Online Documents のエラーコードリストに記載されている「解決方法(開発者向け)」も参考にしてください。リストに記載されているメッセージはエラービューアで表示することもできます。

NFC タグへのコマンド送信

NFC タグへコマンドを送信するには nn::nfc::SendCommandByPassThrough() を呼び出します。

参考：

NFC タグへのコマンド送信には１～２秒程度かかります。

次のフロー図は NFC タグへのコマンド送信時の例です。

図 28. NFC タグへのコマンド送信

NFC タグへの同一セッション複数コマンド送信

NFC タグへ同一セッションで複数のコマンドを送信するには nn::nfc::KeepPassThroughSession() を呼び出してセッションを維持した後に nn::nfc::SendCommandByPassThrough() を呼び出します。

必要なコマンドを送信した後、速やかに nn::nfc::ReleasePassThroughSession() を呼び出してセッションを解放します。

図 29. NFC タグへの同一セッション複数コマンド送信

タグ喪失通知の待ち受け

タグへのアクセスが終了したあとや異なるタグでタッチされたあとに、タグの検知を引き続き行う場合は、再度検知可能な状態に戻します。

ユーザーにはタグを離すように通知します。タグが NFC デバイスから離されるとタグ喪失がアプリケーションに通知されます。

タグ検知の終了は nn::nfc::StopDetection() を呼び出します。

エラービューアによるメッセージの表示

アプリケーションによるエラービューア表示

NFC ライブラリから返されるエラーに対し、エラービューアを使用してメッセージを表示させることができます。

エラービューアに対応しているエラーやメッセージ内容は Nintendo Switch Online Documents のエラーコードリストを確認してください。
