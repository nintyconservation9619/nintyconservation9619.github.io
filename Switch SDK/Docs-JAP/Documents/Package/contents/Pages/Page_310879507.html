概要

参考：

NX Addon 5.3.0以降で使用可能です。

ショップの切り替え、「ダウンロードできる本体」の設定を行います。

使い方

Switching Shops

Valid Shop

開発用に、ショップの切り替えを行うことができます。

設定値が「Dev Shop」のとき、ショップ選択時にDummyShopが立ち上がります。

設定値が「Shop」のとき、ショップ選択時に本来のショップが立ち上がります。

Registering an Active Console

Initial Settings and Registering an Active Console

ショップ初回接続時の設定と「ダウンロードできる本体」の登録をします。ニンテンドーアカウントとリンクしたユーザーアカウントが選択対象になります。

初回のショップアクセス時に行われる処理を実行出来ます。実行した本体が「ダウンロードできる本体」として登録されます。

初回のショップアクセス時には、「ダウンロードできる本体」の登録以外にもショップの利用に必要な設定が行われるため、1つの本体に対して1回は必ず実行する必要があります。

Registering an Active Console

「ダウンロードできる本体」の登録をします。ニンテンドーアカウントとリンクしたユーザーアカウントが選択対象になります。

すでに別の本体で「ダウンロードできる本体」が登録されているニンテンドーアカウントを指定した場合、登録に失敗します。

「ダウンロードできる本体」を変更する場合は、「ダウンロードできる本体」に登録した本体で登録を解除してから、登録対象の本体で「ダウンロードできる本体」の登録を実行してください。

Unregistering an Active Console

「ダウンロードできる本体」の登録解除をします。ニンテンドーアカウントとリンクしたユーザーアカウントが選択対象になります。

参考：

DevMenu Applicationでは解除できません。

Unregistering All Active Consoles

本体の全ての「ダウンロードできる本体」の登録解除をします。ニンテンドーアカウントとリンクしたユーザーアカウントが対象になります。

参考：

DevMenu Applicationでは解除できません。

Checking the Active Console Registration Status

「ダウンロードできる本体」の登録状態の確認をします。ニンテンドーアカウントとリンクしたユーザーアカウントが選択対象になります。

実行した本体が指定したアカウントの「ダウンロードできる本体」として登録されている場合は、下記の内容がダイアログで表示されます。

This console is registered as an active console of the account user index [ユーザーアカウントのindex番号]

他の本体が指定したアカウントの「ダウンロードできる本体」として登録されている場合は、下記の内容がダイアログで表示されます。

Another console is registered as an active console of the account user index [ユーザーアカウントのindex番号]

指定したアカウントの「ダウンロードできる本体」が登録されていない場合は、下記の内容がダイアログで表示されます。

Unregistered
