警告 / エラー

FontConverter(GUI) 版の「警告 / エラー」ダイアログを表示します。
また、ダイアログからより詳細に説明したドキュメントページにジャンプできます。
以下は「警告 / エラー」ダイアログです。

A エラーの内容と簡単な解決方法が記述されています。
テキストはマウスによるドラッグ操作で選択でき、ショートカットキー [Ctrl+C] でコピーできます。
B 関連リンク「こちら」をクリックスると「警告、エラーメッセージ一覧 」ページにジャンプします。
エラーや警告の発生条件と解決方法を図による説明を交えてより詳しく解説しています。
ブラウザのページ内検索機能にコピーした「エラーの内容」のテキストをペーストして検索してください。
