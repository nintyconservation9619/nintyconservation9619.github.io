テクスチャ SRT アニメーション

テクスチャ SRT アニメーションは、"座標"のオフセット U,V 、タイリング U,V 、角度 Wをアニメーションさせて設定します。
それ以外のパラメータには対応していません。

テクスチャ SRT アニメーションはアニメーションを設定したパラメータについてのみデータが出力されます。 データが出力されなかったパラメータは、モデルファイルの状態で表示されます。よって、パラメータが一定値でも モデルファイルの状態と異なる場合は、パラメータにキーを 1 つ以上打ってからテクスチャ SRT アニメーションを出力してください。

参考：

エクスポートオプションのマテリアル圧縮が有効になっていると、マテリアルがまとめられた結果、テクスチャ SRT アニメーションが設定されていないマテリアルまでアニメーションしてしまう場合があります。

その場合は マテリアルアトリビュート設定プラグインで、テクスチャ SRT アニメーションを設定したマテリアルが圧縮されないようにしてください。
