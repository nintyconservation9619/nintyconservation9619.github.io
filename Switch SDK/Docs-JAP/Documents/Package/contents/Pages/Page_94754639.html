トリミング

ここではパーティクルの描画されない範囲を排除する機能「トリミング」について説明します。

概要

「トリミング」は、テクスチャの無駄なアルファ領域を見て、描画面積が最適になる様にパーティクルを生成する機能です。
描画に使用されない余分な描画領域を減らせる為、負荷削減への効果が期待できます。
以下は「 設定項目」の概要です。

トリミング

テクスチャのアルファ形状によって最適な大きさで トリミングするかを「 チェックボックス」ON/OFF で切り替えます。

参考：

以下の機能と併用した場合、この機能は無視されます。

  • パーティクル形状を「プリミティブ」設定時
  • テクスチャパターンアニメ使用時
