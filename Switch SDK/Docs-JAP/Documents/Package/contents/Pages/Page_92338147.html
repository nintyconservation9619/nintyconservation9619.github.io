はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK NX Addon で使用されるコンパイラ、ツールチェーンである Toolchain for NX について説明します。

Toolchain for NX は LLVM/Clang を NX 向けにカスタマイズしたものとなっており、また、標準 C ライブラリとして musl を NX 向けにカスタマイズしたものを使用しています。
これらのツールチェーンの外部ドキュメントへの案内と、NX 向けにカスタマイズしている部分についての説明を扱います。

対象読者

NintendoSDK NX Addon を使ってアプリケーションを開発する開発者を対象とします。
