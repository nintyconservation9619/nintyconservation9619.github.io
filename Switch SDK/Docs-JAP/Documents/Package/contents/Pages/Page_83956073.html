頂点ステート

頂点ステート

頂点ステートは、頂点シェーダー実行前に実行される頂点フェッチについてのいくつかのステートをまとめて表現します。

情報

頂点ステートの情報は nn::gfx::VertexStateInfo として定義されています。

頂点属性ステート

SetVertexAttributeStateInfoArray により頂点属性ステートの配列を設定します。

頂点属性ステートは、利用したい頂点属性の数だけ記述する必要があります。

セマンティックインデックス

SetSemanticIndex により、要素のセマンティックインデックスを設定します。

スロット番号

SetShaderSlot により、シェーダーにおけるどのスロット番号に割り当てるかを設定します。 

バッファーインデックス

SetBufferIndex により、フェッチする頂点バッファーのインデックスを設定します。

オフセット

SetOffset により、頂点バッファーにおける先頭からのオフセットを設定します。

フォーマット

SetFormat により、頂点属性のフォーマットを設定します。

シンボル名

SetNamePtr により、シェーダーにおけるどの頂点属性に割り当てるかの名前を設定します。
このパラメーターに NULL を設定した場合はスロット番号のパラメーターが使われます。 
ポインター先はオブジェクトの初期化後に破棄できます。 

頂点バッファーステート

SetVertexBufferStateInfoArray により頂点バッファーステートの配列を設定します。

頂点バッファーステートは、利用したい頂点属性の数だけ記述する必要があります。
頂点属性をインタリーブさせることにより、頂点属性の数よりも少なくさせることができます。

 ストライド

SetStride によりバッファーのストライドを設定します。

インスタンスの除数

SetDivisor により、いくつの要素ごとにインスタンスを進めるかを表すインスタンスの除数を設定します。
0 を設定することは、データがインスタンス単位ではなく頂点単位であることを意味します。
この設定はこのバッファーを参照するすべての頂点属性に適用されます。 

オブジェクト

頂点ステートのオブジェクトは nn::gfx::VertexState として定義されています。

初期化と破棄

頂点ステートの初期化と破棄はオブジェクトの規則に従います。

頂点ステートのセット

パイプラインを使う実行モデルを利用する場合は、パイプラインの情報に頂点ステートの情報を記述します。

パイプラインを使わない実行モデルを利用する場合は、nn::gfx::CommandBuffer クラスの SetVertexState 関数を呼ぶことによって頂点ステートをセットします。
