スケールフェード

ここでは 放出されるパーティクルを徐々に拡大して出現させる機能「スケールフェード」について説明します。

概要

「スケールフェード」は、エミッタから放出されるすべてのパーティクルをスケール 0 の状態から拡大して出現させる機能です。
「フェードイン時間」に設定した値をかけて徐々に出現させることができます。
以下は「 設定項目」の概要です。

スケールフェード

「スケールフェード」機能の使用 / 未使用を「 チェックボックス」ON/OFF で切り替えます。

関連リンク 「フェードイン時間」についてはこちらを参照してください。
