放出間隔タイプ「距離」

ここではパーティクルが放出される間隔を「距離」で設定する放出間隔タイプ「距離」について説明します。

概要

放出間隔タイプ「距離」は、パーティクルの放出間隔を距離で設定できる機能です。
放出間隔を「距離」で設定することによって、エミッタの移動速度に関係なく、パーティクルの放出間隔を一定にできます。

以下は「 設定項目」の概要です。

「放出間隔タイプ」の設定を「距離」にすると「 放出間隔タイプ「距離」」が表示されます。以下はグループ内の設定項目です。

放出間隔 パーティクルが放出される間隔を移動距離で設定する「放出間隔」について説明します。
最大移動距離/1フレーム エミッタの速度が速いとき、必要以上にパーティクルが放出されないよう制限をかける機能「最大移動距離 / 1フレーム」について説明します。
最小移動距離/1フレーム エミッタが移動しない、または微小な移動のとき、移動したとみなすことでパーティクルを放出できる機能「最小移動距離 / 1フレーム」について説明します。
移動距離切り捨ての閾値 エミッタの速度が速いとき、必要以上にパーティクルが放出されないよう制限をかける機能「移動距離切り捨ての閾値」について説明します。
パーティクル放出最大数 放出されるパーティクル数に制限をかける機能「パーティクル放出最大数」について説明します。
