SSLプロトコル・暗号スイート

Webアプレットがサポートする SSLプロトコル、暗号スイートの情報です。

Nintendo SDK に付属する SSL ライブラリのサポート状況とは異なりますので注意してください。詳しくは 「機能 > ネットワーク > SSL ライブラリ」を参照して下さい。

なお、動画プレイヤーは Nintendo SDK に付属する SSL ライブラリを使っていますので、SSL ライブラリのサポート状況を参照してください。

サポートするSSLプロトコル

サポート対象のプロトコル キー交換アルゴリズム 暗号化アルゴリズム
  • TLSv1.0
  • TLSv1.1
  • TLSv1.2
  • ECDHE
  • RSA
  • AES-GCM （128 bit および 256 bit）
  • AES-CBC （128 bit および 256 bit）
  • 3DES （112 bit）

暗号スイート

Webアプレットがサーバーに送信する暗号スイートのリストです。
