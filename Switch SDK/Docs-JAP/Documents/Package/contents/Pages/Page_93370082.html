エミッタ形状「点」

ここではエミッタ形状「点」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「点」は、パーティクルをワールド座標系の一点から放射状に放出します。
静止したパーティクルを 1つだけ表示させたい時に便利なエミッタ形状です。

「エミッタ形状」を「点」に設定することで利用できます。
「点」形状専用の設定項目はありません。

設定例

以下はエミッタ形状「点」のパーティクル発生位置と速度方向について説明した図です。
ワールド座標系 で XYZ = 0,0,0 の位置に配置したエミッタからパーティクルを放出しています。

A 発生位置 パーティクルはワールド座標系の一点に発生します。
B 速度方向 パーティクルは「点」から放射状に放出されます。
