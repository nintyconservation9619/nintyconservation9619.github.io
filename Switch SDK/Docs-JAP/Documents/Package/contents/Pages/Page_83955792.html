サムネール・プレビュー表示機能

はじめに

Windows エクスプローラー上で、画像ファイルのサムネールとプレビューを表示するための機能です。
インストール・アンインストールについては「サムネールエクスプローラー拡張 ユーザーマニュアル」ページを、
拡張機能の設定変更については「セットアップマネージャー」ページをご覧ください。

使用方法

エクスプローラーの「表示」メニューで「アイコン」「一覧」 「詳細」「並べて表示」「コンテンツ」を切り替えると、表示モードに適した方法でサムネールを表示します。
対応フォーマット : Targa ファイル (.tga), PSD ファイル (.psd)、AI ファイル (.ai)、EPS ファイル(.eps)、テクスチャー中間ファイル(.ftxa,.ftxb)

 

アルファチャンネルのあるテクスチャーは、大アイコン以上の表示以外ではカラーチャンネルと アルファチャンネルを乗算してサムネールを表示します。
このため、アルファチャンネルがゼロの場合は何も表示されません。

Adobe Illustrator 以外で出力された EPS ファイルのサムネールは表示されません。

モデル中間ファイル、レイアウトファイル、エミッタセットのサムネール表示

モデル中間ファイル(.fmdb/.fmda)、レイアウトファイル(.flyt)、エミッタセット(.eset) はファイル内にサムネール情報を持たないため、エクスプローラー上でサムネール表示するためには、指定のフォルダーにサムネール用の画像を作成する必要があります。具体的には、サムネールを作成するファイルを含むフォルダーに 「.NwThumbs」という名前のフォルダーを作成し、そのフォルダー以下に (ファイル拡張子を含むファイル名).(jpg/png) という名前の画像ファイルを作成すると、サムネール表示されるようになります。

例えば、「C:\Models\human.fmdb」のパスのモデル中間ファイルのサムネール表示をしたい場合は、「C:\Models\.NwThumbs\human.fmdb.jpg」というパスの画像ファイルを作成すると、サムネール表示されるようになります。

アイコン表示について

アイコン表示を大アイコン以上にすると、カラーチャンネルとアルファチャンネルのサムネールを別々に表示します。 
また、テクスチャーフォーマット、テクスチャーの幅と高さ、ミップマップレベル、データサイズも表示します。

参考：

ミップマップを最大限に生成していない場合、ミップマップレベルを赤字で表示します。

ツールチップの表示について

エクスプローラ内に表示されているサムネールにマウスオーバーすると、ツールチップにテクスチャー情報が表示されます。
