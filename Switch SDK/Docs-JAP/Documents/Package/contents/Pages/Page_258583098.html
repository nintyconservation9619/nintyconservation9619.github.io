 nn::atk ニュース

評価用の機能追加による処理負荷およびメモリ要求量の増大

評価用の機能追加により、 NXAddon 4.2.0 においてサウンドスレッドの CPU 処理負荷および nn::atk::SoundSystem、 nn::atk::SoundArchivePlayer が要求するメモリ要求量が増大します。
以降のバージョンにおいて処理負荷およびメモリ要求量の増大を抑えるための修正を含める予定ですが、 NXAddon 4.2.0 においては増大が起こる点にご注意ください。
