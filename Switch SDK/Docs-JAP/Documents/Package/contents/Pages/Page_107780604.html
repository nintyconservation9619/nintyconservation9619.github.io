VI ライブラリ

概要

このドキュメントの目的

このドキュメントでは、Nintendo SDK に付属するビデオインターフェース（VI）ライブラリについて説明します。

対象読者

このドキュメントは、Nintendo SDK を使用したアプリケーションの開発を目指している方々を対象としています。

関連ドキュメント

このドキュメントは、nn::vi ネームスペース API のリファレンスと合わせて使用してください。

VI ライブラリに用意されている機能

VI ライブラリは、ディスプレイの機能に関する情報を提供し、ディスプレイプロセスに各種のレイヤを登録します。アプリケーションでは、NVN のテクスチャ群をレイヤに関連付けることができます。ディスプレイプロセスでこのレイヤを使用して、ディスプレイ上の表示に合成できます。
