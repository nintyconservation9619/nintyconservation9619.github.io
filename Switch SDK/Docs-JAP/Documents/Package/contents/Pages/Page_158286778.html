LayoutViewer

 LayoutViewer に新しく追加された機能について紹介します。

デバッグ表示として処理負荷を表示するメーターに対応

LayoutViewer にレイアウトの負荷が解りやすい「処理メーター」が追加されました！

A Calc ランタイムの CPU による計算処理コストを milisecond で表示します。
B Draw ランタイムの CPU による描画処理コストを milisecond で表示します。
C Gpu ランタイムの Gpu コストを milisecond で表示します。
参考：

メーター内の 灰色部分 はレイアウト以外のビューアの表示関連の負荷を表しています。

レイアウト , フォントで新たに追加されたページを紹介します。

LayoutEditor のチュートリアルページ

LayoutEditor のチュートリアル、「リザルト画面の作成（前編）」が追加されました！
「リザルト画面の作成（前編）」では、基本的なペインを使用したレイアウト画面を作成する方法を説明します。

関連リンク チュートリアルページはこちらを参照してください。

FontConverter のチュートリアルページ

FontConverter(GUI版)で以下のようなオリジナルフォントを作成する方法を説明したチュートリアルページが追加されました！

関連リンク チュートリアルページはこちらを参照してください。

注意：

チュートリアルの実施により完成するフォントファイルはパッケージに含まれていません。
以下のバッチファイルを使用することで生成されます。
\Samples\Sources\Resources\Ui2d\Common\setupForNX.bat

上記バッチファイルに以下の画像ファイル「IconFont_16.tga」をドラッグ&ドロップしてください。
