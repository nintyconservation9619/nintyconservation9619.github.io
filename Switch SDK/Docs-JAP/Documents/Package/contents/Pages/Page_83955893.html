 エミッタノード

「エミッタ」ノードの機能について説明します。

基本設定 選択したエミッタの基本設定を行います。
エミッタ エミッタの形状やトランスフォームの設定を行います。
放出 パーティクルの放出に関する設定を行います。
パーティクル パーティクルに関する設定を行います。
コンバイナ パーティクルの見た目に関する設定を行います。

テクスチャ

パーティクルに貼るテクスチャに関する設定を行います。

カラー

エフェクトのカラーやアルファに関する設定を行います。
スケール パーティクルのスケールに関する設定を行います。
回転 パーティクルの回転に関する設定を行います。
Cstmシェーダ オプションでカスタムシェーダ定義ファイルが指定されている場合にユーザーが作成したシェーダの設定を行います。
