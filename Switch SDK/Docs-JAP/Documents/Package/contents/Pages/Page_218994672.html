SoundMaker や AtkPlayer で複数のサウンドを再生する

1. 概要

BGM を再生しながら効果音を再生して確認するなど、複数サウンドを同時に再生したい場合の操作方法です。

SoundMaker と AtkPlayer のどちらも最大 4 つまでサウンドを同時再生できます。

注意：

この同時再生とは、並列に再生が行えるということで、同時に発音開始できるということではありません。

2. サウンドの複数再生方法

2.1 SoundMaker で複数サウンドを再生する

1. プレビュープレイヤーにサウンドを登録します。以下のどちらかの方法でサウンドを登録することができます。

  • サウンドセットタブのリスト領域で、サウンドを選択して再生操作を行う
  • サウンドセットタブのリスト領域で、サウンドのインデックスセルをクリックし、プレビュープレイヤーにドラッグ＆ドロップする

2. プレビュープレイヤーの空いている行にもサウンドを登録します。

3. プレビュープレイヤーにて、サウンドが登録された行の再生ボタンをそれぞれクリックすると、手順 1 のサウンドを再生しつつ、別サウンドを再生できます。


参考：

サウンドが登録された行をクリックして選択状態にすることで、ツールバーの再生ボタンやショートカットキーによる再生を行うことも可能です。


2.2 AtkPlayer で複数サウンドを再生する

2.2.1 AtkPlayer で再生操作を行う

1. AtkPlayer を起動します。

2. SoundArchivePlayer にて、←→で再生するサウンドを切り替えて A ボタン（キー）で再生します。

3. ↑↓でハンドルを変更します。

4. ←→で再生するサウンドを切り替えて A ボタン（キー）で再生すると、手順 1 のサウンドを再生しつつ、別サウンドを再生できます。

2.2.2 SoundMaker で再生操作を行う

1. AtkPlayer を起動します。

2. AtkPlayer と SoundMaker を接続します。

3. SoundMaker のプレビュープレイヤーパネルで行う操作が、AtkPlayer の SoundArchivePlayer パネルに反映されます。本ページ内「SoundMaker で複数サウンドを再生する」の操作を行うと確認できます。

3. 困ったときは

共通

先に再生していたサウンドが停止してしまう、または後発サウンドを再生できない

  • 同時再生可能なサウンド数の上限を超えると、優先度によって再生中のサウンドが停止する、または新たにサウンドを再生できなくなります。 プレイヤーのサウンドリミットや、[プロジェクト]→[プロジェクトの設定] から「サウンドアーカイブ設定タブ1」を開き、サウンドアーカイブにあるサウンド数設定を確認してください。
    •          　
  • AtkPlayer で同一サウンドを複数再生する際は、SoundMaker で対象サウンドの単一再生設定を確認してください。 単一再生を【使用しない】以外に設定していると、同一サウンドを複数再生できない場合があります。
