リピート毎のオフセット

リピート数／リピート毎のオフセット

ここでは「範囲内ループ」で利用する「箱」の内側でパーティクルを繰り返し表示させる機能について説明します。

概要

「リピート数」は放出されたパーティクルを繰り返し描画させる機能です。
また、「リピート毎のオフセット」により一度描画する度にオフセットが加算されることで、パーティクルは前回の描画からずれた位置に表示されます。
オフセット値を十分に不規則な値にすることで、繰り返しによる規則性を緩和できます。
以下は の概要です。

リピート数

「箱」内のパーティクルをここで設定した値分、繰り返し描画させます。

リピート毎のオフセット

コピーしたパーティクル位置にオフセットをかけます。
一度描画する度にオフセットが加算されることでパーティクルは前回の描画からずれた位置に表示されます。
オフセット値を十分に不規則な値にすることで、繰り返しによる規則性を紛らわすことができます。

参考：

単一エミッタで複数回描画する場合、エミッタ／パーティクルの計算処理を一度だけで済ませることができます。
複数エミッタを使用する場合と比べて、その分 CPU 処理コストが軽減されます。但し描画コスト自体は複数回分必要です。
