 フィールドノード

フィールドノードの機能について説明します。

ランダム パーティクルをランダムな方向に移動させるフィールド「ランダム」の設定を行います。
GPUノイズ 「ランダムに見える軌道」に沿って、パーティクルを移動させるフィールド「GPUノイズ」の設定を行います。
磁力 引き寄せたり反発させることで、パーティクルにバネの様な動きを与えるフィールド「磁力」の設定を行います。
スピン パーティクル全体を竜巻状に回転させるフィールド「スピン」の設定を行います。
収束 指定した地点にパーティクルを収束させるフィールド「収束」の設定を行います。
位置に加算 指定した方向にパーティクルを移動させるフィールド「位置に加算」の設定を行います。
シンプルコリジョン 簡易的なコリジョンを設置し、パーティクルと衝突させるフィールド「シンプルコリジョン」の設定を行います。
カールノイズ 空間上にゆらぎを設置することでパーティクルの動きを制御するフィールド「カールノイズ」の設定を行います。
カスタムフィールド ゲームアプリ毎に追加可能なフィールドです。
