フィールド

パーティクルに対して初速度以外の運動を付加することができる「フィールド」について説明します。
フィールドを設定することによって、初速度以外の方向に力を加えることができます。
EffectMaker のフィールド機能は以下の場所にあります。

  • エフェクト制作でよく使用されるフィールド「空気抵抗」や「重力」はエミッタノードの「放出」タブ内にあります。
  • 使用したいときに追加できる「フィールドノード」があります。

関連リンク 以下は関連ページへのリンクです。

空気抵抗 エミッタノードにあるフィールドです。
放出されたパーティクル全体の初速を減衰または加速させる機能「空気抵抗」について説明します。
重力 エミッタノードにあるフィールドです。
放出されたパーティクルに対して重力を与える機能「重力」について説明します。
フィールドノード フィールドノードです。複数種類のフィールドをノードとして追加できます。

フィールドタイプ

フィールドの種類には大きく分けて以下の２種類があります。

位置系フィールド パーティクルの位置を毎フレーム更新するフィールドです。
速度系フィールド パーティクルの速度を毎フレーム更新するフィールドです。

以下は「位置系フィールド」です。それ以外のフィールドは「速度系フィールド」です。
位置系フィールドは「空気抵抗」の影響を受けません。

GPUノイズ 「ランダムに見える軌道」に沿って、パーティクルを移動させるフィールド「GPUノイズ」について説明します。
スピン パーティクル全体を竜巻状に回転させるフィールド「スピン」について説明します。
収束 指定した地点にパーティクルを収束させるフィールド「収束」について説明します。
位置に加算 指定した方向にパーティクルを移動させるフィールド「位置に加算」について説明します。
参考：

「シンプルコリジョン」は、速度系フィールドですが、着地後の挙動は位置を動かして補正しています。

「運動量ランダム」の影響を受けないフィールド

以下は「運動量ランダム」の影響を受けないフィールドです。

ランダム パーティクルの動きを秩序のない挙動にできる「ランダム」について説明します。
GPUノイズ パーティクルの動きがランダムに見えるが規則性のある動きにできる「GPUノイズ」について説明します。
位置に加算 指定した方向にパーティクルを移動させるフィールド「位置に加算」について説明します。
カールノイズ 空間上にゆらぎを設置することでパーティクルの動きを制御する機能「カールノイズ」について説明します。
