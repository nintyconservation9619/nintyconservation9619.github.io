表情の種類

Mii には全部で19種類の表情があります。

各表情は nn::mii::Expression 列挙型で定義されています。

表情の種類 サンプル 説明

nn::mii::Expression_Normal

通常。エディタで作成したままの顔です。
nn::mii::Expression_Smile 笑い、喜び。目がニコニコ目になります。
nn::mii::Expression_Anger 怒り。目と眉毛がつりあがり、口が口角の下がった口になります。
nn::mii::Expression_Sorrow 悲しみ、困り。目と眉毛が垂れて、口が波線になります。
nn::mii::Expression_Surprise 驚き。眉毛が上がり、目が見開いた目になります。
nn::mii::Expression_Blink まばたき。目が線になります。
nn::mii::Expression_OpenMouth 口パク。口が大きく開いた口になります。
nn::mii::Expression_SmileOpenMouth 目がニコニコ目で、口が大きく開いた口になります。
nn::mii::Expression_AngerOpenMouth 目と眉毛がつりあがり、口が大きく開いた口になります。
nn::mii::Expression_SorrowOpenMouth 目と眉毛が垂れて、口が大きく開いた口になります。
nn::mii::Expression_SurpriseOpenMouth 眉毛が上がり、目が見開いた目になり、口が大きく開いた口になります。
nn::mii::Expression_BlinkOpenMouth 目が線になり、口が大きく開いた口になります。
nn::mii::Expression_WinkLeft 左目を閉じてウィンクします。
nn::mii::Expression_WinkRight 右目を閉じてウィンクします。
nn::mii::Expression_WinkLeftOpenMouth 左目を閉じてウィンクし、口が大きく開いた口になります。
nn::mii::Expression_WinkRightOpenMouth 右目を閉じてウィンクし、口が大きく開いた口になります。
nn::mii::Expression_LikeWinkLeft 左目を閉じてウィンクし、笑い口になります。
nn::mii::Expression_LikeWinkRight 右目を閉じてウィンクし、笑い口になります。
nn::mii::Expression_Frustrated 両目を閉じてウィンクし、口が波線になります。
