タイムラインパネル

タイムライン上にデータを表示するパネルです。

現在は、オーディオパフォーマンス、ボイス数の表示にのみ対応しています。

画面説明

  項目 説明
A nn::audio パフォーマンス

Voice, SubMix, FinalMix, Sink の処理時間を表示します。

  • Voice, SubMix, Sink については最初のアイテムの開始から最後のアイテムの完了までの時間です。

凡例には処理の開始から完了までの経過時間を数値で表示します。

グラフには処理の開始時間から完了時間までの範囲を表示します。

単位は usec です。

 

それぞれの項目について、詳しくは オーディオパフォーマンス情報の説明 を参照ください。
B nn::audio ボイス数 発音しているボイスの数を数値とグラフで表示します。
C オレンジ色の縦線 現在の時間を示します。

操作方法

操作 手順
左クリック 現在の時間を、指定位置に移動します。
右ドラッグ 横方向にスクロールします。
Ctrl + ホイール 横方向に拡大、縮小します。
Alt + ホイール 縦方向に拡大、縮小します。
