コマンドライン

コマンドライン

概要

3DEditor をコマンドラインから起動する際に、オプションを指定すると、3DEditor 起動時の設定を変更できます。

コマンドラインから 3DEditor を起動

3DEditor.exe がインストールされているディレクトリをカレントディレクトリにした状態、またはパスが通っている状態で、以下のコマンドを入力すると 3DEditor が起動します。

参考：

3dEditor.exe

コマンドラインオプション

参考：

3dEditor.exe [@引数ファイル] [--version] [--team-settings-file=チーム設定ファイル] [-r] [プロジェクトファイル|中間ファイル|フォルダ] ...

@引数ファイル

引数ファイルに記述した内容を、引数として指定できます。 引数ファイルは複数行に渡って各オプションを記述できます。 行頭に # がある行はコメントとして扱います。 オプション値指定の "" によるエスケープに改行を含めることはできません。

--version

3dEditor のバージョンを表示します。このオプションだけが指定された時はバージョンを表示し、3dEditor を起動しないで終了します。

--team-settings-file

チーム設定ファイルを指定します。 

-r --recursive

フォルダを指定していた場合にそのフォルダ以下の中間ファイルを再帰的に読み込みます。

プロジェクトファイル|中間ファイル|フォルダ

読み込むファイルやフォルダを指定します。プロジェクトファイルを指定する場合はプロジェクトファイル単体を指定してください。

オプション指定例

3DEditor を指定のチーム設定を読み込ませて起動します。

参考：

3DEditor.exe --team-settings-file=GameProject.3dEditor.teamsettings
