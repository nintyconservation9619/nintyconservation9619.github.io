出力の「Image」タブ

Image

「Image」タブではフォントを画像ファイルとして出力します。
ここで出力した画像ファイルはそのまま入力指定のImageで使うことができます。
以下は「WindowsFont」タブの UI 説明です。

A 画像ファイル 出力する画像ファイルのパスを指定します。
B 文字順序 画像中にどのような順番で文字が並んでいるかを選択します。
C セルサイズ

セルの大きさとグリフ出力位置を指定します。

「グリフを中心に配置する」にチェックをつけるとグリフ出力位置がセルの中心となります。
関連リンク 詳細については「セルサイズ」を参照してください。
D 画像形式

出力する画像ファイルの画像フォーマットを指定します。

「拡張子で指定」を選択すると拡張子を元に出力形式を決定します。
E グリッドを描画する チェックをつけるとグリッドを描画します。
F null / 幅線 /　グリッド / 余白

出力される画像ファイル中のそれぞれの部分に色を指定することができます。

  関連リンク 詳細については「出力画像の色の指定」を参照してください。
