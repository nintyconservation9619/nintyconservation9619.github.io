デバイス設定の変更

概要

開発機の設定を変更することができます。

使い方

System Settings

Language

言語設定を変更できます。設定可能な言語は以下になります。

  • American English
  • British English
  • Japanese
  • French
  • German
  • Latin American Spanish
  • Spanish
  • Italian
  • Dutch
  • Canadian French
  • Portuguese
  • Russian

Time Zone

｢Edit｣ ボタンを選択するとネットワーク時計のタイムゾーンを設定できます。

Date Time

日時の設定を行えます。ネットワーク時計を利用するには、ネットワークが利用できる必要があります。

Automatic Correction

日時をネットワーク時計で自動補正するか否かを切替えられます。自動補正が有効時は、日時を手動で変更することができなくなります。

Display Settings

Screen Brightness

開発機の液晶輝度を変更できます。｢Auto Brightness Control｣ を有効にすると、明るさセンサーが検知した周囲の明るさに応じて液晶輝度が自動補正されます。

Screen Resolution

HDMI出力時のディスプレイ解像度を変更できます。

Consumer Electronics Control (CEC)

テレビとの電源連動 (CEC) の有効とするかどうかを設定します。Onにすることで有効になり、Offにすることで無効となります。

本設定は即座に適用されます。

TV Underscan Settings

HDMI出力時のアンダースキャン値(%)を変更できます。設定可能な値は[0 - 20]です。

Sound Settings

TV Sound

テレビから出力される音声のチャンネル数を設定できます。設定可能なチャンネル数は以下になります。デフォルト値は ｢Stereo｣ です。

  • Mono (1 ch)
  • Stereo (2 ch)
  • Surround (5.1 ch)
  • Auto (テレビが対応していれば 5.1 ch、非対応ならば 2 ch)

System Sound (Speaker & HP)

開発機本体から出力される音声(本体内蔵スピーカー・ヘッドホン出力)のチャンネル数を設定できます。設定可能なチャンネル数は以下になります。デフォルト値は ｢Stereo｣ です。

  • Mono (1 ch)
  • Stereo (2 ch)

Mute when headphones are Disconnected

音声の出力先がヘッドホンからスピーカに切り替わった時に、スピーカからの音声出力を自動的に消音するか否かを設定できます。デフォルト値は ｢有効｣ です。

Enable Headphone Output Level Limit

ヘッドホンの音声出力レベルを制限するか否かを設定できます。デフォルト値は ｢無効｣ です。

Sound Test

開発機本体、もしくはテレビからテスト音を再生します。

Output Device

現在の音声出力先とチャンネル数を表示します。

Sleep Settings

Sleep Mode, Idle Time Detectionは DevMenuCommand の ｢ スリープモードの制御 ｣ , ｢無操作状態検出モード｣ と同等の機能です。

Sleep Mode

システムスリープの動作モードを変更できます。スリープモードのデフォルト値は ｢Default｣ です。

  • スリープモードが「Default」の場合は、製品と同様に動作します
    • システム全体がスリープするため、ホストPCとターゲットの間の接続は失われます
  • スリープモードが「Pseudo」の場合は、電源ボタンを押す等をしてもシステムは実際にはスリープせず、1秒程度でスリープから復帰します
    • スリープの前後でホストPCとターゲットの接続を維持することができます
    • スリープ通知等のアプリケーションに対するインタフェースは製品と同様に動作するため、スリープをまたいでアプリケーションをデバッグするのに有用です

コマンド実行後のスリープから変更後の設定が有効になります。

Idle Time Detection

開発用に、無操作状態を検出した際に働く自動スリープや画面焼け軽減機能の有効／無効を変更できます。
本機能は現時点ではデフォルトで｢Off｣となっています。製品では常時有効です。
エージングテスト等の用途では、このコマンドを使って自動スリープを無効に設定する必要があります。

注意：

画面焼け軽減を無効にした状態で、1フレームごとに RGB 値の変化を繰り返すような描画パターンを長時間継続すると、画面に残像やチラツキが現れる場合があります。
発生した場合は、1フレームごとの変化の繰り返しを止めることで徐々に回復します。

変更後の設定を有効にするには、開発機の再起動が必要です。
