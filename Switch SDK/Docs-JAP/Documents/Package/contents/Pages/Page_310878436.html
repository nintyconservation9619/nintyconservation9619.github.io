ネットワーク

  • 概要
  • 使い方
    • Submit network request
    • Testing mode enabled

概要

ネットワーク接続確認やテスト機能の設定を行えます。ネットワーク接続を確立するには、予めネットワーク設定を行う必要があります。
設定方法は ネットワーク接続設定の登録方法 で説明されています。

使い方

Submit network request

ネットワーク接続を試行します。ネットワークを利用可能かどうかは、以下に従ってボタン右側に表示されます。

状態 表示
ネットワーク利用 可 Available
ネットワーク通信 不可 Not available (<Result値>, <エラーコード> )

Result値はGetInnerValueForDebug()を16進数で表記、エラーコードはxxxx-yyyyのフォーマットにて8桁の数字で表記されます。

Testing mode enabled

無線通信および有線通信をON, OFFするテスト機能を利用するか否かを設定できます。詳しくは nifm ライブラリの API リファレンス を参照してください。

  • nn::nifm::SetWirelessCommunicationEnabledForTest()
  • nn::nifm::SetEthernetCommunicationEnabledForTest()
