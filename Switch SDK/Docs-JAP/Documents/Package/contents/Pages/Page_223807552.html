2 Maya シーンファイルの用意

モデルとアニメーションをインポートする手順を説明します。

このチュートリアルではエクスポートする Maya シーンを用意するために既存の fmdb を Maya にインポートします。通常は Maya のシーンファイルを開きます。
パッケージ内にあるサンプルの各ファイル ( モデル、スケルタルアニメーション ) を読み込みます。 

Nintendo Import

Nintendo Import で モデル（fmdb）とスケルタルアニメーション（fskb）を Maya にインポートします。

Nintendo Import ウインドウを開く

Maya の「メニューバー」 → 「Nintendo」→ 「Import...」を開きます。

Nintendo Import ウインドウを設定して読み込む

各項目を設定して読み込みます。

下図と表の順番に操作して読み込みます。

順番 カテゴリ 操作
1 General Options

Process Mode を「Import」にします。

Magnify を「1.0000」にします。Create Texture File のチェックをオンにします。
2 Import Options fmd File

フォルダーアイコンをクリックして以下の「. fmdb」ファイルを指定します。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\SampleTripod.fmdb
3 Import Options fsk File

フォルダーアイコンをクリックして「. fskb」ファイルを指定します。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\ExportSampleTripod.fskb
4 Import Options Import All

「Import All」をクリックします。設定したファイル全てを Maya にインポートします。
参考：

上図 A「Import fmd」や上図 B「Import fsk」をクリックして、個別のファイルをインポートすることもできます。

インポートに成功すると下図のモデルが表示されます。Nindendo Import ウインドウを閉じます。

テクスチャを表示、アニメーションを再生すると以下の図のようになります。
待機、右移動、左移動のアニメーションが連結しています。
