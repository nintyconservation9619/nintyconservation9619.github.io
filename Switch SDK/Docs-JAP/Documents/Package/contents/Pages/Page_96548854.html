磁力（±）

ここでは、磁力の力を設定する機能「磁力（±）」について説明します。

概要

「磁力（±）」は、磁力の大きさを設定できる機能です。
値を大きくすることで、パーティクルが磁力の位置に引かれる力を大きくできます。

以下は「 設定項目」の概要です。

磁力（±）

磁力の大きさを±で設定します。 マイナス方向の値を指定することでパーティクルは磁力に対して反発します。
磁力の「位置」から遠いパーティクルほど、軸方向へ向かう速度は速くなります。

参考：

「 パーティクル時間アニメ」ボタンを押すことにより「 磁力(パーティクル時間アニメ)」が表示さ、カーブエディタでアニメーションを設定できます。
アニメーションは「 CPUで動作させる」ON で有効にできます。

設定例

以下は「磁力」の値を変えた場合の比較画像です。
「● 磁力の位置」はエミッタの中心部と同じ位置にあります。

A 0.025 パーティクルが磁力の位置に引かれ、ばねのように加減速します。
B 0.1

A と比較すると、パーティクルが磁力の位置に向かう速度が速くなります。
C -0.01 A と比較すると、パーティクルが磁力の位置から反発して離れていきます。
