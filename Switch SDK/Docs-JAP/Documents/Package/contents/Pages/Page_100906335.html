概要

  • NintendoWare AtkPlayer (以下、AtkPlayer) は NintendoWare SoundMaker (以下、SoundMaker) で作成したデータを、プレビュー再生するアプリケーションツールです。
    • SoundMaker については、「SoundMaker マニュアル」をご覧ください。

起動方法

AtkPlayer を起動する方法について説明します。

SoundMaker から起動

  • SoundMaker の[プレビュー] - [PC AtkPlayer の起動] メニューを選択
  • または、ツールバーの [PC AtkPlayer の起動] ボタンをクリックします。
  • [PC AtkPlayer の起動] を実行すると次の順で処理されます。
    1. サウンドデータをコンバート
    2. AtkPlayer を起動

SoundArchivePlayer パネル

サウンドアーカイブファイル (.bfsar) に含まれる任意のサウンドを選択して再生するパネルです。(ディスプレイによって選択項目が見えづらいことがありますが、今後改善される予定です。)

各行には以下の情報が表示されます。

項目 再生状態 番号 名前
説明

[PLAY]　再生

[STOP]　停止

[PAUS]　一時停止

 選択しているサウンド ID

選択しているサウンドの名前

他のパネルにフォーカスがある場合、選択中のサウンドハンドルに対して R + A で再生、R + B で停止することができます。

メイン出力が指す出力先は、ターゲットプラットフォームによって変化します。

SoundSystem パネル

メイン出力への出力モードとして SURROUND, STEREO, MONO を選択するパネルです。

Effect パネル

Aux バスにエフェクトを適用するパネルです。

上段に表示されるバス毎のエフェクト設定をエフェクトリストと呼び、それぞれのバスに対して適用するエフェクトの種類とその順番を選択することができます。上の画像を例にすると、AuxB では Delay エフェクトを適用した後、Reverb エフェクトが適用されます。表示されるエフェクトの名前と種類の対応は以下の表によってまとめられます。

エフェクトの名前 エフェクトの種類
None

エフェクトを適用しません
Reverb

nn::atk::EffectReverb エフェクトを適用します
Delay

nn::atk::EffectDelay エフェクトを適用します
I3dl2

nn::atk::EffectI3dl2Reverb エフェクトを適用します

下段に表示されるエフェクトのパラメータ一覧をパラメータリストと呼び、エフェクトリストで選択したエフェクトのパラメータを調整することができます。また、エフェクトリストとパラメータリストの間に表示される Memory は、現在適用しているエフェクトのワークバッファの大きさの合計を表しています。

SoundOutput パネル

サウンドの出力波形をリアルタイムで表示するパネルです。

6 チャンネル分の出力が表示され、それぞれ上から FL, FR, FC, LFE, RL, RR が表示されます。

最下部の表示は以下のような設定を表しています。

設定 意味
[Play]

出力波形のモニタリングを行っています。
[Stop] 出力波形のモニタリングを停止しています。
Time 出力波形の表示時間を表しています。
Amp 出力波形の振幅の表示倍率を表しています。

 

サウンドアーカイブファイルのリロード

AtkPlayer を起動した後に SoundMaker でサウンドデータを編集したとき、その編集内容は自動的に AtkPlayer に反映されません。

反映させるためにはサウンドアーカイブファイルのリロードを行う必要があります。

 

サウンドデータを編集した後に SoundMaker からサウンドアーカイブファイルのコンバートを行います。

 

コンバートが完了した後に AtkPlayer で R + X ボタンを押すとリロードすることができます。

このとき、AtkPlayer の画面右上にある Edit の横に Reloaded の文字が表示されます。

インゲーム編集機能

AtkPlayer は SoundMaker のインゲーム編集機能を利用することができます。

SoundMaker のインゲーム編集機能については、「インゲーム編集機能」をご覧ください。

SoundMaker との接続

AtkPlayer を起動した後に SoundMaker から接続します。

接続が確立すると AtkPlayer の画面右上にある Edit の色が変化します。

接続が確立されている間は、SoundMaker からインゲーム編集を行うことができます。

SoundMaker との切断

インゲーム編集を終了させるには SoundMaker から切断します。
また、再びインゲーム編集を開始するときは SoundMaker から接続します。

サウンドの再生情報の可視化機能

AtkPlayer は Spy.exe (PC ツール) と接続することで、オーディオレンダラの処理負荷や使用しているボイス数などのサウンドの再生情報を可視化することができます。

この機能を利用する場合は AtkPlayer を起動した後に Spy.exe から接続を行ってください。Spy.exe の使い方やサウンドの再生情報に関する詳しい説明は「Spy.exe」をご覧ください。

注意事項

Spy.exe は接続した後のサウンドの変更に対応できません。そのため、インゲーム編集やサウンドアーカイブファイルのリロードを行うと表示される情報に齟齬が生まれることがあります。その場合は AtkPlayer と Spy.exe の接続を切り、再接続することで情報の齟齬が解消されます。

トラブルシューティング

AtkPlayer を起動したときに「SoundSystem の初期化に失敗した」というエラーが発生した

  • nn::atk::SoundSystem の初期化に失敗したときにこのエラーが表示されます。オーディオ再生環境が整っているかどうかを確認してください。例えば、スピーカーやヘッドホンなどのオーディオ再生機器が 1 つも繋がっていない場合はこのエラーが表示されます。
