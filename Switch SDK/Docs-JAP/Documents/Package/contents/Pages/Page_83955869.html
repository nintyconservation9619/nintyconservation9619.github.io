はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている オペレーティングシステム（以後 OS）のライブラリ機能について説明します。


対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象としています。


各章の説明

「マニュアル」では、個々の OS ライブラリ機能を使用する上で理解しておかなければならない特筆事項を説明しています。

「サンプルプログラム」では、具体的なサンプルコードに基づき、個々の OS ライブラリ機能の使い方を説明しています。


関連資料

OS の関数リファレンス と併せて参照して下さい。


OS ライブラリの機能

OS ライブラリは、ソフトウェアの実行を制御する上で有用なマルチスレッド制御機構を提供します。

マルチスレッドとは、複数のソフトウェアが互いに独立した処理を同時実行するための機構です。スレッドを管理するための機能のほか、複数スレッドからの資源要求に対して競合を防ぐための排他機構や、複数スレッドが互いに同期制御したりする機能を提供します。

OS ライブラリが提供する機能

OS ライブラリでは以下の機能を提供しています。

  • スレッド
  • スレッド固有領域
  • ファイバ
  • セマフォ
  • イベント
  • 軽量イベント
  • システムイベント
  • メッセージキュー
  • タイマーイベント
  • 多重待ち
  • ミューテックス
  • 条件変数
  • バリア
  • Readers-Writer ロック
  • 時間管理機能
  • メモリ管理機能
  • ユーザ例外ハンドラ

それぞれの機能については、マニュアル を参照して下さい。
また、具体的な使用方法については、サンプルプログラム を参照して下さい。
