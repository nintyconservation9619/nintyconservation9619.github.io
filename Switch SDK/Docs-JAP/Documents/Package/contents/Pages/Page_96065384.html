設定項目のUI

設定項目にはその機能専用の UI があります。
また、テクスチャやプリミティブを登録する設定項目では、登録したファイルの情報を表示する UI もあります。

ここではエフェクト各設定項目専用の UI について説明します。

値設定専用のUI

EffectMaker の機能に値を入力する UI です。以下の UI が集まり、構成されています。

「 ダイアル」、「  テキストボックス」、「 スピナボタン」、「 アジャストボタン」
ランダム設定
上位設定項目の値に対して乱数値を設定する UI です。
プリミティブのアトリビュート情報
「エミッタ形状」や「パーティクル形状」に登録したプリミティブの詳細な情報を表示する UI です。
テクスチャプレビュー
「テクスチャ」タブで登録したテクスチャファイルのプレビューができる UI です。
パターンテーブル
「テクスチャパターンアニメ」で使用するパターン数とパターンアニメーションの順番を設定できる UI です。
カラー編集のUI
選択しているカラーやアルファの値を編集できる UI です。
グラデーションエディタ

選択しているカラーやアルファにグラデーションでパーティクル時間のアニメーションを編集できる UI です。
ランダムカラー設定のUI
  複数のカラーを登録できる UI です。登録したカラーが放出時のパーティクルカラーとしてランダムに適用されます。
グローバルカラーアニメーションのUI
  グローバルカラーやアルファに列方向を時間、行方向を値としてアニメーションを作成及び編集できる UI です。
カラーの入れ替えボタン
  「カラー 0」と「カラー 1」 または「アルファ　0」と「アルファ 1」で各々挙動タイプと設定内容を入れ替える UI です。
パーティクル時間のカーブエディタ
  横軸を時間、縦軸を値としたグラフでパーティクル時間のアニメーションを編集できる UI です。
