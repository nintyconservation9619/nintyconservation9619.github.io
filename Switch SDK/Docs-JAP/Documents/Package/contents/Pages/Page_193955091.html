フィールド情報の表示

ここではビューアにフィールドやエミッタプラグインノードの情報を表示する機能「フィールド情報の表示」について説明します。

概要

「フィールド情報の表示」は、フィールドとエミッタプラグインノードの情報を表示するデバッグ機能です。
また、フィールドの設定内容が UI 操作に連動して表示されます。
以下は「 設定項目」の概要です。

フィールド情報の表示

「フィールド情報の表示」を「 チェックボックス」ON/OFF で切り替えます。

設定例

以下は「フィールド情報の表示」ONで表示される情報です。

 磁力 フィールド「磁力」を使用したとき、磁力の位置を表示します。
スピン

フィールド「スピン」を使用したとき、回転軸と回転方向を表示します。
収束 フィールド「収束」を使用したとき、収束位置を表示します。
シンプルコリジョン フィールド「シンプルコリジョン」を使用したとき、コリジョン面を表示します。
範囲内ループ エミッタプラグインノード「範囲内ループ」を使用したとき、パーティクルの表示を限定する「箱」を表示します。
