アニメーションコントロール

アニメーションコントロールについて説明します。

関連リンク アニメーションは、ショートカットキーでもコントロールできます。

 

A タイムスライダです。
タイムスライダ上でマウスを左クリックすると、マウス位置のフレームがカレントフレームになります。
マウスを左ドラッグすると、カレントフレームの位置をマウス移動できます。 
B カレントフレームのテキストボックスです。
値を入力すると、タイムスライダ上のカレントフレームが入力した値に移動します。 
C アニメーションの再生方法を設定します。
D アニメーションの再生をコントロールします。
E

1/60 秒間に何フレーム進めるかを設定します。
例： 0.5 にすると半分の速度で再生し、10 であれば10倍速で再生されます。

アニメーション再生方法の種類

再生方法の種類は下記の 3 つになります。

項目 説明
Auto 読み込まれている全モデルのカレントアニメーションセットに含まれているアニメーションの中で、loopがひとつでもあればループ再生になります。
Loop 強制的にアニメーションをループ再生します。
Once 強制的にアニメーションを一回のみの再生にします。

アニメーションコントロールアイコン

アニメーションをコントロールするアイコンの説明です。

アイコン 説明
停止ボタンです。
コマ戻しボタンです。アニメーションが一時停止状態になり、1フレーム戻ります。
再生 / 一時停止ボタンです。
コマ送りボタンです。アニメーションが一時停止状態になり、1フレーム進みます。
