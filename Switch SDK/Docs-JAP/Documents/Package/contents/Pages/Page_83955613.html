波形ファイル出力機能

プロジェクト内にある各種サウンドを、波形ファイルとして出力することができます。

出力可能なサウンドは、以下の種類です。

  • ストリームサウンド
  • ウェーブサウンド
  • シーケンスサウンド

波形ファイルの出力

サウンドセットタブの各種リストから任意のサウンドを選択 して、[プレビュー]→[波形ファイルの出力] を実行します。
「波形ファイルの出力」ダイアログが表示されるので、出力先のファイル名などを指定して、「OK」ボタンをクリックすることで波形ファイルが出力されます。

最大出力時間： 出力する時間の上限を秒数で指定します。
出力チャンネル数： ステレオかモノラルのどちらかを選択できます。

リスト上でのコンテキストメニューの [波形ファイルの出力] からも同様の操作が行えます。

出力するファイルは、複数選択が可能です。

 

波形ファイルは、16bit / 48000Hz  の wav ファイルとして出力されます。
