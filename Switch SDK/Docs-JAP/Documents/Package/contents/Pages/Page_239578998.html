四角形

ピクチャペイン形状「四角形」のプロパティウインドウ

ピクチャペインの形状を「四角形」とした場合のプロパティウインドウ説明です。

 

形状「四角形」のピクチャペイン

頂点カラー

ペインの頂点カラーについて設定します。

頂点カラーを入れる位置を設定する、ペインを簡略化した図形です。

図形中のカラーを指定したい位置をマウス左クリックすると、カラーピッカーが開きます。

カラーピッカーにて頂点カラーを指定します。
すべてを白に設定

編集中のペインの頂点カラー全てに、白 ( RGB表記の場合、R : 255  G : 255  B : 255  A : 255 ) を設定します。

頂点カラーの初期値が白なので、編集のリセットにも使用できます。

下記の「 ～ を変更する」のチェックが外れている場合、設定されません。
RGBを変更する

チェックを入れると RGB を編集できます。

チェックを外すと、 RGB を編集しても反映されません。
Aを変更する

チェックを入れると A ( アルファ ) を編集できます。

チェックを外すと、 A を編集しても反映されません。

テクスチャ座標

テクスチャの座標を設定します。

使用する座標数 :

使用するテクスチャの座標の数を入力します。

プロパティウインドウからテクスチャを追加すると、増やした枚数分「使用する座標数」を増やすことができます。

ここで「使用する座標数」を増やしてから、

[ プロパティウインドウ ] → [ ピクチャペイン ] → [ テクスチャ ] → [ マッピング設定 ] → [ マッピング方法 : ]

を使用して、それぞれのテクスチャがどの座標を使用するのかを設定できます。
編集する座標 :

編集する座標を切り替えます。

複数のテクスチャそれぞれに座標を登録して使用する場合、編集する座標の切り替えをここで行います。
座標を使っているテクスチャ :

座標を使っているテクスチャの画像が表示されます。

上記「編集する座標 : 」の切り替えに同期しています。
一つの座標に複数のテクスチャが設定されている場合は、複数のテクスチャが表示されます。

ペインの全体に対するテクスチャの各頂点の座標を、ピクセル値で設定します。
初期値はテクスチャのサイズにより変化します。

 

一つのテクスチャ座標に大きさの違うテクスチャを複数枚登録している場合、この項目は使用不可となります。

 
テクスチャの座標値を左右反転します。
テクスチャの座標値を上下反転します。
テクスチャの座標値を、時計回りに90度回転します。
