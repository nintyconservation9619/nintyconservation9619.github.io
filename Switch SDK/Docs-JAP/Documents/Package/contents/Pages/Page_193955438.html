EffectMaker

デバッグ表示機能の追加

ビューアノードのデバッグ表示に以下の情報が追加されました！

GPU計測結果の表示

「GPU計測結果の表示」は、エミッタ個々の GPU処理負荷を表示する機能です。
エフェクトの GPU 負荷について、より詳しく知りたいときに使用します。

「GpuCost」は、GPU処理負荷を 全エミッタ分表示します。
以下の情報を表示します。

-Root ビューア全体のGPU処理コストを msec で表示します。
-VfxGpu エフェクト全体のGPU処理コストを msec で表示します。
-Compute コンピュートシェーダのGPU処理コストを msec で表示します。
-EmitterSet XXXX 再生中エミッタセットのGPU処理コストを msec で表示します。
-Emitter XXXX 再生中 各エミッタ のGPU処理コストを msec で表示します。
参考：

EffectMaker のワークスペースで「 表示ボタン」を「 非表示」に切り替えたエミッタの情報は表示されません。

「GpuStatistics」は、GPU内部のカウンタを表示します。

こちらは各エミッタ単位ではなく、エフェクト描画全体のカウンタ値になります。
以下の情報を表示します。

|- InVert

入力頂点の数です。
|- InPrim 入力プリミティブの数です。
|- VtxShaderInvoke 頂点シェーダ呼び出しの数です。
|- ClipInPrim クリッピングステージによって処理されるプリミティブの数です。
|- ClipOutPrim クリッピングステージによって生成されるプリミティブの数です。
|- PixShaderInvoke ピクセルシェーダ呼び出しの数です。
|- SamplePassed フラグメントのテストを通過するサンプルの数です。
参考：

「GpuStatistics」は、プラットフォームが「Windows」のときは表示されません。
関連リンク 「プラットフォーム」については こちら を参照してください。

フィールド情報の表示

「フィールド情報の表示」は、ビューアにフィールドやエミッタプラグインノードの情報を表示する機能です。
以下は「フィールド情報の表示」ONで表示される情報です。

磁力 フィールド「磁力」を使用したとき、磁力の位置を表示します。
スピン

フィールド「スピン」を使用したとき、回転軸と回転方向を表示します。
収束 フィールド「収束」を使用したとき、収束位置を表示します。
シンプルコリジョン フィールド「シンプルコリジョン」を使用したとき、コリジョン面を表示します。
範囲内ループ エミッタプラグインノード「範囲内ループ」を使用したとき、パーティクルの表示を限定する「箱」を表示します。
