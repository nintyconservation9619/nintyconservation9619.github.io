呼び出し開始フレーム

ここではエフェクトを呼び出すタイミングを指定する機能「呼び出し開始フレーム」について説明します。

概要

「呼び出し開始フレーム」は、エフェクトを呼び出すタイミングを指定できる機能です。
ゲーム中で複数のエフェクトデータを組合わせた状態をビューアで確認できます。
以下は「 設定項目」の概要です。

呼び出し開始フレーム

エフェクトを呼び出すタイミングをフレーム数で設定できます。
