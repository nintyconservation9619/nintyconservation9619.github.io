カスタムアトリビュート変換プラグイン

NintendoWare for Revolution (NW4R) 用のカスタムアトリビュートを現在の NintendoWare 用のカスタムアトリビュートに変換するプラグインです。

使用方法

「Nintendo > Convert Old Attribute...」を選択するとウィンドウが表示されます。

使用方法

Search ボタンをクリックすると、NW4R 用のカスタムアトリビュートが設定されたシーン中のノードの一覧が表示されます。

ただし、次のような場合は表示されません。

  • カスタムアトリビュートの値がデフォルト値の場合
  • すでに現在の NintendoWare 用のカスタムアトリビュートが設定されている場合
  • 対応する現在の NintendoWare 用のカスタムアトリビュートがない場合

Apply ボタンをクリックすると、変換が実行されます。
変換後も NW4R 用のカスタムアトリビュートは削除されずに残ります。

以下の NW4R 用のカスタムアトリビュートが変換対象となります。

  • NW4R Set Material Attribute の Compress
  • No Compress Node
  • Frame Extension List
  • NW4R Set Shape Anim Attribute の Position、Normal、Color
