フィルタモード

ここではテクスチャのピクセルにかける処理を設定する機能「フィルタモード」について説明します。

概要

「フィルタモード」は、パーティクルに貼られたテクスチャをビューアで表示する際のピクセルに対してかける処理です。
ポリゴンの表示サイズにあわせて対応するテクセルを決める際の処理方法を変更し、ビューアでの見え方を制御できます。

参考：

「ディスタンスフィールド」による表現を利用する際は 「 Linear 」を使用してください。

以下は「 設定項目」の概要です。

フィルタモード

テクスチャのフィルタ方式を、以下のプリセットから選択します。

Linear 補間によりピクセルのエッジ部分を滑らかに表示します。
Near ピクセル 1 つ 1 つをくっきりと表示します。

設定例

以下は、EffectViewer で実際のテクスチャ解像度より拡大して表示したときの「フィルタモード」各プリセットの比較画像です。

Linear バイリニア法 テクスチャの画素の平均値をとった補間処理を行います。
低解像度（Low Resolution）でもきれいに見えます。
Near ニアレストネイバー法

テクスチャの画素をそのまま拡大します。
低解像度（Low Resolution）ではジャギーが目立ちます。
