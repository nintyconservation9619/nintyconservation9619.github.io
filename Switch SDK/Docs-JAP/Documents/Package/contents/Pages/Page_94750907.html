エミッタ速度継承率

ここではエミッタの移動速度をパーティクル放出時の初速として継承する機能「エミッタ速度継承率」について説明します。

概要

「エミッタ速度継承率」は、エミッタが移動アニメーションする場合、パーティクル放出時点の速度をパーティクルに継承できる機能です。
パーティクル追従タイプが「エミッタに追従しない」設定でエミッタが高速移動する場合にパーティクル同士の隙間が空き、エフェクトとしての一体感がなくなる場合があります。
この機能を使用することで、パーティクルごとの隙間を減らし、ひとかたまりのエフェクトに見せることができます。

参考：

継承速度は前フレームからの移動値から算出される為、エミッタ時間で 1 フレーム目に継承する速度は必ず 0 になります。
以下はエミッタが移動アニメーションしており、毎フレームパーティクルを放出した場合の図です。

エミッタ時間での移動アニメーション（フレーム） 0 1 2 3 4 ...
エミッタの移動速度 0 10 10 10 10 ...
継承する速度（倍率） x 0 x 0.1 x 0.1 x 0.1 x 0.1 ...
パーティクルの初速に加算する速度 0 1 1 1 1 ...

以下は「 設定項目」の概要です。

エミッタ速度継承率

継承される速度は、パーティクル放出時の初速に加算されます。
値はその時のエミッタの移動速度を 1 とした割合で指定します。

設定例

以下は「エミッタ速度継承率」の設定を変更した場合の比較画像です。
エミッタは画面左から右に向かって移動します。
パーティクルは「■ 放出マーク」のタイミングでエミッタから放出されます。

A 0（デフォルト） パーティクルはエミッタ速度の影響を受けない為、放出した場所から動きません。
