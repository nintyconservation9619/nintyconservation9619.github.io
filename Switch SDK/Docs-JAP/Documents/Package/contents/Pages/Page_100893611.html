テクスチャパッケージャ

概要

テクスチャパッケージャは、テクスチャイメージをハードウェア依存のテクスチャパッケージ形式に変換します。

提供形態

.dll, .exe の形式で提供されます。

配置

Windows 環境向け

$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/DdsLoader.dll
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/DdsLoader32.dll
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/NvnImage.dll
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/NvnImage32.dll
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/NvnTexpkg.dll
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/NvnTexpkg.exe
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/NvnTexpkg32.dll
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/NvnTexpkg32.exe
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/TgaLoader.dll
$NINTENDO_SDK_ROOT/Tools/Graphics/NvnTools/TgaLoader32.dll

ヘッダファイル

nvnTool/nvnTool_TexpkgConsumer.h
nvnTool/nvnTool_TexpkgProducerCore.h
nvnTool/nvnTool_TexpkgProducerImage.h 

$NINTENDO_SDK_ROOT/Include/nvnTool

マニュアル

テクスチャパッケージャの詳細については、テクスチャパッケージャマニュアル を参照してください。
