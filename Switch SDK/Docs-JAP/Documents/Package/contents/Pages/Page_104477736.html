XZ 移動速度

ここでは選択しているノードが XZ 平面で移動する時の速度を設定する機能「XZ 移動速度」について説明します。

概要

「XZ 移動速度」は、選択しているノードの XZ 平面円状ルート上の移動速度を設定する機能です。
値を大きくすることによって、移動速度を速くできます。
値が 0 の場合は、自動移動タイプが「無し」と同じになります。
以下は「 設定項目」の概要です。

XZ 移動速度

円状ルートの移動速度を「 ダイヤル」で設定します。

設定例
