エミッタ形状「球（ボリューム）」

ここではパーティクルの放出位置が球体のエミッタ形状「球（ボリューム）」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「球（ボリューム）」は、「球」エミッタ全体から球状に広がった法線方向にパーティクルを放出します。
以下は「 設定項目」の概要です。

「エミッタ形状」を「球（ボリューム）」に設定すると「 球（ボリューム）」が表示されます。以下はグループ内の設定項目です。

空洞率

パーティクルが放出されない領域を、エミッタ中心からの割合で設定します。
