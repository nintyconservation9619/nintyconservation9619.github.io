開始処理

パーティクルの放出開始時の挙動を決める機能が追加されました！
エフェクト出現時にアルファやスケールで徐々に表示させることで、パーティクルが突然表示されることによる違和感を軽減できます。

使用例

「 アニメーションを停止ボタン」を押すことで一旦パーティクルの放出を停止させます。
「 アニメーションを再生ボタン」を押して、パーティクルの放出を開始させることで、「開始処理」の効果を確認できます。
以下は「開始処理」の各機能を ON/OFF した場合の比較画像です。

A フェード OFF フェード処理なしの状態です。
B アルファフェードON 「フェードイン時間」で指定した時間をかけて透明な状態から徐々に出現します。
C スケールフェードON 「フェードイン時間」で指定した時間をかけてサイズ 0 の状態から徐々に出現します。

エフェクトブラウザからパッケージのエフェクトサンプルが参照可能に

エフェクトブラウザからエフェクトのサンプルデータを読み込むことができるようになりました！
以下は読み込み手順です。

1 タブの選択 「 エフェクトブラウザ」ウィンドウで「お気に入り」タブを選択します。
2 「お気に入り」タブの選択 「お気に入り」タブを選択します。
3 「 VfxSample」の選択 お気に入り「 VfxSample」を選択します。リストにサンプルデータが表示されます。
4 サンプルデータの選択 EffectMaker に読み込みたいサンプルデータ（.eset や .prev)ファイルを選択します。
5 ワークスペースへエクスポート ツールバー「 ワークスペースへエクスポート」ボタンをマウス左ボタンクリックします。
読み込みたいファイル上でマウス左ボタンをダブルクリックしても、読み込めます。
参考：

「 VfxSample」は「 エフェクトブラウザ」ツールバーの「 サンプルフォルダをお気に入りに追加」ボタンをマウス左ボタンクリックすることでお気に入りとして追加できます。
