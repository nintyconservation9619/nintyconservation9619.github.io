メインウィンドウ

メインウィンドウ

EffectMaker のウィンドウを構成する、各部位の名称と役割について説明します。
また、各ウィンドウの配置を決める「ドッキング」についても説明しています。
  • A メニューバー
  • B ツールバー
  • C ドッキングエリア
  • D メッセージビュー  

A メニューバー

以下の項目が並びます。項目の上でマウス左クリックによりメニューが表示されます。

ファイル エフェクトの作成や、ファイルの入出力に関する操作を行います。
編集
編集作業「やり直し」「元に戻す」やノード操作を行います。
表示 各ウィンドウ表示に関する操作を行います。
プレビュー
「 ビューアへの接続」に関する操作を行います。
ツール
 EffectMaker の「 オプション」に関する操作を行います。
ヘルプ
 EffectMaker の「 へルプ」や情報を表示します。


B ツールバー

メニュー項目で、使用頻度が多いものをボタンとして表示します。以下はツールバーの説明です。

 開く [ファイル] → [開く] ファイルを開きます。
全て上書き保存 [ファイル] → [全て上書き保存]
すべてのファイルを上書き保存します。
「保存」されたファイルは、プロパティウィンドウ上で編集したことを示す「  編集済みマーク」が消えます。
全てエクスポート [ファイル] → [全てエクスポート]
表示されるダイアログで選択したファイルを指定した場所に出力します。
「保存」と異なる為、出力されたファイルは、プロパティウィンドウ上の「  編集済みマーク」が消えません。
元に戻す [編集] → [元に戻す] 編集前の状態に内容を戻します。
やり直し [編集] → [やり直し] 元に戻した編集をやり直します。
プロパティウィンドウ [表示] → [プロパティウィンドウ] ノードの操作やプレビューができる UI や、ノードの機能設定ができる UI から構成される「プロパティウィンドウ」を表示します。
エフェクトブラウザ [表示] → [エフェクトブラウザ] EffectMaker で使用するファイルに特化した閲覧や検索機能が利用できる「エフェクトブラウザ」を表示します。
カーブエディタウィンドウ [表示] → [カーブエディタ] 横軸を時間、縦軸を値としたグラフでエミッタ時間のアニメーションを作成及び編集できる「カーブエディタ」を表示します。
履歴ウィンドウ [表示] → [履歴] 作業の手順を「履歴」とし一覧表示する「履歴ウィンドウ」を表示します。
検索ウィンドウ [表示] → [検索ウィンドウ] 指定したフォルダのエフェクトファイルから、使用している機能を検索し、設定を一括変換できる「検索ウィンドウ」を表示します。
プラットフォーム - 開発を行っているプラットフォームを選択します。
切断状態→接続 [プレビュー] → [ビューアに接続] ビューアとの通信を開始します。
接続状態→切断 [プレビュー] → [ビューアから切断] ビューアとの通信を終了します。
接続先ビューア - プラットフォームで利用可能なビューアが、プリセットとして表示されますので、接続するビューアを選択します。
ビューアを起動

[プレビュー] → [PCビューアを起動]

[プレビュー] → [実機ビューアを起動]

 「接続先プラットフォーム」と「接続先ビューア」の設定内容でビューアが起動します。
所定の場所に設定したビューアが存在しない場合は選択できません。

サムネールを撮影 [プレビュー] → [サムネールを撮影]
ビューアに表示されているエミッタセットファイル、モデルファイルのサムネールを撮影します。

撮影した画像は、エミッタセットファイルの保存先フォルダに「.NwThumbs」フォルダが作られ、.png ファイルとして保存されます。
保存された .png ファイルは、「エフェクトブラウザ」でファイルのサムネールとして表示されます。

関連リンク 以下は関連ページへのリンクです。

サムネール・プレビュー表示機能モデル中間ファイル(.fmdb/.fmda)、レイアウトファイル(.flyt)、エミッタセット(.eset) のサムネール表示機能について説明します。
エフェクトブラウザEffectMaker で使用するファイルに特化した閲覧や検索機能が利用できる「エフェクトブラウザ」について説明します。
ワークスペースを開く [ファイル] → [ワークスペースを開く] ワークスペースファイルを開きます。
ワークスペースを別名で保存 [ファイル] → [ワークスペースを別名で保存] ワークスペースファイルに名前を付けて保存します。
ワークスペースを上書き保存 [ファイル] → [ワークスペースを上書き保存] ワークスペースファイルを上書き保存します。
ヘルプを開く -

EffectMaker に関するヘルプを以下のプリセットから選択して開きます。

  • メインウィンドウのヘルプを開きます。
  • プロパティウィンドウのヘルプを開きます。
  • 編集中タブ項目のヘルプを開きます。

C ドッキングエリア

「 プロパティウィンドウ」、「 エフェクトブラウザ」、「 カーブエディタウィンドウ」、「 履歴ウィンドウ」
これらのウィンドウは、このエリアにまとめて表示させたり、取り外して独立したウィンドウにできます。
作業内容にあわせて好みの配置にできます。

ドッキングエリア内で配置できます。
ウィンドウをマウス右ボタンクリックで選択すると、ドッキングエリアにアイコンが表示されます。
配置箇所を表す以下のアイコン上までウィンドウを移動させ、ドロップすることで配置できます。

 ドッキングエリアの上辺にスナップするように配置されます。
ドッキングエリアの右辺にスナップするように配置されます。
ドッキングエリアの下辺にスナップするように配置されます。
ドッキングエリアの左辺にスナップするように配置されます。

ドッキングエリア内にウィンドウがおさまるように配置されます。

ドッキングさせたいウィンドウ内で配置できます。
ウィンドウを選択し、そのままドッキングさせたいウィンドウまで移動すると「 アイコン」が表示されます。
配置箇所を表す以下のアイコン上にウィンドウをドロップすることで配置できます。

 ドッキングさせたいウィンドウの上に配置されます。
ドッキングさせたいウィンドウの右に配置されます。
ドッキングさせたいウィンドウの下に配置されます。
ドッキングさせたいウィンドウの左に配置されます。

ウィンドウ同士が重ねて配置されます。
表示させたいウィンドウを「 タブ」で切り替えます。

D メッセージビュー

特定のエラーメッセージや警告メッセージは、このビューに出力します。
マウスの右クリックによりメニューが開きます。以下はメニュー項目です。

全て選択 メッセージをすべて選択します。
コピー 選択したメッセージをコピーします。
全てクリア メッセージビューの内容をすべて削除します。
