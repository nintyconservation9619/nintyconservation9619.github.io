サンプラのテクスチャ情報

マテリアルプロパティウィンドウのサンプラページにある、サンプラリストで選択したサンプラに使用されているテクスチャ情報が表示されます。
A

サンプラで使用するテクスチャの名前です。
B テクスチャ画像の変更ダイアログが開き使用するテクスチャを変更できます。
C サンプラで使用しているテクスチャのフルパスです。
D テクスチャのカラーイメージです。クリックすると詳細画像が表示されます。
E テクスチャのアルファイメージです。クリックすると詳細画像が表示されます。
F テクスチャプロパティウィンドウを開きます。
G 表示深さです。配列テクスチャを選択した場合にアクティブになります。
H DCC ツールのプリセットを表示します。
I テクスチャのプリセットフォーマットを表示します。
J 選択されているテクスチャフォーマットを表示します。
K テクスチャのミップマップを生成する際のフィルタの種類を表示します。
L テクスチャをサンプリングする際の、カラーコンポーネントセレクタの状態です。
M テクスチャ圧縮 ( bc1 / bc2 / bc3 ) をする際に、RGB の誤差に対して、人間の目に合わせた重みづけをするか、しないか、の設定状態を表示します。
N 選択しているテクスチャの R G B A チャンネルに対するガンマ補正の状態を表示します。
O テクスチャの次元を表示します。
P テクスチャの幅です。
Q テクスチャの高さです。
R

テクスチャの深さです。
S テクスチャのデータサイズが表示されます ( パディング や、ミップマップを含む ) 。
T

テクスチャのミップマップレベルです。ミップマップが無い場合は 1 です。
U ミップマップにおけるテクスチャの最小サイズです。
V 実機用イメージを作成する際に取得できる、GX2Surface.tileMode です。
W 実機用イメージ作成時に指定するスウィズル値です。
X 実機用イメージを作成する際に取得できる、GX2Surface.swizzle です。
Y 実機用イメージを作成する際に取得できる、GX2Surface.alignment です。
Z 実機用イメージを作成する際に取得できる、GX2Surface.pitch です。
