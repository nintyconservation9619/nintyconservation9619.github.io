接続タイプ

ここでは、ストライプ先端の接続タイプを設定する機能「接続タイプ」について説明します。

概要

「接続タイプ」は、ストライプの先端をストライプの末端やエミッタに接続するかを設定できる機能です。
以下は、1 から順に描画されてたパーティクルを繋げたストライプを例に、先端と末端について説明した図です。

A ストライプの末端です。 B ストライプの先端です。

以下は「 設定項目」の概要です。

接続タイプ

ポリゴン同士の接続タイプを、以下のプリセットから選択します。

通常 通常時のストライプです。ストライプの計算には最低 2 つのパーティクルが必要です。
末端に繋ぐ ストライプの先端と末端をリング状に接続します。リング状に繋げるには最低 3 つのパーティクルが必要です。
エミッタに繋ぐ ストライプの先端をエミッタの中心に接続します。ストライプの計算には最低 2 つのパーティクルが必要です。
煙のような「一か所から連続して発生する」ようなエフェクトを作成できます。

設定例

以下は「接続タイプ」を設定した場合の比較画像です。

A 通常 デフォルト設定です。ストライプの先端は他の場所に接続しません。
B 末端に繋ぐ ストライプの先端と末端が繋がり、リング状のエフェクトになります。
C エミッタに繋ぐ ストライプの先端とエミッタの中心が繋がります。
