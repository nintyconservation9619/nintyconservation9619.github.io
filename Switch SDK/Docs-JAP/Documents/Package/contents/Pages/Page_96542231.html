ノイズの強さ

ここではパーティクルが利用する波形の振幅を設定する機能「 ノイズの強さ 」について説明します。

概要

「 ノイズの強さ」 は、パーティクルが移動の際に利用する波形の振幅を設定する機能です。
以下図 ↕ の範囲を設定します。

大きな値を設定することによって、パーティクルの移動範囲が大きくなります。


以下は「 設定項目」の概要です。

ノイズの強さ

ノイズの強さの値は「 ダイヤル」や「 テキストボックス」により、エミッタを基準とした座標系 X Y Z で設定します。

参考：

「 パーティクル時間アニメ」ボタンを押すことにより「 ノイズの強さ(パーティクル時間アニメ)」が表示さ、カーブエディタでアニメーションを設定できます。
アニメーションは「 CPUで動作させる」ON で有効にできます。
