データ作成プラグイン

データ作成プラグインとは NintendoWare 用のデータ作成を補助するためのプラグイン群です。

主に、Maya 上のノードに対して NintendoWare 用のアトリビュートを設定したり、設定を検索したりするために使用します。
データ作成プラグインは 「Nintendo」メニューから実行します。

データ作成プラグインの種類

データ作成プラグインの一覧を次の表に示します。

プラグイン名 機能内容
マテリアルアトリビュート設定プラグイン
（Set Material Attribute） 		シェーディンググループにマテリアルのアトリビュートを設定します。
サンプラーアトリビュート設定プラグイン
（Set Sampler Attribute） 		file ノードにサンプラーのアトリビュートを設定します。
ノード圧縮不可設定プラグイン
（Set No Compress Node） 		ノードにノード圧縮不可フラグを設定します。
アニメーションのバイナリー出力設定プラグイン
（Set Animation Binarization） 		ノードにアニメーションのバイナリー出力フラグを設定します。
テクスチャーパターンアニメーション設定プラグイン
（Set Texture Pattern） 		file ノードにテクスチャーパターンアニメーション用に強制的に出力するテクスチャーを設定します。
シェイプアニメーション設定プラグイン
（Set Shape Animation） 		blendShape ノードにシェイプアニメーションで更新する頂点属性を設定します。
カメラの画角設定プラグイン
（Set Camera Fovy） 		カメラの垂直方向の画角（Fovy）を設定します。
ライトアトリビュート設定プラグイン
（Set Light Attribute） 		ライトのシェイプノードにアトリビュートを設定します。
ユーザーデータ設定プラグイン
（Set User Data） 		ノードまたはシェーディンググループにユーザーデータを設定します。
カスタムアトリビュート変換プラグイン
（Convert Old Attribute） 		NW4R 用のカスタムアトリビュートを現在の NintendoWare 用のカスタムアトリビュートに変換します。
