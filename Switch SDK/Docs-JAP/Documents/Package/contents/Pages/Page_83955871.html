シェイプアニメーション設定プラグイン

シェイプアニメーションで更新する頂点属性は、デフォルトでは頂点座標、法線、接線、従法線ですが、このプラグインを使用すると更新する頂点属性を指定できます。
ただし、Maya の画面上には反映されません。

使用方法

「Nintendo > Set Shape Animation...」を選択するとウィンドウが表示されます。

Selected Blend Shape 欄には、選択されている blendShape ノードが表示されます。

参考：

4 つ以上の blendShape ノードが選択されている場合は省略表示となり、右端に選択された数が表示されます。

設定方法

blendShape ノードまたはブレンドシェイプを設定したオブジェクトの transform ノードを選択（複数選択可）すると、Set 欄に選択したノードの現在の設定が表示されます。
複数選択の場合、最初に選択したノードの設定が表示されます。

ウィンドウの各項目を操作して設定を変更します。
複数選択の場合、選択したノードを一括して変更できます（変更した項目だけ反映されます）。

ウィンドウを閉じずに別のノードを選択すると、別のノードを設定することができます。

Position 頂点座標を更新するなら ON にします。デフォルトは ON です。
Normal 法線、接線、従法線を更新するなら ON にします。デフォルトは ON です。
接線、従法線はベースシェイプで法線マッピングまたは異方性シェーダを使用している場合のみ更新されます。
Color 頂点カラーを更新するなら ON にします。デフォルトは OFF です。
ただし、ベースシェイプに頂点カラーが設定されていない場合は ON にしても更新されません。
参考：

Position、Normal、Color がすべて OFF の場合、選択した blendShape ノードによるシェイプアニメーションは出力されません。
