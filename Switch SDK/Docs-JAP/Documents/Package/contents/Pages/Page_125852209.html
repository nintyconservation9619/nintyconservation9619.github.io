5 マテリアルアニメーションの新規作成

カラーアニメーションを作成する手順を説明します。
ディフューズカラー RGB のアニメーションで SampleTripod の目を点滅させます。

このチュートリアルでは以下の UI について触れています。

アニメーションの新規作成 マテリアルアニメーションを新規に作成します。

カーブの作成

 カーブの作成と編集を行います。
カラーピッカーでキーの調整 色を選択することでカーブを調整します。

参考：

完成データは以下の場所にあります。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\Eye_Finish.fmab

 

アニメーションの新規作成

3DEditor のファイルビューから「既定アニメーションセット」を右クリックして「アニメーションの新規作成」→「マテリアル」を選びます。

アニメーションの新規作成ウインドウが開くので下記の設定にします。

名前 Eye
フレーム数 100
ループ再生 On

OK ボタンを押すとファイルビューに Eye.fmab が作成されます。
「 Eye.fmab 」の表示(下図)が追加されます。 

関連リンク 詳しくはファイルビューの基本操作を参照してください。

ディフューズカラーの選択

ファイルビューの Eye.fmab をダブルクリックしてプロパティウインドウを開きます。

以降、このページではプロパティウインドウで操作します。


関連リンク 詳しくはアニメーションプロパティウィンドウを参照してください。

プロパティの左側のリストから「 EyeMt 」→「シェーダパラメータ」→「ディフューズ」を開きます。

下図のように「ディフューズ」→「ディフューズカラー」以下のチェックのみ有効にします。
RGB全てにチェックが入っていることを確認します。 

注意：

このモデルには2つのマテリアル(ディフューズ)があります。「 BodyMt 」と間違えないよう注意してください。

関連リンク 詳しくはカーブツリービューを参照してください。

カーブの作成

メニューから「キーの追加」をクリックします。見つからない場合はウインドウサイズを大きく広げます。

マウス中ボタンクリックして 3 つのキーフレームのカーブを作成します。
RGB 成分がそれぞれ重なった状態でカーブが出来ます。

関連リンク 詳しくはツールバー(カーブエディタ)を参照してください。

キーの編集

各キーフレームを調整します。

下図のようにドラッグで各キーフレームを選択して値を変更します。
RGB のカーブは重なっています。RGB 成分すべてに対して値を変更する必要があります。

ここではテキストボックス（下図）に数値を入力して値を調整します。

下表のように各キーフレームの値を入力します。

フレーム 値
0

0.0
50 1.0
100 0.0

下図のようなカーブになります。（ RGB のカーブ 3 つが同じ位置に重なった状態)

参考：

ショートカットキー「 A 」でカーブ全体をフォーカスできます。
ショートカットキー「 F 」で選択したキーをフォーカスできます。
キー、カーブ以外の場所をクリックまたはドラッグで選択を解除できます。

カラーピッカーでキーの調整

ドラッグでカーブを選択します。キー以外の線の部分をドラッグして選択します。
カーブを選択した状態で下図の赤い枠内の白をクリックするとカラーピッカーが開きます。

カラーピッカーでピンク色をクリックします。クリックすると RGB にキーが打たれます。

下図のようになります。

プロパティウインドウを閉じて、カーブの編集を終了します。

関連リンク 詳しくはタイムスライダー（ カーブエディタ ）およびカーブビューを参照してください。

参考：

マウス操作でカーブを調整することも可能です。以下の方法が有ります。

メニューから「複数移動」をクリックします。
左クリックもしくは左ドラッグして複数のキーやカーブを選択し、中ボタンでドラッグすると移動が出来ます。

メニューから 「最も近いキーの移動」をクリックします。
キーやカーブを選択する必要は無く、中ボタンでキーをドラッグすると移動出来ます。（同時に一つのキーのみ）

ビューアで確認

アニメーションを再生して SampleTripod の目がピンク色に点滅するのを確認します。
