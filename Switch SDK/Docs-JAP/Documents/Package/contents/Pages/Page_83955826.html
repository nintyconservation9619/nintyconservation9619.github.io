セットアップマネージャー

はじめに

セットアップマネージャーは、サムネールエクスプローラー拡張の機能の設定を変更するためのツールです。
ftx, tga, psd 等のファイルのサムネール・プレビュー表示と、アプリケーション関連付けの設定を変更することが出ます。

注意：

セットアップマネージャーを使用せずにアプリケーション関連付けの設定を行った場合、サムネール・プレビュー表示機能が無効化されることがあります。
その場合、再度セットアップマネージャーで設定を変更してください。

標準設定について

インストーラーを使用してインストールを行った後、機能は標準設定で自動的に有効化されます。
セットアップマネージャーを使用することで、標準設定から設定を変更することができます。

インストール時の標準設定の内容は以下の通りです。

拡張子 エクスプローラー拡張 開くアプリケーション
.ftxa/.ftxb 全て（サムネール、プレビュー、ツールチップ） 無し
.tga 全て（サムネール、プレビュー、ツールチップ） 無し
.psd 全て（サムネール、プレビュー、ツールチップ） 無し
.ai 全て（サムネール） 無し
.eps  全て（サムネール） 無し
.eset 全て（サムネール、プレビュー） 無し
.flyt 全て（サムネール、プレビュー） 無し
.fmdb/.fmda 全て（サムネール、プレビュー） 無し

使用方法

セットアップマネージャーは、「%USERPROFILE%/AppData/Local/Nintendo/ThumbnailExplorerExtensions/Setup/」 にインストールされています。
セットアップマネージャーを使用するには Windows の管理者権限が必要になります。

  1. ThumbnailExplorerExtensions/Setup/ExtensionManager.exe を実行して、セットアップマネージャーを起動します。
     
  2. 左のツリーメニューから、設定を行うファイルタイプ ( FTX, TGA, PSD, AI, ESET, EPS, FLYT, FMD ) を選択します。
    「All Files」を選択すると一括で全タイプへの設定を行います。
  3. ファイルタイプに対して有効にしたいエクスプローラー拡張機能 ( サムネール、プレビュー等 ) をチェックします。
    「全て登録する/全て登録から外す」を押すとチェックボックスが一括で変更されます。

  4. ファイルをダブルクリックしたときに「開くアプリケーション」をプルダウンメニューから選択します。「All Files」を選択している場合は、「無し」「リセット」「カスタム」のみ選択可能です。

    項目 説明 備考
    無し ファイルを開く際のアプリケーションを指定しません。
    リセット セットアップマネージャーで設定を行う以前の状態に戻します。
    TextureViewer ファイルを TextureViewer で開きます。 FTX のみ
    Photoshop ファイルを Photoshop で開きます。
    インストールされた Photoshop バージョンが自動的にリストアップされます。    		 FTX, TGA, PSD のみ
    カスタム ファイルを任意のアプリケーションで開きます。
    「参照」ボタンから任意のアプリケーションを選択してください。
  5. OK ボタンを押して、セットアップマネージャーを終了してください。

インストール処理で Windows エクスプローラーを再起動します。※PC の再起動は不要です。
環境によっては、インストール完了までに数分かかることがあります。
インストール先のフォルダは、移動や削除をしないでください。

設定の変更

エクスプローラー拡張の設定を変更する場合は、再度セットアップマネージャーを起動します。

設定を変更すると、セットアップマネージャーのタイトルバーに (*) が付きます。

設定を変更して OK ボタンを押すと、サムネールエクスプローラー拡張の機能の再設定ができます。
