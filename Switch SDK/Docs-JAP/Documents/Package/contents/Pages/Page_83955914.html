カメラ

ビューアで使用しているカメラに関する設定を行います。

ビュー・射影

「 ビュー・射影」は、 接続先のビューアを制御できる機能群です。
以下はグループ内設定項目の一覧です。

カメラの位置（±） カメラの位置を、ワールド座標系で設定できる機能です。
注視点（±） カメラの注視点を、ワールド座標系で設定できる機能です。
カメラの回転（±） カメラをカメラ座標系の Z 軸で回転できる機能です。
ニア カメラから見て、最前面の位置（ニアクリップ）をカメラ座標系で設定します。
「ファー」の設定と合わせることで描画範囲を設定できます。
ファー カメラから見て、最後面の位置（ファークリップ）をカメラ座標系で設定します。
「ニア」の設定と合わせることで描画範囲を設定できます。
カメラタイプ カメラタイプ「パースペクティブ」と「オルソ」から選択する機能です。
