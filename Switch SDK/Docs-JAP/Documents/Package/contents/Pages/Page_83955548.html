バンク

プロジェクトツリーで、バンクのノードを選択することで、サウンドセットタブにバンクを表示することができます。
この時表示されるリストを、バンクリストと呼びます。

バンクとバンクファイル

バンクはバンクファイルを参照します。
シーケンスサウンドが実際に発音するインストルメントは、バンクファイルで定義されます。
バンクファイルはこの後で解説するバンクタブで編集を行います。

バンクの新規作成と追加

バンクを新規作成または追加する場合は、サウンドセットタブのリスト領域を選択して、[プロジェクト]→[アイテムの追加]→[バンク…] を実行します。

バンクリストで、右クリックを行うと現れるコンテキストメニューによる操作でも、バンクを新規作成または追加することができます。

バンクの編集

バンクの編集はバンクリストで行います。
バンクとバンクファイルの章で解説したとおり、音色の本体はバンクから参照されるバンクファイルで定義されます。
バンクリストでは、バンクの名前と、参照するバンクファイル程度の編集しか行いません。

バンクリストの定義要素

ここではバンクリストの各定義要素の意味を解説します。

名前

バンクの名前を指定します。
名前に使用できる文字には制限があり、以下の文字が使えます。

  • 先頭は、半角英字とアンダーバー [a-zA-Z_]
  • 先頭以外は、半角英数字とアンダーバー [a-zA-Z0-9_]

波形アーカイブ

波形アーカイブとの紐付けの設定（共有したい波形アーカイブの選択）をします。

  • 自動（共有）
    【自動】で波形アーカイブが生成されます。
    また、同じ埋め込みグループ内のほかのウェーブサウンドセットやバンクの波形アーカイブが【1 つにまとめられます】。
    （埋め込みグループ内で使用するすべての波形をまとめた波形アーカイブが 1 つ生成されます。）
    波形アーカイブは波形データの【個別ロード】はできず、【一括ロード】されます。
  • 自動（個別）
    【自動】で波形アーカイブを生成します。
    しかし、グループに属してもグループ内のほかのウェーブサウンドセットやバンクとは波形アーカイブが【1 つにはまとめられません】。
    この波形アーカイブは、ほかのどのバンクとも共有されません。【一括ロード】となります。
  • 波形アーカイブを明示指定
    波形アーカイブ内の波形ファイルの【個別ロード】【一括ロード】は、波形アーカイブリストで指定できます。
    しかし、【個別ロード】かつグループに【埋め込み】の場合は、【一括ロード】になります。

グループの「出力方式」と組み合わせた波形アーカイブの解説は、波形アーカイブの自動(共有, 個別)設定 も参照してください。

ファイルパス

バンクファイルのパスを指定します。

ファイル名、およびパス名に使用できる文字には制限があり、以下の文字が使えます。

  • ファイル名の先頭は、半角英字とアンダーバー [a-zA-Z_]
  • ファイル名の先頭以外およびパス名は、半角英数字とアンダーバー [a-zA-Z0-9_]

データサイズ

バンクが参照するバンクファイルの波形データ圧縮変換後のデータサイズを表示します。
表示内容は、コンバートのタイミングで更新されます。

HTMLインストルメントリストの出力

HTML サウンドリストの出力と同様に、インストルメントリストを HTML 形式でファイル出力することができます。

バンクリスト上で、右クリックによるコンテキストメニューから [HTMLインストルメントリストの出力] を実行すると、設定ダイアログが表示されます。 

出力項目を選択してファイル出力します。

HTML サウンドリストと違い、コンバート時には出力されません。
HTML インストルメントリストは、バンクファイル毎にバンクファイルと同じディレクトリに出力されます。
インストルメントのラベル名やコメントを表で一覧することができ、項目毎のソートなども行えます。
