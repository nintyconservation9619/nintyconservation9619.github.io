DrawParam

DrawParam

概要

nn::mii::DrawParam は顔モデルや顔輪郭テクスチャ、マスクテクスチャを構成する 1 パーツを描画するために必要なパラメータを保持しています。各パーツの DrawParam を各クラスから取得して、描画します。

DrawParam の保持する情報

変調の種類

どの変調を用いて描画するかの種類を保持しています。変調の種類に関しては、変調の説明ページを参照していください。

頂点属性バッファ

DrawParam から取得できる頂点属性バッファの種類は nn::mii::DrawParam::AttributeType 列挙体と関連しており、以下の通りです。

種類 説明
nn::mii::DrawParam::AttributeType_Position 位置
nn::mii::DrawParam::AttributeType_Normal 法線
nn::mii::DrawParam::AttributeType_Uv テクスチャ座標
nn::mii::DrawParam::AttributeType_Tangent 接線
nn::mii::DrawParam::AttributeType_Param 特殊パラメータ

各頂点属性の nn::gfx::Buffer やサイズ、ストライドやフォーマットの情報を取得することができます。詳しくは API リファレンスを参照してください。

テクスチャデスクリプタスロット

パーツを描画するためにテクスチャが必要な場合、テクスチャが登録されたデスクリプタスロットのポインタを取得できます。

デスクリプタスロットは nn::mii::DrawParam::GetTextureDescriptorSlot() 関数で取得できます。変調の種類によってはテクスチャを必要としない場合もあります。テクスチャが存在しなければ、GetTextureDescriptorSlot() 関数は nullptr を返します。

定数カラー

変調で使用する定数カラーを取得できます。nn::mii::DrawParam::GetConstantColor() 関数を使用して取得できます。Mii のボディ部分の色は取得できません。お気に入りの色を取得して使用してください。

カリングモード

描画時に指定するカリングモードを取得できます。nn::mii::GetCullMode() 関数を使用して取得してください。取得したカリングモードに合わせて描画の設定を変え、描画してください。
