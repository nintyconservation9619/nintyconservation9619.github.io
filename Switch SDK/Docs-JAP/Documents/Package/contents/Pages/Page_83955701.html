サムネールエクスプローラー拡張

サムネールエクスプローラー拡張　ユーザーマニュアル

概要

「サムネールエクスプローラー拡張」では、Windows エクスプローラーの機能を拡張し、NintendoWare 形式のファイルをエクスプローラー上でプレビューすることができます。サムネールエクスプローラー拡張は NintendoSDK パッケージに含まれ、インストーラー形式で提供されます。

主な機能一覧

  • Windows エクスプローラー上でのサムネール表示
    • NintendoWare テクスチャー中間ファイル(.ftxb/.ftxa)
    • Targa 画像(.tga)
    • Photoshop 画像(.psd)
    • Illustrator 画像(.ai)
    • Illustrator EPS 画像(.eps)
    • エミッタセット(.eset)
    • レイアウトファイル(.flyt)
    • NintendoWare モデル中間ファイル(.fmdb/.fmda)

  • NintendoWare テクスチャー中間ファイル専用ビューアー
    • NintendoWare テクスチャー中間ファイル (.ftxb/.ftxa) を専用のビューアーでプレビューし、詳細なデータを確認できます。

注意：
  • EPS のサムネールは、Adobe Illustrator で Illustrator 9 EPS 以降のフォーマットで保存された EPS ファイルに限って表示されます。
  • AI のサムネールは、Adobe Illustrator で Illustrator CS 以降のフォーマットで保存時のオプションで「PDF 互換ファイルを作成」を有効にして保存された AI ファイルに限って表示されます。

含まれるツール

サムネール・プレビュー表示機能 画像ファイルをエクスプローラー上でサムネール表示するための拡張機能です。
TextureViewer テクスチャー中間ファイル ( .ftxa / .ftxb ) をプレビューするツールです。
セットアップマネージャー 上記ツールの設定を変更するツールです。

インストール・アンインストール方法

注意：
  • サムネールエクスプローラー拡張のインストール・アンインストールには Windows 管理者権限が必要です。
  • インストール・アンインストール時には他の実行中のアプリケーションを終了してください。また、他のエクスプローラー拡張ツールが実行されていないことを確認してください。
  • インストールはエクスプローラー機能拡張を利用するアカウントで進めてください。利用するアカウントと異なるアカウントでインストールすると、 対象となる拡張子を手動で関連付けていた場合に機能が利用できなくなります。

インストール方法

インストーラーの場所

サムネールエクスプローラー拡張インストーラーは NintendoSDK パッケージに含まれます。NintendoSDK パッケージのインストール後、サムネールエクスプローラー拡張インストーラーは 「ルートパス/Installers/ThumbnailExplorerExtensions/」 以下に配置されています。setup_jp.exe を実行することで、標準設定で機能が有効化されます。

設定変更方法

アプリケーションの関連付け設定など、一部機能の設定を手動で変更したい場合はセットアップマネージャーを使用してください。セットアップマネージャーを使用せず設定変更を行った場合、サムネール・プレビュー表示機能が無効化されることがあります。サムネールエクスプローラー拡張のインストール後、セットアップマネージャーは 「%USERPROFILE%/AppData/Local/Nintendo/ThumbnailExplorerExtensions/Setup」 以下に配置されています。詳細は セットアップマネージャー ページを参照して下さい。

アップデート方法

サムネールエクスプローラー拡張は、一台の PC につき 1 バージョンのみインストールが可能です。アップデートをする場合、アップデート対象の NintendoSDK パッケージに含まれるサムネールエクスプローラー拡張のインストーラーを実行してください。

アンインストール方法

サムネールエクスプローラー拡張のアンインストールは以下のいずれかの方法で行うことができます。

  • サムネールエクスプローラー拡張がインストールされている状態で、インストールに使用した setup_jp.exe を実行し、「削除」を選択します。
  • 「Windows コントロール パネル」より「プログラムと機能」を選択し、NintendoSDK-ThumanilExplorerExtensions のアンインストールを実行します。
  • 「%USERPROFILE%/AppData/Local/Nintendo/ThumbnailExplorerExtensions」 に配置される unins000.exe を実行します。

その他

  • ファイルダイアログを使用するアプリケーション全般でサムネールを表示できますが、全ての Windows アプリケーションにおいて正しく動作することは保障していません。ファイルダイアログなどを自作しているアプリケーションなどでは意図しない挙動になる可能性があります。
  • サムネールエクスプローラー拡張は 3D Texture のプレビューには非対応です。
  • エクスプローラー拡張で表示されるテクスチャー中間ファイルの画像は、オリジナルイメージに対しフォーマットなどに応じた簡易な変換を施したものです。テクスチャーコンバーターでコンバートした結果と細かな違いが出ることがあります。

既知の不具合

 Maya のファイルダイアログでサムネールが一種類になる

Maya のファイルダイアログでは、フォルダ内に含まれているファイルのサムネールが他のファイルでも表示されてしまいます。Maya の「プリファレンス」で「ファイル/プロジェクト」の「ファイルダイアログ」 設定を「OS ネイティブ」に設定して回避してください。上記不具合は、以下のバージョンで確認されています。

  • Maya 2011
  • Maya 2012
  • Maya 2013
  • Maya 2014
  • Maya 2015
