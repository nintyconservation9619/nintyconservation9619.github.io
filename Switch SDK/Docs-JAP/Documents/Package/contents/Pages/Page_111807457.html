アニメーション共有情報ウインドウ

アニメーション共有とは

ゲームのメニュー画面などのレイアウト作成する場合を考えると、多くのアイテムで同じアニメーション動作を行うような場合が少なくありません。
たとえば以下のような場合があります。

  • メニューボタンにカーソルが衝突した時に再生されるアニメーション
  • メニューボタンで決定操作が行われた時に再生されるアニメーション

アニメーション共有機能は、このようなデータを効率的に作成するために存在します。
アニメーション共有機能を利用することで、同じアニメーションをたくさん複製したり、プログラマとの間で取り決めをして特別な処理を行うことなくアニメーション共有を実現できます。

参考：

この機能はレイアウトビューアで確認することができません。

アニメーション共有情報ウインドウ

アニメーション共有を設定するウインドウです。
以下の方法で、アニメーション共有情報ウインドウを表示できます。

  • [ メニュー ] → [ ウインドウ ] → [ アニメーション共有情報 ]

以下は「アニメーション共有情報ウインドウ」の UI 説明です。

アニメーション共有情報ウインドウ
A 新規 アニメーション共有情報の登録ダイアログが開きます。
B 編集 アニメーション共有情報の登録ダイアログが開きます。
C 共有グループの作成 アニメーション共有用グループの作成ダイアログが開きます。
D 共有対象をハイライト表示  
E 削除 選択したアニメーション共有情報をリストから削除します。
F アニメーション共有情報  

アニメーション共有情報の登録ダイアログ

以下は共有元ペイン、共有対象グループを設定するダイアログ「アニメーション共有情報の登録」の UI 説明です。

アニメーション共有情報の登録ダイアログ
A 共有元ペイン 共有元ペインをプルダウンリストから選択します。
B 共有対象グループ 共有対象グループをプルダウンリストから選択します。
C コメント アニメーション共有情報にコメントを設定します。

アニメーション共有用グループの作成ダイアログ

アニメーション共有用グループの作成ダイアログ

C 共有元ペインと同一種類のみを対象とする
D 共有対象を検索
