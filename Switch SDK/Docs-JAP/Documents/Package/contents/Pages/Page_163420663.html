有線接続

開発機で有線接続を利用するには、開発機と SDEV クレードル/Switch ドック、SDEV クレードル/Switch ドックと USB-Ethernet アダプターをそれぞれ接続し、SDEV クレードル/Switch ドックには専用の ACアダプターから給電する必要があります。

SDEV クレードル/Switch ドックを用いた有線接続を利用するには、 1.8 以降（1.8 を含む）の SDEV または EDEV を使用してください。

また、1.8 以前（1.8 を含む）の SDEV では、ドックを経由しない USB-Ethernet アダプターの直接接続は、ハードウェアの制約により非対応です。
それ以降の機材についても、現時点での動作は保証されません。

USB-Ethernet アダプターは開発機材に同梱されませんので、別途ご用意ください。

なお開発機に対応する USB-Ethernet アダプターにつきましては、「SDEV 取扱説明書」および「EDEV 取扱説明書」の「USB デバイスを接続する」をご覧ください。

同時に利用できる USB Ethernet アダプターは最大で1個までです。
複数接続してしまった場合は一度 USB Ethernet アダプターをすべて抜去し、利用するものだけを挿しなおしてください。 

注意：

SDEV クレードルおよび SDEV クレードルの電源アダプターの不具合にご注意ください。 USB Ethernet アダプターを認識しない場合、SDEV クレードルの USB 端子への通電状態を再確認してください。 

USB-Ethernet アダプターのセットアップ（SDEV）

SDEV クレードルの USB 端子に USB-Ethernet アダプターを接続することで、通信機能を使うことができます。

次の手順で、USB-Ethernet アダプターを接続してください。

  1. 本体上部から付属の Wii U 本体 ACアダプターのプラグを抜いて電源を OFF にします。
  2. ACアダプターをコンセントから抜きます。
  3. SDEV 本体と SDEV クレードルを接続します。
  4. SDEV クレードルの USB 端子に USB-Ethernet アダプターを接続します。
  5. SDEV クレードルに ACアダプターを接続します。
  6. ACアダプターをコンセントに接続します。
    • このとき、SDEV クレードルのテレビ出力インジケーターが数秒間点灯してから、消灯します。
    • SDEV クレードルのテレビ出力インジケータ―が点灯しなかった場合は ACアダプターをコンセントから抜き、10 秒待ってから接続しなおしてください。
  7. SDEV 本体上部の ACアダプター接続端子に付属の Wii U 本体 ACアダプターのプラグを接続して電源を ON にします。

 

警告：

故障のおそれがありますので、SDEV クレードルは必ず同梱されているものを使用してください。 また、SDEV 1.6 では SDEV クレードルの使用はサポートしていません。
