はじめに

参考：

AlbumDownloader は、NX Addon 5.1.0 にて DevMenuCommand の album サブコマンドに集約されました。
また、NX Addon 6.0.0 からは廃止されてパッケージには含まれないため、DevMenuCommand への移行をお願いします。

AlbumDownloader は開発機のアルバムに保存されている全ファイルを PC にコピーしたり、削除するためのターゲット向けツールです。

以下の操作が実行可能です。

  • 開発機のアルバムに保存されているすべてのファイルを PC にコピーする
  • 開発機のアルバムに保存されているすべてのファイルを削除する

このツールを使用することで、キャプチャーボタンを押して撮影したファイルや、 ALBUM ライブラリ を利用してアプリケーションが保存したファイルを PC 上で確認できます。 

参考：

開発機のアルバムに保存されているファイルを個別にコピー・削除することはできません。

PC から本体のアルバムにファイルをコピーすることはできません。

参考：

AlbumDownloader は本体機能が使用するメモリを一部拝借して動作するようになっているため、AlbumDownloader 実行中は本体機能（主に HOME メニューですが、DevMenu も含む）をあまり操作しないようにして下さい。AlbumDownloader 実行中にこれらの操作を行うとメモリ不足で FATAL エラーが発生する場合があります。そのような場合には、一度本体をリセットして起動し直して下さい。

使用方法

RunOnTarget を利用してツールを起動します。

$ Tools\CommandLineTools\RunOnTarget.exe --suppress-auto-kill TargetTools\NX-NXFP2-a64\AlbumDownloader\Release\AlbumDownloader.nsp -- <action> [<arguments>]


ヘルプの表示

引数なしまたは --help を指定した場合、 AlbumDownloader のヘルプが表示されます。 AlbumDownloader のコマンドを確認できます。

コマンド例

$ RunOnTarget.exe AlbumDownloader.nsp
$ RunOnTarget.exe AlbumDownloader.nsp -- --help

アルバムの全ファイルを PC にコピー

アクションに --download を指定した場合、開発機のアルバムに保存されているすべてのファイルを PC の指定したディレクトリにコピーします。

コピー元となるアルバムのストレージとコピー先の PC 上のディレクトリの指定が必要です。コピー先のディレクトリが空でなかった場合コピーを行いません。 --force オプションを指定した場合にはコピー先のディレクトリが空でなかった場合でもコピーを行います。

引数

引数 必須／任意 値 説明
--storage 必須

以下のいずれか

  • BuiltIn
  • SdCard

コピー元のアルバムのストレージを指定します。

  • BuiltIn … 本体内蔵ストレージ
  • SdCard … SD カード 
--directory 必須

PC 上のディレクトリの絶対パス

  • 例： C:/Album/

コピー先の PC 上のディレクトリの絶対パスを指定します。

ドライブのルート（ C:/ など）は指定できません。
--force 任意

コピー先のディレクトリが空でなかった場合でもコピーを実行します。

ファイル名が重複した場合、上書きコピーします。 

コマンド例

$ RunOnTarget.exe --suppress-auto-kill AlbumDownloader.nsp -- --download --storage BuiltIn --directory C:/Album
$ RunOnTarget.exe --suppress-auto-kill AlbumDownloader.nsp -- --download --storage BuiltIn --directory C:/Album --force


アルバムの全ファイルを削除

アクションに --clean を指定した場合、開発機のアルバムに保存されているすべてのファイルを削除します。

対象とするアルバムのストレージを指定が必要です。

引数

引数 必須／任意 値 説明
--storage 必須

以下のいずれか

  • BuiltIn
  • SdCard

ファイルを削除するアルバムのストレージを指定します。

  • BuiltIn … 本体内蔵ストレージ
  • SdCard … SD カード 

コマンド例

$ RunOnTarget.exe --suppress-auto-kill AlbumDownloader.nsp -- --clean --storage SdCard
