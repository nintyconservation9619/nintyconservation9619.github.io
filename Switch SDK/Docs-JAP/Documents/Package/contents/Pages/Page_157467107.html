Material

nn::mii::Material は 1 体分の Mii のマテリアルを管理するクラスです。

Mii 作成・編集アプレットと同様の Mii の 3D モデルの表示を行いたい場合には、このクラスから各パーツのマテリアルを取得して描画してください。

初期化と終了処理

nn::mii::Material::Initialize() 関数を呼ぶことで初期化を行います。初期化時に渡された nn::mii::CharInfo を元に各パーツのマテリアルを設定します。初期化した内容は nn::mii::Material::Finalize() 関数を呼び終了処理を行うまで変更できません。

各パーツのマテリアルの取得

nn::mii::Material::Get() 関数を使用することで、指定した nn::mii::CharModel::DrawType のマテリアルを取得できます。また、体とパンツ用のマテリアルはそれぞれ nn::mii::Material::GetBody() 関数と nn::mii::Material::GetPants() 関数を使用することで取得できます。

各 Get 関数を使用することで、1 つの DrawParam のマテリアルを保持する nn::mii::DrawParamMaterial が取得できます。詳しくは API リファレンスを参照してください。
