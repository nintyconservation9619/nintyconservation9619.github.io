アニメーションセット

概要

アニメーションセットとは、複数のアニメーションをまとめたものです。
アニメーションセットはモデルにバインドされ、ビューアライブラリで再生するとアニメーションセットに含まれるアニメーションが同時に再生されます。

アニメーションセットの情報はプロジェクトファイルに保存されます。

参考：

この文章や 3DEditor のメニュー等では、アニメーションをアニメーションセットに バインドする という表現を、アニメーションセットにアニメーションを含めるという意味で使っています。

アニメーションセットの名前

アニメーションセットには、印字可能な ASCII 文字からなる文字列で名前を付ける必要があります。

参考：

空文字列や、空白文字のみからなる文字列の名前は許されません。

  • ひとつのモデルにバインドされるアニメーションセットはそれぞれ違う名前でなければいけません。
  • バインドされるモデルが異なる場合はアニメーションセットに同じ名前を付けることができます。
  • 同じ名前のバインドされるモデルが異なるアニメーションセットは区別されます。

アニメーションセットのバインド

アニメーションセットは、ひとつのモデルにバインドする必要があります。

※アニメーションセット自体は、空の状態で問題ありません。

参考：

ひとつのモデルに複数のアニメーションセットをバインドすることは可能です。

アニメーションセットがバインドされているモデルを閉じると、バインドされているアニメーションセットは、全て削除されます。

アニメーションセットの操作

アニメーションセットは、ファイルビューで操作します。

ファイルビュー上で行った操作はアンドゥができます。
