Maya プラグイン

コンフィグファイルの adjust_smooth_skinning のデフォルト値変更

Maya プラグインのコンフィグファイルの adjust_smooth_skinning のデフォルト値を false から true に変更しました。true に設定すると、連続するシェイプ間でスキニング方式が統一されるため、連続するシェイプ間で隙間が発生する問題が発生しなくなります。

既にコンフィグファイルに adjust_smooth_skinning を記述している場合は影響はありませんが、adjust_smooth_skinning をコンフィグファイルに記述していない場合に挙動が変わりますのでご注意ください。

コンフィグファイル

コンフィグファイル Export.ini を編集して、エクスポートプラグインの動作をカスタマイズすることを説明しています。

エクスポートオプションを Pre-Export Script で上書きできるように

Maya プラグインの Pre-Export Script で、出力中のエクスポートオプションを上書きする機能（NintendoOverwriteExportOption プロシージャ）を追加しました。一時的に最適化を無効にすることによって出力時間を短縮し、高速にプレビューしたい場合などに利用することができます。ただし、以下のオプションは上書きできません。

  • preset
  • lod_export
  • pre_export_script
  • post_export_script

一部のコンフィグ設定も上書きできます。

  • optimize_primitive (none / default / force / brute_force / forsyth)
  • quantization_analysis (true / false)
  • delete_near_vertex (文字列)

エクスポートでエラーが起こった際に一時ファイルを保持できるように

Post-Export Script や中間ファイルオプティマイザーでエラーが発生した場合に、テンポラリーフォルダーに出力した中間ファイルを削除しない機能を追加しました。ポストエクスポートスクリプトなどに失敗した際のデバッグをする際に利用できます。

コンフィグファイルで次のように指定します。 

delete_temp_file_on_error="false"

指定しない場合のデフォルト値は true（削除する）です。

Photoshop プラグイン

Unify Image Options と Unify Converter Options のデフォルト値設定を追加

Unify Image Options と Unify Converter Options のデフォルト値設定を追加しました。psd ファイルに設定が含まれていない場合のデフォルト値を設定することができます。

var default_unify_image_options = true;
var default_unify_converter_options = false;
