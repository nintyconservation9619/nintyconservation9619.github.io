1. デフォルトパラメーターでリダクションを行う

どのようなモデルにも適用できる、オーソドックスなパラメーター（以下、デフォルトパラメーター、単に、デフォルト と呼びます）で実行してみます。デフォルトパラメーターでよい結果が得られた場合は、それで問題はありませんが、多くの場合、デフォルトパラメーターは問題の検出に役立ちます。


(注)デフォルトパラメータで実行を行ってもUVアトラスが複数存在する場合には、かならずUVが考慮されます。

デフォルトパラメーター
"--target-rate-lod-level1 0.5 --target-rate-lod-level2 0.25 --property-flag Position --optimize-position-policy OptimalOnSpace"

以下のバッチファイルで実行する事が出来ます。

%NINTENDO_SDK_ROOT%\Samples\Sources\Tools\3dTools\3dIntermediateFileOptimizer\PolygonReductionTutorial\ExecuteDefaultParameter.bat

このパラメーターの意味は次のようなものです。

  • LODレベル １ と ２ に 50% と 25% の削減率の LOD を作成する
    --target-rate-lod-level1 0.5 --target-rate-lod-level2 0.25
  • 頂点位置のみを考慮したリダクションを行う  
    --property-flag Position
  • 処理によって、元のモデルからの体積の変化量が最小になるようにリダクションを行う 
    --optimize-position-policy OptimalOnSpace
