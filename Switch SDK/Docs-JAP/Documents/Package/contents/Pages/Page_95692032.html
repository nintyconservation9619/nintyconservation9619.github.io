シーンアニメーションの固定値入力機能

シーンアニメーション「カメラ」「ライト」「フォグ」のプロパティウィンドウにある全般ページでは、固定値（アニメーションしないコンスタントな値）が設定できます。

参考：

アニメーションが設定されているプロパティは、グレーオフして編集できない状態になります。

カメラアニメーション

カメラアニメーションでは、回転モード、射影モードで使わないプロパティは無効になります。

ライトアニメーション

パラメータが全て固定値の場合、ライトアニメーションのプロパティウィンドウは全般ページをデフォルトにして開きます。

ライトのタイプを切り替えることにより、教示されるパラメータが変わります。

フォグアニメーション

パラメータが全て固定値の場合、フォグアニメーションのプロパティウィンドウは全般ページをデフォルトで開きます。
