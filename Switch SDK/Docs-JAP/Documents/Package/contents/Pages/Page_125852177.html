1 準備

作業を開始するために各ツールの起動手順を説明します。

このチュートリアルでは以下のファイルを利用します。

3DEditor %NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\Start3dEditor.bat

シェーダー定義ファイル

(3DEditorが自動で読み込みます）

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\shader\demo.fsdb
3dSampleViewer 
( ビューア )		 %NINTENDO_SDK_ROOT%\Samples\BuiltApplications\3dSampleViewer\3dSampleViewer.exe

モデル、テクスチャ、

アニメーション中間ファイル

 %NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod 以下のファイル

このチュートリアルで使用するデータは以下の種類があります。

データの種類 拡張子
シェーダー定義ファイル .fsdb
モデル中間ファイル .fmdb
スケルタルアニメーション中間ファイル .fskb
マテリアルアニメーション中間ファイル .fmab
テクスチャ中間ファイル .ftxb
3DEditor プロジェクトファイル .f3pj
