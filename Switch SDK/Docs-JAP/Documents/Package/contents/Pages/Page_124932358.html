入力の「Image」タブ

Image

以下は「Image」タブの UI 説明です。
入力時のフォントファイルとして画像を利用できます。

A 画像ファイル 変換元の画像ファイルを指定します。
B 文字順序 画像中にどのような順番で文字が並んでいるかを選択します。
C カラーフォーマット

画像の各ピクセルの色をどのように解釈するかを指定します。

フォーマット名に「～_SRGB」 と付くものは、リニアワークフローに対応したテクスチャフォーマットです。
D 線形補間対策

チェックをつけるとグリフに対して線形補間による周辺色にじみ問題の対策処理を行います。

この処理はアルファチャンネルをもつ「カラーフォーマット」の場合のみ効果があります。
