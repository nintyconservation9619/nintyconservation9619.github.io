D3Dデバイスの初期化に失敗した旨のエラーが表示された

質問

D3Dデバイスの初期化に失敗した旨のエラーが表示されました。

回答

以下の確認をしてください。

  • ツールを再起動して、もう一回開いてみて再現するか（再現性があるかどうか）
  • 他の人の環境で再現するか（作業者PC環境依存の問題かどうか）
  • 他のデータが開けるか（データ依存の問題かどうか）
  • 新規作成レイアウト（Ctrl+N）で再現するか。また、カーブエディタが表示されているか（最もシンプルな状態で再現するか）


作業者PC環境依存か、データ依存が疑われるときは編集環境のリセットを試してみます。

  • 手順
    • 色見本をエクスポートしておきます（消えてしまうので）
    • LayoutEditor を終了します
    • %LOCALAPPDATA%\Nintendo\LayoutEditor\X.X.X.0（X.X.X.0 は自分の使っているバージョン） 以下 の setting.xml を削除します。
      • 上のパスを、エクスプローラー(Win+E で起動します)のアドレスに貼りつけると簡単にアクセスできます。
    • LayoutEditor を開きます
