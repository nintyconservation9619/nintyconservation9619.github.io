NintendoSDK NX Addon の動作環境

NintendoSDK NX Addon は以下の環境で動作することを確認しています。
以下と異なる環境で使用される場合は、十分に検証を行ったうえで使用して頂くか、予めオフラインインストーラーなどをご準備頂くようにお願いします。
(特に Visual Studio の MSVC コンパイラのバージョンが異なる場合には注意が必要です)

OS
  • Microsoft Windows 7 Professional (64bit) 英語版もしくは日本語版
  • Microsoft Windows 10 Pro (64bit) 英語版もしくは日本語版
警告：

NintendoSDKのインストール・利用は、管理者権限を持つアカウントが必要です。
.NET Framework
  • Microsoft .NET Framework 4.6.1 以降
参考：

インストールされていない場合は、次のリンク先からダウンロード＆インストールしてください。

https://www.microsoft.com/ja-jp/download/details.aspx?id=49981

Visual Studio
  • Microsoft Visual Studio 2015 Professional Edition Update 3 英語版もしくは日本語版
  • Microsoft Visual Studio 2017 Professional Edition (version 15.5) 英語版もしくは日本語版
警告：

Microsoft Visual Studio 2017 は、NintendoSDK NX Addon 4.4.0 以降で使用できます。

Microsoft Visual Studio 2017 version 15.5 は、NintendoSDK NX Addon 5.1.0 以降から対応しています。

NintendoSDK NX Addon 5.1.0 未満の場合は、version 15.4 が対応バージョンとなります。

参考：

NX アプリケーションをビルド・デバッグする際に必要です。
アプリケーションをビルド・デバッグしない場合は不要です。
Windows SDK
  • Windows SDK 8.1 (Visual Studio 2015 を使用する場合)
  • Windows SDK 10.0.15063.0 (Visual Studio 2017 を使用する場合)
参考：

PC 環境でアプリケーションをビルド・デバッグする際に必要です。
アプリケーションをビルド・デバッグしない場合は不要です。

ビデオカード
  • NVIDIA Maxwell 第1世代以降を推奨（NX ハードウェアは Maxwell 第2世代 ）
参考：

PC 環境で NVN を利用する際に必要です。

詳細は、NVN の説明 をご確認ください。
開発機
  • SDEV (SDEV と PC の接続については、NIC による直接続、もしくは、LAN 経由の接続を推奨)  
  • EDEV
警告：

SDEV と EDEV の試作機は使用できません。

参考：

SDEV / EDEV は、SDEV / EDEV 上で NX アプリケーションを実行する際に必要です。
PC 環境のみで開発する場合は不要です。
コントローラー
  • Joy-Con
  • Switch Proコントローラー
参考：

コントローラーのファームウェアの変更履歴は、コントローラーファームウェアのリリースノート を参照してください

参考：

PC 環境で Joy-Con または Switch Pro コントローラーを使用する場合は、PC に Bluetooth が必要です。

NintendoSdk NxAddon で動作確認済みの Bluetooth ドングルは Planex BT-Micro4 です。
