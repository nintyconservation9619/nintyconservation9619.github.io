環境構築ガイド

概要

環境構築ガイドは、NintendoSDK を使用したアプリケーションやライブラリをビルドするにあたり、既存のビルドシステム環境を再利用したい開発者や独自のビルドシステムを構築したい開発者を対象として、必要な情報を提供するドキュメントです。

新規にビルドシステム環境を構築したい場合は、Visual Studio から NintendoSDK の新規プロジェクト作成ウィザードを使用してプロジェクトを作成することを推奨します。
