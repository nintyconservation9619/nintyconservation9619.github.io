パーティクル形状

ここではパーティクルの表示形状を設定する機能「パーティクル形状」について説明します。

概要

「パーティクル形状」は、パーティクルの表示形状を「パーティクル」と「プリミティブ」のどちらかで設定できる機能です。
「パーティクル形状」を変更することによって、多様な形状のパーティクルが利用でき、エフェクト表現の幅を広げることができます。
以下は「 設定項目」の概要です。

パーティクル形状

パーティクルとして使用する形状を、以下のプリセットから選択します。

パーティクル エフェクトランタイムで生成される板状ポリゴンです。
プリミティブ DCC ツールで作成した自由な形状のポリゴンです。
厚みのあるポリゴンも利用できます。
