深度テスト

ここでは描画しようとするパーティクルの深度値をテストする機能「深度テスト」について説明します。

概要

「描画しようとするパーティクルの奥行き値が視点から見てZバッファ（深度バッファ）の値より手前か奥かを、選択した条件に従ってテストできる機能です。
以下は「 設定項目」の概要です。

深度テスト

「深度テスト」の有効/無効を「 チェックボックス」ON/OFF で切り替えます。
設定をONにすると 「 深度テスト」が表示されます。
以下は「 設定項目」の概要です。

テスト通過の条件

深度テストの通過条件を、プリセットから選択します。

Zバッファ書き込み

テストを通過した深度値を深度バッファに書き込むかを「 チェックボックス」ON/OFF で切り替えます。

設定例

以下は「深度テスト」各プリセットの比較画像です。
「 F の文字」を貼った平面パーティクルを通過するように、「□ 赤枠」チェック模様の平面パーティクルを前後に移動させています。
「Zバッファ書き込み」はすべて「 チェックボックス ON」に設定しています。

A 常に通過しない この設定を選択した場合、常にテストを通過しません。
「 F の文字」は常に表示されません。
B 深度値 ＜ Zバッファ 深度値がZバッファの値より小さい場合に通過します。
 「 F の文字」は「□ 赤枠」が手前にあるときに表示されます。
C 深度値 ＝ Zバッファ 深度値がZバッファの値と同じ場合に通過します。
 「 F の文字」は「□ 赤枠」と同じ位置にあるときに表示されます。
D 深度値 ≦ Zバッファ 深度値がZバッファの値と同じか小さい場合に通過します。
 「 F の文字」は「□ 赤枠」が手前か同じ位置にあるときに表示されます。
E 深度値 ＞ Zバッファ

深度値がZバッファの値より大きい場合に通過します。

 「 F の文字」は「□ 赤枠」が奥にあるときに表示されます。
F 深度値 ≠ Zバッファ

深度値がZバッファの値と同じでない場合に通過します。

 「 F の文字」は「□ 赤枠」が同じ位置ではないときに表示されます。
G 深度値 ≧ Zバッファ 深度値がZバッファの値と同じか大きい場合に通過します。
 「 F の文字」は「□ 赤枠」が奥か同じ位置にあるときに表示されます。
H 常に通過 この設定を選択した場合、常にテストを通過します。
 「 F の文字」は常に表示されます。
