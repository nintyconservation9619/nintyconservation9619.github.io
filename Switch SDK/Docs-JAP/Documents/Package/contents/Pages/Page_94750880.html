拡散初速

ここでは指定した軸方向への初速を設定する「拡散初速」について説明します。

概要

「拡散初速」は、指定した軸固定でパーティクル個々に対して放出方向と初速をランダムにする機能です。
パーティクル全体の移動軸を固定した上でパーティクル個々の初速をランダムにすることで見た目にばらつきを与えます。
以下は「 設定項目」の概要です。

拡散初速

パーティクル放出時に拡散させる軸とその方向への速度を設定します。

拡散軸

拡散させる軸はエミッタの座標系で設定します。
以下は「円（等分割）エミッタを傾けた状態を横から見た画像です。

A 拡散初速 X Z軸で 45 度傾けたエミッタを基準に「拡散初速」を X軸に固定して放出します。
B 拡散初速 Y Z軸で 45 度傾けたエミッタを基準に「拡散初速」を Y軸に固定して放出します。
C 拡散初速 Z X軸で 45 度傾けたエミッタを基準に「拡散初速」を Z軸に固定して放出します。

拡散初速

初速に対して拡散初速で設定した値を加算した結果が放出速度になります。
加算値は、設定した値を上限としてランダムに変化します。以下は放出時に決まる初速度の計算式です。

  • 放出時の初速度 = 初速度 +（-「拡散初速」～ +「拡散初速」の計算結果）

以下は「指定方向初速」に対して「拡散初速」を与えたときの説明画像です。

「指定方向初速」 「指定方向初速」+「拡散初速」
A 指定方向初速 Y=0.5
拡散初速 X = 0.2
 Y 方向初速 0.5 に対し、拡散させる軸を X軸に固定し、 -0.2 ～ +0.2 の範囲でランダムに決まった値を足して放出します。
B 指定方向初速 Z=0.5
拡散初速 Y = 0.2		 Z 方向初速 0.5 に対し、拡散させる軸を Y軸に固定し、 -0.2 ～ +0.2 の範囲でランダムに決まった値を足して放出します。
C 指定方向初速 X=0.5
拡散初速 Z = 0.2		 X 方向初速 0.5 に対し、拡散させる軸を Z軸に固定し、 -0.2 ～ +0.2 の範囲でランダムに決まった値を足して放出します。
