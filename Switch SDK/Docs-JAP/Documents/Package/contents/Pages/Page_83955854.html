3D DCC プラグイン出力設定ファイル

概要

3D DCC プラグイン出力設定ファイル（fdes ファイル）は、NintendoWare が提供する 3D DCC ツール用エクスポートプラグインのオプション設定をテキスト形式のファイルにしたものです。
fdes ファイルは 3D DCC ツールに依存しない共通フォーマットになっています。
fdes ファイルの拡張子は fdes です。
fdes ファイルはエクスポートプラグインから入出力可能で、Windows のバッチファイルを使用して中間ファイルを出力する際にも利用します。

構造

fdes ファイルは、エクスポートプラグインのオプション設定項目に対応するキーワードと設定内容を表す値で構成されます。
# から始まる行はコメント行で、エクスポートプラグインに入力する際は無視されます。
fdes ファイルの例にあるコメント行は常に出力するものとします。

fdes ファイルの１行目は次のようなコメント行になっている必要があります。

# NW4F_Export settings

（# の後ろに半角スペースが１つ入ります）

参考：

1 行目が不正な場合はエクスポートプラグインに入力する際にエラーとなります。

入出力

エクスポートプラグインから fdes ファイルを出力する際は、オプションウインドウで設定できるすべての項目（グレー表示を含む）について、fdes ファイルに出力することができます。

エクスポートプラグインに fdes ファイルを入力した際、fdes ファイルに記述された内容のうち、キーワードと値が正常なもののみエクスポートプラグインのオプション設定に反映されます。
fdes ファイル内に不正なキーワードや値があった場合、そのキーワードは無視されます。

キーワード frame_range に playback が指定されている fdes ファイルを、playback の概念のない 3D DCC ツールに入力した際は all と解釈します。
