バイナリファイル出力ウインドウ

レイアウトエディタウインドウのメニューバー「ファイル」から選択できる「バイナリ出力」ついて説明します。

注意

このダイアログは、プログラマーが検証などの用途でバイナリファイルを出力するためのものです。
実制作においては、デザイナーはこのダイアログを利用しないワークフローの構築をお勧めします。

このダイアログで指定する項目は、「レイアウトバイナリコンバータ 」の各種オプションに対応しています ( ただし、すべてのオプションをカバーしているわけではありません ) 。

 

バイナリファイル出力ウインドウ
A 出力ディレクトリ バイナリファイルを出力するディレクトリを指定します。ディレクトリを指定することで出力ボタンが有効になります。
B 出力ファイル名 出力されるバイナリファイルの名前を指定します。
C 出力ディレクトリに出力ファイル名でフォルダを作成 チェックが入っていると、出力ディレクトリに出力ファイル名でフォルダを作成し、そのフォルダ内にバイナリファイルを出力します。
D キー領域以外のカーブ値を固定して出力する

・ツールが処理 ( 互換モード )

・コンバータが処理 ( 推奨 )
E テキストボックスのエスケープ文字を制御コードに変換する

テキストボックス中の文字列のタグ処理に利用する制御コードに変換するかどうかのオプション（--no-convert-cvtrchar）を制御します。

  • タグ処理については「フォントのタグ処理について」をご覧ください
F タグ情報を利用する チェックが外れていると H 出力ファイル種類を個別に選択することができます。
G キーに対する制御

・無し

・フレーム区間内で値が変化しない場合にキーを出力しない ( 最初のアニメーション区間タグを除く )

・フレーム区間内で値が変化しない場合にキーを出力しない
H 出力ファイル種類

出力するファイルの種類を選択できます。

この項目は F タグ情報を利用するのチェックが外れている場合に設定可能です。
