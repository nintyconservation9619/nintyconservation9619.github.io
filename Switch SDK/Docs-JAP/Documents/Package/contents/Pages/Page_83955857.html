テクスチャは無駄なく描く

テクスチャに無駄な余白は設けず、適切なテクスチャサイズで描くことで、描画負荷を抑えることができます。
テクスチャパターンアニメなどで炎のコマを表現する際など、以下の様に最終コマはアルファ抜きの多い絵が描かれがちです。


表現上許されるのであれば、上記例では４コマ目を利用せずに
３コマ目からスケールアニメで消去するなど、代替案を検討してみてください。

多量のパーティクル表現では素直にパーティクルを放出した方が良い場合もあります

１枚のテクスチャに複数の粒を書き込むような表現をするより、 単純にパーティクル数を増やした方がメリットがある場合があります。
GPUパーティクル化できればCPUコストは問題にはなりませんし、 アルファ抜きされる部分が減ることで、GPUコストの削減も期待できます。
