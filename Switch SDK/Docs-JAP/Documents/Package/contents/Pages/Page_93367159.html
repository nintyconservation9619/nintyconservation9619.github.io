放出位置ランダム

ここではパーティクルをランダムな位置から放出する機能「放出位置ランダム」について説明します。

概要

「放出位置ランダム」は、パーティクルを仮想的な球体内のランダムな位置から放出する機能です。
等分割系のエミッタ形状では、分割位置を中心に、パーティクルの放出位置をランダムにできます。
以下は「 設定項目」の概要です。

放出位置ランダム

全エミッタ形状共通の設定項目です。放出位置を中心とした球の半径を指定します。

設定例

以下はエミッタ形状「円（等分割）」で分割数を 4 に設定した時の「放出位置ランダム」の設定例と解説図です。

A
 		等分割した箇所 エミッタ「円」を 4 等分した箇所です。
B 仮想的な球体 放出位置を中心とした仮想的な「球体」です。
C パーティクルの生成位置

パーティクルは B の仮想的な「球体」を実際の放出範囲とし、この範囲内でランダムな位置に生成されます。
D パーティクルの移動方向 エミッタの放出方向に移動します。
同一の「球体」から生成されたパーティクルは、同じ方向に移動します。
