テクスチャーリソースファイル

概要

テクスチャーリソースファイルは、テクスチャーコンバーターによって作成されたバイナリーイメージを表現します。

テクスチャーリソースファイルは複数のテクスチャを扱う仕組みを提供します。

テクスチャーファイルリソース

テクスチャーファイルリソースは nn::gfx::ResTextureFile として定義されています。

テクスチャーファイルリソースはテクスチャーコンバーターによって作成されるバイナリーイメージそのものを表し、リソースファイルクラスの共通機能を利用することができます。

テクスチャーファイルリソースの機能を使う前に、まずテクスチャーファイルリソースを初期化する必要があります。

初期化・終了

Initialize によってテクスチャーファイルリソースを初期化、Finalize によってテクスチャーファイルリソースを終了します。

Initialize 時にメモリープールを指定しなかった場合は、リソース内部のメモリープールが使われます。テクスチャーコンバーターによるリソース作成時に --disable-memory-pool オプションが指定されている場合は、リソース内部にメモリープールを所有していないため、この方法で初期化することはできません。
使用者側のメモリープールがテクスチャーファイルリソースを含んでいる場合は、そのメモリープールを指定して Initialize を呼ぶことによって、リソース内部のメモリープールのかわりに使用者側のメモリープールを使うことができます。

テクスチャー辞書の取得

GetTextureDic により、テクスチャーファイルリソースがもっているすべてのテクスチャーリソースを管理している nn::util::ResDic を取得します。

テクスチャーリソースの取得

GetResTexture により、指定されるインデックスのテクスチャーリソースを取得します。
テクスチャーの名前からテクスチャーリソースを取得したい場合は、テクスチャー辞書の nn::util::ResDic::FindIndex により対象のインデックスを検索してください。

テクスチャーリソース

テクスチャーリソースは nn::gfx::ResTexture として定義されています。

テクスチャー情報の取得

GetTextureInfo により、GFX のテクスチャー情報を取得します。

テクスチャーオブジェクトの取得

GetTexture により、GFX のテクスチャーオブジェクトを取得します。初期化前はテクスチャーオブジェクトの領域には 0 が埋められています。

テクスチャービューオブジェクトの取得

GetTextureView により、GFX のテクスチャービューオブジェクトを取得します。初期化前はテクスチャービューオブジェクトの領域には 0 が埋められています。

初期化・終了

Initialize を呼ぶと、GetTextureInfo によって得られる情報を使って、GetTexture によって得られるテクスチャーオブジェクトと、GetTextureView によって得られるテクスチャーオブジェクトを初期化します。

Finalize を呼ぶと、GetTexture によって得られるテクスチャーオブジェクトと、GetTextureView によって得られるテクスチャーオブジェクトを終了します。

名前の取得

GetName により、テクスチャーの名前を取得します。

データサイズの取得

GetDataSize により、テクスチャーのストレージデータのサイズを取得します。

ユーザーデータ辞書の取得

GetUserData により、テクスチャーリソースがもっているすべてのユーザーデータリソースを管理している nn::util::ResDic を取得します。

ユーザーデータの取得

GetUserData により、指定されるインデックスの nn::gfx::ResUserData を取得します。

ユーザーデスクリプタースロットの設定・取得

テクスチャーリソースにはユーザーによって自由に使うことのできるデスクリプタースロットの領域が確保されています。
SetUserDescriptorSlot によってユーザーデスクリプタースロットを設定、GetUserDescriptorSlot によって設定したユーザーデスクリプタースロットを取得できます。
