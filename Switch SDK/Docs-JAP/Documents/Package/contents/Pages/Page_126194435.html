未使用ファイル

「検索基点パス」以下のエミッタセットから参照されていない未使用のファイルを検索する「未使用ファイル」について説明します。
以下は「未使用ファイル」専用の設定項目です。

A 未使用ファイルを UnUsed フォルダへ移動

この項目にあるボタンをマウス左クリックすることにより、未使用のファイルを UnUsed フォルダに移動します。
B 選択したファイルを移動 ファイルリストから任意で選択したファイルを、UnUsed フォルダに移動します。
C 全て移動 ファイルリストに表示された全てのファイルを、UnUsed フォルダに移動します。
D UnUsed フォルダを開く この項目にある「開く」ボタンをマウス左クリックすることにより、UnUsed フォルダをエクスプローラで開きます。 
\Tools\Graphics\EffectMaker\UnUsed

概要

以下はこのプリセットの図説です。

A 検索基点パス アドレスバーで指定したパスです。検索の基点になります。
B エミッタセット

「検索基点パス」以下のエミッタセットです。

この範囲にあるエミッタセットが参照していない未使用ファイルを検索します。
C 未使用ファイル

B の範囲にあるどのエミッタセットからも参照されていない未使用ファイルです。
本機能の検索対象です。
