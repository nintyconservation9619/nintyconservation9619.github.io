パーティクル時間のカーブエディタ操作

ここではプロパティウィンドウでカーブでパーティクル時間のアニメーションを付ける UI 「カーブエディタ」の操作について説明します。

関連リンク チュートリアル「炎を作る」の「パーティクルスケールをアニメーションさせる」の部分について具体的な操作方法を説明します。

スケールアニメーションの操作

以下は、スケールアニメーション設定前と設定後の比較画像です。

アニメーション設定前 アニメーション設定後

以下の項目でパーティクルサイズをアニメーションさせます。
アニメーションは「カーブエディタ」を使用します。

項目名 時間（%） 値 説明
スケールアニメーション 0 1.0

放出時は値を1.0に設定します。「基準値」がそのまま適用されます。
50 1.0

寿命の半分までは値を1.0に設定します。放出時と同サイズを継続します。
100 0.5

最後のフレームではスケール値を 0.5 に設定します。
透明度のアニメーションと合わせることで、燃焼エネルギーの減少を印象付けられます。

以下の動画は「カーブエディタ」の操作方法です。

  • マウス左ボタンクリックすることで「 アニメーションキー」を追加できます。
  • 追加した「 アニメーションキー」をマウス左ボタンクリックした状態のまま縦方向に移動させることで、キーのスケール値を設定できます。
  • 追加した「 アニメーションキー」をマウス左ボタンクリックした状態のまま横方向に移動させることで、パーティクル寿命に対するキーのタイミングを設定できます。
  • 「アニメーションキー」を削除したい場合はキーの上でマウス右ボタンクリックします。
参考：

パーティクル時間の場合、「時間」の値入力はパーティクル寿命に対する割合で指定します。
キーを直接操作した場合は、「 キーのフレーム表示」に実際のフレーム数が表示されます。
