B Bone Visibility Animation Data ボーンビジビリティー 可視性のオンオフのアニメーションを出力します。
C Color Animation Data カラー

マテリアルのアニメーションを出力します。

D Texture SRT Animation Data テクスチャー SRT フレームの移動、フレームの回転、繰り返し UV のアニメーションを出力します。
E Texture Pattern Animation Data テクスチャーパターン イメージ番号（イメージシーケンスの使用）のアニメーションを出力します。
F Shape Animation Data シェイプ

ブレンドシェイプ（デフォーマの作成）の各シェイプのブレンドのアニメーションを出力します。

(3dSampleViewer では確認することが出来ません)
G Scene Animation Data シーン

(3dSampleViewer ではカメラのみ確認することが出来ます)
H Search Model Animation アニメーションの検索 アニメーションキーを探し、キーが付いてるアニメーションの項目にチェックを入れます。
