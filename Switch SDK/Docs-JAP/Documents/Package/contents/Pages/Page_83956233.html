マテリアル名

3ds Max 上では複数のマテリアルに同じ名前を付けることができますが、中間ファイルではマテリアル名が重複することはありませんのでユニークな名前を設定してください。

同じ名前のマテリアルが複数存在する場合、エクスポートプラグイン内部で最初に取得したマテリアルはそのままの名前で出力されますが、 それ以降の名前が重複するマテリアルにはマテリアル名の後ろに "_1"、"_2" のように アンダーバーと数字が付加されます。

そのままの名前で出力されるマテリアルや数字が付加される順番は、ユーザー側で確認や変更ができません。 DCC ツールのバージョンや環境などによって変わる場合もあります。

マテリアルの種類

Standard マテリアルの"ブリン"、"フォン"、"異方性反射"をお使いください。

それ以外のマテリアルをお使いの場合でも、パラメータが Standard マテリアルと同様のものがある場合には出力できますが、それ以外のパラメータについては対応しません。

異方性反射を使用すると、法線マッピングが設定されていなくても接線と従法線が出力されます（マップチャンネル1から計算します）。

有効なアトリビュート

次のアトリビュートが中間ファイルに反映されます。

また、マテリアルアトリビュート設定プラグインで、NintendoWare 用のアトリビュートを設定できます。

  • 拡散反射光 → ディフューズカラー
  • 不透明度 → 不透明度
  • 周囲光 → アンビエントカラー
  • 自己照明 → エミッションカラー
  • 鏡面反射光 → スペキュラカラー

マルチ/サブオブジェクト

マルチ/サブオブジェクトマテリアルにも対応しています。
マルチ/サブオブジェクトマテリアルが使われている場合、中間ファイル内に格納する頂点データはサブマテリアル毎に分割して出力します。
