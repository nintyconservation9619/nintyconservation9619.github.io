ユーザー設定ファイル

概要

ユーザー設定ファイルは 3DEditor.user.config のことです。

編集することで、ユーザーごとに 3DEditor の動作をカスタマイズできます。

ユーザー設定ファイルでは、以下の項目を設定します。

  • スタートアップパス
  • ファイル更新時の再読み込み設定
  • 編集履歴の設定
  • 外部プログラムの設定
  • 入出力の設定
  • コントロールの設定

※ユーザー設定ファイルのフォーマットは、こちら を参照してください。

ファイル更新時の再読み込み設定

3DEditor が読み込んでいるファイルを他のアプリケーションが更新したときに、3DEditor が更新されたファイルを読み込み直すかユーザー設定で設定します。

  • 設定はファイルの種類ごとに行います。
  • 再読み込みの対象は各種中間ファイルです。
  • 再読み込み時に確認ダイアログを表示させるかを設定します。
    ただし、未保存のファイルを再読み込みするときは、設定に関わらず確認ダイアログが表示されます。
    確認ダイアログではファイルの再読み込みを行うかどうかを選択します。
  • モデルやテクスチャパターンアニメーションを再読み込みしたときに、参照するテクスチャファイルが 3DEditor 上に読み込まれていなければ、テクスチャファイルも読み込みます。

編集履歴の設定

ファイルを閉じたときやファイルをリロードしたときに編集履歴を消去するかどうかや 3DEditor 上に保持する編集履歴の項目数を設定します。

外部プログラムの設定

3DEditor から起動するテキストエディタや接続先アプリケーションのデバッグログを出力するかどうか設定します。

入出力の設定

モデルファイルの読み込み時にマテリアルの修正を自動で行うかどうか、シェーダコンパイル時情報の出力先、読み込み済みファイルの再読込を自動で行うか設定します。

コントロールの設定

マテリアルプロパティウィンドウにあるシェーダページに、スライダコントロールを表示するか設定します。
