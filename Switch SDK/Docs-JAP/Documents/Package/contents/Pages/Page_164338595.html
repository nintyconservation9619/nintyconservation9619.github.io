テクスチャコンバイナ

ペインに登録したテクスチャ画像やカラーとの合成方法を設定します。

詳細なコンバイナモードへ切替 表示を切り替えて、詳細なコンバイナモードの設定を行います。

テクスチャ選択

登録されたテクスチャから、テクスチャコンバイナで設定をするテクスチャを指定します。
カラー合成設定 カラーの合成種類を設定します。インダイレクトの調整（ゆがみ・回転）を行います。
アルファ合成設定 アルファの合成種類を設定します。

詳細なコンバイナモードへ切替

 「詳細なコンバイナモード」から、設定表示を切り替えます。

詳細なコンバイナモードへの切り替え  詳細なコンバイナモード

 

 テクスチャ選択

登録されたテクスチャから、テクスチャコンバイナで設定をするテクスチャを指定します。

設定をするテクスチャを選択します。

 カラー合成設定

カラーの合成種類を設定します。インダイレクトの調整（ゆがみ・回転）を行います。

種類 :

カラー合成設定の種類を指定します。

  • 上書き
  • 乗算
  • 加算
  • 減算
  • 除外 [ 明色から暗色を減算：自身が白色なら反転 ]
  • 覆い焼き [ 前段を明るく：自身が白色に近いほど影響大 ]
  • 焼き込み [ 前段を暗く：自身が黒色に近いほど影響大 ]
  • オーバーレイ [ 前段明度 128 以上：反転を乗算 ( 明るく ) 、127 以下：乗算 ( 暗く ) ]
  • 比較 ( 明 )
  • 比較 ( 暗 )
  • インダイレクト [ 歪みソース = 自分、対象 = 前段 ]
  • インダイレクト [歪みソース = 自分と後段の混合結果、対象 = 前段 ]
  • インダイレクト [歪みソース = 自分 ( 3 段目 )、対象 = 前段 1 と前段 2 ]

インダイレクト調整

( アニメーション可能 )

インダイレクトの調整をします。

  • ゆがみの強さ
  • 回転
注意：
  • 同時に 2 枚以上のインダイレクト設定は未対応です。
  • インダイレクト [ 歪みソース = 自分と後段の混合結果 ] を使う場合は「 3 枚テクスチャの 2 枚目テクスチャにインダイレクトを設定」します。
  • インダイレクト [歪みソース = 自分 ( 3 段目 )、対象 = 前段 1 と前段 2 ] を使う場合は「 3 枚テクスチャの 3 枚目テクスチャにインダイレクトを設定します。
