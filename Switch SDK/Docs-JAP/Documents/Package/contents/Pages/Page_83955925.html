バイナリー中間ファイルコンバーター

バイナリー中間ファイルコンバーターの使用方法と注意事項を説明します。

  • 概要
  • コマンドラインオプション
  • 利用例

概要

テキスト中間ファイル (.f??a) をバイナリー中間ファイル (.f??b) に変換します。

テキストエディター等でテキスト中間ファイルを書き換えた後に、バイナリー中間ファイルに変換する場合に利用します。

実行ファイル (3dIntermediateFileAsciiToBinary.exe) やショートカットに、テキスト中間ファイルやフォルダーをドラッグ＆ドロップすると、テキスト中間ファイルのあるフォルダーにバイナリー中間ファイルを保存します。
テキスト中間ファイル以外のファイルは無視します。

バイナリー中間ファイルを他の場所に出力したい場合は、テキスト中間ファイルをコピーしてから変換するか、変換後のバイナリー中間ファイルを移動してください。

コマンドラインオプション

3dIntermediateFileAsciiToBinary.exe [--args-file ""] [--version] [-h] [-s] [-j ""] [--disable-xsd-check]
[--enable-extension ""] [--disable-extension ""] [中間ファイル|フォルダ]...

テキスト中間ファイル (.f??a) をバイナリー中間ファイル (.f??b) に変換します。

--args-file ""

引数ファイルを指定します。
引数ファイルに記述した内容を、引数として指定できます。
引数ファイルは複数行に渡って各オプションを記述できます。
行頭に # がある行はコメントとして扱います。
オプション値指定の "" によるエスケープに改行を含めることはできません。

--version

バージョン情報を表示します。

-h --help

ヘルプを表示します。

-s --silent

処理中のメッセージ表示を抑制します。

-j "" --jobs ""

実行ジョブ数を指定します。

--disable-xsd-check

スキーマ (.xsd) による中間ファイルのフォーマットチェックを無効化します。
本オプション有効時は、最新版の中間ファイルしか処理できなくなります。

--enable-extension ""

中間ファイルの中で、処理する拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--enable-extension "fmdb,ftxb"

--disable-extension ""

中間ファイルの中で、処理しない拡張子を指定します。 　
複数記述する際は , で区切ります。 　
例）--disable-extension "fmdb,ftxb"

中間ファイル|フォルダー

処理するファイルやフォルダーを指定します。
テキスト中間ファイル以外は無視します。
フォルダーを指定した場合、指定フォルダー以下のファイルを再帰的に処理します。
中間ファイル以外は無視します。

ワイルドカード（?: 一文字、*: 複数文字）に対応しています。
例) "a?b*"
一文字目が a で三文字目が b のファイルを変換します。

利用例

バイナリー中間ファイルコンバーターの利用方法を説明します。

テキスト中間ファイルの変換

text.fmda を text.fmdb に変換します。

3dIntermediateFileAsciiToBinary.exe text.fmda

フォルダーの変換

text フォルダーやサブフォルダー内のテキスト中間ファイルを、バイナリー中間ファイルに変換します。

3dIntermediateFileAsciiToBinary.exe text\
