はじめに

本ドキュメントの目的

本ドキュメントでは、 NintendoSDK に収録されているオーディオおよびビデオに関する圧縮コーデック機能について説明します。

対象読者

NintendoSDK を使ったオーディオおよびビデオに関する圧縮コーデック関係のプログラミングをされる方を対象としています。

関連資料

オーディオおよびビデオに関する圧縮コーデックの API リファレンス と併せて参照して下さい。

CODEC ライブラリの機能

CODEC ライブラリは、オーディオおよびビデオに関する圧縮コーデック機能を提供します。

現在は以下のオーディオ圧縮コーデックに関する機能が利用可能です。

  • aac
  • adpcm
  • opus

CODEC ライブラリの提供形態

以下のページを参照してください。

  • ライブラリの提供形態と利用方法

CODEC ライブラリが提供する機能

CODEC ライブラリでは以下の機能を提供しています。

  • adpcm デコーダ
  • opus デコーダ
  • opus エンコーダ

また、関連する PC ツールとして、以下を提供しています。

  • OpusEncoder

それぞれの機能については、マニュアル、API リファレンス、サンプルプログラム などを参照してください。

ライセンス条項

aac

CODEC ライブラリに含められる aac デコーダは、オープンソースソフトウェア (OSS) fdk-aac を使用しています。これらの機能を使用するアプリケーションの企画や開発には、当該 OSS に係るライセンス条項による制約が追って課せられます。

下記は fdk-aacのライセンス条項を要約したものです。

  • 権利表記
    • fdk-aac の実行コードおよび fdk-aac の実行コードを含むバイナリデータを再配布する場合、 配布物に同梱される文書に fdk-aac ライセンス条項全文 と同じ内容が含まれる必要があります

正式なライセンス条項は fdk-aac ライセンス条項全文 に記載しています。必ずこちらを参照の上、対応を行ってください。

参考：

CODECライブラリの機能はすべてのアプリのバイナリに含まれますので、本体設定内で表記を行います。各アプリでの表記は不要です。

opus

CODEC ライブラリに含められる opus デコーダ、および opus エンコーダ PC ツールは、オープンソースソフトウェア (OSS) libopus (ver.1.1.5) を使用しています。 これらの機能を使用するアプリケーションの企画や開発には、当該 OSS に係るライセンス条項による制約が追って課せられます。

下記は libopus のライセンス条項を要約したものです。

  • 権利表記
    • libopus の実行コードおよび libopus の実行コードを含むバイナリデータを再配布する場合、 配布物に同梱される文書に libopus ライセンス条項全文 と同じ内容が含まれる必要があります
  • 免責事項
    • libopus の著作権者等の名前を事前の許諾なしに宣伝広告で使用してはいけません

正式なライセンス条項は libopus ライセンス条項全文 に記載しています。必ずこちらを参照の上、対応を行ってください。

参考：

CODECライブラリの機能はすべてのアプリのバイナリに含まれますので、本体設定内で表記を行います。各アプリでの表記は不要です。
