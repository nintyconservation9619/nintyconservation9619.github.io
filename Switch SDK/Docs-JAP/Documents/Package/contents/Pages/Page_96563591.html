SDEV の初期化

SDEV を初期化する

専用のツールで SDEV を初期化します。（およそ 2 分で完了します。）

参考：

手順の途中で、Windows ファイアウォールに関する警告が数回表示されますが「アクセスを許可する」を選択してください。

手順 操作
1

SDEV 底面に貼ってあるシールに書かれた値を確認します。

（この値が SDEV の シリアル番号を示しています。以下の例では「XAW02121013686」。）

参考：

ここで確認した値は、手順３で SDEV を選択する際に使用します。
2

NintendoSDK 管理画面の [SDEV を初期化する] ボタンをクリックします。


参考：

NintendoSDK 管理画面は、 Nintendo Dev Interface の「ようこそ」画面の「開発環境の構築・管理」を選択すると表示できます。
3

初期化する SDEV を選択し、初期化をおこないます。


「1. Select SDEV actual equipment」の下にある SDEV 接続リストから「Serial」が手順 1 で確認したものと一致する行をマウスで選択します。


「2. Select firmware variation」の下にあるコンボボックスから利用したいファームウェアのバージョンを選択します。

参考：

最初に選択されているファームウェアをそのまま選択することを強く推奨します。

「+ Prod」「+ ProdWithLog」を選択する場合は、「NintendoSDK ドキュメント」の「環境構築ガイド > より詳しい情報 > 製品機に近いファームウェア(PROD MODE)への初期化」を先に読んでください。


次に「3. Start SDEV initialization」の下にある「Start initialization」のボタンを押すと SDEV の初期化が始まります。
4

ログウィンドウの左下に「 Success 」と表示されたら、「Exit」ボタンを押してツールを終了します。


参考：

ファンの動作音

手順 4 のログウィンドウが表示されると、SDEV に内蔵されている冷却ファンが動作しはじめます。
少し大きな動作音がしますが、故障しているわけではありません。
参考：

他にもセットアップに関する追加情報を「NintendoSDK ドキュメント」の「環境構築ガイド > より詳しい情報 > 開発機セットアップに関する情報あれこれ」のページにまとめました。ご参照ください。

参考：

SDEV が正常に初期化されたかどうか確認するためのビルド済みサンプルが用意されています。

サンプルの実行方法は以下の通りです。

  1. NintendoSDK 管理画面で [サンプルを実行する] ボタンをクリックします。
  2. SDEV が正常に初期化され、VSI for NX が正しくインストールされている場合は、SDEV の液晶に次の画面が表示されます。

※このサンプルは自動的に終了しません。HOMEボタンでメニュー画面に戻ってください。

トラブルシューティング

  • 上記の「手順 2」の GUI アプリケーションが起動しない場合は、NintendoSDK\Tools\CommandLineTools\InitializeSdevWin.exe というファイルが存在するか確認してください。
    存在しない場合は、ウイルス対策ソフトが誤作動し上記ファイルが削除されてしまっている可能性があります。ウイルス対策ソフトが誤作動していないか確認してください。
  • 初回生産分の SDEV MP を使用している場合、シリアル番号が書き込まれておらず、「手順 3」で確認する SDEV 接続リストの「Serial」列が空欄になることがあります。その際は、下記手順に従って該当する SDEV を確認してください。
    1. SDEV を裏返し、背面の LED が見える状態にする
    2. SDEV 接続リストの任意の行を右クリックする
    3. 「Blink LEDs」を選択する
    4. 背面の LED (GC/SD) が点滅するか確認する

    LEDが点滅した場合、その端末が SDEV 接続リストで選択中の SDEV です。

  • SDEV の初期化に失敗すると、「手順 4」で Success のかわりに Fail と表示されます。
    その場合は、以下の手順で何度か試してみてください。それでも成功しない場合は、サポート窓口に問い合わせてください。
    1. Target Manager が起動している場合は終了する
      • Target Manager はタスクトレイに格納されている場合がありますので、タスクトレイも確認してください
    2. SDEV に接続されている Wii U 本体 ACアダプターを抜き挿しする
    3. 上記の「手順2」からやり直す
