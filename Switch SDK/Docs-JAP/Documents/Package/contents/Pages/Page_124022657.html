コントロールの設定ウインドウ

「コントロール設定ウインドウ」は、部品のコントロールの設定をするウインドウです。

コントロール定義を追加することで、新しい振る舞いを定義することができます。
例えば、プログラマが拡張プログラムを用意すると、独自の挙動を持つボタンなどを作ることが可能です。

コントロールの設定ウインドウでは以下の操作ができます。

コントロールの設定ウインドウ
A コントロールの種類

設定するコントロールの種類をプルダウンリストから選択します。

  • 通常レイアウト [ None ]
  • 配置用部品 [ Locator ]
  • 標準ボタン [ NormalButton ]
  • チェックボタン [ CheckButton ]
  • 決定ボタン [ DecisionButton ]
  • 選択ボタン [ SelectCutton ]
  • タッチスクリーン用ボタン [ TouchOffButton ]
  • タッチスクリーン用チェックボタン [ TouchOffCheckButton ]
  • ドラッグボタン [ DragButton ]
  • タッチスクリーン向けドラッグボタン [ TouchDragButton ]
B コントロールの説明 選択したコントロールの説明が表示されます。
C 名前 コントロールの名前を設定します。
