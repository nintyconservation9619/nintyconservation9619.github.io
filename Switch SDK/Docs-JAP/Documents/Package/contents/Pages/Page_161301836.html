ピクチャペイン

ピクチャペイン

プロパティウインドウの中の「ピクチャペイン」では、LayoutEditor のピクチャペイン情報を編集します。

形状

ピクチャペインの形状を設定します。

プロパティウインドウの設定で、形状を「四角形」「角丸四角形」「円」から選択できます。
それぞれの形状は、メッシュの分割数などで調整することができます。

参考：

選択した形状により、プロパティウインドウの設定項目が切り替わります。

形状： ピクチャペインの形状をプルダウンから設定します。
四角形

ピクチャペインの形状を「四角形」にします。

プロパティウインドウの設定が「四角形」専用のものに切り替わります。
角丸四角形

ピクチャペインの形状を「角丸四角形」にします。

プロパティウインドウの設定が「角丸四角形」専用のものに切り替わります。
円

ピクチャペインの形状を「円」にします。

プロパティウインドウの設定が「円」専用のものに切り替わります。
形状「四角形」のピクチャペイン 形状「角丸四角形」のピクチャペイン 形状「円」のピクチャペイン
注意：
  • 「四角形」以外を使用した場合、形状のポリゴンは「ペイン基本」の「サイズ」を元に自動生成されますが、ランタイムでは負荷の問題からレイアウトデータ初期化時に一度だけメッシュを生成して再生成は行わないため
    「サイズ」にアニメーションカーブを設定しても意図した見た目をランタイムで再現することはできません。
  • 「四角形」以外の設定をランタイムで使用するためにはプログラム側で以下の点の対応が必要となります。
    • コンスタントバッファ用の GpuBuffer 内の領域にメッシュ表示のための頂点バッファとインデックスバッファを確保しますので、GpuBuffer に設定する GpuAccessFlag へ nn::gfx::GpuAccess_ConstantBuffer に加え、nn::gfx::GpuAccess_VertexBuffer と nn::gfx::GpuAccess_IndexBuffer を設定してください。
    • GfxUtil の機能を使用するため libnn_gfxUtil をリンクしてください。
