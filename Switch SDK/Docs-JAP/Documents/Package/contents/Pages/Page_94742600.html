エミッタ形状「円柱（ボリューム）」

ここではパーティクルの放出位置が円柱内のエミッタ形状「円柱（ボリューム）」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「円柱（ボリューム）」は、ワールド座標系に配置される円柱全体のランダムな位置から円柱の側面方向にパーティクルを放出します。
以下は「 設定項目」の概要です。

「エミッタ形状」を「円柱（ボリューム）」に設定すると「 円柱（ボリューム）」が表示されます。以下はグループ内の設定項目です。

空洞率

パーティクルが放出されない領域を「 ダイヤル」や「 テキストボックス」によりエミッタ中心から外周までの距離に対する割合で設定します。

設定例

以下はエミッタ形状「円柱（ボリューム）」の「空洞率」を 50% に設定した場合の画像です。

A 空洞率 エミッタ中心からの50%の割合で設定した範囲はパーティクルが放出されない領域です。
円柱の中心部分が空洞になります。
