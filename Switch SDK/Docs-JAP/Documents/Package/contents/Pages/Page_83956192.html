はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK が提供する FS ライブラリとそれによって利用できるファイルシステムサービスの機能について説明します。

対象読者

NintendoSDK の FS ライブラリを使ってアプリケーションを開発される方を対象としています。

関連資料

nn::fs ライブラリの関数リファレンス、サンプル解説書と併せて参照してください。
