はじめに

NintendoSDK は、 動作環境 に示された環境上での動作をサポートしています。

NintendoSDK には、お使いの PC がその環境を満たしているかをチェックするツールがあります。

このドキュメントでは、そのツールの使い方を説明します。

環境チェックツールの使い方

お使いの PC が、NintendoSDK がサポートする環境を満たしているかをチェックするツールとして、GUI アプリケーションと CUI アプリケーションの 2 つが利用できます。

GUI 版 SdkEnvironmentChecker.bat あるいは、
Tools\SdkEnvironmentChecker\SdkEnvironmentChecker.exe
CUI 版

Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe

以下、それぞれの使い方を説明します。

GUI ツール

NintendoSDK フォルダ直下にある SdkEnvironmentChecker.bat をダブルクリックすると、次のような画面が表示されます。

各項目は、以下の内容をチェックします。

項目名 チェック内容 結果
Windows 7 (64bit)
Windows 10 (64bit) 		 NintendoSDK のサポートする OS がインストールされているかを確認します。

"Not available" と表示される場合は、サポート対象の OS がインストールされた環境を用意してください。
値 説明
Available サポート対象の OS がインストールされています。
Not available サポート対象の OS がインストールされていません。
.NET Framework 4.6.1 NintendoSDK のサポートする .NET Framework がインストールされているかを確認します。

"Not installed" と表示される場合は、サポート対象の .NET Framework バージョンをインストールしてください。
値 説明
Installed サポート対象の .NET Framework がインストールされいています。
New version installed サポート対象よりも新しい .NET Framework がインストールされています。
Not installed サポート対象の .NET Framework がインストールされていません。
NVN API Version

NintendoSDK の要求する NVN API と互換性のある NVIDIA Graphics Driver がインストールされているかを確認します。

参考：

PC 環境で NVN を利用する際に必要です。

詳細は、NVN の説明 をご確認ください。



"Driver not installed", "Not compatible" と表示される場合は、 NVN ライブラリ の「GPU ドライバのインストール方法 」に従ってドライバをインストールしてください。


ドライバをインストールしても "Driver not installed" と表示される場合は、 トラブルシューティングの「NVIDIA Graphics Driver をインストールしたのに環境チェックツールの「NVN API Version」の項目で "Driver not installed" と表示される」を参照し、PC の設定を確認してください。

 
"GPU not supported" と表示される場合は、 NVN ライブラリ の  「対応 GPU とサポートされる機能」を確認し、サポートされている GPU を用意してください。



値 説明
Driver not installed NVIDIA Graphics Driver がインストールされていません。
GPU not supported NVIDIA Graphics Driver はインストールされていますが、サポート対象の GPU ではありません。
Compatible NintendoSDK の要求する NVN API と互換性のある NVIDIA Graphics Driver がインストールされています。
Not compatible NintendoSDK の要求する NVN API と互換性のない NVIDIA Graphics Driver がインストールされています
Microsoft Visual C++ XXXX Redistributable (YYY)

NintendoSDK のサポートする Microsoft Visual C++ Redistributable がインストールされているかを確認します。


"Not installed" と表示される場合は、 Installers\VcRuntime から Microsoft Visual C++ Redistributable をインストールしてください。

値 説明
Installed サポート対象の Microsoft Visual C++ Runtime がインストールされいています。
New version installed サポート対象よりも新しい Microsoft Visual C++ Runtime がインストールされいています。
Not installed サポート対象の Microsoft Visual C++ Runtime がインストールされていません。

--exclude オプション

特定の項目をチェック対象から除外したい場合は --exclude オプションを使用します。
--exclude オプションをつけない場合は、全項目をチェックします。

--exclude オプションを使う場合は、NintendoSDK フォルダ直下にある SdkEnvironmentChecker.bat ではなく、Tools\SdkEnvironmentChecker\SdkEnvironmentChecker.exe に対して、オプションを追加して起動してください。

呼び出し例：

> Tools\SdkEnvironmentChecker\SdkEnvironmentChecker.exe --exclude=nvn

現在指定できる値は、次の通りです。

値 意味
nvn チェック対象から NVN API Version のチェックを除外します。


CUI ツール

コマンドライン上で、 Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe を実行すると、次のような出力が得られます。

> Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe
[NG] Microsoft Visual C++ 2015 Redistributable (x86) - 14.0.23918.0
[NG] Microsoft Visual C++ 2015 Redistributable (x64) - 14.0.23918.0

引数なしで起動すると、上記のように [NG] の行のみが標準エラー出力に出力されます。
[NG] の行は、お使いの PC が満たしていない環境を示します。
[NG] が出力された場合は、前項「GUI ツール」のカコミ記事などを参考に足りない環境をインストールしてください。 

上記のほか、[OK] (New version installed) (※ NX Addon 5.0.0 未満では「[OK but newer]」と表記されます) という表示のある行が標準出力に出力されることがあります。
これは、 SDK がサポートしているバージョンより新しいバージョンのプログラムがインストールされていることを示します。

[NG] がひとつでも出力されると、 SdkEnvironmentCheckerConsole.exe は終了コードとして 1 を返します。
[NG] が出力されない場合は、終了コードとして 0 を返します。

また、NX Addon 5.0.0 以降では、関連ファイルの不足などで致命的な問題が発生し、ツールの実行に失敗した場合は終了コードとして -1 を返します。

--verbose オプション

--verbose オプションを追加して起動すると、満たされている環境も含めて表示されます。

[OK] の行は、お使いの PC が満たしている環境です。標準出力に出力されます。
(繰り返しになりますが) [NG] の行は、お使いの PC が満たしていない環境です。標準エラー出力に出力されます。

> Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe --verbose
[OK] (Available)        Windows 7(64bit) Windows 10(64bit)
[OK] (New version installed)    .NET Framework 4.6.1
[NG] (Not compatible)   NVN API Version 52.32
[OK] (Installed)        Microsoft Visual C++ 2013 Redistributable (x86) - 12.0.30501
[OK] (Installed)        Microsoft Visual C++ 2013 Redistributable (x64) - 12.0.30501
[OK] (New version installed)    Microsoft Visual C++ 2017 Redistributable (x86) - 14.0.24210.0
[OK] (New version installed)    Microsoft Visual C++ 2017 Redistributable (x64) - 14.0.24210.0


NX Addon 5.0.0 未満では次のような出力になります。

> Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe --verbose
[OK]           Windows 7(64bit), Windows 10(64bit)
[OK but newer] .NET Framework 4.6.1
[OK]           NVN API Version 52.32
[OK]           Microsoft Visual C++ 2013 Redistributable (x86) - 12.0.30501.0
[OK]           Microsoft Visual C++ 2013 Redistributable (x64) - 12.0.30501.0
[OK but newer] Microsoft Visual C++ 2015 Redistributable (x86) - 14.0.24210.0
[OK but newer] Microsoft Visual C++ 2015 Redistributable (x64) - 14.0.24210.0

[OK but newer] の行は標準出力に出力されます。

--exclude オプション

特定の項目をチェック対象から除外したい場合は --exclude オプションを使用します。
--exclude オプションをつけない場合は、全項目をチェックします。

呼び出し例 (--exclude オプションをつけた場合)

> Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe --exclude=nvn
[OK] (New version installed)    .NET Framework 4.6.1
[OK] (New version installed)    Microsoft Visual C++ 2017 Redistributable (x86) - 14.0.24210.0
[OK] (New version installed)    Microsoft Visual C++ 2015 Redistributable (x64) - 14.0.24210.0


NX Addon 5.0.0 未満では次のような出力になります。

> Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe --exclude=nvn
[OK but newer] .NET Framework 4.6.1
[OK but newer] Microsoft Visual C++ 2015 Redistributable (x86) - 14.0.24210.0
[OK but newer] Microsoft Visual C++ 2015 Redistributable (x64) - 14.0.24210.0


呼び出し例 (--exclude オプションをつけない場合)

> Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe
[OK] (New version installed)    .NET Framework 4.6.1
[NG] (Not compatible)   NVN API Version 52.32
[OK] (New version installed)    Microsoft Visual C++ 2017 Redistributable (x86) - 14.0.24210.0
[OK] (New version installed)    Microsoft Visual C++ 2017 Redistributable (x64) - 14.0.24210.0


NX Addon 5.0.0 未満では次のような出力になります。

> Tools\SdkEnvironmentChecker\SdkEnvironmentCheckerConsole.exe
[OK but newer] .NET Framework 4.6.1
[NG]           NVN API Version 53.100
[OK but newer] Microsoft Visual C++ 2015 Redistributable (x86) - 14.0.24210.0
[OK but newer] Microsoft Visual C++ 2015 Redistributable (x64) - 14.0.24210.0
