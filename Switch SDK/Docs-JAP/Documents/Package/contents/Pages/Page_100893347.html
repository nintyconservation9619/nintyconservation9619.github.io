ツールリファレンス

概要

NVN と共に提供されるグラフィックスツール群について説明します。

まずはじめに、NVN のそれぞれのツールに共通な内容について説明します。

それぞれのツールについての詳細は、それぞれのツールのページを参照してください。

配置

ツールの配置については、それぞれのページを参照してください。

提供形態

ツールは、実機向けライブラリ, .dll, .exe のいずれかの形で提供されます。

実機向けライブラリの提供形態については、以下のページを参照してください。

  • ライブラリファイルの提供形態と利用方法 (NX Addon)

.dll 形式のツールは、Windows 上のプログラムから使用されることを意図しています。

32bit 版のツール

64bit 版と区別するために、32bit 版の .dll, .exe はツール名の末尾に "32" がついています。
これら 32bit 版のルールに従っていないツールは 64bit 版であることを表します。
