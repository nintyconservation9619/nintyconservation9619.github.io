EffectMaker で使用するファイル

EffectMaker で使用するファイルについて説明します。

ワークスペースファイル .work

ワークスペース上の作業状態を記録できるファイルです。
ワークスペースファイルは以下の情報を保持します。

  • 「 プレビューファイル（.prev）」の設定内容
  • ビューアの設定内容と背景用「 テクスチャファイル（ .ftxb）」のリンク情報
  • 「 モデルファイル（.fmdb）」と「アニメーションファイル（ .fskb）」のリンク情報
  • 「 エミッタセットファイル（ .eset）」のリンク情報
エミッタセットファイル .eset エフェクトファイルです。以下の情報を保持します。
  • エミッタセットの設定内容
  • パーティクルの見た目を決める「 テクスチャファイル（ .ftxb）」の名前
  • エミッタやパーティクルとして利用できる「 プリミティブファイル（.fmdb）」の名前
プレビューファイル .prev エミッタセットに対してゲームアプリケーション側で制御可能な内容を EffectViewer で確認できるファイルです。

以下の情報を保持します。

  • 「 プレビューファイル（.prev）」の設定内容
  • 「 エミッタセットファイル（ .eset）」の名前
  連携用モデルファイル .fmdb

ビューアでエフェクトと合わせて確認する為の連携用モデルファイルです。以下の情報を保持します。

  • モデルが利用している「 テクスチャファイル（ .ftxb）」の名前
  • モデルが利用しているスケルタルアニメーションファイル（fmab）の名前
