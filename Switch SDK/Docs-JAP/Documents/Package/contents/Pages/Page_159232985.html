atk の主な変更点

ユーザーエフェクト機能の追加

波形に対して任意の信号処理を行うことができる、ユーザーエフェクトの機能を追加しました。ユーザーは nn::atk::EffectAux を継承したクラスを用意することで、ユーザーエフェクトとして追加することができるようになります。 

詳しくは、AtkEffect サンプルやプログラミングガイドの オーディオエフェクト のドキュメントを参照ください。

メモリプール関連の変更

サンプルのメモリプール対応方法を変更

サンプルでメモリプール対応が必要なバッファについて、個々のバッファをメモリープールにアタッチしていましたが、メモリ効率の改善のためにまとめてアタッチするように修正しました。

アプリケーション側でメモリプール管理が必要なデータの追加

NintendoSDK 0.11.0 のドキュメントで、メモリプール管理が必要になるメモリ領域の種類について
記載漏れがありましたので追記を行いました。具体的には以下の２つです。 

  • メモリサウンドアーカイブ
  • ユーザー管理グループ

詳しくは、プログラミングガイドの メモリー管理 のドキュメントを参照ください。
