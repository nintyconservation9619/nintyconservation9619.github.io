ライブラリ

概要

PC 版 Vulkan ではヘッダファイルのみが提供されます。 ライブラリの静的リンクは不要で Vulkan API のエントリポイントはランタイムでグラフィックスドライバから関数ポインタを取得します。具体的な使用方法は利用されている PC の GPU ベンダによって異なりますので Vulkan を参照して下さい。

NX 版 Vulkan ではヘッダファイルに加えて、専用のライブラリが提供されます。

ヘッダファイル

Vulkan は、C インターフェイスを定義する vulkan/vulkan.h と C++ インターフェイスを定義する vulkan/vulkan.hpp を提供します。

その他の必要なヘッダも合わせて、以下のディレクトリに配置されます。

$NINTENDO_SDK_ROOT/Include/vulkan
