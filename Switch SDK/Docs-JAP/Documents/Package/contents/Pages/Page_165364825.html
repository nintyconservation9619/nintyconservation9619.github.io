「カウンタ」タブの概要

「カウンタ」タブには、プロファイリングでサンプリングされたすべてのカウンタが表示されます。例えば、パフォーマンスカウンタとユーザーが記録したカウンタなどです。

「カウンタ」タブのツールバー

「カウンタ」タブのツールバーには以下の各種コントロールがあります。

「コア」フィルタ処理ドロップダウンと「コア」列

ツールバーの「コア」ドロップダウンをクリックすると、コアすべての表示と個別のコアの表示を切り替えることができます。プロファイルを取得済みのコアのみが「コア」ドロップダウンに表示されます。デフォルトでは「コアすべて」が選択されており、新規プロファイルを取得したり読み込んだりするたびに、「コア」ドロップダウンの選択が「コアすべて」に戻ります。

例えば、コア 1 などの特定のコアを選択した場合、同じコアに属する関数のみが表示されるので、「コア」列は非表示になります。これにより、特定のコアに注目できるほか、タブの横方向のスペースを有効に活用できるようになります。

「グループ」フィルタ処理ドロップダウンと「グループ」列

「カウンタ」ツールバーの「グループ」ドロップダウンでは、全グループ、またはプロファイルに存在するいずれかのグループ（パフォーマンスカウンタなど）のどちらかを選択できます。プロファイルにあるグループのみが「グループ」ドロップダウンの一覧に表示されます。デフォルトでは「全グループ」が選択されており、新規プロファイルを取得したり読み込んだりするたびに、「グループ」ドロップダウンの選択が「全グループ」に戻ります。

パフォーマンスカウンタなどの特定のグループを選択した場合、同じタイプのグループのみが表示されるので、「グループ」列は非表示になります。

特に、パフォーマンスカウンタグループの場合は、どのパフォーマンスカウンタグループがプロファイルで記録されたかについての詳しい情報がドロップダウンに表示されます。例えば、サンプルグループという名前のパフォーマンスカウンタグループがプロファイリングされた場合、「グループ」ドロップダウンには「パフォーマンスカウンタ（サンプルグループ）」のエントリが表示されます。

「すべて選択解除」ボタン

選択したすべての項目を一度に選択解除するには、「すべて選択解除」ボタンをクリックします。

「類似項目の選択」切り替えボタン

「類似項目の選択」ボタンが有効な状態で、ある項目を選択または選択解除すると、他の同じような名前の項目も選択または選択解除されます。

「類似項目の選択」では、複数のコアにわたって類似のカウンタを選択できます。例えば、パフォーマンスカウンタグループをプロファイリングした場合、コア 0 にある先頭のカウンタを 1 回クリックすると、プロファイリングされたすべてのコアで先頭にあるカウンタが選択（または選択解除）されます。

「コピー」ボタン

「コピー」ボタンをクリックすると、現在適用されている並べ替えとフィルタの設定に従って、現在表示されている内容がすべてクリップボードにコピーされます。その内容を、メール、ドキュメント、Excel のスプレッドシートなどに貼り付けることができます。コピーした内容はタブ区切り形式になっているので、Excel のスプレッドシートに貼り付けたときに列の構造が維持されます。

カウンタのフィルタ処理

カウンタは、「カウンタ」タブの下部にある「フィルタ」テキストボックスを使用してフィルタ処理できます。このテキストボックスの動作は、「関数」タブの「フィルタ」テキストボックスと同様です。
