「時間スケール調整」について説明します。

時間スケール調整

　 　　 　
A

操作対象

レイアウト全体

レイアウト全体のカーブのスケールを調整します。

現在編集しているカーブ

現在選択しているカーブのみスケールを調整します。
B 操作範囲

全範囲

操作対象となっているカーブの全てのキーを選択します。
指定範囲

指定範囲の始点と終点を設定し、範囲内のキーを選択します。
C

スケール方法

選択キーだけを変更する

チェックを入れると選択したキーのみスケールを変更します。

新しい始点終点を指定

キーの新しい始点と終点の値を入力しスケールを変更します。

倍率と基点を指定

キーの時間と値を、倍率と基点で指定してスケールを変更します。
