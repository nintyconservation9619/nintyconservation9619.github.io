フェードアウト時間

ここでは Fade コマンド実行時のフェードアウトに要する時間を設定する機能「フェードアウト時間」について説明します。

概要

Fade コマンドが実行された時にここで設定した値をかけてパーティクルを消滅させることができます。
「αフェード」や「スケールフェード」が適用された時に利用できる機能です。
以下は「 設定項目」の概要です。

フェードアウト時間

Fadeコマンドが実行されてからの経過時間を設定します。

設定例

以下は「αフェード」ON で「フェードアウト時間」の設定値を変えた場合の比較画像です。

A 0 Fade コマンドが実行された時点でパーティクルは消滅します。
B 100 100フレームをかけ、アルファフェードとスケールフェードで徐々に消滅します。
