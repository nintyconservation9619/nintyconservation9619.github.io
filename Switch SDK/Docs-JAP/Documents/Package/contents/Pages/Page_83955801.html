機能説明

3DEditor の機能について説明します。

プロジェクト

3DEditor が起動中に、必ずひとつ存在して 3DEditor の作業状況を管理します。
アニメーションセット

複数のアニメーションをまとめたものです。
ユーザー設定

ユーザー設定ファイルを編集することで、3DEditor の動作をカスタマイズできます。
チーム設定

環境パスで指定したチーム設定フォルダ内に各設定ファイルを置くことにより、3DEditor の動作を一部変更することができます。
追加参照パス

追加参照パスは、チーム設定内に設定するファイル入出力設定ファイル ( FileIo.config ) やモデルごとに設定され参照ファイルの検索に使われます。
スタートアップパス

スタートアップパスを設定することにより 3DEditor は起動時に指定したファイルを開きます。
アタッチパス

アタッチパスは、ランタイム連携のモデルアタッチ及びシェーダーアタッチ時に参照するファイルの検索に使われます。
親マテリアル参照パス 親マテリアル参照パスは、親マテリアル追加・編集ダイアログに表示されるマテリアルを持つモデルの検索に使われます。
マテリアル参照

3DEditor のUI 操作で、外部のマテリアルを参照する設定ができます。

コマンドライン

3DEditor をコマンドラインから起動する際に、オプションを指定すると、3DEditor 起動時の設定を変更できます。
3DEditorBridge 3DEditorBridge は 3DEditor と外部ツールとの連携を行うためのコマンドラインツールです。
