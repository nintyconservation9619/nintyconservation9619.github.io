はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK が提供する NFC ライブラリとそれによって利用できる NFC タグへのアクセス機能について説明します。

注意：

本ライブラリを使用するためには、必ず NFC 機能を使用するための環境セットアップを行う必要があります。

注意：

NFC ライブラリには NFP ライブラリと同様の機能があります。そのため、本ドキュメントにおいては内容が重複する部分に関して「NFP ライブラリ」の内容を参照するようにしています。参照の際に「NFP」や「nfp」といった記述を「NFC」や「nfc」に読み替えください。

対象読者

NintendoSDK の NFC ライブラリを使ってアプリケーションを開発される方を対象としています。

各章の説明

「マニュアル」では NFC ライブラリが提供する個々の機能について使用方法を解説します。

関連資料

API リファレンス、サンプル解説書、「NFP ライブラリ」と併せて参照してください。

NFC ライブラリでできること

NFC ライブラリでは以下の機能に対応しています。

機能 備考
NFC タグの検知 NFC タグを検知できます。なお本ドキュメントでは、検知可能な IC タグや IC カードを総称して NFC タグと記載しています。
MIFARE タグの読み書き MIFARE タグの読み込み・書き込みを行う専用の関数を用意しています。MIFARE は Taspo (タスポ) などに利用されている非接触 IC タグの規格の一種です。
NFC タグへのコマンド送信 NFC タグへのコマンドの送信、および NFC タグからのリプライデータの受信が可能です。

また、タグは同時に 1 つしか使えません。タグを同時に2 つ以上を認識させようとすることはサポート対象外となり、正常に動作しません。
