テキストを円形に配置する

ここでは「文字毎の変換」機能を利用して円形上に文字を配置できる機能「テキストを円形に配置する」について説明します。

概要

「テキストを円形に配置する」は、テキストボックスペインで設定した文字列を、円弧に沿って配置できる機能です。
以下は「テキストを円形に配置する」の設定項目の解説です。

A 開始角度 円弧の開始角度と終了角度を指定します。

円の右方向を 0 とし、反時計回りに 360 で一周します。
B 終了角度
C 円の半径 X

円の横方向と縦方向半径を指定します。

縦横のサイズを変えることで、楕円形に文字を沿わせることもできます。
D 円の半径 Y
E 分割数 円をカーブで表すときのキーの数を指定します。
多くすると円弧の形状が滑らかになりますが、処理負荷に影響が出ます。
F 文字の向き

円形に配置された文字毎の傾きからの相対的な角度を指定します。
図では文字の「原点」を「中央」に、「文字の向き」は 90 に設定されています。

使用例

以下は「テキストを円形に配置する」の使用例と設定値です。

LayoutEditor LayoutViewer

設定内容 開始角度：0、終了角度：360、円の半径X,Y：300、分割数：10、文字の向：90

「 テキストボックスペイン」の背面に正円テクスチャを貼った「 ピクチャペイン」を配置しています。

「テキストを円形に配置する」の機能を利用し、正円テクスチャの円に沿うように文字列を並べています。
