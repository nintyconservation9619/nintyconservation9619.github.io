プリミティブファイル

プリミティブファイルは、エミッタセットファイルで利用するプリミティブ用モデルファイルです。
ここでは EffectMaker におけるプリミティブファイルの扱い方について説明します。

プリミティブファイルの保存場所

エミッタセットファイル（ .eset）には、パーティクル形状としてプリミティブを使用した際にモデルファイルの名前のみが保持され、パス情報は保持されません。
その為、 EffectMaker はエミッタセットファイルを開く際、ファイルに保持されているモデルファイル名を元にして同じ名前のモデルファイルを検索します。

モデルファイルの保存場所は、関連付けされたエミッタセットファイルと同じドライブ内の検索可能なフォルダです。

以下の画像は、 EffectMaker で開くエミッタセットファイル（ .eset）を「EmitterSet」、そのファイルに名前が保持されているモデルファイルを「Primitive」とした場合のモデルファイルの検索 順番と検索可能なフォルダを表しています。

  1. EffectMaker は、エミッタセットファイル「EmitterSet.eset」を開く際、同じフォルダにあるモデルファイルを最初に検索します。
  2. 続いてエミッタセットファイル「EmitterSet.eset」のフォルダ直下にある「Primitives」フォルダ内を検索します。
  3. 以降はエミッタセットファイル「EmitterSet.eset」を起点として上位のフォルダにある「Primitives」フォルダを、モデルファイルが見つかるまで検索します。
  4. ドライブ最上位にある「Primitives」フォルダです。検索可能な最後のフォルダです。
    この時点でモデルファイルが見つからなかった場合、 EffectMaker は警告ダイアログを表示します。

EffectMaker は、「Primitives」フォルダ名の大文字小文字を区別しません。
