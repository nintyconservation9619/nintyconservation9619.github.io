NintendoSDK 5.3.0 における変更点を紹介します。

SoundMaker, コマンドラインツール

SoundMaker, SoundProjectWaveConverter の波形化を行う際、終端部分まで波形化を行うように改善

SoundMaker, SoundProjectWaveConverter でサウンドの波形化を行う際、終端部分まで波形化を行うように改善しました。

これまで再生途中で波形化が終了していたものが、再生終了まで波形化を行うようになります。

この修正により、以前のバージョンの SoundMaker, SoundProjectWaveConverter の波形化の出力結果と終端部分が異なる場合がある点にご注意ください。
