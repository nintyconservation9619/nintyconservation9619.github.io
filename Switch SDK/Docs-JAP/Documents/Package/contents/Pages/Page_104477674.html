XZ 移動半径

ここでは選択しているノードが移動する円状ルートの半径を設定する機能「XZ 移動半径」について説明します。

概要

「XZ 移動半径」は、選択しているノードが移動する円状ルートの半径を設定する機能です。
値を大きくすることによって、移動距離を長くできます。
以下は「 設定項目」の概要です。

XZ 移動半径

円状ルートの半径を「 ダイヤル」で設定します。

設定例

以下は「XZ 移動半径」の設定値による比較画像です。

半径 = 8
 半径 = 40
