使用方法

法線マップに snorm_bc5 など XY 成分のみのテクスチャーフォーマットを使用して、ピクセルシェーダで Z 成分を計算する場合、元の法線の Z 成分がマイナスだと正しく表示されません。
このプラグインは法線の Z 成分がマイナスになっているピクセルを検出するために利用します。

処理を行うレイヤーを選択して、[ファイル] → [自動処理] → [Nintendo Select Negative Z Normal] を実行します。

法線の Z 成分がマイナスになっているピクセルが存在すれば選択されます。
存在しなければ "All pixels have positive Z" というメッセージが表示されます。

注意事項

マイナス Z 法線選択プラグインを使用する際の注意事項です。

イメージモード

イメージモードが RGB カラー以外のファイルや、8 bit/チャンネルでないファイルでは実行できません。

レイヤー

非表示のレイヤーを選択して実行するとエラーになります。
有効なピクセルのない完全に透明なレイヤーを選択して実行するとエラーになります。
