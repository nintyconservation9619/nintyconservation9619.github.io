テクスチャ名のフォーマットを表す接尾辞の仕様

はじめに

LayoutConverter でコンバートしたテクスチャ名にはフォーマットを表す接尾辞が付きますが、その仕様について解説します。

フォーマットを表す接尾辞の仕様

接尾辞

フォーマットを表す接尾辞は、以下の 2 ～ 3 文字で表されます。

1 文字目

1 文字目は、インダイレクト用かどうかを表す下記の記号です。

インダイレクト用である +
インダイレクト用ではない ^

2 ～ 3 文字目

2 ～ 3 文字目は、テクスチャフォーマットを表す下記の記号です。

フォーマットを列挙している enum (flyt.TexelFormat) の先頭から順に、「a、b、c、d、…、z、_A、_B、_C、_D、…_Z」という規則で定めます。

Windows ではファイル名の大文字・小文字を区別しないため、大文字のものには先頭に「_」を付加しています。
