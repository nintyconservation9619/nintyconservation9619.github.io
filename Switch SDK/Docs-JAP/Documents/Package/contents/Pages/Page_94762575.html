NX Addon で追加される機能

NintendoSDK ファミリーの各種 NX Addon パッケージを追加することで、次のような機能が利用可能になります。

項目 概要 制限・予定 必要なパッケージ
全般
  • NX API セットのサポート
  • Joy-Con, Switch Pro Controller のサポート


  • NintendoSDK NX Addon
開発機の制御

SDEV / EDEV を制御するためのツールを提供します。
  • NintendoSDK NX Addon
  • NintendoSDK SystemUpdater for NX
ALBUM アプリケーションからアルバムにアクセスする機能を提供するライブラリです。
  • Windows プラットフォームでのビルドはサポートされていますが、アルバム機能は利用できません
  • NintendoSDK NX Addon 
AOC 追加コンテンツを扱う機能を提供するライブラリです。
  • 現在 Windows プラットフォームはサポートされていません
  • NintendoSDK NX Addon
EC 追加コンテンツや期間型サービスアイテムの販売画面へのジャンプ機能などを提供するライブラリです。
  • 現在 Windows プラットフォームはサポートされていません
  • NintendoSDK NX Addon
FS

以下の機能が利用可能になります。

  • 開発用途での SD カード上の FAT32 ファイルシステムの読み書き
  • NintendoSDK NX Addon
GFX

NVN で実装された高速に動作する GFX ライブラリです
  • NintendoSDK NX Addon
  • NVIDIA Graphics Driver
HID

hid ライブラリに以下の機能が追加されます。

  • Joy-Con のサポート

キャプチャーボタンのサポート

キャプチャーボタンの機能として、実行中のアプリケーションのスクリーンショット撮影および動画撮影機能がサポートされます。
  • 動画撮影機能を有効化したアプリケーションには性能面で影響があります。
    詳細は実装に際しての注意事項 (NX Addon)を参照してください。
IRSENSOR モーションIRカメラの機能を使用することができます。

近距離無線通信 amiibo へのアクセスをはじめとする、近距離無線通信を扱う機能を提供します。
  • NintendoSDK NX Addon
LDN 中継器を介さない無線通信を取り扱うローカル通信ライブラリです。
  • 現在 Windows プラットフォームはサポートされていません
  • NintendoSDK NX Addon
MOVIE 動画ファイルを扱う機能を提供するライブラリです。
  • 現在 Windows プラットフォームはサポートされていません。
  • NintendoSDK NX Addon
NVN
  • NX 向け低レベルグラフィックス機能を提供する API です
  • Windows プラットフォームではサポートされる GPU と NintendoSDK が提供するグラフィクスドライバをインストールする必要があります
  • NintendoSDK NX Addon
  • NVIDIA Graphics Driver
OE プログラムの動作環境に関する制御を行なう API です。
以下の機能を含んでいます。
  • プログラムへの通知メッセージに関する機能
  • プログラム動作に関する機能
  • 性能選択に関する機能
  • 現在 Windows プラットフォームはサポートされていません
  • NintendoSDK NX Addon
OpenGL Khronos Group が策定するグラフィックス API です。
  • NintendoSDK NX Addon
PL

プラットフォーム固有の機能を提供するライブラリです。

以下の機能を含んでいます。

  • 共有フォントに関する機能
  • USB インターフェースへの接続を無効にする機能


  • NintendoSDK NX Addon
RO

動的モジュールを扱う機能を提供するライブラリです。
  • 現在 Windows プラットフォームはサポートされていません
  • NintendoSDK NX Addon
VI

ディスプレイへのアクセス機能を提供する API です。
以下の機能を含んでいます。

  • レイヤーの作成や、フォーマットやサイズの設定などの基本的なディスプレイ出力の機能
  • NintendoSDK NX Addon
Vulkan Khronos Group が策定するグラフィックス API です。
  • NintendoSDK NX Addon
NintendoWare
  • グラフィックス系モジュール (nn::g3d, nn::vfx, nn::ui2d など) について、NVN および NVN 上に実装された GFX により高速化された実装を利用できます


  • NintendoSDK NX Addon
  • NintendoSDK NintendoWare NX Addon
  • NVIDIA Graphics Driver
システムメニュー

HOMEボタンや電源ボタンで呼び出せるメニュー全般をシステムメニューと呼びます。

システムメニューには以下の種類があります。

  • エントランス（ロック解除画面を含む）
  • HOMEメニュー
  • フルランチャー
  • クイック設定
  • 電源メニュー
  • NintendoSDK NX Addon
システムアプリ

HOMEメニューから起動することができる本体内蔵のアプリケーションのことをシステムアプリと呼びます。

  • マイページ
  • ニュース
  • アルバム
  • 本体設定

以下のシステムアプリは今後追加予定です。

  • ニンテンドーeショップ
  • NintendoSDK NX Addon
ライブラリアプレット

ライブラリアプレット：
特定の機能ごとにモジュール化された小さなアプリケーションのことです。
NXアプリケーションプレイ中に呼び出し、ライブラリアプレット終了後、NXアプリケーションに戻ることができます。

  • アカウント管理アプレット
  • コントローラーサポートアプレット
  • ソフトウェアキーボードアプレット
  • マイページアプレット
  • Mii作成・管理アプレット（ROM提出には別途申請が必要）
  • amiibo設定アプレット
  • ECアプレット
  • Webアプレット
  • オフラインWebアプレット
  • エラービューアアプレット
  • アルバムアプレット（NX Addon 5.1.0 より）


起動ライブラリ：
ライブラリアプレットを起動するためのライブラリです。


  • NintendoSDK NX Addon
注意：

Mii（似顔絵ライブラリ、Mii作成・管理アプレット）を利用したROMを提出するには別途申請が必要になります。ROM提出希望の方は、任天堂窓口まで問い合わせてください。
