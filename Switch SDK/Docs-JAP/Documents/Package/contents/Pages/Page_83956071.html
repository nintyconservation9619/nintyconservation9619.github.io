ユーザー設定ファイルフォーマット

ユーザー設定ファイルに含まれる要素を説明します。
3dEditor.user.config は XML 形式のファイルです。

ここでは、3dEditor.user.config の例に添って、使用する各要素について説明します。

3dEditor.user.config の例

3dEditor.user.config の例です。

<?xml version="1.0" encoding="utf-8"?>
<UserSetting Version="0.0.0">
    <SearchPath Name="folderB" Recursive="false">C:\folderB</SearchPath>
    <StartUpPath>%DATA_FOLDER%\modelA\modelA.f3pj</StartUpPath>
    <ReloadOptions>
        <Model Reload="true" ShowDialog="true" />
        <Texture Reload="true" ShowDialog="false" />
        <SkeletalAnimation Reload="true" ShowDialog="false" />
        <MaterialAnimation Reload="true" ShowDialog="false" />
        <ShaderParameterAnimation Reload="true" ShowDialog="false" />
        <ColorAnimation Reload="true" ShowDialog="false" />
        <TextureSrtAnimation Reload="true" ShowDialog="false" />
        <TexturePatternAnimation Reload="true" ShowDialog="false" />
        <BoneVisibilityAnimation Reload="true" ShowDialog="false" />
        <MaterialVisibilityAnimation Reload="true" ShowDialog="false" />
        <ShapeAnimation Reload="true" ShowDialog="false" />
        <SceneAnimation Reload="true" ShowDialog="false" />
        <ShaderDefinition Reload="false" ShowDialog="false" />
    </ReloadOptions>
    <EditHistory ClearAfterFileClosed="false" ClearAfterFileReloaded="false" ClearAfterFileSaved="false" MaximumItemCount="10000" />
    <ExternalProgram TextEditor="notepad.exe" TextEditorFileOpenOption="&quot;%1&quot;" EnableRuntimeDebugLog="false" />
    <IO AutoFix="false" AutoReloadUpdatedOnLoad="true" DumpShaderSource="true" MaximumRecentlyUsedFileCount="10" MaximumRecentlyUsedProjectCount="10">
        <DumpShaderSourcePath>C:\dumpFolder</DumpShaderSourcePath>
    </IO>
    <UI FixedSlider="false" />
</UserSetting>

<UserSetting>

ファイルのルート要素です。

属性

名前 説明
Version 3dEditor.user.config のバージョンです。現在のバージョンは 0.0.0 です。

内容

以下の要素を含みます。

  • <SearchPath> : 追加参照パスの設定です。0個以上出現します。
  • <StartUpPath> : スタートアップパスの設定です。0個以上出現します。
  • <ReloadOptions> : ファイル更新時の再読み込みの設定です。最大一つ出現します。
  • <EditHistory> : 編集履歴の設定です。最大一つ出現します。
  • <ExternalProgram> : 外部プログラムの設定です。最大一つ出現します。
  • <IO> : 入出力の設定です。最大一つ出現します。
  • <UI> : コントロールの設定です。最大一つ出現します。

 

<SearchPath>

追加参照パスの設定を含む要素です。

属性

名前 説明
Name 追加参照パス名です。相対パス指定の場合は省略可能です。
Recursive 参照パス以下のフォルダのサブフォルダも検索対象にするか指定します。true または false を設定します。省略した場合は false として扱われます。

内容

フォルダの相対パスまたは絶対パスです。% で囲われた環境変数を含めることができます。

<StartUpPath>

スタートアップパスの設定です。

属性

名前 説明
Base 起点となる追加参照パスの名前です。スタートアップパスが追加参照パスからの相対パスでない場合は省略します。

内容

ファイルまたはフォルダの相対パスまたは絶対パスです。% で囲われた環境変数を含めることができます。

<ReloadOptions>

ファイル更新時の再読み込みの設定です。

内容

以下の要素が最大一つ出現します。

省略した場合は、いずれの種類のファイルの再読み込みも行われません。

  • <Model> : モデルファイルの再読み込み設定です。
  • <Texture> : テクスチャファイルの再読み込み設定です。
  • <SkeletalAnimation> : スケルタルアニメファイルの再読み込み設定です。
  • <ShaderParameterAnimation> : シェーダパラメータアニメファイルの再読み込み設定です。
  • <ColorAnimation> : カラーアニメファイルの再読み込み設定です。
  • <TextureSrtAnimation> : テクスチャ SRT アニメファイルの再読み込み設定です。
  • <TexturePatternAnimation> : テクスチャパターンファイルの再読み込み設定です。
  • <BoneVisibilityAnimation> : ボーンビジビリティアニメファイルの再読み込み設定です。
  • <MaterialVisibilityAnimation> : マテリアルビジビリティアニメファイルの再読み込み設定です。
  • <MaterialAnimation> : マテリアルアニメファイルの再読み込み設定です。
  • <ShapeAnimation> : シェイプアニメファイルの再読み込み設定です。
  • <SceneAnimation> : シーンアニメファイルの再読み込み設定です。
  • <ShaderDefinition> : シェーダ定義ファイルの再読み込み設定です。

<Model> <Texture> <SkeletalAnimation> <ShaderParameterAnimation> <ColorAnimation>
<TextureSrtAnimation> <TexturePatternAnimation> <BoneVisibilityAnimation> <MaterialVisibilityAnimation> <MaterialAnimation> <ShapeAnimation> <SceneAnimation> <ShaderDefinition>

各種ファイルの再読み込み設定です。省略した場合は、ファイルの再読み込みが行われ、未保存のファイルを再読み込みする場合は確認ダイアログが表示されます。

内容

ファイルまたはフォルダの相対パスまたは絶対パスです。% で囲われた環境変数を含めることができます。

属性

名前 説明
Reload 他のアプリケーションでファイルが更新されたときに再読み込みを行うかどうかを指定します。
true または false を設定します。省略した場合は true として扱われます。
ShowDialog 再読み込みを行う前に確認ダイアログを表示するかを指定します。
true または false を設定します。省略した場合は false として扱われます。Reload が true のときのみ有効です。
false に設定しても未保存のファイルを再読み込みする時には、 確認ダイアログが表示されます。

<EditHistory>

編集履歴の設定を含む要素です。

属性

名前 説明
ClearAfterFileClosed ファイルを閉じたときに編集履歴を消去するかどうかを設定します。true または false を設定します。省略した場合は false として扱われます。
ClearAfterFileReloaded ファイルを再読み込みしたときに編集履歴を消去するかどうかを設定します。true または false を設定します。省略した場合は false として扱われます。
ClearAfterFileSaved ファイルを保存したときに編集履歴を消去するかどうかを設定します。true または false を設定します。省略した場合は false として扱われます。
MaximumItemCount 3DEditor が保持する編集履歴の項目数の上限を設定します。0 以上の整数を設定します。省略した場合は 10000 として扱われます。

<ExternalProgram>

外部プログラムの設定を含む要素です。

属性

名前 説明
TextEditor 3DEditor から呼び出すテキストエディタを設定します。テキストエディタの実行ファイルパスを設定します。
省略した場合は システム既定のテキストエディタが呼び出されます。
TextEditorFileOpenOption

テキストエディタ呼び出し時のオプションを設定します。
EnableRuntimeDebugLog 接続先アプリケーションのデバッグログを出力するかどうかを設定します。

<IO>

入出力の設定を含む要素です。

内容

以下の要素を含みます。

  • <DumpShaderSourcePath> : シェーダコンパイル時情報の出力先です。

属性

名前 説明
AutoFix モデル読み込み時にマテリアルの修正を自動で行うかどうかを設定します。 true または false を設定します。省略した場合は false として扱われます。
AutoReloadUpdatedOnLoad すでに読み込まれているファイルを再度開こうとした時にファイルが更新されていた場合、自動的に再読み込みをするかどうかを指定します。 true または false を設定します。省略した場合は false として扱われます。 false に設定すると確認ダイアログが表示されます。
DumpShaderSource シェーダ最適化時にコンパイル時情報を出力するかどうかを設定します。 true または false を設定します。省略した場合は false として扱われます。 コンパイル時情報を出力するにはDumpShaderSourcePathも設定する必要があります。
MaximumRecentlyUsedFileCount ファイルメニューに表示する最近使ったファイルの個数の上限を設定します。
MaximumRecentlyUsedProjectCount ファイルメニューに表示する最近使ったプロジェクトの個数の上限を設定します。

<DumpShaderSourcePath>

シェーダコンパイル時情報の出力先です。

内容

シェーダコンパイル時情報の出力先フォルダの絶対パスです。% で囲われた環境変数を含めることができます。

<UI>

コントロールの設定を含む要素です。

属性

名前 説明
FixedSlider マテリアルプロパティウィンドウのシェーダページにスライダコントロールを表示するかどうかを設定します。
true または false を設定します。省略した場合は false として扱われます。
