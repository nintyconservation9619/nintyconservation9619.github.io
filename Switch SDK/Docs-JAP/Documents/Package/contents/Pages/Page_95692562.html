「ビルボード」と「平面ポリゴン」

ここでは「ビルボード」と「平面ポリゴン」の違いについて説明します。

概要

「平面ポリゴン」はカメラの位置に依存しないパーティクルです。「XY平面のポリゴン」と「XZ平面のポリゴン」の２種類があります。
「ビルボード」はカメラの位置をパーティクルの向きに影響させるパーティクルタイプです。「ビルボード」と「Y軸ビルボード」の２種類があります。
以下の様に円状のテクスチャを貼ったポリゴンをカメラを動かしながら確認する事で「ビルボード」と「平面ポリゴン」の違いが解ります。

A ビルボード

パーティクルの全軸を回転させ、常にカメラ方向を向くパーティクルタイプです。

板ポリゴンに円形状のテクスチャを貼る事で球状に見えます。ボールや雪などのエフェクトに有効です。
B XY平面のポリゴン エミッタ座標系で XY 平面上にパーティクルの XY 平面が一致するようにパーティクルを配置します。
