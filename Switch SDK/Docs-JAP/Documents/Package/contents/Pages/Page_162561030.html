VariableIconBody

概要

nn::mii::VariableIconBody は、体格反映体つきアイコンを描画するための体部分のシェイプデータを保持するクラスです。

Mii の体格反映体つきアイコンを作成したい場合にこのクラスが必要となります。

リソースのロード

似顔絵リソースの中には体部分のデータは含まれていません。別途ファイルとロードする必要があります。

以下のファイルをロードしてください。

$(NINTENDO_SDK_ROOT)/Resources/Mii/Common/iconbody/MiiBody.dat

上記ファイル内に、男女両方の体モデルのデータが含まれています。

初期化/終了処理

初期化と終了処理の注意点に関して説明します。各関数の API リファレンスも合わせて参照してください。

初期化処理

nn::mii::VariableIconBody::Initialize() 関数を使用して初期化を行います。

初期化時に必要な体のリソースデータは、あらかじめ初期化前にメモリ上にロードしておく必要があります。

初期化用のメモリのサイズとアライメントは nn::mii::VariableIconBody::CalculateMemorySize() と nn::mii::VariableIconBody::CalculateMemoryAlignment() 関数を使用することで取得できます。

初期化用のメモリプールのサイズとアライメントは nn::mii::VariableIconBody::CalculateMemoryPoolSize() と nn::mii::VariableIconBody::CalcualteMemoryPoolAlignment() 関数を使用することで取得できます。初期化時に渡すメモリプールのプロパティが CpuCached な場合、初期化後にメモリプールのフラッシュを呼ぶ必要があります。

初期化時に渡したメモリ、メモリプールは終了処理を呼ぶまで保持する必要があります。体のリソースファイルは初期化後に破棄することが可能です。

終了処理

nn::mii::VariableIconBody::Finalize() 関数を呼ぶと終了処理を行います。終了処理が行われた後は初期化に使用したメモリとメモリプールの破棄を行うことができます。

DrawParam

nn::mii::VariableIconBody::GetDrawParam() 関数を使用することで、体の描画用のパラメータが取得できます。

nn::mii::CharInfo を渡して内部に設定されている性別に応じたパラメータを取得することができます。直接性別を指定することも可能です。

Matrix

以下の関数を使用することで体格反映体つきアイコンの描画用の Matrix が取得できます。

  • nn::mii::VariableIconBody::CalculateWorldMatrix()
    指定された性別と体格を元にワールドマトリックスを計算します。
  • nn::mii::VariableIconBody::StoreSkeleton()
    CalculateWorldMatrix() で取得したワールドマトリックスを元にスキニング用マトリックスを取得します。
  • nn::mii::VariableIconBody::StoreHeadWorldMatrix()
    Mii の顔部分（CharModel）を配置するワールドマトリックスを取得します。
  • nn::mii::VariableIconBody::StoreCameraMatrix()
    カメラ用のマトリックスを取得します。
