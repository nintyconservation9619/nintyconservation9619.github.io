カーブエディタウィンドウ

横軸を時間、縦軸を値としたグラフでエミッタ時間のアニメーションを作成及び編集できる「カーブエディタウィンドウ」について説明します。

A リストビュー

エミッタ時間でアニメーションする機能が一覧で表示されます。
「 アニメーション ON」の項目に対して「カーブビュー」でアニメーションを設定します。
B ツールバー

カーブエディタ操作する為のツール群です。

「 全体移動」をマウス左ボタンクリックにより開くウィンドウで指定した移動量分カーブ全体を時間軸方向に移動できます。
C カーブビュー

エフェクトを構成する各ノードをツリー状に表示します。

マウスの右クリックによりワークスペースでの編集メニューが開きます。
