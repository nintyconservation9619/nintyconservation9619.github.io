履歴の空気抵抗

ここでは、空気抵抗によって履歴の全体の速度を減衰または加速させる機能「履歴の空気抵抗」について説明します。

概要

「履歴の空気抵抗」は、「履歴」の速度を加速または減衰させる機能です。
履歴の初速や加速度を設定している場合、「履歴の空気抵抗」の値を設定することでストライプに影響を与えます。
履歴の初速や加速度が 0 の場合は、履歴の空気抵抗を設定してもストライプに影響しません。
以下は「 設定項目」の概要です。

履歴の空気抵抗

「履歴」にかかる空気抵抗を「 ダイヤル」や「 テキストボックス」により設定します。
値が 1.0 より大きいと履歴が加速し、1.0 より小さいと履歴が減衰します。
