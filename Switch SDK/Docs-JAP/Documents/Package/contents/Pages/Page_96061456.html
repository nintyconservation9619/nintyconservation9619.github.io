ソフトパーティクル「フラグメントで処理」

ここでは「ソフトパーティクル」の処理をパーティクルのフラグメント単位で行う機能「フラグメントで処理」について説明します。

概要

ソフトパーティクルの機能「フラグメントで処理」は、「ソフトパーティクル」をプリミティブポリゴン面のピクセル単位で処理する機能です。
頂点数が少ないプリミティブでもアルファの階調がきれいに見えます。
以下はパーティクルのあるフラグメントに対してアルファ値をもとめる計算式です。

  • 深度値 = 描画しようとするパーティクルフラグメントの奥行き値
  • Zバッファ = 既に描画されているピクセルの奥行き値

フラグメントのアルファ値（0.0 ～ 1.0） =  ( 深度値 - Zバッファ ) ÷ 設定値

以下は「 設定項目」の概要です。

「ソフトパーティクル」の設定を「フラグメントで処理」にすると「 フラグメントで処理」が表示されます。以下はグループ内の設定項目です。

関連リンク ソフトパーティクルの詳細についてはこちらを参照してください。

フェード開始距離

パーティクルがモデルと重なった箇所からの距離を「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は視点から「 █ パーティクル面のフラグメント」までの距離と「█ 床面の深度値」との「… 距離の差」によるアルファ値の変化について説明した画像と図です。
「フェード開始距離」は 10 に設定しています。
A … = 0 計算式は、 0 ÷ 設定値 10 = 0.0 でアルファ値は 0 になります。
B … = 5 計算式は、 5 ÷ 設定値 10 = 0.5 でアルファ値は中間の 0.5 になります。
C … = 20 計算式は、 20 ÷ 設定値 10 = 2.0 です。
アルファ値は 0.0 ～ 1.0 に収められる為 1.0 になります。

以下のように「… 距離の差」を短くすると、アルファ値を減衰できます。
