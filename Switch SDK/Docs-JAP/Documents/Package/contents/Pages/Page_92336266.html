オブジェクトビューの表示モードの切り替え

メインウィンドウの表示モードは、下図にある表示切り替えボタンで切り替えることができます。
初回起動時は、スケマティック表示になっています。

参考：

オブジェクトビューを分割表示している場合は、それぞれのビューで表示モードを変更できます。

表示モードの種類

関連リンク 各表示モードで使用できるショートカットキーは、こちらを参照してください。

モード 説明
スケマティック表示 モデルやモデルに含まれるノード（ボーン / シェイプ / マテリアル）の構造を図式的に表示します。
表示されている図を選択して右クリックメニューから編集をおこなったり、F3 キーを押してコメントなど入力することができます。
モデル一覧 3DEditor に読み込まれているモデルをリスト形式で表示します。
マテリアル一覧 モデルに含まれるマテリアルをリスト表示します。

マテリアル一覧表示では下記の機能や情報が表示されます。

  1. 表示する情報が多いのでページ単位のフィルタ機能
  2. マテリアルに対しての最適化シェーダの使用状況を表示

不正なデータが使用されている場合は、マテリアル名と不正なデータが赤字で表示されます。
ボーン一覧 モデルに含まれるボーンをリスト表示します。
シェイプ一覧 モデルに含まれるシェイプをリスト表示します。
シェイプに対応するマテリアル名も一覧に表示されます。
テクスチャ一覧 3DEditor に読み込まれているテクスチャをリスト表示します。

スケマティック表示時の操作

オブジェクトビューをスケマティック表示にした場合は、アイコンやスライダーで表示の操作ができます。

A

オブジェクト全体を中央に表示します。
B

選択したオブジェクトを中央に表示します。
C

オブジェクト全体を整列します。
D

選択したオブジェクト以下を整列します。
E ビューの表示倍率が表示されます。

F

ビューの表示倍率をスライダーで調整します。

G

ビューの 拡大 / 縮小 を ( 10 % 単位 ) で調整します。

一覧表示時の操作

オブジェクトビューを一覧表示にした場合は、以下の操作ができます。

A

現在選択されている表示モードです。右側にある表示モードの切り替えボタンで表示モードを変更します。
B

選択した表示モードの内容が、項目毎に一覧表示されます。項目を左クリックすると、表示されている内容がソートされます。

一覧表示の列項目設定

項目にマウスオーバーして、右クリックメニューから列項目の設定ダイアログを開くことができます。

マテリアル一覧での不正データ表示

オブジェクトビューをマテリアル一覧モードに変更すると、不正なデータが赤字で表示されます。

マテリアル一覧のフィルタ機能

マテリアル一覧では下記のページ単位でフィルタして表示できます。

全て

マテリアルの情報全てが、一覧表示されます。
全般

マテリアルプロパティウィンドウの全般ページの情報が一覧表示されます。
プレビュー

マテリアルプロパティウィンドウのプレビューページの情報が一覧表示されます。
サンプラ マテリアルプロパティウィンドウのサンプラページの情報が一覧表示されます。
シェーダー マテリアルプロパティウィンドウのシェーダーページの情報が一覧表示されます。

最適化シェーダーの使用状況

マテリアル一覧では、3DEditor に読み込まれているマテリアル単位の最適化状況をアイコンの色で確認できます。

参考：

黄色アイコン：シェーダー未送信
青色アイコン：最適化シェーダー送信済み

マテリアル一覧の最適化状態アイコンが表示されている列幅を広げることでコメントが確認できます。

画面分割と表示モード切替を使った操作

ここでは例として、オブジェクトビューを上下2画面に分割してそれぞれ「スケマティック表示」と「マテリアル一覧表示」にしてある場合を例に便利な使用方法を説明します。

下図のように2画面に分割することで、スケマティック表示とマテリアルの名前全部を同時に確認できます。
スケマティック表示のみではマテリアルの名前が一部しか確認できないので、2画面にすると確認しやすく便利です。
