ビューア設定

 ビューアについて設定できます。

接続

「 接続」は、EffectMaker 起動時に、ビューアへ自動的に接続するかを ON/OFF で切り替える機能です。
前回 EffectMaker を終了した時点で使用していたビューアが選択されます。

フレームレート

「 フレームレート」は、ビューアにおけるアニメーションのフレームレートをプリセットから選択できる機能です。

Win版ビューアのウィンドウサイズ

「 Win版ビューアのウィンドウサイズ」は、Windows 版ビューアのウィンドウサイズをプリセットから選択します。

参考：

「Win版ビューアのウィンドウサイズ」で設定した内容は、 EffectMaker  と Windows 版ビューアが接続状態で設定し、
「OK」ボタンを押してダイアログを閉じることで確定できます。確定した設定内容はビューアを再起動することで反映されます。
