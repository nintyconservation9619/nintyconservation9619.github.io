文字毎の変換

できること

テキストボックスペイン中の、文字について1文字毎に平行移動や回転といった変換処理を加えることができます。
また、変換処理をアニメーションすることができます。
たとえば、｢アーチ状に文字を配置する｣、｢文字を波打たせる｣といった特殊な表現が簡単に実現できます。

サンプルデータ

perCharacterTransform にサンプルデータがあります(Samples\Sources\Resources\Ui2d\perCharacterTransform\perCharacterTransform.flyt)。

仕組み

  • テキストボックス毎に、1文字毎の変換（平行移動と回転）をカーブで設定します
  • 変換カーブの評価開始時間（オフセット）と「評価時間幅（幅）」を設定し｢評価する区間｣を決めます
  • 変換カーブについて、「評価する区間」を文字数で均等に分割して、文字ごとの変換量が決まります
  • オフセットをアニメーションさせることができます

注意点

  • 文字毎の変換は、変換のカーブをアニメーションカーブとして扱っているため、部品を使う場合にはアニメーションと同様の制約があります。例えば、部品レイアウト内で文字毎の変換を設定した場合、その部品ペインをボディレイアウトに配置しても文字毎の変換は有効になりません。また、部品ペインに対して文字毎の変換を上書きすることもできません。
