プリファレンスファイルフォーマット

プリファレンスファイルについて説明します。
プリファレンスファイル (preference.xml) はインストーラで自動的にインストールされます。
以下のような要素を利用することにより、FontConverter の設定ができます。

 <PlatformPreference>

プリファレンスファイルのルート要素です。

内容

以下の要素を含みます。

  • <ConverterPlatformName> : プラットフォーム名の設定です。高々1個出現します。

<ConverterPlatformName>

プラットフォーム名の設定です。

内容

プラットフォーム名を指定します。

<IsGpuEncodingEnabled>

GPU エンコーディングを使用するかどうかです。省略時には、「true=GPUを使用する」 が選ばれます。

内容

true(=GPUを使用する) か false(=GPUを使用しない) を指定します。

preference.xml の例

<?xml version="1.0"?>
<PlatformPreference>
  <ConverterPlatformName>Generic</ConverterPlatformName>
  <IsGpuEncodingEnabled>true</IsGpuEncodingEnabled>
</PlatformPreference>
