Atk 再生中のサウンドパネル

Atk で再生中のサウンドの情報を一覧表示するパネルです。

画面説明

  説明
A ツールバーです。リストの表示を変更することができます。
B 再生中のサウンドが表示されるリストビューです。

ツールバー

項目 説明
名前フィルタ 入力された文字でリストのアイテムをフィルタリングします。
対象はサウンド名とプレイヤー名です。
チェックを入れることにより、発音されているアイテムを表示します。
チェックを入れることにより、ミュートされているアイテムを表示します。

リスト列の表示、非表示を設定します。

発音、サウンド名の列は非表示にすることはできません。
再生中のサウンド数を表示します。

リストビュー

再生中のときだけサウンドがリストのアイテムとして表示されますので、再生が終了するとリストからは消えます。
複数のサウンドを同時に再生すると、その数分のアイテムがリストに表示されることになります。

発音の状態を表示します。

再生中であっても音量が0のアイテムはミュートの表示になります。
