表示

レイアウトエディタウインドウのメニューバー追加項目「表示」について説明します。

アウトライン表示 チェックが入っているとペインがアウトラインだけで表示されます。
ペイン境界を隠す

ペイン境界を隠す：ペインの境界線を非表示にします。

ペイン境界線とNullペインを隠す：ペインの境界線をNullペインを非表示にします。

ペイン境界線とNullペインを表示：非表示だったペインの境界線をNullペインを表示します。

選択するごとにこれら 3 つの項目が順番に切り替わります。
ペイン境界線とNullペインを隠す
ペイン境界線とNullペインを表示
境界ペインを表示 チェックが入っていると境界ペインが表示されます。
部品ペインを強調表示 チェックが入っていると部品ペインが薄青で表示され、他のペインと区別しやすくなります。
ズームイン レイアウトウインドウの中心にズームインします。
ズームアウト ズームアウトします。
ウインドウ拡大 レイアウトウインドウを拡大します。
ウインドウ縮小 レイアウトウインドウを縮小します。
全体表示 レイアウト全体が表示されるようにレイアウトウインドウのサイズが調整されます。
選択部分を全体表示 選択したペインをレイアウトウインドウ中央に拡大表示します。
グリッドを表示 チェックが入っているとグリッドが表示されます。
グリッドに吸着 チェックが入っているとペインがグリッドに吸着するようになります。
