ピッチを変更する

1. 概要

SoundMaker や AtkPlayer におけるピッチ設定方法の解説です。

2. ピッチの設定方法

2.1 SoundMaker でのピッチ設定

2.1.1 ストリームサウンド、ウェーブサウンドのピッチ設定

1. SoundMaker で、ストリームサウンドリストまたはウェーブサウンドリストを開きます。

2. サウンドを選択し、リスト領域またはパラメータパネルの「ピッチ」を編集します。数値は倍率で指定します。

3.  SoundMaker の プレビュー機能 でピッチの変化を確認します。

2.1.2 インストルメントのピッチ設定

1. SoundMaker でバンクタブを開きます。

2. インストリストまたはパーカッションリストで編集対象を選択します。

3. リスト領域またはパラメータパネルの「ピッチ(半音)」「ピッチ(セント)」を編集します。入力した数値分だけずれるようにピッチが変化します。

4. SoundMaker の GUI によるインストルメントの再生 でピッチの変化を確認します。

 

参考：

リージョン単位でピッチを変更することも可能です。サンプルマップで変更したいリージョンを選択し、パラメータパネルの下部にある「ピッチ(半音)」「ピッチ(セント)」を編集します。

 

2.2 AtkPlayer でのピッチ設定

1. SoundParameters パネルの Pitch のパラメータを変更します。数値は倍率で指定します。

2. SoundArchivePlayer パネルから再生を行い、ピッチの変化を確認します。
