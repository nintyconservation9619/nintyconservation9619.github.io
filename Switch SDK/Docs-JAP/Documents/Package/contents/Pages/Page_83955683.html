スケール

   このタブではパーティクルのスケールに関する設定を行います。

基本スケール

「 基本スケール」は、パーティクル放出時のスケールを設定する機能群です。
以下はグループ内設定項目の一覧です。

基準値

パーティクル放出時のスケールを設定する機能です。

ランダム量(%)

パーティクル放出時のスケールをランダムに変化させる機能です。

スケール（パーティクル時間アニメ）

「 スケール（パーティクル時間アニメ）」は、カーブエディタでパーティクルスケールのアニメーションを設定する機能です。
以下はグループ内設定項目の一覧です。

スケールアニメーション

カーブエディタでパーティクルスケールのアニメーションを設定する機能です。

その他スケール設定

「 その他スケール設定」は、「カラーコンバイナ」計算式で使用するカラーを設定する機能群です。
以下はグループ内設定項目の一覧です。

スケール制限に画角を連動 「カメラ距離スケール制限」機能使用時に、カメラの画角変化がパーティクルの表示サイズに与える影響を抑える機能です。
カメラ距離スケール制限（近） カメラの前で必要以上にパーティクルが大きくならない様に制限をかける機能です。
カメラ距離スケール制限（遠） カメラから離れていくパーティクルが必要以上に小さくならない様に制限をかける機能です。
速度で頂点を変形 パーティクルの移動速度に比例して、パーティクルのY軸方向にスケールをかける機能です。
