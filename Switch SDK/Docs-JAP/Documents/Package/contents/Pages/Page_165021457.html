FRIENDS ライブラリ

はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されているフレンドライブラリ（FRIENDS）の機能について説明します。

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象としています。

関連資料

FRIENDS の関数リファレンスと併せて参照してください。

FRIENDS ライブラリの機能

FRIENDS ライブラリは、さまざまなユーザー間の関係を管理する機能を提供します。

  • フレンドリスト
    • ネットワーク対戦等でいっしょに遊ぶためのフレンドを管理するリストです。
    • フレンド関係を構築している場合、ユーザーのプレイ状況（プレゼンス）を共有することが可能です。
  • ブロックリスト
    • ネットワーク対戦等で出会ったいっしょに遊びたくないユーザーや、フレンド申請をブロックしたユーザーを管理するリストです。
  • いっしょに遊んだユーザー
    • ネットワーク対戦等で出会ったいっしょに遊んだユーザーを管理しています。

ユーザー識別子として、ネットワークサービスアカウントを使用します。

  • ライブラリの提供形態と利用方法

提供する機能

FRIENDS ライブラリは、以下の機能を提供します。

  • フレンドリストの取得
  • ブロックリストの取得
  • プレゼンスの更新
  • 任意のユーザーのユーザープロフィール（ニックネームとプロフィール画像 URL）の取得
  • いっしょに遊んだユーザーの追加

フレンド関係の構築

フレンド関係の構築は、「マイページ」上で行います。
アプリからフレンド関係の構築（フレンド申請）を行いたい場合、「マイページアプレット」を使用します。

制約事項

FRIENDS ライブラリには次の制約があります。

Windows 環境での利用

Windows 環境ではスタブ実装のみを提供します。
FRIENDS ライブラリを組み込んでビルドを通すことはできますが、実際の処理は行われません。
