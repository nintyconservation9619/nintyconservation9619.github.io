ブレンドプリセット

ここではパーティクル描画時の背景との合成方式を設定する機能「ブレンドプリセット」について説明します。

概要

「ブレンドプリセット」は、パーティクル描画時の背景との合成方式を設定できる機能です。
「Zバッファ、アルファテスト」を「通常」または「Z無視」に設定することで「 ブレンド」が表示されます。
以下は「 設定項目」の概要です。

ブレンドプリセット

以下は説明に使用されている素材イメージと略語とその説明です。

Src_RGB 書き込みテストの判定により使用が決まったパーティクルのカラー（RGB）です。
Src_A 書き込みテストの判定により使用が決まったパーティクルのカラー（A）です。
Fb_RGB 書き込み先の背景カラー（RGB）の値です。

パーティクル描画時の背景との合成方式を、以下のプリセットから選択します。

一般的なαブレンド Src_RGB × Src_A + Fb_RGB × ( 1.0 - Src_A ) パーティクルと背景で、お互いのカラーを維持しながら合成する設定です。
加算半透明 Src_RGB × Src_A + Fb_RGB × 1.0 パーティクルと背景を加算する設定です。
減算半透明 Src_RGB × Src_A - Fb_RGB × 1.0 背景からパーティクルを減算する設定です。
乗算 Src_RGB × 0.0 + Fb_RGB × Src_RGB パーティクルと背景を乗算する設定です。
スクリーン Src_RGB × ( 1.0 - Fb_RGB ) + Fb_RGB × 1.0 パーティクルと背景を加法混色する設定です。
「加算半透明」では明る過ぎる背景に使用すると有効です。
乗算済みα Src_RGB × 1.0 + Fb_RGB × ( 1.0 - Src_A ) テクスチャのアルファ値が小さくなる部分でも正しい明るさが維持できる設定です。
グレアを使用したさい半透明部分で正しい明るさが出ない場合に使用すると有効です。
