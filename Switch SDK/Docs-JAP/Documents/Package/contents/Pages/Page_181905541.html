パターンアニメーションを利用する

ここではパターンアニメ用テクスチャの利用手順について説明します。  
関連リンク 「 パターンアニメ用テクスチャを作る 」 で作成した炎のパターン用テクスチャを利用し、以下のような炎のアニメーションを作成します。

このチュートリアルでは以下の UI 操作について動画で説明しています。

関連リンク 以下は関連ページへのリンクです。

パターンテーブル 「テクスチャパターンアニメ」で使用するパターンを指定し、パターンをアニメーションさせる順番が変更できる UI です。
テクスチャパターンアニメの「再生方法」 登録したテクスチャをパターンアニメとして使用する機能「テクスチャパターンアニメ」の「再生方法」について説明します。

完成データは以下の場所にあります。
\Samples\Applications\Nw4f\eft\SimpleDemo\Resources\Eset\Fire_01.eset

エフェクト作成の準備

あらかじめ EffectMaker と EffectViewer を起動した状態から、エフェクト作成にあたっての準備を行います。

関連リンク EffectMaker と EffectViewer の起動方法については 「作業の流れ」 を参照してください。

エミッタセットの読み込み

以下の手順でテンプレートとなるエフェクトを EffectMaker に読み込みます。

1
メインメニュー項目 [ファイル]→[ 開く]を選択します。
2

ウィンドウが開きますので、以下の場所からエミッタセットファイル（.eset）を選択し、ファイルを開きます。
\Samples\Sources\Resources\Vfx \Eset\Fire_Template.eset

3
ワークスペースに「 Fire_Template 」が読み込まれます。

「デバッグ」タブを選択する

ワークスペース上で「 ビューアノード」を選択します。
プロパティウィンドウの「 タブ」から「デバッグ」タブを選択します。
ビューアのデバッグ機能に関する設定項目が表示されます。

グレアの強さを変更する

グレアのかかり具合を調整する為、以下の「 項目」を設定します。

項目名 値 説明
グレアの強さ 2 グレアは2段階目の強さに変わります。
参考：

このエフェクトはグレア効果を使用します。
関連リンク グレアについては 「グレア」 を参照してください。

「カメラ」タブを選択する

プロパティウィンドウの「 タブ」から「カメラ」タブを選択します。
ビューアのカメラに関する設定項目が表示されます。

関連リンク 以下は関連ページへのリンクです。

ワークスペース

エフェクトを構成する各ノードを階層構造で表示する「ワークスペース」について説明します。
プロパティウィンドウ ノードの操作やプレビューができる「ワークスペース」とノードの機能設定ができる UI から構成される「プロパティウィンドウ」について説明します。

ビューアのカメラを設定する

エフェクトが確認しやすい状態にする為、「 ビュー・投影」の以下の「 項目」を設定します。

項目名 値 説明
カメラの位置 X,Y,Z = 0,4,10 エフェクトがビューア画面に収まる様にカメラの位置と注視点を調整します。
注視点 X,Y,Z = 0,2,0

このエフェクトにテクスチャパターンアニメを設定し、火のエフェクトを作成します。
以降はビューアで変化を確認しながらエフェクトを編集します。

テクスチャパターンアニメの設定

テクスチャファイルを登録し、テクスチャパターンアニメを設定します。

「テクスチャ0」タブを選択する

プロパティウィンドウの「 タブ」から「テクスチャ」タブを選択します。
「 サブタブ」は「テクスチャ0」が選択されていることを確認します。

「テクスチャ0」 タブを設定する。

「 テクスチャ」の以下の「 項目」でテクスチャファイルを登録します。

項目名 設定パス テクスチャ 説明

ファイル指定

  \Samples\Sources\Resources\Vfx
\Eset\Textures\Fire.ftxb

「 ロードボタン」を押すことで開くウィンドウからテクスチャファイルを選択します。
炎の色とシルエットのテクスチャです。

正常に登録された場合、「 テキストボックス」にファイル名が表示されます。

登録したテクスチャの値を設定します。

項目名  値 説明
RGB を二乗して取得 ON テクスチャをリニア変換します。
パターン分割

U = 8

V = 4

テクスチャを横に 8 枚、縦に 4 枚に等分割します。

パターン分割を設定すると、テクスチャ 0 の RGB 画像と α 画像が分割され、番号が振られます。

「テクスチャパターンアニメ」 を設定する

「 テクスチャパターンアニメ」の以下の「 項目」でテクスチャパターンを設定します。

項目名  値 説明
再生方法 ループ テクスチャパターンをループ再生します。
「ループ」「パターンテーブル」が設定可能になりますので、以下のように設定します。
1コマの継続フレーム数 2 1 枚のテクスチャを 2 フレーム継続して表示します。
使用パターンテーブル数 32 テクスチャパターンの枚数を分割した画像の数にします。
テーブル簡易入力 0-31

テクスチャ 0 の 0 番から 31 番まで、番号順に連続再生します。
