使用方法

処理を行うレイヤーの RGB チャンネルを選択して、[フィルター] → [Nintendo] → [Nintendo Normal Map Filter...] を実行します。

  • Operation
    フィルター処理の種類を指定します。

    フィルター処理 説明
    Convert from Height Map 画像の輝度を高さとみなして、その凹凸から法線マップを作成します。
    Normalize 法線マップの各ピクセルの法線を正規化します。
    Scale Slope 接空間法線マップの各ピクセルの法線とベクトル (0, 0, 1) との角度をスケールします。
    法線マップの効果の強さを調整する場合に使用します。
    Combine 接空間法線マップに別の接空間法線マップを合成します。
    低周波な変化を持つ法線マップに高周波な変化を持つ法線マップを合成する場合などに使用します。
  • Positive Z
    法線を加工する際に Z 成分が 0 以上になるように調整するなら ON にします。
    Z 成分がマイナスになっても構わなければ OFF にします。
    snorm_bc5 など XY 成分のみのテクスチャーフォーマットを使用して、ピクセルシェーダで Z 成分を計算する場合は ON にします。
  • Height Scale
    高さマップの高さに乗算するスケール値をパーセントで指定します。値を大きくするほど凹凸が強調されます。
  • Filter Type
    高さマップから法線を計算する際にサンプリングする範囲を指定します。値が大きくなるほど広範囲をサンプリングします。
  • Edge Wrap
    高さマップの画像が繰り返しているとして端のピクセルを処理するなら ON にします。
    画像の端のピクセルを引き伸ばして処理するなら OFF にします。
  • Multiply Alpha
    高さマップにアルファチャンネルの値を乗算して法線を計算するなら ON にします。
  • Slope Scale
    法線とベクトル (0, 0, 1) との角度に乗算するスケール値をパーセントで指定します。値を大きくするほど法線マップの効果が強くなります。
  • Other Normal Map
    合成する他の接空間法線マップ（PSD または TGA ファイル）をフルパスで指定します。
    開いている画像より幅または高さの小さい画像を指定した場合、小さい画像を水平・垂直に繰り返した状態で合成します。
    PSD ファイルは RGB カラー、8 bit/チャンネル、「互換性を優先」を ON にして保存したものを指定してください。PSD ファイルにレイヤーが複数ある場合、全レイヤーを結合した画像を合成します。
  • Combine Scale
    合成する他の接空間法線マップの影響の強さをパーセントで指定します。

OK ボタンをクリックするとフィルターが実行されます。

+ ボタン、- ボタンおよびマウスのホイールでプレビュー画像の拡大率を変えることができます。
Lighting を ON にすると、ライティングした状態をプレビューできます。
その際、プレビュー画像をマウスの右ボタンでクリック（ドラッグ）することで、ライトの方向を変更できます。Rotate Light を ON にすると、ライトの方向が Z 軸を中心に自動的に回転します。

注意事項

法線マップフィルタープラグインを使用する際の注意事項です。

イメージモード

イメージモードが RGB カラー以外のファイルや、8 bit/チャンネルでないファイルでは実行できません。

アルファチャンネル

アルファチャンネルが 2 つ以上存在する場合、チャンネルウィンドウで一番上にあるアルファチャンネルのみ Convert from Height Map 処理で参照します。
