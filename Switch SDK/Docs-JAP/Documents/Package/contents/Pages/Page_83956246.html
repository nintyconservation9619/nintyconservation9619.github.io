親子階層を作成する

ここでは、親子階層の作成手順について説明します。
親子階層構造を使うことによって、関連のある複数のペインをまとめて操作したり、関連して操作することが可能となります。

レイアウトウィンドウで親子階層を作成する

例として、レイアウトファイルを新規作成し、ヌルペインを2つ作成します。


[ 右クリックメニュー ] → [ 親子設定 ] を選びます。
マウスカーソルが変化して、親子設定モードになっていることが確認できます。


親子設定は親ペインから、子供ペインへと直線を結ぶような操作で設定を行います。
まず、親ペイン上でドラッグを開始します。親ペインが正しく選択された場合は、マウスカーソルにむかって直線が描画されます。
この直線を子供ペインに結ぶようにマウスカーソルを子供ペインまで移動します。
マウスカーソル下のペインが子供ペインとしてふさわしい場合は、2 つのペインが赤くハイライト表示されます。


ふさわしいペイン上でマウスボタンを離すと、親子階層が正しく設定されます。

親子としてふさわしくない条件を以下に箇条書きにしておきます。
・指定ペインが自分である。
・指定ペインが自分の親である（”親の親”のような場合も同様です）。

親ペインの位置をマウスで変更してみましょう。子供の位置も親に追従して変化するはずです。
変化しない場合は、[ メニュー ] → [ ペイン ] → [ 親の平行移動操作を子に影響させる ] が有効か確認してみてください。

親の平行移動操作を子に影響させる

[ メニュー ] → [ ペイン ] → [ 親の平行移動操作を子に影響させる ] で、親の平行移動操作を子に影響させるかどうかを切り替えることができます。
また、Ctrl キーを押すことで、操作モードを一時的に反転させることができます。
現在の操作モードは、レイアウトウィンドウのステータスバーのアイコンで確認することができます。

レイアウトウィンドウで親子階層を解除する

親子設定の解除の手順は、設定の時とほぼ同様です。
[ 右クリックメニュー ] → [ 親子解除 ] を選びます。
解除したい親子階層の親ペインと、子ペインを直線で結ぶようにマウス操作を行います。

親子階層ウィンドウを利用する

[ メニュー ] → [ ウィンドウ ] → [ 親子階層 ] をえらび、親子階層ウィンドウを表示します。 親子階層ウィンドウでは、レイアウト中の親子階層関係を一覧することが可能です。

親子階層ウィンドウで親子階層を作成、解除する


まず、子供となるペインに対応するツリー項目をドラッグします。


ドラッグしたツリー要素を、親となるペインに対応するツリー項目にドロップします。


2 つのペインが親子階層を設定するための条件を満たしていれば、親子階層が設定されます。
解除するには、他の親に対して関連付けを行うか、RootPane に対して、親子設定を行ってください。

親子階層に沿ったパラメータの変更

ウインドウ左端の目アイコンと、鍵のアイコンをクリックすることで、指定されたペイン以下の階層をロック状態にするなど、ペイン親子階層に沿ったパラメータの変更を行うことができます。

ノードの複数選択

Ctrl キーや Shift キーを押しながらノードを選択することで、複数ノードを同時に選択することが可能です。
多数のペインに対して選択や、操作を行いたいときに利用します。
