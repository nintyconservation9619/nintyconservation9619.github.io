部品パレットウインドウ

部品ルートは、部品パレットで指定します。
部品パレットは、部品ルート以下の部品レイアウトを一覧表示します。

「部品パレットウインドウ」では以下の操作ができます。

部品パレットウインドウ
A 部品フォルダ

アイコンをクリックすると、フォルダ参照ダイアログが開きます。

フォルダ参照ダイアログから部品フォルダを指定します。
B 部品フォルダパス

現在の部品フォルダのパスが表示されます。

過去に使用した部品フォルダのパスも記憶されています。

ここにフォルダをドロップして部品フォルダを設定することができます。
C 履歴クリア 現在の部品フォルダを除いた履歴を削除します。
D 更新

部品パレットを最新の状態に更新します。

右側に更新した時刻が表示されます。
E 部品レイアウトのみ表示 ボタンを「 ON 」にすると部品フォルダ内の部品レイアウトのみが表示されます。

「 ON 」状態です。

指定した 部品フォルダ 内の、部品レイアウトのみが表示されます。

「 OFF 」状態です。

指定した 部品フォルダ 内の、全てのレイアウトが表示されます。
F 検索

「検索ワード」と、検索対象となる「カテゴリ」を設定することができます。

  • 全て
  • 部品名
  • ファイル名
  • 種類
  • 派生元
  • 説明
  • 更新
  • ペイン数
  • 総ピクセル
G 部品レイアウト一覧

部品レイアウトの情報が表示されます。

  • 部品名
  • ファイル名
  • 種類
  • 派生元
  • 説明
  • 更新
  • ペイン数
  • 総ピクセル

「部品レイアウト一覧」で右クリックすると、以下のメニューが表示されます。
開く 選択したファイルをレイアウトウィンドウで開きます。
部品ペインとして配置 現在開いているレイアウトウィンドウに、選択したファイルを部品として配置します。
エクスプローラで開く 選択したファイルをエクスプローラで開きます。
派生部品を作る 選択したファイルから派生部品を作り、レイアウトウィンドウで開きます。
すべての派生部品を更新 選択したファイルの最新データで派生部品を更新します。
すべてのボディレイアウトで上書きを更新 部品レイアウトの上書き設定を変更した際に、ボディ側で使わなくなった古い上書き情報を一括で破棄します。
派生元への関連付けを修正 「派生元への関連付けを修正」ウィンドウで、派生部品の関連付けを解除したり、派生先を変更したりします。
