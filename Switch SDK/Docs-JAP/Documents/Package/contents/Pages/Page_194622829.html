アトリビュートツリー

カーブエディタウインドウの「アトリビュートツリー」に表示される機能を説明します。

「アトリビュートツリー」は、ペインのアニメーションする要素が表示されます。
ここで選択した要素へのアニメーション作成及び編集をグラフ表示領域で行います。

アトリビュートツリー
A



すべて表示・フィルタなし フィルタリングせず全ての項目を表示します。
基本項目表示 位置、サイズ、スケール、回転、透明度、表示・非表示の基本項目だけを表示します。
キーが打たれた項目表示 キーが打たれた項目だけを表示します。
カスタムフィルタ

選択した任意の項目だけを表示します。

右側のリストから「新規登録」を選択することで項目フィルタをカスタマイズできます。
B アトリビュートツリー

ペインのアニメーションする要素が表示されます。

基本項目以外に表示される項目はペインの種類によって異なります。

フィルタの新規作成

「 カスタムフィルタ」のボタンが ON のとき、右側のテキストボックスをマウス左ボタンクリックすることでリストが表示されます。
リストから「新規登録」を選択することで「フィルタの新規作成」ダイアログが開きます。

フィルタの新規作成ウインドウ
A フィルタ名

ここに入力した名前でリストに表示されます。

フィルタ名を入力しないと OK ボタンが有効になりません。
B すべてをチェック すべての項目のチェックを ON / OFF することができます。
C 選択カーブをチェック 選択しているカーブのチェックを ON / OFF することができます。
D 選択カーブ用アトリビュートツリー ここでチェックをつけた項目がアトリビュートツリーに表示されます。
