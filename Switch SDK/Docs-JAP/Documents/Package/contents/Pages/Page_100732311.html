NintendoWare 3D ニュース

NintendoSDK 0.8.0 で 3D モジュールに追加された機能を紹介します。

3DEditor

  1. カーブビューのコンテキストメニューにキーを挿入する機能を追加
  2. カーブエディタにキースナップ機能を追加
  3. カーブエディタのカレントフレーム移動などのショートカットキーを変更
  4. 3DEditor のカラーピッカー上での色空間切り替え機能の追加

カーブビューのコンテキストメニューにキーを挿入する機能を追加

カーブビューのコンテキストメニューに選択しているカーブにキーを挿入する機能を追加しました。
キーは、カレントフレームの位置に挿入されます。

カーブエディタにキースナップ機能を追加

カーブエディタにキースナップ機能を追加しました。

キースナップを ON にした状態でカーブビュー上のキーを移動すると、カーブビュー上にある他のキーに対して、フレームと値がスナップするようになります。
スナップしたキーに対してカーブビュー上に点線が表示されます。 

便利な使い方

カーブの編集中に他のカーブにあるキーと同じ値や、フレームにキーを移動したいときに使用すると便利です。

C キーでキースナップの ON / OFF を切り替えることができます。

カーブエディタのカレントフレーム移動などのショートカットキーを変更

カーブエディタで使用できるショートカットキーに変更がありました。

機能が削除されたショートカットキー

下記ショートカットキーは、カーブエディタで使用できなくなりました。

操作 説明 状況
[ ↑ ] [ ↓ ]  選択キーの値を編集 Ver 0.8.0 で使用できなくなりました。

機能が追加 / 変更されたショートカットキー

下記ショートカットキーが追加、変更されました。

操作 説明 状況
[ < ] [ > ] カーブビューに表示されている全てのキーでカレントフレームが前後にあるキーの位置にジャンプします。 新しく機能が追加されました。
[ ← ] [ → ] カーブビューでキーの選択状態を左右にある次のキーにシフトします。 前 Ver までは選択キーのフレームを編集する機能でした。

3DEditor のカラーピッカー上での色空間切り替え機能の追加

3DEditor のカラーピッカー上で色空間を切り替える機能を追加しました。

色空間のプリセット sRGB形式 / LinearRGB 形式 / HSV から選択します。

便利な使い方

テクスチャと色空間の異なるマテリアルカラーを一致させるときに便利です。

例：ノンリニアな色空間のDCCツールにおいて、テクスチャのあるピクセルカラーが 0.4 の場合に LinearRGB の値を知りたい。

  1. 3DEditor にあるマテリアルカラーにカラーピッカーの色空間を「sRGB」の状態で  0.4で入力します。
  2. 色空間を「LinearRGB」に変更すると自動的にリニア空間での値 ( 0.13 ) に変換されます。
  3. sRGB の値 0.4 は、LinearRGB では 0.13 と値を知ることができるので、色を合わせるときに便利です。
