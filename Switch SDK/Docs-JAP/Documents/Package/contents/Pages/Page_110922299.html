大きさの違うフォントファイル(bffnt)に差し替えたのに、テキストボックスの表示が変わらない

質問

大きさの違うフォントファイル(bffnt)に差し替えたのに、テキストボックスの表示が変わらないです。

回答

適宜テキストボックスペインのフォントスケール設定を調整してください。
1.0 にすれば指定フォントそのままの大きさになるはずです。
エディタのデフォルト動作として、テキストボックスの見た目サイズが変わらないようにフォントスケールを調整しています。

バージョン 2.4.1以降の LayoutEditor で保存したファイルなら、読み込み時にサイズ変更を検知して警告を表示します。
また、レイアウト中間ファイル(flyt)中の、fontSizeOriginal 要素に、最終ファイル保存時点のフォントの大きさが記録されています。プログラムで一括修正をする際に利用してください。
