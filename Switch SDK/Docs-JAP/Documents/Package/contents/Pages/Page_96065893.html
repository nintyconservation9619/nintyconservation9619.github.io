速度で頂点を変形

ここではパーティクルを速度方向に伸ばす機能「速度で頂点を変形」について説明します。

概要

「速度で頂点を変形」は、パーティクルの移動速度に比例して、パーティクルのY軸方向にスケールをかける機能です。
速度のあるパーティクルに対してこの機能を適用することによって、高速に移動する物体が速度方向に伸びて見える「残像」の効果を表現できます。
以下は「 設定項目」の概要です。

速度で頂点を変形

「速度で頂点を変形」を適用するかを ON/OFF で切り替えます。
設定を ON にすると 「 速度で頂点を変形」が表示されます。以下はグループ内の設定項目です。

加算値

設定した値がパーティクルのY軸方向に加算されます。以下はY軸方向のスケールを求める計算式です。

  • Y 軸方向のスケール : Y軸方向のサイズ + （ 速度 × 「速度をYスケールに加算で指定した値」）
