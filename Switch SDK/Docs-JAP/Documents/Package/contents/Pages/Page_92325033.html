エミッタノードの操作

ここではワークスペースにおけるエミッタノードの操作方法について説明します。

準備

関連リンク クイックスタート 「エフェクトの作成」 を完了させた状態から作業を開始します。

ここまでの手順において、ワークスペースのエミッタセット「 EmitterSet1」直下には以下の様に複数のエミッタが作られています。

エミッタ名の変更

作成したエミッタ名の変更方法について説明します。
ワークスペースのエミッタセット「 EmitterSet1」のエミッタ「 Emitter1_1」を選択した状態で、以下いずれかの操作を行います。

  • メインメニュー項目［ファイル］→［ 名前を変更］を選択します。
  • マウス右ボタンクリックで開くメニューから「 名前変更」を選択します。
  • ショートカットキー F2 を使用します。

名前の変更が可能になりますので、エミッタ名を「Emitter3」と入力し、Enter キーで確定します。
「 Emitter1_1」が「 Emitter3」に変更されます。

エミッタの複製

選択したエミッタの複製方法について説明します。
ワークスペースのエミッタセット「 EmitterSet1」のエミッタ「 Emitter3」を選択した状態で以下いずれかの操作を行います。

  • メインメニュー項目 [編集]→[ 複製]を選択します。
  • マウス右ボタンクリックで開くメニューから「 複製」を選択します。
  • ツールバーにある「 複製」ボタンを押します。
  • ショートカットキー Ctrl + D を使用します。

ワークスペース上にはエミッタセット「 EmitterSet1」の階層下にエミッタ「 Emitter3_Copy1」が追加されます。
ショートカットキー F2 を使用し、エミッタ名を「 Emitter3_Copy1」から「Emitter4」に変更します。

ここまでの手順において、「 EmitterSet1」階層は以下の様に 4 つのエミッタが作られています。

複数エミッタの選択と一括編集

ワークスペース上で選択した複数のエミッタを選択する方法について説明します。
ワークスペースの「 Emitter1」を選択した状態から、以下の操作で複数のエミッタを選択できます。

    • 「 Emitter3」を Ctrl + マウス左ボタンクリックすることにより、「 Emitter1」に加えて「 Emitter3」を選択できます。
    • 「 Emitter3」を Shift + マウス左ボタンクリックすることにより、「 Emitter1」と「 Emitter3」の間にある「 Emitter2」も含めて選択できます。

以下の操作手順で選択した複数エミッタで同一の設定項目を一括編集できます。

1 ここでは「エミッタトランスフォーム」グループ→「位置(±)」Xを「  値 = 10.12」に設定します。
2 「 EmitterSet1」をマウス左ボタンクリックすることで、複数エミッタの選択状態を解除します。
3

同時に選択していたエミッタを確認すると、「位置(±)」Xの値が同じ「  値 = 10.12」であることを確認します。

選択中のエミッタ間で設定項目を共有している為、いずれか一つのエミッタを編集することで同一の設定項目をまとめて変更できます。
