制限

API の呼び出し頻度の制限

サーバー負荷の観点から、一部の API は呼び出し頻度の制限があります。
制限となる API は次節以降で紹介します。

参考：

制限内容はガイドラインを参照してください。

プレゼンスの状態変更

  • nn::friends::DeclareOpenOnlinePlaySession()
  • nn::friends::DeclareCloseOnlinePlaySession()

サーバーに状態変更を通知するため、これらの関数を呼び出すと通信が発生します。

プレゼンスの更新

  • nn::friends::UserPresence::Commit()

サーバーにプレゼンスの更新を通知するため、この関数を呼び出すと通信が発生します。
なお、nn::friends::UserPresence::SetAppValue() や nn::friends::UserPresence::SetDescription() で設定した内容は nn::friends::UserPresence::Commit() を呼び出したタイミングで反映されます。

プロフィールリストの取得

  • nn::friends::GetProfileList()

プロフィールリストはサーバーに問い合わせるため、通信が発生します。
