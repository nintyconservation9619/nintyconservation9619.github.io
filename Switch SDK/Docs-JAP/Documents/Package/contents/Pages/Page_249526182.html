複数台の開発機の初期化はどのようにするべきなのか？

複数台の開発機の初期化方法の最も標準となるものは、「InitializeSdev.exe もしくは InitializeEdev.exe を台数分順次実行していく」です。

例えば IP アドレスが 192.168.10.10, 192.168.10.11, 192.168.10.12 という3台の SDEV を初期化したい場合は bat ファイルにこのように記述すると初期化できます。

@echo off

Tools\CommandLineTools\InitializeSdev.exe --target 192.168.10.10
Tools\CommandLineTools\InitializeSdev.exe --target 192.168.10.11
Tools\CommandLineTools\InitializeSdev.exe --target 192.168.10.12

EDEV の場合も同様です。

この場合初期化ツールは順次実行されることになります。

InitializeSdev ではさらに並列的に同時実行できるしくみがありまして、以下のように --keep-targetmanager-alive オプションをつけると exe 間の同時実行をしようとするようになります。 ただしこれは Trial な機能ですので失敗することがあります。ご了承ください。

InitializeSdev --target 192.168.10.10 --keep-targetmanager-alive

また NX Addon 4.1.0 からはこれらのことを行えるツールスクリプトとして DevKitSetup.ps1 も追加提供しました。こちらもご活用下さい。
