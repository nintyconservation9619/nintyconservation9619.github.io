ペイン情報貼り付け制御ウインドウ

「ペイン情報貼り付け制御ウインドウ」ではペインの任意の情報を選んで、別のペインに貼り付けることができます。

  • [ メニュー ] → [ ウインドウ ] → [ ペイン情報貼り付け制御 ]

「ペイン情報貼り付け制御ウインドウ」を開いた状態で、貼り付ける元データのペインを選択し [ 編集 ] → [ コピー ] を選択すると、元データの種類によって、ウインドウの中の選べる項目が変化します。
貼り付け先のペインを選択して、「ペイン情報貼り付け制御ウインドウ」から貼り付けたい任意の情報を選び、「選択ペインに貼り付け」を選択し貼り付けます。
この時、選べない項目は、グレーで表示されます。

ペイン情報貼り付け制御ウインドウ
A コピー元ペイン名
 貼り付けるデータを持っている、元ペインの名称が表示されます。
B ペイン基本情報 ペインの複製をする基本情報を選択します。
C ピクチャペイン ピクチャペインのデータを複製する場合、設定します。
D テキストボックスペイン テキストボックスペインのデータを複製する場合、設定します。
E ウインドウペイン ウインドウペインのデータを複製する場合、設定します。
F 部品ペイン 部品ペインのデータを複製する場合、設定します。
G マテリアル マテリアルのデータを複製する場合、設定します。
H 頂点カラー

異なる種類のペインで、頂点カラーをコピーする場合に利用します。

「テキストカラー ( テキストボックス ) 」、「頂点カラー ( ピクチャ ) 」、「頂点カラー ( ウインドウ ) 」を相互にコピーできます。

I ベクトル成分の貼り付け制御 ( 全情報共通 )

位置、サイズ、スケール、回転等、ベクトル成分の貼り付けについて制御します。

ベクトル成分の全情報を共通で X 、 Y 、 Z 制御できます。

「ベクトル成分の貼り付け制御」で OFF にした情報は、「選択ペインに貼り付け」で反映されなくなります。
J 色成分の貼り付け制御 ( 全情報共通 )

RGBA 、色成分の貼り付けについて制御します。

色成分の全情報を共通で R 、 G 、 B 、 A 制御できます。

「色成分の貼り付け制御」で OFF にした情報は、「選択ペインに貼り付け」で反映されなくなります。
K アニメーション アニメーションのデータを複製する場合、設定します。
L 貼り付け先のキー情報を置き換える チェックを入れると、貼り付け先のキー情報を全て置き換えます。
チェックを外すと、貼り付け先のキー情報に修正が加えられます。
M すべてチェック 貼り付けることのできるデータ全てにチェックを入れます。
N 選択ペインに貼り付け 貼り付けを実行します。
