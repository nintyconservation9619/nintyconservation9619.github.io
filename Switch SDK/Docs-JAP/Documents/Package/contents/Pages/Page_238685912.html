ユーザ独自コントロール ( 部品 ) を作る手順

ユーザー独自のコントロール ( 部品 ) を作成する手順について説明します。

ユーザ独自コントロール ( 部品 ) の仕様設計・コントロール定義ファイル ( *.flct ) の作成 ( プログラマ作業 )

仕様についてプログラマと相談し、コントロール定義ファイル(*.flct)を作ってもらいます。
コントロール定義ファイルはXML形式で記述されています(サンプルファイルが /Sources/Resources/Ui2d/sampleNW_DefaultControl.flct にあります)。
NW_DefaultControl.flct というファイル名で保存します。

レイアウトを作成

先で作った、NW_DefaultControl.flct というコントロール定義ファイルを部品ルート以下に置くと、LayoutEditor は、コントロール設定の内容を置き換えて、ユーザ独自コントロールを使ったレイアウトを作成することができるようになります。
部品・コントロール設定ウィンドウに、ユーザ独自コントロールが選択肢として追加されています。
デザイナは追加されたコントロールを選んで、レイアウトを作ります。

カスタムビューアを作成する（ プログラマ作業 ）

プログラマがユーザ独自コントロールの動きを実装します。
また、ユーザ独自コントロールを組み込んだ、カスタムビューアを作成します。
カスタムビューアを簡単に実装するための、ユーテリティや参考プログラムが提供されています。
カスタムビューアができると、デザイナはいつも通りのプレビュー実行で簡単に動きが確認できるようになります。
