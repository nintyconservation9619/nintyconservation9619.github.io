3D ニュース

NintendoWare 3D ニュース

NintendoSDK 0.6.0 で3Dモジュールに追加されたいくつかの機能を紹介します。

3DEditor

パス違いの同名テクスチャの読み込みをサポート

パス違いの同名テクスチャが読み込めるようになりました。

ただし、同名テクスチャをひとつのモデルに複数割り当てることはできません。

一覧表示の表示設定保存、読み込み機能追加

一覧表示の表示設定を保存、読み込みできるようになりました。

一覧表示に対象名フィルタ機能追加

一覧表示で対象名をフィルタリングすることができるようになりました。

選択したモデルの点滅機能追加

3DEditorでモデルを選択すると接続先アプリケーションで選択したモデルが点滅するようになりました。

ただし、接続先アプリケーションがディスプレイリストを未使用の場合に限ります。

Photoshop プラグイン

リサイズ補間アルゴリズム選択機能追加

テクスチャエクスポートプラグインでリサイズ時の補間アルゴリズムを選べるようになりました。
