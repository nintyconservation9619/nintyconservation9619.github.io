テクスチャの詳細ウィンドウ

選択したテクスチャイメージ （ カラー / アルファ ） の詳細を確認するウィンドウです。
「テクスチャ情報ウィンドウ」は、「テクスチャ」タブにおいて登録したテクスチャのサムネイルをマウス左ボタンクリックすることで開きます。

A テクスチャイメージの表示倍率です。
B テクスチャイメージを拡大 /　縮小します。
C テクスチャイメージの表示チャンネルを切り替えるボタンです。
D テクスチャイメージをミップマップ表示します。
E ミップマップの表示方向を切り替えます。
F ヘルプを表示します。
G テクスチャのイメージが表示されます。
H テクスチャフォーマットです。
I ミップマップレベルです。
J テクスチャファイルの容量です。byte 数で表示されます。
