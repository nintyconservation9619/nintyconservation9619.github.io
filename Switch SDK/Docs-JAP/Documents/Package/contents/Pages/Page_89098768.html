機能説明

ここでは、MEM ライブラリが提供する各機能を、API の使用方法と合わせて説明します。

機能一覧

機能 概要
StandardAllocator

StandardAllocator クラスは、メモリ領域の確保・解放などを行うための汎用アロケータクラスです。
拡張ヒープに比べて高速に確保・解放を行うことが可能です。 
NumberLineAllocator NumberLineAllocator クラスは管理領域と確保・解放領域を分離して持つことのできる整数アロケータのクラスです。
拡張ヒープ

拡張ヒープ (ExpHeap) は、与えられたメモリ領域を管理し、そのメモリの確保、解放機能を提供します。

フレームヒープやユニットヒープに比べ利用に制限がなく、汎用的なメモリアロケータとして利用できます。

ただし、確保、解放速度の観点から、 拡張ヒープ より StandardAllocator の利用を推奨します。
フレームヒープ

フレームヒープは拡張ヒープに比べ、よりシンプルな機能を提供するメモリアロケータです。

基本的には、メモリブロックの確保と、確保している全てのメモリブロックの解放しかできません。

メモリブロックには管理情報を一切持たないため、高いメモリ効率が得られます。
ユニットヒープ

ユニットヒープは決められたサイズのメモリブロックの確保、解放のみを行えるメモリアロケータです。

拡張ヒープと比べ汎用性に欠けますが、その分高速に動作します。

フレームヒープと同様にメモリブロックに管理領域がないため、高いメモリ効率が得られます。
