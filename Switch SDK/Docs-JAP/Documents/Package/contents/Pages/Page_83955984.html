テクスチャプロパティウィンドウ 全般ページ

選択したテクスチャの全般的なプロパティを確認するウィンドウです。

A 選択したテクスチャへのフルパスです。
B 選択したテクスチャのカラーイメージです。
カラーイメージをクリックすると詳細ウィンドウが開きます。
C 選択したテクスチャのアルファイメージです。
アルファイメージをクリックすると詳細ウィンドウが開きます。
D 表示する配列テクスチャのインデックスです。詳しくは 表示深さについて を参照してください。
E DCC ツールプリセットです。
F テクスチャのプリセットフォーマットです。
G テクスチャフォーマットを表示します。
H テクスチャのミップマップを生成する際のフィルタの種類です。
フィルタには、「ポイントサンプリング」「線形補間」「キュービック補間」があります。
I テクスチャをサンプリングする際の、カラーコンポーネントセレクタです。
J テクスチャ圧縮（ bc1 / bc2 / bc3 ）をする際に、RGB の誤差に対して、人間の目に合わせた重みをつける場合にチェックします。
K

選択しているテクスチャの R G B A チャンネルがリニア変換されているかどうかが表示されます。

DCC ツールからの出力時は、コンフィグ設定に合わせてリニア変換されるかどうかが決定されます。

詳細はグラフィックス中間ファイル仕様書の ftx の texture_info の linear 属性の説明をご覧ください。
L テクスチャの次元が表示されます。
M テクスチャの幅です。
N テクスチャの高さです。
O テクスチャの深さです。
P テクスチャのデータサイズが表示されます ( パディングや、ミップマップを含む ) 。
Q テクスチャのミップマップレベルです。
ミップマップがない場合は、1 を指定します。
R ミップマップの最小サイズの値です。
S テクスチャの用途をヒント情報として表示します。
T 実機用イメージ作成時に指定するスウィズル値です。
U テクスチャの元データのハッシュ値です

表示深さについて

配列テクスチャを読み込んでいた場合にアクティブになります。

プルダウンメニューでインデックスを選択すると、選択したインデックスに対応しているテクスチャイメージを表示します。
全体表示を選択すると、配列されているテクスチャイメージを一覧で表示します。 



テクスチャの深さの注意点

テクスチャの次元や、キューブマップを使用している場合は下記にご注意ください。

  • テクスチャの次元が、一次元テクスチャ / 二次元テクスチャ の配列の場合は、テクスチャの枚数を指定します。
  • キューブマップの場合は、キューブマップの数に合わせて 6 の倍数を指定します。

テクスチャのミップマップレベル

DCC ツールからの出力時は、元画像のサイズから算出した最大値が設定されます。

詳細ウィンドウ

選択したテクスチャイメージ （ カラー / アルファ ） の詳細を確認するウィンドウです。

テクスチャ詳細ウィンドウで使用できるショートカットキーについては、こちらを参照してください。

A 詳細ウィンドウで表示されている倍率です。
B テクスチャイメージを拡大 /　縮小します。
C テクスチャイメージを RGBA 毎にチャンネルを切り替えて確認できます。
D 表示深さ
選択しているテクスチャが配列テクスチャの場合にアクティブになります。
プルダウンメニューからインデックスを選択して、選択したインデックスに対応したテクスチャの詳細を表示します。
E テクスチャイメージをミップマップ表示します。
F ミップマップの表示方向を切り替えます。
G ヘルプを表示します。
H テクスチャイメージです。
カラーイメージ、アルファイメージを確認できます。
I マウスがポイントしている位置です。
J マウスがポイントしている位置のカラー値です。
K テクスチャフォーマットです。
L テクスチャのサイズです。
M ミップマップレベルです。
N テクスチャの総 byte 数です。
O

プレビューされているテクスチャがエンコード済みのテクスチャであるか、そうでないかが表示されます。

エンコード済みのテクスチャであれば、「EncodedImage」、そうでなければ「OriginalImage」と表示されます。
特定のプラットフォーム向けでなかったり、SDK がサポートしていないテクスチャの場合に「OriginalImage」が表示されます。
