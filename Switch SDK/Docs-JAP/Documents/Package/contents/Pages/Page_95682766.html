UV 回転アニメ

ここでは登録したテクスチャの UV を回転する機能「UV 回転アニメ」について説明します。

概要

「UV 回転アニメ」は、登録したテクスチャの UV 座標を回転させることでテクスチャを回転アニメーションができる機能です。
以下は「 設定項目」の概要です。

UV 回転アニメ

テクスチャ UV 座標の「角度」を各軸個別に設定できます。

設定を ON にすると「 UV 回転アニメ」が表示されます。以下はグループ内の設定項目です。

加算値

テクスチャ UV の回転角度に毎フレーム加算する値を設定します。
加算値はエミッタ起動時から適用されます。

初期値

テクスチャ UV 座標の回転角度に初期値を設定します。

初期値ランダム

初期値に指定した範囲の乱数を設定します。
初期値を乱数の上限、下限側はランダムに変化します。以下は乱数の下限をもとめる計算式です。

  • 乱数の下限 : 初期値 - 「初期値ランダム」の計算結果

設定例

以下は「UV 回転アニメ」の各機能の設定例です。
「放出間隔」は 10 に設定しています。

A 加算値

± 各々の方向に対して毎フレーム加算値 1 を設定しています。
負の方向に設定した場合は時計回りに、正方向に設定した場合は反時計回りに毎フレーム 1 度ずつ回転します。
B 初期値 ± 各々の方向に対して 45 度を設定しています。
負の方向に設定した場合は時計回りに、正方向に設定した場合は反時計回りに 45度傾きます。
