ビューアーで、アニメーションが誤って再生されてしまう

質問

ビューアーで、アニメーションが誤って再生されてしまいます。

回答

その１

ビューアー(ビューアーライブラリ)は、アニメーションを 「レイアウト名_区間名」という形で管理しています。

「レイアウト名_区間名」という名前の部品レイアウトが存在すると部品のアニメーションと区別が付きません。

トラブルの例：

  • ボディレイアウト Layout1 と部品レイアウト Layout1_button1 があり、Layout1 に区間 button1 を設定する。

区間名をレイアウト名につなげたときに、部品レイアウト名と同じにならないように区間名を設定してください。

その２

ペイン名が特定の条件を満たした場合に、誤ってアニメーションが関連付けされます。


条件(以下を同時に満たしたときに不具合が起こります)

  • ペイン名の末尾2文字が特定の文字列("LT","RT","LB","RB","L ","R ","T ","B ","C "のいずれか)で終わる
  • 末尾2文字を取り除いた名前が、他のペイン名と一致する


トラブルの例：

  • Pane1 と Pane1LT というペインがあり双方アニメーションしており、本来 Pane1LT につけたアニメーションが、Pane1 について再生された


ペイン名の末尾2文字が、特定文字列で終わらないように修正してください。もしくは、ペイン名の衝突改名によってを回避してください。

（この制限は、LayoutConveter の Windowペインの部位を扱う処理の誤動作によって起こっています）
