コンバイナチャンネル設定の追加

カラーコンバイナとアルファコンバイナの「チャンネル」に、プリセットが追加されました！
「チャンネル」は登録したテクスチャのどの成分を「計算式」への入力値として適用するかを決める機能です。
テクスチャの使い方に幅を持たせることができる為、複数のエミッタ間で同じテクスチャを使い回す場合に便利です。

以下は「カラーコンバイナ」と「アルファコンバイナ」の各「チャンネル」項目ごとの項目名と説明です。

カラーコンバイナ

カラーコンバイナ「チャンネル」のプリセットです。
以下の表は「チャンネル」の項目ごとのイメージと項目名、項目ごとの説明です。

RGB 登録したテクスチャのカラー値が、入力値として適用されます。
1.0 - RGB 定数 1.0 からテクスチャのカラー値を減算したカラーが入力値として適用されます。
A 登録したテクスチャの アルファチャンネルの値です。
1.0 - A 定数 1.0 からテクスチャのアルファ値を減算したカラーが入力値として適用されます。

テクスチャサイズの削減が期待できる

カラーとアルファで同じ画を用いる場合、カラーはリニア、アルファはノンリニアで持つことになり、 テクスチャは 2 チャンネル分の情報が必要でした。
この機能はエフェクトシェーダ内でカラーをリニア化することで 1 チャンネルの情報でもカラーとアルファに対応できます。

EffectMaker

EffectMaker に新しく追加された UI について紹介します。

EffectMaker と ドキュメントの連携

EffectMaker とドキュメントの連携を強化しました！

設定項目「 設定項目」でマウス左ボタンクリックすることで既定のブラウザでドキュメントが開き、その設定項目のヘルプが表示されます。
また、EffectMaker から開くドキュメントファイルはオプション【ユーザー設定】や【プロジェクト設定】の「ドキュメントファイルパス」で変更できます。
