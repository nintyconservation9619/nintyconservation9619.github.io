SystemAudioMonitor

概要

SystemAudioMonitor は開発機のシステムミキサーの出力相当のオーディオデータをキャプチャし、host PC 上で確認するためのツールです。Tools/Audio/SystemAudioMonitor/ 以下に配置されています。

SystemAudioMonitor により取得できるオーディオデータは、ゲーム録画機能 によりキャプチャされるデータと同様ですが、以下が異なります。

  • ゲーム録画機能では 2ch にダウンミックスされた結果をキャプチャしますが、SystemAudioMonitor はダウンミックス前の 6ch データをキャプチャすることができます
  • ゲーム録画機能ではオーディオデータを圧縮しますが、SystemAudioMonitor は圧縮前の PCM データを再生、あるいは .wav ファイルとして保存することができます

使用方法

TargetManager を用いて、開発機に接続します。

その状態で、SystemAudioMonitor.exe を起動します。

起動したのち、左上にある接続ボタンを押すことで、 状態が「未接続」から「接続完了」に変化し、モニタリングが開始されます。
この状態では、PC 上の既定のオーディオ再生デバイスを用いた再生、およびラウドネス値の計測が行われます。
モニタリングの音量は、接続ボタンの下にある音量ボタンでコントロールできます。
終了したい場合は、接続ボタンの隣にある切断ボタンを押すことでモニタリングを停止することが出来ます。

また、モニタリングの開始後、「出力ディレクトリ」を指定したうえで録音ボタンを押すことで、指定したディレクトリへの波形ファイル書き出しを追加で行います。
録音ボタンが存在する領域を右クリックすることでコンテキストメニューが現れ、指定した出力ディレクトリを explorer で開くこともできます。

終了したい場合は、録音ボタンの隣にある停止ボタンを押すことで録音を停止することが出来ます。

注意事項

  • SystemAudioMonitor は開発機環境にのみ対応しており、Windows で動作するアプリケーションの出力をとることはできません
  • SystemAudioMonitor を利用する場合、システムファームウェアの処理負荷が増加し、アプリケーションの動作に影響を及ぼす可能性があります
  • SystemAudioMonitor により取得されるデータに関する仕様は、ゲーム録画機能 の仕様変更などにより予告なく変更される可能性があります

Nintendo.Audio.dll によるキャプチャ機能の使用

SystemAudioMonitor が行っているキャプチャ機能は、 Nintendo.Audio.dll を使用することにより自作の C# ツールなどに組み込むことができます。

.NET アプリでの使用方法は以下の通りです。

  1. Tools/Audio/AudioToolLibraries に Nintendo.Audio.dll が含まれているので、それを「ソリューションエクスプローラ > 参照 > 参照(B)...」で追加します。
  2. キャプチャ機能を実装します。実装の方法については、 Samples/Sources/Applications/AudioToolUtilities 以下にある、 SystemAudioCapture サンプルを参照してください。
