エミッタ形状「円」

ここではパーティクルの放出位置が円周上のエミッタ形状「円」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「円」は、ワールド座標系に配置される円の外周上でランダムな位置から円状に広がった法線方向にパーティクルを放出します。
以下は「 設定項目」の概要です。

エミッタ形状を「円」にすると 「 円」が表示されます。以下はグループ内の設定項目です。

エミッタ半径

円の半径を X Y Z で設定します。

弧の幅（角度）

円の中心を軸に、「弧の開始角度」で指定した角度からの回転幅を角度で設定します。

弧の開始角度

弧の開始位置を角度で設定します。デフォルト値では Z 座標の正方向が開始地点です。

弧の開始角度ランダム

弧の開始角度をランダムにするかどうかを「 チェックボックス」ON/OFF で切り替えます。

設定例

以下はエミッタ形状「円」の各設定項目について設定例の画像と解説図です。

A エミッタ半径 「エミッタ半径」を 50 に設定しています。
B 弧の幅（角度）

「弧の幅」を 90 度に設定しています。
C 弧の開始角度
 「弧の開始角度」を 45 度に設定しています。
