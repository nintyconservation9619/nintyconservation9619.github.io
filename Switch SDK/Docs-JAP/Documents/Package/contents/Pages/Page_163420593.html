NVIDIA CUDA

概要

CUDA は、NX 環境にて CUDA を利用するためのパッケージです。

パッケージ

CUDA は、NintendoSDK NX Addon とは別のパッケージで提供されます。また、他の NintendoSDK パッケージよりも遅れてリリースされる場合があります。

以下にパッケージを一覧します。

パッケージ名 説明
NVIDIA CUDA CUDA を利用するためのパッケージです。

配置

CUDA は、以下のように NintendoSDK を配置したディレクトリ直下に配置されます。

NintendoSDK/
CUDA/

インストール

NDI から CUDA パッケージを取得したあと、setup.exe を実行してください。インストーラが起動し、Windows に CUDA がインストールされます。

ドキュメント

CUDA の基本的な内容については、以下のドキュメントを以下を参照してください。

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\doc\html\index.html

ライブラリ

NX 用のライブラリは、以下のディレクトリにインストールされます。

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\lib\NX-NXFP2-a32
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\lib\NX-NXFP2-a64

サンプル

NX 用のサンプルは、以下のディレクトリにインストールされます。

C:\ProgramData\NVIDIA Corporation\CUDA Samples\v8.0\9_NX
