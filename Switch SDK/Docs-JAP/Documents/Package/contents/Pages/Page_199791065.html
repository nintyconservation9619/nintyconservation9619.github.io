ストライプの機能改善に伴う互換性の問題と回避方法

本リリース以前にストライプ機能を使用して作成されたデータを再生する際に以下の点に注意が必要です。

  • ストライプ利用時に カラータブ / ソフトパーティクル / 遮蔽チェック が動作するようになりました。
  • ストライプ利用時に 回転タブ / Ｚ軸 が動作するようになりました。

この変更により、これまでに作成したデータが意図せず描画結果が変わってしまう可能性があります。

回転

パーティクルの回転に関する設定を行います。ストライプでは Z 軸での回転のみ機能します。

ストライプの設定項目「計算方法」は「ビルボードストライプ」以外に設定してください。

OFF : まったく回転しない場合です。
