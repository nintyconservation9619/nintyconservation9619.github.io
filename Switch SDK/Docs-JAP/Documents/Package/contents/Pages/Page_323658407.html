概要

振動事例集はHD振動の効果的な使い方を紹介する EDEV / SDEV 上で実行できるデモです。


振動事例集では、ゲーム中の様々な場面において、振動を付加する事により

操作感や手応えがどのように変化するかを体験する事が出来ます。 


現在の振動事例集に含まれているデモは、以下の通りです。

名前 説明
UI デモ ボタン、スライダ、ノブ、スイッチなどの UI コンポーネントに振動を付加したデモ

配置場所

振動事例集は Nintendo SDK の NX Addon に含まれており、以下の場所に配置されています。

TargetTools/NX-NXFP2-a64/VibrationDemo/Release/VibrationDemo.nsp

デモの解説

UI デモ

UI デモでは、ボタン、スライダ、ノブ、スイッチなどの

各種 UI  コンポーネントに振動を付加した場合の手応えを体験できます。

操作方法

UI デモでは画面に表示された UI コンポーネント( 以降 UI ) を Joy-Con 及び タッチスクリーン を使用して操作できます。

(一部の UI はジャイロセンサを利用して操作できます。)


画面上には以下の4種類のコンポーネントが、左右2点ずつ用意されています。

  • ボタン
  • スライダ
  • ノブ
  • スイッチ

ボタンとスイッチは、左右同一の挙動です。

スライダとノブは、左右で操作時の挙動が異なります。


操作対象の UI は方向ボタンとスティックの上下左右の操作で変更できます。

操作対象として選択されている UI は、水色の枠線で強調表示されます。

操作対象として選択された状態で A ボタンを押すことで、各 UI の操作を行えます。


スライダ、ノブの場合、A ボタンを押すと枠線の色が水色から黄色へと変化します。

この状態では、方向ボタンやスティックの操作で UI の操作が行えるようになります。

操作対象を変更したい場合は、 B ボタンを押してください。


各 UI の操作方法は以下になります。
UI 名 操作方法
ボタン A ボタンで「ボタンを押し込み」が始まります。
スライダ 方向ボタンまたはスティックの左右操作で、値を変更できます。
ノブ

方向ボタンまたはスティックの左右操作で、値の変更できます。

Aボタンを押した状態で、右 Joy-Con を下図のように回転させることで値の変更できます。
スイッチ A ボタンで ON/OFF を切り替えます。

振動のON/OFFの切り替え
  • L ボタン　振動のON
  • R ボタン　振動のOFF


画面左上で、現在振動がON/OFF設定が確認できます。
