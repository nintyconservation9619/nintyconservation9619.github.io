フラグメントオペレーション

LayoutEditor のペインの中の「フラグメントオペレーション」設定をします。

現在の設定

現在の設定が表示されます。プリセットから選択します。

アルファテスト

（値はアニメーション可能）

アルファテストを設定します。

アルファテストを無効化し、デフォルトを利用するとバイナリサイズが軽減されます。
ブレンド

ブレンド設定をします。

デフォルトを利用して標準的な半透明設定にすると、バイナリサイズが軽減されます。

現在の設定

ペインと、背景の合成方法をプリセットから指定します。
指定した内容で「アルファテスト」や「ブレンド」の設定が自動で切り替わります。

現在の設定 :

プリセットから選択した設定が表示されます。

プリセットから選択した設定を変更すると「カスタム」と表示されます。
プリセット
不透明 アルファテストをしない、完全な不透明物体として表現できる設定です。
抜き 輪郭のはっきりした不透明物体として表現できる設定です。
半透明 ( 初期値 )

ペインと背景で、お互いのカラーを維持しながら合成する設定です。
加算

ペインと背景の画像を加算する設定です。
減算

背景からペインを減算する設定です。
乗算

ペインと背景を乗算する設定です。
半透明 : Max アルファ

キャプチャペイン使用時にアルファ部分が綺麗に合成できる設定です。

 

アルファテスト

アルファ値の比較条件をプルダウンメニューから選択します。
デフォルトを利用するとアルファテストを無効化し、バイナリサイズが軽減されます。

 デフォルト設定を利用する

デフォルト設定を利用する場合「 チェック」を入れます。

デフォルト設定はアルファテストを無効化、バイナリサイズが軽減されます。

条件 :　　　

常にパスしない

この設定を選択した場合、常にテストを通過しません。

アルファ < 値

アルファが値より小さい場合に通過します。

アルファ <= 値

アルファが値と同じか小さい場合に通過します。

アルファ == 値

アルファが値と同じ場合に通過します。

アルファ != 値

アルファが値と同じでない場合に通過します。

アルファ >= 値

アルファが値と同じか大きい場合に通過します。

アルファ > 値

アルファが値より大きい場合に通過します。

常にパスする

この設定を選択した場合、常にテストを通過します。

値 :　　　

値を数値入力か、スライダーで設定します。
注意：

ＧＬ 版は動作しません。

 

ブレンド

テクスチャコンバイナの出力結果 ( Pixel ) と、背景となるフレームバッファ ( FB ) の合成方法を指定します。
デフォルトを利用して標準的な半透明設定にすると、バイナリサイズが軽減されます。

 デフォルト設定を利用する

デフォルト設定を利用する場合「 チェック」を入れます。

デフォルト設定は標準的な半透明設定で、細かな設定はできませんが、バイナリサイズが軽減されます。
