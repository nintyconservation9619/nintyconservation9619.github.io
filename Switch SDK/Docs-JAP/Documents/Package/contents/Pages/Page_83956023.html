終了処理

ここでは再生中のエフェクトを強制的に停止する機能「終了処理」について説明します。

概要

「終了処理」は、ゲームアプリでプログラマが Fade コマンドを実行し、再生中のパーティクルの放出を強制的に止めたり、放出されたパーティクルを強制的に削除できる機能です。
「寿命無限」設定のパーティクルも強制的に削除出来ます。また、削除する際のパーティクルの消し方もアルファ値とスケール値で指定出来ます。
以下は「 終了処理」の「 設定項目」です。

終了処理

Fade コマンド実行時のパーティクルの挙動が設定できます。

放出停止 エミッタからパーティクルの放出を停止する機能です。既に放出されたパーティクルは寿命まで表示されます。
αフェード すべてのパーティクルに対して「フェードアウト時間」で指定した時間をかけて徐々に透明にした後削除する機能です。
スケールフェード すべてのパーティクルのサイズを「フェードアウト時間」で指定した時間をかけて徐々に縮小した後削除する機能です。
フェードアウト時間 フェードアウトに要する時間をフレームで設定する機能です。「αフェード」や「スケールフェード」をONにすることで利用できます。
参考：

終了処理は、プレビューコントロール「 Fade ボタン」を押すことでビューアでも Fade コマンドが呼ばれた結果を確認できます。

設定例

「放出停止」を ON にしたとき、パーティクルの放出は「 Fade ボタン」を押すことで停止します。
以下は「終了処理」の各機能を ON/OFF した場合の放出停止後の比較画像です。

A フェード OFF 既に放出されたパーティクルは寿命まで表示されます。
B アルファフェード ON 「フェードアウト時間」で指定した時間をかけて徐々に透明にした後削除します。
C スケールフェード ON 「フェードアウト時間」で指定した時間をかけて徐々に縮小した後削除します。
