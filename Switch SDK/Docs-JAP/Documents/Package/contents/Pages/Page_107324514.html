エミッタノードの階層化

ここではワークスペースでのエミッタノードの階層化と階層化の解除方法について説明します。

準備

関連リンク クイックスタート 「エフェクトの作成」 を完了させた状態から作業を開始します。

ここまでの手順において、ワークスペースのエミッタセット「 EmitterSet1」直下には以下の様に複数のエミッタが作られています。

階層下にエミッタを追加

選択したエミッタの階層下にエミッタノードを追加し、チャイルドエミッタを作成する方法について説明します。
ワークスペースのエミッタセット「 EmitterSet1」のエミッタ「 Emitter1」を選択した状態で、以下いずれかの操作を行います。

  • マウス右ボタンクリックで開くメニューから「  階層下にエミッタを追加」を選択します。
  • ツールバーにある「  階層下にエミッタを追加」ボタンを押します。

「 Emitter1」の階層下に「 Emitter3」がチャイルドエミッタとして追加されます。

階層下にエミッタをコピー

コピーしたエミッタを階層下にコピーし、チャイルドエミッタを作成する方法について説明します。

ワークスペースのエミッタセット「 EmitterSet1」のエミッタ「 Emitter1」を選択します。
でマウス右クリックボタンで開くメニューから「 コピー」を選択します。
「 Emitter1」を選択したままでマウス右ボタンクリックで開くメニューから「  エミッタを「子」として貼り付け」を選択します。

「 Emitter1」の階層下に「 Emitter1_Copy1」がチャイルドエミッタとして追加されます。
「 Emitter1_Copy1」は「 Emitter1」の設定内容がチャイルドエミッタとして複製されています。

ここまでの手順において、「 EmitterSet1」階層は以下の様に3つのエミッタと2つのチャイルドエミッタが作られています。

階層化の解除

選択したチャイルドエミッタを解除する方法について説明します。
ワークスペースのチャイルドエミッタ「 Emitter3」を選択した状態から、以下いずれかの操作を行います。

    • マウス右ボタンクリックで開くメニューから「  階層化の解除」を選択します。
    • ショートカットキー Ctrl + Shift + P を使用します。

階層上親子関係にあるエミッタ「 Emitter1」とその子階層にあたるチャイルドエミッタ「 Emitter3」の関係が解除されます。
以下はワークスペースのノード配置です。「 Emitter3」は「 Emitter1」と階層上同位に配置されます。
