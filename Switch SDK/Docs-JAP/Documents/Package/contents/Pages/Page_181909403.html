親エミッタ形状の表示

ここではビューアにエミッタの形状とエミッタの軸を表示する機能「親エミッタ形状の表示」について説明します。

概要

「親エミッタ形状の表示」は、EffectViewer にエミッタの形状と軸を表示するデバッグ機能です。
また、「 エミッタ設定」と「 エミッタトランスフォーム」の設定内容が UI 操作に連動して表示されます。
以下は「 設定項目」の概要です。

親エミッタ形状の表示

「親エミッタ形状の表示」を「 チェックボックス」ON/OFF で切り替えます。
