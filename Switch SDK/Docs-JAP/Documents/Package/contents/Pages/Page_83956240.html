レイアウトの新規作成

レイアウトエディタを起動するとレイアウトエディタウインドウが開きます。

[ メニュー ] → [ ファイル ] → [ 新規作成 ] → [ レイアウト ] として、レイアウトファイルを新規作成します。
レイアウトエディタウインドウ画面に ”Untitled.” というタイトルのウィンドウが開きます。このウインドウをレイアウトウインドウと呼びます。
LayoutEditor では、レイアウトウインドウにペインを配置することで、ゲーム画面のレイアウトを作成していきます。

関連リンク レイアウトウィンドウの UI 説明については こちら を参照してください。

テクスチャを登録

ピクチャペインを配置するには、まず使用するテクスチャ（画像ファイル）を登録します。
テクスチャの登録はテクスチャウィンドウで行います。

以下いずれかの操作を行うとテクスチャウィンドウを表示します。

  • メニューから開く  
    [ メニュー ] → [ ウィンドウ ] → [ テクスチャ ] を選ぶとテクスチャウィンドウが開きます。
  • ショートカットで開く
    ショートカット [ F8 ] でテクスチャウィンドウが開きます。

テクスチャの読み込み

テクスチャウインドウにテクスチャを読み込んで、登録します。
テクスチャのフォーマットは TGA ファイルを使用します。

A 読み込みボタン
 テクスチャを選択するための、読み込みダイアログを表示します。
B テクスチャ画像 読み込んで登録されたテクスチャの画像をリストビューに表示します。

以下いずれかの操作を行うと、テクスチャを読み込んで登録します。

  • 読み込みボタンから開く  
    テクスチャウィンドウの読み込みボタンを押すと、テクスチャを選択するための読み込みダイアログが開きます。
    ファイルを選択すると、テクスチャウィンドウのリストビューに読み込んだテクスチャ画像が表示され、登録が完了します。
  • ドロップで開く
    テクスチャウィンドウにテクスチャファイルをドロップして読み込みます。
    テクスチャウィンドウのリストビューに読み込んだテクスチャ画像が表示されテクスチャの登録が完了します。

関連リンク 「テクスチャウィンドウ」の UI 説明については こちら を参照してください。

テクスチャを指定してピクチャペインを作成する

テクスチャウインドウに登録したテクスチャを使用し、ピクチャペインを作成します。

A テクスチャ画像
 選択状態になったテクスチャ画像
B ペイン作成ボタン 有効になったペイン作成ボタン

以下いずれかの操作をテクスチャウインドウで行うと、レイアウトウインドウ上にピクチャペインが作成されます。

  •   ペイン作成ボタンから作成する
    テクスチャウィンドウのリストビューの画像をクリックするとテクスチャ画像が選択状態になります。
    テクスチャ画像を選択状態にするとペイン作成ボタンが有効になるので、ボタンを押してペインを作成します。
  • ドロップで作成する
    テクスチャウインドウからテクスチャファイルをレイアウトウインドウにドロップしてペインを作成します。

レイアウトウィンドウを確認すると、以下のように「ピクチャペイン」が作成されます。

登録したテクスチャのフォーマットを指定する

テクスチャウィンドウで、テクスチャをバイナリ変換するさいのフォーマットを指定します。
以下にカラーフォーマット「BC1」をモノクロフォーマット「BC4L」に変更する手順を例に説明します。

1 テクスチャウィンドウツールバーの「画像表示」を「詳細表示」に変更します。登録したテクスチャの詳細が、一覧で確認できます。
2

「 登録したテクスチャ」の「フォーマット」が「BC1:(RGB+1A)」であることを確認します。
3 マウス左ボタンクリックで「BC1:(RGB+1A)」をクリックし、表示されたフォーマットの一覧からモノクロフォーマットの「BC4L:(I)」を選択します。

レイアウトウィンドウを確認すると、以下のようにピクチャペインのテクスチャがモノクロに変換されています。
