スケールフェードアウト

スケールフェード

ここでは Fade コマンド実行時にパーティクルを徐々に小さくして消滅させる機能「スケールフェード」について説明します。

概要

「スケールフェード」は、ゲームアプリプログラマが Fade コマンドを実行した時にエミッタからパーティクルの放出を停止し、既に放出されたパーティクルを透明にして削除できる機能です。
「フェードアウト時間」に設定した値をかけて徐々に消滅させることができます。
以下は「 設定項目」の概要です。

スケールフェード

「スケールフェード」の使用 / 未使用を「 チェックボックス」ON/OFF で切り替えます。

関連リンク 「フェードアウト時間」については「フェードアウト時間」を参照してください。
