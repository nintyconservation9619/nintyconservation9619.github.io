マテリアルプロパティウィンドウ プレビューページ

選択しているマテリアルのプレビュー情報を確認して編集するページです。

A 最適化処理に含める
チェックが入っていると、モデルを再読み込みする際にマテリアルがシェーダの最適化処理に含まれます。
チェックのデフォルトは、ON です。
※この機能を有効にするには、ランタイム側の実装が必要です。 
B マテリアルの変更に追従してシェーダを再生成する
最適化処理に含めるが ＯＮ の場合に <DefaultValue> で設定することができます。
チェックボックスの ON / OFF のデフォルト値は、チーム設定にある <DefaultValue> で設定できます。 
C シェーダの送信状態
マテリアル単位のシェーダ最適化状況が表示されます。 

マテリアルの変更に追従してシェーダを再生成

マテリアルの変更に追従して、シェーダを再生成するを ON / OFF した際の挙動は以下になります。

チェックボックスを ON にした場合

シェーダパラメータの編集が行われた際に自動的に最適化シェーダを再生成します。

  チェックボックスを OFF にした場合

シェーダパラメータの編集が行われても、自動的に最適化シェーダを再生成しません。
メインウィンドウのツールバーにある「最適化シェーダ再生成」ボタンを押すことで 再生成が行えます。
