チャイルドエミッタの不具合修正による注意点

本バージョンで、チャイルドエミッタの不具合を修正しました。
ここでは、修正した機能と注意点について説明します。

チャイルドエミッタ「パーティクル追従タイプ」の不具合

チャイルドエミッタの基本設定タブ「エミッタSRTに完全追従」の機能に不具合がありました。
「エミッタSRTに完全追従」と「エミッタ位置にのみ追従」は効いていませんでしたが、修正されました。

以下は、「エミッタSRTに完全追従」に設定したデータについて、修正前後の比較動画です。
█ が親エミッタ、█ がチャイルドエミッタです。

修正前 修正後
「エミッタに追従しない」の挙動になっています。 親エミッタに追従しており、「エミッタSRTに完全追従」が効いています。
注意：

データ作成時のデフォルトが「エミッタSRTに完全追従」ですが、修正前は正しく動作していませんでした。
この状態で見た目調整したチャイルドエミッタを使用したエフェクトは、修正により見た目が変わる可能性があります。
以下は修正前に、不正な挙動となる設定の組合わせです。

パーティクル追従タイプ

エミッタSRTに完全追従

 
エミッタの位置にのみ追従
パーティクルにエミッタ割り当て 「 チェックボックス」OFF

チャイルドエミッタ「エミッタトランスフォーム」の不具合

チャイルドのエミッタタブ「エミッタトランスフォーム」の機能に不具合がありました。
エミッタトランスフォームの値がワールド座標系からエミッタ座標系で適用されるように修正されました。

以下は、チャイルドのエミッタのエミッタトランスフォーム「位置」を X = 8 に設定したデータについて、修正前/後の比較動画です。
█ が親エミッタ、█ がチャイルドエミッタです。

修正前 修正後
ワールド座標系で X 方向にオフセットがかかった見え方です。 エミッタ座標系で X 方向にオフセットがかかっており、正しい見え方です。
注意：

修正前にチャイルドのエミッタトランスフォームの設定を行い、見た目合わせでデータを調整した場合は修正後に見た目が変わる可能性があります。
