EffectViewer の操作

以下は EffectViewer （Windows 版）の操作系をまとめた表です。

クラシックコントローラでの操作
R ボタン
4フレーム毎の描画結果を ビューアの exe ファイルと同階層にある「 ScreenCap」に .png ファイルとして出力します。
クラシックコントローラでのカメラ操作
左スティック 注視点を中心として回転します。
右スティック XZ 平面上で平行に移動します。
x / y ボタン ズームアウト/ズームインします。
L + 右スティック Y方向/左右方向 で平行に移動します。
L ボタン Viewer Info を表示/非表示します。
Home ボタン カメラリセット
マウス / キーボードの操作
左ドラッグ 注視点を中心とした回転します。
中ドラッグ 視線 上下 / 左右 方向で平行に移動します。
右ドラッグ ズームアウト/ズームインします。
マウスホイール ズームアウト/ズームインします。
L デバッグ情報 を表示/非表示します。
S カメラをデフォルトの状態にリセットします。
G
関連リンク デバッグ機能 「床の表示」 を ON/OFF できます。
「床」を表示/非表示するデバッグ機能です。
E
関連リンク デバッグ機能 「親エミッタ形状の表示」 を ON/OFF できます。
親エミッタの形状を表示/非表示するデバッグ機能です。
C
関連リンク デバッグ機能 「子エミッタ形状の表示」 を ON/OFF できます。
子エミッタの形状を表示/非表示するデバッグ機能です。
F
関連リンク デバッグ機能 「フィールド情報の表示」 を ON/OFF できます。
フィールドを表示/非表示するデバッグ機能です。
テンキー 4 / 8 NumLock OFF 時にズームアウト/ズームインします。
マウス / キーボードのカメラ操作
R

4 フレーム毎の描画結果をビューアの exe ファイルと同階層にある「 ScreenCap」に .png ファイル として出力します。
Shift + R
１ フレーム毎の描画結果をビューアの exe ファイルと同階層にある「 ScreenCap」に .png ファイル として出力します。
