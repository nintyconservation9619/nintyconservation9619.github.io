似顔絵リソース

似顔絵リソースは、Mii を描画する際に使用するパーツのテクスチャやシェイプデータといったグラフィックスリソースです。シェイプリソースとテクスチャリソースに分かれており、ともにいくつか種類があります。

各似顔絵リソースファイルは、NintendoSDK の以下の場所に配置してあります。

NintendoSDK\Resources\Mii\Common\resource

シェイプリソース

シェイプリソースには以下の種類があります。

リソースファイル名 説明
ShapeHigh.dat 高品質モデルです。
ShapeMiddle.dat 中品質モデルです。

テクスチャリソース

テクスチャリソースは、プラットフォーム（WinGeneric/WinNvn/NX）、品質（Low/Mid）、ガンマ（リニア / sRGB）の種類毎に提供されます。

テクスチャリソースには以下の種類があります。

WinGenericTextureLowLinear.dat Win/Generic 向けリニアガンマ用低品質テクスチャです。
WinGenericTextureLowSRGB.dat Win/Generic 向け sRGB ガンマ用低品質テクスチャです。
WinGenericTextureMidLinear.dat Win/Generic 向けリニアガンマ用中品質テクスチャです。
