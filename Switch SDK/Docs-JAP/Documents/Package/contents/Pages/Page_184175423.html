サイズ固定カメラオフセット

ここではパーティクルの描画位置を、カメラに対して前後にオフセットさせるとき、パースを打ち消す処理を入れる機能「サイズ固定カメラオフセット」について説明します。

概要

「サイズ固定カメラオフセット」は、スクリーン上（画面上）で頂点単位に大きさを調整した後にカメラそのものを手前に出す「カメラ方向にオフセット」の処理を行う機能です。
「Z値をオフセット」の実装方法では、カメラの「ニア」や「ファー」の設定で表示上の不具合が発生することがあります。
「サイズ固定カメラオフセット」は「カメラ方向にオフセット」の処理で上記不具合を解消した上で、パースを打ち消すことができます。

以下は「 設定項目」の概要です。
「パーティクルオフセット」設定を「サイズ固定カメラオフセット」にすると 「 パーティクルオフセット」が表示されます。以下はグループ内の設定項目です。

パーティクルオフセット値

放出位置を基準に、カメラ方向でのオフセット値を「 ダイヤル」や「 テキストボックス」により設定します。
マイナス方向を指定すると、放出位置より奥側にオフセットがかかります。

オフセット値にスケール乗算

エミッタにかかるスケールとパーティクルにスケールをかけた最終的な結果がZ値に乗算するかを「チェックボックス」ON/OFF で切り替えます。
ON に設定することで、スケールをかけたことによるポリゴンの重なりを解消できます。
