派生部品機能

派生部品機能とは

派生部品機能とは、一つの部品レイアウトを元にして複数の部品レイアウトを作りだす機能です。
派生元の部品レイアウトに、新規にペインを追加したり、アニメーションを追加して新しい部品にすることができます。
(開発中の機能につき、仕様変更の可能性があります。)

何がうれしいか

派生部品を使うと、部品を配置した後に行う調整(上書きプロパティや、部品サイズ倍率)ではカバーしきれなかった調整が可能になります。
その結果、より広い範囲で、一つの部品を共通利用することが可能になり、部品を使ったレイアウト作成で得られる、「データ作成や調整が簡単」というメリットをより多く得られるようになります。

派生部品でできること

ペインを追加する

新しくペインを追加することができます。
派生元部品のペインの子供としてぶら下げることも可能です(つまり、スケールアニメや透明度の影響を共有できます)。

たとえば、共通ボタンの上に、アイコンを一つ加えたボタンが作れます。

アニメーションを追加する

派生元部品のペインに対して、アニメーションを追加することができます(ただし、元々アニメーションしていなかった要素に限ります)。


たとえば、カラーアニメを追加すれば挙動がまったく共通の色違いのボタンを作ることができます。

挙動を変更する

派生部品では、部品設定を変更して挙動を変更することができます。
たとえば、機能アニメーションを変更して、元部品のアニメーションに連動してアニメーションするペインを追加することができます。

派生部品ではできないこと

派生元部品自体の改変

  • ペインプロパティの編集
  • ペイン削除や、アニメーションの削除
  • 元部品の親子構造を変更するような親子設定（例：兄弟の間に割って入る追加）
  • アニメーション区間タグやグループの編集や削除

派生部品の派生

多段階の派生はできません。

派生部品の見え方

派生元のペイン


赤いアイコンで表示されます。レイアウトウィンドウでは選択できません。プロパティは編集不能です。

派生元のペインアニメーション


ツリー要素に、赤い錠前アイコンで表示されます。カーブは黒色で表示されます。キー選択や、編集が不能です。

派生元のグループ、区間タグなど

グレーの文字で表示され、文字背景が薄いブルーで表示されます。削除や編集ができません。
