グローバル乗算カラー・アルファ

ここではエミッタから放出されるパーティクル全体のカラーやアルファに影響を与える機能「グローバル乗算カラー・アルファ」について説明します。

概要

グループ「グローバル乗算カラー・アルファ」は、エミッタから放出されるパーティクル全体のカラーやアルファごとに設定した値を乗算できる機能です。
以下は「 設定項目」の概要です。

カラー 0

エミッタから放出される全パーティクルのカラー0 に乗算します。

カラー 1

ミッタから放出される全パーティクルのカラー1 に乗算します。

アルファ 0

エミッタから放出される全パーティクルのアルファ0 に乗算します。

アルファ 1

エミッタから放出される全パーティクルのアルファ1 に乗算します。
