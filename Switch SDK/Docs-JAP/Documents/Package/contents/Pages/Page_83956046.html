カーブエディタ

選択したアニメーションのカーブを確認、編集します。
関連リンク カーブエディタは、アニメーションプロパティウィンドウのカーブを選択して開くことができます。

関連リンク カーブエディタの基本操作は、こちらを参照してください。またカーブエディタの操作に対応したショートカットキーについては、「ショートカット一覧」を参照してください。

A アニメーションのプロパティ項目が表示されます。
B カーブツリービュー
作成するアニメーションで選択可能なノード（設定項目）がツリー表示されます。
右クリックメニューで、ノードやグループ単位のコピー&ペーストができます。
C 編集しているカーブの要素が表示されます。
カーブの「プレラップ」「ポストラップ」が指定 できます。
ボーンビジビリティのみ表示される項目があります。
D ツールバー
カーブやキーを編集する際に使用する機能をアイコンとして表示します。
アイコンにマウスオーバーすると、機能の簡易的な説明がツールチップとして表示されます。
E カーブビュー
カーブツリービューで選択したノードのアニメーションカーブが表示されます。
アニメーションカーブの作成 / 編集を行います。
F タイムスライダー
タイムスライダー領域の上部は、選択されているカーブに対応した情報が表示されます。

ボーンビジビリティのみの表示

ボーンビジビリティのカーブエディタは以下の項目が追加表示されます。
編集しているカーブ要素が表示されます。 

A

中間ファイルオプティマイザのボーン圧縮において圧縮を許可するかどうかを指定できます。
B 3D バイナリコンバータでコンバートする際に対象のカーブを出力後のバイナリリソースに含めるかどうかを指定できます。
