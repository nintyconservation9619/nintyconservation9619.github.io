プロパティウィンドウ

ここでは「プロパティウィンドウ」の UI について説明します。
「プロパティウィンドウ」には以下の種類があります。

共通プロパティ

複合フォント定義ファイル（.fcpx）の基本情報を編集するプロパティウィンドウです。
ペアフォントプロパティ ペアフォントの設定を編集するプロパティウィンドウです。
ビットマップフォントプロパティ

ビットマップフォントの設定を編集するプロパティウィンドウです。
スケーラブルフォントプロパティ スケーラブルフォントの設定を編集するプロパティウィンドウです。
マルチスケーラブルフォントプロパティ マルチスケーラブルフォントの設定を編集するプロパティウィンドウです。
スケーラブルフォント設定プロパティ スケーラブルフォント設定の設定を編集するプロパティウィンドウです。
フォントプロパティ フォントの設定を編集するプロパティウィンドウです。
