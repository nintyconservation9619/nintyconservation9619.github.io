はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている設定ライブラリ（以後Settingsライブラリ）の機能について説明します。

 

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象とします。

 

各章の説明

「マニュアル」では、Settingsライブラリの提供する個々の機能について使用方法を解説します。

 

関連資料

Settingsライブラリの関数リファレンスと併せて参照してください。

 

Settings ライブラリの機能

Settings ライブラリは、本体設定、デバッグ設定、エミュレーション設定など、各種設定を操作するための API を提供します。
