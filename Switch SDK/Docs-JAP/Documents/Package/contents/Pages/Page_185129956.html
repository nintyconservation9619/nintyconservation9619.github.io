概要

本ページでは USB ファームウェアを更新する PC ツールについて説明します。

このツールで、Joy-Con 充電グリップ および Nintendo Switch Pro コントローラに搭載されている USB ファームウェアをアップデートすることが可能です。

USB ファームウェアをアップデートすることでコントローラ充電ケーブルを使用して SDEV / EDEV との有線ペアリングやコントローラの充電が可能になります。

注意：

USB ファームウェアアップデートツールでは、通常のコントローラFWは更新されません。

それらのFW更新方法は以下のドキュメントをご覧ください。

環境構築ガイド > コントローラーの準備 > コントローラーのファームウェアアップデート

必要なツール

  • コントローラ充電ケーブル
    開発機材に同梱されているコントローラ充電ケーブル（USB Type C-Aケーブル）です。
    Joy-Con 充電グリップ および Nintendo Switch Pro コントローラとPCをこのケーブルを使って接続します。 

  • USB ファームウェアアップデータ
    NintendoSDK NX Addon で配布されている USB ファームウェアを更新するためのツールです。 
Tools/Hid/ChargingGripFirmwareUpdater


手順書


  1. ドライバのインストール

    Tools/Hid/ChargingGripFirmwareUpdater/driver 以下にある ”Dfu driver” および ”Virtual comport driver” に移動してください。

    dpinst_amd64.exeをクリックすることで、それぞれドライバをインストールしてください。


  2. 機材の接続

    PCと Joy-Con 充電グリップ および Nintendo Switch Pro コントローラ を コントローラ充電ケーブルで接続してください。

    その際、必ず機材は1つだけ接続してください。


  3. アップデータの起動
    Tools/Hid/ChargingGripFirmwareUpdater/UsbFirmwareUpdater.exe
    を実行してください。

  4. FW 更新を開始する

    以下の場合、Targetを選択して、Startをクリックしてください。

    Virtual COM Portが表示される場合



    Need Update と表示される場合


  5. 問題がなければ以下のように画面が遷移します。


    更新には、１分程度時間がかかります。

    注意：

    ファームウェア更新中は絶対に コントローラ充電ケーブル を切断したりしないでください。
    ファームウェア更新中に切断すると、最悪の場合ファームウェアが壊れ復帰できなくなる可能性があります。 

  6. 「Finished!!」と表示されたら、Exit ボタンを押してアプリを終了してください。
    　



  7. 機材を PC から取り外します。FW の更新作業はこれで終了です。お疲れ様でした。

更新に失敗した（更新中の画面が先に進まない）

以下のような画面が出た場合 FW のアップデートに失敗しています。


一度、コントローラ充電ケーブルを挿抜して、再度アップデート手順をやり直してください。

 その際、以下の様な画面がでた場合はそのまま Start を押してください。
