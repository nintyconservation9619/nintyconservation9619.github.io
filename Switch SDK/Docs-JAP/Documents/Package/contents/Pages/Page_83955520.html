デバイス

概要

デバイスは、ターゲットの物理 GPU デバイスを表現します。
GFX における他のあらゆるオブジェクトはその初期化にデバイスを要求するため、これらを利用するためには、まず最初にデバイスを初期化する必要があります。

いくつかの低レベルグラフィックス API と異なり、GFX ライブラリーは "現在のデバイス" のようなグローバルなステートをもちません。GFX においてデバイスはオブジェクトとしてアプリケーション側で明示的に管理されます。

デバイスを他のオブジェクトの初期化のために使う際、初期化されるオブジェクトはそのデバイスに結び付けられます。別のデバイスにおいてそのオブジェクトを使うことはエラーを引き起こします。

情報

デバイスの情報は nn::gfx::DeviceInfo として定義されています。

API バージョン

SetApiVersion に、nn::gfx::ApiMajorVersion と nn::gfx::ApiMinorVersion を指定してください。

デバッグモード

SetDebugMode により、デバイスをデバッグモードで初期化するかどうかを指定します。

デバッグ機能を無効化するには DebugMode_Disable を、すべてのデバッグ機能を有効化するには DebugMode_Full を使用します。
DebugMode_Enable を指定した場合は、標準的なデバッグ機能のみを有効化し、パフォーマンスに大きく影響するデバッグ機能は有効化しません。 

オブジェクト

デバイスのオブジェクトは nn::gfx::Device として定義されています。

初期化と破棄

デバイスは他のほとんどの GFX クラスと異なり、その初期化にデバイスを必要としません。

初期化は Initialize によって、破棄は Finalize によって行われます。

単一デバイスの制限

現在、複数の物理 GPU デバイスを扱うための機能は提供されておらず、デバイスはある特定の物理 GPU デバイスを表現します。
初期化する対象の物理 GPU デバイスを指定したり、利用可能な物理 GPU デバイスを列挙するための機能はありません。
