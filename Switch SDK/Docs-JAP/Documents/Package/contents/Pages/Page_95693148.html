NintendoWare Audio ニュース

NintendoSDK 0.7.0 における NintendoWare オーディオ開発環境の変更点を紹介します。

NintendoWare のサウンド開発環境の呼称を、"NintendoWare サウンド" から "NintendoWare オーディオ" に変更

NintendoWare のサウンド開発環境の名称、呼称は、これまで "NintendoWare サウンド" でしたが、NintendoSDK 0.7.0 からは "NintendoWare オーディオ" に変更されました。

NintendoSDK の下位レイヤーは、グラフィックス（nn::gfx） / オーディオ（nn::audio） と呼ばれおり、それに合わせる目的であり、役割やスコープはこれまでと変わりません。

コマンドラインツールの配置ディレクトリ変更

WaveConverter など、単体のサウンドリソースに関するコマンドラインツールの配置先を、以下の通り変更しました。
これらのツールを利用される方は、新しい配置先を参照するように対応をお願いします。

ディレクトリ 説明
Tools/Audio/SoundMaker/

サウンドプロジェクトに関するツールを配置します。

  • SoundProjectConverter.exe
  • SoundProjectWaveExporter.exe
  • 他

Tools/Audio/AtkTools/

単体のサウンドリソースに関するツールを配置します。

  • WaveConverter.exe
  • 他

コマンドラインインターフェイスの変更

NintendoSDK でコマンドラインインターフェイス仕様をある程度そろえる目的で、NintendoWare オーディオのコマンドラインインターフェイスを変更しました。
コマンドラインツールを利用される方は、新しい仕様への対応をお願いします。

詳しい変更内容については、NintendoWare Sound の NintendoSDK 移行ガイド を参照してください。

NintendoSDK 0.6.0 とのバイナリリソースファイルの互換性

nw::snd, nn::atk のバイナリリソースファイル（.bfsar, .bfstm など）のエンディアンがリトルエンディアンに変更され、NintendoSDK 0.6.0 との互換性がなくなりました。

NintendoSDK 0.6.0 でコンバートしたバイナリリースファイルは、NintendoSDK 0.7.0 の nw::snd, nn::atk ランタイムでは利用できません。
NintendoSDK 0.7.0 のコンバータで再コンバートをお願いします。
