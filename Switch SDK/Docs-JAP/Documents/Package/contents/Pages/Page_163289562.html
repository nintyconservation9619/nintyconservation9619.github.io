カテゴリ 名称 意味
追加コンテンツのメタデータ 追加コンテンツインデックス

1 以上 2000 以下の数値です。追加コンテンツの識別に使用します。
リリースバージョン 追加コンテンツのリリースバージョンを示す 16 ビットの値です。
アプリケーション ID

追加コンテンツを使用するアプリケーションを識別する 64 ビットの数値です。
必須アプリケーションリリースバージョン

0 以上 の 16 ビットの数値です。

追加コンテンツを使用するために必要なアプリケーションのリリースバージョンです。

アプリケーション ID で指定したアプリケーションのリリースバージョンは、必須アプリケーションリリースバージョンよりも大きい値である必要があります。
タグ

追加コンテンツの内容を表す任意の文字列です。

指定可能な最大文字数は 128 文字です。

追加コンテンツの価格設定を行う際に操作対象の追加コンテンツを選択する必要がありますが、その際の参考情報としてタグの情報が表示されます。

この文字列から追加コンテンツの内容を推測できるように設定することが推奨されます。

1つの nmeta 内で重複する値を指定することはできません。

本体機能ガイドの「共有データ ／ 内蔵フォント」にある日米欧フォントが対応している文字コード以外の文字は受け付けません。
