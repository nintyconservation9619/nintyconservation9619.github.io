エフェクトの作成

ここではエフェクトデータの作成とエミッタの追加方法について説明します。

 準備

あらかじめ EffectMaker を起動した状態から作業を開始します。

関連リンク EffectMaker の起動方法については 「作業の流れ」 を参照してください。

エミッタセットの作成

エフェクトの作成方法について説明します。
エフェクトデータの制作にとりかかる場合、最初にエミッタセットを作成することから始めます。
以下いずれかの操作を行うとダイアログが表示されますので、エミッタセット名が「EmitterSet1」であることを確認し「OK」ボタンを押します。

  • メインメニュー項目 [ファイル]→[新規作成]→[ エミッタセット] を選択します。
  • ワークスペースの「 エフェクト」または「 エミッタセット」を選択した状態でマウス右ボタンクリックで開くメニューから「 エミッタセットを追加」を選択します。
  • ワークスペースのツールバーにある「 エミッタセットを追加」ボタンを押します。
  • ショートカットキー Ctrl + N を使用します。

ワークスペース上にはエミッタセット 「 EmitterSet1」 が作成されます。

プリセットからエミッタセットを作成

エミッタセットをプリセットから作成する方法について説明します。
よく使用するエフェクトデータはプリセット用フォルダに入れておくことでエミッタセット作成時に利用できます。

サンプルとして最初から登録されているプリセット用エミッタセット「SamplePreset_JustOne」を例に説明します。
以下いずれかの操作を行うとダイアログが表示されますので、エミッタセット名が「SamplePreset_JustOne」であることを確認し、「OK」ボタンを押します。

  • メインメニュー項目 [ファイル]→[新規作成]→「 SamplePreset_JustOne」を選択します。
  • ワークスペースの「 エフェクト」 または を選択した状態でマウス右ボタンクリックで開くメニューから「 プリセットから追加」→「 SamplePreset_JustOne」を選択します。

ワークスペース上にはエミッタセット「 SamplePreset_JustOne」が作成されます。
「  SamplePreset_JustOne」直下には以下の様にエミッタ「 Emitter1」が作られています。

プリセット用フォルダの場所はデフォルトで EffectMaker.Application.exe と同じ場所にあります。
また、フォルダの場所はオプション【ユーザー設定】で変更できます。

関連リンク 詳細については 「プリセットフォルダパス」を参照してください。

エミッタの追加

エミッタの追加方法について説明します。
ワークスペースの「 EmitterSet1」 またはその直下の「 Emitter1」を選択した状態で以下いずれかの操作を行うと
ダイアログが表示されますので、エミッタ名が「Emitter2」であることを確認し、「OK」ボタンを押します。

  • メインメニュー項目 [ファイル]→[新規作成]→[ エミッタ] を選択します。
  • マウス右ボタンクリックで開くメニューから「 エミッタを追加」を選択します。
  • ワークスペースのツールバーにある「 エミッタを追加」ボタンを押します。
  • ショートカットキー Ctrl + Shift + N を使用します。

ワークスペース上にはエミッタセット 「 EmitterSet1」 の下にエミッタ「 Emitter2」が追加されています。
「 EmitterSet1」の右側に、エミッタセットの内容が変わったことを示す「  」マークが付いていることを確認します。

参考：

エミッタはエミッタセットの直下に複数追加できます。

関連リンク エミッタノードについては  「エミッタノード」 を参照してください。

プリセットからエミッタを追加

エミッタをプリセットから追加する方法について説明します。
プリセット用フォルダにあるエミッタセットからエミッタのみをプリセットとして利用できます。
ワークスペースの「 EmitterSet1」またはその直下の「 Emitter2」を選択した状態で以下いずれかの操作を行うと
ダイアログが表示されますので、エミッタ名が「 Emitter1_1」であることを確認し、「OK」ボタンを押します。

  • メインメニュー項目 [ファイル]→[新規作成]→「 SamplePreset_JustOne」→「 Emitter1」を選択します。
  • ワークスペースの「 EmitterSet1」を選択した状態でマウス右ボタンクリックで開くメニューから「 プリセットから追加」→「 SamplePreset_JustOne」→「  Emitter1」を選択します。
  • ワークスペースの「 Emitter2」を選択した状態でマウス右ボタンクリックで開くメニューから「 プリセットから追加」→「 SamplePreset_JustOne」→「 Emitter1」を選択します。

ワークスペース上にはエミッタセット「 EmitterSet1」の階層下にエミッタ「 Emitter1_1」が追加されています。
「 EmitterSet1」直下には以下の様に複数のエミッタが作られています。
