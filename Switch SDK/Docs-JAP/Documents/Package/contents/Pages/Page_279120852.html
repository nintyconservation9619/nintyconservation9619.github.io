プロパティウィンドウからアニメーションキーを設定する機能を追加

プロパティウィンドウからアニメーションキーを設定する機能を追加しました !!

「プロパティ」ウィンドウのアニメーションに対応する項目に「 アニメーションマーク」が表示されます。
「アニメーション区間タグ編集」ウィンドウで「 分割モード」のときは、区間タグを選択しているときに限り「 アニメーションマーク」が表示されます。
以下のように「 アニメーションマーク」もしくは「項目名」をマウス右ボタンクリックにより表示されるメニューからアニメーションキーを設定できます。

また、アニメーションマークの色で項目の状態を確認できます。
以下は項目の表示状態とその説明です。

 項目にアニメーションキーが設定されていない状態です。

項目にアニメーションキーが設定されている状態です。
タイムラインのカレントフレームで値が確定されている状態です。

項目の値が不確定な状態です。
タイムラインを操作したりファイルの保存時に 設定した値が無効になります。

タイムラインのカレントフレームにキーを追加すると「 確定状態」になります。

キャプチャペインの編集を補助するショートカットキーの追加

レイアウトウィンドウにおける「 キャプチャペイン」の編集作業に便利です。

レイアウトウィンドウ上で[ Ctrl ] + [ Alt ] + [ C ]

「 キャプチャペイン」と階層下のペインを含めて表示、非表示を切り替えるショートカットです。
キャプチャペインをレイアウトエリアに配置するような作業スタイルでは作業の妨げになる為、本ショートカットの利用をおすすめします。

レイアウトウィンドウ

[ Ctrl ] + マウス左ボタンドラッグによりキャプチャペインのサイズ変更
