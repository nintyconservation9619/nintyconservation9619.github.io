エミッタ形状「円（等分割）」

ここではパーティクルの放出位置が円周上等しい間隔に並ぶエミッタ形状「円（等分割）」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「円（等分割）」は、「円」エミッタを等分割した箇所から円状に広がった法線方向にパーティクルを放出します。

参考：

エミッタ形状が円（等分割）、ライン（等分割）で且つ放出レートのアニメーションを利用したデータにおいては放出レートが固定値になります。
また、放出レートのアニメーション先頭の値は固定値として「分割数」に引き継がれます。

以下は「 設定項目」の概要です。
「エミッタ形状」を「円（等分割）」に設定すると 「 円（等分割）」が表示されます。 以下はグループ内の（等分割）専用の設定項目です。

ランダム角度

パーティクルを円エミッタの分割位置を基準とした円弧上のランダムな位置から放出します。

関連リンク 詳細については「ランダム角度」を参照してください。

放出タイプ

エミッタの法線方向にパーティクルを放出する時の挙動タイプを以下のプリセットから選択します。

一斉 パーティクルは分割した箇所全てから法線方向へ放出されます。
ランダム パーティクルは分割した箇所のランダムな場所から法線方向へ放出されます。
インデックス順 パーティクルは分割した箇所の描画順に法線方向へ放出されます。

分割数

分割数がパーティクルの放出数となり、「放出レート」の設定は無視されます。
