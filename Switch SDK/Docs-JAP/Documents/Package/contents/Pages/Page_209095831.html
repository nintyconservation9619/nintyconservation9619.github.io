履歴の加速度(±)

ここでは、履歴全体にかかる加速度を各軸ごとに設定する機能「履歴の加速度(±)」について説明します。

概要

「履歴の加速度(±)」は、「履歴」全体にかかる加速度を XYZ 各軸ごとに設定する機能です。
履歴の加速度を設定すると、終端部分が特定の方向に徐々に流れていくようなストライプを作成することができます。
以下は「 設定項目」の概要です。

履歴の加速度(±)

「履歴」の加速度を X Y Z 各軸ごとに「 ダイヤル」や「 テキストボックス」により設定します。
値が大きいほど履歴が大きく動くようになります。

設定例

以下は「履歴の加速度」の値を変えた場合のストライプです。
波形の初速は 0 に設定しています。

A Z = 0 ストライプはまっすぐに動きます。
B Z = 0.001 A と比較すると、ストライプの末端が徐々に下方向に動きます。
C Z = 0.0025

B と比較すると、ストライプの末端が大きく下方向に動きます。
