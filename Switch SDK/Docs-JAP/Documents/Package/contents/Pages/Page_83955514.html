スケルタルアニメーション

スケルタルアニメーションは、transform ノードおよび joint ノードの 移動（Translate）、回転（Rotate）、スケール（Scale） をアニメーションさせて設定します。

次のような場合は、必ずベイクしたデータからアニメーションカーブデータを自動作成して出力します。

  • IK、コンストレイン、ダイナミクスなどを用いたアニメーション。
  • パスアニメーション。
  • 回転ピボット（Rotate Pivot）アトリビュートとスケールピボット（Scale Pivot）アトリビュートが異なる。
  • 回転順序（Rotate Order）アトリビュートが xyz でない。
  • 回転軸（Rotate Axis）アトリビュートが (0, 0, 0) でない。
  • ジョイントの方向（Joint Orient）アトリビュートが (0, 0, 0) でない。
  • クオータニオンによる回転の補間をしている。
