ソフトパーティクル

ここではモデルと重なったパーティクルの境目に対して半透明処理を施す機能「ソフトパーティクル」について説明します。

概要

「ソフトパーティクル」は、モデルとパーティクルが重なったとき、パーティクルの境目に半透明処理を施す機能です。
この処理により、境界に見られるラインを目立たなくし、エフェクトを自然に見せることができます。

以下は視点から「 █ パーティクル面」と「█ 床面」が重なったときのアルファ値の変化について説明した画像と図です。

A パーティクルと床面の重なったまたは遮蔽された箇所がアルファ値 0 になります。
B 最も外側の頂点やフラグメントのアルファ値は 1 になります。

「Distance」 はこの機能の設定箇所です。アルファのかかる範囲を設定できます。

C 範囲を設定する前の状態です。
D 範囲を設定した後の状態です。

以下は「 設定項目」の概要です。

ソフトパーティクル

ソフトパーティクルの処理方法を以下のプリセットから選択します。

利用しない ソフトパーティクルを利用しません。
頂点で処理

パーティクルがモデルと重なったとき、パーティクルの頂点に厚みを考慮した半透明処理を施し境目を目立たなくします。
頂点シェーダで処理します。
フラグメントで処理

パーティクルがモデルと重なったとき、パーティクルのフラグメントに半透明処理を施し境目を目立たなくします。

フラグメントシェーダで、ピクセル単位に処理します。
遮蔽チェック

パーティクルがモデルと重なったとき、パーティクル全体のアルファ値を変化させることで、透明にすることができます。

「サンプリング半径」の値を上げることで、遮蔽物との重なりの判定範囲を広げることができます。
