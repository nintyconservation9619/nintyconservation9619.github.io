低レベルグラフィックス API の選択

通常、ここに書いている内容について気にかける必要はありません。

たとえばリソース化するためのツールなどにおいて、低レベルグラフィックス API を選択する必要がある場合などのために使う機能です。

特別用途がない場合は使用しないようにしてください。

選択方法

使用する低レベルグラフィックス API は、nn/gfx.h をインクルードする前にプリプロセッサーマクロを記述することにより、またはコンパイラーにプリプロセッサーマクロを定義させることにより、静的に選択できます。

#define NN_GFX_CONFIG_TARGET_API NN_GFX_CONFIG_API_GL4
#include <nn/gfx.h>

これらマクロはすべて大文字で記述されます。

選択できる内容

以下のいくつかの内容を選択することができます。

  • 対象とする低レベルグラフィックス API
  • 使用する低レベルグラフィックス API
  • 低レベルグラフィックス API の対象バージョン

対象とする低レベルグラフィックス API

NN_GFX_CONFIG_TARGET_API マクロに、対象としたい低レベルグラフィックス API の名前を指定します。

GL4 を指定する場合は以下のように記述します。

#define NN_GFX_CONFIG_TARGET_API NN_GFX_CONFIG_API_GL4

低レベルグラフィックス API の種類だけを指定することも可能です。

#define NN_GFX_CONFIG_TARGET_API NN_GFX_CONFIG_API_GL

この場合、対象のバージョンは、その低レベルグラフィックス API の対象バージョンとして指定されているものになります。

デフォルト

対象とする低レベルグラフィックス API を指定しなかった場合は、デフォルトの低レベルグラフィックス API が選択されます。

デフォルトの低レベルグラフィックス API は、プラットフォームによって異なります。

使用する低レベルグラフィックス API

NN_GFX_CONFIG_INCLUDE_* マクロに 1 を設定することにより、使用する低レベルグラフィックス API を追加することができます。

"*" の部分には低レベルグラフィックス API の名前が入ります。

対象としている低レベルグラフィックス API は必ず追加されるため、明示的に追加する必要はありません。

GL4 を追加する場合は以下のように記述します。

#define NN_GFX_CONFIG_INCLUDE_GL4 1

低レベルグラフィックス API の種類だけを指定することも可能です。

#define NN_GFX_CONFIG_INCLUDE_GL 1

この場合、対象のバージョンは、その低レベルグラフィックス API の対象バージョンとして指定されているものになります。

低レベルグラフィックス API の対象バージョン

NN_GFX_CONFIG_*_TARGET_VERSION マクロにバージョン番号を指定することにより、その低レベルグラフィックス API の対象バージョンを指定することができます。

この機能は、対象とする低レベルグラフィックス API の選択機能や使用する低レベルグラフィックス API の追加機能と組み合わせない限り意味をもちません。

"*" の部分には低レベルグラフィックス API の種類名が入ります。

GL の対象バージョンを 4 とする場合は以下のように記述します。

#define NN_GFX_CONFIG_GL_TARGET_VERSION 4

デフォルト

低レベルグラフィックス API の対象バージョンを指定しなかった場合は、デフォルトのバージョンが選択されます。
