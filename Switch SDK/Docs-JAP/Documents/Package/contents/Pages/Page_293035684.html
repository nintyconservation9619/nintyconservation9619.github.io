派生部品の利用

ここでは「派生部品」の機能を利用し、既にあるセレクトボタンの部品から異なるボタン部品を作成する手順について説明します。
「部品アニメーションの作成」 で作成した部品「Menu_Bar.flyt」を、あるメニュー画面でしか使用しない、以下のようなボタンに派生させて利用するケースを例に説明します。

セレクトボタン右上に「新機能」を意味する「New!!」というマークを付けてみました。

作業の準備

あらかじめ LayoutEditor を起動した状態から、レイアウトデータ作成にあたっての準備を行います。

関連リンク LayoutEditor の起動方法については 「作業の流れ」 を参照してください。

ウインドウを開く

以下のウインドウが開いた状態から作業を開始します。

  • 部品パレット
  • 親子階層
  • プロパティウインドウ

関連リンク 各ウインドウの UI については 「ウインドウ」 を参照してください。

派生部品の作成

以下の手順により部品レイアウト「Menu_Bar.flyt」から「派生部品」となるレイアウトデータを作成します。

1
「部品パレット」ウィンドウで部品レイアウト「Menu_Bar.flyt」を選択します。
2 マウス右クリックで開くメニューから、「派生部品を作る」を選択します。
3

レイアウトウインドウに新規レイアウトデータ「Untitled.flyt」が作成されたことを確認します。
4 「親子階層」ウィンドウを確認すると、「Menu_Bar.flyt」のペインが表示されています。

以下のように「親子階層」ウィンドウには「編集禁止」を示す赤いアイコンで表示されています。
派生元となるセレクトボタンの部品「Menu_Bar.flyt」の内容は変更できません。

レイアウトデータの別名保存

「派生部品」となるレイアウトデータ「Untitled.flyt」を一旦保存します。
メインメニュー[ ファイル ] → [ 別名で保存 ] で表示されるダイアログでファイル名に「Menu_DerivingParts.flyt」を設定します。
「名前を付けて保存」ダイアログの OK ボタンを押し、編集内容を保存します。

以降は「派生部品」「Menu_DerivingParts.flyt」を編集し、部品レイアウト「Menu_Bar.flyt」から派生した異なるセレクトボタンの部品を作成します。

テクスチャの読み込み

ボタン右上に配置する「New!!」の枠

以下のテクスチャファイル（.tga）を「テクスチャ」ウインドウに読み込みます。
Samples/Sources/Resources/Ui2d/Common/Textures/Derivation_Base.tga

「テクスチャ」ウインドウを「詳細表示」に切り替え、テクスチャのフォーマットを以下の様に設定します。

サムネイル

 ファイル名 フォーマット 説明

Derivation_Base.tga A8 :（非圧縮_A） 1 チャンネルの画像をアルファ値として利用できるテクスチャフォーマットです。
セレクトボタン右上に配置する文字列「New!!」の枠として利用します。

関連リンク テクスチャの登録方法については こちら を参照してください。

「ピクチャペイン」の作成　

テクスチャを表示するウインドウ「テクスチャ」から「 ピクチャペイン」を作成します。

「テクスチャ」で「（画像） Derivation_Base.tga」をダブルクリックします。
「親子階層」ウインドウで作成された「 ピクチャペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

 ペイン名 Derivation_Base このペインはセレクトボタン右上に配置する文字列「New!!」の枠になります。
位置 X,Y = 240,16

カラー補間設定

 白カラー補間 R,G,B,A = 0,0,0,200

関連リンク 「ピクチャペイン」の概要については こちら を参照してください。

「テキストボックスペイン」の作成

文字列「New!!」を「 テキストボックスペイン」で作成します。

メインメニュー[ ペイン ] → [ ペイン作成 ] → [ 「テキストボックスペイン」 ] を選択します。
開いた「テキストボックス作成」ダイアログで以下のように設定します。

フォントファイルで指定 Samples/Sources/Resources/Ui2d/Common/Fonts/sample.bffnt
文字列 New!!

「テキストボックス作成」ダイアログの OK ボタンを押します。
「親子階層」ウインドウで作成された「 テキストボックスペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

 ペイン名 T_Derivation

このペインは先に配置した「 ピクチャペイン」「Derivation_Base」の手前に配置する文字列になります。
位置 X,Y = 245,18 ピクチャペイン「Derivation_Base」の中央に配置されるように位置を調整します。

テキストボックスペイン

 斜体傾き(角度指定) -10 ピクチャペイン「Derivation_Base」の形状に合わせ、フォントの斜体角度を調整します。

カラー補間設定

 白カラー補間 R,G,B,A = 255,200,0,255

作成した「New!!」のマークはセレクトボタン右上に付加されますが、セレクトボタンのアニメーションに連動していません。

派生部品に階層に組み入れる

「親子階層」ウィンドウで「 N_Menu」の階層下に作成した「 Derivation_Base」と「 T_Derivation」配置します。
これにより、派生元部品「 N_Menu」のアニメーションを継承できます。
ビューアで確認すると、作成した「New!!」のマークはセレクトボタンのアニメーションに連動しています。

参考：

このように、派生先となるレイアウトデータでは、「親子階層」ウィンドウで派生元ペインに対して新たにペインを組み入れることができます。
この為、上位階層となる派生元のペインがアニメーションしているとき、そのアニメーションを組み入れたペインに継承できます。
この「派生部品」の特徴を活かすことで、共有したいアニメーションを派生元に設定し、これを派生先で利用するといった使い方もできます。

派生の解除

派生元との関連付けを解除したい場合は「部品パレット」ウィンドウから行います。
但し、この操作はやり直しができません。バックアップ等をとってからの利用をおすすめします。

以下に派生元「Menu_Bar.flyt」との関係を解除し、通常のレイアウトデータに変更する手順を説明します。

1

メインメニュー[ ファイル ] → [ 閉じる] で「Menu_DerivingParts.flyt」を閉じます。

LayoutEditor に複数のレイアウトが読み込まれている場合は同様の手順で全て閉じます。
2 「部品パレット」ウィンドウで派生先となる「Menu_DerivingParts.flyt」を選択します。
マウス右クリックメニューの「派生元への関連付けを修正」を選択し、「派生元への関連付けを修正」ダイアログを開きます。
3

ダイアログの派生先ファイル一覧で「Menu_Bar.flyt」が「 チェックボックス ON」であることを確認します。
4

「新しい名前」の選択項目から「 派生関係を消して通常部品にする」を選択します。
5 設定内容を確定する為に「OK」ボタを押し、ダイアログを閉じます。
6 設定内容を確認する為に「部品パレット」ウィンドウで派生先となる「Menu_DerivingParts.flyt」を選択します。
マウス右クリックメニューの「開く」を選択し、「Menu_DerivingParts.flyt」を開きます。
7

「親子階層」ウィンドウを確認すると、「 派生状態を表す赤いアイコン」は「 ロックのかかった状態のアイコン」に戻っています。
「 ロックマーク」をクリックすることで「 編集可能なアイコン」に変わります。
