タイムスライダー（ カーブエディタ ）

カーブエディタのカーブビューにあるタイムスライダー領域の上部は、選択されているカーブに対応した情報が表示されアニメーションの処理をフレーム単位で編集できます。

対応しているアニメーションは以下になります。

  • カラーアニメーション / シーンアニメーション ( ライト / フォグ ) /シェーダーパラメータアニメーション ( カラー )
  • ビジビリティアニメーション
  • テクスチャパターンアニメーション

カラーアニメーション / シーンアニメーション ( ライト / フォグ ) / シェーダーパラメータアニメーション ( カラー )

カラーアニメーションや、シーンアニメーション ( ライト / フォグ ) / シェーダーパラメータアニメーション ( カラー ) を編集する場合は、打たれているキーの色味がカラーバーとして表示されます。

 

キーが打たれているフレームには、カラーアイコンが表示されます。

カラーアイコンは、マウスの左ドラッグで任意のフレームに移動できます。

カラーピッカーで色の設定やキーを追加する

関連リンク カラーバーをクリックか、カラーアイコンをクリックすると、カラーピッカーを使用して色味の変更ができます。
また、カラーバーをクリックしたフレームにキーがない場合はキーが追加され、続いてカラーピッカーで選択した色のキーになります。

カーラーの値をコピー & ペーストする

カラーバー上でマウスを右クリックすると、マウスの位置のカラーをコピーすることができます。

コピー後、貼り付けしたい位置にマウスを動かして「カラーを貼り付け」できます。

カラーに 0 - 1 の範囲外の値が使用されていた場合

カーブエディタでカラーアニメーションなど編集している際に、 0 - 1 の範囲外の値が使用されていた場合は以下の挙動になります。

RGBA カーブのすべてのキー値が 0 - 1 に収まっている カラーバーやキーの色を表すボックスを表示します。
キーの値は0 - 1に収まっていて、カーブが 0 - 1の範囲外 カラーバーは表示されません。
キーの色を表すカラーアイコン（RGBA の一つのキーに対して、正規化した色）は表示します。

ビジビリティアニメーション

ビジビリティアニメーションを編集する場合は、タイムスライダーに、表示（白）と、非表示（黒）を表すバーが表示されます。

  • バーをクリックすると、クリックしたフレームにキーが追加されます。
  • 目アイコンをクリックすると、表示 / 非表示が切り替えられます。
  • 目アイコンは、マウスの左ドラッグで任意のフレームに移動できます。

テクスチャパターンアニメーション

テクスチャパターンアニメーションを編集する場合は、タイムスライダーにテクスチャがサムネイル化されて表示されます。

  • テクスチャが変わるフレームには、テクスチャアイコンが表示されます。
  • テクスチャアイコンは、マウスの左ドラッグで任意のフレームに移動できます。
  • テクスチャのサムネイルにマウスオーバーすると、テクスチャが拡大表示されます。

参考：
関連リンク テクスチャが拡大表示されている状態でマウスを左クリックするか、テクスチャアイコンをクリックするとテクスチャ画像の変更ダイアログが開いてパターンで使用するテクスチャを設定できます。

テクスチャパターンアニメーションの編集

カーブエディタでテクスチャパターンアニメーションの編集を行う際に、ツールバーにある「値をクランプする」機能を ON にすると、読み込んでいるテクスチャのインデックス以外の値が編集できなくなります。

テクスチャパターンアニメーションの編集で、誤ったインデックス設定を防ぐのに便利です。

  • リアルタイム編集中は、値のクランプが常に ON 状態になります。
  • リアルタイム編集中にパターンにないキーが打たれたり、テクスチャの参照が切れた場合は、警告ダイアログが表示され接続中のアプリケーションとの接続が切断されます。
  • アプリケーションとの接続開始時に、不正なテクスチャパターンアニメーションを転送しようとすると警告ダイアログが表示されアプリケーションに接続できません。

ドラッグ & ドロップでテクスチャを追加

関連リンク ファイルビューや、エクスプローラからカーブビューにテクスチャをドラッグ & ドロップすることにより、パターンアニメーションに使用しているテクスチャの追加が行えます。

カーブビューにドロップすると、ドロップしたフレームに既存のパターン番号の最大値に追加される形でキーが追加されます。

例：
既存のパターンが4番まであった場合は、ドロップしたテクスチャが自動的にパターンの5番として追加されカーブ上にキーも追加されます。
