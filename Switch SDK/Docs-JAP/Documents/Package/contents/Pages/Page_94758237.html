グラデーションエディタの操作

ここではプロパティウィンドウでカラーやアルファにアニメーションを付ける UI 「グラデーションエディタ」の操作について説明します。

関連リンク チュートリアル「炎を作る」の「パーティクルのカラーをアニメーションさせる」の部分について具体的な操作方法を説明します。

以下の動画はカラーアニメーションを設定するUIの操作方法です。

  • 「 グラデーションバー」の上でマウス左ボタンクリックすることで「 アニメーションキー」を追加できます。
  • 「 アニメーションキー」をマウス左ボタンクリックしたまま横に移動させます。
    「 グラデーションバー」の長さ全体をパーティクル寿命の長さとしてキーのタイミング（時間）を設定します。
  • 追加した「 アニメーションキー」をマウス左ボタンクリックし、キーの色を設定します。
  • 「 アニメーションキー」を削除したい場合はキーの上にある「 キーの削除ボタン」をマウス左ボタンクリックします。
