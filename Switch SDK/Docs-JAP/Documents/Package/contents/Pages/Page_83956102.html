深度ステンシルビュー

概要

深度ステンシルビューは、テクスチャーを深度ステンシルとしてセットするための機能を提供します。
ビューとしての特性は基本的にテクスチャービューと同じです。テクスチャービューを参照してください。 

深度ステンシルビューは、元のテクスチャーの部分を、配列範囲と特定のミップマップレベルによって表現します。

情報

深度ステンシルビューの情報は nn::gfx::DepthStencilViewInfo として定義されています。

次元、フォーマット、テクスチャーポインター

テクスチャービューとほぼ同じであるため、そちらを参照してください。

元となるテクスチャーのフォーマットは深度ステンシルとして利用可能である必要があります。

元となるテクスチャーは、GPU がアクセスする方法として GpuAccessFlag_DepthStencil を含んで初期化されている必要があります。

サブリソース範囲

EditArrayRange 関数により配列範囲を、SetMipLevel により特定のミップマップレベルを指定します。

オブジェクト

深度ステンシルビューのオブジェクトは nn::gfx::DetphStencilView として定義されています。

初期化と破棄

深度ステンシルビューの初期化と破棄はオブジェクトの規則に従います。

深度ステンシルビューのセット

nn::gfx::CommandBuffer クラスの SetRenderTargets にカラーターゲットビューの配列と合わせて指定することにより、深度ステンシルとして深度ステンシルビューをセットすることができます。
セットする深度ステンシルビューのフォーマットは、レンダーターゲットステートの深度ステンシルターゲットのフォーマットと一致している必要があります。

深度ステンシルビューのクリア

深度ステンシルビューには深度書き込みとステンシル書き込みを通して自由に書き込みを行うことができますが、深度ステンシルの全ピクセルについて一律に同じ深度値とステンシル値を書き込むためのクリア関数が用意されています。

深度ステンシルのクリアは、nn::gfx::CommandBuffer クラスの ClearDepthStencil 関数を通して行われます。すべてのピクセルの深度値、ステンシル値についてどの値でクリアするかを指定します。
このとき、深度ステンシルターゲットにおける特定の配列範囲だけをクリアすることも可能で、それは pArrayRange に望む範囲を指定することで行われます。このパラメーターに NULL を指定した場合はすべての配列範囲をクリアします。

クリアにおける制限

クリア後はレンダーターゲットの設定が無効化されます。必要な場合は再度レンダーターゲットを設定してください。
