追加参照パス

概要

追加参照パスは、チーム設定のファイル入出力設定やモデルやマテリアルアニメーションに設定されている参照ファイルの検索に使われます。
絶対パスが設定された追加参照パスは、スタートアップパスの起点、プロジェクトに含まれるファイルのパスの起点としても利用されます。

参考：

モデルやマテリアルアニメーションを開くときに参照するテクスチャファイルを検索します。
検索で最初に見つかったテクスチャファイルが開かれていなければ、テクスチャファイルを開きます。

モデルを開くときに参照するシェーダ定義と同名のシェーダ定義ファイルが 3DEditor 上に開かれていなければ、シェーダ定義ファイルを検索して開きます。

検索する範囲と順序

検索する範囲と順序は以下の通りです。

参照元マテリアルアニメーション中間ファイルをバインドするモデル中間ファイルに設定された追加参照パスのフォルダ

各中間ファイルの追加参照パス

(アサインユーティリティで割り当てることができます)

4

参照元ファイルが含まれるフォルダ内の textures フォルダ（参照先がテクスチャの場合のみ）

 追加参照パスで指定する検索範囲は 2,3 です。1,4,5 は 3DEditor が自動で検索します。参照元ファイルは 3DEditor で開かれているファイルです。

 

 

追加参照パスの複数設定

追加参照パスが複数設定された場合は、設定した順に検索が行われます。
追加参照パスで指定されたフォルダのサブフォルダを検索対象にするかは、追加参照パスごとに設定できます。

最終的に参照先のファイルが見つからなかったとしてもエラーにはなりません。

参考：

追加参照パスにはフォルダのパスを指定してください。
絶対パスまたは参照元ファイルからの相対パスを設定することができます。
パスの文字列には % で囲われた環境変数を含めることができます。3DEditor は % で囲われた環境変数を環境変数の値に置き換えて解釈します。

設定ファイルで絶対パスを指定する場合

  • 追加参照パス名も設定してください。
  • 追加参照パス名は絶対パスが設定された追加参照パスに対してのみ有効です。
  • 追加参照パス名は重複しないようにしてください。

中間ファイルに対する追加参照パスは、アサインユーティリティまたは 3DEditor のプロパティウィンドウの追加参照パスページから設定できます。

中間ファイルに対する追加参照パスで絶対パスを指定する場合 % で囲われた環境変数から始まるパスを指定してください。
