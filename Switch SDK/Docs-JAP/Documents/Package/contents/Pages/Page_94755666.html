「ビルボード」と「ディレクショナル」

ここでは「ビルボード」と「ディレクショナル」の違いについて説明します。

概要

「ディレクショナル」はパーティクルの速度をパーティクルの向きに影響させるパーティクルタイプです。
以下は「ビルボード」と「ディレクショナルビルボード」を比較した画像です。各エミッタ共に全方向に速度を与えています。

A ビルボード 指向性のあるパーティクルでは違和感があります。
B ディレクショナルビルボード 速度方向にパーティクルが傾く為、指向性のあるパーティクルでは有効な場合があります。
