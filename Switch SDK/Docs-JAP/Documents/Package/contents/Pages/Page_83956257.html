Physique

Physique の出力に対応しています。

ただし、Physique 特有の機能（バルジ、腱、皮膚のスライドなど）は対応していません。一般的なスキンとして出力されます。
出力されるデータは Physique モディファイアの「皮膚の更新」を「リジッド」にしたものとほぼ同じになります。

対応しているオブジェクトは、Bone、Biped、ダミー、ポイントヘルパーとなります。

参考：

Physique モディファイヤを適用した後に、次のような操作を行なわないでください。

  • 頂点数を変更するような操作
  • スキニングに影響しているノードの削除

 モデルファイルに出力される形状

Physique では、フィギュアモードの状態が出力されますが、出力後、Biped のポーズが変更されてしまいます。
エクスポートを行なう前に必ず、Biped にキーを作成するなどしてポーズを保存しておいてください。

参考：

各頂点において、ウェイト値の合計が1.0にならない場合、ウェイト比率を保って合計1.0に調整して出力しています。
