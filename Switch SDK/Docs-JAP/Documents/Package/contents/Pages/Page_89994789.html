プロジェクトで使用する SDK のパスを変更する

はじめに

このページについて

このページでは、NintendoSDK アプリケーションプロジェクト・ライブラリプロジェクトが参照する SDK ディレクトリのパス (SDK ルートパス) の変更方法について説明します。

注意事項

このページでは典型的な方法で作成されたプロジェクトを想定し変更方法を掲載していますが、プロジェクトによっては独自の方法で SDK ルートパスを設定している場合があります。このページで取り上げている方法で SDK ルートパスを正常に変更できない場合は、独自の方法で SDK ルートパスが設定されていないかどうかご確認ください。

NintendoSDK 構成マネージャを使用する

プロジェクトが参照する SDK ルートパスは NintendoSDK 構成マネージャを利用して変更することができます。NintendoSDK 構成マネージャは、NintendoSDK が提供する Visual Studio 拡張機能です。導入方法や使い方については NintendoSDK 構成マネージャ を参照してください。

何らかの理由で NintendoSDK 構成マネージャを使用しない場合は、SDK ルートパスを直接変更することもできます。次の プロジェクトの SDK ルートパス設定を直接変更する を参照してください。ただし、NintendoSDK 構成マネージャを使用して設定を行ったプロジェクトについては、必ず NintendoSDK 構成マネージャを使用して SDK ルートパスを変更してください。

プロジェクトの SDK ルートパス設定を直接変更する

NintendoSDK 用プロジェクトは、利用する SDK ルートパスをなんらかの方法で保持しています。ここでは、典型的なプロジェクトを 2 つ取り上げ、それぞれのプロジェクトに応じた SDK ルートパスの変更方法を説明します。

  • ウィザードで作成したプロジェクトの場合
  • サンプルを別の場所に移動した場合

いずれの方法でも、Visual Studio でプロジェクトファイルを開いている場合は、あらかじめ閉じておいてください。

ウィザードで作成したプロジェクトの場合

ウィザードを利用してプロジェクトを作成した場合、ウィザードの案内にしたがって設定した SDK ルートパスはプロジェクトファイル内に保存されています。

ここで設定した SDK ルートパスを変える手順は以下の通りです：

  1. ウィザードで作成したプロジェクトファイルをテキストエディタ等で開きます。

  2. NintendoSdkRoot のタグの値を指定したいパスに変更します。環境変数によるパスを指定することも可能です。

    例：SDK ルートパスとして "C:\foo\bar" を設定する場合 
    <NintendoSdkRoot>C:\foo\bar</NintendoSdkRoot>
    例：SDK ルートパスとして環境変数 XXX_ROOT に設定されたパスを指定する場合 
    <NintendoSdkRoot>$(XXX_ROOT)</NintendoSdkRoot>

ウィザードの利用に関しては、新規プロジェクトの作成 をご覧ください。

プロジェクトファイルに NintendoSdkRoot タグが存在しない場合...

NintendoSDK 0.15.0 以前のウィザードから作成されたプロジェクトの場合、プロジェクトファイル内に NintendoSdkRoot のタグが存在しないことがあります。その場合は、次に示す サンプルを別の場所に移動した場合 の方法に従って SDK ルートパスを設定してください。

サンプルを別の場所に移動した場合

NintendoSDK のサンプルは、そのサンプルが配置されている NintendoSDK ディレクトリのパスを SDK ルートパスとして使用します。

プログラムの検証時などに、サンプルプログラムをもともとあったディレクトリとは別のディレクトリにコピーして利用する場合は、SDK ルートパスを設定しなおす必要があります。次の 2 つのうち、どちらかの手順で SDK ルートパスを変更することができます。

  • 以下のような XML ファイルを作成し、プロジェクトと同じディレクトリに NintendoSdkRootDefinition.props というファイル名で配置する

    <?xml version="1.0" encoding="utf-8"?>
<Project ToolsVersion="4.0" xmlns="http://schemas.microsoft.com/developer/msbuild/2003">
  <PropertyGroup>
    <NINTENDO_SDK_ROOT>C:\NintendoSDK</NINTENDO_SDK_ROOT>  <!-- NintendoSDK のルートパスを設定してください -->
  </PropertyGroup>
</Project>
  • 環境変数 NINTENDO_SDK_ROOT の値を、変更先の SDK が展開されているパスに設定する
