初速のランダム(%)

ここではパーティクルの初速をランダムにする機能「初速のランダム(%)」について説明します。

概要

「初速のランダム(%)」は、パーティクルが放出された瞬間にパーティクル個々の速度をランダムにする機能です。
同時に複数のパーティクルが放出された際に、初速をランダムにすることで見た目にばらつきを与えます。
以下は「 設定項目」の概要です。

初速度のランダム(%)

パーティクル放出時の初速度をランダムに変化させます。
初速度を乱数の上限、下限側はランダムに変化します。以下は乱数の下限をもとめる計算式です。

  • 乱数の下限 : 初速度 - 「初速度のランダム(%)」の計算結果

設定例

以下は Y 方向のみ初速を 0.5 に設定し、「初速のランダム(%)」の値を変更したときの比較画像です。

A 0 %（デフォルト） 放出毎の初速は変化しない為、一定間隔で放出されています。
B 50 % 放出毎の初速は、下限が初速の半分 0.25 、上限が初速 0.5、この間でランダムに変化します。
C 100 % 放出毎の初速は、下限が初速ゼロ、上限が初速 0.5、この間でランダムに変化します。
