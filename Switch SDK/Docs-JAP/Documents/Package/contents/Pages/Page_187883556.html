ミップマップレベルを制限する機能

「ミップマップレベルを制限」は、ミップマップレベルに制限をかける機能です。
これにより解像度の低すぎるミップマップを参照しないようにできます。

以下は平面パーティクルに登録したテクスチャがミップマップ「なし」か「あり」かで比較した画像です。
パーティクルを Y 軸で回転させることにより画面上テクスチャ横方向の表示面積に変化を与えています。

A ミップマップなし テクスチャをそのまま表示します。 B 「ミップマップあり」と比較してください。
B ミップマップあり 面積がが小さくなるにつれ、解像度の低いミップマップレベルが参照されます。
← の箇所では低すぎる解像度が参照されることにより、縦方向にぼかされた表示になります。
ゲームアプリケーションにより、この現象は望ましくない表示になります。

以下は「ミップマップレベルを制限」を ON/OFF したときの比較画像です。
上記現象が発生するミップマップレベルが参照されない為、問題を抑制できます。

適用したミップマップのレベル
ON 適用されるミップマップレベルが 0 ～ 2 レベルまで制限されています。
表示面積が小さくなっても表示されるミップマップレベルは 2 のままです。
OFF

デフォルト設定です。表示面積が小さくなるれば対応するミップマップレベルで表示されます。
