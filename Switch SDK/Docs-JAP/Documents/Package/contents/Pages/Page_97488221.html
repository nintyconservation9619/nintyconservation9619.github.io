テクスチャリング

ここでは、ストライプに対するテクスチャの貼り方を設定する機能「テクスチャリング」について説明します。

概要

「テクスチャリング」は、ストライプに対するテクスチャの貼り方を設定できる機能です。
以下は「 設定項目」の概要です。

テクスチャリング

ストライプに対するテクスチャの貼り方を、以下のプリセットから選択します。

全て貼る テクスチャ画像がストライプのサイズに合わせて貼られます。
描画範囲に合わせて貼る テクスチャ画像が一定の描画サイズで貼られます。

設定例

以下は「テクスチャリング」の設定を変更した場合の比較画像です。

A 全て貼る ストライプのサイズに合わせてテクスチャが貼られます。
B 描画範囲に合わせて貼る テクスチャが一定の描画サイズで貼られます。
A と比較すると、ストライプ生成時のテクスチャ表示が変わります。
