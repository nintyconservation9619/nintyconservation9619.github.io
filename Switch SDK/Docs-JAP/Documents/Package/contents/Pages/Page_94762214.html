グラデーションエディタ

「グラデーションエディタ」ではパーティクル時間によるカラーやアルファにアニメーションを設定できます。
「 カラータブ」の「 カラー・アルファ設定」で設定できます。

A 挙動タイプ

選択項目から「8キーアニメ」を選択することで、カラーアニメーション可能な「グラデーションエディタ」が表示されます。

アニメーションが有効な状態です。
B キー

「 アニメーションキー」です。キーをマウス左ボタンクリックすることで、その位置（タイミング）のカラーを登録できます。
「 グラデーションバー」の横幅が時間軸とし、マウス左ボタンクリックした位置にキーを設定できます。 
C キーの選択 「 アニメーションキー」はマウス左ボタンクリックすることで選択できます。
D キーの削除 「 アニメーションキー」は「 キーを削除」ボタンをマウス左ボタンクリックすることで削除できます。
E 「再生方法」切り替えボタン 再生方法を「 リニア再生」と「 ループ再生」で切り替えます。
F ループ時のフレームの長さ 「再生方法」でループを ON にした場合に設定できます。アニメーション全体がここで指定したフレームで繰り返し再生されます。
G 開始位置ランダム

パーティクル放出時のカラーを毎回変化させるかを「 チェックボックス」ON/OFF で切り替えます。

ON にすることで、パーティクル放出時に使用されるカラーはグラデーション全体からランダムに選択されます。
H プリセット ボタンを押すことで、よく使用するアルファアニメーションを生成できます。

プリセット

「プリセット」はエフェクトでよく使用するアルファアニメーションをボタン一発で生成できるボタンです。
以下にボタンの種類ごとのアニメーション挙動について説明します。

2キー

パーティクル寿命の範囲でアルファ値 1.0 から 0.0 に変化します。

寿命が短い、または他のエフェクトと同時に発生するなどにより、パーティクルの発生が目立たないエフェクトで使用します。
4キー

パーティクル寿命の 25% までアルファ値 0.0 から 1.0 に変化します。

75%～100% までアルファ値 1.0 から 0.0 に変化します。
寿命が長いとき、パーティクル発生を徐々に出現させることで目立たなくできます。
