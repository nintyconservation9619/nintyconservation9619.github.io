対象プラットフォーム

FontConverter の対応プラットフォームについて説明します。

対象プラットフォームとは

ビットマップフォントバイナリ (.bffnt) は対象プラットフォームが指定されている場合、そのプラットフォームに最適化されたファイルが出力されます。
対象プラットフォームの指定は、 プリファレンスファイルで行います。

 プリファレンスファイル

Tools\Graphics\FontConverter\Preferences\ 内にプラットフォームごとのフォルダが作成されて、その中に preference.xml が配置されます。

─ Preferences ┬ Generic
               └ PlatformAAA

プラットフォームフォルダが複数ある場合はGenericは無視されて、それ以外のフォルダで最初に見つかったフォルダを使用します。
対象プラットフォームを指定したくない場合は、Generic以外の フォルダの preference.xml を削除またはリネームしてください。
