GFX ライブラリ

概要

GFX ライブラリは、GPU を使ったグラフィックス機能を提供するためのライブラリです。

目的

グラフィックス API として現在さまざまな API が存在しており、これらを使うアプリケーションは対象の API に応じたコードを記述する必要があります。

これは対象のグラフィックス API に対してもっともよい性能を引き出しますが、対象のグラフィックス API を変更する際にはアプリケーションのコードを広範囲に書き換えなければならないことも意味します。

GFX ライブラリは、アプリケーションからこれら低レベルグラフィックス API への依存を切り離すために、これら低レベルグラフィックス API を抽象化したグラフィックス API を提供します。

これは、GFX を使うアプリケーションについて、そのコードを書き換えることなく、使用する低レベルグラフィックス API を変更することを可能にします。

これにより、たとえば将来の低レベルグラフィックス API への移植を容易にすることや、同じコードのまま別の低レベルグラフィックス API を試すことなど、ソフトウェア開発コストを下げることを目的としています。

 

方針

低レベルグラフィックス API 間にはいくつもの相違点があり、特にプラットフォームの世代をまたいだ低レベルグラフィックス API の間には機能面での差が大きくあります。

GFX では基本的にこれらを吸収し抽象化することを目指していますが、抽象化が困難と考えられるところや、抽象化のための犠牲が大きいところ、抽象化により性能が大きく低下する見込みのところについては、無理に抽象化することを避けています。

これにより機能が欠けるところについては、GFX では取り扱わないということになります。

そのような機能が必要な箇所では低レベルグラフィックス API を直接呼び出す形で併用してください。
