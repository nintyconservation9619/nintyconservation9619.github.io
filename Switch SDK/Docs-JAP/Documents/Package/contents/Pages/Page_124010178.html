用語の解説

サウンド

NintendoWare オーディオ開発環境 (AudioToolKit) において、音を鳴らすために必要な要素です。

サウンドシステム

サウンドを再生するためにライブラリが内部で用意している仕組みです。

サウンドプロジェクト

SoundMaker で使用するプロジェクトファイルです。
「サウンドプロジェクトとは」を参照してください。

サウンドセット

サウンドプロジェクトに登録される、小分けされたまとまりです。
「サウンドセットとは」を参照してください。

また、ストリームサウンドやウェーブサウンドなどのサウンドセットを構成する用語については、「サウンドセットの構成要素」を参照してください。
