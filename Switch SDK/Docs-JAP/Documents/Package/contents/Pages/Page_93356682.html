プレビューコントロール

「プレビューコントロール」ではビューアに接続している場合に、再生や停止等の操作ができます。

最初のフレームへ移動 初のフレームに移動し、アニメーションを再生します。
停止 ビューアに表示されているエミッタとパーティクルをすべて削除します。
再生/一時停止

アニメーションの再生中にボタンを押すと一時停止します。

再度ボタンを押すと一時停止したところから再生します。
ステップ再生 アニメーションを１フレームずつ再生します。
fade（フェード） コマンドを実行 fade コマンドを実行し、「終了処理」の設定に従った処理を実行します。
/ ループの有効 / 無効

ループ再生させるかどうかを「 / ループの有効 / 無効」で切り替えます。
「 ループを有効」にすることで、再生開始の度に 毎回エフェクトを呼び出します。

また、再生終了の度に fade コマンドが実行されます。

フレームを指定 「 テキストボックス」でループ再生時のフレーム数を指定します。

ボタンは押した後につづけて Enter キーを押すことでも実行できます。
「 ステップ再生」で利用するとストレスなくコマ送りができます。
