SixAxisSensor(NX Addon)

NintendoSDK NX Addon で追加される機能

概要

NintendoSDK NX Addon で追加される SixAxisSensor の機能は以下の通りです。

  • 操作形態の追加
    • Joy-Con 2本持ち操作(NpadStyleJoyDual), Joy-Con(L) 操作(NpadStyleJoyLeft), Joy-Con(R) 操作(NpadStyleJoyRight), 携帯機操作(NpadStyleHandheld) の操作形態が追加されます。

センサーの軸方向

Joy-Con(R) ・ Joy-Con(L) における軸方向は、操作形態に依らず 図2, 図3 の通りに規定されます。

NintendoSDK NX Addon で使用できる操作形態

操作形態の種類 NpadStyleSet 定義 概要
Joy-Con 2本持ち操作 NpadStyleJoyDual

Joy-Con 2本持ち操作をモデルとした操作形態です。

左右の Joy-Con それぞれの 6軸センサー入力を取得できます。

ハンドル取得対象の操作形態を NpadStyleJoyDual として GetSixAxisSensorHandles() を呼び出すと、

出力 pOutValues[0] に Joy-Con(L) 、pOutValues[1] に Joy-Con(R) の SixAxisSensorHandle が格納されます。
Joy-Con(L) 操作 NpadStyleJoyLeft

Joy-Con(L) 操作をモデルとした操作形態です。

Joy-Con(L) の 6軸センサー入力を取得できます。


ハンドル取得対象の操作形態を NpadStyleJoyLeft として GetSixAxisSensorHandles() を呼び出すと、


出力 pOutValues[0] に Joy-Con(L) の SixAxisSensorHandle が格納されます。
Joy-Con(R) 操作 NpadStyleJoyRight 

Joy-Con(R) 操作をモデルとした操作形態です。

Joy-Con(R) の 6軸センサー入力を取得できます。

ハンドル取得対象の操作形態を NpadStyleJoyRight として GetSixAxisSensorHandles() を呼び出すと、

出力 pOutValues[0] に Joy-Con(R) の SixAxisSensorHandle が格納されます。

Joy-Conが本体に装着されていない状態で SixAxisSensor を利用する

本体にJoy-Conが装着されていない場合に SixAxisSensor を利用したい場合は、

Handheld 操作スタイルを有効化するモードを NpadHandheldActivationMode_None として SetNpadHandheldActivationMode() を呼び出すことで、

NpadStyleHandheld の SixAxisSensor の機能を利用できます。
