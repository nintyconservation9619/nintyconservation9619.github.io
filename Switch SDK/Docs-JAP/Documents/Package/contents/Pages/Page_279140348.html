機能アニメーションの確認

ここでは「機能アニメーション」の確認手順について、「部品レイアウト」のセレクトボタンを例に説明します。
「機能アニメーション」により LayoutViewer で実際のメニュー操作ができる為、アニメーションを確認ながら修正することができます。

作業の準備

あらかじめ LayoutEditor を起動した状態から、レイアウトデータ作成にあたっての準備を行います。

関連リンク LayoutEditor の起動方法については 「作業の流れ」 を参照してください。

ウインドウを開く

以下のウインドウが開いた状態から作業を開始します。

  • 部品パレット
  • 部品・コントロール設定
  • 親子階層
  • プロパティウインドウ
  • カーブエディタ

関連リンク 各ウインドウの UI については 「ウインドウ」 を参照してください。

ボディレイアウトを開く

「部品パレット」から「ボディレイアウト」を開きます。

「部品パレット」リストの「Menu.flyt」でマウス右ボタンクリックで開くコンテキストメニューから「開く」を選択します。
LayoutEditor に「Menu.flyt」が読み込まれます。

「部品レイアウト」を開く

「ボディレイアウト」から「部品レイアウト」を開きます。
レイアウトウインドウの下部ツールバーにある「 関連部品を一覧表示」ボタンをマウス左ボタンクリックします。
表示された部品レイアウト「 Menu_Bar」を選択しファイルを開きます。

機能アニメーションの利用

機能アニメーションを利用し、LayoutViewer で実際のメニュー操作しながら、アニメーションの結果を確認します。
「機能アニメーション」を実行するには、メインメニュー項目から [ ツール ] → [ ビューアに転送 ] を選択するか、ショートカット [ Shift ＋ F5 ] を押します。

アニメーションの追加

区間タグ「Down」をブラッシュアップする為にアニメーションを追加します。
「アニメーション区間タグ編集」ウインドウでアニメーション区間タグ「Down」を選択します。

「アニメーション区間タグ編集」ウインドウのツールバーで「 結合モード」であることを確認します。
「カーブエディタ」ウインドウで以下のようにキーを設定します。
