作業の流れ

作業の流れ

ここでは、3DEditor を起動してから作業を完了するまでの一連の流れについて説明します。

説明では、3dSampleViewer を使用しています。 
3dSampleViewer は本ドキュメント内のチュートリアルで使用するビューアです。
実際のゲーム作成では使用できませんのでご注意ください。 

作業開始

3DEditor と 3dSampleViewer を起動してプレビューできる環境を準備します。

3DEditor の起動

NintendoSDK をインストールした下記場所にある 3dEditor.exe をダブルクリックします。

3DEditor

 %NINTENDO_SDK_ROOT%\Tools\Graphics\3dEditor\3dEditor.exe

ビューアの起動

下記場所にある 3dSampleViewer.exe をクリックして SampleViewer を起動します。

3dSampleViewer

 %NINTENDO_SDK_ROOT%\Samples\BuiltApplications\3dSampleViewer\3dSampleViewer.exe
参考：
関連リンク 3DEditor / 3dSampleViewer が起動できない場合は、再度 NintendoSDK オーバービュー／動作環境 で NintendoSDK の動作環境を確認してください。

モデルをプレビューする

プレビュー環境の準備が完了していることを前提に、実際のサンプルデータをビューアに表示する手順を説明します。

（１）プレビューするデータを 3DEditor に読み込みます。

ここでは、例としてパッケージ内の下記場所に用意されているサンプルデータ MaleA を読み込ませます。

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\town\MaleA\MaleA.fmdb

3DEditor のメニュー [ ファイル ]→[ 開く ] でファイルを指定するか、3DEditor 上にドラッグ & ドロップしてファイルを読み込みます。

 

（２）プラットフォームを設定します。

3DEditor のツールバーにあるプラットフォーム切り替え機能で、接続先アプリケーションのプラットフォームを選択します。


（３）ビューアと接続します。

3DEditor のメニュー [ プレビュー ] → [ アプリケーションへ接続 ] か、ツールバーにある [  接続アイコン ] をクリックします。

参考：

ビューアに接続できない場合は以下についてご確認ください。

  • 「 Nintendo Target Manager 」の起動
  • プラットフォームに対応したビューアの起動

（４）ビューアに読み込んだデータが表示されるのを確認します。

MaleA を読み込んでビューアに接続すると下の画像のように MaleAモデルが表示されます。

MaleA が 3dSampleViewer に表示されましたら、3DEditor と 3dSampleViewer は正常に接続されています。

3DEditor で編集

3DEditor で編集できる項目は以下になります。

  • モデルのパラメータ（ビルボード、ユーザーデータなど）
  • マテリアルの固定機能（テクスチャサンプラ）
  • マテリアルへのシェーダー割り当てとパラメータ編集
  • テクスチャの編集（ユーザーデータなど）
  • マテリアルアニメーションの作成と編集
  • ボーンビジビリティアニメーションの作成と編集
  • スケルタルアニメーションの編集(ユーザーデータなど)
  • シーンアニメーションの作成と編集

作業の完了

3DEditor での編集作業を完了させます。

データの保存

3DEditor で編集したパラメータがある場合は、下記のように赤★マークが付きます。

関連リンク メインメニューのファイルにある各保存方法を使用するか、変更したファイルの右クリックメニューにある保存を選択してデータを保存します。

3DEditor を終了する

以下いずれかの操作を行うと、開いているすべてのファイルを閉じて 3DEditor が終了します。

  • メインメニュー項目 [ファイル] → [終了] を選択します。
  • メインウィンドウの「閉じる」ボタンを押します。
  • ショートカットキー [ Ctrl ] + [ Q ] を使用します。
参考：

未保存のファイルがある場合は「変更したファイルの保存」ダイアログが表示されますので、必要がある場合は保存して終了します。
