拡大縮小時にテクスチャが滲まないようにするには

ピクチャペインなどを拡大縮小すると、テクスチャが滲むことがあります。

その対処方法の一つとして、テクスチャをミップマップありで作成しておき、プログラム側でピクチャペインのテクスチャに差し込む方法があります。

このとき描画先領域に対する UV の密度に応じて LOD レベルが決定し、描画する領域よりも大きいミップマップが選ばれて縮小されるケースと、描画する領域よりも小さいミップマップが選ばれて拡大されるケースが発生しますが、常に大きいミップマップが選ばれるように LOD レベルにバイアスをかけるとより綺麗な描画結果になります。

そのような処理にするには GraphicsResource の Setup メソッドの引数の lodBias に負の値(例えば -0.5f)を設定します。

ただしこの場合、大きなミップマップを使うようにすることで処理負荷は増加しますのでご注意ください。
