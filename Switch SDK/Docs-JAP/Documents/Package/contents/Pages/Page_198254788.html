フェード開始α値オフセット

ここでは「ソフトパーティクル」の処理を頂点単位で行うさいに、パーティクルがモデルと重なった箇所の透明度を設定する機能「フェード開始α値オフセット」について説明します。

概要

「フェード開始α値オフセット」は、パーティクルがモデルと重なった箇所からどの程度の範囲を透明にするかを設定できる機能です。
値はスクリーン座標の値を -1 〜 1 とした座標上で指定する為、カメラからの距離によって透明度合いが変化します。
以下は「 設定項目」の概要です。

フェード開始α値オフセット

パーティクルがモデルと重なった箇所からの距離を「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は視点から「 █ パーティクル面の頂点」までの距離と「█ 床面の深度値」との「 … 距離の差」によるアルファ値の変化について説明した画像と図です。
「フェード開始α値オフセット」は 0、「サンプリング半径」は 0 に設定しています。

A パーティクルが床面に遮蔽された箇所のアルファ値は 0 になります。

B フェード開始位置からのオフセットがない為、頂点のアルファ値は 1 になります。

以下は「フェード開始α値オフセット」の値を変更したときの比較画像です。
「 … 距離の差」を短くすると、アルファ値を減衰できます。

A 0.02 値が小さい場合、見下ろすような視点で「 … 距離の差」を短くすると、透明になる度合いがゆるやかになります。
B 0.09 値が大きい場合、見下ろすような視点で「 … 距離の差」を短くすると、透明になる度合いが極端に変化します。
