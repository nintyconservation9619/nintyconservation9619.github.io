ノイズのオフセット（±）

ここでは、ノイズテーブルの位置をずらす機能「ノイズのオフセット（±）」について説明します。

概要

「ノイズのオフセット（±）」は、カールノイズに使用するノイズテーブルの位置を調整できる機能です。
ノイズテーブルのどの位置に発生するかでパーティクルの動きが決定されてしまいます。
オフセットの値を変更することでテーブルの位置をずらし、パーティクルの動きを調整することができます。
以下は「 設定項目」 の概要です。

ノイズのオフセット（±）

ノイズテーブルのオフセット値を「 ダイヤル」や「 テキストボックス」により設定します。

設定例

以下は「ノイズのオフセット（±）」の値を変更した場合の比較画像です。

● 赤 1 デフォルト設定です。
● 緑 -1 デフォルト設定と比較すると、異なる方向へ動きます。
● 白 0 ノイズテーブルをオフセットしない場合のパーティクルの動きです。
● 水色 0.01 オフセットしない場合と比較すると、動く方向が少しずつずれていきます。
