親子設定または親子解除後にペイン間のドラッグ 親子階層の設定または解除をします。
[ Space ] + [ Alt ] + マウス右ボタン以外クリック ズームアウトします。
[ Space ] + [ Ctrl ] + マウス右ボタン以外クリック ズームインします。

ウインドウサイズの変更をします。

[ Shift ] とあわせると、変更量が増大します。

[ Shift ] とあわせると、変更量が増大します。

操作 説明
[ Ctrl ] + [ + ] ズームインします。
[ Ctrl ] + [ - ] ズームアウトします。
[ S ] 選択中のカーブのキー生成をします。
[ A ] 選択カーブの全体表示をします。
[ F ] 選択キーの全体表示をします。

[ Back ]

[ Delete ]

 キーの削除をします。
左ドラッグ 選択します。
左ボタン以外のクリックまたはドラッグ カーブ操作 ( 移動や追加など ) します。
複数選択した状態で [ T ] を押し、操作の枠を右または中ドラッグ キー位置のスケールをかけます。
カーブ操作が通常または最も近いキーを操作 + 方向ボタン 選択されたキーの移動をします。
カーブ操作が通常または最も近いキーを操作 + [ Shift ] + 方向ボタン 選択されたキーの移動を 10 倍します。
カーブ操作がキーの挿入または追加 + 方向ボタン キーの挿入または追加の開始をします。

[ Space ] + [ Alt ] + マウス左ボタン ズームアウトします。 ( [ Alt ] + [ Space ] の順だと効きません）
[ Space ] + [ Shift ] + マウス右水平または垂直ドラッグ 水平または垂直ビュースケール変更をします。 ( [ Shift ] + [ Space ] の順だと効きません）
[ Space ] + マウス左または中ボタン 平行移動します。
[ Space ] + マウス右ボタン水平ドラッグ ビュースケール変更をします。
ホイール ズームイン、ズームアウトをします。

 グループウインドウ

グループウインドウに対応するショートカットキーの一覧です。

操作 説明
[ F2 ] グループのラベルを編集します。
ドラッグ & ドロップ グループウインドウ内の並びを移動します。

親子階層ウインドウ

親子階層ウインドウに対応するショートカットキーの一覧です。

操作 説明
[ F2 ] ラベル編集します。
ドラッグ & ドロップ 移動します。
[ A ] レイアウトウィンドウを全体表示します。
[ F ] レイアウトウィンドウの選択部分を全体表示します。
[ P ] 選択したペインを最後に選択したペインの階層下に配置します。
[ Shift ] + [ P ] 選択したペインの親子関係を解消します。
[ G ] ヌルペインを作成し、選択したペインを階層下に配置します。

[ Alt ] + 「 表示 / 非表示アイコン」クリック

選択ペイン階層下のペインのみを表示し、その他すべてのペインを非表示にします。
再度同操作で直前の状態に戻ります。
