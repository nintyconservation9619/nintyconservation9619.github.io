使用方法

このプラグインは、白黒 2 階調の画像をディスタンスフィールド画像（白黒境界からの最短距離をグレースケールで表した画像）に変換します。
ディスタンスフィールド画像を使用すると、エッジの鮮明さを保ったままテクスチャーを拡大できます。
テクスチャーの品質向上のため、高解像度の画像にこのプラグインを適用した後、縮小して使用することを推奨します。

処理を行う RGB チャンネルまたはグレーチャンネルまたはアルファチャンネルを選択して、[フィルター] → [Nintendo] → [Nintendo Distance Field...] を実行します。

  • Max Distance
    白黒境界からの最大距離（ピクセル）を指定します。
    白黒画像の黒領域は白黒境界からの距離に応じて 0 ～ 128（最大距離以上の場合に 0）に変換されます。
    白黒画像の白領域は白黒境界からの距離に応じて 128 ～ 255（最大距離以上の場合に 255）に変換されます。
    初期状態ではスライダの最大値は 64 ですが、数値を直接入力することによって最大 250 まで指定できます。
  • Edge Wrap
    画像が繰り返しているとして端のピクセルを処理するなら ON にします。
    画像の端のピクセルを引き伸ばして処理するなら OFF にします。
  • Normalize
    最小値が 0、最大値が 255 になるようにディスタンスフィールドの値を正規化するなら ON にします。

OK ボタンをクリックするとフィルターが実行されます。

+ ボタン、- ボタンおよびマウスのホイールでプレビュー画像の拡大率を変えることができます。

画像が 2 階調化されていない場合、128 以上を白、128 未満を黒として処理します。
RGB チャンネルを選択して実行した場合、R 成分を入力として処理し、RGB 成分に同じ値を出力します。

Edge Wrap が OFF の場合、近似アルゴリズムでプレビューを計算するので、実際のフィルター結果とプレビューは多少異なる場合があります。

注意事項

ディスタンスフィールドフィルタープラグインを使用する際の注意点です。

イメージモード

イメージモードがグレースケール、RGB カラー以外のファイルや、8 bit/チャンネルでないファイルでは実行できません。
