カラーピッカー

カラーピッカーは、カラーアニメーションの編集時や、カラータイプのシェーダーパラメーターなどを変更する際に使用します。
プロパティウィンドウや、カーブエディタ内で使用されているカラー表示領域をクリックすることで開きます。
A

現在のカラーが表示されます。
B 現在のカラーをカラーパレットに登録します。
※アルファ成分は登録されません。 
C カラーパレットです。
パレットを選択して左クリックすると、選択した色を現在のカラーとして設定します。
D カラーの色調（Hue）を変更します。
E カラーの彩度（Saturation）と明度（Value）を変更します。
F

カラー成分の表示形式を、sRGB 形式 / リニア RGB 形式 / HSV 形式から選択します。
表示形式が異なるだけで、設定される値に影響はありません。
G

値の表示形式を、整数値形式（0 ～ 255）と 実数値形式（0.0 ～ 1.0）で切り替えます。
カラー成分の表示形式が RGB 形式の場合のみ有効です。表示形式が異なるだけで、設定される値に影響はありません。
H カラーの値を数値入力や、スライダーで設定します。
I カラーのスケール値を数値入力や、スライダーで設定します。
J

カラーのスケール範囲を設定します。ボタン操作で 0 からの最大範囲を設定できます。

デフォルトの範囲は 0 ~ 2 に設定されています。
参考：

カラーパレットに 1.0 を超える値の色は保存できません。1.0 以上の値は 1.0 として保存します。

カラーパレットにカラーのスケール値は保存できません。

HDR カラーの設定

カラーピッカーではカラーのスケールの値を直接入力、もしくはスライダーによる操作により HDR カラーを設定することができます。

カラーの値(カラーピッカー上のカラー値とカラースケールの積)に 1 より大きい値が設定されていると、設定されている HDR カラーの最大値をマテリアルプロパティウィンドウから確認できます。

参考：

マテリアルプロパティウィンドウの RGB の値はリニア RGB 形式で表示されています。sRGB 形式とは値が異なります。
