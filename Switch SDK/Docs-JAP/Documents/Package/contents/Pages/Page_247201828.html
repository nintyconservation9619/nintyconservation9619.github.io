nn::vfx ニュース

 

テクスチャ原点・正規化デバイス座標系切り替え について

以下のグラフィックスAPIに関わる設定をプロジェクト設定ファイルから切り替えるようになりました。

  • フレームバッファやデプスバッファのテクスチャ原点位置切り替え
  • 正規化デバイス座標系切り替え

これらの設定に合わせて起動するエフェクトビューアの挙動も切り替わります。

 nn::font ニュース

マルチスケーラブルフォントの scaleWidth がカーニングに反映されていなかった不具合の修正

マルチスケーラブルフォントでフォントごとに文字幅にスケールを掛ける scaleWidth パラメータが、カーニングに反映されていなかった不具合を修正しました。

scaleWidth パラメータを使用していた場合、これまでと描画結果に差異が出る可能性がありますのでご注意ください。
