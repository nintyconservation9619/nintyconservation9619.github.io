FONTLL2 ライブラリ

はじめに

本ドキュメントの目的

本ドキュメントでは、NintendoSDK に収録されている FONTLL2 ライブラリについて説明します。

対象読者

NintendoSDK を使ってアプリケーションを開発される方を対象としています。

関連資料

nn::fontll2 ライブラリの関数リファレンスと併せて参照してください。

FONTLL2 ライブラリが提供する機能

FONTLL2 ライブラリでは、FONTLL ライブラリの内部エンジンをバージョンアップさせたものです。

FONTLL ライブラリと同様、スケーラブルフォントを扱う低レイヤー API を提供します。

FONTLL ライブラリを使ったときに一部のフォントの描画がおかしくなるなどの問題が発生した場合、FONTLL2 では修正されていることがありますのでお試しください。

FONTLL ライブラリは実行時に動的リンクされますが、FONTLL2 はプログラムに静的リンクされるため、メモリ消費量がライブラリの分だけ増加します。

また、一部のフォントで FONTLL と FONTLL2 の描画結果に差異が出ることがあります。

FONTLL2 ライブラリのインターフェースはほぼ FONTLL と同じため、基本的な使い方については FONTLL ライブラリのマニュアルを参照してください。
