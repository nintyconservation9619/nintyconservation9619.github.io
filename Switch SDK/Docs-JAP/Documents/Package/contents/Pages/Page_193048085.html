マテリアルプロパティウィンドウ 親マテリアルページ
注意：

この機能は今後、予告なく仕様や動作が変更される場合がありますので、利用する際はご注意下さい。

マテリアル参照機能の親マテリアルの設定や確認をするページです。

A

複数の親マテリアルがある場合に選択したマテリアルを上下に移動します。親マテリアルの順序は複数の親マテリアルに重複するパラメーター設定が存在する場合に、採用する設定の優先度に影響します。詳しくは マテリアル参照 の「マテリアルの多重参照」をご覧ください。
B 親マテリアルを新規に追加します。
追加ボタンを押すと親マテリアル編集ウィンドウが表示されます。
C 選択した親マテリアルを削除します。
D 選択した親マテリアルを編集します。
編集ボタンを押すと親マテリアルの編集ウィンドウが表示されます。
E

設定されている親マテリアルのマテリアル名が表示されます。

注意：

現在は親マテリアル選択時のマテリアル名のみが設定したラベル名に置きかわり、「親マテリアルのラベル名（子のマテリアル名）」が表示されます。

F 設定されている親マテリアルを持つモデルファイルパスが表示されます。
G マテリアル参照情報が設定されているシェーダーパラメーターの種類が表示されます。
H

マテリアル参照情報が設定されているシェーダーパラメーターの ID が表示されます。
I

シェーダーパラメーターの値に何を指定しているかが表示されます。親マテリアルの値を参照するか、自身のマテリアルの値で親マテリアルの値を上書きするか、デフォルト値を使用するかが表示されます。※1
K

子マテリアルの制限が表示されます。親マテリアルの値で固定する、必ず子マテリアルで値を設定するなどが表示されます。※1

※1 詳細は マテリアル参照に関する設定 をご覧ください。

親マテリアル編集ウィンドウ

親マテリアルのモデルや親マテリアルの追加と編集ができます。

A

チーム設定に記述したパス以下にあるモデル中間ファイルのマテリアルを D に一覧表示します。

親マテリアルの追加ウィンドウ、変更ウィンドウでチーム設定で指定したカテゴリーから親マテリアルを選択できます。
B

「指定したフォルダ」で指定したパス以下にあるモデル中間ファイルのマテリアルを D に一覧表示します。
C 3DEditor に読み込んでいるモデル中間ファイルのマテリアルを D に一覧表示します。
D リストから親にするマテリアルを選択します。
E OK ボタンを押すとマテリアル参照が追加されます。
F マテリアル参照の編集をキャンセルします。
注意：

同じシェーダーが割り当てられているモデル中間ファイルのマテリアルのみ一覧に表示されます。
