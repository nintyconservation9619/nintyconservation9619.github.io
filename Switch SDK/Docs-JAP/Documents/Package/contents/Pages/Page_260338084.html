NintendoSDK 4.3.0

本バージョンの既知の不具合は以下の通りです。

  • 全般
    • OS のシステム ロケールをデフォルト値以外に変更した場合の動作は保証外
    • API リファレンスの関数などのアルファベット一覧が正しく表示されない
  • friends
    • 機内モードの切り替えなどでネットワーク接続を瞬断させた直後にスリープすると、自分のプレゼンスが数時間オフラインにならないことがある
  • nifm
    • スリープやシャットダウン時に FATAL が発生する
  • libcurl
    • 1 台の PC で複数のアプリケーションからの HTTPS が実行できない
  • Socket
    • nn::socket::Recv() で MSG_DONTWAIT フラグを使用できない
    • nn::socket::Select() もしくは、nn::socket::Poll() が EBADF を返すことがある
  • PERF
    • 特定の PC の環境において、サンプル PerfSimple が停止する
  • SSL
    • nn::ssl::Initialize() と nn::ssl::Finalize() は、メインスレッドで呼んでください
  • オーディオツール
    • nn::atk::viewer ライブラリを使ってインゲーム編集のストリームサウンドを再生中に稀にプログラムが停止する
    • SoundMaker の平均ラウドネス値が正しくない場合がある
    • SoundMaker の最大平均ラウドネス値を設定しても警告アイコンが表示されない
  • 3D-G3D
    • G3dDemo サンプルの Town シーンの実行中に不正アクセスが発生することがある
  • エフェクト-VFX
    • エフェクトデータのチャイルドエミッタの継承設定「パーティクルにエミッタ割当」が OFF の状態で「履歴式ストライプ」「履歴式ストライプ2」のいずれかを設定すると VFX ランタイムが停止する
  • Mii
    • nn::mii::Database::UpdateLatest() が nn::mii::ResultBroken を返さない
  • サムネールエクスプローラー拡張
    • 拡張子 .eset, .flyt, .fmda, .fmdb にアプリケーションが関連付けられているとサムネールが表示されない

全般

OS のシステム ロケールをデフォルト値以外に変更した場合の動作は保証外


OS のシステム ロケールをデフォルト値以外に変更した場合の動作は保証外です。(例えば日本語版 Windows 上でシステム ロケールとして“日本語(日本)”以外を選択した場合)

API リファレンスの関数などのアルファベット一覧が正しく表示されない

API リファレンスで、「名前空間」→「名前空間メンバ」 の下の、「全て」 と 「関数」 と 「列挙値」で表示されるアルファベット一覧を選択しても正しいリストが表示されません。

Socket

nn::socket::Recv() で MSG_DONTWAIT フラグを使用できない

症状

nn::socket::Recv() で MSG_DONTWAIT フラグをセットしても、ノンブロックモードになりません。

回避策

nn::socket::Fcntl() で O_NONBLOCK をセットすることで同等の動作にすることができます。

修正予定

次回以降のリリースで修正予定です。

オーディオツール

nn::atk::viewer ライブラリを使ってインゲーム編集のストリームサウンドを再生中に稀にプログラムが停止する

インゲーム編集対象のストリームサウンドの再生を繰り返したり、インゲーム編集でパラメータを高速に変更し続けたりしていると稀に以下のアボートが発生してプログラムが停止することがあります。 

Abort: 'delimiter != nullptr' in nn::fs::detail::GetMountName()

本件は次回以降のリリースで改善予定です。

SoundMaker の平均ラウドネス値が正しくない場合がある

症状

以下の操作をすると平均ラウドネス値が正しくない場合があります。

  • アプリケーションと接続していない状態で、インゲーム編集パネルの再生ボタンのクリック。
  • メニュー「プレビュー」→「インストルメント波形の参照回数を計測」の実行。
  • メニュー「プレビュー」→「シーケンスサウンドの最大発音数を計測」の実行。
回避策
  • アプリケーションと接続していない状態では、インゲーム編集パネルの再生ボタンをクリックしないようにしてください。
  • メニュー「オプション」→「統計」タブで、平均ラウドネス値の自動計測をオフにしてから「インストルメント波形の参照回数を計測」「シーケンスサウンドの最大発音数を計測」を実行するようにしてください。
修正予定

次回以降のリリースで修正予定です。

3D-G3D

G3dDemo サンプルの Town シーンの実行中に不正アクセスが発生することがある

環境によって Generic 版の G3dDemo サンプルの Town シーンの実行中に不正アクセスが発生することがあります。

この不具合は nn::perf の使用をやめたり、描画するモデル数を変更すると起きなくなることが確認されていますが、詳しい原因はわかっていません。

エフェクト-VFX

エフェクトデータのチャイルドエミッタの継承設定「パーティクルにエミッタ割当」が OFF の状態で「履歴式ストライプ」「履歴式ストライプ2」のいずれかを設定すると VFX ランタイムが停止する

症状

エフェクトデータのチャイルドエミッタの継承設定「パーティクルにエミッタ割当」が OFF の状態で「履歴式ストライプ」「履歴式ストライプ2」のいずれかを設定すると、VFX ランタイム内で不正アクセスが発生する不具合があります。

回避策

エフェクトデータの設定の「パーティクルにエミッタ割当」を ON に設定することで回避できます

修正予定

本件は次回以降のリリースで修正予定です。
