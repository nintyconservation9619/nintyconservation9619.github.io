NintendoWare 3D の概要について説明します。

NintendoWare 3D の構成要素

中間ファイル

NintendoWare 3D におけるモデルやシェーダー、アニメーションなどのアセットです。

関連リンク 3D 中間ファイル仕様書
DCC プラグイン

DCC ツールから中間ファイルを書き出すためのプラグインです。

関連リンク Maya プラグイン、3ds Max プラグイン、Photoshop プラグイン
3DEditor

中間ファイルを作成、編集するための GUI ツールです。

関連リンク 3DEditor
3D コマンドラインツール

中間ファイルを編集、変換するためのコマンドラインツールです。

関連リンク 3D コマンドラインツール

3D ランタイムライブラリー

NintendoWare 3D の機能をゲームアプリケーションに組み込むためのランタイムライブラリーです。

関連リンク 3D ランタイムライブラリーオーバービュー
バイナリーリソース

3D ランタイムライブラリーが参照するモデルやシェーダー、アニメーションのバイナリーファイルです。

中間ファイルから 3D バイナリーコンバーター、3D シェーダーコンバーターによって変換します。

関連リンク 3D バイナリーコンバーター、3D シェーダーコンバーター
ビューアーライブラリー

ビューアーを作成するためのランタイムライブラリーです。

ビューアーライブラリーをゲームアプリケーションに組み込むことで、3DEditor とゲームアプリケーションを連動させ、ゲームアプリケーション上のモデルを編集したり、3DEditor 上に読み込まれたモデルやアニメーションのプレビュー、編集を行うことができます。

関連リンク ビューアーライブラリー
Mii ヘルパーライブラリー

nn::mii::CharModel を NintendoWare 3D のバイナリーリソースにアサインする機能を提供します。

関連リンク Mii ヘルパーライブラリー
