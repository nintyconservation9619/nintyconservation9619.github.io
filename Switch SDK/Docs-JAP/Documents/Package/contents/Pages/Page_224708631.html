パーティクルの重なりチェック

ここでは パーティクルの重なり度合が確認できる機能「パーティクルの重なりチェック」について説明します。

概要

「パーティクルの重なりチェック」は、Effectviewer でパーティクル同士の重なり度合いを確認するデバッグ機能です。

パーティクルの重なりチェック

「パーティクルの重なりチェック」を「 チェックボックス」ON/OFF で切り替えます。
