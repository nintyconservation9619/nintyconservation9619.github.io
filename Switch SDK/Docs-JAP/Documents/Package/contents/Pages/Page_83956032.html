カーブツリービュー

カーブツリービューは、アニメーションに含まれたノードをツリー形式で表示します。
「カーブを編集するノードの選択」や「カーブの表示状態の変更」を操作します。

  • カーブツリービューの基本操作
  • カーブツリービューのアイコン
  • カーブツリービューの右クリックメニュー

A カーブツリービューの表示状態を変更するアイコン群です。
B カーブツリービュー上の文字列を検索してフィルタリングします。
C 選択可能なノードがツリー表示されます。
カーブが作成されているノードを選択すると、カーブビューにカーブが表示されます。
各ノードにあるチェックボックスでカーブの表示 / 非表示を切り替えることができます。

カーブツリービューの操作

カーブツリー上での基本操作を説明します。

カーブツリービュー上の検索

カーブツリーに表示されているノードを文字列で検索してフィルタリングできます。

ノードの選択

カーブツリービューで下図の矢印で示されている四角形をクリックすると、ノードが選択状態になります。
選択状態では、四角形が青色になります。 

複数のノードを選択して同時に編集できます。

上図は、diffuse の R G B に同時にキーを追加したり、編集できる状態です。

参考：

ノードを選択すると、カーブ要素「プレラップ」「ポストラップ」が変更できます。
複数のノードを選択している場合は、選択しているノードのカーブ要素が全て同時に変更されます。

ノードを選択してアニメーションカーブをフォーカスする

ノードをクリックして選択状態にした後に、選択したノードをダブルクリックすると、設定されているアニメーションカーブ全体がカーブビューに収まるようにフォーカスされます。

ノードの並び順をアルファベット順に変更

スケルタルアニメーション、ボーンビジビリティアニメーションのカーブエディタには、ノードをアルファベット順にソートする機能があります。
アイコンでソートの ON /OFF をします。

デフォルトの表示は、インデックス順になっています。

カーブツリービューのアイコン

カーブツリービューで使用されているアイコンを説明します。

表示状態を変更するアイコン

カーブツリービューの表示にフィルターをかけます。

アイコン 説明
フィルタボタンです。
カーブを持つ項目のみ表示させることができます。
3DEditor に読み込まれているデータで、カーブが設定されている項目のみ表示します。
フィルタボタンです。
有効なグループの項目のみ表示します。
※有効なグループとは、fsd の <group> condition のことで、 condition が真のものが有効なグループになります。
<group> の詳細については、グラフィックスツール統合ヘルプ内に別途用意されている中間ファイルフォーマットを参照してください。
フィルタボタンです。
選択しているマテリアルのカーブのみ表示します。
3DEditor のオブジェクトビューをスケマティック表示やマテリアル一覧表示にして、選択した任意のマテリアルに設定してあるカーブのみを表示するようになります。
※このフィルタを ON にすると自動的にカーブエディタのピン機能が ON になります。
