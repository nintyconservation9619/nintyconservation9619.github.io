回転速度減衰率

ここでは放出されたパーティクルの回転速度を減衰または加速させる機能「回転速度減衰率」について説明します。

概要

「回転速度減衰率」は、パーティクルの回転速度に対して毎フレーム指定した値を乗算することで、回転速度を減衰または加速できる機能です。
例えばコマのように、空気抵抗を受けて回転速度が弱まりやがて停止する動きを表現できます。

以下は「 設定項目」の概要です。

回転速度減衰率

パーティクルの回転速度に対して毎フレーム指定した値を乗算することで、回転速度を減衰または加速させます。
