ふち付きフォントで文字が重なる

質問

ふち付きフォントで文字と文字が重なってしまいます。

回答

特にふちが太い場合には、ある程度仕方がない問題です。
文字間隔を大きく設定して回避してください。
また、LA4Packed形式 では文字自体とフチを2回に分けて描画するようにします。
一回目でフチだけを描き、2回目で文字自体だけを描くと、見え方や印象が変わりますので対策として検討してください。
レイアウトモジュールに該当機能があります。
