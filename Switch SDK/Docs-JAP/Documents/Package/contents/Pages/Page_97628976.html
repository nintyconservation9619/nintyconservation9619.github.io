放出時のサイズ

ここではエミッタセットから放出されるすべてのパーティクルに対して設定した値を乗算する機能「放出時のサイズ」について説明します。

概要

「放出時のサイズ」は、エミッタ放出時の「パーティクルのスケール」に乗算する値を設定できる機能です。
値を変更すると、エミッタがパーティクルを放出する時のパーティクルサイズを変更できます。
エミッタから既に放出されているパーティクルのサイズには影響しません。

放出時のサイズ

エミッタから放出するパーティクルの「スケール」に乗算する値を「 ダイヤル」や「 テキストボックス」により設定します。

A X,Y,Z=1 デフォルト設定です。放出時のパーティクルサイズはエミッタタブのスケール値と同じになります。
