ビルドシステム

はじめに

ビルドシステムドキュメントは、NintendoSDK を使用してアプリケーションやライブラリ等をビルドするために必要な情報を提供します。

NintendoSDK を使用した開発に携わるすべての開発者を対象読者としています。

本ドキュメントの構成

ページ 説明
オーバービュー NintendoSDK が提供するビルドシステムの概要を示します。初めて NintendoSDK のビルドシステムを扱う開発者は最初にお読みください。
環境構築ガイド NintendoSDK を使用したアプリケーションやライブラリ等をビルドするために必要な情報を提供します。特に、既存のビルドシステム環境を再利用したい開発者や独自のビルドシステムを構築したい開発者を対象としています。
チュートリアル NintendoSDK を使用したビルドシステムの構築方法を具体的な手順を交えて説明します。
マニュアル NintendoSDK が提供するビルドシステムの使用方法を示します。
リファレンス NintendoSDK が提供するビルドシステムの仕様を列挙します。
