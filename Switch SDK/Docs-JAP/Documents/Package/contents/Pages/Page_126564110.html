複合フォント定義ファイルフォーマット

複合フォント定義ファイルはXML形式のファイルです。

ここでは、複合フォント定義ファイルに含まれる要素を説明します。

関連リンク 複合フォントの概要については「複合フォント」をご確認ください。 

<complexFontDescription>

ファイルのルート要素です。

属性

名前 型 説明
Version string 複合フォント定義ファイルのバージョンです。1.0.0 を指定します。

内容

以下の要素を含みます。

  • <subFont> : 複合フォントの設定です。必ず一つ出現します。

<subFont>

複合フォントの設定です。

内容

以下の要素のいずれか一つを含みます。

  • <pairFont> : ペアフォントの設定です。
  • <bitmapFont> : ビットマップフォントの設定です。
  • <scalableFont> : スケーラブルフォントの設定です。
  • <multiScalableFont> : マルチスケーラブルフォントの設定です。

<pairFont>

ペアフォントの設定です。

内容

以下の要素を含みます。

  • <firstFont> : ペアフォントの一つ目の要素の設定です。必ず一つ出現します。
  • <secondFont> : ペアフォントの二つ目の要素の設定です。必ず一つ出現します。

<firstFont>

ペアフォントの一番目の要素の設定です。

内容

以下の要素のいずれか一つを含みます。

  • <pairFont> : ペアフォントの設定です。
  • <bitmapFont> : ビットマップフォントの設定です。
  • <scalableFont> : スケーラブルフォントの設定です。
  • <multiScalableFont> : マルチスケーラブルフォントの設定です。

<secondFont>

ペアフォントの二番目の要素の設定です。

内容

以下の要素のいずれか一つを含みます。

  • <pairFont> : ペアフォントの設定です。
  • <bitmapFont> : ビットマップフォントの設定です。
  • <scalableFont> : スケーラブルフォントの設定です。
  • <multiScalableFont> : マルチスケーラブルフォントの設定です。

<bitmapFont>

ビットマップフォントの設定です。

内容

以下の要素を一つずつ含みます。

  • <path> : フォントファイルのパスを指定します。
  • <charCodeRangeSet> :  このフォントを表示する文字の範囲を指定します。

<scalableFont>

スケーラブルフォントの設定です。

属性

名前 型 説明
size float フォントのサイズです。省略できません。
alternateChar unsignedInt 代替文字を 10 進数で指定します。省略した場合は 63 ('?' の文字)になります。
lineFeedOffset int 行間のオフセット値を指定します。省略した場合は 0 になります。

内容

以下の要素を含みます。

  • <scalableFontDescription> : スケーラブルフォントの設定です。必ず一つ出現します。

<multiScalableFont>

マルチスケーラブルフォントの設定です。

属性

名前 型 説明
size float フォントのサイズです。省略できません。
alternateChar unsignedInt 代替文字を 10 進数で指定します。省略した場合は 63 ('?' の文字)になります。
lineFeedOffset int 行間のオフセット値を指定します。省略した場合は 0 になります。

内容

以下の要素を含みます。

  • <scalableFontDescriptionSet> : 複合フォントの設定です。必ず一つ出現します。

<scalableFontDescriptionSet>

スケーラブルフォントの配列要素です。

内容

以下の要素を含みます。

  • <scalableFontDescription> : スケーラブルフォントの設定です。一つ以上出現します。

<scalableFontDescription>

スケーラブルフォントの設定です。

属性

名前 型 指定可能な範囲 説明
boldWeight float 0.0 ～ 1.0 ボールドウェイトを指定します(0.0で通常の太さです)。省略できません。
borderWidth int 0 ～ 10 縁取りの太さを指定します(0で縁取り描画がOFFになります)。省略できません。
ttcFontIndex int
ttcフォントの場合に使用するフォントのインデクスを指定します。省略できません。
scaleWidth float
文字幅のスケールを指定します。省略した場合は 1.0 になります。
scaleHeight float
文字の高さのスケールを指定します。省略した場合は 1.0 になります。
ignorePalt boolean
OpenType フォントの全角文字をプロポーショナルにするための情報を無視して等幅で描画するか否かを指定します。省略した場合は false になります。
deleteBearingX boolean

グリフの左右のベアリング(余白)を削除するか否かを指定します。余白が空きすぎるフォントの場合、true にして bearingOffsetX で調整することで緩和できることがあります。

省略した場合は false になります。
bearingOffsetX int
グリフの左右のベアリング(余白)を調整します。 省略した場合は 0 になります。
baselineOffset int
ベースラインの上下を調整します。省略した場合は 0 になります。

内容

以下の要素を含みます。

  • <path> : フォントファイルのパスを指定します。必ず一つ出現します。
  • <charCodeRangeSet> : このフォントを表示する文字の範囲を指定します。 必ず一つ出現します。

<path>

フォントファイルのパスの設定です。

内容

 フォントファイルのパスを指定します。親の要素が、<bitmapFont> の場合は、ffnt, bffnt 形式が指定できます。親の要素が、<scalableFontDescription> の設定の場合は、ttf, otf, bfttf, bfotf 形式が指定できます。

 パスは絶対パスもしくは複合フォント定義ファイルからの相対パスで記述できます。

<charCodeRangeSet>

表示する文字の範囲の設定の配列要素です。

内容

以下の要素を含みます。

  • <charCodeRange> : 表示する文字の範囲を指定します。0個以上出現します。

<charCodeRange>

表示する文字の範囲の設定です。

属性

名前 型 説明
first unsignedInt 文字の範囲の最初を 10 進数で指定します。省略できません。
last unsignedInt 文字の範囲の最後を 10 進数で指定します。省略できません。
