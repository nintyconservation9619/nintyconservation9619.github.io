エフェクトの表示操作

ここではビューアに反映されているエフェクトの表示 / 非表示を切り替える方法について説明します。

準備

 EffectMaker とビューアを起動した状態から作業を開始します。

 関連リンク EffectMaker とビューア の起動方法については 「作業の流れ」 を参照してください。

エミッタセットの読み込み

以下の手順で説明に使用するエフェクトを EffectMaker に読み込みます。

1 メインメニュー項目 [ファイル]→[  開く]を選択します。
2

ウィンドウが開きますので、以下の場所からエミッタセットファイル（.eset）を選択し、ファイルを開きます。

\Samples\Sources\Resources\Vfx\Eset\QuickStart\NodeOperation\Viewer.eset

ビューアを確認する

読み込んだエフェクトがビューアに表示されていることを確認します。
以降はビューアで変化を確認しながらエフェクトを編集します。

エフェクトの表示 / 非表示を切り替える

エフェクトの表示 / 非表示を切り替える方法を、エミッタノードを例に説明します。
以下はワークスペース上での操作手順とビューアの状態を説明しています。

1 ビューア上では「 Emitter_A」のパーティクルが「 Emitter_B」の前面に表示されています。
2 「 Emitter_A」左側の「 表示ボタン」をマウス左ボタンクリックで「 非表示」にします。
ビューアでは「 Emitter_A」は非表示となり、「 Emitter_B」のパーティクルが表示されています。
