6 データの保存

作成したデータの保存の手順を説明します。
保存には様々な方法があります。

このチュートリアルでは以下の UI について触れています。

全て指定場所に保存 編集内容を含む 3DEditor で開いている全てのファイルを指定場所に保存します。
モデルを保存 任意のモデルを保存します。
マテリアルアニメーションの保存 任意のマテリアルアニメーションファイルを保存します。
プロジェクトファイルの保存 3DEditor の作業状態を保存します。

全て指定場所に保存

すべての編集状態を保存します。

3DEditor →「ファイル」→「全て指定場所に保存」から保存先フォルダを選んで「 OK 」ボタンで保存します。

参考：

個別に保存しなくても全て保存されます。

通常作業では「ファイル」→「全て上書き保存」して、編集内容を漏れなく更新します。
ここではサンプルモデル等を上書きしないように「全て指定場所に保存」に保存します。

モデルを保存

モデルを保存します。マテリアルの設定も含まれます。

オブジェクトビューのスケマティック表示からモデルのツリーのトップ「 SampleTripod.fmdb 」を右クリックします。
コンテキストメニューから「別名で保存」を選択します。
保存先を指定して名前を付けて保存します。

参考：

通常作業ではコンテキストメニューから「上書き保存」して更新します。
ここではサンプルモデルを上書きしないように「別名で保存」で保存します。

マテリアルアニメーションの保存

作成したマテリアルアニメーションを保存します。

ファイルビューの「 Eye.fmab 」を右クリックで「別名で保存」を選択します。
保存先を指定して名前を付けて保存します。 

プロジェクトファイルの保存

3DEditor の作業状態を保存します。
個別にファイルを読み込まなくてもプロジェクトファイルを読むと作業状態が復元できます。

ファイルビューの「 Project.f3pj 」を右クリックで「別名で保存」を選択します。
保存先を指定して名前を付けて保存します。 

参考：

ファイルビューの状態が保存されます。
マテリアルやアニメーションの編集内容は保存されません。
