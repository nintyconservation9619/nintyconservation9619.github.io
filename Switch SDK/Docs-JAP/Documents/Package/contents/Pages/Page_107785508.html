使用方法：レイヤの作成

レイヤの作成方法の例として、グラフィックスのサンプルを参照してください。NVN の場合、この初期化の手順は、NvnTutorial ライブラリの TutorialBaseClass.Nx.cpp に記載されています。

1.VI ライブラリを初期化します。

nn::vi::Initialize();

2.ディスプレイを開き、ディスプレイハンドルを取得します。

nn::vi::Display* pDisplay;
nn::vi::OpenDefaultDisplay(&pDisplay);

3.レイヤを作成します。レイヤを作成すると、その可視性が TRUE に設定されます。なお、そのレイヤにアプリケーションで何らかのコンテンツを描画するまでは、レイヤは空なのでディスプレイでは無視されます。

nn::vi::Layer* pLayer

nn::vi::CreateLayer(&pLayer, pDisplay);
// これは、デフォルトのスケーリングモードです
nn::vi::SetLayerScalingMode(pLayer, nn::vi::ScalingMode_FitToLayer);

4.グラフィックス API と共に使用する NativeWindowHandle を取得します。NativeWindowHandle の使用方法の実演については、グラフィックスのサンプルを参照してください。

nn::vi::NativeWindowHandle nativeWindow; 
nn::vi::GetNativeWindow(&nativeWindow, pLayer);
