ACCOUNT ライブラリのエラーコード

エラービューアーに表示されるエラーコードのうち、ACCOUNT ライブラリに由来するものは次の範囲で定義されます。

ACCOUNT ライブラリに由来するエラーコード は 2124-NNNN で表され、 NNNN は 0000 以上 8191 以下となります。

この章では ACCOUNT ライブラリに由来するエラーコードについて、その発生要因を説明します。 
ただし、 ACCOUNT ライブラリに由来するすべてのエラーコードについて説明したものではないことに留意してください。 

注意：

このドキュメントに含まれるエラーコードとその説明は、エンドユーザー向けのものとは異なります。
Nintendo SDK 利用者が開発用途にのみ参照することを想定しています。

ACCOUNT ライブラリのエラーコードと Result 値

ACCOUNT ライブラリに由来する Result 値は、その Module 値が 124 として定義されます。
Module 値が 124 の Result 値に関して、その Description 値は、 ACCOUNT ライブラリに由来するエラーコード 2124-NNNN の NNNN と一致します。

Result 値検索で ACCOUNT ライブラリの該当する Result 値が見つからなかった場合は、本章の内容を参照してください。

注意：

ACCOUNT ライブラリの公開されていない Result 値の Description 値は本体システムのバージョンによって変更される可能性があります。
従って、プログラム上で Description 値による分岐等は行わないでください。
