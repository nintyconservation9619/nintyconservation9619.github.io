エミッタ形状「円（ボリューム）

ここではパーティクルの放出位置が円盤面のエミッタ形状「円（ボリューム）」の設定項目について説明します。
以下は、このエミッタ形状使用時のパーティクルです。

概要

エミッタ形状「円（ボリューム）」は、エミッタ形状「円」の全面から円状に広がる法線方向にパーティクルを放出します。
面の「空洞率」ではパーティクルが放出されない領域を設定できる為、ドーナツ型のエミッタも作成できます。
以下は「 設定項目」の概要です。

「エミッタ形状」を「円（ボリューム）」に設定すると「 円（ボリューム）」が表示されます。以下はグループ内の設定項目です。

空洞率

パーティクルが放出されない領域を、エミッタ中心から外周までの距離に対する割合で設定します。
