カーニング情報ファイル（bffkn）の作成

質問

カーニング情報ファイル（bffkn）の作成方法を教えてください。

回答

現状ユーザー個々にカーニング情報ファイル（bffkn）を作成する方法はありません。また、フォーマットも公開していません。
Bitmap を出力する際に、コンバーターが生成するようになっています。

フォントにカーニングを埋め込む場合は以下の方法で行ってください。

  • Windows フォント からフォントを生成する。
  • Windows フォントから画像を出力して(この際、Windows フォントの持つカーニングが bffkn が出力される)、画像を編集してからフォントを生成する。

なお、Windows フォント がカーニング情報を持っているフォントでないとカーニング情報は出力されませんのでご注意ください。
