atk の主な変更点

バイナリリソースファイルに含まれる波形データのアライメント調整

nn::audio の波形データアライメントが 64 バイトに変更されたことに伴い、atk リソースファイルに含まれる波形データのアライメントを調整しました。

NintendoSDK 0.8.0 以前のコンバータで作成したリソースファイルは、新しい atk ライブラリでは利用できなくなっていますので、新しいコンバータで再コンバートしてください。
影響のあるリソースファイルは以下の通りです。

  • bfwav
  • bfwar
  • bfgrp
  • bfsar

atk::viewer 機能を libnn_atk ライブラリ から libnn_atkviewer ライブラリに分離

ビューア機能 (nn::atk::viewer 名前空間の機能) を、libnn_atk ライブラリから libnn_atkviewer ライブラリに分離しました。 
ヘッダの場所や名前空間に変更はありませんが、ビューア機能を利用しているアプリケーションは、別途 libnn_atkviewer ライブラリへのリンクを追加する必要があります。
