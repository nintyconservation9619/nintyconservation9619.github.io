プロパティウィンドウ

「プロパティウィンドウ」ではノードの操作やプレビューができる UI や、ノードの機能設定ができる UI から構成されています。

A プレビューコントロール

「 ビューアに接続」している場合に、再生や停止等の操作を行います。
B ワークスペースのツールバー ノード操作に関するボタンが配置されています。
C ワークスペース

エフェクトを構成する各ノードを階層構造で表示します。

マウス左ボタンクリックによりそのノードに関係したメニューが開きます。
D タブ

「 タブ」はワークスペースで選択したノードの機能を種類ごとにまとめて表示します。
E グループ 「 タブ」内にある「 グループ」は「 設定項目」をその機能の種類ごとにまとめて表示します。
F 設定項目 「 設定項目」は機能の名前を表示します。
右側には設定項目の値を編集したり、情報を表示する専用の UI が並びます。
参考：

「 設定項目」でマウス左ボタンクリックすることにより既定のブラウザでドキュメントが開き、その設定項目のヘルプが表示されます。
