テクスチャエクスポートプラグインのプリセット

上記表のおすすめテクスチャを作成する場合、Photoshop テクスチャエクスポートプラグインのプリセット機能を利用すると便利です。
コンフィグファイル ExportTexture.jsxinc をテキストエディタで開き、用途に合わせた以下のテキストを上書きすることで利用できます。
関連リンク 詳細については 「テクスチャエクスポートプラグインのコンフィグファイル」 を参照してください。

ノンリニア環境

以下はリニア環境ではないゲームアプリケーションで使用するテクスチャエクスポートプラグインのプリセットです。

var format_presets =
  {
    "vfx_L"              : "unorm_8,rrrr,non,0000",
    "vfx_L_comp"         : "unorm_bc4,rrrr,non,0000",
    "vfx_LA"             : "unorm_8_8,rrrg,non,0000",
    "vfx_LA_comp"        : "unorm_bc5,rrrg,non,0000",
    "vfx_RGBA"           : "unorm_8_8_8_8,rgba,sRGBfetch,0000",
    "vfx_RGBA_comp"      : "unorm_bc7,rgba,sRGBfetch,0000",
  };

リニア環境

以下はリニア環境下のゲームアプリケーションで使用するテクスチャエクスポートプラグインのプリセットです。

var format_presets =
  {
    "vfx_L"         : "unorm_8,rrrr,non,0000",
    "vfx_L_comp"    : "unorm_bc4,rrrr,non,0000",
    "vfx_LA"        : "unorm_8_8,rrrg,linear,0000",
    "vfx_LA_comp"   : "unorm_bc5,rrrg,linear,0000",
    "vfx_linr_RGBA"      : "srgb_8_8_8_8,rgba,sRGBfetch,1110",
    "vfx_linr_RGBA_comp" : "srgb_bc7,rgba,sRGBfetch,1110",
  };

シェーダ用

「パーティクルシェーダ」を使う場合や、「カスタムシェーダ」として法線マップを実装した場合に使用するテクスチャエクスポートプラグインのプリセットです。
通常のテクスチャに追加する形で使用してください。

var format_presets =
  {
    "vfx_flowmap": "unorm_bc5,rggg,indirect,0000",
    "vfx_nml_map"     : "snorm_bc5,,normal,0000",
  };
