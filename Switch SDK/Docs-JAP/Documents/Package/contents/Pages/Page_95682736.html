スフィアマップ

ここでは登録したテクスチャをスフィアマップとして適用する機能「スフィアマップ」について説明します。

概要

「スフィアマップ」は、テクスチャをスフィアマップとして EffectMaker のシェーダに適用できる機能です。
主にパーティクル形状「プリミティブ」の際に利用します。
スフィアマップのしくみ上、使用するプリミティブはある程度ポリゴン分割され、且つ開いた法線の時にその効果が解りやすくなります。
以下は「 設定項目」の概要です。

スフィアマップ

登録したテクスチャをスフィアマップに適用するかを「 チェックボックス」ON/OFF で切り替えます。

設定例

以下は「スフィアマップ」として使用したテクスチャイメージとその使用例です。

A

疑似スペキュラ表現 鏡面反射光の表現に利用した例です。
B

疑似リムライト表現 逆光により、プリミティブの際に「光」が回り込む表現に利用した例です。
C

疑似環境マップ表現
 周囲の風景を描いたスフィアマップテクスチャを利用した例です。
