UV スクロールアニメ

ここでは登録したテクスチャの UV をスクロール移動する機能「UV スクロールアニメ」について説明します。

概要

「UV スクロールアニメ」は登録したテクスチャの UV 座標を移動させることでテクスチャをスクロール移動できる機能です。
「繰り返し数」や「ラップ」などの機能と組み合わせることで、テクスチャのパターンを繰り返し表示したり、テクスチャを徐々に消したりできます。
以下は「 設定項目」の概要です。

UV スクロールアニメ

テクスチャ UV 座標の「位置」を各軸個別に設定できます。
X がテクスチャ座標 U 、 Y がテクスチャ座標 V に相当します。

設定を ON にすると「 UV スクロールアニメ」が表示されます。以下はグループ内の設定項目です。

加算値

テクスチャ UV の位置に毎フレーム加算する値を設定します。
加算値はエミッタ起動時から適用されます。

初期値

テクスチャ UV 座標の位置に初期値を設定します。

初期値ランダム

初期値に指定した範囲の乱数を設定します。
初期値を乱数の上限、下限側はランダムに変化します。以下は乱数の下限をもとめる計算式です。

  • 乱数の下限 : 初期値 - 「初期値ランダム」の計算結果

設定例

以下は「UV スクロールアニメ」の各機能の設定例です。
「放出間隔」は 10 に設定しています。

A 加算値
 指定の軸に対して毎フレーム加算値 0.01 を設定しています。
負の値を設定すると、画像とは反対側にスクロールアニメーションします。
B 初期値 指定の軸に対して初期値として 0.5 を設定しています。
C 初期値ランダム

指定の軸に対してランダム値として 0.5 を設定しています。
放出毎の初期値は「初期値」0.5 に対し、上限を更に +0.5 した範囲内でランダムな値をとります。
