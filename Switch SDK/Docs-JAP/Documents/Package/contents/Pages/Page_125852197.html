3 ビューアーと連携

注意：

3dSampleViewer は 3DEditor チュートリアル用のビューアーになります。実際にゲーム制作で 3DEditor を利用する際は、ビューアーライブラリ をゲームアプリケーションに組み込んで、3DEditor のモデルをプレビューできる環境を用意してください。


3DEditor とビューアー( 3dSampleViewer )を連携してプレビューする手順を説明します。
ビューアーと接続、アニメーションの再生と停止、再生するファイルの変更を行います。

このチュートリアルでは以下の UI に触れています。

PC に接続 ビューアー( 3dSampleViewer )と接続します。
アニメーションコントロール アニメーションの再生、停止を行います。
再生アニメーションの変更 再生するアニメーションを変更します。

ビューアーに接続

3DEditor のツールバーにある「未接続 」をクリックします。

接続に成功すると「 接続済み」という表記になり、SampleTripod のモデルがビューアーに表示されます。

参考：

接続に失敗する場合は以下の事を確認して、再び接続を試みて下さい。

  • Windowsのスタートアップメニューから Nintendo Target Manager を起動する。
  • ツールバーのプラットフォームが「 Win 」になっていることを確認する。
  • 接続先が「 PC 」になっていることを確認する。

アニメーションの再生と停止

3DEditor 画面の下のアニメーションコントロールでアニメーションを再生します。

「再生 / 一時停止ボタン」を押すとアニメーションが開始します。もう一度押すと一時停止します。

「停止ボタン」を押すとアニメーションが停止します。

アニメーションはビューアーの画面を見て確認します。

関連リンク 詳しくはアニメーションの操作を参照してください。

再生アニメーションの変更

再生されるアニメーションは、ファイルビューの「既定アニメーションセット」の一番下のファイルです。

参考：

「アニメーション」にあるファイルと間違えないよう注意して下さい。


 再生するファイルを変更するには、再生させたいファイルをドラッグ＆ドロップして一番下に移動させます。

ビューアーで再生されるアニメーションが替わるのを確認します。

参考：

「アニメーションのプレビュー表示・非表示」を押して非表示にすることで、表示したいアニメーションを選択する方法もあります。
