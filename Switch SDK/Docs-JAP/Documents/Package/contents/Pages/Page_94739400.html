緯度

ここでは球形状エミッタの設定項目「緯度」について説明します。

概要

弧の開き方「緯度」は球系エミッタの時、エミッタを球体の Y 軸正方向（天頂）を基準に南（天底）方向に広げることでドーム状にできる機能です。
パーティクル密度を維持したまま放出範囲を半球状にできます。

以下は「 設定項目」の概要です。
「弧の開き方」を「緯度」にすると「 緯度」が表示されます。以下はグループ内の設定項目です。

軸

ワールド座標系で軸の向きを設定します。天頂の方向が +（プラス）です。

弧の幅（角度）

天頂を弧の開始位置とした幅を角度で設定します。

設定例

以下は「弧の開き方」の「緯度」の設定例と設定内容の解説図です。

A エミッタ半径 「エミッタ半径」を 50 に設定しています。
B 軸

「軸」はプリセットから +Y に選択しています。
球体の天頂位置をエミッタ座標系でY軸正方向に向けた設定です。
C 弧の幅角度
 天頂から天底方向に 90 度広げた設定です。
エミッタはドーム状になります。
