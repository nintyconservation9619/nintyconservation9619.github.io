検索

「検索」タブ固有の UI とその役割について説明します。

A 検索基点パス

「検索」タブではアドレスバーに指定したパスが検索開始時の基点になります。
B 検索設定項目
 「検索」タブではナビゲーションペインに検索の設定項目を表示します。
C ファイルリスト 検索の結果ヒットしたファイルを表示します。

検索設定項目

以下は検索の設定項目です。

A 検索対象ファイル 検索対象を「テクスチャ」と「プリミティブ」から選択します。
B 検索条件

検索条件をプリセットから選択します。プリセットごとに専用の UI が表示されます。
C

検索範囲を検索基点パス以下に絞り込む

検索範囲を「検索基点パス」以下に絞り込むかどうかを「 チェックボックス」ON/OFF で切り替えます。

OFF の場合はドライブの最上位までが検索範囲となります。
D 検索実行 マウス左ボタンクリックで検索を実行します。
E リセット マウス左ボタンクリックで「検索条件」とファイルリストの内容をデフォルトの状態に戻します。

使用可能な機能

以下は「検索」タブで使用可能な機能です。

エクスポート 選択しているファイルをワークスペースにエクスポートします。
コメントの編集 選択しているファイルのコメントを編集します。
ラベルカラーの選択

選択しているファイルのラベルカラーを設定します。
エクスプローラで表示 エクスプローラウィンドウを開き、選択しているフォルダやファイルを表示します。
