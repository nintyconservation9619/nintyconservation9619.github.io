遠距離アルファ

ここではカメラから遠い距離にあるパーティクルに対して半透明処理を施す機能「遠距離アルファ」について説明します。

概要

「遠距離アルファ」は、カメラから遠い距離にあるパーティクルに対して半透明処理を施す機能です。
カメラから遠ざかる不要なパーティクルを、画面に表示された状態でも自然に消去できます。

以下は「 設定項目」の概要です。

遠距離アルファ

遠距離アルファを使用するかを「 チェックボックス」ON/OFF で切り替えます。
ON に設定すると「 遠距離アルファ」が表示されます。

以下は「 設定項目」の概要です。

αが 1 になる距離

アルファ値が 1 になる距離をカメラからの距離で設定します。

αが 0 になる距離

アルファ値が 0 になる距離をカメラからの距離で設定します。

設定例

以下は「遠距離アルファ」の各機能を ON/OFF した場合の比較画像です。

A 

αが 1 になる距離 = 30

αが 0 になる距離 = 50

エフェクトはカメラから 30（3 グリッド）のところから 50（5 グリッド）までの範囲で透明になります。
カメラから遠ざかるほど透明になる設定です。
B

αが 1 になる距離 = 90

αが 0 になる距離 = 100		 デフォルト設定です。
エフェクトはカメラから 90（9 グリッド）のところから 100（10 グリッド）までの範囲で透明になります。
エフェクトがカメラからかなり遠ざかった時に透明になる設定です。
C

αが 1 になる距離 = 50

αが 0 になる距離 = 30

エフェクトはカメラから 50（5 グリッド）のところから 30（3 グリッド）までの範囲で透明になります。

A   とは逆に、カメラに近づくほど透明になる設定です。
