描画パス

ここではゲームアプリケーションで指定する描画パスを設定する機能「描画パス」について説明します。

概要

「描画パス」は、プロジェクト設定ファイルの指定がある場合に、ファイルに記述されたゲームアプリケーション側の描画タイミングを選択できる機能です。
描画パスをエミッタ毎に設定することによって、放出されるパーティクルを、ゲーム内の描画パスに挟みこむことが出来ます。
以下は「 設定項目」の概要です。

描画パス

パーティクルの描画タイミングを登録したプロジェクト設定に記述されたプリセットから選択します。

関連リンク 以下は関連ページへのリンクです。

プロジェクト設定 ゲームプロジェクト毎の設定ファイルを登録する「プロジェクトファイル」について説明します。
プロジェクト設定ファイル プロジェクト設定ファイルの記述内容について説明します。
エフェクト描画順を制御する 「描画パス」によ描画順を制御する機能について説明します。
