ワールド座標とエミッタ座標

エフェクトで扱う座標系について説明します。
座標系によって、パーティクルを配置するさいの基準が変わります。

座標系の種類

 EffectMaker で速度や方向、大きさや角度を設定するさいに扱う座標系には「ワールド座標系」と「エミッタ座標系」があります。

床の表示 「床」を表示させることで、ワールド座標系の軸が確認できます。
親エミッタ形状の表示

エミッタ形状を表示させることで、エミッタ座標系の軸が確認できます。

エミッタ座標系の機能を使うさいに便利です。

座標系を設定する機能の一覧

以下は EffectMaker で座標系を設定する機能の一覧です。

ノード 機能名 説明 影響する項目
位置 回転 スケール
エミッタ 放出 / 指定方向初速 / ワールド座標系で適用 初速方向をワールド座標系で適用します。 ○ ○ ○
放出 / 重力 / ワールド座標系で適用 重力のかかる方向をワールド座標系で適用します。 ○ ○ ○
磁力 位置をエミッタに追従 磁力の位置をエミッタ座標系に追従させるかを設定します。 ○ - -
収束 位置をエミッタに追従 パーティクルの収束位置をエミッタ座標系に追従させるかを設定します。 ○ - -
位置に加算 ワールド座標系で適用 パーティクルを移動させる方向をワールド座標系で適用します。 ○ ○ ○
シンプルコリジョン ワールド座標系で適用 コリジョンの配置位置をワールド座標系で適用します。 ○ ○ ○
カールノイズ ワールド座標系で適用 ゆらぎ要素を持つテーブルをワールド座標系で適用します。 ○ ○ ○

「パーティクル追従タイプ」について

「パーティクル追従タイプ」で「エミッタSRTに完全追従」を選択した場合、エミッタに追従します。
