アニメーション区間タグ、対象グループの指定をデフォルトで必須に変更

アニメーション区間タグを使用する時には、対象グループを指定することがデフォルトで必須となりました !!

デフォルトで必須とした理由は、グループ指定 をする運用が多くを占めており、なおかつ 対象グループ の指定忘れが多数発生しているためです。
( 対象グループの指定 は、所定の設定をすると以前同様に扱うことが出きます。)

 

・対象グループ を指定していない場合、アニメーション区間タグウインドウ に「警告アイコン」を表示するようになりました。

・ビューア転送時など、バイナリ変換した際、エラー通知をするようになりました。

・LayoutEditor のプロジェクト設定ファイルで、エラーチェックを制御できるようになりました。

プロジェクト設定ファイルの、AllowNoGroupAnimationTag 要素 を設定すると、今までどうりに扱うことができます。

 アニメーション区間タグ、一括選択グループ設定が可能に

 複数の「アニメーション区間タグ」を同時に選択状態にして、「関連グループ」をまとめて設定できるようになりました !!

今までは複数の「アニメーション区間タグ」が存在した場合、同じ設定でも、一つ一つの区間タグ毎に「関連グループ」を設定していました。
これからは、同じ関連グループを設定する「アニメーション区間タグ」はまとめて選択状態にし、一括で「関連グループ」設定ができます !

 フォントサイズスケール、入力に「自動で縦横比が連動」追加

[ プロパティウインドウ ] > [ テキストペイン ] > [ 文字の設定 ] の中にある「サイズスケール」に、自動で縦横比が連動するボタンが追加されました !!

サイズスケールの数値入力箇所の横に、ボタンが追加されました。
これはサイズスケールの「縦横比連動ボタン」です。
サイズスケールの「連動」と「非連動」が切り替わります。

「プロパティウインドウ > 文字の設定 > サイズスケール」に追加された「縦横比の連動ボタン」
縦横比の連動ボタン「連動中」 サイズスケールの縦横比の数値が、「 X 」「 Y 」双方連動します。
「 X 」、「 Y 」、どちらのサイズスケールに数値を入力しても、連動して同じ数値となります。
縦横比の連動ボタン「非連動」 サイズスケールの縦横比の数値が、「 X 」「 Y 」で非連動となります。
「 X 」、「 Y 」、どちらのサイズスケールにも別の数値を入力できます。

 ペインをドラッグ時、小数点以下の数値が入らないように変更

ペインをドラッグで位置調整する時に、小数点以下の座標数値が入らないようになりました !

ペインをドラッグして位置調整すると、今までは小数点付きの座標数値が入っていました。 ( グリッドに吸着を使用しない場合 )
小数点以下の座標数値が入らないようになることにより、作業の効率化が図れます。
「ペインをドラッグ → カーブエディタに座標登録」の作業が、さらにスムーズに行えると思います !!
