花火を作る

ここではエミッタの階層化により、以下の様な花火のエフェクトを作成する手順について説明します。

このチュートリアルでは以下のUI操作について説明しています。

階層下にプリセットを追加

良く使用するエミッタをプリセットとして登録しておくことで、UI から階層下に追加して利用できます。

ノードの一括編集

 複数のエミッタ間で同じ設定に揃えたい場合にまとめて編集できます。
参考：

完成データは以下の場所にあります。
\Samples\Sources\Resources\Vfx\Eset\Fireworks.eset

エフェクト作成の準備

あらかじめ EffectMaker と EffectViewer を起動した状態から、エフェクト作成にあたっての準備を行います。

関連リンク EffectMaker と EffectViewer の起動方法については「作業の流れ」を参照してください。

エミッタセットの読み込み

以下の手順でとなるエフェクトを EffectMaker に読み込みます。

1
メインメニュー項目 [ファイル]→[ 開く]を選択します。
2

ウィンドウが開きますので、以下の場所からエミッタセットファイル（.eset）を選択し、ファイルを開きます。
\Samples\Sources\Resources\Vfx\Eset\Fireworks_Template.eset

3

ワークスペースに「 Fireworks_Template」が読み込まれます。

関連リンク 以下は関連ページへのリンクです。

ワークスペース

エフェクトを構成する各ノードを階層構造で表示する「ワークスペース」について説明します。
プロパティウィンドウ ノードの操作やプレビューができる「ワークスペース」とノードの機能設定ができる UI から構成される「プロパティウィンドウ」について説明します。

「カメラ」タブを選択する

ワークスペースウィンドウ上で「 ビューアノード」を選択します。
プロパティウィンドウから「カメラ」タブを選択します。
ビューアのカメラに関する設定項目が表示されます。

ビューアのカメラを設定する

エフェクトが確認しやすい状態にする為、「 ビュー・投影」以下の「 項目」を設定します。

項目名 値 説明
カメラの位置 X,Y,Z = 0,30,100 エフェクトがビューア画面に収まる様にカメラの位置と注視点を調整します。
注視点 X,Y,Z = 0,20,0

ビューアを確認する

以下の様に、宙に打ち上げられた火の玉がビューアに転送されていることを確認します。
このエフェクトを夜空に打ち上げられる「火の玉」として、花火のエフェクトを作成します。
以降はビューアで変化を確認しながらエフェクトを編集します。

階層下に花火の部品を追加する

プリセットフォルダからエミッタ「 Fireworks」の階層下に花火エフェクトの部品となるエミッタを読み込みます。

プリセットフォルダパスの変更

以下の手順でプリセットフォルダパスをデフォルトパスからこのチュートリアル用プリセットが置かれているパスに変更します。

1

メインメニュー項目 [ツール]→[ オプション]を選択します。
2 オプションダイアログを開きますので、ナビゲーションペインの【ユーザー設定】を選択します。
3

「 プリセットフォルダパス」の「 ロードボタン」を押すことで開くウィンドウから、以下の場所を指定します。
\Samples\Sources\Resources\Vfx\Eset\Presets
4

「 OK ボタン」を押すことで設定内容を確定してウィンドウを閉じます。


プリセットを開く場所がデフォルトからこのチュートリアル専用の場所に変更されます。
「 リセットボタン」をおすことでデフォルトパスに戻せます。

関連リンク 詳細については「プリセットフォルダパスの設定」を参照してください。

プリセットから階層下にエミッタを追加する

以下の手順で「 Fireworks_Template」直下の「 FireBall」階層下に花火エフェクトの部品を読み込みます。

1
ワークスペースからエミッタセット「 Fireworks」の階層下にある「 FireBall」をマウス右ボタンクリックで選択します。
2
表示されるメニューから「 階層下にプリセットを追加」→「 Fireworks_Presets」→「 FlareLine」の順で選択します。
3

ダイアログが表示されるので「 OK ボタン」を押すとと「 FireBall」階層下に「 FlareLine」が読み込まれます。
4

同様の手順で他のプリセット、「 TailSmoke」、「 Firework」を選択すると、各々「 FireBall」階層下に読み込まれます。

ここまでの手順において、「 FireBall」の階層下には、以下の様に 3 つのチャイルドエミッタが作られています。

 FlareLine 火の玉のフレア表現用エフェクトです。
TailSmoke 火の玉が燃えることにより出る煙エフェクトです。
Firework 破裂する花火のエフェクトです。
参考：

「 FireBall」の階層下にある「 FlareLine」、「 TailSmoke」、「 Firework」は「子エミッタ（チャイルドエミッタ）」と呼びます。
「 FireBall」は子エミッタに対する「親エミッタ」と呼びます。
関連リンク 詳細については「チャイルドエミッタ」を参照してください。

継承設定をまとめて変更する

チャイルドエミッタの継承設定をまとめて行います。

ノードをまとめて選択する

以下の手順で「 FireBall」階層下のチャイルドエミッタを全て選択します。

1
ワークスペースから「 FlareLine」を選択します。
2
「 Firework」を Shift + マウス左ボタンクリックでチャイルドエミッタをまとめて選択します。

「基本設定」タブを選択する

チャイルドエミッタをまとめて選択した状態で「 基本設定」を選択します。

「基本設定」タブを設定する

チャイルドエミッタの「 継承設定」をまとめて設定します。
以下の「 項目」を「 チェックボックス OFF」で全て無効にします。

項目名 説明
アルファ0 親のアルファ値を継承せず、自身のアルファ値を使用する設定です。
アルファ1
スケール 親のスケール値を継承せず、自身のスケール値を使用する設定です。
回転 親の回転値を継承せず、自身の回転値を使用する設定です。

関連リンク 詳細については「複数エミッタの選択と一括編集」を参照してください。

チャイルドエミッタの継承設定を変更する

チャイルドエミッタ「 FlareLine」の継承設定を変更します。

「FlareLine」を選択する

以下の手順で「 FlareLine」のみを選択します。

1
ワークスペースから親エミッタ「 FireBall」を選択し、チャイルドエミッタの選択状態を解除します。
2
「 FlareLine」を選択します。

「基本設定」タブを選択する

チャイルドエミッタ「 FlareLine」を選択した状態で「 基本設定」を選択します。

「基本設定」タブを設定する

チャイルドエミッタ「 FlareLine」の「 継承設定」を設定します。
以下「 項目」を「 チェックボックス ON」に変更します。

項目名 説明
パーティクルにエミッタ割当
チャイルドエミッタ「 FlareLine」は親エミッタ「 FireBall」から放出されるパーティクルに追従します。
上位の階層より先に描画
チャイルドエミッタ「 FlareLine」は親エミッタ「 FireBall」より先に描画されます。
カラースケール
チャイルドエミッタ「 FlareLine」のカラースケール に親エミッタ「 FireBall」のカラースケールが乗算されます。
カラー「カラー 0」
チャイルドエミッタ「 FlareLine」のカラー 0 に親エミッタ「 FireBall」のカラー 0 が乗算されます。

関連リンク 以下は関連ページへのリンクです。

パーティクルにエミッタ割当 親エミッタから放出されるパーティクルにエミッタを割り当てる機能「パーティクルにエミッタ割当」について説明します。
継承設定「描画パス」 チャイルドエミッタの継承設定「描画パス」について説明します。
継承設定「カラースケール」 チャイルドエミッタの継承設定「カラースケール」について説明します。
継承設定「カラー」 チャイルドエミッタの継承設定「カラー」について説明します。
