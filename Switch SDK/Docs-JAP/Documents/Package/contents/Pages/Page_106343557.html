レイアウトウインドウ

 レイアウトウインドウ

レイアウトウインドウを構成する、各部位の名称と役割について説明します。
  • A  レイアウト画面領域
  • B  座標系原点
  • C  ツールバー
  • D メニューバー追加項目（レイアウトエディタウインドウ）
    レイアウトウインドウを開くと、レイアウトエディタウインドウのメニューバーに項目が追加されます。

A レイアウト画面領域

レイアウト画面領域は、実機に表示される画面と対応しています。
作業に必要な画面領域の大きさを最初に設定し、レイアウトの作成をしましょう。

参考：

レイアウト設定ウインドウで、レイアウトウインドウの画面内カラーや、グリッドの設定等も行う事ができます。

関連リンク 以下は関連ページへのリンクです。

レイアウト設定ウインドウ レイアウト環境設定についてのUI説明です。

B 座標系原点

画面中央に書かれている赤い線は、レイアウト画面領域の座標系原点です。
（レイアウト座標系は XY 軸 2 次元＋Z 軸 1 次元の 3 次元座標です。）

C ツールバー

以下の項目が並びます。

「取り消し / やり直し」の状況の表示です。レイアウト作業する度に、その動作がカウントされて数値が上がって行きます。
取り消しを行うと、括弧内に取り消した回数がカウントされます。取り消しのカウントが増えた分、作業の数値は減ります。
カーソルの座標です。ペインの移動中は Tweaking 、範囲選択中は Selecting と表示されます。
 

レイアウト統計情報を表示します。

カーソルを重ねると統計情報を表示し、アイコンを左クリックすると統計情報をウインドウで表示します。

親ペインの平行移動操作を子に影響するかどうかを表示します。

影響の変更は [ ペイン ] → [ 親の平行移動を子に影響させる ] のチェックで行います。

参考：

アイコンの左クリックや、プロパティウインドウからも変更可能になる予定です。

「パースペクティブ投影」と「正射投影」を切り替えます。
アイコンを左クリックすると、アイコンの形が切り替わり、投影が変化します。

正射投影時のアイコンです。

カメラ視点からの距離に対象の大きさが影響を受けない投影法です。

パースペクティブ投影時のアイコンです。

カメラ視点からの距離により、近くのものを大きく、遠くのものを小さく表示する投影法です。

アイコンを左クリックすると、関連する部品を一覧表示します。

関連部品の一覧表示から対象をクリックすることで、編集対象を切り替えることができます。

関連部品の一覧表示から対象をドラッグ & ドロップすると、別ウインドウで開きます。
コンボボックスとスライダ、＋と－で、ズームイン、ズームアウトを行うことができます。

D メニューバー追加項目

レイアウトウインドウを開くと、レイアウトエディタウインドウのメニューバーに項目が追加されます。
この項目は、アクティブなレイアウトウインドウに対して実行します。

ペイン

各ペインの作成、親子設定、ペインのロック等、ペインの設定に関する操作を行います。
選択

レイアウトウインドウに表示された各ペインの「選択 / 非選択」の切り替え等を行います。
表示

レイアウトウインドウの表示形態の切り替えを行います。
ツール

  ビューア関係等、ツール関係の操作を行います。
