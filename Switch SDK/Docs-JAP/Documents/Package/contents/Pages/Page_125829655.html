お気に入り

「お気に入り」タブ固有の UI の役割について説明します。

A アドレスバー

フォルダのパスを指定できます。
指定したパスの内容を「ファイルリスト」に表示します。
B ナビゲーションペイン

登録されている「 お気に入り」を表示します。
「 お気に入りフォルダ」に「 お気に入り」をまとめ、見やすいように整理することができます。
参考：

「 お気に入りフォルダ」と「 お気に入り」はマウス左ボタンダブルクリックで好きな名前に変更できます。

使用可能な機能

以下は「お気に入り」タブで使用可能な機能です。

エクスポート 選択しているファイルをワークスペースにエクスポートします。
コメントの編集 選択しているファイルのコメントを編集します。
関連リンク 詳細については「コメント」を参照してください。
ラベルカラーの設定

選択しているファイルのラベルカラーを設定します。
関連リンク 詳細については「ラベルカラー」を参照してください。
お気に入りフォルダ

「お気に入り」タブのビュー上に「 お気に入りフォルダ」を作成します。
「 お気に入りフォルダ」は「 お気に入り」をまとめておくことができます。
お気に入りの削除 登録した「 お気に入り」を「お気に入り」タブのビューから削除します。
お気に入りの登録 選択しているファイルやフォルダを「 お気に入り」として登録します。
エクスプローラで表示 エクスプローラウィンドウを開き、選択しているフォルダやファイルを表示します。
サンプルをお気に入りに登録 パッケージ内のサンプルデータを「 お気に入り」として登録します。
