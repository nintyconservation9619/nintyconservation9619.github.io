ボディレイアウトの作成

ここではボタン部品の配置先となる以下のような「ボディレイアウト」の作成手順について説明します。


レイアウト作成の準備

あらかじめ LayoutEditor を起動した状態から、レイアウトデータ作成にあたっての準備を行います。

関連リンク LayoutEditor の起動方法については 「作業の流れ」 を参照してください。

ウインドウを開く

以下のウインドウが開いた状態から作業を開始します。

  • テクスチャ
  • 親子階層
  • プロパティウインドウ

関連リンク 各ウインドウの UI については 「ウインドウ」 を参照してください。

新規レイアウトの作成

メインメニュー[ ファイル ] → [ 新規作成 ] から新規レイアウトを作成します。

関連リンク 新規レイアウトの作成手順については 「レイアウトの作成」 を参照してください。

画面サイズの設定

作成するメニュー画面のサイズを設定します。
設定したサイズにあわせてレイアウトします。

メインメニュー[ ウインドウ ] → [ レイアウト設定 ] を選択します。
開いた「レイアウト設定」ダイアログで以下のように設定します。

環境設定 画面サイズ 幅 1280.0
高さ 720.0

テクスチャの読み込み

以下の場所から以下表のテクスチャファイル（.tga）を「テクスチャ」ウインドウに読み込みます。
Samples/Sources/Resources/Ui2d/Common/Textures/

「テクスチャ」ウインドウを「詳細表示」に切り替え、テクスチャのフォーマットを以下の様に設定します。

サムネイル

 ファイル名 フォーマット 説明

BG_Line.tga BC4L :（I）

テクスチャをグレースケールとして圧縮して使用するフォーマットです。

背景「走査線」の表現に利用します。

Deco_Scroll.tga A8 :（非圧縮_A） テクスチャをアルファとして圧縮せずに使用するフォーマットです。
背景の装飾として利用します。

Deco.tga LA8 :（非圧縮_I+A）

テクスチャをグレースケールとアルファとして圧縮せずに使用するフォーマットです。

LayoutEditor マークとして利用します。

Samples.tga LA8 :（非圧縮_I+A）

テクスチャをグレースケールとアルファとして圧縮せずに使用するフォーマットです。

タイトルとして利用します。

Distortion.tga L8 :（非圧縮_I）

テクスチャをグレースケールとして圧縮せずに使用するフォーマットです。

テクスチャの輝度を「歪み」として利用します。

関連リンク テクスチャの登録方法については こちら を参照してください。

背景の作成

ピクチャペインでメニュー画面の背景を作成します。

テクスチャを表示するウインドウ「テクスチャ」から「 ピクチャペイン」を作成します。
「テクスチャ」で「 BG_Line.tga」をダブルクリックします。
「親子階層」ウインドウで作成された「 ピクチャペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

ペイン基本

ペイン名

BG

このペインはメニュー画面の背景になります。

サイズ

X,Y = 1280,720

画面サイズに合わせます。

ピクチャペイン

頂点カラー

[ALL] R,G,B = 0,0,0

[ LT ] R,G,B = 8,99,85

[ RT ] R,G,B = 3,20,16

頂点カラーで背景の色を設定します。

テクスチャを使わない為、低コストでグラデーションをなめらかにできます。

 テクスチャ

イメージ設定


繰り返し方

U方向, V方向 = リピート, リピート

「走査線」を表現します。

テクスチャを繰り返してスクロールアニメーションできます。

フィルタリング

拡大時, 縮小時 = Near

 「走査線」がはっきり見えるようにテクスチャの補間方法を変更します。
マッピング設定 スケール U, V = 1.000, 100.000 背景に「走査線」が複数並ぶようにテクスチャ座標を縦方向にスケールします。

マッピング設定

マッピング方法

ペイン位置相対の投影マッピング

「親子階層」ウインドウの「 Deco_Scroll」を選択した状態で、「 テクスチャ」の「 追加」ボタンを押します。
ダイアログから、テクスチャ 0 と同じ画像「 Deco_Scroll.tga」を選択し、OK ボタンを押すことで、「 テクスチャ」に「1」が追加されます。
「プロパティウインドウ」で以下のように設定します。

テクスチャ

イメージ設定

繰り返し方

U方向, V方向 = リピート, リピート

テクスチャを繰り返してスクロールアニメーションできます。

マッピング設定

マッピング方法

ペイン位置相対の投影マッピング

 ペインの位置、回転、スケールに影響する投影マッピングです。

SRT 変換行列

スケール U,V = 0.500,0.500

位置 U,V = 0.250,0.250

最初に登録したテクスチャ「0」とテクスチャ半分のサイズ分ずらします。

 テクスチャコンバイナ

詳細なコンバイナモード

「詳細なコンバイナモードへ切替」ボタンを押し、モードを切り替えます。
登録したテクスチャ「0」と「1」の合成方法を設定します。

「詳細なコンバイナモード」の設定

背景装飾「 Deco_Scroll」に登録した 2 枚のテクスチャ「 Deco_Scroll.tga」を「詳細なコンバイナモード」で合成します。

「 テクスチャコンバイナ」の「詳細コンバイナモード」を以下の手順で設定します。

1 「利用するステージ数」を「2」と入力します。
「ステージ選択」でステージ 0 と 1 が選択できます。
2

ステージ選択「0」ボタンを押し、ステージ 0 を設定します。

3

アルファのコンバインモードで「(ソース 0 ＋ソース 1)×ソース 2」を適用します。

テクスチャ 0 と 1 のアルファ値を足したのち、コンスタント 0 のアルファ値を掛け合わせます。
4

アルファのソースを設定します。

ソース 0 は「テクスチャ 0」、ソース 1 は「テクスチャ 1」を適用します。
5

コンスタントカラー「C0」のカラーとアルファを以下のように設定します。
「 R,G,B = 0 , 128 , 128」、「 A = 30」
6 ステージ選択「1」ボタンを押し、ステージ 1 を設定します。
7

アルファのコンバインモードで「ソース 0×ソース 1」を適用します。

前段「ステージ 0」のアルファ値とポリゴンカラーのアルファ値を掛け合わせます。
8

アルファのソース 1 で「ポリゴンカラー」を適用します。
「 ピクチャペイン」の「頂点カラー」で設定したアルファの値を適用します。

以下は「 BG」「 Deco_Scroll」の設定が終わった段階でのビューアの見え方です。

「ヌルペイン」の作成

作成したペインを階層下に配置し、アニメーションさせる為の「 ヌルペイン」を 2 つ作成します。
「ヌルペイン」階層下のペイン差し替えや追加が容易になります。

メインメニュー[ ペイン ] → [ ペイン作成 ] → [「ヌルペイン」 ] を選択します。
「親子階層」ウインドウに作成された「 ヌルペイン」を選択し、「プロパティウインドウ」で以下のように設定します。

レイアウトデータの別名保存

作成、編集したレイアウトデータを保存します。

メインメニュー[ ファイル ] → [ 別名で保存 ] で表示されるダイアログでファイル名に「Menu.flyt」を設定します。
「名前を付けて保存」ダイアログの OK ボタンを押し、編集内容を保存します。
