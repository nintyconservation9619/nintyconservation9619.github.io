環境変数取得

概要

ホスト PC で設定されている環境変数の値を取得する機能です。
API 呼び出し時点で設定されているシステム環境変数を取得します。
そのため、アプリ起動中であっても、システム環境変数を書き換えることで、動的に値を変更することが可能です。 

環境変数取得の使用方法

1. HTC ライブラリを初期化します。

#include <nn/htc.h>

nn::htc::Initialize();

 

2. 必要に応じて環境変数の値のサイズを取得します。

#include <nn/htc.h>

size_t size;
if( nn::htc::GetEnvironmentVariableLength( &size, "NINTENDO_SDK_ROOT" ).IsSuccess() )
{
    // 成功時の処理
}
else
{
    // 失敗時の処理
}

 

3. 環境変数の値を取得します。

#include <nn/htc.h>

size_t readSize;
const size_t bufferSize = 32;
char buffer[bufferSize];
if( nn::htc::GetEnvironmentVariable( &readSize, buffer, bufferSize, "NINTENDO_SDK_ROOT" ).IsSuccess() )
{
    // 成功時の処理
}
else
{
    // 失敗時の処理
}

 

4. HTC ライブラリを使用しない場合は終了処理を行います。

#include <nn/htc.h>

nn::htc::Finalize();
