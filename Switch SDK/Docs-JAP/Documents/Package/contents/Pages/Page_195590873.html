はじめに

ここでは、NIFM ライブラリが提供する個々の機能に関して説明します。

それに先立ち、本節では NIFM ライブラリ全体で共通の事項について説明します。

NIFM ライブラリの名前空間

NIFM ライブラリが提供する型、定数、関数などは全て nn::nifm:: の名前空間に配置されています。

特に指定される場合を除いて nn::nifm:: より深い階層の名前空間にある型、定数、関数などは使用しないで下さい。
これらは予告なく構成や仕様を変更する場合があります。
