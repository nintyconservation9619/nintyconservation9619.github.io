アルファの設定

ここでは「カラーコンバイナ」計算式で使用する Alpha0 や Alpha1 のアルファ値を設定する機能「アルファ 0 挙動タイプ」、「アルファ 1 挙動タイプ」について説明します。

概要

「アルファ 0 挙動タイプ」は、「カラーコンバイナ」計算式で使用する Alpha0 のアルファ値やアルファ値のアニメーションが設定できる機能です。
「アルファ 1 挙動タイプ」は、「カラーコンバイナ」計算式で使用する Alpha1 のアルファ値やアルファ値のアニメーションが設定できる機能です。
以下は「 設定項目」の概要です。

アルファ 0 挙動タイプ、アルファ 1 挙動タイプ

「カラーコンバイナ」計算式の Alpha0 Alpha1 として使用する挙動タイプを、以下のプリセットから選択します。

固定 アニメーションしない時のアルファをカラー編集の UI を使用して設定します。
8キーアニメ 8 つまでキーフレームが登録できるカラーアニメーションの UI を使用してアニメーションを作成します。

  関連リンク 以下は関連ページへのリンクです。

カラー編集のUI カラーやアルファの値を編集できる UI です。
グラデーションエディタ カラーやアルファにグラデーションでパーティクル時間のアニメーションを編集できる UI です。
エフェクトを違和感なく消したい アルファのアニメーションでエフェクトを消す使用ケースについて説明します。
