揺らぎの適用

ここではパーティクルのアルファやスケールに対して「揺らぎ」を与える機能「揺らぎ設定」について説明します。

概要

「揺らぎ設定」は、パーティクルのアルファやスケールに対して波形による「揺らぎ」アニメーションを与える機能です。
波形を利用することで、周期的な明滅効果をアルファやスケールのアニメーションに与えることができます。
