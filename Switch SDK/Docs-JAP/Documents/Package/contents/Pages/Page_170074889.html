Mouse (NX Addon)

NX Addon の導入によって、下記の環境においてもマウス API の利用が可能になります。

SDEV 環境における構成

USB マウス

USB 接続されたマウスの状態から入力状態を決定します。

複数台のマウスが接続された場合、ボタンの押下状態は全てのマウスの論理和が入力状態となり、座標値やホイールの変化差分は全て累積値に反映されます。

最大で 8 台までのマウスが認識され、それ以降に接続されたマウスは単に無視されます。

他にも HID 機器が USB 接続されている場合は、その数の分だけ認識されるマウスの台数の上限が下がります。

Input Director による入力転送

Input Director  でホスト PC から転送されたマウスの状態から入力状態を決定します。

EDEV 環境における構成
