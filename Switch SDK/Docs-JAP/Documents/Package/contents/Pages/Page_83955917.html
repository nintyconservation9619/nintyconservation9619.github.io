基本設定

ビューアで確認する為のモデルファイルとアニメーションファイルの設定を行います。

モデル

「 モデル」は、 接続先のビューアを制御できる機能群です。
以下はグループ内設定項目の一覧です。

モデルの「ファイル指定」 モデルファイル（ .fmdb）を読み込み利用できる機能です。

キャラクタアニメ

「 キャラクタアニメ」は、 出力されたバイナリファイルを読み込むことで、ビューアでそのエフェクトを確認できる機能群です。
以下はグループ内設定項目の一覧です。

アニメーションの「フォルダ指定」 アニメーションファイル（.fskb）が置かれたフォルダを選択することで、フォルダ以下のアニメーションファイルが表示されます。
表示されたファイルから利用したいアニメーションファイルが利用できる機能です。
