「計装プロファイル」タブの概要

「計装プロファイル」タブには、プロファイリング中に記録された計装プロファイルの関数と計装プロファイルのコードブロックがすべて表示されます。「計装プロファイル」タブは、これらいずれかのタイプのデータが記録されている場合にのみ表示されます。

「計装プロファイル」タブのツールバー

「計装プロファイル」タブのツールバーには以下の各種コントロールがあります。

「すべて選択解除」：選択された項目をすべて選択解除します。

「すべて展開」：ビューのすべての項目を展開します。

「すべて非表示」：ビューのすべての項目を非表示にします。

「類似項目の選択」：有効にすると、ある項目を選択または選択解除したときに、他の同じような名前の項目も選択または選択解除されます。「類似項目の選択」では、展開されたブロックのみが選択の対象となります。

「コピー」：現在適用されている並べ替えとフィルタの設定に従って、現在表示されている内容がすべてクリップボードにコピーされます。その内容を、メール、ドキュメント、Excel のスプレッドシートなどに貼り付けることができます。コピーした内容はタブ区切り形式になっているので、Excel のスプレッドシートに貼り付けたときに列の構造が維持されます。

計装プロファイルの項目

計装プロファイルの項目を展開すると、プロファイリング中に記録されたすべてのデータを表示できます。計装プロファイルの項目ごとに、以下の測定値が表示されます。

  • 「関数の呼び出し」または「ブロック」：プロファイリング中に関数またはコードブロックが開始および終了した回数。
  • 「最大再帰深度」：プロファイリング中に検出された再帰深度の最大数を表示します。この測定値は、値が 0 を超える場合にのみ表示されます。
  • 「呼び出しあたりの時間」または「ブロックあたりの時間」：測定された、呼び出しあたりまたはブロックあたりの時間の平均です（時間 = 合計時間/呼び出しの回数）。
  • 「パフォーマンスカウンタデータ」：これらの測定値は、プロファイリング時に選択されたパフォーマンスカウンタに基づく値です。これらの値は、プロファイリングの期間全体にわたるものです。これらの値を呼び出しの回数で除算すると、（比率ではない）より有用な値を得られることがあります。

グラフ表示する計装プロファイルの項目の選択

計装プロファイルの各項目のデータフィールドを選択すると、それらのデータがサンプルグラフ内でグラフ表示されます。

展開されたすべての項目にある類似のデータフィールドを選択するには、「計装プロファイル」タブのツールバーで「類似項目の選択」ボタンを有効にして、対象のデータフィールドを 1 つクリックします。

その他の選択方法

複数の項目を選択または選択解除するには、Shift キーを押したままでそれらの項目をクリックします（Shift キーを押さずに最初の項目を選択または選択解除してから、Shift キーを押したままで最後の項目をクリックします）。これにより、クリックした 2 項目間の項目をまとめて選択または選択解除できます。

1 つの項目のみを選択しておいて、他のすべての項目を選択解除するには、Alt キーを押したままで、選択しておく項目をクリックします。この操作により、クリックした項目が選択されたままになり、他のすべての項目が選択解除されます。
