ピクチャペイン

ここでは登録したテクスチャ画像をレイアウト素材として利用できる「ピクチャペイン」について説明します。

概要

「ピクチャペイン」は、テクスチャ画像をレイアウト素材として配置できる機能です。
複数のテクスチャ画像を使用し、合成方法を工夫することで複雑な表現が可能です。
合成にはポリゴンカラー（黒カラー、白カラー）やコンスタントカラーも利用できます。

ピクチャペインでは主に以下の設定ができます。

  • ペインの位置、スケール、回転、形状
  • 使用するテクスチャ画像
  • 使用する UV 設定
  • ポリゴンカラーやポリゴンアルファの値
  • テクスチャやカラーの合成
  • 背景や他のペインとの合成

以下は「ピクチャペイン」の設定項目です。

ペイン基本 ペインの名前、位置、形状、中心位置などを設定します。
ピクチャペインの形状 ピクチャペインの形状と、形状固有の値を設定します。
テクスチャ テクスチャ画像と、テクスチャ画像の設定、テクスチャパターンを設定します。
カラー補間設定 カラーとアルファに影響を与える入力値を設定します。
テクスチャコンバイナ 2 枚以上の画像の合成方法を設定します。
フラグメントオペレーション 背景や他のペインと合成方法を設定します。
拡張ユーザー情報 ペインにユーザー独自の拡張情報を設定します。

使用例

以下は複数のテクスチャ画像を合成してピクチャペインを作る設定例です。
ボタンを押したときに白く光るアニメーションを表現しています。

テクスチャ 0

 ピクチャペインに使用するカラー画像です。

テクスチャ 1

マスク範囲

カラーアニメーションを加算合成し、光らせる範囲を決めるマスク用テクスチャ画像です。
テクスチャの繰り返し方を「ミラー」に設定することで、マスクが完成します。
合成時に白部分だけが白く光ります。
黒カラー

ボタンを押したときに白く光るカラーアニメーションを「黒カラー」に設定しています。
テクスチャコンバイナ

「黒カラー」をテクスチャ 1 の画像でマスクし、その結果と テクスチャ 0 の画像を加算します。
