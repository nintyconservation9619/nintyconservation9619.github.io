マッピング設定

マッピング設定

マッピング設定は、LayoutEditor のピクチャペインのテクスチャを編集します。

マッピング方法 マッピング方法を選択します。
SRT 変換行列 SRT 変換行列の設定項目です。
投影マッピング 投影マッピングについて設定をします。

 

マッピング方法

マッピング方法をタブから選択します。

UV マッピング ( UV 座標 0 を使用 )

UV マッピングを使用した設定を行います。

正射投影マッピング

正射投影マッピングを使用した設定を行います。

ペイン位置相対の投影マッピング

ペイン位置相対の投影マッピングを使用した設定を行います。
透視投影 透視投影マッピングを使用した設定を行います。
レイアウトのカメラがパースペクティブのさい、投影するテクスチャ画像はスクリーン空間で描画されます。
ペイン位置相対の透視投影マッピング ペイン位置相対の投影マッピングを使用した設定を行います。


SRT 変換行列

SRT 変換行列についての情報を入力します。

Scale U :

U 方向のスケール値を設定します。

Scale V :

V 方向のスケール値を設定します。

Rotate :

回転の値を設定します。
Translate U : U 方向の移動値を設定します。
Translate V : V 方向の移動値を設定します。


注意：

大きなテクスチャを、「 UV スケール値を大きく設定」する等でリピート表示すると、GPU 処理が大変重くなることがあります。

関連リンク 詳細については「パフォーマンスガイド」を参照してください。

 

投影マッピング

投影マッピングについての情報を入力します。

スケール :

投影マッピングのスケールを設定します。

移動 :

移動値の設定をします。
レイアウト全体に投影

マッピング方法「正射投影マッピング」を選択すると「 チェックボックス」が選択可能となります。

マッピング方法「ペイン位置相対の投影マッピング」では無効になります。
ペイン全体に投影
マッピング方法「ペイン位置相対の投影マッピング」を選択すると「 チェックボックス」が選択可能となります。
ピクチャペインでは指定できません。
ペインの回転・スケールを考慮して投影

マッピング方法「ペイン位置相対の投影マッピング」を選択すると「 チェックボックス」が選択可能となります。

ON の方が扱いやすい挙動です。古いデータで互換性を維持するために OFF を指定します。
参考：

投影マッピングでは、UV 座標値が、随時自動計算されます。ペイン 4 隅に設定した UV 座標値は使われません。

投影マッピングでテクスチャに Rotate を加える時には、正方形のテクスチャを推奨します。
正方形ではないテクスチャを使用した場合、歪みが発生する場合があります。
