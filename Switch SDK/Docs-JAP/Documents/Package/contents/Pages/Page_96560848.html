収束力

ここでは、パーティクルが収束位置に向かう力を設定する機能「収束力」について説明します。

概要

「収束力」は、パーティクルを収束位置に向かって移動させる力を設定する機能です。
値が大きいほど収束位置に向かう力が大きくなります。
以下は「 設定項目」の概要です。

収束力

パーティクルの現在位置と収束位置との距離に対し、毎フレームかける係数を設定します。
その計算結果分、パーティクルは収束位置に向かって移動します。

  • 位置に加算する値 : （ 収束位置 - その時のパーティクルの位置 ） × 「収束力」
参考：

「 パーティクル時間アニメ」ボタンを押すことにより「 収束力(パーティクル時間アニメ)」が表示さ、カーブエディタでアニメーションを設定できます。
アニメーションは「 CPUで動作させる」ON で有効にできます。

設定例

以下は「収束力」の値を変更した場合の比較画像です。

A 0.01 パーティクルが収束位置 ● に向かって移動します。
B 0.1 A と比較すると、パーティクルが収束位置 ● に向かう時間が速くなります。
C 0.25 B と比較すると、パーティクルが収束位置 ● に向かう時間がさらに速くなります。
