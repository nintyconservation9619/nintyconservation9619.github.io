部品機能の使い方手順

0. 部品パレット、部品ルート設定

レイアウトエディタが利用する部品をまとめて格納したフォルダを部品フォルダと呼び、フォルダパスを部品ルートと呼びます。
まずは、この部品ルートを設定します。

部品ルートは、部品パレットで指定します。
部品パレットは、部品ルート以下の部品レイアウトを一覧表示します。
ファイルの新規保存やフォルダ構成変更を行った場合は、リロード処理を行って部品パレットを最新状態に更新します。

1. 部品レイアウトを作る

  1. 通常レイアウトを作成する

    まずは、以下の手順で通常レイアウトを作成します。

    • ペインを配置
    • アニメーションを設定 ( 必要なら )
    • アニメーション区間タグと対象グループの設定 ( 必要なら )
  2. 通常レイアウトに部品設定をする

    先ほど作った、通常レイアウトに部品設定をします。
    部品設定は、部品・コントロール設定ウィンドウで行います。
    部品の種類、名前、コメント、機能アニメーション、機能ペイン、上書きプロパティ設定を行います。

  3. 部品レイアウトを部品フォルダに保存

    作成した部品レイアウトを先ほど設定した部品ルート以下に保存します。
    部品パレットのリロードを実行すると、保存したレイアウトが一覧に表示されるはずです。

2. 部品ペインを配置

  1. 部品を配置するレイアウト ( ボディレイアウト ) を作成する

    レイアウトの新規作成をして、部品を配置するレイアウトを作成します。
    このレイアウトを「ボディレイアウト」と呼びます。

  2. 部品ペインの作成

    部品レイアウトを配置するためのペイン、部品ペインを作成します。
    サイズ倍率が設定されると、部品ペインは配下のペインに対してペインサイズをサイズ倍率倍します。
    部品ペインを作るには、[ 部品パレット ] → [ リストで部品レイアウトを選択 ] → [ 右クリックメニュー ] → [ 部品ペインを作成 ] を選びます。
    上記手順で、ボディレイアウトに、部品ペインが配置されます。

3. 部品ペインの調整

  1. サイズ倍率調整

    部品ペインはサイズ倍率を指定して、大きさを調整することができます。
    サイズ倍率は、[ 部品ペインを選択 ] → [ プロパティウィンドウ ] → [ 部品ペインページ ] から設定します。

    他のペインのように、プロパティとして任意のサイズを設定できないの点にご注意ください。サイズのアニメーションをすることもできません。

    サイズ倍率とサイズ制約の違い

    部品ペインを配置後に部品レイアウトにサイズ変更があった場合に違いがでます。
    サイズ制約では、配置時の指定サイズが維持され大きさが変化しません。
    サイズ倍率では、大きさが変化します。

    サイズ倍率の種類

    前述のサイズ倍率の影響を、部品レイアウト側で、ペイン毎に指定することができます。
    サイズ倍率の種類は、[ ペイン選択 ] → [ プロパティパネル ] → [ ペイン基本情報 ] から設定します。

    • A:「無視する」ではサイズ倍率が無視されます。部品の大きさにかかわらず一定の大きさを維持したいペインに指定します
    • B:「サイズをスケール倍する」ではサイズ倍率がサイズに適用されます
    • C:「自分を含む部品外枠との距離を維持」では倍率が適用され変化した部品の外周との距離が、一定値を保つようにペインサイズが変更されます。既定の設定です。
  2. 上書きプロパティ

    部品ペインでは、部品設定で、あらかじめ定めたペインについては内容を上書き変更することができます。
    上書きできるペインについては、部品ペイン選択時に、プロパティウィンドウのカテゴリ項目が表示されます。
    上書きペインを設定するには、部品レイアウトの部品設定を編集します。

関連サンプル

関連サンプル「Ui2dSimple」

部品機能に関連したサンプルがあります。

サンプルデータの場所：NintendoSDK\Samples\Sources\Resources\Ui2d\simple\data

以下はサンプルの内容です。

simple.flyt

部品を配置する「ボディレイアウト」です。 左上のペインが「部品ペイン」になっています。
vi_parts.flyt

「部品レイアウト」です。 部品設定として、「 NormalButton 」が設定されています。

「 simple.flyt 」に使われている部品です。
grandchild.flyt 「 simple.flyt 」に使われている部品「 vi_parts.flyt 」の中の孫部品レイアウトです。
