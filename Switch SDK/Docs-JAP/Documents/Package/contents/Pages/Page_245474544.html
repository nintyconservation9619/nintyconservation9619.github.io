アニメーション区間の分割モード

「アニメーション区間の分割モード」が使用できるようになり、アニメーションの製作が、さらに便利になりました !!

従来のレイアウトエディタでは、ひとつの長いアニメーションカーブを区間タグで範囲を指定して再生していましたが、
「アニメーション区間の分割モード」を利用することで、区間タグ毎に個別にアニメーションカーブを作成することが出来るようになります。

( ビューアでの再生は「区間タグ」を参照しますので、個別にアニメーションを作成しますが、今まで同様に再生することができます )

結合モード、分割モードの違い

今までのアニメーション区間タグ編集を「アニメーション区間の結合モード」、
新しく追加された、アニメーション区間タグ編集を「アニメーション区間の分割モード」と呼びます。

同じアニメーション「 Start 」「 Wait 」「 End 」を使用し、「結合モード」と「分割モード」のアニメーション表示の違いを比較します。

アニメーション区間の「結合モード」を使用した場合のアニメーション作成

アニメーションを一つの繋がったアニメーションとして製作し、アニメーション区間タグによって分類する、今までと同じ方法です。

例：

  • 00 ～ 20 フレームに作られた、合計 20 フレームのアニメーション「 Start 」
  • 20 ～ 70 フレームに作られた、合計 50 フレームのアニメーション「 Wait 」
  • 70 ～ 100 フレームに作られた、合計 30 フレームのアニメーション「 End 」
アニメーション区間の「分割モード」を使用した場合のアニメーション作成

今回追加されたモードです。
アニメーション区間タグによって管理されているアニメーションを、別々に 0 フレーム目から作成します。

例：

  • 00 ～ 20 フレームに作られた、合計 20 フレームのアニメーション「 Start 」
  • 00 ～ 50 フレームに作られた、合計 50 フレームのアニメーション「 Wait 」
  • 00 ～ 30 フレームに作られた、合計 30 フレームのアニメーション「 End 」

 

モードの切り替え

モードを切り替えて、好みの作成方法を選ぶことができます。
また、作業途中のデータも、モードを切り替えて編集 ( ※ ) することができます。

 

「 アニメーション区間タグ編集ウィンドウ」の上部に「モード切り替えボタン」が配置されています。

モード切替ボタン

結合モードボタン

 結合モードを示しています。
分割モードボタン 分割モードを示しています。

「結合モード」から、「分割モード」に切り替えると、アニメーションは「区間タグ」を参照して分割されます。
「分割モード」から、「結合モード」に切り替えると、アニメーションは「区間タグ」を参照して結合されます。

 

参考：

※ 結合操作を行う際、区間範囲外にキーが追加される場合があり、意図しない結果を生み出す可能性があります。
　そのため、基本的には「分割モード」「結合モード」のどちらかに固定した状態で作業していただくことを推奨します。

分割モードでのアニメーション作成

分割モードを使用してのアニメーション作成について説明します。

編集したいペインを選択状態にして「 アニメーション区間タグ編集ウインドウ」から、編集したい範囲を示した「区間タグ」を選択すると、アニメーションウインドウに「区間タグの範囲のアニメーション」が表示されます。
「分割モード」では、区間タグごとにアニメーションカーブを編集します。
区間タグの選択を切り替えることで個別にアニメーションカーブを作成することができます。

分割モードで作成するアニメーションは、全て 0 フレームから始まります。

フィッティング ( 分割モード専用機能 )

フィッティングは、アニメーションカーブの長さに合わせて「区間の幅」を調整する機能です。

「フィッティング」機能は「 カーブエディタウインドウ」のツールバーにある「 フィッティング」ボタンを押すことで実行します。
「 カーブエディタウインドウ」にてアニメーションカーブの長さを変更した時、「 フィッティング」を実行すると「 アニメーション区間タグ編集ウインドウ」の「区間の幅」がアニメーションカーブの長さに調整されます。

フィッティング結果の「区間の幅」は、選択中のタグ区間に含まれる「全てのアニメーションカーブ」から、区間の一番大きいものに合います。

 

参考：
  • タグ区間の開始フレームは、0 フレーム目で固定です。( 分割モードのアニメーションは 0 フレーム目から始まります）
  • タグ区間にアニメーションキーがひとつも無い場合の最小区間幅は 1 です。

作成したアニメーションの拡縮機能 ( 分割モード専用機能 )

作成したアニメーションの拡縮機能は、キーの間隔比率を維持したまま区間の長さを変更する機能です。

作成したアニメーションの拡縮機能は、「区間タグ編集画面」の「フレーム区間」にあるボタンを「 OFF 」「小数点精度」「整数精度」から切り替えます。

「作成したアニメーションの拡縮機能」ボタン
OFF 「作成したアニメーションの拡縮機能」を無効とします。
小数点精度 「作成したアニメーションの拡縮機能」を使用した「拡大 / 縮小」後の結果が小数点単位で設定されます。
整数精度 「作成したアニメーションの拡縮機能」を使用した「拡大 / 縮小」後の結果が整数値に四捨五入されます。
( 四捨五入した結果、キーの数が減る場合があります )

比率連動を有効にした状態で区間の長さを変更すると、キーの間隔比率を維持したままで区間の長さが変化します。

区間幅変更前 ( 区間幅 100 ) 拡縮機能を使用して区間幅 100 を 50 に変更した場合

ドッキングウインドウのロック機能

ドッキングウインドウの状態をロック ( 固定 ) できるようになりました。
「ドッキングを固定する」は、意図せずにドッキングウインドウを動かしてしまう操作を防ぐ、ユーザーの作業を助ける機能です。

「レイアウトエディタウインドウ / メニューバー / ウインドウ / ドッキングを固定する 」

ドッキングを固定する

チェックを入れることで、ドッキングウインドウを固定することができます。

ドッキングウインドウを固定にすることで、作業中に誤ってウインドウを動かしてしまうリスクが解消されます。

解除する場合は、再び「ドッキングを固定する」を選択して、チェックを外します。

「レイアウトエディタウインドウ / メニューバー / ウインドウ / レイアウトウインドウをタブ化しない 」

レイアウトウインドウをタブ化しない

チェックを入れると、レイアウトウインドウを開く際に、個別のウインドウとして開くようになります。
お好みの作業形態に合わせて選択して下さい。

※タブ化の設定を切り替えるには、全てのレイアウトウインドウを閉じる必要があります。
