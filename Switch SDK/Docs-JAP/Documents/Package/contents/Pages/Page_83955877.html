ユーザーデータ設定プラグイン

transform ノード（joint ノード）またはシェーディンググループにユーザーデータを設定します。

使用方法

「Nintendo > Set User Data...」を選択するとウィンドウが表示されます。

Selected Transform Node or Shading Group 欄には、選択されている transform ノードまたはシェーディンググループ（マテリアル）が表示されます。

設定方法

transform ノード（joint ノード）またはシェーディンググループ（shadingEngine ノード）を 1 つだけ選択すると、ウィンドウには選択したノードの現在の設定が表示されます。
（マテリアルノードを選択すると、対応する shadingEngine ノードの設定が表示されます。）

ウィンドウの下部には設定されたユーザーデータのリストが表示されます。

Name ユーザーデータの名前です。
Type

ユーザーデータの型です。
Integer は整数、Float は実数、ASCII は ASCII 文字列、Unicode は Unicode 文字列です。
Size ユーザーデータのサイズ（値の個数）です。
Value

ユーザーデータの内容です。
内容が長い場合は一部を省略して "..." と表示されます。

Add ボタンをクリックすると、新しいユーザーデータを追加できます。リ
ストの項目が選択されている場合は選択した場所に挿入、リストの項目が選択されていない場合はリストの最後に追加となります（リストの一番下にある "-----" をクリックすると選択が解除されます）。

リストの項目を選択して Edit ボタンをクリックするか項目をダブルクリックすると、既存のユーザーデータの名前や内容を変更できます。

リストの項目を選択して Delete ボタンをクリックすると、選択したユーザーデータを削除できます。

リストの項目を選択して Move Up ボタンまたは Move Down ボタンをクリックすると、ユーザーデータの順番を入れ換えることができます。

Display integer value in hexadecimals を ON にすると、整数値のユーザーデータを 16 進数で表示します（入力の際も 16 進数になります）。OFF にすると、10 進数で表示します。

参考：

ウィンドウを閉じずに別のノードを選択すると、別のノードを設定することができます。

Add ボタンまたは Edit ボタンをクリックすると、ユーザーデータ入力ウィンドウが表示されます。

Name

ユーザーデータの名前を入力します。
非 ASCII 文字は使用できません。「_（アンダーバー）」以外の記号も使用できません。
Type

ユーザーデータの型を指定します。
Integer は整数、Float は実数、ASCII String は ASCII 文字列、Unicode String は Unicode 文字列です。
Values

ユーザーデータの値（数値または文字列）を入力します。
改行で区切って複数の値を入力できます（1 行には値を 1 つだけ入力できます）。

OK ボタンをクリックすると、ユーザーデータが追加または変更されます。入力した名前または値が不正な場合は、コマンドラインとスクリプトエディタにエラーが表示されます。

参考：

名前が「nw4f_model_ud」で始まる transform ノードにユーザーデータを設定すると、fmd ファイルの <model> のユーザーデータとして出力されます。
そのノードがロケータなど <bone> として出力されないノードであっても、<model> のユーザーデータは出力されます。
出力対象に「nw4f_model_ud」で始まるノードが複数ある場合、アウトライナでソート順（Sort Order）を「シーンの階層構造（Scene Hierarchy）」にしたときに一番上に表示されるノードが優先されます。

参考：

名前が「nw4f_fsk_ud」で始まる transform ノードにユーザーデータを設定すると、fsk ファイルの <skeletal_anim> のユーザーデータとして出力されます。仕様は nw4f_model_ud と同様です。
