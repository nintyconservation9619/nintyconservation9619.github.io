ツールバー

3DEditor の各機能をアイコンとして表示します。

 

アイコン 対応するメニュー 動作
[ファイル]→[開く] ファイルを開きます。　
[ファイル]→[上書き保存] 選択されたファイルを上書き保存します。　
[ファイル]→[全て指定場所に保存] 読み込まれたファイルを場所を指定して全て保存します。選択すると、場所を指定して保存ダイアログが開きます。
[ファイル]→[全て上書き保存] 3DEditor で編集されたファイルを全て上書き保存します。 編集されていないファイルは保存の対象外になります。
[編集]→[元に戻す] 編集前の状態に内容を戻します。
[編集]→[やり直し] 元に戻した編集をやり直します。
[表示]→[オブジェクトビュー]→[1画面] オブジェクトビューを 1 画面レイアウトにします。　
[表示]→[オブジェクトビュー]→[2画面:上下] オブジェクトビューを 2 画面（上下）レイアウトにします。　
[表示]→[オブジェクトビュー]→[2画面:左右] オブジェクトビューを 2 画面（左右）レイアウトにします。　
[表示]→[オブジェクトビュー]→[3画面:下を分割] オブジェクトビューを 3 画面（下を分割）レイアウトにします。　
[表示]→[オブジェクトビュー]→[3画面:上を分割] オブジェクトビューを 3 画面（上を分割）レイアウトにします。　
[表示]→[オブジェクトビュー]→[3画面:右を分割] オブジェクトビューを 3 画面（右を分割）レイアウトにします。　
[表示]→[オブジェクトビュー]→[3画面:左を分割] オブジェクトビューを 3 画面（左を分割）レイアウトにします。　
[表示]→[オブジェクトビュー]→[4画面] オブジェクトビューを 4 画面レイアウトにします。
[表示]→[プロパティウィンドウ] ファイルビューや、オブジェクトビューで選択されているオブジェクトのプロパティウィンドウを開きます。
-- プルダウンメニューから接続先アプリケーションのプラットフォームを選択します。
[プレビュー]→[アプリケーションに接続] ( 未接続 ) 接続先に指定されているアプリケーションとの通信を開始（接続済みは切断）します。
[プレビュー]→[アプリケーションに接続] ( 接続済み)
[プレビュー]→[接続先を選択] ( PC 接続 ) 接続先を切り替えます。
接続先には開発実機とPCがあり、どちらかに接続中は切り替えできません。
切り替えを行う場合は、アプリケーション（PC や開発実機）との接続を切断してください。
※3DEditor をコマンドラインから起動する際にオプションを指定することでも接続先を切り替えれます。
※接続中は、選択の切り替えができませんのでご注意ください。
[プレビュー]→[接続先を選択] ( 開発実機 接続)
[プレビュー]→[シェーダ最適化] ( 無効状態 ) シェーダ最適化の有効 / 無効を切り替えます。
シェーダ最適化を有効にすると、各モデルで使用されているシェーダプログラムのみランタイムに送信します。
[プレビュー]→[シェーダ最適化] ( 有効状態 )
[プレビュー]→[最適化シェーダ再生成] マテリアルに対して最適化シェーダを再生成する必要がある場合に、手動で最適化シェーダを再生成する場合に使用します。
※シェーダ最適化が OFF の場合は操作できません。
[ツール]→[オートアニメーションバインドモード] オートアニメーションバインドモードの有効 / 無効を切り替えることができます。
メインメニューのツールでも切り替えることができます。
[ヘルプ]→[ヘルプ] 3DEditor ヘルプを開きます。
[ヘルプ]→[バージョン情報] バージョン情報を表示します。
