タイムスライダー

カーブエディタウインドウの「タイムスライダー」に表示される機能を説明します。

「タイムスライダー」は、アトリビュートツリーで選択した要素のカレントフレーム、アニメーション範囲、キーフレームを表示します。

タイムスライダー
レンジスライダー の再生開始フレームと再生終了フレームで設定した範囲が表示されます。

A カレントフレーム カレントフレームが青い線で表示されます。
B カレントフレーム ( 数値 )

カレントフレームが数値で表示されます。

ここに数値を入力することでカレントフレームを移動させることができます。
C キーフレーム キーフレームがオレンジの線で表示されます。
D キーの打たれた範囲

キーが打たれている範囲が薄い青で表示されます。

( 図は、0 フレームから 80 フレームの範囲にキーが打たれていることを示しています )
