はじめに

本ドキュメントでは、似顔絵ライブラリの各機能について説明します。

それに先立ち、本節では似顔絵ライブラリ全体で共通の事項について説明します。

名前空間

似顔絵ライブラリの提供する型、定数、関数などの機能は全て nn::mii:: の名前空間に属します。

特別に指定される場合を除き、nn::mii:: より深い階層の名前空間に属する機能を使用しないでください。これらは予告なく構成や仕様を変更する場合があります。

ヘッダー

nn/mii.h ヘッダーをインクルードすることで似顔絵ライブラリに含まれる全機能を使用することが可能です。

nn/mii.h は以下に配置されています。

$NINTENDO_SDK_ROOT/Include/nn/mii.h

その他のヘッダーファイルは以下に配置されています。nn/mii.h はこれらをインクルードしています。

$NINTENDO_SDK_ROOT/Include/nn/mii/

nn/mii/ ディレクトリーより深い階層のヘッダーファイルをインクルードしないでください。予告なく構成を変更する場合があります。
