機能

マルチエクスポートプラグインを使用すると、シーン内に複数の出力設定を保存して、複数モデルおよび複数アニメーションをまとめて出力できます。

マルチエクスポートプラグインは、次の設定の組み合わせに従って出力します。
この設定の組み合わせを「出力アイテム」と呼びます。

  • 出力する中間ファイル名
  • 中間ファイルを出力するフォルダーのパス
  • マージする fmd ファイルのパス
  • 使用する fdes ファイルのパス
  • 出力するノード名（ディスプレイレイヤ名）
  • 有効にするアニメーションレイヤ名
  • 中間ファイルに対する編集用コメント文
  • アニメーションを出力する範囲
  • ループ再生フラグ
  • 各中間ファイルの出力フラグ
  • 追加オプション
参考：

出力アイテムは、シーン中の "nw4fMultiExp_Settings1" という名前の script ノードに保存されます。

使用方法

メインウィンドウ

「Nintendo > Multi Export...」を選択するとメインウィンドウが表示されます。

ウィンドウには現在のシーンに設定されている出力アイテムのリストが表示されます。

Output File Name 出力する中間ファイル名です。
fdes File 使用する fdes ファイル名です。
Node (Display Layer) 出力するノード名（ディスプレイレイヤ名）です。
Comment 中間ファイルに対する編集用コメント文です。
Start End アニメーションを出力する範囲です。
Loop ループ再生フラグです。
Selection 各中間ファイルの出力フラグです。ON なら次の文字が表示されます。
  M : fmdb ファイル
  T : ftxb ファイル
  S : fskb ファイル
  V : fvbb ファイル
  C : カラーアニメーション中間ファイル（fmab または fclb）
  R : テクスチャー SRT アニメーション中間ファイル（fmab または ftsb）
  P : テクスチャーパターンアニメーション中間ファイル（fmab または ftpb）
  H : fshb ファイル
  N : fsnb ファイル
Options

追加オプションです。 
  • 出力アイテムに出力フォルダーのパス、マージする fmd ファイルのパス、有効にするアニメーションレイヤ名が設定されていれば、リストの項目を選択した際にリストの下に表示されます。
  • Add ボタンをクリックすると、新しい出力アイテムを追加できます。
    リストの項目が選択されている場合は選択した場所に挿入、リストの項目が選択されていない場合はリストの最後に追加となります（リストの一番下にある "-----" をクリックすると選択が解除されます）。
  • リストの項目を選択して Edit ボタンをクリックするか項目をダブルクリックすると、既存の出力アイテムの内容を変更できます。
  • リストの項目を選択して Duplicate ボタンをクリックすると、選択した出力アイテムを複製できます。
  • リストの項目を選択（複数選択可）して Delete ボタンをクリックすると、選択した出力アイテムを削除できます。
  • リストの項目を選択して Move Up ボタンまたは Move Down ボタンをクリックすると、リスト中での出力アイテムの順番を入れ換えることができます（順番は出力される中間ファイルには影響しません）。
  • Column Width で選択したリストの列の幅を、右側のスライダで調整できます。
  • リストの項目を選択（複数選択可）して、Export Selected Items ボタンをクリックすると、選択した出力アイテムを出力します。
  • Export All Items ボタンをクリックすると、すべての出力アイテムを出力します。
  • 複数の出力アイテムが同じフォルダーに同じ名前（同じ拡張子）のファイルを出力する場合、出力時にエラーになります。

リストの項目を選択した状態で右クリックして表示されるメニューから以下の操作ができます。
リストの項目を複数選択した場合、Move Start / End Frame... のみ表示されます。
ユーザーカスタムメニューをコンフィグファイルで追加することもできます。

Select Node 出力アイテムが出力するノードを Maya 上で選択します。このメニューは Node (Display Layer) が空欄でない場合のみ表示されます。
Set Display Layer 出力アイテムが出力するディスプレイレイヤのみ Maya 上で可視にします。このメニューは Node (Display Layer) でディスプレイレイヤ名を指定している場合のみ表示されます。
Load fdes to Scene 出力アイテムの fdes ファイルの設定をシーンにロードして、エクスポートプラグインのオプションウィンドウを表示します（fdes ファイルのパスが空欄の場合は現在のシーンのオプション設定を表示します）。
Set Playback Range 出力アイテムの Start / End Frame の範囲が再生されるように Maya 上のアニメーションの再生範囲を変更します。
Get from Playback Range

Maya 上のアニメーションの再生範囲を取得して、出力アイテムの Start / End Frame に設定します。
Get from Selected Clip タイムエディタで選択したすべてのクリップを含む範囲を取得して、出力アイテムの Start / End Frame に設定します。このメニューは Maya 2017 以降でクリップを作成している場合のみ表示されます。
Move Start / End Frame... 出力アイテム（複数選択可）の Start / End Frame を前後に移動します。ウィンドウで Start / End Frame に加算する移動値（整数値）を指定して、Move ボタンをクリックすると移動します。移動値はマイナスの値も指定できます。Start / End Frame が Range でなければ移動しません。
参考：

fdes File が空欄の場合、シーンにエクスポートプラグインのオプション設定が保存されていないと出力時にエラーになります。
「Nintendo > Export with Setting...」でオプションウィンドウを表示して、左上のメニュー「編集 > 設定の保存（Edit > Save Settings）」を選択するとオプション設定が保存されます。

出力アイテム入力ウィンドウ

メインウィンドウで Add ボタンまたは Edit ボタンをクリックすると、出力アイテム入力ウィンドウが表示されます。

Output File Name

出力する中間ファイル名（拡張子を除く）を入力します。

空欄の場合は現在のシーン名が中間ファイル名となります。
右にある Scene ボタンをクリックすると現在のシーン名が入力されます。
Node ボタンをクリックすると Maya 上で選択中のノード（複数選択されている場合は階層が一番上のもの）の名前が入力されます。
Output Folder 中間ファイルを出力するフォルダーを入力します。
空欄の場合、fdes ファイルの設定（fdes File も空欄の場合は現在のシーンの設定）を使用します。
フルパスでない場合、現在のプロジェクトフォルダーからの相対パスとみなします。
パスには環境変数（%NAME% または $NAME のように記述）が使用できます。
Merge fmd File ON にして fmd ファイルのパスを指定すると、指定した fmd ファイルをマージします。
パスが空欄の場合、上書きする出力先ファイルをマージします。
パスがフルパスでない場合、現在のプロジェクトフォルダーからの相対パスとみなします。
パスには環境変数（%NAME% または $NAME のように記述）が使用できます。
fdes File 使用する fdes ファイルのパスを入力します。
空欄の場合、現在のシーンのオプション設定で出力します。
フルパスでない場合、現在のプロジェクトフォルダーからの相対パスとみなします。
フルパスで入力した場合、現在のプロジェクトフォルダー以下のパスであれば OK ボタンクリック時に相対パスに変換されます。
パスには環境変数（%NAME% または $NAME のように記述）が使用できます。
下にある Recent fdes ボタンをクリックすると、最近入力した fdes ファイルの履歴から選んで入力できます。
Load fdes to Scene ボタンをクリックすると、入力されている fdes ファイルの設定をシーンにロードして、エクスポートプラグインのオプションウィンドウを表示します（パスが空欄の場合は現在のシーンのオプション設定を表示します）。
Node (Display Layer) 特定のノードだけ出力する場合に、出力するノードのノード名を入力します（入力したノードの下の階層のノードも出力されます）。
出力するディスプレイレイヤの名前を入力することも可能です（ディスプレイレイヤに含まれるノードの下の階層のノードも出力されます）。
複数のノード名（ディスプレイレイヤ名）をスペースで区切って記述できます。
fdes ファイルがシーン全体を出力する設定であれば何も入力する必要はありません。
また、fdes ファイルがシーン全体を出力する設定であっても、ノード名（ディスプレイレイヤ名）を入力すると、入力したノード（ディスプレイレイヤ）だけが出力されます。
右にある Get ボタンをクリックすると、Maya 上で選択中のノードの名前が入力されます。
Select ボタンをクリックすると、入力したノードを Maya 上で選択します（ディスプレイレイヤ名を入力している場合はディスプレイレイヤに含まれるノードを選択します）。
下にある Get Display Layer ボタンをクリックすると、Maya 上で可視でテンプレートでないディスプレイレイヤの名前が入力されます。
Set Display Layer ボタンをクリックすると、入力したディスプレイレイヤのみ Maya 上で可視にします。
Anim Layer 特定のアニメーションレイヤのみ有効にして出力する場合に、有効にするアニメーションレイヤ名を入力します。
複数のアニメーションレイヤ名をスペースで区切って記述できます。
入力したアニメーションレイヤに親があれば親のアニメーションレイヤも有効になります。
ベース以外のアニメーションレイヤを無効にするには BaseAnimation を入力します。
空欄の場合、現在有効なアニメーションレイヤを出力します。
右にある Get ボタンをクリックすると、Maya 上で有効なアニメーションレイヤ名が入力されます。
Set ボタンをクリックすると、入力したアニメーションレイヤのみ Maya 上で有効にします。
Comment

中間ファイルに対する編集用コメント文を指定します。
「;（セミコロン）」、「'（シングルクォート）」、「"（ダブルクォート）」は使用できません。

空欄の場合、fdes ファイルで指定した編集用コメント文が反映されます。
Start / End Frame アニメーションを出力する範囲（開始フレームと終了フレーム）を指定します。
指定方法はエクスポートプラグインの Start / End Frame オプションと同じです。
下にある Get ボタンをクリックすると、Maya 上のアニメーションの再生範囲の値が入力されます。
Set ボタンをクリックすると、入力した範囲が再生されるように Maya 上のアニメーションの再生範囲を変更します。
Get Clip ボタンをクリックすると、タイムエディタで選択したすべてのクリップを含む範囲の値が入力されます（このボタンは Maya 2017 以降でクリップを作成している場合のみ表示されます）。 
Loop ループ再生フラグを指定します。
ON にすると、ループ再生するデータとして出力します。
OFF にすると、一度だけ再生するデータとして出力します。
Output File Selection 各中間ファイルの出力フラグを指定します。
Get from fdes ボタンをクリックすると、fdes File で指定した fdes ファイルで出力する設定になっている中間ファイルのみ ON にします（パスが空欄の場合は現在のシーンのオプション設定で出力される中間ファイルのみ ON にします）。
Additional Options

追加オプションを NintendoExportByFdes3 の第 3 引数と同じ書式で指定します。

Edit ボタンをクリックすると、追加オプション編集ウィンドウが表示され、UI で追加オプションを指定できます（詳細は後述）。
1 行に複数のオプションを記述することも、改行して複数行に記述することも可能です。
追加オプションは fdes ファイルのオプション設定より優先されます。
たとえば、fdes ファイルの Frame Precision の設定が 1.0 になっている場合に、0.5 で出力するなら、次のように記述します。

-fp 0.5

出力後にバッチファイルを実行するなら、次のように記述します。

-poes 'system "D:/data/post_export.bat"'

（引数が半角スペースを含む場合は「'（シングルクォート）」で囲みます。）

OK ボタンをクリックすると、出力アイテムが追加または変更されます。

参考：

出力アイテムの fdes ファイルがプリセットを使用する場合、出力アイテムの設定よりプリセットファイルでロックされている設定が常に優先されます。
たとえば、プリセットで Output Folder が「D:/data/」でロックされていれば、出力アイテムの Output Folder が「D:/tmp/」であっても、「D:/data/」に出力されるので注意してください。
Merge fmd File、Start / End Frame、Loop、Output File Selection などについても同様です。


追加オプション編集ウィンドウ

出力アイテム入力ウィンドウで、Additional Options 欄の Edit ボタンをクリックすると、追加オプション編集ウィンドウが表示されます。

参考：

出力アイテム入力ウィンドウの fdes File が空欄の場合、シーンにエクスポートプラグインのオプション設定が保存されていないと追加オプション編集ウィンドウが表示されません（エラーになります）。
「Nintendo > Export with Setting...」でオプションウィンドウを表示して、左上のメニュー「編集 > 設定の保存（Edit > Save Settings）」を選択するとオプション設定が保存されます。


グレー表示のオプションの横にあるペンアイコンを右クリックして、メニューから Edit を選択すると編集可能状態になります。
オプションを指定して OK ボタンをクリックすると、Additional Options が変更されます。

編集可能状態のオプションを右クリックして、メニューから Use fdes を選択するとグレー表示に戻り、fdes ファイル（または現在のシーンのオプション設定）で指定した値が使用されるようになります。

ウィンドウ左下の Show Locked Options を ON にすると、fdes ファイルでロックされているオプションが表示されます。
ロックされているオプションは編集できません（錠前アイコンが表示されます）。
fdes ファイルで指定されている値を確認したい場合に表示してください。

出力アイテム入力ウィンドウに専用の UI があるオプションは、追加オプション編集ウィンドウでは表示されません。
Additional Options 欄にそのようなオプションが入力されていた場合、
追加オプション編集ウィンドウ下部の Other Options 欄に反映されます。
Other Options 欄では NintendoExportByFdes3 の第 3 引数と同じ書式でオプションを指定します。

Merge Animation Folder の ON / OFF を指定せずに Merge Animation Name だけ指定した場合、アニメーション中間ファイルのマージは実行されます。
Divide Submesh の ON / OFF を指定せずに Divide Submesh のモード指定オプションだけ指定した場合、サブメッシュ分割は実行されます。
Polygon Reduction の ON / OFF を指定せずに Polygon Reduction のモード指定オプションだけ指定した場合、ポリゴンリダクションは実行されます。

複数の出力アイテムの設定を一括置換

メインウィンドウでリストの項目を選択（複数選択可）して、「Replace > Replace Selected Items...」メニューを選択すると出力アイテム置換ウィンドウが表示されます。
出力アイテム置換ウィンドウでは、選択した出力アイテムのオプション設定を一括置換できます。
オプションの値にプレフィックスやサフィックスを追加することもできます。

Target で対象のオプションを指定します。
Mode で置換モードを指定します。

Replace 特定の文字列（From）をすべて別の文字列（To）で置換します。
Replace (RegExp)

正規表現で指定した文字列（From）を別の文字列（To）で置換します。

元の文字列中の複数部分がマッチする場合、最初の部分のみ置換します。
Overwrite Value で指定した文字列で完全に上書きします。
Add Prefix 元の文字列の先頭に Prefx で指定した文字列を追加します。
Add Suffix 元の文字列の末尾に Suffix で指定した文字列を追加します。

Target が Output File Selection の場合、チェックボックスで各中間ファイルの出力フラグを指定します（常に上書きとなります）。
Apply ボタンをクリックすると置換が実行されます。

Mode が Replace (RegExp) の場合、たとえば From が「^human」なら元の文字列の先頭の「human」のみ置換します。From が「run$」なら元の文字列の末尾の「run」のみ置換します。

マルチエクスポート設定のエクスポート／インポート

メインウィンドウの「Settings」メニューから以下の操作ができます。

Import Settings from ndmes File... 出力アイテムの設定を ndmes ファイルからインポートします。
既存の出力アイテムが存在する場合、それらの後に追加します。
Export All Settings to ndmes File... すべての出力アイテムの設定を ndmes ファイルにエクスポートします。
Export Selected Settings to ndmes File... リストで選択（複数選択可）した出力アイテムの設定を ndmes ファイルにエクスポートします。

fdes ファイルのパスが空欄の出力アイテムが存在する場合、シーンの単体エクスポート設定も ndmes ファイルにエクスポートします。

インポートする ndmes ファイルにシーンの単体エクスポート設定が含まれる場合、その設定もインポートするかどうかをダイアログで選択します。
シーンの単体エクスポート設定をインポートした場合、既存の出力アイテムに fdes ファイルのパスが空欄のものがあれば、そのエクスポート設定にも影響することに注意してください。

バッチファイルによるマルチエクスポート

Windows のバッチファイルを使用して、シーンに保存された出力アイテムをすべて出力するには次のようにします。

call StartMaya-2017.bat -batch -proj "C:/Project1" -file "Scene1.mb" -command "NintendoMultiExportDirect"

指定した出力アイテムのみ出力するには、次のようにします。

call StartMaya-2017.bat -batch -proj "C:/Project1" -file "Scene1.mb" -command "NintendoMultiExportDirect2 \"out1,out3\""

NintendoMultiExportDirect2 の引数には、出力する出力アイテムの Output File Name を指定します。複数の Output File Name を「,（カンマ）」で区切って指定できます。同じ Output File Name の出力アイテムが複数存在する場合はすべて出力されます。
