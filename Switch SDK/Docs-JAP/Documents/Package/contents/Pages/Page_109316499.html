概要

InitializeSdevWin は開発機 SDEV のファームウェアを初期化するためのGUIツールです。

このツールのコマンドライン版が InitializeSdev です。自動テストで用いる場合はそちらをご利用ください。

パッケージ

InitializeSdevWin は NintendoSDK NX Addon パッケージに含まれています。

配置

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/InitializeSdevWin.exe

なお動作のためにはこの exe ファイル以外にも依存するファイルがいくつかあります。

注意点

本ツールで SDEV を初期化すると、NX アプリケーションを開発機にインストールして実行する でインストールしたアプリケーションや、セーブデータがすべて削除されます。

オプション形式

この実行ファイルに指定する必要のあるコマンドラインオプションはありません。

exe ファイルを引数なしで実行するとウィンドウが表示されます。

表示画面

メインウィンドウ

InitializeSdevWin を起動すると表示される画面です。

使い方は 1 で初期化する開発機を指定し、2 でファームウェアバージョンを指定し、3の初期化開始ボタンを押すというものです。
初期化開始ボタンを押すとログウィンドウが表示され初期化が開始されます。

なお1の画面に初期化したい開発機が表示されない場合は“Re-Scan”ボタンを押して再検索してください。
またメニューの"Tools"→"Change network settings" を選ぶと“ネットワーク接続”が表示されますので NIC の設定が変更できます。

コンテキストメニュー

SDEV 接続リストに表示されている行を右クリックするとコンテキストメニューが表示されます。

  • Start initialization
    • 指定した SDEV の初期化を開始します。ウィンドウ下にある同名のボタンと同一の動作をします
  • Blink LEDs
    • 指定した SDEV 本体裏面の LED を点滅させます。一覧にシリアル番号が表示されないことで本体を特定できない場合にお使いください。
  • Write serial number to SDEV
    • 指定した SDEV にシリアル番号情報を書き込みます。

ログウィンドウ

初期化を実行すると表示される画面です。

プログレスバーで経過が表示され、ログウィンドウにログが表示されます。
初期化が完了すると画面左下の結果表示欄に結果が表示され、画面右下の終了ボタンが有効になります。

その他機能の説明

ファームウェア種類選択コンボボックスからファームの種類を選択することで後述する機能も使えます。

製品機に近いファームによる初期化

ファームウェア種類選択コンボボックスで "+ Prod" を含む項目を選択すると製品機に近い状態になるファームを使って初期化します。

また "+ ProdWithLog" を含む項目を選択すると microSD カードからのログ出力機能付きの製品機に近いファームで初期化します。

注意：

「製品機に近いファームによる初期化」機能は SDEV MP のみで利用できます。それ以外の SDEV 本体で実行するとエラーになりますのでご注意ください。
