UI 説明

エディタのユーザーインターフェイス（UI）各部の概要を説明します。

メインウィンドウ EffectMaker のウィンドウを構成する、各部位の名称と役割について説明します。
プロパティウィンドウ EffectMaker のノードを設定する為の「プロパティウィンドウ」について説明します。
エフェクトブラウザ EffectMaker で使用するファイルを閲覧、検索する「エフェクトブラウザ」について説明します。
カーブエディタウィンドウ EffectMaker のエミッタ時間アニメを作成、編集する「カーブエディタウィンドウ」について説明します。
履歴ウィンドウ エフェクトの編集履歴を一覧表示し、履歴間を往き来できる「履歴ウィンドウ」について説明します。
検索ウィンドウ 指定したフォルダのエフェクトファイルから、使用している機能を検索し、設定を一括変換できる「検索ウィンドウ」について説明します。
テクスチャの詳細ウィンドウ エフェクトで使用するテクスチャの詳細情報を表示する「詳細ウィンドウ」について説明します。
EffectViewer EffectMaker 用ビューア「EffectViewer」 について説明します。

関連リンク EffectMaker で使用できるキーボードショートカットは「ショートカット一覧」 で確認できます。
