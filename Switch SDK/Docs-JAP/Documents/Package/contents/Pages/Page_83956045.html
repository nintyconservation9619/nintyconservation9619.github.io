ポップアップスライダー

3DEditor は、ポップアップスライダーを使用して入力値をアナログ的に編集します。
関連リンク スライダーの操作に対応したショートカットキーについては、こちら を参照してください。

A スライダーの範囲内で、数値を直接入力して設定します。
B ボタンを押すと、スライダーウィンドウがポップアップ表示されます。
スライダーウィンドウ以外の領域をクリックするとウィンドウは閉じられます。
C 値をスライダーウィンドウが表示された時の値に戻します。
D 設定する範囲を１段階縮小します。
この操作は、設定範囲が可変な値の編集時にのみ可能です。
－（マイナス）キーでも同様な操作が行えます。
E 設定する範囲を１段階拡大します。
この操作は、設定範囲が可変な値の編集時にのみ可能です。
＋（プラス）キーでも同様な操作が行えます。
F 値をスライダーで変更します。
スライダーの領域内をマウスでポイントすると、ポイントした位置に対応する値がツールチップ表示されます。

スライダー範囲を縮小した場合

ポップアップスライダーの範囲縮小機能を使用した場合に、既に設定してある値よりも範囲が狭くなった場合は、値は変更されず、スライダーに「赤枠」が表示されます。
