ウインドウペインのプロパティウインドウから設定する「フレームサイズ」を他のアニメーションと同様に設定できます。
カーブエディタウインドウにてウインドウペインの「フレームサイズ」アニメーションを編集します。

ウインドウペインのプロパティウインドウ、「カスタムフレームサイズ」

カスタムフレームサイズに数値入力をして、フレームサイズを変更します。

テクスチャサイズに制限されない自由な大きさのフレームを設定することができます。

「設定」アイコンをクリックすると、カスタムフレームサイズが入力できるようになります。

( アイコンをクリックすると、アイコンは「リセット」に切り替わります。)

「リセット」アイコンをクリックすると、カスタムフレームサイズが初期値に戻ります。

( アイコンをクリックすると、アイコンは「設定」に切り替わります。)

「上、下、左、右」は、それぞれのフレームの位置を指します。

それぞれの位置のフレームに対して、カスタムフレームサイズを設定します。
上下を同期 チェックを入れると、上下のカスタムフレームサイズ入力が同期します。
左右を同期 チェックを入れると、左右のカスタムフレームサイズ入力が同期します。

他のアニメーションと同様に、フレームサイズにアニメーション設定ができます。

  • フレームサイズ / 下
  • フレームサイズ / 左
  • フレームサイズ / 右

カーブエディタウインドウからアニメーション設定をした場合、プロパティウインドウの「カスタムフレームサイズ」は自動的に「設定」に切り替わります。

部品上書き機能「テクスチャだけ上書き」に対応

部品上書き機能、「テクスチャだけ上書き」が対応されました !!
「テクスチャだけ上書き」は、「カラー補間だけ上書き」と同様の機能で、テクスチャだけを上書きする機能です。

「テクスチャだけ上書き」を使用すると、カラー補間等は上書きせずにテクスチャだけを上書きします。

今までは、部品上書き機能を使用してテクスチャの上書きをしたい場合、「ピクチャペインを上書き」「ウインドウペインを上書き」等を使用する必要があり、カラー補間も、その他のマテリアル情報も、全てが上書き対象となっていました。
このために、ボディレイアウト側の部品ペインに設定されていた「カラー補間」等の設定が ( 上書き対象に含まれるため ) そのまま使用できない状態になっていました。
「テクスチャだけ上書き」を使用することで、テクスチャの上書き以外の設定を、ボディレイアウト側の部品ペイン設定のまま、使用することができます。

部品上書き機能「テクスチャだけ上書き」を使用することで、作業をより効率的に行う事ができます。

プロパティウインドウに追加された「テクスチャだけ上書き」
A   テクスチャだけ上書き 部品上書き機能を使用して「テクスチャだけ上書き」にするチェックボックスです。
B 上書き設定を変更

「 テクスチャだけ上書き」のチェックを入れたら、「上書き設定を変更」をクリックし、設定変更を決定します。

C に表示される階層構造が切り替わります。
C 階層構造 「テクスチャ」項目から、上書き内容の設定を行います。
