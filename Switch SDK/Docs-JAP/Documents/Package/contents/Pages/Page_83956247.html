ピクチャペインを利用する

画像を画面に配置するために使用される、ピクチャペインに関して説明をおこないます。

テクスチャを読み込む

ピクチャペインを配置するには、まずピクチャペインが使用する画像ファイルをテクスチャとして登録する必要があります。
テクスチャの登録はテクスチャウィンドウで行います。

テクスチャウィンドウの表示

まず、レイアウトを新規作成します。
次に、[ メニュー ] → [ ウィンドウ ] → [ テクスチャ ] を選びテクスチャウィンドウを表示状態にします。

画像ファイルの読み込み


テクスチャウィンドウの 読み込みボタン を押します。
画像ファイルを選択する、読み込みダイアログが表示されます。
画像ファイルのフォーマットは TGA ファイルを使用します。
ファイルを選択すると、テクスチャウィンドウ左側のリストビューに、読み込んだ画像 ( B ) が表示されテクスチャの読み込みが完了します。

なお、テクスチャウィンドウにファイルをドロップすることでも同じ結果を得られます。

テクスチャを指定してピクチャペインを作成する


テクスチャウィンドウのリストビューの画像をクリックすると テクスチャ画像が選択状態 ( A ) になります。
テクスチャ画像を選択すると ペイン作成ボタン が有効になるので、ボタンを押してペインを生成します。


レイアウトウィンドウにテクスチャを表示したピクチャペインが作成されたことが確認できます。

プロパティを変更する


ピクチャペインを選択して、プロパティウィンドウを表示します。
プロパティウィンドウ左側のツリーから、ピクチャペイン情報 ( A ) を選択します。
右側のページ(B)が ピクチャペイン情報編集ページ に切り替わります。

以下のような変更を行うことが可能です。

  • 使用するテクスチャを変更する
  • 頂点カラーを変更してペイン 4 隅の頂点の色を変更する
  • テクスチャ座標を変更して画像の貼り方を変える
