カスタムシェーダ

ゲームアプリで用意したプログラマブルシェーダを登録したり、登録したシェーダの内容を確認できます。
この設定項目は主にゲームプログラマが利用します。

カスタムシェーダ定義ファイルパス

「 カスタムシェーダ定義ファイルパス」は、ゲームプロジェクトで使用するシェーダ定義ファイルを登録する機能です。
ゲームアプリケーション独自のシェーダを記述したファイルが使用できます。

シェーダコンパイル時の追加設定

「 シェーダコンパイル時の追加設定」は登録したシェーダ定義ファイルの内容を表示します。

追加のインクルードディレクトリ

プロジェクト設定ファイルに記述された、カスタムシェーダ内で使用可能なインクルードパスがここに表示されます。

追加のプリプロセッサ定義

プロジェクト設定ファイルに記述された、シェーダコンパイル時に追加で与えられるプリプロセッサ定義がここに表示されます。

プリプロセスを有効にする

プロジェクト設定ファイルに記述された、シェーダコンパイル時にプリプロセスしたソースコードを渡すかどうかの設定を表示します。

カスタムシェーダ設定

「 カスタムシェーダ設定」は、シェーダ定義ファイル内容を表示する機能です。
