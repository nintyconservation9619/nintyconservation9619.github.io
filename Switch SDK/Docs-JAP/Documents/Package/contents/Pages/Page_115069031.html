Maya プラグイン

 Maya プラグイン に新しく追加された機能について紹介します。

マルチエクスポートプラグインのウィンドウにユーザーカスタム UI 機能

マルチエクスポートダイアログを、ユーザーがカスタマイズできるようになりました！
これにより、「マルチエクスポートダイアログに、バージョン管理システムのサブミットボタンを追加する」、といったカスタマイズが可能になりました。

マルチエクスポートプラグインで、出力アイテムリストの列幅調整機能

マルチエクスポートダイアログの出力アイテムリストの幅が固定で、項目によっては入りきらないことがありましたが、幅を UI 上で変えられるようになりました。
ウィンドウ右上の Column Width のプルダウンメニューとスライダーを使って、それぞれの列幅を調整できます。

エクスポートプラグインのエラー（警告）発生時に、詳細な説明の表示

Maya を日本語の UI で使用している場合は、以下のように説明が日本語で表示されるようになりました！

英語 UI によるエラー表示例 日本語 UI によるエラー表示例
Make sure that you clear the In-Between option when creating blend shapes ブレンドシェイプ作成時にインビトウィーンオプションを OFF にしてください

3DEditor に新しく追加された UI について紹介します。

カーブエディタでのキーの表示の改善

カーブエディタで、キーの状態が分かりやすくなるように表示を改善しました！
以下は改善箇所の説明です。

キーの色をカーブと統一

以下のように、キーがカーブの色になります。
これにより、表示されているキーがどのカーブのものか一目でわかるようになりました！

キーを選択時の強調表示

以下のように、キーを選択したときにキーが白く表示されます
これにより、キーの選択状態が分かりやすくなりました！

近接キーの強調表示

以下のように、複数のキーが近接したときにそれらのキーが円で囲まれます。
これにより、複数のキーが同一時間に重なっているミスがあったときに見つけやすくなりました！

マテリアルプロパティウィンドウのシェーダページで float3, float4 のコントロールをカラー用のコントロールに切り替えられるように

マテリアルの設定において 3 個また 4 個の値が並ぶ float3、float4 のコントロールが、値コントロールと、カラーコントロールで切り替えられるようになりました！
これにより、カラーピッカーで直感的に色指定できる UI も利用できます。
以下のようにラベルをマウス右クリックしたメニューから「カラーコントロールを表示」を選択することで切り替えます。

チュートリアル用のサンプルビューア

チュートリアル用のサンプルビューアが同梱されるようになりました！

今までは、モデルやアニメーションをプレビューするためにビューア環境の準備が必要で、デザイナが独力で試すことが難しい状況でしたが、サンプルビューアを使うことで簡単にチュートリアルを始められるようになりました。

サンプルビューアは、3DEditor と接続して、モデルやアニメーションのプレビューが見られます。

マテリアルアニメーション中間ファイルフォーマットの対応

マテリアルアニメーション中間ファイルフォーマット(fma)の仕様が追加されました。
将来、fsp(fcl、fts)、ftp、fvm フォーマットは廃止されますので、これらのフォーマットを fma フォーマットに変換する必要があります。
