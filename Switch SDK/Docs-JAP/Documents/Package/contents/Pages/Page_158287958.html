Maya プラグインに頂点カラーの RA 成分のみ出力する機能を追加

頂点カラーの出力成分指定で 5 を指定することにより、RA 成分のみを出力できるようになりました！
使用例として、頂点カラーへのアンビエントオクルージョン情報の焼き付けで、RGB 成分に同じ値が入るケースでは R 成分のみを出力できる為データ容量を削減できます。

使用方法として、例えば、カラーセット名に下記のような名前をつけると RA 成分のみ出力されます。

 nw4f_color0_5f

file ノードのフレーム拡張子アトリビュートに直接キーを設定しない場合でもテクスチャパターンアニメーションが出力可能に

テクスチャパターンアニメーション設定プラグインが改善されました！
file ノードのフレーム拡張子アトリビュートに直接キーを設定しない場合(ユーティリティノードを経由する場合など)でもテクスチャパターンアニメーションを出力可能になりました。

例えば以下のように「乗算除算ノード」等のユーティリティ系ノードと「file ノード」のイメージ番号（フレーム拡張子アトリビュート）を接続している場合でもエクスポートできます。
