UV 割り当て

ここでは、ストライプに貼るテクスチャの UV 割り当てを設定する機能「UV 割り当て」について説明します。

概要

「UV 割り当て」は、ストライプに対するテクスチャの貼り方を設定できる機能です。
「テクスチャリング」の設定が「描画範囲に合わせて貼る」のとき、テクスチャの UV 割り当てが動作します。
関連リンク 「テクスチャリング」についてはこちらを参照してください。

以下は「 設定項目」の概要です。

UV 割り当て

ストライプの UV 割り当てを以下のプリセットから選択します。

履歴毎に固定割合 1 つのストライプ全体にテクスチャを貼ります。
履歴距離に依存 ストライプに使用する「履歴」の距離に依存してテクスチャを貼ります。

設定例

以下は「UV 割り当て」のプリセットを設定した場合の比較画像です。
