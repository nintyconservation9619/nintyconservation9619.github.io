IconBody

概要

nn::mii::IconBody は、体つきアイコンを描画するための体部分のシェイプデータを保持するクラスです。体つきのアイコンを作成したい場合にこのクラスが必要となります。

リソースのロード

似顔絵リソースの中には体部分のデータは含まれていません。別途ファイルとロードする必要があります。

以下のファイルをロードしてください。

$(NINTENDO_SDK_ROOT)/Resources/Mii/Common/iconbody/IconBody.dat

上記ファイル内に、男女両方の体モデルのデータが含まれています。

初期化/終了処理

初期化と終了処理の注意点に関して説明します。各関数の API リファレンスも合わせて参照してください。

初期化処理

nn::mii::IconBody::Initialize() 関数を使用して初期化を行います。

初期化時に必要となる体のリソースデータは、あらかじめ初期化前にメモリ上にロードしておく必要があります。

初期化用のメモリのサイズとアライメントは nn::mii::IconBody::CalculateMemorySize() と nn::mii::IconBody::CalculateMemoryAlignment() 関数を使用することで取得できます。

初期化用のメモリプールのサイズとアライメントは nn::mii::IconBody::CalculateMemoryPoolSize() と nn::mii::IconBody::CalcualteMemoryPoolAlignment() 関数を使用することで取得できます。初期化時に渡すメモリプールのプロパティが CpuCached な場合、初期化後にメモリプールのフラッシュを呼ぶ必要があります。

初期化時に渡したメモリ、メモリプールは終了処理を呼ぶまで保持する必要があります。体のリソースデータは初期化後に破棄することが可能です。

終了処理

nn::mii::IconBody::Finalize() 関数を呼ぶと終了処理を行います。終了処理が行われた後は初期化に使用したメモリとメモリプールの破棄を行うことができます。

DrawParam

nn::mii::IconBody::GetDrawParam() 関数を使用することで、体の描画に必要がパラメータが取得できます。CharInfo を渡して内部に設定されている性別に応じたパラメータを取得することができます。直接性別を指定することも可能です。
