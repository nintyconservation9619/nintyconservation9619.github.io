はじめに

本ドキュメントでは、HID のツールについて説明します。


配置

HID のツールは以下に配置されています。

Tools/Hid
TargetTools/NX-NXFP2-a64/ControllerFirmwareUpdater/Release


ツール一覧

HID が提供するツールの一覧です。

ツール名 種別 パス 説明
HID Shell Windows 用ツール (.exe) Tools/CommandLineTools/Hid HID ライブラリから扱うことのできるデバイスについて、 PC 上からの、ロギング、プレイバック、オートパイロットを可能にします。
Input Director Windows 用ツール (.exe) Tools/Hid/InputDirector

PC 上の HID 入力を開発機へ転送します。 転送された値は開発機の HID 入力として扱われます。
振動コンバータ Windows 用ツール (.exe) Tools/Hid/VibrationConverterConsole

音声ファイルを元にして、振動バイナリファイルを作成します。
振動図鑑

ターゲット向けツール (.nsp)

 TargetTools/NX-NXFP2-a64/VibrationCollection/Release

振動図鑑は振動と音声を同時に再生可能な ターゲット上で実行できるプレビューツールです。

振動図鑑にはプリセットとして振動ファイルと音声ファイルが用意されています。メニューに表示された振動と音声を選択することで、両者を組み合わせたときにどのような感触になるのか、簡単に試すことが可能です。
振動事例集 ターゲット向けツール (.nsp) TargetTools/NX-NXFP2-a64/VibrationDemo/Release HD振動の効果的な使い方を紹介する EDEV / SDEV 上で実行できるデモです。
