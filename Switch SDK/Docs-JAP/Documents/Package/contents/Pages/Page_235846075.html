HlslCrossCompiler

HlslCrossCompiler の解説ページはこちらです。本ページでは NX 固有の機能について説明します。

オプション

--hlslcc-option-flags

NX では下記のフラグを使用する事ができます。

フラグ名 フラグ値 説明
HLSLCC_FLAG_SEPARATE_TEXTURE_TYPES 0x20000000 if set, this shader uses sampler/texture objects

各フラグの詳細

HLSLCC_FLAG_SEPARATE_TEXTURE_TYPES

参考：

NintendoSDK 5.0.0 から利用できる機能です。

GL_KHR_vulkan_glsl のセパレートサンプラー、テクスチャー相当の機能を使用した GLSL ソースコードにコンバートします。

uniform sampler、uniform texture を使用するため HLSL に近い SamplerState、Texture の取り回しが可能になります。
