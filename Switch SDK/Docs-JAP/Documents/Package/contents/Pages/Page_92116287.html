エフェクトブラウザ

EffectMaker で使用するファイルに特化した閲覧や検索機能が利用できる「エフェクトブラウザ」について説明します。
関連リンク EffectMaker で使用するファイルについては 「中間ファイル」 を参照してください。

A アドレスバー ファイルパスを表示、入力できます。「検索」タブでは検索基準となるパスを指定します。
B タブ

「 タブ」はエフェクトブラウザで利用できる機能をマウス左ボタンクリックで切り替えます。
C ナビゲーションペイン

「 タブ」の機能目的ごとの情報や設定項目を表示します。
D 検索ボックス 「 テキストボックス」にキーワードを入力することで、ファイルリストに表示するファイルを絞り込みます。
E ツールバー ファイル操作に関するツールのボタンが配置されています。
F オプション 「ファイルリスト」でのフィルタリング方法やファイルの表示方法を変更できます。
G ファイルリスト  「ナビゲーションペイン」の選択または設定結果に応じたフォルダやファイルを一覧表示します。


A アドレスバー

  ファイルパスの機能について説明します。

「 前に戻る」と 「 次に進む」のボタン は、マウス左ボタンクリックにより、アドレスバーに入力したパスの履歴を移動できます。
 「 テキストボックス」右端の「履歴表示ボタン」は、マウス左ボタンクリックによりリストアップされるパスの履歴から直接移動先パスを選択できます。
「 フォルダーの参照ボタン」はマウス左ボタンクリックでフォルダが選択できるダイアログが開きます。

B タブ

エフェクトブラウザの機能です。
「 タブ」をマウス左ボタンクリックで切り替えます。以下の機能が利用できます。

コンピュータ

階層構造で表示されたビューを利用してフォルダやファイルにアクセスできます。
また、よく使用するフォルダを「お気に入り」として登録できます。
お気に入り ビューで「お気に入り」の表示や登録、編集ができます。
また「お気に入りフォルダ」により、お気に入りをグループ化できます。
検索 エフェクトに関するデータを設定した条件に従って検索できます。

C ナビゲーションペイン

「 タブ」の機能ごとに情報や設定項目を表示します。
以下は「 タブ」機能ごとの表示内容です。

コンピュータ 階層をツリー状に表示します。
お気に入り 「 お気に入りフォルダ」や「 お気に入り」を表示します。
検索 検索条件の設定項目を表示します。

またファイルの種類によっては以下のように選択したファイルの情報や画像をサムネイルとして表示します。

テクスチャファイル プリミティブファイル
選択したテクスチャファイルの情報と画像をサムネイル表示します。
関連リンク 表示される情報は こちら を参照してください。

選択したファイルと同名の .jpg または .png 画像があるとき、
その画像をサムネイル表示します。
参考：

EffectViewer では、描画結果を .png ファイルとして出力する機能があります。
この機能を利用し、EffectMaker に読み込んだ プリミティブファイルをサムネイルとして利用できます。
関連リンク 詳細については、 「EffectViewer の操作」 を参照してください。

参考：

プリミティブのサムネイル画像は以下の方法で用意できます。

  1. プリミティブファイルを「 EffectMaker」 の「 モデルノード」に登録します。
  2. 「 EffectViewer （PC）」を起動し、登録したモデルが表示されていることを確認します。
  3. EffectViewer ウィンドウ上でキーボードショートカット Rキー を押すことでキャプチャできます。
  4.  エクスプローラで、起動したビューアの場所に「 ScreenCap」があります。
  5. フォルダ内にキャプチャ画像ファイル（.png）が生成されています。
  6. 生成された画像ファイルをプリミティブファイルと同じ場所に同じ名前で配置します。
  7. 「 EffectBrowser」 のファイルリストでプリミティブファイルを選択し、サムネイルが表示されることを確認してください。

D 検索ボックス

「 テキストボックス」に入力したキーワードでファイルを絞り込み、「ファイルリスト」に表示させます。

参考：

入力するテキストにはワイルドカードが使用できます。
例えばお気に入りの「 VfxSample」で「*.eset」と入力すると、「ファイルリスト」には「 .eset ファイル」のみが表示されます。

E ツールバー

ファイルやフォルダに対して操作可能なツール群です。マウス左クリックすることで実行されます。
以下は各ツールの説明です。タブによって使用できるなツールが異なります。

エクスポート 選択しているファイルをワークスペースにエクスポートします。
コメントの編集 選択しているファイルのコメントを編集します。
関連リンク コメントについては 「コメント」 を参照してください。
ラベルカラーの設定

選択しているファイルのラベルカラーを設定します。
関連リンク ラベルカラーについては 「ラベルカラー」 を参照してください。
お気に入りフォルダ

「お気に入り」タブのビュー上にお気に入りフォルダを作成します。
「 お気に入りフォルダ」は「 お気に入り」をまとめておくことができます。
お気に入りの削除 登録した「 お気に入り」を「お気に入り」タブのビューから削除します。
お気に入りの登録 選択しているファイルやフォルダを「 お気に入り」として登録します。
エクスプローラで表示 エクスプローラウィンドウを開き、選択しているフォルダやファイルを表示します。
サンプルをお気に入りに登録 パッケージ内のサンプルデータを「 お気に入り」として登録します。

F オプション

「 フィルター」は、「ファイルリスト」が縮小表示のときに表示する情報を限定できます。
▼をマウス左ボタンクリックで表示されるメニューから限定したい項目選択します。「リセット」を選択するとデフォルト状態に戻せます。

「 拡大 / 縮小ボタン」は、タブによって以下の様に結果が異なります。

コンピュータ / お気に入りタブ Textures または Primitives フォルダ内のファイルを大きく表示します。
検索タブ 検索の結果、表示されるファイルを大きく表示します。

G ファイルリスト

 「ナビゲーションペイン」の選択または設定結果に応じ、フォルダやファイルを一覧表示します。
以下は「 タブ」機能ごとの表示内容です。

コンピュータ 「ナビゲーションペイン」で選択したフォルダ内のフォルダやファイルを表示します。
お気に入り 「 お気に入りフォルダ」や「 お気に入り」内のフォルダやファイルを表示します。
検索 検索の結果ヒットしたフォルダやファイルを表示します。

ファイルやフォルダは付随する情報を「ツールバー」やマウス右クリックで表示されるメニューから編集できます。
操作可能なツールは 「ツールバー」 を参照してください。
