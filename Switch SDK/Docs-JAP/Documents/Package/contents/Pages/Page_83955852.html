サンプラアトリビュート設定プラグイン

ビットマップに NintendoWare 用のサンプラのアトリビュートを設定します。ただし、3ds Max の画面上には反映されません。設定の検索もできます。
「Nintendo > Set Sampler Attribute」を選択するとウィンドウが表示されます。

このウインドウを使って、複数のビットマップに対して同時に設定を変更したり、検索することができます。 ノードを複数選択することで、それらに割り当てられているマテリアルを選択します。

Selected Material 欄には、選択されている ノードに割り当てられたマテリアルに含まれるビットマップが表示されます。

設定方法

ビットマップを選択（複数選択可）すると、Set 欄に選択したビットマップの現在の設定が表示されます。
複数選択の場合、一番上のビットマップの設定が表示されます。

ウィンドウの各項目に入力して Set をクリックして変更します。複数選択の場合、選択したビットマップを一括して変更できます。
ウィンドウを閉じずに別のノードを選択すると、別のビットマップを設定することができます。

  • Name
    サンプラの名前を入力します。
    空欄にするとビットマップに応じた名前で出力されます。デフォルトは空欄です。
  • Hint
    サンプラのヒント情報を入力します。
    サンプラの名前を入力している場合に入力できます。
    空欄にするとビットマップに応じたヒント情報で出力されます。デフォルトは空欄です。
参考：

同じマテリアルで使用しているサンプラは、名前やヒント情報が重複してはいけません。

参考：

サンプラーアトリビュートは、Max のマップに設定されます。ビットマップファイル自体に設定されるわけではありません。

検索方法

Search 欄の Name に文字列を入力して、右の Search ボタンをクリックすると、サンプラの名前に入力した文字列が含まれる ビットマップがすべて選択されます。
「*（アスタリスク）」を入力して検索すると、サンプラの名前が空欄でない ビットマップがすべて選択されます。
何も入力しないで検索すると、サンプラの名前が空欄になっているビットマップがすべて選択されます。

Hint の検索についても同様です。
