テキストボックスペインを利用する

文字列を画面に配置する場合に使用する、テキストボックスペインの説明をおこないます。
テキストボックスペインは、バイナリフォントファイルを使用して文字列の配置が可能です。

フォントを作成する

事前準備として、バイナリフォントファイルを作成しておきます。
バイナリフォントの作成方法については、関連ドキュメントを参照してください。

FontConverter ( GUI版 ) FontConverter ( GUI版 ) のサンプルを用い、フォントの作成手順について説明します。( チュートリアル )

テキストボックスペインを作成する

レイアウトファイルを新規作成し、[右クリックメニュー]→[ペインの作成]→[テキストボックスペイン]を選択します。
テキストボックス作成ダイアログが表示されます。



ここで、あらかじめ作成しておいた、バイナリフォントファイルを指定します。
指定した情報が正しければ、OK ボタンが有効になっているはずです。
OK ボタンを押してペインを作成します。
正しく、文字列がレイアウト領域に表示されているはずです。


文字が表示されない、または、表示されていない文字があるという場合は、フォントファイルに含まれていない文字を使用していないか確認してみてください。
アルファベット文字しか含まないフォントデータで漢字を指定している場合などが当てはまります。

テキストボックスペインのプロパティを調整する

プロパティウィンドウを開き、左側項目ツリーからテキストボックスペイン情報項目を選択します。

制御文字列の利用

テキストボックスの文字列中で '^' 文字は制御文字列として利用することが可能です。
レイアウトコンバータは、'^' 文字以降の 2 文字を 16 進数の数字(0x00 ～ 0x1F)として解釈し、制御コードに置き換える変換処理を行います。
制御コードを実機プログラム中で解釈することで、テキストボックスペインの挙動をユーザ拡張することが可能となります。
これによって、たとえば、文字列の途中で文字色や、文字サイズを変更したり、画像イメージを表示したりするようなユーザ拡張処理を行うことが可能です。

なお、不正な指定があった場合には、'^' 文字が消去され、続く 2 文字が通常の文字として表示されます。
コンバータにオプションを指定することで、変換処理を無効にすることも可能です。
