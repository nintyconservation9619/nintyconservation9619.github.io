開発機の制御

はじめに

本ドキュメントでは、開発機に対して以下の仕事を行うツールについて説明します。

  • 開発機 (SDEV) の管理
  • 開発機 (SDEV) の初期化
  • 開発機 (EDEV) の初期化
  • アプリケーションの実行
  • 自動テスト
  • アプリケーションのインストール
  • セーブデータの管理
  • コントローラの管理
  • アルバムの管理
  • ログの受信と保存および閲覧
  • 本体設定値の管理
