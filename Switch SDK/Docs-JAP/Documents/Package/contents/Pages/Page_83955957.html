コントロール

コントロールとは

レイアウトの振る舞いを定義する仕組みです。
具体的には、「機能ペイン」（特別な役割を持ったペインを定義する）と「機能アニメ」（特別な役割を持ったアニメーション区間タグを定義する）からなります。

どんなことができるのか

コントロール定義を追加することで、新しい振る舞いを定義することができます。
例えば、プログラマが拡張プログラムを用意すると、独自の挙動を持つボタンなどを作ることが可能です。

また、プログラマがユーザ独自の部品に対応したビューアを作ることで、デザイナがプログラマの手を借りずに最終的な動きを確認でき、効率よく試行錯誤できる環境を作れます。
