ディレクショナルビルボード

ここではパーティクルタイプ「ディレクショナルビルボード」について説明します。

概要

パーティクルタイプ「ディレクショナルビルボード」は、パーティクルの移動する直前のフレームとの位置を見て、その移動方向へパーティクルを傾けて表示します。
パーティクルの Z 面はカメラのレンズ面と平行になるように Y 軸のみ回転して表示します。
例えば、四方に破裂する火花のエフェクトなど、速度方向にスケールをかけることができます。
