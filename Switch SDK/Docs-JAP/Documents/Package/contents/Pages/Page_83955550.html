エラーおよび警告

ここでは、エクスポートプラグインのエラーと警告について説明します。

エラーの確認方法

エクスポートを行ったときにエラーや警告が起きた場合、リスナーウィンドウに出力されます。

また一部のエラーはメッセージボックスで表示されます。

エラーの種類と対処方法

現在実装しているエラーリストをアルファベット順に以下に記載します。

エラーを検出した場合、Export 処理は中止します。

  • Animation file to merge is not found
    エクスポートオプションの Merge Animation Folder が ON かつフォルダを指定していますが、マージ対象のアニメーション中間ファイルが 1 つも存在しません。
  • Can't open file
    ファイルが存在しない場合、ファイルが開けない場合、ファイルに上書きできない場合などに発生します。
  • Comment is wrong
    エクスポートオプションの Comment に禁止文字が含まれています。Comment には、;（セミコロン）、"（ダブルクォート）は使用できません。
  • DDS file is wrong
    DDS ファイルの内容またはフォーマットが不正です。対応しているフォーマットを確認してください。
  • Image type is not supported
    サポートしていない画像ファイルが使用されています。現在 TGA ファイルと DDS ファイルのみテクスチャファイルとして使用できます。
  • Influence object is not outputted
    スキニングに影響するノードが出力されていません。 エクスポートオプションの Export Target が Selection でスキニングに影響するノードが選択されていない場合や、スキンモディファイア及びPhysiqueモディファイアを適用していてもエンベロープが頂点に割り当てられていない場合に発生します。
  • Merge Animation Folder is wrong
    エクスポートオプションの Merge Animation Folder に禁止文字が含まれる場合に発生します。Merge Animation Folder には、全角文字、半角カナ文字、;（セミコロン）は使用できません。
  • Merge Animation Name is wrong
    エクスポートオプションの Merge Animation Name に禁止文字が含まれる場合に発生します。Merge Animation Name には、ファイル名に使用できない文字、全角文字、半角カナ文字、;（セミコロン）は使用できません。
  • No effective node
    出力される有効なノードが 1 つもありません。
  • No output file
    中間ファイルの出力指定が全てオフになっている場合や、テクスチャデータの出力指定のみオンで出力するテクスチャがない場合に発生します。
  • Output File Name is wrong
    Export オプションの Output File Name が空の場合や、ファイル名に使用できない文字が含まれる場合に発生します。
  • Output Folder is wrong
    Export オプションの Output Folder が空の場合や、ファイル名に使用できない文字が含まれる場合に発生します。
  • Path of fmd file to merge is wrong
    エクスポートオプションの Merge fmd File が ON かつ fmd ファイルのパスを指定していて、マージする fmd ファイルが存在しない場合や、パスに禁止文字が含まれる場合に発生します。パスには、全角文字、半角カナ文字、;（セミコロン）は使用できません。
  • Post-Export Script failed
    出力後に実行するスクリプト（MAX Script）でエラーが発生しました。リスナーウィンドウでエラーの詳細を確認してください。
  • Pre-Export Script failed
  • 出力前に実行するスクリプト（MAX Script）でエラーが発生しました。リスナーウィンドウでエラーの詳細を確認してください。
  • Preset Name is wrong
    プリセット名が空の場合や、禁止文字が含まれる場合に発生します。プリセット名には、ファイル名に使用できない文字、全角文字、半角カナ文字、「;（セミコロン）」は使用できません。また「None」というプリセット名を使用することもできません。
  • Sampler hint is wrong
    サンプラのヒント情報に禁止文字が含まれています。
  • Sampler hint is duplicate
    同じマテリアルでサンプラのヒント情報が重複しています。サンプラアトリビュート設定プラグインで重複しないヒント情報を設定してください。
  • Sampler name is wrong
    サンプラの名前に禁止文字が含まれています。
  • Sampler name is duplicate
    同じマテリアルでサンプラの名前が重複しています。サンプラアトリビュート設定プラグインで重複しない名前を設定してください。
  • Texture size is wrong
    テクスチャ画像の幅と高さが対象プラットフォームで使用できるサイズになっていません。
  • TGA file is wrong
    TGA ファイルの内容が不正です。
  • The number of weighted influence is over
     コンフィグファイルで指定した値を超える数のノードにスキニングのウェイト値が分配されています。
  • Wrong weighted vertices exist
    スキニングのウェイト値の合計が 1.0 にならない頂点が存在します。ウェイト値を修正してください。

警告の種類と対処方法

現在実装している警告リストをアルファベット順に以下に記載します。

警告を検出しても、Export 処理は続行します。

  • ftx file to merge is not found
    エクスポートオプションの Merge ftx File が ON ですが、マージ対象の ftx ファイルが 1 つも存在しません。マージしないで出力します。
  • Inherits of node link info is off
    ノードの [階層] - [リンク情報] - [継承] のいずれかが OFF になっています。ON にしないと正しく表示されません。
  • Map channel does not exist
    マップチャンネル n が存在しないオブジェクトに、チャンネル n のマップが設定がされています。
    この状態では正常に出力できないため、強制的にマップチャンネル 1 のUV情報を貼り付けて出力します。
  • Material name is changed
    マテリアルの名前が重複しているため、中間ファイルに出力されるマテリアル名が変更されました。
  • Morpher modifier must be under skin(physique) modifier
    モーファーがスキン（あるいは physique ）より上にあります。
    モーファーはスキンや Physique より下にある必要があります。
  • シェイプアニメーションは出力されません。
  • Morpher target topology is different from original mesh
    モーファーのターゲットシェイプの頂点数、面の数、頂点カラーの数が、ベースシェイプと異なっています。
  • Node has no material
  • 出力されるノードにマテリアルが割り当てられていません。ダミーのマテリアルが出力されます。
  • Node name is changed
    同じ名前のノードが複数存在する場合やノード名に禁止文字が含まれる場合は、中間ファイルに出力されるノード名が変更されます。
  • Shader is not supported
    エクスポートプラグインが対応していないシェーダが使用されています。この場合、プラグインが自動的にデフォルトのマテリアルを中間ファイルに追加して出力します。
  • Texture file doesn't exist.
    割り当てられているテクスチャーが存在しません。 マテリアルに割り当てられたテクスチャーファイル名にパスが含まれている場合、出力時にそのパスに保存されている画像を参照します。
    そのパスに画像が存在しない場合に発生します。 また、テスクチャーファイル名にパスが含まれていない場合は、現在の max ファイルが保存されているディレクトリに画像を参照します。
  • Texture name is changed
    テクスチャ名に禁止文字が含まれています。禁止文字は「_（アンダーバー）」に変換して出力されます。
  • Texture type is not supported
  • ビットマップ、法線マップ、合成マップ以外のテクスチャマップは使用できません
  • There are multiple texture files with the same name
    出力対象にファイル名が同じでフォルダまたは拡張子の異なるテクスチャファイルが複数存在します。 出力される ftx ファイルは 1 つだけなので注意してください。
  • Zero normal exist
    長さが 0 の法線ベクトルが存在します。ライティングが正常に行えません。「法線を編集」モディファイアやスムージンググループなどで修正してください。
  • Zero tangent or binormal exists
    長さが 0 の接線または従法線が存在したため、(0, 1, 0) に置き換えて出力しました。法線や UV 座標を修正してください。
