AudioRedirector


警告：

現状 AudioRedirector は評価版として提供されるものです。今後予告なく大きな仕様変更を行う可能性があることにご注意ください。

概要

AudioRedirector は host PC で再生している音を開発機に転送し再生するためのツールです。DAW ツールなどで制作中のサウンドを、開発機の本体スピーカで再生して確認する、などの用途を想定しています。

ツールは以下に配置されています。

  • Tools/Audio/AudioRedirector/AudioRedirector.exe
  • TargetTools/NX-NXFP2-a64/AudioRedirectorServer/AudioRedirectorServer.nsp

使用方法

開発機側のセットアップ

AudioRedirectorServer.nsp を実行します。

PC 側のセットアップ

TargetManager を起動し、開発機に接続します。htcs エントリの「ポート名」に "AudioRedirector" が表示されていることを確認します。

この状態で AudioRedirector.exe を起動します。成功すると、以下のログが表示されます。

start redirecting, press any key to exit

また AudioRedirectorServer.nsp の表示は以下のように変わります。

PC で再生した音が開発機から再生されるようになります。AudioRedirector.exe を終了するには、任意のキーを入力してください。

その他

何らかのエラーが発生した場合、AudioRedirector.exe および AudioRedirectorServer.nsp 共にエラーメッセージを表示します。この場合、上記手順を最初からやり直してください。

制限事項

  • PC 上の既定のサウンドデバイスのサンプルレートが 48kHz/2ch である場合のみをサポートしています
    • この制限事項については、今後の改善を検討中です
