リストの操作

サウンドプロジェクトが取り扱うアイテムは、全てサウンドセットタブにリストとして表示され、リストの一行が、ひとつのアイテムに対応します。
ここでは、リストにおける共通の操作について解説します。

インデックスセルの操作

リストの先頭には、アイコンを配置したセルがあります。このセルをインデックスセルと呼びます。

インデックスセルは、リストの行が対応するアイテムの種類を示しています。
リスト自体は表計算ソフトに似た操作でセル単位での操作も可能ですが、インデックスセルにより行単位（アイテム単位）での操作も同時に可能にしています。

バンクリストにおいて、インデックスセルをダブルクリックすると、参照するバンクファイルのタブを開くことができます。

インデックスセルをクリックするとその行が選択され、選択行のインデックスセルをドラッグする事により、リスト内で行の移動（並び替え）を行う事ができます。

未選択の行に対してインデックスセルをクリックしたままドラッグすると、その範囲の行が選択状態になります。

グループへの登録／解除

グループに登録可能なアイテムのリストでは、リストのコンテキストメニューからの操作でも、グループにアイテム登録を行なう事ができます。

グループに登録したいアイテムのリストで、右クリックして現れるコンテキストメニューから[グループに登録]を実行します。
専用のダイアログが現れ、全てのグループの一覧が表示されるので、登録したいグループにチェックを入れて、OK を押してください。

既に登録済みのグループには、あらかじめチェックが入っており、そのチェックを外すことによりグループ登録を解除できます。

また、複数アイテムを選択してグループ登録を行った場合、グループに登録されているものと、されていないものが混在したグループには、グレーのチェック表示がされます。
一度グレーのチェックを外すと、キャンセルしない限り、元の混合状態には戻せません。

 

セルの一括編集

一括編集ダイアログを利用して、複数のセルを一括して編集することができます。
編集したい複数のセルを選択して、以下の方法により一括編集ダイアログを開きます。

  • Shift + F2 キーを押す
  • メニューの [編集]→[一括編集] を実行
  • リスト上の右クリックによるコンテキストメニューの [一括編集] を実行

セルに適用したい数値と、数値の適用方法を選択します。

  • 直接指定　…　指定した数値がそのまま適用されます。チェックボックスでは "true", "false" で on, off の指定ができます。
  • 増減で指定　…　-20 や +150 など、指定した数値が元の値に加算されます。
  • 割合で指定　…　0.01 や 20.0 など、指定した数値が元の値に倍率として適用されます。

それぞれのセルの上限を超える、または下限を下回る値となる設定は結果に反映されません。

 

圧縮方式、プレイヤー、バンク等、コンボボックスで編集する項目は、コンボボックスに表示される値をそのまま直接指定で入力することにより、一括編集が可能です。

アイテムの名前については、重複を避けるため複数選択してあっても一番上のセルしか編集できません。

アイテム名の補完

リスト上でプレイヤー、バンク、グループの項目を編集する際に、直接アイテム名を入力できます。

また直接入力時は、前方一致補完が利用できます。

リスト項目の並び替えと表示／非表示切り替え

並び替え

リストの [名前] や [データサイズ] などの項目は、各項目名をドラッグ＆ドロップすることにより、表示順序を並び替えることができます。

また、下記の【リスト項目の設定ダイアログ】上での各項目のドラッグ＆ドロップでも同様の並び替え操作が行えます。

リスト項目の設定

リストの各項目について、各項目の表示／非表示を選択できます。 利用頻度の低い項目を非表示にする事により、リスト項目が多く、横に長くなり一覧性に難が生じてしまうアイテムを見易くします。

サウンドセットタブのリスト項目名の部分を右クリックすることにより現れるコンテキストメニューから、表示・非表示にしたい項目をクリックすることにより、リストの表示に反映されます。

 

コンテキストメニュー上段の「リスト項目の設定...」を選択すると 【リスト項目の設定ダイアログ】が表示され、表示／非表示の設定を一括して行うことができます。

また、ダイアログリスト内の各項目をドラッグ、もしくは「上へ移動」、「下へ移動」ボタンを押すことにより、表示順序の並び替えを行うことが可能です。

 

非表示の項目は、セルの範囲選択の対象となりません。
選択セルが非表示にされた場合、選択は解除されます。

リスト項目設定の登録

現在のリスト項目設定（カラムの並びやソートの状態）をリストの種類毎に登録することができます。
保存情報は、"項目の表示/非表示"、"項目の並び順" と "ソートの状態" です。

リスト項目名の部分を右クリックして表示されるコンテキストメニューから「リスト項目設定の登録...」を選択すると、【リスト項目設定の登録】ダイアログが開きます。

 

登録名を設定することにより、現在のリスト項目の設定状態を保存します。

リスト項目設定の適用

登録済みのリスト項目設定を現在のリストに適用します。

リスト項目名の部分を右クリックして表示されるコンテキストメニューから、「リスト項目設定の適用」のサブメニューに表示される名前を選択すると、登録した時のリスト項目の状態に変更されます。

 

設定の適用は、コンテキストメニューの他にメインツールバーの [リスト項目:] ボタンからも実行できます。

 

フォーカスがあるリストのリスト項目設定がボタンに表示されます。
登録していないと、ボタンが [---] となりグレー表示になります。
また、登録してあってもリストからフォーカスが外れるとグレー表示になります。

登録済みリスト項目の修正

オプションダイアログの「リスト項目」タブで、登録済みリスト項目の名前変更や削除が行えます。

リスト項目設定ファイルは、 %APPDATA%\Nintendo\SoundMaker\バージョン\user.columns に保存されます。

リスト幅の最適化

リストの項目名のセル幅を調整する場所（右境界）でダブルクリックをすると、リストの表示内容に応じてセル幅を最適に調整します。

リストのソート表示

リストの項目名をクリックすると、各項目の内容でリストをソートします。
クリックする度に、昇順・降順・ソート無し(本来の並び) の順で切り替わりソートされ直します。

『インデックスセル』の項目でのソートは、「本来の並び」で表示されます。

「本来の並び」とは、新規に作成したり、インデックスセルをドラッグで並び替えてユーザーが作成した並び順を指します。

ここでの各項目のソートは、表示上の並び替えだけなので、プロジェクトに保存される並びは、「本来の並び」順です。

また、ファンクションキーの F12 で、リストの表示を更新する事ができます。
特定の項目でソートして表示している時に、アイテムの挿入や新規作成を行った場合などに利用します。
この操作で行うのはリストの表示の更新のみです。
プロジェクトファイルの更新が必要なケースでは、ファイルメニューの「プロジェクトの再読込」を利用してください。

コピー＆ペースト

キー操作やリスト上の右クリックにより現れるコンテキストメニューから、セル単位または行単位にコピー＆ペースト操作が行えます。

 

セル単位

セル単位でのコピーは、コンテキストメニューの「セルのコピー」または Ctrl + Shift + C で行います。
コピーしたセルは、破線で囲まれます。

行単位

行単位でコピーした場合は、コンテキストメニューの「項目を選択して貼り付け」が有効になります。
コピー元とペースト先が異なるリストの場合、または複数行をコピーしている場合は、「項目を選択して貼り付け」は無効になります。

貼り付けを実行すると、項目を選択するためのダイアログが表示されます。
上書きしたい項目にチェックを入れ実行します。

 

 

アイテムの一時無効化

リスト上でアイテム（行）を無効化することにより、プロジェクトからアイテムを削除することなく、一時的に必要のないものを無効にすることができます。
無効化したアイテムは、コンバートの対象となりませんが、アイテムのパラメータはファイルに保存されますので、簡単に復元させることができます。

設定を行いたい行で、右クリックによるコンテキストメニューから、またはメインウインドウの[編集]メニューから、「アイテムを無効にする」、「アイテムを有効にする」を選択します。
無効化されたアイテムは、テキストがグレー表示に変わり、インデックスセルや参照アイテムの項目に "無効マーク" が付きます。
プロジェクトツリーにも表示されるアイテムについては、プロジェクトツリー側の表示も同様に無効化アイコンが付き、アイテム名がグレー表示になります。

 

無効化の対象は、各種サウンド（ストリームサウンドトラック、ウェーブサウンドセット、シーケンスサウンドセットを含む）、バンク、プレイヤー、グループ、波形アーカイブ及びインストリストです。
いずれのサウンドも無効化された状態でもプレビュー再生は可能です。

アイテムの色付け

リスト上でアイテム（行）単位に色付けすることができます。

色指定は、選択された行での右クリックによるコンテキストメニューから、またはメインウインドウの[編集]メニューから、「アイテムのカラー」を選択し、サブメニューで色を決定します。

 

配色の情報は、HTMLサウンドリスト、HTMLインストルメントリストにも反映します。

色付けが行えるのは以下のリストです。

  • ストリームサウンドリスト
  • ストリームサウンドトラックリスト
  • ウェーブサウンドセットリスト
  • ウェーブサウンドリスト
  • バンクリスト
  • シーケンスサウンドセットリスト
  • シーケンスサウンドリスト
  • 波形アーカイブリスト
  • プレイヤーリスト
  • グループリスト
  • インストリスト

グループアイテムリストは、各アイテムに本来設定されている色が表示されます。

サブメニュー内の色を表している文字列は、「プロジェクト設定ダイアログ」の [カラー] タブで変更することができます。

数値セルのバーの表示／非表示切り替え

上下限のある数値を表すセルにバーを表示できます。バーは最大値に対する比率です。

 

デフォルトは非表示ですが、メインメニュー [ツール] - [オプション] から「オプションダイアログ」を開いて、バーの表示／非表示の設定を行います。

更新点の有無

リストで編集を行い、未保存の更新点があるサウンドセットでは、プロジェクトツリーのサウンドセットノードの右側に 、またバンクはバンクタブのバンクファイル名右側に"＊" を表示して、更新の有無を示します。

 

波形ファイル情報の更新

[表示]→[波形ファイルの情報を更新] を実行することにより、リスト上に表示されている波形ファイルの情報を更新することができます。
またこの操作には Ctrl + R のショートカットキーが割り振られています。

 

更新される情報は以下の項目です。

  • 再生時間
  • ティック数
  • サンプルレート
  • ビットレート
  • 量子化ビット数
  • チャンネル数

アイテムの警告表示

アイテムの参照ラベルのリンクが切れていたり、アイテム名が重複している場合など、リスト上に警告アイコンを表示して注意を促します。

 

 

アイテムの上限数

実機側のランタイムで扱えるストリームサウンドやバンクなどの各アイテムの上限数は 16777216 (2の24乗) 個です。
各アイテムを 16777217 個以上作った場合のコンバータや実機側の nn::atk ライブラリの動作は不定です。

SoundMaker 上ではアイテムの個数制限はかけておらず、メモリの許す限り登録することが可能です。

ビューの分割

サウンドセットタブ内のビューは 2 分割して表示することができます。 ここではビューの分割機能について解説します。

用語解説

  • ［ビュー］　…　サウンドセットタブとバンクタブのタブコントロールを含む編集画面です。
  • ［上ビュー、下ビュー、左ビュー、右ビュー］　…　ビューを分割した際の各ビューを指します。
  • ［アクティブビュー］　…　操作対象となるビューです。最後にフォーカスを持ったビューがアクティブビューとなります。
  • ［アクティブタブ］　…　アクティブビュー内の選択タブです。ビュー分割時、選択タブは上下（左右）ビューにひとつずつあります。

ビューの分割

ビュー分割

ビューを 上下 または 左右 に 2 分割できます。

分割は、[ウインドウ]→[上下に分割]、[左右に分割] 、またはツールバーの [上下に分割]、[左右に分割] ボタンにより行います。

 

 

シングルビューからのビュー分割直後の動作は、ビューを分割して、新たに開いたビューにアクティブタブと同じタブをもうひとつ開きます。

分割した各ビューは、それぞれ全て同じ仕様かつ独立して動作します。

 

［上下に分割］

分割されていない状態　…　現在のビューを上、新しいビューを下に表示します。
左右分割している状態　…　左ビューを上、右ビューを下に表示します。

［左右に分割］

分割されていない状態　…　現在のビューを左、新しいビューを右に表示します。
上下分割している状態　…　上ビューを左、下ビューを右に表示します。

 

分割解除

ビューの分割を解除するには、消したいビューにあるタブをすべて閉じることにより行います。

タブ上のコンテキストメニューから「すべてのタブを閉じる」を実行することにより、簡易に行うことができます。

 

タブの操作

タブ操作

同じビュー内で複数のタブが開かれている場合、タブのドラッグ＆ドロップにより、並び替えが可能です。
また、他のビューへの移動はできません。

タブ上を右クリックして現れるコンテキストメニューから、そのタブを「閉じる」またはそのビュー内の「他のタブをすべて閉じる」と「タブをすべて閉じる」ことができます。
メインメニュー [ウインドウ]→[タブをすべて閉じる] からの操作では、分割表示されている両方の全タブが同時に閉じられます。

 

 

マウスの中ボタンクリックで、コンテキストメニューの「閉じる」と同操作が可能です。

ショートカットキー

"共通の操作 − リストの操作" の「タブ操作ショートカット一覧」を参照してください。

操作の対象

プロジェクトツリーや検索ウインドウ等にて、タブ表示を切り替える操作が行われた場合、アクティブビューが表示切替の対象となります。

ビューの最小化

分割されたビューを閉じずに最小化することができます。

ビューを分割するとタブ表示エリアの端に現れる [最小化] ボタンをクリックすると、対象のビューはタブ部分の表示だけに縮小されます。
最小化されたビューは、[最小化] ボタンから切り替わった [元に戻す] ボタンをクリックすることにより、元のサイズに戻ります。

 

 

[最小化] と [元に戻す] 切り替え動作は、以下の方法でも行えます。

  • ビュー分割の境界のスプリッタをダブルクリックする。
  • 上下分割中に [上下に分割] ボタンをクリックする。
  • 左右分割中に [左右に分割] ボタンをクリックする。

最小化は、アクティブビューでない方のビューが最小化されます。

 

最小化されたビューがある状態で、以下の操作を行った場合、ビューは元のサイズに戻ります。

  • 最小化したビューのタブをダブルクリックする。
  • 上下分割中に [左右に分割] ボタンをクリックする。
  • 左右分割中に [上下に分割] ボタンをクリックする。

 

   

ファイルについて

ファイル名、およびパス名に使用できる文字には制限があり、以下の文字が使えます。

  • ファイル名の先頭は、半角英字とアンダーバー [a-zA-Z_]
  • ファイル名の先頭以外およびパス名は、半角英数字とアンダーバー [a-zA-Z0-9_]

ディレクトリ名は 248 文字未満、ファイル名まで含めたパスは 260 文字未満である必要があります。

プロジェクトファイルの種類

SoundMaker が扱うプロジェクトファイルには、以下の種類があります。

  • サウンドプロジェクトファイル
  • サウンドセットファイル
  • バンクファイル

参照ファイルの種類

SoundMaker がアイテムに参照ファイルとして登録し、再生・コンバートできるファイルには、以下の種類があります。

  • wav ファイル（波形）
  • aiff ファイル（波形）
  • opus ファイル（波形）
  • 標準 MIDI ファイル（シーケンス）
  • テキストシーケンスファイル（シーケンス）

上記以外の種類のファイルもアイテムとして登録することができますが、SoundMakerでの再生やコンバートはできません。

アイテムリストのコンテキストメニューから [参照ファイルを開く] を選択することにより、可能であれば対応アプリケーションが開きます。

  • オプションダイアログの「外部エディタ」タブで、参照ファイルを開くアプリケーションを指定することもできます。

参照ファイルパスの保存方法

アイテムが参照するファイルパスの保存は、参照するファイルが「サウンドプロジェクトファイルと同じドライブの時」には相対パスで、同じく参照するファイルが「サウンドプロジェクトファイルと違うドライブの時」には絶対パスで行われます。

波形ファイル選択ダイアログでのプレビュー再生

波形ファイルを選択するダイアログ上で、  選択した波形ファイルをプレビュー再生することができます。

「ファイル選択時に自動再生する」のチェックが ON の場合、リストでファイルを選択すると同時に自動再生が行われます。
