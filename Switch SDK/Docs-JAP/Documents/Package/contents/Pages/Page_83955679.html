パーティクル

  このタブではパーティクルに関する設定を行います。

パーティクル寿命

「 パーティクル寿命」は、パーティクルの生成から破棄までの時間「寿命」について設定する機能群です。
以下はグループ内設定項目の一覧です。

寿命無限

パーティクルの寿命を無限設定する機能です。
寿命ランダム (%)   パーティクル個々の寿命をランダムにする機能です。

パーティクル設定

「 パーティクル設定」は、パーティクルの種類選択や、姿勢制御の為の軸について設定する機能群です。
以下はグループ内設定項目の一覧です。

パーティクル形状 パーティクルの表示形状を設定する機能です。
トリミング パーティクルの描画されない範囲をトリミングする機能です。
パーティクルタイプ

パーティクルの向きを設定する機能です。
パーティクルの中心 パーティクルをスケールまたは回転させる中心位置を設定する機能です。
パーティクルオフセット ゲーム内のオブジェクトとパーティクルの重なりを調整する機能です。
Zファイティング抑制 パーティクルが別のポリゴンと同じ位置に描画されたときのちらつきを抑える機能です。

揺らぎ設定

「 揺らぎ設定」は、パーティクルの設定要素を波形によってアニメーションさせる機能です。
以下はグループ内設定項目の一覧です。

揺らぎの適用 パーティクルのアルファやスケールなどの設定要素に対して揺らぎを与える機能です。
