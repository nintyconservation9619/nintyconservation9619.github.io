NintendoSDK

NintendoSDK は任天堂プラットフォームの開発環境です。NintendoSDK はプラットフォーム間で共通の機能やプラットフォーム固有機能を追加する能力を提供します。複数のプラットフォーム間で開発環境を使うことはコードや知見の再利用を可能にし、他の任天堂プラットフォームでの開発を容易にします。

NintendoSDK は以下のプラットフォーム共通機能を提供します。

  • プラットフォーム間で共通の API
  • プラットフォーム間で共通の各種ツール類
  • プラットフォーム間で共通のコンパイラツールチェインおよびデバッガソフトウェア
  • 上記機能の Windows 上での互換環境

以下の機能はプラットフォーム固有のもので、アドオンパッケージによって取り込まれなければなりません。

  • グラフィックスの下位レイヤなど性能が重要とされる機能
  • プラットフォーム固有のハードウェアを利用する機能
  • 互換機能を提供することが困難な過去のプラットフォームの機能
  • 上記機能の Windows 上での互換環境

NintendoSDK NX Addon

NintendoSDK の NX プラットフォーム向けの開発環境です。NintendoSDK NX Addon は NX プラットフォームでの開発に必要です。

参考：

現在のバージョンでは、MegaZarf 形式のファイル経由でインストールした場合は NX Addon のすべてのパッケージがインストールされることが保証されます。

フィードバック

NintendoSDK は、より良い開発環境を提供するために、開発者からのフィードバックを求めています。開発の初期段階でフィードバックをいただければ、少ない変更コストでフィードバックを反映することができます。

フィードバックの連絡先については、NintendoSDK の公開元にお問い合わせください。
