パーティクルの中心

ここではパーティクルをスケールまたは回転させる中心位置を設定する機能「パーティクルの中心」について説明します。

概要

「パーティクルの中心」は、パーティクルをスケール、回転させる中心位置をパーティクルの中央を基準に X Y平面上で設定できる機能です。
1 以上の値を設定することで中心を表示面より外側に設定できる為、大きく回転させることができます。
以下は「 設定項目」の概要です。

パーティクルの中心

パーティクルの中心位置をパーティクル全体を 1 として「 ダイヤル」や「 テキストボックス」により設定します。
-1.0 ～ 1.0 以外の数値を指定することで 「パーティクルの中心」を、パーティクルの外に設定できます。

設定例

以下は「パーティクルの中心」の設定を変更し、Z 軸で正方向に回転させたときの比較画像です。

A XY = -1 パーティクル平面の右上を中心に回転しています。
B XY = 0 パーティクル平面の中央を中心に回転しています。
C XY = 1
 パーティクル平面の左下を中心に回転しています。
