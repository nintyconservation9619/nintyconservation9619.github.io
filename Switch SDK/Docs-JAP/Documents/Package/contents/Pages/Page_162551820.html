Atk サウンド再生履歴パネル

Atk で再生したサウンドの履歴情報をリストビューで一覧表示するパネルです。

画面説明

  説明
A ツールバーです。リストの表示を変更することができます。
B 再生を開始したサウンドが表示されるリストビューです。
C 現在の時間のリストアイテムをオレンジ色で表示します。

ツールバー

項目 説明
名前フィルタ 文字列を入力することでリスト表示のフィルタリングをします。フィルタ対象は名前とプレイヤーです。

リスト列の表示、非表示の設定します。

Time / AppFrame と名前の列は非表示にすることはできません。

リストビュー

各リスト列のカラムをクリックすると昇順、降順ソートすることができます。

リスト列 説明
Time / AppFrame

再生を開始した時間です。

時間単位の切り替えにあわせて Time または AppFrame に切り替わります。

  • Time は時間で、書式は「時:分:秒.ミリ秒」です。
  • AppFrame はアプリケーションフレームで、単位はフレームです。
名前 サウンド名です。
プレイヤー プレイヤー名です。
音量

音量です。

現在値を数値とバーで表示します。バーは最大値に対する比率です。

プレイヤー

優先度

プレイヤー優先度です。

現在値を数値とバーで表示します。バーは最大値に対する比率です。
サウンド種別 サウンドの種類です。
