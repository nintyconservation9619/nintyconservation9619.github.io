NintendoWare for Cafe から NintendoSDK NintendoWare で仕様の変更がある機能

エフェクトデータの修正が必要な機能

以下は NintendoWare for Cafe から NintendoSDK NintendoWare で仕様変更の為、データ調整が必要な機能の一覧です。

ノード名 タブ名 項目名 NintendoWare for Cafe から NintendoSDK NintendoWare へ読み込んだ時の差異
エミッタ 基本設定 パーティクルソート
  • 「昇順」に設定されている場合、NintendoSDK NintendoWareでは全て「ソートしない」に変更されます。
  • 「昇順」「降順」の見え方がNintendoSDK NintendoWareでは逆になります。
    以下のように変更することで、見た目が揃います。
    • 「昇順」→「降順」に変更
    • 「降順」→「昇順」に変更
エミッタ パーティクル 寿命無限 NintendoSDK NintendoWare では寿命無限のみ ON では機能しません。
「寿命無限」の機能を有効にする場合は放出タブの「ワンタイム」を ON に設定します。
エミッタ パーティクル パーティクルタイプ

「ディレクショナルポリゴン」の向きが NintendoWare for Cafe と NintendoSDK NintendoWare で異なります。
以下のように変更することで、見た目が揃います。

  • パーティクルを X 軸で 180 度回転させる
  • エミッタを X 軸 または Z軸で 180 度回転させる
エミッタ パーティクル パーティクルオフセット

「Z値をオフセット」がこれまでは単位のない恣意的なオフセット値でしたが、単位がワールド座標系での距離に仕様変更されました。

パラメータの機械的な調整ができないものなので、意図しないオフセットになっている場合は、オフセット値を調整してください。
エミッタ コンバイナ カラーバッファ屈折

NintendoWare for Cafe の最新版で意図通りの描画結果になっていれば問題ありませんが、

NintendoWare for Cafe のバージョンアップにより、係数の効き具合に仕様変更が入っています。

最新でない NintendoWare for Cafe から一気にアップデートを行う場合は、見え方が意図通りかどうか確認してください。
エミッタ コンバイナ ソフトパーティクル/フラグメントで処理

NintendoSDK NintendoWareには、「ボリューム値」の機能が削除されています。

NintendoWare for Cafe 側で、「ボリューム値」に 0 以外の値が設定されている場合、NintendoSDK NintendoWareと表示が異なります。
エミッタ カラー カラー0～1挙動タイプ 連結式ストライプのカラーを設定したとき、NintendoSDK NintendoWareでは反映されません。
NintendoSDK NintendoWareの連結式ストライプにカラーを設定したい場合は、カラータブの「グローバル乗算カラー・アルファ」に設定し直してください。
チャイルド 基本設定 チャイルド追従タイプ/親のエミッタに追従 チェックが ON/OFF 関わらず、NintendoSDK NintendoWare では基本設定のパーティクル追従タイプが「エミッタに追従しない」に設定されます。
NintendoWare for Cafeで親のエミッタに追従する設定の場合、NintendoSDK NintendoWareで、基本設定の「パーティクル追従タイプ」を変更してください。
チャイルド 基本設定 継承設定/パーティクルにエミッタ割当

NintendoSDK NintendoWare の初期バージョンでは、「パーティクルにエミッタ割当」が ON の状態の方式に仕様変更されましたが、
その後の更新に伴い、「パーティクルにエミッタ割当」のオプションが追加され、このチェックを OFF にすることで、 NintendoWare for Cafe 相当の挙動を設定することが可能になりました。

中間ファイルアップデートを通すと自動的に「パーティクルにエミッタ割当」のチェックが ON になりますので、描画内容や描画負荷を考慮の上、必要に応じてチェックを OFF にするなどの調整を行ってください。

エフェクトデータの修正が不要な機能

以下は NintendoWare for Cafe から NintendoSDK NintendoWare でデータ仕様変更のあった機能の一覧です。
データを NintendoSDK NintendoWare EffectMaker に読み込むことで適切な値に変更される為、データ修正は不要です。

 ノード名 タブ名 項目名 NintendoWare for Cafe からNintendoSDK NintendoWare へ読み込んだ時の差異
エミッタ 放出 時間/放出間隔 数値の表記が変更されています。
例　NintendoWare for Cafe「10」→ NintendoSDK NintendoWare 「11」
エミッタ 放出 重力/大きさ（エミッタ時間アニメ） 数値の表記が変更されています。
例　NintendoWare for Cafe「0.1」→ NintendoSDK NintendoWare 「10」
フィールド カールノイズ 影響の強さ 数値の表記が変更されています。
例　NintendoWare for Cafe「0.01」→ NintendoSDK NintendoWare 「0.1」
フィールド カールノイズ ノイズの移動速度 数値の表記が変更されています。
例　NintendoWare for Cafe「1.0」→ NintendoSDK NintendoWare 「10」
チャイルド 基本設定 チャイルド有効 チャイルドを使用したデータを読み込むと、継承設定の「パーティクルにエミッタ割当」が ON に変更されます。
チャイルド 基本設定 親のフィールドを適用/親のフィールドを適用

チェックを使用したデータは、NintendoSDK NintendoWareで子に親と同じ数値のフィールドが作成されます。

NintendoWare for Cafe で作成したデータとほぼ同じ見た目になります。
チャイルド 放出 放出間隔 数値の表記が変更されています。
例　NintendoWare for Cafe「10」→NintendoSDK NintendoWare 「11」
チャイルド 放出 重力/大きさ（エミッタ時間アニメ） 数値の表記が変更されています。
例　NintendoWare for Cafe「0.1」→NintendoSDK NintendoWare 「10」
