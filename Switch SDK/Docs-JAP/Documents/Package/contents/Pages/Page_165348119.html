 エミッタ拡張パラメータノード

ここではゲームアプリで利用できるデータが設定できる「エミッタ拡張パラメータ」について説明します。
ゲームアプリの機能やワークフローで利用できます。

フラグ

「 フラグ」は、ユーザーデータをフラグで設定できます。
以下はグループ内設定項目の一覧です。

ビット [0] ～ [15]

16 個のフラグを「 チェックボックス」ON/OFF で設定します。

数値

「 整数パラメータ」は、ユーザーデータをスライダで設定できます。
以下はグループ内設定項目の一覧です。

数値1 ～ 数値2  0 ～ 1000000 の 範囲で「 ダイヤル」や「 テキストボックス」により整数値を設定します。

数値

「 実数パラメータ」は、ユーザーデータをスライダで設定できます。
以下はグループ内設定項目の一覧です。

数値1 ～ 数値2  0 ～ 1000000 の 範囲で「 ダイヤル」や「 テキストボックス」により実数値を設定します。

設定例

以下は、ゲームアプリで「エミッタ拡張パラメータ」を使った例です。

UI の 例 ゲームにおける使用イメージ
A フォグを使用 = ON

このエミッタのパーティクルはゲーム中のフォグの影響を受けます。
B 距離減衰 = 100 フォグのかかり具合を設定します。
