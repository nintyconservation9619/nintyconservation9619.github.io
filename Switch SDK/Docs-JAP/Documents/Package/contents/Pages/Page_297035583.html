整列ペイン

「整列ペイン」は、整列ペインの階層下にあるペインの調整方法を設定をするページです。

整列方法（水平） 水平方向で整列方法を設定します。
デフォルトマージン 整列するペイン同士の間隔を設定します。
端のペインを拡張する 整列する子ペインのうち、端にあるペインのサイズを、整列ペインの端まで拡張するかを設定します。
配下のペインを整列する レイアウトウィンドウで子ペインを整列します。

整列方法（水平）

階層下にあるペインを整列するとき、整列ペインのどの位置に合わせて整列するかをラジオボタンで設定します。

左

 整列ペインの左端に合わせて左詰めで整列します。
中央 整列ペインの中央部に合わせてセンタリングします。
右 整列ペインの右端とに合わせて右詰めで整列します。

デフォルトマージン

整列するペイン同士の間隔を設定します。

デフォルトマージン

 整列するペイン同士の間隔を入力します。

端のペインを拡張する

階層下にあるペインの横幅を拡張して整列するかを設定します。

端のペインを拡張する

チェック無しの場合（初期設定）

階層下にあるペインの横幅を変えずに整列します。

端のペインを拡張する

チェック有りの場合

階層下にあるペインの横幅を、整列ペインの端まで拡張して整列します。

整列方法（水平）の設定によって、拡張するペインが変わります。
整列方向（水平）：左 右端にあるペインの横幅を整列ペインの右端まで拡張します。
整列方向（水平）：中央

左右端にあるペインの横幅を整列ペインの左右端まで拡張します。
整列方向（水平）：右 左端にあるペインの横幅を整列ペインの左端まで拡張します。

配下のペインを整列する

ボタンを押すことで設定した内容で階層下のペインをレイアウトウィンドウ上で整列します。
