加算値（±）

ここでは、パーティクルの位置に加算する座標値を設定する機能「加算値（±）」について説明します。

概要

「加算値（±）」は、パーティクルの位置に加算または減算する値を、XYZ の座標系ごとに設定する機能です。
値を大きくすることによって、フレームごとの移動距離を大きくできます。
以下は「 設定項目」の概要です。

加算値（±）

パーティクルの位置に対して毎フレーム加算または減算する値を、XYZ 座標系それぞれ「 ダイヤル」や「 テキストボックス」により設定します。

参考：

「 パーティクル時間アニメ」ボタンを押すことにより「 加算値(パーティクル時間アニメ)」が表示さ、カーブエディタでアニメーションを設定できます。
アニメーションは「 CPUで動作させる」ON で有効にできます。
