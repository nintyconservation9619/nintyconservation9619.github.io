テキストボックスペイン

テキストボックスペインは、LayoutEditor で作成されたテキストボックスペインを編集するウィンドウです。

フォント情報 使用するフォントについての情報を入力します。
文字列 テキストボックスペインで作成する文字列を入力します。
文字の設定 文字列に使用する文字の設定を行います。
文字列の位置 テキストボックスペインの文字列の位置について設定をします。
行ごとに幅を指定

テキストボックスペインの幅を行単位で編集します。
テキストカラー 文字列に色を付けることができます。
文字列の長さを強制的に指定 文字列の長さを強制的に指定する設定をします。
影 文字に影を付けることができます。
文字毎の変換 文字毎の変換についての情報を入力します。

フォント情報

使用するフォントについての情報を入力します。

フォントファイル名 :

使用するフォントを指定します。

カーニング情報 :

フォントがカーニング情報 ( 文字間隔の微調整 ) を持っているかどうかを表示しています。

特殊フチ形式 :

Packed4 形式で作成されたフチ付きフォントを使用する場合に「ON 」になります。

「 ON 」になった場合、以下の「フチ形式」から選択します。

  • 通常
  • フチを消す
  • 2 回に分けて描く

文字列

テキストボックスペインで作成する文字列を、ここで入力します。

文字列入力枠

ここに入力した文字列が、レイアウト領域に表示されます。

ペインサイズの調整

文字列の表示サイズに合わせてペインの大きさが調整されます。

テキスト ID :

テキストボックスの文字列に対しての「検索キー文字列」として利用します。

( 翻訳された文字列をデーターベースから検索する際に利用します )

参考：

 プログラム上で、TextSearcher というクラスによって利用されますので、プログラマーと相談して利用してください。

 

文字の設定

文字列に使用する文字の設定を行います。

サイズ絶対値 :

文字のサイズ絶対値が表示されます。

ここでの入力は、サイズスケールの値にも影響を与えます。

初期値 :

使用する文字の初期値が表示されます。

サイズスケール :

文字のサイズスケール ( 初期値を 1 としてのスケール ) 値が表示されます。

ここでの入力は、サイズ絶対値にも影響を与えます。

・縦横比の連動ボタン「連動中」

クリックする度に、「連動」と「非連動」が切り替わります。


サイズスケールの縦横比の数値が、「 X 」「 Y 」双方連動します。

「 X 」、「 Y 」、どちらのサイズスケールに数値を入力しても、連動して同じ数値となります。

・縦横比の連動ボタン「非連動」

クリックする度に、「連動」と「非連動」が切り替わります。


サイズスケールの縦横比の数値が、「 X 」「 Y 」で非連動となります。

「 X 」、「 Y 」、どちらのサイズスケールにも別の数値を入力できます。

文字間隔 :

文字と文字の間隔を設定します。
斜体傾き ( 角度指定 ) :

文字の傾きを指定します。

「 -60.00 ～ 60.00 」の範囲で設定できます。

ここでの入力は、比率の値に影響を与えます。
斜体傾き ( 比率 ) :

文字に傾きを与えることで変更された、比率を表示します。

「斜体傾き ( 角度指定 ) 」で設定した数値と連動しています。

文字列の位置

テキストボックスペインの文字列の位置について設定をします。

水平方向 :

文字列の水平方向の位置を指定します。

  • 左
  • 中央
  • 右

垂直方向 :

文字列の垂直方向の位置を指定します。

  • 上
  • 中央
  • 下

行そろえ :

文字列が 2 行以上になった場合、行をどの位置にそろえるか指定します。

  • 左
  • 中央
  • 右

中央揃えのにじみ防止機能を有効にする

この機能を有効にすると、中央揃えにした文字列が滲まないように、座標の小数点以下を切り上げて整数座標にするようになります。

このため文字が滲まなくなる代わりに、図のように文字列が中央からずれて描画される副作用があります。


(1) が有効、(2) が無効にした場合の本機能の描画結果です。



この機能は解像度が低かった過去のプラットフォームでは有用でしたが、解像度が高い最近のプラットフォームでは座標が小数になってもそれほど目立たなくなっています。

過去との互換性のために残している機能ですので、「通常は無効、滲みが気になるときに限り有効」にしてください。

初期状態では無効になっています。

行ごとに幅を指定

テキストボックスペインの幅を行単位で編集します。
16 行まで設定できます。

行ごとに幅を指定
有効
 機能の ON / OFF スイッチ チェックボックスにチェックを入れると、機能が有効となります。
サイズ X: テキストボックスペインの X の大きさ

テキストボックスペインの X 方向のサイズを表示します。
プロパティウインドウの「ペイン基本」で設定した X 方向のサイズです。

「行単位での幅調整」で幅の調整をすると、各行のサイズが変化するので参考数値となります。
A 左からの行の移動設定 左から右方向に px 単位で行の幅を設定します。
B 幅を指定した結果の行の幅表示

「行ごとに幅を指定」を実行して、各行の幅が変化した結果を表示します。

表示される数値は、各行の移動設定の数値入力と連動します。
C 右からの行の移動設定 右から左方向に px 単位で行の幅を設定します。

テキストカラー

文字列に色を付けることができます。

上端 :

文字の上端部分に色を設定します。

下端 :

文字の下端部分に色を設定します。

図からも色を設定できます。

図の色指定をしたい部分をマウス左クリックすると、カラーピッカーウインドウが開きます。

  • T : 文字の上端部分に色を設定します。
  • ALL : 文字の全体に色を設定します。
  • B : 文字の下端部分に色を設定します。

文字列の長さを強制的に指定

文字列の長さを強制的に指定する設定をします。

指定する

チェックボックスにチェックを入れると、強制的に指定する設定が ON になります。

指定する文字列長 :

設定が ON になった場合、指定する文字列の長さを設定します。

影

文字に影を付けることができます。
文字に影を付ける場合、影の設定をします。

有効

チェックボックスにチェックを入れることで、影を有効にします。

オフセット :

影の位置設定をします。

スケール :

影の大きさを設定します。

斜体傾き ( 角度指定 ) :

影を傾ける角度を入力します。

「 -60.00 ～ 60.00 」の範囲で設定できます。
斜体傾き ( 比率 ) :

影に傾きを与えることで変更された、比率を表示します。

「斜体傾き ( 角度指定 ) 」で設定した数値と連動しています。
上端 : 影の上端部分に色を設定します。
下端 : 影の下端部分に色を設定します。

図からも色を設定できます。

図の色指定をしたい部分をマウス左クリックすると、カラーピッカーウインドウが開きます。

  • T : 影の上端部分に色を設定します。
  • ALL : 影の全体に色を設定します。
  • B : 影の下端部分に色を設定します。
黒カラー補間 : 影に使用されている文字の、黒い部分のカラー補間をします。
白カラー補間 : 影に使用されている文字の、白い部分のカラー補間をします。

文字毎の変換

文字毎の変換についての情報を入力します。

テキストボックスペイン中の、文字について1文字毎に平行移動や回転といった変換処理を加えることができます。
また、変換処理をアニメーションすることができます。
｢アーチ状に文字を配置する｣、｢文字を波打たせる｣といった特殊な表現が簡単に実現できます。

有効

チェックボックスにチェックを入れることで、有効にします。
オフセット/時間幅アニメーションの簡単設定
文字毎の変換設定を簡単にできる機能です。
「 有効」にチェックを入れると、ボタンが押せるようになります。
ボタンを押すと「文字毎の変換のオフセット/時間幅アニメーションの簡単設定」ウインドウが開きます。

オフセット時間 :

文字毎のアニメーションを再生するにあたって、「文字が参照するアニメーション」を作成した、時間枠の設定です。

「 00 フレーム ～ 設定フレームまで」


ここで設定した、アニメーションの時間枠が「オフセット」となります。

「オフセット」は、カーブエディタの「文字毎の変換：時間オフセット」にて、再生の速さやタイミング等の動きを指定することができます。

参考：

オフセットとしたアニメーションは、カーブエディタで「文字毎の変換：時間オフセット」カーブを編集することで、アニメーション再生の早さや再生タイミングを自由に設定することができます。

時間幅 :

個々の文字の動きに時間差を与えます。
アニメーションが個々の文字に影響を与えるタイミングを設定できます。

参考：

時間幅の数値に -60 と入力した場合、その影響により全ての文字のアニメーションが終了するのに 60 フレーム多く時間がかかります。

最後に影響を受ける文字のアニメーションが終了する時間が 60 フレーム遅くなります。

影響を受ける文字が多かった場合は、最後の文字の動きから逆算し、均等に時間幅が設定されます。
ループ :

ループの設定をします。

  • ワンタイム
  • ループ
原点 ( 垂直方向 ) : 文字ごとの垂直方向の原点を設定します。
文字をアニメーションさせるさいの原点になります。
  • 中央
  • 下
高さのオフセット 文字を動かす時の、原点の高さのオフセットを数値入力します。
「原点 ( 垂直方向 ) 」で指定した原点の位置からのオフセットです。
文字幅に応じて分割する

通常文字幅に関係なく等幅に配置されますが、チェックを入れると文字幅に応じて文字間隔を調整します。
文字毎に文字幅が異なるフォントが並んだとき、幅の狭い文字は文字と文字の間隔を詰めることで綺麗に見えます。

 影のアルファを文字のアルファに自動追従させる

 文字に「影」の機能を利用したさいに、チェックを入れると文字のアルファ値に追従できます。
文字のアルファ値をアニメーションさせて消す場合など、「影」も違和感なく消すことができます。
テキストを円形に配置する

テキストを円弧に沿って配置することができます。
正しく円形に配置する場合は「変換の原点をテキストボックスの中心にする」のチェックを入れてください。

変換の原点をテキストボックスの中心にする

通常文字毎の中心が原点ですが、チェックを入れると字毎の原点をテキストボックスの中心位置 1 箇所に設定します。

アニメーション開始位置を文字毎の原点から開始したい場合はチェックをオフにします。
