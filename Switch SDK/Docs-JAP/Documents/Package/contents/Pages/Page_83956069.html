GFX ライブラリーの初期化と終了

概要

GFX ライブラリーにおける他のあらゆる機能を使う前に、まず GFX ライブラリーを初期化する必要があります。
GFX ライブラリーを初期化したあと、もはや GFX ライブラリーの機能が必要なくなれば、GFX ライブラリーの終了を行う必要があります。GFX ライブラリーの終了処理後に、ふたたび GFX ライブラリーを初期化することなしに GFX ライブラリーの他の機能を使うことは、未定義の振る舞いを引き起こします。

GFX ライブラリーの初期化と終了の機能は、他のあらゆる GFX の機能と異なり、クラスではなく関数として提供されます。 
これらの関数はスレッドセーフではないことに注意してください。

初期化

GFX ライブラリーの初期化は nn::gfx::Initialize によって行われます。

終了

GFX ライブラリーの終了は nn::gfx::Finalize によって行われます。
