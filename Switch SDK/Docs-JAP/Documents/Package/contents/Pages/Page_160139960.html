アクセサリー

アクセサリー

ここでは、アクセサリの作成について説明します。

作成可能なアクセサリは、ヘアバンドや王冠といった髪に装着できるものとなります。髪以外のアクセサリについては対象外となります。

アクセサリ作成前の確認

すべての髪モデルにはアクセサリ装着用の Null が3箇所あります。アクセサリは set_front、set_side、set_top に装着することができます。

以下、それぞれの Null にアクセサリを装着した例を示します。すべての Null を同時に使うアクセサリを作成してもかまいません。

set_front（ヘアバンド） set_side（ネコの耳） set_top（王冠）

アクセサリの作成

ここでは、アクセサリの作成手順について説明します。

最初にデザイナーがアクセサリのモデルを作成し、それをプログラマが実装するという順序で説明します。

アクセサリのモデルデータの作成（デザイナー作業）

まず、3DCG ソフトウェアで坊主頭のサンプル Hair_Sample を表示してください。

坊主頭の Null に装着できるアクセサリを作成します。その際、アクセサリのサンプル HairAcce_Sample を参考にしてください。

HairAcce_Sample の詳細

  • Acce_front（ヘアバンド）
  • Acce_Side（ネコの耳）
  • Acce_top（王冠）

髪のモデルのサンプルは、坊主頭（Hair_Sample）のみとなります。そのほかの髪型のモデルは用意していません。そのため、アプリ開発側の判断で特定の髪型にあったアクセサリの形状を作成する場合は、各自で確認する必要があります。

アクセサリの表示（プログラマ作業）

似顔絵ライブラリを使用して Mii の顔モデルを表示させ、合わせてデザイナーが作成したアクセサリを表示してください。

顔モデルに装着する際には、アクセサリの回転と移動の情報を取得し、それを基準に装着します。nn::mii::CharModel::GetHeadPartsTransform() 関数を使用することで回転と移動の情報が取得できます。

Hair_Sample では Acce_Side の Null は右耳側のものしかありませんが、nn::mii::CharModel::GetHeadPartsTransform() 関数の引数に nn::mii::HeadPartsType_Left を指定することで左耳側の回転と移動の情報を取得することができます。

アクセサリ作成後の確認

実際のアプリで Mii のすべての髪型にアクセサリを装着し、確認を行ってください。

特定の髪型でアクセサリが沈んでいたり、髪とアクセサリが重なっていたりする可能性があります。その場合はアプリ開発側の判断でアクセサリの形状を修正してください。
