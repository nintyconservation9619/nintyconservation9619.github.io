変更の目的

NintendoSDK 0.7.0 にてグラフィックス系、オーディオ系のモジュールは、将来の任天堂プラットフォームに合わせて より最適な設計、より最適なパフォーマンス を実現するため依存する低レイヤライブラリの変更を行いました。具体的には、グラフィックス系モジュールは、nn::gfx ライブラリ、オーディオ系モジュールは、nn::audio ライブラリを利用した実装に変更され、名前空間も nw名前空間 から nn名前空間 に移行しました。

また、システム系のモジュールは NintendoSDK 0.7.0 リリース にて 廃止 されました。こちらも、将来の任天堂プラットフォームにとって より最適なモジュール として、NintendoSDK 内モジュール として用意されています。ただし、これまで提供してきた全ての機能が用意されるわけではありません。

本ドキュメントでは、NintendoSDK 0.7.0以降、変更後の NintendoWare を 新 NintendoWare と呼びます。

旧 NintendoWare ライブラリについて

旧 NintendoWare は 移行期間として NintendoSDK 0.7.0 ではリリース されましたが、NintendoSDK 0.8.0  にて 廃止 され、移行措置として グラフィックス系モジュール と オーディオ系モジュール に関しては、 NintendoSDK 0.8.0 にて  旧 NintendoWare 互換ライブラリ をリリースすることを告知していましたが、新 NitnendoWare のクオリティアップを優先するため、申し訳ありませんが リリースを延期させてください。延期後のスケジュールは未定です。

名前空間の移行

NintendoSDK 0.7.0 にて、グラフィックス系、オーディオ系モジュール は　nw 名前空間 から nn 名前空間 へ移行しました。
また、名前空間の移行に伴い、名前空間の名称の見直しや統合も行いました。

  　　新 NintendoWare　　 旧 NintedoWare 備考
グラフィック系モジュール nn::g3d nw::g3d  
nn::g3d::viewer nw::g3d::edit  
nn::ui2d nw::lyt / nw::ctrl nw::lyt / nw::ctrl は、nn::ui2d に統合されます。
nn::ui2d::viewer nw::vwrlyt  
nn::font nw::sys::font / nw::scfont nw::sys::font / nw::scfont は、nn::font に統合されます。
nn::fontll scffnd  
nn::vfx nw::eft  
nn::vfx::viewer nw::eftvw  
オーディオ系モジュール nn::atk nw::snd  
nn::atk::viewer nw::snd::edit / nw::snd::ctrl nw::snd::edit / nw::snd::ctrl は、nn::atk::viewer に統合されます。

 

システム系のモジュールの廃止について

廃止 されたシステム系モジュール ( nw::sys / dev / math / util ) を利用しているユーザーは、NintendoSDK モジュールで用意される代替モジュールへ NintendoSDK 0.8.0 リリースまでに乗り換えていただく必要があります。

主な代替モジュールは以下の通りです。

　　　　 廃止されるシステム系モジュール 代替モジュール ヘッダ
マトリクス nw::math::MTX34 / nw::math::MTX44 など nn::util::Matrix4x3f / nn::util::Matrix4x4f など  nn/util/util_MathTypes.h
ベクトル nw::math::VEC3 / nw::math::VEC4 など nn::util::Vector3f / nn::util::Vector4f など nn/util/util_MathTypes.h
数学関数 nw::math::SinRad など nn::util::SinCosEst　など nn/util/util_Arithmetic.h

 nw::dev::FileDeviceManager など nn::fs モジュール nn/fs.h
