カスタムシェーダサンプルの追加

エフェクトに法線マップ、ライトマップを反映させるカスタムシェーダサンプルが追加されました！

使用例

プリミティブパーティクルを使用したエフェクトでは、法線マップやライトマップの反映状況を EffectViewer で確認できます。
「 オプション」設定の【プロジェクト設定】にサンプルプロジェクトファイル「ProjConfig.xml」を設定することで、サンプルとして用意されたカスタムシェーダが利用できます。
以下はサンプルの場所です。

\Samples\Sources\Applications\VfxCustomShader\Resources\

以下はカスタムシェーダサンプルの利用方法です。

  1.  EffectMaker メインメニュー「ツール」から「オプション」を選択し、オプション設定ダイアログを開きます。
  2. 【プロジェクト設定】を選択します。
  3. 「 プロジェクト設定ファイルパス」にサンプルプロジェクトファイル ProjConfig.xml を登録します。
  4. OK ボタンでオプションの設定内容を確定します。
  5. 「 Eset」内のエフェクトファイル「 CustomShader.eset」を EffectMaker に読み込みます。
  6. ワークスペースの「 Emitter1」を選択します。
  7. 「 Cstmシェーダタブ」を選択します。
  8. 「 カスタムシェーダ設定」のプリセットから「Custom Shader Sample」を選択すると、サンプルの設定項目が表示されます。
  9. メニューバー「 ビューアを起動」ボタンをマウス左ボタンでクリックすると、ビューアが起動します。
  10. 起動した「 EffectViewer」を確認すると、以下のようにサンプルの内容が確認できます。

以下はサンプルで用意されたシェーダの説明です。

A カラーの変更 「 パラメータ」「 カラー」で設定した RGB 値をエミッタに乗算します。
B 描画位置の変更 「 パラメータ」「 加算する位置」の XYZ 値をエミッタ位置に加算することでエフェクトの描画位置を調整します。
C ライティング シンプルなライティング処理を適用します。
D 法線カラー ビューアでパーティクルの法線を色として確認できます。
E 法線マップ

法線マップを適用します。法線マップ用テクスチャは「 テクスチャ」タブの「 テクスチャ1」に登録してください。
F ライトマップ 「 パラメータ」「 IBL カラー0」空のカラー、「 IBL カラー1」を地面のカラーとしたシンプルなライトマップを適用します。

毎フレーム継承

親エミッタのアルファ値をチャイルドエミッタのアルファに継承する機能「毎フレーム継承」が追加されました！
この機能をON に設定した場合、パーティクルは常に親エミッタのアルファ値を自身のアルファ値に乗算します。

使用例

B 継承設定 ON 毎フレーム継承 OFF  ● は放出時の 親エミッタ ● のアルファ値を継承します。

エフェクトブラウザ対応フィルの追加

「 ワークスペースファイル（.work）」が「 エフェクトブラウザ」に対応しました。

エフェクトブラウザでは以下のように表示されます。
エフェクトブラウザで「  ラベルカラー」や「 コメント」が設定できます。

使用例

ワークスペース上の作業状態は、ワークスペースファイル（ .work）として保存できます。
保存したワークスペースファイル（.work）を開くことで、保存した時の状態へ復帰させることができます。

「 エフェクトブラウザ」「お気に入り」タブにはサンプルとしてお気に入り「 VfxSample」があります。
この中にある「 Afterburner.work」でワークスペースファイルの使い方を説明します。
  1. 「 エフェクトブラウザ」「お気に入り」タブを選択します。
  2. ナビゲーションペインのお気に入り「 VfxSample」を選択します。
  3. ファイルリストに表示された「 Afterburner」を選択した状態で、ツールバーの「 エクスポート」をマウス左ボタンクリックします。
  4. 表示を「 プロパティウィンドウ」に切り替えます。
  5. ワークスペースに「 Afterburner.eset」と関連するファイル「 Afterburner」と「 Afterburner」が一発で読み込まれます。
  6. メニューバー「 ビューアを起動」ボタンをマウス左ボタンでクリックすると、ビューアが起動します。
  7. 起動した「 EffectViewer」を確認すると、以下のように「 ワークスペースファイル」に保存されたモデルとエフェクトの連携状態を確認できます。
