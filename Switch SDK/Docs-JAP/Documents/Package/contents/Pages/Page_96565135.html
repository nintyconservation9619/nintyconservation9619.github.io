影響の強さ（±）

ここではパーティクルがカールノイズの影響を受ける強さを設定する機能「影響の強さ（±）」について説明します。

概要

「影響の強さ（±）」は、パーティクルがカールノイズの影響を受ける強さを XYZ 軸ごとに設定できる機能です。
パーティクルの動きに作用させたい方向を決めることができます。
以下は「 設定項目」の概要です。

影響の強さ（±）

パーティクルがカールノイズの影響を受ける強さの値を「 ダイヤル」や「 テキストボックス」により設定します。
値が大きいほどパーティクルの速度に影響を与えます。値をマイナスにすると逆方向に影響を与えます。値が 0 の場合は影響しません。
