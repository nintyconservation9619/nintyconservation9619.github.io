マルチエクスポートプラグインの追加オプション編集ウィンドウ

マルチエクスポートプラグインの追加オプションを UI から編集できるようになりました。

参考：

出力アイテム入力ウィンドウの fdes File が空欄の場合、シーンにエクスポートプラグインのオプション設定が保存されていないと追加オプション編集ウィンドウが表示されません（エラーになります）。
「Nintendo > Export with Setting...」でオプションウィンドウを表示して、左上のメニュー「編集 > 設定の保存（Edit > Save Settings）」を選択するとオプション設定が保存されます。


 

関連リンク　詳細は マルチエクスポートプラグインの追加オプション編集ウィンドウ をご覧ください。
