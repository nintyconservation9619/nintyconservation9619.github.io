機能説明

 EffectMaker の機能について、設定例を交えて説明します。

エミッタセットノード 複数のエミッタやプレビューをワークスペース上でひとつにまとめることができる「エミッタセットノード」について説明します。

エミッタノード

パーティクルの生成と放出時の設定ができる「エミッタノード」について説明します。
パーティクルの挙動計算を CPU で行うか、GPU 、GPU+SO で行うかでアイコンの表示が異なります。
フィールドノード エミッタから放出されたパーティクルの速度に変化を与える「フィールドノード」について説明します。
エミッタプラグインノード 既存のパーティクルシステムに対する追加機能群「エミッタプラグインノード」について説明します。
カスタムアクションノード エミッタ単位でユーザー独自の情報が付加できる「カスタムアクションノード」について説明します。
エミッタ拡張パラメータノード ゲームアプリで利用できるデータが設定できる「エミッタ拡張パラメータ」について説明します。
プレビューノード エミッタセットに対してゲームアプリケーション側で 制御可能な内容をビューアで確認できる「プレビューノード」について説明します。
ビューアノード ビューアの機能や表示を制御する「ビューアノード」について説明します。
モデル設定ノード EffectMaker にモデルやアニメーションを登録できる「モデルノード」について説明します。
オプション EffectMaker やビューアの使用環境を設定できる「オプション」の機能について説明します。
