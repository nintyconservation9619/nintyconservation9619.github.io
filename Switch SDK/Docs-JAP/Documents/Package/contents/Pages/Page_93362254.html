パーティクルソート

ここでは放出されたパーティクルごとの並び順を決める機能「パーティクルソート」について説明します。

概要

「パーティクルソート」は、放出されたパーティクルごとの並べ方を決める機能です。
並べ方を変えることによって、放出されたパーティクルごとの描画順番を変更出来ます。
以下は「 設定項目」の概要です。

パーティクルソート

パーティクル同士の描画順序を以下のプリセットから選択します。

ソートしない
ソート処理を行わない為、最も負荷のかからない設定です。
描画順が正しくない状態でも表現上問題のないエフェクトで使用します。
昇順 後から放出されたパーティクルが手前に描画されるようにソートします。
降順 最初に放出されたパーティクルが手前に描画されるようにソートします。
Zソート カメラからの距離に応じてパーティクルを奥から順に描画します。
参考：

GPU エミッタでは、パーティクルの発生位置をもとに Z ソートされます。
関連リンク GPU エミッタについては 「エフェクトのエミッタ」 を参照してください。

設定例

以下は「パーティクルソート」各プリセットの比較画像です。

A ソートしない ソート処理を行っていない為、描画順が正しくないときがあります。
B 昇順 後から放出されたパーティクルが手前に描画されています。
C 降順
 最初に放出されたパーティクルが手前に描画されています。
D Zソート
 カメラを回した場合、常にカメラから近いパーティクルが手前に描画されています。
