NintendoWare Audio ニュース

NintendoSDK 0.10.0 で オーディオモジュールに追加された機能を紹介します。

SoundMaker

 ストリームサウンドに Opus フォーマットのファイルを登録できるようになりました

OpusEncoder で事前にエンコードした Opus ファイルを SoundMaker のストリームサウンドリストにドラッグ&ドロップするなどして、Opus のストリームサウンドとして登録することができます。
ただし、現在は、シングルトラック、2ch までのサポートとなります。

詳しくは、「圧縮オーディオ」 を参照ください。

AtkPlayer 

エフェクトのプレビューができるようになりました

サウンドエフェクトを適用することや、それぞれのエフェクトのパラメータを調整してプレビューすることができる Effect パネルが AtkPlayer に追加されました。

詳しくは、「AtkPlayer」を参照ください。

波形のプレビューができるようになりました

再生中のサウンドの波形をモニタリングすることができる SoundOutput パネルが AtkPlayer に追加されました。

詳しくは、「AtkPlayer」を参照ください。
