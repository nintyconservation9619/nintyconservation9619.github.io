グレア/ブルームのチカラを借りる

発光表現にはグレア/ブルームを利用し、ブレンドタイプ「加算半透明」を使用した重ね書きによる発光表現は避けましょう。
以下のようにエミッタ単位で複製し、輝度をあげるような表現では、単純に２倍の塗面積が必要になります。

グレア/ブルーム表現が利用可能であれば、カラー値やカラースケール値に対して、1.0　以上の値を入力することで塗面積を節約しつつ、発行表現が可能になります。
