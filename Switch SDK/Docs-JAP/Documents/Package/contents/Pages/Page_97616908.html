開始前に強制計算

ここではエフェクト呼び出し時に、設定した時間分あらかじめ計算処理を回しておく機能「開始前に強制計算」について説明します。
雪などのエフェクトを最初から空間に満たしておきたい場合などに使用します。

概要

「開始前に強制計算」は、エフェクト呼び出し時に、設定した時間分あらかじめ計算処理を回すことができる機能です。
雪などのエフェクトを最初から空間に満たしておきたい場合などに使用します。
以下は「 設定項目」の概要です。

開始前に強制計算

事前に計算処理を回しておく時間をフレーム数で設定します。

以下は「開始前に強制計算」の値を変更したときの比較画像です。

A 0 エフェクトを通常どおり呼び出しています。
