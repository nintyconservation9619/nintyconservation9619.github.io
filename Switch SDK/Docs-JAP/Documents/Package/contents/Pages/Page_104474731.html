カメラ距離スケール制限（遠）

「 エミッタノード」→「 スケール」→「 その他スケール設定」に「 カメラ距離スケール制限（遠）」が追加されました！
カメラから離れていくパーティクルが必要以上に小さくならない様に制限をかける機能です。

例えばゲームのキャラクタ情報など、ゲームとして必要な情報がカメラ距離に関係なくプレイヤーから認識できる表示サイズにできます。

以下は「カメラ距離スケール制限（遠）」を「 チェックボックス」ON/OFF したときの比較画像です。
この機能を使用した場合、「1P」と書かれたパーティクルはカメラから設定値 120 離れた地点を境に一定の大きさを維持します。

「 エミッタノード」→「 コンバイナ」→「 パーティクルシェーダ」に「 カメラが遠いほどに強く歪む」が追加されました！
パーティクルがカメラから離れるほど、屈折のかかり方を強くできます。

以下は屈折係数が同じ設定で、「カメラが遠いほどに強く歪む」を「 チェックボックス」ON/OFF したときの比較画像です。
この機能を使用した場合、カメラからの距離に関係なく歪み方が一定に見えます。

エミッタ拡張パラメータ

ゲームアプリで利用できる固有のデータをエフェクトに付加できる機能「  エミッタ拡張パラメータ」が追加されました！

ゲームアプリ固有の機能として利用できる値を設定できます。また、その値に沿った形で EffectMaker に UI を付加できます。
ユーザーは UI で設定した値を編集、エフェクトデータとして保存できます。

「  エミッタ拡張パラメータ」は以下いずれかの操作を行います。

  • マウス右ボタンクリックで開くメニューから「  「エミッタ拡張パラメータ」を追加」を選択します。
  • ツールバーにある「  「エミッタ拡張パラメータ」を追加」ボタンを押します。

以下は「エミッタ拡張パラメータ」のプロパティ設定画面です。
EffectMaker で扱うことのできる UI を使った新たな 設定項目が追加されています。

以下は「エミッタ拡張パラメータ」で使用できる EffectMaker の基本 UI とその説明です。

A チェックボックス 2 つの選択肢からどちらか 1 つを選択する UI です。
UI を配置するスペース消費量が少ない利点があります。

2 つ以上の複数ある選択肢からどれか 1 つを選択する UI です。

「階層化の解除」で親子関係を解除する

ワークスペースで親子関係にあるエミッタの階層化を簡単に解除できるようになりました！
以下のように「 Emitter1」の階層下にあるチャイルドエミッタ「 Emitter3」を選択した状態から操作方法について説明します。

以下いずれかの操作を行います。

  • マウス右ボタンクリックで開くメニューから「  階層化の解除」を選択します。
  • ショートカットキー Ctrl + Shift + P を使用します。

階層上親子関係にあるエミッタ「 Emitter1」とその子階層にあたるチャイルドエミッタ「 Emitter3」の関係が解除されます。
以下はワークスペースのノード配置です。「 Emitter3」は「 Emitter1」と階層上同位に配置されます。

「モデル設定ノード」の対応

「モデル設定ノード」に登録したモデルデータが、ビューアで確認できるようになりました！
ビューアで、登録したモデルとエフェクトを連携させて表示することで、サイズや色合わせができます。

「  モデルノード」→「 基本設定」→「 モデル」→「 ファイル指定」に登録したモデルデータがビューアに表示されます。
また、「 キャラアニメ」→「 フォルダ指定」で登録したフォルダ以下のアニメーションを「 アニメーションファイル」から選択することで、ビューアのモデルに反映されるようになりました。

以下のように、ビューアではアニメーションしているモデルとエフェクトを連携させて確認できます。

モデル：人 、エフェクト：炎

モデル：魚 、エフェクト：泡
