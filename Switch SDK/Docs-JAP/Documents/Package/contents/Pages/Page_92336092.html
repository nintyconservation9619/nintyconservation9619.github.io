NintendoWare オーディオ開発環境 (AudioToolkit) とは？

nn::audio の上位レイヤーとして、SE、BGM などのゲームサウンドを扱うための開発環境です。

AudioToolkit は、次の要素から構成されています。

nn::atk

ゲーム上で SE、BGM などのサウンドを再生するクラスライブラリです。

nn::atk と nw::snd (※) の違いについては、NintendoWare オーディオ の変更点 を参照ください。


※nw::snd は、3DS, Wii U 世代の NintendoWare オーディオライブラリです。
SoundMaker ゲーム上で扱うサウンドデータをオーサリングする GUI ツールです。
AtkPlayer SoundMaker で編集したサウンドデータをプレビューするツールです。各種パラメータを調整しつつプレビューすることが可能です。
各種コマンドラインツール SoundMaker などで編集したサウンドリソースファイルを、ランタイムライブラリが扱えるバイナリリソースファイルに変換するコマンドラインツール群です。
