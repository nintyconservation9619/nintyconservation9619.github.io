データの利用

データの読み込み

データ配信キャッシュストレージに保存されたデータをアプリケーションが読み込むためのステップを説明します。

  1. nn::bcat::Initialize() を呼び出し、BCATライブラリを初期化します。
  2. nn::bcat::MountDeliveryCacheStorage() を呼び出し、本体システムに対してデータ配信キャッシュストレージの利用開始を宣言します。
  3. nn::bcat::EnumerateDeliveryCacheDirectory() を呼び出し、ディレクトリを列挙します。
    • この手順を省略して、あとの手順においてディレクトリの名前を直接指定することもできます。
  4. ディレクトリごとに nn::bcat::DeliveryCacheDirectory::Read() を呼び出し、ファイルを列挙します。
    • この手順を省略して、あとの手順においてファイルの名前を直接指定することもできます。
  5. nn::bcat::DeliveryCacheFile::Open() 、および  nn::bcat::DeliveryCacheFile::Read() を呼び出し、ファイルを読み込みます。
  6. nn::bcat::UnmountDeliveryCacheStorage() を呼び出し、本体システムに対してデータ配信キャッシュストレージの利用終了を宣言します。


BcatSimple サンプルデモもあわせて参照してください。

データの書き込み

データ配信キャッシュストレージにアプリケーションがデータを直接書き込むことはできません。
