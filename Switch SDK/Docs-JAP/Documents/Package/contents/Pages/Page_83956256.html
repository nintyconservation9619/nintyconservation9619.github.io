プレビュー

ビューアライブラリ を組み込んだアプリケーション（PC や開発実機）を使用したモデルやアニメーションのプレビューに関するメニューです。

参考：

シェイプアニメーションのプレビューやリアルタイム編集はビューアライブラリでは非対応です。

A

アプリケーションへの接続（接続中は切断）を行います。
B 自動接続を ON にしておくと、3DEditor を起動した際にアプリケーション ( PC または、開発実機 ) に自動的に接続します。
接続先は、「接続先を選択」で変更できます。
C

接続先を PC または、開発実機から選択して切り替えます。
※接続先を切り替える場合は、アプリケーションへの接続を一度切断する必要があります。
D

シェーダ最適化の有効 / 無効を切り替えます。
シェーダ最適化を有効にすると、各モデルで使用されているシェーダプログラムのみをもつ最適化されたシェーダをランタイムに送信します。
E 最適化シェーダの再生成を行います。
アプリケーションに接続されている場合に再生成が可能です。
