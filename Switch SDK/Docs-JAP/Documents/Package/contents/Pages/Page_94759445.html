振幅

ここでは揺らぎの大きさを設定する機能「振幅」について説明します。

概要

波形の設定項目は、「波形タイプ」で選択した波形を「正弦波」で設定できます。
以下は の概要です。
揺らぎの適用項目いずれかを ON にすると  グループ『揺らぎの適用』が表示されます。以下はグループ内の設定項目です。

振幅

揺らぎの大きさを設定します。以下の式で求めた最小値から最大値の間で揺らぎます。
最小値 : 揺らぎを適用したパラメータの値 - 「振幅」の値
最大値 : 揺らぎを適用したパラメータの値
