マテリアルプロパティウィンドウのシェーダーページに任意のテキストを挿入する機能の追加

シェーダーソースのアノテーションにテキストブロックを記述することで、下図のように指定位置に任意のテキストを表示することができるようになりました。

例：シェーダーアノテーションテに以下のキストブロックを追加 するとテキストが表示されます。 

// @@ textblock text="テキストが挿入されます" order="0" group="diffuse_group"

文字列型の描画情報に対してチェックボックスUIの対応

文字列型の描画情報はこれまではコンボボックスUIだけ対応していましたが、2値の文字列型の描画情報について、下図のようにチェックボックスを表示できるようになりました。

例：以下のように描画情報のシェーダーアノテーションのitemにcheckを指定するとチェックボックスが表示されます。3DEditor側でチェックボックスをON、OFFした場合、接続先アプリケーションには文字列で「true」か「false」が送信されます。 

// @@ renderinfo="checkbox_test" type="string" default="false" choice="false, true" optional="false" item="check" label="チェック状態でランタイム側にtrueかfalseの文字列が送られます。"

マルチエクスポートプラグインのコンテキストメニューカスタマイズ機能の追加

コンフィグファイルに以下を記述することで、マルチエクスポートプラグインの出力アイテムリストを右クリックしたときにユーザーカスタムメニューが表示されるようになりました。

multi_export_user_custom_list_menu0="メニュー名,プロシージャ名,タイプ,注釈"


例：出力アイテムリストを右クリックしたときに「My Export」が表示されます。

multi_export_user_custom_list_menu0="My Export,MyNwMultiExportCustomData,multi,Export custom data"

Cullボーン圧縮、Mergeボーン圧縮使用時にボーンビジビリティーアニメーションを維持できるよう仕様改善


Cullボーン圧縮、Mergeボーン圧縮使用時に圧縮不可に設定したボーンに対してボーンビジビリティーアニメーションを作成した場合、これまではボーンビジビリティーアニメーションを維持できませんでしたが、維持できるようにボーン圧縮の仕様を変更しました。具体的な圧縮仕様の説明はボーン圧縮を参照してください。

例：下図は Group にビジビリティーアニメーションを付けて圧縮不可にして出力した際のスケマティック表示です。
　　シェイプは Group の下になり、ビジビリティーアニメーションが維持されるようになりました。
