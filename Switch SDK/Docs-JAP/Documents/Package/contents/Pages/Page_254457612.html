係数アニメ

ここでは「カラーバッファ屈折」の屈折度合いと「テクスチャ歪み」の歪み度合いをアニメーションさせる機能「係数アニメ」について説明します。

概要

「係数アニメ」は、「カラーバッファ屈折」の屈折度合いと「テクスチャ歪み」の歪み度合いをアニメーションさせる機能です。
以下は「 設定項目」の概要です。

係数アニメ

「係数」に乗算する値を「 グラデーションエディタ」でアニメーションさせます。

関連リンク 以下は関連ページへのリンクです。

カラーバッファ屈折 テクスチャの RGB 値を利用して背景を屈折させる機能「カラーバッファ屈折」について説明します。
テクスチャ歪み テクスチャの RGB 値を利用してその他 テクスチャを歪める機能「カラーバッファ屈折」について説明します。
グラデーションエディタ 屈折度合いや歪み度合いをパーティクル時間によりアニメーションさせる UI「グラデーションエディタ」について説明します。
