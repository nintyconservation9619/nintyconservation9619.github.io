Use 3DEditor の廃止

Use 3DEditor が廃止になりました。

代替方法については NintendoWare 3D の NintendoSDK 移行ガイド を参照してください。

Maya エクスポートプラグインのオプションウィンドウにカスタム UI を作成できる機能を追加

Maya エクスポートプラグインのオプションウィンドウにユーザが任意の UI を作成できる機能を追加しました。

詳細については エクスポートプラグインの user_custom_ui の項目  を参照してください。

3DEditor

  • デバッグログウィンドウの表示/非表示を切り替えられる機能を追加
  • ツールバーにプラットフォームを切り替える機能を追加
  • 3DEditorBridge ツールを追加

デバッグログウィンドウの表示/非表示を切り替えられる機能を追加

デバッグログウィンドウの表示/非表示を切り替えます。

メインメニューの [ツール] → [デバッグログウィンドウを表示] で表示/非表示を操作します。

参考：

主に問題の調査で使うウィンドウになります。
特定の環境で必ず発生する不具合があった場合に表示しておくと便利です。

ツールバーにプラットフォーム切り替え機能を追加

3DEditor のツールバーに接続先アプリケーションのプラットフォームを選択する機能を追加しました。

参考：

プラットフォームを間違えていると、ビューアと接続できないなど問題が発生する場合がありますのでご注意ください。

3DEditorBridge ツールを追加

3DEditor を外部から操作することができるコマンドラインツール 3DEditorBridge を追加しました。

3DEditor とその他のツールの連携を行うために利用することができます。

詳細は 3DEditorBridge を参照してください。
