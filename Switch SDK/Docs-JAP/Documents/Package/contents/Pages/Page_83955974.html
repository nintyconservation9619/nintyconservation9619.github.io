アニメーション区間タグ機能を利用する

ここでは、アニメーション区間タグ機能の利用方法を説明します。

アニメーション区間タグ編集ウィンドウ

「アニメーション区間タグ編集」ウィンドウでは、アニメーション区間タグの登録、削除、編集を行うことができます。
 [ メニュー ] → [ ウィンドウ ] → [ アニメーション区間タグ編集 ] で以下ウィンドウが開きます。

  

区間タグの追加

以下の手順で「アニメーション区間タグ編集」ウィンドウに区間タグを追加します。

1

「追加ボタン」をクリックし、ダイアログを開き、新規に追加する区間タグの情報を設定します。

区間タグ名が「Tag_00」であることを確認します。
2   ここではデフォルト設定のまま「OK」ボタンをクリックし、ダイアログを閉じます。
3 「アニメーション区間タグ編集」ウィンドウで、「Tag_00」が追加されていることを確認します。

以下は「アニメーション区間タグ編集」ウィンドウの基本的な設定手順です。

1

区間タグ「Tag_00」を選択します。
2

アニメーションの再生区間として切り取りたいフレーム数を設定します。
 0（開始フレーム） ～ 1 （終了フレーム）に設定されていることを確認します。
3

「関連グループ指定」ボタンを押すと関連グループ ( ※ ) を設定するダイアログが表示されます。 
「グループ」の設定がない為「アニメーションさせる対象ペインが限定されていない」として「 マーク」が表示されています。
4

「対象ペインのキーから自動設定」ボタンを押します。
このボタンを押すことにより、対象ペインに設定されたアニメーションのみをバイナリ出力の対象に設定します。
ここではペインにアニメーションが設定されていない為、全て「チェックボックス Off」になります。
参考：

 ( ※ ) 関連グループを設定することによって、区間タグがあらわすアニメーションの対象ペインを限定することができます。
この機能によって、画面の特定部分だけが変化するようなアニメーションを作成することができます。
 この機能を反映したアニメーション再生を LayoutViewer で確認できます。

アニメーションのモード切り替え

モードを切り替えて、好みのアニメーション作成方法を選ぶことができます。
また、作業途中のデータも、モードを切り替えて編集 ( ※ ) することができます。

「アニメーション区間タグ編集ウィンドウ」の上部に以下のような「モード切り替えボタン」が配置されています。

 結合モードボタン 結合モードを示しています。
分割モードボタン 分割モードを示しています。

「結合モード」から、「分割モード」に切り替えると、アニメーションは「区間タグ」を参照して分割されます。
「分割モード」から、「結合モード」に切り替えると、アニメーションは「区間タグ」を参照して結合されます。

参考：

※ 結合操作を行う際、区間範囲外にキーが追加される場合があり、意図しない結果を生み出す可能性があります。
そのため、基本的には「分割モード」「結合モード」のどちらかに固定した状態で作業していただくことを推奨します。

分割モードでのアニメーション作成

分割モードを使用してのアニメーション作成について説明します。

編集したいペインを選択状態にして「 アニメーション区間タグ編集ウインドウ」から、編集したい範囲を示した「区間タグ」を選択すると、「 カーブエディタウインドウ」に「区間タグの範囲のアニメーション」が表示されます。
「分割モード」では、区間タグごとにアニメーションカーブを編集します。
区間タグの選択を切り替えることで個別にアニメーションカーブを作成することができます。

分割モードで作成するアニメーションは、全て 0 フレームから始まります。

作成したアニメーションの拡縮機能

作成したアニメーションの拡縮機能は、分割モード専用の機能です。
キーの間隔比率を維持したまま区間の長さを変更できます。
「アニメーション区間タグ編集」ウィンドウの「フレーム区間」にあるボタンを「 OFF 」「小数点精度」「整数精度」から切り替えます。
以下はボタンの説明です。

 OFF 「作成したアニメーションの拡縮機能」を無効とします。
小数点精度 「作成したアニメーションの拡縮機能」を使用した「拡大 / 縮小」後の結果が小数点単位で設定されます。
整数精度

「作成したアニメーションの拡縮機能」を使用した「拡大 / 縮小」後の結果が整数値に四捨五入されます。

( 四捨五入した結果、キーの数が減る場合があります )

比率連動を有効にした状態で区間の長さを変更すると、キーの間隔比率を維持したままで区間の長さが変化します。

区間幅変更前 ( 区間幅 100 ) 比率連動を有効にした状態で区間幅を 100 から 50 にした場合
