開発環境でのニンテンドーアカウント作成手順

はじめに

ここでは、開発環境（dd1）でのニンテンドーアカウントの作成手順を案内しています。

ニンテンドーアカウントの作成

ニンテンドーアカウントログインページへのアクセス

ニンテンドーアカウントを新しく作成するには、PC等のブラウザーでニンテンドーアカウントログインページへアクセスしてください。
ニンテンドーアカウントログインページのURLは、別途案内します。不明な場合は任天堂の窓口まで問い合わせてください。NXからニンテンドーアカウントを作成することはできません。

このページにはBasic認証がかけられています。 ログインIDおよびパスワードは別途案内します。不明な場合は任天堂の窓口まで問い合わせてください。
なお、NXからニンテンドーアカウントにログインする際も、同じようにBasic認証がかけられていますので、同じログインIDとパスワードを利用してください。

以下のページが表示されたら、「新規作成」ボタンを押し、ニンテンドーアカウントの作成ページに進んでください。なお、画面は実物とは一部異なる場合があります。

 

ニンテンドーアカウントの情報入力

下記の説明を読んだうえで、ニンテンドーアカウントの作成に必要な情報を入力し、「作成する」ボタンを押してください。

 

  説明
ほかのアカウントと連携して作成

3DSやWii Uの開発機から作成したニンテンドーネットワークIDと連携することでもニンテンドーアカウントを作成することができます。

開発環境では、Facebook、Google+、Twitterと連携することはできません。詳細は後述の「ほかのアカウントとの連携」を参照してください。
ニックネーム 任意のニックネームを入力してください。
メールアドレス

ニンテンドーアカウントに登録するメールアドレスを入力してください。

他のニンテンドーアカウントと同じメールアドレスを登録することはできません。

開発環境に登録したメールアドレスに対して、実際にメールを送信することはありません。

そのため、実際に受信可能なメールアドレスである必要はありませんが、他人のメールアドレスや適当なメールアドレスを登録すると混乱を招くため、以下のように自身のメールアドレスのユーザーネームの末尾に "+" を付けた拡張メールアドレスの形式で入力してください。

（例）

自身のメールアドレス : username@exmaple.com

登録するメールアドレス : username+0001@example.com
パスワード

任意のパスワードを入力してください。パスワードを忘れてしまった場合、開発環境ではパスワードを再発行する手段がありません。

そのため入力したパスワードは忘れないようにしてください。

もし、パスワードを忘れてしまいログインできなくなった場合は、新しく別のニンテンドーアカウントを作成するか、任天堂の窓口に問い合わせてください。
生年月日

任意の生年月日を入力してください。

13歳未満となる生年月日を入力すると、子供用のニンテンドーアカウント作成ページに誘導されます。

その際は、画面に表示される案内に従ってニンテンドーアカウントの作成を進めてください。
性別 任意の性別を選んでください。
国 / 地域

任意の国・地域を選んでください。
タイムゾーン 任意のタイムゾーンを選んでください。 
キャンペーンメールの受信

どちらでも構いません。チェックを付けたとしてもキャンペーンメールが送られることはありません。 
利用規約およびプライバシーポリシーへの同意 チェックを付けてください。 

メールアドレスの疎通確認

通常、ニンテンドーアカウントの作成を完了させるためにはメールアドレスの疎通確認が必要ですが、開発環境では画面の左上に表示される「認証コード」を入力することで、ニンテンドーアカウントの作成を完了させることができます。
もし、画面の左上に「認証コード」が表示され無い場合は、メールアドレスがすでに他のニンテンドーアカウントに登録済みの可能性があります。その場合は、別のメールアドレスでニンテンドーアカウントの作成をやり直してください。

なお、メールアドレスの疎通確認に限らず、そのほかのメール受信が必要な手続きについても、画面左上に表示される情報をもとに進めることができます。 

ニンテンドーアカウントの管理

ニンテンドーアカウントのログインページからログインすると、以下のような管理ページにアクセスすることができます。ここでは、プロフィールの変更等を行うことができます。変更する際は、画面に表示される説明に従って操作してください。

 

 

ほかのアカウントとの連携

ニンテンドーアカウントは外部サービスのアカウントと連携させることができますが、開発環境で連携できるのは、ニンテンドーネットワークIDのみです。Facebook、Google+、Twitter と連携することはできません。
また、連携できるニンテンドーネットワークIDは、3DSやWii Uの開発機から作成した開発環境のニンテンドーネットワークIDのみです。本番環境のニンテンドーネットワークIDと連携することはできません。

ニンテンドーネットワークIDとの連携は、ニンテンドーアカウントの管理ページ内にある「ほかのアカウントとの連携」メニューなどから行うことができます。
連携にはニンテンドーネットワークIDでのログインが必要ですので、画面の説明に従ってログインしてください。

開発機上のユーザーアカウントとの連携

作成したアカウントを開発機上のユーザーアカウントと連携する方法については「ニンテンドーアカウントとの連携」ページを参照してください。

連携時の注意点

ユーザー情報の送信を許可したニンテンドーアカウントを開発機上のユーザーアカウントと連携した場合、製品機と同じようにユーザー情報の送信が有効になります。ユーザー情報にはシステム上で発生したエラーに関する情報などが含まれます。
