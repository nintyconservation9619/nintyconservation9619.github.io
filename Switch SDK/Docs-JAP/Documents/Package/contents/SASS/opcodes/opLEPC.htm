LEPC : Load Effective Program Counter

Format

SPA 5.0:
        {@{!}Pg}   LEPC   Rd   {&req_6}   {?sched}   ;

Description

LEPC loads the effective Program Counter of the current (LEPC) instruction into register Rd. The U32 Program Counter (PC) is in bytes, and is relative to the 40b virtual base address (PROGRAM_BASE) of all shader programs in this context.

Examples:

LEPC  R5;
